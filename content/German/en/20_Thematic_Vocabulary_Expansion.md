# 20. Thematic Vocabulary Expansion (Thematische Wortschatzerweiterung)

## Learning Objectives

- Build vocabulary across seven essential thematic domains: academics, politics, environment, technology, media, health, and economy
- Understand and apply German word-building strategies: compounding, prefixes, and suffixes
- Recognize how compound nouns are formed and determine their gender
- Use common prefixes (un-, ur-, miss-) and suffixes (-ung, -heit/-keit, -schaft, -lich, -bar, -ig, -isch) productively
- Convert between word classes: noun to verb, verb to noun, adjective to noun
- Apply nominalization (das Lesen, das Schreiben) in context

## 8. Word Building: Compound Nouns

German freely combines nouns (and other word classes) to create new words. This is one of the most productive features of the language.

### 8.1 Basic Structure

The **last** component determines the gender, plural form, and general meaning. Everything before it modifies or specifies.

```
Modifier(s) + Base Noun = Compound Noun
Gender = Gender of Base Noun
```

| Components | Compound | Gender | English |
|-----------|----------|--------|---------|
| Kranken + Haus | **Krankenhaus** | das (Haus) | hospital |
| Bundes + Regierung | **Bundesregierung** | die (Regierung) | federal government |
| Geschwindigkeit + s + Begrenzung | **Geschwindigkeitsbegrenzung** | die (Begrenzung) | speed limit |
| Wohnzimmer + Tür | **Wohnzimmertür** | die (Tür) | living room door |
| Geburts + Tag | **Geburtstag** | der (Tag) | birthday |
| Haupt + Bahnhof | **Hauptbahnhof** | der (Bahnhof) | main train station |
| Fahr + Rad | **Fahrrad** | das (Rad) | bicycle |

### 8.2 Connecting Elements (Fugenelemente)

Compound nouns often require a connecting element between parts:

| Fugenelement | Example | Components |
|-------------|---------|-----------|
| **-s-** | Arbeit**s**platz | Arbeit + Platz (workplace) |
| **-s-** | Geburt**s**tag | Geburt + Tag (birthday) |
| **-n-** | Blume**n**laden | Blume + Laden (flower shop) |
| **-en-** | Studenten**en**wohnheim | Student + Wohnheim (dormitory) |
| **-er-** | Kind**er**garten | Kind + Garten (kindergarten) |
| **-e-** | Hund**e**hütte | Hund + Hütte (doghouse) |
| (none) | Haustür | Haus + Tür (front door) |

> There are no strict rules for which connecting element to use. The choice is historically determined and must be learned with each compound.

### 8.3 Multi-Part Compounds

German compounds can chain indefinitely (though in practice, three or four parts are most common):

| Compound | Components | English |
|----------|-----------|---------|
| **Handschuh** | Hand + Schuh | glove (hand-shoe) |
| **Kühlschrank** | kühl + Schrank | refrigerator (cool-cabinet) |
| **Staubsauger** | Staub + Sauger | vacuum cleaner (dust-sucker) |
| **Flugzeugträger** | Flug + Zeug + Träger | aircraft carrier |
| **Lebensversicherungsgesellschaft** | Lebens + Versicherungs + Gesellschaft | life insurance company |
| **Donaudampfschifffahrtsgesellschaft** | Donau + Dampf + Schiff + Fahrt + s + Gesellschaft | Danube steamship company |

## 10. Word Building: Suffixes

Suffixes are powerful tools for deriving new words from existing ones. They often determine the word class and gender.

### 10.1 Noun Suffixes

| Suffix | Gender | Forms From | Example | English |
|--------|--------|-----------|---------|---------|
| **-ung** | die | verbs | die Wohn**ung** (← wohnen) | apartment |
| | | | die Erklär**ung** (← erklären) | explanation |
| | | | die Entscheid**ung** (← entscheiden) | decision |
| **-heit** | die | adjectives | die Frei**heit** (← frei) | freedom |
| | | | die Krank**heit** (← krank) | illness |
| | | | die Gesund**heit** (← gesund) | health |
| **-keit** | die | adj. ending in -ig, -lich, -bar, -sam | die Möglich**keit** (← möglich) | possibility |
| | | | die Freundlich**keit** (← freundlich) | friendliness |
| | | | die Einsamkeit (← einsam) | loneliness |
| **-schaft** | die | nouns, adjectives | die Freund**schaft** (← Freund) | friendship |
| | | | die Wissen**schaft** (← Wissen) | science |
| | | | die Land**schaft** (← Land) | landscape |
| | | | die Gesell**schaft** (← Geselle) | society |

> **Rule:** All nouns ending in **-ung, -heit, -keit, -schaft** are **feminine** (die). No exceptions.

## 12. Nominalization

Nominalization is the process of turning verbs, adjectives, or other parts of speech into nouns. In German, all nominalized words are capitalized and are usually neuter (das).

### 12.1 Infinitive Nominalization

Any German infinitive can be turned into a neuter noun by capitalizing it:

| Infinitive | Nominalized Form | Example Sentence |
|-----------|-----------------|-----------------|
| lesen | **das Lesen** | **Das Lesen** macht mir Spaß. (Reading is fun for me.) |
| schreiben | **das Schreiben** | **Das Schreiben** fällt ihm schwer. (Writing is difficult for him.) |
| schwimmen | **das Schwimmen** | **Das Schwimmen** ist gesund. (Swimming is healthy.) |
| reisen | **das Reisen** | **Das Reisen** bildet. (Traveling educates.) |
| kochen | **das Kochen** | Ich liebe **das Kochen**. (I love cooking.) |
| laufen | **das Laufen** | **Das Laufen** hält fit. (Running keeps you fit.) |

### 12.2 Nominalized Adjectives (Review)

Adjectives used as nouns take standard adjective endings (covered in Lesson 17):

| German | English |
|--------|---------|
| **das Gute** und **das Böse** | good and evil |
| **etwas Neues** | something new |
| **nichts Besonderes** | nothing special |
| **das Wichtigste** | the most important thing |
| **alles Gute!** | all the best! |

### 12.3 Nominalized Participles

| Participle | Nominalized | English |
|-----------|------------|---------|
| reisend (traveling) | **der/die Reisende** | the traveler |
| studierend (studying) | **der/die Studierende** | the student |
| angestellt (employed) | **der/die Angestellte** | the employee |
| erwachsen (grown-up) | **der/die Erwachsene** | the adult |
| vorsitzend (presiding) | **der/die Vorsitzende** | the chairperson |

### 12.4 Nominalization in Context

Nominalization is extremely common in formal and academic German:

| Verbal Style (less formal) | Nominalized Style (more formal) |
|---------------------------|-------------------------------|
| Wir müssen die Kosten **senken**. | **Die Senkung** der Kosten ist notwendig. |
| Man muss das Problem **lösen**. | **Die Lösung** des Problems ist dringend. |
| Die Bevölkerung **wächst**. | **Das Wachstum** der Bevölkerung nimmt zu. |
| Die Regierung **entscheidet** morgen. | **Die Entscheidung** der Regierung fällt morgen. |
