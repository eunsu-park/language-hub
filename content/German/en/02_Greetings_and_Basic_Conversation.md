## 1. Formal vs Informal Address: du and Sie

### 1.1 The Core Distinction

German has two ways to say "you" — and choosing the wrong one can be a serious social misstep. This distinction is far more important in German culture than in English.

```
┌─────────────────────────────────────────────────────────────────┐
│                 du vs Sie — Choosing Correctly                   │
│                                                                 │
│   du (informal, lowercase)           Sie (formal, CAPITALIZED)  │
│   ─────────────────────              ──────────────────────     │
│   Friends                            Strangers (adults)         │
│   Family members                     Colleagues (often)         │
│   Children (among themselves)        Officials, police          │
│   Fellow students                    Shopkeepers, waiters       │
│   Close colleagues                   Professors, doctors        │
│   Pets/animals                       Anyone you don't know well │
│   God (in prayer)                    Elderly people             │
│                                                                 │
│   ※ When in doubt, ALWAYS use Sie. It is never offensive       │
│     to be too formal. Being too informal can be rude.           │
│                                                                 │
│   ※ The shift from Sie to du is called "das Duzen."            │
│     The older or higher-ranking person offers it first:         │
│     "Wir können uns duzen." (We can use du with each other.)   │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Verb Forms with du and Sie

The pronoun changes the verb conjugation. Sie (formal) always uses the same form as the third person plural (sie = they):

| Pronoun | Meaning | Verb: kommen (to come) | Verb: heißen (to be called) |
|---------|---------|----------------------|---------------------------|
| du | you (informal, singular) | du kommst | du heißt |
| ihr | you (informal, plural) | ihr kommt | ihr heißt |
| Sie | you (formal, singular & plural) | Sie kommen | Sie heißen |

```
Informal:
  Woher kommst du?         (Where do you come from?)
  Wie heißt du?            (What's your name?)

Formal:
  Woher kommen Sie?        (Where do you come from?)
  Wie heißen Sie?          (What is your name?)
```

### 1.3 Modern Trends

```
Workplace: Many modern German companies use du among all employees
           (especially tech, startups, IKEA). Traditional companies
           and government offices still use Sie.

Internet:  du is standard on social media and forums.

Age:       Young people (under ~25) almost always use du with
           each other, even when meeting for the first time.

Rule of thumb: If someone is over 30 and you don't know them,
               start with Sie until they offer du.
```

## 2. Greetings by Time of Day

### 2.1 Standard Greetings

| German | Pronunciation | English | When to use |
|--------|--------------|---------|-------------|
| Guten Morgen | /ˈɡuːtən ˈmɔʁɡən/ | Good morning | Until ~10-11 AM |
| Guten Tag | /ˈɡuːtən taːk/ | Good day / Hello | All day (default formal) |
| Guten Abend | /ˈɡuːtən ˈaːbənt/ | Good evening | From ~6 PM |
| Gute Nacht | /ˈɡuːtə naxt/ | Good night | Only when going to bed |

### 2.2 Informal Greetings

| German | Pronunciation | English | Notes |
|--------|--------------|---------|-------|
| Hallo | /haˈloː/ | Hello / Hi | Universal informal greeting |
| Hi | /haɪ/ | Hi | Very casual, from English |
| Hey | /hɛɪ/ | Hey | Very casual |
| Na? | /naː/ | Hey? / Well? | Very casual, expects a response |
| Na, wie geht's? | /naː viː ɡeːts/ | Hey, how's it going? | Common casual opener |
| Was geht? | /vas ɡeːt/ | What's up? | Youth slang |

### 2.3 Casual Shortened Forms

In everyday speech, Germans often shorten greetings:

```
Guten Morgen → Morgen!      (just "Morning!")
Guten Tag    → Tag!         (just "Day!")
Guten Abend  → N'Abend!    (shortened "Evening!")
```

## 3. Farewells

### 3.1 Standard Farewells

| German | Pronunciation | English | Formality |
|--------|--------------|---------|-----------|
| Auf Wiedersehen | /aʊf ˈviːdɐˌzeːən/ | Goodbye (lit. "until seeing again") | Formal |
| Auf Wiederhören | /aʊf ˈviːdɐˌhøːʁən/ | Goodbye (on phone: "until hearing again") | Formal (phone) |
| Tschüss | /tʃʏs/ | Bye | Informal, very common |
| Tschüssi | /ˈtʃʏsi/ | Bye-bye | Informal, slightly playful |
| Bis bald | /bɪs balt/ | See you soon | Neutral |
| Bis später | /bɪs ˈʃpɛːtɐ/ | See you later | Neutral |
| Bis morgen | /bɪs ˈmɔʁɡən/ | See you tomorrow | Neutral |
| Bis dann | /bɪs dan/ | See you then / Until then | Neutral |
| Ciao | /tʃaʊ/ | Bye (from Italian) | Informal, common in southern Germany |
| Mach's gut | /maxs ɡuːt/ | Take care (lit. "do it well") | Informal |
| Schönen Tag noch | /ˈʃøːnən taːk nɔx/ | Have a nice day | Polite, common |
| Schönes Wochenende | /ˈʃøːnəs ˈvɔxənˌɛndə/ | Have a nice weekend | Polite |

### 3.2 Using Farewells in Context

```
Formal (at a store, office, with strangers):
  A: Auf Wiedersehen! Schönen Tag noch!
  B: Danke, gleichfalls! Auf Wiedersehen!
     (Thanks, likewise! Goodbye!)

Informal (with friends):
  A: Tschüss! Bis morgen!
  B: Tschüss! Mach's gut!

On the phone:
  A: Auf Wiederhören!
  B: Auf Wiederhören! Tschüss!
```

## 4. Self-Introduction

### 4.1 Saying Your Name

There are several ways to introduce yourself:

```
Formal:
  Mein Name ist Anna Schmidt.     (My name is Anna Schmidt.)
  Ich heiße Thomas Müller.        (I am called Thomas Muller.)
  Ich bin Frau/Herr Weber.        (I am Mrs./Mr. Weber.)
  Gestatten, Müller.              (Allow me — Muller.) [very formal]
  Darf ich mich vorstellen?       (May I introduce myself?) [very formal]
    Mein Name ist ...

Informal:
  Ich bin Anna.                   (I'm Anna.)
  Ich heiße Thomas.               (I'm called Thomas.)
  Hi, ich bin die Anna.           (Hi, I'm Anna.) [with article, casual]
```

### 4.2 Saying Where You Are From

```
Ich komme aus ...                 (I come from ...)
  Ich komme aus Deutschland.      (I come from Germany.)
  Ich komme aus den USA.          (I come from the USA.)
  Ich komme aus Südkorea.         (I come from South Korea.)
  Ich komme aus Japan.            (I come from Japan.)

Ich bin ...                       (I am ...)
  Ich bin Deutscher/Deutsche.     (I am German.) [m./f.]
  Ich bin Amerikaner/Amerikanerin. (I am American.) [m./f.]
  Ich bin Koreaner/Koreanerin.    (I am Korean.) [m./f.]
```

### 4.3 Saying Where You Live

```
Ich wohne in ...                  (I live in ...)
  Ich wohne in Berlin.           (I live in Berlin.)
  Ich wohne in München.          (I live in Munich.)
  Ich wohne in der Schweiz.      (I live in Switzerland.)
```

### 4.4 Saying What You Do

```
Ich bin ... (von Beruf).          (I am ... [by profession].)
  Ich bin Student/Studentin.      (I am a student.) [m./f.]
  Ich bin Lehrer/Lehrerin.        (I am a teacher.) [m./f.]
  Ich bin Ingenieur/Ingenieurin.  (I am an engineer.) [m./f.]
  Ich bin Arzt/Ärztin.           (I am a doctor.) [m./f.]
  Ich bin Programmierer/Programmiererin. (I am a programmer.) [m./f.]

Note: Unlike English, German does NOT use an article before professions:
  ✓ Ich bin Student.              (I am a student.)
  ✗ Ich bin ein Student.          (Incorrect — no "ein" needed)
```

### 4.5 A Complete Self-Introduction

```
Formal:
  Guten Tag. Mein Name ist Anna Schmidt.
  Ich komme aus Deutschland, aus München.
  Ich wohne jetzt in Berlin.
  Ich bin Ingenieurin von Beruf.
  Freut mich.

  (Good day. My name is Anna Schmidt.
   I come from Germany, from Munich.
   I live in Berlin now.
   I am an engineer by profession.
   Pleased to meet you.)

Informal:
  Hi! Ich bin Thomas.
  Ich komme aus Wien, aber ich wohne in Hamburg.
  Ich bin Student.

  (Hi! I'm Thomas.
   I come from Vienna, but I live in Hamburg.
   I'm a student.)
```

## 5. Asking and Answering Basic Questions

### 5.1 Essential Questions

| Question (formal) | Question (informal) | English |
|-------------------|---------------------|---------|
| Wie heißen Sie? | Wie heißt du? | What is your name? |
| Woher kommen Sie? | Woher kommst du? | Where do you come from? |
| Wo wohnen Sie? | Wo wohnst du? | Where do you live? |
| Was machen Sie beruflich? | Was machst du beruflich? | What do you do for work? |
| Wie geht es Ihnen? | Wie geht es dir? / Wie geht's? | How are you? |
| Sprechen Sie Englisch? | Sprichst du Englisch? | Do you speak English? |
| Wie alt sind Sie? | Wie alt bist du? | How old are you? |

### 5.2 Answering "Wie geht es Ihnen/dir?"

```
┌─────────────────────────────────────────────────────────────────┐
│   Positive responses:                                           │
│   Sehr gut, danke!          Very well, thank you!               │
│   Gut, danke.               Good, thanks.                       │
│   Mir geht es gut.          I'm doing well.                     │
│   Prima!                    Great!                               │
│   Super!                    Super!                               │
│   Ausgezeichnet!            Excellent!                           │
│                                                                 │
│   Neutral responses:                                            │
│   Es geht.                  It's okay. / So-so.                 │
│   Ganz gut.                 Pretty good.                        │
│   Nicht schlecht.            Not bad.                            │
│                                                                 │
│   Negative responses:                                           │
│   Nicht so gut.             Not so good.                        │
│   Schlecht.                 Bad.                                 │
│   Es geht mir nicht gut.   I'm not doing well.                  │
│                                                                 │
│   Always add: Und Ihnen? (formal) / Und dir? (informal)        │
│   = And you?                                                    │
└─────────────────────────────────────────────────────────────────┘
```

### 5.3 Sample Conversation — Formal

```
A: Guten Tag! Wie heißen Sie?
   (Good day! What is your name?)

B: Guten Tag! Mein Name ist Thomas Müller. Und Sie?
   (Good day! My name is Thomas Muller. And you?)

A: Ich heiße Anna Schmidt. Freut mich!
   (I am called Anna Schmidt. Pleased to meet me!)

B: Freut mich auch! Woher kommen Sie?
   (Pleased to meet you too! Where do you come from?)

A: Ich komme aus Berlin. Und Sie?
   (I come from Berlin. And you?)

B: Ich komme aus Wien, aus Österreich.
   (I come from Vienna, from Austria.)

A: Wie schön! Was machen Sie beruflich?
   (How nice! What do you do for work?)

B: Ich bin Lehrer. Und Sie?
   (I'm a teacher. And you?)

A: Ich bin Ärztin.
   (I'm a doctor.)
```

### 5.4 Sample Conversation — Informal

```
A: Hey! Ich bin Lisa. Wie heißt du?
   (Hey! I'm Lisa. What's your name?)

B: Hi! Ich bin Max. Woher kommst du?
   (Hi! I'm Max. Where do you come from?)

A: Aus Hamburg. Und du?
   (From Hamburg. And you?)

B: Ich komme aus München. Wie geht's?
   (I come from Munich. How's it going?)

A: Gut, danke! Und dir?
   (Good, thanks! And you?)

B: Auch gut, danke!
   (Also good, thanks!)
```

### 8.2 Special Notes

```
Important observations:

1. zwei vs zwo: On the phone, Germans often say "zwo" instead of
   "zwei" to avoid confusion with "drei." Both mean 2.

2. sechs drops the "s" before -zehn: sechzehn (not "sechszehn")

3. sieben drops the "en" before -zehn: siebzehn (not "siebenzehn")

4. eins vs ein: "eins" is used for counting; "ein/eine" is used
   before nouns: ein Buch (one/a book), eine Katze (one/a cat)

5. Numbers 13-19 follow the pattern: unit + zehn
   dreizehn = drei + zehn = "three-ten" = 13
```

## 9. Regional Greeting Variations

### 9.1 Northern Germany

```
Moin!          — Universal greeting (morning, noon, and night!)
Moin Moin!     — Extended version (some say this is "too much")
Tschüss!       — Standard informal farewell (originally northern)
```

### 9.2 Southern Germany (Bavaria, Baden-Württemberg)

```
Grüß Gott!     — "God greet [you]" (standard greeting, all day)
Servus!         — Hello / Goodbye (both! from Latin "at your service")
Pfüati / Pfiat di! — Bavarian farewell (from "behüte dich Gott")
Habe die Ehre!  — "Have the honor" (formal, old-fashioned)
```

### 9.3 Austria

```
Grüß Gott!     — Standard greeting (very common)
Servus!         — Hello/Goodbye (very common, informal)
Baba!           — Bye! (Austrian informal)
Wiederschaun!   — Shortened Auf Wiedersehen (Austrian pronunciation)
```

### 9.4 Switzerland (German-speaking)

```
Grüezi!        — Standard greeting (from "Grüß [Sie] Gott")
Grüezi mitenand! — Greeting to a group ("Grüezi together")
Hoi!           — Hi! (informal, very common)
Uf Widerluege! — Auf Wiedersehen (Swiss German)
Adieu / Ade    — Goodbye (French influence)
Tschau!        — Ciao (very common)
Sali!          — Hi! (Basel region)
```

### 9.5 Overview Table

| Region | Hello (formal) | Hello (informal) | Goodbye |
|--------|---------------|-----------------|---------|
| Northern Germany | Guten Tag | Moin! | Tschüss! |
| Southern Germany | Grüß Gott! | Servus! | Servus! / Pfüati! |
| Austria | Grüß Gott! | Servus! | Baba! / Servus! |
| Switzerland | Grüezi! | Hoi! / Sali! | Ade! / Tschau! |
| Everywhere | Hallo | Hi / Hey | Tschüss / Ciao |
