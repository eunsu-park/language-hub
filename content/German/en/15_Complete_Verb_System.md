## 1. Overview: German Verb Classification

German verbs can be classified in two ways: by **conjugation type** (how they form their past tenses) and by **function** (what role they play in a sentence).

```
+--------------------------------------------------------------------+
|                    GERMAN VERB SYSTEM                               |
+--------------------------------------------------------------------+
|                                                                    |
|  By conjugation type:                                              |
|  +-----------------+  +-----------------+  +-----------------+     |
|  | WEAK (schwach)  |  | STRONG (stark)  |  | MIXED           |    |
|  | Regular         |  | Irregular       |  | (gemischt)      |    |
|  |                 |  |                 |  |                 |    |
|  | Praet: -te      |  | Praet: vowel   |  | Vowel change    |    |
|  | PP: ge-...-t    |  |   change        |  | + weak endings  |    |
|  |                 |  | PP: ge-...-en   |  | PP: ge-...-t    |    |
|  | machen          |  | gehen           |  | denken          |    |
|  | machte          |  | ging            |  | dachte          |    |
|  | gemacht         |  | gegangen        |  | gedacht         |    |
|  +-----------------+  +-----------------+  +-----------------+     |
|                                                                    |
|  By function:                                                      |
|  +-----------------+  +-----------------+  +-----------------+     |
|  | AUXILIARY        |  | MODAL           |  | FULL VERBS      |    |
|  | sein, haben     |  | koennen,        |  | All others      |    |
|  | werden          |  | muessen,        |  |                 |    |
|  |                 |  | duerfen,        |  |                 |    |
|  | Form compound   |  | sollen,         |  |                 |    |
|  | tenses, passive |  | wollen, moegen  |  |                 |    |
|  +-----------------+  +-----------------+  +-----------------+     |
|                                                                    |
|  Special categories:                                               |
|  - Reflexive verbs (sich waschen)                                  |
|  - Separable prefix verbs (aufstehen)                              |
|  - Inseparable prefix verbs (verstehen)                            |
|  - Impersonal verbs (es regnet, es gibt)                           |
+--------------------------------------------------------------------+
```

## 2. Weak (Regular) Verbs

Weak verbs are the **largest and most predictable** verb class. The vast majority of German verbs (and all newly created verbs) are weak.

### Principal Parts Pattern

```
Infinitive  -->  Praeteritum  -->  Partizip II
machen      -->  machte        -->  gemacht
spielen     -->  spielte       -->  gespielt
lernen      -->  lernte        -->  gelernt
arbeiten    -->  arbeitete     -->  gearbeitet
```

### Full Conjugation: machen (to make/do)

| Tense | ich | du | er/sie/es | wir | ihr | sie/Sie |
|-------|-----|-----|-----------|-----|-----|---------|
| Praesens | mache | machst | macht | machen | macht | machen |
| Praeteritum | machte | machtest | machte | machten | machtet | machten |
| Perfekt | habe gemacht | hast gemacht | hat gemacht | haben gemacht | habt gemacht | haben gemacht |
| Plusquam. | hatte gemacht | hattest gemacht | hatte gemacht | hatten gemacht | hattet gemacht | hatten gemacht |
| Futur I | werde machen | wirst machen | wird machen | werden machen | werdet machen | werden machen |
| Futur II | werde gemacht haben | wirst gemacht haben | wird gemacht haben | werden gemacht haben | werdet gemacht haben | werden gemacht haben |

### Key Features of Weak Verbs

1. **No vowel change** in any tense
2. **-te** suffix in Praeteritum (added to the stem)
3. **ge-...-t** pattern in Partizip II
4. Stem ending in **-t, -d, -chn, -ffn, -gn** adds extra **-e-** before the ending:
   - arbeit**e**te (not *arbeittte*)
   - red**e**te
   - oeffn**e**te
   - zeichn**e**te

### Common Weak Verbs

| Infinitive | Praeteritum | Partizip II | Meaning |
|-----------|------------|-------------|---------|
| machen | machte | gemacht | to make, do |
| spielen | spielte | gespielt | to play |
| lernen | lernte | gelernt | to learn |
| kaufen | kaufte | gekauft | to buy |
| arbeiten | arbeitete | gearbeitet | to work |
| hoeren | hoerte | gehoert | to hear |
| sagen | sagte | gesagt | to say |
| fragen | fragte | gefragt | to ask |
| wohnen | wohnte | gewohnt | to live (reside) |
| brauchen | brauchte | gebraucht | to need |

## 3. Strong (Irregular) Verbs

Strong verbs change their **stem vowel** (Ablaut) in past tenses and use different endings than weak verbs.

### Principal Parts Pattern

```
Infinitive  -->  Praeteritum  -->  Partizip II
gehen       -->  ging          -->  gegangen
sprechen    -->  sprach        -->  gesprochen
schreiben   -->  schrieb       -->  geschrieben
trinken     -->  trank         -->  getrunken
```

### Key Features of Strong Verbs

1. **Stem vowel changes** between tenses (Ablaut)
2. **No -te suffix** in Praeteritum (just the changed stem, no ending for ich/er)
3. **ge-...-en** pattern in Partizip II (ending in **-en**, not **-t**)
4. Many have **present tense stem changes** in du/er forms

### Present Tense Stem Changes (du and er/sie/es only)

| Change | Infinitive | du | er/sie/es | Meaning |
|--------|-----------|-----|-----------|---------|
| **e -> i** | geben | g**i**bst | g**i**bt | to give |
| **e -> i** | sprechen | spr**i**chst | spr**i**cht | to speak |
| **e -> i** | helfen | h**i**lfst | h**i**lft | to help |
| **e -> i** | nehmen | n**i**mmst | n**i**mmt | to take |
| **e -> ie** | lesen | l**ie**st | l**ie**st | to read |
| **e -> ie** | sehen | s**ie**hst | s**ie**ht | to see |
| **e -> ie** | empfehlen | empf**ie**hlst | empf**ie**hlt | to recommend |
| **a -> ae** | fahren | f**ae**hrst | f**ae**hrt | to drive |
| **a -> ae** | schlafen | schl**ae**fst | schl**ae**ft | to sleep |
| **a -> ae** | tragen | tr**ae**gst | tr**ae**gt | to carry, wear |
| **a -> ae** | fallen | f**ae**llst | f**ae**llt | to fall |
| **au -> aeu** | laufen | l**aeu**fst | l**aeu**ft | to run |

> **Important**: These present tense changes occur ONLY in 2nd and 3rd person singular (du, er/sie/es). All other forms (ich, wir, ihr, sie/Sie) keep the infinitive stem.

## 4. Ablaut Series: Vowel Change Patterns

Strong verbs follow identifiable vowel-change patterns called **Ablaut series**. Learning these patterns helps you predict the forms of unfamiliar strong verbs.

### The Major Ablaut Series

| # | Pattern | Infinitive | Praeteritum | Partizip II | More Examples |
|---|---------|-----------|------------|-------------|---------------|
| 1 | ei -> ie -> ie | schreiben | schr**ie**b | geschr**ie**ben | bleiben, treiben, steigen, reiten (first group) |
| 2 | ei -> i -> i | greifen | gr**i**ff | gegr**i**ffen | schneiden, reiten, pfeifen, streiten |
| 3 | e -> a -> o | sprechen | spr**a**ch | gespr**o**chen | helfen, nehmen, treffen, sterben, werfen |
| 4 | i -> a -> u | trinken | tr**a**nk | getr**u**nken | singen, finden, binden, klingen, springen |
| 5 | i -> a -> o | schwimmen | schw**a**mm | geschw**o**mmen | beginnen, gewinnen, spinnen |
| 6 | ie -> o -> o | fliegen | fl**o**g | gefl**o**gen | biegen, ziehen, verlieren, fliessen, schiessen |
| 7 | a -> u -> a | fahren | f**u**hr | gef**a**hren | tragen, graben, schlagen, wachsen |
| 8 | a -> ie -> a | fallen | f**ie**l | gef**a**llen | halten, schlafen, lassen, raten, braten |
| 9 | e -> a -> e | geben | g**a**b | geg**e**ben | lesen, sehen, essen, vergessen, treten |
| 10 | (various) | gehen | g**i**ng | geg**a**ngen | (irregular — no clear series) |

### How to Use the Ablaut Series

When you encounter a new strong verb, try to match it to a series:

1. Look at the **infinitive vowel** (e, i, a, ei, ie, etc.)
2. Check if the verb follows a known pattern
3. The Partizip II vowel can often be predicted from the series

> **Example**: You see the verb *singen* (to sing). Infinitive vowel is **i**. Looking at Series 4 (i -> a -> u): sang, gesungen. Correct!

## 6. Mixed Verbs

Mixed verbs combine features of **both** strong and weak verbs: they have a **stem vowel change** (like strong verbs) but use **weak verb endings** (-te in Praeteritum, ge-...-t in Partizip II).

### The Mixed Verbs (Complete List)

| Infinitive | Praeteritum | Partizip II | Meaning |
|-----------|------------|-------------|---------|
| bringen | **brachte** | **gebracht** | to bring |
| denken | **dachte** | **gedacht** | to think |
| kennen | **kannte** | **gekannt** | to know (be acquainted with) |
| nennen | **nannte** | **genannt** | to name, call |
| rennen | **rannte** | **gerannt** | to run |
| senden | **sandte** | **gesandt** | to send* |
| wenden | **wandte** | **gewandt** | to turn* |
| wissen | **wusste** | **gewusst** | to know (a fact) |

> *senden and wenden also have regular weak forms (sendete/gesendet, wendete/gewendet) that are equally acceptable and increasingly common.

### Comparison: Weak vs. Strong vs. Mixed

| Feature | Weak | Strong | Mixed |
|---------|------|--------|-------|
| Vowel change? | No | Yes | Yes |
| Praeteritum ending | -te | no ending (ich/er) | -te |
| Partizip II ending | -t (gemacht) | -en (gegangen) | -t (gedacht) |
| Example | machen/machte/gemacht | gehen/ging/gegangen | denken/dachte/gedacht |

### Special Case: wissen

**wissen** has a unique present tense conjugation that resembles modal verbs:

| Person | Present |
|--------|---------|
| ich | **weiss** |
| du | **weisst** |
| er/sie/es | **weiss** |
| wir | **wissen** |
| ihr | **wisst** |
| sie/Sie | **wissen** |

> Note: ich weiss and er weiss have the same form (like modals), and the singular stem differs from the plural/infinitive stem.

## 7. All Six Tenses Reviewed

### Tense Overview Table

| Tense | Formation | Example (ich, fahren) | Primary Use |
|-------|----------|----------------------|-------------|
| **Praesens** | stem + ending | ich fahre | present, habitual, near future |
| **Praeteritum** | stem change or -te | ich fuhr | written past narrative; sein/haben/modals in speech |
| **Perfekt** | haben/sein + Partizip II | ich bin gefahren | spoken past (most common in conversation) |
| **Plusquamperfekt** | hatte/war + Partizip II | ich war gefahren | action before another past action ("had done") |
| **Futur I** | werden + Infinitive | ich werde fahren | future, probability, assumption |
| **Futur II** | werden + PP + haben/sein | ich werde gefahren sein | future perfect, assumption about completed action |

### Praesens (Present Tense)

Used for: current actions, habitual actions, general truths, and scheduled future events.

| Weak: machen | Strong: fahren |
|-------------|---------------|
| ich mache | ich fahre |
| du machst | du faehrst |
| er macht | er faehrt |
| wir machen | wir fahren |
| ihr macht | ihr fahrt |
| sie machen | sie fahren |

### Praeteritum (Simple Past)

Used primarily in **written narrative** and for **sein, haben, modal verbs** in speech.

| Weak: machen | Strong: fahren | Mixed: denken |
|-------------|---------------|--------------|
| ich machte | ich fuhr | ich dachte |
| du machtest | du fuhrst | du dachtest |
| er machte | er fuhr | er dachte |
| wir machten | wir fuhren | wir dachten |
| ihr machtet | ihr fuhrt | ihr dachtet |
| sie machten | sie fuhren | sie dachten |

> **Key difference**: Weak verbs add -te to the stem. Strong verbs change the vowel and add NO ending for ich/er. Mixed verbs change the vowel AND add -te.

### Perfekt (Present Perfect)

The **most common past tense in spoken German**. Formed with haben or sein + Partizip II.

- Ich **habe** einen Brief **geschrieben**. (I wrote a letter.)
- Er **ist** nach Berlin **gefahren**. (He drove to Berlin.)
- Wir **haben** das Problem **verstanden**. (We understood the problem.)

### Plusquamperfekt (Past Perfect)

Describes an action that happened **before** another past action. Uses hatte/war + Partizip II.

- **Nachdem** er **gegessen hatte**, ging er spazieren.
  (After he had eaten, he went for a walk.)
- Sie **war** schon **abgefahren**, als ich ankam.
  (She had already departed when I arrived.)

### Futur I (Future)

Formed with werden + Infinitive. Also used for **probability/assumption**.

- Ich **werde** morgen **kommen**. (I will come tomorrow.)
- Es **wird** wohl regnen. (It will probably rain. — assumption)
- Er **wird** jetzt schon zu Hause **sein**. (He is probably already home. — assumption)

> **Practical note**: In spoken German, the present tense + time word often replaces Futur I:
> Ich **komme** morgen. = Ich **werde** morgen **kommen**.

### Futur II (Future Perfect)

Formed with werden + Partizip II + haben/sein. Expresses completed future actions or assumptions about the past.

- Bis morgen **werde** ich das Buch **gelesen haben**. (By tomorrow I will have read the book.)
- Er **wird** das wohl **vergessen haben**. (He has probably forgotten that. — assumption)
- Sie **wird** schon **angekommen sein**. (She will have already arrived.)

## 8. Haben vs. Sein in Perfekt

The choice between **haben** and **sein** as the auxiliary in Perfekt (and Plusquamperfekt) follows clear rules:

### Use sein with:

1. **Verbs of movement from A to B** (change of location):
   - gehen, kommen, fahren, fliegen, laufen, reisen, fallen, steigen, schwimmen

2. **Verbs of change of state**:
   - werden (to become), sterben (to die), aufwachen (to wake up), einschlafen (to fall asleep), wachsen (to grow), entstehen (to come into being)

3. **sein** and **bleiben** themselves:
   - Ich **bin** muede **gewesen**. (I was tired.)
   - Er **ist** zu Hause **geblieben**. (He stayed home.)

4. **geschehen, passieren, gelingen, misslingen** (to happen, to succeed, to fail):
   - Was **ist** passiert? (What happened?)

### Use haben with:

1. **All transitive verbs** (verbs with a direct accusative object):
   - Ich **habe** das Buch **gelesen**. (I read the book.)
   - Er **hat** den Brief **geschrieben**. (He wrote the letter.)

2. **All reflexive verbs**:
   - Ich **habe** mich gefreut. (I was happy.)
   - Er **hat** sich gewaschen. (He washed himself.)

3. **Verbs expressing duration or position** (not movement A to B):
   - Ich **habe** drei Stunden **geschlafen**. (I slept for three hours.)
   - Sie **hat** am Tisch **gesessen**. (She sat at the table.)

4. **Impersonal verbs**:
   - Es **hat** geregnet. (It rained.)
   - Es **hat** geschneit. (It snowed.)

### Summary Decision Flowchart

```
Does the verb have an accusative object?
   YES --> HABEN
   NO  --> Does it express movement from A to B?
           YES --> SEIN (gehen, kommen, fahren...)
           NO  --> Does it express a change of state?
                   YES --> SEIN (werden, sterben, aufwachen...)
                   NO  --> Is it sein or bleiben?
                           YES --> SEIN
                           NO  --> HABEN
```

### Tricky Cases

| Verb | With haben | With sein |
|------|-----------|-----------|
| **fahren** | Ich **habe** das Auto **gefahren**. (I drove the car. — with object) | Ich **bin** nach Berlin **gefahren**. (I drove to Berlin. — movement A to B) |
| **schwimmen** | Er **hat** eine Stunde **geschwommen**. (He swam for an hour. — duration) | Er **ist** ans Ufer **geschwommen**. (He swam to the shore. — A to B) |
| **fliegen** | Der Pilot **hat** das Flugzeug **geflogen**. (The pilot flew the plane. — with object) | Wir **sind** nach Japan **geflogen**. (We flew to Japan. — A to B) |

## 9. Passive Voice: werden + Partizip II

The passive voice shifts the focus from the **doer** (agent) to the **action** or the **receiver** of the action.

### Formation: Vorgangspassiv (Process Passive)

```
werden (conjugated) + Partizip II
```

### Active vs. Passive

| Active | Passive |
|--------|---------|
| Der Lehrer **erklaert** die Lektion. | Die Lektion **wird** (vom Lehrer) **erklaert**. |
| (The teacher explains the lesson.) | (The lesson is explained [by the teacher].) |
| Die Studenten **lesen** das Buch. | Das Buch **wird** (von den Studenten) **gelesen**. |
| (The students read the book.) | (The book is being read [by the students].) |

### Passive in All Tenses

| Tense | Active (er liest das Buch) | Passive (das Buch...) |
|-------|---------------------------|----------------------|
| Praesens | Er liest das Buch. | Das Buch **wird** gelesen. |
| Praeteritum | Er las das Buch. | Das Buch **wurde** gelesen. |
| Perfekt | Er hat das Buch gelesen. | Das Buch **ist** gelesen **worden**. |
| Plusquamperfekt | Er hatte das Buch gelesen. | Das Buch **war** gelesen **worden**. |
| Futur I | Er wird das Buch lesen. | Das Buch **wird** gelesen **werden**. |

> **Note on Perfekt passive**: The Partizip II of werden in the passive is **worden** (not *geworden*):
> Das Buch **ist** gelesen **worden**. (NOT *geworden*)

### The Agent (by whom)

The agent (doer) in a passive sentence is introduced by **von** + dative:

- Das Haus **wird von dem Arbeiter** gebaut. (The house is being built by the worker.)
- Der Brief **wurde von meiner Mutter** geschrieben. (The letter was written by my mother.)

For instruments or means, use **durch** + accusative:

- Die Stadt **wurde durch ein Erdbeben** zerstoert. (The city was destroyed by an earthquake.)

### Impersonal Passive (es wird + Partizip II)

German can form a passive even from intransitive verbs using an impersonal construction:

- **Es wird** hier nicht **geraucht**. (There is no smoking here.)
- **Es wurde** viel **getanzt**. (There was a lot of dancing.)

### Zustandspassiv (State Passive): sein + Partizip II

The **Zustandspassiv** describes the **result** of a completed action (not the process):

| Vorgangspassiv (process) | Zustandspassiv (state/result) |
|--------------------------|-------------------------------|
| Die Tuer **wird** geoeffnet. (is being opened) | Die Tuer **ist** geoeffnet. (is open) |
| Das Auto **wird** repariert. (is being repaired) | Das Auto **ist** repariert. (is repaired) |

## 10. Konjunktiv II: Formation and Uses

The Konjunktiv II (subjunctive II) expresses **hypothetical, unreal, or polite** situations.

### Formation: Two Methods

**Method 1: From Praeteritum stem** (for sein, haben, modals, and some strong verbs)

Take the Praeteritum stem, add umlaut where possible, and add subjunctive endings (-e, -est, -e, -en, -et, -en):

| Infinitive | Praeteritum | Konjunktiv II |
|-----------|------------|---------------|
| sein | war | **waere** |
| haben | hatte | **haette** |
| koennen | konnte | **koennte** |
| muessen | musste | **muesste** |
| duerfen | durfte | **duerfte** |
| sollen | sollte | **sollte** (no umlaut) |
| wollen | wollte | **wollte** (no umlaut) |
| kommen | kam | **kaeme** |
| gehen | ging | **ginge** |
| wissen | wusste | **wuesste** |
| geben | gab | **gaebe** |
| fahren | fuhr | **fuehre** |

**Method 2: wuerde + Infinitive** (for most verbs in spoken German)

```
wuerde (conjugated) + Infinitive
```

| Person | wuerde + machen |
|--------|----------------|
| ich | wuerde machen |
| du | wuerdest machen |
| er/sie/es | wuerde machen |
| wir | wuerden machen |
| ihr | wuerdet machen |
| sie/Sie | wuerden machen |

> **When to use which**: Use the one-word form (waere, haette, koennte, etc.) for **sein, haben, modals, and common strong verbs**. Use **wuerde + infinitive** for **weak verbs** and any verb where the Konjunktiv II form sounds identical to the Praeteritum (e.g., machte = Praet. AND Konj. II — use *wuerde machen* to avoid ambiguity).

### Primary Uses of Konjunktiv II

**1. Unreal conditions (if...would)**

- **Wenn** ich reich **waere**, **wuerde** ich eine Weltreise **machen**.
  (If I were rich, I would take a world trip.)
- **Wenn** er Zeit **haette**, **kaeme** er mit.
  (If he had time, he would come along.)
- **Wenn** ich das **gewusst haette**, **haette** ich dir **geholfen**.
  (If I had known that, I would have helped you.)

**2. Polite requests and suggestions**

- **Koennten** Sie mir bitte helfen? (Could you please help me?)
- Ich **haette** gern einen Kaffee. (I would like a coffee.)
- **Wuerden** Sie bitte das Fenster oeffnen? (Would you please open the window?)
- Ich **wuerde** Ihnen empfehlen, frueh zu kommen. (I would recommend you come early.)

**3. Wishes**

- **Wenn** ich doch nur fliegen **koennte**! (If only I could fly!)
- Ich **wuenschte**, ich **waere** jetzt am Strand. (I wish I were at the beach now.)

**4. Konjunktiv II Past (haette/waere + Partizip II)**

For unreal situations in the past:

- Ich **haette** das nicht **gemacht**. (I would not have done that.)
- Er **waere** gern **gekommen**. (He would have liked to come.)
- Wenn ich das **gewusst haette**, **haette** ich anders **gehandelt**.
  (If I had known that, I would have acted differently.)

## 11. Verb Valency: Intransitive, Transitive, Ditransitive

**Verb valency** describes how many and what type of objects a verb requires.

### Intransitive Verbs (no accusative object)

| Verb | Example | Note |
|------|---------|------|
| schlafen | Er **schlaeft**. | No object possible |
| kommen | Sie **kommt** morgen. | No ACC object |
| sterben | Der Mann **ist gestorben**. | No object |
| lachen | Die Kinder **lachen**. | No object |
| arbeiten | Ich **arbeite** jeden Tag. | No ACC object |

> Some intransitive verbs take a **dative** object: helfen + DAT, danken + DAT, gefallen + DAT.
> Ich helfe **dem Mann**. (dative, not accusative)

### Transitive Verbs (one accusative object)

| Verb | Example | Object |
|------|---------|--------|
| sehen | Ich **sehe den** Mann. | den Mann (ACC) |
| lesen | Er **liest ein** Buch. | ein Buch (ACC) |
| kaufen | Sie **kauft einen** Kuchen. | einen Kuchen (ACC) |
| verstehen | Wir **verstehen die** Frage. | die Frage (ACC) |
| besuchen | Ich **besuche meinen** Freund. | meinen Freund (ACC) |

### Ditransitive Verbs (dative + accusative object)

| Verb | Example | DAT Object | ACC Object |
|------|---------|-----------|-----------|
| geben | Ich gebe **dem Mann** **das Buch**. | dem Mann | das Buch |
| schicken | Sie schickt **ihrer Mutter** **einen Brief**. | ihrer Mutter | einen Brief |
| zeigen | Er zeigt **uns** **den Weg**. | uns | den Weg |
| erklaeren | Sie erklaert **den Schuelern** **die Grammatik**. | den Schuelern | die Grammatik |
| schenken | Ich schenke **dir** **eine Blume**. | dir | eine Blume |
| empfehlen | Der Arzt empfiehlt **dem Patienten** **Ruhe**. | dem Patienten | Ruhe |

### Valency and Passive

Only **transitive** verbs (with an accusative object) can form a full personal passive. The accusative object becomes the subject of the passive sentence:

| Active (transitive) | Passive |
|--------------------|---------|
| Der Lehrer erklaert **die Lektion**. | **Die Lektion** wird erklaert. |
| Sie liest **das Buch**. | **Das Buch** wird gelesen. |

Intransitive verbs can only form the **impersonal passive**:

| Active (intransitive) | Impersonal Passive |
|----------------------|-------------------|
| Die Leute tanzen. | **Es wird** getanzt. |
| Man arbeitet hier. | Hier **wird** gearbeitet. |

## 12. Modal Verbs: Complete Review

### The Six Modal Verbs

| Modal | Meaning | Praesens (ich) | Praeteritum (ich) | Partizip II |
|-------|---------|---------------|-------------------|-------------|
| **koennen** | can, to be able to | kann | konnte | gekonnt |
| **muessen** | must, to have to | muss | musste | gemusst |
| **duerfen** | may, to be allowed to | darf | durfte | gedurft |
| **sollen** | should, to be supposed to | soll | sollte | gesollt |
| **wollen** | to want to | will | wollte | gewollt |
| **moegen** | to like | mag | mochte | gemocht |

### Present Tense Conjugation

All modals share a distinctive pattern: singular forms differ from plural, and 1st/3rd person singular are identical:

| Person | koennen | muessen | duerfen | sollen | wollen | moegen |
|--------|---------|---------|---------|--------|--------|--------|
| ich | kann | muss | darf | soll | will | mag |
| du | kannst | musst | darfst | sollst | willst | magst |
| er/sie/es | kann | muss | darf | soll | will | mag |
| wir | koennen | muessen | duerfen | sollen | wollen | moegen |
| ihr | koennt | muesst | duerft | sollt | wollt | moegt |
| sie/Sie | koennen | muessen | duerfen | sollen | wollen | moegen |

### Modal Sentence Structure

```
Subject + Modal (position 2) + ... + Infinitive (end)
```

- Ich **kann** gut schwimmen. (I can swim well.)
- Du **musst** morgen frueh aufstehen. (You must get up early tomorrow.)
- Wir **duerfen** hier nicht rauchen. (We may not smoke here.)

### Modals in Perfekt (Double Infinitive)

When used with another verb, modals use the **double infinitive** construction:

```
haben + ... + Infinitive + Modal-Infinitive
```

- Ich **habe** nicht kommen **koennen**. (I couldn't come.)
- Sie **hat** das nicht machen **wollen**. (She didn't want to do that.)

> **Practical tip**: Because the double infinitive sounds awkward, modals are almost always used in **Praeteritum** in spoken German:
> Ich **konnte** nicht kommen. (preferred over: Ich habe nicht kommen koennen.)

### moechten (would like)

**moechten** is the Konjunktiv II form of moegen, widely used as a polite "would like." It conjugates like a modal: ich/er = moechte, du = moechtest, wir/sie = moechten.

- Ich **moechte** einen Kaffee, bitte. (I would like a coffee, please.)
- **Moechten** Sie bestellen? (Would you like to order?)

## 13. Auxiliary Verbs: sein, haben, werden

### sein (to be)

| Tense | ich | du | er/sie/es | wir | ihr | sie/Sie |
|-------|-----|-----|-----------|-----|-----|---------|
| Praesens | bin | bist | ist | sind | seid | sind |
| Praeteritum | war | warst | war | waren | wart | waren |
| Konjunktiv II | waere | waerst | waere | waeren | waert | waeren |
| Partizip II | gewesen |

### haben (to have)

| Tense | ich | du | er/sie/es | wir | ihr | sie/Sie |
|-------|-----|-----|-----------|-----|-----|---------|
| Praesens | habe | hast | hat | haben | habt | haben |
| Praeteritum | hatte | hattest | hatte | hatten | hattet | hatten |
| Konjunktiv II | haette | haettest | haette | haetten | haettet | haetten |
| Partizip II | gehabt |

### werden (to become / auxiliary)

| Tense | ich | du | er/sie/es | wir | ihr | sie/Sie |
|-------|-----|-----|-----------|-----|-----|---------|
| Praesens | werde | wirst | wird | werden | werdet | werden |
| Praeteritum | wurde | wurdest | wurde | wurden | wurdet | wurden |
| Konjunktiv II | wuerde | wuerdest | wuerde | wuerden | wuerdet | wuerden |
| Partizip II | geworden (full verb) / worden (passive) |

### The Three Functions of werden

| Function | Structure | Example |
|----------|----------|---------|
| Full verb: "to become" | werden + adjective/noun | Er **wird** Arzt. |
| Future auxiliary | werden + Infinitive | Ich **werde** gehen. |
| Passive auxiliary | werden + Partizip II | Das Buch **wird** gelesen. |

## 14. Complete Verb Classification Table

```
+----------------------------------------------------------------+
|              GERMAN VERB CLASSIFICATION OVERVIEW                |
+----------------------------------------------------------------+
|                                                                |
|  WEAK VERBS (schwache Verben)                                  |
|  - Largest group; all new verbs are weak                       |
|  - Pattern: machen / machte / gemacht                          |
|  - No vowel change; -te Praeteritum; ge-...-t Partizip II     |
|                                                                |
|  STRONG VERBS (starke Verben)                                  |
|  - About 200 common verbs                                      |
|  - Pattern: gehen / ging / gegangen                            |
|  - Vowel change (Ablaut); no -te; ge-...-en Partizip II       |
|  - Many have present tense stem change (du/er)                 |
|                                                                |
|  MIXED VERBS (gemischte Verben)                                |
|  - Small group: denken, bringen, kennen, nennen, rennen,      |
|    senden, wenden, wissen                                      |
|  - Vowel change + weak endings: dachte / gedacht               |
|                                                                |
|  MODAL VERBS (Modalverben)                                     |
|  - koennen, muessen, duerfen, sollen, wollen, moegen          |
|  - ich/er forms identical; no ending in sg. present            |
|  - Praeteritum preferred over Perfekt in speech                |
|                                                                |
|  AUXILIARY VERBS (Hilfsverben)                                 |
|  - sein (Perfekt with movement/change of state)                |
|  - haben (Perfekt with most verbs)                             |
|  - werden (future, passive, Konjunktiv II with wuerde)        |
|                                                                |
+----------------------------------------------------------------+
```

## 15. Practice Exercises

### Exercise 1: Verb Classification

Classify each verb as weak (W), strong (S), or mixed (M). Give the Praeteritum and Partizip II.

1. lernen (to learn) -> Type: ________ | Praet.: ________ | PP: ________
2. sprechen (to speak) -> Type: ________ | Praet.: ________ | PP: ________
3. denken (to think) -> Type: ________ | Praet.: ________ | PP: ________
4. fahren (to drive) -> Type: ________ | Praet.: ________ | PP: ________
5. bringen (to bring) -> Type: ________ | Praet.: ________ | PP: ________
6. spielen (to play) -> Type: ________ | Praet.: ________ | PP: ________
7. schreiben (to write) -> Type: ________ | Praet.: ________ | PP: ________
8. kennen (to know sb.) -> Type: ________ | Praet.: ________ | PP: ________

<details>
<summary>Answers</summary>

1. **W** (weak) | lernte | gelernt (regular: no vowel change, -te, ge-...-t)
2. **S** (strong) | sprach | gesprochen (vowel change e->a->o, ge-...-en)
3. **M** (mixed) | dachte | gedacht (vowel change + weak endings -te, ge-...-t)
4. **S** (strong) | fuhr | gefahren (vowel change a->u->a, ge-...-en)
5. **M** (mixed) | brachte | gebracht (vowel change + weak endings)
6. **W** (weak) | spielte | gespielt (regular)
7. **S** (strong) | schrieb | geschrieben (vowel change ei->ie->ie)
8. **M** (mixed) | kannte | gekannt (vowel change + weak endings)

</details>

### Exercise 2: Haben or Sein?

Choose the correct auxiliary for the Perfekt.

1. Ich ________ nach Berlin gefahren. (fahren — movement A to B)
2. Er ________ das Auto gefahren. (fahren — with accusative object)
3. Wir ________ drei Stunden geschlafen. (schlafen — duration)
4. Sie ________ Aerztin geworden. (werden — change of state)
5. Ich ________ das Buch gelesen. (lesen — transitive)
6. Der Hund ________ weggelaufen. (weglaufen — movement)
7. Er ________ zu Hause geblieben. (bleiben — always sein)
8. Was ________ passiert? (passieren — always sein)

<details>
<summary>Answers</summary>

1. **bin** (movement from A to B)
2. **habe** (transitive — has accusative object "das Auto")
3. **haben** (no movement, just duration)
4. **ist** (change of state)
5. **habe** (transitive verb)
6. **ist** (movement away)
7. **ist** (bleiben always uses sein)
8. **ist** (passieren always uses sein)

</details>

### Exercise 3: Complete the Tense Drill

Conjugate "schreiben" (to write, strong, haben) through all tenses for "er".

1. Praesens: Er ________
2. Praeteritum: Er ________
3. Perfekt: Er ________
4. Plusquamperfekt: Er ________
5. Futur I: Er ________
6. Futur II: Er ________

<details>
<summary>Answers</summary>

1. Er **schreibt**. (present — no stem change for schreiben)
2. Er **schrieb**. (Praeteritum — vowel change ei -> ie)
3. Er **hat geschrieben**. (Perfekt — haben + Partizip II)
4. Er **hatte geschrieben**. (Plusquamperfekt — hatte + Partizip II)
5. Er **wird schreiben**. (Futur I — werden + Infinitive)
6. Er **wird geschrieben haben**. (Futur II — werden + PP + haben)

</details>

### Exercise 4: Active to Passive

Rewrite each active sentence in the passive voice (same tense).

1. Der Koch kocht die Suppe. (Praesens) -> ________
2. Die Schueler lasen das Buch. (Praeteritum) -> ________
3. Man hat den Brief geschrieben. (Perfekt) -> ________
4. Der Mechaniker wird das Auto reparieren. (Futur I) -> ________

<details>
<summary>Answers</summary>

1. Die Suppe **wird** (vom Koch) **gekocht**. (Praesens passive)
2. Das Buch **wurde** (von den Schuelern) **gelesen**. (Praeteritum passive)
3. Der Brief **ist** geschrieben **worden**. (Perfekt passive — worden, not geworden)
4. Das Auto **wird** (vom Mechaniker) **repariert werden**. (Futur I passive)

</details>

### Exercise 5: Mixed Challenge — Translate into German

Translate each sentence, paying attention to verb type, tense, auxiliary selection, and voice.

1. She can speak three languages. (modal, present)
2. If I were rich, I would travel the world. (Konjunktiv II)
3. The house was built by the workers. (passive, Praeteritum)
4. After he had eaten, he went for a walk. (Plusquamperfekt + Praeteritum)
5. I would like to order, please. (moechten)
6. He has been learning German for two years. (seit + present tense)
7. The letter has been written. (Perfekt passive)
8. If I had known that, I would have helped you. (Konj. II past)
9. He brought her a flower. (ditransitive, Praeteritum, mixed verb)
10. It is getting cold and it will rain tomorrow. (werden as full verb + Futur I)

<details>
<summary>Answers</summary>

1. Sie **kann** drei Sprachen **sprechen**. (modal + infinitive at end)
2. **Wenn** ich reich **waere**, **wuerde** ich die Welt **bereisen**. (Konj. II: waere + wuerde + infinitive)
3. Das Haus **wurde** von den Arbeitern **gebaut**. (Praeteritum passive)
4. **Nachdem** er **gegessen hatte**, **ging** er spazieren. (Plusquamperfekt in nachdem-clause, Praet. in main clause)
5. Ich **moechte** bitte **bestellen**. (moechten + infinitive)
6. Er **lernt** Deutsch **seit** zwei Jahren. (seit + present tense — German uses present for ongoing actions with seit)
7. Der Brief **ist** geschrieben **worden**. (Perfekt passive — worden, not geworden)
8. **Wenn** ich das **gewusst haette**, **haette** ich dir **geholfen**. (Konj. II past: haette + PP)
9. Er **brachte** ihr eine Blume. (bringen is mixed: brachte; dative + accusative)
10. Es **wird** kalt und es **wird** morgen **regnen**. (werden as "become" + werden + infinitive for future)

</details>
