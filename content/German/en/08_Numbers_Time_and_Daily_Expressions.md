# 8. Numbers, Time, and Daily Expressions (Zahlen, Uhrzeit und Alltagsausdrücke)

## Learning Objectives

- Count from 0 to 1,000+ and understand the reversed German number word order

- Name all days of the week and months of the year in German
- Express dates correctly using ordinal numbers
- Describe seasons and weather with common expressions
- Use prices and currency expressions confidently
- Apply common daily time-related expressions and idioms

## 1. Cardinal Numbers 0-100

### 1.1 Numbers 0-12

These must be memorized individually:

| Number | German | Number | German |
|--------|--------|--------|--------|
| 0 | null | 7 | sieben |
| 1 | eins | 8 | acht |
| 2 | zwei | 9 | neun |
| 3 | drei | 10 | zehn |
| 4 | vier | 11 | elf |
| 5 | fünf | 12 | zwölf |
| 6 | sechs | | |

> **Note**: *eins* becomes *ein-* when used before a noun and declines like the indefinite article: **ein** Buch, **eine** Katze, **einen** Hund.

### 1.2 Numbers 13-19

Formed with the pattern: **unit + zehn** (like English "-teen"):

| Number | German | Literal |
|--------|--------|---------|
| 13 | dreizehn | three-ten |
| 14 | vierzehn | four-ten |
| 15 | fünfzehn | five-ten |
| 16 | **sechzehn** | six-ten (sechs loses the -s) |
| 17 | **siebzehn** | seven-ten (sieben loses the -en) |
| 18 | achtzehn | eight-ten |
| 19 | neunzehn | nine-ten |

> **Watch out**: *sech**zehn*** (not sechszehn) and *sieb**zehn*** (not siebenzehn).

### 1.3 Tens (20-90)

| Number | German | Number | German |
|--------|--------|--------|--------|
| 20 | zwanzig | 60 | sechzig |
| 30 | **dreißig** | 70 | siebzig |
| 40 | vierzig | 80 | achtzig |
| 50 | fünfzig | 90 | neunzig |

> **Note**: 30 is *drei**ßig*** (with *ß*), not *dreißzig*.

### 1.4 Numbers 21-99: The Reversed Order

This is where German differs fundamentally from English. In German, the **ones digit comes before the tens digit**, connected by **und** (and):

```
English:  twenty-one    (tens first, then ones)
German:   einundzwanzig  (ones first, "und", then tens)
          ein-und-zwanzig = one-and-twenty
```

| Number | German | Breakdown |
|--------|--------|-----------|
| 21 | **einundzwanzig** | ein + und + zwanzig |
| 32 | **zweiunddreißig** | zwei + und + dreißig |
| 45 | **fünfundvierzig** | fünf + und + vierzig |
| 53 | **dreiundfünfzig** | drei + und + fünfzig |
| 67 | **siebenundsechzig** | sieben + und + sechzig |
| 78 | **achtundsiebzig** | acht + und + siebzig |
| 84 | **vierundachtzig** | vier + und + achtzig |
| 99 | **neunundneunzig** | neun + und + neunzig |

> **Tip**: Think of the German nursery rhyme "99 Luftballons" — *neun*und*neunzig* Luftballons.

### 1.5 The Number 100

- 100 = **(ein)hundert** — The *ein* is often dropped.

## 2. Numbers 100-1000 and Beyond

### 2.1 Hundreds

| Number | German |
|--------|--------|
| 100 | (ein)hundert |
| 200 | zweihundert |
| 300 | dreihundert |
| 400 | vierhundert |
| 500 | fünfhundert |
| 600 | sechshundert |
| 700 | siebenhundert |
| 800 | achthundert |
| 900 | neunhundert |

Compound hundreds follow the pattern: **hundert + ones-und-tens**:

- 125 = (ein)hundert**fünfundzwanzig**
- 347 = dreihundert**siebenundvierzig**
- 891 = achthundert**einundneunzig**

> **Note**: German writes compound numbers as **one word** (no spaces): *dreihundertsiebenundvierzig*.

### 2.2 Thousands and Beyond

| Number | German |
|--------|--------|
| 1,000 | (ein)tausend |
| 2,000 | zweitausend |
| 10,000 | zehntausend |
| 100,000 | hunderttausend |
| 1,000,000 | eine Million |
| 1,000,000,000 | eine Milliarde |

Examples of large numbers:

- 1.234 = (ein)tausendzweihundertvierunddreißig
- 2.025 = zweitausendfünfundzwanzig
- 3.456.789 = drei Millionen vierhundertsechsundfünfzigtausendsiebenhundertneunundachtzig

> **Important**: German uses a **period** (.) for thousands and a **comma** (,) for decimals — the opposite of English:
> - German: 1.000,50 = English: 1,000.50

## 3. Ordinal Numbers

Ordinal numbers answer the question "Which one in order?" (Der wievielte?).

### 3.1 Formation Rules

| Range | Rule | Suffix |
|-------|------|--------|
| 1st-19th | stem + **-te** | -te |
| 20th+ | stem + **-ste** | -ste |

| Ordinal | German | Note |
|---------|--------|------|
| 1st | **erste** | irregular |
| 2nd | **zweite** | regular |
| 3rd | **dritte** | irregular (extra -t) |
| 4th | **vierte** | regular |
| 5th | **fünfte** | regular |
| 6th | **sechste** | regular |
| 7th | **sieb(en)te** | irregular (loses -en) |
| 8th | **achte** | irregular (no extra -t) |
| 9th | **neunte** | regular |
| 10th | **zehnte** | regular |
| 20th | **zwanzigste** | -ste |
| 21st | **einundzwanzigste** | -ste |
| 100th | **hundertste** | -ste |

### 3.2 Ordinals as Adjectives

Ordinal numbers function as adjectives and **must take adjective endings**:

- Heute ist der **erste** Januar. — Today is the first of January. (weak, masc. nom.)
- Er ist am **dritten** Mai geboren. — He was born on the third of May. (weak, masc. dat.)
- Sie wohnt im **zweiten** Stock. — She lives on the second floor. (weak, masc. dat.)
- Das ist mein **erstes** Auto. — That is my first car. (mixed, neut. nom.)

### 3.3 Written Abbreviations

Ordinals are abbreviated with a **period** after the number:

- 1\. = erste (1st)
- 2\. = zweite (2nd)
- 3\. = dritte (3rd)
- 21\. = einundzwanzigste (21st)

## 4. Telling Time

### 4.1 Asking for the Time

- **Wie spät ist es?** — What time is it? (lit. "How late is it?")
- **Wie viel Uhr ist es?** — What time is it? (lit. "How many o'clock is it?")
- **Um wie viel Uhr...?** — At what time...?

### 4.2 Formal Time (24-Hour Clock)

The 24-hour clock is standard in Germany for schedules, announcements, and official contexts:

| Time | German |
|------|--------|
| 8:00 | Es ist **acht Uhr**. |
| 9:15 | Es ist **neun Uhr fünfzehn**. |
| 14:30 | Es ist **vierzehn Uhr dreißig**. |
| 17:45 | Es ist **siebzehn Uhr fünfundvierzig**. |
| 20:00 | Es ist **zwanzig Uhr**. |
| 0:00 | Es ist **null Uhr** / **Mitternacht**. |

### 4.3 Informal Time (12-Hour Clock)

In everyday conversation, Germans use the 12-hour system with these key terms:

| German | English | Used for |
|--------|---------|----------|
| **Viertel nach** | quarter past | :15 |
| **halb** | half (before next hour!) | :30 |
| **Viertel vor** | quarter to | :45 |
| **nach** | past/after | :01-:29 |
| **vor** | to/before | :31-:59 |

> **Critical difference**: In German, **halb** refers to **half before** the next hour, not the current hour:
>
> - **halb drei** = 2:30 (half [before] three), **NOT** 3:30
> - **halb acht** = 7:30 (half [before] eight), **NOT** 8:30

| Time | German (informal) | Literal |
|------|-------------------|---------|
| 3:00 | drei (Uhr) | three o'clock |
| 3:05 | fünf nach drei | five past three |
| 3:10 | zehn nach drei | ten past three |
| 3:15 | Viertel nach drei | quarter past three |
| 3:20 | zwanzig nach drei | twenty past three |
| 3:25 | fünf vor halb vier | five before half four |
| 3:30 | **halb vier** | half four (= half before 4) |
| 3:35 | fünf nach halb vier | five past half four |
| 3:40 | zwanzig vor vier | twenty to four |
| 3:45 | Viertel vor vier | quarter to four |
| 3:50 | zehn vor vier | ten to four |
| 3:55 | fünf vor vier | five to four |

### 4.4 Time of Day

| German | English | Approximate Time |
|--------|---------|-----------------|
| morgens | in the morning | 6:00-10:00 |
| vormittags | late morning | 10:00-12:00 |
| mittags | at noon | 12:00-14:00 |
| nachmittags | in the afternoon | 14:00-18:00 |
| abends | in the evening | 18:00-22:00 |
| nachts | at night | 22:00-6:00 |

Examples:
- Der Kurs beginnt **um neun Uhr morgens**. — The course starts at 9 in the morning.
- Wir essen **um halb eins** zu Mittag. — We eat lunch at 12:30.

## 5. Days of the Week

All days of the week in German are **masculine** (der):

| German | English | Abbreviation |
|--------|---------|-------------|
| der **Montag** | Monday | Mo. |
| der **Dienstag** | Tuesday | Di. |
| der **Mittwoch** | Wednesday | Mi. |
| der **Donnerstag** | Thursday | Do. |
| der **Freitag** | Friday | Fr. |
| der **Samstag** / der **Sonnabend** | Saturday | Sa. |
| der **Sonntag** | Sunday | So. |

> **Note**: *Samstag* is used in southern Germany, Austria, and Switzerland. *Sonnabend* is used in northern and eastern Germany. Both are correct.

### Common Expressions with Days

| German | English |
|--------|---------|
| am Montag | on Monday |
| montags | on Mondays (every Monday) |
| jeden Montag | every Monday |
| letzten Freitag | last Friday |
| nächsten Dienstag | next Tuesday |
| von Montag bis Freitag | from Monday to Friday |

- **Am Samstag** gehe ich einkaufen. — On Saturday I go shopping.
- **Dienstags** habe ich Deutschkurs. — On Tuesdays I have German class.

|--------|---------|--------|---------|

## 7. Dates

### 7.1 Asking the Date

- **Der Wievielte ist heute?** — What is today's date? (lit. "The how-many-th is today?")
- **Welches Datum haben wir heute?** — What date do we have today?
- **Den Wievielten haben wir heute?** — What date is it today? (accusative form)

### 7.2 Stating the Date

Dates use **ordinal numbers**:

- Heute ist **der fünfte März**. — Today is the fifth of March.
- Heute ist **der 5. März**. — Today is March 5th.
- Morgen ist **der sechste März**. — Tomorrow is the sixth of March.

### 7.3 Dates with Prepositions

| Situation | Preposition | Case | Example |
|-----------|-------------|------|---------|
| On a date | **am** (an + dem) | Dative | **Am 5. März** habe ich Geburtstag. |
| From a date | **vom** (von + dem) | Dative | **Vom 1. bis zum 15. Juli** bin ich im Urlaub. |
| Until a date | **bis zum** | Dative | **Bis zum 31. Dezember** muss das fertig sein. |

### 7.4 Writing Full Dates

German dates are written in **day.month.year** order:

- 05.03.2026 = der 5. März 2026
- 24.12.2025 = der 24. Dezember 2025
- 01.01.2000 = der 1. Januar 2000

> **Reading years**: Years are read differently depending on the century:
> - 1990 = **neunzehnhundertneunzig** (nineteen hundred ninety)
> - 2000 = **zweitausend** (two thousand)
> - 2026 = **zweitausendsechsundzwanzig** (two thousand twenty-six)

## 8. Seasons and Weather

### 8.1 The Four Seasons

All seasons are **masculine** (der):

| German | English |
|--------|---------|
| der **Frühling** (der **Lenz**) | spring |
| der **Sommer** | summer |
| der **Herbst** | autumn / fall |
| der **Winter** | winter |

- **Im Frühling** blühen die Blumen. — In spring, the flowers bloom.
- **Im Sommer** ist es heiß. — In summer, it is hot.
- **Im Herbst** fallen die Blätter. — In autumn, the leaves fall.
- **Im Winter** schneit es oft. — In winter, it often snows.

### 8.2 Weather Expressions

German weather expressions often use **es** (it) as the subject:

| German | English |
|--------|---------|
| **Es regnet.** | It is raining. |
| **Es schneit.** | It is snowing. |
| **Es donnert.** | It is thundering. |
| **Es ist sonnig.** | It is sunny. |
| **Es ist bewölkt.** | It is cloudy. |
| **Es ist neblig.** | It is foggy. |
| **Es ist windig.** | It is windy. |
| **Es ist heiß.** | It is hot. |
| **Es ist warm.** | It is warm. |
| **Es ist kühl.** | It is cool. |
| **Es ist kalt.** | It is cold. |
| **Es friert.** | It is freezing. |

### 8.3 Temperature

- **Wie ist das Wetter heute?** — How is the weather today?
- **Wie viel Grad sind es?** — How many degrees is it?
- Es sind **zwanzig Grad**. — It is twenty degrees.
- Es sind **minus fünf Grad**. — It is minus five degrees.

> **Note**: Germany uses **Celsius**, not Fahrenheit. 20°C is a pleasant spring day; 30°C is quite hot.

## 9. Prices and Currency

### 9.1 Euro and Cent

The currency in Germany, Austria, and most of the EU is the **Euro** (€):

- 1 Euro = 100 Cent
- The symbol **€** is placed after the number in German: 5,99 €

| German | English |
|--------|---------|
| ein Euro | one euro |
| zwei Euro | two euros |
| fünfzig Cent | fifty cents |
| drei Euro fünfzig | three euros fifty |

> **Note**: *Euro* does not change in the plural in German: ein Euro, zwei **Euro** (not Euros).

### 9.2 Asking About Prices

| German | English |
|--------|---------|
| **Was kostet das?** | What does that cost? |
| **Wie viel kostet das?** | How much does that cost? |
| **Was kosten die Äpfel?** | What do the apples cost? |
| **Das macht...** | That comes to... |

### 9.3 Price Examples

- Das Buch kostet **neunzehn Euro neunundneunzig** (19,99 €). — The book costs 19.99 euros.
- Ein Kaffee kostet **drei Euro fünfzig** (3,50 €). — A coffee costs 3.50 euros.
- Die Fahrkarte kostet **zwölf Euro achtzig** (12,80 €). — The ticket costs 12.80 euros.

### 9.4 Paying

| German | English |
|--------|---------|
| Kann ich mit Karte zahlen? | Can I pay by card? |
| Ich zahle bar. | I pay in cash. |
| Stimmt so. | Keep the change. |
| Können Sie wechseln? | Can you make change? |
