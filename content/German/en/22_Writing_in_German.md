## 1. German Writing Conventions

Before diving into specific text types, review these fundamental rules of German writing:

### Capitalization (Großschreibung)

| Rule | Example |
|------|---------|
| All nouns are capitalized | *der Tisch*, *die Freiheit*, *das Auto* |
| Sentence beginnings | *Er kommt morgen.* |
| Formal "You" — *Sie/Ihnen/Ihr* | *Können Sie mir helfen?* |
| In letters: *Du/Dir/Dein* (optional but common) | *Wie geht es Dir?* |
| Nominalized verbs and adjectives | *das Lesen* (reading), *der Alte* (the old man) |
| Languages, nationalities (as nouns) | *Er spricht Deutsch.* / *die Deutschen* |

### The Eszett (ß) and Double-S (ss)

After the 1996 spelling reform:
- **ß** follows a long vowel or diphthong: *Straße*, *groß*, *heißen*
- **ss** follows a short vowel: *dass*, *muss*, *Schloss*
- Switzerland does not use ß at all — always *ss*

### Umlauts (ä, ö, ü)

Umlauts are not optional decorations — they change meaning:
- *schon* (already) vs. *schön* (beautiful)
- *schwul* (gay) vs. *schwül* (humid)
- *Mutter* (mother) vs. *Mütter* (mothers)

If you cannot type umlauts, use: ae, oe, ue (e.g., *Muenchen* for *München*).

## 2. Informal Letters and Emails

### Structure

```
+----------------------------------------------------------+
|  INFORMAL LETTER / EMAIL                                  |
+----------------------------------------------------------+
|                                                          |
|  Place, Date           Berlin, den 5. März 2025          |
|                                                          |
|  Greeting              Liebe Anna, / Lieber Max,         |
|                        Hallo Lisa,                       |
|                                                          |
|  Opening               Wie geht es Dir?                  |
|                        Vielen Dank für Deine E-Mail!     |
|                                                          |
|  Body                  [Your message]                    |
|                                                          |
|  Closing               Ich freue mich auf Deine Antwort! |
|                                                          |
|  Sign-off              Liebe Grüße, / Viele Grüße,      |
|                        Bis bald! / Herzliche Grüße,      |
|                                                          |
|  Name                  Dein Thomas / Deine Maria         |
+----------------------------------------------------------+
```

### Key Points

- **Du/Dein/Dir**: In personal letters, these are traditionally capitalized (though lowercase is now also accepted). Capitalizing shows respect and warmth.
- **Greeting comma**: After *Liebe Anna,* there is a **comma**, and the next line begins with a **lowercase** letter (unless it is a noun or *Ich*).
- **Tone**: Conversational, personal, use of exclamation marks and questions.

### Common Openings

| German | English |
|--------|---------|
| Wie geht es Dir? | How are you? |
| Ich hoffe, Dir geht es gut! | I hope you are well! |
| Vielen Dank für Deine Nachricht! | Thanks for your message! |
| Es tut mir leid, dass ich so lange nicht geschrieben habe. | Sorry I haven't written in so long. |
| Es war schön, von Dir zu hören! | It was nice to hear from you! |

### Common Closings

| German | English |
|--------|---------|
| Liebe Grüße | With love / Best wishes |
| Viele Grüße | Many greetings / Best regards |
| Herzliche Grüße | Warm regards |
| Bis bald! | See you soon! |
| Schreib mir bald! | Write me soon! |
| Ich freue mich auf Deine Antwort! | I look forward to your reply! |

### Example: Writing to a Friend About Vacation Plans

> Berlin, den 12. Juni 2025
>
> Lieber Jan,
>
> wie geht es Dir? Ich hoffe, Du hattest eine schöne Woche!
>
> Ich schreibe Dir, weil ich schon unseren Sommerurlaub plane. Was hältst Du davon, im August nach Kroatien zu fahren? Meine Freundin war letztes Jahr dort und hat mir wunderschöne Fotos gezeigt. Die Strände sollen fantastisch sein, und es ist viel günstiger als Griechenland.
>
> Wir könnten für eine Woche eine kleine Ferienwohnung mieten und mit dem Auto hinfahren. Was meinst Du? Hast Du im August Zeit?
>
> Schreib mir bald, damit wir planen können!
>
> Viele Grüße,
> Dein Stefan

## 3. Formal Letters and Emails

### Structure

```
+----------------------------------------------------------+
|  FORMAL LETTER / EMAIL                                    |
+----------------------------------------------------------+
|                                                          |
|  Sender                Anna Schmidt                      |
|  (Absender)            Musterstraße 12                   |
|                        10115 Berlin                      |
|                                                          |
|  Recipient             Firma XY GmbH                     |
|  (Empfänger)           Personalabteilung                 |
|                        Hauptstraße 45                    |
|                        80331 München                     |
|                                                          |
|  Place, Date           Berlin, den 5. März 2025          |
|                                                          |
|  Subject               Betreff: Bewerbung als ...        |
|                                                          |
|  Greeting              Sehr geehrte Damen und Herren,    |
|                        Sehr geehrter Herr Müller,        |
|                        Sehr geehrte Frau Dr. Schmidt,    |
|                                                          |
|  Body                  [Formal message — Sie form]       |
|                                                          |
|  Closing formula       Mit freundlichen Grüßen           |
|                                                          |
|  Signature             Anna Schmidt                      |
|  Attachment note       Anlage: Lebenslauf, Zeugnisse     |
+----------------------------------------------------------+
```

### Key Points

- **Sie/Ihnen/Ihr**: Always capitalized in formal writing. Never use *du* in formal correspondence.
- After *Sehr geehrte Damen und Herren,* the next line begins with a **lowercase** letter (same comma rule as informal).
- **Betreff** (subject line): Concise, no period at the end.
- **Mit freundlichen Grüßen** — no comma after this closing (unlike English).

### Common Formal Openings

| German | English |
|--------|---------|
| Sehr geehrte Damen und Herren, | Dear Sir or Madam, |
| Sehr geehrter Herr [Name], | Dear Mr. [Name], |
| Sehr geehrte Frau [Name], | Dear Mrs./Ms. [Name], |
| Sehr geehrte Frau Dr. [Name], | Dear Dr. [Name], |

### Common Formal Closings

| German | English |
|--------|---------|
| Mit freundlichen Grüßen | Yours sincerely / Best regards |
| Mit freundlichem Gruß | Kind regards |
| Hochachtungsvoll | Respectfully yours (very formal) |

### Common Formal Phrases

| German | English |
|--------|---------|
| Hiermit bewerbe ich mich um... | I hereby apply for... |
| Bezugnehmend auf Ihre Anzeige... | Referring to your advertisement... |
| Ich möchte Sie darüber informieren, dass... | I would like to inform you that... |
| Für Rückfragen stehe ich Ihnen gerne zur Verfügung. | I am happy to answer any questions. |
| Ich freue mich auf Ihre Rückmeldung. | I look forward to your response. |
| Vielen Dank im Voraus. | Thank you in advance. |

## 4. Essay Writing: Aufsatz and Erörterung

### General Essay Structure

Every well-written German essay follows this three-part structure:

| Part | German | Function | Approximate Length |
|------|--------|----------|-------------------|
| Introduction | **Einleitung** | Hook, context, thesis statement | 10-15% |
| Body | **Hauptteil** | Arguments, evidence, examples | 70-80% |
| Conclusion | **Schluss** | Summary, final position, outlook | 10-15% |

### Einleitung (Introduction) Strategies

Start your essay with one of these hooks:

| Strategy | Example |
|----------|---------|
| Question | *Sollte die Schule erst um 9 Uhr beginnen?* |
| Statistic | *Laut einer Studie nutzen 95% aller Jugendlichen soziale Medien.* |
| Quote | *Goethe sagte einmal: "Es ist nicht genug, zu wissen, man muss auch anwenden."* |
| Provocative statement | *Das Smartphone hat die Art, wie wir kommunizieren, grundlegend verändert.* |

Then state your thesis:
- *In diesem Aufsatz werde ich argumentieren, dass...*
- *Im Folgenden sollen die Vor- und Nachteile von ... untersucht werden.*

### Erörterung (Argumentative Essay)

The *Erörterung* is a classic German essay format taught in schools:

**Linear Erörterung** (one-sided): Present arguments supporting one position.

**Dialektische Erörterung** (two-sided): Present both pro and contra arguments, then state your own position.

```
+----------------------------------------------------------+
|  DIALEKTISCHE ERÖRTERUNG                                  |
+----------------------------------------------------------+
|                                                          |
|  Einleitung                                              |
|  → Present the topic, give context                       |
|  → State the central question (Leitfrage)                |
|                                                          |
|  Hauptteil                                               |
|  → Pro-Argumente (arguments for)                         |
|    - Argument 1 + Beispiel                               |
|    - Argument 2 + Beispiel                               |
|    - Argument 3 + Beispiel                               |
|  → Contra-Argumente (arguments against)                  |
|    - Argument 1 + Beispiel                               |
|    - Argument 2 + Beispiel                               |
|    - Argument 3 + Beispiel                               |
|                                                          |
|  Schluss                                                 |
|  → Weigh arguments, state your position                  |
|  → Outlook or call to action                             |
+----------------------------------------------------------+
```

> **Tip**: In a dialektische Erörterung, present the weaker side first and the side you agree with second, so it leaves a stronger impression.

### Bildbeschreibung (Picture Description)

A structured approach for describing images in German:

1. **Allgemeine Beschreibung**: *Auf dem Bild sieht man...* (In the picture one can see...)
2. **Vordergrund**: *Im Vordergrund befindet sich...* (In the foreground there is...)
3. **Hintergrund**: *Im Hintergrund erkennt man...* (In the background one can see...)
4. **Details**: *Auf der linken/rechten Seite...* (On the left/right side...)
5. **Interpretation**: *Das Bild zeigt vermutlich...* (The picture probably shows...)
6. **Persönliche Reaktion**: *Dieses Bild erinnert mich an...* (This picture reminds me of...)

## 6. Descriptive Writing: Bildbeschreibung

### Useful Phrases for Image Description

**Location**:
- *Im Vordergrund / Im Hintergrund* — In the foreground / background
- *In der Mitte des Bildes* — In the center of the picture
- *Auf der linken / rechten Seite* — On the left / right side
- *Am oberen / unteren Rand* — At the top / bottom edge
- *Neben / Hinter / Vor* — Next to / Behind / In front of

**Description**:
- *Man sieht / Man erkennt* — One can see / recognize
- *Es scheint, als ob...* — It seems as if...
- *Die Person trägt...* — The person is wearing...
- *Die Szene spielt sich ab in...* — The scene takes place in...

**Interpretation**:
- *Das Bild vermittelt den Eindruck, dass...* — The image gives the impression that...
- *Der Fotograf / Die Fotografin wollte wahrscheinlich zeigen, dass...* — The photographer probably wanted to show that...
- *Das Bild könnte symbolisch für ... stehen.* — The image could be symbolic of...

## 7. German Punctuation Rules

German punctuation differs from English in several important ways.

### Comma Rules (Kommaregeln)

**Mandatory comma before subordinate clauses**:

Every subordinating conjunction requires a comma:

| Conjunction | Example |
|------------|---------|
| **dass** | *Ich weiß**,** dass er morgen kommt.* |
| **weil** | *Sie bleibt zu Hause**,** weil sie krank ist.* |
| **wenn** | *Ruf mich an**,** wenn du fertig bist.* |
| **obwohl** | *Er kam**,** obwohl er müde war.* |
| **ob** | *Ich frage mich**,** ob das stimmt.* |
| **als** | *Er war traurig**,** als sie ging.* |
| **während** | *Ich lese**,** während du kochst.* |
| **damit** | *Ich spare**,** damit ich reisen kann.* |

> **This is mandatory** — unlike English, where commas before "because" or "if" are sometimes optional, German requires them before every subordinate clause.

**Comma between main clauses** connected by *und* or *oder*:

- Optional if the clauses share the same subject: *Er kam nach Hause und (er) setzte sich auf das Sofa.*
- Recommended for clarity when clauses have different subjects: *Er kam nach Hause, und sie ging in die Küche.*

**Comma in relative clauses**:

Always set off relative clauses with commas:
- *Der Mann**,** der neben mir steht**,** ist mein Bruder.*
- *Das Buch**,** das ich gestern gelesen habe**,** war spannend.*

**Comma in lists (no Oxford comma)**:

German does **not** use the Oxford comma:
- *Ich kaufe Brot, Butter, Käse und Milch.* (NOT: *Brot, Butter, Käse, und Milch*)

**Comma with infinitive clauses with *zu***:

A comma is used when the infinitive clause contains additional words:
- *Er beschloss**,** nach Berlin zu fahren.*
- *Sie hatte keine Lust**,** den ganzen Tag zu arbeiten.*

### Quotation Marks (Anführungszeichen)

German uses **bottom-top** quotation marks, different from English:

- German: **„Hallo, wie geht es Ihnen?"**
- English: "Hello, how are you?"

In print and formal writing:
- Primary quotes: „..." (opening at bottom, closing at top)
- Quotes within quotes: ‚...' (single bottom-top)

> **Note**: In digital communication and many keyboards, standard "..." quotes are often used instead, and this is widely accepted.

### Other Punctuation Differences

| Rule | German | English |
|------|--------|---------|
| Decimal separator | **Komma**: 3,50 € | **Period**: $3.50 |
| Thousands separator | **Punkt**: 1.000.000 | **Comma**: 1,000,000 |
| Time | **Doppelpunkt**: 14:30 Uhr | **Colon**: 2:30 PM |
| Date format | **TT.MM.JJJJ**: 05.03.2025 | MM/DD/YYYY: 03/05/2025 |

## 8. Common Writing Mistakes

### Word Order Errors

**Mistake 1**: Forgetting verb-second (V2) in main clauses.

- Wrong: *Gestern ich bin ins Kino gegangen.*
- Correct: *Gestern **bin** ich ins Kino gegangen.*

**Mistake 2**: Forgetting verb-final in subordinate clauses.

- Wrong: *Ich weiß, dass er kommt morgen.*
- Correct: *Ich weiß, dass er morgen **kommt**.*

**Mistake 3**: Wrong word order after adverbs like *deshalb*, *trotzdem*.

- Wrong: *Deshalb ich bleibe zu Hause.*
- Correct: *Deshalb **bleibe** ich zu Hause.* (subject-verb inversion)

### Case Errors

**Mistake 4**: Using the wrong case after prepositions.

- Wrong: *Ich gehe in **der** Park.* (dative)
- Correct: *Ich gehe in **den** Park.* (accusative — movement toward)
- Also correct: *Ich bin in **dem** Park.* (dative — location)

### Article/Gender Confusion

**Mistake 5**: Wrong gender for common nouns.

Commonly confused:
- *das* Mädchen (neuter — because of the -chen diminutive, NOT feminine)
- *die* E-Mail (feminine in German, though "mail" feels neuter)
- *der* Computer (masculine)

### Comma Errors

**Mistake 6**: Missing comma before subordinate clauses.

- Wrong: *Ich glaube dass er Recht hat.*
- Correct: *Ich glaube**,** dass er Recht hat.*

**Mistake 7**: Adding an Oxford comma.

- Wrong: *Äpfel, Birnen, und Kirschen*
- Correct: *Äpfel, Birnen und Kirschen*

### Spelling Errors

**Mistake 8**: Confusing *das* (article/pronoun) and *dass* (conjunction).

- *Das* Buch ist gut. (article: the)
- Ich weiß, *dass* er kommt. (conjunction: that)
- Tip: If you can replace it with *dieses* or *welches*, use *das*. Otherwise, *dass*.

## 9. Text Messaging and Social Media

German text messaging uses many abbreviations and informal conventions:

### Common Abbreviations

| Abbreviation | Full Form | English |
|-------------|-----------|---------|
| LG | Liebe Grüße | Best wishes |
| VG | Viele Grüße | Many greetings |
| MfG | Mit freundlichen Grüßen | Best regards |
| HDL | Hab dich lieb | Love you (platonic) |
| GLG | Ganz liebe Grüße | Much love |
| WE | Wochenende | Weekend |
| vllt | vielleicht | maybe |
| eigtl | eigentlich | actually |
| bzgl | bezüglich | regarding |
| ggf | gegebenenfalls | if applicable |
| usw | und so weiter | and so on |
| zB / z.B. | zum Beispiel | for example |
| mMn | meiner Meinung nach | in my opinion |

### Informal Texting Style

- Lowercase throughout: *hey, kommst du heute abend?*
- Missing umlauts: *schoen* instead of *schön*
- Emoticons and emojis widely used
- Short sentences, fragments: *Bin gleich da.* (Am there soon.)
- English loanwords common: *cool, sorry, nice, okay*

> **Note**: Text messaging conventions should NOT be used in formal writing, exams, or business communication.

### Exercise 4: Write a Short Essay Outline

Write an outline (Gliederung) for this topic: *"Sollten Handys in der Schule verboten werden?"* (Should phones be banned in school?)

Include: Einleitung, 2-3 Pro arguments, 2-3 Contra arguments, Schluss.

<details>
<summary>Model Answer</summary>

**Einleitung**:
- Hook: *Heutzutage besitzt fast jeder Schüler ein Smartphone.* (Nowadays almost every student owns a smartphone.)
- Central question: *Sollten Handys in der Schule verboten werden?*

**Pro (Für ein Verbot)**:
1. Handys lenken vom Unterricht ab. (Phones distract from lessons.)
2. Cybermobbing kann während der Schulzeit passieren. (Cyberbullying can occur during school hours.)

1. Smartphones können als Lernmittel genutzt werden. (Smartphones can be used as learning tools.)

**Schluss**:

### Exercise 5: Identify and Fix the Errors

Each sentence contains one or more mistakes. Find and correct them.

1. *Gestern ich habe meine Mutter besucht.*
2. *Ich weiß, dass er kommt morgen nach Berlin.*
3. *Das buch was ich lese ist sehr interessant.*
4. *Sie hat mir gesagt das sie nicht kommen kann.*
5. *In den Park gehen wir oft am Wochenende spazieren.*

<details>
<summary>Answers</summary>

1. **Gestern habe ich meine Mutter besucht.** (V2 rule: verb must be in second position after *Gestern*)
2. **Ich weiß, dass er morgen nach Berlin kommt.** (verb goes to the end in a *dass*-clause)
3. **Das Buch, das ich lese, ist sehr interessant.** (*Buch* capitalized; relative pronoun *das* not *was*; commas around relative clause)
4. **Sie hat mir gesagt, dass sie nicht kommen kann.** (*dass* not *das*; comma before subordinate clause)
5. This sentence is actually **correct**! *In dem/den Park* depends on meaning. *In den Park gehen wir...* is correct (V2 with prepositional phrase in first position). However, it should be: **In dem Park gehen wir oft am Wochenende spazieren.** (dative — location, not movement to) OR the original is fine if they are going into the park.

</details>
