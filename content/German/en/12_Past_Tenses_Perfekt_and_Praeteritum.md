# 12. Past Tenses: Perfekt and Praeteritum (Perfekt und Präteritum)

- Conjugate common verbs in the Präteritum (weak, strong, and modal verbs)
- Form the Plusquamperfekt (past perfect) using hatte/war + Partizip II
- Choose the appropriate past tense based on context (speech vs. writing)
- Memorize the Partizip II forms of the 20+ most common irregular verbs

## 1. Overview of German Past Tenses

German has three past tenses. Understanding when to use each one is essential:

```
+------------------------------------------------------------------+
|                    GERMAN PAST TENSES                             |
+------------------------------------------------------------------+
|                                                                    |
|  PERFEKT (Present Perfect)                                        |
|  haben/sein (present) + Partizip II                               |
|  "Ich habe gegessen." (I have eaten / I ate.)                     |
|  USE: Spoken German, conversation, informal writing               |
|                                                                    |
|  PRÄTERITUM (Simple Past / Imperfect)                             |
|  One-word past form                                               |
|  "Ich aß." (I ate.)                                               |
|  USE: Written narratives, formal writing, sein/haben/modals       |
|                                                                    |
|  PLUSQUAMPERFEKT (Past Perfect / Pluperfect)                      |
|  hatte/war + Partizip II                                          |
|  "Ich hatte gegessen." (I had eaten.)                             |
|  USE: Action completed before another past action                 |
|                                                                    |
+------------------------------------------------------------------+
```

> **Key insight**: Unlike English, where "I ate" and "I have eaten" differ in meaning, the German Perfekt and Präteritum typically express the **same time reference**. The choice between them is primarily about **register** (spoken vs. written), not meaning.

---

## 2. Perfekt: Structure and Formation

The Perfekt is the **most common past tense in spoken German**. It is formed with two parts:

```
Subject + haben/sein (conjugated, position 2) + ... + Partizip II (end of clause)
```

**Examples**:

- Ich **habe** gestern viel **gearbeitet**. (I worked a lot yesterday.)
- Sie **ist** nach Berlin **geflogen**. (She flew to Berlin.)
- Wir **haben** den Film **gesehen**. (We saw the film.)
- Er **ist** um 7 Uhr **aufgestanden**. (He got up at 7 o'clock.)

> The Partizip II always goes to the **end** of the main clause. This is one of the most distinctive features of German sentence structure.

## 3. Partizip II: Regular (Weak) Verbs

Regular (weak) verbs form the Partizip II with the pattern:

```
ge- + verb stem + -t
```

| Infinitive | Stem | Partizip II | Meaning |
|-----------|------|-------------|---------|
| machen | mach- | **ge**mach**t** | made/done |
| spielen | spiel- | **ge**spiel**t** | played |
| lernen | lern- | **ge**lern**t** | learned |
| kaufen | kauf- | **ge**kauf**t** | bought |
| kochen | koch- | **ge**koch**t** | cooked |
| wohnen | wohn- | **ge**wohn**t** | lived |
| hören | hör- | **ge**hör**t** | heard |
| sagen | sag- | **ge**sag**t** | said |

### Stems Ending in -t, -d, -chn, -ffn, -gn

These add **-et** instead of **-t** for pronunciation:

| Infinitive | Partizip II |
|-----------|-------------|
| arbeiten | **ge**arbeit**et** |
| warten | **ge**wart**et** |
| reden | **ge**red**et** |
| öffnen | **ge**öffn**et** |
| regnen | **ge**regn**et** |

---

## 4. Partizip II: Strong (Irregular) Verbs

Strong verbs form the Partizip II with:

```
ge- + changed stem + -en
```

The vowel in the stem typically changes (this is called **Ablaut**):

| Infinitive | Partizip II | Vowel Change | Meaning |
|-----------|-------------|--------------|---------|
| gehen | **ge**gang**en** | e → a | gone |
| sprechen | **ge**sproch**en** | e → o | spoken |
| schreiben | **ge**schrieb**en** | ei → ie | written |
| trinken | **ge**trunk**en** | i → u | drunk |
| fahren | **ge**fahr**en** | (no change) | driven |
| lesen | **ge**les**en** | (no change) | read |
| singen | **ge**sung**en** | i → u | sung |
| finden | **ge**fund**en** | i → u | found |
| helfen | **ge**holf**en** | e → o | helped |
| nehmen | **ge**nomm**en** | e → o | taken |

> **There is no shortcut**: Strong verb forms must be memorized. See Section 11 for a comprehensive table.

## 5. Partizip II: Special Cases

### Verbs Ending in -ieren (No ge-)

Verbs borrowed from French/Latin that end in **-ieren** do NOT get the **ge-** prefix:

| Infinitive | Partizip II | Meaning |
|-----------|-------------|---------|
| studieren | studier**t** | studied |
| telefonieren | telefonier**t** | telephoned |
| reparieren | reparier**t** | repaired |
| funktionieren | funktionier**t** | functioned |
| fotografieren | fotografier**t** | photographed |

### Inseparable Prefix Verbs (No ge-)

Verbs with inseparable prefixes (be-, emp-, ent-, er-, ge-, miss-, ver-, zer-) do NOT get **ge-**:

| Infinitive | Partizip II | Meaning |
|-----------|-------------|---------|
| verstehen | verstand**en** | understood |
| bekommen | bekomm**en** | received |
| erzählen | erzähl**t** | told |
| beginnen | begonn**en** | begun |
| empfehlen | empfohl**en** | recommended |
| entdecken | entdeck**t** | discovered |
| vergessen | vergess**en** | forgotten |
| zerstören | zerstör**t** | destroyed |
| gehören | gehör**t** | belonged |
| misslingen | misslungen | failed |

### Separable Prefix Verbs (ge- Between Prefix and Stem)

Verbs with separable prefixes insert **ge-** between the prefix and the stem:

| Infinitive | Partizip II | Meaning |
|-----------|-------------|---------|
| aufmachen | auf**ge**mach**t** | opened |
| anfangen | an**ge**fang**en** | begun |
| aufstehen | auf**ge**stand**en** | stood up |
| einkaufen | ein**ge**kauf**t** | shopped |
| zurückkommen | zurück**ge**komm**en** | came back |
| mitnehmen | mit**ge**nomm**en** | taken along |
| anrufen | an**ge**ruf**en** | called |
| fernsehen | fern**ge**seh**en** | watched TV |

### Summary: Where Does ge- Go?

```
Regular verb:       ge + stem + t       →  gemacht
Strong verb:        ge + stem + en      →  gegangen
-ieren verb:        stem + t            →  studiert       (NO ge-)
Inseparable prefix: prefix + stem + t/en →  verstanden     (NO ge-)
Separable prefix:   prefix + ge + stem  →  aufgemacht     (ge- in middle)
```

## 6. Haben vs. Sein: Choosing the Auxiliary

Most verbs use **haben**. A smaller group uses **sein**. Here are the rules:

### Use SEIN with:

1. **Verbs of movement from A to B**: gehen, fahren, fliegen, kommen, laufen, reisen, schwimmen, fallen
2. **Verbs of change of state**: werden (become), sterben (die), einschlafen (fall asleep), aufwachen (wake up), wachsen (grow), geboren sein (be born)
3. **sein** and **bleiben** (to be and to stay): Ich **bin** dort gewesen. / Er **ist** geblieben.
4. **passieren** and **geschehen** (to happen): Was **ist** passiert?

### Use HABEN with:

1. **All transitive verbs** (verbs that take a direct object): Ich habe das Buch gelesen.
2. **Reflexive verbs**: Er hat sich gewaschen.
3. **Modal verbs**: Sie hat schwimmen können.
4. **Impersonal verbs**: Es hat geregnet. (It rained.)
5. **Everything else** not in the sein list.

### Quick Reference Table

| Auxiliary | Verb Type | Example |
|-----------|----------|---------|
| **sein** | Movement A→B | Er **ist** nach Hause **gegangen**. |
| **sein** | Change of state | Sie **ist** Ärztin **geworden**. |
| **sein** | sein, bleiben | Ich **bin** in Berlin **gewesen**. |
| **haben** | Transitive | Wir **haben** Pizza **gegessen**. |
| **haben** | Reflexive | Ich **habe** mich **gefreut**. |
| **haben** | Most verbs | Er **hat** gut **geschlafen**. |

> **Tricky case**: Some movement verbs use **haben** when the focus is on the activity rather than the destination:
> - Ich **bin** geschwommen. (I swam [to somewhere].)
> - Ich **habe** eine Stunde geschwommen. (I swam for an hour [activity focus].)
>
> In practice, this distinction is becoming less strict, and regional variation exists.

## 7. Präteritum: Formation and Use

The Präteritum is the **simple past** tense. While less common in speech (except for sein, haben, and modals), it is the standard past tense in **written narratives**.

### Weak (Regular) Verbs

Add these endings to the verb stem:

| Person | Ending | Example: machen |
|--------|--------|----------------|
| ich | -**te** | ich mach**te** |
| du | -**test** | du mach**test** |
| er/sie/es | -**te** | er mach**te** |
| wir | -**ten** | wir mach**ten** |
| ihr | -**tet** | ihr mach**tet** |
| sie/Sie | -**ten** | sie mach**ten** |

> Stems ending in -t, -d, -chn, -ffn add an extra **e**: arbeitete, redete, öffnete.

### Strong (Irregular) Verbs

Strong verbs change their stem vowel and use **different endings** (no -te):

| Person | Ending | Example: gehen → ging | Example: sprechen → sprach |
|--------|--------|----------------------|---------------------------|
| ich | — (zero) | ich **ging** | ich **sprach** |
| du | -**st** | du **gingst** | du **sprachst** |
| er/sie/es | — (zero) | er **ging** | er **sprach** |
| wir | -**en** | wir **gingen** | wir **sprachen** |
| ihr | -**t** | ihr **gingt** | ihr **spracht** |
| sie/Sie | -**en** | sie **gingen** | sie **sprachen** |

> Note: The 1st and 3rd person singular have **no ending** (zero ending) in strong Präteritum.

### Sein and Haben in Präteritum

These are used in Präteritum even in spoken German:

| Person | sein (was) | haben (had) |
|--------|-----------|-------------|
| ich | **war** | **hatte** |
| du | **warst** | **hattest** |
| er/sie/es | **war** | **hatte** |
| wir | **waren** | **hatten** |
| ihr | **wart** | **hattet** |
| sie/Sie | **waren** | **hatten** |

---

## 8. Präteritum of Modal Verbs

Modal verbs are almost always used in the **Präteritum** in spoken German (not Perfekt). Note that the **umlauts disappear**:

| Modal | Present (ich) | Präteritum (ich) | Meaning (past) |
|-------|--------------|-------------------|-----------------|
| können | kann | **konnte** | could, was able to |
| müssen | muss | **musste** | had to |
| dürfen | darf | **durfte** | was allowed to |
| sollen | soll | **sollte** | was supposed to |
| wollen | will | **wollte** | wanted to |
| mögen | mag | **mochte** | liked |

### Full Conjugation Pattern (Example: können)

| Person | Präteritum |
|--------|-----------|
| ich | konnte |
| du | konntest |
| er/sie/es | konnte |
| wir | konnten |
| ihr | konntet |
| sie/Sie | konnten |

> **Tip**: All modal Präteritum forms follow the weak verb pattern (-te endings) and lose their umlauts. The 1st and 3rd person singular are identical.

**Examples**:

- Ich **konnte** gestern nicht kommen. (I couldn't come yesterday.)
- Sie **musste** früh aufstehen. (She had to get up early.)
- Wir **durften** nicht ins Kino gehen. (We were not allowed to go to the cinema.)
- Er **wollte** Arzt werden. (He wanted to become a doctor.)
- Du **solltest** das nicht sagen. (You shouldn't have said that.)

## 9. When to Use Which Past Tense

### Perfekt vs. Präteritum: A Practical Guide

| Context | Preferred Tense | Example |
|---------|----------------|---------|
| **Everyday conversation** | Perfekt | Was hast du gestern gemacht? |
| **Casual writing** (emails, messages) | Perfekt | Ich habe dir geschrieben. |
| **sein, haben** (even in speech) | Präteritum | Ich **war** müde. / Ich **hatte** Hunger. |
| **Modal verbs** (even in speech) | Präteritum | Ich **konnte** nicht schlafen. |
| **Novels, fairy tales, news** | Präteritum | Der König **lebte** in einem Schloss. |
| **Academic/formal writing** | Präteritum | Die Studie **zeigte**, dass... |
| **Reporting a sequence** (written) | Präteritum | Er **kam**, **sah** und **siegte**. |

### Regional Differences

- **Northern Germany**: Präteritum is used more even in speech
- **Southern Germany, Austria, Switzerland**: Perfekt is strongly preferred in speech; Präteritum can sound formal or literary

> **Practical rule for learners**: Use **Perfekt** for everything in conversation, except switch to **Präteritum** for **sein**, **haben**, and **modal verbs**. This will sound natural in any region.

## 10. Plusquamperfekt: The Past Perfect

The Plusquamperfekt expresses an action that was completed **before** another past action — equivalent to English "had done."

```
hatte/war (Präteritum) + Partizip II
```

### Formation

| Auxiliary | Plusquamperfekt |
|-----------|----------------|
| haben → hatte | Ich **hatte** das Buch **gelesen**. (I had read the book.) |
| sein → war | Er **war** schon **gegangen**. (He had already left.) |

### Usage: Sequencing Past Events

The Plusquamperfekt shows what happened **first** (background), while Perfekt or Präteritum shows what happened **next**:

- **Nachdem** ich gefrühstückt **hatte**, ging ich zur Arbeit.
  (After I **had** eaten breakfast, I went to work.)

- Er **war** schon eingeschlafen, als ich nach Hause **kam**.
  (He **had** already fallen asleep when I came home.)

- Sie **hatte** den Brief schon geschrieben, bevor er **anrief**.
  (She **had** already written the letter before he called.)

> **Trigger words**: nachdem (after), bevor (before), als (when), schon (already), bereits (already)

## 12. Practice Exercises

### Exercise 1: Form the Partizip II

Give the Partizip II of the following verbs.

1. spielen (to play) → ________
2. trinken (to drink) → ________
3. verstehen (to understand) → ________
4. anfangen (to begin) → ________
5. studieren (to study) → ________
6. aufräumen (to tidy up) → ________
7. bekommen (to receive) → ________
8. lesen (to read) → ________

<details>
<summary>Answers</summary>

1. **gespielt** (regular: ge- + spiel + -t)
2. **getrunken** (strong: ge- + trunk + -en)
3. **verstanden** (inseparable prefix: no ge-)
4. **angefangen** (separable prefix: an + ge + fang + -en)
5. **studiert** (-ieren verb: no ge-)
6. **aufgeräumt** (separable prefix: auf + ge + räum + -t)
7. **bekommen** (inseparable prefix: no ge-)
8. **gelesen** (strong: ge- + les + -en)

</details>

### Exercise 2: Haben or Sein?

Choose the correct auxiliary.

1. Wir ________ (haben/sein) nach Paris geflogen.
2. Er ________ (haben/sein) den ganzen Tag gearbeitet.
3. ________ (Haben/Sein) du schon gegessen?
4. Sie ________ (haben/sein) gestern spät gekommen.
5. Ich ________ (haben/sein) mich sehr gefreut.
6. Die Kinder ________ (haben/sein) schnell eingeschlafen.

<details>
<summary>Answers</summary>

1. **sind** (fliegen = movement A→B, uses sein)
2. **hat** (arbeiten = no movement/state change, uses haben)
3. **Hast** (essen = transitive, uses haben)
4. **ist** (kommen = movement, uses sein)
5. **habe** (sich freuen = reflexive verb, uses haben)
6. **sind** (einschlafen = change of state, uses sein)

</details>

### Exercise 3: Präteritum Conjugation

Put the verbs into the Präteritum.

1. Ich ________ (sein) gestern krank.
2. Wir ________ (haben) keine Zeit.
3. Er ________ (können) nicht schwimmen.
4. Sie (pl.) ________ (müssen) lange warten.
5. Das Kind ________ (wollen) ein Eis.
6. Ich ________ (gehen) nach Hause.
7. Er ________ (sprechen) sehr leise.

<details>
<summary>Answers</summary>

1. **war** (sein → war)
2. **hatten** (haben → hatten)
3. **konnte** (können → konnte)
4. **mussten** (müssen → mussten)
5. **wollte** (wollen → wollte)
6. **ging** (gehen → ging, strong verb)
7. **sprach** (sprechen → sprach, strong verb)

</details>

### Exercise 4: Perfekt Sentences

Rewrite the sentences in the Perfekt.

1. Ich trinke Kaffee. → ________
2. Sie fährt nach München. → ________
3. Wir kaufen ein neues Auto. → ________
4. Er steht um 6 Uhr auf. → ________
5. Die Kinder spielen im Garten. → ________

<details>
<summary>Answers</summary>

1. Ich **habe** Kaffee **getrunken**. (haben + strong Partizip II)
2. Sie **ist** nach München **gefahren**. (sein + movement)
3. Wir **haben** ein neues Auto **gekauft**. (haben + regular Partizip II)
4. Er **ist** um 6 Uhr **aufgestanden**. (sein + separable prefix + change of state)
5. Die Kinder **haben** im Garten **gespielt**. (haben + regular Partizip II)

</details>

### Exercise 5: Plusquamperfekt

Complete the sentences using the Plusquamperfekt.

1. Nachdem ich ________ (essen), ging ich spazieren.
2. Er war müde, weil er schlecht ________ (schlafen).
3. Bevor sie anrief, ________ ich schon ________ (gehen).
4. Wir kamen zu spät, weil der Zug schon ________ (abfahren).

<details>
<summary>Answers</summary>

1. Nachdem ich **gegessen hatte**, ging ich spazieren. (hatte + gegessen)
2. Er war müde, weil er schlecht **geschlafen hatte**. (hatte + geschlafen)
3. Bevor sie anrief, **war** ich schon **gegangen**. (war + gegangen — movement)
4. Wir kamen zu spät, weil der Zug schon **abgefahren war**. (war + abgefahren — movement)

</details>

### Exercise 6: Mixed — Translate into German

Use the Perfekt unless a specific tense is indicated.

1. I was tired yesterday. (use Präteritum for sein)
2. She has written a letter.
3. We flew to Berlin last week.
4. He couldn't come. (use Präteritum for modal)
5. After I had eaten, I went to bed.

<details>
<summary>Answers</summary>

1. Ich **war** gestern müde. (Präteritum of sein)
2. Sie **hat** einen Brief **geschrieben**. (Perfekt with haben)
3. Wir **sind** letzte Woche nach Berlin **geflogen**. (Perfekt with sein — movement)
4. Er **konnte** nicht kommen. (Präteritum of modal können)
5. Nachdem ich **gegessen hatte**, **bin** ich ins Bett **gegangen**. (Plusquamperfekt + Perfekt)

</details>
