# 18. Passive Voice and Advanced Constructions (Passiv und fortgeschrittene Konstruktionen)

## Learning Objectives

- Form the Vorgangspassiv (process passive) with werden in all tenses
- Form the Zustandspassiv (state passive) with sein and distinguish it from process passive
- Use von + Dativ and durch + Akkusativ to express the agent in passive sentences
- Apply passive alternatives: man, sich lassen, sein + zu + Infinitiv
- Build relative clauses (Relativsätze) with correct pronoun case and word order
- Understand extended participial attributes in written German
- Use zu + Infinitiv constructions (um...zu, ohne...zu, (an)statt...zu)
- Recognize and form Konjunktiv I for indirect speech

## 1. Vorgangspassiv (Process Passive)

The Vorgangspassiv describes a **process** or **action being done**. It is formed with **werden + Partizip II**.

### 1.1 Formation Across Tenses

| Tense | Formation | Example |
|-------|-----------|---------|
| Präsens | wird + Partizip II | Das Buch **wird gelesen**. (The book is being read.) |
| Präteritum | wurde + Partizip II | Das Buch **wurde gelesen**. (The book was read.) |
| Perfekt | ist + Partizip II + **worden** | Das Buch **ist gelesen worden**. (The book has been read.) |
| Plusquamperfekt | war + Partizip II + worden | Das Buch **war gelesen worden**. (The book had been read.) |
| Futur I | wird + Partizip II + werden | Das Buch **wird gelesen werden**. (The book will be read.) |

> **Key point:** In Perfekt passive, the past participle of *werden* is **worden** (NOT *geworden*). This is the only place in German grammar where this shortened form appears.

### 1.2 Conjugation in Präsens Passiv

| Person | Passive Form |
|--------|-------------|
| ich | **werde** eingeladen |
| du | **wirst** eingeladen |
| er/sie/es | **wird** eingeladen |
| wir | **werden** eingeladen |
| ihr | **werdet** eingeladen |
| sie/Sie | **werden** eingeladen |

### 1.3 Passive with Modal Verbs

When a modal verb is used in a passive construction, the structure is:

```
Modal (conjugated) + Partizip II + werden (infinitive)
```

| German | English |
|--------|---------|
| Das Buch **muss gelesen werden**. | The book must be read. |
| Der Brief **kann morgen geschickt werden**. | The letter can be sent tomorrow. |
| Das Problem **sollte sofort gelöst werden**. | The problem should be solved immediately. |
| Das Fenster **darf nicht geöffnet werden**. | The window must not be opened. |

### 1.4 Active to Passive Transformation

| Active | Passive |
|--------|---------|
| Der Lehrer **korrigiert** die Prüfungen. | Die Prüfungen **werden** (vom Lehrer) **korrigiert**. |
| Man **baut** eine neue Brücke. | Eine neue Brücke **wird gebaut**. |
| Die Polizei **hat** den Dieb **gefangen**. | Der Dieb **ist** (von der Polizei) **gefangen worden**. |

> The accusative object of the active sentence becomes the nominative subject of the passive sentence. The former subject becomes the agent (von + Dativ) or is omitted.

### 1.5 Impersonal Passive (es-Passive)

Some passive sentences have no subject. **Es** can serve as a placeholder in first position:

| German | English |
|--------|---------|
| **Es wird** hier nicht **geraucht**. | There is no smoking here. |
| **Es wurde** viel **gelacht**. | There was a lot of laughing. |
| **Es wird** sonntags nicht **gearbeitet**. | There is no work on Sundays. |
| Hier **wird** getanzt. | Dancing happens here. (no *es* because *hier* fills first position) |

> Impersonal passive is possible even with intransitive verbs: *Es wurde getanzt und gesungen.* (There was dancing and singing.)

## 2. Zustandspassiv (State Passive)

The Zustandspassiv describes a **state** or **result** of a completed action. It is formed with **sein + Partizip II**.

### 2.1 Vorgangspassiv vs. Zustandspassiv

| Vorgangspassiv (Process) | Zustandspassiv (State) |
|--------------------------|----------------------|
| Das Fenster **wird geöffnet**. | Das Fenster **ist geöffnet**. |
| (The window is being opened.) | (The window is open.) |
| Die Tür **wurde geschlossen**. | Die Tür **war geschlossen**. |
| (The door was being closed.) | (The door was closed [= in a closed state].) |
| Die Arbeit **wird erledigt**. | Die Arbeit **ist erledigt**. |
| (The work is being done.) | (The work is done [= finished].) |

### 2.2 More Examples

| German | English |
|--------|---------|
| Das Auto **ist repariert**. | The car is repaired (= fixed). |
| Der Tisch **ist gedeckt**. | The table is set. |
| Die E-Mail **ist geschrieben**. | The email is written (= done). |
| Das Museum **ist geschlossen**. | The museum is closed. |
| Die Rechnung **ist bezahlt**. | The bill is paid. |

### 2.3 Tenses of Zustandspassiv

| Tense | Example |
|-------|---------|
| Präsens | Die Tür **ist** geschlossen. |
| Präteritum | Die Tür **war** geschlossen. |
| Perfekt | (rarely used — Präteritum preferred) |
| Futur I | Die Tür **wird** geschlossen **sein**. |

## 3. Agent in Passive: von vs. durch

### 3.1 von + Dativ (by a person or direct agent)

Used when the agent is a **person** or **direct cause**:

| German | English |
|--------|---------|
| Das Buch wurde **von** Goethe **geschrieben**. | The book was written by Goethe. |
| Der Patient wird **vom** Arzt **untersucht**. | The patient is examined by the doctor. |
| Die Kinder werden **von** den Eltern **abgeholt**. | The children are picked up by the parents. |

### 3.2 durch + Akkusativ (by means of / through)

Used when the agent is an **intermediary, instrument, or impersonal force**:

| German | English |
|--------|---------|
| Das Haus wurde **durch** ein Erdbeben **zerstört**. | The house was destroyed by an earthquake. |
| Die Nachricht wurde **durch** einen Boten **übermittelt**. | The message was conveyed by a messenger. |
| Die Stadt wurde **durch** einen Brand **verwüstet**. | The city was devastated by a fire. |

### 3.3 Comparison

| von (personal agent) | durch (means/instrument) |
|---------------------|------------------------|
| Der Brief wurde **von** meiner Kollegin geschrieben. | Der Brief wurde **durch** einen Kurier zugestellt. |
| (written by my colleague — direct doer) | (delivered by a courier — means of delivery) |

## 4. Passive Alternatives

German offers several ways to express passive meaning without using the werden-passive.

### 4.1 man (one / people / they)

The pronoun **man** is the most common passive alternative in everyday German:

| Passive | With man |
|---------|----------|
| Hier **wird** Deutsch **gesprochen**. | Hier **spricht man** Deutsch. |
| Das **wurde** schnell **erledigt**. | Das **hat man** schnell **erledigt**. |
| Es **wird** viel **diskutiert**. | **Man diskutiert** viel. |

### 4.2 sich lassen + Infinitiv (can be done)

Expresses that something **can be done** (possibility):

| German | English |
|--------|---------|
| Das **lässt sich** machen. | That can be done. |
| Das Problem **lässt sich** leicht **lösen**. | The problem can be easily solved. |
| Das **lässt sich** nicht **ändern**. | That can't be changed. |
| Diese Frage **lässt sich** schwer **beantworten**. | This question is hard to answer. |

### 4.3 sein + zu + Infinitiv (must/can be done)

Expresses necessity or possibility, depending on context:

| German | Meaning | English |
|--------|---------|---------|
| Die Aufgabe **ist** bis morgen **zu erledigen**. | necessity | The task must be completed by tomorrow. |
| Das Formular **ist** auszufüllen. | necessity | The form must be filled out. |
| Die Aussicht **ist** nicht **zu übertreffen**. | possibility (negative) | The view cannot be surpassed. |

### 4.4 Summary Comparison

| Construction | Example | Nuance |
|-------------|---------|--------|
| Vorgangspassiv | Das wird gemacht. | Neutral: It is being done. |
| man | Man macht das. | Colloquial: People do that. |
| sich lassen | Das lässt sich machen. | Possibility: That can be done. |
| sein + zu | Das ist zu machen. | Necessity: That is to be done. |

## 5. Relative Clauses (Relativsätze)

Relative clauses provide additional information about a noun. They are introduced by **relative pronouns** and the conjugated verb goes to the **end** of the clause.

### 5.1 Relative Pronoun Forms

The relative pronoun agrees with its antecedent in **gender and number**, but its **case** is determined by its function within the relative clause.

| Case | Masculine | Feminine | Neuter | Plural |
|------|-----------|----------|--------|--------|
| Nominativ | der | die | das | die |
| Akkusativ | den | die | das | die |
| Dativ | dem | der | dem | denen |
| Genitiv | dessen | deren | dessen | deren |

> These forms are identical to the definite articles except in the **dative plural** (denen) and **genitive** (dessen/deren).

### 5.2 Relative Clauses by Case

**Nominativ** — the relative pronoun is the subject:

| German | English |
|--------|---------|
| Der Mann, **der** dort **steht**, ist mein Lehrer. | The man who is standing there is my teacher. |
| Die Frau, **die** dort **wohnt**, ist Ärztin. | The woman who lives there is a doctor. |
| Das Kind, **das** dort **spielt**, ist mein Sohn. | The child who is playing there is my son. |

**Akkusativ** — the relative pronoun is a direct object:

| German | English |
|--------|---------|
| Der Film, **den** ich gestern **gesehen habe**, war gut. | The film that I saw yesterday was good. |
| Die Frau, **die** ich **kenne**, arbeitet bei Siemens. | The woman whom I know works at Siemens. |
| Das Buch, **das** du mir **empfohlen hast**, ist toll. | The book you recommended to me is great. |

**Dativ** — the relative pronoun is an indirect object:

| German | English |
|--------|---------|
| Der Kollege, **dem** ich **geholfen habe**, ist dankbar. | The colleague whom I helped is grateful. |
| Die Freundin, **der** ich das Buch **gegeben habe**, lebt in Wien. | The friend to whom I gave the book lives in Vienna. |
| Die Kinder, **denen** wir **Geschenke gebracht haben**, waren glücklich. | The children to whom we brought gifts were happy. |

**Genitiv** — the relative pronoun shows possession (whose):

| German | English |
|--------|---------|
| Der Mann, **dessen** Auto vor der Tür **steht**, ist mein Nachbar. | The man whose car is in front of the door is my neighbor. |
| Die Frau, **deren** Tochter ich **kenne**, arbeitet hier. | The woman whose daughter I know works here. |
| Die Leute, **deren** Haus wir **gekauft haben**, sind umgezogen. | The people whose house we bought have moved. |

### 5.3 Relative Pronouns with Prepositions

When a preposition is required, it is placed **before** the relative pronoun:

| German | English |
|--------|---------|
| Das Buch, **über das** ich **spreche**, ist sehr interessant. | The book about which I'm speaking is very interesting. |
| Die Stadt, **in der** ich **lebe**, ist schön. | The city in which I live is beautiful. |
| Der Freund, **mit dem** ich **reise**, kommt aus Berlin. | The friend with whom I'm traveling comes from Berlin. |
| Die Firma, **für die** ich **arbeite**, ist international. | The company for which I work is international. |
| Der Grund, **aus dem** er **gegangen ist**, ist unklar. | The reason for which he left is unclear. |

### 5.4 Word Order in Relative Clauses

The conjugated verb goes to the **end** of the relative clause:

```
..., Relativpronomen + ... + Verb (am Ende).
```

| German | English |
|--------|---------|
| Das ist der Mann, **der gestern angekommen ist**. | That is the man who arrived yesterday. |
| Ich kenne die Frau, **die dort an der Ecke steht**. | I know the woman who is standing there on the corner. |
| Das Haus, **in dem wir früher gewohnt haben**, wird renoviert. | The house in which we used to live is being renovated. |

## 6. Extended Participial Attributes

This construction is characteristic of formal written German (academic texts, journalism, official documents). An entire relative clause is compressed into an adjectival phrase before the noun.

### 6.1 Structure

```
article + [adverbial modifiers] + participle (with adjective ending) + noun
```

### 6.2 Conversion from Relative Clause

| Relative Clause | Participial Attribute |
|-----------------|----------------------|
| die Familie, **die in Berlin lebt** | die **in Berlin lebende** Familie |
| der **gestern geschriebene** Brief | der Brief, **der gestern geschrieben wurde** |
| das **noch nicht gelöste** Problem | das Problem, **das noch nicht gelöst worden ist** |
| die **vor drei Jahren gebaute** Brücke | die Brücke, **die vor drei Jahren gebaut wurde** |
| die **an der Konferenz teilnehmenden** Wissenschaftler | die Wissenschaftler, **die an der Konferenz teilnehmen** |

### 6.3 Rules

- **Partizip I** (laufend, schlafend) replaces an **active** relative clause (present meaning)
- **Partizip II** (gelesen, geöffnet) replaces a **passive** relative clause (completed action)
- The participle takes the same adjective ending it would have in normal attributive position
- All modifiers (adverbs, prepositional phrases) go between the article and the participle

### 6.4 Reading Strategy

When encountering a long participial attribute, work from the outside in:

1. Find the **article** (der, die, das, ein, etc.)
2. Find the **noun** it belongs to
3. Everything between the article and noun is the participial attribute
4. Identify the **participle** (just before the noun) — it carries the main meaning
5. Read the modifiers as context for the participle

Example: **die** seit vielen Jahren in Deutschland **lebenden** ausländischen **Studenten**
- article: die
- noun: Studenten
- participle: lebenden (living)
- modifiers: seit vielen Jahren (for many years), in Deutschland (in Germany), ausländischen (foreign)
- = "the foreign students who have been living in Germany for many years"

## 7. zu + Infinitiv Constructions

German uses **zu + Infinitiv** in several important constructions.

### 7.1 um ... zu + Infinitiv (in order to)

Expresses purpose. The subject of both clauses must be the same.

| German | English |
|--------|---------|
| Ich lerne Deutsch, **um** in Deutschland **zu arbeiten**. | I'm learning German (in order) to work in Germany. |
| Er spart Geld, **um** ein Haus **zu kaufen**. | He saves money (in order) to buy a house. |
| Sie ging in die Bibliothek, **um** ein Buch **zu leihen**. | She went to the library to borrow a book. |

> With separable verbs, **zu** goes between the prefix and the stem: **um ein**zu**kaufen**, **um an**zu**rufen**, **um auf**zu**stehen**.

### 7.2 ohne ... zu + Infinitiv (without)

| German | English |
|--------|---------|
| Er ging, **ohne** sich **zu verabschieden**. | He left without saying goodbye. |
| Sie hat bestanden, **ohne** viel **zu lernen**. | She passed without studying much. |
| Man kann kein Deutsch lernen, **ohne** Grammatik **zu üben**. | You can't learn German without practicing grammar. |

### 7.3 (an)statt ... zu + Infinitiv (instead of)

| German | English |
|--------|---------|
| **Anstatt zu arbeiten**, spielt er Computerspiele. | Instead of working, he plays computer games. |
| **Statt** ein Taxi **zu nehmen**, sind wir gelaufen. | Instead of taking a taxi, we walked. |
| Er hat ferngesehen, **anstatt zu lernen**. | He watched TV instead of studying. |

### 7.4 Adjective + zu + Infinitiv

| German | English |
|--------|---------|
| Es ist **schwer zu verstehen**. | It is hard to understand. |
| Das Buch ist **leicht zu lesen**. | The book is easy to read. |
| Es ist **wichtig**, regelmäßig **zu üben**. | It is important to practice regularly. |
| Es ist **unmöglich**, alles **zu wissen**. | It is impossible to know everything. |

### 7.5 Verb + zu + Infinitiv

Many verbs require **zu** before a dependent infinitive:

| Verb | Example | English |
|------|---------|---------|
| anfangen zu | Er **fängt an zu** sprechen. | He starts to speak. |
| aufhören zu | Hör **auf zu** reden! | Stop talking! |
| versuchen zu | Ich **versuche** ihn **zu** erreichen. | I'm trying to reach him. |
| vergessen zu | Vergiss nicht, mich **anzurufen**! | Don't forget to call me! |
| hoffen zu | Wir **hoffen**, Sie bald **wiederzusehen**. | We hope to see you again soon. |
| beginnen zu | Es **beginnt zu** regnen. | It's starting to rain. |
| scheinen zu | Er **scheint zu** schlafen. | He seems to be sleeping. |
| vorhaben zu | Ich **habe vor**, morgen **abzureisen**. | I plan to leave tomorrow. |

> **Note:** Modal verbs (können, müssen, etc.) and a few other verbs (lassen, sehen, hören, gehen, fahren, lernen) use the infinitive **without** zu: Ich **kann** schwimmen. Er **lässt** mich warten.

## 8. Konjunktiv I: Indirect Speech

Konjunktiv I is used primarily in **indirect speech** (indirekte Rede), especially in journalism, academic writing, and formal reporting. It signals that the speaker is reporting someone else's words without vouching for their truth.

### 8.1 Formation

Konjunktiv I is formed from the **infinitive stem** + subjunctive endings:

| Person | sein | haben | kommen | wissen |
|--------|------|-------|--------|--------|
| ich | sei | habe | komme | wisse |
| du | sei(e)st | habest | kommest | wissest |
| er/sie/es | **sei** | **habe** | **komme** | **wisse** |
| wir | seien | haben | kommen | wissen |
| ihr | sei(e)t | habet | kommet | wisset |
| sie/Sie | seien | haben | kommen | wissen |

> The most important form is the **third person singular** (er/sie/es), which is always distinct from the indicative (ist→sei, hat→habe, kommt→komme).

### 8.2 When Konjunktiv I = Indicative

When Konjunktiv I is identical to the indicative (as in *ich*, *wir*, *sie/Sie* for most verbs), use **Konjunktiv II** as a replacement:

| Person | Indicative | Konjunktiv I | Replacement (Konj. II) |
|--------|-----------|-------------|----------------------|
| ich | komme | komme (same!) | **käme** |
| wir | kommen | kommen (same!) | **kämen** / **würden kommen** |
| sie (pl.) | kommen | kommen (same!) | **kämen** / **würden kommen** |

### 8.3 Usage in Indirect Speech

| Direct Speech | Indirect Speech (Konjunktiv I) |
|--------------|-------------------------------|
| Er sagt: „Ich **bin** krank." | Er sagt, er **sei** krank. |
| Sie sagt: „Ich **habe** keine Zeit." | Sie sagt, sie **habe** keine Zeit. |
| Er sagt: „Ich **komme** morgen." | Er sagt, er **komme** morgen. |
| Sie sagte: „Ich **war** in Berlin." | Sie sagte, sie **sei** in Berlin **gewesen**. |
| Er sagte: „Ich **habe** das nicht **gewusst**." | Er sagte, er **habe** das nicht **gewusst**. |

### 8.4 Konjunktiv I in Journalism

In German news reporting, Konjunktiv I is standard:

| German | English |
|--------|---------|
| Der Minister **sagte**, die Reform **sei** notwendig. | The minister said the reform was necessary. |
| Die Polizei **teilte mit**, der Verdächtige **habe** die Stadt **verlassen**. | The police reported that the suspect had left the city. |
| Der Bericht **besagt**, die Zahlen **seien** gestiegen. | The report states that the numbers have risen. |

> In everyday spoken German, Konjunktiv I is rarely used. Speakers typically use the indicative with **dass**: *Er sagt, **dass** er krank **ist**.* Konjunktiv I is mainly a feature of formal written German.

## 9. Practice Exercises

### Exercise 1: Vorgangspassiv

Transform from active to passive (Präsens).

1. Der Koch bereitet das Essen zu.
2. Die Studenten schreiben die Prüfung.
3. Man spricht hier Deutsch.
4. Der Mechaniker repariert das Auto.

<details>
<summary>Answers</summary>

1. Das Essen **wird** (vom Koch) **zubereitet**.
2. Die Prüfung **wird** (von den Studenten) **geschrieben**.
3. Hier **wird** Deutsch **gesprochen**.
4. Das Auto **wird** (vom Mechaniker) **repariert**.

</details>

### Exercise 2: Passive Tenses

Rewrite the sentence in the indicated tense.

Base: Das Haus wird gebaut. (Präsens)

1. Präteritum: ______
2. Perfekt: ______
3. With modal (müssen): ______
4. Futur I: ______

<details>
<summary>Answers</summary>

1. Das Haus **wurde gebaut**.
2. Das Haus **ist gebaut worden**.
3. Das Haus **muss gebaut werden**.
4. Das Haus **wird gebaut werden**.

</details>

### Exercise 3: Vorgangspassiv vs. Zustandspassiv

Choose the correct form: wird (Vorgang) or ist (Zustand).

1. Das Fenster ______ gerade geöffnet. (Someone is opening it right now.)
2. Das Fenster ______ geöffnet. (It is in an open state.)
3. Die Suppe ______ gerade gekocht. (Someone is cooking it now.)
4. Das Essen ______ fertig gekocht. (It is ready/done.)
5. Der Brief ______ gerade geschrieben. (Someone is writing it.)
6. Der Brief ______ schon geschrieben. (It has been done already.)

<details>
<summary>Answers</summary>

1. **wird** (process — someone is doing it)
2. **ist** (state — result)
3. **wird** (process)
4. **ist** (state)
5. **wird** (process)
6. **ist** (state)

</details>

### Exercise 4: Relative Clauses

Fill in the correct relative pronoun.

1. Der Mann, ______ dort steht, ist mein Bruder. (Nom. m)
2. Die Frau, ______ ich gestern getroffen habe, ist Lehrerin. (Akk. f)
3. Das Kind, ______ ich ein Buch geschenkt habe, war glücklich. (Dat. n)
4. Der Kollege, ______ Auto kaputt ist, fährt mit dem Bus. (Gen. m)
5. Die Leute, mit ______ ich gesprochen habe, waren sehr nett. (Dat. pl)
6. Das Thema, über ______ wir diskutieren, ist wichtig. (Akk. n)

<details>
<summary>Answers</summary>

1. **der** (Nom. masculine)
2. **die** (Akk. feminine)
3. **dem** (Dat. neuter)
4. **dessen** (Gen. masculine)
5. **denen** (Dat. plural)
6. **das** (Akk. neuter, after preposition *über*)

</details>

### Exercise 5: zu + Infinitiv Constructions

Combine the sentences using um...zu, ohne...zu, or (an)statt...zu.

1. Er lernt Deutsch. Er will in Deutschland arbeiten. (um...zu)
2. Sie ging weg. Sie hat sich nicht verabschiedet. (ohne...zu)
3. Er spielt Videospiele. Er sollte lernen. (anstatt...zu)
4. Wir sind früh aufgestanden. Wir wollten den Sonnenaufgang sehen. (um...zu)

<details>
<summary>Answers</summary>

1. Er lernt Deutsch, **um** in Deutschland **zu arbeiten**.
2. Sie ging weg, **ohne** sich **zu verabschieden**.
3. **Anstatt zu lernen**, spielt er Videospiele.
4. Wir sind früh aufgestanden, **um** den Sonnenaufgang **zu sehen**.

</details>

### Exercise 6: Passive Alternatives

Rewrite using the indicated alternative.

1. Die Tür wird um 8 Uhr geschlossen. (man) → ______
2. Das kann leicht erklärt werden. (sich lassen) → ______
3. Der Bericht muss bis Freitag abgegeben werden. (sein + zu) → ______
4. Dieses Wort wird so ausgesprochen. (man) → ______

<details>
<summary>Answers</summary>

1. **Man schließt** die Tür um 8 Uhr.
2. Das **lässt sich** leicht **erklären**.
3. Der Bericht **ist** bis Freitag **abzugeben**.
4. **Man spricht** dieses Wort so **aus**.

</details>

### Exercise 7: Konjunktiv I (Indirect Speech)

Rewrite in indirect speech using Konjunktiv I.

1. Er sagt: „Ich bin müde." → Er sagt, er ______ müde.
2. Sie sagt: „Ich habe keine Zeit." → Sie sagt, sie ______ keine Zeit.
3. Er sagt: „Ich komme morgen." → Er sagt, er ______ morgen.
4. Sie sagt: „Ich war gestern in Berlin." → Sie sagt, sie ______ gestern in Berlin ______.

<details>
<summary>Answers</summary>

1. Er sagt, er **sei** müde.
2. Sie sagt, sie **habe** keine Zeit.
3. Er sagt, er **komme** morgen.
4. Sie sagt, sie **sei** gestern in Berlin **gewesen**.

</details>
