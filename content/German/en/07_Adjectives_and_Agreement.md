## 1. Predicate vs. Attributive Adjectives

German adjectives behave differently depending on their position in the sentence.

### 1.1 Predicate Adjectives (No Ending)

When an adjective comes **after** a verb like *sein* (to be), *werden* (to become), or *bleiben* (to remain), it takes **no ending** at all:

- Das Haus ist **groß**. — The house is big.
- Der Mann ist **alt**. — The man is old.
- Die Blumen sind **schön**. — The flowers are beautiful.
- Das Wetter wird **kalt**. — The weather is getting cold.

> **Rule**: Predicate adjectives are easy — they never change form.

### 1.2 Attributive Adjectives (Before the Noun — Must Decline)

When an adjective comes **before** a noun, it **must** take an ending that reflects the gender, number, and case of the noun:

- Das **große** Haus — The big house
- Der **alte** Mann — The old man
- Die **schönen** Blumen — The beautiful flowers
- Ein **kalter** Tag — A cold day

The ending depends on three factors:
1. **Gender** of the noun (masculine, feminine, neuter)
2. **Case** (nominative, accusative, dative, genitive)
3. **What precedes the adjective** (definite article, indefinite article, or nothing)

This gives us three declension patterns: **weak**, **mixed**, and **strong**.

## 2. Weak Declension (After Definite Articles)

Use the **weak declension** when the adjective follows a **definite article** (der, die, das) or a similar word that already shows the gender/case signal clearly (dieser, jeder, jener, welcher, alle).

### 2.1 Weak Declension Table

| | Masculine | Feminine | Neuter | Plural |
|---|-----------|----------|--------|--------|
| **Nominative** | der alt**e** Mann | die alt**e** Frau | das alt**e** Haus | die alt**en** Häuser |
| **Accusative** | den alt**en** Mann | die alt**e** Frau | das alt**e** Haus | die alt**en** Häuser |
| **Dative** | dem alt**en** Mann | der alt**en** Frau | dem alt**en** Haus | den alt**en** Häusern |
| **Genitive** | des alt**en** Mannes | der alt**en** Frau | des alt**en** Hauses | der alt**en** Häuser |

### 2.2 The Pattern

The weak declension uses only two endings:

- **-e** in five positions (nominative masculine/feminine/neuter, accusative feminine/neuter)
- **-en** everywhere else

```
+----------------------------------------------+
|  WEAK DECLENSION (after der/die/das)          |
+----------------------------------------------+
|         Masc.   Fem.   Neut.   Plural        |
|  Nom.    -e      -e     -e      -en          |
|  Acc.    -en     -e     -e      -en          |
|  Dat.    -en     -en    -en     -en          |
|  Gen.    -en     -en    -en     -en          |
+----------------------------------------------+
|  Memory: -e in nom. (all genders) +           |
|          acc. fem./neut. — everything else -en |
+----------------------------------------------+
```

### 2.3 Examples

- **Nom.**: Der **neue** Computer funktioniert gut. — The new computer works well.
- **Acc.**: Ich kaufe den **neuen** Computer. — I buy the new computer.
- **Dat.**: Er arbeitet mit dem **neuen** Computer. — He works with the new computer.
- **Nom. pl.**: Die **kleinen** Kinder spielen draußen. — The small children play outside.

## 3. Mixed Declension (After Indefinite Articles)

Use the **mixed declension** after an **indefinite article** (ein, eine) or possessive pronouns (mein, dein, sein, ihr, unser, euer, Ihr) and the negative article (kein).

### 3.1 Why "Mixed"?

The indefinite article does not always show the gender/case clearly. In those positions, the adjective must carry a **strong ending** to compensate. In other positions where the article does show the signal, the adjective uses a **weak ending**.

### 3.2 Mixed Declension Table

| | Masculine | Feminine | Neuter | Plural (keine/meine) |
|---|-----------|----------|--------|--------|
| **Nominative** | ein alt**er** Mann | eine alt**e** Frau | ein alt**es** Haus | keine alt**en** Häuser |
| **Accusative** | einen alt**en** Mann | eine alt**e** Frau | ein alt**es** Haus | keine alt**en** Häuser |
| **Dative** | einem alt**en** Mann | einer alt**en** Frau | einem alt**en** Haus | keinen alt**en** Häusern |
| **Genitive** | eines alt**en** Mannes | einer alt**en** Frau | eines alt**en** Hauses | keiner alt**en** Häuser |

### 3.3 The Pattern

```
+----------------------------------------------+
|  MIXED DECLENSION (after ein/mein/kein)       |
+----------------------------------------------+
|         Masc.   Fem.   Neut.   Plural        |
|  Nom.    -er     -e     -es     -en          |
|  Acc.    -en     -e     -es     -en          |
|  Dat.    -en     -en    -en     -en          |
|  Gen.    -en     -en    -en     -en          |
+----------------------------------------------+
|  Key insight: Strong endings (-er, -es) appear |
|  where ein/mein/kein lack a gender signal      |
|  (nom. masc. & nom./acc. neuter)              |
+----------------------------------------------+
```

### 3.4 Examples

- **Nom.**: Ein **junger** Mann steht an der Tür. — A young man stands at the door.
- **Acc.**: Sie sucht einen **guten** Job. — She is looking for a good job.
- **Nom. neut.**: Mein **kleines** Kind schläft. — My small child is sleeping.
- **Acc. fem.**: Er kauft eine **rote** Rose. — He buys a red rose.
- **Dat.**: Ich wohne in einem **alten** Haus. — I live in an old house.

## 4. Strong Declension (No Article)

Use the **strong declension** when **no article** precedes the adjective. The adjective itself must carry the full gender/case signal.

### 4.1 Strong Declension Table

| | Masculine | Feminine | Neuter | Plural |
|---|-----------|----------|--------|--------|
| **Nominative** | alt**er** Mann | alt**e** Frau | alt**es** Haus | alt**e** Häuser |
| **Accusative** | alt**en** Mann | alt**e** Frau | alt**es** Haus | alt**e** Häuser |
| **Dative** | alt**em** Mann | alt**er** Frau | alt**em** Haus | alt**en** Häusern |
| **Genitive** | alt**en** Mannes | alt**er** Frau | alt**en** Hauses | alt**er** Häuser |

### 4.2 The Pattern

```
+----------------------------------------------+
|  STRONG DECLENSION (no article)               |
+----------------------------------------------+
|         Masc.   Fem.   Neut.   Plural        |
|  Nom.    -er     -e     -es     -e           |
|  Acc.    -en     -e     -es     -e           |
|  Dat.    -em     -er    -em     -en          |
|  Gen.    -en     -er    -en     -er          |
+----------------------------------------------+
|  The endings mirror the definite article       |
|  endings (der/die/das) — the adjective takes  |
|  over the article's job!                      |
+----------------------------------------------+
```

### 4.3 When Does This Occur?

Strong declension appears after:
- No article at all: **Kalter** Kaffee schmeckt nicht. (Cold coffee doesn't taste good.)
- Numbers: Zwei **kleine** Kinder spielen. (Two small children play.)
- *viel* / *wenig* (without endings): **viel guter** Wein (much good wine)
- *einige*, *mehrere*, *andere*: einige **neue** Bücher (some new books)

### 4.4 Examples

- **Nom.**: **Frischer** Wind weht heute. — Fresh wind is blowing today.
- **Acc.**: Ich trinke **kalten** Tee. — I drink cold tea.
- **Dat.**: Mit **großem** Interesse habe ich das gelesen. — I read that with great interest.
- **Nom. pl.**: **Kleine** Kinder brauchen viel Schlaf. — Small children need a lot of sleep.

## 5. Declension Summary and Strategy

### 5.1 The Core Principle

The noun phrase must always show the gender/case signal **somewhere** — either in the article or in the adjective ending:

1. **Definite article present** (der/die/das) → Article shows the signal → Adjective takes **weak** endings (-e or -en)
2. **Indefinite article present** (ein/mein/kein) → Article sometimes shows signal, sometimes not → Adjective uses **mixed** endings (weak where article signals, strong where it does not)
3. **No article** → Adjective must show the signal → Adjective takes **strong** endings (matching the definite article endings)

### 5.2 Quick Decision Flowchart

```
Is there an article before the adjective?
├── YES: Is it der/die/das (or dieser/jeder/welcher)?
│   └── YES → WEAK declension (-e / -en)
│   └── NO (ein/mein/kein) → MIXED declension
└── NO → STRONG declension
```

## 6. Comparison: Komparativ and Superlativ

### 6.1 Comparative (Komparativ)

Add **-er** to the adjective stem:

| Adjective | Comparative | English |
|-----------|-------------|---------|
| schnell | schnell**er** | faster |
| klein | klein**er** | smaller |
| langsam | langsam**er** | slower |
| billig | billig**er** | cheaper |
| schön | schön**er** | more beautiful |

**Comparative in sentences**:

- Er ist **schneller** als ich. — He is faster than me.
- Dieses Buch ist **interessanter** als jenes. — This book is more interesting than that one.
- Berlin ist **größer** als München. — Berlin is bigger than Munich.

> **Note**: German always uses the **-er** suffix, even with long adjectives. There is no equivalent of English "more interesting" — it is always *interessanter*.

**Comparative as attributive adjective** (before a noun, it still takes declension endings):

- Ein **schnelleres** Auto. — A faster car. (neuter nom./acc.)
- Der **ältere** Bruder. — The older brother. (masc. nom.)

### 6.2 Superlative (Superlativ)

Add **-st** or **-est** to the adjective stem:

| Adjective | Superlative | English |
|-----------|-------------|---------|
| schnell | schnell**st-** | fastest |
| klein | klein**st-** | smallest |
| schön | schön**st-** | most beautiful |
| alt | älte**st-** | oldest |
| kalt | kälte**st-** | coldest |

> **Use -est** when the adjective ends in **-d, -t, -s, -ß, -sch, -z**: der kälte**st**e Tag, das intere**ss**ante**st**e Buch.

**Two superlative forms**:

1. **Attributive** (before noun): Takes adjective endings
   - Das ist der **schnellste** Zug. — That is the fastest train.
   - Sie ist die **beste** Schülerin. — She is the best student.

2. **Predicate** (after *sein*): Uses **am + -sten**
   - Dieser Zug ist **am schnellsten**. — This train is the fastest.
   - Im Winter ist es **am kältesten**. — In winter it is the coldest.

### 6.3 Comparison Structures

| Structure | German | English |
|-----------|--------|---------|
| more ... than | Komparativ + **als** | -er ... than |
| as ... as | **so** + Positiv + **wie** | as ... as |
| not as ... as | **nicht so** + Positiv + **wie** | not as ... as |

Examples:
- Er ist **größer als** sein Vater. — He is taller than his father.
- Sie ist **so intelligent wie** ihr Bruder. — She is as intelligent as her brother.
- Dieses Hotel ist **nicht so teuer wie** das andere. — This hotel is not as expensive as the other one.

## 7. Irregular Comparisons and Umlaut Changes

### 7.1 Completely Irregular Forms

| Positive | Komparativ | Superlativ | English |
|----------|------------|------------|---------|
| gut | **besser** | **am besten** / der **beste** | good → better → best |
| viel | **mehr** | **am meisten** / der **meiste** | much → more → most |
| gern | **lieber** | **am liebsten** | gladly → rather → most gladly |
| hoch | **höher** | **am höchsten** / der **höchste** | high → higher → highest |
| nah | **näher** | **am nächsten** / der **nächste** | near → nearer → nearest |

### 7.2 Umlaut in Comparison

Many common one-syllable adjectives gain an **umlaut** (a→ä, o→ö, u→ü) in comparative and superlative:

| Positive | Komparativ | Superlativ | English |
|----------|------------|------------|---------|
| alt | **älter** | **am ältesten** | old → older → oldest |
| groß | **größer** | **am größten** | big → bigger → biggest |
| jung | **jünger** | **am jüngsten** | young → younger → youngest |
| kalt | **kälter** | **am kältesten** | cold → colder → coldest |
| warm | **wärmer** | **am wärmsten** | warm → warmer → warmest |
| lang | **länger** | **am längsten** | long → longer → longest |
| kurz | **kürzer** | **am kürzesten** | short → shorter → shortest |
| stark | **stärker** | **am stärksten** | strong → stronger → strongest |
| klug | **klüger** | **am klügsten** | clever → cleverer → cleverest |
| arm | **ärmer** | **am ärmsten** | poor → poorer → poorest |
| krank | **kränker** | **am kränksten** | sick → sicker → sickest |
| dumm | **dümmer** | **am dümmsten** | dumb → dumber → dumbest |

> **Note**: Not all one-syllable adjectives take an umlaut. For example: *bunt* → *bunter* (not *bünter*), *schlank* → *schlanker* (not *schlänker*). You must memorize which ones take the umlaut.

## 8. Adjectives as Nouns

German frequently turns adjectives into nouns. These **substantivierte Adjektive** (nominalized adjectives) are capitalized and still decline like attributive adjectives:

### 8.1 With Definite Articles

| German | English | Declension |
|--------|---------|------------|
| der **Deutsche** | the German (man) | weak masc. nom. |
| die **Deutsche** | the German (woman) | weak fem. nom. |
| die **Deutschen** | the Germans | weak plural |
| der **Alte** | the old man | weak masc. nom. |
| die **Alte** | the old woman | weak fem. nom. |
| der **Kranke** | the sick person (m.) | weak masc. nom. |
| der **Bekannte** | the acquaintance (m.) | weak masc. nom. |
| die **Verwandte** | the relative (f.) | weak fem. nom. |

### 8.2 With Indefinite Articles

| German | English | Declension |
|--------|---------|------------|
| ein **Deutscher** | a German (man) | mixed masc. nom. |
| eine **Deutsche** | a German (woman) | mixed fem. nom. |
| ein **Bekannter** | an acquaintance (m.) | mixed masc. nom. |
| eine **Bekannte** | an acquaintance (f.) | mixed fem. nom. |

### 8.3 With etwas, nichts, viel, wenig

After *etwas* (something), *nichts* (nothing), *viel* (much), *wenig* (little), the adjective-noun takes **strong neuter** endings:

- etwas **Neues** — something new
- nichts **Besonderes** — nothing special
- viel **Gutes** — much good (many good things)
- wenig **Interessantes** — little of interest

## 10. Practice Exercises

### Exercise 1: Predicate or Attributive?

Decide if the adjective needs an ending. Write the correct form.

1. Das Wetter ist ________ (kalt).
2. Ein ________ Tag (kalt).
3. Die Suppe ist ________ (heiß).
4. Ich trinke ________ Suppe (heiß, with eine).
5. Die Kinder sind ________ (müde).
6. Die ________ Kinder schlafen (müde).

<details>
<summary>Answers</summary>

1. Das Wetter ist **kalt**. (predicate — no ending)
2. Ein **kalter** Tag. (attributive, mixed nom. masc. — -er)
3. Die Suppe ist **heiß**. (predicate — no ending)
4. Ich trinke eine **heiße** Suppe. (attributive, mixed acc. fem. — -e)
5. Die Kinder sind **müde**. (predicate — no ending)
6. Die **müden** Kinder schlafen. (attributive, weak nom. pl. — -en)

</details>

### Exercise 2: Weak Declension (der/die/das)

Fill in the correct adjective ending.

1. Der neu________ Film ist gut. (masc. nom.)
2. Ich lese das interessant________ Buch. (neut. acc.)
3. Die klein________ Kinder spielen. (pl. nom.)
4. Er kauft den teur________ Wein. (masc. acc.)
5. Wir wohnen in dem alt________ Haus. (neut. dat.)

<details>
<summary>Answers</summary>

1. Der neu**e** Film ist gut. (weak masc. nom. = -e)
2. Ich lese das interessant**e** Buch. (weak neut. acc. = -e)
3. Die klein**en** Kinder spielen. (weak pl. nom. = -en)
4. Er kauft den teur**en** Wein. (weak masc. acc. = -en)
5. Wir wohnen in dem alt**en** Haus. (weak neut. dat. = -en)

</details>

### Exercise 3: Mixed Declension (ein/mein/kein)

Fill in the correct adjective ending.

1. Ein jung________ Mann wartet draußen. (masc. nom.)
2. Sie hat ein klein________ Kind. (neut. acc.)
3. Ich suche einen gut________ Arzt. (masc. acc.)
4. Er wohnt in einer groß________ Stadt. (fem. dat.)
5. Mein alt________ Auto ist kaputt. (neut. nom.)

<details>
<summary>Answers</summary>

1. Ein jung**er** Mann wartet draußen. (mixed masc. nom. = -er)
2. Sie hat ein klein**es** Kind. (mixed neut. acc. = -es)
3. Ich suche einen gut**en** Arzt. (mixed masc. acc. = -en)
4. Er wohnt in einer groß**en** Stadt. (mixed fem. dat. = -en)
5. Mein alt**es** Auto ist kaputt. (mixed neut. nom. = -es)

</details>

### Exercise 4: Strong Declension (No Article)

Fill in the correct adjective ending.

1. Kalt________ Wasser ist erfrischend. (neut. nom.)
2. Ich trinke gern warm________ Milch. (fem. acc.)
3. Bei schlecht________ Wetter bleiben wir zu Hause. (neut. dat.)
4. Frisch________ Brot schmeckt am besten. (neut. nom.)
5. Deutsch________ Bier ist weltberühmt. (neut. nom.)

<details>
<summary>Answers</summary>

1. Kalt**es** Wasser ist erfrischend. (strong neut. nom. = -es)
2. Ich trinke gern warm**e** Milch. (strong fem. acc. = -e)
3. Bei schlecht**em** Wetter bleiben wir zu Hause. (strong neut. dat. = -em)
4. Frisch**es** Brot schmeckt am besten. (strong neut. nom. = -es)
5. Deutsch**es** Bier ist weltberühmt. (strong neut. nom. = -es)

</details>

### Exercise 5: Comparatives and Superlatives

Complete the sentence with the correct comparative or superlative form.

1. Berlin ist ________ als München. (groß)
2. Mein Bruder ist ________ als ich. (alt)
3. Dieses Restaurant ist ________ (am + gut).
4. Sie spricht ________ Deutsch als ich. (gut)
5. Der Mount Everest ist der ________ Berg der Welt. (hoch)
6. Im Sommer sind die Tage ________ als im Winter. (lang)

<details>
<summary>Answers</summary>

1. Berlin ist **größer** als München.
2. Mein Bruder ist **älter** als ich.
3. Dieses Restaurant ist **am besten**.
4. Sie spricht **besser** Deutsch als ich.
5. Der Mount Everest ist der **höchste** Berg der Welt.
6. Im Sommer sind die Tage **länger** als im Winter.

</details>

### Exercise 6: Translation Practice

Translate into German.

1. The old house is beautiful. (use predicate adjective)
2. I see the small child. (das Kind, weak declension)
3. She has a new car. (das Auto, mixed declension)
4. Cold coffee tastes bad. (strong declension)
5. Something new is happening. (etwas + adjective as noun)

<details>
<summary>Answers</summary>

1. Das alte Haus ist **schön**.
2. Ich sehe das **kleine** Kind.
3. Sie hat ein **neues** Auto.
4. **Kalter** Kaffee schmeckt schlecht.
5. **Etwas Neues** passiert.

</details>

## Quick Reference

```
+----------------------------------------------------------+
|         ADJECTIVE DECLENSION CHEAT SHEET                  |
+----------------------------------------------------------+
|                                                          |
|  PREDICATE (after sein/werden): NO ending                |
|    Das Haus ist groß.                                    |
|                                                          |
|  ATTRIBUTIVE (before noun): MUST have ending             |
|                                                          |
|  WEAK (after der/die/das):  mostly -e and -en            |
|    Nom: -e (all)  Acc: -en (m), -e (f/n)                |
|    Dat/Gen: always -en                                   |
|                                                          |
|  MIXED (after ein/mein/kein):                            |
|    Same as weak EXCEPT:                                  |
|    Nom. masc: -er, Nom/Acc. neut: -es                   |
|                                                          |
|  STRONG (no article): mirrors der/die/das endings        |
|    Nom: -er (m), -e (f), -es (n), -e (pl)               |
|                                                          |
|  COMPARISON:                                             |
|    Komparativ: + -er (größer, besser)                    |
|    Superlativ: am + -sten / der + -ste                   |
|    Many short adj. take umlaut: alt→älter                |
|                                                          |
+----------------------------------------------------------+
```
