## 1. Formal vs. Informal in Business

### Siezen vs. Duzen

One of the most important distinctions in German business life is when to use **Sie** (formal "you") vs. **du** (informal "you").

| | **Siezen** (Sie) | **Duzen** (du) |
|---|---|---|
| When | Default in business, formal settings | After explicit offer, casual companies |
| With whom | Clients, superiors, strangers, officials | Close colleagues, after being offered "du" |
| Signal phrases | *Können Sie mir helfen?* | *Kannst du mir helfen?* |
| Written form | Always capitalized: *Sie*, *Ihnen*, *Ihr* | Usually lowercase: *du*, *dir*, *dein* |

### The Du-Angebot (Offer to Use "du")

The transition from *Sie* to *du* is a deliberate social act:

- The **higher-ranking** person offers the *du* to the lower-ranking person
- The **older** person offers it to the younger person
- Among peers, it can be mutual
- Once offered and accepted, you cannot go back to *Sie*

> **Important**: Never assume *du* in a business setting. Wait for it to be offered. Using *du* prematurely is considered disrespectful.

### Modern Trends

Many modern companies, especially in tech and startups, use *du* by default (*Duz-Kultur*):
- IKEA Germany uses *du* with all customers
- Many startups have *du* as company policy
- Traditional industries (law, banking, government) still strongly prefer *Sie*

## 2. Job Application: Bewerbung

A German job application (*Bewerbung*) consists of several documents:

```
+----------------------------------------------------------+
|  BEWERBUNGSUNTERLAGEN (Application Documents)             |
+----------------------------------------------------------+
|                                                          |
|  1. Anschreiben (Cover letter)                           |
|     → One page, tailored to the specific job              |
|                                                          |
|  2. Lebenslauf (CV / Résumé)                             |
|     → Tabular format, reverse chronological               |
|     → Usually includes a photo (Bewerbungsfoto)           |
|                                                          |
|  3. Zeugnisse (References / Certificates)                |
|     → School diplomas, university degrees                 |
|     → Arbeitszeugnisse (employer references)              |
|     → Training certificates                               |
|                                                          |
|  4. Optional: Arbeitsproben (Work samples)               |
+----------------------------------------------------------+
```

### Lebenslauf (CV) — German Style

The German CV differs from American/British formats:

| Feature | German CV | US/UK CV |
|---------|-----------|----------|
| Photo | **Yes** — professional headshot (common) | No (anti-discrimination) |
| Personal details | Date of birth, nationality, marital status (optional but common) | Not included |
| Format | **Tabellarisch** (tabular, two-column) | Paragraph or bullet point |
| Order | **Reverse chronological** (newest first) | Reverse chronological |
| Length | 1-2 pages | 1 page (US) / 2 pages (UK) |
| Signature | **Yes** — signed and dated at the bottom | No |

### Lebenslauf Structure

| Section | German | Content |
|---------|--------|---------|
| Personal details | **Persönliche Daten** | Name, address, phone, email, date/place of birth |
| Work experience | **Berufserfahrung** | Jobs in reverse chronological order |
| Education | **Ausbildung / Studium** | Degrees, schools, dates |
| Skills | **Kenntnisse und Fähigkeiten** | Languages, IT skills, certifications |
| Interests | **Interessen / Hobbys** | Optional, keep professional |

### Anschreiben (Cover Letter) Structure

The cover letter should follow this format:

```
+----------------------------------------------------------+
|  ANSCHREIBEN STRUCTURE                                    |
+----------------------------------------------------------+
|                                                          |
|  Absender (your contact info)                            |
|  Empfänger (company address)                             |
|  Ort, Datum (place, date)                                |
|                                                          |
|  Betreff: Bewerbung als [Position]                       |
|                                                          |
|  Sehr geehrte/r [Name],                                  |
|                                                          |
|  Paragraph 1: WHY this company/position?                 |
|  → How did you find the job?                             |
|  → What attracts you to this role?                       |
|                                                          |
|  Paragraph 2: WHO are you?                               |
|  → Your qualifications and experience                    |
|  → Relevant skills and achievements                      |
|                                                          |
|  Paragraph 3: WHAT can you offer?                        |
|  → How will you contribute to the company?               |
|  → Match your skills to the job requirements              |
|                                                          |
|  Closing: Availability, interview request                |
|                                                          |
|  Mit freundlichen Grüßen                                 |
|  [Signature]                                             |
|  [Name]                                                  |
|  Anlagen: Lebenslauf, Zeugnisse                          |
+----------------------------------------------------------+
```

| German | English |
|--------|---------|
| Mit großem Interesse habe ich Ihre Stellenanzeige gelesen. | I read your job advertisement with great interest. |
| Hiermit bewerbe ich mich um die Stelle als... | I hereby apply for the position of... |

## 3. Job Interview: Vorstellungsgespräch

### Common Interview Questions and Sample Answers

**Question 1**: *Erzählen Sie uns etwas über sich.* (Tell us something about yourself.)

> *Gerne. Mein Name ist Anna Schmidt. Ich bin 28 Jahre alt und habe an der TU Berlin Informatik studiert. Nach meinem Masterabschluss habe ich drei Jahre als Softwareentwicklerin bei der Firma XY gearbeitet, wo ich vor allem an Webapplikationen mit Python und JavaScript gearbeitet habe.*

**Question 2**: *Warum möchten Sie bei uns arbeiten?* (Why do you want to work for us?)

> *Ihr Unternehmen ist bekannt für seine innovativen Produkte im Bereich Künstliche Intelligenz. Ich bin von Ihrer Unternehmenskultur beeindruckt und sehe hier die Möglichkeit, meine Fähigkeiten in einem dynamischen Umfeld weiterzuentwickeln.*

**Question 3**: *Was sind Ihre Stärken?* (What are your strengths?)

> *Zu meinen Stärken zähle ich meine analytische Denkweise, meine Teamfähigkeit und meine Bereitschaft, mich in neue Themen einzuarbeiten. In meiner letzten Position habe ich erfolgreich ein Team von fünf Personen geleitet.*

**Question 4**: *Was sind Ihre Schwächen?* (What are your weaknesses?)

> *Ich neige manchmal dazu, zu perfektionistisch zu sein und zu viel Zeit auf Details zu verwenden. Ich habe jedoch gelernt, Prioritäten besser zu setzen und mich auf das Wesentliche zu konzentrieren.*

**Question 5**: *Wo sehen Sie sich in fünf Jahren?* (Where do you see yourself in five years?)

> *In fünf Jahren möchte ich eine leitende Position in der Softwareentwicklung übernehmen und an strategischen Entscheidungen mitwirken. Ich möchte mich fachlich und persönlich weiterentwickeln.*

```
+----------------------------------------------------------+
|  MEETING STRUCTURE                                        |
+----------------------------------------------------------+
|                                                          |
|  1. Begrüßung (Welcome)                                 |
|     "Guten Morgen, vielen Dank, dass Sie alle da sind."  |
|                                                          |
|  2. Tagesordnung (Agenda)                                |
|     "Auf der Tagesordnung stehen heute drei Punkte..."   |
|                                                          |

|                                                          |

|                                                          |

```

## 7. Telephone Conversations

### Answering the Phone

**Business**: State your company name and your name:
- *Firma DataTech, Schmidt am Apparat. Guten Tag!*
- *DataTech GmbH, Sie sprechen mit Frau Müller. Was kann ich für Sie tun?*

**Personal**: State your last name:
- *Schmidt.* (very common, just the last name)
- *Hier ist Schmidt.*

### Key Telephone Phrases

**Identifying yourself**:

| German | English |
|--------|---------|
| Hier spricht [Name] von [Firma]. | This is [Name] from [Company]. |
| Mein Name ist [Name], ich rufe an wegen... | My name is [Name], I am calling about... |
| Guten Tag, hier ist [Name]. | Hello, this is [Name]. |

**Asking to speak to someone**:

| German | English |
|--------|---------|
| Könnte ich bitte mit Herrn/Frau [Name] sprechen? | Could I please speak with Mr./Mrs. [Name]? |
| Könnten Sie mich bitte mit der [Abteilung] verbinden? | Could you please connect me with the [department]? |
| Ist Herr/Frau [Name] zu sprechen? | Is Mr./Mrs. [Name] available? |

**If the person is unavailable**:

| German | English |
|--------|---------|
| Es tut mir leid, Herr/Frau [Name] ist gerade nicht erreichbar. | I am sorry, Mr./Mrs. [Name] is not available right now. |
| Er/Sie ist gerade in einer Besprechung. | He/She is currently in a meeting. |
| Möchten Sie eine Nachricht hinterlassen? | Would you like to leave a message? |
| Soll er/sie Sie zurückrufen? | Should he/she call you back? |

**Leaving a message**:

| German | English |
|--------|---------|
| Könnten Sie ihm/ihr bitte ausrichten, dass... | Could you please tell him/her that... |
| Bitte richten Sie aus, dass ich angerufen habe. | Please pass on that I called. |
| Er/Sie kann mich unter der Nummer ... erreichen. | He/She can reach me at the number ... |
| Ich rufe später noch einmal an. | I will call again later. |

**Ending the call**:

| German | English |
|--------|---------|
| Vielen Dank für Ihre Hilfe. | Thank you for your help. |
| Auf Wiederhören! | Goodbye! (on the phone) |
| Ich wünsche Ihnen einen schönen Tag. | I wish you a nice day. |

> **Note**: On the phone, Germans say *Auf Wiederhören* (literally "until hearing again"), not *Auf Wiedersehen* (until seeing again).

### Sample Phone Conversation

> **Empfang**: Firma TechStar, guten Tag. Sie sprechen mit Frau Braun. Was kann ich für Sie tun?
>
> **Anrufer**: Guten Tag, Frau Braun. Hier spricht Thomas Weber von Digital Solutions. Könnte ich bitte mit Herrn Müller aus der IT-Abteilung sprechen?
>
> **Empfang**: Einen Moment bitte, ich verbinde Sie... Es tut mir leid, Herr Müller ist gerade in einer Besprechung. Möchten Sie eine Nachricht hinterlassen?
>
> **Anrufer**: Ja, gerne. Könnten Sie ihm bitte ausrichten, dass ich wegen des Angebots angerufen habe? Er kann mich unter 030-98765432 zurückrufen.
>
> **Empfang**: Selbstverständlich, ich richte es ihm aus. Kann ich sonst noch etwas für Sie tun?
>
> **Anrufer**: Nein, das ist alles. Vielen Dank, Frau Braun. Auf Wiederhören!
>
> **Empfang**: Gerne! Auf Wiederhören!

## 9. German Business Culture

### Key Cultural Values

| Value | German | Description |
|-------|--------|-------------|
| Punctuality | **Pünktlichkeit** | Arrive 5 minutes early. Being late is disrespectful. |
| Directness | **Direktheit** | Germans communicate clearly and directly. This is not rudeness — it is efficiency. |
| Thoroughness | **Gründlichkeit** | Work is expected to be thorough and well-prepared. |
| Reliability | **Zuverlässigkeit** | If you say you will do something, you must deliver. |
| Separation of work/private | **Trennung Beruf/Privat** | Germans generally keep work and personal life separate. |

### Business Etiquette

**Handshake (Handschlag)**:
- Firm handshake upon meeting and leaving
- Shake hands with everyone present, including at the beginning and end of meetings
- Make eye contact during the handshake

**Business cards (Visitenkarte)**:
- Exchange business cards at the beginning of a meeting
- Read the card before putting it away (shows respect)
- In Germany, academic titles (Dr., Prof.) are important and should be included on the card and used in address

**Titles and forms of address**:
- Use *Herr* + last name or *Frau* + last name
- If the person has a doctoral title: *Herr Dr. Schmidt*, *Frau Dr. Müller*
- A Professor: *Herr Professor Schmidt* or *Herr Prof. Dr. Schmidt*

### Work-Life Balance

German labor law strongly protects employees:

| Aspect | Details |
|--------|---------|
| **Urlaub** (Vacation) | Minimum 20 days per year by law (many companies offer 25-30) |
| **Arbeitszeit** (Working hours) | Typically 35-40 hours per week; 8-hour workday standard |
| **Mittagspause** (Lunch break) | Usually 30-60 minutes |
| **Überstunden** (Overtime) | Legally regulated; cannot exceed 10 hours per workday |
| **Krankheitstage** (Sick days) | Paid sick leave from day 1; doctor's note required after day 3 |
| **Elternzeit** (Parental leave) | Up to 3 years per parent; job protection guaranteed |
| **Feierabend** | End of workday; respected as personal time |

### Meetings Culture

- **Agenda** (*Tagesordnung*): Meetings always have a clear agenda distributed in advance
- **Minutes** (*Protokoll*): Meeting minutes are recorded and distributed afterward
- **Decisions**: Decisions are made collectively and documented; individual action items have clear owners and deadlines
- **Small talk**: Brief, appropriate small talk at the start (weather, journey), but meetings get to business quickly

1. Meeting a new client for the first time
2. Your colleague who started on the same day as you and you have lunch together every day
3. Emailing the CEO of a partner company
4. Your boss says: "Wir können uns ruhig duzen."
5. At a startup where everyone uses first names

<details>
<summary>Answers</summary>

1. **Sie** — Always use Sie with new business contacts until offered otherwise.
2. **It depends** — If you are close and one of you has offered du, then du. Otherwise, Sie is safer until the du is explicitly offered.
3. **Sie** — Formal business correspondence always uses Sie.
4. **Du** — Your boss has made the Du-Angebot; accept and switch.
5. **Du** — If the company culture is du, follow it; but confirm on your first day.

### Exercise 6: Business Culture — True or False

1. In Germany, it is acceptable to arrive 10 minutes late to a business meeting.
2. Germans typically separate their work life from their private life.
3. The minimum vacation by German law is 30 days per year.
4. You should always use academic titles (Dr., Prof.) when addressing someone in business.
5. "Auf Wiederhören" is the correct way to say goodbye on the phone.
6. At the end of a German meeting, a Protokoll is usually written.

<details>
<summary>Answers</summary>

1. **Falsch** — Punctuality is extremely important; being late is considered disrespectful.
2. **Richtig** — Germans generally maintain a clear separation between Beruf and Privatleben.
3. **Falsch** — The legal minimum is 20 days (though many companies offer more).
4. **Richtig** — Titles are important in German business culture and should be used.
5. **Richtig** — *Auf Wiederhören* (until hearing again) is used on the phone, *Auf Wiedersehen* (until seeing again) is used in person.
6. **Richtig** — Meeting minutes (Protokoll) are standard practice.

</details>
