## 1. 서술 형용사 vs. 수식 형용사

독일어 형용사는 문장에서의 위치에 따라 다르게 작용합니다.

### 1.1 서술 형용사(Predicate Adjectives) (어미 없음)

형용사가 *sein* (이다), *werden* (되다), *bleiben* (남다)과 같은 동사 **뒤에** 올 때는 어미가 **전혀** 붙지 않습니다:

- Das Haus ist **groß**. — 그 집은 크다.
- Der Mann ist **alt**. — 그 남자는 늙었다.
- Die Blumen sind **schön**. — 그 꽃들은 아름답다.
- Das Wetter wird **kalt**. — 날씨가 추워지고 있다.

> **규칙**: 서술 형용사는 쉽습니다 — 형태가 절대 변하지 않습니다.

### 1.2 수식 형용사(Attributive Adjectives) (명사 앞 — 반드시 어미 변화)

형용사가 명사 **앞에** 올 때는, 명사의 성, 수, 격을 반영하는 어미를 **반드시** 붙여야 합니다:

- Das **große** Haus — 큰 집
- Der **alte** Mann — 늙은 남자
- Die **schönen** Blumen — 아름다운 꽃들
- Ein **kalter** Tag — 추운 날

어미는 세 가지 요인에 따라 결정됩니다:
1. 명사의 **성** (남성, 여성, 중성)
2. **격** (주격, 대격, 여격, 소유격)
3. **형용사 앞에 오는 것** (정관사, 부정관사, 또는 아무것도 없음)

이것이 세 가지 어미 변화 패턴을 만듭니다: **약변화**, **혼합변화**, **강변화**.

## 2. 약변화 (정관사 뒤)

형용사가 **정관사** (der, die, das) 또는 성/격 신호를 명확히 보여주는 유사한 단어(dieser, jeder, jener, welcher, alle) 뒤에 올 때 **약변화**를 사용합니다.

### 2.1 약변화 표

| | 남성 | 여성 | 중성 | 복수 |
|---|-----------|----------|--------|--------|
| **주격** | der alt**e** Mann | die alt**e** Frau | das alt**e** Haus | die alt**en** Häuser |
| **대격** | den alt**en** Mann | die alt**e** Frau | das alt**e** Haus | die alt**en** Häuser |
| **여격** | dem alt**en** Mann | der alt**en** Frau | dem alt**en** Haus | den alt**en** Häusern |
| **소유격** | des alt**en** Mannes | der alt**en** Frau | des alt**en** Hauses | der alt**en** Häuser |

### 2.2 패턴

약변화는 두 가지 어미만 사용합니다:

- **-e**: 다섯 자리 (주격 남성/여성/중성, 대격 여성/중성)
- **-en**: 나머지 모든 자리

```
+----------------------------------------------+
|  WEAK DECLENSION (after der/die/das)          |
+----------------------------------------------+
|         Masc.   Fem.   Neut.   Plural        |
|  Nom.    -e      -e     -e      -en          |
|  Acc.    -en     -e     -e      -en          |
|  Dat.    -en     -en    -en     -en          |
|  Gen.    -en     -en    -en     -en          |
+----------------------------------------------+
|  Memory: -e in nom. (all genders) +           |
|          acc. fem./neut. — everything else -en |
+----------------------------------------------+
```

### 2.3 예시

- **주격**: Der **neue** Computer funktioniert gut. — 새 컴퓨터가 잘 작동한다.
- **대격**: Ich kaufe den **neuen** Computer. — 나는 새 컴퓨터를 산다.
- **여격**: Er arbeitet mit dem **neuen** Computer. — 그는 새 컴퓨터로 일한다.
- **주격 복수**: Die **kleinen** Kinder spielen draußen. — 어린 아이들이 밖에서 놀고 있다.

## 3. 혼합변화 (부정관사 뒤)

**부정관사** (ein, eine) 또는 소유 대명사(mein, dein, sein, ihr, unser, euer, Ihr)와 부정관사(kein) 뒤에서 **혼합변화**를 사용합니다.

### 3.1 왜 "혼합"인가?

부정관사는 성/격을 항상 명확하게 보여주지 않습니다. 그런 자리에서 형용사가 **강변화 어미**를 취해 보완해야 합니다. 관사가 신호를 보여주는 다른 자리에서는 형용사가 **약변화 어미**를 사용합니다.

### 3.2 혼합변화 표

| | 남성 | 여성 | 중성 | 복수 (keine/meine) |
|---|-----------|----------|--------|--------|
| **주격** | ein alt**er** Mann | eine alt**e** Frau | ein alt**es** Haus | keine alt**en** Häuser |
| **대격** | einen alt**en** Mann | eine alt**e** Frau | ein alt**es** Haus | keine alt**en** Häuser |
| **여격** | einem alt**en** Mann | einer alt**en** Frau | einem alt**en** Haus | keinen alt**en** Häusern |
| **소유격** | eines alt**en** Mannes | einer alt**en** Frau | eines alt**en** Hauses | keiner alt**en** Häuser |

### 3.3 패턴

```
+----------------------------------------------+
|  MIXED DECLENSION (after ein/mein/kein)       |
+----------------------------------------------+
|         Masc.   Fem.   Neut.   Plural        |
|  Nom.    -er     -e     -es     -en          |
|  Acc.    -en     -e     -es     -en          |
|  Dat.    -en     -en    -en     -en          |
|  Gen.    -en     -en    -en     -en          |
+----------------------------------------------+
|  Key insight: Strong endings (-er, -es) appear |
|  where ein/mein/kein lack a gender signal      |
|  (nom. masc. & nom./acc. neuter)              |
+----------------------------------------------+
```

### 3.4 예시

- **주격**: Ein **junger** Mann steht an der Tür. — 젊은 남자가 문 앞에 서 있다.
- **대격**: Sie sucht einen **guten** Job. — 그녀는 좋은 직장을 찾고 있다.
- **주격 중성**: Mein **kleines** Kind schläft. — 내 어린 아이가 자고 있다.
- **대격 여성**: Er kauft eine **rote** Rose. — 그는 빨간 장미를 산다.
- **여격**: Ich wohne in einem **alten** Haus. — 나는 오래된 집에 산다.

## 4. 강변화 (관사 없음)

형용사 앞에 **관사가 없을** 때 **강변화**를 사용합니다. 형용사 자체가 성/격 신호를 완전히 담당해야 합니다.

### 4.1 강변화 표

| | 남성 | 여성 | 중성 | 복수 |
|---|-----------|----------|--------|--------|
| **주격** | alt**er** Mann | alt**e** Frau | alt**es** Haus | alt**e** Häuser |
| **대격** | alt**en** Mann | alt**e** Frau | alt**es** Haus | alt**e** Häuser |
| **여격** | alt**em** Mann | alt**er** Frau | alt**em** Haus | alt**en** Häusern |
| **소유격** | alt**en** Mannes | alt**er** Frau | alt**en** Hauses | alt**er** Häuser |

### 4.2 패턴

```
+----------------------------------------------+
|  STRONG DECLENSION (no article)               |
+----------------------------------------------+
|         Masc.   Fem.   Neut.   Plural        |
|  Nom.    -er     -e     -es     -e           |
|  Acc.    -en     -e     -es     -e           |
|  Dat.    -em     -er    -em     -en          |
|  Gen.    -en     -er    -en     -er          |
+----------------------------------------------+
|  The endings mirror the definite article       |
|  endings (der/die/das) — the adjective takes  |
|  over the article's job!                      |
+----------------------------------------------+
```

### 4.3 언제 사용하나요?

강변화는 다음 경우에 나타납니다:
- 관사가 전혀 없을 때: **Kalter** Kaffee schmeckt nicht. (차가운 커피는 맛이 없다.)
- 숫자 뒤: Zwei **kleine** Kinder spielen. (어린 아이 둘이 놀고 있다.)
- *viel* / *wenig* (어미 없이): **viel guter** Wein (많은 좋은 와인)
- *einige*, *mehrere*, *andere*: einige **neue** Bücher (몇 권의 새 책들)

### 4.4 예시

- **주격**: **Frischer** Wind weht heute. — 오늘 신선한 바람이 분다.
- **대격**: Ich trinke **kalten** Tee. — 나는 차가운 차를 마신다.
- **여격**: Mit **großem** Interesse habe ich das gelesen. — 나는 큰 관심을 가지고 그것을 읽었다.
- **주격 복수**: **Kleine** Kinder brauchen viel Schlaf. — 어린 아이들은 잠이 많이 필요하다.

## 5. 어미 변화 요약과 전략

### 5.1 핵심 원리

명사구는 항상 성/격 신호를 **어딘가에** 보여주어야 합니다 — 관사에서든 형용사 어미에서든:

1. **정관사가 있을 때** (der/die/das) → 관사가 신호를 보여줌 → 형용사는 **약변화** 어미 (-e 또는 -en)
2. **부정관사가 있을 때** (ein/mein/kein) → 관사가 때때로 신호를 보여주고 때때로 안 보여줌 → 형용사는 **혼합변화** 어미 (관사가 신호를 보여주면 약변화, 그렇지 않으면 강변화)
3. **관사가 없을 때** → 형용사가 신호를 보여주어야 함 → 형용사는 **강변화** 어미 (정관사 어미와 일치)

### 5.2 빠른 판단 순서도

```
Is there an article before the adjective?
├── YES: Is it der/die/das (or dieser/jeder/welcher)?
│   └── YES → WEAK declension (-e / -en)
│   └── NO (ein/mein/kein) → MIXED declension
└── NO → STRONG declension
```

## 6. 비교: Komparativ과 Superlativ

### 6.1 비교급 (Komparativ)

형용사 어간에 **-er**을 붙입니다:

| 형용사 | 비교급 | 영어 |
|-----------|-------------|---------|
| schnell | schnell**er** | 더 빠른 |
| klein | klein**er** | 더 작은 |
| langsam | langsam**er** | 더 느린 |
| billig | billig**er** | 더 싼 |
| schön | schön**er** | 더 아름다운 |

**문장에서의 비교급**:

- Er ist **schneller** als ich. — 그는 나보다 빠르다.
- Dieses Buch ist **interessanter** als jenes. — 이 책은 저것보다 더 흥미롭다.
- Berlin ist **größer** als München. — 베를린은 뮌헨보다 크다.

> **참고**: 독일어는 긴 형용사에서도 항상 **-er** 접미사를 사용합니다. 영어의 "more interesting"에 해당하는 것이 없으며, 항상 *interessanter*입니다.

**수식 형용사로서의 비교급** (명사 앞에서, 여전히 어미 변화를 취합니다):

- Ein **schnelleres** Auto. — 더 빠른 자동차. (중성 주격/대격)
- Der **ältere** Bruder. — 형 (더 나이 많은 형제). (남성 주격)

### 6.2 최상급 (Superlativ)

형용사 어간에 **-st** 또는 **-est**를 붙입니다:

| 형용사 | 최상급 | 영어 |
|-----------|-------------|---------|
| schnell | schnell**st-** | 가장 빠른 |
| klein | klein**st-** | 가장 작은 |
| schön | schön**st-** | 가장 아름다운 |
| alt | älte**st-** | 가장 나이 많은 |
| kalt | kälte**st-** | 가장 추운 |

> **-est 사용**: 형용사가 **-d, -t, -s, -ß, -sch, -z**로 끝날 때: der kälte**st**e Tag, das intere**ss**ante**st**e Buch.

**두 가지 최상급 형태**:

1. **수식형** (명사 앞): 형용사 어미를 취합니다
   - Das ist der **schnellste** Zug. — 그것은 가장 빠른 기차이다.
   - Sie ist die **beste** Schülerin. — 그녀는 가장 뛰어난 학생이다.

2. **서술형** (*sein* 뒤): **am + -sten**을 사용합니다
   - Dieser Zug ist **am schnellsten**. — 이 기차가 가장 빠르다.
   - Im Winter ist es **am kältesten**. — 겨울에 가장 춥다.

### 6.3 비교 구문

| 구문 | 독일어 | 영어 |
|-----------|--------|---------|
| ~보다 더 ... | Komparativ + **als** | -er ... than |
| ~만큼 ... | **so** + 원급 + **wie** | as ... as |
| ~만큼 ...하지 않은 | **nicht so** + 원급 + **wie** | not as ... as |

예시:
- Er ist **größer als** sein Vater. — 그는 아버지보다 키가 크다.
- Sie ist **so intelligent wie** ihr Bruder. — 그녀는 오빠만큼 똑똑하다.
- Dieses Hotel ist **nicht so teuer wie** das andere. — 이 호텔은 다른 호텔만큼 비싸지 않다.

## 7. 불규칙 비교와 움라우트 변화

### 7.1 완전 불규칙 형태

| 원급 | 비교급(Komparativ) | 최상급(Superlativ) | 영어 |
|----------|------------|------------|---------|
| gut | **besser** | **am besten** / der **beste** | 좋은 → 더 좋은 → 가장 좋은 |
| viel | **mehr** | **am meisten** / der **meiste** | 많은 → 더 많은 → 가장 많은 |
| gern | **lieber** | **am liebsten** | 기꺼이 → 더 기꺼이 → 가장 기꺼이 |
| hoch | **höher** | **am höchsten** / der **höchste** | 높은 → 더 높은 → 가장 높은 |
| nah | **näher** | **am nächsten** / der **nächste** | 가까운 → 더 가까운 → 가장 가까운 |

### 7.2 비교에서의 움라우트(Umlaut)

많은 흔한 단음절 형용사가 비교급과 최상급에서 **움라우트** (a→ä, o→ö, u→ü)를 얻습니다:

| 원급 | 비교급(Komparativ) | 최상급(Superlativ) | 영어 |
|----------|------------|------------|---------|
| alt | **älter** | **am ältesten** | 나이 든 → 더 나이 든 → 가장 나이 든 |
| groß | **größer** | **am größten** | 큰 → 더 큰 → 가장 큰 |
| jung | **jünger** | **am jüngsten** | 젊은 → 더 젊은 → 가장 젊은 |
| kalt | **kälter** | **am kältesten** | 추운 → 더 추운 → 가장 추운 |
| warm | **wärmer** | **am wärmsten** | 따뜻한 → 더 따뜻한 → 가장 따뜻한 |
| lang | **länger** | **am längsten** | 긴 → 더 긴 → 가장 긴 |
| kurz | **kürzer** | **am kürzesten** | 짧은 → 더 짧은 → 가장 짧은 |
| stark | **stärker** | **am stärksten** | 강한 → 더 강한 → 가장 강한 |
| klug | **klüger** | **am klügsten** | 영리한 → 더 영리한 → 가장 영리한 |
| arm | **ärmer** | **am ärmsten** | 가난한 → 더 가난한 → 가장 가난한 |
| krank | **kränker** | **am kränksten** | 아픈 → 더 아픈 → 가장 아픈 |
| dumm | **dümmer** | **am dümmsten** | 어리석은 → 더 어리석은 → 가장 어리석은 |

> **참고**: 모든 단음절 형용사가 움라우트를 취하는 것은 아닙니다. 예를 들어: *bunt* → *bunter* (*bünter*가 아님), *schlank* → *schlanker* (*schlänker*가 아님). 어떤 것이 움라우트를 취하는지 암기해야 합니다.

## 8. 명사화된 형용사

독일어는 형용사를 명사로 자주 전환합니다. 이 **substantivierte Adjektive** (명사화된 형용사)는 대문자로 쓰며 여전히 수식 형용사처럼 어미가 변화합니다:

### 8.1 정관사와 함께

| 독일어 | 영어 | 어미 변화 |
|--------|---------|------------|
| der **Deutsche** | 독일 남성 | 약변화 남성 주격 |
| die **Deutsche** | 독일 여성 | 약변화 여성 주격 |
| die **Deutschen** | 독일인들 | 약변화 복수 |
| der **Alte** | 늙은 남자 | 약변화 남성 주격 |
| die **Alte** | 늙은 여자 | 약변화 여성 주격 |
| der **Kranke** | 아픈 사람 (남) | 약변화 남성 주격 |
| der **Bekannte** | 지인 (남) | 약변화 남성 주격 |
| die **Verwandte** | 친척 (여) | 약변화 여성 주격 |

### 8.2 부정관사와 함께

| 독일어 | 영어 | 어미 변화 |
|--------|---------|------------|
| ein **Deutscher** | 독일 남성 한 명 | 혼합변화 남성 주격 |
| eine **Deutsche** | 독일 여성 한 명 | 혼합변화 여성 주격 |
| ein **Bekannter** | 지인 한 명 (남) | 혼합변화 남성 주격 |
| eine **Bekannte** | 지인 한 명 (여) | 혼합변화 여성 주격 |

### 8.3 etwas, nichts, viel, wenig과 함께

*etwas* (무언가), *nichts* (아무것도 ~않다), *viel* (많은), *wenig* (적은) 뒤에서, 형용사 명사는 **강변화 중성** 어미를 취합니다:

- etwas **Neues** — 새로운 무언가
- nichts **Besonderes** — 특별한 것은 없음
- viel **Gutes** — 많은 좋은 것
- wenig **Interessantes** — 흥미로운 것이 거의 없음

## 10. 연습 문제

### 연습 1: 서술 형용사인가 수식 형용사인가?

형용사에 어미가 필요한지 결정하세요. 올바른 형태를 쓰세요.

1. Das Wetter ist ________ (kalt).
2. Ein ________ Tag (kalt).
3. Die Suppe ist ________ (heiß).
4. Ich trinke ________ Suppe (heiß, eine와 함께).
5. Die Kinder sind ________ (müde).
6. Die ________ Kinder schlafen (müde).

<details>
<summary>정답</summary>

1. Das Wetter ist **kalt**. (서술 — 어미 없음)
2. Ein **kalter** Tag. (수식, 혼합변화 주격 남성 — -er)
3. Die Suppe ist **heiß**. (서술 — 어미 없음)
4. Ich trinke eine **heiße** Suppe. (수식, 혼합변화 대격 여성 — -e)
5. Die Kinder sind **müde**. (서술 — 어미 없음)
6. Die **müden** Kinder schlafen. (수식, 약변화 주격 복수 — -en)

</details>

### 연습 2: 약변화 (der/die/das)

올바른 형용사 어미를 넣으세요.

1. Der neu________ Film ist gut. (남성 주격)
2. Ich lese das interessant________ Buch. (중성 대격)
3. Die klein________ Kinder spielen. (복수 주격)
4. Er kauft den teur________ Wein. (남성 대격)
5. Wir wohnen in dem alt________ Haus. (중성 여격)

<details>
<summary>정답</summary>

1. Der neu**e** Film ist gut. (약변화 남성 주격 = -e)
2. Ich lese das interessant**e** Buch. (약변화 중성 대격 = -e)
3. Die klein**en** Kinder spielen. (약변화 복수 주격 = -en)
4. Er kauft den teur**en** Wein. (약변화 남성 대격 = -en)
5. Wir wohnen in dem alt**en** Haus. (약변화 중성 여격 = -en)

</details>

### 연습 3: 혼합변화 (ein/mein/kein)

올바른 형용사 어미를 넣으세요.

1. Ein jung________ Mann wartet draußen. (남성 주격)
2. Sie hat ein klein________ Kind. (중성 대격)
3. Ich suche einen gut________ Arzt. (남성 대격)
4. Er wohnt in einer groß________ Stadt. (여성 여격)
5. Mein alt________ Auto ist kaputt. (중성 주격)

<details>
<summary>정답</summary>

1. Ein jung**er** Mann wartet draußen. (혼합변화 남성 주격 = -er)
2. Sie hat ein klein**es** Kind. (혼합변화 중성 대격 = -es)
3. Ich suche einen gut**en** Arzt. (혼합변화 남성 대격 = -en)
4. Er wohnt in einer groß**en** Stadt. (혼합변화 여성 여격 = -en)
5. Mein alt**es** Auto ist kaputt. (혼합변화 중성 주격 = -es)

</details>

### 연습 4: 강변화 (관사 없음)

올바른 형용사 어미를 넣으세요.

1. Kalt________ Wasser ist erfrischend. (중성 주격)
2. Ich trinke gern warm________ Milch. (여성 대격)
3. Bei schlecht________ Wetter bleiben wir zu Hause. (중성 여격)
4. Frisch________ Brot schmeckt am besten. (중성 주격)
5. Deutsch________ Bier ist weltberühmt. (중성 주격)

<details>
<summary>정답</summary>

1. Kalt**es** Wasser ist erfrischend. (강변화 중성 주격 = -es)
2. Ich trinke gern warm**e** Milch. (강변화 여성 대격 = -e)
3. Bei schlecht**em** Wetter bleiben wir zu Hause. (강변화 중성 여격 = -em)
4. Frisch**es** Brot schmeckt am besten. (강변화 중성 주격 = -es)
5. Deutsch**es** Bier ist weltberühmt. (강변화 중성 주격 = -es)

</details>

### 연습 5: 비교급과 최상급

올바른 비교급 또는 최상급 형태로 문장을 완성하세요.

1. Berlin ist ________ als München. (groß)
2. Mein Bruder ist ________ als ich. (alt)
3. Dieses Restaurant ist ________ (am + gut).
4. Sie spricht ________ Deutsch als ich. (gut)
5. Der Mount Everest ist der ________ Berg der Welt. (hoch)
6. Im Sommer sind die Tage ________ als im Winter. (lang)

<details>
<summary>정답</summary>

1. Berlin ist **größer** als München.
2. Mein Bruder ist **älter** als ich.
3. Dieses Restaurant ist **am besten**.
4. Sie spricht **besser** Deutsch als ich.
5. Der Mount Everest ist der **höchste** Berg der Welt.
6. Im Sommer sind die Tage **länger** als im Winter.

</details>

### 연습 6: 번역 연습

독일어로 번역하세요.

1. The old house is beautiful. (서술 형용사 사용)
2. I see the small child. (das Kind, 약변화)
3. She has a new car. (das Auto, 혼합변화)
4. Cold coffee tastes bad. (강변화)
5. Something new is happening. (etwas + 명사화된 형용사)

<details>
<summary>정답</summary>

1. Das alte Haus ist **schön**.
2. Ich sehe das **kleine** Kind.
3. Sie hat ein **neues** Auto.
4. **Kalter** Kaffee schmeckt schlecht.
5. **Etwas Neues** passiert.

</details>

```
+----------------------------------------------------------+
|         ADJECTIVE DECLENSION CHEAT SHEET                  |
+----------------------------------------------------------+
|                                                          |
|  PREDICATE (after sein/werden): NO ending                |
|    Das Haus ist groß.                                    |
|                                                          |
|  ATTRIBUTIVE (before noun): MUST have ending             |
|                                                          |
|  WEAK (after der/die/das):  mostly -e and -en            |
|    Nom: -e (all)  Acc: -en (m), -e (f/n)                |
|    Dat/Gen: always -en                                   |
|                                                          |
|  MIXED (after ein/mein/kein):                            |
|    Same as weak EXCEPT:                                  |
|    Nom. masc: -er, Nom/Acc. neut: -es                   |
|                                                          |
|  STRONG (no article): mirrors der/die/das endings        |
|    Nom: -er (m), -e (f), -es (n), -e (pl)               |
|                                                          |
|  COMPARISON:                                             |
|    Komparativ: + -er (größer, besser)                    |
|    Superlativ: am + -sten / der + -ste                   |
|    Many short adj. take umlaut: alt→älter                |
|                                                          |
+----------------------------------------------------------+
```
