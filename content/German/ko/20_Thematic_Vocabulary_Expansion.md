# 20. 주제별 어휘 확장 (Thematische Wortschatzerweiterung)

## 학습 목표

- 7개 핵심 주제 영역에 걸쳐 어휘를 구축한다: 학술, 정치, 환경, 기술, 미디어, 건강, 경제
- 독일어 단어 형성 전략을 이해하고 적용한다: 복합어, 접두사, 접미사
- 복합 명사의 형성 방법을 인식하고 그 성(Gender)을 판별한다
- 일반적인 접두사(un-, ur-, miss-)와 접미사(-ung, -heit/-keit, -schaft, -lich, -bar, -ig, -isch)를 생산적으로 사용한다
- 품사 간 전환을 수행한다: 명사→동사, 동사→명사, 형용사→명사
- 명사화(Nominalization: das Lesen, das Schreiben)를 맥락에 맞게 적용한다

## 8. 단어 형성: 복합 명사

독일어는 명사(및 다른 품사)를 자유롭게 결합하여 새로운 단어를 만든다. 이것은 독일어에서 가장 생산적인 특징 중 하나이다.

### 8.1 기본 구조

**마지막** 구성 요소가 성, 복수형, 전반적인 의미를 결정한다. 그 앞의 모든 것은 수식하거나 구체화한다.

```
Modifier(s) + Base Noun = Compound Noun
Gender = Gender of Base Noun
```

| 구성 요소 | 복합어 | 성 | 영어 |
|-----------|----------|--------|---------|
| Kranken + Haus | **Krankenhaus** | das (Haus) | hospital |
| Bundes + Regierung | **Bundesregierung** | die (Regierung) | federal government |
| Geschwindigkeit + s + Begrenzung | **Geschwindigkeitsbegrenzung** | die (Begrenzung) | speed limit |
| Wohnzimmer + Tür | **Wohnzimmertür** | die (Tür) | living room door |
| Geburts + Tag | **Geburtstag** | der (Tag) | birthday |
| Haupt + Bahnhof | **Hauptbahnhof** | der (Bahnhof) | main train station |
| Fahr + Rad | **Fahrrad** | das (Rad) | bicycle |

### 8.2 연결 요소 (Fugenelemente)

복합 명사에는 구성 요소 사이에 연결 요소가 필요한 경우가 많다:

| Fugenelement | 예 | 구성 요소 |
|-------------|---------|-----------|
| **-s-** | Arbeit**s**platz | Arbeit + Platz (직장) |
| **-s-** | Geburt**s**tag | Geburt + Tag (생일) |
| **-n-** | Blume**n**laden | Blume + Laden (꽃집) |
| **-en-** | Studenten**en**wohnheim | Student + Wohnheim (기숙사) |
| **-er-** | Kind**er**garten | Kind + Garten (유치원) |
| **-e-** | Hund**e**hütte | Hund + Hütte (개집) |
| (없음) | Haustür | Haus + Tür (현관문) |

> 어떤 연결 요소를 사용할지에 대한 엄격한 규칙은 없다. 선택은 역사적으로 결정되어 있으며 각 복합어와 함께 배워야 한다.

### 8.3 다중 구성 복합어

독일어 복합어는 무한히 연결할 수 있다 (실제로는 3~4개 구성이 가장 일반적이다):

| 복합어 | 구성 요소 | 영어 |
|----------|-----------|---------|
| **Handschuh** | Hand + Schuh | glove (손-신발) |
| **Kühlschrank** | kühl + Schrank | refrigerator (냉-장) |
| **Staubsauger** | Staub + Sauger | vacuum cleaner (먼지-빨아들이는 것) |
| **Flugzeugträger** | Flug + Zeug + Träger | aircraft carrier |
| **Lebensversicherungsgesellschaft** | Lebens + Versicherungs + Gesellschaft | life insurance company |
| **Donaudampfschifffahrtsgesellschaft** | Donau + Dampf + Schiff + Fahrt + s + Gesellschaft | Danube steamship company |

## 10. 단어 형성: 접미사

접미사는 기존 단어에서 새로운 단어를 파생하기 위한 강력한 도구이다. 접미사는 종종 품사와 성을 결정한다.

### 10.1 명사 접미사

| 접미사 | 성 | 형성 원천 | 예 | 영어 |
|--------|--------|-----------|---------|---------|
| **-ung** | die | 동사 | die Wohn**ung** (← wohnen) | apartment |
| | | | die Erklär**ung** (← erklären) | explanation |
| | | | die Entscheid**ung** (← entscheiden) | decision |
| **-heit** | die | 형용사 | die Frei**heit** (← frei) | freedom |
| | | | die Krank**heit** (← krank) | illness |
| | | | die Gesund**heit** (← gesund) | health |
| **-keit** | die | -ig, -lich, -bar, -sam으로 끝나는 형용사 | die Möglich**keit** (← möglich) | possibility |
| | | | die Freundlich**keit** (← freundlich) | friendliness |
| | | | die Einsamkeit (← einsam) | loneliness |
| **-schaft** | die | 명사, 형용사 | die Freund**schaft** (← Freund) | friendship |
| | | | die Wissen**schaft** (← Wissen) | science |
| | | | die Land**schaft** (← Land) | landscape |
| | | | die Gesell**schaft** (← Geselle) | society |

> **규칙:** **-ung, -heit, -keit, -schaft**로 끝나는 모든 명사는 **여성**(die)이다. 예외 없음.

### 10.2 형용사 접미사

| 접미사 | 의미 | 예 | 영어 |
|--------|---------|---------|---------|
| **-lich** | ~의 성질을 가진 | freund**lich** (← Freund) | friendly |
| | | herz**lich** (← Herz) | heartfelt / cordial |
| | | täg**lich** (← Tag) | daily |
| | | glaub**lich** (← glauben) | believable |
| **-bar** | ~할 수 있는 | mach**bar** (← machen) | feasible / doable |
| | | trink**bar** (← trinken) | drinkable |
| | | sicht**bar** (← Sicht) | visible |
| | | vorstell**bar** (← vorstellen) | imaginable |
| **-ig** | ~을 가진 / ~의 특징을 가진 | fleiß**ig** (← Fleiß) | diligent |
| | | eis**ig** (← Eis) | icy |
| | | richt**ig** (← Richt-) | correct |
| | | ruhig (← Ruhe) | calm |
| **-isch** | ~에 관한 (사람에 대해서는 다소 부정적) | typ**isch** (← Typ) | typical |
| | | kindisch (← Kind) | childish (negative) |
| | | europä**isch** (← Europa) | European |
| | | iron**isch** (← Ironie) | ironic |

### 10.3 접미사 대비: -lich vs. -isch

| -lich (대체로 긍정적/중립적) | -isch (사람에 대해 대체로 부정적) |
|-------------------------------|----------------------------------|
| kind**lich** (어린이다운 — 중립적) | kind**isch** (유치한 — 부정적) |
| weib**lich** (여성적인 — 중립적) | weib**isch** (계집애 같은 — 부정적) |
| herr**lich** (훌륭한) | herr**isch** (거만한) |

| essen | **das Essen** | food / eating |
| trinken | **das Trinken** | drinking |
| leben | **das Leben** | life / living |
| wissen | **das Wissen** | knowledge |

**-ung** 접미사 사용 (여성):

| 동사 | 명사 (-ung) | 영어 |
|------|------------|---------|

## 12. 명사화

명사화(Nominalization)는 동사, 형용사 또는 다른 품사를 명사로 바꾸는 과정이다. 독일어에서 명사화된 모든 단어는 대문자로 시작하며, 보통 중성(das)이다.

### 12.1 부정사 명사화

어떤 독일어 부정사든 대문자로 시작하면 중성 명사가 될 수 있다:

| 부정사 | 명사화된 형태 | 예문 |
|-----------|-----------------|-----------------|
| lesen | **das Lesen** | **Das Lesen** macht mir Spaß. (읽기는 나에게 즐겁다.) |
| schreiben | **das Schreiben** | **Das Schreiben** fällt ihm schwer. (쓰기는 그에게 어렵다.) |
| schwimmen | **das Schwimmen** | **Das Schwimmen** ist gesund. (수영은 건강에 좋다.) |
| reisen | **das Reisen** | **Das Reisen** bildet. (여행은 교양을 넓힌다.) |
| kochen | **das Kochen** | Ich liebe **das Kochen**. (나는 요리를 좋아한다.) |
| laufen | **das Laufen** | **Das Laufen** hält fit. (달리기는 건강을 유지시킨다.) |

### 12.2 명사화된 형용사 (복습)

명사로 사용된 형용사는 표준 형용사 어미를 취한다 (17과에서 다룸):

| 독일어 | 영어 |
|--------|---------|
| **das Gute** und **das Böse** | good and evil |
| **etwas Neues** | something new |
| **nichts Besonderes** | nothing special |
| **das Wichtigste** | the most important thing |
| **alles Gute!** | all the best! |

### 12.3 명사화된 분사

| 분사 | 명사화 | 영어 |
|-----------|------------|---------|
| reisend (여행하는) | **der/die Reisende** | the traveler |
| studierend (공부하는) | **der/die Studierende** | the student |
| angestellt (고용된) | **der/die Angestellte** | the employee |
| erwachsen (다 자란) | **der/die Erwachsene** | the adult |
| vorsitzend (의장을 맡는) | **der/die Vorsitzende** | the chairperson |

### 12.4 맥락에서의 명사화

명사화는 격식체 및 학술 독일어에서 매우 흔하다:

| 동사적 표현 (덜 격식적) | 명사화된 표현 (더 격식적) |
|---------------------------|-------------------------------|
| Wir müssen die Kosten **senken**. | **Die Senkung** der Kosten ist notwendig. |
| Man muss das Problem **lösen**. | **Die Lösung** des Problems ist dringend. |
| Die Bevölkerung **wächst**. | **Das Wachstum** der Bevölkerung nimmt zu. |
| Die Regierung **entscheidet** morgen. | **Die Entscheidung** der Regierung fällt morgen. |

## 13. 연습 문제

### 연습 1: 주제별 어휘 — 빈칸 채우기

어휘 섹션에서 올바른 단어를 선택하시오.

1. Der ______ hat das neue Gesetz verabschiedet. (독일 연방 의회)
2. Die ______ ______ sind eine Alternative zu fossilen Brennstoffen. (재생 에너지)
3. Der ______ schützt persönliche Daten im Internet. (데이터 보호)
4. Er muss ein ______ vom Arzt bekommen, um das Medikament zu kaufen. (처방전)
5. Der ______ des Unternehmens ist dieses Jahr um 20% gestiegen. (매출)

<details>
<summary>정답</summary>

1. **Bundestag**
2. **erneuerbaren Energien**
3. **Datenschutz**
4. **Rezept**
5. **Umsatz**

</details>

### 연습 2: 복합 명사 — 만들고 번역하기

주어진 구성 요소를 결합하여 복합 명사를 만든 후 번역하시오.

1. Kranken + Haus = ______
2. Haupt + Stadt = ______
3. Flug + Hafen = ______
4. Kühl + Schrank = ______
5. Hand + Schuh = ______
6. Geburts + Tag = ______

<details>
<summary>정답</summary>

1. **Krankenhaus** (병원) — das
2. **Hauptstadt** (수도) — die
3. **Flughafen** (공항) — der
4. **Kühlschrank** (냉장고) — der
5. **Handschuh** (장갑) — der
6. **Geburtstag** (생일) — der

</details>

### 연습 3: 성 판별

각 복합 명사의 성은 무엇인가? 기억하라: 마지막 구성 요소가 성을 결정한다.

1. die/der/das Bundesregierung?
2. die/der/das Schlafzimmer?
3. die/der/das Autobahn?
4. die/der/das Briefkasten?
5. die/der/das Krankenversicherung?

<details>
<summary>정답</summary>

1. **die** Bundesregierung (die Regierung)
2. **das** Schlafzimmer (das Zimmer)
3. **die** Autobahn (die Bahn)
4. **der** Briefkasten (der Kasten)
5. **die** Krankenversicherung (die Versicherung)

</details>

### 연습 4: 접두사

올바른 접두사(un-, ur-, miss-)를 추가하여 문장을 완성하시오.

1. Das ist ein ______verständnis — ich habe das nie gesagt! (오해)
2. Der Wald ist ______alt — er existiert seit Tausenden von Jahren. (아주 오래된)
3. Leider ist das ______möglich. (불가능한)
4. Sie ______traut seinen Worten. (불신하다)
5. Das Ergebnis war sehr ______befriedigend. (불만족스러운)

<details>
<summary>정답</summary>

1. **Miss**verständnis
2. **ur**alt
3. **un**möglich
4. **miss**traut
5. **un**befriedigend

</details>

### 연습 5: 접미사 — 단어 파생하기

지시된 접미사를 사용하여 새로운 단어를 만드시오.

1. wohnen + -ung = ______ (아파트)
2. frei + -heit = ______ (자유)
3. möglich + -keit = ______ (가능성)
4. Freund + -schaft = ______ (우정)
5. trinken + -bar = ______ (마실 수 있는)
6. Tag + -lich = ______ (매일의)
7. Eis + -ig = ______ (얼음 같은)
8. Europa + -isch = ______ (유럽의)

<details>
<summary>정답</summary>

1. die **Wohnung**
2. die **Freiheit**
3. die **Möglichkeit**
4. die **Freundschaft**
5. **trinkbar**
6. **täglich**
7. **eisig**
8. **europäisch**

</details>

### 연습 6: 명사화

명사화를 사용하여 문장을 더 격식적으로 다시 쓰시오.

1. Man muss die Umwelt schützen. → ______ der Umwelt ist wichtig.
2. Er hat sich entschieden. → ______ war schwierig.
3. Die Bevölkerung wächst. → ______ der Bevölkerung ist ein Thema.
4. Wir müssen die Daten analysieren. → ______ der Daten ist notwendig.

<details>
<summary>정답</summary>

1. **Der Schutz** der Umwelt ist wichtig.
2. **Die Entscheidung** war schwierig.
3. **Das Wachstum** der Bevölkerung ist ein Thema.
4. **Die Analyse** der Daten ist notwendig.

</details>

### 연습 7: 주제별 문장

이 과에서 배운 어휘를 사용하여 독일어로 번역하시오.

1. The government passed a new law.
2. We need to reduce CO2 emissions.
3. Artificial intelligence is changing the economy.
4. She needs to make a doctor's appointment.
5. The company's revenue increased by 15%.

<details>
<summary>정답</summary>

1. **Die Regierung hat ein neues Gesetz verabschiedet.**
2. **Wir müssen die CO₂-Emissionen reduzieren.**
3. **Künstliche Intelligenz verändert die Wirtschaft.**
4. **Sie muss einen Arzttermin vereinbaren.**
5. **Der Umsatz des Unternehmens ist um 15% gestiegen.**

</details>
