# 13. 분리동사와 비분리동사 (Trennbare und untrennbare Verben)

## 학습 목표

- 강세 패턴을 통해 분리 접두사 동사와 비분리 접두사 동사를 구별할 수 있다
- 어순 규칙을 적용할 수 있다: 분리 접두사(Separable Prefix)는 주절 끝으로 이동한다
- 종속절에서는 분리 접두사가 분리되지 않음을 인식할 수 있다
- 두 동사 유형 모두에서 Partizip II와 zu + 부정사(Infinitiv)를 올바르게 형성할 수 있다
- 이중 접두사 동사(durch-, ueber-, um-, unter-, wieder-)를 식별하고 의미에 따라 분리 여부를 판단할 수 있다
- 가장 흔한 분리동사 20개를 자신감 있게 문장에서 사용할 수 있다
- 가장 흔한 비분리동사 15개를 자신감 있게 문장에서 사용할 수 있다
- 모든 시제(현재, Perfekt, Praeteritum)에서 접두사 동사를 올바르게 다룰 수 있다
- 분리동사를 화법조동사(Modal Verb)와 올바르게 결합할 수 있다

## 1. 접두사 동사란 무엇인가?

독일어는 기본 동사에 **접두사(Praefix)**를 붙여 새로운 동사를 빈번하게 만든다. 접두사는 의미를 변화시키며, 때로는 극적으로 달라진다:

| 기본 동사 | 접두사 동사 | 새로운 의미 |
|-----------|------------|-------------|
| stehen (서다) | **auf**stehen | 일어나다 |
| stehen (서다) | **ver**stehen | 이해하다 |
| kommen (오다) | **an**kommen | 도착하다 |
| kommen (오다) | **be**kommen | 받다 |
| fahren (운전하다) | **ab**fahren | 출발하다 |
| fahren (운전하다) | **er**fahren | 경험하다, 알게 되다 |
| gehen (가다) | **aus**gehen | 외출하다 |
| gehen (가다) | **ent**gehen | 피하다, 면하다 |
| schreiben (쓰다) | **auf**schreiben | 적어 두다 |
| schreiben (쓰다) | **be**schreiben | 묘사하다 |

핵심 질문은: **접두사가 문장에서 동사로부터 분리되는가?**

```
+--------------------------------------------------------------+
|  SEPARABLE (trennbar)           INSEPARABLE (untrennbar)     |
|  Prefix is STRESSED             Stem is STRESSED              |
|  AUFstehen                      verSTEHen                     |
|  Prefix separates in            Prefix NEVER separates        |
|  main clauses                                                 |
|  Partizip II: aufGEstanden      Partizip II: verstanden       |
|  zu + Inf: aufZUstehen          zu + Inf: zu verstehen        |
+--------------------------------------------------------------+
```

> **핵심 통찰**: 강세 패턴이 모든 것을 알려준다. 강세가 어디에 있는지 들을 수 있다면, 그 동사가 분리동사인지 비분리동사인지 알 수 있다.

## 2. 분리 접두사 동사 (Trennbare Verben)

### 분리 접두사의 종류

다음 접두사들은 **항상 분리**된다. 대부분이 그 자체로 독립적인 단어(전치사 또는 부사)이기도 하다:

| 접두사 | 일반적 의미 | 동사 예시 | 번역 |
|--------|-------------|----------|------|
| **ab-** | 떨어져, 아래로 | abfahren | 출발하다 |
| **an-** | 위에, ~에, ~로, 시작 | ankommen | 도착하다 |
| **auf-** | 위로, 열림 | aufstehen | 일어나다 |
| **aus-** | 밖으로, 꺼짐, 완료 | ausgehen | 외출하다 |
| **bei-** | 옆에, 함께 | beitragen | 기여하다 |
| **ein-** | 안으로 | einladen | 초대하다 |
| **fest-** | 단단한, 고정된 | festhalten | 꽉 잡다 |
| **her-** | 화자 쪽으로 | herkommen | 이리 오다 |
| **hin-** | 화자에게서 멀어져 | hingehen | 그리로 가다 |
| **los-** | 느슨한, 시작 | losfahren | 출발하다 |
| **mit-** | 함께 | mitkommen | 같이 오다 |
| **nach-** | ~을 따라, 이후 | nachdenken | 숙고하다, 생각하다 |
| **vor-** | 앞으로, 이전 | vorstellen | 소개하다 |
| **weg-** | 멀리 | weggehen | 가버리다 |
| **zu-** | 닫힘, ~을 향해 | zumachen | 닫다 |
| **zurueck-** | 뒤로 | zurueckkommen | 돌아오다 |
| **zusammen-** | 함께 | zusammenarbeiten | 협력하다 |
| **weiter-** | 더, 계속하여 | weitermachen | 계속하다 |
| **vorbei-** | 지나서, 곁을 | vorbeikommen | 들르다, 방문하다 |
| **heraus-** | 밖으로 (화자 쪽) | herausfinden | 알아내다 |

> **강세 규칙**: 분리 접두사는 항상 **강세**를 받는다. 들을 수 있다:
> **AUF**stehen, **AN**kommen, **AUS**gehen. 접두사에 강세가 있으면 분리동사이다.

### 분리 접두사가 독립 단어인 이유

대부분의 분리 접두사는 그 자체로 전치사나 부사로 기능할 수 있다:

| 접두사 | 전치사/부사로서 | 동사 접두사로서 |
|--------|----------------|----------------|
| an | an der Wand (벽에) | ankommen (도착하다) |
| auf | auf dem Tisch (탁자 위에) | aufstehen (일어나다) |
| aus | aus dem Haus (집 밖으로) | ausgehen (외출하다) |
| mit | mit dem Bus (버스로) | mitkommen (같이 오다) |
| vor | vor dem Haus (집 앞에) | vorstellen (소개하다) |

이것이 분리 접두사가 강세를 받고 절의 끝에서 독립적으로 설 수 있는 이유이다.

## 3. 주절에서의 분리동사

**주절(Hauptsatz)**에서 분리 접두사는 동사로부터 분리되어 절의 **끝**으로 이동한다.

### 현재 시제(Praesens)

```
Subject + conjugated stem (position 2) + ... + prefix (end)
```

| 원형 | 주절 | 번역 |
|------|------|------|
| aufstehen | Ich **stehe** um 7 Uhr **auf**. | 나는 7시에 일어난다. |
| ankommen | Der Zug **kommt** um 9 Uhr **an**. | 기차가 9시에 도착한다. |
| einkaufen | Wir **kaufen** im Supermarkt **ein**. | 우리는 슈퍼마켓에서 장을 본다. |
| anrufen | Er **ruft** seine Mutter **an**. | 그는 어머니에게 전화한다. |
| fernsehen | Die Kinder **sehen** abends **fern**. | 아이들은 저녁에 TV를 본다. |
| mitkommen | **Kommst** du **mit**? | 같이 갈래? |
| zumachen | **Mach** bitte die Tuer **zu**! | 문을 닫아 주세요! |
| vorstellen | Ich **stelle** mich **vor**. | 나는 자기소개를 한다. |
| weitermachen | Wir **machen** morgen **weiter**. | 우리는 내일 계속한다. |

### 과거 시제(Praeteritum)

동일한 분리 규칙이 단순 과거에서도 적용된다:

| 원형 | Praeteritum 주절 | 번역 |
|------|-------------------|------|
| aufstehen | Ich **stand** um 7 Uhr **auf**. | 나는 7시에 일어났다. |
| ankommen | Der Zug **kam** puenktlich **an**. | 기차가 정시에 도착했다. |
| anrufen | Er **rief** seine Freundin **an**. | 그는 여자친구에게 전화했다. |
| anfangen | Das Konzert **fing** um 8 Uhr **an**. | 콘서트가 8시에 시작했다. |
| einladen | Sie **lud** ihre Freunde **ein**. | 그녀는 친구들을 초대했다. |
| ausgehen | Wir **gingen** gestern Abend **aus**. | 우리는 어젯밤에 외출했다. |

### 현재완료(Perfekt)

Perfekt에서는 전체 Partizip II가 끝에 함께 남는다 (분리가 필요 없다 — 접두사가 이미 Partizip II의 일부이다):

| 원형 | Perfekt | 번역 |
|------|---------|------|
| aufstehen | Ich **bin** um 7 Uhr **aufgestanden**. | 나는 7시에 일어났다. |
| ankommen | Der Zug **ist** puenktlich **angekommen**. | 기차가 정시에 도착했다. |
| anrufen | Er **hat** seine Freundin **angerufen**. | 그는 여자친구에게 전화했다. |
| einkaufen | Wir **haben** im Supermarkt **eingekauft**. | 우리는 슈퍼마켓에서 장을 봤다. |
| einladen | Sie **hat** uns zum Essen **eingeladen**. | 그녀는 우리를 식사에 초대했다. |
| mitnehmen | **Hast** du den Regenschirm **mitgenommen**? | 우산을 가져갔니? |

### 명령문(Imperativ)

명령문에서도 접두사는 끝으로 간다:

| 동사 | Imperativ (du) | Imperativ (ihr) | Imperativ (Sie) |
|------|----------------|-----------------|-----------------|
| aufstehen | **Steh auf**! | **Steht auf**! | **Stehen Sie auf**! |
| zumachen | **Mach** die Tuer **zu**! | **Macht** die Tuer **zu**! | **Machen Sie** die Tuer **zu**! |
| anrufen | **Ruf** mich **an**! | **Ruft** mich **an**! | **Rufen Sie** mich **an**! |
| mitkommen | **Komm mit**! | **Kommt mit**! | **Kommen Sie mit**! |

## 4. 종속절에서의 분리동사

**종속절(Nebensatz)**에서 접두사는 분리되지 **않는다**. 동사는 하나로 합쳐진 채 절의 끝으로 이동한다.

```
..., conjunction + Subject + ... + prefix+conjugated verb (end)
```

### 비교표

| 주절 (분리됨) | 종속절 (합쳐져 있음) |
|---------------|---------------------|
| Ich **stehe** um 7 Uhr **auf**. | ..., weil ich um 7 Uhr **aufstehe**. |
| Er **kommt** morgen **an**. | ..., dass er morgen **ankommt**. |
| Wir **kaufen** heute **ein**. | ..., obwohl wir heute **einkaufen**. |
| Sie **ruft** ihn **an**. | Ich weiss, dass sie ihn **anruft**. |
| Der Film **faengt** um 8 **an**. | ..., bevor der Film um 8 **anfaengt**. |

### 완전한 문장 예시

- Ich bin muede, **weil** ich jeden Tag um 5 Uhr **aufstehe**.
  (매일 5시에 일어나기 때문에 피곤하다.)

- Er sagt, **dass** er morgen **zurueckkommt**.
  (그는 내일 돌아온다고 말한다.)

- Ich weiss nicht, **ob** sie heute **mitkommt**.
  (그녀가 오늘 같이 오는지 모르겠다.)

- **Wenn** du das Fenster **aufmachst**, wird es kalt.
  (창문을 열면 추워질 것이다.)

- **Obwohl** er jeden Tag frueh **aufsteht**, kommt er immer zu spaet.
  (매일 일찍 일어나지만 그는 항상 늦게 온다.)

- Ich freue mich, **dass** du morgen **vorbeikommst**.
  (내일 네가 들른다니 기쁘다.)

> **중요**: 종속절 내의 Perfekt에서는 Partizip II가 합쳐진 채로 남고, 조동사가 맨 끝으로 간다:
>
> - ..., weil ich gestern frueh **aufgestanden bin**.
> - ..., dass er seine Mutter **angerufen hat**.

## 5. 화법조동사와 분리동사

분리동사가 화법조동사(koennen, muessen, wollen, sollen, duerfen, moegen)와 함께 사용될 때, 분리동사는 절의 끝에서 부정사 형태로 합쳐진 채 남는다. 화법조동사가 2번 위치에서 활용된다.

```
Subject + modal (pos. 2) + ... + separable verb infinitive (end)
```

| 화법조동사 + 분리동사 | 문장 | 번역 |
|----------------------|------|------|
| muessen + aufstehen | Ich **muss** frueh **aufstehen**. | 나는 일찍 일어나야 한다. |
| koennen + mitkommen | **Kannst** du **mitkommen**? | 같이 올 수 있어? |
| wollen + ausgehen | Wir **wollen** heute Abend **ausgehen**. | 우리는 오늘 저녁에 외출하고 싶다. |
| sollen + anrufen | Du **sollst** deine Mutter **anrufen**. | 네 어머니에게 전화해야 한다. |
| duerfen + fernsehen | Die Kinder **duerfen** nicht **fernsehen**. | 아이들은 TV를 봐서는 안 된다. |
| moechten + einkaufen | Ich **moechte** im Zentrum **einkaufen**. | 시내에서 쇼핑하고 싶다. |

> **핵심 규칙**: 화법조동사와 함께 사용될 때 분리동사는 절대 분리되지 않는다. 절의 끝에서 부정사 형태의 한 단어로 남는다.

## 6. 분리동사의 Partizip II와 zu + 부정사

### Partizip II

**ge-** 가 접두사와 어간 **사이에** 삽입된다:

```
prefix + ge + stem + t/en
```

| 원형 | Partizip II | 유형 |
|------|-------------|------|
| aufmachen | auf**ge**macht | 약변화 |
| anfangen | an**ge**fangen | 강변화 |
| aufstehen | auf**ge**standen | 강변화 |
| einkaufen | ein**ge**kauft | 약변화 |
| zurueckkommen | zurueck**ge**kommen | 강변화 |
| mitnehmen | mit**ge**nommen | 강변화 |
| anrufen | an**ge**rufen | 강변화 |
| vorstellen | vor**ge**stellt | 약변화 |
| einladen | ein**ge**laden | 강변화 |
| abfahren | ab**ge**fahren | 강변화 |
| fernsehen | fern**ge**sehen | 강변화 |
| zumachen | zu**ge**macht | 약변화 |

### zu + 부정사(Infinitiv)

특정 동사, 형용사, 구문 뒤에서 **zu**가 필요한 경우, **zu**는 접두사와 동사 사이에 삽입되며, 한 단어로 쓴다:

```
prefix + zu + infinitive    (한 단어!)
```

| 표준 부정사 | zu + 부정사 |
|------------|------------|
| aufstehen | **auf**zu**stehen** |
| anfangen | **an**zu**fangen** |
| anrufen | **an**zu**rufen** |
| einkaufen | **ein**zu**kaufen** |
| mitkommen | **mit**zu**kommen** |
| weggehen | **weg**zu**gehen** |
| fernsehen | **fern**zu**sehen** |

**문장 속 예시**:

- Es ist schwer, jeden Tag um 5 Uhr **aufzustehen**. (매일 5시에 일어나는 것은 힘들다.)
- Ich versuche, dich morgen **anzurufen**. (내일 너에게 전화하려고 한다.)
- Vergiss nicht, Milch **einzukaufen**. (우유 사는 것을 잊지 마.)
- Er hat keine Lust, **mitzukommen**. (그는 같이 가고 싶지 않다.)
- Ich habe vor, morgen frueh **anzufangen**. (내일 일찍 시작할 계획이다.)
- Es ist wichtig, den Kindern **zuzuhoeren**. (아이들의 말을 듣는 것이 중요하다.)

## 7. 비분리 접두사 동사 (Untrennbare Verben)

### 비분리 접두사의 종류

다음 접두사들은 **항상 비분리**이다. 이들은 독립적인 단어가 **아니다**:

| 접두사 | 일반적 의미 | 동사 예시 | 번역 |
|--------|-------------|----------|------|
| **be-** | 자동사를 타동사로 전환 | besuchen | 방문하다 |
| **emp-** | 수용, 감정 | empfehlen | 추천하다 |
| **ent-** | 제거, 탈출, 시작 | entdecken | 발견하다 |
| **er-** | 달성, 결과, 시작 | erklaeren | 설명하다 |
| **ge-** | 완료, 결과 | gehoeren | ~에 속하다 |
| **miss-** | 잘못, 실패 | missverstehen | 오해하다 |
| **ver-** | 변화, 강화, 오류 | verstehen | 이해하다 |
| **zer-** | 파괴, 분해 | zerstoeren | 파괴하다 |

> **강세 규칙**: 비분리 접두사는 **강세가 없다**. 강세는 어간에 온다:
> ver**STE**hen, be**KOM**men, er**ZAEH**len. 어간에 강세가 있으면 비분리동사이다.

### 비분리 접두사 암기법

> **"Be Emp Ent Er Ge Miss Ver Zer"** — 하나의 그룹으로 기억하라. 이 접두사들은 독일어에서 독립 단어로 존재할 수 없다.

### 비분리동사의 핵심 특성

1. 절의 유형에 관계없이 **절대 분리되지 않는다**
2. Partizip II에 **ge-가 없다**: verstanden (verstanden이지, *ge*verstanden이 아니다)
3. **일반적인 zu + 부정사**: zu verstehen, zu bekommen (*zu ver stehen*이 아니다)

### 문장 속 예시

- Ich **verstehe** die Frage nicht. (질문을 이해하지 못하겠다.)
- Er **besucht** seine Grosseltern. (그는 조부모를 방문한다.)
- Sie **erzaehlt** eine Geschichte. (그녀는 이야기를 들려준다.)
- Wir **bekommen** morgen Besuch. (내일 손님이 온다.)
- Das Kind **entdeckt** einen Frosch. (아이가 개구리를 발견한다.)
- Ich **empfehle** dieses Restaurant. (이 레스토랑을 추천한다.)

### 비분리동사의 Partizip II

| 원형 | Partizip II | 참고 |
|------|-------------|------|
| verstehen | **verstanden** | ge- 없음 |
| bekommen | **bekommen** | ge- 없음 |
| erzaehlen | **erzaehlt** | ge- 없음 |
| beginnen | **begonnen** | ge- 없음 |
| empfehlen | **empfohlen** | ge- 없음 |
| entdecken | **entdeckt** | ge- 없음 |
| vergessen | **vergessen** | ge- 없음 |
| zerstoeren | **zerstoert** | ge- 없음 |
| gefallen | **gefallen** | ge- 없음 |
| misslingen | **misslungen** | ge- 없음 |
| erfahren | **erfahren** | ge- 없음 |
| beschreiben | **beschrieben** | ge- 없음 |

## 8. 비분리 접두사의 의미 상세 분석

각 비분리 접두사가 기본 동사에 어떤 작용을 하는지 이해하면 낯선 단어의 의미를 추측하는 데 도움이 된다.

### be- (타동사화 / 행위를 대상에 지향시킴)

접두사 **be-** 는 자동사를 타동사로 바꾸거나, 행위를 직접 목적어 쪽으로 방향 전환한다:

| 기본 동사 | be- 동사 | 의미 변화 |
|-----------|---------|-----------|
| antworten (+ 3격) | **be**antworten (+ 4격) | (질문에) 대답하다 |
| kommen | **be**kommen | 받다, 얻다 |
| suchen | **be**suchen | 방문하다 |
| zahlen | **be**zahlen | (무엇에 대해) 지불하다 |
| wohnen | **be**wohnen | 거주하다 |
| treten (밟다) | **be**treten | 들어서다 |

### ver- (변화, 강화, 또는 오류)

| 의미 범주 | 예시 | 번역 |
|-----------|------|------|
| **변화** | verkaufen | 팔다 (소유권 변화) |
| **강화** | verbessern | 개선하다 (더 좋게 만들다) |
| **오류** | sich verlaufen | 길을 잃다 (잘못된 길로 가다) |
| **사라짐** | verschwinden | 사라지다 |
| **완료** | verbrauchen | 다 소비하다 |

### er- (달성 또는 시작)

| 의미 범주 | 예시 | 번역 |
|-----------|------|------|
| **달성** | erreichen | 도달하다, 달성하다 |
| **결과** | erfahren | 알게 되다, 경험하다 |
| **시작** | erklingen | 울리기 시작하다 |
| **죽음에 이름** | ertrinken | 익사하다 |

### ent- (제거, 탈출, 또는 시작)

| 의미 범주 | 예시 | 번역 |
|-----------|------|------|
| **제거** | entfernen | 제거하다 |
| **탈출** | entkommen | 탈출하다 |
| **시작** | entstehen | 생겨나다 |
| **역전** | entdecken | 발견하다 (드러내다) |

### zer- (파괴, 분해)

| 예시 | 번역 |
|------|------|
| zerbrechen | (조각내어) 부수다 |
| zerstoeren | 파괴하다 |
| zerreissen | 찢어 버리다 |
| zerschneiden | 잘라 버리다 |

## 9. 이중 접두사 동사: 분리인가 비분리인가?

일부 접두사는 의미에 따라 **분리 또는 비분리** 둘 다 될 수 있다. 문자적/물리적 의미는 대체로 분리이고, 비유적/추상적 의미는 비분리이다.

| 접두사 | 분리 (문자적/물리적) | 비분리 (비유적/추상적) |
|--------|---------------------|----------------------|
| **durch-** | durchfahren (통과하여 운전하다) | durchfahren (쉬지 않고 계속 가다) |
| **ueber-** | uebersetzen (배로 건너다) | uebersetzen (번역하다) |
| **um-** | umziehen (옷을 갈아입다 / 이사하다) | umgeben (둘러싸다) |
| **unter-** | untergehen (가라앉다) | unterschreiben (서명하다) |
| **wieder-** | wiederkommen (돌아오다) | wiederholen (반복하다) |

### 구별하는 방법

| 특징 | 분리 | 비분리 |
|------|------|--------|
| **강세** | **접두사**에 | **어간**에 |
| **의미** | 문자적, 물리적 | 비유적, 추상적 |
| **Partizip II** | 접두사 + ge + 어간 | 접두사 + 어간 (ge 없음) |
| **zu + 부정사** | 접두사 + zu + 어간 | zu + 접두사 + 어간 |

### durch-의 상세 예시

**분리** (문자적: 물리적 공간을 통과):
- Er **faehrt** hier **durch**. (그는 여기를 통과하여 운전한다.)
- Partizip II: **durchgefahren**

**비분리** (비유적: 견디다, 완수하다):
- Er **durchfaehrt** die ganze Nacht. (그는 밤새 계속 운전한다.)
- Partizip II: **durchfahren** (ge- 없음)

### ueber-의 상세 예시

**분리** (문자적: 물리적으로 건너감):
- Das Boot **setzt** ueber. (배가 건너간다.)
- Partizip II: **uebergesetzt**

**비분리** (비유적: 번역):
- Er **uebersetzt** den Text. (그는 텍스트를 번역한다.)
- Partizip II: **uebersetzt** (ge- 없음)

### um-의 상세 예시

**분리** (문자적: 바꾸다/이동하다):
- Ich **ziehe** mich **um**. (나는 옷을 갈아입는다.)
- Partizip II: **umgezogen**
- Wir **ziehen** naechste Woche **um**. (우리는 다음 주에 이사한다.)

**비분리** (비유적: 둘러싸다):
- Die Polizei **umstellt** das Gebaeude. (경찰이 건물을 포위한다.)
- Partizip II: **umstellt** (ge- 없음)

### unter-의 상세 예시

**분리** (문자적: 아래로 가다):
- Die Sonne **geht unter**. (해가 진다.)
- Partizip II: **untergegangen**

**비분리** (비유적: 서명하다, 조사하다):
- Er **unterschreibt** den Vertrag. (그는 계약서에 서명한다.)
- Partizip II: **unterschrieben** (ge- 없음)

### wieder-의 상세 예시

**분리** (문자적: 돌아오다):
- Er **kommt** morgen **wieder**. (그는 내일 다시 온다.)
- Partizip II: **wiedergekommen**

**비분리** (비유적: 반복하다):
- Sie **wiederholt** die Lektion. (그녀는 수업을 반복한다.)
- Partizip II: **wiederholt** (ge- 없음)

## 12. 모든 시제에서의 분리동사

여기서는 분리가 어떻게 작용하는지 보여주기 위해 **anfangen** (시작하다)을 모든 시제에 걸쳐 완전히 활용한다:

### 현재(Praesens)

| 인칭 | 활용 |
|------|------|
| ich | fange ... an |
| du | faengst ... an |
| er/sie/es | faengt ... an |
| wir | fangen ... an |
| ihr | fangt ... an |
| sie/Sie | fangen ... an |

> Der Film **faengt** um acht Uhr **an**.

### 과거(Praeteritum)

| 인칭 | 활용 |
|------|------|
| ich | fing ... an |
| du | fingst ... an |
| er/sie/es | fing ... an |
| wir | fingen ... an |
| ihr | fingt ... an |
| sie/Sie | fingen ... an |

> Der Film **fing** um acht Uhr **an**.

### 현재완료(Perfekt)

> Der Film **hat** um acht Uhr **angefangen**.

### 과거완료(Plusquamperfekt)

> Der Film **hatte** um acht Uhr **angefangen**.

### 미래(Futur I)

> Der Film **wird** um acht Uhr **anfangen**.

### 화법조동사와 함께

> Der Film **soll** um acht Uhr **anfangen**.

### 종속절에서 (현재)

> ..., weil der Film um acht Uhr **anfaengt**.

### 종속절에서 (Perfekt)

> ..., weil der Film um acht Uhr **angefangen hat**.

## 13. 요약: 빠른 참조표

```
+----------------------------------------------------------------+
|                  PREFIX VERB QUICK REFERENCE                    |
+----------------------------------------------------------------+
|                                                                |
|  ALWAYS SEPARABLE:                                             |
|  ab- an- auf- aus- bei- ein- fest- her- hin- los-            |
|  mit- nach- vor- weg- zu- zurueck- zusammen- weiter-         |
|  vorbei- heraus-                                               |
|  -> Prefix stressed: AUFstehen                                 |
|  -> Main clause: prefix goes to END                            |
|  -> Subord. clause: verb stays together                        |
|  -> With modal: infinitive stays together                      |
|  -> Partizip II: auf-GE-standen                                |
|  -> zu + Inf: aufZUstehen                                      |
|                                                                |
|  ALWAYS INSEPARABLE:                                           |
|  be- emp- ent- er- ge- miss- ver- zer-                        |
|  -> Stem stressed: verSTEHen                                   |
|  -> Never separates                                            |
|  -> Partizip II: verstanden (NO ge-)                           |
|  -> zu + Inf: zu verstehen                                     |
|                                                                |
|  DUAL (depends on meaning):                                    |
|  durch- ueber- um- unter- wieder-                              |
|  -> Literal/physical = separable (prefix stressed)             |
|  -> Figurative/abstract = inseparable (stem stressed)          |
|                                                                |
+----------------------------------------------------------------+
```

### 판단 순서도

```
Is the prefix be-, emp-, ent-, er-, ge-, miss-, ver-, or zer-?
   |
   YES --> INSEPARABLE (always)
   |
   NO --> Is the prefix ab-, an-, auf-, aus-, bei-, ein-, etc.?
          |
          YES --> SEPARABLE (always)
          |
          NO --> Is the prefix durch-, ueber-, um-, unter-, wieder-?
                 |
                 YES --> Check meaning:
                         Literal/physical? --> SEPARABLE
                         Figurative/abstract? --> INSEPARABLE
```

### 연습 1: 분리인가 비분리인가?

각 동사를 분리(S), 비분리(I), 이중(D)으로 분류하고 이유를 설명하시오.

2. verstehen

7. wiederholen

<summary>정답</summary>

9. It is hard to get up early.
10. He says that he is coming back tomorrow.

<details>
<summary>정답</summary>

1. Ich **stehe** jeden Morgen um 6 Uhr **auf**. (주절에서 분리)
2. Sie **versteht** die Frage nicht. (비분리 — 합쳐져 있음)
3. **Kannst** du **mitkommen**? (화법조동사 + 분리 부정사가 합쳐져 있음)
4. Er **hat** gestern seine Mutter **angerufen**. (Perfekt: 분리 Partizip II)
5. Ich bin muede, weil ich immer frueh **aufstehe**. (종속절 — 합쳐져 있음)
6. **Vergiss** deinen Schluessel nicht! (비분리 — 합쳐져 있음)
7. Die Kinder **duerfen** nicht **fernsehen**. (화법조동사 + 분리 부정사)
8. Sie **uebersetzt** das Buch. (비분리 — 합쳐져 있음)
9. Es ist schwer, frueh **aufzustehen**. (zu가 접두사와 어간 사이에)
10. Er sagt, dass er morgen **zurueckkommt**. (종속절 — 합쳐져 있음)
