# 21. 독해력 (Leseverstehen)

## 학습 목표

- 다양한 수준의 독일어 텍스트를 읽기 위한 효과적인 전략 개발
- 독일어 합성어(Komposita) 분해 기법 습득
- 훑어보기(Überfliegen) 및 찾아보기(Suchendes Lesen) 전략 연습
- 문맥 단서를 활용하여 모르는 단어의 의미 추론
- 다양한 독일어 텍스트 유형과 구조적 특징 인식
- 문장과 단락 간 아이디어를 연결하는 응집 장치(Cohesive Devices) 이해
- 학술 독해 능력 구축: 논지 파악, 사실과 의견 구별
- A2-B1 수준의 실제 독일어 텍스트를 활용한 독해 연습

## 1. 독해 전략 개요

```
+----------------------------------------------------------+
|           READING STRATEGIES FOR GERMAN                   |
+----------------------------------------------------------+
|                                                          |
|  BEFORE reading (Vor dem Lesen):                         |
|  1. Look at title, headings, images, captions            |
|  2. Predict content from context                         |
|  3. Activate your background knowledge                   |
|  4. Identify the text type (article? letter? story?)     |
|                                                          |
|  DURING reading (Während des Lesens):                    |
|  5. Don't stop at every unknown word                     |
|  6. Break down compound words into parts                 |
|  7. Use context to guess meaning                         |
|  8. Pay attention to connectors and signal words          |
|  9. Notice the verb position — it reveals clause type     |
|                                                          |
|  AFTER reading (Nach dem Lesen):                         |
|  10. Summarize the main idea in your own words           |
|  11. Review key vocabulary                               |
|  12. Re-read difficult passages                          |
|  13. Ask: Who? What? When? Where? Why? How?              |
+----------------------------------------------------------+
```

> **핵심 원칙**: 텍스트를 이해하기 위해 모든 단어를 알 필요는 없습니다. 독일어 텍스트는 긴 합성어와 복잡한 문장 구조 때문에 어려워 보일 수 있지만, 실제 의미는 처음 보이는 것보다 훨씬 쉽게 파악할 수 있는 경우가 많습니다.

## 2. 훑어보기와 찾아보기

### 훑어보기 (Überfliegen)

목적: 텍스트의 **전체적인 내용**을 빠르게 파악합니다.

독일어 텍스트를 훑어보는 방법:
- 제목(*Überschrift*)과 부제목(*Unterüberschriften*)을 읽습니다
- 각 단락의 첫 번째 문장을 읽습니다 — 독일어 단락은 일반적으로 핵심 내용으로 시작합니다
- 굵은 글씨, 이탤릭체, 대문자로 된 단어를 찾습니다 (참고: 독일어의 모든 명사는 대문자로 시작하므로 다른 서식에 집중하세요)
- 마지막 단락을 읽습니다 — 보통 요약이나 결론이 포함됩니다
- 이미지, 차트, 캡션을 살펴봅니다

### 찾아보기 (Suchendes Lesen)

목적: **특정 정보**를 빠르게 찾습니다.

독일어 텍스트에서 찾아보기 방법:
- 읽기 시작하기 전에 무엇을 찾을 것인지 결정합니다
- 텍스트 위로 빠르게 시선을 이동합니다
- 숫자, 고유명사, 날짜, 키워드를 찾습니다
- 독일어 어순에서 핵심 정보가 문장 끝에 올 수 있다는 것을 기억합니다 (종속절에서 동사가 문미에 위치)

### 정독 (Genaues Lesen)

목적: 텍스트를 **상세하게** 이해합니다.

사용 시기:
- 학술 텍스트, 지시문, 계약서
- 시험 독해 지문
- 훑어보기를 통해 중요한 부분을 파악한 후

### 각 전략별 팁

| 전략 | 속도 | 이해도 목표 | 적합한 용도 |
|------|------|------------|------------|
| Überfliegen (훑어보기) | 빠름 | 전체적인 내용 (50-60%) | 기사, 긴 텍스트, 미리보기 |
| Suchendes Lesen (찾아보기) | 빠름 | 특정 정보만 | 시간표, 목록, 참고 자료 |
| Genaues Lesen (정독) | 느림 | 완전한 이해 (90%+) | 학습 자료, 시험 지문, 지시문 |

## 3. 합성어 다루기

독일어 학습자에게 가장 큰 도전 중 하나는 독일어의 합성 명사에 대한 선호입니다. 독일어는 단어를 자유롭게 결합하여 새로운 단어를 만들며, 때로는 놀라울 정도로 긴 단어가 탄생합니다.

### 규칙: 오른쪽에서 왼쪽으로 읽기

합성어에서 **마지막 단어**가 성(性)과 일반적인 범주를 결정합니다. 오른쪽에서 왼쪽으로 읽으세요:

| 합성어 | 구성 요소 | 의미 |
|--------|----------|------|
| **Haus**tür | Haus + Tür | 집 문 (현관문) |
| **Küchen**tisch | Küche + n + Tisch | 부엌 탁자 |
| **Flug**hafen | Flug + Hafen | 비행 항구 (공항) |
| **Hand**schuh | Hand + Schuh | 손 신발 (장갑) |
| **Kranken**haus | krank + en + Haus | 아픈 사람의 집 (병원) |
| **Kühl**schrank | kühl + Schrank | 차가운 장 (냉장고) |

> 가장 오른쪽의 명사가 **핵심어**(Grundwort)입니다. 그 앞의 모든 것은 핵심어를 수식하거나 한정합니다.

### 긴 단어 분해하기

유명한 예시를 분석해 봅시다:

**Donaudampfschifffahrtsgesellschaft**

단계별로 분해합니다:
- **Donau** — 도나우 (강 이름)
- **Dampf** — 증기
- **Schiff** — 배
- **fahrt** — 여행/운항
- **s** — 연결 요소 (Fugen-s)
- **Gesellschaft** — 회사/단체

결과: "도나우 증기선 운항 회사"

### 연결 요소 (Fugenelemente)

독일어 합성어에는 때때로 작은 연결 요소가 삽입됩니다:

| 요소 | 예시 | 구성 요소 |
|------|------|----------|
| **-s-** | Arbeit**s**platz | Arbeit + Platz (직장) |
| **-n-** | Blume**n**laden | Blume + Laden (꽃집) |
| **-en-** | Frau**en**arzt | Frau + Arzt (산부인과 의사) |
| **-er-** | Kind**er**garten | Kind + Garten (아이들의 정원) |
| (없음) | Haustür | Haus + Tür (현관문) |

### 연습: 분해해 보세요

다음 합성어를 분해해 보세요:

1. Schlafzimmer
2. Fahrkarte
3. Weihnachtsmarkt
4. Geburtstagsfeier
5. Straßenbahn

<details>
<summary>정답</summary>

1. **Schlaf** (잠) + **Zimmer** (방) = 침실
2. **Fahr** (운전/여행) + **Karte** (카드/표) = 승차권
3. **Weihnacht** (크리스마스) + **s** + **Markt** (시장) = 크리스마스 시장
4. **Geburtstag** (생일) + **s** + **Feier** (축하) = 생일 파티
5. **Straße** (거리) + **n** + **Bahn** (열차/궤도) = 노면전차/트램

</details>

## 4. 문맥 단서 활용하기

모르는 단어를 만났을 때, 이 전략들을 사용하면 사전 없이도 의미를 파악할 수 있습니다.

### 유형 1: 정의 단서

텍스트에서 직접 단어를 설명합니다:

> "Das **Abitur**, also die Abschlussprüfung am Gymnasium, ist Voraussetzung für ein Studium."
> (Abitur, 즉 김나지움의 졸업 시험은 대학 진학의 전제 조건입니다.)

신호 표현: **also** (즉/다시 말해), **das heißt** (그것은 ~을 의미한다), **d.h.** (즉), **und zwar** (구체적으로)

### 유형 2: 유의어 단서

비슷한 의미의 단어가 가까이에 나타납니다:

> "Er war völlig **erschöpft**, total müde nach dem langen Arbeitstag."
> (그는 긴 근무일 후에 완전히 지쳐 있었다, 완전히 피곤한 상태였다.)

### 유형 3: 반의어 단서

반대되는 단어가 대조를 제공합니다:

> "Sie war nicht **schüchtern**, sondern sehr selbstbewusst."
> (그녀는 수줍어하지 않았다, 오히려 매우 자신감이 있었다.)

신호 단어: **sondern** (오히려), **im Gegenteil** (반대로), **nicht...sondern** (~이 아니라 오히려)

### 유형 4: 예시 단서

예시가 의미를 보여줍니다:

> "In Deutschland gibt es viele **Feiertage**, zum Beispiel Weihnachten, Ostern und den Tag der Deutschen Einheit."
> (독일에는 많은 공휴일이 있습니다. 예를 들어 크리스마스, 부활절, 독일 통일의 날 등입니다.)

신호 표현: **zum Beispiel** (예를 들어), **wie** (~와 같은), **beispielsweise** (예를 들면)

### 유형 5: 어족 단서

이미 아는 관련 단어를 인식합니다:

| 아는 단어 | 모르는 단어 | 추론 |
|----------|-----------|------|
| fahren (운전하다) | der Fahrer | 운전사 |
| lesen (읽다) | die Lesung | 낭독회 |
| freund (친구) | freundlich | 친절한 |
| arbeiten (일하다) | der Arbeitnehmer | 직원 (일을 받는 사람) |
| kaufen (사다) | das Kaufhaus | 백화점 |

## 5. 텍스트 구조 인식하기

독일어 텍스트는 일반적으로 명확한 구조적 패턴을 따릅니다. 이를 알면 정보를 어디서 찾을 수 있는지 예측하는 데 도움이 됩니다.

### 표준 텍스트 구조

```
+----------------------------------------------------------+
|  Einleitung (Introduction)                                |
|  → Main topic, thesis, or hook                            |
|  → Answers: What is this text about?                      |
+----------------------------------------------------------+
|  Hauptteil (Body)                                         |
|  → Arguments, evidence, details, examples                 |
|  → Often organized by:                                    |
|    - Chronology (first...then...finally)                  |
|    - Pro/Con (on one hand...on the other hand)            |
|    - Cause and effect (because...therefore)               |
+----------------------------------------------------------+
|  Schluss (Conclusion)                                     |
|  → Summary, final opinion, outlook, call to action        |
|  → Answers: So what? What's the takeaway?                 |
+----------------------------------------------------------+
```

### 단락 수준의 신호

텍스트의 흐름을 이해하기 위해 단락 시작 부분에서 다음 단어들을 찾아보세요:

| 신호 | 기능 | 영어 |
|------|------|------|
| Zunächst... | 첫 번째 요점 도입 | First of all... |
| Außerdem... | 정보 추가 | Furthermore... |
| Einerseits...andererseits | 양면 제시 | On one hand...on the other |
| Im Gegensatz dazu... | 대조 도입 | In contrast to this... |
| Zusammenfassend... | 결론 신호 | In summary... |
| Abschließend... | 마무리 신호 | In conclusion... |

## 6. 텍스트 유형과 특징

### Zeitungsartikel (뉴스 기사)

**구조**: 역피라미드형 — 가장 중요한 정보가 먼저, 세부 사항이 뒤따름

**특징**:
- 헤드라인(*Schlagzeile*): 짧고 주목을 끌며, 관사를 생략할 수 있음
- 리드 문단(*Vorspann*): 누가, 무엇을, 언제, 어디서, 왜
- 격식 있고 객관적인 언어
- 3인칭 시점
- 관련 당사자의 인용문
- 서술에 과거 시제(Präteritum) 사용

**헤드라인 스타일 예시**: "Bundesregierung beschließt neues Klimapaket"

### Briefe und E-Mails (편지와 이메일)

**격식체(formell)**:
- 인사말: *Sehr geehrte Damen und Herren,* / *Sehr geehrter Herr Schmidt,*
- *Sie* (격식 "당신") 사용
- 맺음말: *Mit freundlichen Grüßen*
- 객관적이고 정중한 어조

**비격식체(informell)**:
- 인사말: *Liebe Anna,* / *Lieber Max,* / *Hallo!*
- *du/ihr* 사용
- 맺음말: *Liebe Grüße* / *Viele Grüße* / *Bis bald!*
- 편안하고 개인적인 어조

### Sachtext (정보 텍스트)

**특징**:
- 객관적이고 사실적인 언어
- 전문 용어(*Fachwörter*)
- 수동태가 자주 사용됨: *Es wird geschätzt, dass...* (... 으로 추정된다)
- 통계, 데이터, 참고 자료
- 중립적인 어조 — 개인적 의견 없음

### Literarischer Text (문학 텍스트)

**특징**:
- 서술 시점 (1인칭 또는 3인칭)
- 서술적이고 비유적인 언어(*Bildsprache*)
- 은유, 직유: *wie ein offenes Buch* (열린 책처럼)
- 리듬과 효과를 위한 다양한 문장 구조
- 감정적인 어휘와 심상
- 과거 시제: 주요 서술에 Präteritum, 회상에 Plusquamperfekt

### Gebrauchstext (실용 텍스트)

**특징**:
- 명확하고 직접적인 지시
- 명령형 또는 부정사 형태: *Drücken Sie den Knopf* / *Knopf drücken*
- 목록, 번호 매긴 단계, 글머리 기호
- 도표 또는 삽화
- 예시: 레시피(*Rezepte*), 설명서, 광고(*Werbung*), 일정표

## 7. 연습 텍스트 1: 뉴스 기사

다음 기사를 읽고 질문에 답하세요.

---

**Mehr Fahrradwege für Berliner Innenstadt**

*Berlin, 8. März 2025*

Die Berliner Senatsverwaltung hat einen neuen Plan für den Ausbau des Fahrradnetzes in der Hauptstadt vorgestellt. Bis 2028 sollen insgesamt 120 Kilometer neue Fahrradwege entstehen, vor allem in den Bezirken Mitte, Kreuzberg und Prenzlauer Berg.

"Wir wollen, dass Berlin eine echte Fahrradstadt wird", sagte Verkehrssenatorin Müller bei der Pressekonferenz am Dienstag. "Jeder Berliner soll sicher und bequem mit dem Fahrrad zur Arbeit fahren können."

Der Plan sieht vor, dass die neuen Wege mindestens 2,50 Meter breit sein müssen und von der Autostraße durch Poller getrennt werden. Außerdem werden an wichtigen Kreuzungen neue Ampeln speziell für Radfahrer installiert.

Nicht alle sind begeistert. Der ADAC kritisierte den Plan: "Mehr Fahrradwege bedeuten weniger Platz für Autos. Das führt zu noch mehr Staus in der Innenstadt." Umweltverbände hingegen begrüßten die Initiative als "längst überfällig".

---

5. 누가 이 계획을 비판했고 그 이유는 무엇인가요?
6. 환경 단체는 어떻게 반응했나요?

2. Mitte, Kreuzberg, Prenzlauer Berg.
3. 최소 2.50미터 너비.
4. 볼라드(Poller)로 자동차 도로와 분리됩니다.
5. ADAC(독일 자동차 클럽)이 비판했습니다. 자전거 도로가 늘어나면 자동차를 위한 공간이 줄어들어 교통 체증이 더 심해지기 때문입니다.

</details>

### 참/거짓 문제

각 문장이 *richtig* (참)인지 *falsch* (거짓)인지 판단하세요:

1. 베를린은 2025년까지 120km의 새로운 Fahrradwege를 건설할 것이다.

5. 새 자전거 도로는 자동차 도로와 분리될 것이다.

<details>
<summary>정답</summary>

1. **Falsch** — 2025년이 아니라 2028년까지입니다.
2. **Richtig** — "bei der Pressekonferenz am Dienstag."
3. **Richtig** — "an wichtigen Kreuzungen neue Ampeln speziell für Radfahrer."
4. **Falsch** — ADAC은 이 계획을 비판했습니다.
5. **Richtig** — 볼라드(Poller)로 분리될 것입니다.

</details>

## 9. 연습 텍스트 3: 문학 작품 발췌

다음 발췌문을 읽고 질문에 답하세요.

---

Es war ein kalter Novemberabend, und der Wind trieb die letzten Blätter über die leeren Straßen. Thomas saß allein in seinem Lieblingscafé am Fenster und beobachtete die Passanten, die sich mit hochgezogenen Kragen gegen die Kälte schützten.

Er wartete schon seit einer Stunde. Auf dem Tisch vor ihm stand eine Tasse Kaffee, die längst kalt geworden war. Neben der Tasse lag ein kleines Päckchen, sorgfältig in blaues Papier eingewickelt. Von Zeit zu Zeit schaute er auf die Uhr und dann zur Tür.

Um halb acht stand er schließlich auf, legte Geld auf den Tisch und steckte das Päckchen in seine Manteltasche. Er trat hinaus in die kalte Nacht und ging langsam nach Hause. Das Päckchen blieb ungeöffnet.

---

### 어휘 목록

| 독일어 | 영어 |
|--------|------|
| treiben | 몰다, (바람이) 날리다 |
| das Blatt (복수: Blätter) | 잎/나뭇잎 |
| beobachten | 관찰하다, 지켜보다 |
| der Passant | 행인 |
| hochgezogen | 올려 세운 |
| der Kragen | 옷깃 |
| sich schützen gegen | ~로부터 자신을 보호하다 |
| das Päckchen | 작은 꾸러미 |
| sorgfältig | 정성스럽게, 주의 깊게 |
| eingewickelt | 포장된, 싸인 |
| von Zeit zu Zeit | 때때로 |
| schließlich | 마침내, 결국 |
| die Manteltasche | 코트 주머니 |
| hinaustreten | 밖으로 나가다 |
| ungeöffnet | 개봉되지 않은 |

### 독해 질문

1. 어떤 계절이며 날씨는 어떤가요?
2. Thomas는 어디에 앉아 있고 무엇을 하고 있나요?
3. 그는 얼마나 오래 기다렸나요?
4. 커피 옆 테이블 위에 무엇이 있나요?
5. Thomas는 7시 30분에 무엇을 하나요?
6. 꾸러미는 어떻게 되었나요?

<details>
<summary>정답</summary>

1. 추운 11월 저녁이며, 바람이 빈 거리 위로 나뭇잎을 날리고 있습니다.
2. 그는 단골 카페의 창가에 혼자 앉아 행인들을 관찰하고 있습니다.
3. 이미 한 시간째 기다리고 있습니다(seit einer Stunde).
4. 파란 종이로 정성스럽게 포장된 작은 꾸러미가 있습니다.
5. 자리에서 일어나 테이블에 돈을 놓고, 꾸러미를 코트 주머니에 넣은 후 집으로 갑니다.
6. 꾸러미는 개봉되지 않은 채 남아 있습니다(ungeöffnet).

</details>

### 문학 분석

이 발췌문은 독일어 산문에서 흔히 사용되는 여러 문학 기법을 사용합니다:

- **날씨를 활용한 장면 설정**: 추운 11월 저녁이 분위기(외로움, 실망)를 반영합니다
- **과거 시제(Präteritum)**: 서술 전체에 사용됩니다 (*saß*, *wartete*, *stand auf*)
- **과거완료(Plusquamperfekt)**: *kalt geworden war* (차가워져 있었다) — 서술 시점 이전에 완료된 행위
- **의미심장한 디테일**: 식은 커피와 개봉되지 않은 꾸러미는 기다림과 이루어지지 않은 기대의 이야기를 암시합니다

## 10. 학술 독해 전략

### 논지와 논거 파악하기

독일어 학술 텍스트에서 논지(*These*)는 보통 서론에서 제시되고 결론에서 다시 언급됩니다. 논거(*Argumente*)는 본론 단락에 나타납니다.

논지 신호를 찾아보세요:
- *In diesem Aufsatz wird argumentiert, dass...* (이 논문에서는 ~을 주장합니다)
- *Die zentrale These lautet...* (핵심 논지는 ~입니다)
- *Ziel dieser Arbeit ist es zu zeigen, dass...* (이 연구의 목표는 ~을 보여주는 것입니다)

### 사실과 의견 구별하기

| 사실 (Tatsache) | 의견 (Meinung) |
|-----------------|----------------|
| *Laut Statistik...* (통계에 따르면...) | *Meiner Meinung nach...* (내 의견으로는...) |
| *Studien zeigen, dass...* (연구에 따르면...) | *Ich bin der Ansicht, dass...* (나는 ~라고 생각합니다) |
| *Es ist bekannt, dass...* (~은 알려져 있다) | *Man könnte argumentieren...* (~라고 주장할 수 있다) |
| *Im Jahr 2023 betrug...* (2023년에 ~이었다) | *Es scheint, dass...* (~인 것 같다) |
| *Die Daten belegen...* (데이터가 증명한다) | *Vermutlich...* (아마도...) |

### 논증 구조 파악을 위한 독해

```
+----------------------------------------------------------+
|  ARGUMENT MAPPING                                         |
+----------------------------------------------------------+
|                                                          |
|  1. THESE (Thesis):                                      |
|     What is the author claiming?                         |
|                                                          |
|  2. ARGUMENTE (Arguments):                               |
|     What reasons support the claim?                      |
|                                                          |
|  3. BELEGE (Evidence):                                   |
|     What data, examples, or quotes are used?             |
|                                                          |
|  4. GEGENARGUMENTE (Counterarguments):                   |
|     Does the author address opposing views?              |
|                                                          |
|  5. SCHLUSSFOLGERUNG (Conclusion):                       |
|     What does the author conclude?                       |
+----------------------------------------------------------+
```

## 11. 응집 장치

접속어(*Konnektoren*)를 이해하는 것은 독일어 텍스트의 흐름을 따라가는 데 필수적입니다. 이 단어들은 아이디어 간의 관계를 나타냅니다.

### 원인과 결과 (Ursache und Wirkung)

| 접속어 | 영어 | 예시 |
|--------|------|------|
| **deshalb** | therefore | *Es regnete, deshalb blieben wir zu Hause.* |
| **deswegen** | for that reason | *Sie war müde, deswegen ging sie früh ins Bett.* |
| **daher** | hence, therefore | *Die Preise steigen, daher müssen wir sparen.* |
| **weil** | because (종속절) | *Wir blieben zu Hause, weil es regnete.* |
| **da** | since, as (종속절) | *Da es spät war, gingen wir nach Hause.* |

### 대조 (Gegensatz)

| 접속어 | 영어 | 예시 |
|--------|------|------|
| **trotzdem** | nevertheless | *Es regnete, trotzdem gingen wir spazieren.* |
| **jedoch** | however | *Das Ergebnis war jedoch anders als erwartet.* |
| **allerdings** | however, though | *Die Idee ist gut, allerdings teuer.* |
| **obwohl** | although (종속절) | *Obwohl es regnete, gingen wir spazieren.* |
| **während** | while, whereas | *Während er liest, hört sie Musik.* |

### 추가 (Ergänzung)

| 접속어 | 영어 | 예시 |
|--------|------|------|
| **außerdem** | furthermore, besides | *Er spricht Deutsch, außerdem lernt er Spanisch.* |
| **darüber hinaus** | beyond that | *Darüber hinaus gibt es noch andere Vorteile.* |
| **zusätzlich** | additionally | *Zusätzlich wird ein neues Programm eingeführt.* |
| **sowohl...als auch** | both...and | *Sowohl Kinder als auch Erwachsene sind willkommen.* |

### 순서 (Reihenfolge)

| 접속어 | 영어 | 예시 |
|--------|------|------|
| **zunächst** | first of all | *Zunächst muss man die Grundlagen verstehen.* |
| **dann / danach** | then / after that | *Dann kam der schwierigste Teil.* |
| **schließlich** | finally | *Schließlich fanden sie eine Lösung.* |
| **zuletzt** | lastly | *Zuletzt möchte ich zusammenfassen.* |

> **어순 참고**: *deshalb*, *trotzdem*, *außerdem*과 같은 접속어는 **부사**입니다 — 위치 1에 오며 동사가 위치 2에 옵니다 (도치 어순). *weil*, *obwohl*, *da*와 같은 단어는 **종속 접속사**입니다 — 동사가 절의 **끝**으로 이동합니다.

## 12. 연습 문제

### 연습 1: 합성어 분해

각 합성어를 구성 요소로 분해하고 영어 의미를 제시하세요.

1. Reisebüro
2. Buchhandlung
3. Frühstückstisch
4. Einwanderungspolitik
5. Krankenversicherung

<details>
<summary>정답</summary>

1. **Reise** (여행) + **Büro** (사무실) = 여행사(travel agency)
2. **Buch** (책) + **Handlung** (사업/거래) = 서점(bookstore)
3. **Frühstück** (아침 식사) + **s** + **Tisch** (탁자) = 아침 식탁(breakfast table)
4. **Einwanderung** (이민) + **s** + **Politik** (정책) = 이민 정책(immigration policy)
5. **Kranken** (환자/병든) + **Versicherung** (보험) = 건강 보험(health insurance)

</details>

### 연습 2: 문맥 단서 연습

문맥 단서를 활용하여 굵은 글씨 단어의 의미를 추측하세요.

1. "Nach dem langen Winter waren alle **erleichtert**, als endlich die Sonne wieder schien."
2. "Die Stadt hat ein neues Gesetz **verabschiedet**, das ab nächstem Monat gilt."
3. "Der **Strom** fiel plötzlich aus, und das ganze Haus war dunkel."

<details>
<summary>정답</summary>

1. **erleichtert** = 안도한(relieved) (단서: 긴 겨울이 끝나고 마침내 해가 다시 빛났을 때 모두가 이렇게 느꼈습니다)
2. **verabschiedet** = 통과시킨/제정한(passed/enacted) (단서: 법률이고 다음 달부터 시행됩니다 — 입법 맥락)
3. **Strom** = 전기/전력(electricity/power) (단서: 갑자기 나갔고 집 전체가 어두워졌습니다)

</details>

### 연습 3: 텍스트 유형 식별

각 발췌문이 어떤 유형의 텍스트에서 나온 것인지 답하세요.

1. "Sehr geehrte Frau Schneider, hiermit bewerbe ich mich um die ausgeschriebene Stelle als Projektmanager..."
2. "SONDERANGEBOT! Nur diese Woche: Alle Winterjacken 50% reduziert!"
3. "Laut einer aktuellen Studie des Deutschen Instituts für Wirtschaftsforschung stieg das BIP um 1,3%..."
4. "Die alte Frau saß am Fenster und blickte hinaus in den Garten, wo die Rosen in voller Blüte standen..."
5. "Schritt 1: Öffnen Sie die Verpackung. Schritt 2: Verbinden Sie das Kabel mit dem Gerät."

<details>
<summary>정답</summary>

1. **격식 편지** (Bewerbung — 입사 지원서)
2. **광고** (Werbung — 특별 할인)
3. **정보/뉴스 텍스트** (Sachtext — 통계가 포함된 연구 인용)
4. **문학 텍스트** (Literarischer Text — 서술적, 묘사적)
5. **실용 텍스트** (Gebrauchstext — 지시/설명서)

</details>

### 연습 4: 응집 장치

적절한 접속어를 빈칸에 채우세요: *deshalb, trotzdem, außerdem, allerdings, weil*

1. Ich kann heute nicht kommen, ________ ich krank bin.
2. Das Wetter war schlecht, ________ gingen wir wandern.
3. Die Mieten in München sind sehr hoch, ________ ziehen viele Leute ins Umland.
4. Der Kurs ist sehr interessant, ________ auch ziemlich anspruchsvoll.
5. Wir brauchen mehr Personal, ________ sollten wir die Öffnungszeiten verlängern.

<details>
<summary>정답</summary>

1. **weil** — Ich kann heute nicht kommen, weil ich krank bin. (왜냐하면)
2. **trotzdem** — Das Wetter war schlecht, trotzdem gingen wir wandern. (그럼에도 불구하고)
3. **deshalb** — Die Mieten in München sind sehr hoch, deshalb ziehen viele Leute ins Umland. (그래서)
4. **allerdings** — Der Kurs ist sehr interessant, allerdings auch ziemlich anspruchsvoll. (하지만)
5. **außerdem** — Wir brauchen mehr Personal, außerdem sollten wir die Öffnungszeiten verlängern. (게다가)

</details>

### 연습 5: 독해력 — 참 또는 거짓

이 수업에서 배운 내용을 바탕으로 각 문장이 *richtig*인지 *falsch*인지 판단하세요:

1. 독일어 합성어에서는 첫 번째 단어가 성(性)을 결정한다.
2. 훑어보기(Skimming)란 완전한 이해를 위해 모든 단어를 주의 깊게 읽는 것이다.
3. *obwohl*은 동사를 절 끝으로 보낸다.
4. 격식체 독일어 편지는 *Sie*를, 비격식체 편지는 *du*를 사용한다.
5. 독일어 뉴스 기사는 서술에 일반적으로 Präteritum을 사용한다.

<details>
<summary>정답</summary>

1. **Falsch** — 마지막 단어(Grundwort)가 성을 결정합니다.
2. **Falsch** — 훑어보기(Überfliegen)는 전체적인 내용을 파악하기 위해 빠르게 읽는 것입니다.
3. **Richtig** — *obwohl*은 종속 접속사이며, 동사가 절 끝으로 이동합니다.
4. **Richtig** — 격식체 = *Sie*, 비격식체 = *du/ihr*.
5. **Richtig** — 저널리즘 독일어는 서술 과거에 Präteritum을 사용합니다.

</details>
