# 12. 과거 시제: Perfekt와 Präteritum (Perfekt und Präteritum)

## 학습 목표

- Perfekt(회화체 과거)와 Präteritum(서술적 과거)의 차이를 이해할 수 있다
- 규칙 동사, 강변화 동사, 분리 동사, 비분리 동사의 Partizip II를 형성할 수 있다
- Perfekt 구문에서 올바른 조동사(haben vs. sein)를 선택할 수 있다
- 일반적인 동사들의 Präteritum(약변화, 강변화, 화법 조동사)을 활용할 수 있다
- hatte/war + Partizip II를 사용하여 Plusquamperfekt(과거완료)를 형성할 수 있다
- 문맥(구어 vs. 문어)에 따라 적절한 과거 시제를 선택할 수 있다
- 가장 일반적인 20개 이상의 불규칙 동사의 Partizip II 형태를 암기할 수 있다

## 1. 독일어 과거 시제 개요

독일어에는 세 가지 과거 시제가 있습니다. 각각을 언제 사용하는지 이해하는 것이 필수적입니다:

```
+------------------------------------------------------------------+
|                    GERMAN PAST TENSES                             |
+------------------------------------------------------------------+
|                                                                    |
|  PERFEKT (Present Perfect)                                        |
|  haben/sein (present) + Partizip II                               |
|  "Ich habe gegessen." (I have eaten / I ate.)                     |
|  USE: Spoken German, conversation, informal writing               |
|                                                                    |
|  PRÄTERITUM (Simple Past / Imperfect)                             |
|  One-word past form                                               |
|  "Ich aß." (I ate.)                                               |
|  USE: Written narratives, formal writing, sein/haben/modals       |
|                                                                    |
|  PLUSQUAMPERFEKT (Past Perfect / Pluperfect)                      |
|  hatte/war + Partizip II                                          |
|  "Ich hatte gegessen." (I had eaten.)                             |
|  USE: Action completed before another past action                 |
|                                                                    |
+------------------------------------------------------------------+
```

> **핵심 통찰**: 영어에서는 "I ate"와 "I have eaten"의 의미가 다르지만, 독일어의 Perfekt와 Präteritum은 일반적으로 **같은 시간대**를 표현합니다. 이 둘 사이의 선택은 주로 의미가 아닌 **사용역**(구어 vs. 문어)에 달려 있습니다.

## 2. Perfekt: 구조와 형성

Perfekt는 **구어체 독일어에서 가장 일반적인 과거 시제**입니다. 두 부분으로 구성됩니다:

```
Subject + haben/sein (conjugated, position 2) + ... + Partizip II (end of clause)
```

**예문**:

- Ich **habe** gestern viel **gearbeitet**. (나는 어제 많이 일했다.)
- Sie **ist** nach Berlin **geflogen**. (그녀는 베를린으로 날아갔다.)
- Wir **haben** den Film **gesehen**. (우리는 그 영화를 봤다.)
- Er **ist** um 7 Uhr **aufgestanden**. (그는 7시에 일어났다.)

> Partizip II는 항상 주절의 **끝**에 위치합니다. 이것은 독일어 문장 구조의 가장 특징적인 특성 중 하나입니다.

## 3. Partizip II: 규칙(약변화) 동사

규칙(약변화) 동사는 다음 패턴으로 Partizip II를 형성합니다:

```
ge- + verb stem + -t
```

| 원형(Infinitive) | 어간(Stem) | Partizip II | 의미 |
|-----------|------|-------------|---------|
| machen | mach- | **ge**mach**t** | 하다/만들다 |
| spielen | spiel- | **ge**spiel**t** | 놀다/연주하다 |
| lernen | lern- | **ge**lern**t** | 배우다 |
| kaufen | kauf- | **ge**kauf**t** | 사다 |
| kochen | koch- | **ge**koch**t** | 요리하다 |
| wohnen | wohn- | **ge**wohn**t** | 살다/거주하다 |
| hören | hör- | **ge**hör**t** | 듣다 |
| sagen | sag- | **ge**sag**t** | 말하다 |

### -t, -d, -chn, -ffn, -gn으로 끝나는 어간

발음을 위해 **-t** 대신 **-et**를 추가합니다:

| 원형(Infinitive) | Partizip II |
|-----------|-------------|
| arbeiten | **ge**arbeit**et** |
| warten | **ge**wart**et** |
| reden | **ge**red**et** |
| öffnen | **ge**öffn**et** |
| regnen | **ge**regn**et** |

## 4. Partizip II: 강변화(불규칙) 동사

강변화 동사는 다음 패턴으로 Partizip II를 형성합니다:

```
ge- + changed stem + -en
```

어간의 모음이 보통 변합니다 (이를 **Ablaut**라고 합니다):

| 원형(Infinitive) | Partizip II | 모음 변화 | 의미 |
|-----------|-------------|--------------|---------|
| gehen | **ge**gang**en** | e → a | 가다 |
| sprechen | **ge**sproch**en** | e → o | 말하다 |
| schreiben | **ge**schrieb**en** | ei → ie | 쓰다 |
| trinken | **ge**trunk**en** | i → u | 마시다 |
| fahren | **ge**fahr**en** | (변화 없음) | 운전하다/가다 |
| lesen | **ge**les**en** | (변화 없음) | 읽다 |
| singen | **ge**sung**en** | i → u | 노래하다 |
| finden | **ge**fund**en** | i → u | 찾다 |
| helfen | **ge**holf**en** | e → o | 돕다 |
| nehmen | **ge**nomm**en** | e → o | 가져가다 |

> **지름길은 없습니다**: 강변화 동사 형태는 암기해야 합니다. 포괄적인 표는 섹션 11을 참조하세요.

## 5. Partizip II: 특수한 경우

### -ieren으로 끝나는 동사 (ge- 없음)

프랑스어/라틴어에서 차용되어 **-ieren**으로 끝나는 동사는 **ge-** 접두사를 붙이지 않습니다:

| 원형(Infinitive) | Partizip II | 의미 |
|-----------|-------------|---------|
| studieren | studier**t** | 공부하다 |
| telefonieren | telefonier**t** | 전화하다 |
| reparieren | reparier**t** | 수리하다 |
| funktionieren | funktionier**t** | 기능하다 |
| fotografieren | fotografier**t** | 사진 찍다 |

### 비분리 접두사 동사 (ge- 없음)

비분리 접두사(be-, emp-, ent-, er-, ge-, miss-, ver-, zer-)가 있는 동사는 **ge-**를 붙이지 않습니다:

| 원형(Infinitive) | Partizip II | 의미 |
|-----------|-------------|---------|
| verstehen | verstand**en** | 이해하다 |
| bekommen | bekomm**en** | 받다 |
| erzählen | erzähl**t** | 이야기하다 |
| beginnen | begonn**en** | 시작하다 |
| empfehlen | empfohl**en** | 추천하다 |
| entdecken | entdeck**t** | 발견하다 |
| vergessen | vergess**en** | 잊다 |
| zerstören | zerstör**t** | 파괴하다 |
| gehören | gehör**t** | ~에 속하다 |
| misslingen | misslungen | 실패하다 |

### 분리 접두사 동사 (접두사와 어간 사이에 ge-)

분리 접두사가 있는 동사는 접두사와 어간 사이에 **ge-**를 삽입합니다:

| 원형(Infinitive) | Partizip II | 의미 |
|-----------|-------------|---------|
| aufmachen | auf**ge**mach**t** | 열다 |
| anfangen | an**ge**fang**en** | 시작하다 |
| aufstehen | auf**ge**stand**en** | 일어서다 |
| einkaufen | ein**ge**kauf**t** | 쇼핑하다 |
| zurückkommen | zurück**ge**komm**en** | 돌아오다 |
| mitnehmen | mit**ge**nomm**en** | 가져가다 |
| anrufen | an**ge**ruf**en** | 전화하다 |
| fernsehen | fern**ge**seh**en** | TV를 보다 |

### 요약: ge-는 어디에 위치하는가?

```
Regular verb:       ge + stem + t       →  gemacht
Strong verb:        ge + stem + en      →  gegangen
-ieren verb:        stem + t            →  studiert       (NO ge-)
Inseparable prefix: prefix + stem + t/en →  verstanden     (NO ge-)
Separable prefix:   prefix + ge + stem  →  aufgemacht     (ge- in middle)
```

## 6. Haben vs. Sein: 조동사 선택

대부분의 동사는 **haben**을 사용합니다. 더 적은 수의 동사 그룹이 **sein**을 사용합니다. 규칙은 다음과 같습니다:

### SEIN과 함께 사용하는 경우:

1. **A에서 B로의 이동 동사**: gehen, fahren, fliegen, kommen, laufen, reisen, schwimmen, fallen
2. **상태 변화 동사**: werden (되다), sterben (죽다), einschlafen (잠들다), aufwachen (깨어나다), wachsen (자라다), geboren sein (태어나다)
3. **sein**과 **bleiben** (있다/머무르다): Ich **bin** dort gewesen. / Er **ist** geblieben.
4. **passieren**과 **geschehen** (일어나다): Was **ist** passiert?

### HABEN과 함께 사용하는 경우:

1. **모든 타동사** (직접 목적어를 취하는 동사): Ich habe das Buch gelesen.
2. **재귀 동사**: Er hat sich gewaschen.
3. **화법 조동사**: Sie hat schwimmen können.
4. **비인칭 동사**: Es hat geregnet. (비가 왔다.)
5. **sein 목록에 없는 나머지 모든 동사**.

### 빠른 참조표

| 조동사 | 동사 유형 | 예문 |
|-----------|----------|---------|
| **sein** | 이동 A→B | Er **ist** nach Hause **gegangen**. |
| **sein** | 상태 변화 | Sie **ist** Ärztin **geworden**. |
| **sein** | sein, bleiben | Ich **bin** in Berlin **gewesen**. |
| **haben** | 타동사 | Wir **haben** Pizza **gegessen**. |
| **haben** | 재귀 동사 | Ich **habe** mich **gefreut**. |
| **haben** | 대부분의 동사 | Er **hat** gut **geschlafen**. |

> **까다로운 경우**: 일부 이동 동사는 활동 자체에 초점이 맞춰져 있을 때 **haben**을 사용합니다:
> - Ich **bin** geschwommen. (나는 (어딘가로) 수영했다.)
> - Ich **habe** eine Stunde geschwommen. (나는 한 시간 동안 수영했다 [활동 초점].)
>
> 실제로 이 구별은 점점 덜 엄격해지고 있으며 지역적 차이가 존재합니다.

## 7. Präteritum: 형성과 용법

Präteritum은 **단순과거** 시제입니다. 회화에서는 (sein, haben, 화법 조동사를 제외하고) 덜 일반적이지만, **서술적 문어체**에서는 표준 과거 시제입니다.

### 약변화(규칙) 동사

동사 어간에 다음 어미를 추가합니다:

| 인칭 | 어미 | 예: machen |
|--------|--------|----------------|
| ich | -**te** | ich mach**te** |
| du | -**test** | du mach**test** |
| er/sie/es | -**te** | er mach**te** |
| wir | -**ten** | wir mach**ten** |
| ihr | -**tet** | ihr mach**tet** |
| sie/Sie | -**ten** | sie mach**ten** |

> -t, -d, -chn, -ffn으로 끝나는 어간은 추가 **e**를 넣습니다: arbeitete, redete, öffnete.

### 강변화(불규칙) 동사

강변화 동사는 어간 모음이 변하고 **다른 어미**(no -te)를 사용합니다:

| 인칭 | 어미 | 예: gehen → ging | 예: sprechen → sprach |
|--------|--------|----------------------|---------------------------|
| ich | — (무어미) | ich **ging** | ich **sprach** |
| du | -**st** | du **gingst** | du **sprachst** |
| er/sie/es | — (무어미) | er **ging** | er **sprach** |
| wir | -**en** | wir **gingen** | wir **sprachen** |
| ihr | -**t** | ihr **gingt** | ihr **spracht** |
| sie/Sie | -**en** | sie **gingen** | sie **sprachen** |

> 참고: 강변화 Präteritum에서 1인칭과 3인칭 단수는 **어미가 없습니다** (무어미).

### Präteritum에서의 Sein과 Haben

이 동사들은 구어체에서도 Präteritum으로 사용됩니다:

| 인칭 | sein (이었다) | haben (가졌다) |
|--------|-----------|-------------|
| ich | **war** | **hatte** |
| du | **warst** | **hattest** |
| er/sie/es | **war** | **hatte** |
| wir | **waren** | **hatten** |
| ihr | **wart** | **hattet** |
| sie/Sie | **waren** | **hatten** |

## 8. 화법 조동사의 Präteritum

화법 조동사는 구어체에서도 거의 항상 **Präteritum**으로 사용됩니다 (Perfekt가 아님). **움라우트가 사라진다**는 점에 주의하세요:

| 화법 조동사 | 현재형 (ich) | Präteritum (ich) | 과거 의미 |
|-------|--------------|-------------------|-----------------|
| können | kann | **konnte** | ~할 수 있었다 |
| müssen | muss | **musste** | ~해야 했다 |
| dürfen | darf | **durfte** | ~해도 됐다 |
| sollen | soll | **sollte** | ~해야 했다 |
| wollen | will | **wollte** | ~하고 싶었다 |
| mögen | mag | **mochte** | 좋아했다 |

### 전체 활용 패턴 (예: können)

| 인칭 | Präteritum |
|--------|-----------|
| ich | konnte |
| du | konntest |
| er/sie/es | konnte |
| wir | konnten |
| ihr | konntet |
| sie/Sie | konnten |

> **팁**: 모든 화법 조동사의 Präteritum 형태는 약변화 동사 패턴(-te 어미)을 따르며 움라우트를 잃습니다. 1인칭과 3인칭 단수는 동일합니다.

**예문**:

- Ich **konnte** gestern nicht kommen. (나는 어제 올 수 없었다.)
- Sie **musste** früh aufstehen. (그녀는 일찍 일어나야 했다.)
- Wir **durften** nicht ins Kino gehen. (우리는 영화관에 가면 안 됐다.)
- Er **wollte** Arzt werden. (그는 의사가 되고 싶었다.)
- Du **solltest** das nicht sagen. (너는 그 말을 하지 말았어야 했다.)

## 9. 어떤 과거 시제를 사용할 것인가

### Perfekt vs. Präteritum: 실용 가이드

| 문맥 | 선호 시제 | 예문 |
|---------|----------------|---------|
| **일상 대화** | Perfekt | Was hast du gestern gemacht? |
| **비격식 문어체** (이메일, 메시지) | Perfekt | Ich habe dir geschrieben. |
| **sein, haben** (구어에서도) | Präteritum | Ich **war** müde. / Ich **hatte** Hunger. |
| **화법 조동사** (구어에서도) | Präteritum | Ich **konnte** nicht schlafen. |
| **소설, 동화, 뉴스** | Präteritum | Der König **lebte** in einem Schloss. |
| **학술/격식 문어체** | Präteritum | Die Studie **zeigte**, dass... |
| **순서대로 보고** (문어체) | Präteritum | Er **kam**, **sah** und **siegte**. |

### 지역적 차이

- **북부 독일**: Präteritum이 구어에서도 더 많이 사용됨
- **남부 독일, 오스트리아, 스위스**: 구어에서 Perfekt가 강하게 선호됨; Präteritum은 격식적이거나 문학적으로 들릴 수 있음

> **학습자를 위한 실용 규칙**: 대화에서는 모든 것에 **Perfekt**를 사용하되, **sein**, **haben**, **화법 조동사**에 대해서만 **Präteritum**으로 전환하세요. 이렇게 하면 어떤 지역에서든 자연스럽게 들립니다.

## 10. Plusquamperfekt: 과거완료

Plusquamperfekt는 다른 과거 행위 **이전에** 완료된 행위를 표현합니다 — 영어의 "had done"에 해당합니다.

```
hatte/war (Präteritum) + Partizip II
```

### 형성

| 조동사 | Plusquamperfekt |
|-----------|----------------|
| haben → hatte | Ich **hatte** das Buch **gelesen**. (나는 그 책을 읽었었다.) |
| sein → war | Er **war** schon **gegangen**. (그는 이미 갔었다.) |

### 용법: 과거 사건의 순서 정하기

Plusquamperfekt는 **먼저** 일어난 일(배경)을 보여주고, Perfekt나 Präteritum은 **그 다음에** 일어난 일을 보여줍니다:

- **Nachdem** ich gefrühstückt **hatte**, ging ich zur Arbeit.
  (아침을 먹은 **후**, 나는 일하러 갔다.)

- Er **war** schon eingeschlafen, als ich nach Hause **kam**.
  (내가 집에 **왔을 때**, 그는 이미 잠들어 **있었다**.)

- Sie **hatte** den Brief schon geschrieben, bevor er **anrief**.
  (그가 전화하기 **전에**, 그녀는 이미 편지를 쓴 **상태였다**.)

> **신호 단어**: nachdem (~ 후에), bevor (~전에), als (~할 때), schon (이미), bereits (이미)

## 11. 일반적인 불규칙 Partizip II 형태

이 표는 가장 중요한 강변화 및 혼합변화 동사와 그 Partizip II 형태를 나열합니다. 체계적으로 암기하세요.

| 원형(Infinitive) | Präteritum | Partizip II | 조동사 | 의미 |
|-----------|------------|-------------|-----------|---------|
| beginnen | begann | begonnen | haben | 시작하다 |

## 12. 연습 문제

### 연습 1: Partizip II 형성

다음 동사의 Partizip II를 쓰세요.

1. spielen (놀다) → ________
2. trinken (마시다) → ________
3. verstehen (이해하다) → ________
4. anfangen (시작하다) → ________
5. studieren (공부하다) → ________
6. aufräumen (정리하다) → ________
7. bekommen (받다) → ________
8. lesen (읽다) → ________

<details>
<summary>정답</summary>

1. **gespielt** (규칙: ge- + spiel + -t)
2. **getrunken** (강변화: ge- + trunk + -en)
3. **verstanden** (비분리 접두사: ge- 없음)
4. **angefangen** (분리 접두사: an + ge + fang + -en)
5. **studiert** (-ieren 동사: ge- 없음)
6. **aufgeräumt** (분리 접두사: auf + ge + räum + -t)
7. **bekommen** (비분리 접두사: ge- 없음)
8. **gelesen** (강변화: ge- + les + -en)

</details>

### 연습 2: Haben인가 Sein인가?

올바른 조동사를 선택하세요.

1. Wir ________ (haben/sein) nach Paris geflogen.
2. Er ________ (haben/sein) den ganzen Tag gearbeitet.
3. ________ (Haben/Sein) du schon gegessen?
4. Sie ________ (haben/sein) gestern spät gekommen.
5. Ich ________ (haben/sein) mich sehr gefreut.
6. Die Kinder ________ (haben/sein) schnell eingeschlafen.

<details>
<summary>정답</summary>

1. **sind** (fliegen = 이동 A→B, sein 사용)
2. **hat** (arbeiten = 이동/상태 변화 없음, haben 사용)
3. **Hast** (essen = 타동사, haben 사용)
4. **ist** (kommen = 이동, sein 사용)
5. **habe** (sich freuen = 재귀 동사, haben 사용)
6. **sind** (einschlafen = 상태 변화, sein 사용)

</details>

### 연습 3: Präteritum 활용

동사를 Präteritum으로 변환하세요.

1. Ich ________ (sein) gestern krank.
2. Wir ________ (haben) keine Zeit.
3. Er ________ (können) nicht schwimmen.
4. Sie (pl.) ________ (müssen) lange warten.
5. Das Kind ________ (wollen) ein Eis.
6. Ich ________ (gehen) nach Hause.
7. Er ________ (sprechen) sehr leise.

<details>
<summary>정답</summary>

1. **war** (sein → war)
2. **hatten** (haben → hatten)
3. **konnte** (können → konnte)
4. **mussten** (müssen → mussten)
5. **wollte** (wollen → wollte)
6. **ging** (gehen → ging, 강변화 동사)
7. **sprach** (sprechen → sprach, 강변화 동사)

</details>

### 연습 4: Perfekt 문장

문장을 Perfekt로 다시 쓰세요.

1. Ich trinke Kaffee. → ________
2. Sie fährt nach München. → ________
3. Wir kaufen ein neues Auto. → ________
4. Er steht um 6 Uhr auf. → ________
5. Die Kinder spielen im Garten. → ________

<details>
<summary>정답</summary>

1. Ich **habe** Kaffee **getrunken**. (haben + 강변화 Partizip II)
2. Sie **ist** nach München **gefahren**. (sein + 이동)
3. Wir **haben** ein neues Auto **gekauft**. (haben + 규칙 Partizip II)
4. Er **ist** um 6 Uhr **aufgestanden**. (sein + 분리 접두사 + 상태 변화)
5. Die Kinder **haben** im Garten **gespielt**. (haben + 규칙 Partizip II)

</details>

### 연습 5: Plusquamperfekt

Plusquamperfekt를 사용하여 문장을 완성하세요.

1. Nachdem ich ________ (essen), ging ich spazieren.
2. Er war müde, weil er schlecht ________ (schlafen).
3. Bevor sie anrief, ________ ich schon ________ (gehen).
4. Wir kamen zu spät, weil der Zug schon ________ (abfahren).

<details>
<summary>정답</summary>

1. Nachdem ich **gegessen hatte**, ging ich spazieren. (hatte + gegessen)
2. Er war müde, weil er schlecht **geschlafen hatte**. (hatte + geschlafen)
3. Bevor sie anrief, **war** ich schon **gegangen**. (war + gegangen — 이동)
4. Wir kamen zu spät, weil der Zug schon **abgefahren war**. (war + abgefahren — 이동)

</details>

### 연습 6: 혼합 — 독일어로 번역하기

특별히 시제가 지시되지 않은 한 Perfekt를 사용하세요.

1. I was tired yesterday. (sein은 Präteritum 사용)
2. She has written a letter.
3. We flew to Berlin last week.
4. He couldn't come. (화법 조동사는 Präteritum 사용)
5. After I had eaten, I went to bed.

<details>
<summary>정답</summary>

1. Ich **war** gestern müde. (sein의 Präteritum)
2. Sie **hat** einen Brief **geschrieben**. (haben을 사용한 Perfekt)
3. Wir **sind** letzte Woche nach Berlin **geflogen**. (sein을 사용한 Perfekt — 이동)
4. Er **konnte** nicht kommen. (화법 조동사 können의 Präteritum)
5. Nachdem ich **gegessen hatte**, **bin** ich ins Bett **gegangen**. (Plusquamperfekt + Perfekt)

</details>
