# 15. 완전한 동사 체계 (Das vollstaendige Verbsystem)

## 학습 목표

- 독일어 동사를 약변화(규칙), 강변화(불규칙), 혼합변화로 분류할 수 있다
- 각 동사 유형의 기본형 3단계(원형, Praeteritum, Partizip II)를 식별할 수 있다
- 강변화 동사의 주요 Ablaut 계열(모음 변화 패턴)을 인식할 수 있다
- 6개 시제 전체에 걸쳐 동사를 활용할 수 있다: 현재, Praeteritum, Perfekt, Plusquamperfekt, Futur I, Futur II
- Perfekt와 Plusquamperfekt에서 haben vs. sein 선택 규칙을 적용할 수 있다

- 조동사(sein, haben, werden)와 화법조동사(Modal Verb)를 모든 시제에 걸쳐 활용할 수 있다

## 1. 개요: 독일어 동사 분류

독일어 동사는 두 가지 방식으로 분류할 수 있다: **활용 유형**(과거 시제 형성 방식)과 **기능**(문장에서의 역할).

```
+--------------------------------------------------------------------+
|                    GERMAN VERB SYSTEM                               |
+--------------------------------------------------------------------+
|                                                                    |
|  By conjugation type:                                              |
|  +-----------------+  +-----------------+  +-----------------+     |
|  | WEAK (schwach)  |  | STRONG (stark)  |  | MIXED           |    |
|  | Regular         |  | Irregular       |  | (gemischt)      |    |
|  |                 |  |                 |  |                 |    |
|  | Praet: -te      |  | Praet: vowel   |  | Vowel change    |    |
|  | PP: ge-...-t    |  |   change        |  | + weak endings  |    |
|  |                 |  | PP: ge-...-en   |  | PP: ge-...-t    |    |

```

## 2. 약변화 (규칙) 동사

약변화 동사는 **가장 크고 가장 예측 가능한** 동사 부류이다. 독일어 동사의 대다수(그리고 새로 만들어지는 모든 동사)가 약변화이다.

### 기본형 3단계 패턴

```
Infinitive  -->  Praeteritum  -->  Partizip II
machen      -->  machte        -->  gemacht
spielen     -->  spielte       -->  gespielt
lernen      -->  lernte        -->  gelernt
arbeiten    -->  arbeitete     -->  gearbeitet
```

### 완전 활용: machen (만들다/하다)

| 시제 | ich | du | er/sie/es | wir | ihr | sie/Sie |
|------|-----|-----|-----------|-----|-----|---------|
| 현재 | mache | machst | macht | machen | macht | machen |
| Praeteritum | machte | machtest | machte | machten | machtet | machten |
| Perfekt | habe gemacht | hast gemacht | hat gemacht | haben gemacht | habt gemacht | haben gemacht |
| 과거완료 | hatte gemacht | hattest gemacht | hatte gemacht | hatten gemacht | hattet gemacht | hatten gemacht |
| Futur I | werde machen | wirst machen | wird machen | werden machen | werdet machen | werden machen |
| Futur II | werde gemacht haben | wirst gemacht haben | wird gemacht haben | werden gemacht haben | werdet gemacht haben | werden gemacht haben |

### 약변화 동사의 핵심 특징

1. 어떤 시제에서도 **모음 변화 없음**
2. Praeteritum에서 **-te** 접미사 (어간에 추가)
3. Partizip II에서 **ge-...-t** 패턴
4. 어간이 **-t, -d, -chn, -ffn, -gn**으로 끝나면 어미 앞에 **-e-** 추가:
   - arbeit**e**te (*arbeittte*가 아님)
   - red**e**te
   - oeffn**e**te
   - zeichn**e**te

### 흔한 약변화 동사

| 원형 | Praeteritum | Partizip II | 의미 |
|------|------------|-------------|------|
| machen | machte | gemacht | 만들다, 하다 |
| spielen | spielte | gespielt | 놀다 |
| lernen | lernte | gelernt | 배우다 |
| kaufen | kaufte | gekauft | 사다 |
| arbeiten | arbeitete | gearbeitet | 일하다 |
| hoeren | hoerte | gehoert | 듣다 |
| sagen | sagte | gesagt | 말하다 |
| fragen | fragte | gefragt | 묻다 |
| wohnen | wohnte | gewohnt | 살다 (거주하다) |
| brauchen | brauchte | gebraucht | 필요하다 |

## 3. 강변화 (불규칙) 동사

강변화 동사는 과거 시제에서 **어간 모음**이 변하며(Ablaut), 약변화 동사와 다른 어미를 사용한다.

### 기본형 3단계 패턴

```
Infinitive  -->  Praeteritum  -->  Partizip II
gehen       -->  ging          -->  gegangen
sprechen    -->  sprach        -->  gesprochen
schreiben   -->  schrieb       -->  geschrieben
trinken     -->  trank         -->  getrunken
```

### 강변화 동사의 핵심 특징

1. 시제 간 **어간 모음 변화** (Ablaut)
2. Praeteritum에서 **-te 접미사 없음** (변화된 어간만, ich/er에 어미 없음)
3. Partizip II에서 **ge-...-en** 패턴 (**-t**가 아닌 **-en**으로 끝남)
4. 많은 동사가 du/er 형태에서 **현재 시제 어간 변화**를 가짐

### 현재 시제 어간 변화 (du와 er/sie/es만 해당)

| 변화 | 원형 | du | er/sie/es | 의미 |
|------|------|-----|-----------|------|
| **e -> i** | geben | g**i**bst | g**i**bt | 주다 |
| **e -> i** | sprechen | spr**i**chst | spr**i**cht | 말하다 |
| **e -> i** | helfen | h**i**lfst | h**i**lft | 돕다 |
| **e -> i** | nehmen | n**i**mmst | n**i**mmt | 가져가다 |
| **e -> ie** | lesen | l**ie**st | l**ie**st | 읽다 |
| **e -> ie** | sehen | s**ie**hst | s**ie**ht | 보다 |
| **e -> ie** | empfehlen | empf**ie**hlst | empf**ie**hlt | 추천하다 |
| **a -> ae** | fahren | f**ae**hrst | f**ae**hrt | 운전하다 |
| **a -> ae** | schlafen | schl**ae**fst | schl**ae**ft | 자다 |
| **a -> ae** | tragen | tr**ae**gst | tr**ae**gt | 나르다, 입다 |
| **a -> ae** | fallen | f**ae**llst | f**ae**llt | 떨어지다 |
| **au -> aeu** | laufen | l**aeu**fst | l**aeu**ft | 달리다 |

> **중요**: 이러한 현재 시제 변화는 2인칭 단수와 3인칭 단수(du, er/sie/es)에서만 일어난다. 다른 모든 형태(ich, wir, ihr, sie/Sie)는 원형 어간을 유지한다.

## 4. Ablaut 계열: 모음 변화 패턴

강변화 동사는 **Ablaut 계열**이라 불리는 식별 가능한 모음 변화 패턴을 따른다. 이 패턴을 배우면 낯선 강변화 동사의 형태를 예측하는 데 도움이 된다.

### 주요 Ablaut 계열

| # | 패턴 | 원형 | Praeteritum | Partizip II | 추가 예시 |
|---|------|------|------------|-------------|----------|
| 1 | ei -> ie -> ie | schreiben | schr**ie**b | geschr**ie**ben | bleiben, treiben, steigen, reiten (첫 번째 그룹) |
| 2 | ei -> i -> i | greifen | gr**i**ff | gegr**i**ffen | schneiden, reiten, pfeifen, streiten |
| 3 | e -> a -> o | sprechen | spr**a**ch | gespr**o**chen | helfen, nehmen, treffen, sterben, werfen |
| 4 | i -> a -> u | trinken | tr**a**nk | getr**u**nken | singen, finden, binden, klingen, springen |
| 5 | i -> a -> o | schwimmen | schw**a**mm | geschw**o**mmen | beginnen, gewinnen, spinnen |
| 6 | ie -> o -> o | fliegen | fl**o**g | gefl**o**gen | biegen, ziehen, verlieren, fliessen, schiessen |
| 7 | a -> u -> a | fahren | f**u**hr | gef**a**hren | tragen, graben, schlagen, wachsen |
| 8 | a -> ie -> a | fallen | f**ie**l | gef**a**llen | halten, schlafen, lassen, raten, braten |
| 9 | e -> a -> e | geben | g**a**b | geg**e**ben | lesen, sehen, essen, vergessen, treten |
| 10 | (다양) | gehen | g**i**ng | geg**a**ngen | (불규칙 — 명확한 계열 없음) |

### Ablaut 계열 활용법

새로운 강변화 동사를 만났을 때 계열에 맞춰 보라:

1. **원형 모음**을 확인하라 (e, i, a, ei, ie 등)
2. 알려진 패턴을 따르는지 확인하라
3. Partizip II 모음은 종종 계열에서 예측할 수 있다

> **예시**: 동사 *singen* (노래하다)을 만났다. 원형 모음이 **i**이다. 계열 4 (i -> a -> u)를 보면: sang, gesungen. 맞다!

## 6. 혼합변화 동사

혼합변화 동사는 강변화와 약변화 **양쪽**의 특징을 결합한다: 강변화 동사처럼 **어간 모음이 변하지만**, 약변화 동사의 **어미**를 사용한다 (Praeteritum에서 -te, Partizip II에서 ge-...-t).

### 혼합변화 동사 (전체 목록)

| 원형 | Praeteritum | Partizip II | 의미 |
|------|------------|-------------|------|
| bringen | **brachte** | **gebracht** | 가져오다 |
| denken | **dachte** | **gedacht** | 생각하다 |
| kennen | **kannte** | **gekannt** | 알다 (알고 지내다) |
| nennen | **nannte** | **genannt** | 이름 붙이다, 부르다 |
| rennen | **rannte** | **gerannt** | 달리다 |
| senden | **sandte** | **gesandt** | 보내다* |
| wenden | **wandte** | **gewandt** | 돌리다* |
| wissen | **wusste** | **gewusst** | 알다 (사실을) |

> *senden과 wenden은 규칙적인 약변화 형태(sendete/gesendet, wendete/gewendet)도 있으며, 똑같이 허용되고 점점 더 흔해지고 있다.

### 비교: 약변화 vs. 강변화 vs. 혼합변화

| 특징 | 약변화 | 강변화 | 혼합변화 |
|------|--------|--------|---------|
| 모음 변화? | 없음 | 있음 | 있음 |
| Praeteritum 어미 | -te | 어미 없음 (ich/er) | -te |
| Partizip II 어미 | -t (gemacht) | -en (gegangen) | -t (gedacht) |
| 예시 | machen/machte/gemacht | gehen/ging/gegangen | denken/dachte/gedacht |

### 특수 경우: wissen

**wissen**은 화법조동사와 유사한 독특한 현재 시제 활용을 가진다:

| 인칭 | 현재 |
|------|------|
| ich | **weiss** |
| du | **weisst** |
| er/sie/es | **weiss** |
| wir | **wissen** |
| ihr | **wisst** |
| sie/Sie | **wissen** |

> 참고: ich weiss와 er weiss는 동일한 형태이며(화법조동사와 같음), 단수 어간이 복수/원형 어간과 다르다.

## 7. 6개 시제 총정리

### 시제 개요표

| 시제 | 형성 방식 | 예시 (ich, fahren) | 주요 용법 |
|------|----------|-------------------|----------|
| **현재(Praesens)** | 어간 + 어미 | ich fahre | 현재, 습관, 가까운 미래 |
| **Praeteritum** | 어간 변화 또는 -te | ich fuhr | 문어체 과거 서술; 구어에서 sein/haben/화법조동사 |
| **Perfekt** | haben/sein + Partizip II | ich bin gefahren | 구어 과거 (대화에서 가장 흔함) |
| **Plusquamperfekt** | hatte/war + Partizip II | ich war gefahren | 다른 과거 행위 이전의 행위 ("had done") |
| **Futur I** | werden + 원형 | ich werde fahren | 미래, 추측, 가정 |
| **Futur II** | werden + PP + haben/sein | ich werde gefahren sein | 미래완료, 완료된 행위에 대한 추측 |

### 현재(Praesens)

용도: 현재 행위, 습관적 행위, 일반적 진리, 예정된 미래 사건.

| 약변화: machen | 강변화: fahren |
|---------------|---------------|
| ich mache | ich fahre |
| du machst | du faehrst |
| er macht | er faehrt |
| wir machen | wir fahren |
| ihr macht | ihr fahrt |
| sie machen | sie fahren |

### Praeteritum (단순 과거)

주로 **문어체 서술**과 구어에서 **sein, haben, 화법조동사**에 사용된다.

| 약변화: machen | 강변화: fahren | 혼합변화: denken |
|---------------|---------------|----------------|
| ich machte | ich fuhr | ich dachte |
| du machtest | du fuhrst | du dachtest |
| er machte | er fuhr | er dachte |
| wir machten | wir fuhren | wir dachten |
| ihr machtet | ihr fuhrt | ihr dachtet |
| sie machten | sie fuhren | sie dachten |

> **핵심 차이**: 약변화 동사는 어간에 -te를 추가한다. 강변화 동사는 모음이 변하고 ich/er에 어미가 없다. 혼합변화 동사는 모음이 변하고 -te도 추가한다.

### Perfekt (현재완료)

**구어 독일어에서 가장 흔한 과거 시제**. haben 또는 sein + Partizip II로 형성된다.

- Ich **habe** einen Brief **geschrieben**. (나는 편지를 썼다.)
- Er **ist** nach Berlin **gefahren**. (그는 베를린으로 갔다.)
- Wir **haben** das Problem **verstanden**. (우리는 그 문제를 이해했다.)

### Plusquamperfekt (과거완료)

다른 과거 행위**이전에** 일어난 행위를 묘사한다. hatte/war + Partizip II를 사용한다.

- **Nachdem** er **gegessen hatte**, ging er spazieren.
  (식사 후에 그는 산책하러 갔다.)
- Sie **war** schon **abgefahren**, als ich ankam.
  (내가 도착했을 때 그녀는 이미 출발한 후였다.)

### Futur I (미래)

werden + 원형으로 형성된다. **추측/가정**에도 사용된다.

- Ich **werde** morgen **kommen**. (내일 올 것이다.)
- Es **wird** wohl regnen. (아마 비가 올 것이다. — 추측)
- Er **wird** jetzt schon zu Hause **sein**. (그는 아마 이미 집에 있을 것이다. — 추측)

> **실용적 참고**: 구어 독일어에서는 현재 시제 + 시간 표현이 종종 Futur I을 대체한다:
> Ich **komme** morgen. = Ich **werde** morgen **kommen**.

### Futur II (미래완료)

werden + Partizip II + haben/sein으로 형성된다. 완료된 미래 행위나 과거에 대한 추측을 표현한다.

- Bis morgen **werde** ich das Buch **gelesen haben**. (내일까지 그 책을 다 읽었을 것이다.)
- Er **wird** das wohl **vergessen haben**. (그는 아마 그것을 잊었을 것이다. — 추측)
- Sie **wird** schon **angekommen sein**. (그녀는 이미 도착했을 것이다.)

## 8. Perfekt에서의 Haben vs. Sein

Perfekt(와 Plusquamperfekt)에서 조동사로 **haben**과 **sein** 중 어느 것을 선택할지는 명확한 규칙을 따른다:

### sein을 사용하는 경우:

1. **A에서 B로의 이동 동사** (장소 변화):
   - gehen, kommen, fahren, fliegen, laufen, reisen, fallen, steigen, schwimmen

2. **상태 변화 동사**:
   - werden (되다), sterben (죽다), aufwachen (깨다), einschlafen (잠들다), wachsen (자라다), entstehen (생겨나다)

3. **sein**과 **bleiben** 자체:
   - Ich **bin** muede **gewesen**. (나는 피곤했다.)
   - Er **ist** zu Hause **geblieben**. (그는 집에 있었다.)

4. **geschehen, passieren, gelingen, misslingen** (일어나다, 성공하다, 실패하다):
   - Was **ist** passiert? (무슨 일이 있었니?)

### haben을 사용하는 경우:

1. **모든 타동사** (직접 대격 목적어가 있는 동사):
   - Ich **habe** das Buch **gelesen**. (나는 그 책을 읽었다.)
   - Er **hat** den Brief **geschrieben**. (그는 편지를 썼다.)

2. **모든 재귀동사**:
   - Ich **habe** mich gefreut. (나는 기뻤다.)
   - Er **hat** sich gewaschen. (그는 씻었다.)

3. **지속이나 위치를 표현하는 동사** (A에서 B로의 이동이 아닌):
   - Ich **habe** drei Stunden **geschlafen**. (나는 세 시간 잤다.)
   - Sie **hat** am Tisch **gesessen**. (그녀는 탁자에 앉아 있었다.)

4. **비인칭 동사**:
   - Es **hat** geregnet. (비가 왔다.)
   - Es **hat** geschneit. (눈이 왔다.)

### 판단 순서도 요약

```
Does the verb have an accusative object?
   YES --> HABEN
   NO  --> Does it express movement from A to B?
           YES --> SEIN (gehen, kommen, fahren...)
           NO  --> Does it express a change of state?
                   YES --> SEIN (werden, sterben, aufwachen...)
                   NO  --> Is it sein or bleiben?
                           YES --> SEIN
                           NO  --> HABEN
```

### 주의해야 할 경우

| 동사 | haben과 함께 | sein과 함께 |
|------|-------------|-------------|
| **fahren** | Ich **habe** das Auto **gefahren**. (차를 운전했다. — 목적어 있음) | Ich **bin** nach Berlin **gefahren**. (베를린으로 갔다. — A에서 B로 이동) |
| **schwimmen** | Er **hat** eine Stunde **geschwommen**. (한 시간 수영했다. — 지속) | Er **ist** ans Ufer **geschwommen**. (해안으로 수영했다. — A에서 B로) |
| **fliegen** | Der Pilot **hat** das Flugzeug **geflogen**. (조종사가 비행기를 조종했다. — 목적어 있음) | Wir **sind** nach Japan **geflogen**. (일본으로 날아갔다. — A에서 B로) |

## 9. 수동태: werden + Partizip II

수동태(Passive Voice)는 초점을 **행위자**(agent)에서 **행위** 또는 행위의 **대상**으로 전환한다.

### 형성: Vorgangspassiv (과정 수동태)

```
werden (conjugated) + Partizip II
```

### 능동태 vs. 수동태

| 능동태 | 수동태 |
|--------|--------|
| Der Lehrer **erklaert** die Lektion. | Die Lektion **wird** (vom Lehrer) **erklaert**. |
| (선생님이 수업을 설명한다.) | (수업이 [선생님에 의해] 설명된다.) |
| Die Studenten **lesen** das Buch. | Das Buch **wird** (von den Studenten) **gelesen**. |
| (학생들이 책을 읽는다.) | (책이 [학생들에 의해] 읽힌다.) |

### 모든 시제의 수동태

| 시제 | 능동태 (er liest das Buch) | 수동태 (das Buch...) |
|------|--------------------------|---------------------|
| 현재 | Er liest das Buch. | Das Buch **wird** gelesen. |
| Praeteritum | Er las das Buch. | Das Buch **wurde** gelesen. |
| Perfekt | Er hat das Buch gelesen. | Das Buch **ist** gelesen **worden**. |
| Plusquamperfekt | Er hatte das Buch gelesen. | Das Buch **war** gelesen **worden**. |
| Futur I | Er wird das Buch lesen. | Das Buch **wird** gelesen **werden**. |

> **Perfekt 수동태에 대한 참고**: 수동태에서 werden의 Partizip II는 **worden**이다 (*geworden*이 아님):
> Das Buch **ist** gelesen **worden**. (*geworden*이 아님)

### 행위자 (누구에 의해)

수동 문장에서 행위자(하는 사람)는 **von** + 여격으로 도입된다:

- Das Haus **wird von dem Arbeiter** gebaut. (집이 노동자에 의해 지어진다.)
- Der Brief **wurde von meiner Mutter** geschrieben. (편지가 어머니에 의해 쓰였다.)

도구나 수단에는 **durch** + 대격을 사용한다:

- Die Stadt **wurde durch ein Erdbeben** zerstoert. (도시가 지진에 의해 파괴되었다.)

### 비인칭 수동태 (es wird + Partizip II)

독일어에서는 자동사로도 비인칭 구문을 사용하여 수동태를 만들 수 있다:

- **Es wird** hier nicht **geraucht**. (여기서는 흡연이 금지이다.)
- **Es wurde** viel **getanzt**. (많이 춤을 추었다.)

### 상태 수동태(Zustandspassiv): sein + Partizip II

**Zustandspassiv**는 완료된 행위의 **결과**(과정이 아닌)를 묘사한다:

| Vorgangspassiv (과정) | Zustandspassiv (상태/결과) |
|----------------------|--------------------------|
| Die Tuer **wird** geoeffnet. (열리는 중이다) | Die Tuer **ist** geoeffnet. (열려 있다) |
| Das Auto **wird** repariert. (수리되는 중이다) | Das Auto **ist** repariert. (수리된 상태이다) |

## 10. Konjunktiv II: 형성과 용법

Konjunktiv II(접속법 제2식)는 **가정적, 비현실적, 또는 공손한** 상황을 표현한다.

### 형성: 두 가지 방법

**방법 1: Praeteritum 어간에서** (sein, haben, 화법조동사, 일부 강변화 동사에 적용)

Praeteritum 어간을 가져와 가능한 곳에 움라우트(Umlaut)를 추가하고, 접속법 어미(-e, -est, -e, -en, -et, -en)를 붙인다:

| 원형 | Praeteritum | Konjunktiv II |
|------|------------|---------------|
| sein | war | **waere** |
| haben | hatte | **haette** |
| koennen | konnte | **koennte** |
| muessen | musste | **muesste** |
| duerfen | durfte | **duerfte** |
| sollen | sollte | **sollte** (움라우트 없음) |
| wollen | wollte | **wollte** (움라우트 없음) |
| kommen | kam | **kaeme** |
| gehen | ging | **ginge** |
| wissen | wusste | **wuesste** |
| geben | gab | **gaebe** |
| fahren | fuhr | **fuehre** |

**방법 2: wuerde + 원형** (구어 독일어에서 대부분의 동사에 적용)

```
wuerde (conjugated) + Infinitive
```

| 인칭 | wuerde + machen |
|------|----------------|
| ich | wuerde machen |
| du | wuerdest machen |
| er/sie/es | wuerde machen |
| wir | wuerden machen |
| ihr | wuerdet machen |
| sie/Sie | wuerden machen |

> **어느 것을 언제 사용하나**: **sein, haben, 화법조동사, 흔한 강변화 동사**에는 단일 단어 형태(waere, haette, koennte 등)를 사용한다. **약변화 동사**와 Konjunktiv II 형태가 Praeteritum과 동일한 동사(예: machte = Praet. 이자 Konj. II — 모호함을 피하기 위해 *wuerde machen* 사용)에는 **wuerde + 원형**을 사용한다.

### Konjunktiv II의 주요 용법

**1. 비현실 조건 (if...would)**

- **Wenn** ich reich **waere**, **wuerde** ich eine Weltreise **machen**.
  (내가 부자라면 세계 여행을 할 텐데.)
- **Wenn** er Zeit **haette**, **kaeme** er mit.
  (그에게 시간이 있다면 같이 올 텐데.)
- **Wenn** ich das **gewusst haette**, **haette** ich dir **geholfen**.
  (그것을 알았더라면 너를 도왔을 텐데.)

**2. 공손한 요청과 제안**

- **Koennten** Sie mir bitte helfen? (저를 도와주실 수 있으시겠습니까?)
- Ich **haette** gern einen Kaffee. (커피 한 잔 부탁합니다.)
- **Wuerden** Sie bitte das Fenster oeffnen? (창문을 열어주시겠습니까?)
- Ich **wuerde** Ihnen empfehlen, frueh zu kommen. (일찍 오시길 권합니다.)

**3. 소원**

- **Wenn** ich doch nur fliegen **koennte**! (날 수만 있다면!)
- Ich **wuenschte**, ich **waere** jetzt am Strand. (지금 해변에 있었으면 좋겠다.)

**4. Konjunktiv II 과거 (haette/waere + Partizip II)**

과거의 비현실 상황에 사용:

- Ich **haette** das nicht **gemacht**. (나는 그것을 하지 않았을 것이다.)
- Er **waere** gern **gekommen**. (그는 기꺼이 왔을 것이다.)
- Wenn ich das **gewusst haette**, **haette** ich anders **gehandelt**.
  (그것을 알았더라면 다르게 행동했을 것이다.)

## 11. 동사 결합가: 자동사, 타동사, 이중 목적어 동사

**동사 결합가(Verb Valency)**는 동사가 몇 개의, 어떤 유형의 목적어를 필요로 하는지를 기술한다.

### 자동사 (대격 목적어 없음)

| 동사 | 예시 | 참고 |
|------|------|------|
| schlafen | Er **schlaeft**. | 목적어 불가 |
| kommen | Sie **kommt** morgen. | 대격 목적어 없음 |
| sterben | Der Mann **ist gestorben**. | 목적어 없음 |
| lachen | Die Kinder **lachen**. | 목적어 없음 |
| arbeiten | Ich **arbeite** jeden Tag. | 대격 목적어 없음 |

> 일부 자동사는 **여격** 목적어를 취한다: helfen + 여격, danken + 여격, gefallen + 여격.
> Ich helfe **dem Mann**. (여격, 대격이 아님)

### 타동사 (대격 목적어 하나)

| 동사 | 예시 | 목적어 |
|------|------|--------|
| sehen | Ich **sehe den** Mann. | den Mann (대격) |
| lesen | Er **liest ein** Buch. | ein Buch (대격) |
| kaufen | Sie **kauft einen** Kuchen. | einen Kuchen (대격) |
| verstehen | Wir **verstehen die** Frage. | die Frage (대격) |
| besuchen | Ich **besuche meinen** Freund. | meinen Freund (대격) |

### 이중 목적어 동사 (여격 + 대격 목적어)

| 동사 | 예시 | 여격 목적어 | 대격 목적어 |
|------|------|-----------|-----------|
| geben | Ich gebe **dem Mann** **das Buch**. | dem Mann | das Buch |
| schicken | Sie schickt **ihrer Mutter** **einen Brief**. | ihrer Mutter | einen Brief |
| zeigen | Er zeigt **uns** **den Weg**. | uns | den Weg |
| erklaeren | Sie erklaert **den Schuelern** **die Grammatik**. | den Schuelern | die Grammatik |
| schenken | Ich schenke **dir** **eine Blume**. | dir | eine Blume |
| empfehlen | Der Arzt empfiehlt **dem Patienten** **Ruhe**. | dem Patienten | Ruhe |

### 결합가와 수동태

**타동사**(대격 목적어를 가진)만 완전한 인칭 수동태를 형성할 수 있다. 대격 목적어가 수동 문장의 주어가 된다:

| 능동태 (타동사) | 수동태 |
|----------------|--------|
| Der Lehrer erklaert **die Lektion**. | **Die Lektion** wird erklaert. |
| Sie liest **das Buch**. | **Das Buch** wird gelesen. |

자동사는 **비인칭 수동태**만 형성할 수 있다:

| 능동태 (자동사) | 비인칭 수동태 |
|---------------|-------------|
| Die Leute tanzen. | **Es wird** getanzt. |
| Man arbeitet hier. | Hier **wird** gearbeitet. |

## 12. 화법조동사: 완전 복습

### 6개의 화법조동사

| 화법조동사 | 의미 | 현재 (ich) | Praeteritum (ich) | Partizip II |
|----------|------|-----------|-------------------|-------------|
| **koennen** | ~할 수 있다 | kann | konnte | gekonnt |
| **muessen** | ~해야 한다 | muss | musste | gemusst |
| **duerfen** | ~해도 된다 | darf | durfte | gedurft |
| **sollen** | ~해야 한다 (의무) | soll | sollte | gesollt |
| **wollen** | ~하고 싶다 | will | wollte | gewollt |
| **moegen** | 좋아하다 | mag | mochte | gemocht |

### 현재 시제 활용

모든 화법조동사는 독특한 패턴을 공유한다: 단수 형태가 복수와 다르고, 1인칭/3인칭 단수가 동일하다:

| 인칭 | koennen | muessen | duerfen | sollen | wollen | moegen |
|------|---------|---------|---------|--------|--------|--------|
| ich | kann | muss | darf | soll | will | mag |
| du | kannst | musst | darfst | sollst | willst | magst |
| er/sie/es | kann | muss | darf | soll | will | mag |
| wir | koennen | muessen | duerfen | sollen | wollen | moegen |
| ihr | koennt | muesst | duerft | sollt | wollt | moegt |
| sie/Sie | koennen | muessen | duerfen | sollen | wollen | moegen |

### 화법조동사 문장 구조

```
Subject + Modal (position 2) + ... + Infinitive (end)
```

- Ich **kann** gut schwimmen. (나는 수영을 잘 할 수 있다.)
- Du **musst** morgen frueh aufstehen. (너는 내일 일찍 일어나야 한다.)
- Wir **duerfen** hier nicht rauchen. (여기서 담배를 피워서는 안 된다.)

### Perfekt에서의 화법조동사 (이중 부정사)

다른 동사와 함께 사용될 때, 화법조동사는 **이중 부정사(Double Infinitive)** 구문을 사용한다:

```
haben + ... + Infinitive + Modal-Infinitive
```

- Ich **habe** nicht kommen **koennen**. (올 수 없었다.)
- Sie **hat** das nicht machen **wollen**. (그녀는 그것을 하고 싶지 않았다.)

> **실용적 팁**: 이중 부정사가 어색하게 들리기 때문에, 화법조동사는 구어 독일어에서 거의 항상 **Praeteritum**으로 사용된다:
> Ich **konnte** nicht kommen. (Ich habe nicht kommen koennen보다 선호됨.)

### moechten (~하고 싶다)

**moechten**은 moegen의 Konjunktiv II 형태로, 공손한 "would like"로 널리 사용된다. 화법조동사처럼 활용된다: ich/er = moechte, du = moechtest, wir/sie = moechten.

- Ich **moechte** einen Kaffee, bitte. (커피 한 잔 주세요.)
- **Moechten** Sie bestellen? (주문하시겠습니까?)

## 13. 조동사: sein, haben, werden

### sein (~이다)

| 시제 | ich | du | er/sie/es | wir | ihr | sie/Sie |
|------|-----|-----|-----------|-----|-----|---------|
| 현재 | bin | bist | ist | sind | seid | sind |
| Praeteritum | war | warst | war | waren | wart | waren |
| Konjunktiv II | waere | waerst | waere | waeren | waert | waeren |
| Partizip II | gewesen |

### haben (가지다)

| 시제 | ich | du | er/sie/es | wir | ihr | sie/Sie |
|------|-----|-----|-----------|-----|-----|---------|
| 현재 | habe | hast | hat | haben | habt | haben |
| Praeteritum | hatte | hattest | hatte | hatten | hattet | hatten |
| Konjunktiv II | haette | haettest | haette | haetten | haettet | haetten |
| Partizip II | gehabt |

### werden (되다 / 조동사)

| 시제 | ich | du | er/sie/es | wir | ihr | sie/Sie |
|------|-----|-----|-----------|-----|-----|---------|
| 현재 | werde | wirst | wird | werden | werdet | werden |
| Praeteritum | wurde | wurdest | wurde | wurden | wurdet | wurden |
| Konjunktiv II | wuerde | wuerdest | wuerde | wuerden | wuerdet | wuerden |
| Partizip II | geworden (본동사) / worden (수동태) |

### werden의 세 가지 기능

| 기능 | 구조 | 예시 |
|------|------|------|
| 본동사: "되다" | werden + 형용사/명사 | Er **wird** Arzt. |
| 미래 조동사 | werden + 원형 | Ich **werde** gehen. |
| 수동태 조동사 | werden + Partizip II | Das Buch **wird** gelesen. |

```
+----------------------------------------------------------------+
|                                                                |
|  WEAK VERBS (schwache Verben)                                  |
|  - Largest group; all new verbs are weak                       |
|  - Pattern: machen / machte / gemacht                          |
|  - No vowel change; -te Praeteritum; ge-...-t Partizip II     |
|                                                                |
|  STRONG VERBS (starke Verben)                                  |
|  - About 200 common verbs                                      |
|  - Pattern: gehen / ging / gegangen                            |
|  - Vowel change (Ablaut); no -te; ge-...-en Partizip II       |
|  - Many have present tense stem change (du/er)                 |
|                                                                |
|  MIXED VERBS (gemischte Verben)                                |
|  - Small group: denken, bringen, kennen, nennen, rennen,      |
|    senden, wenden, wissen                                      |
|  - Vowel change + weak endings: dachte / gedacht               |
|                                                                |
|  MODAL VERBS (Modalverben)                                     |
|  - koennen, muessen, duerfen, sollen, wollen, moegen          |
|  - ich/er forms identical; no ending in sg. present            |
|  - Praeteritum preferred over Perfekt in speech                |
|                                                                |
|  AUXILIARY VERBS (Hilfsverben)                                 |
|  - sein (Perfekt with movement/change of state)                |
|  - haben (Perfekt with most verbs)                             |
|  - werden (future, passive, Konjunktiv II with wuerde)        |
|                                                                |
+----------------------------------------------------------------+
```

## 15. 연습 문제

### 연습 1: 동사 분류

각 동사를 약변화(W), 강변화(S), 혼합변화(M)로 분류하시오. Praeteritum과 Partizip II를 쓰시오.

1. lernen (배우다) -> 유형: ________ | Praet.: ________ | PP: ________
2. sprechen (말하다) -> 유형: ________ | Praet.: ________ | PP: ________
3. denken (생각하다) -> 유형: ________ | Praet.: ________ | PP: ________
4. fahren (운전하다) -> 유형: ________ | Praet.: ________ | PP: ________
5. bringen (가져오다) -> 유형: ________ | Praet.: ________ | PP: ________
6. spielen (놀다) -> 유형: ________ | Praet.: ________ | PP: ________
7. schreiben (쓰다) -> 유형: ________ | Praet.: ________ | PP: ________
8. kennen (알다) -> 유형: ________ | Praet.: ________ | PP: ________

<details>
<summary>정답</summary>

1. **W** (약변화) | lernte | gelernt (규칙적: 모음 변화 없음, -te, ge-...-t)
2. **S** (강변화) | sprach | gesprochen (모음 변화 e->a->o, ge-...-en)
3. **M** (혼합변화) | dachte | gedacht (모음 변화 + 약변화 어미 -te, ge-...-t)
4. **S** (강변화) | fuhr | gefahren (모음 변화 a->u->a, ge-...-en)
5. **M** (혼합변화) | brachte | gebracht (모음 변화 + 약변화 어미)
6. **W** (약변화) | spielte | gespielt (규칙적)
7. **S** (강변화) | schrieb | geschrieben (모음 변화 ei->ie->ie)
8. **M** (혼합변화) | kannte | gekannt (모음 변화 + 약변화 어미)

</details>

### 연습 2: Haben인가 Sein인가?

Perfekt에 맞는 올바른 조동사를 선택하시오.

1. Ich ________ nach Berlin gefahren. (fahren — A에서 B로 이동)
2. Er ________ das Auto gefahren. (fahren — 대격 목적어 있음)
3. Wir ________ drei Stunden geschlafen. (schlafen — 지속)
4. Sie ________ Aerztin geworden. (werden — 상태 변화)
5. Ich ________ das Buch gelesen. (lesen — 타동사)
6. Der Hund ________ weggelaufen. (weglaufen — 이동)
7. Er ________ zu Hause geblieben. (bleiben — 항상 sein)
8. Was ________ passiert? (passieren — 항상 sein)

<details>
<summary>정답</summary>

1. **bin** (A에서 B로의 이동)
2. **habe** (타동사 — 대격 목적어 "das Auto" 있음)
3. **haben** (이동 아님, 지속만)
4. **ist** (상태 변화)
5. **habe** (타동사)
6. **ist** (이동)
7. **ist** (bleiben은 항상 sein 사용)
8. **ist** (passieren은 항상 sein 사용)

</details>

### 연습 3: 시제 훈련 완성하기

"schreiben" (쓰다, 강변화, haben)을 "er"에 대해 모든 시제로 활용하시오.

1. 현재: Er ________
2. Praeteritum: Er ________
3. Perfekt: Er ________
4. Plusquamperfekt: Er ________
5. Futur I: Er ________
6. Futur II: Er ________

<details>
<summary>정답</summary>

1. Er **schreibt**. (현재 — schreiben은 어간 변화 없음)
2. Er **schrieb**. (Praeteritum — 모음 변화 ei -> ie)
3. Er **hat geschrieben**. (Perfekt — haben + Partizip II)
4. Er **hatte geschrieben**. (Plusquamperfekt — hatte + Partizip II)
5. Er **wird schreiben**. (Futur I — werden + 원형)
6. Er **wird geschrieben haben**. (Futur II — werden + PP + haben)

</details>

### 연습 4: 능동태를 수동태로

각 능동태 문장을 동일한 시제의 수동태로 다시 쓰시오.

1. Der Koch kocht die Suppe. (현재) -> ________
2. Die Schueler lasen das Buch. (Praeteritum) -> ________
3. Man hat den Brief geschrieben. (Perfekt) -> ________
4. Der Mechaniker wird das Auto reparieren. (Futur I) -> ________

<details>
<summary>정답</summary>

1. Die Suppe **wird** (vom Koch) **gekocht**. (현재 수동태)
2. Das Buch **wurde** (von den Schuelern) **gelesen**. (Praeteritum 수동태)
3. Der Brief **ist** geschrieben **worden**. (Perfekt 수동태 — worden, geworden이 아님)
4. Das Auto **wird** (vom Mechaniker) **repariert werden**. (Futur I 수동태)

</details>

### 연습 5: 종합 도전 — 독일어로 번역하기

동사 유형, 시제, 조동사 선택, 태에 주의하며 각 문장을 번역하시오.

1. She can speak three languages. (화법조동사, 현재)
2. If I were rich, I would travel the world. (Konjunktiv II)
3. The house was built by the workers. (수동태, Praeteritum)
4. After he had eaten, he went for a walk. (Plusquamperfekt + Praeteritum)
5. I would like to order, please. (moechten)
6. He has been learning German for two years. (seit + 현재 시제)
7. The letter has been written. (Perfekt 수동태)
8. If I had known that, I would have helped you. (Konj. II 과거)
9. He brought her a flower. (이중 목적어, Praeteritum, 혼합변화 동사)
10. It is getting cold and it will rain tomorrow. (werden 본동사 + Futur I)

<details>
<summary>정답</summary>

1. Sie **kann** drei Sprachen **sprechen**. (화법조동사 + 원형이 끝에)
2. **Wenn** ich reich **waere**, **wuerde** ich die Welt **bereisen**. (Konj. II: waere + wuerde + 원형)
3. Das Haus **wurde** von den Arbeitern **gebaut**. (Praeteritum 수동태)
4. **Nachdem** er **gegessen hatte**, **ging** er spazieren. (nachdem-절에서 Plusquamperfekt, 주절에서 Praet.)
5. Ich **moechte** bitte **bestellen**. (moechten + 원형)
6. Er **lernt** Deutsch **seit** zwei Jahren. (seit + 현재 시제 — 독일어에서는 seit와 함께 진행 중인 행위에 현재 시제를 사용)
7. Der Brief **ist** geschrieben **worden**. (Perfekt 수동태 — worden, geworden이 아님)
8. **Wenn** ich das **gewusst haette**, **haette** ich dir **geholfen**. (Konj. II 과거: haette + PP)
9. Er **brachte** ihr eine Blume. (bringen은 혼합변화: brachte; 여격 + 대격)
10. Es **wird** kalt und es **wird** morgen **regnen**. (werden "되다" + werden + 원형 미래)

</details>
