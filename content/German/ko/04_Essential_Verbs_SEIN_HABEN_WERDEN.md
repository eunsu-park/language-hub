## 1. 독일어 동사 기초

### 1.1 부정사 형태

모든 독일어 동사에는 "사전 형태"인 부정사(Infinitiv)가 있습니다. 거의 모든 독일어 부정사는 **-en**으로 끝납니다:

```
┌─────────────────────────────────────────────────────────────────┐
│                  German Infinitive Forms                         │
│                                                                 │
│   Standard ending: -en                                          │
│     kommen     (to come)        gehen      (to go)              │
│     sprechen   (to speak)       machen     (to do/make)         │
│     spielen    (to play)        lernen     (to learn)           │
│     wohnen     (to live)        arbeiten   (to work)            │
│     lesen      (to read)        schreiben  (to write)           │
│                                                                 │
│   Less common ending: -n (instead of -en)                       │
│     sein       (to be)          tun        (to do)              │
│     wandern    (to hike)        ändern     (to change)          │
│     sammeln    (to collect)     klingeln   (to ring)            │
│                                                                 │
│   The infinitive has two parts:                                 │
│     STEM + ENDING                                               │
│     komm + en  →  kommen                                        │
│     lern + en  →  lernen                                        │
│     spiel + en →  spielen                                       │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 활용(Konjugation)이란?

활용(Konjugation)이란 주어에 맞게 동사 형태를 변화시키는 것입니다. 영어에서는 활용이 최소한입니다: "I speak, you speak, he speak**s**." 독일어에서는 모든 인칭에 따라 동사 어미가 변합니다:

```
Infinitive: kommen (to come)
Stem: komm-

  ich komm-e       (I come)
  du komm-st       (you come)
  er/sie/es komm-t (he/she/it comes)
  wir komm-en      (we come)
  ihr komm-t       (you all come)
  sie/Sie komm-en  (they/you formal come)

Regular conjugation endings:
  ┌──────────┬──────────┐
  │ Singular │ Plural   │
  ├──────────┼──────────┤
  │ -e       │ -en      │
  │ -st      │ -t       │
  │ -t       │ -en      │
  └──────────┴──────────┘
```

### 1.3 SEIN, HABEN, WERDEN이 특별한 이유

이 세 동사는 독일어의 기초입니다:

```
┌─────────────────────────────────────────────────────────────────┐
│   Why These Three Verbs Matter                                   │
│                                                                 │
│   1. Most frequently used verbs in German                       │
│   2. Highly irregular — do NOT follow regular conjugation       │
│   3. Auxiliary verbs — help form other tenses:                  │
│                                                                 │
│      sein    → Perfekt tense (movement/state verbs)             │
│               "Ich bin gekommen." (I have come.)                │
│                                                                 │
│      haben   → Perfekt tense (most other verbs)                 │
│               "Ich habe gegessen." (I have eaten.)              │
│                                                                 │
│      werden  → Futur I (future tense)                           │
│               "Ich werde kommen." (I will come.)                │
│               → Passiv (passive voice)                          │
│               "Das Haus wird gebaut." (The house is built.)     │
│                                                                 │
│   4. Essential from day one — you cannot form basic sentences   │
│      without them                                               │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 인칭대명사

### 2.1 모든 독일어 인칭대명사

| 인칭 | 독일어 | 영어 | 비고 |
|--------|--------|---------|-------|
| 1인칭 단수 | **ich** | I | 항상 소문자 (문장 시작 시 제외) |
| 2인칭 단수 (비격식) | **du** | you | 친구, 가족, 아이들에게 사용 |
| 3인칭 단수 남성 | **er** | he / it | 남성 명사 대신 사용: *der Tisch* → *er* |
| 3인칭 단수 여성 | **sie** | she / it | 여성 명사 대신 사용: *die Lampe* → *sie* |
| 3인칭 단수 중성 | **es** | it | 중성 명사 대신 사용: *das Buch* → *es* |
| 1인칭 복수 | **wir** | we | |
| 2인칭 복수 (비격식) | **ihr** | you all | *du*의 복수형 |
| 3인칭 복수 | **sie** | they | "she"와 같은 단어 — 문맥으로 구분 |
| 2인칭 격식 (단수 & 복수) | **Sie** | you (격식) | 항상 대문자! |

### 2.2 du/ihr와 Sie의 구분

```
┌─────────────────────────────────────────────────────────────────┐
│   Informal (du/ihr) vs Formal (Sie)                              │
│                                                                 │
│   Use DU (singular) / IHR (plural) with:                        │
│     - Friends and family                                        │
│     - Children and young people                                 │
│     - Fellow students                                           │
│     - People who offer "du" (Duzen)                             │
│     - Animals                                                   │
│     - God (in prayers)                                          │
│                                                                 │
│   Use SIE with:                                                 │
│     - Strangers and acquaintances                               │
│     - In professional/business settings                         │
│     - With older people (until they offer "du")                 │
│     - Authority figures                                         │
│     - Customer service                                          │
│                                                                 │
│   Rule of thumb: When in doubt, use Sie. The other person      │
│   will offer "du" if they prefer it.                            │
│   "Wollen wir uns duzen?" = Shall we use du with each other?   │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 같은 모양인 세 단어: sie / sie / Sie

```
sie (lowercase, singular context) = she
  Sie kommt aus Berlin.           She comes from Berlin.
  → verb is singular (kommt)

sie (lowercase, plural context) = they
  Sie kommen aus Berlin.          They come from Berlin.
  → verb is plural (kommen)

Sie (CAPITALIZED, any position) = you (formal)
  Sie kommen aus Berlin.          You come from Berlin. (formal)
  → verb is plural (kommen), same as "they"

How to tell them apart:
  - Capitalization: Sie (formal you) is always capitalized
  - Verb form: sie (she) + singular verb vs sie (they) + plural verb
  - Context of the conversation
```

### 2.4 대명사 대체로서의 er, sie, es

영어와 달리 독일어는 명사의 **문법적 성**에 따라 *er*, *sie*, *es*를 사용합니다:

```
Der Tisch ist groß.   → Er ist groß.    (It is big.)     — der → er
Die Lampe ist neu.     → Sie ist neu.    (It is new.)     — die → sie
Das Buch ist gut.      → Es ist gut.     (It is good.)    — das → es
Das Mädchen ist nett.  → Es ist nett.    (She is nice.)   — das → es!

Note: das Mädchen (girl) → es (not sie!), because the noun is neuter.
```

## 3. SEIN — ~이다

### 3.1 현재 시제 활용

*Sein*은 독일어에서 가장 불규칙한 동사입니다. 모든 형태가 어간과 완전히 다릅니다.

| 인칭 | 활용 | 영어 | 발음 |
|--------|------------|---------|---------------|
| ich | **bin** | I am | [bɪn] |
| du | **bist** | you are | [bɪst] |
| er/sie/es | **ist** | he/she/it is | [ɪst] |
| wir | **sind** | we are | [zɪnt] |
| ihr | **seid** | you all are | [zaɪt] |
| sie/Sie | **sind** | they are / you are (격식) | [zɪnt] |

```
┌─────────────────────────────────────────────────────────────────┐
│   SEIN Conjugation — Memorize This!                              │
│                                                                 │
│        Singular              Plural                             │
│   ┌──────────────────┬──────────────────┐                       │
│   │ ich    bin       │ wir    sind      │                       │
│   │ du     bist      │ ihr    seid      │                       │
│   │ er/sie/es ist    │ sie/Sie sind     │                       │
│   └──────────────────┴──────────────────┘                       │
│                                                                 │
│   Compare with English:                                         │
│     I am        → ich bin                                       │
│     you are     → du bist / Sie sind                            │
│     he/she/it is → er/sie/es ist                                │
│     we are      → wir sind                                      │
│     they are    → sie sind                                      │
│                                                                 │
│   Every form is different — there is no pattern to derive       │
│   one from another. Pure memorization required!                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 SEIN으로 신원 표현하기

```
Who someone is:
  Ich bin Thomas.                    I am Thomas.
  Sie ist meine Schwester.          She is my sister.
  Wir sind Studenten.               We are students.
  Das ist mein Freund.              That is my friend.
  Wer bist du?                      Who are you?

Introducing yourself:
  Ich bin Anna. Und du?             I am Anna. And you?
  Ich bin der neue Kollege.         I am the new colleague.
  Mein Name ist Müller.             My name is Müller.
```

### 3.3 SEIN으로 국적과 출신 표현하기

```
Nationality (NO article!):
  Ich bin Deutscher.                 I am German. (male)
  Ich bin Deutsche.                  I am German. (female)
  Er ist Amerikaner.                 He is American.
  Sie ist Japanerin.                 She is Japanese.
  Wir sind Koreaner.                 We are Korean.

Origin with "aus" (from):
  Ich bin aus Deutschland.           I am from Germany.
  Woher bist du?                     Where are you from?
  Sie ist aus Seoul.                 She is from Seoul.
  Wir sind aus Berlin.               We are from Berlin.
```

### 3.4 SEIN으로 직업 표현하기

```
Profession (NO article in German!):
  Er ist Arzt.                       He is a doctor.
  Sie ist Lehrerin.                  She is a teacher.
  Ich bin Ingenieur.                 I am an engineer.
  Wir sind Studenten.               We are students.
  Was bist du von Beruf?             What is your profession?

Compare:
  English: He is A doctor.   (article "a")
  German:  Er ist Arzt.      (no article)

  English: She is AN engineer. (article "an")
  German:  Sie ist Ingenieurin. (no article)
```

### 3.5 SEIN으로 묘사와 상태 표현하기

```
Description (with adjectives):
  Das Wetter ist schön.             The weather is nice.
  Der Film ist interessant.         The film is interesting.
  Die Kinder sind müde.             The children are tired.
  Das Essen ist lecker.             The food is delicious.
  Die Stadt ist groß.               The city is big.

Location:
  Ich bin in Berlin.                 I am in Berlin.
  Wo bist du?                        Where are you?
  Die Bücher sind auf dem Tisch.     The books are on the table.
  Er ist zu Hause.                   He is at home.
  Wir sind im Kino.                  We are at the cinema.

Time:
  Es ist drei Uhr.                   It is three o'clock.
  Wie spät ist es?                   What time is it?
  Es ist spät.                       It is late.
```

### 3.6 SEIN을 사용하는 주요 표현

| 독일어 | 영어 |
|--------|---------|
| Ich bin müde. | I am tired. |
| Ich bin hungrig. | I am hungry. |
| Ich bin krank. | I am sick. |
| Ich bin fertig. | I am done/finished. |
| Ich bin bereit. | I am ready. |
| Es ist kalt/warm/heiß. | It is cold/warm/hot. |
| Das ist richtig. | That is correct. |
| Das ist falsch. | That is wrong. |
| Es ist wichtig. | It is important. |
| Das ist toll! | That is great! |
| Mir ist kalt. | I am cold. (직역: "나에게 추운 것이다.") |
| Mir ist schlecht. | I feel sick. (직역: "나에게 나쁜 것이다.") |
| Es ist mir egal. | I don't care. (직역: "나에게 같은 것이다.") |
| Wie alt bist du? | How old are you? |
| Ich bin 25 Jahre alt. | I am 25 years old. |

### 3.7 SEIN을 사용한 의문문

```
Yes/No questions (verb first):
  Bist du Student?                   Are you a student?
  Ist sie deine Schwester?           Is she your sister?
  Sind Sie Herr Müller?              Are you Mr. Müller?
  Ist das richtig?                   Is that correct?
  Seid ihr müde?                     Are you all tired?

W-questions (question word + verb):
  Wer ist das?                       Who is that?
  Was ist das?                       What is that?
  Wo ist die Toilette?               Where is the toilet?
  Wie ist das Wetter?                How is the weather?
  Wie alt bist du?                   How old are you?
  Woher sind Sie?                    Where are you from?
  Warum bist du müde?                Why are you tired?
```

## 4. HABEN — ~을 가지다

### 4.1 현재 시제 활용

*Haben*은 du와 er/sie/es 형태에서 불규칙합니다 — 어간의 -b-가 탈락합니다.

| 인칭 | 활용 | 영어 | 발음 |
|--------|------------|---------|---------------|
| ich | **habe** | I have | [haːbə] |
| du | **hast** | you have | [hast] |
| er/sie/es | **hat** | he/she/it has | [hat] |
| wir | **haben** | we have | [haːbən] |
| ihr | **habt** | you all have | [haːpt] |
| sie/Sie | **haben** | they have / you have (격식) | [haːbən] |

```
┌─────────────────────────────────────────────────────────────────┐
│   HABEN Conjugation                                              │
│                                                                 │
│        Singular              Plural                             │
│   ┌──────────────────┬──────────────────┐                       │
│   │ ich    habe      │ wir    haben     │                       │
│   │ du     hast      │ ihr    habt      │                       │
│   │ er/sie/es hat    │ sie/Sie haben    │                       │
│   └──────────────────┴──────────────────┘                       │
│                                                                 │
│   Note the irregular forms:                                     │
│     haben → du hast  (not *habst — the b drops)                │
│     haben → er hat   (not *habt  — the b drops)                │
│                                                                 │
│   The stem changes: hab- → ha- in du and er/sie/es forms       │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 HABEN으로 소유 표현하기

```
Basic possession:
  Ich habe ein Auto.                I have a car.
  Hast du Geschwister?              Do you have siblings?
  Er hat zwei Kinder.               He has two children.
  Wir haben eine Wohnung in Berlin. We have an apartment in Berlin.
  Sie hat einen Hund.               She has a dog.

Quantity:
  Ich habe drei Bücher.             I have three books.
  Sie hat viel Geld.                She has a lot of money.
  Wir haben keine Zeit.             We have no time.
  Wie viele Kinder haben Sie?       How many children do you have?

With abstract nouns:
  Ich habe eine Frage.              I have a question.
  Hast du eine Idee?                Do you have an idea?
  Sie haben Recht.                  You are right. (lit. "You have right")
  Er hat ein Problem.               He has a problem.
```

### 4.3 HABEN으로 나이 표현하기

독일어에서 나이는 **sein**을 사용하며, **haben**을 사용하지 않습니다(영어와 동일):

```

Ich bin 25 Jahre alt.               I am 25 years old. (with sein)

BUT for age as a possession concept:
  Das Kind hat drei Jahre.          The child is three years old.
  → Less common; "Das Kind ist drei Jahre alt" is preferred.

Note: German uses SEIN for age (like English), unlike French/Spanish
which use "have." But many HABEN expressions replace English "be":
```

### 4.4 HABEN을 사용하는 주요 표현

영어에서 "to be"를 사용하는 많은 표현이 독일어에서는 "haben"을 사용합니다:

| 독일어 | 직역 | 자연스러운 영어 |
|--------|---------|-----------------|
| Ich habe Hunger. | I have hunger. | I am hungry. |
| Ich habe Durst. | I have thirst. | I am thirsty. |
| Ich habe Angst. | I have fear. | I am afraid. |
| Ich habe Lust (auf ...). | I have desire. | I feel like (doing something). |
| Ich habe Recht. | I have right. | I am right. |
| Ich habe Unrecht. | I have un-right. | I am wrong. |
| Ich habe Glück. | I have luck. | I am lucky. |
| Ich habe Pech. | I have bad luck. | I am unlucky. |
| Ich habe es eilig. | I have it hurry. | I am in a hurry. |
| Er hat Geburtstag. | He has birthday. | It's his birthday. |
| Ich habe Zeit. | I have time. | I have time. / I am free. |
| Ich habe keine Ahnung. | I have no idea. | I have no idea. |
| Ich habe Heimweh. | I have home-pain. | I am homesick. |
| Ich habe Langeweile. | I have boredom. | I am bored. |

```
┌─────────────────────────────────────────────────────────────────┐
│   Key Difference from English                                    │
│                                                                 │
│   English: I AM hungry/thirsty/afraid/right/lucky               │
│   German:  Ich HABE Hunger/Durst/Angst/Recht/Glück             │
│                                                                 │
│   Think: "I HAVE hunger" not "I AM hungry"                      │
│                                                                 │
│   Both "Ich habe Hunger" and "Ich bin hungrig" are correct,    │
│   but the HABEN version is much more natural and common         │
│   in everyday German speech.                                    │
│                                                                 │
│   More examples:                                                │
│     Hast du Hunger?              Are you hungry?                │
│     Wir haben großen Durst.      We are very thirsty.           │
│     Sie hat Angst vor Spinnen.   She is afraid of spiders.      │
│     Du hast Recht!               You are right!                 │
│     Ich habe heute Geburtstag.   It is my birthday today.       │
└─────────────────────────────────────────────────────────────────┘
```

### 4.5 HABEN을 사용한 의문문

```
Yes/No questions:
  Hast du Zeit?                     Do you have time?
  Haben Sie eine Frage?             Do you have a question? (formal)
  Hat er Kinder?                    Does he have children?
  Habt ihr Hunger?                  Are you (all) hungry?
  Hat sie ein Auto?                 Does she have a car?

W-questions:
  Was hast du?                      What do you have?
  Wie viele Kinder haben Sie?       How many children do you have?
  Wer hat mein Buch?                Who has my book?
  Wie viel Zeit haben wir?          How much time do we have?
```

## 5. WERDEN — ~이 되다 / ~할 것이다

### 5.1 현재 시제 활용

*Werden*은 du와 er/sie/es 형태에서 어간 모음이 변합니다 (e → i).

| 인칭 | 활용 | 영어 | 발음 |
|--------|------------|---------|---------------|
| ich | **werde** | I become / I will | [veːɐdə] |
| du | **wirst** | you become / you will | [vɪʁst] |
| er/sie/es | **wird** | he/she/it becomes / will | [vɪʁt] |
| wir | **werden** | we become / we will | [veːɐdən] |
| ihr | **werdet** | you all become / will | [veːɐdət] |
| sie/Sie | **werden** | they/you (격식) become / will | [veːɐdən] |

```
┌─────────────────────────────────────────────────────────────────┐
│   WERDEN Conjugation                                             │
│                                                                 │
│        Singular              Plural                             │
│   ┌──────────────────┬──────────────────┐                       │
│   │ ich    werde     │ wir    werden    │                       │
│   │ du     wirst     │ ihr    werdet    │                       │
│   │ er/sie/es wird   │ sie/Sie werden   │                       │
│   └──────────────────┴──────────────────┘                       │
│                                                                 │
│   Note the stem changes:                                        │
│     werden → du wirst  (e → i, d drops)                        │
│     werden → er wird   (e → i, e drops)                        │
│                                                                 │
│   Three main uses:                                              │
│     1. "to become" (main verb)                                  │
│     2. Future tense auxiliary (werden + infinitive)              │
│     3. Passive voice auxiliary (werden + past participle)        │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 WERDEN = "~이 되다" — 상태의 변화

현재 시제에서 *werden*은 주로 "~이 되다" 또는 "~해지다"를 의미합니다:

```
Change of state:
  Es wird kalt.                     It's getting cold.
  Es wird dunkel.                   It's getting dark.
  Es wird warm.                     It's getting warm.
  Er wird alt.                      He is getting old.
  Das Kind wird groß.               The child is growing up.
  Sie wird Ärztin.                  She is becoming a doctor.
  Ich werde müde.                   I'm getting tired.
  Er wird langsam nervös.           He is slowly getting nervous.

Weather and time:
  Es wird Frühling.                 Spring is coming. (It's becoming spring.)
  Es wird spät.                     It's getting late.
  Es wird bald Nacht.               It's going to be night soon.
  Morgen wird es regnen.            Tomorrow it's going to rain.

Career and transformation:
  Was willst du werden?             What do you want to become?
  Ich will Arzt werden.             I want to become a doctor.
  Er wird Vater.                    He is going to be a father.
  Sie wird immer schöner.           She is becoming more and more beautiful.
```

### 5.3 WERDEN = 미래 조동사 (미리보기)

*Werden* + 부정사(Infinitiv)는 미래 시제(Futur I)를 형성합니다. 여기서는 간략히 소개하며, 이후 수업에서 자세히 다룹니다:

```
Structure: werden (conjugated) + infinitive (at END of sentence)

  Ich werde Deutsch lernen.         I will learn German.
  Du wirst es verstehen.            You will understand it.
  Er wird morgen kommen.            He will come tomorrow.
  Wir werden nach Berlin fahren.    We will drive to Berlin.
  Sie werden es schaffen.           They will manage it.
  Werden Sie morgen kommen?         Will you come tomorrow? (formal)

Note: The infinitive goes to the END of the sentence!
  Ich WERDE morgen nach Berlin FAHREN.
  ↑ conjugated                  ↑ infinitive at end
```

### 5.4 WERDEN = 수동태 (미리보기)

*Werden*은 수동태를 형성하는 데에도 사용됩니다(이후 수업에서 다룸):

```
Structure: werden (conjugated) + past participle

  Das Haus wird gebaut.             The house is being built.
  Die Tür wird geöffnet.            The door is being opened.
  Das Buch wird gelesen.            The book is being read.
  Die Kinder werden gerufen.        The children are being called.
```

### 5.5 WERDEN을 사용하는 주요 표현

| 독일어 | 영어 |
|--------|---------|
| Es wird schon. | It'll be fine. / It'll work out. |
| Was willst du werden? | What do you want to become? (직업) |
| Mir wird schlecht. | I'm feeling sick. |
| Mir wird schwindlig. | I'm getting dizzy. |
| Es wird Zeit. | It's about time. |
| Was wird aus dir? | What will become of you? |
| Es wird nichts. | It's not going to work. / Nothing will come of it. |
| Wird gemacht! | Will do! / Consider it done! |

### 5.6 WERDEN과 영어 "become" / "get" 비교

```
┌─────────────────────────────────────────────────────────────────┐
│   "Werden" covers multiple English words:                        │
│                                                                 │
│   "become":  Sie wird Lehrerin.    She is becoming a teacher.   │
│   "get":     Es wird kalt.        It is getting cold.           │
│   "grow":    Er wird alt.         He is growing old.            │
│   "turn":    Sie wird rot.        She is turning red.           │
│   "will":    Ich werde kommen.    I will come.                  │
│   "be" (passive): Es wird gebaut. It is being built.            │
│                                                                 │
│   German "werden" is one of the most versatile verbs!           │
└─────────────────────────────────────────────────────────────────┘
```

## 6. SEIN, HABEN, WERDEN 비교

### 6.1 나란히 놓은 활용표

| 인칭 | sein (~이다) | haben (~을 가지다) | werden (~이 되다) |
|--------|-------------|-----------------|-------------------|
| ich | bin | habe | werde |
| du | bist | hast | wirst |
| er/sie/es | ist | hat | wird |
| wir | sind | haben | werden |
| ihr | seid | habt | werdet |
| sie/Sie | sind | haben | werden |

### 6.2 어떤 동사를 사용할까

```
┌─────────────────────────────────────────────────────────────────┐
│   SEIN (to be) — Use for:                                        │
│   - Identity:    Ich bin Anna.                                   │
│   - Profession:  Er ist Lehrer. (no article!)                   │
│   - Nationality: Sie ist Deutsche.                               │
│   - Description: Das Haus ist groß.                              │
│   - Location:    Wir sind in Berlin.                             │
│   - Origin:      Sie ist aus Japan.                              │
│   - Time:        Es ist 3 Uhr.                                  │
│   - Age:         Ich bin 25 Jahre alt.                           │
│                                                                 │
│   HABEN (to have) — Use for:                                     │
│   - Possession:  Ich habe ein Auto.                              │
│   - Physical states: Ich habe Hunger/Durst/Angst.                │
│   - Being right: Du hast Recht.                                  │
│   - Luck:        Ich habe Glück.                                 │
│   - Quantity:    Er hat drei Kinder.                              │
│                                                                 │
│   WERDEN (to become) — Use for:                                  │
│   - Change of state: Es wird kalt.                               │
│   - Future career:   Sie wird Ärztin.                            │
│   - Getting/growing: Er wird alt.                                │
│   - Future tense:    Ich werde kommen.                           │
│   - Passive voice:   Das Haus wird gebaut.                       │
└─────────────────────────────────────────────────────────────────┘
```

### 6.3 Sein과 Haben — 헷갈리는 경우

일부 표현은 독일어와 영어에서 다릅니다:

| 영어 ("be" 사용) | 독일어 |
|---------------------|--------|
| I **am** hungry | Ich **habe** Hunger |
| I **am** thirsty | Ich **habe** Durst |
| I **am** afraid | Ich **habe** Angst |
| I **am** right | Ich **habe** Recht |
| I **am** lucky | Ich **habe** Glück |
| I **am** in a hurry | Ich **habe** es eilig |
| How old **are** you? | Wie alt **bist** du? (sein 사용!) |
| I **am** 25 years old | Ich **bin** 25 Jahre alt (sein 사용!) |

> **참고**: 독일어에서 나이는 (영어처럼) *sein*을 사용합니다: *Ich **bin** zwanzig Jahre alt.* (나는 스무 살입니다.) 이것은 "have"를 나이에 사용하는 프랑스어, 스페인어와 다릅니다.

### 6.4 Sein과 Werden — 상태 vs 변화

```
SEIN = current state (how things ARE)
WERDEN = change (how things are CHANGING)

  Es ist kalt.     →  It is cold.        (state: it is cold right now)
  Es wird kalt.    →  It is getting cold. (change: it was warm, now becoming cold)

  Er ist alt.      →  He is old.         (state: he is old)
  Er wird alt.     →  He is getting old.  (change: he is aging)

  Sie ist müde.    →  She is tired.      (state: she is tired now)
  Sie wird müde.   →  She is getting tired. (change: she was awake, now tiring)

  Er ist Arzt.     →  He is a doctor.    (state: he is already a doctor)
  Er wird Arzt.    →  He is becoming a doctor. (change: he is studying medicine)
```

## 7. nicht와 kein을 이용한 부정

### 7.1 nicht — "~이 아니다"

*Nicht*는 동사, 형용사, 부사, 특정 요소를 부정합니다:

```
Negating verbs:
  Ich bin nicht müde.               I am not tired.
  Er kommt nicht.                   He is not coming.
  Wir haben nicht viel Zeit.        We don't have much time.
  Sie wird nicht kommen.            She will not come.

Negating adjectives:
  Das ist nicht richtig.            That is not correct.
  Das Buch ist nicht interessant.   The book is not interesting.
  Er ist nicht krank.               He is not sick.

Negating specific elements:
  Ich komme nicht aus Berlin.       I don't come from Berlin.
  Er ist nicht mein Bruder.         He is not my brother.
  Nicht ich, sondern er.            Not I, but he.
  Er kommt nicht morgen.            He is not coming tomorrow.
```

### 7.2 kein — "~이 아닌 / ~이 없는"

*Kein*은 *ein/eine*이나 관사가 없는 명사를 부정합니다:

```
With indefinite article (ein → kein):
  Ich habe ein Auto. → Ich habe kein Auto.
  (I have a car.)      (I don't have a car.)

  Das ist eine Katze. → Das ist keine Katze.
  (That is a cat.)       (That is not a cat.)

With no article (plural/uncountable → kein):
  Ich habe Geld. → Ich habe kein Geld.
  (I have money.)  (I have no money.)

  Er hat Kinder. → Er hat keine Kinder.
  (He has children.) (He has no children.)

  Ich habe Zeit. → Ich habe keine Zeit.
  (I have time.)    (I have no time.)
```

### 7.3 nicht와 kein — 선택 가이드

```
┌─────────────────────────────────────────────────────────────────┐
│   Use KEIN when:                                                 │
│   - Negating a noun with ein/eine:  ein Buch → kein Buch       │
│   - Negating a noun with no article: Geld → kein Geld          │
│                                                                 │
│   Use NICHT when:                                                │
│   - Negating a verb:          Ich komme nicht.                   │
│   - Negating an adjective:    Er ist nicht groß.                 │
│   - Negating a noun with definite article:                       │
│     Das ist nicht der Lehrer. (NOT *kein der Lehrer*)            │
│   - Negating a possessive:    Das ist nicht mein Buch.           │
│   - Negating a specific element: nicht hier, nicht morgen        │
│                                                                 │
│   Simple test:                                                   │
│   Can you replace the word with "kein/keine"?                    │
│   Yes → use kein.  No → use nicht.                               │
└─────────────────────────────────────────────────────────────────┘
```

### 7.4 kein 변화 (주격)

| 성 | 긍정형 | 부정형 |
|--------|----------|----------|
| 남성(Maskulin) | ein | kein |
| 여성(Feminin) | eine | keine |
| 중성(Neutrum) | ein | kein |
| 복수 | — | keine |
