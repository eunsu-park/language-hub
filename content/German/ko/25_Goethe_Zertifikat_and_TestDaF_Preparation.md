## 1. 독일어 자격증 개요

국제적으로 인정되는 네 가지 주요 독일어 자격증이 있습니다:

| 자격증 | 기관 | 레벨 | 주요 용도 |
|--------|------|------|-----------|
| **Goethe-Zertifikat** | Goethe-Institut | A1 - C2 | 범용: 이민, 취업, 학업 |
| **TestDaF** | TestDaF-Institut | B2 - C1 (TDN 3-5) | 독일 대학 입학 |
| **ÖSD** | Österreichisches Sprachdiplom | A1 - C2 | 오스트리아 독일어; 오스트리아 이민 |
| **telc Deutsch** | telc GmbH | A1 - C2 | 범용; 시민권 신청 |

### 어떤 시험을 봐야 할까?

```
+----------------------------------------------------------+
|  CHOOSING YOUR EXAM                                       |
+----------------------------------------------------------+
|                                                          |
|  Want to STUDY at a German university?                   |
|  → TestDaF (TDN 4 in all sections) or                   |
|  → Goethe-Zertifikat C1/C2 or DSH                       |
|                                                          |
|  Want to WORK in Germany?                                |
|  → Goethe-Zertifikat B1/B2 (depending on profession)    |
|  → telc Deutsch B1/B2                                    |
|                                                          |
|  Want to IMMIGRATE / get citizenship?                    |
|  → Goethe-Zertifikat B1 (+ Einbürgerungstest)           |
|  → telc Deutsch B1                                       |
|                                                          |
|  Want to IMMIGRATE to Austria?                           |
|  → ÖSD (recognized for Austrian residency)               |
|                                                          |
|  Want to prove general proficiency?                      |
|  → Goethe-Zertifikat at your level                       |
|  → telc at your level                                    |
+----------------------------------------------------------+
```

## 2. CEFR 레벨과 그 의미

모든 독일어 시험은 **CEFR** (유럽 공통 언어 참조 기준)에 맞추어져 있습니다:

| 레벨 | 명칭 | 할 수 있는 것 |
|------|------|--------------|
| **A1** | Breakthrough | 기본적인 표현을 이해하고 사용; 자기소개 가능 |
| **A2** | Waystage | 단순하고 일상적인 과제에서 소통; 배경 설명 가능 |
| **B1** | Threshold | 대부분의 여행 상황 처리 가능; 경험, 희망, 의견 설명 가능 |
| **B2** | Vantage | 원어민과 유창하게 상호작용; 다양한 주제에 대해 상세한 텍스트 작성 가능 |
| **C1** | Effective Operational Proficiency | 유창하고 즉흥적으로 의견 표현; 전문적, 학술적 목적으로 언어를 유연하게 사용 |
| **C2** | Mastery | 거의 모든 것을 이해; 정확하고 즉흥적으로 표현 |

### 독일 대학이 요구하는 것

대부분의 독일 대학은 다음 중 하나를 요구합니다:
- **TestDaF**: 4개 영역 모두 TDN 4 (일부 프로그램은 한 영역에서 TDN 3 허용)
- **Goethe-Zertifikat C1** (이전 ZMP) 또는 **C2** (이전 ZOP)
- **DSH** (Deutsche Sprachprüfung für den Hochschulzugang): 레벨 DSH-1, DSH-2, DSH-3
- **telc Deutsch C1 Hochschule**

## 3. Goethe-Zertifikat: 구조와 레벨

Goethe-Institut은 CEFR 6개 레벨 모두에서 시험을 제공합니다:

| 시험 | CEFR | 시간 | 영역 | 합격 점수 |
|------|------|------|------|-----------|
| Goethe-Zertifikat A1: Start Deutsch 1 | A1 | ~80분 | Lesen, Hören, Schreiben, Sprechen | 총 60% |
| Goethe-Zertifikat A2 | A2 | ~80분 | Lesen, Hören, Schreiben, Sprechen | 총 60% |
| Goethe-Zertifikat B1 | B1 | ~190분 | Lesen, Hören, Schreiben, Sprechen | 모듈별 60% |
| Goethe-Zertifikat B2 | B2 | ~250분 | Lesen, Hören, Schreiben, Sprechen | 모듈별 60% |
| Goethe-Zertifikat C1 | C1 | ~190분 | Lesen, Hören, Schreiben, Sprechen | 모듈별 60% |
| Goethe-Zertifikat C2: GDS | C2 | ~240분 | Lesen, Hören, Schreiben, Sprechen | 모듈별 60% |

> **중요**: B1부터 각 모듈 (Lesen, Hören, Schreiben, Sprechen)을 개별적으로 60% 이상 통과해야 합니다. 강한 영역으로 약한 영역을 보충할 수 없습니다.

## 4. Goethe B1 시험 상세

### Lesen (읽기) — 65분

| 과제 | 유형 | 문항 수 | 내용 |
|------|------|---------|------|
| Teil 1 | 블로그/포럼 게시물 | 6문항 | 관심사/필요에 따라 사람을 텍스트에 매칭 |
| Teil 2 | 신문 기사 | 6문항 | 객관식 (문항당 3개 선택지) |
| Teil 3 | 규정/규칙 | 7문항 | 참 (Richtig) 또는 거짓 (Falsch) |
| Teil 4 | 독자 편지/의견 | 7문항 | 진술을 올바른 사람에게 매칭 |
| Teil 5 | 지침/안내서 | 4문항 | 객관식 (3개 선택지) |

### Hören (듣기) — 40분

| 과제 | 유형 | 문항 수 | 내용 |
|------|------|---------|------|
| Teil 1 | 짧은 대화 | 10문항 | 참 (Richtig) 또는 거짓 (Falsch) |
| Teil 2 | 라디오 보도/인터뷰 | 5문항 | 객관식 (3개 선택지) |
| Teil 3 | 대화 | 7문항 | 참 (Richtig) 또는 거짓 (Falsch) |
| Teil 4 | 토론/논쟁 | 8문항 | 의견을 발화자에게 매칭 |

### Schreiben (쓰기) — 60분

| 과제 | 유형 | 내용 |
|------|------|------|
| Teil 1 | 비공식 또는 준공식 이메일 | 상황에 대응하여 약 80단어 작성; 3가지 구체적 요점 다루기 |
| Teil 2 | 토론 포럼 게시물 | 약 80단어 작성; 주어진 주제에 대한 의견 표현 |
| Teil 3 | 공식 이메일 | 약 40단어 작성; 짧은 공식 요청 또는 답변 |

### Sprechen (말하기) — 15분 (2인 시험)

| 과제 | 유형 | 내용 |
|------|------|------|
| Teil 1 | 함께 계획하기 | 파트너와 함께 무언가를 토론하고 계획하기 (예: 행사 조직) |
| Teil 2 | 프레젠테이션 | 주어진 주제에 대해 짧은 프레젠테이션 (~3-4분) 발표; 파트너가 질문 |
| Teil 3 | 피드백과 질문 | 파트너의 프레젠테이션에 질문과 피드백으로 응답 |

## 6. TestDaF 시험 상세

**TestDaF** (Test Deutsch als Fremdsprache)는 학술적 독일어 능력을 위해 특별히 설계된 시험입니다.

### 채점: TDN 레벨

각 영역은 3단계 척도로 독립적으로 채점됩니다:

| TDN 레벨 | 대응 레벨 | 의미 |
|----------|-----------|------|
| **TDN 5** | C1 | 최고 레벨; 대학 요건 초과 |
| **TDN 4** | B2-C1 | 대부분의 독일 대학의 표준 요건 |
| **TDN 3** | B2 | 표준 이하; 일부 대학은 조건부로 수용 |
| TDN 3 미만 | B2 이하 | 대학 입학에 불충분 |

> 대부분의 대학은 **4개 영역 모두 TDN 4**를 요구합니다 (흔히 "4x4"로 표기).

### Leseverstehen (독해) — 60분

| 과제 | 유형 | 문항 수 | 내용 |
|------|------|---------|------|
| Aufgabe 1 | 짧은 텍스트 + 매칭 | 10문항 | 10개의 짧은 설명을 텍스트에 매칭 (예: 강좌 목록, 행사) |
| Aufgabe 2 | 저널리즘 텍스트 | 10문항 | 객관식 또는 참/거짓/언급 없음 |
| Aufgabe 3 | 학술/과학 텍스트 | 10문항 | 빈칸 채우기 또는 복잡한 질문에 답변 매칭 |

### Hörverstehen (청해) — 40분

| 과제 | 유형 | 문항 수 | 내용 |
|------|------|---------|------|
| Aufgabe 1 | 일상 대화 (캠퍼스) | 8문항 | 단답형 또는 메모 |
| Aufgabe 2 | 인터뷰/토론 | 10문항 | 참/거짓/언급 없음 |
| Aufgabe 3 | 강의/학술 발표 | 7문항 | 단답형 또는 메모; 1회 청취 |

### Schriftlicher Ausdruck (서면 표현) — 60분

**하나의 과제**: 학술 주제에 대해 체계적이고 논증적인 텍스트 (~최소 250단어) 작성.

프롬프트에 제공되는 것:
- 주제/질문 (예: "대학 교육이 무료여야 하는가?")
- 포함할 데이터가 있는 그래프 또는 차트
- 토론할 두 가지 이상의 입장

해야 할 것:
1. 그래프/차트의 데이터 설명
2. 찬성과 반대 논거 제시
3. 본국의 상황과 비교
4. 자신의 입장 표명

### Mündlicher Ausdruck (구술 표현) — 35분

이것은 **컴퓨터 기반** 말하기 시험입니다 (대면이 아님):

| 과제 | 유형 | 준비/발화 시간 | 내용 |
|------|------|----------------|------|
| 1 | 캠퍼스 상황 | 30초 준비 / 30초 발화 | 정보를 얻기 위해 전화하기 |
| 2 | 캠퍼스 상황 | 60초 준비 / 60초 발화 | 조언을 주거나 정보 공유하기 |
| 3 | 그림 묘사 | 60초 준비 / 60초 발화 | 그래프 또는 차트 묘사하기 |
| 4 | 입장 표명 | 60초 준비 / 90초 발화 | 의견을 표명하고 정당화하기 |
| 5 | 장단점 | 60초 준비 / 90초 발화 | 장점과 단점 토론하기 |
| 6 | 가상 상황 | 60초 준비 / 90초 발화 | 특정 상황에서 어떻게 할 것인지 토론 |
| 7 | 추천 | 60초 준비 / 90초 발화 | 추천하고 이유 설명하기 |

> **참고**: 컴퓨터 기반이므로 마이크에 대고 말합니다. 대화 상대가 없습니다. 프롬프트를 듣고 할당된 시간 내에 응답해야 합니다.

## 7. 영역별 전략

### 읽기 전략 (Lesen / Leseverstehen)

**시간 관리**:
- 전체 텍스트를 먼저 읽지 마세요 — 질문을 먼저 읽고 텍스트를 훑어보세요
- 매칭 과제의 경우: 결정하기 전에 모든 선택지를 읽으세요
- 객관식의 경우: 명백히 틀린 답부터 제거하세요
- 과제당 시간을 분배하세요; 하나의 질문에 너무 오래 머물지 마세요

**소거법**:
1. 질문을 주의 깊게 읽기
2. 모든 답변 선택지 읽기
3. 명백히 틀린 답 제거
4. 남은 선택지 사이에서 텍스트 구절 다시 확인
5. 텍스트에 의해 가장 직접적으로 뒷받침되는 답 선택

**키워드 스캐닝**:
- 질문의 핵심 단어에 밑줄 긋기
- 텍스트에서 그 단어 또는 동의어 검색
- 정답은 보통 텍스트의 **의역**이지 단어 그대로의 복사가 아닙니다

> **함정 주의**: 답변 선택지가 텍스트와 정확히 같은 단어를 사용한다면 주의하세요 — 유인책(distractor)일 수 있습니다. 시험 출제자는 종종 정답을 바꿔 표현합니다.

### 쓰기 전략 (Schreiben / Schriftlicher Ausdruck)

**논증적 글쓰기를 위한 구조 템플릿**:

```
+----------------------------------------------------------+
|  WRITING TEMPLATE (250+ words)                            |
+----------------------------------------------------------+
|                                                          |
|  Einleitung (2-3 sentences):                             |
|  - Introduce the topic                                   |
|  - State the central question                            |
|  - "Im Folgenden möchte ich... erörtern."               |
|                                                          |
|  Grafikbeschreibung (if applicable, 3-4 sentences):      |
|  - "Die Grafik zeigt..."                                 |
|  - Describe main trends, numbers                         |
|  - "Auffällig ist, dass..."                              |
|                                                          |
|  Pro-Argumente (3-4 sentences):                          |
|  - "Dafür spricht, dass..."                              |
|  - "Ein weiterer Vorteil ist..."                         |
|                                                          |
|  Contra-Argumente (3-4 sentences):                       |
|  - "Dagegen spricht, dass..."                            |
|  - "Andererseits muss man bedenken, dass..."             |
|                                                          |
|  Situation im Heimatland (2-3 sentences):                |
|  - "In meinem Heimatland..."                             |
|  - "Im Vergleich zu Deutschland..."                      |
|                                                          |
|  Schluss (2-3 sentences):                                |
|  - "Zusammenfassend lässt sich sagen, dass..."           |
|  - State your own position                               |
|  - "Meiner Meinung nach..."                              |
+----------------------------------------------------------+
```

**작문용 전환 표현**:

| 기능 | 표현 |

**Digitales Lernen: Revolution oder Illusion?**

Die Corona-Pandemie hat die Digitalisierung des Bildungswesens drastisch beschleunigt. Innerhalb weniger Wochen mussten Schulen und Universitäten auf Online-Unterricht umstellen. Was zunächst als Notlösung galt, hat sich in vielen Bereichen als dauerhafte Alternative etabliert.

Befürworter des digitalen Lernens betonen die Flexibilität: Studierende können im eigenen Tempo lernen, Vorlesungen wiederholt ansehen und von überall auf der Welt teilnehmen. Außerdem ermöglichen digitale Werkzeuge wie interaktive Quizze und Lernplattformen eine individuellere Betreuung.

Kritiker hingegen warnen vor den Schattenseiten. Die soziale Interaktion zwischen Lernenden und Lehrenden leide erheblich, was sich negativ auf die Motivation und das psychische Wohlbefinden auswirke. Zudem verstärke digitales Lernen bestehende Ungleichheiten: Nicht alle Familien verfügen über die nötige technische Ausstattung.

Eine aktuelle Studie der Universität München zeigt, dass 62% der befragten Studierenden eine Mischung aus Präsenz- und Online-Unterricht bevorzugen. Nur 15% wünschen sich rein digitalen Unterricht, während 23% den traditionellen Präsenzunterricht vorziehen.

## 9. 쓰기 샘플 문제

### TestDaF 스타일 — Schriftlicher Ausdruck

**주제**: *Sollten Studierende neben dem Studium arbeiten?* (학생들이 학업과 병행하여 일해야 하는가?)

**제공된 데이터**: 설문 조사에 따르면 독일 학생의 68%가 아르바이트를 합니다: 40%는 경제적 이유, 18%는 경력 경험, 10%는 개인 발전을 위해. 평균 근무 시간은 주 12시간입니다. 주 15시간 이상 일하는 학생은 표준 수학 기간 (*Regelstudienzeit*)을 초과할 확률이 두 배입니다.

**과제**: 다음을 포함하는 논증적 텍스트 (최소 250단어)를 작성하세요:
1. 데이터 설명
2. 학업 중 일하는 것에 대한 찬반 논거 제시
3. 본국의 상황과 비교
4. 자신의 입장 표명

<details>
<summary>모범 답안 개요</summary>

**Einleitung**:
*Das Thema, ob Studierende neben dem Studium arbeiten sollten, ist seit langem Gegenstand einer kontroversen Debatte. Im Folgenden möchte ich diese Frage anhand aktueller Daten und verschiedener Argumente erörtern.*

**Grafikbeschreibung**:
*Die vorliegenden Daten zeigen, dass 68% der deutschen Studierenden neben dem Studium arbeiten. Der Hauptgrund ist finanzieller Natur (40%), gefolgt von dem Wunsch nach Berufserfahrung (18%) und persönlicher Weiterentwicklung (10%). Auffällig ist, dass Studierende, die mehr als 15 Stunden pro Woche arbeiten, doppelt so häufig die Regelstudienzeit überschreiten.*

**Pro-Argumente**:
*Dafür spricht zunächst, dass viele Studierende auf das Einkommen angewiesen sind, um ihren Lebensunterhalt zu finanzieren. Darüber hinaus sammeln Studierende durch Nebenjobs wertvolle Berufserfahrung, die ihnen beim späteren Berufseinstieg hilft.*

**Contra-Argumente**:
*Andererseits zeigen die Daten, dass zu viel Arbeit das Studium verlängert. Außerdem kann der Doppelstress aus Arbeit und Studium zu Erschöpfung und schlechteren Noten führen.*

**Vergleich mit dem Heimatland**:
*In meinem Heimatland ist die Situation ähnlich / anders, da...*

**Schluss**:
*Zusammenfassend bin ich der Ansicht, dass Nebenjobs grundsätzlich positiv sind, solange sie 15 Stunden pro Woche nicht überschreiten. Ein gesundes Gleichgewicht zwischen Arbeit und Studium ist der Schlüssel zum Erfolg.*

</details>

## 10. 말하기 샘플 문제

### TestDaF 스타일 — Aufgabe 4 (입장 표명)

**프롬프트**: 다음을 듣습니다: *"An vielen Universitäten wird darüber diskutiert, ob die Anwesenheitspflicht bei Vorlesungen abgeschafft werden sollte. Nehmen Sie Stellung dazu."*

**준비 시간**: 60초
**발화 시간**: 90초

**이 과제에 유용한 표현**:

| 단계 | 표현 |
|------|------|
| 도입 | *Ich möchte zu dieser Frage Stellung nehmen.* / *Meiner Meinung nach...* |
| 논거 1 | *Erstens denke ich, dass... / Ein wichtiger Punkt ist...* |

<details>
<summary>모범 답안</summary>

*Ich möchte zu dieser Frage Stellung nehmen. Meiner Meinung nach sollte die Anwesenheitspflicht bei Vorlesungen abgeschafft werden, und zwar aus mehreren Gründen.*

*Erstens sind Studierende erwachsene Menschen, die selbst entscheiden können, wie sie am besten lernen. Manche Studierende lernen effektiver mit Büchern oder Online-Materialien als in einer überfüllten Vorlesung.*

*Insgesamt bin ich jedoch der Meinung, dass die Vorteile überwiegen. Die Qualität der Lehre sollte so hoch sein, dass Studierende freiwillig kommen — nicht weil sie müssen, sondern weil sie wollen.*

</details>

### 연습 2: 읽기 — 참, 거짓, 또는 언급 없음?

8절의 "Digitales Lernen" 텍스트를 바탕으로 결정하세요: **Richtig (R)**, **Falsch (F)**, 또는 **Steht nicht im Text (N)**:

2. 디지털 학습 도구는 학생에 대한 더 개별적인 지원을 가능하게 한다.
3. 디지털 학습은 전통적 교실 수업보다 비용이 더 든다.
4. 설문 대상 학생의 62%가 순수 온라인 수업을 선호한다.
5. 비평가들은 디지털 학습이 동기 부여와 정신 건강에 해를 끼칠 수 있다고 말한다.

<details>
<summary>정답</summary>

1. **Falsch** — *innerhalb weniger Wochen* (몇 주 이내에) 이루어졌지, 점진적으로가 아닙니다.
2. **Richtig** — 텍스트에서 디지털 도구가 *individuellere Betreuung ermöglichen*이라고 합니다.
3. **Steht nicht im Text** — 텍스트에서 비용을 언급하지 않습니다.
4. **Falsch** — 62%는 *Mischung* (조합)을 선호합니다, 순수 온라인이 아닙니다. 순수 온라인을 원하는 것은 15%만입니다.
5. **Richtig** — 텍스트에서 사회적 상호작용이 줄어들어 *sich negativ auf die Motivation und das psychische Wohlbefinden auswirke*라고 합니다.

</details>

### 연습 3: 데이터 묘사하기

다음 데이터를 독일어로 3-4문장으로 묘사하세요:

> **설문 조사: "독일인들은 여가 시간에 무엇을 하는가?" (2024)**
> - Sport treiben: 45%
> - Fernsehen: 38%
> - Lesen: 29%
> - Freunde treffen: 52%
> - Gartenarbeit: 18%

<details>
<summary>모범 답안</summary>

*Die Umfrage zeigt, wie Deutsche ihre Freizeit verbringen. Die beliebteste Aktivität ist das Treffen von Freunden mit 52%, gefolgt von Sport mit 45%. Auffällig ist, dass Fernsehen (38%) beliebter als Lesen (29%) ist. Die am wenigsten verbreitete Aktivität ist Gartenarbeit mit nur 18%.*

</details>

### 연습 4: 말하기 — 구조화된 응답

다음 프롬프트에 응답하세요. 90초 동안 말하세요 (또는 응답을 적으세요):

*"Heutzutage nutzen immer mehr Menschen soziale Medien. Manche sagen, soziale Medien verbinden die Menschen, andere meinen, sie isolieren sie. Nehmen Sie Stellung."*

(오늘날 점점 더 많은 사람들이 소셜 미디어를 사용합니다. 어떤 사람들은 소셜 미디어가 사람들을 연결한다고 말하고, 다른 사람들은 고립시킨다고 말합니다. 입장을 밝히세요.)

<details>
<summary>모범 답안</summary>

*Ich möchte zu dieser Frage Stellung nehmen. Es stimmt, dass soziale Medien unser Leben stark verändert haben, und ich vertrete die Ansicht, dass sie sowohl verbindende als auch isolierende Effekte haben.*

*Einerseits ermöglichen soziale Medien den Kontakt zu Menschen auf der ganzen Welt. Man kann mit Freunden und Familie in Verbindung bleiben, auch wenn man weit entfernt lebt. Darüber hinaus können soziale Medien Menschen mit gemeinsamen Interessen zusammenbringen.*

*Andererseits zeigen Studien, dass exzessive Nutzung sozialer Medien zu Einsamkeit führen kann. Viele Menschen ersetzen echte persönliche Gespräche durch oberflächliche Online-Interaktionen. Außerdem kann der ständige Vergleich mit anderen auf Plattformen wie Instagram das Selbstwertgefühl negativ beeinflussen.*

*Insgesamt denke ich, dass soziale Medien ein Werkzeug sind — es kommt darauf an, wie man sie nutzt. Ein bewusster und maßvoller Umgang ist der Schlüssel.*

</details>

### 연습 5: 시험 어휘

각 시험 용어를 영어 번역과 매칭하세요:

1. Leseverstehen → ________
2. Hörverstehen → ________
3. Schriftlicher Ausdruck → ________
4. Mündlicher Ausdruck → ________
5. Prüfungsergebnis → ________
6. Modellsatz → ________
7. Anmeldefrist → ________
8. Teilnehmer → ________

선택지: a) written expression, b) registration deadline, c) reading comprehension, d) oral expression, e) participant, f) practice exam/model test, g) exam result, h) listening comprehension

<details>
<summary>정답</summary>

1. Leseverstehen → **c) reading comprehension**
2. Hörverstehen → **h) listening comprehension**
3. Schriftlicher Ausdruck → **a) written expression**
4. Mündlicher Ausdruck → **d) oral expression**
5. Prüfungsergebnis → **g) exam result**
6. Modellsatz → **f) practice exam/model test**
7. Anmeldefrist → **b) registration deadline**
8. Teilnehmer → **e) participant**

</details>

### 연습 6: 오류 유형 식별하기

각 문장에는 쓰기 시험에서 감점될 오류가 있습니다. 오류 유형 (어순, 격, 철자, 쉼표, 또는 어조)을 식별하고 수정하세요:

1. *Ich denke das die Universität mehr Kurse anbieten sollte.*
2. *Wegen dem schlechten Wetter blieben alle zu Hause.*
3. *Meiner Meinung nach das ist ein großes Problem.*
4. *Die Studie zeigt dass viele Studierende neben dem Studium jobben.*
5. *Hey, ich wollte fragen ob Sie mir die Unterlagen schicken können.* (공식 이메일)

<details>
<summary>정답</summary>

1. **쉼표 오류**: *Ich denke**,** dass die Universität mehr Kurse anbieten sollte.* (*dass* 앞에 쉼표)
2. **격 오류**: *Wegen **des** schlechten Wetters blieben alle zu Hause.* (*wegen*은 2격(속격)을 취하며, 3격(여격)이 아닙니다. 참고: *wegen dem*은 구어에서 흔하지만 공식 문서에서는 틀립니다.)
3. **어순 오류**: *Meiner Meinung nach **ist das** ein großes Problem.* (*Meiner Meinung nach* 뒤에 동사가 두 번째 위치에 와야 함)
4. **쉼표 오류**: *Die Studie zeigt**,** dass viele Studierende neben dem Studium jobben.* (*dass* 앞에 쉼표)
5. **어조 오류**: *Sehr geehrte Damen und Herren, ich möchte Sie bitten, mir die Unterlagen zuzusenden.* ("Hey"는 공식 이메일에서 부적절함)

</details>
