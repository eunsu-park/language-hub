## 1. 비즈니스에서의 공식 vs. 비공식

### Siezen vs. Duzen

독일 비즈니스 생활에서 가장 중요한 구분 중 하나는 **Sie** (공식적 "당신")와 **du** (비공식적 "너")를 언제 사용하느냐입니다.

| | **Siezen** (Sie) | **Duzen** (du) |
|---|---|---|
| 사용 시점 | 비즈니스, 공식적 상황에서 기본 | 명시적 제안 이후, 캐주얼한 회사 |
| 대상 | 고객, 상사, 낯선 사람, 공무원 | 친밀한 동료, "du" 제안을 받은 후 |
| 표현 예시 | *Können Sie mir helfen?* | *Kannst du mir helfen?* |
| 서면 형태 | 항상 대문자: *Sie*, *Ihnen*, *Ihr* | 보통 소문자: *du*, *dir*, *dein* |

### Du-Angebot ("du" 사용 제안)

*Sie*에서 *du*로의 전환은 의도적인 사회적 행위입니다:

- **직급이 높은** 사람이 낮은 사람에게 *du*를 제안합니다
- **나이가 많은** 사람이 젊은 사람에게 제안합니다
- 동료 간에는 상호적일 수 있습니다
- 한번 제안하고 수락하면 다시 *Sie*로 돌아갈 수 없습니다

> **중요**: 비즈니스 환경에서 *du*를 절대 먼저 사용하지 마세요. 제안받을 때까지 기다리세요. *du*를 먼저 사용하는 것은 무례한 것으로 간주됩니다.

### 현대적 추세

많은 현대 기업, 특히 기술 및 스타트업 분야에서는 *du*를 기본으로 사용합니다 (*Duz-Kultur*):
- IKEA 독일은 모든 고객에게 *du*를 사용합니다
- 많은 스타트업이 *du*를 회사 방침으로 채택합니다
- 전통적 산업 (법률, 은행, 정부)은 여전히 *Sie*를 강하게 선호합니다

## 2. 입사 지원: Bewerbung

독일의 입사 지원서 (*Bewerbung*)는 여러 문서로 구성됩니다:

```
+----------------------------------------------------------+
|  BEWERBUNGSUNTERLAGEN (Application Documents)             |
+----------------------------------------------------------+
|                                                          |
|  1. Anschreiben (Cover letter)                           |
|     → One page, tailored to the specific job              |
|                                                          |
|  2. Lebenslauf (CV / Résumé)                             |
|     → Tabular format, reverse chronological               |
|     → Usually includes a photo (Bewerbungsfoto)           |
|                                                          |
|  3. Zeugnisse (References / Certificates)                |
|     → School diplomas, university degrees                 |
|     → Arbeitszeugnisse (employer references)              |
|     → Training certificates                               |
|                                                          |
|  4. Optional: Arbeitsproben (Work samples)               |
+----------------------------------------------------------+
```

### Lebenslauf (이력서) — 독일 스타일

독일 이력서는 미국/영국 형식과 다릅니다:

| 특징 | 독일 이력서 | 미국/영국 이력서 |
|------|-------------|------------------|
| 사진 | **예** — 전문적 두상 사진 (일반적) | 아니오 (차별 방지) |
| 개인 정보 | 생년월일, 국적, 결혼 여부 (선택적이지만 일반적) | 포함하지 않음 |
| 형식 | **Tabellarisch** (표 형식, 2열) | 문단 또는 글머리 기호 |
| 순서 | **역연대순** (최신 먼저) | 역연대순 |
| 길이 | 1-2 페이지 | 1 페이지 (미국) / 2 페이지 (영국) |
| 서명 | **예** — 하단에 서명 및 날짜 | 아니오 |

### Lebenslauf 구조

| 섹션 | 독일어 | 내용 |
|------|--------|------|
| 개인 정보 | **Persönliche Daten** | 이름, 주소, 전화번호, 이메일, 생년월일/출생지 |
| 경력 | **Berufserfahrung** | 역연대순 경력 사항 |
| 학력 | **Ausbildung / Studium** | 학위, 학교, 날짜 |
| 기술 | **Kenntnisse und Fähigkeiten** | 언어, IT 기술, 자격증 |
| 관심사 | **Interessen / Hobbys** | 선택 사항, 전문적으로 유지 |

### Anschreiben (자기소개서) 구조

자기소개서는 다음 형식을 따라야 합니다:

```
+----------------------------------------------------------+
|  ANSCHREIBEN STRUCTURE                                    |
+----------------------------------------------------------+
|                                                          |
|  Absender (your contact info)                            |
|  Empfänger (company address)                             |
|  Ort, Datum (place, date)                                |
|                                                          |
|  Betreff: Bewerbung als [Position]                       |
|                                                          |
|  Sehr geehrte/r [Name],                                  |
|                                                          |
|  Paragraph 1: WHY this company/position?                 |
|  → How did you find the job?                             |
|  → What attracts you to this role?                       |
|                                                          |
|  Paragraph 2: WHO are you?                               |
|  → Your qualifications and experience                    |
|  → Relevant skills and achievements                      |
|                                                          |
|  Paragraph 3: WHAT can you offer?                        |
|  → How will you contribute to the company?               |
|  → Match your skills to the job requirements              |
|                                                          |
|  Closing: Availability, interview request                |
|                                                          |
|  Mit freundlichen Grüßen                                 |
|  [Signature]                                             |
|  [Name]                                                  |
|  Anlagen: Lebenslauf, Zeugnisse                          |
+----------------------------------------------------------+
```

### 유용한 자기소개서 문구

| 독일어 | 영어 |
|--------|------|
| Mit großem Interesse habe ich Ihre Stellenanzeige gelesen. | I read your job advertisement with great interest. |
| Hiermit bewerbe ich mich um die Stelle als... | I hereby apply for the position of... |
| In meiner bisherigen Tätigkeit als... konnte ich umfangreiche Erfahrung sammeln. | In my previous role as... I was able to gain extensive experience. |
| Ich bringe fundierte Kenntnisse in ... mit. | I bring solid knowledge in ... |
| Meine Stärken liegen in... | My strengths lie in... |
| Über eine Einladung zu einem persönlichen Gespräch freue ich mich sehr. | I would be delighted to receive an invitation for a personal interview. |
| Ich stehe Ihnen ab dem [Datum] zur Verfügung. | I am available from [date]. |
| Meine Gehaltsvorstellung liegt bei ... Euro brutto pro Jahr. | My salary expectation is ... euros gross per year. |

## 3. 면접: Vorstellungsgespräch

### 일반적인 면접 질문과 모범 답변

**질문 1**: *Erzählen Sie uns etwas über sich.* (자기소개를 해주세요.)

> *Gerne. Mein Name ist Anna Schmidt. Ich bin 28 Jahre alt und habe an der TU Berlin Informatik studiert. Nach meinem Masterabschluss habe ich drei Jahre als Softwareentwicklerin bei der Firma XY gearbeitet, wo ich vor allem an Webapplikationen mit Python und JavaScript gearbeitet habe.*

**질문 2**: *Warum möchten Sie bei uns arbeiten?* (왜 저희 회사에서 일하고 싶으신가요?)

> *Ihr Unternehmen ist bekannt für seine innovativen Produkte im Bereich Künstliche Intelligenz. Ich bin von Ihrer Unternehmenskultur beeindruckt und sehe hier die Möglichkeit, meine Fähigkeiten in einem dynamischen Umfeld weiterzuentwickeln.*

**질문 3**: *Was sind Ihre Stärken?* (당신의 강점은 무엇인가요?)

> *Zu meinen Stärken zähle ich meine analytische Denkweise, meine Teamfähigkeit und meine Bereitschaft, mich in neue Themen einzuarbeiten. In meiner letzten Position habe ich erfolgreich ein Team von fünf Personen geleitet.*

**질문 4**: *Was sind Ihre Schwächen?* (당신의 약점은 무엇인가요?)

> *Ich neige manchmal dazu, zu perfektionistisch zu sein und zu viel Zeit auf Details zu verwenden. Ich habe jedoch gelernt, Prioritäten besser zu setzen und mich auf das Wesentliche zu konzentrieren.*

**질문 5**: *Wo sehen Sie sich in fünf Jahren?* (5년 후 어디에 있을 거라고 보시나요?)

> *In fünf Jahren möchte ich eine leitende Position in der Softwareentwicklung übernehmen und an strategischen Entscheidungen mitwirken. Ich möchte mich fachlich und persönlich weiterentwickeln.*

## 5. 회의: Besprechungen

### 회의 구조

```
+----------------------------------------------------------+
|  MEETING STRUCTURE                                        |
+----------------------------------------------------------+
|                                                          |
|  1. Begrüßung (Welcome)                                 |
|     "Guten Morgen, vielen Dank, dass Sie alle da sind."  |
|                                                          |
|  2. Tagesordnung (Agenda)                                |
|     "Auf der Tagesordnung stehen heute drei Punkte..."   |
|                                                          |
|  3. Diskussion (Discussion)                              |
|     Each agenda point is discussed                       |
|                                                          |
|  4. Zusammenfassung (Summary)                            |
|     "Fassen wir zusammen..."                             |
|                                                          |
|  5. Nächste Schritte (Next Steps)                        |
|     "Bis nächste Woche sollten wir..."                   |
|                                                          |
|  6. Verabschiedung (Closing)                             |
|     "Vielen Dank für Ihre Zeit."                         |
+----------------------------------------------------------+
```

### 의견 제시

| 독일어 | 영어 |
|--------|------|
| Meiner Meinung nach... | In my opinion... |
| Ich bin der Ansicht, dass... | I am of the view that... |
| Ich denke / glaube / finde, dass... | I think / believe / find that... |
| Aus meiner Sicht... | From my perspective... |
| Wenn ich mich nicht irre... | If I am not mistaken... |

### 동의

| 독일어 | 영어 |
|--------|------|
| Ich stimme Ihnen / dir zu. | I agree with you. |
| Da haben Sie völlig Recht. | You are absolutely right. |
| Das sehe ich genauso. | I see it exactly the same way. |
| Das ist ein guter Punkt. | That is a good point. |
| Einverstanden. | Agreed. |

### 정중한 반대

| 독일어 | 영어 |
|--------|------|
| Da bin ich anderer Meinung. | I have a different opinion on that. |
| Ich sehe das etwas anders. | I see that a bit differently. |
| Das kann ich so nicht bestätigen. | I cannot confirm that. |
| Mit Verlaub, ich sehe das anders. | With respect, I see it differently. |
| Ich verstehe Ihren Punkt, aber... | I understand your point, but... |

### 명확한 설명 요청

| 독일어 | 영어 |
|--------|------|
| Könnten Sie das bitte genauer erklären? | Could you please explain that in more detail? |
| Was meinen Sie genau damit? | What exactly do you mean by that? |
| Habe ich Sie richtig verstanden, dass...? | Did I understand you correctly that...? |
| Können Sie ein Beispiel geben? | Can you give an example? |
| Entschuldigung, das habe ich nicht ganz verstanden. | Excuse me, I did not quite understand that. |

## 6. 비즈니스 이메일 에티켓

### 이메일 구조

| 요소 | 독일어 | 예시 |
|------|--------|------|
| 제목줄 | **Betreff** | *Betreff: Besprechung am 10. März* |

| Wie besprochen, sende ich Ihnen... | As discussed, I am sending you... |

**요청**:

| 독일어 | 영어 |
|--------|------|
| Könnten Sie mir bitte ... zusenden? | Could you please send me ...? |

| Wären Sie so freundlich, ... | Would you be so kind as to ... |
| Bitte teilen Sie mir mit, ob... | Please let me know whether... |

**정보 제공**:

| 독일어 | 영어 |

**사과**:

| 독일어 | 영어 |

## 7. 전화 대화

### 전화 받기

**비즈니스**: 회사명과 자신의 이름을 밝히세요:
- *Firma DataTech, Schmidt am Apparat. Guten Tag!*
- *DataTech GmbH, Sie sprechen mit Frau Müller. Was kann ich für Sie tun?*

**개인**: 성만 밝히세요:
- *Schmidt.* (매우 일반적, 성만)
- *Hier ist Schmidt.*

### 주요 전화 표현

**자기 소개**:

| 독일어 | 영어 |
|--------|------|
| Hier spricht [Name] von [Firma]. | This is [Name] from [Company]. |
| Mein Name ist [Name], ich rufe an wegen... | My name is [Name], I am calling about... |
| Guten Tag, hier ist [Name]. | Hello, this is [Name]. |

**통화 요청**:

| 독일어 | 영어 |
|--------|------|
| Könnte ich bitte mit Herrn/Frau [Name] sprechen? | Could I please speak with Mr./Mrs. [Name]? |
| Könnten Sie mich bitte mit der [Abteilung] verbinden? | Could you please connect me with the [department]? |
| Ist Herr/Frau [Name] zu sprechen? | Is Mr./Mrs. [Name] available? |

**상대방이 부재중인 경우**:

| 독일어 | 영어 |
|--------|------|
| Es tut mir leid, Herr/Frau [Name] ist gerade nicht erreichbar. | I am sorry, Mr./Mrs. [Name] is not available right now. |
| Er/Sie ist gerade in einer Besprechung. | He/She is currently in a meeting. |
| Möchten Sie eine Nachricht hinterlassen? | Would you like to leave a message? |
| Soll er/sie Sie zurückrufen? | Should he/she call you back? |

**메시지 남기기**:

| 독일어 | 영어 |
|--------|------|
| Könnten Sie ihm/ihr bitte ausrichten, dass... | Could you please tell him/her that... |
| Bitte richten Sie aus, dass ich angerufen habe. | Please pass on that I called. |
| Er/Sie kann mich unter der Nummer ... erreichen. | He/She can reach me at the number ... |
| Ich rufe später noch einmal an. | I will call again later. |

**통화 종료**:

| 독일어 | 영어 |
|--------|------|
| Vielen Dank für Ihre Hilfe. | Thank you for your help. |
| Auf Wiederhören! | Goodbye! (전화에서) |
| Ich wünsche Ihnen einen schönen Tag. | I wish you a nice day. |

> **참고**: 전화에서 독일인들은 *Auf Wiederhören* (직역: "다시 들을 때까지")이라 말하며, *Auf Wiedersehen* (다시 볼 때까지)은 사용하지 않습니다.

### 전화 대화 예시

> **Empfang**: Firma TechStar, guten Tag. Sie sprechen mit Frau Braun. Was kann ich für Sie tun?
>
> **Anrufer**: Guten Tag, Frau Braun. Hier spricht Thomas Weber von Digital Solutions. Könnte ich bitte mit Herrn Müller aus der IT-Abteilung sprechen?
>
> **Empfang**: Einen Moment bitte, ich verbinde Sie... Es tut mir leid, Herr Müller ist gerade in einer Besprechung. Möchten Sie eine Nachricht hinterlassen?
>
> **Anrufer**: Ja, gerne. Könnten Sie ihm bitte ausrichten, dass ich wegen des Angebots angerufen habe? Er kann mich unter 030-98765432 zurückrufen.
>
> **Empfang**: Selbstverständlich, ich richte es ihm aus. Kann ich sonst noch etwas für Sie tun?
>
> **Anrufer**: Nein, das ist alles. Vielen Dank, Frau Braun. Auf Wiederhören!
>
> **Empfang**: Gerne! Auf Wiederhören!

## 8. 프레젠테이션: Präsentationen

### 프레젠테이션 구조

| 단계 | 독일어 용어 | 목적 |
|------|-------------|------|
| 도입 | **Einleitung** | 환영, 자기소개, 주제 제시 |
| 개요 | **Überblick / Gliederung** | 구조 개요 설명 |
| 본론 | **Hauptteil** | 내용, 데이터, 논거 제시 |
| 요약 | **Zusammenfassung** | 핵심 요점 정리 |
| 질의응답 | **Fragerunde** | 청중 질문에 답변 |

### 유용한 프레젠테이션 표현

**도입**:

| 독일어 | 영어 |
|--------|------|
| Guten Morgen / Guten Tag, meine Damen und Herren. | Good morning / Good day, ladies and gentlemen. |
| Ich möchte Ihnen heute ... vorstellen. | I would like to present ... to you today. |
| Das Thema meiner Präsentation ist... | The topic of my presentation is... |
| Mein Vortrag gliedert sich in drei Teile. | My presentation is divided into three parts. |

**전환**:

| 독일어 | 영어 |
|--------|------|
| Kommen wir nun zum nächsten Punkt. | Let us now move to the next point. |
| Ich möchte nun auf ... eingehen. | I would now like to address ... |
| Wie ich bereits erwähnt habe... | As I already mentioned... |
| In diesem Zusammenhang möchte ich betonen, dass... | In this context, I would like to emphasize that... |

**시각 자료 참조**:

| 독일어 | 영어 |
|--------|------|
| Wie Sie auf dieser Folie sehen können... | As you can see on this slide... |
| Die Grafik zeigt... | The graph shows... |
| Wenn Sie sich die Tabelle ansehen... | If you look at the table... |
| Die Zahlen verdeutlichen, dass... | The numbers illustrate that... |

**마무리**:

| 독일어 | 영어 |
|--------|------|
| Zusammenfassend lässt sich sagen, dass... | In summary, it can be said that... |
| Abschließend möchte ich festhalten, dass... | In conclusion, I would like to note that... |
| Vielen Dank für Ihre Aufmerksamkeit. | Thank you for your attention. |
| Haben Sie noch Fragen? | Do you have any questions? |
| Ich beantworte gerne Ihre Fragen. | I am happy to answer your questions. |

## 9. 독일 비즈니스 문화

### 핵심 문화 가치

| 가치 | 독일어 | 설명 |
|------|--------|------|
| 시간 엄수 | **Pünktlichkeit** | 5분 일찍 도착하세요. 늦는 것은 무례한 것입니다. |
| 직접성 | **Direktheit** | 독일인들은 명확하고 직접적으로 소통합니다. 이것은 무례함이 아니라 효율성입니다. |
| 철저함 | **Gründlichkeit** | 업무는 철저하고 잘 준비된 것이 기대됩니다. |
| 신뢰성 | **Zuverlässigkeit** | 무엇을 하겠다고 말하면 반드시 이행해야 합니다. |
| 업무/사생활 분리 | **Trennung Beruf/Privat** | 독일인들은 일반적으로 업무와 사생활을 분리합니다. |

### 비즈니스 에티켓

**악수 (Handschlag)**:
- 만남과 헤어질 때 단단한 악수
- 회의 시작과 끝에 참석자 모두와 악수
- 악수 중 눈을 맞추기

**명함 (Visitenkarte)**:
- 회의 시작 시 명함 교환
- 명함을 치우기 전에 읽기 (존중을 표시)
- 독일에서 학위 칭호 (Dr., Prof.)는 중요하며 명함에 포함하고 호칭에 사용해야 합니다

**칭호와 호칭**:
- *Herr* + 성 또는 *Frau* + 성 사용
- 박사 학위가 있는 경우: *Herr Dr. Schmidt*, *Frau Dr. Müller*
- 교수인 경우: *Herr Professor Schmidt* 또는 *Herr Prof. Dr. Schmidt*

### 일과 삶의 균형

독일 노동법은 직원을 강력하게 보호합니다:

| 항목 | 세부 사항 |
|------|-----------|
| **Urlaub** (휴가) | 법적 최소 연 20일 (많은 기업은 25-30일 제공) |
| **Arbeitszeit** (근무 시간) | 일반적으로 주 35-40시간; 하루 8시간 근무가 표준 |
| **Mittagspause** (점심 휴식) | 보통 30-60분 |
| **Überstunden** (초과 근무) | 법적으로 규제됨; 하루 10시간을 초과할 수 없음 |
| **Krankheitstage** (병가) | 첫날부터 유급 병가; 3일 이후 의사 진단서 필요 |
| **Elternzeit** (육아 휴직) | 부모 각각 최대 3년; 일자리 보호 보장 |
| **Feierabend** | 퇴근 시간; 개인 시간으로 존중됨 |

### 회의 문화

- **안건** (*Tagesordnung*): 회의에는 항상 사전에 배포되는 명확한 안건이 있습니다
- **회의록** (*Protokoll*): 회의 후 회의록을 작성하여 배포합니다
- **결정**: 결정은 집단적으로 이루어지고 문서화됩니다; 개별 행동 항목에는 명확한 담당자와 기한이 있습니다
- **잡담**: 시작 시 짧고 적절한 잡담 (날씨, 이동 중 경험)이 있지만, 회의는 빠르게 본론에 들어갑니다

### 연습 1: Sie 또는 Du?

각 상황에서 *Sie*를 사용해야 할지 *du*를 사용해야 할지 결정하세요:

1. 새로운 고객을 처음 만날 때
2. 같은 날 입사하여 매일 함께 점심을 먹는 동료
3. 파트너 회사의 CEO에게 이메일을 보낼 때
4. 상사가 말합니다: "Wir können uns ruhig duzen."
5. 모든 사람이 이름을 부르는 스타트업에서

<details>
<summary>정답</summary>

1. **Sie** — 새로운 비즈니스 연락처에는 다른 제안을 받을 때까지 항상 Sie를 사용합니다.
2. **상황에 따라** — 가깝고 둘 중 한 명이 du를 제안했다면 du. 그렇지 않으면 명시적으로 du가 제안될 때까지 Sie가 안전합니다.
3. **Sie** — 공식 비즈니스 서신은 항상 Sie를 사용합니다.
4. **Du** — 상사가 Du-Angebot을 했으므로 수락하고 전환합니다.
5. **Du** — 회사 문화가 du이면 따르세요; 하지만 첫날에 확인하세요.

</details>

### 연습 6: 비즈니스 문화 — 참 또는 거짓

1. 독일에서는 비즈니스 회의에 10분 늦게 도착하는 것이 허용됩니다.
2. 독일인들은 일반적으로 업무 생활과 사생활을 분리합니다.
3. 독일 법률상 최소 휴가일수는 연 30일입니다.
4. 비즈니스에서 누군가를 호칭할 때 항상 학위 칭호 (Dr., Prof.)를 사용해야 합니다.
5. "Auf Wiederhören"은 전화에서 작별 인사를 하는 올바른 방법입니다.
6. 독일 회의가 끝나면 보통 Protokoll이 작성됩니다.

<details>
<summary>정답</summary>

1. **거짓** — 시간 엄수는 매우 중요하며, 늦는 것은 무례한 것으로 간주됩니다.
2. **참** — 독일인들은 일반적으로 Beruf(직업)과 Privatleben(사생활)을 명확히 분리합니다.
3. **거짓** — 법적 최소 일수는 20일입니다 (많은 기업이 더 많이 제공하지만).
4. **참** — 독일 비즈니스 문화에서 칭호는 중요하며 사용되어야 합니다.
5. **참** — *Auf Wiederhören* (다시 들을 때까지)은 전화에서, *Auf Wiedersehen* (다시 볼 때까지)은 대면에서 사용합니다.
6. **참** — 회의록 (Protokoll)은 표준 관행입니다.

</details>
