## 1. 독일어 작문 관례

특정 텍스트 유형을 다루기 전에, 독일어 작문의 기본 규칙을 살펴보겠습니다:

### 대문자 표기 (Großschreibung)

| 규칙 | 예시 |
|------|------|
| 모든 명사는 대문자로 시작 | *der Tisch*, *die Freiheit*, *das Auto* |
| 문장 시작 | *Er kommt morgen.* |
| 공식적 "당신" — *Sie/Ihnen/Ihr* | *Können Sie mir helfen?* |
| 편지에서: *Du/Dir/Dein* (선택 사항이지만 일반적) | *Wie geht es Dir?* |
| 명사화된 동사와 형용사 | *das Lesen* (읽기), *der Alte* (노인) |
| 언어, 국적 (명사일 때) | *Er spricht Deutsch.* / *die Deutschen* |

### 에스체트 (ß)와 이중 S (ss)

1996년 철자법 개혁 이후:
- **ß**는 장모음이나 이중모음 뒤에 사용: *Straße*, *groß*, *heißen*
- **ss**는 단모음 뒤에 사용: *dass*, *muss*, *Schloss*
- 스위스에서는 ß를 전혀 사용하지 않고 항상 *ss*를 사용

### 움라우트 (ä, ö, ü)

움라우트는 선택적 장식이 아니라 의미를 변화시킵니다:
- *schon* (이미) vs. *schön* (아름다운)
- *schwul* (동성애의) vs. *schwül* (습한)
- *Mutter* (어머니) vs. *Mütter* (어머니들)

움라우트를 입력할 수 없는 경우 ae, oe, ue로 대체합니다 (예: *München* 대신 *Muenchen*).

## 2. 비공식 편지와 이메일

### 구조

```
+----------------------------------------------------------+
|  INFORMAL LETTER / EMAIL                                  |
+----------------------------------------------------------+
|                                                          |
|  Place, Date           Berlin, den 5. März 2025          |
|                                                          |
|  Greeting              Liebe Anna, / Lieber Max,         |
|                        Hallo Lisa,                       |
|                                                          |
|  Opening               Wie geht es Dir?                  |
|                        Vielen Dank für Deine E-Mail!     |
|                                                          |
|  Body                  [Your message]                    |
|                                                          |
|  Closing               Ich freue mich auf Deine Antwort! |
|                                                          |
|  Sign-off              Liebe Grüße, / Viele Grüße,      |
|                        Bis bald! / Herzliche Grüße,      |
|                                                          |
|  Name                  Dein Thomas / Deine Maria         |
+----------------------------------------------------------+
```

### 핵심 포인트

- **Du/Dein/Dir**: 개인 편지에서 이들은 전통적으로 대문자로 표기합니다 (소문자도 현재 허용됨). 대문자 표기는 존중과 따뜻함을 나타냅니다.
- **인사말 쉼표**: *Liebe Anna,* 뒤에 **쉼표**가 오고, 다음 줄은 **소문자**로 시작합니다 (명사나 *Ich*인 경우 제외).
- **어조**: 대화체, 개인적, 느낌표와 질문을 자주 사용합니다.

### 일반적인 시작 표현

| 독일어 | 영어 |
|--------|------|
| Wie geht es Dir? | How are you? |
| Ich hoffe, Dir geht es gut! | I hope you are well! |
| Vielen Dank für Deine Nachricht! | Thanks for your message! |
| Es tut mir leid, dass ich so lange nicht geschrieben habe. | Sorry I haven't written in so long. |
| Es war schön, von Dir zu hören! | It was nice to hear from you! |

### 일반적인 맺음 표현

| 독일어 | 영어 |
|--------|------|
| Liebe Grüße | With love / Best wishes |
| Viele Grüße | Many greetings / Best regards |
| Herzliche Grüße | Warm regards |
| Bis bald! | See you soon! |
| Schreib mir bald! | Write me soon! |
| Ich freue mich auf Deine Antwort! | I look forward to your reply! |

### 예시: 친구에게 휴가 계획에 대해 쓰기

> Berlin, den 12. Juni 2025
>
> Lieber Jan,
>
> wie geht es Dir? Ich hoffe, Du hattest eine schöne Woche!
>
> Ich schreibe Dir, weil ich schon unseren Sommerurlaub plane. Was hältst Du davon, im August nach Kroatien zu fahren? Meine Freundin war letztes Jahr dort und hat mir wunderschöne Fotos gezeigt. Die Strände sollen fantastisch sein, und es ist viel günstiger als Griechenland.
>
> Wir könnten für eine Woche eine kleine Ferienwohnung mieten und mit dem Auto hinfahren. Was meinst Du? Hast Du im August Zeit?
>
> Schreib mir bald, damit wir planen können!
>
> Viele Grüße,
> Dein Stefan

## 3. 공식 편지와 이메일

### 구조

```
+----------------------------------------------------------+
|  FORMAL LETTER / EMAIL                                    |
+----------------------------------------------------------+
|                                                          |
|  Sender                Anna Schmidt                      |
|  (Absender)            Musterstraße 12                   |
|                        10115 Berlin                      |
|                                                          |
|  Recipient             Firma XY GmbH                     |
|  (Empfänger)           Personalabteilung                 |
|                        Hauptstraße 45                    |
|                        80331 München                     |
|                                                          |
|  Place, Date           Berlin, den 5. März 2025          |
|                                                          |
|  Subject               Betreff: Bewerbung als ...        |
|                                                          |
|  Greeting              Sehr geehrte Damen und Herren,    |
|                        Sehr geehrter Herr Müller,        |
|                        Sehr geehrte Frau Dr. Schmidt,    |
|                                                          |
|  Body                  [Formal message — Sie form]       |
|                                                          |
|  Closing formula       Mit freundlichen Grüßen           |
|                                                          |
|  Signature             Anna Schmidt                      |
|  Attachment note       Anlage: Lebenslauf, Zeugnisse     |
+----------------------------------------------------------+
```

### 핵심 포인트

- **Sie/Ihnen/Ihr**: 공식 문서에서 항상 대문자로 표기합니다. 공식 서신에서는 절대 *du*를 사용하지 마세요.
- *Sehr geehrte Damen und Herren,* 뒤의 다음 줄은 **소문자**로 시작합니다 (비공식과 동일한 쉼표 규칙).
- **Betreff** (제목줄): 간결하게, 끝에 마침표를 붙이지 않습니다.
- **Mit freundlichen Grüßen** — 이 맺음말 뒤에 쉼표를 붙이지 않습니다 (영어와 다름).

### 일반적인 공식 시작 표현

| 독일어 | 영어 |
|--------|------|
| Sehr geehrte Damen und Herren, | Dear Sir or Madam, |
| Sehr geehrter Herr [Name], | Dear Mr. [Name], |
| Sehr geehrte Frau [Name], | Dear Mrs./Ms. [Name], |
| Sehr geehrte Frau Dr. [Name], | Dear Dr. [Name], |

### 일반적인 공식 맺음 표현

| 독일어 | 영어 |
|--------|------|
| Mit freundlichen Grüßen | Yours sincerely / Best regards |
| Mit freundlichem Gruß | Kind regards |
| Hochachtungsvoll | Respectfully yours (매우 격식적) |

### 일반적인 공식 문구

| 독일어 | 영어 |
|--------|------|
| Hiermit bewerbe ich mich um... | I hereby apply for... |
| Bezugnehmend auf Ihre Anzeige... | Referring to your advertisement... |
| Ich möchte Sie darüber informieren, dass... | I would like to inform you that... |
| Für Rückfragen stehe ich Ihnen gerne zur Verfügung. | I am happy to answer any questions. |
| Ich freue mich auf Ihre Rückmeldung. | I look forward to your response. |
| Vielen Dank im Voraus. | Thank you in advance. |

### 예시: 입사 지원서

> Anna Schmidt
> Musterstraße 12
> 10115 Berlin
> Tel: 030-12345678
> E-Mail: a.schmidt@email.de
>
> Firma DataTech GmbH
> Personalabteilung
> z. Hd. Herrn Müller
> Industriestraße 22
> 80331 München
>
> Berlin, den 5. März 2025
>
> **Betreff: Bewerbung als Software-Entwicklerin**
>
> Sehr geehrter Herr Müller,
>
> mit großem Interesse habe ich Ihre Stellenanzeige auf der Website Ihres Unternehmens gelesen. Hiermit bewerbe ich mich um die Position als Software-Entwicklerin in Ihrem Unternehmen.
>
> Ich habe an der Technischen Universität Berlin Informatik studiert und meinen Masterabschluss im September 2024 erhalten. Während meines Studiums habe ich Erfahrung in der Softwareentwicklung mit Python und Java gesammelt und an mehreren Projekten im Bereich Datenanalyse mitgewirkt.
>
> Besonders reizt mich an der ausgeschriebenen Stelle die Möglichkeit, an innovativen Projekten im Bereich Künstliche Intelligenz mitzuarbeiten. Ich bin überzeugt, dass meine Fähigkeiten und meine Leidenschaft für Technologie Ihr Team bereichern werden.
>
> Über eine Einladung zu einem persönlichen Gespräch freue ich mich sehr. Für Rückfragen stehe ich Ihnen jederzeit zur Verfügung.
>
> Mit freundlichen Grüßen
>
> Anna Schmidt
>
> Anlagen: Lebenslauf, Zeugnisse, Arbeitsproben

## 4. 에세이 작문: Aufsatz와 Erörterung

### 일반적인 에세이 구조

잘 쓰인 독일어 에세이는 다음 세 부분 구조를 따릅니다:

| 부분 | 독일어 | 기능 | 대략적 분량 |
|------|--------|------|-------------|
| 서론 | **Einleitung** | 주제 도입, 맥락, 논제 제시 | 10-15% |
| 본론 | **Hauptteil** | 논거, 증거, 예시 | 70-80% |
| 결론 | **Schluss** | 요약, 최종 입장, 전망 | 10-15% |

### Einleitung (서론) 전략

다음 도입부 중 하나로 에세이를 시작하세요:

| 전략 | 예시 |
|------|------|
| 질문 | *Sollte die Schule erst um 9 Uhr beginnen?* |
| 통계 | *Laut einer Studie nutzen 95% aller Jugendlichen soziale Medien.* |
| 인용 | *Goethe sagte einmal: "Es ist nicht genug, zu wissen, man muss auch anwenden."* |
| 도발적 진술 | *Das Smartphone hat die Art, wie wir kommunizieren, grundlegend verändert.* |

그런 다음 논제를 제시하세요:
- *In diesem Aufsatz werde ich argumentieren, dass...*
- *Im Folgenden sollen die Vor- und Nachteile von ... untersucht werden.*

### Erörterung (논증적 에세이)

*Erörterung*는 독일 학교에서 가르치는 전통적인 에세이 형식입니다:

**Lineare Erörterung** (일방적): 하나의 입장을 지지하는 논거를 제시합니다.

**Dialektische Erörterung** (양면적): 찬성과 반대 논거를 모두 제시한 후 자신의 입장을 밝힙니다.

```
+----------------------------------------------------------+
|  DIALEKTISCHE ERÖRTERUNG                                  |
+----------------------------------------------------------+
|                                                          |
|  Einleitung                                              |
|  → Present the topic, give context                       |
|  → State the central question (Leitfrage)                |
|                                                          |
|  Hauptteil                                               |
|  → Pro-Argumente (arguments for)                         |
|    - Argument 1 + Beispiel                               |
|    - Argument 2 + Beispiel                               |
|    - Argument 3 + Beispiel                               |
|  → Contra-Argumente (arguments against)                  |
|    - Argument 1 + Beispiel                               |
|    - Argument 2 + Beispiel                               |
|    - Argument 3 + Beispiel                               |
|                                                          |
|  Schluss                                                 |
|  → Weigh arguments, state your position                  |
|  → Outlook or call to action                             |
+----------------------------------------------------------+
```

> **팁**: Dialektische Erörterung에서는 약한 측을 먼저 제시하고, 동의하는 측을 나중에 제시하면 더 강한 인상을 남깁니다.

### Bildbeschreibung (그림 묘사)

독일어로 이미지를 묘사하는 체계적 접근법:

1. **Allgemeine Beschreibung**: *Auf dem Bild sieht man...* (그림에서 ...을 볼 수 있다)
2. **Vordergrund**: *Im Vordergrund befindet sich...* (전경에 ...이 있다)
3. **Hintergrund**: *Im Hintergrund erkennt man...* (배경에서 ...을 알아볼 수 있다)
4. **Details**: *Auf der linken/rechten Seite...* (왼쪽/오른쪽에...)
5. **Interpretation**: *Das Bild zeigt vermutlich...* (그림은 아마도 ...을 보여준다)
6. **Persönliche Reaktion**: *Dieses Bild erinnert mich an...* (이 그림은 나에게 ...을 떠올리게 한다)

## 5. 연결어와 전환 표현

### 순서 표현

| 독일어 | 영어 |
|--------|------|
| erstens, zweitens, drittens | firstly, secondly, thirdly |
| zunächst / zuerst | first of all |
| dann / danach | then / after that |
| anschließend | subsequently |
| schließlich / zuletzt | finally / lastly |

### 정보 추가

| 독일어 | 영어 |
|--------|------|
| außerdem | furthermore, besides |
| darüber hinaus | beyond that, moreover |
| zusätzlich | additionally |
| hinzu kommt, dass... | in addition... |
| nicht nur...sondern auch | not only...but also |

### 대조

| 독일어 | 영어 |
|--------|------|
| im Gegensatz dazu | in contrast to that |
| andererseits | on the other hand |
| einerseits...andererseits | on one hand...on the other |
| trotzdem / dennoch | nevertheless |
| jedoch / allerdings | however |
| obwohl | although |
| während | while, whereas |

### 원인과 결과

| 독일어 | 영어 |
|--------|------|
| deshalb / deswegen / daher | therefore |
| folglich / infolgedessen | consequently |
| aus diesem Grund | for this reason |

| 독일어 | 영어 |

## 6. 묘사 작문: Bildbeschreibung

### 이미지 묘사에 유용한 표현

**위치**:
- *Im Vordergrund / Im Hintergrund* — 전경에 / 배경에
- *In der Mitte des Bildes* — 그림 중앙에
- *Auf der linken / rechten Seite* — 왼쪽 / 오른쪽에
- *Am oberen / unteren Rand* — 상단 / 하단 가장자리에
- *Neben / Hinter / Vor* — 옆에 / 뒤에 / 앞에

**묘사**:
- *Man sieht / Man erkennt* — ...을 볼 수 있다 / 알아볼 수 있다
- *Es scheint, als ob...* — ...인 것 같다
- *Die Person trägt...* — 그 사람은 ...을 입고 있다
- *Die Szene spielt sich ab in...* — 장면은 ...에서 펼쳐진다

**해석**:
- *Das Bild vermittelt den Eindruck, dass...* — 이 이미지는 ...이라는 인상을 준다
- *Der Fotograf / Die Fotografin wollte wahrscheinlich zeigen, dass...* — 사진작가는 아마도 ...을 보여주고 싶었을 것이다
- *Das Bild könnte symbolisch für ... stehen.* — 이 이미지는 ...을 상징할 수 있다

## 7. 독일어 구두점 규칙

독일어 구두점은 영어와 여러 중요한 점에서 다릅니다.

### 쉼표 규칙 (Kommaregeln)

**종속절 앞에 필수 쉼표**:

모든 종속 접속사 앞에 쉼표가 필요합니다:

| 접속사 | 예시 |
|--------|------|
| **dass** | *Ich weiß**,** dass er morgen kommt.* |
| **weil** | *Sie bleibt zu Hause**,** weil sie krank ist.* |
| **wenn** | *Ruf mich an**,** wenn du fertig bist.* |
| **obwohl** | *Er kam**,** obwohl er müde war.* |
| **ob** | *Ich frage mich**,** ob das stimmt.* |
| **als** | *Er war traurig**,** als sie ging.* |
| **während** | *Ich lese**,** während du kochst.* |
| **damit** | *Ich spare**,** damit ich reisen kann.* |

> **이것은 필수입니다** — 영어에서는 "because"나 "if" 앞에 쉼표가 선택적인 경우가 있지만, 독일어에서는 모든 종속절 앞에 쉼표를 반드시 사용해야 합니다.

***und*나 *oder*로 연결된 주절 사이의 쉼표**:

- 절들이 같은 주어를 공유할 경우 선택 사항: *Er kam nach Hause und (er) setzte sich auf das Sofa.*
- 절들의 주어가 다를 경우 명확성을 위해 권장: *Er kam nach Hause, und sie ging in die Küche.*

**관계절에서의 쉼표**:

관계절은 항상 쉼표로 구분합니다:
- *Der Mann**,** der neben mir steht**,** ist mein Bruder.*
- *Das Buch**,** das ich gestern gelesen habe**,** war spannend.*

**목록에서의 쉼표 (옥스퍼드 쉼표 없음)**:

독일어에서는 옥스퍼드 쉼표를 사용하지 **않습니다**:
- *Ich kaufe Brot, Butter, Käse und Milch.* (*Brot, Butter, Käse, und Milch*이 아님)

***zu* 부정사절에서의 쉼표**:

부정사절에 추가 단어가 포함된 경우 쉼표를 사용합니다:
- *Er beschloss**,** nach Berlin zu fahren.*
- *Sie hatte keine Lust**,** den ganzen Tag zu arbeiten.*

### 인용부호 (Anführungszeichen)

독일어는 영어와 다른 **아래-위** 인용부호를 사용합니다:

- 독일어: **„Hallo, wie geht es Ihnen?"**
- 영어: "Hello, how are you?"

인쇄 및 공식 문서에서:
- 기본 인용부호: „..." (아래에서 열고, 위에서 닫음)
- 인용부호 안의 인용: ‚...' (단일 아래-위)

> **참고**: 디지털 통신과 많은 키보드에서는 표준 "..." 인용부호가 대신 사용되는 경우가 많으며, 이는 널리 허용됩니다.

### 기타 구두점 차이

| 규칙 | 독일어 | 영어 |
|------|--------|------|
| 소수점 구분자 | **쉼표**: 3,50 € | **마침표**: $3.50 |
| 천 단위 구분자 | **마침표**: 1.000.000 | **쉼표**: 1,000,000 |
| 시간 | **콜론**: 14:30 Uhr | **콜론**: 2:30 PM |
| 날짜 형식 | **TT.MM.JJJJ**: 05.03.2025 | MM/DD/YYYY: 03/05/2025 |

---

## 8. 흔한 작문 실수

### 어순 오류

**실수 1**: 주절에서 동사 두 번째 위치(V2) 규칙 위반.

- 틀림: *Gestern ich bin ins Kino gegangen.*
- 올바름: *Gestern **bin** ich ins Kino gegangen.*

**실수 2**: 종속절에서 동사 끝 위치 규칙 위반.

- 틀림: *Ich weiß, dass er kommt morgen.*
- 올바름: *Ich weiß, dass er morgen **kommt**.*

**실수 3**: *deshalb*, *trotzdem* 같은 부사 뒤의 어순 오류.

- 틀림: *Deshalb ich bleibe zu Hause.*
- 올바름: *Deshalb **bleibe** ich zu Hause.* (주어-동사 도치)

### 격 오류

**실수 4**: 전치사 뒤에 잘못된 격 사용.

- 틀림: *Ich gehe in **der** Park.* (여격)
- 올바름: *Ich gehe in **den** Park.* (대격 — 방향으로의 이동)
- 역시 올바름: *Ich bin in **dem** Park.* (여격 — 장소)

### 관사/성 혼동

**실수 5**: 일반적인 명사의 성별 오류.

자주 혼동되는 것들:
- *das* Mädchen (중성 — -chen 축소 접미사 때문, 여성이 아님)
- *die* E-Mail (독일어에서 여성, "mail"이 중성처럼 느껴지지만)
- *der* Computer (남성)

### 쉼표 오류

**실수 6**: 종속절 앞에 쉼표 누락.

- 틀림: *Ich glaube dass er Recht hat.*
- 올바름: *Ich glaube**,** dass er Recht hat.*

**실수 7**: 옥스퍼드 쉼표 추가.

- 틀림: *Äpfel, Birnen, und Kirschen*
- 올바름: *Äpfel, Birnen und Kirschen*

### 철자 오류

**실수 8**: *das* (관사/대명사)와 *dass* (접속사) 혼동.

- *Das* Buch ist gut. (관사: 그)
- Ich weiß, *dass* er kommt. (접속사: ~라는 것)
- 팁: *dieses*나 *welches*로 대체할 수 있으면 *das*를 사용하세요. 그렇지 않으면 *dass*입니다.

## 9. 문자 메시지와 소셜 미디어

독일어 문자 메시지에서는 많은 약어와 비공식적 관례를 사용합니다:

### 일반적인 약어

| 약어 | 전체 형태 | 영어 |
|------|-----------|------|
| LG | Liebe Grüße | Best wishes |
| VG | Viele Grüße | Many greetings |
| MfG | Mit freundlichen Grüßen | Best regards |
| HDL | Hab dich lieb | Love you (우정) |
| GLG | Ganz liebe Grüße | Much love |
| WE | Wochenende | Weekend |
| vllt | vielleicht | maybe |
| eigtl | eigentlich | actually |
| bzgl | bezüglich | regarding |
| ggf | gegebenenfalls | if applicable |
| usw | und so weiter | and so on |
| zB / z.B. | zum Beispiel | for example |
| mMn | meiner Meinung nach | in my opinion |

### 비공식 문자 스타일

- 전체 소문자 사용: *hey, kommst du heute abend?*
- 움라우트 생략: *schön* 대신 *schoen*
- 이모티콘과 이모지 광범위하게 사용
- 짧은 문장, 문장 조각: *Bin gleich da.* (곧 갈게.)
- 영어 외래어 일반적: *cool, sorry, nice, okay*

> **참고**: 문자 메시지 관례는 공식 문서, 시험, 비즈니스 커뮤니케이션에서 사용해서는 안 됩니다.

독일어 친구 *Max*에게 독일 방문 계획에 대한 이메일을 작성하세요. 다음을 포함하세요:
- 인사말
- 안부 묻기
- 방문 시기 알리기
- 함께 할 활동 제안
- 맺음말

<details>
<summary>모범 답안</summary>

> Lieber Max,
>
> wie geht es Dir? Ich hoffe, Du hattest ein schönes Wochenende!
>
> Ich wollte Dir sagen, dass ich im Juli für zwei Wochen nach Deutschland komme! Ich habe schon einen Flug nach Frankfurt gebucht und komme am 10. Juli an.
>

### 연습 2: 공식 이메일 작성하기

뮌헨의 호텔에 객실 예약을 위한 공식 이메일을 작성하세요. 다음을 포함하세요:
- 공식 인사말
- 숙박 날짜 (7월 15-20일)
- 객실 유형 (더블룸, Doppelzimmer)
- 조식 포함 여부 문의
- 주차장 문의
- 공식 맺음말

<details>
<summary>모범 답안</summary>

> Betreff: Zimmerreservierung vom 15. bis 20. Juli
>
> Sehr geehrte Damen und Herren,
>
> ich möchte gerne ein Doppelzimmer in Ihrem Hotel für den Zeitraum vom 15. bis 20. Juli 2025 reservieren (5 Nächte).
>
> Könnten Sie mir bitte mitteilen, ob für diesen Zeitraum ein Zimmer verfügbar ist und wie hoch der Preis pro Nacht ist? Außerdem würde ich gerne wissen, ob das Frühstück im Preis inbegriffen ist und ob es Parkmöglichkeiten für Hotelgäste gibt.
>
> Vielen Dank im Voraus für Ihre Rückmeldung.
>
> Mit freundlichen Grüßen
>
> [Name]

</details>

### 연습 3: 구두점 교정하기

다음 문장의 구두점 오류를 수정하세요:

1. Ich glaube dass er morgen kommt.
2. Wir kaufen Brot, Käse, Wurst, und Marmelade.
3. Er sagte "Ich komme morgen."
4. Der Preis beträgt 1,500.00 Euro.
5. Wann kommst du, wenn du fertig bist ruf mich an.

<details>
<summary>정답</summary>

1. Ich glaube**,** dass er morgen kommt. (*dass* 앞에 쉼표)
2. Wir kaufen Brot, Käse, Wurst und Marmelade. (옥스퍼드 쉼표 제거)
3. Er sagte**:** **„**Ich komme morgen.**"** (*sagte* 뒤에 콜론, 독일어 인용부호)
4. Der Preis beträgt 1**.**500**,**00 Euro. (독일어: 천 단위에 마침표, 소수점에 쉼표)
5. Wann kommst du**?** **W**enn du fertig bist**,** ruf mich an**!** (두 개의 별도 문장, 종속절이 먼저 올 때 주절 앞에 쉼표)

</details>

### 연습 4: 짧은 에세이 개요 작성하기

다음 주제에 대한 개요(Gliederung)를 작성하세요: *"Sollten Handys in der Schule verboten werden?"* (학교에서 휴대폰을 금지해야 하는가?)

포함할 내용: Einleitung, 찬성 논거 2-3개, 반대 논거 2-3개, Schluss.

<details>
<summary>모범 답안</summary>

**Einleitung**:
- 도입부: *Heutzutage besitzt fast jeder Schüler ein Smartphone.* (오늘날 거의 모든 학생이 스마트폰을 소유하고 있다.)
- 핵심 질문: *Sollten Handys in der Schule verboten werden?*

**Pro (금지 찬성)**:
1. Handys lenken vom Unterricht ab. (휴대폰은 수업에서 주의를 분산시킨다.)
2. Cybermobbing kann während der Schulzeit passieren. (사이버 괴롭힘이 수업 시간에 발생할 수 있다.)
3. Ohne Handys würden Schüler mehr miteinander reden. (휴대폰 없이는 학생들이 서로 더 많이 대화할 것이다.)

**Contra (금지 반대)**:
1. Smartphones können als Lernmittel genutzt werden. (스마트폰은 학습 도구로 활용될 수 있다.)
2. Eltern müssen ihre Kinder im Notfall erreichen können. (부모는 긴급 상황에서 자녀에게 연락할 수 있어야 한다.)
3. Ein Verbot ist schwer durchzusetzen. (금지 조치는 시행하기 어렵다.)

**Schluss**:
- *Meiner Meinung nach sollten Handys nicht komplett verboten werden, sondern es sollte klare Regeln für die Nutzung geben.* (제 의견으로는 휴대폰을 완전히 금지해서는 안 되며, 사용에 대한 명확한 규칙이 있어야 합니다.)

</details>

### 연습 5: 오류 찾고 수정하기

각 문장에는 하나 이상의 실수가 있습니다. 찾아서 수정하세요.

1. *Gestern ich habe meine Mutter besucht.*
2. *Ich weiß, dass er kommt morgen nach Berlin.*
3. *Das buch was ich lese ist sehr interessant.*
4. *Sie hat mir gesagt das sie nicht kommen kann.*
5. *In den Park gehen wir oft am Wochenende spazieren.*

<details>
<summary>정답</summary>

1. **Gestern habe ich meine Mutter besucht.** (V2 규칙: *Gestern* 뒤에 동사가 두 번째 위치에 와야 함)
2. **Ich weiß, dass er morgen nach Berlin kommt.** (*dass*-절에서 동사가 끝에 위치)
3. **Das Buch, das ich lese, ist sehr interessant.** (*Buch* 대문자 표기; 관계대명사 *was*가 아닌 *das*; 관계절 주위에 쉼표)
4. **Sie hat mir gesagt, dass sie nicht kommen kann.** (*das*가 아닌 *dass*; 종속절 앞에 쉼표)
5. 이 문장은 사실 **올바릅니다**! *In dem/den Park*는 의미에 따라 달라집니다. *In den Park gehen wir...*는 올바릅니다 (전치사구가 첫 번째 위치에 오는 V2). 그러나 다음과 같이 쓸 수도 있습니다: **In dem Park gehen wir oft am Wochenende spazieren.** (여격 — 위치, 이동이 아님) 또는 공원으로 들어가는 것이라면 원래 문장도 맞습니다.

</details>

### 연습 6: 비공식을 공식으로 변환하기

다음 비공식 메시지를 공식 이메일로 다시 작성하세요:

>
> Bitte teilen Sie mir mit, ob dies möglich ist. Vielen Dank im Voraus.
>
> Mit freundlichen Grüßen
>
> [Name]

</details>
