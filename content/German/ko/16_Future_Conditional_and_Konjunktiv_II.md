## 1. Futur I: werden + Infinitiv

Futur I는 활용된 조동사 **werden**과 절 끝에 오는 **부정사(Infinitiv)**로 형성됩니다.

### 1.1 werden의 활용

| 인칭 | werden |
|--------|--------|
| ich | werde |
| du | wirst |
| er/sie/es | wird |
| wir | werden |
| ihr | werdet |
| sie/Sie | werden |

### 1.2 형성

```
Subject + werden (conjugated) + ... + Infinitiv
```

| 독일어 | 영어 |
|--------|---------|
| Ich **werde** morgen nach Berlin **fliegen**. | I will fly to Berlin tomorrow. |
| Sie **wird** nächstes Jahr Medizin **studieren**. | She will study medicine next year. |
| Wir **werden** bald **umziehen**. | We will move soon. |

### 1.3 용법: 미래 사건

Futur I는 계획, 예측, 의도를 표현합니다:

- **Ich werde** nächsten Sommer nach Japan **reisen**. (나는 다음 여름에 일본으로 여행할 것이다.)
- **Es wird** morgen **regnen**. (내일 비가 올 것이다.)
- **Er wird** die Prüfung **bestehen**. (그는 시험에 합격할 것이다.)

> **중요:** 구어체 독일어에서는 확정된 미래 계획을 표현할 때, 특히 시간 표현이 미래 참조를 분명히 할 때 Futur I보다 **현재 시제**(Präsens)가 훨씬 더 일반적입니다:
>
> - Ich **fliege** morgen nach Berlin. (나는 내일 베를린으로 간다.) — 더 자연스러움
> - Ich **werde** morgen nach Berlin **fliegen**. — 더 격식적 / 강조적

### 1.4 용법: 가정과 추정

Futur I는 **현재**에 대한 가정이나 추정을 표현하는 데 자주 사용되며, 종종 조사 **wohl** (아마도)과 함께 쓰입니다:

| 독일어 | 영어 |
|--------|---------|
| Er **wird wohl** krank **sein**. | He is probably sick. |
| Sie **wird** schon **wissen**, was sie tut. | She probably knows what she's doing. |
| Das **wird** stimmen. | That's probably right. |
| Er **wird wohl** zu Hause **sein**. | He's probably at home. |

## 2. Futur II: werden + Partizip II + haben/sein

Futur II는 영어의 미래완료("will have done")에 해당하지만, 과거에 대한 추정을 표현하는 데 더 일반적으로 사용됩니다.

### 2.1 형성

```
Subject + werden (conjugated) + ... + Partizip II + haben/sein
```

**haben** 또는 **sein**의 선택은 Perfekt 시제와 같은 규칙을 따릅니다.

| 독일어 | 영어 |
|--------|---------|
| Ich **werde** das Buch **gelesen haben**. | I will have read the book. |
| Sie **wird** schon **angekommen sein**. | She will have already arrived. |
| Bis morgen **werden** wir alles **erledigt haben**. | By tomorrow we will have taken care of everything. |

### 2.2 용법: 미래완료

미래의 특정 시점까지 완료될 행위를 설명합니다:

- **Bis nächsten Freitag werde** ich den Bericht **geschrieben haben**. (다음 금요일까지 나는 보고서를 다 쓸 것이다.)
- **In zwei Jahren wird** er sein Studium **abgeschlossen haben**. (2년 후 그는 학업을 마칠 것이다.)

### 2.3 용법: 과거에 대한 추정

이것이 실제로 Futur II의 더 일반적인 용법입니다 — 아마 이미 일어났을 것에 대한 가정을 표현합니다:

| 독일어 | 영어 |
|--------|---------|
| Er **wird** es **vergessen haben**. | He has probably forgotten it. |
| Sie **wird** den Zug **verpasst haben**. | She has probably missed the train. |
| Das **wird** ein Missverständnis **gewesen sein**. | That was probably a misunderstanding. |
| Er **wird wohl** schon **gegangen sein**. | He has probably already left. |

## 3. Konjunktiv II: 합성형

Konjunktiv II(접속법 II)는 가정적 상황, 비현실적 조건, 소원, 정중한 요청을 표현합니다. 합성형은 Präteritum 어간에서 직접 파생됩니다.

### 3.1 형성

```
Präteritum stem + umlaut (if possible) + subjunctive endings
```

**접속법 어미:**

| 인칭 | 어미 |
|--------|--------|
| ich | -e |
| du | -est |
| er/sie/es | -e |
| wir | -en |
| ihr | -et |
| sie/Sie | -en |

### 3.2 일반적인 합성 Konjunktiv II 형태

이 고빈도 형태들은 암기해야 합니다. würde를 사용하지 않고 합성형으로 사용됩니다:

| 원형(Infinitiv) | Präteritum | Konjunktiv II | 영어 |
|-----------|-----------|---------------|---------|
| sein | war | **wäre** | would be |
| haben | hatte | **hätte** | would have |
| werden | wurde | **würde** | would (become) |
| können | konnte | **könnte** | could |
| müssen | musste | **müsste** | would have to |
| dürfen | durfte | **dürfte** | might / would be allowed to |
| sollen | sollte | **sollte** | should |
| wollen | wollte | **wollte** | would want |
| gehen | ging | **ginge** | would go |
| kommen | kam | **käme** | would come |
| geben | gab | **gäbe** | would give |
| wissen | wusste | **wüsste** | would know |
| tun | tat | **täte** | would do |
| lassen | ließ | **ließe** | would let |
| brauchen | brauchte | **bräuchte** | would need |

### 3.3 합성형 예문

| 독일어 | 영어 |
|--------|---------|
| Ich **wäre** gern in Deutschland. | I would like to be in Germany. |
| **Hätten** Sie einen Moment Zeit? | Would you have a moment? |
| Das **könnte** schwierig sein. | That could be difficult. |
| Ich **müsste** eigentlich arbeiten. | I really should be working. |
| Das **ginge** auch anders. | That could also be done differently. |
| Wenn er nur **käme**! | If only he would come! |
| Es **gäbe** eine andere Möglichkeit. | There would be another possibility. |
| Ich **wüsste** gern, ob... | I would like to know whether... |

## 4. Konjunktiv II: 우회형 (würde + Infinitiv)

대부분의 동사에서 합성 Konjunktiv II 형태는 Präteritum 직설법과 동일합니다 (예: *ich spielte* = "나는 놀았다" 그리고 "나는 놀 것이다"). 모호함을 피하기 위해 독일어에서는 우회 구문을 사용합니다:

### 4.1 형성

```
würde (conjugated) + ... + Infinitiv
```

| 인칭 | würde |
|--------|-------|
| ich | würde |
| du | würdest |
| er/sie/es | würde |
| wir | würden |
| ihr | würdet |
| sie/Sie | würden |

### 4.2 예문

| 독일어 | 영어 |
|--------|---------|
| Ich **würde** gern nach Paris **reisen**. | I would like to travel to Paris. |
| Was **würdest** du an meiner Stelle **tun**? | What would you do in my place? |
| Er **würde** das nie **sagen**. | He would never say that. |
| Wir **würden** euch gern **einladen**. | We would like to invite you. |
| **Würden** Sie mir bitte **helfen**? | Would you please help me? |

## 5. 합성형 vs. 우회형: 언제 어떤 것을 사용할 것인가

### 5.1 항상 합성형 사용

다음 동사에는 합성 Konjunktiv II를 사용하세요 — würde 형태는 부자연스럽게 들립니다:

- **sein** → wäre (~~würde sein~~ 아님)
- **haben** → hätte (~~würde haben~~ 아님)
- **werden** → würde (이것이 würde 형태 자체)
- **모든 화법 조동사** → könnte, müsste, dürfte, sollte, wollte (~~würde können~~ 아님)

### 5.2 합성형 선호 (특히 문어체에서)

Konjunktiv II가 독특하고 널리 사용되는 일반적인 강변화 동사들:

- ginge, käme, gäbe, wüsste, täte, ließe, bräuchte, fände, hielte, ließe, stünde/stände

### 5.3 würde + Infinitiv 사용

다른 모든 동사, 특히:

- Konjunktiv II = Präteritum인 약변화 동사: ich spielte → ich **würde spielen**
- 덜 일반적인 강변화 동사: ich **würde singen** (고어적인 *sänge*보다)
- "항상 합성형" 목록에 없는 모든 동사에 대한 구어체

### 5.4 요약표

| 범주 | 예 | 사용 형태 |
|----------|---------|-----------|
| sein, haben | wäre, hätte | 항상 합성형 |
| 화법 조동사 | könnte, müsste | 항상 합성형 |
| 일반적인 강변화 동사 | ginge, käme, gäbe | 합성형 선호 |
| 약변화 동사 | spielen, arbeiten | 항상 würde + Inf. |
| 덜 일반적인 강변화 동사 | singen, trinken | 보통 würde + Inf. |

## 6. 조건문 (Bedingungssätze)

독일어에는 세 가지 주요 유형의 조건문이 있습니다. **wenn**이 있는 종속절은 활용된 동사를 절 끝으로 보냅니다.

### 6.1 유형 1: 현실 조건 (Reale Bedingung — 직설법)

가능하거나 그럴 듯한 것을 표현합니다. 직설법을 사용합니다.

```
Wenn + ... + Verb(end), Verb + Subject + ...
```

| 독일어 | 영어 |
|--------|---------|
| **Wenn** es **regnet**, **bleibe** ich zu Hause. | If it rains, I'll stay home. |
| **Wenn** du **kommst**, **freue** ich mich. | If you come, I'll be happy. |
| **Wenn** man viel **übt**, **lernt** man schneller. | If you practice a lot, you learn faster. |
| Ich **kaufe** das Buch, **wenn** es nicht zu teuer **ist**. | I'll buy the book if it's not too expensive. |

### 6.2 유형 2: 비현실 현재 조건 (Irreale Bedingung der Gegenwart — Konjunktiv II)

가정적이거나 있을 법하지 않거나 현재 사실에 반하는 것을 표현합니다.

```
Wenn + ... + Konj. II(end), Konj. II + Subject + ...
```

| 독일어 | 영어 |
|--------|---------|
| **Wenn** ich reich **wäre**, **würde** ich um die Welt **reisen**. | If I were rich, I would travel around the world. |
| **Wenn** ich mehr Zeit **hätte**, **würde** ich Klavier **lernen**. | If I had more time, I would learn piano. |
| **Wenn** er Deutsch **könnte**, **würde** er in Berlin **arbeiten**. | If he could speak German, he would work in Berlin. |
| **Wenn** ich du **wäre**, **würde** ich das nicht **machen**. | If I were you, I wouldn't do that. |

### 6.3 유형 3: 비현실 과거 조건 (Irreale Bedingung der Vergangenheit — Konjunktiv II Perfekt)

과거 사실에 반하는 것을 표현합니다. hätte/wäre + Partizip II를 사용합니다.

```
Wenn + ... + Partizip II + hätte/wäre(end), hätte/wäre + Subject + ... + Partizip II
```

| 독일어 | 영어 |
|--------|---------|
| **Wenn** ich das **gewusst hätte**, **wäre** ich **gekommen**. | If I had known that, I would have come. |
| **Wenn** er früher **aufgestanden wäre**, **hätte** er den Zug **erreicht**. | If he had gotten up earlier, he would have caught the train. |
| **Wenn** wir mehr **geübt hätten**, **hätten** wir die Prüfung **bestanden**. | If we had practiced more, we would have passed the exam. |
| **Wenn** sie mich **gefragt hätte**, **hätte** ich ihr **geholfen**. | If she had asked me, I would have helped her. |

### 6.4 wenn 생략

격식적이거나 문학적인 독일어에서는 **wenn**을 생략할 수 있습니다. 이 경우 동사가 첫 번째 위치로 이동합니다:

| wenn 포함 | wenn 생략 |
|-----------|-------------|
| **Wenn** ich reich **wäre**, ... | **Wäre** ich reich, ... |
| **Wenn** ich das **gewusst hätte**, ... | **Hätte** ich das **gewusst**, ... |
| **Wenn** er **käme**, ... | **Käme** er, ... |

## 7. 소원 표현하기

Konjunktiv II는 현실에 반하는 상황에 대한 소원을 표현하는 데 사용됩니다.

### 7.1 Ich wünschte... (~하면 좋겠다...)

| 독일어 | 영어 |
|--------|---------|
| Ich **wünschte**, ich **hätte** mehr Geld. | I wish I had more money. |
| Ich **wünschte**, ich **wäre** jünger. | I wish I were younger. |
| Ich **wünschte**, er **würde** mich anrufen. | I wish he would call me. |
| Ich **wünschte**, ich **hätte** das **gewusst**. | I wish I had known that. |

> 참고: **wünschte** 자체가 Konjunktiv II 형태입니다 (*wünschen*에서 파생).

### 7.2 Wenn ... doch / nur / bloß ...!

조사 **doch**, **nur**, **bloß**는 소원을 강화합니다. 결합할 수도 있습니다:

| 독일어 | 영어 |
|--------|---------|
| **Wenn** ich **doch** fliegen **könnte**! | If only I could fly! |
| **Wenn** er **nur** hier **wäre**! | If only he were here! |
| **Wenn** wir **bloß** mehr Zeit **hätten**! | If only we had more time! |
| **Wenn** ich das **doch nur gewusst hätte**! | If only I had known that! |

### 7.3 Hätte / Wäre + bloß / nur ...!

wenn 없이 — 동사가 첫 번째 위치로 이동합니다:

| 독일어 | 영어 |
|--------|---------|
| **Hätte** ich **bloß** zugehört! | If only I had listened! |
| **Wäre** ich **nur** zu Hause geblieben! | If only I had stayed home! |
| **Hätte** ich das **doch** früher gewusst! | If only I had known that earlier! |

## 8. 정중한 요청

Konjunktiv II는 직접적인 요청을 완화하여 더 정중하게 만듭니다. 비교:

| 직접적 (직설법) | 정중한 (Konjunktiv II) |
|--------------------|-----------------------|
| **Können** Sie mir helfen? | **Könnten** Sie mir helfen? |
| **Haben** Sie einen Stift? | **Hätten** Sie einen Stift? |
| Helfen Sie mir! | **Würden** Sie mir bitte helfen? |
| Ich **will** ein Glas Wasser. | Ich **hätte** gern ein Glas Wasser. |
| Ich **brauche** Ihre Hilfe. | Ich **bräuchte** Ihre Hilfe. |

### 일반적인 정중한 표현

| 독일어 | 영어 |
|--------|---------|
| **Könnten** Sie das bitte wiederholen? | Could you please repeat that? |
| **Hätten** Sie einen Moment Zeit? | Would you have a moment? |
| **Würden** Sie bitte das Fenster öffnen? | Would you please open the window? |
| **Dürfte** ich Sie etwas fragen? | Might I ask you something? |
| **Wäre** es möglich, einen Termin zu machen? | Would it be possible to make an appointment? |
| Ich **hätte** gern eine Tasse Kaffee. | I would like a cup of coffee. |
| Es **wäre** nett, wenn Sie mir helfen **könnten**. | It would be nice if you could help me. |

## 9. als ob + Konjunktiv II

접속사 **als ob** (마치 ~인 것처럼)은 뒤에 Konjunktiv II가 옵니다. als ob이 종속절을 도입하기 때문에 활용된 동사는 절 끝으로 갑니다.

### 9.1 als ob + Konjunktiv II (현재 의미)

| 독일어 | 영어 |
|--------|---------|
| Er tut so, **als ob** er krank **wäre**. | He acts as if he were sick. |
| Sie sieht aus, **als ob** sie nicht geschlafen **hätte**. | She looks as if she hadn't slept. |
| Es scheint, **als ob** er alles **wüsste**. | It seems as if he knew everything. |
| Er redet, **als ob** er ein Experte **wäre**. | He talks as if he were an expert. |

### 9.2 als + Konjunktiv II (ob 없이)

**ob**이 생략되면 활용된 동사가 **als** 바로 뒤에 옵니다:

| als ob 포함 | als (ob 없이) |
|------------|-----------------|
| Er tut so, **als ob** er krank **wäre**. | Er tut so, **als wäre** er krank. |
| Sie sieht aus, **als ob** sie weinen **würde**. | Sie sieht aus, **als würde** sie weinen. |
| Es klingt, **als ob** es wahr **wäre**. | Es klingt, **als wäre** es wahr. |

> 두 형태 모두 동등하게 올바릅니다. 더 짧은 **als + 동사** 형태는 구어와 문어 모두에서 매우 일반적입니다.

### 9.3 als ob과 과거 참조

"마치 ~했었던 것처럼"을 표현하려면 Konjunktiv II Perfekt를 사용합니다:

| 독일어 | 영어 |
|--------|---------|
| Er sieht aus, **als ob** er einen Geist **gesehen hätte**. | He looks as if he had seen a ghost. |
| Sie benimmt sich, **als ob** nichts **passiert wäre**. | She behaves as if nothing had happened. |

<summary>정답</summary>

1. He will arrive tomorrow. **(F)**
2. She is probably tired. **(P)**
3. We will get married next year. **(F)**
4. That's probably right. **(P)**
5. I will finish reading the book soon. **(F)**

</details>

### 연습 2: Futur II

과거에 대한 추정을 표현하기 위해 Futur II를 사용하여 다시 쓰세요.

1. Er hat wahrscheinlich den Bus verpasst. → Er wird ...
2. Sie ist wahrscheinlich schon gegangen. → Sie wird ...
3. Das war wahrscheinlich ein Fehler. → Das wird ...
4. Er hat es wahrscheinlich vergessen. → Er wird ...

<details>
<summary>정답</summary>

1. Er **wird** den Bus **verpasst haben**.
2. Sie **wird** schon **gegangen sein**.
3. Das **wird** ein Fehler **gewesen sein**.
4. Er **wird** es **vergessen haben**.

</details>

### 연습 3: Konjunktiv II — 합성형인가 würde인가?

올바른 형태를 선택하세요. 적절한 경우 합성 Konjunktiv II를 사용하세요.

1. Wenn ich reich ______ (sein), würde ich reisen.
2. ______ (können) Sie mir bitte helfen?
3. Was ______ du an meiner Stelle ______ (machen)?
4. Ich ______ (haben) gern ein Stück Kuchen.
5. Es ______ (geben) noch eine andere Lösung.
6. Wenn er nur ______ (kommen)!

<details>
<summary>정답</summary>

1. **wäre** (sein은 항상 합성형)
2. **Könnten** (화법 조동사는 항상 합성형)
3. **würdest** ... **machen** (약변화 패턴 동사에는 würde + Inf.)
4. **hätte** (haben은 항상 합성형)
5. **gäbe** (일반적인 합성형)
6. **käme** (일반적인 합성형)

</details>

### 연습 4: 조건문

올바른 형태로 문장을 완성하세요.

1. Wenn es morgen regnet, ______ ich zu Hause ______. (bleiben — 유형 1)
2. Wenn ich Deutsch sprechen ______, ______ ich in Berlin arbeiten. (können — 유형 2)
3. Wenn ich das ______ ______, ______ ich dir geholfen. (wissen — 유형 3)
4. Wenn er mehr ______ ______, ______ er die Prüfung bestanden. (lernen — 유형 3)
5. Wenn du morgen kommst, ______ wir ins Kino ______. (gehen — 유형 1)

<details>
<summary>정답</summary>

1. Wenn es morgen regnet, **bleibe** ich zu Hause. (현재 직설법)
2. Wenn ich Deutsch sprechen **könnte**, **würde** ich in Berlin arbeiten.
3. Wenn ich das **gewusst hätte**, **hätte** ich dir geholfen.
4. Wenn er mehr **gelernt hätte**, **hätte** er die Prüfung bestanden.
5. Wenn du morgen kommst, **gehen** wir ins Kino. (현재 직설법)

</details>

### 연습 5: 소원과 정중한 요청

A) "Wenn ... doch nur ...!"를 사용하여 소원으로 다시 쓰세요.

1. Ich habe kein Geld. → Wenn ich doch nur ...
2. Ich bin nicht in Deutschland. → Wenn ich doch nur ...
3. Ich habe das nicht gewusst. → Wenn ich das doch nur ...

B) Konjunktiv II를 사용하여 정중하게 바꾸세요.

4. Können Sie mir den Weg zeigen? → ______
5. Haben Sie ein Zimmer frei? → ______

<details>
<summary>정답</summary>

1. **Wenn** ich **doch nur** Geld **hätte**!
2. **Wenn** ich **doch nur** in Deutschland **wäre**!
3. **Wenn** ich das **doch nur gewusst hätte**!
4. **Könnten** Sie mir den Weg zeigen?
5. **Hätten** Sie ein Zimmer frei?

</details>

### 연습 6: als ob + Konjunktiv II

올바른 Konjunktiv II 형태로 완성하세요.

1. Er tut so, als ob er alles ______ (wissen).
2. Sie sieht aus, als ob sie ______ (weinen) ______ (würde).
3. Es klingt, als ob es wahr ______ (sein).
4. Er benimmt sich, als ob er der Chef ______ (sein).
5. Sie redet, als ob sie noch nie dort ______ ______ (sein — 과거).

<details>
<summary>정답</summary>

1. Er tut so, als ob er alles **wüsste**.
2. Sie sieht aus, als ob sie weinen **würde**.
3. Es klingt, als ob es wahr **wäre**.
4. Er benimmt sich, als ob er der Chef **wäre**.
5. Sie redet, als ob sie noch nie dort **gewesen wäre**.

</details>
