## 2. Cardinal Numbers 31-1,000,000

### 31-99: Tens + Units (one word)

| Number | Italian |
|--------|---------|
| 31 | trent**uno** |
| 38 | trent**otto** |
| 42 | quaranta**due** |
| 55 | cinquanta**cinque** |
| 63 | sessanta**tré** |
| 77 | settanta**sette** |
| 88 | ottant**otto** |
| 99 | novanta**nove** |

> **Rule**: When adding *uno* or *otto*, the tens word drops its final vowel: trent**a** + uno = trent**uno**, quarant**a** + otto = quarant**otto**.

### Hundreds

| Number | Italian |
|--------|---------|
| 100 | cento |
| 200 | duecento |
| 300 | trecento |
| 400 | quattrocento |
| 500 | cinquecento |
| 600 | seicento |
| 700 | settecento |
| 800 | ottocento |
| 900 | novecento |

> **Note**: Unlike Spanish, Italian hundreds do NOT change for gender. *Duecento* whether referring to masculine or feminine nouns: *duecento persone*, *duecento libri*.

### Combining Hundreds

```
101 = centouno (or centuno)
115 = centoquindici
200 = duecento
350 = trecentocinquanta
999 = novecentonovantanove

※ All written as ONE word up to 999
※ Cento does NOT change to "centi" in compounds
```

### Large Numbers

| Number | Italian |
|--------|---------|
| 1,000 | mille |
| 2,000 | duemila |
| 10,000 | diecimila |
| 100,000 | centomila |
| 1,000,000 | un milione (di + noun) |
| 2,000,000 | due milioni (di + noun) |
| 1,000,000,000 | un miliardo (di + noun) |

```
Key notes:
  mille → mila (plural): duemila, tremila, diecimila
  milione/miliardo are NOUNS → need "di" before another noun:
    un milione di euro (one million euros)
    tre milioni di persone (three million people)
    un miliardo di dollari (one billion dollars)

  ※ Numbers are written as one word up to 999:
    cinquecentoventitré (523)
  ※ After mille, continue as one word:
    milleduecentotrenta (1,230)
```

### UNO as an Adjective

When *uno* precedes a noun, it follows article rules:
- **un** libro (a/one book — m. before consonant)
- **uno** studente (a/one student — m. before s+cons.)
- **una** casa (a/one house — f. before consonant)
- **un'**amica (a/one friend — f. before vowel)

In compound numbers, *-uno* can optionally drop -o before a noun:
- ventun libri OR ventuno libri (21 books)

## 3. Ordinal Numbers

| Number | Masculine | Feminine | Abbreviated |
|--------|-----------|----------|-------------|
| 1st | primo | prima | 1° / 1ª |
| 2nd | secondo | seconda | 2° / 2ª |
| 3rd | terzo | terza | 3° / 3ª |
| 4th | quarto | quarta | 4° |
| 5th | quinto | quinta | 5° |
| 6th | sesto | sesta | 6° |
| 7th | settimo | settima | 7° |
| 8th | ottavo | ottava | 8° |
| 9th | nono | nona | 9° |
| 10th | decimo | decima | 10° |

> From 11th onward, add **-esimo** to the cardinal number (drop final vowel):
> undici → undic**esimo** (11th), dodici → dodic**esimo** (12th), venti → vent**esimo** (20th)

**Usage:**
- Ordinals agree in gender and number like regular adjectives
- Always placed **before** the noun: *il primo giorno* (the first day)
- Used for centuries: *il Quattrocento* = the 1400s (15th century)
- Used for popes/kings: *Giovanni Paolo Secondo* (John Paul II)

> **Italian centuries by name** (cultural vocabulary):
> il Duecento (1200s), il Trecento (1300s), il Quattrocento (1400s),
> il Cinquecento (1500s), il Seicento (1600s), il Settecento (1700s),
> l'Ottocento (1800s), il Novecento (1900s)

## 4. Telling Time

### 4.1 Basic Structure

> **Telling Time in Italian**
>
> - 1:00 → È l'una. (singular: "è l'una")
> - 2:00 → Sono le due. (plural: "sono le")
> - 3:15 → Sono le tre e un quarto. (= Sono le tre e quindici.)
> - 4:30 → Sono le quattro e mezza. (= Sono le quattro e trenta.)
> - 5:45 → Sono le sei meno un quarto. (= Sono le cinque e quarantacinque.)
> - 6:10 → Sono le sei e dieci.
> - 7:50 → Sono le otto meno dieci.
> - 12:00 → È mezzogiorno. (noon)
> - 00:00 → È mezzanotte. (midnight)
>
> - "È" for 1:00, noon, midnight; "Sono" for all others
> - "Meno" = minus (for minutes before the next hour)
> - 24-hour clock is common in formal/written contexts

### 4.2 Time of Day

| Italian | English |
|---------|---------|
| di mattina / della mattina | in the morning (AM) |
| del pomeriggio | in the afternoon (PM, ~1-6) |
| di sera / della sera | in the evening (PM, ~6-10) |
| di notte / della notte | at night (PM, ~10-6) |

**Examples:**
- Sono le otto **di mattina**. (It's 8 AM.)
- Sono le tre **del pomeriggio**. (It's 3 PM.)
- È l'una **del pomeriggio**. (It's 1 PM.)
- Sono le dieci **di sera**. (It's 10 PM.)

### 4.3 Asking and Answering

- **Che ore sono?** / **Che ora è?** (What time is it?)
- **A che ora...?** (At what time...?)
  - A che ora comincia la lezione? — **All'una.** / **Alle nove.** (At one. / At nine.)

> **Note**: "At" a time uses *a + article*: **alle** due (at two), **alle** tre (at three), **all'**una (at one), **a** mezzogiorno (at noon).

## 5. Days of the Week

| Italian | English | Note |
|---------|---------|------|
| lunedì | Monday | (from Luna — Moon) |
| martedì | Tuesday | (from Marte — Mars) |
| mercoledì | Wednesday | (from Mercurio — Mercury) |
| giovedì | Thursday | (from Giove — Jupiter) |
| venerdì | Friday | (from Venere — Venus) |
| sabato | Saturday | |
| domenica | Sunday | |

> Days are **not capitalized** in Italian (unless at the start of a sentence).
> All days except *sabato* and *domenica* end in **-dì** (from Latin *dies* = day).
> Use the **definite article** for habitual actions: **il** lunedì (on Mondays), **la** domenica (on Sundays).
>
> - **Lunedì vado in palestra.** (On Monday I go to the gym.) — specific
> - **Il lunedì vado in palestra.** (On Mondays I go to the gym.) — habitual

## 7. Expressing Dates

### 7.1 Format

```
article + number + month + year

il 15 marzo 2026
(March 15, 2026)

il 25 dicembre
(December 25)
```

- The first of the month uses **primo** (ordinal):
  - il **primo** gennaio (January 1st)
  - But all other dates use cardinal numbers: il **due** gennaio, il **tre** febbraio

### 7.2 Asking the Date

- **Che giorno è oggi?** (What day is today?)
- **Quanti ne abbiamo oggi?** (What's the date? — lit: How many do we have today?)
- **Oggi è il 8 marzo 2026.** (Today is March 8, 2026.)
- **Oggi è lunedì, il 8 marzo.** (Today is Monday, March 8.)

### 7.3 Years

```
2026: duemilaventisei
1999: millenovecentonovantanove
2000: duemila (l'anno duemila)
1492: millequattrocentonovantadue

※ Years are read as regular numbers (not split like English "nineteen ninety-nine")
※ "In the year..." = nell'anno... or nel + year
   nel 2026 (in 2026)
   nel 1990 (in 1990)
```

### 7.4 Birth Date

```
Quand'è il tuo compleanno?    → When is your birthday?
Il mio compleanno è il 5 luglio. → My birthday is July 5.
Sono nato/a il 15 marzo 1995. → I was born on March 15, 1995.
```

## 9. Expressing Frequency

| Italian | English |
|---------|---------|
| sempre | always |
| quasi sempre | almost always |
| tutti i giorni / ogni giorno | every day |
| spesso | often |
| di solito / generalmente | usually |
| a volte / qualche volta | sometimes |
| di tanto in tanto | every now and then |
| raramente / di rado | rarely |
| quasi mai | almost never |
| mai | never |

**Frequency with Time:**
- tre volte **al** giorno (three times a day)
- una volta **alla** settimana (once a week)
- due volte **al** mese (twice a month)
- ogni anno / una volta **all'**anno (every year / once a year)
- ogni due settimane (every two weeks)

## 10. Practical Phrases for Daily Life

### 10.1 Shopping

```
Quanto costa?               → How much does it cost?
Quanto costano?             → How much do they cost?
Quanto viene?               → How much is it? (informal)
È troppo caro.              → It's too expensive.
Posso provarlo/a?           → Can I try it on?
C'è uno sconto?             → Is there a discount?
Posso pagare con la carta?  → Can I pay by card?
Mi può fare lo scontrino?   → Can you give me the receipt?
Vorrei... / Mi dia...       → I would like... / Give me...
```

### 10.2 At a Restaurant

```
Un tavolo per due, per favore.  → A table for two, please.
Il menù, per favore.           → The menu, please.
Cosa mi consiglia?             → What do you recommend?
Prendo / Per me...             → I'll have / For me...
Il conto, per favore.          → The check, please.
È incluso il coperto?          → Is the cover charge included?
Dov'è il bagno?                → Where is the restroom?
```

### 10.3 Directions

```
Dov'è...?                    → Where is...?
Come si arriva a...?          → How do you get to...?
È lontano / vicino?           → Is it far / near?
Vada dritto.                  → Go straight.
Giri a destra / a sinistra.   → Turn right / left.
Al semaforo...                → At the traffic light...
All'incrocio...               → At the intersection...
È a cinque minuti a piedi.    → It's five minutes on foot.
Mi sono perso/a.              → I'm lost.
```

### 10.4 Weather

```
Che tempo fa?                 → What's the weather like?
Fa caldo / freddo.            → It's hot / cold.
Fa bel tempo / brutto tempo.  → The weather is nice / bad.
Piove. / Sta piovendo.        → It's raining.
Nevica. / Sta nevicando.      → It's snowing.
Tira vento.                   → It's windy.
C'è il sole.                  → It's sunny.
È nuvoloso.                   → It's cloudy.
Che caldo! / Che freddo!      → How hot! / How cold!
```

## Quick Reference

> **Quick Reference**
>
> - **NUMBERS**: 0-10 (memorize), 11-19 (unique/dici-), 20-99 (tens+units as one word, drop vowel before 1/8). 100=cento, 1000=mille/mila, 1M=milione(di), 1B=miliardo
> - **TIME**: È l'una / Sono le [due-dodici] + e [minuti] / meno [minuti] / e un quarto / e mezza. A che ora? → Alle [ore] / All'una
> - **DAYS**: lunedì-venerdì (end in -dì), sabato, domenica. Not capitalized. Il lunedì = on Mondays.
> - **DATES**: il + number + month (+ year). Exception: il primo (1st of month)
> - **ORDINALS**: 1°-10° (unique), 11°+ = cardinal + -esimo
> - **KEY PHRASES**: Quanto costa? Dov'è...? Che tempo fa? Buon appetito! Salute! In bocca al lupo! — Crepi!
