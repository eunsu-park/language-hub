# 04. Essential Verbs: ESSERE, AVERE, FARE, ANDARE (Verbi Essenziali)

## Learning Objectives
- Master the four most important Italian verbs in the present tense
- Understand ESSERE vs STARE — two verbs both meaning "to be"
- Learn AVERE idioms that replace English "to be" expressions
- Master FARE expressions for everyday actions
- Understand ANDARE and the construction andare a + infinitive
- Learn C'È / CI SONO (there is / there are)

## 1. Why These Four Verbs?

```
┌─────────────────────────────────────────────────────────────────┐
│   These four verbs appear in almost every Italian sentence.     │
│   Master them and you can express most basic ideas.             │
│                                                                 │
│   ESSERE  → identity, origin, traits, time     ("to be")       │
│   AVERE   → possession, age, many idioms       ("to have")     │
│   FARE    → actions, weather, expressions       ("to do/make")  │
│   ANDARE  → movement, intentions               ("to go")       │
│                                                                 │
│   + STARE  → health, progressive, position     ("to be/stay")  │
│   + C'È/CI SONO  → existence                  ("there is/are") │
│                                                                 │
│   Challenge for English speakers:                               │
│   English "to be" → ESSERE or STARE (you must choose!)         │
│   English "to be" → sometimes AVERE (avere fame = "to be       │
│                     hungry", not "to have hunger")              │
│   English "to do/make" → one verb in Italian: FARE             │
└─────────────────────────────────────────────────────────────────┘
```

## 2. ESSERE — "To Be" (Identity, Essence)

### 2.1 Present Tense Conjugation

| Subject | Conjugation | Example |
|---------|-------------|---------|
| io | **sono** | Sono studente. (I'm a student.) |
| tu | **sei** | Sei intelligente. (You're smart.) |
| lui/lei/Lei | **è** | È professore. (He's a teacher.) |
| noi | **siamo** | Siamo coreani. (We're Korean.) |
| voi | **siete** | Siete amici. (You all are friends.) |
| loro | **sono** | Sono le tre. (It's three o'clock.) |

> **Note**: *io sono* and *loro sono* have the same conjugation. Context and subject pronouns clarify meaning.

### 2.2 Uses of ESSERE

```
┌─────────────────────────────────────────────────────────────────┐
│   1. Identity and Description                                    │
│      Sono Maria. (I'm Maria.)                                   │
│      Lei è alta e simpatica. (She is tall and friendly.)        │
│                                                                 │
│   2. Occupation (no article!)                                   │
│      Sono ingegnere. (I'm an engineer.)                         │
│      È professoressa. (She's a professor.)                      │
│                                                                 │
│   3. Nationality and Origin                                     │
│      Sono coreano. (I'm Korean.)                                │
│      Siamo di Seoul. (We're from Seoul.)                        │
│      Il caffè è di Colombia. (The coffee is from Colombia.)    │
│                                                                 │
│   4. Time and Dates                                             │
│      Sono le due. (It's two o'clock.)                           │
│      È lunedì. (It's Monday.)                                  │
│      È il 15 marzo. (It's March 15.)                           │
│                                                                 │
│   5. Characteristics (inherent qualities)                       │
│      Il ghiaccio è freddo. (Ice is cold.)                      │
│      La neve è bianca. (Snow is white.)                        │
│                                                                 │
│   6. Material / Composition                                     │
│      L'anello è d'oro. (The ring is gold.)                     │
│      Il tavolo è di legno. (The table is wooden.)              │
│                                                                 │
│   7. Location of events                                         │
│      La festa è a casa mia. (The party is at my house.)        │
│      Il concerto è al teatro. (The concert is at the theater.) │
│                                                                 │
│   8. Possession                                                 │
│      È il mio libro. (It's my book.)                           │
│      Di chi è questa borsa? (Whose bag is this?)               │
│                                                                 │
│   9. Auxiliary verb (for passato prossimo with certain verbs)   │
│      Sono andato a Roma. (I went to Rome.)                     │
│      È arrivata ieri. (She arrived yesterday.)                 │
└─────────────────────────────────────────────────────────────────┘
```

## 3. STARE — "To Be" (State, Condition)

### 3.1 Present Tense Conjugation

| Subject | Conjugation | Example |
|---------|-------------|---------|
| io | **sto** | Sto bene. (I'm fine.) |
| tu | **stai** | Come stai? (How are you?) |
| lui/lei/Lei | **sta** | Sta meglio oggi. (He's better today.) |
| noi | **stiamo** | Stiamo a casa. (We're staying at home.) |
| voi | **state** | State zitti! (Be quiet!) |
| loro | **stanno** | Stanno arrivando. (They're arriving.) |

### 3.2 Uses of STARE

```
┌─────────────────────────────────────────────────────────────────┐
│   1. Health / How someone is feeling                            │
│      Come stai? — Sto bene, grazie. (How are you? — I'm fine.) │
│      Non sto molto bene. (I'm not doing very well.)            │
│                                                                 │
│   2. Progressive form (stare + gerund)                          │
│      Sto mangiando. (I'm eating.)                              │
│      Stanno studiando. (They're studying.)                     │
│      Cosa stai facendo? (What are you doing?)                  │
│                                                                 │
│   3. Staying / Remaining                                        │
│      Stasera sto a casa. (Tonight I'm staying home.)           │
│      Sta' fermo! (Stay still!)                                 │
│                                                                 │
│   4. Being about to (stare per + infinitive)                   │
│      Sto per uscire. (I'm about to go out.)                    │
│      Sta per piovere. (It's about to rain.)                    │
│                                                                 │
│   5. Fixed expressions                                          │
│      Stare attento (to be careful/pay attention)               │
│      Stare zitto (to be quiet)                                 │
│      Stare tranquillo (to be calm)                             │
│      Stare in piedi (to be standing)                           │
│      Stare seduto (to be sitting)                              │
└─────────────────────────────────────────────────────────────────┘
```

## 4. ESSERE vs STARE Comparison

### 4.1 The Key Distinction

```
ESSERE = what something IS (essence, identity, inherent quality)
STARE = how something IS (current state, health, temporary position)

Italian ESSERE vs STARE is DIFFERENT from Spanish SER vs ESTAR:
  Italian uses ESSERE for location: Roma è in Italia. ✓
  Spanish uses ESTAR for location: Roma está en Italia. ✓

  Italian uses ESSERE for adjectives: Sono contento. ✓ (most cases)
  Italian uses STARE mainly for health: Sto bene. ✓
```

### 4.2 Direct Comparison

| Context | ESSERE | STARE |
|---------|--------|-------|
| Identity | Sono Maria. (I'm Maria.) | — |
| Health | — | Come stai? (How are you?) |
| Feeling/state | Sono felice. (I'm happy.) | Sto bene. (I'm well.) |
| Location | Roma è in Italia. (Rome is in Italy.) | Sto a casa. (I'm at/staying home.) |
| Progressive | — | Sto studiando. (I'm studying.) |
| About to | — | Sto per partire. (I'm about to leave.) |
| Occupation | Sono dottore. (I'm a doctor.) | — |
| Time | Sono le tre. (It's three.) | — |

### 4.3 Quick Decision Chart

```
Is it about...?
  Identity / Nationality / Profession  → ESSERE
  Location (of a place or thing)       → ESSERE
  Time / Date                          → ESSERE
  Inherent quality                     → ESSERE
  How you're doing (health)            → STARE
  Progressive action (-ando/-endo)     → STARE
  About to happen (per + infinitive)   → STARE
  Staying / Remaining somewhere        → STARE
  Fixed expressions (attento, zitto)   → STARE
```

## 5. AVERE — "To Have" and Idiomatic Uses

### 5.1 Present Tense Conjugation

| Subject | Conjugation | Example |
|---------|-------------|---------|
| io | **ho** | Ho un gatto. (I have a cat.) |
| tu | **hai** | Hai tempo? (Do you have time?) |
| lui/lei/Lei | **ha** | Ha 30 anni. (He's 30 years old.) |
| noi | **abbiamo** | Abbiamo lezione. (We have class.) |
| voi | **avete** | Avete fame? (Are you hungry?) |
| loro | **hanno** | Hanno due figli. (They have two children.) |

> **Note**: The h in *ho*, *hai*, *ha*, *hanno* is silent. It serves only to distinguish these verb forms from other words: *o* (or), *ai* (to the), *a* (to), *anno* (year).

### 5.2 Literal Uses

```
Possession:    Ho una macchina. (I have a car.)
Relationships: Ha tre fratelli. (He has three siblings.)
Age:           Ho venticinque anni. (I'm 25 years old.)
               ※ Italian says "I have 25 years", NOT "I am 25"
```

### 5.3 AVERE Idioms (Where English uses "to be")

```
┌─────────────────────────────────────────────────────────────────┐
│   Physical sensations:                                          │
│   avere fame         → to be hungry     (lit: have hunger)      │
│   avere sete         → to be thirsty    (lit: have thirst)      │
│   avere freddo       → to be cold       (lit: have cold)        │
│   avere caldo        → to be hot        (lit: have heat)        │
│   avere sonno        → to be sleepy     (lit: have sleep)       │
│                                                                 │
│   Emotional states:                                             │
│   avere paura (di)   → to be afraid (of) (lit: have fear)      │
│   avere vergogna     → to be ashamed    (lit: have shame)       │
│   avere pazienza     → to be patient    (lit: have patience)    │
│   avere fretta       → to be in a hurry (lit: have hurry)       │
│                                                                 │
│   Other common idioms:                                          │
│   avere ragione      → to be right      (lit: have reason)      │
│   avere torto        → to be wrong      (lit: have wrong)       │
│   avere fortuna      → to be lucky      (lit: have luck)        │
│   avere bisogno di   → to need          (lit: have need of)     │
│   avere voglia di    → to feel like     (lit: have desire of)   │
│   avere luogo        → to take place    (lit: have place)       │
│   avere senso        → to make sense    (lit: have sense)       │
│   avere a che fare con → to have to do with                     │
│                                                                 │
│   ※ Unlike Spanish, Italian does NOT say "avere anni"          │
│     alone — you must include the number:                        │
│     Ho trent'anni. (I'm 30 years old.)                         │
└─────────────────────────────────────────────────────────────────┘
```

### 5.4 AVERE as Auxiliary (Compound Tenses Preview)

```
AVERE (conjugated) + past participle = compound tenses
(used with most transitive verbs)

Present perfect (passato prossimo):
  ho parlato     → I have spoken / I spoke
  hai mangiato   → you have eaten / you ate
  ha visto       → he/she has seen / saw
  abbiamo letto  → we have read / read

※ This will be covered in detail in Lesson 11.
※ ESSERE is the auxiliary for intransitive verbs of motion/change.
```

## 6. FARE — "To Do/Make" and Expressions

### 6.1 Present Tense Conjugation

| Subject | Conjugation | Example |
|---------|-------------|---------|
| io | **faccio** | Faccio colazione. (I have breakfast.) |
| tu | **fai** | Cosa fai? (What are you doing?) |
| lui/lei/Lei | **fa** | Fa caldo. (It's hot.) |
| noi | **facciamo** | Facciamo una passeggiata. (We take a walk.) |
| voi | **fate** | Fate attenzione! (Pay attention!) |
| loro | **fanno** | Fanno i compiti. (They do homework.) |

### 6.2 FARE Expressions

```
┌─────────────────────────────────────────────────────────────────┐
│   Daily activities:                                             │
│   fare colazione        → to have breakfast                     │
│   fare pranzo/cena      → to have lunch/dinner                  │
│   fare la spesa         → to do grocery shopping                │
│   fare le compere/gli acquisti → to go shopping                 │
│   fare una passeggiata  → to take a walk                        │
│   fare il bagno/la doccia → to take a bath/shower              │
│   fare i compiti        → to do homework                        │
│   fare le valigie       → to pack (suitcases)                  │
│   fare la fila/la coda  → to stand in line                     │
│                                                                 │
│   Weather (impersonal — always "fa"):                           │
│   fa caldo              → it's hot                              │
│   fa freddo             → it's cold                             │
│   fa bel tempo          → the weather is nice                   │
│   fa brutto tempo       → the weather is bad                    │
│                                                                 │
│   Other expressions:                                            │
│   fare una domanda      → to ask a question                     │
│   fare attenzione       → to pay attention                      │
│   fare un viaggio       → to take a trip                        │
│   fare un regalo        → to give a gift                        │
│   fare una foto         → to take a photo                       │
│   fare tardi            → to be late                            │
│   fare in fretta        → to hurry                              │
│   fare parte di         → to be part of                         │
│   fare finta di         → to pretend                            │
│   fare un favore        → to do a favor                         │
│   farcela               → to manage / to make it                │
│   farsi male            → to hurt oneself                       │
│                                                                 │
│   ※ FARE is one of the most versatile verbs in Italian.        │
│     English splits "to do" and "to make" — Italian uses one.   │
└─────────────────────────────────────────────────────────────────┘
```

## 7. ANDARE — "To Go"

### 7.1 Present Tense Conjugation

| Subject | Conjugation | Example |
|---------|-------------|---------|
| io | **vado** | Vado a scuola. (I go to school.) |
| tu | **vai** | Dove vai? (Where are you going?) |
| lui/lei/Lei | **va** | Va in Italia. (He goes to Italy.) |
| noi | **andiamo** | Andiamo al cinema! (Let's go to the movies!) |
| voi | **andate** | Andate a casa. (You all go home.) |
| loro | **vanno** | Vanno in vacanza. (They go on vacation.) |

### 7.2 ANDARE + Prepositions

```
ANDARE A + place/activity:
  Vado a scuola.          (I go to school.)
  Vado a casa.            (I go home.)
  Vado a Roma.            (I go to Rome.)
  Vado al cinema.         (I go to the movies.) (a + il = al)
  Vado a mangiare.        (I go to eat.) (a + infinitive)

ANDARE IN + country/region/means of transport:
  Vado in Italia.         (I go to Italy.)
  Vado in Toscana.        (I go to Tuscany.)
  Vado in macchina.       (I go by car.)
  Vado in treno.          (I go by train.)
  Vado in bicicletta.     (I go by bicycle.)
  Vado in vacanza.        (I go on vacation.)

ANDARE A + infinitive (intention/purpose):
  Vado a studiare.        (I'm going to study.)
  Andiamo a mangiare!     (Let's go eat!)
  Vai a dormire.          (Go to sleep.)

ANDARE DA + person:
  Vado dal dottore.       (I go to the doctor.)
  Vado da Maria.          (I go to Maria's place.)
  Vado dai miei genitori. (I go to my parents'.)
```

### 7.3 ANDARE vs VENIRE

```
ANDARE = to go (away from the speaker)
VENIRE = to come (toward the speaker)

  A: Domani vado a Roma. (Tomorrow I'm going to Rome.)
  B: Vengo anch'io! (I'm coming too!)

  Vieni a casa mia? (Are you coming to my house?)
  Sì, vengo alle tre. (Yes, I'm coming at three.)
```

## 8. C'È and CI SONO — "There Is/Are"

```
┌─────────────────────────────────────────────────────────────────┐
│   c'è = there is (singular — ci + è)                           │
│   ci sono = there are (plural)                                  │
│                                                                 │
│   C'è un libro sul tavolo.        → There is a book on the table.│
│   Ci sono molti studenti.         → There are many students.   │
│   C'è un bar qui vicino?          → Is there a bar nearby?     │
│   Non c'è problema.               → There's no problem.        │
│   Ci sono tre camere.             → There are three rooms.     │
│                                                                 │
│   Past:   c'era (there was), c'erano (there were)              │
│   Future: ci sarà (there will be), ci saranno (there will be)  │
│                                                                 │
│   C'È vs ESSERE for location:                                  │
│   C'è un banco in via Roma.      → There's a bank on via Roma. │
│   (introduces existence — indefinite)                           │
│   La banca è in via Roma.        → The bank is on via Roma.    │
│   (specifies location of known thing — definite)               │
│                                                                 │
│   ※ C'è/Ci sono NEVER changes to agree with the noun.         │
│   ※ Use indefinite articles/adjectives after c'è/ci sono:     │
│     C'è un gatto.   Ci sono tre gatti.                         │
│     (NOT: C'è il gatto — use "il gatto è..." instead)         │
└─────────────────────────────────────────────────────────────────┘
```

## 9. Common Mistakes and Tips

### 9.1 Typical Errors

```
✗ Sono freddo.             ✓ Ho freddo. (I'm cold = AVERE idiom)
✗ Sto professore.          ✓ Sono professore. (profession = ESSERE)
✗ Sto a Roma.              ✓ Sono a Roma. (location = ESSERE in Italian!)
  BUT: Sto a casa.         ✓ (staying home = STARE, fixed expression)
✗ Ho 25.                   ✓ Ho 25 anni. (need "anni" for age)
✗ Sono fame.               ✓ Ho fame. (hungry = AVERE idiom)
✗ Ci è un libro.           ✓ C'è un libro. (there is = c'è)
✗ Il partito è a casa mia. ✓ La festa è a casa mia. (la festa, not il partito)
✗ Faccio un domanda.       ✓ Faccio una domanda. (domanda is feminine)
```

### 9.2 ESSERE vs STARE — Key Differences from Spanish

```
If you know Spanish, beware of these differences:

Spanish ESTAR for location → Italian ESSERE:
  ESP: Estoy en Roma.     → ITA: Sono a Roma. ✓ (NOT Sto a Roma)
  ESP: La tienda está aquí. → ITA: Il negozio è qui. ✓

Spanish ESTAR for temporary states → Italian ESSERE:
  ESP: Estoy cansado.     → ITA: Sono stanco. ✓ (NOT Sto stanco)
  ESP: Estoy contento.    → ITA: Sono contento. ✓ (NOT Sto contento)

Italian STARE is mainly for:
  Health: Sto bene / Sto male
  Progressive: Sto mangiando
  About to: Sto per partire
  Staying: Sto a casa
  Fixed expressions: Stare attento, stare zitto
```

### 9.3 Memory Tips

```
ESSERE = identity, location, description, time, events
         (wider use than Spanish ESTAR — covers location too!)

STARE = health + progressive + "about to" + staying + fixed expressions
        (narrower use than Spanish ESTAR)

AVERE for physical sensations:
  "In Italian, you don't ARE hungry/cold/sleepy —
   you HAVE hunger/cold/sleepiness"

FARE for everything you DO or MAKE:
  "When in doubt about 'do' or 'make' in Italian, try FARE"
```
