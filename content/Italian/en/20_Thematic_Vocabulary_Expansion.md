## 5. Diminutives (-ino, -etto, -ello)

Italian diminutives express smallness, affection, or endearment. They are extremely common in everyday speech.

### 5.1 -ino / -ina (Most Common)

| Base Word | Diminutive | Meaning |
|-----------|-----------|---------|
| ragazzo (boy) | ragazzino | little boy |
| gatto (cat) | gattino | kitten |
| tavolo (table) | tavolino | small table / café table |
| pensiero (thought) | pensierino | little thought / small gift |
| caro (dear) | carino | cute |
| poco (little) | pochino | a tiny bit |
| bene (well) | benino | fairly well |
| vicino (near) | vicinino | very near |

### 5.2 -etto / -etta

| Base Word | Diminutive | Meaning |
|-----------|-----------|---------|
| casa (house) | casetta | little house / cottage |
| giardino (garden) | giardinetto | small garden |
| posto (place) | posticino / postetto | cozy little spot |
| vecchio (old) | vecchietto | little old man |
| giovane (young) | giovanetto | young lad |
| fiero (proud) | fieretto | a bit proud (gently) |
| basso (low/short) | bassetto | rather short |

### 5.3 -ello / -ella

| Base Word | Diminutive | Meaning |
|-----------|-----------|---------|
| albero (tree) | alberello | small tree / sapling |
| paese (village) | paesello | little village |
| storia (story) | storiella | short story / anecdote |
| rondine (swallow) | rondinella | little swallow |
| asino (donkey) | asinello | little donkey |
| fontana (fountain) | fontanella | small fountain / baby's fontanelle |

### 5.4 Other Diminutives

| Suffix | Example | Meaning |
|--------|---------|---------|
| -uccio/-uccia | casa → casuccia | poor little house (slightly pejorative + affection) |
| -uzzo/-uzza | pietra → pietruzza | small stone / pebble |
| -icino/-icina | cuore → cuoricino | little heart |
| -olino/-olina | topo → topolino | little mouse (also: Mickey Mouse) |

> **Important**: Diminutives can change a word's gender. For example, *la finestra* (f.) → *il finestrino* (m., car window). Always learn the article with the diminutive.

## 6. Augmentatives (-one/-ona)

Augmentatives express largeness or intensification.

| Base Word | Augmentative | Meaning |
|-----------|-------------|---------|
| libro (book) | librone | big book |
| uomo (man) | omone | big man |
| donna (woman) | donnone | large/imposing woman |
| porta (door) | portone | main entrance / large door |
| minestra (soup) | minestrone | thick vegetable soup |
| festa (party) | festone | big party / garland |
| naso (nose) | nasone | big nose |
| pigro (lazy) | pigrone | very lazy person |
| chiacchiera (chat) | chiacchierone | big talker / chatterbox |
| bacio (kiss) | bacione | big kiss |

> **Gender Note**: Augmentatives with *-one* are always masculine, even if the base word is feminine:
> - la donna (f.) → il donnone (m.)
> - la porta (f.) → il portone (m.)
> - la palla (f.) → il pallone (m.) -- football!

## 7. Pejoratives (-accio/-accia)

Pejoratives express negative qualities -- ugliness, unpleasantness, or poor quality.

| Base Word | Pejorative | Meaning |
|-----------|-----------|---------|
| tempo (weather/time) | tempaccio | terrible weather |
| parola (word) | parolaccia | bad word / swear word |
| ragazzo (boy) | ragazzaccio | bad/naughty boy |
| carattere (character) | caratteraccio | bad temper |
| figura (figure/impression) | figuraccia | bad impression / embarrassment |
| giorno (day) | giornaccia | terrible day |
| strada (road) | stradaccia | bad road |
| uomo (man) | omaccio | nasty/rough man |
| lettera (letter) | letteraccia | nasty letter |

### Mixed Suffixes for Nuance

| Suffix Combo | Example | Nuance |
|-------------|---------|--------|
| -ino + affection | *amore* → *amorino* | little love, cupid |
| -one + emphasis | *febbre* → *febbrone* | raging fever |
| -accio + disgust | *caffè* → *caffèaccio* | terrible coffee |
| -etto + lightness | *viaggio* → *viaggetto* | short/quick trip |

## 8. Word Formation: Prefixes

### 8.1 Negation and Reversal Prefixes

| Prefix | Meaning | Examples |
|--------|---------|---------|
| **in-/im-** | not, un- | felice → **in**felice (unhappy), possibile → **im**possibile (impossible) |
| **dis-** | un-, dis- | fare → **dis**fare (to undo), ordine → **dis**ordine (disorder) |
| **s-** | un-, de-, dis- | fortunato → **s**fortunato (unlucky), caricare → **s**caricare (to unload/download) |
| **a-** | without | normale → **a**normale (abnormal), morale → **a**morale (amoral) |

### 8.2 Intensification Prefixes

| Prefix | Meaning | Examples |
|--------|---------|---------|
| **ri-** | re-, again | fare → **ri**fare (to redo), leggere → **ri**leggere (to reread) |
| **stra-** | extra, super | ordinario → **stra**ordinario (extraordinary), cotto → **stra**cotto (overcooked) |
| **super-** | super, above | mercato → **super**mercato (supermarket), uomo → **super**uomo (superman) |
| **ultra-** | ultra, beyond | moderno → **ultra**moderno (ultramodern) |
| **iper-** | hyper, over | attivo → **iper**attivo (hyperactive), mercato → **iper**mercato (hypermarket) |

### 8.3 Spatial and Temporal Prefixes

| Prefix | Meaning | Examples |
|--------|---------|---------|
| **pre-** | before | vedere → **pre**vedere (to foresee), guerra → **pre**guerra (prewar) |
| **post-** | after | guerra → **post**guerra (postwar), moderno → **post**moderno (postmodern) |
| **sotto-** | under, below | terra → **sotto**terra (underground), titolo → **sotto**titolo (subtitle) |
| **sopra-** | above, over | tutto → **sopra**ttutto (above all), vivvere → **sopra**vvivere (to survive) |
| **inter-** | between | nazionale → **inter**nazionale (international) |
| **extra-** | outside | terrestre → **extra**terrestre (extraterrestrial) |

## 9. Word Formation: Suffixes

### 9.1 Noun-Forming Suffixes

| Suffix | Meaning | Examples |
|--------|---------|---------|
| **-zione/-sione** | action, result | informare → informa**zione** (information), decidere → deci**sione** (decision) |
| **-mento** | result of action | trattare → tratta**mento** (treatment), inquinare → inquina**mento** (pollution) |
| **-tore/-trice** | agent (person who does) | lavorare → lavora**tore** (worker, m.), lavorare → lavora**trice** (worker, f.) |
| **-ista** | specialist/adherent | giornale → giornal**ista** (journalist), arte → art**ista** (artist) |
| **-eria** | place/shop | libro → libr**eria** (bookstore), pasta → past**eria** (pasta/pastry shop) |
| **-ura** | result / abstract quality | aprire → apert**ura** (opening), colto → colt**ura** (culture) |

### 9.2 Adjective-Forming Suffixes

| Suffix | Meaning | Examples |
|--------|---------|---------|
| **-oso/-osa** | full of | fama → fam**oso** (famous), pericolo → pericol**oso** (dangerous) |
| **-bile** | -able, -ible | mangiare → mangia**bile** (edible), incredere → incredi**bile** (incredible) |

## 11. False Friends: Italian-Spanish

For learners who also know Spanish, these are common traps:

| Italian | Spanish Look-alike | Italian Meaning | Spanish Meaning |
|---------|-------------------|----------------|-----------------|
| **burro** | burro (donkey) | butter | donkey (asino in Italian) |
| **caldo** | caldo (broth) | hot/warm | broth (brodo in Italian) |
| **gamba** | gamba (shrimp) | leg | shrimp (gambero in Italian) |
| **guardia** | guardia (guard) | guard (same) | guard (same -- but *guardare* = to look, not to guard) |
| **largo** | largo (long) | wide | long (lungo in Italian) |
| **salire** | salir (to leave) | to go up | to leave (uscire in Italian) |
| **subire** | subir (to go up) | to suffer/undergo | to go up (salire in Italian) |
| **aceto** | aceite (oil) | vinegar | oil (olio in Italian) |
| **guardare** | guardar (to keep) | to look at / watch | to keep (tenere in Italian) |
| **pronto** | pronto (soon) | ready / hello (phone) | soon (presto in Italian) |
| **tienda** | tienda (store, Sp.) | tenda (tent / curtain) | store = negozio in Italian |
| **embarazada** | (Spanish: pregnant) | -- | *imbarazzata* = embarrassed (not pregnant: *incinta*) |

### Exercise 3: False Friends

Choose the correct Italian translation. Beware of false friends!

1. "Actually, I don't agree."
   - a) Attualmente, non sono d'accordo.
   - b) In realtà, non sono d'accordo.

2. "I went to the library."
   - a) Sono andato alla libreria.
   - b) Sono andato alla biblioteca.

3. "My parents are coming tomorrow."
   - a) I miei parenti vengono domani.
   - b) I miei genitori vengono domani.

4. "The factory is closed on Sundays."
   - a) La fattoria è chiusa la domenica.
   - b) La fabbrica è chiusa la domenica.

5. "She is very sensitive."
   - a) È molto sensibile.
   - b) È molto sensata.

6. "It's very cold today." (Temperature)
   - a) È molto caldo oggi.
   - b) Fa molto freddo oggi.

<details>
<summary>Answers</summary>

1. **b) In realtà, non sono d'accordo.** (*Attualmente* means "currently," not "actually.")
2. **b) Sono andato alla biblioteca.** (*Libreria* means "bookstore," not "library.")
3. **b) I miei genitori vengono domani.** (*Parenti* means "relatives," not "parents.")
4. **b) La fabbrica è chiusa la domenica.** (*Fattoria* means "farm," not "factory.")
5. **a) È molto sensibile.** (This is a correct match -- *sensibile* does mean "sensitive." The false friend is *sensibile* ≠ "sensible"; "sensible" = *sensato/a*.)
6. **b) Fa molto freddo oggi.** (*Caldo* means "hot," not "cold"!)

</details>

5. informazione = _________ + _________ (meaning: ________)

<details>
<summary>Answers</summary>

1. **lavorare** + **-tore** = worker (one who works)
2. **libro** + **-eria** = bookstore (place of books)
3. **giornale** + **-ista** = journalist (specialist in newspapers)
4. **pericolo** + **-oso** = dangerous (full of danger)
5. **informare** + **-zione** = information (result of informing)

</details>

### Exercise 7: Italian-Spanish False Friends

For Spanish speakers: choose the correct Italian word.

1. "Butter" in Italian:
   - a) burro  b) manteca  c) mantequilla

2. "Hot" in Italian:
   - a) caliente  b) caldo  c) calor

3. "To go up" in Italian:
   - a) subire  b) salire  c) subir

4. "Vinegar" in Italian:
   - a) aceite  b) aceto  c) vinagre

5. "Ready" or "Hello (on the phone)" in Italian:
   - a) presto  b) pronto  c) listo

<details>
<summary>Answers</summary>

1. **a) burro** (In Italian, *burro* = butter, not donkey!)
2. **b) caldo** (In Italian, *caldo* = hot/warm, not broth like in Spanish)
3. **b) salire** (In Italian, *salire* = to go up; *subire* = to suffer/undergo)
4. **b) aceto** (In Italian, *aceto* = vinegar; the Spanish *aceite* means oil = *olio*)
5. **b) pronto** (In Italian, *pronto* = ready or phone greeting; *presto* = soon)

</details>
