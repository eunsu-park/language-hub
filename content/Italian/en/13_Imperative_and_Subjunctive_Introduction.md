## 1. Imperative: Overview

The imperativo (imperative) is used for commands, requests, instructions, and suggestions. Italian has imperative forms for four persons:

```
+----------------------------------------------------------+
|  IMPERATIVE PERSONS                                       |
|                                                          |
|  tu    — informal singular (friends, family)              |
|  Lei   — formal singular (strangers, superiors)           |
|  noi   — inclusive "let's" (noi)                          |
|  voi   — plural (group address)                           |
|                                                          |
|  Note: The "Lei" form borrows from the subjunctive.       |
|  Note: There is NO imperative for io or loro in modern    |
|        standard Italian.                                  |
+----------------------------------------------------------+
```

---

## 2. Regular Imperative Forms

### -ARE Verbs (e.g., parlare — to speak)

| Person | Form | Example |
|--------|------|---------|
| tu | parl**a** | Parla più piano! (Speak more quietly!) |
| Lei | parl**i** | Parli più piano! (Speak more quietly! — formal) |
| noi | parl**iamo** | Parliamo italiano! (Let's speak Italian!) |
| voi | parl**ate** | Parlate più forte! (Speak louder!) |

### -ERE Verbs (e.g., scrivere — to write)

| Person | Form | Example |
|--------|------|---------|
| tu | scriv**i** | Scrivi il tuo nome! (Write your name!) |
| Lei | scriv**a** | Scriva il Suo nome! (Write your name! — formal) |
| noi | scriv**iamo** | Scriviamo una lettera! (Let's write a letter!) |
| voi | scriv**ete** | Scrivete tutto! (Write everything!) |

### -IRE Verbs (e.g., dormire — to sleep)

| Person | Form | Example |
|--------|------|---------|
| tu | dorm**i** | Dormi bene! (Sleep well!) |
| Lei | dorm**a** | Dorma bene! (Sleep well! — formal) |
| noi | dorm**iamo** | Dormiamo un po'! (Let's sleep a bit!) |
| voi | dorm**ite** | Dormite presto! (Go to sleep early!) |

### -IRE Verbs with -ISC- (e.g., capire — to understand, finire — to finish)

| Person | Form | Example |
|--------|------|---------|
| tu | cap**isci** | Capisci bene! (Understand well!) |
| Lei | cap**isca** | Capisca bene! (Understand well! — formal) |
| noi | cap**iamo** | Capiamo la situazione! (Let's understand the situation!) |
| voi | cap**ite** | Capite? (Do you understand? / Understand!) |

### Summary Table

| Person | -ARE | -ERE | -IRE | -IRE (-isc-) |
|--------|------|------|------|---------------|
| **tu** | -a | -i | -i | -isci |
| **Lei** | -i | -a | -a | -isca |
| **noi** | -iamo | -iamo | -iamo | -iamo |
| **voi** | -ate | -ete | -ite | -ite |

> **Key observation**: For **tu**, the -ARE imperative ends in **-a** (like the present indicative). For -ERE/-IRE, the tu form is identical to the present indicative tu form. The **Lei** form always comes from the **subjunctive**.

## 3. Irregular Imperatives

Several common verbs have irregular imperative forms, especially for **tu** and **Lei**.

### Most Important Irregular Imperatives

| Infinitive | tu | Lei | noi | voi |
|------------|-----|------|------|------|
| **essere** (to be) | sii | sia | siamo | siate |
| **avere** (to have) | abbi | abbia | abbiamo | abbiate |
| **andare** (to go) | va' / vai | vada | andiamo | andate |
| **fare** (to do/make) | fa' / fai | faccia | facciamo | fate |
| **dire** (to say/tell) | di' | dica | diciamo | dite |
| **dare** (to give) | da' / dai | dia | diamo | date |
| **stare** (to stay/be) | sta' / stai | stia | stiamo | state |
| **venire** (to come) | vieni | venga | veniamo | venite |
| **uscire** (to go out) | esci | esca | usciamo | uscite |
| **tenere** (to hold) | tieni | tenga | teniamo | tenete |
| **sapere** (to know) | sappi | sappia | sappiamo | sappiate |
| **scegliere** (to choose) | scegli | scelga | scegliamo | scegliete |

> **Note on truncated forms**: *Va'*, *fa'*, *di'*, *da'*, *sta'* are shortened forms. The full forms *vai*, *fai*, *dai*, *stai* are also accepted and increasingly common. The apostrophe indicates truncation, not an accent.

### Examples in Context

| Italian | English |
|---------|---------|
| **Sii** gentile! | Be kind! |
| **Abbi** pazienza! | Be patient! (lit. Have patience!) |
| **Va'** / **Vai** a casa! | Go home! |
| **Fa'** / **Fai** attenzione! | Pay attention! / Be careful! |
| **Di'** la verità! | Tell the truth! |
| **Da'** / **Dai** una mano a tuo fratello! | Give your brother a hand! |
| **Sta'** / **Stai** zitto! | Be quiet! |
| **Vieni** qui! | Come here! |
| **Esci** di qui! | Get out of here! |

## 4. Negative Imperative

### Tu Form: NON + INFINITIVE

The negative imperative for **tu** is unique: use **non** + the **infinitive** (NOT the conjugated form).

| Affirmative | Negative |
|------------|---------|
| Parla! (Speak!) | **Non parlare!** (Don't speak!) |
| Scrivi! (Write!) | **Non scrivere!** (Don't write!) |
| Dormi! (Sleep!) | **Non dormire!** (Don't sleep!) |
| Va'! (Go!) | **Non andare!** (Don't go!) |
| Di'! (Tell!) | **Non dire niente!** (Don't say anything!) |
| Fa'! (Do!) | **Non fare così!** (Don't do that!) |

### Lei / Noi / Voi: NON + Imperative Form

For all other persons, simply place **non** before the regular imperative form.

| Affirmative | Negative |
|------------|---------|
| Parli! (Lei) | **Non parli!** |
| Parliamo! (noi) | **Non parliamo!** |
| Parlate! (voi) | **Non parlate!** |
| Vada! (Lei) | **Non vada!** |

## 5. Pronoun Placement with Imperatives

### Affirmative: Pronouns ATTACH to the End

With **tu, noi, voi** affirmative imperatives, pronouns attach to the end of the verb.

| Italian | English |
|---------|---------|
| Scrivimi! (Scrivi + mi) | Write to me! |
| Dimmi! (Di' + mi) | Tell me! |
| Dammelo! (Da' + me + lo) | Give it to me! |
| Fallo! (Fa' + lo) | Do it! |
| Chiamalo! (Chiama + lo) | Call him! |
| Andiamoci! (Andiamo + ci) | Let's go there! |
| Portatelo! (Portate + lo) | Bring it! (pl.) |

> **Doubling rule**: With the truncated forms *da'*, *di'*, *fa'*, *sta'*, *va'*, the initial consonant of the pronoun **doubles**: da' + mi → **dammi**, di' + lo → **dillo**, fa' + lo → **fallo**, sta' + ci → **stacci**.
> Exception: **gli** never doubles: di' + gli → **digli** (not *digligli*).

### Formal (Lei): Pronouns PRECEDE the Verb

| Italian | English |
|---------|---------|
| Mi scriva! | Write to me! (formal) |
| Lo faccia! | Do it! (formal) |
| Mi dica! | Tell me! (formal) |
| Glielo dia! | Give it to him/her! (formal) |
| Si accomodi! | Please sit down! (formal) |

### Negative Tu: Two Options

With negative **tu** commands, the pronoun can either attach or precede:

| Option 1 (attached) | Option 2 (preceding) | English |
|---------------------|---------------------|---------|
| Non dirlo! | Non lo dire! | Don't say it! |
| Non farlo! | Non lo fare! | Don't do it! |
| Non preoccuparti! | Non ti preoccupare! | Don't worry! |

## 7. Congiuntivo Presente: Formation

The congiuntivo (subjunctive) expresses subjectivity: wishes, doubts, opinions, emotions, and uncertainty. It always appears in a **dependent clause** introduced by **che**.

### Regular Congiuntivo Presente

| Person | -ARE (parlare) | -ERE (scrivere) | -IRE (dormire) | -IRE -isc- (capire) |
|--------|----------------|-----------------|----------------|---------------------|
| che io | parl**i** | scriv**a** | dorm**a** | cap**isca** |
| che tu | parl**i** | scriv**a** | dorm**a** | cap**isca** |
| che lui/lei/Lei | parl**i** | scriv**a** | dorm**a** | cap**isca** |
| che noi | parl**iamo** | scriv**iamo** | dorm**iamo** | cap**iamo** |
| che voi | parl**iate** | scriv**iate** | dorm**iate** | cap**iate** |
| che loro | parl**ino** | scriv**ano** | dorm**ano** | cap**iscano** |

> **Key observation**: For -ARE verbs, the singular forms all end in **-i** and the **loro** form ends in **-ino**. For -ERE/-IRE verbs, the singular forms all end in **-a** and **loro** ends in **-ano**. The **io, tu, lui/lei** forms are **identical** — context or pronouns disambiguate.

### Congiuntivo vs. Indicativo: An Example

| Indicative (fact) | Subjunctive (opinion/wish) |
|-------------------|--------------------------|
| So che Marco **parla** italiano. (I know Marco speaks Italian.) | Penso che Marco **parli** italiano. (I think Marco speaks Italian.) |
| È vero che Maria **viene**. (It's true Maria is coming.) | Spero che Maria **venga**. (I hope Maria comes.) |

## 9. Subjunctive Triggers: Wishes and Desires

The subjunctive is required after expressions of wish, desire, or preference in the main clause.

| Trigger Expression | Example |
|-------------------|---------|
| **volere che** (to want that) | Voglio che tu **venga** alla festa. (I want you to come to the party.) |
| **desiderare che** (to wish that) | Desidero che tutto **vada** bene. (I wish everything goes well.) |
| **sperare che** (to hope that) | Spero che **faccia** bel tempo. (I hope the weather is nice.) |
| **preferire che** (to prefer that) | Preferisco che tu **stia** qui. (I prefer that you stay here.) |
| **augurare che** (to wish that) | Mi auguro che **sia** vero. (I wish it's true.) |
| **aspettare che** (to wait for ... to) | Aspetto che lui **finisca**. (I'm waiting for him to finish.) |

> **Important**: When the subject of both clauses is the **same**, use the **infinitive** instead of the subjunctive:
> - Voglio **andare**. (I want to go.) — same subject
> - Voglio che tu **vada**. (I want you to go.) — different subjects

---

## 10. Subjunctive Triggers: Doubt and Opinion

| Trigger Expression | Example |
|-------------------|---------|
| **pensare che** (to think that) | Penso che **sia** una buona idea. (I think it's a good idea.) |
| **credere che** (to believe that) | Credo che **abbia** ragione. (I believe he's right.) |
| **dubitare che** (to doubt that) | Dubito che **venga**. (I doubt he'll come.) |
| **non essere sicuro/a che** | Non sono sicuro che **possa** venire. (I'm not sure he can come.) |
| **immaginare che** (to imagine that) | Immagino che **sia** difficile. (I imagine it's difficult.) |
| **supporre che** (to suppose that) | Suppongo che **abbia** avuto problemi. (I suppose he had problems.) |
| **sembrare che** (to seem that) | Sembra che **piova**. (It seems like it's raining.) |
| **parere che** (to appear that) | Pare che **sia** partito. (It appears he has left.) |

> **Note**: *Sapere che* (to know that) and *essere sicuro che* (to be sure that) take the **indicative** because they express certainty, not doubt:
> - So che **è** vero. (I know it's true.) — indicative
> - Non so se **sia** vero. (I don't know if it's true.) — subjunctive

---

## 11. Subjunctive Triggers: Emotion

| Trigger Expression | Example |
|-------------------|---------|
| **essere felice/contento/a che** | Sono felice che tu **sia** qui. (I'm happy you're here.) |
| **essere triste che** | È triste che lei **parta**. (It's sad that she's leaving.) |
| **avere paura che** | Ho paura che **succeda** qualcosa. (I'm afraid something will happen.) |
| **dispiacere che** | Mi dispiace che non **possa** venire. (I'm sorry he can't come.) |
| **essere sorpreso/a che** | Sono sorpreso che non lo **sappia**. (I'm surprised he doesn't know.) |
| **essere arrabbiato/a che** | È arrabbiato che non **l'abbia** chiamato. (He's angry I didn't call him.) |
| **piacere che** | Mi piace che tu **stia** bene. (I'm glad you're doing well.) |
| **vergognarsi che** | Si vergogna che tutti lo **sappiano**. (He's ashamed everyone knows.) |

### Impersonal Expressions That Trigger Subjunctive

| Expression | Example |
|------------|---------|
| **È necessario che** | È necessario che tu **venga**. (It's necessary that you come.) |
| **È importante che** | È importante che **capiscano**. (It's important they understand.) |
| **È possibile che** | È possibile che **piova**. (It's possible it'll rain.) |
| **È impossibile che** | È impossibile che **sia** vero. (It's impossible it's true.) |
| **È probabile che** | È probabile che **arrivi** in ritardo. (It's likely he'll arrive late.) |
| **È meglio che** | È meglio che tu **stia** zitto. (It's better that you stay quiet.) |
| **È strano che** | È strano che non **risponda**. (It's strange he doesn't answer.) |
| **Bisogna che** | Bisogna che **studiamo** di più. (We need to study more.) |
| **Basta che** | Basta che tu **faccia** il tuo dovere. (It's enough that you do your duty.) |

### Exercise 4: Indicative or Subjunctive?

Choose the correct form.

1. So che Marco (parlare — parla / parli) italiano.
2. Penso che Maria (essere — è / sia) intelligente.
3. È vero che loro (venire — vengono / vengano) domani.
4. Spero che tu (stare — stai / stia) bene.
5. Voglio che voi (capire — capite / capiate) la situazione.
6. È possibile che (piovere — piove / piova) domani.

8. Sono sicuro che tu (avere — hai / abbia) ragione.

<details>
<summary>Answers</summary>

1. **parla** (indicative — *so che* = certainty)

5. **capiate** (subjunctive — *voglio che* = desire)
6. **piova** (subjunctive — *è possibile che* = possibility)
7. **sappia** (subjunctive — *dubito che* = doubt)
8. **hai** (indicative — *sono sicuro che* = certainty)
