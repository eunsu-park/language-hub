## 1. 부사 형성: -MENTE

대부분의 이탈리아어 부사는 형용사의 여성 단수형에 **-mente**를 추가하여 형성됩니다.

### 규칙적 형성

| 형용사 (남성) | 형용사 (여성) | 부사 | 한국어 |
|-------------|-------------|------|--------|
| lento | lent**a** | lenta**mente** | 느리게 |
| chiaro | chiar**a** | chiara**mente** | 명확하게 |
| rapido | rapid**a** | rapida**mente** | 빠르게 |
| sicuro | sicur**a** | sicura**mente** | 확실히/틀림없이 |
| vero | ver**a** | vera**mente** | 정말로/진실로 |
| sincero | sincer**a** | sincera**mente** | 진심으로 |
| perfetto | perfett**a** | perfetta**mente** | 완벽하게 |
| completo | complet**a** | completa**mente** | 완전히 |
| fortunato | fortunat**a** | fortunata**mente** | 다행히 |
| ovvio | ovvi**a** | ovvia**mente** | 분명히 |

### -LE 또는 -RE로 끝나는 형용사: 마지막 -E를 탈락

| 형용사 | 부사 | 한국어 |
|--------|------|--------|
| facile | facil**mente** | 쉽게 |
| difficile | difficil**mente** | 어렵게 |
| gentile | gentil**mente** | 친절하게 |
| normale | normal**mente** | 보통으로/정상적으로 |
| speciale | special**mente** | 특별히 |
| regolare | regolar**mente** | 규칙적으로 |
| particolare | particolar**mente** | 특히 |
| probabile | probabilmente | 아마도 |

> **예외**: -le 또는 -re로 끝나는 형용사에서 마지막 -e 앞에 모음 군집이 있으면, 완전한 -mente가 추가됩니다: *leale → lealmente* (충성스럽게).

### 부사로도 쓰이는 형용사 (-mente 없이)

일부 형용사는 변화 없이 부사로 사용됩니다:

| 단어 | 형용사로서 | 부사로서 |
|------|-----------|---------|
| **forte** | un uomo forte (강한 남자) | Parla forte! (크게 말해!) |
| **piano** | un piano tavolo (평평한 테이블) | Parla piano! (조용히 말해!) |
| **chiaro** | un cielo chiaro (맑은 하늘) | Parla chiaro! (분명하게 말해!) |
| **veloce** | un treno veloce (빠른 기차) | Corre veloce. (그는 빠르게 달린다.) |
| **duro** | un lavoro duro (힘든 일) | Lavora duro. (그는 열심히 일한다.) |
| **alto** | un muro alto (높은 벽) | Parla alto! (크게 말해!) |
| **basso** | un tono basso (낮은 톤) | Parla basso. (낮은 소리로 말해.) |
| **giusto** | la risposta giusta (올바른 답) | Hai detto giusto. (올바르게 말했어.) |

## 2. 범주별 일반 부사

### 시간 (Avverbi di Tempo)

| 이탈리아어 | 한국어 | 예시 |
|-----------|--------|------|
| **già** | 이미 | Ho già mangiato. (이미 먹었다.) |
| **ancora** | 아직 / 여전히 / 다시 | Non è ancora arrivato. (아직 도착하지 않았다.) |
| **sempre** | 항상 | Vai sempre al bar? (항상 바에 가니?) |
| **mai** | 절대 / 한 번도 | Non piove mai. (비가 절대 오지 않는다.) |
| **spesso** | 자주 | Vado spesso al cinema. (자주 영화관에 간다.) |
| **subito** | 바로/즉시 | Vengo subito! (바로 갈게!) |
| **appena** | 방금 | Sono appena arrivato. (방금 도착했다.) |
| **prima** | 전에 / 먼저 | Arrivo prima di te. (너보다 먼저 도착한다.) |
| **dopo** | 후에 / 나중에 | Ne parliamo dopo. (나중에 이야기하자.) |
| **poi** | 그 다음에 / 나중에 | E poi che è successo? (그 다음에 무슨 일이 있었어?) |
| **adesso / ora** | 지금 | Cosa fai adesso? (지금 뭐 하고 있어?) |
| **presto** | 곧 / 일찍 | Ci vediamo presto. (곧 보자.) |
| **tardi** | 늦게 | È arrivato tardi. (그는 늦게 도착했다.) |
| **ormai** | 이제/이미 | Ormai è troppo tardi. (이제 너무 늦었다.) |
| **finora / finalmente** | 지금까지 / 마침내 | Finora tutto bene. (지금까지 다 잘 됐다.) |
| **talvolta / a volte** | 때때로 | A volte mi sento solo. (때때로 외로움을 느낀다.) |

### 장소 (Avverbi di Luogo)

| 이탈리아어 | 한국어 | 예시 |
|-----------|--------|------|
| **qui / qua** | 여기 | Vieni qui! (여기로 와!) |
| **lì / là** | 거기 | Il libro è lì. (책은 거기에 있다.) |
| **su** | 위로 | Guarda su! (위를 봐!) |
| **giù** | 아래로 | Vai giù! (아래로 가!) |
| **dentro** | 안에/안으로 | Vieni dentro! (안으로 들어와!) |
| **fuori** | 밖에/밖으로 | Andiamo fuori. (밖으로 나가자.) |
| **davanti** | 앞에 | La macchina è davanti. (차가 앞에 있다.) |
| **dietro** | 뒤에 | Il giardino è dietro. (정원은 뒤에 있다.) |
| **vicino** | 가까이 | Abito vicino. (가까이 산다.) |
| **lontano** | 멀리 | Roma è lontano. (로마는 멀다.) |
| **sopra** | 위에 | Il gatto è sopra il tavolo. (고양이가 테이블 위에 있다.) |
| **sotto** | 아래에 | Il cane è sotto il tavolo. (개가 테이블 아래에 있다.) |
| **dappertutto / ovunque** | 어디에나 | C'è gente dappertutto. (어디에나 사람들이 있다.) |
| **da nessuna parte** | 아무 데도 | Non vado da nessuna parte. (아무 데도 가지 않는다.) |
| **altrove** | 다른 곳에 | Cerchiamo altrove. (다른 곳에서 찾아보자.) |

### 양태 (Avverbi di Modo)

| 이탈리아어 | 한국어 | 예시 |
|-----------|--------|------|
| **bene** | 잘 | Parli bene l'italiano. (이탈리아어를 잘 말한다.) |
| **male** | 나쁘게/안 좋게 | Mi sento male. (몸이 안 좋다.) |
| **così** | 이렇게/그렇게 | Non fare così! (그렇게 하지 마!) |
| **insieme** | 함께 | Andiamo insieme. (함께 가자.) |
| **apposta / di proposito** | 일부러 | L'ha fatto apposta. (그는 일부러 했다.) |
| **volentieri** | 기꺼이 | Vengo volentieri! (기꺼이 갈게!) |
| **per caso** | 우연히 | L'ho incontrato per caso. (우연히 그를 만났다.) |
| **in fretta** | 서둘러 | Mangia sempre in fretta. (그는 항상 서둘러 먹는다.) |
| **piano piano** | 조금씩 | Piano piano imparo. (조금씩 배우고 있다.) |

### 정도/수량 (Avverbi di Quantità)

| 이탈리아어 | 한국어 | 예시 |
|-----------|--------|------|
| **molto** | 매우/많이 | È molto bello. (매우 아름답다.) |
| **poco** | 적게/별로 | Mangia poco. (적게 먹는다.) |
| **troppo** | 너무/너무 많이 | Parli troppo. (너무 많이 말한다.) |
| **abbastanza** | 충분히/꽤 | È abbastanza grande. (충분히 크다.) |
| **tanto** | 매우/그만큼 | Ti voglio tanto bene. (너를 정말 많이 사랑해.) |
| **quanto** | 얼마나 | Quanto costa? (얼마예요?) |
| **quasi** | 거의 | È quasi mezzanotte. (거의 자정이다.) |
| **appena** | 겨우/방금 | L'ho appena visto. (방금 그를 봤다.) |
| **solo / soltanto** | 단지/오직 | Ho solo cinque euro. (5유로밖에 없다.) |
| **almeno** | 적어도 | Almeno dimmi la verità. (적어도 진실을 말해.) |
| **piuttosto** | 오히려/꽤 | È piuttosto difficile. (꽤 어렵다.) |
| **proprio** | 정말/정확히 | È proprio così. (정말 그렇다.) |

### 긍정, 부정, 의심

| 이탈리아어 | 한국어 | 예시 |
|-----------|--------|------|
| **sì** | 네 | Sì, vengo. (네, 갈게요.) |
| **no** | 아니오 | No, non posso. (아니요, 못 해요.) |
| **certo / certamente** | 물론/확실히 | Certo che vengo! (물론 갈게!) |
| **sicuramente** | 확실히 | Sicuramente pioverà. (확실히 비가 올 것이다.) |
| **forse / magari** | 아마/혹시 | Forse domani. (아마 내일.) |
| **probabilmente** | 아마도 | Probabilmente ha ragione. (아마도 그가 맞을 것이다.) |
| **neanche / nemmeno / neppure** | ~조차 아닌 | Neanche io. (나도 아니야.) |
| **mica** | 전혀 (~않다) | Non è mica facile. (전혀 쉽지 않다.) |
| **affatto** | 전혀 (부정과 함께) | Non mi piace affatto. (전혀 마음에 들지 않는다.) |
| **per niente** | 전혀 (~않다) | Non è stanco per niente. (전혀 피곤하지 않다.) |

## 3. 부사의 위치

### 일반 규칙

```
+----------------------------------------------------------+
|  부사 위치 가이드라인                                        |
|                                                          |
|  1. 양태 부사: 동사 뒤                                     |
|     Parla bene l'italiano.                               |
|                                                          |
|  2. 시간/빈도: 유동적 (문장 처음, 동사 뒤, 문장 끝)          |
|     Sempre vado al bar. / Vado sempre al bar.            |
|                                                          |
|  3. 정도 부사: 수식하는 형용사/부사 앞                       |
|     È molto bello. / Parla troppo velocemente.           |
|                                                          |
|  4. 복합 시제에서: 보통 조동사와 과거 분사 사이                |
|     Ho già mangiato. / Non ho mai visto.                 |
|                                                          |
|  5. 부정 부사: NON은 동사 앞, 부정어는 보통                  |
|     동사 뒤 또는 조동사와 과거 분사 사이                     |
|     Non vado mai. / Non ho mai capito.                   |
+----------------------------------------------------------+
```

### 복합 시제에서의 위치

| 부사 | 위치 | 예시 |
|------|------|------|
| **già** | 조동사와 과거 분사 사이 | Ho **già** mangiato. |
| **mai** | 조동사와 과거 분사 사이 | Non ho **mai** visto Roma. |
| **ancora** | 조동사와 과거 분사 사이 | Non è **ancora** arrivato. |
| **sempre** | 조동사와 과거 분사 사이 | Ha **sempre** lavorato duro. |
| **appena** | 조동사와 과거 분사 사이 | Sono **appena** arrivato. |
| **anche** | 수식하는 단어 앞 | **Anche** io ho mangiato. / Ho mangiato **anche** io. |

## 4. 비교급과 최상급 부사

### 비교급

| 구조 | 예시 | 한국어 |
|------|------|--------|
| **più** + 부사 + **di/che** | Parla **più velocemente di** me. | 그는 나보다 더 빨리 말한다. |
| **meno** + 부사 + **di/che** | Arriva **meno spesso di** prima. | 그는 전보다 덜 자주 온다. |
| **(così)... come** / **(tanto)... quanto** | Parla **(così) bene come** te. | 그는 너만큼 잘 말한다. |

### 불규칙 비교급

| 부사 | 비교급 | 한국어 |
|------|--------|--------|
| **bene** (잘) | **meglio** (더 잘) | Parla meglio di me. (그는 나보다 더 잘 말한다.) |
| **male** (나쁘게) | **peggio** (더 나쁘게) | Sta peggio di ieri. (어제보다 상태가 더 나쁘다.) |
| **molto** (많이) | **più** / **di più** (더 많이) | Studia di più! (더 많이 공부해!) |
| **poco** (적게) | **meno** / **di meno** (더 적게) | Lavora di meno. (그는 더 적게 일한다.) |

### 최상급

| 유형 | 구조 | 예시 | 한국어 |
|------|------|------|--------|
| **상대** | il più / il meno + 부사 | Corre **il più velocemente** possibile. | 가능한 한 가장 빨리 달린다. |
| **절대** | 부사 + -issimo | Benissimo! (bene → benissimo) | 아주 잘! |
| **절대** | molto + 부사 | Parla **molto bene**. | 매우 잘 말한다. |

### 일반 부사의 절대 최상급

| 부사 | 최상급 | 한국어 |
|------|--------|--------|
| bene | **benissimo** | 아주 잘 / 훌륭히 |
| male | **malissimo** | 아주 나쁘게 / 최악으로 |
| molto | **moltissimo** | 매우 많이 |
| poco | **pochissimo** | 아주 적게 |
| tardi | **tardissimo** | 아주 늦게 |
| presto | **prestissimo** | 아주 일찍 / 아주 곧 |
| spesso | **spessissimo** | 아주 자주 |
| vicino | **vicinissimo** | 아주 가까이 |
| lontano | **lontanissimo** | 아주 멀리 |

## 5. 등위 접속사

등위 접속사는 **동등한 중요성**을 가진 단어, 구, 절을 연결합니다.

### 연결 (추가)

| 접속사 | 한국어 | 예시 |
|--------|--------|------|
| **e** / **ed** (모음 앞) | 그리고 | Marco **e** Anna sono amici. |
| **anche** | 또한/~도 | **Anche** io vengo. (나도 간다.) |
| **inoltre** / **in più** | 게다가/더욱이 | È bravo, **inoltre** è simpatico. |
| **né... né** | ~도 아니고 ~도 아닌 | **Né** io **né** lui sappiamo. |
| **sia... sia** / **sia... che** | ~와 ~모두 | **Sia** Marco **sia** Anna vengono. |

> **Ed**는 **e**로 시작하는 단어 앞에 사용됩니다: Marco **ed** Elena (음조 조화를 위해). 선택 사항이지만 선호됩니다.

### 대립 (대조)

| 접속사 | 한국어 | 예시 |
|--------|--------|------|
| **ma** | 그러나 | È stanco **ma** felice. (피곤하지만 행복하다.) |
| **però** | 그러나/하지만 | È caro, **però** è buono. (비싸지만 맛있다.) |
| **tuttavia** | 그럼에도 불구하고 | È difficile; **tuttavia**, ci proviamo. |
| **eppure** | 그럼에도 | Lo sapeva, **eppure** non ha detto niente. |
| **anzi** | 오히려 | Non è brutto, **anzi**, è bellissimo! |
| **bensì** | 오히려 (격식) | Non è stupido, **bensì** svogliato. |
| **invece** | 대신/반면에 | Io resto, tu **invece** vai. |

### 이접 (선택)

| 접속사 | 한국어 | 예시 |
|--------|--------|------|
| **o** / **oppure** | 또는 | Vuoi tè **o** caffè? |
| **altrimenti** | 그렇지 않으면 | Sbrigati, **altrimenti** perdiamo il treno! |
| **o... o** | ~이거나 ~이거나 | **O** vieni **o** resti. |

### 결론 (결과)

| 접속사 | 한국어 | 예시 |
|--------|--------|------|
| **quindi** / **dunque** | 그러므로/따라서 | Era tardi, **quindi** siamo andati a casa. |
| **perciò** | 그래서/그 때문에 | Piove, **perciò** resto a casa. |
| **di conseguenza** | 결과적으로 | Non ha studiato, **di conseguenza** non ha superato l'esame. |
| **allora** | 그러면/그래서 | **Allora**, che facciamo? |

### 설명 (명확화)

| 접속사 | 한국어 | 예시 |
|--------|--------|------|
| **cioè** | 즉/다시 말해 | È poliglotta, **cioè** parla molte lingue. |
| **infatti** | 사실/실제로 | Pensavo fosse facile; **infatti**, non lo è. |
| **ossia** | 즉/다시 말해 | È un medico, **ossia** un chirurgo. |

## 6. 종속 접속사

종속 접속사는 종속절을 도입합니다.

### 원인 (이유)

| 접속사 | 한국어 | 법 | 예시 |
|--------|--------|-----|------|
| **perché** | ~때문에 | 직설법 | Non esco **perché** piove. |
| **poiché / dato che / siccome** | ~이므로 / ~이기 때문에 | 직설법 | **Siccome** piove, resto a casa. |
| **visto che** | ~인 것을 보면 | 직설법 | **Visto che** sei qui, parliamo. |
| **per il fatto che** | ~라는 사실 때문에 | 직설법 | L'ho fatto **per il fatto che** era necessario. |

> **참고**: *Siccome*, *dato che*, *poiché*는 보통 문장 처음에 옵니다. *Perché*는 보통 두 번째 절에 나타납니다.

### 시간

| 접속사 | 한국어 | 예시 |
|--------|--------|------|
| **quando** | ~할 때 | **Quando** arrivi, chiamami. |
| **mentre** | ~하는 동안 | **Mentre** mangiavo, leggevo il giornale. |
| **appena / non appena** | ~하자마자 | **Appena** arriva, usciamo. |
| **prima che** | ~하기 전에 (+ 접속법) | Parti **prima che** piova. |
| **dopo che** | ~한 후에 | **Dopo che** ha mangiato, è uscito. |
| **finché / fino a quando** | ~할 때까지 / ~하는 한 | Aspetto **finché** non arrivi. |
| **da quando** | ~이래로 | **Da quando** sono arrivato, piove. |
| **ogni volta che** | ~할 때마다 | **Ogni volta che** lo vedo, mi sorride. |

### 양보 (~에도 불구하고)

| 접속사 | 한국어 | 법 | 예시 |
|--------|--------|-----|------|
| **benché / sebbene** | ~에도 불구하고 | 접속법 | **Sebbene** piova, esco. |
| **nonostante (che)** | ~에도 불구하고 | 접속법 | **Nonostante** sia stanco, lavora. |
| **malgrado** | ~에도 불구하고 | 접속법 | **Malgrado** faccia freddo, usciamo. |
| **anche se** | ~일지라도 / 비록 ~이지만 | 직설법 | **Anche se** piove, esco. |
| **pur** + 제룬디오 | ~에도 불구하고 | — | **Pur essendo** stanco, lavora. |

### 목적 (~하기 위해)

| 접속사 | 한국어 | 법 | 예시 |
|--------|--------|-----|------|
| **affinché / perché** (= ~하도록) | ~하기 위해 | 접속법 | Parlo piano **affinché** tu **capisca**. |
| **in modo che** | ~하도록 / ~하게끔 | 접속법 | Spiego **in modo che** tutti **capiscano**. |
| **per** + 부정사 | ~하기 위해 | — | Studio **per** imparare. |

### 조건

| 접속사 | 한국어 | 법 | 예시 |
|--------|--------|-----|------|
| **se** | 만약 | periodo ipotetico 규칙 참조 | **Se** piove, resto a casa. |
| **a meno che (non)** | ~하지 않는 한 | 접속법 | Vengo **a meno che non** piova. |
| **purché / a patto che / a condizione che** | ~하는 조건으로 | 접속법 | Ti aiuto **purché** tu studi. |
| **nel caso (in cui)** | ~하는 경우에 | 접속법 | **Nel caso** piova, porta l'ombrello. |

### 비교

| 접속사 | 한국어 | 예시 |
|--------|--------|------|
| **come** | ~처럼 / ~같이 | È come pensavo. (내가 생각했던 대로다.) |
| **come se** | 마치 ~인 것처럼 (+ 접속법) | Parla **come se** fosse italiano. |
| **più... di quanto** | ~보다 더 | È più difficile **di quanto** pensassi. |
| **meno... di quanto** | ~보다 덜 | Costa meno **di quanto** credessi. |
| **tanto... quanto** | ~만큼 ~한 | È **tanto** bravo **quanto** suo padre. |

### 결과

| 접속사 | 한국어 | 예시 |
|--------|--------|------|
| **così... che** | 너무 ~해서 ~하다 | È **così** stanco **che** non riesce a parlare. |
| **tanto... che** | 너무 ~해서 ~하다 | Ha mangiato **tanto... che** sta male. |
| **talmente... che** | 정도가 ~해서 ~하다 | È **talmente** bravo **che** tutti lo ammirano. |

## 7. 담화 표지: 대화

담화 표지는 대화를 구조화하고, 태도를 나타내며, 대화의 흐름을 관리하는 단어와 구입니다.

### 대화 시작 및 관리

| 표지 | 기능 | 예시 |
|------|------|------|
| **Allora** | 그래서 / 자 (화제 개시) | **Allora**, come stai? |
| **Dunque** | 그래서 / 자 (보다 격식적) | **Dunque**, parliamo di lavoro. |
| **Senti / Senta** | 있잖아 / 저기요 (주의 환기) | **Senti**, ti devo dire una cosa. |
| **Guarda / Guardi** | 봐 / 보세요 (비격식 강조) | **Guarda**, non è colpa mia! |
| **Ecco** | 자 / 그러니까 | **Ecco**, il problema è questo. |
| **Niente** | 아무것도 아닌데 / 그냥 (간투사) | **Niente**, volevo solo salutarti. |
| **Diciamo** | 말하자면 | **Diciamo** che non è andata benissimo. |

### 동의와 강화

| 표지 | 기능 | 예시 |
|------|------|------|
| **Appunto** | 바로 그거야 / 정확히 | "È troppo caro." — "**Appunto!**" |
| **Esatto** | 맞아 / 정확히 | "Dovremmo partire alle sei." — "**Esatto.**" |
| **Già** | 그래 / 맞아 (동의) | "Piove sempre." — "**Già...**" |
| **Eh sì** | 아 맞아 / 그래 | "È stato un anno difficile." — "**Eh sì.**" |
| **Per forza** | 당연하지 / 물론이지 | **Per forza** che è stanco, ha lavorato 12 ore! |
| **D'accordo** | 알겠어 / 좋아 | "Ci vediamo alle tre?" — "**D'accordo.**" |

### 머뭇거림과 간투

| 표지 | 기능 | 예시 |
|------|------|------|
| **Cioè** | 그러니까 / 즉 | È... **cioè**... non so come dire. |
| **Praticamente** | 사실상 / 기본적으로 | **Praticamente**, è successo questo... |
| **In pratica** | 실제로는 / 기본적으로 | **In pratica**, non ha funzionato. |
| **Come dire** | 어떻게 말할까 / 말하자면 | È, **come dire**, un po' strano. |
| **Diciamo** | 말하자면 | **Diciamo** che poteva andare meglio. |
| **Mah** | 글쎄... / 음... | **Mah**, non saprei. |
| **Boh** | 몰라 / 글쎄 | "Viene Marco?" — "**Boh!**" |
| **Tipo** | ~같은 (구어) | Era **tipo** un'ora fa. |

### 놀라움이나 반응 표현

| 표지 | 기능 | 예시 |
|------|------|------|
| **Davvero? / Sul serio?** | 정말? / 진짜? | "Mi sposo!" — "**Davvero?!**" |
| **Ma dai!** | 설마! / 말도 안 돼! | **Ma dai**, non ci credo! |
| **Magari!** | 그러면 좋겠다! | "Vuoi un milione?" — "**Magari!**" |
| **Peccato!** | 아쉽다! / 안됐다! | "Non posso venire." — "**Peccato!**" |
| **Meno male!** | 다행이다! | "L'esame è stato cancellato." — "**Meno male!**" |
| **Che bello!** | 좋다! / 멋지다! | **Che bello**, sei tornato! |
| **Macché!** | 무슨 소리! / 천만에! | "Sei arrabbiato?" — "**Macché!**" |
| **Figurati! / Si figuri!** | 별말씀을! / 천만에요! | "Grazie!" — "**Figurati!**" |

## 8. 담화 표지: 텍스트 구성

이 표지들은 논증, 에세이, 발표, 격식 글쓰기를 구조화하는 데 도움이 됩니다.

### 순서 매기기

| 표지 | 한국어 | 용도 |
|------|--------|------|
| **Innanzitutto / Prima di tutto** | 우선/먼저 | 첫 번째 요점 |
| **In primo luogo** | 첫째로 | 격식 나열 |
| **In secondo luogo** | 둘째로 | 두 번째 요점 |
| **Poi / Dopo / Successivamente** | 그 다음에/이후에 | 이어서 |
| **Inoltre / Per di più** | 게다가/더욱이 | 정보 추가 |
| **Infine / Per finire** | 마지막으로 | 마지막 요점 |
| **In conclusione / Per concludere** | 결론적으로 | 마무리 |

### 추가와 확장

| 표지 | 한국어 | 예시 |
|------|--------|------|
| **Inoltre** | 게다가 | **Inoltre**, bisogna considerare i costi. |
| **Per di più** | 더욱이 | **Per di più**, non abbiamo tempo. |
| **Non solo... ma anche** | ~뿐만 아니라 ~도 | **Non solo** è bravo, **ma anche** simpatico. |
| **Oltre a ciò** | 그 외에도 | **Oltre a ciò**, ci sono altri problemi. |
| **A questo proposito** | 이와 관련하여 | **A questo proposito**, vorrei aggiungere... |
| **Per quanto riguarda** | ~에 관해서는 | **Per quanto riguarda** il budget... |

### 대조와 비교

| 표지 | 한국어 | 예시 |
|------|--------|------|
| **D'altra parte / D'altro canto** | 반면에/한편으로는 | **D'altra parte**, ha anche ragione. |
| **Al contrario** | 반대로 | **Al contrario**, penso che sia giusto. |
| **Invece** | 대신/반면에 | Io lavoro, lui **invece** dorme. |
| **Da un lato... dall'altro** | 한편으로는... 다른 한편으로는 | **Da un lato** è caro, **dall'altro** è di qualità. |
| **Rispetto a** | ~에 비해 | **Rispetto a** ieri, oggi fa più caldo. |
| **Mentre** | 반면에/~하는 동안 | Marco studia, **mentre** Luigi gioca. |

### 요약과 결론

| 표지 | 한국어 | 예시 |
|------|--------|------|
| **In conclusione** | 결론적으로 | **In conclusione**, il progetto è fattibile. |
| **In sintesi / In breve** | 요약하면/간단히 말해 | **In sintesi**, abbiamo tre opzioni. |
| **Insomma** | 결국/요컨대 | **Insomma**, è stata una bella esperienza. |
| **In definitiva** | 궁극적으로/결국 | **In definitiva**, ha avuto ragione lui. |
| **Per riassumere** | 요약하자면 | **Per riassumere**, i punti principali sono... |
| **Tutto sommato** | 전체적으로 보면 | **Tutto sommato**, è andata bene. |
| **Dopotutto** | 결국/어쨌든 | **Dopotutto**, siamo ancora amici. |

### 예시 들기

| 표지 | 한국어 | 예시 |
|------|--------|------|
| **Per esempio / Ad esempio** | 예를 들어 | **Per esempio**, in Italia si mangia bene. |
| **Come** | ~같은 / ~처럼 | Lingue **come** l'italiano e lo spagnolo. |
| **In particolare** | 특히 | **In particolare**, mi riferisco a... |
| **Tra cui / Tra l'altro** | ~중에 / 그중에 | Ha visitato molte città, **tra cui** Roma. |

## 9. 담화 표지: 태도 표현

### 확신과 강조

| 표지 | 한국어 | 예시 |
|------|--------|------|
| **Senza dubbio** | 의심할 여지 없이 | **Senza dubbio**, è il migliore. |
| **Di sicuro / Sicuramente** | 확실히/틀림없이 | **Di sicuro** verrà. |
| **Evidentemente** | 분명히 | **Evidentemente**, non ha capito. |
| **Chiaramente** | 명확하게 | **Chiaramente**, questo è un errore. |
| **In effetti** | 사실/실제로 | **In effetti**, hai ragione. |
| **Effettivamente** | 확실히/정말로 | **Effettivamente**, è proprio così. |

### 불확실성과 완화

| 표지 | 한국어 | 예시 |
|------|--------|------|
| **Forse** | 아마 | **Forse** ha ragione. |
| **Magari** | 혹시/어쩌면 | **Magari** domani. |
| **Probabilmente** | 아마도 | **Probabilmente** pioverà. |
| **Pare che / Sembra che** | ~인 것 같다 | **Pare che** sia partito. |
| **A quanto pare** | 보아하니 | **A quanto pare**, non viene. |
| **Per così dire** | 말하자면 | È, **per così dire**, un genio. |
| **In un certo senso** | 어떤 의미에서는 | **In un certo senso**, hai ragione. |

### 개인적 입장 표현

| 표지 | 한국어 | 예시 |
|------|--------|------|
| **Secondo me / A mio parere** | 내 생각에는 | **Secondo me**, è troppo caro. |
| **Per me** | 나로서는 | **Per me** va bene. |
| **A mio avviso** | 내 견해로는 (격식) | **A mio avviso**, è necessario. |
| **Personalmente** | 개인적으로 | **Personalmente**, preferisco il caffè. |
| **Onestamente / Francamente** | 솔직히 | **Onestamente**, non mi piace. |
| **Per la verità / A dire il vero** | 사실을 말하면 | **A dire il vero**, non lo sapevo. |

10. **chiaramente**

### 연습 2: 비교급과 최상급

빈칸을 채우세요.

1. Marco parla italiano ________ di me. (더 잘)
2. Oggi sto ________ di ieri. (더 나쁘게)
3. Lei corre ________ velocemente ________ lui. (더... ~보다)
4. Dovresti studiare di ________. (더 많이)
5. È il ristorante ________ della città. (최고의 — meglio 사용)
6. Ha risposto ________. (아주 잘 — 절대 최상급)

<details>
<summary>정답</summary>

1. **meglio**
2. **peggio**
3. **più** ... **di**
4. **più**
5. **migliore** (레스토랑에 대한 형용사, 부사 meglio가 아님)
6. **benissimo**

</details>

### 연습 3: 접속사

올바른 접속사를 채우세요.

1. Non sono venuto ________ ero malato. (~때문에)
2. ________ faccia freddo, esco lo stesso. (~에도 불구하고 — + 접속법)
3. Studio ________ tu possa capire. (~하도록 — + 접속법)
4. Vengo ________ non piova. (~하지 않는 한)
5. È ________ intelligente ________ simpatico. (~와 ~모두)
6. Ha lavorato molto, ________ è stanco. (그러므로)
7. ________ piove, prendo l'ombrello. (만약)
8. Non mangia ________ carne ________ pesce. (~도 아니고 ~도 아닌)

<details>
<summary>정답</summary>

1. **perché**
2. **Benché** / **Sebbene**
3. **affinché** / **perché**
4. **a meno che non**
5. **sia** ... **sia** / **che**
6. **quindi** / **perciò** / **dunque**
7. **Se**
8. **né** ... **né**

</details>

### 연습 4: 담화 표지

각 빈칸에 가장 적절한 담화 표지를 선택하세요.

1. ________, vorrei ringraziare tutti per essere venuti. (우선)
2. Il progetto è interessante. ________, costa troppo. (그러나)
3. ________ i costi, dobbiamo riparlarne. (~에 관해서는)
4. ________, l'esperimento è stato un successo. (결론적으로)
5. Ha visitato molte città, ________ Roma e Firenze. (~중에)
6. ________, non sapevo che fosse così complicato. (사실을 말하면)
7. "Hai vinto la lotteria!" — "________!" (설마! / 말도 안 돼!)
8. "Grazie mille per l'aiuto." — "________!" (별말씀을!)

<details>
<summary>정답</summary>

1. **Innanzitutto** / **Prima di tutto**
2. **Tuttavia** / **Però**
3. **Per quanto riguarda**
4. **In conclusione**
5. **tra cui**
6. **A dire il vero** / **Per la verità**
7. **Ma dai!**
8. **Figurati!** / **Di niente!**

</details>

### 연습 5: 부사 배치

문장 내 올바른 위치에 부사를 배치하세요.

1. Ho mangiato. (già) → ________
2. Non sono stato a Roma. (mai) → ________
3. È arrivato. (appena) → ________
4. Parla l'italiano. (bene) → ________
5. La lezione è difficile. (molto) → ________
6. Vado al cinema. (spesso) → ________

<details>
<summary>정답</summary>

1. Ho **già** mangiato.
2. Non sono **mai** stato a Roma.
3. È **appena** arrivato.
4. Parla **bene** l'italiano.
5. La lezione è **molto** difficile.
6. Vado **spesso** al cinema.

</details>

### 연습 6: 이탈리아어로 번역하기

적절한 접속사, 부사, 담화 표지를 사용하세요.

1. 비가 오지만 나간다. (양보)
2. 우선, 자기소개를 하겠습니다.
3. 그는 나보다 이탈리아어를 더 잘 말한다.
4. 내 생각에 이 레스토랑이 도시에서 최고다.
5. 시간이 없다; 그러므로 갈 수 없다.
6. 솔직히 말해서, 문제를 이해하지 못하겠다.
7. 그녀는 토요일과 일요일 모두 일한다.
8. 결론적으로, 프로젝트는 성공이었다.

<details>
<summary>정답</summary>

1. Benché / Sebbene piova, esco (lo stesso).
2. Innanzitutto / Prima di tutto, lasciatemi presentare.
3. Parla l'italiano meglio di me.
4. Secondo me, questo ristorante è il migliore della città.
5. Non ho tempo; quindi / perciò non posso venire.
6. Onestamente / Francamente, non capisco il problema.
7. Lavora sia il sabato sia / che la domenica.
8. In conclusione, il progetto è stato un successo.

</details>
