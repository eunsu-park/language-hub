## 1. Passato Prossimo: 구성

Passato prossimo(근과거)는 구어 이탈리아어에서 가장 흔한 과거 시제입니다. 과거의 **완료된** 동작을 설명합니다.

```
+----------------------------------------------------------+
|  PASSATO PROSSIMO = 조동사 + 과거 분사                     |
|                                                          |
|  조동사: AVERE 또는 ESSERE의 현재형                        |
|  + participio passato                                    |
|                                                          |
|  Ho mangiato.    (나는 먹었다.)                           |
|  Sono andato/a.  (나는 갔다.)                             |
+----------------------------------------------------------+
```

### 현재 시제의 조동사

| 인칭 | avere | essere |
|------|-------|--------|
| io | ho | sono |
| tu | hai | sei |
| lui/lei/Lei | ha | è |
| noi | abbiamo | siamo |
| voi | avete | siete |
| loro | hanno | sono |

## 2. 규칙 과거 분사

| 부정사 어미 | 과거 분사 어미 | 예시 |
|------------|--------------|------|
| **-ARE** | **-ato** | parlare → parl**ato** (말한) |
| **-ERE** | **-uto** | credere → cred**uto** (믿은) |
| **-IRE** | **-ito** | dormire → dorm**ito** (잔) |

### Avere를 사용한 예시

| 이탈리아어 | 한국어 |
|-----------|--------|
| Ho **parlato** con Maria. | Maria와 이야기했다. |
| Hai **mangiato** la pizza? | 피자 먹었어? |
| Ha **venduto** la macchina. | 그/그녀는 차를 팔았다. |
| Abbiamo **creduto** a tutto. | 우리는 모든 것을 믿었다. |
| Avete **capito** la lezione? | 수업을 이해했나요? |
| Hanno **dormito** bene. | 그들은 잘 잤다. |

## 4. 조동사 선택: Avere vs. Essere

올바른 조동사를 선택하는 것은 이탈리아어에서 매우 중요합니다.

### AVERE를 사용하는 경우:

- **타동사** (직접 목적어를 취하는 동사):
  - Ho mangiato **la pizza**. (나는 피자를 먹었다.)
  - Ha letto **il libro**. (그는 책을 읽었다.)
  - Abbiamo visto **un film**. (우리는 영화를 봤다.)

### ESSERE를 사용하는 경우:

- **이동 및 상태 변화의 자동사**
- **재귀 동사** (항상)
- **비인칭 동사** (piacere, sembrare, bastare...)

### 빠른 판단 도표

```
+----------------------------------------------------------+
|  동사가 직접 목적어를 취하는가?                              |
|    예 → AVERE 사용                                        |
|    아니오 → 아래 목록 확인:                                |
|                                                          |
|  재귀 동사인가? (mi lavo, si veste...)                    |
|    예 → ESSERE 사용                                       |
|                                                          |
|  이동/변화 동사인가?                                      |
|    예 → 보통 ESSERE                                       |
|    아니오 → 보통 AVERE                                    |
|                                                          |
|  예외: 일부 동사는 의미에 따라 둘 다 사용!                  |
|  - Ho corso per un'ora. (한 시간 동안 달렸다 — 활동)       |
|  - Sono corso a casa. (집으로 달려갔다 — 목적지)           |
+----------------------------------------------------------+
```

## 5. Essere 동사와 일치

| loro (남성 복수) | -i | Loro **sono andati**. |

| noi (남성/혼합) | -i | Noi **siamo partiti**. |

## 6. 반과거: 규칙 활용

Imperfetto(반과거)는 과거의 **진행 중**, **습관적**, 또는 **배경** 동작을 설명합니다.

### 규칙 반과거 어미

| 인칭 | -ARE (parlare) | -ERE (credere) | -IRE (dormire) |
|------|----------------|----------------|----------------|
| io | parl**avo** | cred**evo** | dorm**ivo** |
| tu | parl**avi** | cred**evi** | dorm**ivi** |
| lui/lei/Lei | parl**ava** | cred**eva** | dorm**iva** |
| noi | parl**avamo** | cred**evamo** | dorm**ivamo** |
| voi | parl**avate** | cred**evate** | dorm**ivate** |
| loro | parl**avano** | cred**evano** | dorm**ivano** |

> **핵심 특징**: imperfetto는 이탈리아어에서 가장 규칙적인 시제 중 하나입니다. 세 동사 그룹 모두 어미가 거의 동일합니다 — 특성 모음(a/e/i)만 바꾸면 됩니다.

## 7. 반과거: 불규칙 동사

imperfetto에서 진정으로 불규칙한 동사는 매우 적습니다.

### Essere (to be)

| 인칭 | 형태 |
|------|------|
| io | **ero** |
| tu | **eri** |
| lui/lei/Lei | **era** |
| noi | **eravamo** |
| voi | **eravate** |
| loro | **erano** |

### 기타 반과거 불규칙 동사

이 동사들은 축약되거나 고어적 어간을 사용하지만 규칙적인 반과거 어미를 붙입니다:

| 부정사 | 어간 | io | tu | lui/lei | noi | voi | loro |
|--------|------|-----|-----|---------|------|------|-------|
| **fare** | face- | facevo | facevi | faceva | facevamo | facevate | facevano |
| **dire** | dice- | dicevo | dicevi | diceva | dicevamo | dicevate | dicevano |
| **bere** | beve- | bevevo | bevevi | beveva | bevevamo | bevevate | bevevano |
| **tradurre** | traduce- | traducevo | traducevi | traduceva | traducevamo | traducevate | traducevano |
| **porre** | pone- | ponevo | ponevi | poneva | ponevamo | ponevate | ponevano |
| **trarre** | trae- | traevo | traevi | traeva | traevamo | traevate | traevano |

> **참고**: 이 어간들은 원래의 라틴어 형태(facere, dicere, bevere 등)에서 유래합니다. 반과거 어미(-vo, -vi, -va, -vamo, -vate, -vano)는 완전히 규칙적입니다.

## 8. 각 시제의 사용 시점

### Passato Prossimo: 완료된 동작

Passato prossimo를 사용하는 경우:

| 용법 | 예시 |
|------|------|
| **단일 완료 동작** | Ieri ho mangiato una pizza. (어제 피자를 먹었다.) |
| **시작/끝이 명확한 동작** | Ho lavorato dalle 9 alle 5. (9시부터 5시까지 일했다.) |
| **연속적인 완료 사건** | Mi sono alzato, ho fatto colazione e sono uscito. (일어나서 아침을 먹고 나갔다.) |
| **구체적인 횟수** | Ho visto quel film tre volte. (그 영화를 세 번 봤다.) |
| **상태 변화** | È diventato famoso. (그는 유명해졌다.) |

### Imperfetto: 진행 중/습관적 동작

Imperfetto를 사용하는 경우:

| 용법 | 예시 |
|------|------|
| **습관적 동작** | Da bambino giocavo sempre fuori. (어릴 때 항상 밖에서 놀았다.) |
| **지속적 상태** | Faceva caldo quel giorno. (그날은 더웠다.) |
| **신체적/감정적 묘사** | Era stanca e aveva fame. (그녀는 피곤하고 배가 고팠다.) |
| **과거의 나이** | Avevo dieci anni. (나는 열 살이었다.) |
| **시각과 날씨** | Erano le tre. Pioveva. (3시였다. 비가 오고 있었다.) |
| **동시 진행 동작** | Mentre leggevo, lui cucinava. (내가 읽는 동안 그가 요리하고 있었다.) |
| **공손한 요청 (완곡)** | Volevo chiederti una cosa. (한 가지 물어보려고 했어요.) |

### 나란히 비교

| Passato Prossimo | Imperfetto |
|------------------|------------|
| Ieri **ho mangiato** al ristorante. (어제 레스토랑에서 먹었다.) | Da bambino **mangiavo** sempre la pasta. (어릴 때 항상 파스타를 먹었다.) |
| **Sono andato** a Roma lunedì. (월요일에 로마에 갔다.) | Ogni estate **andavo** al mare. (매년 여름 바다에 갔다/가곤 했다.) |
| **Ha piovuto** per due ore. (두 시간 동안 비가 왔다.) | **Pioveva** quando sono uscito. (내가 나갔을 때 비가 오고 있었다.) |

## 10. 의미가 달라지는 동사

일부 동사는 passato prossimo를 사용하느냐 imperfetto를 사용하느냐에 따라 의미가 달라집니다.

| 동사 | Imperfetto (상태) | Passato Prossimo (사건/변화) |
|------|-------------------|-------------------------------|
| **conoscere** | Conoscevo Maria. (Maria를 **알고 있었다**.) | Ho conosciuto Maria. (Maria를 **만났다**.) |
| **sapere** | Sapevo la verità. (진실을 **알고 있었다**.) | Ho saputo la verità. (진실을 **알게 되었다**.) |
| **potere** | Potevo farlo. (**할 수 있었다** / 할 수 있는 상태였다.) | Ho potuto farlo. (그것을 해내는 데 **성공했다**.) |
| **volere** | Volevo partire. (떠나고 **싶었다**.) | Ho voluto partire. (떠나기를 **고집했다**.) |
| **dovere** | Dovevo studiare. (공부**해야 하는 상황이었다**.) | Ho dovuto studiare. (공부**해야 했다** — 그리고 했다.) |

## 11. 서술에서의 활용: 함께 사용하기

이야기에서 두 시제는 함께 작동합니다:
- **Imperfetto**는 배경을 설정합니다 (배경, 묘사, 진행 중인 동작)
- **Passato prossimo**는 줄거리를 진행시킵니다 (사건, 행동, 변화)

### 서술 예시

> **Era** una bella giornata di primavera. Il sole **splendeva** e **faceva** caldo.
> Maria **camminava** per il centro della città. **Portava** un vestito blu e **aveva** una borsa grande.
> All'improvviso **ha visto** un vecchio amico dall'altra parte della strada. **Ha attraversato** la piazza e lo **ha chiamato**.
> "Marco! Che sorpresa!" **ha detto**.
> Marco **si è girato** e **ha sorriso**. Non **si vedevano** da cinque anni.
> **Sono andati** in un bar vicino e **hanno parlato** per ore. Marco **le ha raccontato** che **viveva** a Londra e che **lavorava** come architetto.

**분석**:

| Imperfetto (배경) | Passato Prossimo (사건) |
|-------------------|------------------------|
| Era una bella giornata (아름다운 날이었다) | Ha visto un amico (친구를 보았다) |
| Il sole splendeva (태양이 빛나고 있었다) | Ha attraversato la piazza (광장을 건넜다) |
| Camminava per il centro (시내를 걷고 있었다) | Ha detto (말했다) |
| Portava un vestito blu (파란 드레스를 입고 있었다) | Si è girato (돌아보았다) |
| Viveva a Londra (런던에 살고 있었다) | Sono andati in un bar (바에 갔다) |

## 12. 연습 문제

### 연습 1: Passato Prossimo 활용하기

올바른 조동사와 일치를 사용하여 동사를 passato prossimo로 활용하세요.

1. Maria (andare) ________ a Roma.
2. Noi (mangiare) ________ una pizza.
3. I ragazzi (partire) ________ ieri sera.
4. Tu (scrivere) ________ una lettera?
5. Le ragazze (svegliarsi) ________ tardi.
6. Io (f.) (nascere) ________ nel 1990.
7. Voi (vedere) ________ quel film?
8. Lui (rimanere) ________ a casa.

<details>
<summary>정답</summary>

1. Maria **è andata** a Roma.
2. Noi **abbiamo mangiato** una pizza.
3. I ragazzi **sono partiti** ieri sera.
4. Tu **hai scritto** una lettera?
5. Le ragazze **si sono svegliate** tardi.
6. Io **sono nata** nel 1990.
7. Voi **avete visto** quel film?
8. Lui **è rimasto** a casa.

</details>

### 연습 2: Passato Prossimo인가 Imperfetto인가?

괄호 안의 동사에 알맞은 시제를 선택하세요.

1. Quando (essere) ________ bambino, (giocare) ________ sempre a calcio.
2. Ieri (piovere) ________ tutto il giorno.
3. Mentre (io/studiare) ________, il telefono (suonare) ________.
4. Ogni estate (noi/andare) ________ al mare.
5. L'anno scorso (io/visitare) ________ Firenze per la prima volta.
6. (Fare) ________ freddo e (nevicare) ________ quando (noi/arrivare) ________.
7. Da giovane mio nonno (lavorare) ________ in fabbrica.
8. Ieri sera (io/conoscere) ________ il fratello di Marco.

<details>
<summary>정답</summary>

1. **ero** ... **giocavo** (습관적 과거)
2. **ha piovuto** / **è piovuto** (완료된 사건 — 하루 종일이지만 명확한 끝이 있음)
3. **studiavo** ... **ha suonato** (진행 중인 동작이 사건에 의해 중단됨)
4. **andavamo** (습관적 과거)
5. **ho visitato** (단일 완료 사건)
6. **Faceva** ... **nevicava** ... **siamo arrivati/e** (배경 묘사 + 사건)
7. **lavorava** (습관적 과거)
8. **ho conosciuto** (만남 — 단일 사건)

</details>

### 연습 4: 이야기 완성하기

빈칸을 passato prossimo 또는 imperfetto로 채우세요.

> (Essere) ________ sabato sera. Marco (avere) ________ fame e (volere) ________ uscire a cena. (Chiamare) ________ la sua amica Lucia e le (chiedere) ________ se (volere) ________ venire con lui. Lucia (dire) ________ di sì. (Andare - loro) ________ in un ristorante nel centro. Il ristorante (essere) ________ pieno di gente e (esserci) ________ musica dal vivo. (Ordinare - loro) ________ la pasta e il pesce. La cena (essere) ________ ottima. Dopo cena (fare - loro) ________ una passeggiata e (tornare) ________ a casa a mezzanotte.

<details>
<summary>정답</summary>

**Era** sabato sera. Marco **aveva** fame e **voleva** uscire a cena. **Ha chiamato** la sua amica Lucia e le **ha chiesto** se **voleva** venire con lui. Lucia **ha detto** di sì. **Sono andati** in un ristorante nel centro. Il ristorante **era** pieno di gente e **c'era** musica dal vivo. **Hanno ordinato** la pasta e il pesce. La cena **è stata** ottima. Dopo cena **hanno fatto** una passeggiata e **sono tornati** a casa a mezzanotte.

</details>
