## 1. 이탈리아어 동사 기초

### 1.1 부정사 형태

모든 이탈리아어 동사는 **-are**, **-ere**, 또는 **-ire**로 끝나는 부정사를 갖습니다:

```
┌─────────────────────────────────────────────────────────────────┐
│   그룹 1: -ARE 동사 (~전체의 60% — 가장 큰 그룹)                   │
│   parlare (말하다), studiare (공부하다), lavorare (일하다)          │
│                                                                 │
│   그룹 2: -ERE 동사 (~25%)                                       │
│   credere (믿다), vendere (팔다), leggere (읽다)                  │
│                                                                 │
│   그룹 3a: -IRE 동사 — 규칙 (~15%)                               │
│   dormire (자다), partire (떠나다), aprire (열다)                  │
│                                                                 │
│   그룹 3b: -IRE 동사 — -ISC- 삽입형                               │
│   capire (이해하다), finire (끝내다), preferire (선호하다)          │
│                                                                 │
│   활용법: 어미(-are/-ere/-ire)를 제거하여 어간을 얻은 후,           │
│   적절한 인칭/수 어미를 붙입니다.                                   │
│                                                                 │
│   parlare → parl- + -o = parlo (나는 말한다)                      │
│   credere → cred- + -o = credo (나는 믿는다)                      │
│   dormire → dorm- + -o = dormo (나는 잔다)                        │
│   capire  → cap-  + -isc- + -o = capisco (나는 이해한다)          │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 주어 대명사

| 단수 | 복수 |
|------|------|
| io (나) | noi (우리) |
| tu (너, 비격식) | voi (너희) |
| lui (그) | loro (그들, 남) |
| lei (그녀) | loro (그들, 여) |
| Lei (당신, 격식) | Loro (여러분, 격식 — 드묾) |

> **Pro-drop**: 이탈리아어는 주어 탈락 언어입니다 — 동사 어미가 주어를 나타내므로 주어 대명사는 보통 생략됩니다. *Parlo italiano* = "나는 이탈리아어를 한다" (*io* 불필요). 강조, 대조, 명확성을 위해 대명사를 사용합니다.

## 2. -ARE 동사 활용

### 2.1 패턴: parlare (말하다)

| 주어 | 어간 + 어미 | 전체 형태 | 의미 |
|------|-----------|----------|------|
| io | parl + **-o** | parl**o** | 나는 말한다 |
| tu | parl + **-i** | parl**i** | 너는 말한다 |
| lui/lei/Lei | parl + **-a** | parl**a** | 그/그녀는 말한다 |
| noi | parl + **-iamo** | parl**iamo** | 우리는 말한다 |
| voi | parl + **-ate** | parl**ate** | 너희는 말한다 |
| loro | parl + **-ano** | parl**ano** | 그들은 말한다 |

### 2.2 예시: studiare (공부하다)

```
Studio italiano tutti i giorni.    → 매일 이탈리아어를 공부한다.
Studi all'università?              → 대학에서 공부해?
Lei studia medicina.               → 그녀는 의학을 공부한다.
Studiamo insieme.                  → 함께 공부한다.
Studiano molto per l'esame.        → 시험을 위해 많이 공부한다.
```

### 2.3 필수 -ARE 동사 10개

| 동사 | 의미 | 예시 |
|------|------|------|
| parlare | 말하다 | Parlo due lingue. (두 언어를 한다.) |
| studiare | 공부하다 | Studio l'italiano. (이탈리아어를 공부한다.) |
| lavorare | 일하다 | Lavoro in un ufficio. (사무실에서 일한다.) |
| comprare | 사다 | Compro il pane. (빵을 산다.) |
| chiamare | 부르다/전화하다 | Chiamo mia madre. (어머니께 전화한다.) |
| guardare | 보다/시청하다 | Guardo la televisione. (텔레비전을 본다.) |
| ascoltare | 듣다 | Ascolto la musica. (음악을 듣는다.) |
| camminare | 걷다 | Cammino nel parco. (공원에서 걷는다.) |
| cucinare | 요리하다 | Cucino la cena. (저녁을 요리한다.) |
| mangiare | 먹다 | Mangio la pasta. (파스타를 먹는다.) |

### 2.4 철자 참고: -CARE와 -GARE 동사

**-care**와 **-gare**로 끝나는 동사는 경음을 유지하기 위해 **i** 앞에 **h**를 추가합니다:

```
cercare (찾다):   cerco, cerchi, cerca, cerchiamo, cercate, cercano
                         ↑                ↑
                  i 앞에 h 추가!

pagare (지불하다): pago, paghi, paga, paghiamo, pagate, pagano
                        ↑              ↑
                  i 앞에 h 추가!

giocare (놀다):   gioco, giochi, gioca, giochiamo, giocate, giocano
```

## 3. -ERE 동사 활용

### 3.1 패턴: credere (믿다)

| 주어 | 어간 + 어미 | 전체 형태 | 의미 |
|------|-----------|----------|------|
| io | cred + **-o** | cred**o** | 나는 믿는다 |
| tu | cred + **-i** | cred**i** | 너는 믿는다 |
| lui/lei/Lei | cred + **-e** | cred**e** | 그/그녀는 믿는다 |
| noi | cred + **-iamo** | cred**iamo** | 우리는 믿는다 |
| voi | cred + **-ete** | cred**ete** | 너희는 믿는다 |
| loro | cred + **-ono** | cred**ono** | 그들은 믿는다 |

### 3.2 예시: scrivere (쓰다)

```
Scrivo nel mio diario.              → 일기에 쓴다.
Scrivi delle email?                 → 이메일을 써?
Lei scrive poesie.                  → 그녀는 시를 쓴다.
Scriviamo lettere.                  → 편지를 쓴다.
Scrivono messaggi.                  → 메시지를 쓴다.
```

### 3.3 필수 -ERE 동사 10개

| 동사 | 의미 | 예시 |
|------|------|------|
| credere | 믿다 | Credo di sì. (그렇다고 생각한다.) |
| vendere | 팔다 | Vende macchine. (차를 판다.) |
| leggere | 읽다 | Leggo un libro. (책을 읽는다.) |
| scrivere | 쓰다 | Scrivo una lettera. (편지를 쓴다.) |
| prendere | 가져가다/마시다 | Prendo un caffè. (커피를 마신다.) |
| mettere | 놓다 | Metto il libro sul tavolo. (책을 테이블에 놓는다.) |
| chiudere | 닫다 | Chiudo la porta. (문을 닫는다.) |
| rispondere | 대답하다 | Rispondo al telefono. (전화를 받는다.) |
| correre | 달리다 | Corro la mattina. (아침에 달린다.) |
| vivere | 살다 | Vivo a Milano. (밀라노에 산다.) |

> **참고**: 많은 흔한 -ERE 동사가 불규칙입니다. 일부는 어미보다 어간에 강세가 있습니다: *lèggere*, *scrìvere*, *prèndere* (어간 첫 음절에 강세). 이것은 활용 패턴에는 영향을 주지 않습니다.

## 5. -ISC- 삽입 -IRE 동사

### 5.1 패턴: capire (이해하다)

큰 -IRE 동사 그룹이 **noi**와 **voi**를 제외한 모든 형태에서 어간과 어미 사이에 **-isc-**를 삽입합니다.

| 주어 | 형태 | 의미 |
|------|------|------|
| io | cap**isco** | 나는 이해한다 |
| tu | cap**isci** | 너는 이해한다 |
| lui/lei/Lei | cap**isce** | 그/그녀는 이해한다 |
| noi | cap**iamo** | 우리는 이해한다 (-isc- 없음!) |
| voi | cap**ite** | 너희는 이해한다 (-isc- 없음!) |
| loro | cap**iscono** | 그들은 이해한다 |

> **-ISC- "부츠" 패턴**: -isc-는 io, tu, lui/lei, loro에 나타나며 — 활용표에서 부츠 모양을 형성합니다. Noi와 voi는 규칙적입니다.

### 5.2 흔한 -ISC- 동사 (반드시 암기!)

| 동사 | 의미 | 1인칭 형태 |
|------|------|-----------|
| capire | 이해하다 | capisco |
| finire | 끝내다 | finisco |
| preferire | 선호하다 | preferisco |
| pulire | 청소하다 | pulisco |
| costruire | 건설하다 | costruisco |
| spedire | 보내다/발송하다 | spedisco |
| gestire | 관리하다 | gestisco |
| colpire | 치다/때리다 | colpisco |
| unire | 통합하다 | unisco |
| proibire | 금지하다 | proibisco |
| guarire | 치유하다 | guarisco |
| suggerire | 제안하다 | suggerisco |
| restituire | 돌려주다 | restituisco |
| contribuire | 기여하다 | contribuisco |

> **확실한 규칙 없음**: -IRE 동사가 -isc-를 취하는지 아닌지 결정하는 철자 규칙이 없습니다. 어떤 동사가 이 그룹에 속하는지 외워야 합니다. 확실하지 않으면 사전에 표시되어 있습니다.

## 6. 흔한 불규칙 동사: 조동사

### 6.1 POTERE (~할 수 있다)

| 주어 | 형태 | 예시 |
|------|------|------|
| io | **posso** | Posso aiutarti. (도와줄 수 있어.) |
| tu | **puoi** | Puoi venire? (올 수 있어?) |
| lui/lei | **può** | Non può dormire. (잠을 못 자.) |
| noi | **possiamo** | Possiamo uscire. (나갈 수 있어.) |
| voi | **potete** | Potete entrare. (들어올 수 있어.) |
| loro | **possono** | Non possono venire. (올 수 없어.) |

### 6.2 VOLERE (원하다)

| 주어 | 형태 | 예시 |
|------|------|------|
| io | **voglio** | Voglio un caffè. (커피 원해.) |
| tu | **vuoi** | Cosa vuoi fare? (뭐 하고 싶어?) |
| lui/lei | **vuole** | Vuole partire. (떠나고 싶어한다.) |
| noi | **vogliamo** | Vogliamo mangiare. (먹고 싶다.) |
| voi | **volete** | Volete bere qualcosa? (뭔가 마실래?) |
| loro | **vogliono** | Vogliono andare. (가고 싶어한다.) |

### 6.3 DOVERE (~해야 한다)

| 주어 | 형태 | 예시 |
|------|------|------|
| io | **devo** | Devo studiare. (공부해야 한다.) |
| tu | **devi** | Devi mangiare. (먹어야 해.) |
| lui/lei | **deve** | Deve andare. (가야 한다.) |
| noi | **dobbiamo** | Dobbiamo partire. (떠나야 한다.) |
| voi | **dovete** | Dovete pagare. (지불해야 한다.) |
| loro | **devono** | Devono lavorare. (일해야 한다.) |

### 6.4 SAPERE (알다 [사실/방법])

| 주어 | 형태 | 예시 |
|------|------|------|
| io | **so** | Non so nuotare. (수영할 줄 모른다.) |
| tu | **sai** | Sai dov'è? (어디인지 알아?) |
| lui/lei | **sa** | Sa parlare italiano. (이탈리아어를 할 줄 안다.) |
| noi | **sappiamo** | Sappiamo la verità. (진실을 안다.) |
| voi | **sapete** | Sapete cucinare? (요리할 줄 알아?) |
| loro | **sanno** | Non sanno niente. (아무것도 모른다.) |

### 6.5 SAPERE vs CONOSCERE

```
SAPERE = 사실, 정보, 방법을 알다
  So che Roma è in Italia. (로마가 이탈리아에 있다는 걸 안다.)
  Sai nuotare? (수영할 줄 알아? / 수영할 수 있어?)
  Non so la risposta. (답을 모른다.)

CONOSCERE = 사람, 장소, 사물을 알다/친숙하다
  Conosco Maria. (Maria를 안다.)
  Conosci Roma? (로마를 알아? / 로마에 가본 적 있어?)
  Non conosco questo ristorante. (이 레스토랑을 모른다.)

※ CONOSCERE 활용 (규칙 -ERE):
  conosco, conosci, conosce, conosciamo, conoscete, conoscono
```

## 7. 기본 문장 구성

### 7.1 어순: 주어-동사-목적어 (SVO)

```
┌─────────────────────────────────────────────────────────────────┐
│   기본 패턴: 주어 + 동사 + 목적어/보어                              │
│                                                                 │
│   (Io) studio italiano.        → 이탈리아어를 공부한다.             │
│   Maria mangia una mela.       → Maria는 사과를 먹는다.           │
│   (Noi) viviamo a Roma.        → 로마에 산다.                     │
│                                                                 │
│   ※ 주어 대명사는 보통 생략 (pro-drop):                            │
│   Parlo italiano. = 이탈리아어를 한다. (io 암시)                    │
│   Studia medicina. = 의학을 공부한다.                              │
│                                                                 │
│   ※ 이탈리아어 어순은 강조를 위해 유연합니다:                        │
│   Leggo un libro. = Un libro leggo. ("책을"에 강조)               │
│   L'italiano, lo studio. (이탈리아어, 그걸 공부한다 — 주제화)       │
│   둘 다 문법적으로 올바릅니다.                                      │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 세부사항 추가

```
시간 표현:
  Studio italiano la mattina.          → 아침에 이탈리아어를 공부한다.
  Lavoro tutti i giorni.               → 매일 일한다.
  Mangio sempre alle due.              → 항상 2시에 먹는다.

장소:
  Studio in biblioteca.                → 도서관에서 공부한다.
  Lavoro in un ufficio.                → 사무실에서 일한다.

방법:
  Parla italiano molto bene.           → 이탈리아어를 아주 잘한다.
  Camminiamo lentamente.               → 천천히 걷는다.

빈도:
  sempre (항상), spesso (자주), a volte / qualche volta (때때로),
  raramente (드물게), quasi mai (거의 안), mai (결코)
```

## 8. 의문문 만들기

### 8.1 의문사

| 이탈리아어 | 한국어 | 예시 |
|-----------|--------|------|
| Che? / Che cosa? / Cosa? | 무엇? | Cosa studi? (뭘 공부해?) |
| Chi? | 누구? | Chi parla? (누가 말해?) |
| Quando? | 언제? | Quando mangi? (언제 먹어?) |
| Dove? | 어디? | Dove abiti? (어디 살아?) |
| Perché? | 왜? / 때문에 | Perché studi? (왜 공부해?) |
| Come? | 어떻게? | Come stai? (어떻게 지내?) |
| Quanto/a? | 얼마? | Quanto costa? (얼마예요?) |
| Quanti/e? | 몇? | Quanti anni hai? (몇 살이야?) |
| Quale? / Quali? | 어떤? | Quale preferisci? (어떤 걸 선호해?) |

> **Perché**는 "왜?" (질문)와 "때문에" (대답) 두 가지를 모두 의미합니다. 문맥으로 구별됩니다.

### 8.2 예/아니오 질문을 만드는 세 가지 방법

```
방법 1: 억양만으로 (끝에서 음높이 올리기 — 가장 흔함!)
  Parli italiano?           → 이탈리아어 해?
  (평서문과 같은 어순, 올리는 억양)

방법 2: 주어와 동사 도치 (격식/문어)
  Parla Lei italiano?       → 이탈리아어를 하십니까? (격식)
  Studia Maria medicina?    → Maria는 의학을 공부합니까?

방법 3: 부가 의문문 추가
  Parli italiano, no?       → 이탈리아어 하지, 그렇지?
  Parli italiano, vero?     → 이탈리아어 하지, 맞지?
```

> **참고**: 스페인어와 달리, 이탈리아어는 거꾸로 된 부호를 사용하지 않습니다. 의문문은 마지막의 `?`만 사용합니다.

## 9. 부정문

### 9.1 기본 부정: 동사 앞에 *non* 배치

```
Parlo italiano.        → 이탈리아어를 한다.
Non parlo italiano.    → 이탈리아어를 못 한다.

Ha tempo.              → 시간이 있다.
Non ha tempo.          → 시간이 없다.

È il mio libro.       → 내 책이다.
Non è il mio libro.   → 내 책이 아니다.
```

### 9.2 부정어

| 부정어 | 긍정 반대어 | 의미 |
|--------|-----------|------|
| non | sì | ~아니다 |
| niente / nulla | qualcosa | 아무것도 / 뭔가 |
| nessuno | qualcuno | 아무도 / 누군가 |
| mai | sempre | 결코 / 항상 |
| nessuno/a | qualche / alcuni | 어떤 ~도 아닌 / 어떤 |
| neanche / nemmeno / neppure | anche | ~도 아니다 / ~도 |
| né...né | o...o | ~도 ~도 아닌 / ~이거나 |

### 9.3 이중 부정 (이탈리아어에서는 올바름!)

```
┌─────────────────────────────────────────────────────────────────┐
│   스페인어처럼, 부정어가 동사 뒤에 올 때                             │
│   이중 부정이 필수입니다.                                          │
│                                                                 │
│   Non ho niente.         → 아무것도 없다. (직역: 아무것도 안 가졌다)│
│   Non vedo nessuno.      → 아무도 안 보인다.                      │
│   Non mangio mai carne.  → 결코 고기를 먹지 않는다.                │
│   Non voglio né caffè né tè. → 커피도 차도 원하지 않는다.         │
│                                                                 │
│   대안 (부정어가 동사 앞 — "non" 없이):                            │
│   Mai mangio carne.      → 결코 고기를 먹지 않는다.                │
│   Nessuno parla.         → 아무도 말하지 않는다.                   │
│   Niente funziona.       → 아무것도 작동하지 않는다.               │
│                                                                 │
│   두 패턴 모두 올바름:                                             │
│   Non viene mai. = Mai viene. (그/그녀는 결코 오지 않는다.)        │
│                                                                 │
│   Neanche / Nemmeno / Neppure (~도 아닌 / ~조차 아닌):           │
│   Non parlo neanche inglese. (영어조차 못 한다.)                  │
│   Nemmeno io! (나도 아니야! / 나도 마찬가지!)                      │
└─────────────────────────────────────────────────────────────────┘
```

## 10. 현재 시제의 용법

이탈리아어 현재 시제는 여러 영어 시제를 포함합니다:

```
1. 습관적 행동:
   Studio italiano tutti i giorni.
   → 매일 이탈리아어를 공부한다.

2. 현재 상태:
   Vivo a Milano.
   → 밀라노에 산다.

3. 일반적 진리:
   La Terra gira intorno al Sole.
   → 지구는 태양 주위를 돈다.

4. 가까운 미래 (시간 표현과 함께):
   Domani parto per Roma.
   → 내일 로마로 떠난다. (= 떠날 것이다)

5. 진행 중인 행동 (비격식, 진행형 대신):
   Che leggi? (= Che stai leggendo?)
   → 뭐 읽고 있어?

6. 역사적 현재 (생생한 서술):
   Nel 1492, Colombo scopre l'America.
   → 1492년, 콜럼버스가 아메리카를 발견한다.

7. 지시와 안내:
   Vai dritto e poi giri a destra.
   → 직진해서 오른쪽으로 돌아.
```

## 11. 요약: 모든 그룹 비교

```
┌────────┬──────────┬──────────┬──────────┬──────────┐
│ 인칭   │ -ARE     │ -ERE     │ -IRE     │ -IRE     │
│        │ (parlare)│ (credere)│ (dormire)│ (capire) │
├────────┼──────────┼──────────┼──────────┼──────────┤
│ io     │ parl-o   │ cred-o   │ dorm-o   │ cap-isco │
│ tu     │ parl-i   │ cred-i   │ dorm-i   │ cap-isci │
│ lui    │ parl-a   │ cred-e   │ dorm-e   │ cap-isce │
│ noi    │ parl-iamo│ cred-iamo│ dorm-iamo│ cap-iamo │
│ voi    │ parl-ate │ cred-ete │ dorm-ite │ cap-ite  │
│ loro   │ parl-ano │ cred-ono │ dorm-ono │ cap-iscono│
├────────┴──────────┴──────────┴──────────┴──────────┤
│ 패턴:                                               │
│ • "io"는 항상 -o로 끝남 (또는 -isco)                  │
│ • "noi"는 항상 -iamo로 끝남                          │
│ • -ARE: a 주제 (a, i, a, iamo, ate, ano)            │
│ • -ERE: e 주제 (o, i, e, iamo, ete, ono)            │
│ • -IRE: -ERE와 동일하나 voi에서 다름 (-ite vs -ete)   │
│ • -ISC-: io, tu, lui, loro에서만 -isc- 추가           │
└─────────────────────────────────────────────────────┘
```
