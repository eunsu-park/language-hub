## 1. 표준 이탈리아어와 토스카나어 기반

```
+----------------------------------------------------------+
|        표준 이탈리아어 — L'ITALIANO STANDARD               |
+----------------------------------------------------------+
|                                                          |
|  기반: 14세기 문학적 토스카나어 (fiorentino)               |
|  주요 작가: Dante, Petrarca, Boccaccio (le tre corone)   |
|                                                          |
|  체계화:                                                  |
|    Pietro Bembo — "Prose della volgar lingua" (1525)     |
|    Accademia della Crusca — 1583년 설립                   |
|    (최초의 이탈리아어 사전, "Vocabolario" — 1612)          |
|                                                          |
|  통일 당시 (1861):                                       |
|    이탈리아인 중 약 2.5%만 표준 이탈리아어를 구사           |
|    나머지는 지역 방언을 모어로 사용                         |
|                                                          |
|  현재:                                                   |
|    약 95%의 이탈리아인이 표준 이탈리아어를 구사             |
|    약 50%가 여전히 방언을 정기적으로 사용 (ISTAT 조사)      |
+----------------------------------------------------------+
```

### 왜 토스카나어인가?

토스카나 문학 전통의 명성, 특히 다음 작가들의 작품 때문이다:

| 작가 | 주요 작품 | 공헌 |
|--------|-----------|-------------|
| **Dante Alighieri** (1265-1321) | *La Divina Commedia* | 라틴어보다 속어(토스카나어)의 문학적 힘을 입증 |
| **Francesco Petrarca** (1304-1374) | *Il Canzoniere* | 토스카나어로 서정시 전통을 정제 |
| **Giovanni Boccaccio** (1313-1375) | *Il Decamerone* | 토스카나어 산문을 문학적 표준으로 확립 |

> **핵심 포인트**: 표준 이탈리아어는 단순히 현대 피렌체어가 아니다. 14세기 피렌체어에 기반한 문학적 표준으로, 이후 다른 지역, 특히 로마와 밀라노의 기여로 수정되고 풍부해졌다.

```
+----------------------------------------------------------+
```

### 등어선: 구분하는 선들

| 등어선 | 위치 | 구분 |
|----------|----------|-----------|
| **La Spezia-Rimini 선** | La Spezia (리구리아)에서 Rimini (로마냐)까지 | 북부 방언과 중부 방언을 구분 |
| **Roma-Ancona 선** | Roma (라치오)에서 Ancona (마르케)까지 | 중부 방언과 남부 방언을 구분 |

이 선들을 기준으로 달라지는 특징들:

| 특징 | 북부 | 중부 | 남부 |
|---------|-------|--------|-------|
| 모음 사이 폐쇄음의 유성화 | 흔함 (*amiga* ← *amica*) | 드묾 | 드묾 |
| 이중 자음 | 자주 단순화됨 | 보존됨 | 강화됨 |
| 어말 모음 | 자주 탈락 | 보존됨 | 때때로 변형됨 |
| 모음 체계 | 최대 15개 모음 | 7개 모음 (표준과 동일) | 5개 모음 |

## 3. 북부 방언 (Dialetti Settentrionali)

### 3.1 Piemontese

피에몬테 지방에서 사용되며 토리노(Torino)를 중심으로 한다.

| 특징 | 예시 | 표준 이탈리아어 |
|---------|---------|-----------------|
| 어말 모음 탈락 | *can* | cane (개) |
| 비모음 (프랑스어 영향) | *pan* [pã] | pane (빵) |
| *u*가 [y]로 발음 (프랑스어 *u*처럼) | *lun-a* [ˈlyːna] | luna (달) |
| 주어 대명사 접어 | *Mi i son* | Io sono (나는 ~이다) |

**피에몬테어 예문:**
- *A l'é na bela giornà.* = E una bella giornata. (아름다운 날이다.)

### 3.2 Lombardo

롬바르디아 지방에서 사용되며 밀라노(Milano)를 중심으로 한다.

| 특징 | 예시 | 표준 이탈리아어 |
|---------|---------|-----------------|
| 모음 비음화 | *pà* [pã] | pane (빵) |
| *u* → [y] | *lüna* | luna |
| 어말 모음 탈락 | *lavorà* | lavorare (일하다) |
| *minga*를 사용한 부정 | *El vegn minga.* | Non viene. (그는 오지 않는다.) |

**밀라노어 예문:**
- *Sem andaa a cà.* = Siamo andati a casa. (우리는 집에 갔다.)

### 3.3 Veneto

베네토 지방에서 사용되며 베네치아(Venezia)를 중심으로 한다.

| 특징 | 예시 | 표준 이탈리아어 |
|---------|---------|-----------------|
| 치간음 [θ] 또는 [ð] | *sì* [θi] | sì |
| 이중 자음 소실 | *belo* | bello (아름다운) |
| *-tion* → *-sion* | *nasion* | nazione (국가) |
| 의문 접사 *-lo* | *Vien-lo?* | Viene? (그가 오나요?) |

**베네치아어 예문:**
- *No go mai visto niente.* = Non ho mai visto niente. (나는 아무것도 본 적이 없다.)

### 3.4 Emiliano-Romagnolo

에밀리아로마냐 지방에서 사용된다.

| 특징 | 예시 | 표준 이탈리아어 |
|---------|---------|-----------------|
| 광범위한 모음 축소 | *ragàs* | ragazzi (소년들) |
| 강한 비모음 | *pan* [pã] | pane |
| 어말 모음 탈락 | *tèmp* | tempo (시간) |

## 4. 중부 방언 (Dialetti Centrali)

### 4.1 Toscano (토스카나어)

표준 이탈리아어에 가장 가까운 방언이지만 독특한 특징이 있다:

| 특징 | 설명 | 예시 |
|---------|------------|---------|
| **Gorgia toscana** | 모음 사이 무성 폐쇄음의 기음화: /k/ → [h], /t/ → [θ], /p/ → [ɸ] | *la casa* → [la ˈhaːsa] |
| **마찰음화** | /k/ → [h] "la coca-cola" → [la ˌhoha ˈhɔːla] | 일상 대화에서 흔함 |
| **음운론적 이중 자음화** | 특정 단어 뒤에서 초성 자음 강화 | *a casa* → [akˈkaːsa] |

**Gorgia toscana 예시:**

| 표준 발음 | 토스카나 발음 | 단어 |
|-----------------------|---------------------|------|
| [la ˈkaːsa] | [la ˈhaːsa] | la casa (집) |
| [la ˈtorta] | [la ˈθorta] | la torta (케이크) |
| [il ˈpane] | [il ˈɸane] | il pane (빵) |

> **참고**: Gorgia toscana는 토스카나 사투리의 가장 인식하기 쉬운 특징이다. 피렌체 사람들은 종종 "C를 마신다" (*bere le C*)고 농담한다.

### 4.2 Romano (로마어)

로마의 방언으로, 현대 표준 이탈리아어에 큰 영향을 미쳤다 (특히 비격식체에서):

| 특징 | 예시 | 표준 이탈리아어 |
|---------|---------|-----------------|
| 유음화: 자음 앞 *l* → *r* | *er sole* | il sole (태양) |
| 강세 모음 뒤 이중 자음화 | *annamo* | andiamo (가자) |
| *-nd-* → *-nn-* | *quanno* | quando (~할 때) |
| *noi/voi* → *noantri/voantri* | *noantri annamo* | noi andiamo (우리는 간다) |
| 이중모음 단순화 | *bono* | buono (좋은) |

**흔한 로마 방언 표현:**

| 로마어 | 표준 이탈리아어 | 한국어 |
|-------|-----------------|---------|
| *Daje!* | Dai! / Forza! | 어서! / 가자! |
| *Aò!* | Ehi! | 야! |
| *'Sto / 'Sta* | Questo / Questa | 이 |
| *Che te frega?* | Che te ne importa? | 네가 무슨 상관이야? |
| *Mortacci tua!* | (강한 욕설) | (강한 로마 감탄사) |

## 5. 남부 방언 (Dialetti Meridionali)

### 5.1 Napoletano (나폴리어)

가장 널리 사용되고 문화적으로 풍부한 방언 중 하나로, 강력한 문학 및 음악 전통을 가지고 있다.

| 특징 | 예시 | 표준 이탈리아어 |
|---------|---------|-----------------|
| 비강세 어말 모음의 슈와 [ə] | *bell* [bɛllə] | bello (아름다운) |
| 어두 모음 변화: *u-* → *o-* | *overò* | vero (진짜의) |
| *mb-* → *mm-* | *'a mamma* | la mamma (엄마) |
| 정관사: *'o, 'a, 'e* | *'o guaglione* | il ragazzo (소년) |
| *tenere*를 사용한 과거 시제 | *Tengo fatto* | Ho fatto (나는 했다) |

**나폴리어 예문:**

| 나폴리어 | 표준 이탈리아어 | 한국어 |
|-----------|-----------------|---------|
| *Comm stai?* | Come stai? | 어떻게 지내? |
| *Nun saccio niente.* | Non so niente. | 아무것도 모르겠어. |
| *Uè, guagliò!* | Ehi, ragazzo! | 야, 꼬마야! |

### 5.2 Siciliano (시칠리아어)

유네스코에 의해 독립된 언어로 인정받았다. Scuola Siciliana (13세기)부터 이어지는 풍부한 문학 전통이 있다.

| 특징 | 예시 | 표준 이탈리아어 |
|---------|---------|-----------------|
| 5모음 체계 (/e/ vs /ɛ/, /o/ vs /ɔ/ 구분 없음) | *nivi* | neve (눈) |
| *-nd-* → *-nn-* | *quannu* | quando (~할 때) |
| *-mb-* → *-mm-* | *amma* | gamba (다리) |
| *ll* → *dd* (권설음) | *beddu* | bello (아름다운) |
| 정관사: *u, a, i* | *u picciriddu* | il bambino (아이) |
| 동사 어미 차이 | *parrari* | parlare (말하다) |

**시칠리아어 예문:**

| 시칠리아어 | 표준 이탈리아어 | 한국어 |
|----------|-----------------|---------|
| *Comu stai?* | Come stai? | 어떻게 지내? |
| *Nun sacciu nenti.* | Non so niente. | 아무것도 모르겠어. |
| *Chi bedda!* | Che bella! | 정말 아름다워! |

### 5.3 기타 남부 방언

| 방언 | 지역 | 주목할 특징 |
|---------|--------|----------------|
| **Abruzzese** | Abruzzo | 대부분의 비강세 모음이 슈와로 대체 |
| **Pugliese** | Puglia | 강한 모음 축소, 독특한 억양 |
| **Calabrese** | Calabria | 남부와 시칠리아 특징의 혼합 |
| **Lucano** | Basilicata | 고어 어휘, 나폴리어와 칼라브리아어 사이의 과도기적 특징 |
| **Salentino** | 남부 Puglia | 풀리아어보다 시칠리아어에 더 가까움 |

## 6. 도서 방언 (Dialetti Insulari)

### 6.1 Sardo (사르데냐어)

사르데냐어는 가장 보수적인 로망스어로, 많은 라틴어 특징을 보존하고 있다:

| 특징 | 사르데냐어 | 라틴어 | 이탈리아어 |
|---------|-----------|-------|---------|
| *e/i* 앞 라틴어 *k* 보존 | *kentu* | centum | cento (백) |
| 라틴어 *-ll-* 유지 | *stella* | stella | stella (별) |
| *ipse*에서 유래한 정관사 | *su/sa* | ipse/ipsa | il/la |
| 라틴어 *-t* 보존 | *amat* | amat | ama (그/그녀가 사랑한다) |

**사르데냐어 변종:**
- **Logudorese** (중북부) -- 가장 보수적
- **Campidanese** (남부) -- 이탈리아어의 영향을 더 많이 받음
- **Gallurese**와 **Sassarese** (북부) -- 코르시카어에 더 가까움

> **중요**: 사르데냐어는 이탈리아 법률 (법률 482/1999)에 의해 소수 언어 (*lingua minoritaria*)로 공식 인정되었다.

## 7. 지역별 발음 차이

### 7.1 주요 음성 현상

| 현상 | 설명 | 지역 |
|-----------|-------------|-------|
| **Gorgia toscana** | /k, t, p/ → [h, θ, ɸ] (모음 사이) | 토스카나 |
| **Raddoppiamento fonosintattico** | 특정 단어 뒤에서 초성 자음 이중화 | 중부/남부 이탈리아 |
| **연음화** | 모음 사이 자음 약화 | 북부 이탈리아 |
| **슈와 삽입** | 비강세 모음 → [ə] | 남부 이탈리아 |
| **s 유성화** | /s/ → [z] (유성 자음 앞) | 북부 이탈리아 |
| **파찰음 변이** | "zero" 같은 단어에서 [ts] vs [dz] | 지역에 따라 다름 |

### 7.2 Raddoppiamento Fonosintattico (RF)

특정 유발 단어 뒤에서 다음 단어의 초성 자음이 이중화되는 현상:

| 유발어 | 예시 | 발음 |
|---------|---------|--------------|
| *a* (전치사) | *a casa* | [akˈkaːsa] |
| *e* (접속사) | *e bene* | [ebˈbɛːne] |
| *come* | *come mai* | [komemˈmai] |
| *qualche* | *qualche cosa* | [kwalkekˈkɔːsa] |
| *città* (끝강세 단어) | *città bella* | [tʃittabˈbɛlla] |

> **지역별 참고**: RF는 중부/남부 이탈리아어 발화에서는 자연스럽지만 북부 이탈리아어 발음에서는 나타나지 않는다. 이것은 북부 억양과 남부 억양을 구분하는 가장 분명한 표지 중 하나이다.

### 7.3 표준 이탈리아어의 지역별 발음

## 10. 방언과 표준어 사이의 코드 전환

### 10.1 이탈리아인들이 전환하는 경우

현대 이탈리아인들은 표준 이탈리아어와 지역 방언 사이를 빈번하게 전환한다:

| 상황 | 전형적 선택 |
|---------|---------------|
| 업무, 공식 회의 | 표준 이탈리아어 |
| 가족 모임 | 방언 또는 혼합 |
| 친구들과 (비격식) | 지역 이탈리아어 또는 방언 |
| 소셜 미디어 | 혼합 (표준 + 방언 표현) |
| 강한 감정 표현 | 방언 (욕설, 감탄사) |
| 지역 음식, 전통 논의 | 방언 용어 선호 |

### 10.2 지역 이탈리아어 (Italiano Regionale)

순수 방언과 표준 이탈리아어 사이에는 **italiano regionale**라 불리는 중간층이 존재한다 -- 지역적 발음, 어휘, 구문을 가진 표준 이탈리아어:

| 지역 | 지역 이탈리아어 특징 | 표준 |
|--------|------------------------|----------|
| 밀라노 | *Piuttosto che* ("oppure"의 의미로) | *Piuttosto che* (~보다 차라리) |
| 로마 | *Er* ← *il*, 격려의 의미로 *daje* | 표준 관사, *dai* |
| 나폴리 | *Tengo* ← *ho* (나는 ~을 가지고 있다) | *Ho* |
| 시칠리아 | *Minchia!* 일반적 감탄사로 | (동등한 표현 없음) |
| 사르데냐 | *Eja* ← *sì* | *Sì* |

### 10.3 세대별 차이

| 세대 | 방언 사용 |
|-----------|------------|
| 1940년 이전 출생 | 방언이 모어; 학교에서 이탈리아어 학습 |
| 1940-1970 출생 | 이중 언어; 집에서 방언, 공공장소에서 이탈리아어 |
| 1970-1990 출생 | 이탈리아어 우세; 방언에 대한 수동적 지식 |
| 1990년 이후 출생 | 이탈리아어 모어; 방언은 고정 표현에 한정 |

3. **(b) 지역 이탈리아어** -- 반격식, 지역 맥락
4. **(b) 지역 이탈리아어** 또는 방언 표현이 섞인 **(a) 표준어**
5. **(c) 방언** -- 강한 감정은 종종 방언 사용을 유발하며, 특히 감탄사에서 그러하다
