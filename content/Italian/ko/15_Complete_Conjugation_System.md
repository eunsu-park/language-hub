## 1. 이탈리아어 동사 체계 개요

> **이탈리아어 동사 분류**
>
> **어미별**:
> - -ARE (가장 흔함: 동사의 ~60%)
> - -ERE (두 번째로 큰 그룹)
> - -IRE (두 하위 그룹: 규칙 + -isc-)
>
> **규칙성별**:
> - 규칙
> - 불규칙 (고유 어간/패턴)
> - -ISC- 동사 (capire, finire, preferire)
> - 재귀 동사 (-si 어미: lavarsi)
>
> **법별**:
> - Indicativo (사실, 확실성)
> - Congiuntivo (주관성, 의심)
> - Condizionale (가정, 공손)
> - Imperativo (명령)
>
> **비정형**:
> - Infinito (parlare)
> - Gerundio (parlando)
> - Participio (parlato, parlante)

### 직설법 시제 한눈에 보기

| 시제 | 이탈리아어 이름 | 용법 |
|------|---------------|------|
| 현재 | Presente | 현재 동작, 습관, 일반적 진실 |
| 반과거 | Imperfetto | 과거 습관, 묘사, 진행 중인 상태 |
| 근과거 | Passato prossimo | 완료된 과거 동작 (구어) |
| 원과거 | Passato remoto | 완료된 과거 동작 (문학/역사) |
| 대과거 | Trapassato prossimo | 다른 과거 동작 이전의 동작 |
| 단순 미래 | Futuro semplice | 미래 동작, 추측 |
| 전미래 | Futuro anteriore | 완료된 미래, 과거 추측 |

## 5. Passato Remoto: 규칙 활용

Passato remoto(원과거)는 **먼 과거의 완료된 동작**에 사용되며, 특히 문학 작품과 남부 이탈리아 구어에서 사용됩니다. 북부 이탈리아와 일상 대화에서는 passato prossimo가 선호됩니다.

### 규칙 어미

| 인칭 | -ARE (parlare) | -ERE (credere) | -IRE (dormire) |
|------|----------------|----------------|----------------|
| io | parl**ai** | cred**ei** (cred**etti**) | dorm**ii** |
| tu | parl**asti** | cred**esti** | dorm**isti** |
| lui/lei | parl**ò** | cred**é** (cred**ette**) | dorm**ì** |
| noi | parl**ammo** | cred**emmo** | dorm**immo** |
| voi | parl**aste** | cred**este** | dorm**iste** |
| loro | parl**arono** | cred**erono** (cred**ettero**) | dorm**irono** |

> **참고**: -ERE 동사는 io, lui/lei, loro에서 두 가지 가능한 어미 세트가 있습니다. 둘 다 올바르지만, -etti/-ette/-ettero 형태가 규칙 -ERE 동사에서 더 흔합니다.

## 7. Gerundio와 진행형 (Stare + Gerundio)

### Gerundio 구성

| 부정사 어미 | Gerundio 어미 | 예시 |
|------------|-------------|------|
| **-ARE** | **-ando** | parlare → parl**ando** |
| **-ERE** | **-endo** | scrivere → scriv**endo** |
| **-IRE** | **-endo** | dormire → dorm**endo** |

### 불규칙 Gerundio

| 부정사 | Gerundio |
|--------|---------|
| fare | **facendo** |
| dire | **dicendo** |
| bere | **bevendo** |
| tradurre | **traducendo** |
| porre | **ponendo** |
| trarre | **traendo** |

### Stare + Gerundio (진행형)

이탈리아어는 진행 중인 동작에 **stare + gerundio**를 사용합니다 (영어의 "be + -ing"처럼).

| 시제 | 구성 | 예시 | 한국어 |
|------|------|------|--------|
| **현재 진행형** | sto + gerundio | Sto **parlando**. | 말하고 있다. |
| **과거 진행형** | stavo + gerundio | Stavo **dormendo**. | 자고 있었다. |
| **미래 진행형** | starò + gerundio | Starò **lavorando**. | 일하고 있을 것이다. |

### 예시

| 이탈리아어 | 한국어 |
|-----------|--------|
| Cosa **stai facendo**? | 뭐 하고 있어? |
| **Sto leggendo** un libro. | 책을 읽고 있다. |
| **Stava piovendo** quando sono uscito. | 내가 나갔을 때 비가 오고 있었다. |
| I bambini **stanno giocando** in giardino. | 아이들이 정원에서 놀고 있다. |
| Non mi disturbare, **sto studiando**! | 방해하지 마, 공부 중이야! |

### 부사절로서의 Gerundio

Gerundio는 영어의 "while/by/when + -ing"처럼 기능할 수도 있습니다:

| 이탈리아어 | 한국어 |
|-----------|--------|
| **Camminando** per strada, ho incontrato Marco. | 길을 걷다가 Marco를 만났다. |
| **Studiando** molto, ha superato l'esame. | 많이 공부해서 시험에 합격했다. |
| **Non avendo** soldi, non posso comprarlo. | 돈이 없어서 살 수 없다. |

## 8. Participio Passato: 일치 규칙

### 규칙 1: ESSERE와 함께 — 항상 주어에 일치

| 이탈리아어 | 한국어 |
|-----------|--------|
| Maria **è andata**. (여성 단수) | Maria가 갔다. |
| I ragazzi **sono partiti**. (남성 복수) | 소년들이 떠났다. |
| Le ragazze **sono arrivate**. (여성 복수) | 소녀들이 도착했다. |

### 규칙 2: AVERE와 함께 — 선행 직접 목적어 대명사에 일치

| 대명사 없음 (일치 없음) | 대명사 있음 (일치) |
|----------------------|------------------|
| Ho mangiato **la pizza**. | **L'**ho mangiata. (la pizza = 여성 단수) |
| Ho visto **i film**. | **Li** ho visti. (i film = 남성 복수) |
| Ho letto **le lettere**. | **Le** ho lette. (le lettere = 여성 복수) |
| Ho comprato **il libro**. | **L'**ho comprato. (il libro = 남성 단수) |

### 규칙 3: AVERE와 함께 — 간접 목적어 대명사에는 일치하지 않음

| 이탈리아어 | 한국어 |
|-----------|--------|
| Le ho **parlato**. (*parlata*가 아님) | 그녀에게 말했다. (*le* = 간접 목적어) |
| Gli ho **scritto**. (*scritti*가 아님) | 그에게 편지를 썼다. (*gli* = 간접 목적어) |

### 규칙 4: NE와 함께 — 일치

| 이탈리아어 | 한국어 |
|-----------|--------|
| Quante mele hai comprato? — **Ne** ho comprat**e** tre. | 사과 몇 개 샀어? — 세 개 샀어. |
| Quanti libri hai letto? — **Ne** ho lett**i** due. | 책 몇 권 읽었어? — 두 권 읽었어. |

## 9. 부정사 구문

### 전치사 + 부정사

| 구문 | 예시 | 한국어 |
|------|------|--------|
| **di** + 부정사 | Ho deciso **di partire**. | 떠나기로 결정했다. |
| **a** + 부정사 | Comincio **a capire**. | 이해하기 시작한다. |
| **per** + 부정사 | Studio **per imparare**. | 배우기 위해 공부한다. |
| **senza** + 부정사 | È uscito **senza parlare**. | 말없이 나갔다. |
| **prima di** + 부정사 | **Prima di uscire**, chiudi la porta. | 나가기 전에 문을 닫아. |
| **dopo** + 과거 부정사 | **Dopo aver mangiato**, siamo usciti. | 먹은 후 나갔다. |
| **invece di** + 부정사 | **Invece di studiare**, guarda la TV. | 공부 대신 TV를 본다. |

### DI + 부정사가 따르는 동사

| 동사 | 예시 |
|------|------|
| decidere di | Ho deciso di partire. |
| cercare di | Cerco di capire. |
| smettere di | Ha smesso di fumare. |
| dimenticare di | Ho dimenticato di chiamare. |
| promettere di | Ha promesso di venire. |
| pensare di | Penso di restare. |
| avere paura di | Ho paura di volare. |
| avere bisogno di | Ho bisogno di riposare. |

### A + 부정사가 따르는 동사

| 동사 | 예시 |
|------|------|
| cominciare a | Comincio a capire. |
| continuare a | Continua a piovere. |
| imparare a | Imparo a cucinare. |
| riuscire a | Non riesco a dormire. |
| andare a | Vado a fare la spesa. |
| venire a | Vieni a trovarmi! |
| aiutare a | Mi aiuti a portare le borse? |
| insegnare a | Mi insegna a guidare. |

### 부정사가 직접 따르는 동사 (전치사 없음)

| 동사 | 예시 |
|------|------|
| potere | Posso venire. |
| dovere | Devo partire. |
| volere | Voglio mangiare. |
| sapere | So nuotare. |
| preferire | Preferisco restare. |
| desiderare | Desidero viaggiare. |
| fare | Faccio riparare la macchina. |
| lasciare | Lascia parlare Maria. |

## 10. 수동태

### ESSERE + Participio Passato 구문

가장 일반적인 수동태 구문입니다. 과거 분사는 주어에 일치합니다.

| 능동 | 수동 |
|------|------|
| Marco mangia la pizza. | La pizza **è mangiata** da Marco. |
| Marco ha mangiato la pizza. | La pizza **è stata mangiata** da Marco. |
| Marco mangerà la pizza. | La pizza **sarà mangiata** da Marco. |

### VENIRE + Participio Passato 구문

**단순 시제**에서만 사용됩니다 (복합 시제 불가). 동작이 일어나는 것을 강조합니다.

| 이탈리아어 | 한국어 |
|-----------|--------|
| La porta **viene chiusa** alle otto. | 문은 8시에 닫힌다. (습관적) |
| Questa legge **verrà approvata** domani. | 이 법안은 내일 승인될 것이다. |

> *Venire* 수동태는 복합 시제에서 사용할 수 없습니다: *La porta è venuta chiusa*는 불가.

### ANDARE + Participio Passato 구문

의무나 필요성을 표현합니다 ("~해야 한다" / "~되어야 한다").

| 이탈리아어 | 한국어 |
|-----------|--------|
| Questo lavoro **va fatto** subito. | 이 일은 즉시 해야 한다. |
| Le regole **vanno rispettate**. | 규칙은 지켜야 한다. |
| Il latte **va conservato** in frigo. | 우유는 냉장고에 보관해야 한다. |

## 11. SI Impersonale와 SI Passivante

### SI Impersonale (비인칭 SI)

SI impersonale는 영어의 "one," "you" (일반적), "people," "they" (일반적)에 해당합니다. 동사는 항상 **3인칭 단수**입니다.

| 이탈리아어 | 한국어 |
|-----------|--------|
| In Italia **si mangia** bene. | 이탈리아에서는 잘 먹는다. / 사람들이 잘 먹는다. |
| **Si dice** che sia ricco. | 그가 부자라고 한다. |
| Come **si dice** "cat" in italiano? | 이탈리아어로 "cat"을 어떻게 말하나요? |
| Qui non **si può** fumare. | 여기서는 흡연할 수 없습니다. |
| **Si deve** studiare per imparare. | 배우려면 공부해야 한다. |

> **형용사와 함께**: SI impersonale는 동사가 단수임에도 **복수** 형용사/명사를 취합니다:
> - Se si è **stanchi**, bisogna riposare. (피곤하면 쉬어야 한다.)
> - Quando si è **giovani**, si ha più energia. (젊을 때 에너지가 더 많다.)

### SI Passivante (수동 SI)

SI passivante는 주어가 동사 뒤에 오는 수동 구문입니다. 동사는 주어의 수에 일치합니다.

| 이탈리아어 | 한국어 |
|-----------|--------|
| **Si vendono** appartamenti. | 아파트 매매. / 아파트가 판매됩니다. |
| Qui **si parla** italiano. | 여기서는 이탈리아어가 사용됩니다. |
| **Si affittano** camere. | 방 임대. |
| In questo negozio **si vendono** scarpe. | 이 가게에서 신발이 판매됩니다. |
| Queste cose non **si fanno**! | 이런 일은 하지 않는다! |

### SI Impersonale vs. SI Passivante

| SI Impersonale | SI Passivante |
|----------------|---------------|
| 동사는 항상 **단수** | 동사가 주어에 일치 (복수 가능) |
| 진정한 문법적 주어 없음 | 문법적 주어 있음 |
| Si mangia bene. (잘 먹는다.) | Si mangiano gli spaghetti. (스파게티가 먹힌다.) |
| Si deve lavorare. (일해야 한다.) | Si devono pagare le tasse. (세금을 내야 한다.) |

## 12. 연습 문제

### 연습 1: Passato Remoto — 활용하기

Passato remoto 형태를 쓰세요.

1. (io) essere → ________
2. (lui) fare → ________
3. (loro) dire → ________
4. (tu) scrivere → ________
5. (noi) vedere → ________
6. (io) nascere → ________
7. (lei) venire → ________
8. (loro) avere → ________
9. (io) leggere → ________
10. (lui) prendere → ________

<details>
<summary>정답</summary>

1. **fui**
2. **fece**
3. **dissero**
4. **scrivesti**
5. **vedemmo**
6. **nacqui**
7. **venne**
8. **ebbero**
9. **lessi**
10. **prese**

</details>

### 연습 2: 진행형

*stare + gerundio*를 사용하여 다시 쓰세요.

1. Mangio una pizza. → ________
2. Leggevi un libro. → ________
3. Cosa fai? → ________
4. I bambini giocano. → ________
5. Pioveva molto. → ________

<details>
<summary>정답</summary>

1. **Sto mangiando** una pizza.
2. **Stavi leggendo** un libro.
3. Cosa **stai facendo**?
4. I bambini **stanno giocando**.
5. **Stava piovendo** molto.

</details>

### 연습 3: 과거 분사 일치

과거 분사의 올바른 어미를 넣으세요.

1. Maria è andat___ a Roma.
2. Ho comprato le mele. → Le ho comprat___.
3. I ragazzi sono partit___ ieri.
4. Hai visto la mostra? — Sì, l'ho vist___.
5. Quanti libri hai letto? — Ne ho lett___ tre.
6. Le ho parlat___. (그녀에게 말했다.)

<details>
<summary>정답</summary>

1. andat**a** (essere → Maria에 일치, 여성 단수)
2. comprat**e** (le = 여성 복수 직접 목적어 대명사)
3. partit**i** (essere → i ragazzi에 일치, 남성 복수)
4. vist**a** (l' = la mostra, 여성 단수 직접 목적어)
5. lett**i** (ne가 i libri를 대체, 남성 복수)
6. parlat**o** (le = 간접 목적어, 일치 없음)

</details>

### 연습 4: 수동태

능동태를 수동태로 바꾸세요.

1. Marco ha scritto la lettera.
2. Il professore spiega la lezione.
3. Costruiranno un nuovo ponte.
4. (의무) Bisogna rispettare le regole.

<details>
<summary>정답</summary>

1. La lettera **è stata scritta** da Marco.
2. La lezione **è spiegata** / **viene spiegata** dal professore.
3. Un nuovo ponte **sarà costruito**.
4. Le regole **vanno rispettate**.

</details>

### 연습 5: SI Impersonale / SI Passivante

SI 구문을 사용하여 번역하세요.

1. 여기서는 이탈리아어가 사용됩니다.
2. 병원에서 흡연해서는 안 된다.
3. 이탈리아어로 "house"를 어떻게 말하나요?
4. 방 임대. (방들이 임대된다.)
5. 이탈리아에서 사람들은 잘 먹는다.
6. 이 신발은 할인가로 판매됩니다.

<details>
<summary>정답</summary>

1. Qui **si parla** italiano.
2. Non **si deve** fumare in ospedale.
3. Come **si dice** "house" in italiano?
4. **Si affittano** camere.
5. In Italia **si mangia** bene.
6. Queste scarpe **si vendono** in saldo.

</details>

### 연습 6: 부정사 구문

올바른 전치사(di, a, per, 또는 없음)를 넣으세요.

1. Ho deciso ________ partire domani.
2. Comincio ________ capire l'italiano.
3. Studio ________ superare l'esame.
4. Posso ________ aiutarti?
5. Ho dimenticato ________ chiamarlo.
6. Imparo ________ cucinare la pasta.
7. Preferisco ________ restare a casa.
8. Ha smesso ________ piovere.

<details>
<summary>정답</summary>

1. **di**
2. **a**
3. **per**
4. (없음 — potere + 부정사 직접)
5. **di**
6. **a**
7. (없음 — preferire + 부정사 직접)
8. **di**

</details>
