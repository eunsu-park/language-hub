## 1. 시간대별 인사

> **시간대별 인사**
>
> 06:00 — 13:00: Buongiorno (좋은 아침)
> 13:00 — 18:00: Buon pomeriggio (좋은 오후, 덜 흔함)
> 18:00 — 06:00: Buonasera (좋은 저녁)
>
> Buonanotte = 안녕히 주무세요 (잠자리에 들 때 작별 인사만)
>
> - Buongiorno는 이른 오후까지 광범위하게 사용됩니다
> - Buonasera는 오후 5-6시경 시작 (지역에 따라 다름)
> - Ciao!는 언제든 가능 (비격식만)

| 시간 | 인사 | 사용법 |
|------|------|--------|
| 아침 (~6:00-13:00) | Buongiorno! | 격식 & 비격식 |
| 오후 (~13:00-18:00) | Buon pomeriggio! | 격식 (덜 흔함) |
| 늦은 오후~저녁 | Buonasera! | 격식 & 비격식 |
| 밤 (작별) | Buonanotte! | 잠자리에 들 때 |
| 언제든 | Ciao! | 비격식만 |
| 언제든 | Salve! | 중립 (격식/비격식 사이) |

> **참고**: *Buongiorno*와 *buonasera*는 한 단어 또는 두 단어(*buon giorno*, *buona sera*)로 쓸 수 있습니다. 현대 용법에서는 한 단어 형태가 더 흔합니다.

## 2. 비격식 인사

### 2.1 흔한 비격식 인사

| 이탈리아어 | 한국어 | 비고 |
|-----------|--------|------|
| Ciao! | 안녕! / 잘 가! | 보편적 비격식 — 인사와 작별 모두 사용 |
| Come stai? | 어떻게 지내? | 표준 비격식 |
| Come va? | 잘 지내? | 매우 흔함, 격의 없음 |
| Che si dice? | 무슨 일이야? (직역: 뭐라고 하지?) | 구어체 |
| Tutto bene? | 다 잘 돼? | 가벼운 안부 |
| Come te la passi? | 잘 지내고 있어? | 친근함 |
| Che fai (di bello)? | 뭐 하고 있어? | 근황 물을 때 |
| Ehi! / Ehilà! | 야! | 매우 격의 없음 |

### 2.2 지역별 비격식 인사

| 표현 | 지역 | 의미 |
|------|------|------|
| Uè! | 남부 이탈리아 (나폴리) | 야! |
| Ciao bello/bella! | 일반 (애정 표현) | 안녕 이쁜이! (친구에게) |
| Che mi racconti? | 일반 | 무슨 소식이야? (직역: 나에게 뭘 말해주지?) |
| 'Ngiorno! | 남부 (비격식) | Buongiorno의 축약형 |
| Oi! / Ohi! | 사르데냐 / 일부 지역 | 야! |
| Mandi! | 프리울리 | 안녕/잘 가 |

## 3. 격식 vs 비격식: Tu vs Lei

### 3.1 두 가지 "당신"

> **TU (비격식 "너/당신")** — 친구 & 동료, 가족, 어린이, 같은 나이 급우, 비격식 상황
> **LEI (격식 "당신")** — 모르는 사람, 어르신, 상사 / 윗사람, 공무원 / 경찰, 격식 상황
>
> - **동사**: TU = 2인칭 단수 / LEI = 3인칭 단수
> - Come stai? (어떻게 지내?) / Come sta? (어떻게 지내십니까?)
> - Hai tempo? (시간 있어?) / Ha tempo?
>
> - Lei는 글에서 lei (그녀)와 구별하기 위해 항상 대문자로 씁니다. 이것은 이탈리아어 고유의 관습입니다.
> - 확실하지 않으면 LEI를 사용하세요 — tu로 전환하는 것을 "dare del tu"라 하며 보통 상호 합의가 필요합니다.
> - Voi (너희/여러분)는 격식·비격식 모두에서 복수로 사용됩니다. Loro (격식 복수 "당신들")는 존재하지만 드뭅니다.

### 3.2 빠른 동사 형태 비교

| 상황 | Tu (비격식) | Lei (격식) |
|------|------------|-----------|
| 어떻게 지내세요? | Come **stai**? | Come **sta**? |
| 어디 출신이세요? | Di dove **sei**? | Di dov'**è**? |
| 이름이 뭐예요? | Come **ti** chiami? | Come **si** chiama? |
| ~어를 하세요? | **Parli** italiano? | **Parla** italiano? |
| 도와주실 수 있어요? | **Puoi** aiutarmi? | **Può** aiutarmi? |
| ~하시겠어요? | **Vuoi** un caffè? | **Vuole** un caffè? |

### 3.3 "Dare del tu" — 비격식으로 전환하기

```
누군가 Lei에서 tu로 전환하고 싶을 때 이렇게 말합니다:
  "Diamoci del tu!"    → 서로 tu를 쓰자!
  "Possiamo darci del tu?"  → tu를 써도 될까요?
  "Mi puoi dare del tu."    → 나에게 tu를 써도 돼요.

이것은 이탈리아 문화에서 일종의 사교적 의식입니다.
관계가 더 친근하거나 동등해졌음을 나타냅니다.
```

## 4. 자기소개

### 4.1 기본 구성 요소

```
이름:    Mi chiamo [이름].         → 내 이름은 [이름]입니다.
         (Sono [이름].)           → ([이름]입니다.)

출신:    Sono di [도시/나라].      → 저는 [도시/나라] 출신입니다.
         Sono [국적].             → 저는 [국적]입니다.

나이:    Ho [숫자] anni.          → 저는 [숫자]살입니다.

직업:    Lavoro come [직업].      → 저는 [직업]으로 일합니다.
         Lavoro a/per [회사].     → 저는 [회사]에서/를 위해 일합니다.

학업:    Studio [과목].           → 저는 [과목]을 공부합니다.
         Sono studente/studentessa. → 저는 학생입니다 (남/여).

언어:    Parlo [언어].            → 저는 [언어]를 합니다.
         Sto imparando l'italiano. → 이탈리아어를 배우고 있습니다.
```

### 4.2 주요 국적

| 이탈리아어 (남/여) | 한국어 |
|-------------------|--------|
| coreano / coreana | 한국인 |
| americano / americana | 미국인 |
| inglese / inglese | 영국인 (형태 동일) |
| francese / francese | 프랑스인 (형태 동일) |
| tedesco / tedesca | 독일인 |
| giapponese / giapponese | 일본인 (형태 동일) |
| cinese / cinese | 중국인 (형태 동일) |
| spagnolo / spagnola | 스페인인 |
| italiano / italiana | 이탈리아인 |
| brasiliano / brasiliana | 브라질인 |

### 4.3 주요 직업

| 이탈리아어 (남/여) | 한국어 |
|-------------------|--------|
| professore / professoressa | 교사/교수 |
| ingegnere / ingegnera | 엔지니어 |
| medico / medico (또는 medica) | 의사 |
| avvocato / avvocata (또는 avvocatessa) | 변호사 |
| studente / studentessa | 학생 |
| programmatore / programmatrice | 프로그래머 |
| designer / designer | 디자이너 |
| imprenditore / imprenditrice | 기업가 |

### 4.4 자기소개 예시 대화

```
대화 1 (비격식 — tu):
A: Ciao! Come ti chiami?
B: Mi chiamo Min-jun. E tu?
A: Sono Maria. Di dove sei?
B: Sono della Corea del Sud. Sono studente.
A: Che bello! Piacere, Min-jun!
B: Piacere mio, Maria.

대화 2 (격식 — Lei):
A: Buongiorno. Come si chiama?
B: Mi chiamo Park Min-jun. Piacere.
A: Piacere. Sono la dottoressa Rossi. Di dov'è?
B: Sono di Seoul, Corea del Sud.
A: E cosa fa qui a Roma?
B: Studio italiano all'università.
```

## 5. "어떻게 지내세요?"에 대한 응답

### 5.1 응답 스펙트럼

```
매우 긍정적:
  Benissimo!            → 아주 잘 지내요!
  Alla grande!          → 최고예요! (구어체)
  Da dio!               → 환상적이에요! (구어체)
  Meravigliosamente!    → 훌륭해요!
  Molto bene!           → 아주 잘 지내요!

긍정적:
  Bene, grazie.         → 잘 지내요, 감사합니다.
  Tutto bene.           → 다 좋아요.
  Non c'è male.         → 나쁘지 않아요.
  Abbastanza bene.      → 꽤 잘 지내요.

중립:
  Così così.            → 그저 그래요.
  Insomma...            → 글쎄요... (그저 그런 느낌)
  Si tira avanti.       → 그럭저럭 지내요.
  Non mi lamento.       → 불평할 건 없어요.

부정적:
  Non molto bene.       → 별로 안 좋아요.
  Un po' stanco/a.      → 좀 피곤해요.
  Male.                 → 안 좋아요.
  Malissimo.            → 최악이에요.
```

### 5.2 완전한 대화 교환

```
비격식:
A: Ciao! Come stai?           → 안녕! 어떻게 지내?
B: Bene, grazie. E tu?        → 잘 지내, 고마워. 너는?
A: Molto bene, grazie.        → 아주 잘 지내, 고마워.

격식:
A: Buongiorno. Come sta?      → 안녕하세요. 어떻게 지내십니까?
B: Bene, grazie. E Lei?       → 잘 지냅니다, 감사합니다. 당신은요?
A: Bene, grazie.              → 잘 지냅니다, 감사합니다.

격의 없는:
A: Ciao! Come va?             → 야! 어때?
B: Bene, e te?                → 좋아, 너는?
A: Tutto bene!                → 다 좋아!
```

## 6. 작별 인사

> **격식 작별**: Arrivederci. / ArrivederLa. (매우 격식) / A presto. / Buona giornata. / Buona serata.
> **비격식 작별**: Ciao! / Ci vediamo! / A dopo! / Stammi bene! / A presto!
>
> **시간별**:
> - A domani. (내일 봐요.)
> - A lunedì. (월요일에 봐요.)
> - Alla prossima. (다음에 또요.)
> - Buonanotte. (안녕히 주무세요. — 잠자리 작별만)
>
> - Ciao는 인사와 작별 모두에 사용 (비격식)
> - Arrivederci는 격식/중립 작별에 항상 안전
> - ArrivederLa는 극존칭 (Lei 형태)

## 7. 공손한 표현

### 7.1 감사 표현

| 표현 | 의미 | 격식도 |
|------|------|--------|
| Grazie. | 감사합니다. | 보편적 |
| Grazie mille. | 정말 감사합니다. | 따뜻함 |
| Grazie tante. | 대단히 감사합니다. | 따뜻함 |
| Ti/Le ringrazio. | 감사드립니다. (비격식/격식) | 더 개인적 |
| Molto gentile. | 정말 친절하시군요. | 격식 |
| Grazie di tutto. | 모든 것에 감사합니다. | 따뜻함 |

**감사에 대한 응답:**
- **Prego.** — 천만에요. (가장 흔함)
- **Di niente. / Di nulla.** — 아무것도 아니에요.
- **Non c'è di che.** — 별말씀을요.
- **Figurati! (tu) / Si figuri! (Lei)** — 별말씀을요! / 전혀요!
- **È un piacere.** — 기쁨입니다.
- **Ma di che!** — 뭘요! (구어체)

### 7.2 사과 표현

> - **Scusa (tu) / Scusi (Lei)** → 미안해요 / 실례합니다 (관심 끌기, 사소한 일)
> - **Mi scusi (Lei) / Scusami (tu)** → 실례합니다 (더 정중/개인적)
> - **Mi dispiace (molto).** → (매우) 죄송합니다. (진심 어린 사과, 동정 표현)
> - **Chiedo scusa.** → 사과드립니다. (더 격식)
> - **Permesso.** → 실례합니다. (지나갈 때, 방/엘리베이터에 들어갈 때)
> - **È stata colpa mia.** → 제 잘못이었습니다.
> - **Non era mia intenzione.** → 의도한 것이 아니었습니다.

**사과에 대한 응답:**
- **Non fa niente.** — 아무것도 아니에요.
- **Non ti preoccupare. (tu) / Non si preoccupi. (Lei)** — 걱정 마세요.
- **Va bene.** — 괜찮아요.
- **Nessun problema.** — 문제없어요.
- **Figurati! (tu) / Si figuri! (Lei)** — 신경 쓰지 마세요!

### 7.3 요청 표현

| 정중도 | 이탈리아어 | 한국어 |
|--------|-----------|--------|
| 직접적 | Puoi aiutarmi? | 도와줄 수 있어? |
| 정중 | Potresti aiutarmi? | 도와줄 수 있을까? |
| 격식 | Potrebbe aiutarmi? | 도와주실 수 있으신가요? (Lei) |
| 매우 정중 | Sarebbe possibile...? | ~이 가능할까요? |
| 격식 요청 | Vorrei... | ~하고 싶습니다... |
| 극존칭 | Desidererei... | ~을 원합니다... (매우 격식) |

> **Per favore** (부탁합니다)와 **per cortesia** (더 격식적인 "부탁합니다")를 어떤 요청에든 붙일 수 있습니다.

## 8. 기본 대화 패턴

### 8.1 바 (카페)에서

```
Cliente: Buongiorno!
Barista: Buongiorno! Cosa prende?
Cliente: Un caffè, per favore.
Barista: Macchiato, lungo, ristretto?
Cliente: Un espresso normale, grazie.
Barista: Altro?
Cliente: No, basta così. Grazie.
Barista: Un euro e venti.
Cliente: Ecco a Lei. Grazie.
Barista: Grazie a Lei. Buona giornata!
```

> **문화 참고**: 이탈리아에서는 보통 바에 서서(*al banco*) 에스프레소를 마시는데, 이것이 테이블에 앉는 것(*al tavolo*)보다 저렴합니다. 많은 바가 *banco*와 *tavolo* 서비스에 다른 가격을 적용합니다.

### 8.2 길 묻기

```
A: Scusi, dov'è la stazione?
B: Vada dritto e giri a destra alla seconda strada.
A: È lontano?
B: No, sono cinque minuti a piedi.
A: Grazie mille.
B: Di niente.
```

### 8.3 새로운 사람 만나기

```
A: Ciao! Sei nuovo qui?
B: Sì, mi chiamo Marco. Piacere!
A: Piacere, Marco. Sono Lucia. Di dove sei?
B: Sono di Milano. E tu?
A: Di qui, di Roma. Benvenuto!
B: Grazie! Tutti sono molto gentili.
```

### 8.4 전화 통화

```
전화 받기: Pronto? (여보세요? — 표준 전화 인사)
묻기: Pronto, chi parla? (여보세요, 누구시죠?)
요청: Potrei parlare con...? (~와 통화할 수 있을까요?)
메시지 남기기: Posso lasciare un messaggio? (메시지를 남겨도 될까요?)
```

## 9. 유용한 첫날 표현

### 9.1 생존 표현

> **첫날을 위한 필수 표현**
>
> - **Per favore.** → 부탁합니다.
> - **Grazie.** → 감사합니다.
> - **Prego.** → 천만에요.
> - **Sì. / No.** → 네. / 아니오.
> - **Non capisco.** → 이해하지 못합니다.
> - **Può ripetere, per favore?** → 다시 말씀해 주시겠어요?
> - **Come si dice ___ in italiano?** → 이탈리아어로 ___을 뭐라 하나요?
> - **Che cosa significa ___?** → ___은 무슨 뜻인가요?
> - **Più lentamente, per favore.** → 더 천천히요, 부탁합니다.
> - **Non parlo (bene) l'italiano.** → 이탈리아어를 (잘) 못합니다.
> - **Parlo un po' di italiano.** → 이탈리아어를 조금 합니다.
> - **Parla inglese?** → 영어 하시나요?
> - **Dov'è il bagno?** → 화장실이 어디인가요?
> - **Quanto costa?** → 얼마인가요?
> - **Il conto, per favore.** → 계산서 부탁합니다.
> - **Aiuto!** → 도와주세요!
> - **Mi sono perso/a.** → 길을 잃었습니다. (남/여)
> - **Posso avere...?** → ~을 받을 수 있을까요?

### 9.2 기차역에서

```
Viaggiatore: Buongiorno. Un biglietto per Firenze, per favore.
Bigliettaio: Solo andata o andata e ritorno?
Viaggiatore: Andata e ritorno, per favore.
Bigliettaio: Per quando?
Viaggiatore: Per oggi. A che ora parte il prossimo treno?
Bigliettaio: Alle 14:30. Binario 5.
Viaggiatore: Perfetto. Quanto costa?
Bigliettaio: Quarantacinque euro.
Viaggiatore: Ecco. Grazie.
Bigliettaio: Buon viaggio!
```

### 9.3 레스토랑에서

```
Cameriere: Buonasera! Quanti siete?
Cliente: Siamo in due.
Cameriere: Prego, accomodatevi. Ecco il menù.
  (... dopo un po' ...)
Cameriere: Avete scelto?
Cliente: Sì. Per me, la pasta alla carbonara.
Cameriere: E da bere?
Cliente: Un bicchiere di vino rosso, per favore.
Cameriere: Perfetto. Subito!
  (... dopo il pasto ...)
Cliente: Il conto, per favore.
Cameriere: Ecco a Lei. Sono ventidue euro.
Cliente: Posso pagare con la carta?
Cameriere: Certamente. Grazie e arrivederci!
```

### 연습 1: 적절한 인사 선택
각 상황에 가장 적합한 인사를 고르세요.

   → Buongiorno, professore.

   → Buonasera.

3. 아무 때나 친구에게 문자할 때
   → Ciao! Come va?

4. 오후 8시에 격식 저녁 식사에 도착할 때
   → Buonasera.

5. 잠자리에 드는 사람에게 작별할 때

   → Salve!

### 연습 4: 이탈리아어로 번역
다음을 이탈리아어로 표현하세요.

<details>
<summary>정답 보기</summary>

```
1. 실례합니다, 화장실이 어디인가요?
   → Scusi, dov'è il bagno?

2. 이해하지 못합니다. 다시 말씀해 주시겠어요?
   → Non capisco. Può ripetere?

3. 죄송합니다, 이탈리아어를 잘 못합니다.
   → Mi dispiace, non parlo bene l'italiano.

4. 얼마인가요?
   → Quanto costa?

   → Grazie mille! — Prego!
```

</details>
