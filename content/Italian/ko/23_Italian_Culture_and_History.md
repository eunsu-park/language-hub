## 1. 이탈리아 약사

### 1.1 연표

> **이탈리아 역사의 주요 시대**
>
> - **BC 753**: 로마의 전설적 건국
> - **BC 509**: 로마 공화정 수립
> - **BC 27**: 로마 제정 시작 (아우구스투스)
> - **AD 476**: 서로마 제국 멸망
>
> - **500-1000**: 초기 중세 (침입, 봉건제)
> - **1000-1300**: 도시 국가의 부상 (comuni)
> - **1300-1600**: 르네상스 (Rinascimento)
> - **1500-1700**: 외세 지배 (스페인, 오스트리아, 프랑스)
>
> - **1815**: 빈 회의 (이탈리아 국가들 복원)
> - **1848-1861**: 리소르지멘토 (이탈리아 통일 운동)
> - **1861**: 이탈리아 왕국 선포
> - **1870**: 로마가 수도가 됨
>
> - **1915-1918**: 제1차 세계대전 (이탈리아 연합국 참전)
> - **1922-1943**: 무솔리니 치하 파시즘
> - **1940-1945**: 제2차 세계대전
> - **1946**: 이탈리아 공화국 (국민투표)
> - **1957**: 유럽경제공동체(EEC, 현 EU) 창립 회원국
> - **1958-1963**: "Miracolo economico" (경제 기적)
> - **2002**: 유로화 도입

### 1.2 주요 역사 시대

| 시대 | 기간 | 주요 인물/사건 | 문화적 영향 |
|--------|-------|-------------------|----------------|
| **로마 제국** | BC 27 - AD 476 | 아우구스투스, 율리우스 카이사르, 네로 | 라틴어, 법률, 도로, 수도교, 건축 |
| **중세 도시 국가** | 1000-1400 | 베네치아, 피렌체, 제노바, 밀라노 | 무역, 은행업, 초기 민주주의 (comuni) |
| **르네상스** | ~1300-1600 | 다빈치, 미켈란젤로, 마키아벨리 | 예술, 과학, 인문주의, 건축 |
| **리소르지멘토** | 1815-1871 | 가리발디, 마치니, 카보우르, 비토리오 에마누엘레 2세 | 국가 통일, 이탈리아 창건 |
| **파시즘 시대** | 1922-1943 | 무솔리니, 로마 진군 (1922) | 독재 통치, 제2차 세계대전 참전 |
| **이탈리아 공화국** | 1946-현재 | 1946년 6월 2일 국민투표 | 민주주의, 경제 기적, EU 창립 회원국 |

### 1.3 리소르지멘토의 영웅들

| 인물 | 역할 | 유명한 업적 |
|--------|------|-----------|
| **주세페 가리발디** | 군사 지도자 | *Spedizione dei Mille* (천인 원정대) -- 남부 이탈리아 정복 |
| **주세페 마치니** | 정치 사상가 | *Giovine Italia* (청년 이탈리아) 창설 -- 통일의 이념적 아버지 |
| **카밀로 벤소 디 카보우르** | 사르데냐 수상 | 통일의 외교적 설계자, 프랑스와 협상 |
| **비토리오 에마누엘레 2세** | 사르데냐 왕, 이후 이탈리아 왕 | 통일 이탈리아 초대 국왕 (1861) |

>

## 3. 지역별 이탈리아 요리

### 3.1 북부 요리 (Cucina del Nord)

| 특징 | 설명 |
|---------|-------------|
| **주식 탄수화물** | 쌀, 폴렌타, 생 달걀 파스타 (건면보다 많음) |
| **지방** | 버터(burro)가 올리브 오일보다 더 일반적 |
| **고기** | 송아지, 돼지, 가공육 (bresaola, speck) |
| **치즈** | Parmigiano-Reggiano, Gorgonzola, fontina, taleggio |
| **대표 요리** | Risotto alla milanese, polenta, vitello tonnato, bollito misto |

### 3.2 중부 요리 (Cucina del Centro)

| 특징 | 설명 |
|---------|-------------|
| **주식 탄수화물** | 생면과 건면 모두, 빵 |
| **지방** | 엑스트라 버진 올리브 오일 (특히 토스카나와 움브리아) |
| **고기** | 돼지, 멧돼지(cinghiale), bistecca alla fiorentina |
| **치즈** | Pecorino toscano, pecorino romano |
| **대표 요리** | Bistecca alla fiorentina, ribollita, cacio e pepe, amatriciana, carbonara |

### 3.3 남부 요리 (Cucina del Sud)

| 특징 | 설명 |
|---------|-------------|
| **주식 탄수화물** | 건면 (특히 orecchiette, paccheri 같은 모양) |
| **지방** | 엑스트라 버진 올리브 오일 (Puglia가 이탈리아 최대 생산지) |
| **단백질** | 해산물, mozzarella di bufala |
| **채소** | 토마토, 가지, 피망, 아티초크 |
| **대표 요리** | Pizza napoletana, pasta alla norma, caponata, 'nduja |

## 5. 음악: 오페라에서 현대 팝까지

### 5.1 오페라 (L'Opera Lirica)

이탈리아는 오페라의 발상지입니다 (16세기 후반, 피렌체).

| 작곡가 | 시대 | 유명한 오페라 |
|----------|-------|---------------|
| **클라우디오 몬테베르디** | 1567-1643 | *L'Orfeo* (최초의 위대한 오페라, 1607) |
| **조아키노 로시니** | 1792-1868 | *Il Barbiere di Siviglia*, *Guglielmo Tell* |
| **가에타노 도니제티** | 1797-1848 | *L'Elisir d'Amore*, *Lucia di Lammermoor* |
| **주세페 베르디** | 1813-1901 | *La Traviata*, *Aida*, *Rigoletto*, *Nabucco* |
| **자코모 푸치니** | 1858-1924 | *La Bohème*, *Tosca*, *Madama Butterfly*, *Turandot* |

> **문화적 참고**: 베르디의 *Nabucco* 중 합창곡 "Va, pensiero"는 이탈리아 통일의 비공식 국가가 되었습니다. 베르디의 이름 자체가 약어가 되었습니다: **V.E.R.D.I.** = *Vittorio Emanuele Re D'Italia* (이탈리아의 왕 비토리오 에마누엘레).

### 5.2 이탈리아어에서 유래한 음악 용어

전 세계적으로 사용되는 많은 음악 용어가 이탈리아어에서 유래했습니다:

| 이탈리아어 용어 | 의미 | 영어 사용 맥락 |
|-------------|---------|---------------|
| piano / forte | 여리게 / 세게 | 다이내믹스 |
| crescendo | 점점 크게 | 점진적 증가 |
| allegro | 쾌활하게, 빠르게 | 빠른 빠르기 |
| andante | 걸어가듯이 | 보통 빠르기 |
| adagio | 느긋하게 | 느린 빠르기 |
| staccato | 끊어서 | 짧고 분리된 음표 |
| legato | 이어서 | 부드럽고 연결된 음표 |
| soprano / alto / tenore / basso | 성부 범위 | 성악 분류 |
| a cappella | 성당 양식으로 | 악기 없는 노래 |
| libretto | 작은 책 | 오페라의 가사 |

## 6. 이탈리아 영화

### 6.1 네오리얼리즘 (Neorealismo) -- 1940-1950년대

비전문 배우와 실제 장소를 활용하여 이탈리아의 일상을 묘사한 혁명적 운동:

| 감독 | 주요 작품 | 주제 |

## 10. 이탈리아 일상 문화

### 10.1 커피 문화 (Il Caffè)

| 종류 | 설명 | 시간대 |
|------|------------|------|
| **Espresso** | 작고, 진하고, 블랙 | 언제든지 |
| **Cappuccino** | 에스프레소 + 스팀 밀크 거품 | 아침 식사 때만 (식사 후에는 절대 안 됨) |
| **Caffè macchiato** | 에스프레소 + 우유 한 방울 | 언제든지 |
| **Caffè lungo** | 약간 긴 에스프레소 | 언제든지 |
| **Caffè corretto** | 에스프레소 + 그라파/리큐어 | 점심/저녁 식사 후 |
| **Caffè d'orzo** | 보리 커피 (카페인 없음) | 언제든지 |

> **문화적 규칙**: 이탈리아 바에서 카운터(*al bancone*)에 서서 마시는 것이 테이블(*al tavolo*)에 앉는 것보다 저렴합니다. 먼저 계산대(*cassa*)에서 주문하고 결제한 다음, 영수증(*scontrino*)을 바에 가져갑니다.

### 10.2 산책 문화 (La Passeggiata)

*passeggiata*는 이탈리아인들이 마을의 주요 거리를 이른 저녁에 산책하는 전통입니다:

- **시간**: 보통 오후 6시-8시 (저녁 식사 전)
- **목적**: 사교, 보고 보이기, 윈도우 쇼핑
- **참여자**: 모든 사람 -- 가족, 커플, 친구, 어르신
- **장소**: 중심 거리(*corso*), 광장(piazza), 해변 산책로(lungomare)
- **복장**: 이탈리아인들은 산책 때 보통 잘 차려입음 (*fare bella figura*)

### 10.3 식사 (I Pasti)

| 식사 | 시간 | 설명 |
|------|------|------------|
| **Colazione** (아침) | 7:00-9:00 | 가벼움: 바에서 카푸치노 + 코르네토 (크루아상) |
| **Pranzo** (점심) | 12:30-14:00 | 전통적으로 주요 식사 (primo + secondo + contorno) |
| **Merenda** (간식) | 16:00-17:00 | 오후 간식, 특히 어린이용 |
| **Aperitivo** | 18:00-20:00 | 식전 음료 (Aperol Spritz, Negroni)와 stuzzichini (안주) |
| **Cena** (저녁) | 20:00-21:30 | 점심보다 가벼울 수도, 또는 비슷할 수도 있음 |

### 10.4 좋은 인상 만들기 (Fare Bella Figura)

*Fare bella figura* (좋은 인상 만들기)는 이탈리아 문화에서 가장 중요한 개념 중 하나입니다:

| 측면 | 설명 |
|--------|------------|
| **외모** | 잘 차려입는 것은 허영이 아님 -- 타인에 대한 존중의 표시 |
| **행동** | 공공장소에서 예의 바르고, 관대하며, 침착하게 행동하기 |
| **사교** | 손님을 잘 대접하고, 초대받으면 선물 가져가기 |
| **반대** | *Fare brutta figura* = 나쁜 인상 만들기 (무슨 수를 써서든 피해야!) |

### 연습 4: 문화 참/거짓

1. 이탈리아인들은 점심 후에 카푸치노를 흔히 마신다.
2. Ferragosto (8월 15일)는 대부분의 이탈리아인들이 여름 휴가를 가는 때이다.
3. 이탈리아 바에서 테이블에 앉는 것은 카운터에 서는 것과 가격이 같다.
4. *Fare bella figura*는 나쁜 인상을 만든다는 의미이다.
5. La Befana는 1월 6일에 선물을 가져다주는 노파이다.
6. 산책(passeggiata)은 보통 이른 아침에 이루어진다.
7. 렌즈콩은 전통적으로 행운을 위해 새해 전야에 먹는다.
8. 1861년 이탈리아 통일 당시 대부분의 이탈리아인들은 표준 이탈리아어를 유창하게 구사했다.

<details>
<summary>정답</summary>

1. **거짓** -- 카푸치노는 아침 식사 음료뿐입니다. 식사 후에는 에스프레소를 마십니다.
2. **참** -- Ferragosto는 이탈리아 여름 휴가의 절정; 온 나라가 사실상 문을 닫습니다.
3. **거짓** -- 테이블(*al tavolo*)에 앉는 것이 카운터(*al bancone*)에 서는 것보다 비쌉니다.
