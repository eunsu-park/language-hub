## 1. 성 일치

이탈리아어 형용사는 수식하는 명사의 **성** (남성/여성)과 **수** (단수/복수)에 일치해야 합니다. 이를 *concordanza*라고 합니다.

### 패턴 1: 4형 형용사 (-o / -a / -i / -e) — 가장 일반적

| | 남성 | 여성 |
|---|---|---|
| **단수** | alt**o** | alt**a** |
| **복수** | alt**i** | alt**e** |

| 남성 | 여성 | 한국어 |
|-----------|----------|---------|
| alto | alta | 키가 큰 |
| basso | bassa | 키가 작은 |
| bello | bella | 아름다운 |
| piccolo | piccola | 작은 |
| nuovo | nuova | 새로운 |
| vecchio | vecchia | 오래된 |
| ricco | ricca | 부유한 |
| bravo | brava | 잘하는/유능한 |

> **규칙**: -o (남.단) → -a (여.단) → -i (남.복) → -e (여.복)

### 패턴 2: 2형 형용사 (-e / -i) — 성에 대해 불변

이 형용사들은 남성과 여성에서 **같은 형태**입니다. 수에 대해서만 변화합니다.

| | 단수 | 복수 |
|---|---|---|
| **남성 & 여성** | grand**e** | grand**i** |

| 형용사 | 한국어 | 예시 |
|-----------|---------|---------|
| grande | 큰 | un uomo **grande** / una casa **grande** |
| intelligente | 지적인 | un ragazzo **intelligente** / una ragazza **intelligente** |
| triste | 슬픈 | il bambino **triste** / la bambina **triste** |
| felice | 행복한 | un giorno **felice** / una sera **felice** |
| facile | 쉬운 | un esame **facile** / una domanda **facile** |
| difficile | 어려운 | un problema **difficile** / una questione **difficile** |
| giovane | 젊은 | un uomo **giovane** / una donna **giovane** |
| gentile | 친절한 | un gesto **gentile** / una persona **gentile** |
| forte | 강한 | un caffè **forte** / una donna **forte** |
| dolce | 달콤한 | un sapore **dolce** / una voce **dolce** |

### 패턴 3: 국적 형용사 (혼합형)

| 남성 단수 | 여성 단수 | 남성 복수 | 여성 복수 | 한국어 |
|-----------|----------|--------|--------|---------|
| italiano | italiana | italiani | italiane | 이탈리아의 |
| francese | francese | francesi | francesi | 프랑스의 |
| inglese | inglese | inglesi | inglesi | 영국의 |
| tedesco | tedesca | tedeschi | tedesche | 독일의 |
| spagnolo | spagnola | spagnoli | spagnole | 스페인의 |
| giapponese | giapponese | giapponesi | giapponesi | 일본의 |
| coreano | coreana | coreani | coreane | 한국의 |

> **참고**: -ese로 끝나는 국적은 2형 (남/여 동일)입니다. -o/-a로 끝나는 것은 4형입니다.

## 2. 수 일치

### 모음 어미 → 마지막 모음 변경

| 단수 → 복수 | 패턴 |
|---|---|
| -o → -i | alto → alti |
| -a → -e | alta → alte |
| -e → -i | grande → grandi |

### 철자 변화

```
-co → -chi (경음 유지, 대부분의 경우):
  bianco → bianchi       ricco → ricchi
  단, simpatico → simpatici (끝에서 세 번째 음절 강세일 때)

-ca → -che (항상):
  bianca → bianche       ricca → ricche

-go → -ghi (보통):
  lungo → lunghi         largo → larghi

-ga → -ghe (항상):
  lunga → lunghe         larga → larghe
```

### 혼합 성별 그룹

형용사가 **혼합 성별** 그룹을 수식할 때는 **남성 복수**를 사용합니다:

- I ragazzi e le ragazze sono **alti**. (소년들과 소녀들은 키가 크다.)
- Marco e Maria sono **italiani**. (Marco와 Maria는 이탈리아인이다.)

## 4. 형용사 위치

### 기본: 명사 뒤

대부분의 형용사는 이탈리아어에서 명사 **뒤에** 옵니다 (영어와 반대):

| 이탈리아어 | 한국어 |
|---------|---------|
| una macchina **rossa** | 빨간 자동차 |
| una casa **grande** | 큰 집 |
| un libro **interessante** | 흥미로운 책 |
| una persona **gentile** | 친절한 사람 |
| un ragazzo **alto** | 키 큰 소년 |

### 명사 앞에 오는 경우 (특수한 경우)

일부 일반적인 형용사는 특히 강조나 문체적 효과를 위해 명사 **앞에** 올 수 있습니다:

| 범주 | 예시 |
|----------|---------|
| 양 | molto, poco, tanto, troppo, ogni, qualche |
| 순서 | primo, secondo, ultimo, prossimo |
| 미/질 | bello, buono, brutto, cattivo, bravo |
| 크기/나이 | grande, piccolo, giovane, vecchio, nuovo |
| 본질적 특성 | la bianca neve (하얀 눈) — 문어체 |

> **경험 법칙**: 짧고 일반적이며 주관적인 형용사는 명사 앞에 올 수 있습니다. 길고 기술적이거나 분류하는 형용사는 뒤에 옵니다.

### 남성 단수 명사 앞에서의 단축 형태

일부 형용사는 **남성 단수 명사** 앞에서 마지막 모음이 탈락하거나 형태가 변합니다:

| 원형 | 단축형 | 예시 |
|-----------|-----------|---------|
| buono | **buon** | un **buon** amico (좋은 친구) |
| bello | **bel** | un **bel** giorno (아름다운 날) |
| grande | **gran** | un **gran** uomo (위대한 사람) |

> **참고**: 이 단축 형태는 자음으로 시작하는 남성 단수 명사 앞에서 사용됩니다 (s+자음, z 등 제외). 자세한 내용은 5절을 참조하세요.

## 5. 특수 형용사: BELLO, BUONO, GRANDE, QUELLO

### 5.1 BELLO (아름다운) — 정관사 패턴을 따름

명사 **앞에** 올 때, BELLO는 정관사를 반영하는 형태를 취합니다:

| 관사 | BELLO 형태 | 예시 |
|---------|-----------|---------|
| il → | **bel** | un **bel** ragazzo (잘생긴 소년) |
| lo → | **bello** | un **bello** spettacolo (아름다운 공연) |
| l' → | **bell'** | un **bell'**uomo (잘생긴 남자) |
| la → | **bella** | una **bella** donna (아름다운 여자) |
| l' → | **bell'** | una **bell'**idea (멋진 아이디어) |
| i → | **bei** | dei **bei** ragazzi (잘생긴 소년들) |
| gli → | **begli** | dei **begli** occhi (아름다운 눈들) |
| le → | **belle** | delle **belle** case (아름다운 집들) |

> 명사 뒤에 올 때, BELLO는 규칙적입니다: un ragazzo **bello**, una donna **bella**, ragazzi **belli**, donne **belle**.

### 5.2 BUONO (좋은) — 부정관사 패턴을 따름

명사 **앞에** 올 때, BUONO는 부정관사와 유사한 형태를 취합니다 (단수만):

| 관사 | BUONO 형태 | 예시 |
|---------|-----------|---------|
| un → | **buon** | un **buon** libro (좋은 책) |
| uno → | **buono** | un **buono** studente (좋은 학생) |
| una → | **buona** | una **buona** idea (좋은 아이디어) |
| un' → | **buon'** | una **buon'**amica (좋은 친구, 여) |

복수: 규칙적 — **buoni** (남.복), **buone** (여.복)
- dei **buoni** libri (좋은 책들)
- delle **buone** amiche (좋은 친구들)

### 5.3 GRANDE (큰/위대한)

명사 앞에서 GRANDE는 대부분의 자음 앞에서 **gran**으로 단축될 수 있습니다 (양성 모두):

```
un gran uomo (위대한 사람)       una gran donna (위대한 여성)
un gran problema (큰 문제)       una gran festa (멋진 파티)

s+자음, z 앞: grande 유지
  un grande spettacolo    una grande storia

모음 앞: grand' 사용 가능
  un grand'amico          una grand'amica

명사 뒤: 항상 규칙적
  un uomo grande (큰 사람 — 물리적 크기)
```

### 5.4 QUELLO (저/그) — 정관사 패턴을 따름

QUELLO는 정관사 형태를 정확히 반영합니다 (BELLO와 동일):

| 관사 | QUELLO 형태 | 예시 |
|---------|-----------|---------|
| il → | **quel** | **quel** ragazzo (저 소년) |
| lo → | **quello** | **quello** studente (저 학생) |
| l' → | **quell'** | **quell'**uomo (저 남자) |
| la → | **quella** | **quella** donna (저 여자) |
| l' → | **quell'** | **quell'**idea (저 아이디어) |
| i → | **quei** | **quei** ragazzi (저 소년들) |
| gli → | **quegli** | **quegli** studenti (저 학생들) |
| le → | **quelle** | **quelle** donne (저 여자들) |

> **QUESTO** (이/이것)는 더 간단합니다: questo/questa/questi/queste (규칙적 4형). 모음 앞: quest' (선택적).

## 6. 위치에 따른 의미 변화

일부 형용사는 명사 앞에 오는지 뒤에 오는지에 따라 의미가 달라집니다:

| 형용사 | 명사 앞 | 명사 뒤 |
|-----------|-------------|------------|
| **grande** | 위대한, 중요한: un **gran** uomo (위대한 사람) | 큰, 거대한: un uomo **grande** (큰 사람) |
| **povero** | 불쌍한: il **povero** ragazzo (불쌍한 소년) | 가난한: un ragazzo **povero** (가난한 소년) |
| **vecchio** | 오랜: un **vecchio** amico (오랜 친구) | 나이든: un amico **vecchio** (나이 든 친구) |
| **nuovo** | 또 다른, 다른: una **nuova** macchina (새[다른] 차) | 새것인: una macchina **nuova** (새[신품] 차) |
| **unico** | 유일한: l'**unico** figlio (외아들) | 독특한: un figlio **unico** (독특한 아들) |
| **stesso** | 같은: lo **stesso** libro (같은 책) | 자체: il libro **stesso** (책 자체) |
| **certo** | 어떤/모종의: un **certo** signore (어떤 신사) | 확실한: una notizia **certa** (확실한 뉴스) |
| **diverso** | 여러/다양한: **diversi** libri (여러 책) | 다른: libri **diversi** (다른 책들) |
| **alto** | 고위의: un **alto** funzionario (고위 관료) | 키 큰: un uomo **alto** (키 큰 사람) |
| **semplice** | 단순한/단지: una **semplice** domanda (단지 질문일 뿐) | 간단한: una domanda **semplice** (간단한 질문) |

## 7. 비교급

### 7.1 ~보다 더 (più... di / più... che)

**Più... di** — 같은 특성에 대해 두 개의 다른 명사/대명사를 비교:
- Maria è **più alta di** Giovanni. (Maria는 Giovanni보다 키가 크다.)
- Roma è **più grande di** Firenze. (로마는 피렌체보다 크다.)

**Più... che** — 같은 명사의 두 가지 특성을 비교하거나, 전치사/부정사 앞에서:
- È **più intelligente che** bello. (그는 잘생긴 것보다 더 지적이다.)
- Ci sono **più** turisti a Roma **che** a Napoli. (나폴리보다 로마에 관광객이 더 많다.)
- È **più facile** parlare **che** scrivere. (쓰는 것보다 말하는 것이 더 쉽다.)

```
┌─────────────────────────────────────────────────────────────────┐
│   DI 아니면 CHE?                                                 │
│                                                                 │
│   PIÙ...DI = 두 개의 명사/대명사 비교 (같은 특성)                    │
│   Maria è più alta DI Giovanni.                                │
│   Questo è più caro DI quello.                                 │
│                                                                 │
│   PIÙ...CHE = 두 가지 특성 비교 (같은 주어)                        │
│             = 전치사, 부정사, 부사 앞                               │
│   È più bello CHE intelligente.                                │
│   Studia più a casa CHE in biblioteca.                         │
│   È più facile leggere CHE scrivere.                           │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 ~보다 덜 (meno... di / meno... che)

- Giovanni è **meno alto di** Maria. (Giovanni는 Maria보다 덜 크다.)
- Questo ristorante è **meno caro di** quello. (이 레스토랑은 저것보다 덜 비싸다.)

### 7.3 ~만큼 (tanto... quanto / così... come)

- Pietro è **(tanto) intelligente quanto** Anna. (Pietro는 Anna만큼 지적이다.)
- La mia casa è **(così) grande come** la tua. (내 집은 네 집만큼 크다.)

> **참고**: *tanto*와 *così*는 종종 생략됩니다: Pietro è intelligente **quanto** Anna.

### 7.4 ~만큼 많은 (tanto/a/i/e... quanto)

*Tanto*는 수식하는 명사에 일치합니다:
- Ho **tanti** libri **quanti** (ne hai) tu. (나는 너만큼 많은 책을 가지고 있다.)
- Non ho **tanta** pazienza **quanta** (ne ha) lei. (나는 그녀만큼 인내심이 없다.)

## 8. 최상급

### 8.1 상대 최상급 (가장 ~한 / 가장 덜 ~한)

```
관사 + più/meno + 형용사 + di + 집단
```

- È **la più alta della** classe. (그녀는 반에서 가장 키가 크다.)
- È **il meno caro di** tutti. (그것은 모든 것 중 가장 덜 비싸다.)
- Sono **i più importanti del** mondo. (그들은 세계에서 가장 중요하다.)
- Roma è **la città più bella d'**Italia. (로마는 이탈리아에서 가장 아름다운 도시이다.)

### 8.2 절대 최상급 (-issimo/a/i/e)

비교 없이 극도의 정도를 표현합니다 ("매우, 매우"):

| 형용사 | 최상급 | 의미 |
|-----------|-------------|---------|
| alto | alt**issimo** | 극히 키가 큰 |
| bello | bell**issimo** | 극히 아름다운 |
| buono | buon**issimo** | 극히 좋은 |
| grande | grand**issimo** | 극히 큰 |
| ricco | ricch**issimo** | 극히 부유한 |
| facile | facil**issimo** | 극히 쉬운 |
| lungo | lungh**issimo** | 극히 긴 |

> **철자 변화**: ricco → ricch**issimo** (ch 이중), lungo → lungh**issimo** (h 추가), largo → largh**issimo**.

### 8.3 "매우"를 표현하는 다른 방법

```
molto + 형용사:        È molto bello. (그는 매우 잘생겼다.)
tanto + 형용사:        È tanto gentile. (그는 매우 친절하다.)
assai + 형용사:        È assai difficile. (그것은 꽤 어렵다.)
estremamente:          È estremamente importante. (극히 중요한)
incredibilmente:       È incredibilmente bello. (믿을 수 없이 아름다운)

접두사 arci-/stra-/ultra-:
  arcistufo (진저리가 나는)
  straricco (엄청 부유한)
  ultramoderno (초현대적인)
```

## 9. 불규칙 비교급과 최상급

| 형용사 | 비교급 | 상대 최상급 | 절대 최상급 |
|-----------|-------------|---------------------|---------------------|
| buono (좋은) | **migliore** (더 좋은) | **il/la migliore** (가장 좋은) | **ottimo** (훌륭한) |
| cattivo (나쁜) | **peggiore** (더 나쁜) | **il/la peggiore** (가장 나쁜) | **pessimo** (최악의) |
| grande (큰) | **maggiore** (더 큰/나이 많은) | **il/la maggiore** (가장 큰) | **massimo** (최대의) |
| piccolo (작은) | **minore** (더 작은/어린) | **il/la minore** (가장 작은) | **minimo** (최소의) |
| alto (높은) | **superiore** (더 높은) | **il/la superiore** (가장 높은) | **supremo/sommo** |
| basso (낮은) | **inferiore** (더 낮은) | **il/la inferiore** (가장 낮은) | **infimo** |

### 사용법 참고

```
MIGLIORE / PEGGIORE (더 좋은 / 더 나쁜):
  Questo vino è migliore di quello. (이 와인이 저것보다 좋다.)
  Il tempo è peggiore di ieri. (날씨가 어제보다 나쁘다.)
  ※ 규칙형도 존재합니다: più buono, più cattivo
     (음식/맛에서 더 흔함: Questa pizza è più buona.)

MAGGIORE / MINORE (형/누나 / 동생 — 형제/나이 비교):
  Mio fratello maggiore (형/오빠)
  La sorella minore (여동생)
  ※ 물리적 크기에는 più grande / più piccolo 사용:
     Questa casa è più grande. (이 집이 더 크다.)
```

## 11. 연습 문제

### 연습 1: 성과 수 일치

형용사를 명사에 일치시키세요.

1. Le case ________ (bianco)
2. Un uomo ________ (italiano)
3. Una donna ________ (francese)
4. I problemi ________ (difficile)
5. Le ragazze ________ (intelligente)
6. Un giorno ________ (felice)
7. Dei fiori ________ (bello)

<details>
<summary>정답 보기</summary>

1. Le case **bianche**
2. Un uomo **italiano**
3. Una donna **francese**
4. I problemi **difficili**
5. Le ragazze **intelligenti**
6. Un giorno **felice**
7. Dei fiori **belli** (또는: dei **bei** fiori)

</details>

### 연습 2: BELLO와 QUELLO 형태

올바른 BELLO 또는 QUELLO 형태를 넣으세요.

1. Guarda ________ tramonto! (bello)
2. ________ studentessa è brava. (quello)
3. Ho visto dei ________ occhi. (bello)
4. ________ zaino è mio. (quello)
5. Che ________ idea! (bello)
6. ________ uomini sono italiani. (quello)

<details>
<summary>정답 보기</summary>

1. Guarda **quel bel** tramonto! (il → quel/bel)
2. **Quella** studentessa è brava.
3. Ho visto dei **begli** occhi. (gli → begli)
4. **Quello** zaino è mio. (lo → quello)
5. Che **bell'**idea! (l' → bell')
6. **Quegli** uomini sono italiani. (gli → quegli)

</details>

### 연습 3: 비교급 — DI 아니면 CHE?

**di** 또는 **che**를 넣으세요.

1. Maria è più alta ________ Giovanni.
2. È più intelligente ________ bello.
3. Roma è più grande ________ Firenze.
4. È più facile parlare ________ scrivere.
5. Ci sono più studenti a Milano ________ a Venezia.
6. Questo è più caro ________ quello.

<details>
<summary>정답 보기</summary>

1. **di** (두 명사 비교 — 같은 특성)
2. **che** (두 특성 비교 — 같은 주어)
3. **di** (두 명사 비교)
4. **che** (부정사 앞)
5. **che** (전치사 앞)
6. **di** (두 명사 비교)

</details>

### 연습 4: 최상급과 불규칙형

이탈리아어로 번역하세요.

1. 그녀는 반에서 가장 키가 크다.
2. 이것은 도시에서 가장 좋은 레스토랑이다.
3. 이 커피는 극도로 뜨겁다. (-issimo 사용)
4. 형(오빠)은 로마에 산다.
5. 날씨가 어제보다 나쁘다.

<details>
<summary>정답 보기</summary>

1. (Lei) è **la più alta della** classe.
2. Questo è **il miglior** ristorante **della** città. (또는: il ristorante migliore della città)
3. Questo caffè è **caldissimo**.
4. Mio fratello **maggiore** vive a Roma.
5. Il tempo è **peggiore di** ieri. (또는: Il tempo è peggio di ieri.)

</details>

### 연습 5: 위치에 따른 의미 변화

각 쌍을 번역하되 형용사 위치에 주의하세요.

1. a) 위대한 여성 → ________
   b) 큰 여성 → ________
2. a) 불쌍한 소년 → ________
   b) 가난한 소년 → ________
3. a) 오랜 친구 → ________
   b) 나이 든 친구 → ________

<details>
<summary>정답 보기</summary>

1. a) Una **gran** donna (위대한) / b) Una donna **grande** (큰)
2. a) Il **povero** ragazzo (불쌍한) / b) Un ragazzo **povero** (가난한)
3. a) Un **vecchio** amico (오랜) / b) Un amico **vecchio** (나이 든)

</details>
