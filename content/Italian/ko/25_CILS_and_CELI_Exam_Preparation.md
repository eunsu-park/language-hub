## 1. 이탈리아어 능력 시험 개요

```
+----------------------------------------------------------+
|     이탈리아어 능력 시험                                     |
+----------------------------------------------------------+
|                                                          |
|  국제적으로 인정되는 세 가지 주요 시험:                       |
|                                                          |
|  1. CILS — Certificazione di Italiano come              |
|            Lingua Straniera                              |
|     시행 기관: Università per Stranieri di Siena          |
|                                                          |
|  2. CELI — Certificato di Conoscenza della              |
|            Lingua Italiana                               |
|     시행 기관: Università per Stranieri di Perugia        |
|                                                          |
|  3. PLIDA — Progetto Lingua Italiana Dante Alighieri    |
|     시행 기관: Società Dante Alighieri                    |
|                                                          |

```

### CEFR 레벨 설명

| CEFR 레벨 | 설명 | 이탈리아 시험 동급 | 할 수 있는 것 |
|-----------|-------------|----------------------|-----------------|
| **A1** | 돌파 / 입문 | CILS A1 / CELI Impatto / PLIDA A1 | 기본 인사, 간단한 질문, 생존 표현 |
| **A2** | 기초 / 초급 | CILS A2 / CELI 1 / PLIDA A2 | 일상 생활, 간단한 대화, 기초 작문 |
| **B1** | 문턱 / 중급 | CILS Uno-B1 / CELI 2 / PLIDA B1 | 여행, 의견 표현, 경험 서술, 시민권 요건 |
| **B2** | 우위 / 중상급 | CILS Due-B2 / CELI 3 / PLIDA B2 | 복잡한 토론, 직업적 맥락, 대학 등록 |
| **C1** | 효과적 능숙 / 고급 | CILS Tre-C1 / CELI 4 / PLIDA C1 | 유창한 표현, 학술/직업 담화, 미묘한 이해 |
| **C2** | 숙달 / 능숙 | CILS Quattro-C2 / CELI 5 / PLIDA C2 | 원어민 수준 능력, 복잡한 텍스트, 세련된 표현 |

## 2. CILS 시험 (시에나 외국인 대학)

### 2.1 개요

```
+----------------------------------------------------------+
|  CILS — Certificazione di Italiano come Lingua Straniera |
+----------------------------------------------------------+
|                                                          |
|  시행 기관: Università per Stranieri di Siena             |
|  인정: 이탈리아 교육부 (MIUR)                              |
|  유효기간: 평생 (만료 없음)                                 |
|  레벨: A1, A2, B1 (Uno), B2 (Due), C1 (Tre),            |
|        C2 (Quattro)                                      |
|  특별 버전: CILS A1/A2 Integrazione                       |
|    (이탈리아 체류 허가를 위한 이민자용)                       |
|                                                          |
|  시험 회차: 6월과 12월                                     |
|  (일부 레벨은 4월과 10월에도 가능)                           |
|                                                          |
|  섹션 (5개 섹션):                                          |
|  1. Ascolto (듣기)                                        |
|  2. Comprensione della lettura (읽기)                     |
|  3. Analisi delle strutture di comunicazione (문법)       |
|  4. Produzione scritta (쓰기)                             |
|  5. Produzione orale (말하기)                              |
|                                                          |
|  합격: 섹션별 최소 11/20점 (55%)                           |
|  불합격 섹션만 재시험 가능.                                  |
+----------------------------------------------------------+
```

### 2.2 CILS 시험 시간 (레벨별)

| 레벨 | 듣기 | 읽기 | 문법 | 쓰기 | 말하기 |
|-------|-----------|---------|---------|---------|---------|
| A1 | 20분 | 30분 | 30분 | 30분 | 10분 |
| A2 | 20분 | 30분 | 30분 | 30분 | 10분 |
| B1 (Uno) | 30분 | 40분 | 40분 | 60분 | 10분 |
| B2 (Due) | 30분 | 50분 | 45분 | 90분 | 10분 |
| C1 (Tre) | 35분 | 60분 | 45분 | 90분 | 15분 |
| C2 (Quattro) | 35분 | 60분 | 45분 | 120분 | 15분 |

### 2.3 CILS 섹션 세부 사항

| 섹션 | 문제 유형 |
|---------|-----------|
| **Ascolto** | 오디오 녹음(대화, 독백, 안내방송) 기반 객관식, 참/거짓, 빈칸 채우기 |
| **Comprensione della lettura** | 문서(기사, 광고, 이메일, 문학) 기반 객관식, 연결하기, 참/거짓 |
| **Analisi delle strutture** | 빈칸 채우기, 문장 변환, 동사 활용, 단어 형성 |
| **Produzione scritta** | B1: 짧은 이메일/편지 + 설명; B2: 에세이/기사 + 격식 편지; C1-C2: 복잡한 텍스트 |
| **Produzione orale** | B1: 자기 소개, 역할극, 사진 설명; B2-C2: 토론, 논증, 상호작용 |

## 3. CELI 시험 (페루자 외국인 대학)

### 3.1 개요

```
+----------------------------------------------------------+
|  CELI — Certificato di Conoscenza della Lingua Italiana  |
+----------------------------------------------------------+
|                                                          |
|  시행 기관: Università per Stranieri di Perugia           |
|  인정: 이탈리아 교육부 (MIUR)                              |
|  유효기간: 평생 (만료 없음)                                 |
|  레벨:                                                    |
|    CELI Impatto (A1)                                     |
|    CELI 1 (A2)                                           |
|    CELI 2 (B1)                                           |
|    CELI 3 (B2)                                           |
|    CELI 4 (C1)                                           |
|    CELI 5 (C2)                                           |
|                                                          |
|  시험 회차: 6월과 11월                                     |
|  (CELI Impatto와 CELI 1은 3월에도 가능)                    |
|                                                          |
|  섹션 (4개 섹션):                                          |
|  섹션 A: 읽기 이해 + 문법                                  |
|  섹션 B: 작문                                             |
|  섹션 C: 듣기 이해                                        |
|  섹션 D: 구술                                             |
|                                                          |
|  합격: 각 섹션 별도 채점.                                   |
|  최소 점수로 각 섹션 합격.                                  |
|  불합격 섹션은 개별 재시험 가능.                              |
+----------------------------------------------------------+
```

### 3.2 CELI 시험 시간 (레벨별)

| 레벨 | 읽기 + 문법 | 쓰기 | 듣기 | 말하기 |
|-------|------------------|---------|-----------|---------|
| CELI Impatto (A1) | 60분 | 30분 | 15분 | 10분 |
| CELI 1 (A2) | 90분 | 40분 | 15분 | 10분 |
| CELI 2 (B1) | 120분 | 60분 | 20분 | 15분 |
| CELI 3 (B2) | 135분 | 90분 | 25분 | 15분 |
| CELI 4 (C1) | 150분 | 120분 | 30분 | 15분 |
| CELI 5 (C2) | 180분 | 150분 | 30분 | 20분 |

### 3.3 CILS와의 주요 차이점

| 특징 | CILS | CELI |
|---------|------|------|
| 문법 섹션 | 별도 섹션 (Analisi delle strutture) | 읽기와 통합 (섹션 A) |
| 섹션 수 | 5 | 4 |
| 시행 기관 | 시에나 | 페루자 |
| 특별 버전 | CILS Integrazione (이민자용) | CELI i (이민자용) |

### 6.2 B1 레벨의 의미

B1 레벨에서 할 수 있어야 하는 것:

| 기술 | B1 설명 |
|-------|--------------|
| **듣기** | 익숙한 주제(직장, 학교, 여가)에 대한 표준 발화의 요점 이해 |
| **읽기** | 일상 주제와 개인적 관심사에 관한 텍스트 이해 |
| **쓰기** | 익숙한 주제와 개인 경험에 대한 간단한 연결 텍스트 작성 |
| **말하기** | 여행 중 대부분의 상황 처리, 경험 서술, 의견에 대한 이유 제시 |
| **문법** | 현재, 과거, 미래 시제 올바르게 사용; 조건법; 기초 접속법; 관계절 |
| **어휘** | 일상 주제를 다루는 ~3,000-4,000 단어 |

## 7. 듣기 섹션 전략

### 7.1 일반 전략

| 전략 | 설명 |
|----------|-------------|
| **먼저 질문 읽기** | 오디오 재생 전에 모든 질문을 읽어 무엇을 들어야 할지 파악 |
| **첫 번째 청취: 전체 의미** | 모든 단어를 이해하려 하지 말고 전체 메시지에 집중 |
| **두 번째 청취: 세부 사항** | 각 질문에 답하는 데 필요한 정보에 집중 |
| **핵심 단어** | 질문의 핵심 단어를 파악하고 그것 또는 동의어를 청취 |
| **방해 요소** | 오답 선택지에 종종 오디오의 단어가 포함됨에 주의 |
| **당황하지 않기** | 한 답을 놓쳤다고 다음 문제 집중에 영향을 주지 말 것 |

### 7.2 레벨별 일반적인 오디오 유형

| 레벨 | 오디오 유형 |
|-------|-----------|
| **A1-A2** | 짧은 대화, 안내방송, 전화 메시지, 간단한 설명 |
| **B1** | 라디오 프로그램, 대화, 뉴스 요약, 인터뷰 |
| **B2** | 강의, 토론, 장시간 인터뷰, 뉴스 보도 |
| **C1-C2** | 학술 강의, 복잡한 토론, 문학 낭독, 전문 내용 |

### 7.3 듣기 연습 팁

| 팁 | 연습 방법 |
|-----|----------------|
| RAI 라디오/TV | RAI Radio 1을 듣거나 RAI News를 시청하여 실제 이탈리아어 청취 |
| 팟캐스트 | 학습자를 위한 "Coffee Break Italian", "News in Slow Italian" |
| 속도 연습 | 느리고 명확한 발화부터 시작, 점차 자연스러운 속도로 증가 |
| 자막 | 이탈리아어 자막으로 이탈리아 영화 시청 (영어 아님!) |
| 받아쓰기 | 들은 것을 받아 적고 원문과 비교 |

## 8. 읽기 섹션 전략

### 8.1 일반 전략

| 전략 | 설명 |
|----------|-------------|
| **먼저 훑어보기** | 전체 텍스트를 빠르게 읽어 대의 파악 |
| **질문 읽기** | 세부 읽기 전에 필요한 정보가 무엇인지 파악 |
| **핵심 단어** | 질문과 텍스트에서 핵심 단어에 밑줄이나 동그라미 |
| **소거법** | 객관식에서 명백히 틀린 답부터 제거 |
| **번역하지 않기** | 이탈리아어로 생각; 단어별로 모국어로 번역 금지 |
| **문맥 단서** | 주변 단어를 사용하여 모르는 어휘 추측 |
| **시간 관리** | 한 문제에 너무 오래 머물지 말고, 넘기고 나중에 돌아오기 |

### 8.2 일반적인 텍스트 유형

| 레벨 | 텍스트 유형 |
|-------|-----------|
| **A1-A2** | 표지판, 메뉴, 간단한 이메일, 짧은 광고, 양식 작성 |
| **B1** | 신문 기사, 비격식 편지, 브로슈어, 블로그 글 |
| **B2** | 복잡한 기사, 격식 편지, 문학 발췌, 에세이 |
| **C1-C2** | 학술 논문, 문학 텍스트, 법률 문서, 전문 기사 |

> **Vivere in Italia: il fascino delle piccole città**
>

Secondo un recente sondaggio, il 45% degli expatriati in Italia preferisce vivere in centri con meno di 50.000 abitanti.
>

   - a) è andato  b) sia andato  c) andava  d) andrà

3. Sebbene ________ molto, non ha superato l'esame.
   - a) ha studiato  b) studia  c) abbia studiato  d) studierà

4. Mi hanno detto che la riunione ________ rinviata.
   - a) è  b) è stata  c) sia  d) fosse stata

5. Non conosco nessuno che ________ il cinese e l'arabo.
   - a) parla  b) parli  c) parlando  d) parlare

6. Prima di ________, controlla i documenti.
   - a) parti  b) partire  c) partendo  d) parta

<details>
<summary>정답</summary>

1. **b) parlassi** -- *Se* + 접속법 반과거, 가정 조건문 (제2유형 조건문)
2. **b) sia andato** -- *Credo che*는 접속법 유발; 과거 사건 → *접속법 과거*
3. **c) abbia studiato** -- *Sebbene*은 항상 접속법 유발; 완료된 행위 → 접속법 과거
4. **b) è stata** -- *hanno detto che* + 직설법으로 전달된 보고 (사실 보고)
5. **b) parli** -- *Nessuno che* + 접속법 (불확실성/비존재 표현)
6. **b) partire** -- *Prima di* + 부정사 (무엇을 하기 전에)

</details>

### 연습 5: 전략 적용

각 시나리오에 대해 가장 적합한 시험 전략을 선택하세요.

1. 듣기 섹션 중 3번 문제의 답을 놓침.
   - a) 오디오 되감기  b) 다음 문제에 계속 집중  c) 시험장 나가기

2. 읽기 섹션에서 모르는 단어를 만남.
   - a) 사전 찾기  b) 텍스트 전체 건너뛰기  c) 문맥 단서를 사용하여 의미 추측

3. 쓰기 섹션에서 10분 일찍 마침.
   - a) 즉시 나가기  b) 문법, 철자, 악센트 오류 검토  c) 페이지를 채우기 위해 텍스트 추가

4. 말하기 섹션에서 시험관의 질문을 이해하지 못함.
   - a) 추측하고 아무거나 답하기  b) 침묵하기  c) "Potrebbe ripetere, per favore?"라고 묻기

5. 시험 3개월 전이고 현재 A2 수준.
   - a) C2에 등록  b) B1에 등록하고 집중 학습  c) A1에 등록

<details>
<summary>정답</summary>

1. **b) 다음 문제에 계속 집중** -- 한 답을 놓쳤다고 나머지 집중을 망치지 마세요.
2. **c) 문맥 단서를 사용하여 의미 추측** -- 사전 사용 불가; 문맥 단서가 최선의 도구입니다.
3. **b) 문법, 철자, 악센트 오류 검토** -- 항상 여분의 시간을 교정에 사용하세요!
4. **c) "Potrebbe ripetere, per favore?"라고 묻기** -- 설명 요청은 완전히 허용되며 의사소통 능력을 보여줍니다.
5. **b) B1에 등록하고 집중 학습** -- 현재 수준보다 한 단계 위의 현실적 목표가 최선입니다.

</details>
