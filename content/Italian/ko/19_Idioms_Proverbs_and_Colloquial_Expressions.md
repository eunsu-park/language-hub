## 1. 신체 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|----------------|---------------|
| In bocca al lupo! | 늑대의 입 안으로! | 행운을 빌어! |
| Costare un occhio della testa | 머리에서 눈 하나의 값 | 엄청나게 비싸다 |
| Non avere peli sulla lingua | 혀에 털이 없다 | 직설적으로 말하다, 솔직하다 |
| Avere le mani in pasta | 반죽에 손이 들어가 있다 | 관여하다 / 모든 일에 끼어들다 |
| Avere le mani bucate | 구멍 뚫린 손을 가지다 | 돈을 물 쓰듯 하다 |
| Stare con le mani in mano | 손에 손을 얹고 서 있다 | 아무것도 하지 않다, 빈둥거리다 |
| Dare una mano | 손을 주다 | 도와주다 |
| Non vedere l'ora | 시간을 보지 못하다 | 간절히 기다리다 |
| Avere il sangue freddo | 차가운 피를 가지다 | 압박 속에서도 침착하다 |
| Perdere la testa | 머리를 잃다 | 미치다 / 열렬히 사랑에 빠지다 |
| Rompere le scatole | 상자를 부수다 | 귀찮게 하다, 성가시게 하다 |
| Essere in gamba | 다리 안에 있다 | 똑똑하다, 유능하다 |
| Avere fegato | 간을 가지다 | 배짱이 있다, 용기가 있다 |
| Fare orecchie da mercante | 상인의 귀를 만들다 | 못 들은 척하다 |
| Restare a bocca aperta | 입이 벌어진 채로 남다 | 놀라다 |

### 예문

- **In bocca al lupo** per l'esame di domani! -- **Crepi il lupo!** (내일 시험 행운을 빌어! -- 고마워! [직역: 늑대가 죽기를!])
- Quel ristorante **costa un occhio della testa**. (그 식당은 엄청나게 비싸다.)
- Marco **non ha peli sulla lingua**: dice sempre quello che pensa. (Marco는 직설적이다: 항상 생각하는 대로 말한다.)
- Lei **ha le mani in pasta** in tutti i progetti dell'ufficio. (그녀는 사무실의 모든 프로젝트에 관여하고 있다.)
- **Non vedo l'ora** di andare in vacanza! (휴가 가는 게 너무 기다려져!)
- Smettila di **rompermi le scatole**! (그만 좀 귀찮게 해!)

> **문화 참고**: 누군가 *In bocca al lupo!*라고 말하면, 전통적인 대답은 *Crepi il lupo!* (늑대가 죽기를!) 또는 간단히 *Crepi!*이다. *Grazie*로 대답하면 미신을 믿는 이탈리아인들에게 불길하게 여겨진다.

## 2. 음식 관련 관용구

이탈리아 문화는 음식과 깊이 연결되어 있으며, 이는 언어에도 반영된다:

| 관용구 | 직역 | 실제 의미 |
|-------|----------------|---------------|
| Rendere pan per focaccia | 빵으로 포카치아를 갚다 | 보복하다, 앙갚음하다 |
| Essere al verde | 초록에 있다 | 무일푼이다 |
| Avere il prosciutto sugli occhi | 눈 위에 프로슈토가 있다 | 명백한 것을 보지 못하다 |
| Non è farina del tuo sacco | 네 자루의 밀가루가 아니다 | 네 자신의 작품/아이디어가 아니다 |
| Essere come il prezzemolo | 파슬리 같다 | 어디에나 있다, 항상 나타나다 |
| Buono come il pane | 빵처럼 좋은 | 매우 친절하고 선량한 |
| Mangiare la foglia | 잎을 먹다 | 속임수를 간파하다 |
| Avere le mani in pasta | 반죽에 손이 들어가 있다 | 깊이 관여하다 |
| Fare polpette di qualcuno | 누군가를 미트볼로 만들다 | 누군가를 심하게 이기다 |
| Se non è zuppa è pan bagnato | 수프가 아니면 젖은 빵이다 | 거기서 거기다, 오십보백보 |
| La ciliegina sulla torta | 케이크 위의 체리 | 금상첨화 |
| Dire pane al pane e vino al vino | 빵을 빵이라, 포도주를 포도주라 부르다 | 있는 그대로 말하다 |

### 예문

- Sono **al verde** questo mese; non posso uscire. (이번 달에 무일푼이야; 외출할 수 없어.)
- Hai **il prosciutto sugli occhi**! Non vedi che ti sta mentendo? (눈에 프로슈토를 붙이고 있구나! 그가 거짓말하는 게 안 보여?)
- Quella ragazza è **come il prezzemolo**: la trovi dappertutto! (그 여자는 파슬리 같아: 어디서든 마주쳐!)
- Il tuo vicino è **buono come il pane**. (네 이웃은 빵처럼 좋아 -- 매우 친절하다.)

## 3. 동물 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|----------------|---------------|
| In bocca al lupo | 늑대의 입 안으로 | 행운을 빌어 |
| Prendere due piccioni con una fava | 잠두콩 하나로 비둘기 두 마리 잡기 | 일석이조 |
| Essere una volpe | 여우이다 | 교활하다, 약삭빠르다 |
| Avere una memoria da elefante | 코끼리의 기억력을 가지다 | 뛰어난 기억력을 가지다 |
| Essere un pesce fuor d'acqua | 물 밖의 물고기이다 | 어색하게 느끼다 |
| Chi dorme non piglia pesci | 자는 자는 물고기를 잡지 못한다 | 부지런해야 기회를 잡는다 |
| Essere un coniglio | 토끼이다 | 겁쟁이이다 |
| Fare la gatta morta | 죽은 고양이 흉내를 내다 | 순진한 척하다 |
| A caval donato non si guarda in bocca | 선물 받은 말의 입을 들여다보지 마라 | 선물을 비판하지 마라 |
| Ogni scarrafone è bello a mamma soja | 모든 바퀴벌레는 자기 엄마에게 아름답다 | 어머니는 항상 자식에게서 아름다움을 본다 |
| Quando il gatto non c'è, i topi ballano | 고양이가 없으면 쥐가 춤춘다 | 윗사람이 없으면 아랫사람이 방종한다 |
| Menare il can per l'aia | 타작마당에서 개를 끌고 다니다 | 빙빙 돌려 말하다 |

### 예문

- Con quel trucco abbiamo **preso due piccioni con una fava**. (그 방법으로 일석이조를 거두었다.)
- In quella situazione mi sentivo **un pesce fuor d'acqua**. (그 상황에서 물 밖의 물고기 같았다.)
- Non **menare il can per l'aia** -- dimmi la verità! (빙빙 돌리지 말고 -- 진실을 말해!)
- **A caval donato non si guarda in bocca!** (선물 받은 말의 입은 들여다보지 마라!)

## 4. 색상 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|----------------|---------------|
| Vedere rosso | 빨간색을 보다 | 격노하다 |
| Essere al verde | 초록에 있다 | 무일푼이다 |
| Passare la notte in bianco | 밤을 흰색 속에서 보내다 | 밤새 잠을 못 자다 |
| Essere nero | 검다 | 기분이 매우 나쁘다 |
| Avere una fifa blu | 파란 공포를 가지다 | 매우 두려워하다 |
| Vedere tutto rosa | 모든 것을 분홍색으로 보다 | 모든 것을 낙관적으로 보다 |
| Mettere nero su bianco | 흰색 위에 검은색을 놓다 | 글로 남기다, 문서화하다 |
| Cronaca rosa | 분홍 기사 | 연예 / 가십 뉴스 |
| Cronaca nera | 검은 기사 | 범죄 뉴스 |
| Diventare rosso come un peperone | 고추처럼 빨갛게 되다 | 심하게 얼굴이 붉어지다 |
| Principe azzurro | 파란 왕자 | 백마 탄 왕자 |

### 예문

- Quando ho visto il danno alla macchina, ho **visto rosso**. (차에 난 손상을 봤을 때 격노했다.)
- Ho **passato la notte in bianco** a causa dell'esame. (시험 때문에 밤새 잠을 못 잤다.)
- **Mettiamolo nero su bianco** così non ci sono malintesi. (오해가 없도록 글로 남기자.)
- È **diventata rossa come un peperone** quando l'hanno chiamata. (그녀의 이름이 불렸을 때 얼굴이 새빨갛게 됐다.)

## 5. 돈과 일 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|----------------|---------------|
| Fare il portoghese | 포르투갈인 행세를 하다 | 돈을 안 내고 몰래 들어가다 |
| Non avere il becco di un quattrino | 동전의 부리도 없다 | 완전히 무일푼이다 |
| Costare un patrimonio | 재산만큼 비싸다 | 매우 비싸다 |
| Guadagnarsi il pane | 자신의 빵을 벌다 | 생계를 꾸리다 |
| Battere la fiacca | 게으름을 두드리다 | 빈둥거리다, 게으름 피우다 |
| Fare un buco nell'acqua | 물에 구멍을 내다 | 실패하다 |
| Tirare la cinghia | 허리띠를 당기다 | 허리띠를 졸라매다, 절약하다 |
| Avere l'acqua alla gola | 목까지 물이 차다 | 심각한 곤경에 처하다 (특히 경제적으로) |

### 예문

- Hanno provato ad entrare al cinema **facendo i portoghesi**. (그들은 돈을 안 내고 영화관에 들어가려 했다.)
- Dopo il licenziamento, avevano **l'acqua alla gola**. (해고 후 그들은 심각한 경제적 곤경에 처했다.)
- Dobbiamo **tirare la cinghia** fino alla fine del mese. (월말까지 허리띠를 졸라매야 한다.)

> **문화 참고**: *Fare il portoghese*는 18세기 로마에서 포르투갈 대사가 로마에 거주하는 모든 포르투갈인을 무료 행사에 초대했는데, 많은 비포르투갈인들이 무료로 입장하기 위해 포르투갈인인 척했다는 이야기에서 유래했다고 한다.

## 6. 날씨와 자연 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|----------------|---------------|
| Piove sul bagnato | 젖은 곳에 비가 온다 | 엎친 데 덮친 격 (불운이 겹치다) |
| Fare il bello e il cattivo tempo | 좋은 날씨와 나쁜 날씨를 만들다 | 모든 것을 좌지우지하다 |
| Una tempesta in un bicchier d'acqua | 물 한 잔 속의 폭풍 | 사소한 일에 호들갑 떨기 |
| Avere la testa fra le nuvole | 구름 사이에 머리가 있다 | 멍하니 공상에 잠기다 |
| Sentirsi un pesce fuor d'acqua | 물 밖의 물고기 같이 느끼다 | 어색하게 느끼다 |
| Dopo la pioggia viene il sereno | 비 뒤에 맑은 하늘이 온다 | 고진감래 |

### 예문

- Prima ha perso il lavoro, poi si è rotto il braccio... **piove sul bagnato**! (먼저 직장을 잃고, 팔까지 부러졌어... 엎친 데 덮친 격이야!)
- Non fare **una tempesta in un bicchier d'acqua** -- è un problema piccolo. (사소한 일에 호들갑 떨지 마 -- 작은 문제야.)
- Marco **ha sempre la testa fra le nuvole**: non ascolta mai. (Marco는 항상 멍하니 공상에 잠겨 있다: 절대 듣지를 않아.)

## 7. 관계와 감정 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|----------------|---------------|
| Avere un colpo di fulmine | 벼락을 맞다 | 첫눈에 반하다 |
| Prendere in giro | 빙빙 돌리다 | 놀리다 |
| Andare d'accordo | 합의 속에 가다 | 사이가 좋다 |
| Non poterne più | 더 이상 할 수 없다 | 지겹다, 참을 수 없다 |
| Andare a genio | 천재에게 가다 | 마음에 들다, 좋아하다 |
| Essere cotto/a di qualcuno | 누군가에게 익다 | 누군가에게 푹 빠지다 |
| Fare le corna a qualcuno | 누군가에게 뿔을 만들다 | 바람을 피우다 |
| Avere un diavolo per capello | 머리카락마다 악마가 있다 | 극도로 화가 나다 |
| Legarsela al dito | 손가락에 묶다 | 원한을 품다 |

### 예문

- Quando l'ho vista per la prima volta, è stato **un colpo di fulmine**. (그녀를 처음 봤을 때, 첫눈에 반했다.)
- Non **prendermi in giro**! (나를 놀리지 마!)
- Io e la mia collega **andiamo molto d'accordo**. (나와 동료는 사이가 매우 좋다.)
- **Non ne posso più** di questo caldo! (이 더위를 더 이상 참을 수 없어!)
- Giorgio è **cotto** di Maria. (Giorgio는 Maria에게 푹 빠져 있다.)

## 8. 흔한 이탈리아 속담

### 8.1 삶과 지혜에 관한 속담

| 이탈리아어 | 직역 | 한국어 대응 표현 |
|---------|--------------------|--------------------|
| Chi dorme non piglia pesci. | 자는 자는 물고기를 잡지 못한다. | 부지런해야 기회를 잡는다. |
| Tra il dire e il fare c'è di mezzo il mare. | 말과 행동 사이에 바다가 있다. | 말이야 쉽지. |
| L'appetito vien mangiando. | 식욕은 먹으면서 온다. | 가지면 가질수록 더 원한다. |
| Chi la fa l'aspetti. | 행한 자는 기대하라. | 뿌린 대로 거둔다. |
| Chi va piano va sano e va lontano. | 천천히 가는 자가 건강하고 멀리 간다. | 급할수록 돌아가라. |
| Sbagliando s'impara. | 실수하면서 배운다. | 실패는 성공의 어머니. |
| Meglio tardi che mai. | 안 하는 것보다 늦는 게 낫다. | 늦더라도 안 하는 것보다 낫다. |
| Il mattino ha l'oro in bocca. | 아침은 입에 금을 물고 있다. | 일찍 일어나는 새가 벌레를 잡는다. |
| Non tutto il male vien per nuocere. | 모든 나쁜 것이 해치려고 오지는 않는다. | 전화위복. |

### 8.2 관계에 관한 속담

| 이탈리아어 | 직역 | 한국어 대응 표현 |
|---------|--------------------|--------------------|
| Chi trova un amico trova un tesoro. | 친구를 찾는 자는 보물을 찾는다. | 좋은 친구는 보물과 같다. |
| L'abito non fa il monaco. | 옷이 수도사를 만들지 않는다. | 겉모습으로 판단하지 마라. |
| Moglie e buoi dei paesi tuoi. | 아내와 소는 자기 마을에서. | 잘 아는 것을 고수하라 (가까운 곳에서 결혼하라). |
| Lontano dagli occhi, lontano dal cuore. | 눈에서 멀면 마음에서도 멀다. | 눈에서 멀어지면 마음에서도 멀어진다. |
| Dimmi con chi vai e ti dirò chi sei. | 누구와 다니는지 말해라, 네가 누군지 말해주겠다. | 친구를 보면 그 사람을 알 수 있다. |

### 8.3 돈과 일에 관한 속담

| 이탈리아어 | 직역 | 한국어 대응 표현 |
|---------|--------------------|--------------------|
| Chi più spende meno spende. | 더 쓰는 자가 덜 쓴다. | 싼 게 비지떡. |
| I soldi non fanno la felicità. | 돈이 행복을 만들지 않는다. | 돈으로 행복을 살 수 없다. |

## 10. 이탈리아 제스처와 비언어적 의사소통

이탈리아의 의사소통은 제스처가 매우 풍부한 것으로 유명하다. 이러한 제스처는 무작위가 아니라 모든 이탈리아인이 이해하는 구체적인 의미를 가지고 있다.

### 10.1 필수 이탈리아 제스처

```
+----------------------------------------------------------+
|          이탈리아 제스처 가이드                             |
+----------------------------------------------------------+
|                                                          |
|  1. "MA CHE VUOI?" (뭘 원하는 거야?)                     |
|     손가락 끝을 모아 위로 향하고,                          |
|     손을 위아래로 반복적으로 흔든다.                        |
|     → 가장 상징적인 이탈리아 제스처.                       |
|     → 의미: 혼란, 좌절, "뭐라고?!"                       |
|                                                          |
|  2. "PERFETTO" (완벽해)                                   |
|     엄지와 검지로 원을 만들고 (OK 사인),                   |
|     나머지 손가락을 편다.                                  |
|     → 의미: 맛있다, 훌륭하다, 완벽하다                    |
|                                                          |
|  3. "NON ME NE FREGA" (상관없어)                          |
|     손가락 뒷면으로 턱을 밖으로 튕긴다.                    |
|     → 의미: 상관없어 / 내 문제가 아냐                     |
|                                                          |
|  4. "FURBETTO" (약삭빠른 놈)                              |
|     검지로 아래 눈꺼풀을 당긴다.                           |
|     → 의미: 조심해 / 지켜보고 있어 / 알고 있어             |
|                                                          |
|  5. "PAURA" (두려움)                                      |
|     몸 앞에서 양손을 떤다.                                |
|     → 의미: 무서워 / 소름 끼쳐 / 와                       |
|                                                          |
|  6. "HO FAME" (배고파)                                    |
|     손바닥을 편 채 옆면으로 배를 반복적으로 친다.           |
|     → 의미: 배고프다                                      |
|                                                          |
|  7. "ANDIAMO!" (가자!)                                    |
|     손바닥을 위로 향하고 자신 쪽으로 손짓한다.              |
|     → 의미: 어서 / 가자                                   |
|                                                          |
|  8. "BASTA!" (그만!)                                      |
|     양손 바닥을 아래로 향해 밑으로 밀어낸다.                |
|     → 의미: 그만 / 됐어 / 진정해                          |
|                                                          |
|  9. "LE CORNA" (뿔)                                      |
|     검지와 새끼손가락을 펴고, 가운데와                     |
|     약지를 접는다 (록 뿔 모양).                            |
|     → 의미: 불운을 막기 (미신)                             |
|     → 또한: "네 파트너가 바람피우고 있어"                   |
|       (누군가에게 향할 때)                                 |
|                                                          |
|  10. "CHE BARBA!" (정말 지루해!)                           |
|      상상의 수염을 쓰다듬는 손동작.                        |
|      → 의미: 정말 지루해 / 귀찮아                          |
+----------------------------------------------------------+
```

### 10.2 제스처의 맥락별 사용

| 상황 | 제스처 | 언어적 동의어 |
|-----------|---------|-------------------|
| 누군가 혼란스러운 질문을 할 때 | 손가락 모으기 ("Che vuoi?") | Ma che dici? / Cosa vuoi? |
| 음식이 맛있을 때 | 입 근처에서 OK 사인 | Buonissimo! / Squisito! |
| 관심 없을 때 | 턱 튕기기 | Non me ne frega niente. |
| 누군가에게 조심하라고 경고할 때 | 아래 눈꺼풀 당기기 | Stai attento! / Occhio! |
| 서두르라고 할 때 | 손바닥 위로 향한 손짓 | Dai, sbrigati! / Muoviti! |
| 뭔가 미쳤다고 표현할 때 | 관자놀이에서 손가락 돌리기 | È matto/a! / È fuori di testa! |

### 10.3 신체적 거리와 접촉

| 문화적 규범 | 설명 |
|--------------|-------------|
| **인사 키스** | 친구와 가족 사이에 볼에 두 번 키스 (오른쪽 볼 먼저) |
| **개인 공간** | 북유럽 / 미국 기준보다 가까움 |
| **대화 중 접촉** | 팔, 어깨 -- 매우 흔하며 침범으로 간주되지 않음 |
| **눈 맞춤** | 직접적이고 지속적; 눈 맞춤을 피하면 회피적으로 보일 수 있음 |
| **악수** | 단단하게, 눈 맞춤과 함께; 직업적 환경에서 흔함 |

### 연습 5: 제스처 식별

1. 손가락 끝을 모두 위로 향하게 모아 손을 흔든다.
2. 상대를 보면서 손가락으로 아래 눈꺼풀을 당긴다.
3. 손가락 뒷면으로 턱을 밖으로 튕긴다.
4. 먹은 후 입 근처에서 OK 사인 (엄지와 검지로 원)을 만든다.
