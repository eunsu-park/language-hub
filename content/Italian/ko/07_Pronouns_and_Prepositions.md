## 1. 주격 대명사

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 | **io** (나) | **noi** (우리) |
| 2인칭 비격식 | **tu** (너) | **voi** (너희) |
| 2인칭 격식 | **Lei** (당신) | **Loro** (여러분 — 드묾, 격식) |
| 3인칭 남성 | **lui** (그) | **loro** (그들) |
| 3인칭 여성 | **lei** (그녀) | **loro** (그들) |

> **대명사 생략**: 이탈리아어는 대명사 생략 언어입니다 — 동사 어미가 주어를 나타내므로 주격 대명사는 보통 **생략**됩니다.
> - *Parlo italiano.* (이탈리아어를 한다.) — "io" 생략
> - **강조, 대조, 또는 명확성**을 위해 주격 대명사를 사용합니다:
>   - **Io** parlo italiano, ma **lui** parla francese.

## 2. 직접목적어 대명사

직접목적어 대명사는 동작을 직접 받는 **사물이나 사람**을 대체합니다.

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 | **mi** (나를) | **ci** (우리를) |
| 2인칭 비격식 | **ti** (너를) | **vi** (너희를) |
| 2인칭 격식 | **La** (당신을, 남/여) | **Li/Le** (여러분을, 남/여) |
| 3인칭 남성 | **lo** (그를/그것을) | **li** (그들을, 남) |
| 3인칭 여성 | **la** (그녀를/그것을) | **le** (그들을, 여) |

**예시:**
- Vedi **il libro**? → **Lo** vedi? (그것을 보니?)
- Ho comprato **le mele**. → **Le** ho comprate. (그것들을 샀다.)
- Conosco **Maria**. → **La** conosco. (그녀를 안다.)
- Aspetto **gli amici**. → **Li** aspetto. (그들을 기다린다.)

> **복합 시제에서의 일치**: lo/la/li/le가 근과거의 동사 앞에 올 때, 과거분사가 성과 수에 **일치**합니다:
> - Ho comprato **la torta**. → **L'**ho comprat**a**. (그것을 샀다.)
> - Ho visto **i ragazzi**. → **Li** ho vist**i**. (그들을 봤다.)

## 3. 간접목적어 대명사

간접목적어 대명사는 동작이 **누구에게/누구를 위해** 행해지는지를 나타냅니다.

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 | **mi** (나에게) | **ci** (우리에게) |
| 2인칭 비격식 | **ti** (너에게) | **vi** (너희에게) |
| 2인칭 격식 | **Le** (당신에게) | **Loro** (여러분에게 — 드묾) |
| 3인칭 남성 | **gli** (그에게) | **gli** (그들에게 — 남 & 여) |
| 3인칭 여성 | **le** (그녀에게) | **gli** (그들에게 — 남 & 여) |

> **참고**: 현대 이탈리아어에서 **gli**는 "그에게"와 "그들에게" (남 & 여) 모두의 간접목적어로 쓰입니다. 격식체 *loro* (동사 뒤에 위치)는 점점 드물어지고 있습니다.

**예시:**
- **Mi** ha dato un regalo. (그가 나에게 선물을 줬다.)
- **Gli** ho scritto una lettera. (그에게 편지를 썼다.)
- **Le** parlo domani. (그녀에게 내일 말할 것이다.)
- **Gli** abbiamo spiegato. (우리가 그들에게 설명했다.)

## 4. 대명사 위치 규칙

### 활용 동사 앞

대명사는 활용 동사 **앞에** 옵니다:
- **Lo** vedo. (그를 본다.)
- **Mi** chiama. (그가 나를 부른다.)
- Non **ti** conosco. (너를 모른다.)

### 부정사, 제룬디오, 긍정 명령에 부착

대명사는 부정사 (마지막 -e를 탈락), 제룬디오, 긍정 명령의 **뒤에** 부착됩니다:

| 구조 | 예시 |
|-----------|---------|
| 부정사 | Voglio **vederlo**. = **Lo** voglio vedere. (그것을 보고 싶다.) |
| 제룬디오 | Sto **facendolo**. = **Lo** sto facendo. (그것을 하고 있다.) |
| 긍정 명령 | **Dimmi**! (말해줘!) **Guardalo**! (그것을 봐!) |

### 부정 명령: 동사 앞

- Non **mi** lo dire! (나에게 말하지 마!)
- Non **lo** fare! (그것을 하지 마!)

### 조동사 (potere, volere, dovere)와 함께

두 가지 동일하게 올바른 방법:
- **Lo** posso fare. = Posso **farlo**. (그것을 할 수 있다.)
- **Ti** devo parlare. = Devo **parlarti**. (너에게 말해야 한다.)

## 5. 결합 (이중) 대명사

간접목적어와 직접목적어 대명사가 함께 나타날 때:

```
┌──────────────────────────────────────────────────────────────┐
│         이중 대명사 순서와 형태                                   │
│                                                              │
│  간접목적어 (IO)가 항상 직접목적어 (DO) 앞에 옵니다               │
│                                                              │
│  IO + DO:   lo      la      li      le      ne              │
│  ─────────────────────────────────────────────────           │
│  mi →       me lo   me la   me li   me le   me ne           │
│  ti →       te lo   te la   te li   te le   te ne           │
│  gli/le →   glielo  gliela  glieli  gliele  gliene          │
│  ci →       ce lo   ce la   ce li   ce le   ce ne           │
│  vi →       ve lo   ve la   ve li   ve le   ve ne           │
│  gli →      glielo  gliela  glieli  gliele  gliene          │
│  (그들에게)                                                    │
│                                                              │
│  주요 변화:                                                    │
│  • mi → me, ti → te, ci → ce, vi → ve (lo/la 등 앞)         │
│  • gli/le → glie- (직접목적어에 부착되어 한 단어로)               │
│  • gli (그들에게) → glie- (gli = 그에게와 같은 형태)             │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

**예시:**
- Mi dai il libro? → **Me lo** dai? (그것을 나에게 줄래?)
- Gli ho dato le chiavi. → **Gliele** ho date. (그에게 그것들을 줬다.)
- Ti compro una pizza. → **Te la** compro. (너에게 그것을 사줄게.)
- Ci manda i documenti. → **Ce li** manda. (그가 우리에게 그것들을 보낸다.)

## 6. 재귀 대명사

재귀 대명사는 주어가 **자기 자신에게** 동작을 수행함을 나타냅니다.

| 인칭 | 대명사 | 예시 |
|--------|---------|---------|
| io | **mi** | Mi sveglio. (일어난다.) |
| tu | **ti** | Ti lavi. (너는 씻는다.) |
| lui/lei/Lei | **si** | Si veste. (그/그녀는 옷을 입는다.) |
| noi | **ci** | Ci prepariamo. (우리는 준비한다.) |
| voi | **vi** | Vi divertite. (너희는 즐긴다.) |
| loro | **si** | Si alzano. (그들은 일어난다.) |

### 일반적인 재귀 동사

| 이탈리아어 | 한국어 | 이탈리아어 | 한국어 |
|---------|---------|---------|---------|
| svegliarsi | 잠에서 깨다 | pettinarsi | 머리를 빗다 |
| alzarsi | 일어나다 | vestirsi | 옷을 입다 |
| lavarsi | 씻다 | truccarsi | 화장하다 |
| farsi la doccia | 샤워하다 | mettersi | (옷을) 입다 |
| sedersi | 앉다 | sentirsi | 느끼다 |
| addormentarsi | 잠들다 | chiamarsi | ~라고 불리다 (이름) |
| divertirsi | 즐기다 | arrabbiarsi | 화내다 |
| preoccuparsi | 걱정하다 | ricordarsi | 기억하다 |

### 상호 용법

재귀 대명사는 상호 동작 ("서로")도 표현합니다:
- Ci vediamo domani. (내일 만나자.)
- Si amano molto. (그들은 서로 매우 사랑한다.)
- Vi conoscete? (서로 아시나요?)

## 7. 소사 CI

**CI**는 이탈리아어에서 가장 다용도인 소사 중 하나입니다. 여러 의미를 가집니다:

```
┌─────────────────────────────────────────────────────────────────┐
│   1. CI = "거기" (장소를 대체)                                     │
│      Vai a Roma? — Sì, ci vado domani.                         │
│      (로마에 가니? — 응, 내일 거기에 가.)                           │
│      Sei stato in Italia? — Sì, ci sono stato.                 │
│      (이탈리아에 가봤니? — 응, 거기에 가봤어.)                       │
│                                                                 │
│   2. CI → C'È / CI SONO = "~이 있다"                            │
│      C'è un problema. (문제가 있다.)                              │
│      Ci sono molti studenti. (많은 학생들이 있다.)                 │
│                                                                 │
│   3. CI = "우리를" (직접/간접/재귀 대명사)                          │
│      Ci vede. (그가 우리를 본다.)                                  │
│      Ci parla. (그가 우리에게 말한다.)                              │
│      Ci divertiamo. (우리는 즐긴다.)                              │
│                                                                 │
│   4. CI = "그것에 대해 / 그것에" (a + 명사를 대체)                  │
│      Credi a questa storia? — No, non ci credo.                │
│      (이 이야기를 믿니? — 아니, 안 믿어.)                          │
│      Pensi al futuro? — Sì, ci penso spesso.                  │
│      (미래에 대해 생각하니? — 응, 자주 생각해.)                      │
│                                                                 │
│   5. CI 관용 표현:                                               │
│      volerci = (시간/노력이) 걸리다                                │
│      Ci vogliono due ore. (2시간이 걸린다.)                       │
│      metterci = (개인적으로) 걸리다                                │
│      Ci metto un'ora. (나는 1시간이 걸린다.)                       │
│      entrarci = ~와 관계가 있다                                    │
│      Non c'entra. (그것은 관계없다.)                              │
│      farcela = 해내다                                             │
│      Ce la faccio! (할 수 있어!)                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 8. 소사 NE

**NE**는 "그것의/그들의," "약간," 또는 "그것에 대해"를 대체합니다:

```
┌─────────────────────────────────────────────────────────────────┐
│   1. NE = "그것의/그들의" (di + 명사, 부분을 대체)                  │
│      Vuoi del pane? — Sì, ne voglio un po'.                   │
│      (빵 좀 원하니? — 응, 좀 원해.)                               │
│      Quanti libri hai? — Ne ho tre.                            │
│      (책이 몇 권 있니? — 세 권 있어.)                              │
│                                                                 │
│   2. NE = "그것에 대해" (di + 주제를 대체)                         │
│      Parli del film? — Sì, ne parlo.                           │
│      (영화에 대해 말하니? — 응, 그것에 대해 말하고 있어.)              │
│                                                                 │
│   3. NE = "거기에서" (da + 장소를 대체)                            │
│      Vieni dall'ufficio? — Sì, ne vengo ora.                   │
│      (사무실에서 오니? — 응, 지금 거기에서 와.)                      │
│                                                                 │
│   4. NE 관용 표현:                                               │
│      andarsene = 떠나다 / 가버리다                                │
│      Me ne vado. (나는 떠난다.)                                   │
│      Non ne posso più. (더 이상 못 참겠다.)                       │
│      Che ne pensi? (그것에 대해 어떻게 생각하니?)                   │
│                                                                 │
│   ※ NE가 근과거와 함께 쓰일 때, 과거분사가                          │
│     수량에 일치합니다:                                             │
│     Quante mele hai comprato? — Ne ho comprate tre.            │
│     (사과를 몇 개 샀니? — 세 개 샀어.)                             │
└─────────────────────────────────────────────────────────────────┘
```

## 9. 소유 형용사와 대명사

### 소유 형용사 (명사 앞, 관사와 함께)

| 인칭 | 남성 단수 | 여성 단수 | 남성 복수 | 여성 복수 |
|--------|-----------|----------|-----------|----------|
| io | **il mio** | **la mia** | **i miei** | **le mie** |
| tu | **il tuo** | **la tua** | **i tuoi** | **le tue** |
| lui/lei | **il suo** | **la sua** | **i suoi** | **le sue** |
| Lei (격식) | **il Suo** | **la Sua** | **i Suoi** | **le Sue** |
| noi | **il nostro** | **la nostra** | **i nostri** | **le nostre** |
| voi | **il vostro** | **la vostra** | **i vostri** | **le vostre** |
| loro | **il loro** | **la loro** | **i loro** | **le loro** |

> **영어/스페인어와의 주요 차이**: 이탈리아어 소유형용사는 **정관사**와 함께 사용됩니다: *il mio libro* (내 책), *mio libro*가 아닙니다.

### 예외: 가족 구성원 (단수, 관사 없이)

단수이며 수식어가 없는 가족 구성원 명사에는 관사를 **생략**합니다:
- **mio** padre (내 아버지) — *il mio padre*가 아님
- **sua** madre (그/그녀의 어머니)
- **nostro** fratello (우리 형제)

단, 관사를 **사용하는** 경우:
- 복수: **i miei** fratelli (내 형제들)
- Loro: **il loro** padre (그들의 아버지)
- 수식어 있을 때: **il mio** caro padre (내 사랑하는 아버지)
- 축소사: **la mia** sorellina (내 여동생)

### 소유 대명사 (명사 대체, 관사와 함께)

- È il tuo libro? → Sì, è **il mio**. (응, 내 것이야.)
- La mia casa è grande. **La tua** è piccola. (내 집은 크다. 네 집은 작다.)
- Questi sono **i nostri**. (이것들은 우리 것이다.)

## 10. 지시 대명사

| 거리 | 남성 | 여성 | 중성 |
|----------|-----------|----------|--------|
| 가까운 (이것) | **questo / questi** | **questa / queste** | — |
| 먼 (저것) | **quello / quelli** | **quella / quelle** | — |

명사 앞의 형용사로 쓸 때 QUELLO는 형태가 변합니다 (6과 참조):
- **Quel** libro è interessante. (저 책은 흥미롭다.)
- **Quella** casa è bella. (저 집은 아름답다.)
- **Quei** ragazzi sono simpatici. (저 소년들은 호감이 간다.)

대명사로 (독립적으로) 쓸 때:
- **Questo** è il mio. (이것은 내 것이다.)
- **Quello** è il tuo. (저것은 네 것이다.)
- Preferisco **questi** a **quelli**. (이것들이 저것들보다 좋다.)

> **Ciò** = "이것/저것" (추상적 개념): *Ciò non è vero.* (그것은 사실이 아니다.)

## 11. 필수 전치사

### 11.1 단순 전치사

| 전치사 | 의미 | 예시 |
|-------------|---------|---------|
| **di** | ~의, ~에서, ~에 대해 | Sono **di** Roma. (로마 출신이다.) |
| **a** | ~에, ~으로 | Vado **a** scuola. (학교에 간다.) |
| **da** | ~에서, ~로부터, ~의 집에 | Vengo **da** Roma. (로마에서 왔다.) |
| **in** | ~안에, ~속으로, ~(나라)에 | Vivo **in** Italia. (이탈리아에 산다.) |
| **con** | ~와 함께 | Vado **con** Marco. (Marco와 간다.) |
| **su** | ~위에, ~에 대해 | Il libro è **su**l tavolo. (테이블 위에) |
| **per** | ~을 위해, ~을 통해 | È **per** te. (너를 위한 거야.) |
| **tra / fra** | ~사이에, ~후에 (시간) | **Tra** un'ora. (한 시간 후에.) |

### 11.2 전치관사 (복습)

di, a, da, in, su는 정관사와 결합합니다 (전체 표는 3과 참조):
- **al** cinema (a + il)
- **della** casa (di + la)
- **nel** giardino (in + il)
- **sugli** scaffali (su + gli)
- **dall'**aeroporto (da + l')

### 11.3 복합 전치사

| 전치사 | 의미 | 예시 |
|-------------|---------|---------|
| davanti a | ~앞에 | davanti **alla** casa |
| dietro (a) | ~뒤에 | dietro **la** porta |
| sopra | ~위에, ~꼭대기에 | sopra **il** tavolo |
| sotto | ~아래에 | sotto **il** letto |
| accanto a / vicino a | ~옆에, ~가까이에 | vicino **alla** scuola |
| lontano da | ~에서 멀리 | lontano **dalla** città |
| dentro | ~안에 | dentro **la** borsa |
| fuori (da) | ~바깥에 | fuori **dalla** finestra |
| prima di | ~전에 | prima **di** mangiare |
| dopo (di) | ~후에 | dopo **la** lezione |
| invece di | ~대신에 | invece **di** dormire |
| insieme a / con | ~와 함께 | insieme **a** Marco |

### 11.4 A vs IN vs DA — 이동과 위치

```
┌─────────────────────────────────────────────────────────────────┐
│   A = ~에/~으로 (도시, 특정 장소)                                  │
│   Vado a Roma. (로마에 간다.)                                    │
│   Vado a scuola. (학교에 간다.)                                  │
│   Sono a casa. (집에 있다.)                                      │
│   Vado al cinema. (영화관에 간다.)                                │
│                                                                 │
│   IN = ~안에/~으로 (나라, 지역, 교통수단,                           │
│                    불특정 장소)                                    │
│   Vado in Italia. (이탈리아에 간다.)                               │
│   Vado in Toscana. (토스카나에 간다.)                              │
│   Vado in macchina. (차로 간다.)                                  │
│   Sono in ufficio. (사무실에 있다.)                                │
│   Vado in banca. (은행에 간다.)                                   │
│                                                                 │
│   DA = ~에서 / ~의 집에 / ~에 의해                                 │
│   Vengo da Roma. (로마에서 왔다.)                                 │
│   Vado dal dottore. (의사에게 간다.)                               │
│   Vado da Marco. (Marco의 집에 간다.)                             │
│   È stato fatto da lui. (그에 의해 만들어졌다.)                     │
│   Abito qui da tre anni. (여기에 3년째 살고 있다.)                  │
│                                                                 │
│   ※ 참고: "da"는 기간 표현 (영어 "for/since"):                    │
│     Studio italiano da sei mesi.                               │
│     (이탈리아어를 6개월째 공부하고 있다.)                            │
└─────────────────────────────────────────────────────────────────┘
```

## 12. 연습 문제

### 연습 1: 대명사로 대체하기

밑줄 친 부분을 올바른 대명사로 대체하세요.

1. Vedo **Maria** tutti i giorni. → ________ vedo tutti i giorni.
2. Ho comprato **il libro** ieri. → ________ ho comprato ieri.
3. Ha dato **il regalo** **a sua madre**. → ________ ________ ha dato.
4. Sto leggendo **il romanzo**. → Sto ________.

<details>
<summary>정답 보기</summary>

1. **La** vedo tutti i giorni. (직접목적어: Maria → la)
2. **L'**ho comprato ieri. (직접목적어: il libro → lo → l')
3. **Glielo** ha dato. (간접: a sua madre → le → glie-; 직접: il regalo → lo)
4. Sto **leggendolo**. (또는: **Lo** sto leggendo.)

</details>

### 연습 2: CI 아니면 NE?

**ci** 또는 **ne**를 넣으세요.

1. Vai a Roma domani? — Sì, ________ vado.
2. Quanti fratelli hai? — ________ ho due.
3. Credi a questa storia? — No, non ________ credo.
4. Vuoi del vino? — Sì, ________ voglio un bicchiere.
5. Pensi al futuro? — ________ penso spesso.
6. Vieni dall'ufficio? — Sì, ________ vengo adesso.
7. Sei stato in Giappone? — Sì, ________ sono stato.
8. Parliamo del film? — Sì, ________ parliamo.

<details>
<summary>정답 보기</summary>

1. **ci** ("a Roma" 대체 — 장소)
2. **Ne** ("di fratelli" 대체 — 부분적 수량)
3. **ci** ("a questa storia" 대체 — a + 명사)
4. **ne** ("del vino" 대체 — 부분)
5. **Ci** ("al futuro" 대체 — a + 명사)
6. **ne** ("dall'ufficio" 대체 — da + 장소)
7. **ci** ("in Giappone" 대체 — 장소)
8. **ne** ("del film" 대체 — di + 명사)

</details>

### 연습 3: 소유형

올바른 소유 형태를 넣으세요.

1. ________ padre è medico. (나의)
2. ________ amici sono italiani. (그의)
3. ________ sorelle vivono a Milano. (우리의)
4. Dov'è ________ macchina? (너의, 비격식)
5. ________ madre è insegnante. (그녀의)

<details>
<summary>정답 보기</summary>

1. **Mio** padre è medico. (가족 단수 — 관사 없음)
2. **I suoi** amici sono italiani. (복수, 가족 아님 — 관사 필요)
3. **Le nostre** sorelle vivono a Milano. (복수 가족 — 관사 필요)
4. Dov'è **la tua** macchina? (가족 아님 — 관사 필요)
5. **Sua** madre è insegnante. (가족 단수 — 관사 없음)

</details>

### 연습 4: 전치사 — A, IN, 아니면 DA?

올바른 전치사를 넣으세요 (필요시 관사 포함).

1. Vado ________ Italia quest'estate.
2. Abito ________ Roma.
3. Vengo ________ Corea del Sud.
4. Vado ________ dottore.
5. Viaggio ________ treno.
6. Studio italiano ________ due anni.
7. Andiamo ________ cinema stasera?
8. Sono ________ ufficio.

<details>
<summary>정답 보기</summary>

1. **in** Italia (나라 → in)
2. **a** Roma (도시 → a)
3. **dalla** Corea del Sud (~에서 → da + la)
4. **dal** dottore (~의 곳에 → da + il)
5. **in** treno (교통수단 → in)
6. **da** due anni (기간 → da)
7. **al** cinema (a + il = al)
8. **in** ufficio (in + 관사 없는 장소)

</details>

### 연습 5: 결합 대명사

이중 대명사를 사용하여 다시 쓰세요.

1. Mi dai il libro? → ________ dai?
2. Ti compro la pizza. → ________ compro.
3. Gli mando le foto. → ________ mando.
4. Ci porta i documenti. → ________ porta.

<details>
<summary>정답 보기</summary>

1. **Me lo** dai? (mi → me + il libro → lo)
2. **Te la** compro. (ti → te + la pizza → la)
3. **Gliele** mando. (gli + le foto → le = gliele)
4. **Ce li** porta. (ci → ce + i documenti → li)

</details>
