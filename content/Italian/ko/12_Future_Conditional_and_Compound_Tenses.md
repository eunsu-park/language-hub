## 1. Futuro Semplice: 규칙 활용

Futuro semplice(단순 미래)는 부정사 어간을 변형하고 미래 어미를 추가하여 만듭니다. 참고: **-ARE** 동사는 특성 **-a-**가 **-e-**로 변합니다.

### 규칙 미래 어미

| 인칭 | -ARE (parlare → parler-) | -ERE (credere → creder-) | -IRE (dormire → dormir-) |
|------|--------------------------|--------------------------|--------------------------|
| io | parler**ò** | creder**ò** | dormir**ò** |
| tu | parler**ai** | creder**ai** | dormir**ai** |
| lui/lei/Lei | parler**à** | creder**à** | dormir**à** |
| noi | parler**emo** | creder**emo** | dormir**emo** |
| voi | parler**ete** | creder**ete** | dormir**ete** |
| loro | parler**anno** | creder**anno** | dormir**anno** |

> **핵심**: 어미(-ò, -ai, -à, -emo, -ete, -anno)는 세 동사 그룹 모두 **동일**합니다. 어간만 다릅니다.

### -ARE 추가 예시

| 부정사 | 미래 어간 | io 형태 | lui/lei 형태 |
|--------|---------|---------|-------------|
| mangiare | manger- | mangerò | mangerà |
| lavorare | lavorer- | lavorerò | lavorerà |
| giocare | giocher- | giocherò | giocherà |
| pagare | pagher- | pagherò | pagherà |
| cercare | cercher- | cercherò | cercherà |
| cominciare | comincer- | comincerò | comincerà |

> **철자 참고**: **-care**와 **-gare**로 끝나는 동사는 경음을 유지하기 위해 **h**를 추가합니다: giocare → gioc**h**erò, pagare → pag**h**erò. **-ciare**와 **-giare** 동사는 **i**를 탈락시킵니다: cominciare → cominc**e**rò (*comincierò*가 아님).

## 2. Futuro Semplice: 불규칙 동사

불규칙 미래 동사는 축약되거나 변형된 어간을 가지지만 동일한 규칙 어미를 사용합니다.

### 일반적인 불규칙 미래 어간

| 부정사 | 미래 어간 | 전체 활용 (io/tu/lui/noi/voi/loro) |
|--------|---------|------------------------------------------|
| **essere** | sar- | sarò, sarai, sarà, saremo, sarete, saranno |
| **avere** | avr- | avrò, avrai, avrà, avremo, avrete, avranno |
| **fare** | far- | farò, farai, farà, faremo, farete, faranno |
| **andare** | andr- | andrò, andrai, andrà, andremo, andrete, andranno |
| **venire** | verr- | verrò, verrai, verrà, verremo, verrete, verranno |
| **vedere** | vedr- | vedrò, vedrai, vedrà, vedremo, vedrete, vedranno |
| **dovere** | dovr- | dovrò, dovrai, dovrà, dovremo, dovrete, dovranno |
| **potere** | potr- | potrò, potrai, potrà, potremo, potrete, potranno |
| **volere** | vorr- | vorrò, vorrai, vorrà, vorremo, vorrete, vorranno |
| **sapere** | sapr- | saprò, saprai, saprà, sapremo, saprete, sapranno |
| **vivere** | vivr- | vivrò, vivrai, vivrà, vivremo, vivrete, vivranno |
| **rimanere** | rimarr- | rimarrò, rimarrai, rimarrà, rimarremo, rimarrete, rimarranno |
| **tenere** | terr- | terrò, terrai, terrà, terremo, terrete, terranno |
| **bere** | berr- | berrò, berrai, berrà, berremo, berrete, berranno |
| **dare** | dar- | darò, darai, darà, daremo, darete, daranno |
| **stare** | star- | starò, starai, starà, staremo, starete, staranno |

### 패턴 그룹

```
+----------------------------------------------------------+
|  -r 앞의 모음 탈락:                                       |
|  avere → avr-   dovere → dovr-   potere → potr-          |
|  sapere → sapr-  vedere → vedr-   vivere → vivr-          |
|  andare → andr-                                           |
|                                                          |
|  -r 이중화:                                               |
|  venire → verr-   volere → vorr-   rimanere → rimarr-    |
|  tenere → terr-   bere → berr-                            |
|                                                          |
|  특수 어간:                                               |
|  essere → sar-    fare → far-    dare → dar-              |
|  stare → star-                                            |
+----------------------------------------------------------+
```

## 3. 미래 시제의 용법

| 용법 | 예시 | 한국어 |
|------|------|--------|
| **미래 계획** | Domani andrò al mare. | 내일 바다에 갈 것이다. |
| **예측** | Pioverà stasera. | 오늘 저녁 비가 올 것이다. |
| **약속** | Ti chiamerò domani. | 내일 전화할게. |
| **추측/가능성** | Saranno le tre. | 3시쯤 될 것이다. |
| **quando/se 이후 (미래 맥락)** | Quando arriverai, chiamami. | 도착하면 전화해. |

> **중요**: 영어와 달리 이탈리아어는 행동이 미래일 때 **quando** (when), **se** (if), **appena** (as soon as), **finché** (until) 뒤에 미래 시제를 사용합니다:
> - Quando **arriverò**, ti **chiamerò**. (도착하면 전화할게.)
> - Se **pioverà**, **resteremo** a casa. (비가 오면 집에 있을 거야.)

## 4. Condizionale Presente: 규칙 활용

Condizionale presente(현재 조건법)는 미래와 동일한 어간을 공유하지만 다른 어미를 사용합니다.

### 규칙 조건법 어미

| 인칭 | -ARE (parlare → parler-) | -ERE (credere → creder-) | -IRE (dormire → dormir-) |
|------|--------------------------|--------------------------|--------------------------|
| io | parler**ei** | creder**ei** | dormir**ei** |
| tu | parler**esti** | creder**esti** | dormir**esti** |
| lui/lei/Lei | parler**ebbe** | creder**ebbe** | dormir**ebbe** |
| noi | parler**emmo** | creder**emmo** | dormir**emmo** |
| voi | parler**este** | creder**este** | dormir**este** |
| loro | parler**ebbero** | creder**ebbero** | dormir**ebbero** |

> **암기 도움**: 조건법 어미는 **-ei, -esti, -ebbe, -emmo, -este, -ebbero**입니다. 모든 동사 그룹에서 동일합니다.

## 6. 조건법의 용법

| 용법 | 예시 | 한국어 |
|------|------|--------|
| **공손한 요청** | Vorrei un caffè, per favore. | 커피 한 잔 주세요. |
| **소망/바람** | Mi piacerebbe vivere in Italia. | 이탈리아에서 살고 싶습니다. |
| **조언** | Dovresti studiare di più. | 더 공부해야 해. |
| **가정 상황** | Mangerei volentieri una pizza. | 기꺼이 피자를 먹겠다. |
| **간접 화법 (과거의 미래)** | Ha detto che verrebbe. | 그가 올 것이라고 말했다. |
| **완곡한 표현** | Direi di sì. | 그렇다고 말하겠습니다. |
| **불확실성** | Secondo il giornale, ci sarebbero 100 feriti. | 신문에 따르면 100명의 부상자가 있을 것이라고 한다. |

### 공손한 표현에서의 조건법

| 조건법 | 현재형 (덜 공손) |
|--------|-----------------|
| Vorrei... (~을 원합니다...) | Voglio... (~을 원한다...) |
| Potrebbe aiutarmi? (도와주실 수 있으신가요?) | Può aiutarmi? (도와줄 수 있어?) |
| Sarebbe possibile...? (~이 가능할까요?) | È possibile...? (~이 가능해?) |
| Le dispiacerebbe...? (~해 주시겠어요?) | Le dispiace...? (~해 줄래?) |

## 7. Trapassato Prossimo

Trapassato prossimo(대과거)는 다른 과거 동작 **이전에** 발생한 동작을 설명합니다.

```
+----------------------------------------------------------+
|  TRAPASSATO PROSSIMO = avere/essere의 IMPERFETTO          |
|                       + PARTICIPIO PASSATO                |
|                                                          |
|  Avevo già mangiato quando è arrivato.                   |
|  (그가 도착했을 때 나는 이미 먹은 후였다.)                  |
|                                                          |
|  타임라인:  [mangiato] -----> [arrivato] -----> 현재       |
|             (이전)            (이후 과거)                  |
+----------------------------------------------------------+
```

### 구성

| 인칭 | AVERE 사용 (mangiare) | ESSERE 사용 (andare, 남성) |
|------|----------------------|--------------------------|
| io | avevo mangiato | ero andato/a |
| tu | avevi mangiato | eri andato/a |
| lui/lei | aveva mangiato | era andato/a |
| noi | avevamo mangiato | eravamo andati/e |
| voi | avevate mangiato | eravate andati/e |
| loro | avevano mangiato | erano andati/e |

### 예시

| 이탈리아어 | 한국어 |
|-----------|--------|
| **Avevo già finito** quando sei arrivato. | 네가 도착했을 때 나는 이미 끝낸 후였다. |
| Non **aveva mai visto** il mare. | 그는 바다를 본 적이 없었다. |
| **Eravamo già partiti** quando ha chiamato. | 그가 전화했을 때 우리는 이미 떠난 후였다. |
| **Si era addormentata** presto. | 그녀는 일찍 잠들어 있었다. |
| Non **avevo capito** che era importante. | 나는 그것이 중요하다는 것을 이해하지 못했었다. |

## 8. Futuro Anteriore

Futuro anteriore(전미래)는 다른 미래 동작 전에 **완료되어 있을** 동작을 설명합니다.

```
+----------------------------------------------------------+
|  FUTURO ANTERIORE = avere/essere의 FUTURO                 |
|                    + PARTICIPIO PASSATO                   |
|                                                          |
|  Quando arriverai, avrò già mangiato.                    |
|  (네가 도착할 때 나는 이미 먹은 후일 것이다.)               |
|                                                          |
|  타임라인:  현재 -----> [mangiato] -----> [arriverai]      |
+----------------------------------------------------------+
```

### 구성

| 인칭 | AVERE 사용 (finire) | ESSERE 사용 (partire, 남성) |
|------|--------------------|-----------------------------|
| io | avrò finito | sarò partito/a |
| tu | avrai finito | sarai partito/a |
| lui/lei | avrà finito | sarà partito/a |
| noi | avremo finito | saremo partiti/e |
| voi | avrete finito | sarete partiti/e |
| loro | avranno finito | saranno partiti/e |

### 예시

| 이탈리아어 | 한국어 |
|-----------|--------|
| Quando arriverai, **avrò già cucinato**. | 네가 도착할 때 나는 이미 요리를 끝냈을 것이다. |
| Entro domani **avremo finito** il progetto. | 내일까지 우리는 프로젝트를 끝냈을 것이다. |
| Quando **sarai tornato**, ne parleremo. | 네가 돌아오면 그것에 대해 이야기하자. |

### 과거 추측을 위한 Futuro Anteriore

Futuro anteriore는 과거에 대한 추측도 표현할 수 있습니다:

| 이탈리아어 | 한국어 |
|-----------|--------|
| **Avrà perso** il treno. | 기차를 놓쳤을 것이다. |
| **Saranno arrivati** ieri sera. | 어젯밤에 도착했을 것이다. |
| **Avrà dimenticato** la riunione. | 회의를 잊어버렸을 것이다. |

## 9. Condizionale Passato

Condizionale passato(과거 조건법)는 **일어났을 법한** 일을 표현합니다.

```
+----------------------------------------------------------+
|  CONDIZIONALE PASSATO = avere/essere의 CONDIZIONALE       |
|                        + PARTICIPIO PASSATO               |
|                                                          |
|  Avrei mangiato, ma non avevo fame.                      |
|  (먹었을 텐데, 배가 안 고팠다.)                            |
+----------------------------------------------------------+
```

### 구성

| 인칭 | AVERE 사용 (mangiare) | ESSERE 사용 (andare, 남성) |
|------|----------------------|--------------------------|
| io | avrei mangiato | sarei andato/a |
| tu | avresti mangiato | saresti andato/a |
| lui/lei | avrebbe mangiato | sarebbe andato/a |
| noi | avremmo mangiato | saremmo andati/e |
| voi | avreste mangiato | sareste andati/e |
| loro | avrebbero mangiato | sarebbero andati/e |

### 용법

| 용법 | 예시 | 한국어 |
|------|------|--------|
| **이루어지지 않은 소망** | Sarei venuto, ma ero malato. | 갔을 텐데, 아팠다. |
| **후회** | Avrei dovuto studiare di più. | 더 공부했어야 했다. |
| **과거에 대한 조언** | Avresti dovuto dirmelo. | 나에게 말했어야 했다. |
| **간접 화법 (과거의 미래)** | Ha detto che sarebbe venuto. | 그가 올 것이라고 말했다. |
| **공손한 완화** | Avrei una domanda. | 질문이 하나 있습니다. |

## 10. 특수 용법: 추측의 미래

이탈리아어 미래 시제의 가장 독특한 용법 중 하나는 현재에 대한 **추측이나 가능성**을 표현하는 것입니다.

| 추측의 미래 | 동등한 의미 |
|------------|-----------|
| Che ore **saranno**? — **Saranno** le tre. | 몇 시쯤 됐을까? — 3시쯤 될 거야. |
| Quanti anni **avrà**? | 몇 살쯤 될까? |
| **Sarà** stanco. | 피곤할 거야. |
| **Costerà** caro. | 아마 비쌀 거야. |
| Dove **sarà** Marco? | Marco가 어디에 있을까? |
| Chi **sarà** a quest'ora? | 이 시간에 누구일까? |

### 과거 추측을 위한 Futuro Anteriore

| 과거 추측 | 동등한 의미 |
|----------|-----------|
| **Avrà dimenticato**. | 잊어버렸을 것이다. |
| **Sarà uscito/a**. | 나갔을 것이다. |
| **Avranno avuto** un problema. | 문제가 있었을 것이다. |
| Non **sarà stato** facile. | 쉽지 않았을 것이다. |

### 연습 5: Condizionale Passato

후회나 이루어지지 않은 행동을 표현하세요.

1. 전화했어야 했다. → ________
2. 우리는 갔을 텐데 비가 왔다. → ________
3. 그녀는 오고 싶었을 것이다. → ________
4. 나에게 말했어야 했다. (tu) → ________
5. 그들은 집을 샀을 텐데 너무 비쌌다. → ________

<details>
<summary>정답</summary>

1. Avrei dovuto chiamare.
2. Saremmo andati/e, ma pioveva.
3. Le sarebbe piaciuto venire. / Sarebbe voluta venire.
4. Avresti dovuto dirmelo.
5. Avrebbero comprato la casa, ma era troppo cara.

</details>

### 연습 6: 혼합 시제 번역

적절한 시제를 사용하여 이탈리아어로 번역하세요.

1. 내일 로마에 갈 것이다.
2. 일본 여행을 하고 싶다.
3. 네가 도착할 때 나는 이미 끝냈을 것이다.
4. 그녀는 지금 집에 있을 것이다 (추측의 미래 사용).
5. 그는 나에게 전화할 것이라고 말했다.
6. 내년까지 우리는 졸업했을 것이다.
7. 집에 있는 것이 더 좋았을 텐데.

<details>
<summary>정답</summary>

1. Domani andrò a Roma.
2. Mi piacerebbe viaggiare in Giappone. / Vorrei viaggiare in Giappone.
3. Quando arriverai, avrò già finito.
4. Sarà a casa adesso.
5. Ha detto che mi avrebbe chiamato.
6. Entro l'anno prossimo, ci saremo laureati/e.
7. Avrei preferito restare / rimanere a casa.

</details>
