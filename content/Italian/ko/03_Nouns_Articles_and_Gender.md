## 1. 문법적 성이란?

모든 이탈리아어 명사에는 문법적 성이 있습니다: **남성** 또는 **여성**. 이것은 명사와 함께 쓰이는 관사, 형용사, 대명사에 영향을 줍니다.

```
┌─────────────────────────────────────────────────────────────────┐
│   영어: "the book", "the table", "the house"                    │
│   (명사와 관계없이 같은 관사)                                      │
│                                                                 │
│   이탈리아어: "il libro", "la tavola", "la casa"                 │
│   (관사가 명사의 성과 첫소리에 맞게 변화)                           │
│                                                                 │
│   중요한 이유:                                                   │
│   • 관사가 일치해야 함: il libro / la tavola                      │
│   • 형용사가 일치해야 함: libro nuovo / tavola nuova               │
│   • 대명사가 일치해야 함: lo compro / la compro                   │
│   • 이탈리아어에는 7개의 다른 정관사가 있습니다 (단 2개가 아님!)      │
└─────────────────────────────────────────────────────────────────┘
```

> **한국어 화자를 위한 참고**: 한국어 명사에는 문법적 성이 없습니다 (책, 탁자는 성 중립). 이탈리아어에서는 모든 명사의 성을 관사와 함께 외워야 합니다. "libro"만이 아닌 **il libro**로 학습하세요.

## 2. 성 규칙

### 2.1 남성 명사 표지

```
전형적 남성:
  -o 어미:     il libro (책), il gatto (고양이), il ragazzo (소년)
  -ore 어미:   il dottore (의사), l'amore (사랑), il colore (색)
  -iere 어미:  il cameriere (웨이터), il barbiere (이발사)
  -ment- 어미: il momento, il sentimento
  요일:        il lunedì, il martedì, il mercoledì...
  월:          gennaio, febbraio (보통 관사 없이, 남성)
  언어:        l'italiano, l'inglese, il coreano
  나무:        il melo (사과나무), il pero (배나무)
```

### 2.2 여성 명사 표지

```
전형적 여성:
  -a 어미:      la casa (집), la tavola (테이블), la donna (여자)
  -zione 어미:  la nazione (국가), la stazione (역)
  -sione 어미:  la televisione, la decisione
  -tà 어미:     la città (도시), l'università (대학)
  -tù 어미:     la gioventù (청춘), la virtù (미덕)
  -ezza 어미:   la bellezza (아름다움), la dolcezza (달콤함)
  -ura 어미:    la natura (자연), la cultura (문화)
  -ità 어미:    la qualità (질), la quantità (양)
  과일:         la mela (사과), la pera (배)
```

### 2.3 -E로 끝나는 명사 (어느 성이든 가능 — 암기 필수)

```
┌─────────────────────────────────────────────────────────────────┐
│   남성 -e 명사:                                                  │
│   il pane (빵)             il nome (이름)                        │
│   il giornale (신문)        il mare (바다)                        │
│   il pesce (생선)           il fiore (꽃)                         │
│   il sole (태양)            il ponte (다리)                       │
│                                                                 │
│   여성 -e 명사:                                                  │
│   la notte (밤)            la chiave (열쇠)                      │
│   la classe (수업)          la gente (사람들)                     │
│   la mente (마음)           la madre (어머니)                     │
│   la canzone (노래)         la legge (법률)                      │
│                                                                 │
│   ※ 확실한 규칙이 없습니다 — -e 명사의 성은 개별적으로                │
│     학습해야 합니다. 관사와 함께 외우세요!                           │
└─────────────────────────────────────────────────────────────────┘
```

### 2.4 중요한 예외

```
-a로 끝나는 남성 단어 (반드시 외우세요!):
  il problema (문제)          il programma (프로그램)
  il sistema (시스템)          il tema (주제/작문)
  il cinema (영화관)           il clima (기후)
  il panorama (파노라마)       il dramma (드라마)
  il dilemma (딜레마)          il telegramma (전보)
  il poeta (시인)             il pilota (조종사)

  ※ 그리스어에서 온 많은 -ma 단어가 남성입니다 (스페인어와 동일)

-o로 끝나는 여성 단어 (반드시 외우세요!):
  la mano (손)               la radio (라디오)
  la foto (사진)              la moto (오토바이)
  la auto (자동차, 축약형)     la eco (메아리)

중성 (같은 형태, 관사가 의미를 바꿈):
  il/la cantante (가수)        il/la giornalista (기자)
  il/la turista (관광객)        il/la artista (예술가)
  il/la collega (동료)          il/la pianista (피아니스트)
  il/la dentista (치과의사)     il/la atleta (운동선수)
```

## 3. 정관사 — 7가지 형태

이탈리아어에는 **7개의 정관사**가 있습니다 — 이것은 이 언어의 가장 독특한 특징 중 하나입니다. 선택은 성, 수, 그리고 다음 단어의 첫소리에 따라 달라집니다.

### 3.1 완전한 체계

| | 대부분의 자음 앞 | s+자음, z, gn, ps, x, y 앞 | 모음 앞 |
|---|---|---|---|
| **남성 단수** | **il** | **lo** | **l'** |
| **남성 복수** | **i** | **gli** | **gli** |
| **여성 단수** | **la** | **la** | **l'** |
| **여성 복수** | **le** | **le** | **le** |

### 3.2 남성 관사와 예시

```
IL (대부분의 자음 앞):
  il libro (그 책)            il gatto (그 고양이)
  il ragazzo (그 소년)         il tavolo (그 테이블)
  il telefono (그 전화기)      il treno (그 기차)

LO (s+자음, z, gn, ps, x, y, 그리고 때로 i+모음 앞):
  lo studente (그 학생)        lo zaino (그 배낭)
  lo gnomo (그 노움)           lo psicologo (그 심리학자)
  lo xilofono (그 실로폰)      lo yogurt (그 요거트)
  lo specchio (그 거울)        lo sport (스포츠)

L' (모음 앞 — 남성):
  l'uomo (그 남자)             l'amico (그 친구)
  l'albero (그 나무)           l'italiano (그 이탈리아인)
  l'orologio (그 시계)         l'aereo (그 비행기)

I (il의 복수):
  i libri (그 책들)            i gatti (그 고양이들)
  i ragazzi (그 소년들)        i treni (그 기차들)

GLI (lo와 l'의 복수):
  gli studenti (그 학생들)      gli zaini (그 배낭들)
  gli uomini (그 남자들)        gli amici (그 친구들)
  gli sport                    gli italiani (그 이탈리아인들)
```

### 3.3 여성 관사와 예시

```
LA (자음 앞):
  la casa (그 집)              la donna (그 여자)
  la scuola (그 학교)          la stazione (그 역)
  la zia (그 이모/고모)         la studentessa (그 여학생)

L' (모음 앞 — 여성):
  l'amica (그 여자 친구)       l'acqua (그 물)
  l'idea (그 아이디어)          l'ora (그 시간)
  l'università (그 대학)

LE (모든 여성 복수):
  le case (그 집들)            le donne (그 여자들)
  le amiche (그 여자 친구들)    le idee (그 아이디어들)
```

### 3.4 정관사를 사용하는 경우 (영어와 다른 점)

```
이탈리아어에서 "the"를 쓰지만 영어에서는 안 쓰는 경우:
  Mi piace il cioccolato.      → 나는 초콜릿을 좋아한다. (일반)
  La vita è bella.             → 삶은 아름답다.
  I gatti sono indipendenti.   → 고양이는 독립적이다.
  L'italiano è musicale.       → 이탈리아어는 음악적이다.
  Il lunedì vado in palestra.  → 월요일마다 헬스장에 간다.
  Parlo l'italiano.            → 이탈리아어를 한다. (언어와 함께)

이탈리아어에서 "the"를 안 쓰지만 영어에서는 쓰는 경우:
  Sono studente.               → 나는 학생이다. (직업)
  Ho bisogno di caffè.        → 커피가 필요하다. (일부)
```

## 4. 부정관사

### 4.1 "a/an"의 네 가지 형태

| | 대부분의 자음 앞 | s+자음, z, gn, ps, x, y 앞 | 모음 앞 |
|---|---|---|---|
| **남성** | **un** | **uno** | **un** |
| **여성** | **una** | **una** | **un'** |

```
UN:
  un libro (한 책)             un amico (한 친구, 남)
  un gatto (한 고양이)          un italiano (한 이탈리아인)

UNO:
  uno studente (한 학생)        uno zaino (한 배낭)
  uno psicologo (한 심리학자)    uno gnomo (한 노움)

UNA:
  una casa (한 집)             una donna (한 여자)
  una scuola (한 학교)          una studentessa (한 여학생)

UN' (모음 앞 여성 — 아포스트로피 주의):
  un'amica (한 친구, 여)        un'idea (한 아이디어)
  un'ora (한 시간)              un'università (한 대학)

※ UN (모음 앞 남성)에는 아포스트로피가 없습니다:
  un amico ✓    un'amico ✗
  un'amica ✓    una amica ✗
```

> **중요**: 이탈리아어에는 "some"에 해당하는 복수 부정관사가 없습니다. "some"을 위해 *alcuni/e* (몇몇), *qualche* (몇몇, + 단수 명사), 또는 *dei/delle* (부분관사)를 사용합니다.

### 4.2 부분관사 (일부/약간)

부분관사는 **di + 정관사**로 형성됩니다:

| | 의미 | 예시 |
|---|---|---|
| del | 약간의 (남성 단수) | del pane (빵 좀) |
| dello | 약간의 (남성 단수) | dello zucchero (설탕 좀) |
| dell' | 약간의 (모음 앞) | dell'acqua (물 좀) |
| della | 약간의 (여성 단수) | della pasta (파스타 좀) |
| dei | 약간의 (남성 복수) | dei libri (책 몇 권) |
| degli | 약간의 (남성 복수) | degli amici (친구 몇 명) |
| delle | 약간의 (여성 복수) | delle mele (사과 몇 개) |

## 5. 복수형 형성

### 5.1 규칙 복수

```
┌─────────────────────────────────────────────────────────────────┐
│   남성 명사:                                                     │
│   -o → -i     il libro → i libri    il gatto → i gatti         │
│   -e → -i     il fiore → i fiori    il pesce → i pesci         │
│                                                                 │
│   여성 명사:                                                     │
│   -a → -e     la casa → le case     la donna → le donne        │
│   -e → -i     la notte → le notti   la chiave → le chiavi      │
│                                                                 │
│   요약:                                                          │
│   ┌─────────┬───────────┬───────────┐                          │
│   │ 어미    │ 단수      │ 복수      │                           │
│   ├─────────┼───────────┼───────────┤                          │
│   │ 남 -o   │ libro     │ libr-i    │                          │
│   │ 여 -a   │ casa      │ cas-e     │                          │
│   │ 남/여-e │ fiore (남)│ fior-i    │                           │
│   │         │ notte (여)│ nott-i    │                           │
│   └─────────┴───────────┴───────────┘                          │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 복수형의 철자 변화

```
-co/-go → -chi/-ghi (경음 유지, 보통 끝에서 두 번째 음절 강세):
  amico → amici (예외!)     그러나: tedesco → tedeschi
  parco → parchi             lago → laghi
  fungo → funghi             albergo → alberghi

-co/-go → -ci/-gi (끝에서 세 번째 음절 강세일 때):
  medico → medici            amico → amici
  sindaco → sindaci          monaco → monaci

-ca/-ga → -che/-ghe (항상 경음 유지):
  amica → amiche             banca → banche
  bottega → botteghe         collega → colleghe (여)

-cia/-gia → -cie/-gie (모음 뒤에 올 때):
  camicia → camicie          ciliegia → ciliegie

-cia/-gia → -ce/-ge (자음 뒤에 올 때):
  arancia → arance           pioggia → piogge
  provincia → province       spiaggia → spiagge
```

### 5.3 불변 명사 (복수에서 변화 없음)

```
악센트가 있는 마지막 모음:
  la città → le città         il caffè → i caffè
  l'università → le università    la virtù → le virtù

단음절 명사:
  il re → i re (왕)           la gru → le gru (두루미)

자음 어미 (외래어):
  il bar → i bar              lo sport → gli sport
  il film → i film            il computer → i computer
  l'autobus → gli autobus

축약형:
  la foto → le foto           la moto → le moto
  la radio → le radio         il cinema → i cinema
  l'auto → le auto
```

### 5.4 불규칙 복수

```
남성 단수 → 여성 복수 (!) — 종종 신체 부위:
  il braccio → le braccia (팔)
  il dito → le dita (손가락)
  il ginocchio → le ginocchia (무릎)
  il labbro → le labbra (입술)
  l'osso → le ossa (뼈)
  l'uovo → le uova (달걀)
  il paio → le paia (켤레)
  il miglio → le miglia (마일)

기타 불규칙:
  l'uomo → gli uomini (남자들)
  il bue → i buoi (황소들)
```

## 6. 전치관사

이탈리아어 문법의 가장 독특한 특징 중 하나: 특정 전치사가 정관사와 결합하여 하나의 단어를 형성합니다.

### 6.1 전체 표

| 전치사 \ 관사 | il | lo | la | l' | i | gli | le |
|---|---|---|---|---|---|---|---|
| **di** (~의) | del | dello | della | dell' | dei | degli | delle |
| **a** (~에, ~으로) | al | allo | alla | all' | ai | agli | alle |
| **da** (~에서, ~로부터) | dal | dallo | dalla | dall' | dai | dagli | dalle |
| **in** (~안에) | nel | nello | nella | nell' | nei | negli | nelle |
| **su** (~위에) | sul | sullo | sulla | sull' | sui | sugli | sulle |

> **결합하지 않는 전치사**: *con* (~와 함께), *per* (~을 위해), *tra/fra* (~사이에). 이들은 분리 상태 유지: *con il*, *per la*, *tra i*.
>
> 예외: *con + il = col*과 *con + i = coi*가 존재하지만 현대 문어에서는 덜 흔합니다.

### 6.2 전치관사 예시

```
DI (~의, ~에서, ~에 대해):
  il libro del professore     (교수의 책)
  la porta della casa         (집의 문)
  la fine dell'estate        (여름의 끝)
  le pagine dei libri         (책들의 페이지)

A (~에, ~으로):
  Vado al cinema.            (영화관에 간다.)
  Siamo alla stazione.       (역에 있다.)
  Scrivo agli amici.         (친구들에게 쓴다.)
  Arrivo alle tre.           (3시에 도착한다.)

DA (~에서, ~로부터, ~의 집에):
  Vengo dal lavoro.          (직장에서 온다.)
  Passo dalla nonna.         (할머니 집에 들른다.)
  Dai miei genitori.         (부모님 댁에서.)

IN (~안에, ~속으로):
  nel giardino              (정원 안에)
  nella borsa               (가방 안에)
  nei negozi                (가게들 안에)
  negli Stati Uniti         (미국에)

SU (~위에, ~에 대해):
  sul tavolo                (테이블 위에)
  sulla sedia               (의자 위에)
  sugli scaffali            (선반 위에)
  sulle montagne            (산 위에)
```

## 7. 필수 명사: 가족

| 이탈리아어 | 한국어 | 성 |
|-----------|--------|-----|
| la famiglia | 가족 | 여 |
| il padre / la madre | 아버지 / 어머니 | 남 / 여 |
| il figlio / la figlia | 아들 / 딸 | 남 / 여 |
| il fratello / la sorella | 형제(오빠) / 자매(언니) | 남 / 여 |
| il nonno / la nonna | 할아버지 / 할머니 | 남 / 여 |
| lo zio / la zia | 삼촌(외삼촌) / 이모(고모) | 남 / 여 |
| il cugino / la cugina | 사촌 | 남 / 여 |
| il marito / la moglie | 남편 / 아내 | 남 / 여 |
| il nipote / la nipote | 조카-손자 / 조카-손녀 | 남 / 여 |
| il suocero / la suocera | 시아버지(장인) / 시어머니(장모) | 남 / 여 |
| il cognato / la cognata | 처남(시숙) / 처제(시누이) | 남 / 여 |
| il bambino / la bambina | 아이 (남아/여아) | 남 / 여 |
| i genitori | 부모님 | 남 복수 |
| i figli | 자녀 (아들과 딸) | 남 복수 |

**예문:**
- Ho due fratelli. — 형제가 둘 있습니다.
- Mia madre è insegnante. — 어머니는 선생님입니다.
- I miei nonni abitano a Busan. — 조부모님은 부산에 사십니다.

> **참고**: 남성 복수는 양성을 포함할 수 있습니다: *i fratelli* = 형제들, 또는 형제자매. *nipote*라는 단어는 남녀 형태가 같으며 관사만 바뀝니다.

## 10. 연습 문제

### 연습 1: 성 식별
*il/lo/l'* 인가요 *la/l'* 인가요?

<details>
<summary>정답 보기</summary>

```
1. ___ libro      → il libro (남성, -o 어미)
2. ___ tavola     → la tavola (여성, -a 어미)
3. ___ città      → la città (여성, -tà 어미)
4. ___ problema   → il problema (남성, 그리스어 -ma 예외)
5. ___ mano       → la mano (여성, 예외!)
6. ___ studente   → lo studente (남성, s + 자음)
7. ___ canzone    → la canzone (여성, -zione 어미)
8. ___ acqua      → l'acqua (여성, 모음)
9. ___ foto       → la foto (여성, 예외)
10. ___ fiore     → il fiore (남성, -e 명사 — 암기 필수)
```

</details>

### 연습 2: 관사 맞추기
올바른 관사를 넣으세요 (il, lo, la, l', i, gli, le, un, uno, una, un').

<details>
<summary>정답 보기</summary>

```
1. Ho ___ sorella e ___ fratello.     → una, un
2. ___ libri sono su ___ tavolo.      → I, il (→ I libri sono sul tavolo.)
3. È ___ bravo studente.              → un (uno 아님 — un bravo studente)
4. ___ mani di ___ bambina.           → Le, la (→ Le mani della bambina.)
5. Vado a ___ scuola.                 → la (→ Vado alla scuola.)
6. Vengo da ___ mercato.              → il (→ Vengo dal mercato.)
7. ___ zaino è grande.                → Lo
8. ___ amiche di Maria.               → Le
9. Ho ___ amica italiana.             → un'
10. ___ uomini e ___ donne.           → Gli, le
```

</details>

### 연습 3: 복수형 만들기
복수형으로 바꾸세요.

<details>
<summary>정답 보기</summary>

```
1. il libro      → i libri
2. la casa       → le case
3. lo studente   → gli studenti
4. l'amico       → gli amici
5. il pesce      → i pesci
6. la città      → le città (불변 — 악센트)
7. il braccio    → le braccia (불규칙!)
8. il film       → i film (불변 — 외래어)
9. l'uomo        → gli uomini (불규칙!)
10. la chiave    → le chiavi
```

</details>

### 연습 4: 전치관사
전치사와 관사를 올바른 형태로 결합하세요.

<details>
<summary>정답 보기</summary>

```
1. di + il = del         → il libro del professore
2. a + la = alla         → Vado alla stazione.
3. da + il = dal         → Vengo dal lavoro.
4. in + la = nella       → nella cucina
5. su + il = sul         → sul tavolo
6. di + gli = degli      → degli studenti
7. a + i = ai            → Scrivo ai miei amici.
8. in + l' = nell'       → nell'ufficio
9. da + la = dalla       → dalla nonna
10. su + le = sulle      → sulle montagne
```

</details>

### 연습 5: 문장 만들기
배운 명사와 관사를 사용하여 문장을 만드세요.

<details>
<summary>정답 보기</summary>

```
1. (가족 / 살다 / 도시)
   → La mia famiglia vive in città.

2. (가다 / 공원 / 오후)
   → Vado al parco nel pomeriggio.

3. (집 / 갖고 있다 / 방들)
   → La casa ha cinque stanze.

4. (도서관 / 근처 / 대학)
   → La biblioteca è vicino all'università.

5. (책 / 위에 / 테이블)
   → Il libro è sul tavolo.

6. (오다 / ~에서 / 시장)
   → Vengo dal mercato.
```

</details>
