## 1. 명령법: 개요

Imperativo(명령법)는 명령, 요청, 지시, 제안에 사용됩니다. 이탈리아어에는 네 가지 인칭의 명령법 형태가 있습니다:

```
+----------------------------------------------------------+
|  명령법 인칭                                               |
|                                                          |
|  tu    — 비격식 단수 (친구, 가족)                          |
|  Lei   — 격식 단수 (낯선 사람, 상사)                       |
|  noi   — 포함적 "~합시다" (noi)                            |
|  voi   — 복수 (그룹 대상)                                  |
|                                                          |
|  참고: "Lei" 형태는 접속법에서 차용됩니다.                   |
|  참고: 현대 표준 이탈리아어에서 io나 loro에 대한             |
|        명령법은 없습니다.                                   |
+----------------------------------------------------------+
```

## 2. 규칙 명령법 형태

### -ARE 동사 (예: parlare — 말하다)

| 인칭 | 형태 | 예시 |
|------|------|------|
| tu | parl**a** | Parla più piano! (더 조용히 말해!) |
| Lei | parl**i** | Parli più piano! (더 조용히 말씀하세요! — 격식) |
| noi | parl**iamo** | Parliamo italiano! (이탈리아어로 이야기합시다!) |
| voi | parl**ate** | Parlate più forte! (더 크게 말해!) |

### -ERE 동사 (예: scrivere — 쓰다)

| 인칭 | 형태 | 예시 |
|------|------|------|
| tu | scriv**i** | Scrivi il tuo nome! (이름을 써!) |
| Lei | scriv**a** | Scriva il Suo nome! (성함을 써 주세요! — 격식) |
| noi | scriv**iamo** | Scriviamo una lettera! (편지를 씁시다!) |
| voi | scriv**ete** | Scrivete tutto! (전부 써!) |

### -IRE 동사 (예: dormire — 자다)

| 인칭 | 형태 | 예시 |
|------|------|------|
| tu | dorm**i** | Dormi bene! (잘 자!) |
| Lei | dorm**a** | Dorma bene! (편히 주무세요! — 격식) |
| noi | dorm**iamo** | Dormiamo un po'! (좀 잡시다!) |
| voi | dorm**ite** | Dormite presto! (일찍 자!) |

### -ISC- 포함 -IRE 동사 (예: capire — 이해하다, finire — 끝내다)

| 인칭 | 형태 | 예시 |
|------|------|------|
| tu | cap**isci** | Capisci bene! (잘 이해해!) |
| Lei | cap**isca** | Capisca bene! (잘 이해해 주세요! — 격식) |
| noi | cap**iamo** | Capiamo la situazione! (상황을 이해합시다!) |
| voi | cap**ite** | Capite? (이해해? / 이해해!) |

### 요약표

| 인칭 | -ARE | -ERE | -IRE | -IRE (-isc-) |
|------|------|------|------|---------------|
| **tu** | -a | -i | -i | -isci |
| **Lei** | -i | -a | -a | -isca |
| **noi** | -iamo | -iamo | -iamo | -iamo |
| **voi** | -ate | -ete | -ite | -ite |

> **핵심 관찰**: **tu**의 경우, -ARE 명령법은 **-a**로 끝납니다 (현재 직설법과 동일). -ERE/-IRE의 tu 형태는 현재 직설법 tu 형태와 동일합니다. **Lei** 형태는 항상 **접속법**에서 가져옵니다.

## 3. 불규칙 명령법

여러 일상 동사들이 특히 **tu**와 **Lei**에서 불규칙 명령법 형태를 가집니다.

### 가장 중요한 불규칙 명령법

| 부정사 | tu | Lei | noi | voi |
|--------|-----|------|------|------|
| **essere** (~이다) | sii | sia | siamo | siate |
| **avere** (가지다) | abbi | abbia | abbiamo | abbiate |
| **andare** (가다) | va' / vai | vada | andiamo | andate |
| **fare** (하다/만들다) | fa' / fai | faccia | facciamo | fate |
| **dire** (말하다) | di' | dica | diciamo | dite |
| **dare** (주다) | da' / dai | dia | diamo | date |
| **stare** (있다/머무르다) | sta' / stai | stia | stiamo | state |
| **venire** (오다) | vieni | venga | veniamo | venite |
| **uscire** (나가다) | esci | esca | usciamo | uscite |
| **tenere** (잡다) | tieni | tenga | teniamo | tenete |
| **sapere** (알다) | sappi | sappia | sappiamo | sappiate |
| **scegliere** (선택하다) | scegli | scelga | scegliamo | scegliete |

> **축약형 참고**: *Va'*, *fa'*, *di'*, *da'*, *sta'*는 축약형입니다. 완전형 *vai*, *fai*, *dai*, *stai*도 사용되며 점점 일반화되고 있습니다. 아포스트로피는 축약을 나타내며 악센트가 아닙니다.

### 문맥 속 예시

| 이탈리아어 | 한국어 |
|-----------|--------|
| **Sii** gentile! | 친절하게 해! |
| **Abbi** pazienza! | 인내심을 가져! (직역: 인내를 가져라!) |
| **Va'** / **Vai** a casa! | 집에 가! |
| **Fa'** / **Fai** attenzione! | 주의해! / 조심해! |
| **Di'** la verità! | 진실을 말해! |
| **Da'** / **Dai** una mano a tuo fratello! | 형/동생을 도와줘! |
| **Sta'** / **Stai** zitto! | 조용히 해! |
| **Vieni** qui! | 여기로 와! |
| **Esci** di qui! | 여기서 나가! |

## 4. 부정 명령법

### Tu 형태: NON + 부정사

**tu**의 부정 명령법은 독특합니다: **non** + **부정사**를 사용합니다 (활용형이 아님).

| 긍정 | 부정 |
|------|------|
| Parla! (말해!) | **Non parlare!** (말하지 마!) |
| Scrivi! (써!) | **Non scrivere!** (쓰지 마!) |
| Dormi! (자!) | **Non dormire!** (자지 마!) |
| Va'! (가!) | **Non andare!** (가지 마!) |
| Di'! (말해!) | **Non dire niente!** (아무 말도 하지 마!) |
| Fa'! (해!) | **Non fare così!** (그렇게 하지 마!) |

### Lei / Noi / Voi: NON + 명령법 형태

다른 인칭에서는 일반 명령법 형태 앞에 **non**을 놓으면 됩니다.

| 긍정 | 부정 |
|------|------|
| Parli! (Lei) | **Non parli!** |
| Parliamo! (noi) | **Non parliamo!** |
| Parlate! (voi) | **Non parlate!** |
| Vada! (Lei) | **Non vada!** |

## 5. 명령법에서의 대명사 배치

### 긍정: 대명사가 동사 뒤에 부착

**tu, noi, voi** 긍정 명령법에서 대명사는 동사 끝에 붙습니다.

| 이탈리아어 | 한국어 |
|-----------|--------|
| Scrivimi! (Scrivi + mi) | 나에게 써 줘! |
| Dimmi! (Di' + mi) | 나에게 말해! |
| Dammelo! (Da' + me + lo) | 그걸 나에게 줘! |
| Fallo! (Fa' + lo) | 그것을 해! |
| Chiamalo! (Chiama + lo) | 그에게 전화해! |
| Andiamoci! (Andiamo + ci) | 거기에 갑시다! |
| Portatelo! (Portate + lo) | 그것을 가져와! (복수) |

> **자음 이중화 규칙**: 축약형 *da'*, *di'*, *fa'*, *sta'*, *va'*에서 대명사의 첫 자음이 **이중화**됩니다: da' + mi → **dammi**, di' + lo → **dillo**, fa' + lo → **fallo**, sta' + ci → **stacci**.
> 예외: **gli**는 절대 이중화되지 않습니다: di' + gli → **digli** (*digligli*가 아님).

### 격식체 (Lei): 대명사가 동사 앞에 위치

| 이탈리아어 | 한국어 |
|-----------|--------|
| Mi scriva! | 저에게 써 주세요! (격식) |
| Lo faccia! | 그것을 하세요! (격식) |
| Mi dica! | 말씀하세요! (격식) |
| Glielo dia! | 그것을 그/그녀에게 주세요! (격식) |
| Si accomodi! | 앉으세요! (격식) |

### 부정 Tu: 두 가지 옵션

부정 **tu** 명령에서 대명사는 부착하거나 앞에 놓을 수 있습니다:

| 옵션 1 (부착) | 옵션 2 (앞에) | 한국어 |
|--------------|-------------|--------|
| Non dirlo! | Non lo dire! | 그것을 말하지 마! |
| Non farlo! | Non lo fare! | 그것을 하지 마! |
| Non preoccuparti! | Non ti preoccupare! | 걱정하지 마! |

## 7. Congiuntivo Presente: 구성

Congiuntivo(접속법)는 주관성을 표현합니다: 소망, 의심, 의견, 감정, 불확실성. 항상 **che**로 도입되는 **종속절**에서 나타납니다.

### 규칙 Congiuntivo Presente

| 인칭 | -ARE (parlare) | -ERE (scrivere) | -IRE (dormire) | -IRE -isc- (capire) |
|------|----------------|-----------------|----------------|---------------------|
| che io | parl**i** | scriv**a** | dorm**a** | cap**isca** |
| che tu | parl**i** | scriv**a** | dorm**a** | cap**isca** |
| che lui/lei/Lei | parl**i** | scriv**a** | dorm**a** | cap**isca** |
| che noi | parl**iamo** | scriv**iamo** | dorm**iamo** | cap**iamo** |
| che voi | parl**iate** | scriv**iate** | dorm**iate** | cap**iate** |
| che loro | parl**ino** | scriv**ano** | dorm**ano** | cap**iscano** |

> **핵심 관찰**: -ARE 동사의 경우 단수 형태가 모두 **-i**로 끝나고 **loro** 형태는 **-ino**로 끝납니다. -ERE/-IRE 동사의 경우 단수 형태가 모두 **-a**로 끝나고 **loro**는 **-ano**로 끝납니다. **io, tu, lui/lei** 형태는 **동일**합니다 — 문맥이나 대명사로 구별합니다.

### 접속법 vs. 직설법: 예시

| 직설법 (사실) | 접속법 (의견/소망) |
|-------------|------------------|
| So che Marco **parla** italiano. (Marco가 이탈리아어를 한다는 것을 안다.) | Penso che Marco **parli** italiano. (Marco가 이탈리아어를 한다고 생각한다.) |
| È vero che Maria **viene**. (Maria가 오는 것은 사실이다.) | Spero che Maria **venga**. (Maria가 오기를 바란다.) |

## 9. 접속법 트리거: 소망과 바람

주절에서 소망, 바람, 선호를 표현하는 경우 종속절에 접속법이 필요합니다.

| 트리거 표현 | 예시 |
|------------|------|
| **volere che** (~을 원하다) | Voglio che tu **venga** alla festa. (네가 파티에 오기를 원한다.) |
| **desiderare che** (~을 바라다) | Desidero che tutto **vada** bene. (모든 것이 잘 되기를 바란다.) |
| **sperare che** (~을 희망하다) | Spero che **faccia** bel tempo. (날씨가 좋기를 바란다.) |
| **preferire che** (~을 선호하다) | Preferisco che tu **stia** qui. (네가 여기 있는 것을 선호한다.) |
| **augurare che** (~을 기원하다) | Mi auguro che **sia** vero. (그것이 사실이기를 바란다.) |
| **aspettare che** (~을 기다리다) | Aspetto che lui **finisca**. (그가 끝내기를 기다리고 있다.) |

> **중요**: 두 절의 주어가 **같을** 때는 접속법 대신 **부정사**를 사용합니다:
> - Voglio **andare**. (가고 싶다.) — 동일 주어
> - Voglio che tu **vada**. (네가 가기를 원한다.) — 다른 주어

## 10. 접속법 트리거: 의심과 의견

| 트리거 표현 | 예시 |
|------------|------|
| **pensare che** (~라고 생각하다) | Penso che **sia** una buona idea. (좋은 생각이라고 생각한다.) |
| **credere che** (~라고 믿다) | Credo che **abbia** ragione. (그가 맞다고 믿는다.) |
| **dubitare che** (~을 의심하다) | Dubito che **venga**. (그가 올지 의심한다.) |
| **non essere sicuro/a che** | Non sono sicuro che **possa** venire. (그가 올 수 있는지 확실하지 않다.) |
| **immaginare che** (~라고 상상하다) | Immagino che **sia** difficile. (어려울 거라고 상상한다.) |
| **supporre che** (~라고 추정하다) | Suppongo che **abbia** avuto problemi. (문제가 있었을 거라고 추정한다.) |
| **sembrare che** (~인 것 같다) | Sembra che **piova**. (비가 올 것 같다.) |
| **parere che** (~인 듯하다) | Pare che **sia** partito. (떠난 듯하다.) |

> **참고**: *Sapere che* (알다)와 *essere sicuro che* (확신하다)는 확신을 표현하므로 의심이 아닌 **직설법**을 취합니다:
> - So che **è** vero. (그것이 사실임을 안다.) — 직설법
> - Non so se **sia** vero. (그것이 사실인지 모른다.) — 접속법

## 11. 접속법 트리거: 감정

| 트리거 표현 | 예시 |
|------------|------|
| **essere felice/contento/a che** | Sono felice che tu **sia** qui. (네가 여기 있어서 행복하다.) |
| **essere triste che** | È triste che lei **parta**. (그녀가 떠나는 것이 슬프다.) |
| **avere paura che** | Ho paura che **succeda** qualcosa. (무슨 일이 일어날까 두렵다.) |
| **dispiacere che** | Mi dispiace che non **possa** venire. (그가 올 수 없어서 유감이다.) |
| **essere sorpreso/a che** | Sono sorpreso che non lo **sappia**. (그가 모른다는 것에 놀랐다.) |
| **essere arrabbiato/a che** | È arrabbiato che non **l'abbia** chiamato. (내가 전화하지 않아서 화가 났다.) |
| **piacere che** | Mi piace che tu **stia** bene. (네가 잘 지내서 기쁘다.) |
| **vergognarsi che** | Si vergogna che tutti lo **sappiano**. (모두가 아는 것이 부끄럽다.) |

### 접속법을 요구하는 비인칭 표현

| 표현 | 예시 |
|------|------|
| **È necessario che** | È necessario che tu **venga**. (네가 와야 한다.) |
| **È importante che** | È importante che **capiscano**. (그들이 이해하는 것이 중요하다.) |
| **È possibile che** | È possibile che **piova**. (비가 올 수 있다.) |
| **È impossibile che** | È impossibile che **sia** vero. (그것이 사실일 수 없다.) |
| **È probabile che** | È probabile che **arrivi** in ritardo. (그가 늦게 도착할 가능성이 높다.) |
| **È meglio che** | È meglio che tu **stia** zitto. (네가 조용히 있는 게 낫다.) |
| **È strano che** | È strano che non **risponda**. (그가 대답하지 않는 것이 이상하다.) |
| **Bisogna che** | Bisogna che **studiamo** di più. (우리가 더 공부할 필요가 있다.) |
| **Basta che** | Basta che tu **faccia** il tuo dovere. (네 의무를 다하면 충분하다.) |

### 연습 3: 대명사 배치

### 연습 4: 직설법인가 접속법인가?

올바른 형태를 선택하세요.

1. So che Marco (parlare — parla / parli) italiano.
2. Penso che Maria (essere — è / sia) intelligente.
3. È vero che loro (venire — vengono / vengano) domani.
4. Spero che tu (stare — stai / stia) bene.
5. Voglio che voi (capire — capite / capiate) la situazione.
6. È possibile che (piovere — piove / piova) domani.
7. Dubito che lui (sapere — sa / sappia) la risposta.
8. Sono sicuro che tu (avere — hai / abbia) ragione.

<details>
<summary>정답</summary>

1. **parla** (직설법 — *so che* = 확신)
2. **sia** (접속법 — *penso che* = 의견)
3. **vengono** (직설법 — *è vero che* = 사실)
4. **stia** (접속법 — *spero che* = 소망)
5. **capiate** (접속법 — *voglio che* = 바람)
6. **piova** (접속법 — *è possibile che* = 가능성)
7. **sappia** (접속법 — *dubito che* = 의심)
8. **hai** (직설법 — *sono sicuro che* = 확신)

</details>

### 연습 5: 접속법으로 완성하기

1. Spero che voi (venire) ________ alla festa.

4. Voglio che Marco (dire) ________ la verità.

6. Ho paura che non (potere — loro) ________ venire.
7. È meglio che noi (andare) ________ via.
8. Credo che lui non (sapere) ________ niente.

<details>
<summary>정답</summary>

1. **veniate**
2. **faccia**
3. **abbia**
4. **dica**
5. **sia** / **ci sia**
6. **possano**
7. **andiamo**
8. **sappia**

</details>
