이탈리아어 축소사는 작음, 애정, 친밀감을 표현한다. 일상 대화에서 매우 흔하게 사용된다.

### 5.4 기타 축소사

| 접미사 | 예시 | 의미 |
|--------|---------|---------|
| -uccio/-uccia | casa → casuccia | 불쌍한 작은 집 (약간 비하적 + 애정) |
| -uzzo/-uzza | pietra → pietruzza | 작은 돌 / 조약돌 |
| -icino/-icina | cuore → cuoricino | 작은 하트 |
| -olino/-olina | topo → topolino | 작은 쥐 (미키 마우스이기도 함) |

> **중요**: 축소사는 단어의 성을 바꿀 수 있다. 예를 들어, *la finestra* (여성) → *il finestrino* (남성, 차 창문). 축소사와 함께 항상 관사를 배워야 한다.

## 6. 확대사 (-one/-ona)

확대사는 크기나 강도를 표현한다.

| 기본 단어 | 확대사 | 의미 |
|-----------|-------------|---------|
| libro (책) | librone | 큰 책 |
| uomo (남자) | omone | 큰 남자 |
| donna (여자) | donnone | 큰/위풍당당한 여자 |
| porta (문) | portone | 정문 / 큰 문 |
| minestra (수프) | minestrone | 걸쭉한 야채 수프 |
| festa (파티) | festone | 큰 파티 / 화환 |
| naso (코) | nasone | 큰 코 |
| pigro (게으른) | pigrone | 매우 게으른 사람 |
| chiacchiera (수다) | chiacchierone | 수다쟁이 |

> **성 참고**: *-one* 확대사는 기본 단어가 여성이더라도 항상 남성이다:
> - la donna (여성) → il donnone (남성)
> - la porta (여성) → il portone (남성)
> - la palla (여성) → il pallone (남성) -- 축구공!

## 7. 비하사 (-accio/-accia)

비하사는 부정적 특성 -- 추함, 불쾌함, 열악한 품질을 표현한다.

| 기본 단어 | 비하사 | 의미 |

| ragazzo (소년) | ragazzaccio | 나쁜/장난꾸러기 소년 |

### 혼합 접미사를 통한 뉘앙스

| 접미사 조합 | 예시 | 뉘앙스 |
|-------------|---------|--------|

| -etto + 가벼움 | *viaggio* → *viaggetto* | 짧은/빠른 여행 |

## 8. 단어 형성: 접두사

### 8.1 부정과 역전 접두사

| 접두사 | 의미 | 예시 |
|--------|---------|---------|
| **in-/im-** | ~이 아닌, 비- | felice → **in**felice (불행한), possibile → **im**possibile (불가능한) |
| **dis-** | 비-, 탈- | fare → **dis**fare (풀다), ordine → **dis**ordine (무질서) |
| **s-** | 비-, 탈-, 해제- | fortunato → **s**fortunato (불운한), caricare → **s**caricare (내리다/다운로드하다) |
| **a-** | ~없이 | normale → **a**normale (비정상적인), morale → **a**morale (비도덕적인) |

### 8.2 강화 접두사

| 접두사 | 의미 | 예시 |
|--------|---------|---------|
| **ri-** | 재-, 다시 | fare → **ri**fare (다시 하다), leggere → **ri**leggere (다시 읽다) |
| **stra-** | 초-, 극히 | ordinario → **stra**ordinario (비범한), cotto → **stra**cotto (너무 익힌) |
| **super-** | 초-, 위 | mercato → **super**mercato (슈퍼마켓), uomo → **super**uomo (초인) |
| **ultra-** | 초-, 넘어서 | moderno → **ultra**moderno (초현대적인) |
| **iper-** | 과-, 초과 | attivo → **iper**attivo (과잉 활동적인), mercato → **iper**mercato (대형마트) |

### 8.3 공간 및 시간 접두사

| 접두사 | 의미 | 예시 |
|--------|---------|---------|
| **pre-** | ~전 | vedere → **pre**vedere (예견하다), guerra → **pre**guerra (전쟁 전) |
| **post-** | ~후 | guerra → **post**guerra (전쟁 후), moderno → **post**moderno (포스트모던) |
| **sotto-** | ~아래 | terra → **sotto**terra (지하), titolo → **sotto**titolo (자막) |
| **sopra-** | ~위, ~이상 | tutto → **sopra**ttutto (무엇보다), vivvere → **sopra**vvivere (살아남다) |
| **inter-** | ~사이 | nazionale → **inter**nazionale (국제적인) |
| **extra-** | ~밖 | terrestre → **extra**terrestre (외계의) |

## 9. 단어 형성: 접미사

### 9.1 명사 형성 접미사

| 접미사 | 의미 | 예시 |
|--------|---------|---------|
| **-zione/-sione** | 행위, 결과 | informare → informa**zione** (정보), decidere → deci**sione** (결정) |
| **-mento** | 행위의 결과 | trattare → tratta**mento** (치료), inquinare → inquina**mento** (오염) |
| **-tore/-trice** | 행위자 (하는 사람) | lavorare → lavora**tore** (노동자, 남), lavorare → lavora**trice** (노동자, 여) |
| **-ista** | 전문가/지지자 | giornale → giornal**ista** (기자), arte → art**ista** (예술가) |
| **-eria** | 장소/상점 | libro → libr**eria** (서점), pasta → past**eria** (파스타/과자 가게) |
| **-ura** | 결과 / 추상적 특성 | aprire → apert**ura** (개방), colto → colt**ura** (문화) |

### 9.2 형용사 형성 접미사

| 접미사 | 의미 | 예시 |
|--------|---------|---------|
| **-oso/-osa** | ~이 가득한 | fama → fam**oso** (유명한), pericolo → pericol**oso** (위험한) |
| **-bile** | ~할 수 있는 | mangiare → mangia**bile** (먹을 수 있는), incredere → incredi**bile** (믿을 수 없는) |
| **-ale** | ~에 관한 | nazione → nazion**ale** (국가의), natura → natur**ale** (자연의) |
| **-ivo/-iva** | ~하는 경향의 | creare → creat**ivo** (창의적인), sportare → sport**ivo** (스포츠적인) |
| **-esco/-esca** | ~풍의 | Dante → dant**esco** (단테풍의), pittore → pittor**esco** (그림 같은) |

## 10. 거짓 친구: 이탈리아어-영어

겉모습은 비슷하지만 의미가 다른 단어들:

| 이탈리아어 단어 | 닮은 영어 | 실제 의미 | 영어 의미의 이탈리아어 |
|-------------|-------------------|---------------|---------------------------|
| **attualmente** | actually | 현재, 지금 | in realtà (사실은) |
| **camera** | camera | 방 / 침실 | macchina fotografica (카메라) |
| **confrontare** | to confront | 비교하다 | affrontare (대면하다) |
| **eventualmente** | eventually | 혹시, 필요하다면 | alla fine (결국) |
| **fattoria** | factory | 농장 | fabbrica (공장) |
| **libreria** | library | 서점 | biblioteca (도서관) |
| **magazzino** | magazine | 창고 / 저장소 | rivista (잡지) |
| **morbido** | morbid | 부드러운, 유연한 | morboso (병적인) |
| **parente** | parent | 친척 | genitore (부모) |
| **pretendere** | to pretend | 요구하다, 기대하다 | fingere (~인 척하다) |
| **sensibile** | sensible | 민감한 | sensato (분별 있는) |
| **simpatico** | sympathetic | 상냥한, 호감 가는 | comprensivo (동정적인) |
| **stampare** | to stamp | 인쇄하다 | timbrare (도장 찍다) |
| **bravo** | brave | 잘하는, 능숙한 | coraggioso (용감한) |
| **caldo** | cold | 뜨거운, 따뜻한 | freddo (차가운) |
| **firma** | firm (회사) | 서명 | ditta (회사) |
| **largo** | large | 넓은 | grande (큰) |
| **rumore** | rumor | 소음 | voce / diceria (소문) |
| **assistere** | to assist | 참석하다 / 목격하다 | aiutare (돕다) |

## 11. 거짓 친구: 이탈리아어-스페인어

스페인어도 아는 학습자를 위한 흔한 함정들:

| 이탈리아어 | 스페인어 유사어 | 이탈리아어 의미 | 스페인어 의미 |
|---------|-------------------|----------------|-----------------|
| **burro** | burro (당나귀) | 버터 | 당나귀 (이탈리아어로 asino) |
| **caldo** | caldo (국물) | 뜨거운/따뜻한 | 국물 (이탈리아어로 brodo) |
| **gamba** | gamba (새우) | 다리 | 새우 (이탈리아어로 gambero) |
| **guardia** | guardia (경비원) | 경비원 (동일) | 경비원 (동일 -- 하지만 *guardare* = 보다, 지키다가 아님) |
| **largo** | largo (긴) | 넓은 | 긴 (이탈리아어로 lungo) |
| **salire** | salir (나가다) | 올라가다 | 나가다 (이탈리아어로 uscire) |
| **subire** | subir (올라가다) | 겪다/당하다 | 올라가다 (이탈리아어로 salire) |
| **aceto** | aceite (기름) | 식초 | 기름 (이탈리아어로 olio) |
| **guardare** | guardar (보관하다) | 보다 / 시청하다 | 보관하다 (이탈리아어로 tenere) |
| **pronto** | pronto (곧) | 준비된 / 여보세요 (전화) | 곧 (이탈리아어로 presto) |
| **tienda** | tienda (가게, 스페인어) | tenda (텐트 / 커튼) | 가게 = 이탈리아어로 negozio |
| **embarazada** | (스페인어: 임신한) | -- | *imbarazzata* = 당황한 (임신한이 아님: *incinta*) |

## 12. 연습 문제

### 연습 1: 기술 어휘

다음을 이탈리아어로 번역하시오.

1. 스마트폰에 앱을 다운로드해야 해요. → ________
2. 첨부 파일이 있는 이메일을 보내줄 수 있어요? → ________
3. Wi-Fi 연결이 매우 느려요. → ________
4. 비밀번호를 잊어버렸어요. → ________
5. 소프트웨어를 업데이트해 주세요. → ________

<details>
<summary>정답</summary>

1. **Devo scaricare l'app sul mio smartphone/cellulare.**
2. **Puoi mandarmi l'email con l'allegato?**
3. **La connessione Wi-Fi è molto lenta.**
4. **Ho dimenticato la mia password.** / **Ho dimenticato la parola d'ordine.**
5. **Per favore, aggiorna il software.**

</details>

### 연습 2: 축소사, 확대사, 비하사

표시된 접미사를 사용하여 올바른 파생어를 만드시오.

1. casa + (-etta) → ________ (의미: ________)
2. libro + (-one) → ________ (의미: ________)
3. giorno + (-accio) → ________ (의미: ________)
4. gatto + (-ino) → ________ (의미: ________)
5. parola + (-accia) → ________ (의미: ________)
6. tavolo + (-ino) → ________ (의미: ________)
7. donna + (-one) → ________ (의미: ________)
8. storia + (-ella) → ________ (의미: ________)
9. ragazzo + (-ino) → ________ (의미: ________)
10. tempo + (-accio) → ________ (의미: ________)

<details>
<summary>정답</summary>

1. **casetta** (작은 집 / 오두막)
2. **librone** (큰 책)
3. **giornaccia** (끔찍한 날)
4. **gattino** (새끼 고양이)
5. **parolaccia** (욕설 / 나쁜 말)
6. **tavolino** (작은 테이블 / 카페 테이블)
7. **donnone** (큰/위풍당당한 여자 -- 참고: 남성 관사 *il* 사용)
8. **storiella** (짧은 이야기 / 일화)
9. **ragazzino** (어린 소년)
10. **tempaccio** (끔찍한 날씨)

</details>

### 연습 3: 거짓 친구

올바른 이탈리아어 번역을 고르시오. 거짓 친구에 주의하세요!

1. "사실, 동의하지 않아요."
   - a) Attualmente, non sono d'accordo.
   - b) In realtà, non sono d'accordo.

2. "도서관에 갔어요."
   - a) Sono andato alla libreria.
   - b) Sono andato alla biblioteca.

3. "내 부모님이 내일 오세요."
   - a) I miei parenti vengono domani.
   - b) I miei genitori vengono domani.

4. "공장은 일요일에 문을 닫아요."
   - a) La fattoria è chiusa la domenica.
   - b) La fabbrica è chiusa la domenica.

5. "그녀는 매우 민감해요."
   - a) È molto sensibile.
   - b) È molto sensata.

6. "오늘 매우 추워요." (기온)
   - a) È molto caldo oggi.
   - b) Fa molto freddo oggi.

<details>
<summary>정답</summary>

1. **b) In realtà, non sono d'accordo.** (*Attualmente*는 "사실"이 아니라 "현재"를 의미한다.)
2. **b) Sono andato alla biblioteca.** (*Libreria*는 "도서관"이 아니라 "서점"을 의미한다.)
3. **b) I miei genitori vengono domani.** (*Parenti*는 "부모"가 아니라 "친척"을 의미한다.)
4. **b) La fabbrica è chiusa la domenica.** (*Fattoria*는 "공장"이 아니라 "농장"을 의미한다.)
5. **a) È molto sensibile.** (이것은 올바른 대응이다 -- *sensibile*는 실제로 "민감한"을 의미한다. 거짓 친구는 *sensibile* ≠ "sensible(분별 있는)"; "분별 있는" = *sensato/a*.)
6. **b) Fa molto freddo oggi.** (*Caldo*는 "차가운"이 아니라 "뜨거운"을 의미한다!)

</details>

### 연습 4: 접두사 만들기

올바른 접두사를 추가하여 반대 또는 수정된 의미를 만드시오.

1. ______felice (불행한) → ________
2. ______caricare (내리다 / 다운로드하다) → ________
3. ______fare (다시 하다) → ________

5. ______possibile (불가능한) → ________
6. ______fortunato (불운한) → ________
7. ______vedere (예견하다) → ________
8. ______titolo (자막) → ________

<details>
<summary>정답</summary>

1. **in**felice
2. **s**caricare
3. **ri**fare
4. **stra**ordinario
5. **im**possibile
6. **s**fortunato
7. **pre**vedere
8. **sotto**titolo

</details>

### 연습 5: 환경 어휘

빈칸에 올바른 이탈리아어 단어를 채우시오.

5. La ________ è essenziale per proteggere l'ambiente. (지속가능성은 환경을 보호하는 데 필수적이다.)

2. **raccolta differenziata**
3. **energia solare**
4. **emissioni**
5. **sostenibilità**

</details>

### 연습 6: 단어 형성

접미사를 식별하고 단어의 형성을 설명하시오.

1. lavoratore = _________ + _________ (의미: ________)
2. libreria = _________ + _________ (의미: ________)
3. giornalista = _________ + _________ (의미: ________)
4. pericoloso = _________ + _________ (의미: ________)
5. informazione = _________ + _________ (의미: ________)

<details>
<summary>정답</summary>

1. **lavorare** + **-tore** = 노동자 (일하는 사람)
2. **libro** + **-eria** = 서점 (책의 장소)
3. **giornale** + **-ista** = 기자 (신문 전문가)
4. **pericolo** + **-oso** = 위험한 (위험이 가득한)
5. **informare** + **-zione** = 정보 (알림의 결과)

</details>

### 연습 7: 이탈리아어-스페인어 거짓 친구

스페인어 화자를 위해: 올바른 이탈리아어 단어를 고르시오.

1. 이탈리아어로 "버터":
   - a) burro  b) manteca  c) mantequilla

2. 이탈리아어로 "뜨거운":
   - a) caliente  b) caldo  c) calor

3. 이탈리아어로 "올라가다":
   - a) subire  b) salire  c) subir

4. 이탈리아어로 "식초":
   - a) aceite  b) aceto  c) vinagre

5. 이탈리아어로 "준비된" 또는 "여보세요 (전화)":
   - a) presto  b) pronto  c) listo

<details>
<summary>정답</summary>

1. **a) burro** (이탈리아어에서 *burro* = 버터, 당나귀가 아니다!)
2. **b) caldo** (이탈리아어에서 *caldo* = 뜨거운/따뜻한, 스페인어처럼 국물이 아니다)
3. **b) salire** (이탈리아어에서 *salire* = 올라가다; *subire* = 겪다/당하다)
4. **b) aceto** (이탈리아어에서 *aceto* = 식초; 스페인어 *aceite*는 기름 = *olio*)
5. **b) pronto** (이탈리아어에서 *pronto* = 준비된 또는 전화 인사; *presto* = 곧)

</details>
