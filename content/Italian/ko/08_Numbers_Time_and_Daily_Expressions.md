## 2. 기수 31-1,000,000

### 31-99: 십의 자리 + 일의 자리 (한 단어)

| 숫자 | 이탈리아어 |
|--------|---------|
| 31 | trent**uno** |
| 38 | trent**otto** |
| 42 | quaranta**due** |
| 55 | cinquanta**cinque** |
| 63 | sessanta**tré** |
| 77 | settanta**sette** |
| 88 | ottant**otto** |
| 99 | novanta**nove** |

> **규칙**: *uno*나 *otto*를 더할 때 십의 자리 단어의 마지막 모음이 탈락합니다: trent**a** + uno = trent**uno**, quarant**a** + otto = quarant**otto**.

### 십의 자리

| 30 | trenta | 70 | settanta |
|----|--------|----|----------|
| 40 | quaranta | 80 | ottanta |
| 50 | cinquanta | 90 | novanta |
| 60 | sessanta | | |

### 백의 자리

| 숫자 | 이탈리아어 |
|--------|---------|
| 100 | cento |
| 200 | duecento |
| 300 | trecento |
| 400 | quattrocento |
| 500 | cinquecento |
| 600 | seicento |
| 700 | settecento |
| 800 | ottocento |
| 900 | novecento |

> **참고**: 스페인어와 달리 이탈리아어의 백의 자리는 성에 따라 변하지 않습니다. 남성이든 여성이든 *duecento*: *duecento persone*, *duecento libri*.

### 백의 자리 결합

```
101 = centouno (또는 centuno)
115 = centoquindici
200 = duecento
350 = trecentocinquanta
999 = novecentonovantanove

※ 999까지는 모두 한 단어로 씁니다
※ Cento는 합성어에서 "centi"로 변하지 않습니다
```

### 큰 숫자

| 숫자 | 이탈리아어 |
|--------|---------|
| 1,000 | mille |
| 2,000 | duemila |
| 10,000 | diecimila |
| 100,000 | centomila |
| 1,000,000 | un milione (di + 명사) |
| 2,000,000 | due milioni (di + 명사) |
| 1,000,000,000 | un miliardo (di + 명사) |

```
주요 사항:
  mille → mila (복수): duemila, tremila, diecimila
  milione/miliardo는 명사 → 다른 명사 앞에 "di"가 필요:
    un milione di euro (백만 유로)
    tre milioni di persone (삼백만 명)
    un miliardo di dollari (십억 달러)

  ※ 999까지의 숫자는 한 단어로 씁니다:
    cinquecentoventitré (523)
  ※ mille 뒤에도 한 단어로 이어집니다:
    milleduecentotrenta (1,230)
```

### 형용사로서의 UNO

*uno*가 명사 앞에 올 때 관사 규칙을 따릅니다:
- **un** libro (책 한 권 — 남. 자음 앞)
- **uno** studente (학생 한 명 — 남. s+자음 앞)
- **una** casa (집 한 채 — 여. 자음 앞)
- **un'**amica (친구 한 명 — 여. 모음 앞)

합성 숫자에서 *-uno*는 명사 앞에서 -o가 선택적으로 탈락할 수 있습니다:
- ventun libri 또는 ventuno libri (책 21권)

## 3. 서수

| 숫자 | 남성 | 여성 | 약어 |
|--------|-----------|----------|-------------|
| 1번째 | primo | prima | 1° / 1ª |
| 2번째 | secondo | seconda | 2° / 2ª |
| 3번째 | terzo | terza | 3° / 3ª |
| 4번째 | quarto | quarta | 4° |
| 5번째 | quinto | quinta | 5° |
| 6번째 | sesto | sesta | 6° |
| 7번째 | settimo | settima | 7° |
| 8번째 | ottavo | ottava | 8° |
| 9번째 | nono | nona | 9° |
| 10번째 | decimo | decima | 10° |

> 11번째부터는 기수에 **-esimo**를 추가합니다 (마지막 모음 탈락):
> undici → undic**esimo** (11번째), dodici → dodic**esimo** (12번째), venti → vent**esimo** (20번째)

**사용법:**
- 서수는 일반 형용사처럼 성과 수에 일치합니다
- 항상 명사 **앞에** 위치: *il primo giorno* (첫째 날)
- 세기에 사용: *il Quattrocento* = 1400년대 (15세기)
- 교황/왕에 사용: *Giovanni Paolo Secondo* (요한 바오로 2세)

> **이탈리아어 세기 이름** (문화 어휘):
> il Duecento (1200년대), il Trecento (1300년대), il Quattrocento (1400년대),
> il Cinquecento (1500년대), il Seicento (1600년대), il Settecento (1700년대),
> l'Ottocento (1800년대), il Novecento (1900년대)

## 4. 시간 말하기

### 4.1 기본 구조

```
┌──────────────────────────────────────────────────────────────┐
│              이탈리아어 시간 표현                                 │
│                                                              │
│  1:00  È l'una.            (단수: "è l'una")                 │
│  2:00  Sono le due.        (복수: "sono le")                  │
│  3:15  Sono le tre e un quarto.                             │
│        (= Sono le tre e quindici.)                          │
│  4:30  Sono le quattro e mezza.                             │
│        (= Sono le quattro e trenta.)                        │
│  5:45  Sono le sei meno un quarto.                          │
│        (= Sono le cinque e quarantacinque.)                 │
│  6:10  Sono le sei e dieci.                                 │
│  7:50  Sono le otto meno dieci.                             │
│  12:00 È mezzogiorno. (정오)                                  │
│  00:00 È mezzanotte. (자정)                                   │
│                                                              │
│  ※ 1시, 정오, 자정에는 "È"; 나머지는 모두 "Sono"              │
│  ※ "Meno" = 빼기 (다음 시간 전 분)                            │
│  ※ 격식/문어에서는 24시간제가 일반적                             │
└──────────────────────────────────────────────────────────────┘
```

### 4.2 시간대

| 이탈리아어 | 한국어 |
|---------|---------|
| di mattina / della mattina | 아침에 (오전) |
| del pomeriggio | 오후에 (~1-6시) |
| di sera / della sera | 저녁에 (~6-10시) |
| di notte / della notte | 밤에 (~10-6시) |

**예시:**
- Sono le otto **di mattina**. (오전 8시이다.)
- Sono le tre **del pomeriggio**. (오후 3시이다.)
- È l'una **del pomeriggio**. (오후 1시이다.)
- Sono le dieci **di sera**. (저녁 10시이다.)

### 4.3 묻고 답하기

- **Che ore sono?** / **Che ora è?** (몇 시인가요?)
- **A che ora...?** (몇 시에...?)
  - A che ora comincia la lezione? — **All'una.** / **Alle nove.** (1시에. / 9시에.)

> **참고**: "~시에"는 *a + 관사*를 사용: **alle** due (2시에), **alle** tre (3시에), **all'**una (1시에), **a** mezzogiorno (정오에).

## 5. 요일

| 이탈리아어 | 한국어 | 비고 |
|---------|---------|------|
| lunedì | 월요일 | (Luna — 달에서) |
| martedì | 화요일 | (Marte — 화성에서) |
| mercoledì | 수요일 | (Mercurio — 수성에서) |
| giovedì | 목요일 | (Giove — 목성에서) |
| venerdì | 금요일 | (Venere — 금성에서) |
| sabato | 토요일 | |
| domenica | 일요일 | |

> 이탈리아어에서 요일은 **대문자로 시작하지 않습니다** (문장 시작 제외).
> *sabato*와 *domenica*를 제외한 모든 요일은 **-dì**로 끝납니다 (라틴어 *dies* = 날에서).
> 습관적 행동에는 **정관사**를 사용: **il** lunedì (매주 월요일마다), **la** domenica (매주 일요일마다).
>
> - **Lunedì vado in palestra.** (월요일에 헬스장에 간다.) — 특정
> - **Il lunedì vado in palestra.** (매주 월요일마다 헬스장에 간다.) — 습관

## 6. 월과 계절

### 월

| 이탈리아어 | 한국어 | 이탈리아어 | 한국어 |
|---------|---------|---------|---------|
| gennaio | 1월 | luglio | 7월 |
| febbraio | 2월 | agosto | 8월 |
| marzo | 3월 | settembre | 9월 |
| aprile | 4월 | ottobre | 10월 |
| maggio | 5월 | novembre | 11월 |
| giugno | 6월 | dicembre | 12월 |

> 이탈리아어에서 월은 **대문자로 시작하지 않습니다**.
> 월과 함께 **in** 또는 **a**를 사용: *in gennaio* 또는 *a gennaio* (1월에).

### 계절

| 이탈리아어 | 한국어 |
|---------|---------|
| la primavera | 봄 |
| l'estate (여.) | 여름 |
| l'autunno (남.) | 가을 |
| l'inverno (남.) | 겨울 |

> 계절과 함께 **in**을 사용: *in primavera* (봄에), *in estate* (여름에), *in autunno* (가을에), *in inverno* (겨울에).
> 관사를 사용하는 대안: *d'estate* (여름에), *d'inverno* (겨울에).

## 7. 날짜 표현

### 7.1 형식

```
관사 + 숫자 + 월 + 년도

il 15 marzo 2026
(2026년 3월 15일)

il 25 dicembre
(12월 25일)
```

- 매월 1일에는 **primo** (서수)를 사용:
  - il **primo** gennaio (1월 1일)
  - 하지만 다른 날짜에는 기수를 사용: il **due** gennaio, il **tre** febbraio

### 7.2 날짜 묻기

- **Che giorno è oggi?** (오늘 무슨 요일인가요?)
- **Quanti ne abbiamo oggi?** (오늘 며칠인가요? — 직역: 오늘 몇 개나 가지고 있나요?)
- **Oggi è il 8 marzo 2026.** (오늘은 2026년 3월 8일이다.)
- **Oggi è lunedì, il 8 marzo.** (오늘은 월요일, 3월 8일이다.)

### 7.3 연도

```
2026: duemilaventisei
1999: millenovecentonovantanove
2000: duemila (l'anno duemila)
1492: millequattrocentonovantadue

※ 연도는 일반 숫자처럼 읽습니다 (영어처럼 "nineteen ninety-nine"으로 나누지 않음)
※ "~년에" = nell'anno... 또는 nel + 연도
   nel 2026 (2026년에)
   nel 1990 (1990년에)
```

### 7.4 생년월일

```
Quand'è il tuo compleanno?    → 생일이 언제인가요?
Il mio compleanno è il 5 luglio. → 내 생일은 7월 5일이다.
Sono nato/a il 15 marzo 1995. → 1995년 3월 15일에 태어났다.
```

## 9. 빈도 표현

| 이탈리아어 | 한국어 |
|---------|---------|
| sempre | 항상 |
| quasi sempre | 거의 항상 |
| tutti i giorni / ogni giorno | 매일 |
| spesso | 자주 |
| di solito / generalmente | 보통 |
| a volte / qualche volta | 때때로 |
| di tanto in tanto | 이따금 |
| raramente / di rado | 드물게 |
| quasi mai | 거의 ~않다 |
| mai | 절대 ~않다 |

**시간과 함께하는 빈도:**
- tre volte **al** giorno (하루에 세 번)
- una volta **alla** settimana (일주일에 한 번)
- due volte **al** mese (한 달에 두 번)
- ogni anno / una volta **all'**anno (매년 / 일 년에 한 번)
- ogni due settimane (2주마다)

## 10. 일상생활 실용 표현

### 10.1 쇼핑

```
Quanto costa?               → 얼마인가요?
Quanto costano?             → (복수) 얼마인가요?
Quanto viene?               → 얼마예요? (비격식)
È troppo caro.              → 너무 비싸요.
Posso provarlo/a?           → 입어봐도 될까요?
C'è uno sconto?             → 할인이 있나요?
Posso pagare con la carta?  → 카드로 결제할 수 있나요?
Mi può fare lo scontrino?   → 영수증 주시겠어요?
Vorrei... / Mi dia...       → ~을 원합니다... / ~을 주세요...
```

### 10.2 레스토랑에서

```
Un tavolo per due, per favore.  → 두 명 자리 부탁합니다.
Il menù, per favore.           → 메뉴 주세요.
Cosa mi consiglia?             → 추천해 주시겠어요?
Prendo / Per me...             → 저는 ~으로 할게요 / 저는...
Il conto, per favore.          → 계산서 부탁합니다.
È incluso il coperto?          → 자릿세가 포함되어 있나요?
Dov'è il bagno?                → 화장실이 어디인가요?
```

### 10.3 길 묻기

```
Dov'è...?                    → ~이 어디인가요?
Come si arriva a...?          → ~에 어떻게 가나요?
È lontano / vicino?           → 멀어요 / 가까워요?
Vada dritto.                  → 직진하세요.
Giri a destra / a sinistra.   → 오른쪽 / 왼쪽으로 도세요.
Al semaforo...                → 신호등에서...
All'incrocio...               → 교차로에서...
È a cinque minuti a piedi.    → 도보로 5분 거리입니다.
Mi sono perso/a.              → 길을 잃었어요.
```

### 10.4 날씨

```
Che tempo fa?                 → 날씨가 어때요?
Fa caldo / freddo.            → 더워요 / 추워요.
Fa bel tempo / brutto tempo.  → 날씨가 좋아요 / 나빠요.
Piove. / Sta piovendo.        → 비가 와요.
Nevica. / Sta nevicando.      → 눈이 와요.
Tira vento.                   → 바람이 불어요.
C'è il sole.                  → 해가 나요.
È nuvoloso.                   → 흐려요.
Che caldo! / Che freddo!      → 정말 덥다! / 정말 춥다!
```

┌──────────────────────────────────────────────────────────────┐
│   숫자: 0-10 (암기), 11-19 (고유/dici-),                      │
│   20-99 (십의 자리+단위를 한 단어로, 1/8 앞에서 모음 탈락)        │
│   100=cento, 1000=mille/mila, 1M=milione(di), 1B=miliardo   │
│                                                              │
│   시간: È l'una / Sono le [due-dodici]                       │

│   A che ora? → Alle [시] / All'una                           │
│                                                              │
│   요일: lunedì-venerdì (-dì로 끝남), sabato, domenica        │
│   대문자 아님. Il lunedì = 매주 월요일마다.                      │
│                                                              │
│   날짜: il + 숫자 + 월 (+ 연도)                                │

│                                                              │
│   서수: 1°-10° (고유), 11°+ = 기수 + -esimo                  │
│                                                              │
│   핵심 표현: Quanto costa? Dov'è...? Che tempo fa?           │
