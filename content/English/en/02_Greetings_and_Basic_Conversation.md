# 02. Greetings and Basic Conversation

## Learning Objectives
- Learn common English greetings and farewells for different situations
- Master self-introductions in English
- Understand subject pronouns (I/you/he/she/it/we/they)
- Conjugate the BE verb (am/is/are) correctly
- Form basic SVO (Subject-Verb-Object) sentences
- Understand that English has NO honorific system like Korean (존댓말/반말)

## 1. Greetings

### 1.1 Basic Greetings

```
┌─────────────────────────────────────────────────────────────────┐
│                    English Greetings                             │
│                                                                 │
│   Formal / Neutral:                                             │
│   "Hello."             /həˈloʊ/       안녕하세요                  │
│   "Good morning."      /gʊd ˈmɔːrnɪŋ/ 좋은 아침이에요             │
│   "Good afternoon."    /gʊd æftərˈnuːn/ 좋은 오후예요             │
│   "Good evening."      /gʊd ˈiːvnɪŋ/  좋은 저녁이에요             │
│                                                                 │
│   Informal / Casual:                                            │
│   "Hi."                /haɪ/          안녕                       │
│   "Hey."               /heɪ/          야, 안녕                   │
│   "Hey there."         /heɪ ðer/      안녕                       │
│   "What's up?"         /wɒts ʌp/      뭐 해? / 별일 없어?         │
│   "How's it going?"    /haʊz ɪt goʊɪŋ/ 잘 지내?                 │
│                                                                 │
│   ※ "Good night" is a FAREWELL, not a greeting!                │
│     You say it when LEAVING, not when arriving.                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 "How are you?" and Responses

| Question | Formality | Response |
|----------|-----------|----------|
| How are you? | Formal/Neutral | I'm fine, thank you. And you? |
| How are you doing? | Neutral | I'm doing well, thanks. |
| How's it going? | Informal | Good, thanks. You? |
| What's up? | Very informal | Not much. / Nothing much. |
| How have you been? | Neutral (after time apart) | I've been good, thanks. |

> **Korean speaker note**: In Korean, "밥 먹었어?" (Have you eaten?) is a common greeting. In English, "Have you eaten?" is a literal question, NOT a greeting. Don't use it as one — people will think you're inviting them to lunch!

### 1.3 Time-Based Greetings

```
Morning (sunrise ~ noon):     "Good morning!"
Afternoon (noon ~ ~5-6 PM):   "Good afternoon!"
Evening (after ~5-6 PM):      "Good evening!"   (greeting)
Night (when leaving/bedtime):  "Good night!"     (farewell ONLY)
```

## 3. Introductions

### 3.1 Introducing Yourself

```
Formal:
  "Hello, my name is Minjun Kim."
  "I'm Minjun Kim. Nice to meet you."
  "I'm from Seoul, South Korea."
  "I work at Samsung." / "I'm a student at Seoul National University."

Informal:
  "Hi, I'm Minjun."
  "Hey, I'm Minjun. Nice to meet you."
  "I'm from Korea."
```

### 3.2 Introducing Others

```
Formal:
  "I'd like to introduce my colleague, Ms. Park."
  "This is my friend, Sujin."
  "Have you met Professor Lee?"

Informal:
  "This is my friend Sujin."
  "Sujin, this is Tom. Tom, Sujin."
  "Hey, meet my friend Sujin."
```

### 3.3 Name Conventions

```
┌─────────────────────────────────────────────────────────────────┐

│                                                                 │
│   Korean: 성 + 이름 (Family name first)                          │
│   김민준 = Kim (성) + Minjun (이름)                               │
│                                                                 │
│   English: First name + Last name (Given name first)            │
│   John Smith = John (first/given) + Smith (last/family)         │
│                                                                 │

│   "My name is Minjun Kim." (given name first)                   │
│                                                                 │

```

## 4. Subject Pronouns

### 4.1 The 7 Subject Pronouns

```
┌─────────────────────────────────────────────────────────────────┐
│              English Subject Pronouns                            │
│                                                                 │
│   Singular:                                                     │
│   I     /aɪ/     나, 저           (1st person)                   │
│   you   /juː/    너, 당신, 여러분  (2nd person)                   │
│   he    /hiː/    그 (남성)        (3rd person, male)             │
│   she   /ʃiː/    그녀 (여성)      (3rd person, female)           │
│   it    /ɪt/     그것             (3rd person, thing/animal)     │
│                                                                 │
│   Plural:                                                       │
│   we    /wiː/    우리             (1st person plural)            │
│   they  /ðeɪ/    그들, 그것들      (3rd person plural)            │
│                                                                 │
│   ※ "you" = both singular AND plural (same word!)              │
│   ※ "you" = both formal AND informal (same word!)              │
│   ※ "I" is ALWAYS capitalized, even in the middle of a sentence│
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Critical Differences from Korean

```
Difference 1: Subject pronouns are REQUIRED in English
  Korean: (나는) 학생이다.  → Subject often dropped
  English: I am a student. → "I" is REQUIRED (cannot say "Am a student")

Difference 2: "you" covers everything
  Korean: 너, 당신, 선생님, 교수님, 사장님... (many forms)
  English: you, you, you, you, you... (one form for ALL)

Difference 3: "he" and "she" are mandatory
  Korean: 그 사람 / 걔 (gender-neutral)
  English: MUST choose he (male) or she (female)
  ※ Modern English also uses "they" as singular gender-neutral

Difference 4: "it" for things and animals
  Korean: No equivalent — just use the noun or drop it
  English: "Where is the book?" → "It is on the table."
  ※ Pets are often "he" or "she", not "it"
```

## 5. The BE Verb: am/is/are

### 5.1 Conjugation

```
┌─────────────────────────────────────────────────────────────────┐
│              The BE Verb — Present Tense                         │
│                                                                 │
│   I          am      (I'm)        나는 ~이다                     │
│   you        are     (you're)     너는/당신은 ~이다               │
│   he/she/it  is      (he's/she's/it's)  그/그녀/그것은 ~이다      │
│   we         are     (we're)      우리는 ~이다                   │
│   they       are     (they're)    그들은 ~이다                   │
│                                                                 │
│   Full form:  I am a student.    (formal/written)               │
│   Contracted: I'm a student.     (spoken/informal)              │
│                                                                 │
│   ※ Korean "이다" is a particle attached to the noun.           │
│     English "be" is a separate word BEFORE the complement.      │
│     Korean: 학생이다 (student + is)                               │
│     English: am a student (is + a + student)                    │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 Uses of BE

```
1. Identity (신분/정체):
   I am a student.           나는 학생이다.
   She is a doctor.          그녀는 의사이다.

2. Description (묘사):
   He is tall.               그는 키가 크다.
   The sky is blue.          하늘이 파랗다.

3. Origin / Nationality (출신/국적):
   I am from Korea.          나는 한국 출신이다.
   They are Korean.          그들은 한국 사람이다.

4. Location (위치):
   She is at home.           그녀는 집에 있다.
   The books are on the table.  책들이 탁자 위에 있다.

5. Age (나이):
   I am 25 years old.        나는 25살이다.
   ※ Korean: 25살이다 (number + 살)
      English: am 25 years old (be + number + years old)

6. Feelings / States (감정/상태):
   I am happy.               나는 행복하다.
   We are tired.             우리는 피곤하다.
```

### 5.3 Negative Form

```
Full form:              Contracted:
I am not tired.         I'm not tired.
You are not late.       You're not late.   / You aren't late.
He is not here.         He's not here.     / He isn't here.
She is not Korean.      She's not Korean.  / She isn't Korean.
It is not cold.         It's not cold.     / It isn't cold.
We are not ready.       We're not ready.   / We aren't ready.
They are not students.  They're not students. / They aren't students.

※ "I amn't" does NOT exist! Only "I'm not."
```

### 5.4 Question Form

```
Statement → Question: Move "be" BEFORE the subject

Statement:           Question:
You are a student.   Are you a student?      (당신은 학생입니까?)
He is Korean.        Is he Korean?           (그는 한국 사람입니까?)
They are ready.      Are they ready?         (그들은 준비됐습니까?)

Short answers:
Are you a student?   → Yes, I am. / No, I'm not.
Is she Korean?       → Yes, she is. / No, she isn't.
Are they ready?      → Yes, they are. / No, they aren't.

※ Korean makes questions by adding -까/-니 to the verb ending.
  English makes questions by MOVING the verb before the subject.
  This word-order change is one of the hardest habits to build!
```

## 6. Basic SVO Sentences

### 6.1 English Sentence Structure

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean: SOV (Subject + Object + Verb)                         │
│   나는    사과를    먹는다.                                       │
│   I      apple    eat                                           │
│                                                                 │
│   English: SVO (Subject + Verb + Object)                        │
│   I       eat     an apple.                                     │
│   주어     동사     목적어                                        │
│                                                                 │
│   ※ The verb comes BEFORE the object in English!               │
│     This is the OPPOSITE of Korean word order.                  │
│     Korean: 나는 영어를 공부한다                                   │
│     English: I study English (NOT "I English study")            │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 Simple SVO Examples

| Korean (SOV) | English (SVO) | Common Mistake |
|-------------|---------------|----------------|
| 나는 물을 마신다 | I drink water | ~~I water drink~~ |
| 그녀는 책을 읽는다 | She reads a book | ~~She a book reads~~ |
| 우리는 영어를 공부한다 | We study English | ~~We English study~~ |
| 그들은 축구를 한다 | They play soccer | ~~They soccer play~~ |
| 나는 음악을 듣는다 | I listen to music | ~~I music listen~~ |

### 6.3 SVO with BE

```
With "be", the complement comes after:

S + BE + Noun:       I am a teacher.
S + BE + Adjective:  She is beautiful.
S + BE + Location:   They are in Seoul.

These are NOT SVO (no object) but S + V + C (complement).
The pattern is the same: subject FIRST, verb SECOND.
```

## 7. Polite Expressions

### 7.1 Essential Polite Words

```
Please      /pliːz/      ~해 주세요           (when requesting)
Thank you   /θæŋk juː/   감사합니다           (when receiving)
Thanks      /θæŋks/      고마워요 (informal)
You're welcome /jʊr ˈwelkəm/ 천만에요         (responding to thanks)
Sorry       /ˈsɒri/      미안합니다           (apologizing)
Excuse me   /ɪkˈskjuːz miː/ 실례합니다        (getting attention / passing by)
Pardon?     /ˈpɑːrdn/    네? / 뭐라고요?       (didn't hear)
```

### 7.2 "Please" Usage

```
Adding "please" makes a request polite:
  "Close the door."        → somewhat direct
  "Close the door, please." → polite
  "Please close the door." → polite
  "Could you close the door, please?" → very polite

※ Korean uses 존댓말 verb endings for politeness (-세요, -습니다).
  English uses "please", "could you", "would you" + polite tone.
  "Please" alone is NOT as powerful as Korean 존댓말.
  For true politeness, use: "Could you...?" or "Would you mind...?"
```

### 7.3 Thanking and Responding

```
Thanking (감사):
  Thank you.                       감사합니다
  Thank you very much.             대단히 감사합니다
  Thanks a lot.                    정말 고마워요
  Thanks.                          고마워
  I appreciate it.                 감사드립니다 (formal)

Responding to thanks (응답):
  You're welcome.                  천만에요
  No problem.                      별말씀을요
  Don't mention it.                천만에요
  Sure.                            그럼요 (casual)
  My pleasure.                     기꺼이요 (formal)
  Anytime.                         언제든지요
```

## 8. Formality in English vs Korean Honorifics

### 8.1 Korean Has 7 Speech Levels — English Has ~2-3

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean Speech Levels (7단계):                                  │
│   하십시오체, 해요체, 하오체, 하게체, 해체, 해라체, etc.              │
│   → Verb endings change dramatically                            │
│   → 먹다: 드십시오, 드세요, 먹어요, 먹어, 먹어라...                 │
│                                                                 │
│   English "Formality" (2-3 levels):                             │
│   Formal:    "Could you please provide the document?"           │
│   Neutral:   "Can you give me the document?"                    │
│   Informal:  "Give me the document."                            │
│                                                                 │
│   ※ English does NOT change verb forms for politeness!          │
│     "eat" is "eat" whether talking to a child or a president.  │
│     Politeness comes from: word choice, "please/could/would",  │
│     and tone of voice — NOT verb endings.                       │
│                                                                 │
│   This is LIBERATING for Korean speakers but also CONFUSING:    │
│   You might sound "too casual" by forgetting "please/could"    │
│   or "too formal" by being overly polite in casual situations.  │
└─────────────────────────────────────────────────────────────────┘
```

### 8.2 How English Shows Politeness

```
Instead of verb endings, English uses:

1. Modal verbs:
   Direct:  "Open the window."
   Polite:  "Can you open the window?"
   Very polite: "Could you open the window?"
   Most polite: "Would you mind opening the window?"

2. Word choice:
   Direct:  "I want coffee."
   Polite:  "I'd like coffee, please."
   Formal:  "I would appreciate a coffee."

3. Indirect questions:
   Direct:  "Where is the station?"
   Polite:  "Could you tell me where the station is?"
   Formal:  "I was wondering if you could tell me where the station is."

4. Titles:
   Casual:  "Hey John"
   Neutral: "Hi John"
   Formal:  "Hello, Mr. Smith"
```

## 9. Common Conversation Patterns

### 9.1 Meeting Someone for the First Time

```
A: Hi, I'm Minjun. Nice to meet you.
B: Hi Minjun, I'm Sarah. Nice to meet you too.
A: Where are you from?
B: I'm from New York. How about you?
A: I'm from Seoul, Korea.
B: Oh, nice! What do you do?
A: I'm a software engineer. And you?
B: I'm a teacher.
A: That's great. It was nice meeting you.
B: You too. See you around!
```

### 9.2 At a Coffee Shop

```
Barista: Hi, what can I get you?
You:     Hi, can I have a latte, please?
Barista: What size? Small, medium, or large?
You:     Medium, please.
Barista: Anything else?
You:     No, that's all. Thank you.
Barista: That'll be $4.50.
You:     Here you go.
Barista: Thanks. Have a great day!
You:     You too!
```

### 9.3 Asking for Help

```
Formal:
  "Excuse me, could you help me?"
  "I'm sorry to bother you, but could you tell me...?"

Informal:
  "Hey, can you help me with this?"
  "Do you know where the bathroom is?"

※ Korean speakers often say "Excuse me" too forcefully.
  Say it softly and with a slight rise in intonation: "Excuse me? ↗"
```

### 9.4 Phone / Video Call

```
Answering:
  "Hello?"                        (rising intonation)
  "Hi, this is Minjun speaking."  (identifying yourself)

Calling someone:
  "Hi, this is Minjun. Is Sarah there?"
  "Hi, may I speak to Sarah, please?"

Ending:
  "Okay, talk to you later. Bye!"
  "Alright, take care. Goodbye."
```
