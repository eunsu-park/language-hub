## 1. Introduction to Voice

### 1.1 What is Voice?

```
┌─────────────────────────────────────────────────────────────────┐
│                    Active vs Passive Voice                       │
│                                                                 │
│   Active Voice:  The subject PERFORMS the action                │
│   → "The chef cooked the meal."                                 │
│     Subject (chef) → Action (cooked) → Object (meal)           │
│                                                                 │
│   Passive Voice: The subject RECEIVES the action               │
│   → "The meal was cooked by the chef."                          │
│     Subject (meal) ← Action (was cooked) ← Agent (chef)        │
│                                                                 │
│   Korean Comparison:                                            │
│   Active:  요리사가 음식을 요리했다.                                │
│   Passive: 음식이 (요리사에 의해) 요리되었다.                       │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Why Use the Passive Voice?

The passive voice is used when:
- The **action** is more important than who did it
- The **doer** is unknown, obvious, or unimportant
- You want to sound **formal or objective** (academic writing, news reports)
- You want to **avoid blaming** someone directly

```
Unknown agent:    "My car was stolen last night."
                  (We don't know who stole it.)

Obvious agent:    "The suspect was arrested."
                  (Obviously by the police.)

Formal/objective: "The experiment was conducted in 2024."
                  (Scientific writing style.)

Avoiding blame:   "Mistakes were made."
                  (Who made them? Not specified.)
```

## 2. Passive Voice Formation

### 2.1 The Basic Formula

```
┌─────────────────────────────────────────────────────────────────┐
│           Passive Voice = BE + Past Participle (PP)             │
│                                                                 │
│   Subject + BE (conjugated) + Past Participle (+ by agent)     │
│                                                                 │
│   "The letter   was          written       by Maria."           │
│    └─subject─┘  └─BE─┘       └──PP──┘     └─agent─┘            │
│                                                                 │
│   The form of BE changes to match the tense:                   │
│   • Present:  am/is/are + PP                                   │
│   • Past:     was/were + PP                                    │
│   • Future:   will be + PP                                     │
│   • Perfect:  have/has been + PP                               │
│   • Modal:    can/must/should be + PP                          │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 Regular and Irregular Past Participles

Remember: the past participle is the third form of the verb.

| Type | Base Form | Past Tense | Past Participle |
|------|-----------|------------|-----------------|
| Regular | cook | cooked | **cooked** |
| Regular | clean | cleaned | **cleaned** |
| Regular | play | played | **played** |
| Irregular | write | wrote | **written** |
| Irregular | break | broke | **broken** |
| Irregular | take | took | **taken** |
| Irregular | see | saw | **seen** |
| Irregular | give | gave | **given** |
| Irregular | make | made | **made** |
| Irregular | eat | ate | **eaten** |

> **Common Irregular Past Participles to Memorize**: done, gone, known, shown, spoken, stolen, chosen, driven, forgotten, hidden, risen, worn, torn, sworn, thrown, blown, grown, drawn, flown, frozen

## 3. Active to Passive Conversion

### 3.1 Step-by-Step Process

```
Step 1: Identify the subject, verb, and object in the active sentence
Step 2: Move the object to the subject position
Step 3: Change the verb to "be + past participle"
Step 4: Move the original subject to "by + agent" (optional)

Example:
  Active:  [Shakespeare] [wrote] [Hamlet].
            subject       verb    object

  Step 1:  Subject = Shakespeare, Verb = wrote, Object = Hamlet
  Step 2:  Hamlet → subject position
  Step 3:  wrote → was written (past tense of "be" + PP)
  Step 4:  Shakespeare → by Shakespeare

  Passive: Hamlet was written by Shakespeare.
```

### 3.2 More Examples

| Active | Passive |
|--------|---------|
| The dog chased the cat. | The cat **was chased** by the dog. |
| Someone has stolen my phone. | My phone **has been stolen**. |
| They will announce the results. | The results **will be announced**. |
| People speak English worldwide. | English **is spoken** worldwide. |
| The company is building a new office. | A new office **is being built** by the company. |

### 3.3 Sentences That Cannot Be Passive

Only **transitive verbs** (verbs with a direct object) can be made passive.

```
Cannot be passive (intransitive verbs):
  ✗ "The baby cried."       → No object, no passive
  ✗ "She arrived early."    → No object, no passive
  ✗ "He slept for hours."   → No object, no passive
  ✗ "The sun rose."         → No object, no passive

Can be passive (transitive verbs):
  ✓ "She wrote a letter."   → "A letter was written by her."
  ✓ "They ate the cake."    → "The cake was eaten by them."
```

## 4. The Agent with "By"

### 4.1 When to Include the Agent

```
┌─────────────────────────────────────────────────────────────────┐
│                     Include "by + agent" when:                  │
│                                                                 │
│   1. The agent provides NEW or IMPORTANT information           │
│      "The Mona Lisa was painted by Leonardo da Vinci."         │
│                                                                 │
│   2. The identity of the agent is SURPRISING                   │
│      "The window was broken by the teacher!"                   │
│                                                                 │
│   3. You need to SPECIFY who is responsible                    │
│      "This report was written by the marketing team."          │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 When to Omit the Agent

```
┌─────────────────────────────────────────────────────────────────┐
│                     Omit "by + agent" when:                     │
│                                                                 │
│   1. The agent is UNKNOWN                                      │
│      "My wallet was stolen." (by someone unknown)              │
│                                                                 │
│   2. The agent is OBVIOUS                                      │
│      "He was arrested." (obviously by police)                  │
│                                                                 │
│   3. The agent is UNIMPORTANT / GENERAL                        │
│      "English is spoken in many countries."                    │
│      (by people — too general to matter)                       │
│                                                                 │
│   4. The agent is "people," "someone," "they" (generic)        │
│      "The building was constructed in 1990."                   │
│      (by construction workers — obvious)                       │
└─────────────────────────────────────────────────────────────────┘
```

## 5. Passive Voice Across Tenses

### 5.1 Complete Tense Table

| Tense | Active | Passive |
|-------|--------|---------|
| Simple Present | They **make** cars here. | Cars **are made** here. |
| Present Continuous | They **are building** a bridge. | A bridge **is being built**. |
| Present Perfect | They **have finished** the project. | The project **has been finished**. |
| Simple Past | She **wrote** the report. | The report **was written** by her. |
| Past Continuous | They **were cleaning** the office. | The office **was being cleaned**. |
| Past Perfect | He **had completed** the task. | The task **had been completed**. |
| Simple Future | They **will deliver** the package. | The package **will be delivered**. |
| Future Perfect | They **will have built** the house. | The house **will have been built**. |
| Modal (can) | You **can see** the mountains. | The mountains **can be seen**. |
| Modal (must) | You **must submit** the form. | The form **must be submitted**. |
| Modal (should) | They **should fix** the problem. | The problem **should be fixed**. |

### 5.2 Passive with Modals

```
Modal + be + past participle

  can be done       "This can be done quickly."
  could be seen     "The stars could be seen clearly."
  must be finished  "The work must be finished by Friday."
  should be sent    "The email should be sent today."
  may be delayed    "The flight may be delayed."
  might be changed  "The plan might be changed."
  has to be paid    "The bill has to be paid."
  ought to be told  "She ought to be told the truth."
```

## 6. Special Passive Constructions

### 6.1 Get + Past Participle (Informal Passive)

In informal English, **get** can replace **be** in passive constructions.

```
Formal (be):     "He was fired from his job."
Informal (get):  "He got fired from his job."

Common "get" passives:
  get married     "They got married last June."
  get divorced    "She got divorced in 2020."
  get hurt        "Nobody got hurt in the accident."
  get lost        "We got lost in the city."
  get arrested    "The thief got arrested."
  get paid        "I get paid every two weeks."
  get stuck       "The car got stuck in the mud."
  get killed      "Three people got killed in the crash."
```

> **Note**: "Get" passives often imply something negative or unexpected.

### 6.2 Have/Get Something Done (Causative Passive)

Used when you **arrange for someone else** to do something for you.

```
┌─────────────────────────────────────────────────────────────────┐
│   have + object + past participle                               │
│   get  + object + past participle                               │
│                                                                 │
│   "I had my hair cut."                                          │
│   = I arranged for someone to cut my hair.                     │
│   ≠ I cut my own hair.                                         │
│                                                                 │
│   "She got her car repaired."                                   │
│   = She arranged for someone to repair her car.                │
│                                                                 │
│   More examples:                                               │
│   "I need to have my eyes tested."                              │
│   "We had the house painted last summer."                      │
│   "You should get your teeth checked."                         │
│   "He's getting his suit dry-cleaned."                         │
└─────────────────────────────────────────────────────────────────┘
```

### 6.3 It is said that... / He is said to...

```
Reporting what people say/believe (formal):

  Pattern 1: It + passive reporting verb + that-clause
    "It is said that he is very rich."
    "It is believed that the earth is 4.5 billion years old."
    "It is reported that the company will close."
    "It was thought that the disease was incurable."

  Pattern 2: Subject + passive reporting verb + to-infinitive
    "He is said to be very rich."
    "The earth is believed to be 4.5 billion years old."
    "The company is reported to be closing."
    "The disease was thought to be incurable."

  Common reporting verbs for this pattern:
    say, believe, think, know, report, consider, expect, understand,
    claim, allege, assume, estimate, feel, suppose
```

## 7. Introduction to Reported Speech

### 7.1 Direct vs Reported (Indirect) Speech

```
┌─────────────────────────────────────────────────────────────────┐
│   Direct Speech:   Exact words inside quotation marks          │
│   → She said, "I am tired."                                    │
│                                                                 │
│   Reported Speech: Restating someone's words without quotes    │
│   → She said (that) she was tired.                              │
│                                                                 │
│   Key Changes:                                                  │
│   1. Quotation marks are removed                               │
│   2. Pronouns change (I → she)                                 │
│   3. Tenses shift back (am → was) — "backshifting"             │
│   4. Time/place words may change (today → that day)            │
│   5. "That" is optional but common                             │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 Why Reported Speech Matters

Reported speech is essential for:
- Retelling conversations
- Writing news articles
- Academic writing (citing sources)
- Everyday storytelling ("He told me that...")

## 8. Backshifting Rules

### 8.1 Tense Changes

When the reporting verb is in the **past tense** (said, told, asked), the tenses in the reported clause shift back:

```
┌───────────────────────────┬──────────────────────────────────┐
│   Direct Speech            │   Reported Speech               │
├───────────────────────────┼──────────────────────────────────┤
│ Present Simple             │ Past Simple                      │
│ "I like coffee."           │ She said she liked coffee.       │
├───────────────────────────┼──────────────────────────────────┤
│ Present Continuous         │ Past Continuous                  │
│ "I am studying."           │ He said he was studying.         │
├───────────────────────────┼──────────────────────────────────┤
│ Present Perfect            │ Past Perfect                     │
│ "I have finished."         │ She said she had finished.       │
├───────────────────────────┼──────────────────────────────────┤
│ Past Simple                │ Past Perfect                     │
│ "I went to Seoul."         │ He said he had gone to Seoul.    │
├───────────────────────────┼──────────────────────────────────┤
│ Past Continuous            │ Past Perfect Continuous          │
│ "I was reading."           │ She said she had been reading.   │
├───────────────────────────┼──────────────────────────────────┤
│ will                       │ would                            │
│ "I will call you."         │ He said he would call me.        │
├───────────────────────────┼──────────────────────────────────┤
│ can                        │ could                            │
│ "I can help."              │ She said she could help.         │
├───────────────────────────┼──────────────────────────────────┤
│ may                        │ might                            │
│ "I may come."              │ He said he might come.           │
├───────────────────────────┼──────────────────────────────────┤
│ must                       │ had to                           │
│ "I must leave."            │ She said she had to leave.       │
├───────────────────────────┼──────────────────────────────────┤
│ shall                      │ should / would                   │
│ "Shall I open it?"         │ He asked if he should open it.   │
└───────────────────────────┴──────────────────────────────────┘
```

### 8.2 No Backshifting (Exceptions)

Backshifting is **optional** or **unnecessary** when:

```
1. The information is STILL TRUE:
   Direct:   "The earth revolves around the sun."
   Reported: He said (that) the earth revolves around the sun.
             (Still true — no need to backshift)

2. The reporting verb is PRESENT TENSE:
   Direct:   "I'm coming to the party."
   Reported: She says (that) she's coming to the party.
             (No backshift needed)

3. Modals that DON'T change: would, could, should, might, ought to
   Direct:   "You should study harder."
   Reported: He said I should study harder.
             (No change)

4. Past Perfect stays Past Perfect:
   Direct:   "I had already eaten."
   Reported: She said she had already eaten.
             (Already the "furthest back" tense)
```

### 8.3 Time and Place Changes

| Direct Speech | Reported Speech |
|---------------|-----------------|
| today | that day |
| tonight | that night |
| yesterday | the day before / the previous day |
| tomorrow | the next day / the following day |
| last week | the week before / the previous week |
| next month | the following month |
| ago | before / earlier |
| now | then / at that time |
| here | there |
| this | that |
| these | those |
| come | go |

```
Direct:   "I will see you here tomorrow."
Reported: He said he would see me there the following day.

Direct:   "I finished this project yesterday."
Reported: She said she had finished that project the day before.
```

## 9. Reporting Questions

### 9.1 Yes/No Questions

```
Formula: asked + if / whether + subject + verb (statement order)

Direct:   "Do you like Korean food?"
Reported: She asked (me) if/whether I liked Korean food.

Direct:   "Can you swim?"
Reported: He asked if I could swim.

Direct:   "Have you been to London?"
Reported: She asked whether I had been to London.

Direct:   "Will you come to the party?"
Reported: He asked if I would come to the party.

Important: Do NOT use question word order in reported questions!
  ✗ She asked if did I like Korean food.
  ✓ She asked if I liked Korean food.

  ✗ He asked could I swim.
  ✓ He asked if I could swim.
```

### 9.2 Wh-Questions

```
Formula: asked + question word + subject + verb (statement order)

Direct:   "Where do you live?"
Reported: She asked (me) where I lived.

Direct:   "What time does the train leave?"
Reported: He asked what time the train left.

Direct:   "Why are you crying?"
Reported: She asked why I was crying.

Direct:   "How long have you been waiting?"
Reported: He asked how long I had been waiting.

Important: Statement word order, NOT question word order!
  ✗ She asked where did I live.
  ✓ She asked where I lived.

  ✗ He asked what was my name.
  ✓ He asked what my name was.
```

### 9.3 Reporting Commands and Requests

```
Formula: told/asked/ordered + person + (not) + to-infinitive

Commands:
  Direct:   "Close the door."
  Reported: He told me to close the door.

  Direct:   "Don't touch that!"
  Reported: She told us not to touch that.

Requests:
  Direct:   "Could you please help me?"
  Reported: She asked me to help her.

  Direct:   "Please don't be late."
  Reported: He asked us not to be late.

Advice:
  Direct:   "You should see a doctor."
  Reported: She advised me to see a doctor.
```

## 10. Say vs Tell

### 10.1 Key Differences

```
┌─────────────────────────────────────────────────────────────────┐
│   SAY                          │   TELL                         │
│                                │                                │
│   • No indirect object needed  │   • Needs an indirect object   │
│   • say (that)...              │   • tell + person + (that)...  │
│   • say + direct speech        │   • Cannot use with direct     │
│                                │     speech quotes              │
│                                │                                │
│   ✓ He said (that) he was      │   ✓ He told me (that) he was   │
│     tired.                     │     tired.                     │
│   ✓ He said, "I'm tired."     │   ✗ He told, "I'm tired."      │
│   ✗ He said me that...        │   ✓ He told me that...          │
│   ✓ He said to me that...     │                                │
│     (possible but less common) │                                │
└─────────────────────────────────────────────────────────────────┘
```

### 10.2 Summary Rules

```
SAY:  said / said that / said to someone
      He said he was happy.
      He said that he was happy.
      He said to me that he was happy.

TELL: told + PERSON / told + PERSON + that
      He told me he was happy.
      He told me that he was happy.
      ✗ He told that he was happy. (WRONG — needs a person!)
```

### 10.3 Fixed Expressions with Tell

```
tell the truth    "Always tell the truth."
tell a lie        "She told a lie."
tell a story      "Grandma told us a story."
tell a joke       "He told a funny joke."
tell the time     "Can you tell the time?"
tell the difference "Can you tell the difference?"
tell a secret     "Don't tell anyone the secret."
```

## 12. Korean Speaker Notes

### 12.1 Passive Voice Comparison

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean uses different passive constructions:                  │
│                                                                 │
│   1. -되다 (doeda) attached to Sino-Korean nouns:              │
│      건설되다 (to be constructed), 발견되다 (to be discovered)    │
│                                                                 │
│   2. -받다 (batda) "receive" passive:                           │
│      사랑받다 (to be loved), 존경받다 (to be respected)           │
│                                                                 │
│   3. Verb suffix passive (-이/-히/-리/-기):                     │
│      잡다 → 잡히다 (catch → be caught)                          │
│      열다 → 열리다 (open → be opened)                           │
│                                                                 │
│   Key Differences:                                             │
│   • Korean passive is less common than in English              │
│   • Korean speakers often overuse active voice in English      │
│   • English passive is more structured (be + PP) and          │
│     can be applied to almost any transitive verb               │
│   • Korean word order (SOV) vs English passive (SVO)           │
│     makes conversion tricky                                    │
│                                                                 │
│   Common Error:                                                │
│   ✗ "The meeting was held" written as "The meeting held"       │
│   ✓ Always include the BE verb in English passive!             │
└─────────────────────────────────────────────────────────────────┘
```

### 12.2 Reported Speech Comparison

```
Korean reported speech uses:
  -고 하다 / -라고 하다 / -다고 하다 / -냐고 하다

  "나는 행복하다"고 그가 말했다.
  → He said (that) he was happy.

Key Differences:
  • Korean does NOT backshift tenses in reported speech
  • Korean uses different quote particles for statements (-다고),
    questions (-냐고), commands (-라고), suggestions (-자고)
  • English uses the same "that" for all statement reports
  • Korean learners often forget to backshift tenses!

Common Korean Speaker Errors:
  ✗ He said he is happy. (forgetting to backshift)
  ✓ He said he was happy.

  ✗ She asked where is the station. (keeping question word order)
  ✓ She asked where the station was.

  ✗ He told that he was tired. (missing person after "tell")
  ✓ He told me that he was tired.
```

### Exercise 6: Mixed Practice

Change to reported speech, choosing the correct form.

1. "Please close the window," he said to me.
2. "Don't eat too much sugar," the doctor said to the patient.
3. "Let's go to the beach," she said.
4. "I can't come because I'm sick," he told me.
5. "Where were you last night?" she asked him.

<details>
<summary>Click to reveal answers</summary>

```
1. He asked me to close the window.
   (Or: He told me to close the window.)
2. The doctor advised the patient not to eat too much sugar.
   (Or: The doctor told the patient not to eat too much sugar.)
3. She suggested going to the beach.
   (Or: She suggested that they go to the beach.)
4. He told me (that) he couldn't come because he was sick.
5. She asked him where he had been the night before / the previous night.
```

## Quick Reference

```
┌─────────────────────────────────────────────────────────────────┐
│                Passive Voice & Reported Speech                  │
│                                                                 │
│   PASSIVE VOICE:                                               │
│   Formula: Subject + BE (conjugated) + Past Participle         │
│   Present:  is/are + PP    Past:  was/were + PP                │
│   Perfect:  has/have been + PP   Future: will be + PP          │
│   Modal:    can/must/should be + PP                            │
│   Omit "by" when agent is unknown, obvious, or unimportant    │
│                                                                 │
│   REPORTED SPEECH:                                             │
│   Statements: said (that) / told + person + (that)             │
│   Questions:  asked + if/whether (yes/no)                      │
│               asked + wh-word (wh-questions)                   │
│   Commands:   told/asked + person + (not) to + infinitive      │
│                                                                 │
│   BACKSHIFTING: present→past, past→past perfect,               │
│   will→would, can→could, may→might, must→had to               │
│                                                                 │
│   SAY vs TELL:                                                 │
│   say — no person needed: "He said he was happy."              │
│   tell — person required: "He told me he was happy."           │
└─────────────────────────────────────────────────────────────────┘
```
