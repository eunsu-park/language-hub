## 1. Introduction to Reading Strategies

### 1.1 Why Reading Strategies Matter

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean speakers reading in English often:                    │
│                                                                 │
│   ✗ Try to understand every single word                        │
│   ✗ Mentally translate to Korean word by word                  │
│   ✗ Read at the same speed regardless of purpose               │
│   ✗ Give up when encountering unknown vocabulary               │
│                                                                 │
│   Effective readers:                                           │
│                                                                 │
│   ✓ Adjust reading speed to match their purpose               │
│   ✓ Use context clues to guess unknown words                  │
│   ✓ Skim for main ideas before reading in detail              │
│   ✓ Make predictions and inferences as they read              │
│   ✓ Identify text structure to navigate efficiently            │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Three Core Strategies

```
SKIMMING:   Read quickly for the MAIN IDEA / general overview
            Speed: very fast (~800-1000 words per minute)
            Purpose: "What is this text about?"

SCANNING:   Search quickly for SPECIFIC INFORMATION
            Speed: very fast (eyes jump around the text)
            Purpose: "Where does it mention X?"

INTENSIVE:  Read carefully for COMPLETE UNDERSTANDING
            Speed: slow (~200-300 words per minute)
            Purpose: "What exactly does this mean?"
```

## 2. Skimming

### 2.1 How to Skim

```
Step 1: Read the TITLE and any SUBTITLES
Step 2: Read the FIRST PARAGRAPH (introduction)
Step 3: Read the FIRST SENTENCE of each paragraph (topic sentence)
Step 4: Read the LAST PARAGRAPH (conclusion)
Step 5: Note any bold, italic, or highlighted text
Step 6: Look at images, charts, and captions

Time: 1-2 minutes for a full-page article

After skimming, you should be able to answer:
  • What is the main topic?
  • What is the author's general position?
  • What are the key subtopics?
```

### 2.2 When to Skim

```
• Deciding whether a text is relevant for your research
• Getting an overview before reading in detail
• Reviewing material before an exam
• Reading news headlines and article summaries
• Previewing a textbook chapter
```

## 3. Scanning

### 3.1 How to Scan

```
Step 1: Know EXACTLY what you're looking for
        (a name, date, number, keyword, etc.)
Step 2: Move your eyes quickly across and down the page
Step 3: Look for visual clues (capital letters, numbers,
        bold text, bullet points)
Step 4: Stop when you find the target information
Step 5: Read the surrounding context for full understanding

Scanning is like using Ctrl+F (Find) in a document!
```

### 3.2 When to Scan

```
• Finding a specific date or statistic
• Looking up a phone number or address
• Finding a word in a dictionary or glossary
• Looking for a name in a list
• Finding a particular section in a long document
• Checking a bus schedule or timetable
```

## 4. Reading for Detail

### 4.1 Intensive Reading Techniques

```
SQ3R Method:
  Survey  → Skim the text first (get the big picture)
  Question → Turn headings into questions ("What is X?")
  Read    → Read carefully to answer your questions
  Recite  → Summarize what you read in your own words
  Review  → Review notes and key points later

Active Reading Markers:
  ★ = important point
  ? = I don't understand this
  ! = surprising or interesting
  ↔ = connects to something else I know
  ✓ = I agree
  ✗ = I disagree
```

### 4.2 Paragraph Structure

```
Most English paragraphs follow this structure:

  ┌─ Topic sentence (main idea) ──────────────────────┐
  │  "Social media has transformed how we communicate." │
  ├─ Supporting details ──────────────────────────────┤
  │  "For example, platforms like Twitter allow instant │
  │   information sharing across the globe."           │
  │  "Furthermore, Facebook has connected over 2       │
  │   billion users worldwide."                        │
  ├─ Concluding sentence (optional) ──────────────────┤
  │  "Clearly, social media has become an integral     │
  │   part of modern communication."                   │
  └───────────────────────────────────────────────────┘

  Tip: The topic sentence is usually the FIRST sentence
  of the paragraph. Look there for the main idea!
```

## 5. Making Inferences

### 5.1 What Is an Inference?

An inference is a conclusion drawn from evidence in the text, even when it is not explicitly stated.

```
Text: "Sarah grabbed her umbrella and put on her raincoat
       before leaving the house."

Inference: It was raining or about to rain.
(The text doesn't say "it was raining," but we can infer it.)

Text: "The man looked at his watch for the third time and
       tapped his foot impatiently."

Inference: He was waiting for someone/something and they were late.
```

### 5.2 How to Make Inferences

```
1. Look at what the text SAYS (explicit information)
2. Use your BACKGROUND KNOWLEDGE
3. Connect the two to reach a CONCLUSION

Text says + What I know = Inference

"The streets were empty, and all the shops had their
 shutters down."
 + I know shops close and streets empty at night or
   during an emergency.
 = It was either very late at night or there was an emergency.
```

### 5.3 Inference vs Fact vs Opinion

```
FACT:      Can be verified; is stated directly in the text.
           "Seoul has a population of approximately 10 million."

INFERENCE: Not stated directly but supported by text evidence.
           "Given the crowded subway, the city must have a
            large population." (supported by evidence)

OPINION:   A personal belief or judgment; cannot be proven.
           "Seoul is the best city in the world."

On tests, be careful to distinguish these three!
```

## 6. Text Types and Structures

### 6.1 Common Text Types

```
┌──────────────┬──────────────────────────────────────────────────┐
│ Text Type    │ Characteristics                                  │
├──────────────┼──────────────────────────────────────────────────┤
│ Narrative    │ Tells a story; has characters, setting, plot;   │
│              │ uses past tenses; chronological order            │
├──────────────┼──────────────────────────────────────────────────┤
│ Descriptive  │ Describes a person/place/thing; uses adjectives │
│              │ and sensory details; creates mental images       │
├──────────────┼──────────────────────────────────────────────────┤
│ Expository   │ Explains or informs; presents facts; uses       │
│              │ definitions, examples, and data                  │
├──────────────┼──────────────────────────────────────────────────┤
│ Persuasive / │ Argues a point; uses evidence and reasoning;    │
│ Argumentative│ has thesis, supporting arguments, counterpoint   │
├──────────────┼──────────────────────────────────────────────────┤
│ Instructional│ Gives directions; uses imperatives and          │
│              │ numbered steps; recipes, manuals                 │
└──────────────┴──────────────────────────────────────────────────┘
```

### 6.2 Organizational Patterns

```
Chronological:    Events in time order
                  Signal: first, then, next, finally, in 2020

Cause and Effect: Why something happened and its results
                  Signal: because, therefore, as a result, due to

Compare/Contrast: Similarities and differences between items
                  Signal: similarly, however, whereas, on the other hand

Problem/Solution: A problem is presented, then solutions
                  Signal: the issue is, one solution, to address this

Classification:   Items sorted into categories
                  Signal: types, categories, groups, classes, kinds
```

## 7. Understanding Context Clues

### 7.1 Types of Context Clues

```
1. DEFINITION clue: The meaning is directly stated.
   "Photosynthesis — the process by which plants convert
    sunlight into energy — is essential for life."

2. SYNONYM clue: A similar word is used nearby.
   "The child was elated; she was so happy she couldn't
    stop smiling."

3. ANTONYM clue: An opposite word provides contrast.
   "Unlike her gregarious sister, she was very shy."

4. EXAMPLE clue: Examples clarify the meaning.
   "Renewable energy sources, such as solar, wind, and
    hydroelectric power, are becoming more popular."

5. LOGIC clue: General knowledge helps determine meaning.
   "After three days without water in the desert, the
    hikers were completely parched."
    (Logic: no water + desert = very thirsty/dry)
```

### 7.2 Practice Strategy

```
When you encounter an unknown word:
  1. Don't look it up immediately
  2. Read the FULL sentence (and the ones before/after)
  3. Look for context clues
  4. Guess the meaning
  5. Check if your guess makes sense in context
  6. Continue reading
  7. Look up the word later to confirm
```

## 9. Critical Reading

### 9.1 What Is Critical Reading?

```
Critical reading goes beyond understanding WHAT a text says
to evaluating HOW and WHY it says it.

Questions to ask:
  • Who wrote this? What is their expertise or bias?
  • What is the author's purpose? (inform, persuade, entertain)
  • What evidence is provided? Is it sufficient?
  • Are there any logical fallacies or biased language?
  • What is NOT being said? (gaps or omissions)
  • How would someone with a different perspective respond?
```

### 9.2 Identifying Author's Purpose

```
To INFORM:     Presents facts objectively
               "The population of Seoul is approximately 10 million."

To PERSUADE:   Tries to convince the reader
               "We must act now to address climate change."

To ENTERTAIN:  Tells a story or amuses
               "The fox looked at the grapes and said, 'They're
                probably sour anyway.'"

To INSTRUCT:   Gives directions or steps
               "First, preheat the oven to 180°C."
```

### 9.3 Recognizing Bias and Persuasion Techniques

```
LOADED LANGUAGE:  Using emotional words to influence
  Biased:  "The reckless government slashed the budget."
  Neutral: "The government reduced the budget by 15%."

CHERRY-PICKING:   Only showing evidence that supports one side
  "9 out of 10 dentists recommend X."
  (What about the 10th dentist? How was the study done?)

GENERALIZATION:   Making broad claims from limited evidence
  "Young people today are all addicted to their phones."
  (All of them? Based on what evidence?)

APPEAL TO AUTHORITY: Using an authority figure inappropriately
  "A famous actor says this medicine works."
  (An actor is not a medical expert.)

AD HOMINEM:       Attacking the person instead of the argument
  "You can't trust his research because he's not from
   a top university."
```

## 10. Reading Speed Tips

### 10.1 Common Speed Blockers

```
┌─────────────────────────────────────────────────────────────────┐
│   1. Subvocalization (reading every word aloud in your head)   │
│      → Practice reading groups of words, not individual ones   │
│                                                                 │
│   2. Regression (going back to re-read)                        │
│      → Use a finger or pointer to keep moving forward          │
│                                                                 │
│   3. Word-by-word translation to Korean                        │
│      → Try to understand meaning in English directly           │
│                                                                 │
│   4. Stopping at every unknown word                            │
│      → Use context clues; look up words later                  │
│                                                                 │
│   5. Lack of vocabulary                                        │
│      → Build vocabulary systematically (see Lesson 20)         │
└─────────────────────────────────────────────────────────────────┘
```

### 10.2 Speed Improvement Techniques

```
1. Read in CHUNKS (groups of 3-5 words), not word by word
   Instead of: "The | boy | ran | to | the | store"
   Read as:    "The boy | ran to | the store"

2. EXPAND your peripheral vision
   Practice seeing more words at each fixation

3. READ EXTENSIVELY (read a lot of easy, enjoyable material)
   Graded readers, news articles, blogs on topics you enjoy

4. TIME yourself regularly to track progress
   Aim for 200-300 wpm for detailed reading
   Aim for 400-600 wpm for light/recreational reading

5. Don't REGRESS — keep moving forward
   If you missed something, your brain often fills in the gaps
```
