## 1. English Verbs vs Korean Verbs

### 1.1 Key Differences

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean Verbs                    English Verbs                  │
│   ─────────────                   ─────────────                  │
│   Come at END of sentence         Come AFTER subject             │
│   나는 사과를 먹는다                I eat an apple                 │
│                                                                 │
│   Change form for politeness      DON'T change for politeness   │
│   먹는다/먹어요/드십니다             eat/eat/eat                    │
│                                                                 │
│   DON'T change for person         Change for 3rd person (he -s) │
│   나는 먹는다/그는 먹는다            I eat / he eatS               │
│                                                                 │
│   Subject can be dropped          Subject is REQUIRED            │
│   (나는) 먹는다                     I eat (cannot drop "I")        │
│                                                                 │
│   No do-support                   Need "do" for questions/neg   │
│   먹니? / 안 먹는다                 Do you eat? / I don't eat     │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Verb Forms Overview

```
English verbs have 3-5 forms:

Regular verb (play):
  Base form:        play        (기본형)
  3rd person -s:    plays       (3인칭 단수)
  Past:             played      (과거)
  Past participle:  played      (과거분사)
  -ing form:        playing     (현재분사)

Irregular verb (go):
  Base form:        go
  3rd person -s:    goes
  Past:             went        (불규칙!)
  Past participle:  gone        (불규칙!)
  -ing form:        going
```

## 2. The 8 Essential Verbs

### 2.1 BE — 이다/있다

```
The most important and most irregular English verb.
See Lesson 02 for full BE verb conjugation.

Present: am / is / are
Past:    was / were
Uses:    identity, description, location, state

  I am Korean.              나는 한국인이다.
  She is a doctor.          그녀는 의사이다.
  They are at home.         그들은 집에 있다.
  It is cold.               (날씨가) 춥다.
```

### 2.2 HAVE — 가지다/있다

```
┌─────────────────────────────────────────────────────────────────┐
│   Present: have / has (3rd person)                               │
│   Past:    had                                                   │
│                                                                 │
│   Meaning 1: Possession (소유)                                   │
│   I have a car.              나는 차가 있다.                      │
│   She has two children.      그녀는 아이가 둘 있다.               │
│                                                                 │
│   Meaning 2: Experience/Action (경험/행동)                        │
│   I have breakfast at 7.     나는 7시에 아침을 먹는다.            │
│   She has a shower.          그녀는 샤워를 한다.                  │
│   We have a meeting.         우리는 회의가 있다.                  │
│                                                                 │
│   Meaning 3: Auxiliary verb (조동사 — for perfect tenses)        │
│   I have eaten lunch.        나는 점심을 먹었다. (현재완료)        │
│                                                                 │
│   ※ "have" vs "있다":                                           │
│     Korean 있다 = both "have" and "exist/be located"            │
│     "I have a book" (소유) — use HAVE                           │
│     "The book is on the table" (위치) — use BE, not HAVE!       │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 DO — 하다

```
Present: do / does (3rd person)
Past:    did

Meaning 1: General action (일반적 행동)
  I do my homework.           나는 숙제를 한다.
  She does the dishes.        그녀는 설거지를 한다.
  We do exercise.             우리는 운동을 한다.

Meaning 2: Auxiliary verb for questions/negatives (조동사)
  Do you speak English?       영어를 하시나요?
  I don't like spicy food.    나는 매운 음식을 안 좋아한다.

Common "do" collocations:
  do homework    do the dishes    do the laundry
  do business    do research      do a favor
  do one's best  do exercise      do the shopping
```

### 2.4 GO — 가다

```
Present: go / goes
Past:    went (very irregular!)

  I go to school.             나는 학교에 간다.
  She goes to work by bus.    그녀는 버스로 출근한다.
  We go shopping on weekends. 우리는 주말에 쇼핑을 간다.

Common "go" patterns:
  go to + place:    go to school, go to work, go to the store
  go + -ing:        go shopping, go swimming, go hiking, go fishing
  go home:          (NO "to" — go home, NOT "go to home")
  go back:          go back to Korea (돌아가다)
  go out:           go out for dinner (외출하다)

  ※ "go to home" ✗ → "go home" ✓ (home is an adverb here)
```

### 2.5 GET — 얻다/되다/도착하다

```
Present: get / gets
Past:    got

"Get" is one of the most versatile English verbs:

  get + noun:       "I get a letter."       (받다/얻다)
  get + adjective:  "I get tired."          (되다 — state change)
  get + to + place: "I get to work at 9."   (도착하다)
  get + up:         "I get up at 7."        (일어나다)
  get + on/off:     "Get on the bus."       (타다/내리다)

Common "get" expressions:
  get married (결혼하다)     get lost (길을 잃다)
  get angry (화나다)         get better (나아지다)
  get ready (준비하다)       get dressed (옷을 입다)
  get a job (취직하다)       get home (집에 도착하다)
```

### 2.6 MAKE — 만들다

```
Present: make / makes
Past:    made

  I make breakfast.           나는 아침을 만든다.
  She makes a lot of money.   그녀는 돈을 많이 번다.

Common "make" collocations:
  make breakfast/lunch/dinner  make a mistake
  make a decision              make money
  make friends                 make noise
  make progress                make an effort
  make a phone call            make a reservation

  ※ make vs do:
    make = create/produce something  (만들다/생산하다)
    do   = perform an activity       (수행하다)
    make breakfast ✓    do breakfast ✗
    do homework ✓       make homework ✗
```

### 2.7 TAKE — 가져가다/걸리다

```
Present: take / takes
Past:    took

  I take the bus to work.     나는 버스를 타고 출근한다.
  It takes 30 minutes.        30분이 걸린다.
  Take your umbrella.         우산을 가져가.

Common "take" collocations:
  take a shower/bath (샤워/목욕하다)
  take a photo/picture (사진을 찍다)
  take a break/rest (쉬다)
  take a test/exam (시험을 보다)
  take medicine (약을 먹다)
  take notes (노트를 하다)
  take a walk (산책하다)
  take time (시간이 걸리다)
  take a taxi/bus (택시/버스를 타다)
```

### 2.8 GIVE — 주다

```
Present: give / gives
Past:    gave

  I give her a gift.          나는 그녀에게 선물을 준다.
  She gives me advice.        그녀는 나에게 조언을 준다.

Two word orders:
  give + person + thing:   "Give me the book."
  give + thing + to + person: "Give the book to me."

Common "give" collocations:
  give advice (조언하다)      give information (정보를 주다)
  give a speech (연설하다)    give permission (허락하다)
  give a hand (돕다)          give up (포기하다)
```

## 3. Simple Present Tense

### 3.1 Formation

```
┌─────────────────────────────────────────────────────────────────┐
│           Simple Present Tense — Formation                      │
│                                                                 │
│   I          eat                                                │
│   you        eat                                                │
│   he/she/it  eatS    ← only change! Add -s for 3rd person sg  │
│   we         eat                                                │
│   they       eat                                                │
│                                                                 │
│   ※ Korean verbs don't change for person:                      │
│     나는 먹는다 / 그는 먹는다 / 우리는 먹는다 (same 먹는다)         │
│     I eat / he eatS / we eat (different for "he/she/it")        │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 When to Use Simple Present

```
1. Habits and routines (습관):
   "I drink coffee every morning."     나는 매일 아침 커피를 마신다.
   "She goes to gym three times a week." 그녀는 일주일에 세 번 헬스장에 간다.

2. Facts and general truths (사실):
   "The Earth goes around the Sun."    지구는 태양을 돈다.
   "Water boils at 100°C."            물은 100도에서 끓는다.

3. Permanent situations (항구적 상황):
   "I live in Seoul."                  나는 서울에 산다.
   "She works at a hospital."         그녀는 병원에서 일한다.

4. Schedules and timetables (일정):
   "The train leaves at 9 AM."        기차는 아침 9시에 출발한다.
   "The movie starts at 7."           영화는 7시에 시작한다.
```

## 4. Third Person Singular -s

### 4.1 Spelling Rules

```
Rule 1: Most verbs → add -s
  eat → eats       play → plays       read → reads
  work → works     like → likes       live → lives

Rule 2: Verbs ending in -s, -sh, -ch, -x, -z, -o → add -es
  pass → passes    wash → washes      watch → watches
  fix → fixes      go → goes          do → does

Rule 3: Verbs ending in consonant + y → change y to -ies
  study → studies  carry → carries    fly → flies
  try → tries      cry → cries        worry → worries
  ※ Vowel + y → just -s: play → plays, say → says, enjoy → enjoys

Irregular:
  have → has       (NOT "haves")
  be → is          (NOT "bes")
```

### 4.2 Pronunciation of -s/-es

```
Same 3 rules as plural nouns:

/s/  after voiceless: /p/, /t/, /k/, /f/, /θ/
  eats /iːts/    works /wɜːrks/    laughs /læfs/

/z/  after voiced: /b/, /d/, /g/, /v/, /m/, /n/, /l/, vowels
  reads /riːdz/   lives /lɪvz/     plays /pleɪz/    runs /rʌnz/

/ɪz/ after sibilants: /s/, /z/, /ʃ/, /ʒ/, /tʃ/, /dʒ/
  passes /ˈpæsɪz/  washes /ˈwɒʃɪz/  watches /ˈwɒtʃɪz/
```

## 5. Do-Support: Questions

### 5.1 The Do-Support System

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean makes questions by changing the verb ending:            │
│   먹는다 (statement) → 먹니? / 먹습니까? (question)               │
│                                                                 │
│   English needs "DO" to make questions with regular verbs:      │
│   You eat pizza. → Do you eat pizza?                            │
│                                                                 │
│   Why? Because English questions need INVERSION (verb before    │
│   subject), but regular verbs can't move. So English adds       │
│   "do/does" as a helper to carry the inversion.                 │
│                                                                 │
│   ※ BE verb does NOT need do-support!                          │
│     "Are you Korean?" (NOT "Do you are Korean?")                │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 Yes/No Questions

```
Formula: Do/Does + subject + base verb + ...?

I eat pizza.           → Do I eat pizza?
You like music.        → Do you like music?
He works here.         → Does he work here?     (he → does, verb loses -s)
She speaks Korean.     → Does she speak Korean?  (speaks → speak)
It costs $10.          → Does it cost $10?
We live in Seoul.      → Do we live in Seoul?
They play soccer.      → Do they play soccer?

※ CRITICAL: After "does", the verb returns to BASE FORM!
  Does she speakS? ✗ → Does she speak? ✓
  Does he goES?    ✗ → Does he go?     ✓
```

### 5.3 Wh-Questions

```
Formula: Wh-word + do/does + subject + base verb + ...?

Where do you live?           너는 어디에 사니?
What does she eat?           그녀는 뭘 먹니?
When do they arrive?         그들은 언제 도착하니?
How do you go to work?       너는 어떻게 출근하니?
Why does he study English?   그는 왜 영어를 공부하니?
How often do you exercise?   너는 얼마나 자주 운동하니?

※ Exception: When "who/what" IS the subject → NO do-support!
  "Who lives here?"        (NOT "Who does live here?")
  "What happens next?"     (NOT "What does happen next?")
  ← Because "who/what" is already the subject
```

### 5.4 Short Answers

```
Do you like coffee?     → Yes, I do. / No, I don't.
Does she work here?     → Yes, she does. / No, she doesn't.
Do they speak Korean?   → Yes, they do. / No, they don't.

※ Korean answer pattern is different:
  "커피 좋아해?" → "응" / "아니"
  English needs the auxiliary: "Yes, I DO." / "No, I DON'T."
  Just "Yes." / "No." sounds abrupt (but is technically OK).
```

## 6. Do-Support: Negatives

### 6.1 Forming Negatives

```
Formula: Subject + do/does + not + base verb + ...

I eat meat.             → I do not eat meat.        / I don't eat meat.
You like spicy food.    → You do not like spicy food./ You don't like spicy food.
He works on Sunday.     → He does not work on Sunday./ He doesn't work on Sunday.
She speaks Chinese.     → She does not speak Chinese./ She doesn't speak Chinese.
It costs a lot.         → It does not cost a lot.   / It doesn't cost a lot.
We live in Busan.       → We do not live in Busan.  / We don't live in Busan.
They play tennis.       → They do not play tennis.  / They don't play tennis.

※ Same rule: After "does not/doesn't", verb returns to BASE FORM!
  She doesn't speakS ✗ → She doesn't speak ✓
```

### 6.2 Korean vs English Negation

```
Korean negation: 안 + verb  OR  verb + 지 않다
  안 먹는다  /  먹지 않는다

English negation: subject + don't/doesn't + base verb
  I don't eat.

  ※ Korean can put 안 BEFORE the verb.
    English puts "don't" BETWEEN subject and verb.
    Korean: 나는 안 먹는다        (S + 안 + V)
    English: I don't eat.         (S + don't + V)

  ※ Never use double negatives in standard English:
    "I don't eat nothing." ✗
    "I don't eat anything." ✓
    (Korean: 나는 아무것도 안 먹는다 — double negative is normal!)
```

## 7. Adverbs of Frequency

### 7.1 Frequency Scale

```
┌─────────────────────────────────────────────────────────────────┐
│   always       (항상)           100%   ████████████████████     │
│   usually      (보통)            90%   ██████████████████       │
│   often        (자주)            70%   ████████████████         │
│   sometimes    (가끔)            50%   ████████████             │
│   occasionally (때때로)           30%   ████████                 │
│   rarely       (거의 안)          10%   ████                     │
│   seldom       (좀처럼 안)        10%   ████                     │
│   never        (절대 안)           0%   (none)                   │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 Position in Sentence

```
Frequency adverbs go BEFORE the main verb but AFTER "be":

Before main verb:
  I always eat breakfast.        나는 항상 아침을 먹는다.
  She usually walks to work.     그녀는 보통 걸어서 출근한다.
  They never drink alcohol.      그들은 절대 술을 안 마신다.
  He sometimes plays guitar.     그는 가끔 기타를 친다.

After BE verb:
  I am always hungry.            나는 항상 배고프다.
  She is usually on time.        그녀는 보통 제시간에 온다.
  They are never late.           그들은 절대 늦지 않는다.

  ※ Korean puts frequency words at the beginning or before verb.
    English puts them in a specific middle position.
    "I always eat" ✓   "Always I eat" ✗   "I eat always" ✗
```

## 8. Simple Present Uses

### 8.1 Summary of All Uses

```
1. Habits/Routines:    "I wake up at 7 every day."
2. Facts/Truths:       "The sun rises in the east."
3. Permanent states:   "I live in Seoul."
4. Schedules:          "The bus arrives at 3:15."
5. Instructions:       "You mix the flour and eggs."
6. Sports commentary:  "He passes to Kim, Kim shoots, goal!"
7. Storytelling:       "So she walks in and says..."

※ Simple present does NOT describe actions happening RIGHT NOW!
  "I eat lunch." = I eat lunch (every day, as a habit)
  "I am eating lunch." = I am eating lunch (right now)
  → Korean doesn't always distinguish these clearly.
```

## Quick Reference

```
┌─────────────────────────────────────────────────────────────────┐
│            Simple Present Quick Guide                            │
│                                                                 │
│   Form: I/you/we/they + base verb                               │
│         he/she/it + base verb + S                               │
│                                                                 │
│   -s rules: most → +s, -sh/-ch/-x/-ss/-o → +es,               │
│             consonant+y → ies, have → has                       │
│                                                                 │
│   Questions: Do/Does + subject + base verb?                     │
│   Negatives: Subject + don't/doesn't + base verb               │
│   ※ After does/doesn't → base verb (NO -s!)                   │
│                                                                 │
│   Uses: habits, facts, schedules, permanent states              │
│   ※ NOT for actions happening right now (→ present continuous) │
│                                                                 │
│   Key collocations:                                             │
│   make: breakfast, money, decision, friends, mistake            │
│   do: homework, dishes, laundry, research, exercise             │
│   take: shower, photo, break, test, medicine, notes             │
│   have: breakfast, meeting, headache, fun                       │
│   get: up, ready, married, lost, better, home                   │
└─────────────────────────────────────────────────────────────────┘
```
