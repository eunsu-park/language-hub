# English — Overview

## About This Topic

This topic provides a comprehensive, progressive curriculum for learning English designed specifically for Korean speakers (한국어 화자를 위한 영어 학습). The 25 lessons are organized into 5 stages that build upon each other, moving from foundational concepts to practical real-world application.

English is spoken by approximately 1.5 billion people worldwide, making it the most widely used language for international communication, science, technology, business, and media.

## Prerequisites

- No prior English knowledge required (the course begins from zero)
- Native or fluent Korean is assumed — explanations reference Korean grammar for comparison
- Basic understanding of grammar concepts in Korean is helpful

## Difficulty

- **Tier**: Beginner (starts from scratch, progresses to advanced)
- **Estimated scope**: A1 through B2/C1 CEFR levels

## Why English is Challenging for Korean Speakers (한국어 화자가 영어를 어려워하는 이유)

```
┌─────────────────────────────────────────────────────────────────┐
│           Top 5 Challenges for Korean Speakers                  │
│                                                                 │
│   1. Word Order: Korean SOV → English SVO                       │
│      나는 사과를 먹는다 → I eat an apple                          │
│                                                                 │
│   2. Articles (a/an/the): Korean has NO articles                │
│      This is the #1 hardest grammar point                       │
│                                                                 │
│   3. Pronunciation: R/L, th, f/v, stress patterns               │
│      Sounds that don't exist in Korean                          │
│                                                                 │
│   4. Tense System: 12 tenses vs Korean 3                        │
│      Present perfect is especially difficult                    │
│                                                                 │
│   5. Prepositions: in/on/at vs Korean postpositions             │
│      에, 에서, 으로 don't map 1:1 to English                     │
└─────────────────────────────────────────────────────────────────┘
```

## Learning Path

```
Stage 1: Foundations (01-05)  [CEFR A1]
  Build the basics: pronunciation, greetings, articles, essential verbs, word order
  ↓
Stage 2: Elementary (06-10)  [CEFR A2]
  Expand: past tense, prepositions, adjectives, numbers, daily vocabulary
  ↓
Stage 3: Intermediate Grammar (11-15)  [CEFR B1]
  Master verb system: continuous, perfect tenses, modals, passive, full tense system
  ↓
Stage 4: Advanced (16-20)  [CEFR B2]
  Refine: relative clauses, conditionals, phrasal verbs, discourse, vocabulary
  ↓
Stage 5: Practical Application (21-25)  [CEFR B2-C1]
  Apply: reading, writing, culture, business English, exam preparation
```

## How to Use This Course

1. **Follow the order**: Lessons build on each other. Complete each stage before moving to the next.
2. **Do the exercises**: Every lesson includes practice exercises with answers. Work through them.
3. **Review regularly**: Language learning requires repetition. Review previous lessons periodically.
4. **Practice actively**: Don't just read — write out sentences, say words aloud, use new vocabulary.
5. **Compare with Korean**: Each lesson highlights Korean-English differences. Use your Korean knowledge as a bridge.
6. **Reference lessons**: Lessons 15 (tense system), 18 (phrasal verbs), and 20 (vocabulary) serve as useful references to return to.

## Korean-English Quick Comparison (한국어-영어 빠른 비교)

```
┌────────────────────────────────────────────────────────────────────┐
│   Feature          │ Korean (한국어)         │ English              │
│────────────────────│────────────────────────│──────────────────────│
│   Word Order       │ SOV (주어-목적어-동사)   │ SVO (주어-동사-목적어)│
│   Articles         │ None (없음)            │ a/an/the (관사)       │
│   Plurals          │ Usually unmarked       │ -s/-es required       │
│   Verb Tenses      │ ~3 basic tenses        │ 12 tenses             │
│   Prepositions     │ Postpositions (조사)    │ Prepositions          │
│   Honorifics       │ Complex (존댓말)        │ Minimal               │
│   Subject Pronouns │ Often dropped          │ Required              │
│   Relative Clauses │ Before noun            │ After noun            │
│   Adjectives       │ Before noun + 한/ㄴ     │ Before noun           │
│   Question Form    │ Add -까/-니 to verb     │ Invert / do-support   │
└────────────────────────────────────────────────────────────────────┘
```
