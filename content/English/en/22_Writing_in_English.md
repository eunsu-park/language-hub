## 1. Introduction to English Writing

### 1.1 Korean vs English Writing Style

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean Writing Pattern (기-승-전-결):                          │
│   Introduction → Development → Turn → Conclusion               │
│   Indirect approach: background first, main point at the end   │
│                                                                 │
│   English Writing Pattern:                                     │
│   Main point first → Support → Details → Conclusion            │
│   Direct approach: thesis statement up front                   │
│                                                                 │
│   KEY DIFFERENCE:                                              │
│   Korean readers expect the main point LATER                   │
│   English readers expect the main point FIRST                  │
│                                                                 │
│   Korean: "The weather was bad, and traffic was terrible,      │
│            and many people were affected, so I was late."       │
│   English: "I was late because of bad weather and traffic."    │
│            (Main point first, reasons after)                   │
└─────────────────────────────────────────────────────────────────┘
```

## 2. Email Writing

### 2.1 Email Format

```
┌─────────────────────────────────────────────────────────────────┐
│   To:      recipient@email.com                                  │
│   Cc:      copied.person@email.com                              │
│   Subject: Clear, specific subject line                        │
│                                                                 │
│   Greeting,                                                    │
│                                                                 │
│   Opening line (purpose of the email).                         │
│                                                                 │
│   Body paragraphs (details, requests, information).            │
│                                                                 │
│   Closing line (action items, thanks, next steps).             │
│                                                                 │
│   Sign-off,                                                    │
│   Your Name                                                    │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 Formal Email Example

```
Subject: Request for Meeting — Q1 Budget Review

Dear Mr. Johnson,

I hope this email finds you well.

I am writing to request a meeting to discuss the Q1 budget
review. Could we schedule a 30-minute meeting sometime
next week? I am available on Tuesday and Thursday afternoon.

Please find the preliminary budget report attached for
your review.

I look forward to hearing from you.

Best regards,
Minjun Kim
Financial Analyst, ABC Corporation
```

### 2.3 Informal Email Example

```
Subject: Dinner this Saturday?

Hey Sarah,

How's it going?

A few of us are planning to grab dinner this Saturday
at that new Italian place downtown. Want to join?
We're thinking around 7 PM.

Let me know if you can make it!

Cheers,
Jihye
```

### 2.4 Greetings and Sign-offs

```
FORMAL GREETINGS:
  Dear Mr./Ms./Dr. [Last Name],
  Dear Sir/Madam, (when name unknown)
  To Whom It May Concern, (very formal, name unknown)
  Dear Hiring Manager, (job applications)

SEMI-FORMAL:
  Dear [First Name],
  Hello [First Name],

INFORMAL:
  Hi [Name],
  Hey [Name],

FORMAL SIGN-OFFS:
  Sincerely, / Yours sincerely,
  Best regards, / Kind regards,
  Respectfully,

SEMI-FORMAL:
  Best, / All the best,
  Thank you, / Thanks,
  Regards,

INFORMAL:
  Cheers, / Talk soon,
  Take care, / See you,
```

### 2.5 Useful Email Phrases

```
OPENING:
  "I am writing to inquire about..."
  "I am writing in reference to..."
  "Thank you for your email regarding..."
  "Following up on our conversation..."
  "I hope this email finds you well."

REQUESTING:
  "Could you please...?"
  "I would appreciate it if you could..."
  "Would it be possible to...?"
  "I was wondering if you could..."

ATTACHING FILES:
  "Please find attached..."
  "I have attached the document for your reference."
  "The requested file is attached."

APOLOGIZING:
  "I apologize for the delay in responding."
  "I'm sorry for any inconvenience caused."
  "Please accept my apologies for..."

CLOSING:
  "I look forward to hearing from you."
  "Please don't hesitate to contact me if you have questions."
  "Thank you for your time and consideration."
  "I would be happy to discuss this further."
```

## 3. Essay Structure

### 3.1 The Five-Paragraph Essay

```
┌─────────────────────────────────────────────────────────────────┐
│   Paragraph 1: INTRODUCTION                                   │
│   • Hook (attention-grabbing opening)                          │
│   • Background information                                    │
│   • Thesis statement (main argument — last sentence)          │
│                                                                 │
│   Paragraph 2: BODY PARAGRAPH 1                               │
│   • Topic sentence (first supporting point)                   │
│   • Evidence / examples                                       │
│   • Analysis / explanation                                    │
│                                                                 │
│   Paragraph 3: BODY PARAGRAPH 2                               │
│   • Topic sentence (second supporting point)                  │
│   • Evidence / examples                                       │
│   • Analysis / explanation                                    │
│                                                                 │
│   Paragraph 4: BODY PARAGRAPH 3                               │
│   • Topic sentence (third supporting point)                   │
│   • Evidence / examples                                       │
│   • Analysis / explanation                                    │
│                                                                 │
│   Paragraph 5: CONCLUSION                                     │
│   • Restate thesis (in different words)                       │
│   • Summarize key points                                      │
│   • Final thought / call to action / broader implication      │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Writing a Strong Thesis Statement

```
A thesis statement:
  • States the MAIN ARGUMENT of your essay
  • Appears at the END of the introduction
  • Is SPECIFIC (not vague)
  • Is ARGUABLE (not just a fact)
  • Previews the structure of the essay

Weak thesis:
  "Pollution is bad."
  (Too vague. Everyone agrees. Not arguable.)

Strong thesis:
  "Urban air pollution can be significantly reduced through
   three key measures: expanding public transportation,
   implementing stricter emissions standards, and increasing
   green spaces in cities."
  (Specific, arguable, previews three body paragraphs.)
```

### 3.3 Types of Hooks

```
Question:       "Have you ever wondered why some languages
                 are easier to learn than others?"

Statistic:      "According to a 2024 study, 1.5 billion people
                 are currently learning English worldwide."

Quote:          "As Nelson Mandela once said, 'If you talk to a
                 man in a language he understands, that goes to
                 his head. If you talk to him in his own language,
                 that goes to his heart.'"

Anecdote:       "When I first arrived in New York, I couldn't
                 order a cup of coffee without pointing."

Bold statement: "English is the most important skill a Korean
                 student can develop today."
```

## 4. Paragraph Organization

### 4.1 The PEEL Method

```
P — Point:       State the main idea (topic sentence)
E — Evidence:    Provide facts, data, or examples
E — Explanation: Analyze the evidence, explain its significance
L — Link:        Connect back to the thesis or transition

Example:
  [P] Regular exercise has been shown to improve mental health.
  [E] A study by Harvard Medical School found that 30 minutes
      of moderate exercise three times per week reduced symptoms
      of depression by up to 47%.
  [E] This suggests that physical activity triggers the release
      of endorphins and serotonin, chemicals that directly
      affect mood and well-being.
  [L] Therefore, incorporating exercise into daily routines
      should be a priority for anyone seeking to improve their
      mental health.
```

### 4.2 Transition Words Between Paragraphs

```
Adding:         Furthermore, Moreover, In addition, Similarly
Contrasting:    However, On the other hand, In contrast, Nevertheless
Cause/Effect:   Therefore, Consequently, As a result, Thus
Example:        For example, For instance, Specifically
Time:           First, Next, Then, Finally, Meanwhile
Concluding:     In conclusion, To sum up, Overall, In summary
```

## 5. Punctuation Rules

### 5.1 Period (.)

```
• End of a declarative sentence: "I live in Seoul."
• Abbreviations: Dr., Mr., Mrs., Jr., etc.
• Note: Modern style often omits periods in abbreviations:
  US, UK, PhD, CEO, FBI
```

### 5.2 Comma (,)

```
1. Lists:
   "I bought apples, oranges, and bananas."
   (Oxford comma before "and" is recommended but optional)

2. Before coordinating conjunctions joining independent clauses:
   "I was tired, but I kept working."

3. After introductory elements:
   "However, the results were surprising."
   "In 2020, the world changed dramatically."
   "After finishing dinner, we went for a walk."

4. Non-essential information:
   "My sister, who lives in London, is visiting."
   "Seoul, the capital of South Korea, is vibrant."

5. Direct address:
   "John, can you help me?"
   "Thank you, everyone, for coming."

6. Dates:
   "March 7, 2026"  BUT  "7 March 2026" (no comma in British)

COMMON ERROR:
  ✗ Comma splice: "I was tired, I went to bed."
  ✓ "I was tired, so I went to bed."
  ✓ "I was tired. I went to bed."
  ✓ "I was tired; I went to bed."
```

### 5.3 Semicolon (;)

```
1. Join two related independent clauses (without a conjunction):
   "She studied hard; she passed the exam."

2. Before adverbs like however, therefore, moreover:
   "I was tired; however, I kept working."

3. In complex lists:
   "We visited Seoul, South Korea; Tokyo, Japan; and
    Beijing, China."
```

### 5.4 Colon (:)

```
1. Before a list:
   "You need three things: pen, paper, and patience."

2. Before an explanation:
   "There was one problem: we had no money."

3. Before a quote (formal):
   "The teacher said: 'Study hard.'"

4. Time:
   "The meeting starts at 3:00 PM."
```

### 5.5 Apostrophe (')

```
1. Contractions:
   don't, can't, I'm, she's, they're, it's (it is)

2. Possession:
   "John's book" / "the students' projects" (plural possessive)
   "the children's toys" (irregular plural)

COMMON ERROR:
  its = possessive ("The cat licked its paw.")
  it's = it is ("It's raining.")
  ✗ "The dog wagged it's tail." (WRONG)
  ✓ "The dog wagged its tail."
```

### 5.6 Quotation Marks (" ")

```
1. Direct speech:
   "She said, 'I'm coming tomorrow.'"

2. Titles of articles, chapters, songs:
   "I read an article called 'The Future of AI.'"

3. Highlighting unusual usage:
   "The so-called 'expert' had no qualifications."

American vs British:
  American: "double quotes" primary, 'single' inside
  British:  'single quotes' primary, "double" inside
```

## 6. Capitalization Rules

### 6.1 When to Capitalize

```
1. First word of a sentence:
   "The meeting starts at 9."

2. Proper nouns (names of specific people, places, organizations):
   "Kim Minjun lives in Seoul, South Korea."
   "She works at Samsung Electronics."

3. Titles before names:
   "President Yoon" / "Dr. Park" / "Professor Lee"

4. Days, months, holidays (NOT seasons):
   "Monday, January, Christmas"
   "summer, winter, spring, fall" (lowercase)

5. Nationalities, languages, religions:
   "Korean, English, Buddhism, Christianity"

6. Titles of works (capitalize major words):
   "The Lord of the Rings"
   "To Kill a Mockingbird"

7. The pronoun "I":
   "My friend and I went to the park."

NOT capitalized (unlike Korean conventions):
  ✗ "I study English every Day."
  ✓ "I study English every day."
  ✗ "She is a Doctor."
  ✓ "She is a doctor." (no capital for general occupations)
```

## 7. Common Writing Mistakes by Korean Speakers

### 7.1 Top 10 Writing Errors

```
1. ARTICLE ERRORS (a/an/the/zero article)
   ✗ "I want to be doctor."
   ✓ "I want to be a doctor."
   ✗ "I went to the school yesterday."
   ✓ "I went to school yesterday."
   (Korean has no articles — this is the #1 error)

2. SUBJECT-VERB AGREEMENT
   ✗ "The students is studying."
   ✓ "The students are studying."
   ✗ "Everyone have a book."
   ✓ "Everyone has a book."

3. WORD ORDER (Korean SOV → English SVO)
   ✗ "I yesterday the movie watched."
   ✓ "I watched the movie yesterday."
   ✗ "Very much I like English."
   ✓ "I like English very much."

4. PLURALS (Korean doesn't mark plurals the same way)
   ✗ "I have three book."
   ✓ "I have three books."
   ✗ "Many student came."
   ✓ "Many students came."

5. PRONOUN CONFUSION
   ✗ "My sister, he is a teacher."
   ✓ "My sister, she is a teacher."
   (Korean 그/그녀 is less strictly differentiated)

6. TENSE CONSISTENCY
   ✗ "Yesterday I go to the store and buy some food."
   ✓ "Yesterday I went to the store and bought some food."

7. PREPOSITION ERRORS
   ✗ "I'm interested for music."
   ✓ "I'm interested in music."
   ✗ "She arrived to the airport."
   ✓ "She arrived at the airport."

8. COMMA SPLICES
   ✗ "I was tired, I went to bed."
   ✓ "I was tired, so I went to bed."

9. INDIRECT WRITING STYLE
   ✗ Starting essays with long background before the point
   ✓ State the main point (thesis) clearly in the introduction

10. RUN-ON SENTENCES
    ✗ "I went to Seoul and I met my friend and we had lunch
       and then we went shopping and after that we went home."
    ✓ "I went to Seoul and met my friend. We had lunch and
       went shopping. After that, we went home."
```

## 8. Academic Writing Tips

### 8.1 Do's and Don'ts

```
DO:
  ✓ Use formal language (avoid contractions, slang)
  ✓ Write in third person (avoid "I think" in formal essays)
  ✓ Use hedging language ("may," "suggests," "appears to")
  ✓ Cite sources properly
  ✓ Use topic sentences in every paragraph
  ✓ Proofread carefully

DON'T:
  ✗ Use contractions (don't → do not)
  ✗ Use "etc." or "and so on" (be specific)
  ✗ Start sentences with "And," "But," "So" (in formal writing)
  ✗ Use emotional language or exclamation marks
  ✗ Use "a lot of" (use "numerous," "many," "significant")
  ✗ Use first person unless the task allows it
```

### 8.2 Useful Academic Phrases

```
INTRODUCING:
  "This essay examines..."
  "The purpose of this paper is to..."
  "This study investigates the relationship between..."

PRESENTING EVIDENCE:
  "According to Smith (2023),..."
  "Research conducted by Kim (2022) demonstrates that..."
  "The data suggests that..."

ANALYZING:
  "This finding indicates that..."
  "It can be inferred that..."
  "A possible explanation for this is..."

CONCLUDING:
  "In conclusion, this study has shown that..."
  "The evidence presented suggests that..."
  "Further research is needed to..."
```

## 9. Editing and Proofreading

### 9.1 Self-Editing Checklist

```
CONTENT:
  □ Does the introduction have a clear thesis statement?
  □ Does each body paragraph have a topic sentence?
  □ Is there enough evidence to support each point?
  □ Does the conclusion summarize and provide closure?

ORGANIZATION:
  □ Do paragraphs flow logically?
  □ Are transition words used between paragraphs?
  □ Is the main point stated early (not buried at the end)?

GRAMMAR:
  □ Subject-verb agreement?
  □ Correct tense usage and consistency?
  □ Correct article usage (a/an/the)?
  □ Correct prepositions?

MECHANICS:
  □ Correct spelling?
  □ Correct punctuation (commas, periods, apostrophes)?
  □ Correct capitalization?
  □ No comma splices or run-on sentences?

STYLE:
  □ Appropriate register (formal/informal)?
  □ No unnecessary repetition?
  □ Varied sentence structure?
  □ Clear and concise language?
```

### 9.2 Proofreading Tips

```
1. Read your work ALOUD — your ear catches errors your eyes miss
2. Read BACKWARDS (sentence by sentence) to catch spelling errors
3. Focus on ONE type of error at a time (first tenses, then articles, etc.)
4. Take a BREAK before proofreading (fresh eyes find more errors)
5. Use a CHECKLIST (like the one above)
6. Have SOMEONE ELSE read it
```
