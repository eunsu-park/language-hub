```

│   Native speakers:    ~400 million                              │

│   Official language in: ~60 countries                           │

│   • Canadian English (CaE) — ~38 million                       │
│   • Indian English (InE) — ~125 million L2                     │

│   • Nigerian English — ~80 million L2                          │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 No "Right" English

```
Important principle:
  No variety of English is "more correct" than another.
  American, British, Australian, Indian English — all are valid.

However:
  • Be CONSISTENT within a single document
    (Don't mix American and British spelling)
  • Know your AUDIENCE
    (Academic papers in the UK should use British English)
  • Most international contexts use American English
    (due to media, technology, and business influence)
  • Korean standardized tests (TOEIC, TOEFL) use American English
  • IELTS accepts both American and British English
```

## 4. American vs British: Grammar

### 4.1 Key Grammar Differences

```
┌─────────────────────────────────────────────────────────────────┐
│   1. PRESENT PERFECT vs SIMPLE PAST                            │
│                                                                 │
│   BrE: "I've just eaten." / "Have you seen the news?"          │
│   AmE: "I just ate."      / "Did you see the news?"            │
│                                                                 │
│   American English uses Simple Past more often where            │
│   British English uses Present Perfect.                         │
│                                                                 │
│   2. COLLECTIVE NOUNS                                          │
│                                                                 │
│   BrE: "The team are playing well." (plural verb OK)           │
│   AmE: "The team is playing well." (singular verb)             │
│                                                                 │
│   British English can treat collective nouns as plural.         │
│   American English treats them as singular.                     │
│                                                                 │
│   3. "GOT" vs "GOTTEN"                                         │
│                                                                 │
│   BrE: "I've got a car." (have/possess)                        │
│   AmE: "I've gotten better." (become/obtained)                  │
│   AmE: "I've got a car." (also used for possession)            │
│                                                                 │
│   "Gotten" is American only. British English uses "got"         │
│   for all meanings.                                             │
│                                                                 │
│   4. "SHALL" vs "WILL"                                         │
│                                                                 │
│   BrE: "Shall we go?" (suggestion — common)                   │
│   AmE: "Should we go?" or "Do you want to go?"                │
│         ("Shall" is rare in American English)                   │
│                                                                 │
│   5. "HAVE" as a main verb                                     │
│                                                                 │
│   BrE: "Have you got a pen?"   / "I haven't got time."        │
│   AmE: "Do you have a pen?"    / "I don't have time."         │
│                                                                 │
│   6. PREPOSITIONS                                              │
│                                                                 │
│   AmE: "on the weekend"        BrE: "at the weekend"          │
│   AmE: "Monday through Friday" BrE: "Monday to Friday"        │
│   AmE: "different from/than"   BrE: "different from/to"       │
│   AmE: "write (someone)"      BrE: "write to (someone)"      │
└─────────────────────────────────────────────────────────────────┘
```

## 5. American vs British: Pronunciation

### 5.1 Key Sound Differences

```
1. R-pronunciation (rhotic vs non-rhotic):
   AmE: pronounces /r/ in all positions (rhotic)
        "car" /kɑːr/, "water" /wɔːtər/, "here" /hɪr/
   BrE: drops /r/ except before vowels (non-rhotic)
        "car" /kɑː/, "water" /wɔːtə/, "here" /hɪə/

2. /æ/ vs /ɑː/ (bath words):
   AmE: "bath" /bæθ/, "can't" /kænt/, "dance" /dæns/
   BrE: "bath" /bɑːθ/, "can't" /kɑːnt/, "dance" /dɑːns/

3. T-flapping (American):
   AmE: "water" /wɔːdər/, "better" /bedər/, "city" /sɪdi/
   BrE: "water" /wɔːtə/, "better" /betə/, "city" /sɪti/
   (In AmE, /t/ between vowels sounds like a quick /d/)

4. /ɒ/ vs /ɑː/ (lot words):
   AmE: "hot" /hɑːt/, "not" /nɑːt/, "lot" /lɑːt/
   BrE: "hot" /hɒt/, "not" /nɒt/, "lot" /lɒt/

5. Stress differences:
   AmE: AD-dress (noun)     BrE: a-DRESS (both)
   AmE: ga-RAGE             BrE: GA-rage
   AmE: week-END            BrE: WEEK-end
```

## 6. Australian English

### 6.1 Key Features

```
Pronunciation:
  • "day" → sounds like /daɪ/
  • Rising intonation at end of statements
    (sounds like asking a question)
  • "i" sound → /ɔɪ/: "right" sounds like "roight"

Vocabulary:
  arvo = afternoon           "See you this arvo."
  barbie = barbecue          "Let's have a barbie."
  brekkie = breakfast        "What's for brekkie?"
  cheers = thanks            "Cheers, mate."
  mate = friend              "G'day, mate."
  no worries = no problem    "No worries, mate."
  reckon = think             "I reckon it'll rain."
  thongs = flip-flops        "Grab your thongs for the beach."
  ute = pickup truck         "Load it into the ute."

Shortened words (very common):
  afternoon → arvo           university → uni
  breakfast → brekkie        barbecue → barbie
  mosquito → mozzie          sunglasses → sunnies
  Australian → Aussie        documentation → doco
```

## 7. Indian English

### 7.1 Key Features

```
Vocabulary (unique to Indian English):
  lakh = 100,000             "The car costs 5 lakhs."
  crore = 10,000,000         "Revenue of 2 crores."
  prepone = move earlier     "Let's prepone the meeting."
    (opposite of postpone — not used in other varieties)
  do the needful = take      "Please do the needful."
    the necessary action
  passed out = graduated     "I passed out of IIT."
    (In other varieties: fainted)
  revert = reply             "Please revert at the earliest."
    (Other varieties: revert = go back to)

Grammar features:
  • Progressive with stative verbs:
    "I am having a car." (standard: "I have a car.")
  • Tag questions with "isn't it?" or "no?":
    "You're coming, isn't it?" / "He's late, no?"
  • "Itself" for emphasis:
    "Today itself we should finish." (= today, not later)
  • Reduplication for emphasis:
    "Come come, sit sit." / "It was small small pieces."
```

## 8. Other Varieties

### 8.1 Canadian English

```
  • Spelling: mix of British and American
    (colour but organize, centre but realize)
  • "eh?" as a tag question: "Nice day, eh?"
  • "about" → sounds like "aboot" to non-Canadians
  • "sorry" → pronounced more like "sore-ee"
  • toque = knit hat
  • loonie/toonie = 1/2 dollar coins
```

### 8.2 South African English

```
  • "just now" = in a while (not immediately!)
  • "now now" = soon
  • "robot" = traffic light
  • "braai" = barbecue
  • "howzit" = hello / how are you
  • Influenced by Afrikaans and African languages
```

### 8.3 Singapore English (Singlish)

```
  • "lah" particle at end of sentences: "Can lah!"
  • "can" or "cannot" as complete answers
  • Influenced by Chinese, Malay, Tamil
  • "Where got?" = "That's not true"
  • "steady lah" = well done
```

## 9. Cultural Communication Norms

### 9.1 High-Context vs Low-Context Communication

```
┌─────────────────────────────────────────────────────────────────┐
│   HIGH-CONTEXT (Korean, Japanese, Chinese):                    │
│   • Meaning is implied, not directly stated                    │
│   • Context, tone, and relationship matter more than words     │
│   • "Reading the air" (눈치) is important                      │
│   • Indirect refusals: "That might be difficult..."            │
│                                                                 │
│   LOW-CONTEXT (American, German, Australian):                  │
│   • Meaning is explicitly stated                               │
│   • Words carry the message directly                           │
│   • "Say what you mean, mean what you say"                     │
│   • Direct refusals: "No, I can't do that."                    │
│                                                                 │
│   MEDIUM-CONTEXT (British):                                    │
│   • More indirect than American but more direct than Korean    │
│   • Uses politeness strategies and understatement              │
│   • "That's an interesting idea" may mean "I disagree"         │
└─────────────────────────────────────────────────────────────────┘
```

### 9.2 Cultural Tips by Country

```
UNITED STATES:
  • Small talk is important (weather, sports, weekend plans)
  • First names used quickly, even in business
  • "How are you?" is a greeting, not a real question
    Answer: "Good, thanks!" (not a long answer)
  • Directness is valued — say what you mean
  • Positive feedback is common ("Great job!")

UNITED KINGDOM:
  • Understatement and irony are common
  • "Not bad" often means "quite good"
  • "I'll bear it in mind" may mean "I won't do it"
  • Queuing (standing in line) is sacred
  • Tea-related social rituals matter

AUSTRALIA:
  • Very casual and informal culture
  • Tall poppy syndrome (don't show off)
  • Self-deprecating humor is common
  • "How are you going?" = "How are you?"
  • Mateship (friendship, loyalty) is valued
```

## 10. Directness vs Indirectness

### 10.1 The British Understatement

```
What a British person says    vs    What they mean:
─────────────────────────────────────────────────────
"That's quite good."               "That's excellent."
"That's interesting."              "I disagree / that's strange."
"I hear what you say."             "I disagree completely."
"With the greatest respect..."     "I think you're wrong."
"I'll bear it in mind."           "I've already forgotten it."
"That's a very brave proposal."    "You're crazy."
"I'm sure it's my fault."         "It's definitely your fault."
"I almost agree."                  "I don't agree at all."
"Could we consider alternatives?"  "Your idea won't work."
"Not bad."                         "Quite good / excellent."
```

### 10.2 Korean Communication Style in English

```
Korean speakers should be aware that:

1. Saying "yes" when you mean "maybe" or "no" confuses
   English speakers. Be clearer:
   Instead of: "Yes..." (meaning 글쎄요)
   Say: "I'll think about it." / "I'm not sure yet."

2. Not saying "no" directly can cause misunderstandings:
   Instead of: silence or vague response
   Say: "I'm afraid I can't." / "Unfortunately, that won't work."

3. Self-deprecation is misunderstood:
   Korean: "My English is not good" (겸손)
   English speakers take this literally.
   Better: "I'm still working on improving my English."

4. Apologizing too much can seem insincere:
   Korean speakers say "sorry" (죄송합니다) very often.
   In English, excessive apology weakens the word.
   Use "excuse me" or "thank you" instead when appropriate:
   Instead of: "Sorry for being late."
   Try: "Thank you for waiting."
```

### Exercise 4: Cultural Awareness

Choose the most appropriate response for the cultural context.

1. American colleague: "How are you?" — You should:
   a) Give a detailed answer about your health
   b) Say "Good, thanks! And you?"
   c) Say nothing

2. British colleague says "That's quite brave of you" about your idea:
   a) Say "Thank you!" enthusiastically
   b) Consider that they might think it's risky/foolish
   c) Ignore the comment

3. You disagree with your Australian colleague's suggestion:
   a) Say nothing (Korean style)
   b) Say "I think that's a bad idea"
   c) Say "That's a good point, but have you considered...?"

<details>
<summary>Click to reveal answers</summary>

```
1. b) "Good, thanks! And you?" — This is a greeting, not a
   real inquiry. Keep it short and positive.

2. b) Consider that they might think it's risky/foolish.
   British "brave" often means "risky" or "unwise" in this context.

3. c) "That's a good point, but have you considered...?"
   This acknowledges their idea while expressing your view.
   Being respectfully direct is appropriate in most English-
   speaking work environments.
```
