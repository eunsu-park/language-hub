## 1. Family

```
┌─────────────────────────────────────────────────────────────────┐
│   English           Korean        Notes                         │
│   ─────────         ──────        ─────                         │
│   father / dad      아버지 / 아빠  "dad" = informal              │
│   mother / mom      어머니 / 엄마  "mom" (AmE), "mum" (BrE)     │
│   parents           부모님                                       │
│   son               아들                                        │
│   daughter           딸                                         │
│   brother            형/오빠/남동생  ※ Korean has 4 words!       │
│   sister             언니/누나/여동생 ※ Korean has 4 words!       │
│   husband            남편                                       │
│   wife               아내                                       │
│   children           자녀/아이들                                  │
│                                                                 │
│   ※ Korean distinguishes older/younger AND speaker's gender:   │
│     형 (older brother, male speaker)                            │
│     오빠 (older brother, female speaker)                         │
│     누나 (older sister, male speaker)                            │
│     언니 (older sister, female speaker)                          │
│     남동생 (younger brother)                                     │
│     여동생 (younger sister)                                      │
│                                                                 │
│   English has only: brother, sister (+ older/younger/big/little)│
│     "my older brother" or "my big brother"                      │
│     "my younger sister" or "my little sister"                   │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Extended Family

```
grandfather / grandpa    할아버지         grandmother / grandma  할머니
grandparents             조부모님         grandchildren          손주들
grandson                 손자             granddaughter          손녀
uncle                    삼촌/외삼촌       aunt                   이모/고모
nephew                   조카 (남)         niece                  조카 (여)
cousin                   사촌
father-in-law            장인/시아버지     mother-in-law          장모/시어머니
brother-in-law           처남/매형/시동생   sister-in-law         시누이/올케/형수

stepfather               새아버지         stepmother             새어머니
half-brother             이복형제         half-sister            이복자매

※ Korean has MORE specific family terms than English:
  Korean 삼촌 = father's brother, 외삼촌 = mother's brother
  English just says "uncle" for both!
  Korean 이모 = mother's sister, 고모 = father's sister
  English just says "aunt" for both!
```

## 2. Food and Cooking

### 2.1 Basic Food Categories

```
Meat (고기):           chicken, beef, pork, lamb, fish, seafood
Vegetables (채소):      carrot, onion, potato, tomato, lettuce, broccoli,
                       pepper, garlic, cucumber, spinach, mushroom
Fruits (과일):          apple, banana, orange, grape, strawberry, watermelon,
                       peach, mango, pineapple, lemon, cherry, pear
Grains (곡물):          rice, bread, pasta, noodles, cereal, oatmeal
Dairy (유제품):         milk, cheese, butter, yogurt, cream, eggs
Drinks (음료):          water, coffee, tea, juice, soda, beer, wine
```

### 2.2 Cooking Verbs

```
boil     끓이다     "Boil the water."          물을 끓여라.
fry      튀기다     "Fry the egg."             달걀을 부쳐라.
bake     굽다(오븐)  "Bake a cake."            케이크를 구워라.
grill    굽다(불)   "Grill the meat."          고기를 구워라.
steam    찌다       "Steam the vegetables."    야채를 쪄라.
roast    구워지다    "Roast the chicken."       닭을 구워라.
stir     젓다       "Stir the soup."           수프를 저어라.
chop     자르다     "Chop the onions."         양파를 썰어라.
slice    얇게 썰다  "Slice the bread."         빵을 썰어라.
peel     껍질을 벗기다 "Peel the potato."       감자 껍질을 벗겨라.
mix      섞다       "Mix the ingredients."     재료를 섞어라.
pour     붓다       "Pour the water."          물을 부어라.
```

### 2.3 Taste and Texture

```
Taste (맛):
  sweet (달다)      sour (시다)       salty (짜다)
  bitter (쓰다)     spicy/hot (맵다)   bland (싱겁다)
  savory (고소하다)  delicious (맛있다) terrible (맛없다)
  rich (진하다)      mild (순하다)

Texture (식감):
  crispy (바삭한)    soft (부드러운)    chewy (쫄깃한)
  crunchy (아삭한)   tender (연한)      tough (질긴)
  juicy (즙이 많은)  creamy (크리미한)   sticky (끈적한)
```

### 2.4 Restaurant English

```
"A table for two, please."          2인 테이블 부탁합니다.
"Can I see the menu?"               메뉴판 볼 수 있을까요?
"I'd like the steak, please."       스테이크를 주세요.
"How would you like your steak?"    스테이크 어떻게 구워 드릴까요?
  → rare / medium-rare / medium / well-done
"Could I have the check/bill?"      계산서 주세요.
"Is the tip included?"              팁이 포함되어 있나요?

※ Korean 서비스 (service) = English "free/complimentary"
  Korean 셀프 (self) = English "self-service"
```

## 3. Clothing

### 3.1 Common Clothing Items

```
Top:     shirt, T-shirt, blouse, sweater, hoodie, jacket, coat
Bottom:  pants (AmE) / trousers (BrE), jeans, shorts, skirt, dress
Shoes:   shoes, sneakers, boots, sandals, heels, slippers
Accessories: hat, cap, scarf, gloves, belt, tie, glasses, watch
Underwear: underwear, socks, bra

※ "pants" = trousers (AmE) but underwear (BrE)!
  American "pants" = British "trousers"
  British "pants" = American "underwear"
```

### 3.2 Clothing Verbs and Expressions

```
put on (입다/착용하다):     "Put on your coat."       코트를 입어.
take off (벗다):            "Take off your shoes."    신발을 벗어.
wear (입고 있다 — state):   "She's wearing a dress."  그녀는 원피스를 입고 있다.
try on (입어보다):          "Can I try this on?"      이거 입어봐도 돼요?
fit (맞다):                 "Does it fit?"            사이즈가 맞나요?
match (어울리다):           "This shirt matches."     이 셔츠가 어울린다.

※ "wear" vs "put on":
  "I'm wearing a hat." = I have it on right now (상태)
  "I'm putting on a hat." = I'm in the process of placing it on (동작)
```

### 3.3 Sizes and Shopping

```
Sizes: XS (extra small), S (small), M (medium), L (large), XL (extra large)

"What size are you?"                 사이즈가 어떻게 되세요?
"Do you have this in a medium?"      이거 미디엄 사이즈 있어요?
"It's too big/small/tight/loose."    너무 커요/작아요/꽉 끼어요/헐렁해요.
"How much is this?"                  이거 얼마예요?
"I'll take it."                      이걸로 할게요.
"Can I return this?"                 반품할 수 있나요?
```

## 7. Weather Expressions

### 7.1 Weather Vocabulary

```
sunny    화창한       cloudy    흐린        rainy     비 오는
windy    바람 부는    snowy     눈 오는     foggy     안개 낀
stormy   폭풍우의    humid     습한        dry       건조한
hot      더운        warm      따뜻한      cool      시원한
cold     추운        freezing  몹시 추운

"It's sunny today."          오늘 화창하다.
"It's raining."              비가 온다.
"It's snowing."              눈이 온다.
"It's windy."                바람이 분다.
```

### 7.2 Temperature

```
"It's hot."          덥다.
"It's warm."         따뜻하다.
"It's cool."         시원하다 / 선선하다.
"It's cold."         춥다.
"It's freezing."     몹시 춥다 / 꽁꽁 얼 정도로 춥다.
"It's 30 degrees."   30도이다.
"It's below zero."   영하이다.

※ English uses "it" as a dummy subject for weather:
  "It is raining." (비가 온다 — "it" has no real meaning)
  Korean: 비가 온다 (비 = rain, subject)
  English: "Rains." ✗ → "It rains." ✓ (must have "it")
```

### 7.3 Weather-Related Expressions

```
"What's the weather like?"     날씨가 어때?
"What's the forecast?"         일기 예보가 뭐야?
"It looks like rain."          비가 올 것 같다.
"The weather is getting better." 날씨가 좋아지고 있다.
"It's pouring."                비가 쏟아진다 / 폭우가 온다.
"It's drizzling."              이슬비가 온다.
"There's a thunderstorm."      천둥번개가 친다.
"It cleared up."               날이 개었다.
```

### 8.5 make vs do Quick Guide

```
┌─────────────────────────────────────────────────────────────────┐
│   MAKE = create / produce something new                         │
│   (결과물이 있는 것: 만들다)                                      │
│   make breakfast (아침을 만들다)                                  │
│   make a decision (결정을 만들다)                                 │
│   make money (돈을 만들다/벌다)                                   │
│                                                                 │
│   DO = perform an activity / task                               │
│   (활동을 수행하는 것: 하다)                                      │
│   do homework (숙제를 하다)                                      │
│   do the dishes (설거지를 하다)                                   │
│   do exercise (운동을 하다)                                      │
│                                                                 │
│   ※ "make homework" ✗ → "do homework" ✓                       │
│   ※ "do a mistake" ✗ → "make a mistake" ✓                     │
│   ※ "make the dishes" ✗ → "do the dishes" ✓                   │
│   These must be memorized — there is no perfect rule!          │
└─────────────────────────────────────────────────────────────────┘
```
