### 1.1 English as a Global Language

```
┌─────────────────────────────────────────────────────────────────┐
│                  English: A Global Language                      │
│                                                                 │
│   Native speakers: ~400 million                                 │
│   Total speakers:  ~1.5 billion (including L2)                  │
│   Official in:     67 countries                                 │
│   UN official language: Yes (1 of 6)                            │
│                                                                 │
│   Language family: Indo-European → Germanic → West Germanic     │
│   Major varieties: American (AmE), British (BrE), Australian    │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Why English Pronunciation is Hard for Korean Speakers

Unlike Spanish or Italian, English spelling is NOT phonetic. The same letter can have multiple sounds, and the same sound can be spelled multiple ways.

```
The word "ough" has 7+ pronunciations:
  through  /θruː/     (우 sound)
  though   /ðoʊ/      (오우 sound)
  tough    /tʌf/      (어프 sound)
  cough    /kɒf/      (오프 sound)
  thought  /θɔːt/     (오트 sound)
  bough    /baʊ/      (아우 sound)
  hiccough /ˈhɪkʌp/   (업 sound)
```

> **For Korean speakers (한국어 화자 참고)**: Korean (한글) has near-perfect letter-to-sound correspondence. English does NOT. You must learn each word's pronunciation individually — you cannot reliably guess from spelling alone.

### 1.3 IPA: Your Pronunciation Guide

The International Phonetic Alphabet (IPA) gives each sound exactly one symbol. This course uses IPA notation throughout.

```
Why IPA matters for Korean speakers:
  Korean spelling: 읽다 → 일다 (spelling ≠ pronunciation)
  English spelling: knight → /naɪt/ (k, g, h are all silent!)

  IPA removes ambiguity. Learn to read IPA = always know
  the correct pronunciation from a dictionary.
```

## 3. Vowel System: Monophthongs

### 3.1 The 12 Pure Vowels (Monophthongs)

English has 12 monophthong vowels — more than double Korean's basic vowels. They differ in tongue position AND length.

```
┌─────────────────────────────────────────────────────────────────┐
│                  English Vowel Chart (RP/GA)                    │
│                                                                 │
│            Front        Central        Back                     │
│   High     iː (seat)                   uː (food)               │
│            ɪ  (sit)                     ʊ  (book)               │
│   Mid      e  (bed)     ə  (about)     ɔː (law)                │
│                          ʌ  (cup)                               │
│   Low      æ  (cat)     ɑː (father)   ɒ  (lot, BrE)           │
│                                                                 │
│   Short vowels: ɪ, e, æ, ʌ, ɒ, ʊ, ə                           │
│   Long vowels:  iː, ɑː, ɔː, uː, ɜː (nurse)                  │
│                                                                 │
│   ※ Length matters! "ship" /ʃɪp/ ≠ "sheep" /ʃiːp/              │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Short Vowels

| IPA | Example | Korean Approx. | Key Difference from Korean |
|-----|---------|----------------|---------------------------|
| /ɪ/ | s**i**t, b**i**g, f**i**sh | 이 (shorter) | More relaxed than Korean 이, mouth less spread |
| /e/ | b**e**d, r**e**d, s**e**t | 에 | Very similar to Korean 에 |
| /æ/ | c**a**t, b**a**d, h**a**nd | 애 (wider) | Mouth opens wider than Korean 애, jaw drops more |
| /ʌ/ | c**u**p, b**u**s, l**u**ck | 어 (shorter) | Similar to short Korean 어 but more central |
| /ɒ/ | h**o**t, d**o**g, l**o**t (BrE) | 아/오 사이 | Does not exist in Korean; between 아 and 오 |
| /ʊ/ | b**oo**k, p**u**t, g**oo**d | 우 (shorter) | More relaxed than Korean 우, lips less rounded |
| /ə/ | **a**bout, sof**a**, banan**a** | 으/어 사이 | Schwa — the most common English sound! |

### 3.3 Long Vowels

| IPA | Example | Korean Approx. | Key Note |
|-----|---------|----------------|----------|
| /iː/ | s**ee**, t**ea**, m**ea**t | 이 (길게) | Longer than Korean 이, tongue slightly higher |
| /ɑː/ | f**a**ther, c**a**r, st**a**rt | 아 (길게) | Like Korean 아 but held longer |
| /ɔː/ | l**aw**, c**au**ght, w**a**ll | 오 (길게) | Like Korean 오 but more open, jaw drops |
| /uː/ | f**oo**d, bl**ue**, sh**oe** | 우 (길게) | Like Korean 우 but held longer |
| /ɜː/ | n**ur**se, b**ir**d, w**or**d | 어 (길게) | Like holding Korean 어, tongue central |

### 3.4 The Schwa /ə/ — Most Important Sound in English

```
┌─────────────────────────────────────────────────────────────────┐
│   The Schwa /ə/ — The Most Frequent English Sound               │
│                                                                 │
│   What: A short, weak, neutral "uh" sound                       │
│   When: In UNSTRESSED syllables                                 │
│   Why important: Appears in almost every English word            │
│                                                                 │
│   Examples (schwa underlined):                                  │
│   a-BOUT   /ə.ˈbaʊt/     (첫 음절이 약하게 "어")                 │
│   ba-NA-na  /bə.ˈnæ.nə/  (1번째, 3번째 음절 모두 schwa)          │
│   TO-day    /tə.ˈdeɪ/     (첫 음절이 약하게 "터")                 │
│   po-LICE   /pə.ˈliːs/   (첫 음절이 약하게 "퍼")                 │
│                                                                 │
│   ※ Korean speakers tend to pronounce every syllable clearly.  │
│     In English, unstressed syllables REDUCE to schwa.           │
│     This is essential for natural-sounding English.             │
└─────────────────────────────────────────────────────────────────┘
```

## 4. Vowel System: Diphthongs

### 4.1 The 8 Diphthongs

A diphthong is a vowel that glides from one sound to another within one syllable.

| IPA | Example | Korean Approx. | Movement |
|-----|---------|----------------|----------|
| /eɪ/ | d**ay**, m**a**ke, r**ai**n | 에이 | e → ɪ |
| /aɪ/ | t**i**me, b**uy**, cr**y** | 아이 | a → ɪ |
| /ɔɪ/ | b**oy**, n**oi**se, t**oy** | 오이 | ɔ → ɪ |
| /aʊ/ | h**ou**se, n**ow**, c**ow** | 아우 | a → ʊ |
| /oʊ/ | g**o**, h**o**me, kn**ow** | 오우 | o → ʊ |
| /ɪə/ | h**ere**, n**ear**, b**eer** (BrE) | 이어 | ɪ → ə |
| /eə/ | th**ere**, c**are**, h**air** (BrE) | 에어 | e → ə |
| /ʊə/ | p**ure**, t**our**, c**ure** (BrE) | 우어 | ʊ → ə |

> **Note**: In General American (AmE), /ɪə/, /eə/, /ʊə/ are often replaced by vowel + /r/: /ɪr/, /er/, /ʊr/.

### 4.2 Common Confusion Pairs

```
Pairs Korean speakers confuse:

  /æ/ vs /e/           bat /bæt/ vs bet /bet/
  /ʌ/ vs /ɑː/         cut /kʌt/ vs cart /kɑːrt/
  /ɪ/ vs /iː/         ship /ʃɪp/ vs sheep /ʃiːp/
  /ʊ/ vs /uː/         full /fʊl/ vs fool /fuːl/
  /ɔː/ vs /oʊ/        caught /kɔːt/ vs coat /koʊt/

  ※ Vowel LENGTH changes meaning in English!
  Korean vowel length distinction (말 vs 말:) is disappearing,
  so Korean speakers often ignore English vowel length.
```

## 5. Consonant System

### 5.1 English Consonants Overview

English has 24 consonant phonemes. Most map well to Korean, but several do not.

```
┌─────────────────────────────────────────────────────────────────┐
│              English Consonant Phonemes (24)                     │
│                                                                 │
│   Stops:       /p/ /b/ /t/ /d/ /k/ /g/                         │
│   Fricatives:  /f/ /v/ /θ/ /ð/ /s/ /z/ /ʃ/ /ʒ/ /h/            │
│   Affricates:  /tʃ/ /dʒ/                                       │
│   Nasals:      /m/ /n/ /ŋ/                                      │
│   Liquids:     /l/ /r/                                          │
│   Glides:      /w/ /j/                                          │
│                                                                 │
│   ★ Sounds NOT in Korean: /f/ /v/ /θ/ /ð/ /z/ /ʒ/ /r/         │
│     These need special practice!                                │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 Stops (Plosives)

| IPA | Korean Similar | Example | Notes |
|-----|---------------|---------|-------|
| /p/ | ㅍ (aspirated) / ㅂ (plain) | **p**en, sto**p** | Word-initial: aspirated like ㅍ |
| /b/ | ㅂ (voiced) | **b**ig, ca**b** | Voiced — vibrate vocal cords |
| /t/ | ㅌ (aspirated) / ㄷ (plain) | **t**en, ca**t** | Word-initial: aspirated like ㅌ |
| /d/ | ㄷ (voiced) | **d**og, be**d** | Voiced — vibrate vocal cords |
| /k/ | ㅋ (aspirated) / ㄱ (plain) | **c**at, ba**ck** | Word-initial: aspirated like ㅋ |
| /g/ | ㄱ (voiced) | **g**et, bi**g** | Voiced — vibrate vocal cords |

> **Korean speaker tip**: English distinguishes voiced vs voiceless (/b/ vs /p/, /d/ vs /t/, /g/ vs /k/). Korean distinguishes plain vs aspirated vs tense (ㅂ/ㅍ/ㅃ). These are DIFFERENT systems! Practice voicing (vocal cord vibration) for /b/, /d/, /g/.

### 5.3 Fricatives

| IPA | Korean Similar | Example | How to Make It |
|-----|---------------|---------|----------------|
| /f/ | ㅍ (NO!) | **f**ish, li**f**e | Upper teeth touch lower lip, blow air |
| /v/ | ㅂ (NO!) | **v**ery, lo**v**e | Same as /f/ but with voice |
| /θ/ | ㅅ (NO!) | **th**ink, ba**th** | Tongue tip between teeth, blow air |
| /ð/ | ㄷ (NO!) | **th**is, brea**th**e | Same as /θ/ but with voice |
| /s/ | ㅅ | **s**un, bu**s** | Very similar to Korean ㅅ |
| /z/ | ㅈ (NO!) | **z**oo, bu**zz** | Same as /s/ but with voice |
| /ʃ/ | 쉬 | **sh**ip, fi**sh** | Similar to Korean 쉬 |
| /ʒ/ | 쥬 (NO!) | vi**si**on, plea**s**ure | Same as /ʃ/ but with voice |
| /h/ | ㅎ | **h**ot, **h**ello | Very similar to Korean ㅎ |

## 6. Sounds That Don't Exist in Korean

### 6.1 R vs L — The #1 Challenge (R/L 구분)

```
┌─────────────────────────────────────────────────────────────────┐
│              R /r/ vs L /l/ — Critical Distinction               │
│                                                                 │
│   Korean ㄹ = ONE sound (a tap/flap, like Spanish "r")          │
│   English = TWO completely different sounds                     │
│                                                                 │
│   /l/ (lateral):                                                │
│   • Tongue TIP touches the ridge behind upper teeth             │
│   • Air flows over the SIDES of the tongue                      │
│   • "light" /laɪt/    "bell" /bel/    "play" /pleɪ/            │
│                                                                 │
│   /r/ (approximant):                                            │
│   • Tongue tip curls BACK, does NOT touch anything              │
│   • Tongue sides touch upper molars                             │
│   • "right" /raɪt/    "car" /kɑːr/    "pray" /preɪ/            │
│                                                                 │
│   Minimal pairs:                                                │
│   right /raɪt/  vs  light /laɪt/                                │
│   road /roʊd/   vs  load /loʊd/                                 │
│   read /riːd/   vs  lead /liːd/                                 │
│   row /roʊ/     vs  low /loʊ/                                   │
│   rock /rɒk/    vs  lock /lɒk/                                  │
│   pray /preɪ/   vs  play /pleɪ/                                 │
│   fry /fraɪ/    vs  fly /flaɪ/                                  │
│   grow /groʊ/   vs  glow /gloʊ/                                 │
│                                                                 │
│   ※ Practice: Say "la la la" (tongue touches ridge)            │
│     Then "ra ra ra" (tongue curls back, touches NOTHING)        │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 /θ/ and /ð/ — The "th" Sounds

```
┌─────────────────────────────────────────────────────────────────┐
│   Voiceless th /θ/:  think, three, bath, math, tooth            │
│   • Tongue tip between teeth, blow air (no voice)               │
│   • Korean speakers say ㅅ → "sink" instead of "think"          │
│                                                                 │
│   Voiced th /ð/:     this, that, the, mother, breathe           │
│   • Tongue tip between teeth, blow air WITH voice               │
│   • Korean speakers say ㄷ → "dis" instead of "this"            │
│                                                                 │
│   Minimal pairs:                                                │
│   think /θɪŋk/ vs sink /sɪŋk/ (th vs s)                        │
│   three /θriː/ vs tree /triː/ (th vs t)                        │
│   path /pæθ/   vs pass /pæs/  (th vs s)                         │
│   then /ðen/   vs den /den/   (th vs d)                         │
│   they /ðeɪ/   vs day /deɪ/   (th vs d)                         │
│                                                                 │
│   ※ Practice: Stick tongue out slightly between teeth.         │
│     Feel air blow over tongue. This sound does NOT exist        │
│     in Korean, so it requires dedicated practice.               │
└─────────────────────────────────────────────────────────────────┘
```

### 6.3 /f/ and /v/ — Lip-Teeth Sounds

```
/f/ (voiceless labiodental fricative):
  • Upper teeth touch LOWER LIP, blow air
  • fish, five, life, off, phone
  • Korean speakers substitute ㅍ (/p/) → "pish" for "fish"
  • ※ ㅍ uses BOTH lips. /f/ uses teeth + lip. Different!

/v/ (voiced labiodental fricative):
  • Same position as /f/ but with vocal cord vibration
  • very, live, van, love, video
  • Korean speakers substitute ㅂ (/b/) → "bery" for "very"
  • ※ ㅂ uses BOTH lips. /v/ uses teeth + lip. Different!

Minimal pairs:
  fan /fæn/   vs  pan /pæn/   (f vs p)
  fine /faɪn/ vs  pine /paɪn/ (f vs p)
  vine /vaɪn/ vs  wine /waɪn/ (v vs w)
  vest /vest/ vs  best /best/ (v vs b)
  very /ˈveri/ vs berry /ˈberi/ (v vs b)
```

### 6.4 /z/ — Voiced S

```
/z/ (voiced alveolar fricative):
  • Same mouth position as /s/ but with vocal cord vibration
  • zoo, buzz, zero, please, dogs
  • Korean has no /z/ sound
  • Korean speakers substitute ㅈ (/dʒ/) → "joo" for "zoo"

Minimal pairs:
  sip /sɪp/   vs  zip /zɪp/
  bus /bʌs/   vs  buzz /bʌz/
  price /praɪs/ vs prize /praɪz/
  ice /aɪs/   vs  eyes /aɪz/

※ Many English words use /z/ where you see "s":
  dogs /dɒgz/    is /ɪz/      has /hæz/
  boys /bɔɪz/    was /wɒz/    his /hɪz/
```

## 7. Word Stress

### 7.1 What is Word Stress?

```
┌─────────────────────────────────────────────────────────────────┐
│              English Word Stress — Key Concept                   │
│                                                                 │
│   Every English word of 2+ syllables has ONE stressed syllable  │
│   The stressed syllable is: LOUDER, LONGER, HIGHER in pitch     │
│                                                                 │
│   ba-NA-na    /bə.ˈnæ.nə/     (2nd syllable stressed)          │
│   COM-pu-ter  /kəm.ˈpjuː.tər/ (2nd syllable stressed)         │
│   BEAU-ti-ful /ˈbjuː.tɪ.fəl/  (1st syllable stressed)         │
│                                                                 │
│   ※ Korean is a syllable-timed language (each syllable has     │
│     roughly equal weight). English is stress-timed (stressed    │
│     syllables are louder/longer, unstressed syllables are       │
│     compressed). This is WHY English sounds "uneven" to         │
│     Korean ears.                                                │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 Stress Patterns by Word Type

```
Two-syllable nouns — stress usually on FIRST syllable:
  TA-ble    WA-ter    DOC-tor    MU-sic    PIC-ture

Two-syllable verbs — stress usually on SECOND syllable:
  be-GIN    for-GET   ar-RIVE    de-CIDE   re-PEAT

Same word, different stress = different meaning:
  RE-cord (noun: 기록)    vs  re-CORD (verb: 기록하다)
  PRE-sent (noun: 선물)   vs  pre-SENT (verb: 제시하다)
  OB-ject (noun: 물체)    vs  ob-JECT (verb: 반대하다)
  CON-tract (noun: 계약)  vs  con-TRACT (verb: 수축하다)
  PRO-duce (noun: 농산물)  vs  pro-DUCE (verb: 생산하다)
```

### 7.3 Stress in Longer Words

```
Suffixes that ATTRACT stress (stress on suffix or syllable before):
  -tion / -sion    → edu-CA-tion, de-CI-sion
  -ic              → fan-TAS-tic, e-co-NOM-ic
  -ity             → u-ni-VER-si-ty, e-lec-TRI-ci-ty
  -ial / -ious     → of-FI-cial, am-BI-tious
  -graphy          → pho-TOG-ra-phy, ge-OG-ra-phy

Suffixes that DON'T change stress:
  -ment            → de-VE-lop → de-VE-lop-ment
  -ness            → HAP-py → HAP-pi-ness
  -ful             → BEAU-ty → BEAU-ti-ful
  -less            → CARE → CARE-less
  -ly              → QUICK → QUICK-ly
```

## 8. Sentence Stress and Rhythm

### 8.1 Content Words vs Function Words

```
┌─────────────────────────────────────────────────────────────────┐
│   Content words (STRESSED) — carry meaning:                     │
│   Nouns:      DOG, TABLE, KOREA                                 │
│   Main verbs: EAT, STUDY, WANT                                 │
│   Adjectives: BIG, HAPPY, FAST                                  │
│   Adverbs:    QUICKLY, ALWAYS, VERY                             │
│   Negatives:  NOT, NEVER, NO                                    │
│   Wh-words:   WHAT, WHERE, WHO                                  │
│                                                                 │
│   Function words (unstressed) — grammar glue:                   │
│   Articles:    a, an, the                                       │
│   Pronouns:    I, you, he, she, it, we, they                    │
│   Prepositions: in, on, at, to, for, of                         │
│   Auxiliaries:  am, is, are, was, do, have, can, will            │
│   Conjunctions: and, but, or, that                              │
│                                                                 │
│   Example:                                                      │
│   "I want to GO to the STORE and BUY some BREAD."               │
│    ·  ····  ·  ··  ·  ···  ····  ···  ···  ····  ····           │
│    weak     weak  STRONG weak STRONG weak  STRONG                │
└─────────────────────────────────────────────────────────────────┘
```

### 8.2 Stress-Timed Rhythm

```
English is STRESS-TIMED:
  The time between stressed syllables is roughly equal,
  regardless of how many unstressed syllables are between them.

  DOGS    CHASE    CATS           (3 stresses, ~1 sec)
  The DOGS   are CHAS-ing  the CATS   (3 stresses, ~1 sec)
  The BIG DOGS  have been CHAS-ing  the SMALL CATS  (still ~1 sec between stresses)

  ※ Korean is SYLLABLE-TIMED: each syllable takes roughly equal time.
    This is why Korean speakers sound "robotic" in English —
    they give equal weight to every syllable.
    Solution: STRESS content words, COMPRESS function words.
```

## 9. Connected Speech

### 9.1 Linking

When speaking naturally, English words are NOT pronounced separately. They connect.

```
Consonant-to-Vowel linking (자음 → 모음 연결):
  "turn off"     → "tur-NOFF"     /tɜːr.ˈnɒf/
  "pick it up"   → "pi-KI-TUP"   /pɪ.kɪ.ˈtʌp/
  "an apple"     → "a-NA-pple"    /ə.ˈnæ.pəl/

Vowel-to-Vowel linking (모음 → 모음 연결):
  "go away"      → "go-WA-way"   (insert /w/)
  "she asked"    → "she-YASKED"  (insert /j/)
  "to eat"       → "to-WEAT"     (insert /w/)

Consonant-to-Consonant linking (자음 → 자음 연결):
  "good day"     → "goo-DAY"     (first /d/ unreleased)
  "hot tea"      → "ho-TEA"      (first /t/ merges)
  "big game"     → "bi-GAME"     (first /g/ unreleased)
```

### 9.2 Reduction

Unstressed function words change dramatically in fast speech.

```
Word        Full Form    Reduced Form    Example
and         /ænd/        /ən/ or /n/     bread 'n butter
to          /tuː/        /tə/            want to → wanna /ˈwɒnə/
for         /fɔːr/       /fər/           "what's it for?"
of          /ɒv/         /əv/ or /ə/     cup of tea → cuppa tea
you         /juː/        /jə/            "did you" → "didja"
can         /kæn/        /kən/           "I can go" → /aɪ kən goʊ/
have        /hæv/        /əv/ or /ə/     "should have" → "shoulda"
```

### 9.3 Elision (Sound Deletion)

```
Common dropped sounds:
  "probably"    → "probly"      /ˈprɒbli/     (middle syllable lost)
  "comfortable" → "comfterble"  /ˈkʌmftəbəl/  (syllable lost)
  "Wednesday"   → "Wensday"    /ˈwenzdeɪ/     (/d/ dropped)
  "next day"    → "nex day"    /ˈneksdeɪ/     (/t/ dropped)
  "must be"     → "mus be"     /ˈmʌsbi/       (/t/ dropped)
```

## 10. Intonation Patterns

### 10.1 Falling Intonation (↘)

```
Used for: statements, wh-questions, commands

  Statements:
    "I live in Seoul."
    ──────────╲
              ↓ (falls at end)

  Wh-questions:
    "Where do you live?"
    ╱──────────╲
    ↑ (high on "where")  ↓ (falls at end)

  Commands:
    "Close the door."
    ──────╲
          ↓
```

### 10.2 Rising Intonation (↗)

```
Used for: yes/no questions, showing surprise, listing (non-final items)

  Yes/No questions:
    "Do you speak English?"
    ──────────────╱
                  ↑ (rises at end)

  Surprise:
    "Really?"
    ────╱
        ↑

  Lists (non-final):
    "I bought apples↗, oranges↗, and bananas↘."
```

### 10.3 Fall-Rise Intonation (↘↗)

```
Used for: uncertainty, politeness, contrast, "but..."

  "I like the color... (but not the size)"
  ────╲──╱
      ↓  ↑

  "It's okay..."  (implying reservation)
  ───╲──╱
```

> **For Korean speakers**: Korean intonation is relatively flat compared to English. Korean marks questions with particles (까, 니) while English uses intonation + word order. Practice exaggerating your pitch range — what feels "too dramatic" in Korean is often normal in English.

## 11. Common Pronunciation Mistakes by Korean Speakers

### 11.1 Systematic Error Patterns

| Error | What Korean Speakers Say | Correct | Fix |
|-------|-------------------------|---------|-----|
| /f/ → /p/ | "pish" | "fish" /fɪʃ/ | Teeth on lower lip |
| /v/ → /b/ | "bery" | "very" /ˈveri/ | Teeth on lower lip + voice |
| /θ/ → /s/ | "sink" | "think" /θɪŋk/ | Tongue between teeth |
| /ð/ → /d/ | "dis" | "this" /ðɪs/ | Tongue between teeth + voice |
| /z/ → /dʒ/ | "joo" | "zoo" /zuː/ | Like /s/ but with voice |
| /r/ → /l/ | "light" | "right" /raɪt/ | Curl tongue back, don't touch |
| /l/ → /r/ | "rice" | "lice" /laɪs/ | Tongue tip touches ridge |
| Adding vowels | "seh-tuh-rike" | "strike" /straɪk/ | No vowels between consonants |

### 11.2 Final Consonant Release

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean consonant rules at syllable end (받침):                │
│   • Only 7 sounds allowed: ㄱ,ㄴ,ㄷ,ㄹ,ㅁ,ㅂ,ㅇ                  │
│   • Consonants are unreleased (no burst of air)                 │
│                                                                 │
│   English consonant rules at word end:                          │
│   • ANY consonant can end a word                                │
│   • Consonants CAN be released (with air burst)                 │
│   • Consonant CLUSTERS are common: texts /teksts/               │
│                                                                 │
│   Common Korean speaker errors:                                 │
│   "dog" → "dok" (devoicing final /g/ to /k/)                   │
│   "bed" → "bet" (devoicing final /d/ to /t/)                   │
│   "five" → "paibeu" (adding vowel, changing /f/ to /p/)        │
│   "bus" → "beoseu" (adding vowels)                              │
│                                                                 │
│   ※ Practice releasing final consonants:                       │
│     "cat" — let the /t/ pop out with air                        │
│     "dog" — keep the /g/ voiced at the end                      │
│     "love" — end with /v/ (teeth on lip), don't add 으          │
└─────────────────────────────────────────────────────────────────┘
```

### 11.3 Consonant Clusters

```
Korean allows max 1 consonant at syllable start, 1 at end.
English allows up to 3 at start, 4+ at end.

English clusters Korean speakers struggle with:
  Beginning:  str- (strike), spr- (spring), skr- (scream)
  Middle:     -nstr- (construct), -mpl- (simple)
  End:        -lk (milk), -nks (thanks), -ksts (texts)

  ※ Do NOT insert vowels between consonants!
  "strike" ≠ "suh-tuh-rai-kuh"  →  /straɪk/ (one syllable!)
  "spring" ≠ "suh-puh-ring"     →  /sprɪŋ/  (one syllable!)
  "milk"   ≠ "mil-kuh"          →  /mɪlk/   (one syllable!)
```
