- Understand the order of multiple adjectives before a noun

## 1. Adjectives in English vs Korean

### 1.1 Key Differences

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean adjectives BEHAVE LIKE VERBS:                          │
│   크다 (to be big), 작다 (to be small), 빠르다 (to be fast)     │
│   They conjugate: 크다, 큰, 컸다, 클 것이다                      │
│                                                                 │
│   English adjectives are SEPARATE WORDS (no conjugation):       │
│   big, small, fast — they NEVER change form for tense           │
│   "She is big." "She was big." "She will be big."              │
│   → "big" stays "big" in all tenses!                           │
│                                                                 │
│   Korean: 큰 집 (adjective modifies noun with ㄴ/은)             │
│   English: a big house (adjective goes directly before noun)    │
│                                                                 │
│   Korean: 그 집은 크다 (adjective acts as the main verb)         │
│   English: The house IS big. (adjective needs BE verb)          │
│   ※ "The house big." ✗ — MUST have a linking verb!            │
└─────────────────────────────────────────────────────────────────┘
```

## 2. Adjective Position

### 2.1 Before the Noun (Attributive)

```
Adjective + Noun:
  a tall man                  키 큰 남자
  a beautiful day             아름다운 날
  an expensive car            비싼 차
  a cold drink                차가운 음료
  an interesting book         재미있는 책

  ※ Korean: 큰 + 집 (adjective stem + ㄴ + noun)
    English: big + house (adjective + noun, nothing in between)
```

### 2.2 After a Linking Verb (Predicative)

```
Subject + Linking Verb + Adjective:
  She is tall.                그녀는 키가 크다.
  The food smells delicious.  음식 냄새가 맛있다.
  He looks tired.             그는 피곤해 보인다.
  The music sounds beautiful. 음악이 아름답게 들린다.
  This feels soft.            이것은 부드럽게 느껴진다.

Linking verbs: be, seem, appear, look, sound, smell, taste, feel, become, get

  ※ After linking verbs, use adjective (NOT adverb):
    "She looks beautiful." ✓  (adjective after look)
    "She looks beautifully." ✗ (this would mean "she uses her eyes beautifully")
```

### 2.3 Adjectives That Can Only Be Used in One Position

```
Only BEFORE noun (attributive only):
  main, chief, principal, former, latter, elder, mere, utter, only, sole
  "the main reason" ✓     "The reason is main." ✗
  "the former president" ✓ "The president is former." ✗

Only AFTER verb (predicative only):
  afraid, alive, alone, asleep, awake, aware, glad, ill (BrE), well
  "She is afraid." ✓      "an afraid girl" ✗ → "a frightened girl" ✓
  "He is asleep." ✓       "an asleep baby" ✗ → "a sleeping baby" ✓
  "She is alone." ✓       "an alone person" ✗ → "a lonely person" ✓
```

## 3. Order of Multiple Adjectives

### 3.1 The OSASCOMP Rule

```
┌─────────────────────────────────────────────────────────────────┐
│   When multiple adjectives come before a noun, they follow      │
│   a specific order (native speakers do this intuitively):       │
│                                                                 │
│   O - Opinion    (beautiful, nice, ugly, delicious)             │
│   S - Size       (big, small, tall, short, long)                │
│   A - Age        (old, young, new, ancient)                     │
│   S - Shape      (round, square, flat, wide)                    │
│   C - Color      (red, blue, green, dark)                       │
│   O - Origin     (Korean, American, Japanese, French)           │
│   M - Material   (wooden, metal, cotton, plastic)               │
│   P - Purpose    (sleeping [bag], running [shoes])              │
│                                                                 │
│   Examples:                                                     │
│   "a beautiful big old round red Korean wooden dining table"    │
│    Opinion  Size Age Shape Color Origin Material Purpose Noun   │
│                                                                 │
│   In practice, you rarely use more than 2-3 adjectives:        │
│   "a beautiful old house"      (Opinion + Age)                  │
│   "a small red car"            (Size + Color)                   │
│   "a tall young Korean man"    (Size + Age + Origin)            │
│                                                                 │
│   ※ Korean: 아름다운 큰 오래된 빨간 한국의 나무 식탁              │
│     Korean order is more flexible, but English order is fixed!  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Quick Examples

```
"a nice little old town"          (Opinion + Size + Age)         ✓
"a little nice old town"          (Size + Opinion + Age)         ✗

"a big black leather bag"         (Size + Color + Material)      ✓
"a black big leather bag"         (Color + Size + Material)      ✗

"an expensive Japanese car"       (Opinion + Origin)             ✓
"a Japanese expensive car"        (Origin + Opinion)             ✗
```

## 4. Comparative Forms (-er / more)

### 4.1 Rules for Forming Comparatives

```
┌─────────────────────────────────────────────────────────────────┐
│   One-syllable adjectives → add -er:                            │
│   tall → taller     old → older     fast → faster               │
│   cheap → cheaper   long → longer   young → younger             │
│                                                                 │
│   One-syllable ending CVC → double + -er:                       │
│   big → bigger      hot → hotter    thin → thinner              │
│   fat → fatter      wet → wetter    sad → sadder                │
│                                                                 │
│   Two-syllable ending -y → change to -ier:                      │
│   happy → happier   easy → easier   busy → busier               │
│   early → earlier   heavy → heavier  funny → funnier            │
│                                                                 │
│   Two+ syllable adjectives → more + adjective:                  │
│   beautiful → more beautiful                                    │
│   expensive → more expensive                                    │
│   important → more important                                    │
│   interesting → more interesting                                │
│   comfortable → more comfortable                                │
│                                                                 │
│   ※ NEVER double: "more bigger" ✗ "more taller" ✗             │
│     Use EITHER -er OR more, NEVER both!                        │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Using Comparatives

```
Pattern: adjective-er / more + adjective + than

  Seoul is bigger than Busan.
  서울은 부산보다 더 크다.

  English is more difficult than Spanish.
  영어는 스페인어보다 더 어렵다.

  She is taller than me/I.
  그녀는 나보다 더 키가 크다.

  This book is more interesting than that one.
  이 책은 저 책보다 더 재미있다.

  ※ Korean: ~보다 더 (보다 comes AFTER the compared noun)
    English: more/~er ... THAN (than comes AFTER the adjective)
    Korean: 서울은 부산보다 크다     (부산보다 → 크다)
    English: Seoul is bigger than Busan  (bigger → than Busan)
```

### 4.3 Comparative Modifiers

```
much / far / a lot + comparative (훨씬):
  "Seoul is MUCH bigger than Busan."
  "This is FAR more expensive."
  "She is A LOT taller than him."

a little / a bit / slightly + comparative (약간):
  "She is A LITTLE taller than me."
  "This is SLIGHTLY more expensive."

even + comparative (심지어, 더욱):
  "Today is EVEN colder than yesterday."

  ※ "very bigger" ✗ — Cannot use "very" with comparatives!
    Use "much" instead: "much bigger" ✓
```

## 5. Superlative Forms (-est / most)

### 5.1 Rules for Forming Superlatives

```
One-syllable → the + adjective-est:
  tall → the tallest     old → the oldest     fast → the fastest
  big → the biggest      hot → the hottest

Two-syllable ending -y → the + adjective-iest:
  happy → the happiest   easy → the easiest   busy → the busiest

Two+ syllables → the most + adjective:
  beautiful → the most beautiful
  important → the most important
  interesting → the most interesting

  ※ Superlatives ALWAYS use "the":
    "She is the tallest."    ✓
    "She is tallest."        ✗ (needs "the")
```

### 5.2 Using Superlatives

```
Pattern: the + adjective-est / the most + adjective + (in/of)

  Seoul is the biggest city in Korea.
  서울은 한국에서 가장 큰 도시이다.

  She is the smartest student in the class.
  그녀는 반에서 가장 똑똑한 학생이다.

  This is the most expensive restaurant in the city.
  이것은 도시에서 가장 비싼 식당이다.

  He is the tallest of the three brothers.
  그는 세 형제 중 가장 키가 크다.

  ※ "in" for places/groups: in the class, in the world, in Korea
    "of" for a specific set: of the three, of all students
```

## 6. Irregular Comparatives and Superlatives

### 6.1 The Irregulars

```
┌─────────────────────────────────────────────────────────────────┐
│   Adjective    Comparative    Superlative                       │
│   ─────────    ───────────    ───────────                       │
│   good         better         the best                          │
│   bad          worse          the worst                         │
│   far          farther/further the farthest/furthest            │
│   little       less           the least                         │
│   much/many    more           the most                          │
│   old          older/elder    the oldest/eldest                  │
│                                                                 │
│   ※ "gooder" ✗  "badder" ✗  "more good" ✗  "more bad" ✗      │
│     These are ALWAYS irregular — memorize them!                │
│                                                                 │
│   farther = physical distance: "Seoul is farther than Suwon."  │
│   further = additional/more:   "We need further information."  │
│   elder = family members:      "my elder brother" (형/언니)      │
│   older = general:             "She is older than me."         │
└─────────────────────────────────────────────────────────────────┘
```

## 7. as...as Comparisons

### 7.1 Equal Comparison: as + adjective + as

```
Positive (같은 정도):
  She is as tall as her mother.
  그녀는 어머니만큼 키가 크다.

  This book is as interesting as that one.
  이 책은 저 책만큼 재미있다.

  He runs as fast as his brother.
  그는 형만큼 빨리 달린다.

Negative (다른 정도):
  She is not as tall as her mother.
  그녀는 어머니만큼 키가 크지 않다.

  English is not as easy as Spanish.
  영어는 스페인어만큼 쉽지 않다.

  ※ Korean: ~만큼 (만큼 comes AFTER the compared noun)
    English: as...as (compared noun goes AFTER second "as")
```

### 7.2 Common as...as Expressions

```
as soon as possible (ASAP)     가능한 한 빨리
as much as possible            가능한 한 많이
as far as I know               내가 아는 한
as long as                     ~하는 한
as well as                     ~뿐만 아니라
as good as new                 새것이나 다름없는
as easy as pie                 매우 쉬운
as cold as ice                 얼음처럼 차가운
```

## 8. Adverbs

### 8.1 Adverb Formation: -ly

```
Most adjectives → add -ly to form adverb:
  slow → slowly         quick → quickly       careful → carefully
  quiet → quietly       loud → loudly         beautiful → beautifully
  happy → happily       easy → easily         lucky → luckily
  comfortable → comfortably                   terrible → terribly

Spelling changes:
  -le ending → change -le to -ly:   simple → simply, gentle → gently
  -y ending → change -y to -ily:    happy → happily, easy → easily
  -ic ending → add -ally:           basic → basically, automatic → automatically
  -ll ending → add -y:              full → fully
  -ue ending → drop -e, add -ly:    true → truly

  ※ NOT all -ly words are adverbs!
    friendly, lovely, lonely, silly, ugly, lively = ADJECTIVES
    "She is friendly." ✓ (adjective)
    "She acted in a friendly way." ✓ (not "friendlily")
```

### 8.2 Irregular Adverbs

```
┌─────────────────────────────────────────────────────────────────┐
│   Adjective → Adverb (same form or irregular):                  │
│                                                                 │
│   good → well         "She speaks English well."                │
│   fast → fast         "He runs fast."  (NOT "fastly")          │
│   hard → hard         "She works hard." (NOT "hardly")         │
│   late → late         "He arrived late."                        │
│   early → early       "She woke up early."                     │
│   straight → straight "Go straight."                           │
│                                                                 │
│   ※ hard vs hardly:                                            │
│     "She works hard."    = 열심히 일한다 (diligently)            │
│     "She hardly works."  = 거의 일 안 한다 (barely)              │
│     These mean OPPOSITE things!                                │
│                                                                 │
│   ※ late vs lately:                                            │
│     "He arrived late."   = 늦게 도착했다 (not on time)           │
│     "I've been busy lately." = 최근에 바빴다 (recently)          │
│     Different meanings!                                         │
└─────────────────────────────────────────────────────────────────┘
```

### 8.3 Adverb Position

```
Manner adverbs (how) → after verb/object:
  "She sings beautifully."            (after verb)
  "He speaks English fluently."       (after object)
  ※ NOT "She beautifully sings." (OK in literary style, unusual)

Frequency adverbs → before main verb, after BE:
  "I always eat breakfast."           (before main verb)
  "She is always happy."              (after BE)

Degree adverbs → before adjective/adverb:
  "She is very tall."                 (before adjective)
  "He runs very fast."                (before adverb)
  "It was really interesting."        (before adjective)
  "She quite likes it."               (before verb — British)
```

### 8.4 Comparative and Superlative Adverbs

```
Short adverbs: -er / -est
  fast → faster → fastest
  hard → harder → hardest
  early → earlier → earliest

Long adverbs (-ly): more / most
  quickly → more quickly → most quickly
  carefully → more carefully → most carefully
  beautifully → more beautifully → most beautifully

Irregular:
  well → better → best
  badly → worse → worst
  far → farther/further → farthest/furthest
  much → more → most
  little → less → least
```

## 9. enough and too

### 9.1 enough

```
enough + noun (충분한 + 명사):
  "I have enough money."          나는 돈이 충분하다.
  "There isn't enough time."      시간이 충분하지 않다.

adjective/adverb + enough (형용사/부사 + 충분히):
  "She is old enough to drive."   그녀는 운전할 만큼 나이가 든다.
  "He runs fast enough."          그는 충분히 빨리 달린다.

  ※ enough goes AFTER adjective/adverb but BEFORE noun:
    "old enough" ✓   "enough old" ✗
    "enough money" ✓  "money enough" ✗

enough + to + verb:
  "This bag is big enough to carry all my books."
  이 가방은 내 책을 다 넣을 만큼 충분히 크다.
```

### 9.2 too

```
too + adjective/adverb (너무 ~한):
  "This coffee is too hot."       이 커피는 너무 뜨겁다.
  "He speaks too quickly."        그는 너무 빨리 말한다.

too + adjective + to + verb:
  "She is too young to drive."    그녀는 운전하기에 너무 어리다.
  "It's too cold to swim."        수영하기에 너무 춥다.

too much + uncountable noun:
  "I ate too much food."          나는 음식을 너무 많이 먹었다.

too many + countable noun:
  "There are too many people."    사람이 너무 많다.

  ※ too vs very:
    "very" = neutral intensifier (매우): "She is very tall." (just a fact)
    "too" = excessively, problematic (너무): "She is too tall." (it's a problem)
    Korean 너무 is used positively: "너무 예쁘다!" (very pretty)
    English "too" is ALWAYS negative: "too pretty" = problematically pretty
```

## Quick Reference

```
┌─────────────────────────────────────────────────────────────────┐
│          Adjectives & Adverbs Quick Guide                       │
│                                                                 │
│   Adjective position: before noun / after linking verb          │
│   Multiple adj order: Opinion-Size-Age-Shape-Color-Origin-      │
│                       Material-Purpose (OSASCOMP)              │
│                                                                 │
│   Comparative: short adj → -er + than / long adj → more + than │
│   Superlative: short adj → the -est / long adj → the most      │
│   Irregular: good-better-best, bad-worse-worst                 │
│   ※ NEVER "more better" or "most best"!                       │
│                                                                 │
│   Adverbs: adjective + -ly (slowly, carefully, beautifully)    │
│   Irregular: good→well, fast→fast, hard→hard                   │
│   ※ hard ≠ hardly, late ≠ lately                              │
│                                                                 │
│   as...as = same degree    not as...as = less degree            │
│   enough: adj + enough / enough + noun                         │
│   too: too + adj (excessive, negative meaning)                  │
│   ※ English "too" ≠ Korean 너무 (너무 is often positive)       │
└─────────────────────────────────────────────────────────────────┘
```
