## 1. Introduction to Conditionals

### 1.1 What Are Conditionals?

Conditionals express the relationship between a **condition** (if-clause) and its **result** (main clause).

```
┌─────────────────────────────────────────────────────────────────┐
│                   Structure of Conditionals                     │
│                                                                 │
│   IF-CLAUSE (condition)    +    MAIN CLAUSE (result)           │
│   "If it rains,"               "I will stay home."             │
│                                                                 │
│   The order can be reversed:                                   │
│   MAIN CLAUSE + IF-CLAUSE (no comma needed)                    │
│   "I will stay home if it rains."                              │
│                                                                 │
│   Comma rule:                                                  │
│   If + clause, main clause.    ← comma after if-clause         │
│   Main clause if + clause.     ← no comma                     │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Overview of All Conditional Types

```
┌────────┬──────────────────┬──────────────────┬─────────────────┐
│ Type   │ If-clause        │ Main clause      │ Meaning         │
├────────┼──────────────────┼──────────────────┼─────────────────┤
│ Zero   │ present simple   │ present simple   │ always true     │
│ First  │ present simple   │ will + base      │ real/possible   │
│ Second │ past simple      │ would + base     │ unreal present  │
│ Third  │ past perfect     │ would have + PP  │ unreal past     │
│ Mixed  │ (combinations)   │ (combinations)   │ mixed time      │
└────────┴──────────────────┴──────────────────┴─────────────────┘
```

## 2. Zero Conditional

### 2.1 Formation

```
If + present simple, present simple

Used for: general truths, scientific facts, things that ALWAYS happen
```

### 2.2 Examples

```
Scientific facts:
  "If you heat water to 100°C, it boils."
  "If you mix red and blue, you get purple."

General truths:
  "If you don't eat, you get hungry."
  "If people exercise regularly, they stay healthier."

Habitual results:
  "If I drink too much coffee, I can't sleep."
  "If it rains, the streets flood." (this always happens)

Instructions/rules:
  "If the light turns red, stop."
  "If the alarm goes off, leave the building immediately."
```

### 2.3 "If" vs "When" in Zero Conditional

```
In the zero conditional, "if" and "when" are often interchangeable:

  "If/When you press this button, the door opens."
  "If/When I feel stressed, I go for a walk."

Use "when" for things that will definitely happen:
  "When water reaches 100°C, it boils." (certain)

Use "if" for things that may or may not happen:
  "If the fire alarm rings, evacuate." (may never happen)
```

## 3. First Conditional

### 3.1 Formation

```
If + present simple, will + base verb

Used for: real/possible future situations
         things that are likely or possible to happen
```

### 3.2 Examples

```
Likely future:
  "If it rains tomorrow, I will take an umbrella."
  "If you study hard, you will pass the exam."
  "If she calls, I will tell her the news."

Warnings:
  "If you touch that wire, you will get a shock."
  "If you don't hurry, you will miss the bus."

Promises:
  "If you help me, I will buy you dinner."
  "If I get the job, I will celebrate."

Negotiations:
  "If you lower the price, I will buy two."
```

### 3.3 Variations in the Main Clause

```
The main clause doesn't always use "will":

  can:     "If you finish early, you can leave."
  may:     "If the weather is nice, we may go hiking."
  might:   "If she studies, she might get an A."
  should:  "If you feel sick, you should see a doctor."
  must:    "If you want to pass, you must study."
  imperative: "If you see John, tell him I called."
```

### 3.4 Important Rule

```
┌─────────────────────────────────────────────────────────────────┐
│   NEVER use "will" in the if-clause!                           │
│                                                                 │
│   ✗ "If it will rain, I will stay home."                       │
│   ✓ "If it rains, I will stay home."                           │
│                                                                 │
│   ✗ "If you will study, you will pass."                        │
│   ✓ "If you study, you will pass."                             │
│                                                                 │
│   Exception: "will" in if-clause for willingness/requests:     │
│   ✓ "If you will wait a moment, I'll check for you."          │
│     (= if you are willing to wait)                             │
└─────────────────────────────────────────────────────────────────┘
```

## 4. Second Conditional

### 4.1 Formation

```
If + past simple, would + base verb

Used for: unreal/hypothetical present or future situations
         things that are unlikely, imaginary, or contrary to fact
```

### 4.2 Examples

```
Hypothetical situations:
  "If I had more money, I would travel the world."
  (I don't have more money — imaginary)

  "If I were you, I would accept the offer."
  (I'm not you — impossible)

  "If I won the lottery, I would buy a house."
  (unlikely to happen)

Imaginary changes:
  "If I spoke Japanese, I would work in Tokyo."
  (I don't speak Japanese)

  "If she lived closer, we would see each other more often."
  (she doesn't live close)

Dreams:
  "If I could fly, I would visit every country."
  "If I had a time machine, I would go back to the 1990s."
```

### 4.3 "Were" vs "Was" in Second Conditional

```
┌─────────────────────────────────────────────────────────────────┐
│   Formally, use "were" for ALL subjects (subjunctive mood):    │
│                                                                 │
│   "If I were rich, I would donate to charity."                 │
│   "If she were here, she would know what to do."               │
│   "If he were taller, he could play basketball."               │
│                                                                 │
│   In informal English, "was" is common with I/he/she/it:      │
│   "If I was rich..." (informal but widely accepted)            │
│                                                                 │
│   The phrase "If I were you" is ALWAYS "were":                 │
│   ✓ "If I were you, I would study harder."                     │
│   ✗ "If I was you, I would study harder." (avoid)              │
└─────────────────────────────────────────────────────────────────┘
```

### 4.4 First vs Second Conditional Contrast

```
First Conditional (possible/likely):
  "If it rains tomorrow, I will bring an umbrella."
  (It might rain — real possibility.)

Second Conditional (unlikely/hypothetical):
  "If it rained in the Sahara, everyone would be shocked."
  (Very unlikely — hypothetical.)

First:  "If I have time, I will help you." (I might have time)
Second: "If I had time, I would help you." (I don't have time)

First:  "If she asks, I will tell her." (she might ask)
Second: "If she asked, I would tell her." (she probably won't ask)
```

## 5. Third Conditional

### 5.1 Formation

```
If + past perfect (had + PP), would have + past participle

Used for: unreal past situations
         things that did NOT happen (imagining a different past)
```

### 5.2 Examples

```
Imagining a different past:
  "If I had studied harder, I would have passed the exam."
  (I didn't study hard → I didn't pass.)

  "If she had taken the earlier train, she wouldn't have been late."
  (She didn't take the earlier train → she was late.)

  "If we had known about the problem, we would have fixed it."
  (We didn't know → we didn't fix it.)

Regrets:
  "If I had been more careful, I wouldn't have broken my leg."
  "If I had listened to my mother, I would have saved more money."

Alternative history:
  "If Columbus hadn't discovered America, history would have been different."
  "If the internet hadn't been invented, life would be very different."
```

### 5.3 Contractions in Third Conditional

```
Spoken English often contracts these forms:

  "If I'd known, I'd have helped."
  = "If I had known, I would have helped."

  "If she'd called, I'd have answered."
  = "If she had called, I would have answered."

  Note: "I'd" can mean "I had" (if-clause) or "I would" (main clause)
  Context determines the meaning!
```

### 5.4 Variations

```
could have / might have (instead of would have):

  "If I had practiced, I could have won."
  (= I would have been able to win)

  "If she had told me, I might have helped."
  (= There's a chance I would have helped)
```

## 6. Mixed Conditionals

### 6.1 Past Condition → Present Result

```
If + past perfect, would + base verb

Meaning: Something didn't happen in the past, and it affects NOW.

  "If I had studied medicine, I would be a doctor now."
  (I didn't study medicine → I'm not a doctor now.)

  "If she had accepted the job, she would live in New York now."
  (She didn't accept → she doesn't live there now.)

  "If I hadn't missed the flight, I would be in Tokyo right now."
  (I missed it → I'm not in Tokyo now.)
```

### 6.2 Present Condition → Past Result

```
If + past simple, would have + past participle

Meaning: A general/permanent condition affected a past event.

  "If I spoke Chinese, I would have understood the conversation."
  (I don't speak Chinese → I didn't understand.)

  "If he were braver, he would have asked her out."
  (He's not brave → he didn't ask her out.)

  "If I weren't so shy, I would have talked to her."
  (I am shy → I didn't talk to her.)
```

### 6.3 Summary

```
┌─────────────────────────────────────────────────────────────────┐
│   MIXED CONDITIONALS                                           │
│                                                                 │
│   Type 1: Past → Present                                      │
│   If + had + PP, ... would + base verb                         │
│   "If I had saved money, I would be rich now."                 │
│                                                                 │
│   Type 2: Present → Past                                      │
│   If + past simple, ... would have + PP                        │
│   "If I were taller, I would have played basketball."          │
└─────────────────────────────────────────────────────────────────┘
```

## 7. Wish and If Only

### 7.1 Wish + Past Simple (Present Wishes)

Used to express wishes about the **present** (things you want to be different now).

```
"I wish I had more money."        (I don't have enough money now.)
"I wish I were taller."           (I'm not tall.)
"I wish I spoke French."          (I don't speak French.)
"I wish I lived by the sea."      (I don't live by the sea.)
"I wish it weren't raining."      (It is raining.)

Note: Use "were" (not "was") for all subjects (formal):
  "I wish I were..." / "I wish she were..." / "I wish it were..."
```

### 7.2 Wish + Past Perfect (Past Wishes/Regrets)

Used to express wishes about the **past** (things you regret).

```
"I wish I had studied harder."        (I didn't study hard → regret.)
"I wish I hadn't said that."          (I said it → regret.)
"I wish I had gone to the concert."   (I didn't go → regret.)
"I wish we had met sooner."           (We didn't → regret.)
```

### 7.3 Wish + Would (Complaints/Desires for Change)

Used to express **annoyance** about someone else's behavior or a desire for something to change.

```
"I wish you would stop smoking."       (You keep smoking — annoying.)
"I wish it would stop raining."        (I want the rain to stop.)
"I wish they would be quieter."        (They're being loud.)
"I wish he would listen to me."        (He doesn't listen.)

Rules:
  • Usually about OTHER people/things (not yourself)
  • ✗ "I wish I would be taller." (WRONG)
  • ✓ "I wish I were taller." (Correct)
  • Expresses frustration or desire for change
```

### 7.4 If Only

"If only" is a stronger, more emotional form of "wish."

```
Present wish:
  "If only I had a car!"              = "I wish I had a car."
  "If only I knew the answer!"        = "I wish I knew the answer."

Past regret:
  "If only I had listened to you!"    = "I wish I had listened."
  "If only we hadn't missed the train!" = "I wish we hadn't missed it."

Complaint:
  "If only he would call!"            = "I wish he would call."
  "If only it would stop snowing!"    = "I wish it would stop."
```

### 7.5 Wish Summary Table

| Structure | Time | Meaning | Example |
|-----------|------|---------|---------|
| wish + past simple | Present | Unreal present | I wish I **spoke** English fluently. |
| wish + past perfect | Past | Regret | I wish I **had studied** more. |
| wish + would | Future | Complaint/desire for change | I wish it **would** stop raining. |
| if only + past simple | Present | Strong present wish | If only I **were** rich! |
| if only + past perfect | Past | Strong regret | If only I **had known**! |

## 8. Alternative Conditional Words

### 8.1 Unless (= if not)

```
"unless" = "if ... not"

  "Unless you study, you will fail."
  = "If you don't study, you will fail."

  "I won't go unless you come with me."
  = "I won't go if you don't come with me."

  "Unless it rains, we'll have the picnic outside."
  = "If it doesn't rain, we'll have the picnic outside."

  ⚠ Don't use "unless" with negative main clauses + negative meaning:
  ✗ "Unless you don't stop, I'll call the police."
  ✓ "Unless you stop, I'll call the police."
```

### 8.2 Provided that / Providing that

```
"provided (that)" = "only if" / "on the condition that"
More formal than "if"

  "I'll lend you the money provided (that) you pay me back."
  "You can go out provided that you finish your homework first."
  "The event will proceed provided the weather is good."
```

### 8.3 As long as / So long as

```
"as long as" = "on the condition that" / "provided that"

  "You can borrow my car as long as you drive carefully."
  "I don't mind waiting as long as it's not too long."
  "As long as you're happy, I'm happy."
```

### 8.4 On condition that / In case

```
"on condition that" = formal version of "if"
  "I'll help you on condition that you help me in return."

"in case" = as a precaution / because it might happen
  "Take an umbrella in case it rains."
  (= because it might rain)
  ≠ "Take an umbrella if it rains."
  (= only when it rains)
```

### 8.5 Even if / Whether or not

```
"even if" = the result is the same regardless:
  "Even if it rains, we'll go hiking."
  "I wouldn't buy it even if I had the money."

"whether or not" = regardless of the condition:
  "Whether or not you agree, I'm going."
  "I'll call you whether or not I get the result."
```

### 8.6 Suppose / What if

```
"suppose" / "what if" = imagine this situation:
  "Suppose you won the lottery, what would you do?"
  "What if the plan fails?"
  "Suppose it rains, do we have a backup plan?"
```

## 9. Polite Conditionals

### 9.1 Using Conditionals for Politeness

```
Second conditional for polite requests:
  Direct:    "Can you help me?"
  Polite:    "Could you help me?"
  Very polite: "Would you mind helping me?"

  "It would be great if you could send me the report."
  "I would appreciate it if you would reply soon."
  "If you wouldn't mind, could you close the window?"

"I wonder if..." (very polite):
  "I wonder if you could help me."
  "I was wondering if you could spare a moment."
```

### 9.2 Common Polite Patterns

```
"Would you mind if I...?"
  "Would you mind if I opened the window?"
  "Would you mind if I sat here?"
  (Note: past tense "opened/sat" makes it more polite)

"If it's not too much trouble..."
  "If it's not too much trouble, could you send me a copy?"

"Would it be possible to...?"
  "Would it be possible to reschedule the meeting?"
```

## 10. Korean Speaker Notes

### 10.1 Key Differences

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean conditionals use -(으)면:                              │
│                                                                 │
│   비가 오면, 집에 있을 거야.                                      │
│   "If it rains, I will stay home."                              │
│                                                                 │
│   Korean does NOT change tense for hypothetical conditions:    │
│   내가 부자면, 여행을 갈 텐데.                                     │
│   "If I were rich, I would travel." (no past tense in Korean)  │
│                                                                 │
│   This is why Korean speakers often make these errors:         │
│                                                                 │
│   ✗ "If I am rich, I would travel."                            │
│     (mixing first and second conditional)                      │
│   ✓ "If I were rich, I would travel."                          │
│                                                                 │
│   ✗ "If I study harder, I would pass."                         │
│     (mixing first and second conditional)                      │
│   ✓ "If I studied harder, I would pass." (hypothetical)        │
│   ✓ "If I study harder, I will pass." (real possibility)       │
└─────────────────────────────────────────────────────────────────┘
```

### 10.2 Common Korean Speaker Errors

```
Error 1: Using "will" in the if-clause
  ✗ "If it will rain, I will stay home."
  ✓ "If it rains, I will stay home."

Error 2: Not backshifting for hypothetical
  ✗ "If I have more time, I would help you."
  ✓ "If I had more time, I would help you."

Error 3: Forgetting "have" in third conditional
  ✗ "If I had studied, I would passed."
  ✓ "If I had studied, I would have passed."

Error 4: Using "would" in the if-clause
  ✗ "If I would know, I would tell you."
  ✓ "If I knew, I would tell you."

Error 5: Confusing "wish" tense
  ✗ "I wish I can speak English better."
  ✓ "I wish I could speak English better."
```

## Quick Reference

```
┌─────────────────────────────────────────────────────────────────┐
│                  Conditionals Quick Guide                       │
│                                                                 │
│   Zero:  If + present, present         (always true)           │
│   1st:   If + present, will + base     (real/possible)         │
│   2nd:   If + past, would + base       (unreal present)        │
│   3rd:   If + past perfect, would have + PP (unreal past)      │
│   Mixed: Combinations of 2nd and 3rd                           │
│                                                                 │
│   WISH:                                                        │
│   Present: wish + past simple   "I wish I knew"               │
│   Past:    wish + past perfect  "I wish I had known"           │
│   Change:  wish + would         "I wish he would stop"         │
│                                                                 │
│   ALTERNATIVES:                                                │
│   unless = if not                                              │
│   provided that / as long as = on the condition that           │
│   even if = regardless                                         │
│   in case = as a precaution                                    │
│                                                                 │
│   NEVER use "will" or "would" in the if-clause!                │
└─────────────────────────────────────────────────────────────────┘
```
