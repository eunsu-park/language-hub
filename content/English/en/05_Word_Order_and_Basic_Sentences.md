## 1. SVO vs SOV: The Fundamental Difference

### 1.1 The Core Contrast

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean: SOV (Subject + Object + Verb)                         │
│   나는     사과를      먹는다                                     │
│   S        O          V                                         │
│   I        apple      eat                                       │
│                                                                 │
│   English: SVO (Subject + Verb + Object)                        │
│   I        eat        an apple                                  │
│   S        V          O                                         │
│                                                                 │
│   The VERB moves from the END to the MIDDLE.                    │
│   Everything after the verb is rearranged.                      │
│                                                                 │
│   Korean: 나는 [어제] [학교에서] [영어를] [공부했다]                 │
│           S    Time    Place     Object   Verb                   │
│                                                                 │
│   English: I [studied] [English] [at school] [yesterday]        │
│            S  Verb     Object    Place       Time                │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Why Word Order Matters More in English

```
Korean has particles (조사) that mark grammatical roles:
  나는 (topic)  사과를 (object)  먹는다 (verb)
  사과를 나는 먹는다    → Same meaning (object-first OK due to 를)
  먹는다 나는 사과를    → Unusual but understandable due to particles

English uses POSITION to mark grammatical roles:
  "I eat an apple."    → I = subject, apple = object
  "An apple eats me."  → apple = subject, me = object (DIFFERENT meaning!)

  ※ In English, if you change the word order, you change the meaning.
    There are no Korean-style particles to save you.
    POSITION = MEANING in English.
```

### 1.3 Multi-Element Sentences

```
Korean order: S + Time + Place + Manner + Object + Verb
  나는 매일 학교에서 열심히 영어를 공부한다.
  I   daily school-at hard  English study

English order: S + Verb + Object + Manner + Place + Time
  I study English hard at school every day.
  S V      O       Manner Place   Time

  ※ Time/Place phrases are flexible in English:
    "Every day I study English at school."  ✓ (time at front OK)
    "At school, I study English every day." ✓ (place at front OK)
    But the core S + V + O is FIXED.
```

## 2. Statement Word Order

### 2.1 Basic Patterns

```
Pattern 1: S + V (Subject + Intransitive Verb)
  I sleep.                    나는 잔다.
  She runs.                   그녀는 달린다.
  The baby cries.             아기가 운다.

Pattern 2: S + V + O (Subject + Verb + Object)
  I eat rice.                 나는 밥을 먹는다.
  She reads books.            그녀는 책을 읽는다.
  He plays the guitar.        그는 기타를 친다.

Pattern 3: S + V + C (Subject + Verb + Complement)
  I am happy.                 나는 행복하다.
  She is a teacher.           그녀는 선생님이다.
  They look tired.            그들은 피곤해 보인다.

Pattern 4: S + V + O + O (Subject + Verb + Indirect Object + Direct Object)
  I gave her a book.          나는 그녀에게 책을 줬다.
  She told me the truth.      그녀는 나에게 사실을 말했다.

Pattern 5: S + V + O + C (Subject + Verb + Object + Complement)
  I call him Tom.             나는 그를 톰이라 부른다.
  They made me happy.         그들은 나를 행복하게 했다.
```

### 2.2 English Has Fixed Slots

```
┌─────────────────────────────────────────────────────────────────┐
│   Slot 1    Slot 2    Slot 3    Slot 4    Slot 5    Slot 6     │
│   (Time)    Subject   Verb      Object    Place     Time       │
│                                                                 │
│   Yesterday  I        met       Tom       at school             │
│              She      drinks    coffee    in the cafe  daily    │
│   Every day  he       walks                to work              │
│              They     play      soccer    in the park  on Sun.  │
│                                                                 │
│   ※ Subject-Verb-Object is NEVER reordered in statements.     │
│     "Coffee she drinks" ✗ (not standard English)                │
│     "She drinks coffee" ✓                                       │
└─────────────────────────────────────────────────────────────────┘
```

## 3. Question Word Order

### 3.1 Yes/No Questions

```
Two methods for making yes/no questions:

Method 1: BE/Auxiliary verb — move it BEFORE the subject (Inversion)
  She IS happy.          → IS she happy?
  They ARE coming.       → ARE they coming?
  He CAN swim.           → CAN he swim?
  She HAS finished.      → HAS she finished?

Method 2: Regular verbs — add DO/DOES before subject (Do-support)
  You like coffee.       → DO you like coffee?
  She plays tennis.      → DOES she play tennis?
  They work here.        → DO they work here?

  ※ Korean question formation:
    Statement: 그녀는 행복하다
    Question:  그녀는 행복하니? / 그녀는 행복합니까?
    → Just change the ENDING. No word order change!

    English CHANGES WORD ORDER for questions.
    This is one of the hardest habits to build.
```

### 3.2 Wh-Questions

```
Formula: Wh-word + auxiliary/do + subject + verb?

Where do you live?            어디에 사니?
What does she eat?            뭘 먹니?
When do they arrive?          언제 도착하니?
Why is he sad?                왜 슬프니?
How are you?                  어떻게 지내니?
Which book do you want?       어떤 책을 원하니?
Whose car is this?            이것은 누구의 차니?
How much does it cost?        얼마나 하니?
How many books do you have?   몇 권의 책이 있니?
```

### 3.3 Subject Questions (No Inversion!)

```
When the wh-word IS the subject → NO do-support, NO inversion:

  "Who lives here?"       (NOT "Who does live here?")
  → "Who" is the subject of "lives"

  "What happened?"        (NOT "What did happen?")
  → "What" is the subject of "happened"

  "Which bus goes to Seoul?" (NOT "Which bus does go?")
  → "Which bus" is the subject of "goes"

Compare:
  Subject question:  "Who called you?"     (누가 너에게 전화했니?)
  Object question:   "Who did you call?"   (너는 누구에게 전화했니?)
  → Same word "who" but different word order!
```

### 3.4 Tag Questions

```
Tag questions seek confirmation (맞지? / 그렇지?):

Positive statement → negative tag:
  "You are Korean, aren't you?"       (한국인이지, 그렇지?)
  "She likes coffee, doesn't she?"    (커피 좋아하지, 그렇지?)
  "They came yesterday, didn't they?" (어제 왔지, 그렇지?)

Negative statement → positive tag:
  "You aren't tired, are you?"        (피곤하지 않지, 그렇지?)
  "He doesn't smoke, does he?"       (담배 안 피우지, 그렇지?)

  ※ Korean just adds 그렇지? or 맞지? — much simpler!
```

## 4. Negation

### 4.1 Negation with BE

```
Subject + BE + not:
  I am not tired.              나는 피곤하지 않다.
  She is not here.             그녀는 여기 없다.
  They are not ready.          그들은 준비되지 않았다.

Contractions:
  I'm not / she isn't (she's not) / they aren't (they're not)
```

### 4.2 Negation with DO/DOES

```
Subject + do/does + not + base verb:
  I do not like fish.          → I don't like fish.
  She does not eat meat.       → She doesn't eat meat.
  They do not work weekends.   → They don't work weekends.
```

### 4.3 Negation with Other Auxiliaries

```
Subject + auxiliary + not:
  I cannot swim.               → I can't swim.
  She will not come.           → She won't come.
  They have not finished.      → They haven't finished.
  He should not go.            → He shouldn't go.
```

### 4.4 Negative Words

```
┌─────────────────────────────────────────────────────────────────┐
│   English Negative Words                                        │
│                                                                 │
│   no        = 아니 / 없는      "I have no money."               │
│   not       = ~않다            "I do not know."                  │
│   never     = 절대 ~않다       "I never smoke."                  │
│   nothing   = 아무것도 ~않다   "I know nothing." / "I don't      │
│                                 know anything."                  │
│   nobody    = 아무도 ~않다     "Nobody came."                     │
│   no one    = 아무도 ~않다     "No one knows."                    │
│   nowhere   = 아무 데도 ~않다  "I have nowhere to go."           │
│   neither   = 둘 다 ~않다      "Neither answer is correct."      │
│                                                                 │
│   ※ CRITICAL RULE: English allows only ONE negative per clause!│
│     "I don't know nothing."  ✗ (double negative = WRONG)        │
│     "I don't know anything." ✓                                  │
│     "I know nothing."        ✓                                  │
│                                                                 │
│     Korean allows double/triple negatives:                      │
│     "나는 아무것도 안 먹지 않았다" (multiple negatives OK)          │
│     English: ONE negative per clause only!                      │
└─────────────────────────────────────────────────────────────────┘
```

## 5. There is / There are

### 5.1 Basic Usage

```
"There is/are" expresses EXISTENCE (존재):

Korean: 탁자 위에 책이 있다.
English: There is a book on the table.

  ※ Korean: [location] + [thing] + 있다
    English: There + is/are + [thing] + [location]
    → Opposite order!

There is  + singular noun:    There is a cat in the garden.
There is  + uncountable noun: There is water on the floor.
There are + plural noun:      There are three books on the table.
```

### 5.2 Forms

```
Positive:
  There is (There's) a problem.       문제가 있다.
  There are many people here.         여기 사람이 많다.

Negative:
  There is not (There isn't) any milk.   우유가 없다.
  There are not (There aren't) any chairs. 의자가 없다.
  There is no milk.                      우유가 없다.

Question:
  Is there a bathroom?                   화장실이 있나요?
  Are there any questions?               질문 있나요?

Short answers:
  Is there a park nearby? → Yes, there is. / No, there isn't.
  Are there any shops?    → Yes, there are. / No, there aren't.
```

### 5.3 there vs it

```
"There" = introduces NEW information (existence):
  "There is a book on the table."    (new info: a book exists)

"It" = refers to something already known:
  "It is on the table."             (we already know about the book)

  ※ Korean uses 있다 for both situations.
    English distinguishes:
    "있다 (새 정보)" → there is/are
    "있다 (알고 있는 것)" → it is / they are
```

## 6. Imperatives (Commands)

### 6.1 Formation

```
┌─────────────────────────────────────────────────────────────────┐
│   Imperatives = base verb (no subject needed!)                  │
│                                                                 │
│   Positive:                                                     │
│   "Sit down."                앉아. / 앉으세요.                    │
│   "Open the door."           문을 열어.                           │
│   "Be quiet."                조용히 해.                           │
│   "Come here."               여기로 와.                           │
│   "Listen to me."            내 말을 들어.                        │
│                                                                 │
│   Negative (Don't + base verb):                                 │
│   "Don't sit down."          앉지 마.                            │
│   "Don't open the door."     문을 열지 마.                        │
│   "Don't be late."           늦지 마.                            │
│   "Don't worry."             걱정하지 마.                         │
│                                                                 │
│   Polite imperative (add "please"):                             │
│   "Please sit down."         앉아 주세요.                        │
│   "Don't run, please."       뛰지 말아 주세요.                    │
│                                                                 │
│   ※ English imperatives have NO subject "you":                 │
│     "You sit down!" — This sounds angry/emphatic!               │
│     "Sit down." — Normal command                                │
│     "Sit down, please." — Polite command                        │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 Special Imperative: Let's

```
"Let's" = suggestion for "we" (우리 ~하자):

  "Let's go."              가자.
  "Let's eat lunch."       점심 먹자.
  "Let's start."           시작하자.

Negative:
  "Let's not argue."       싸우지 말자.
  "Let's not be late."     늦지 말자.
```

## 7. Exclamations

### 7.1 What/How Exclamations

```
What + (a/an) + adjective + noun + (S + V)!
  "What a beautiful day!"            정말 아름다운 날이다!
  "What a big house!"                정말 큰 집이다!
  "What an interesting story!"       정말 재미있는 이야기다!
  "What beautiful flowers!"          정말 아름다운 꽃들이다! (plural — no a/an)
  "What terrible weather!"           정말 끔찍한 날씨다! (uncountable — no a/an)

How + adjective/adverb + (S + V)!
  "How beautiful!"                   정말 아름답다!
  "How fast he runs!"                그는 정말 빨리 달린다!
  "How kind of you!"                 정말 친절하시네요!

  ※ Korean uses 정말/진짜/너무 + adjective: "정말 예쁘다!"
    English uses "What a..." or "How..." structures.
    "What a...!" is more common in everyday speech.
```

## 8. Word Order with Adverbs and Prepositional Phrases

### 8.1 Adverb Position

```
Frequency adverbs → before main verb, after BE:
  "I always eat breakfast."       (before eat)
  "She is always happy."          (after is)

Manner adverbs → after verb/object:
  "She sings beautifully."        (after verb)
  "He speaks English well."       (after object)
  ※ NOT "He well speaks English" (Korean order)

Time/Place → usually at the end (Place before Time):
  "I study English at school every day."
                     (Place)    (Time)
  ※ Flexible: "Every day, I study English at school." ✓
```

### 8.2 Place before Time (장소 → 시간)

```
Korean: Time → Place (시간 먼저)
  나는 매일 학교에서 공부한다.
  I   daily  school-at study

English: Place → Time (장소 먼저)
  I study at school every day.
          (Place)   (Time)

More examples:
  "I was born in Seoul in 1995."      (place → time)
  "She works at a hospital on weekdays." (place → time)
  "We met at a cafe last Friday."     (place → time)
```

### 8.3 SVOMPT Order

```
Full sentence order: Subject + Verb + Object + Manner + Place + Time

  S     V       O          M           P              T
  She   plays   the piano  beautifully at home        every evening.
  I     read    the book   carefully   in the library yesterday.
  He    drives  his car    slowly      on the highway in winter.

  ※ This is a GUIDELINE. Time and place phrases are flexible:
    "Yesterday I read the book carefully in the library." ✓
    But S + V + O order is always fixed.
```

## 9. Common Word Order Errors by Korean Speakers

### 9.1 Error Collection

```
┌─────────────────────────────────────────────────────────────────┐
│   Error (Korean order)              Correct (English order)     │
│   ───────────────────               ─────────────────────       │
│   "I English study."               "I study English."          │
│   (SOV → should be SVO)                                        │
│                                                                 │
│   "Yesterday I to school went."    "Yesterday I went to school."│
│   (verb at end → verb in middle)                                │
│                                                                 │
│   "She very is pretty."            "She is very pretty."        │
│   (adverb before be → after be)                                │
│                                                                 │
│   "You what eat?"                  "What do you eat?"           │
│   (Korean question order)                                       │
│                                                                 │
│   "I tomorrow will go."            "I will go tomorrow."        │
│   (time in middle → time at end)                                │
│                                                                 │
│   "In Seoul I live."               "I live in Seoul."           │
│   (place first → place after verb)                              │
│                                                                 │
│   "Coffee to me please give."      "Please give me coffee."    │
│   (full Korean order)                                           │
│                                                                 │
│   "Don't late be."                 "Don't be late."             │
│   (Korean order: 늦지 마)                                        │
└─────────────────────────────────────────────────────────────────┘
```

### 9.2 The Subject-Drop Problem

```
Korean frequently drops the subject:
  (나는) 배고프다.   → Just "배고프다" is fine in Korean.

English REQUIRES the subject:
  "Am hungry." ✗
  "I am hungry." ✓

  "Is raining." ✗
  "It is raining." ✓  (even "it" — a meaningless placeholder — is required!)

  "Like coffee." ✗
  "I like coffee." ✓

  ※ The ONLY exception: imperatives
    "Sit down." ✓ (subject "you" is implied in commands)
```
