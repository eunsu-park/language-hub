## 1. Introduction to Vocabulary Building

### 1.1 Why Systematic Vocabulary Learning?

```
┌─────────────────────────────────────────────────────────────────┐
│   Native English speakers know ~20,000-35,000 word families.   │
│                                                                 │
│   For effective communication, learners need:                  │
│   • ~3,000 word families for everyday conversation             │
│   • ~5,000-6,000 for understanding news and media              │
│   • ~8,000-9,000 for reading authentic texts                   │
│   • ~10,000+ for academic and professional contexts            │
│                                                                 │
│   Strategy: Learn words in CONTEXT and through WORD FAMILIES   │
│   instead of memorizing isolated word-translation pairs.       │
│                                                                 │
│   Example word family:                                         │
│   employ → employer, employee, employment, unemployment,       │
│           employable, unemployable, self-employed               │
│   One root = 7+ words learned!                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 6. Word Formation: Prefixes

### 6.1 Negative Prefixes

```
un-    (not, opposite)
  happy → unhappy        usual → unusual
  fair → unfair          do → undo

in-    (not) — changes to il-, im-, ir- before certain letters
  active → inactive      visible → invisible
  legal → illegal        mature → immature
  regular → irregular    responsible → irresponsible
  complete → incomplete  patient → impatient

dis-   (not, opposite, apart)
  agree → disagree       appear → disappear
  connect → disconnect   advantage → disadvantage
  honest → dishonest     like → dislike

non-   (not)
  fiction → nonfiction   sense → nonsense
  profit → nonprofit    violent → nonviolent

mis-   (wrongly, badly)
  understand → misunderstand    lead → mislead
  spell → misspell              use → misuse
  judge → misjudge              behave → misbehave
```

### 6.2 Positive and Directional Prefixes

```
re-    (again, back)
  do → redo              write → rewrite
  build → rebuild        start → restart
  cycle → recycle        use → reuse

pre-   (before)
  view → preview         heat → preheat
  historic → prehistoric war → prewar
  pay → prepay           school → preschool

post-  (after)
  war → postwar          modern → postmodern
  graduate → postgraduate

over-  (too much)
  eat → overeat          work → overwork
  load → overload        react → overreact
  sleep → oversleep      estimate → overestimate

under- (too little, below)
  estimate → underestimate      pay → underpay
  value → undervalue            line → underline
  ground → underground          age → underage

out-   (more than, beyond)
  do → outdo             perform → outperform
  number → outnumber     grow → outgrow
  run → outrun           live → outlive
```

### 6.3 Size and Degree Prefixes

```
super- (above, beyond)    superhero, supernatural, supervise
sub-   (under, below)     subway, subtitle, subconscious
semi-  (half, partly)     semicircle, semifinal, semicolon
multi- (many)             multilingual, multimedia, multinational
mono-  (one, single)      monolingual, monopoly, monotone
bi-    (two)              bilingual, bicycle, biweekly
tri-   (three)            triangle, trilingual, trilogy
inter- (between)          international, interact, internet
trans- (across)           transport, translate, transform
co-    (together)         cooperate, coexist, coworker
anti-  (against)          antibody, antisocial, antibiotic
auto-  (self)             automatic, autobiography, autonomous
```

## 7. Word Formation: Suffixes

### 7.1 Noun Suffixes

```
-tion / -sion (action, state)
  educate → education          decide → decision
  inform → information         express → expression
  communicate → communication  conclude → conclusion

-ment (result, action)
  develop → development        achieve → achievement
  manage → management          agree → agreement
  improve → improvement        govern → government

-ness (state, quality)
  happy → happiness           dark → darkness
  kind → kindness             aware → awareness
  sad → sadness               weak → weakness

-ity / -ty (state, quality)
  real → reality              creative → creativity
  electric → electricity      able → ability
  diverse → diversity         secure → security

-er / -or (person who does)
  teach → teacher             act → actor
  drive → driver              invent → inventor
  lead → leader               visit → visitor

-ist (person with a belief/skill)
  science → scientist         art → artist
  tour → tourist              piano → pianist
  journal → journalist        novel → novelist

-ance / -ence (state, quality)
  perform → performance       exist → existence
  assist → assistance         differ → difference
  import → importance         depend → dependence
```

### 7.2 Adjective Suffixes

```
-able / -ible (can be done)
  afford → affordable         access → accessible
  rely → reliable             flex → flexible
  comfort → comfortable       response → responsible

-ful (full of)
  hope → hopeful              care → careful
  beauty → beautiful          success → successful
  power → powerful            peace → peaceful

-less (without)
  hope → hopeless             care → careless
  home → homeless             use → useless
  end → endless               meaning → meaningless

-ous / -ious / -eous (having the quality of)
  danger → dangerous          ambition → ambitious
  courage → courageous        religion → religious
  fame → famous               mystery → mysterious

-al / -ial (relating to)
  nature → natural            finance → financial
  music → musical             industry → industrial
  person → personal           commerce → commercial

-ive (having the nature of)
  act → active                create → creative
  attract → attractive        effect → effective
  impress → impressive        expense → expensive
```

### 7.3 Verb Suffixes

```
-ize / -ise (to make, to become)
  modern → modernize          organ → organize
  standard → standardize      real → realize
  special → specialize        custom → customize

-ify (to make, to cause)
  simple → simplify           intense → intensify
  class → classify            beauty → beautify
  just → justify              identity → identify

-en (to make, to become)
  wide → widen                deep → deepen
  strength → strengthen       short → shorten
  dark → darken               bright → brighten
```

### 7.4 Adverb Suffix

```
-ly (in a ... manner)
  quick → quickly             careful → carefully
  happy → happily             serious → seriously
  unfortunate → unfortunately easy → easily
```

### 8.3 How to Decode Unknown Words

```
Example: "telecommunication"
  tele (far) + com (together) + munic (share) + ation (noun)
  = sharing information over distances

Example: "autobiography"
  auto (self) + bio (life) + graph (write) + y (noun)
  = writing about one's own life

Example: "unprecedented"
  un (not) + pre (before) + ced (go) + ent (adjective) + ed
  = never having gone before = never happened before

Example: "biodegradable"
  bio (life) + de (down) + grad (step) + able (can be)
  = can be broken down by living organisms
```

## 9. Other Word-Building Strategies

### 9.1 Compounding

```
Two words combined to form a new word:

Noun + Noun:     airport, bedroom, newspaper, bookshelf
Adj + Noun:      blackboard, greenhouse, highway, software
Noun + Verb:     sunrise, rainfall, headache, earthquake
Verb + Noun:     washing machine, swimming pool, driving license
Adj + Adj:       bittersweet, well-known, red-hot
```

### 9.2 Conversion (Zero Derivation)

```
Using a word as a different part of speech without changing form:

Noun → Verb:     "Email me." / "Google it." / "Bookmark this page."
Verb → Noun:     "a run" / "a swim" / "a try" / "a look"
Adj → Verb:      "clean the room" / "dry your hair" / "empty the box"
Adj → Noun:      "the poor" / "the rich" / "the elderly"
```

### 9.3 Blending

```
Parts of two words combined:

brunch    = breakfast + lunch
smog      = smoke + fog
internet  = inter(national) + net(work)
email     = electronic + mail
blog      = web + log
podcast   = iPod + broadcast
frenemy   = friend + enemy
```

### 9.4 Clipping

```
Shortening a longer word:

exam      = examination
lab       = laboratory
ad        = advertisement
flu       = influenza
phone     = telephone
gym       = gymnasium
fridge    = refrigerator
info      = information
memo      = memorandum
photo     = photograph
```

## 10. Korean Speaker Notes

### 10.1 Cognates and False Friends

```
Many English words exist as loanwords in Korean (Konglish),
but some have shifted meaning:

  아르바이트 (arbeit) → part-time job (not "arbeit" from German)
    English: "part-time job" or "gig"

  핸들 (handle) → steering wheel
    English: "steering wheel" (a handle is a door/bag part)

  노트북 (notebook) → laptop computer
    English: "laptop" (a notebook is a paper book for writing)

  컨닝 (cunning) → cheating on an exam
    English: "cheating" (cunning means clever/sly)

  서비스 (service) → free extra item
    English: "complimentary" or "on the house"

  원룸 (one room) → studio apartment
    English: "studio apartment" or "studio"

  오바이트 (overeat) → vomiting
    English: "throw up" or "vomit" (overeat = eat too much)
```

### 10.2 Word Formation Differences

```
Korean often uses Sino-Korean compounds where English uses
Latin/Greek roots:

  교육 (教育) = education (educ + ation)
  경제 (經濟) = economy (eco + nomy)
  민주 (民主) = democracy (demo + cracy)
  과학 (科學) = science (scient + ia)

Understanding this parallel helps:
  Korean Sino-Korean = English Latin/Greek roots
  Both build complex words from meaningful building blocks!
```
