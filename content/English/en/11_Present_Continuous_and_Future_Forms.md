- Understand three ways to express the future: will, going to, present continuous
- Learn when to use each future form with clear guidelines
- Use common future time expressions naturally

## 1. Present Continuous Formation

### 1.1 Structure

```
┌─────────────────────────────────────────────────────────────────┐
│   Present Continuous = BE (am/is/are) + verb-ING               │
│                                                                 │
│   I          am eating      (I'm eating)                        │
│   you        are eating     (you're eating)                     │
│   he/she/it  is eating      (he's/she's/it's eating)           │
│   we         are eating     (we're eating)                      │
│   they       are eating     (they're eating)                    │
│                                                                 │
│   Negative: am/is/are + NOT + verb-ing                         │
│   I'm not eating.    She isn't eating.    They aren't eating.  │
│                                                                 │
│   Question: Am/Is/Are + subject + verb-ing?                    │
│   Are you eating?    Is she eating?    What are they eating?   │
│                                                                 │
│   ※ Korean has -고 있다 (하고 있다, 먹고 있다) for continuous,   │
│     but Korean speakers use it less frequently than English.    │
│     English REQUIRES present continuous for ongoing actions.    │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Spelling Rules for -ing

```
Rule 1: Most verbs → add -ing
  eat → eating      play → playing     read → reading
  go → going        study → studying   work → working

Rule 2: Verbs ending in -e → drop -e, add -ing
  make → making     write → writing    come → coming
  live → living     dance → dancing    have → having
  ※ Exception: be → being, see → seeing, agree → agreeing

Rule 3: Short CVC verbs → double final consonant + -ing
  sit → sitting     run → running      swim → swimming
  get → getting     cut → cutting      stop → stopping
  put → putting     begin → beginning  plan → planning
  ※ Don't double: w, x, y (show → showing, fix → fixing)

Rule 4: Verbs ending in -ie → change to -ying
  die → dying       lie → lying        tie → tying

Rule 5: Verbs ending in -c → add -king
  panic → panicking   picnic → picnicking
```

## 2. Present Continuous Uses

### 2.1 Actions Happening RIGHT NOW

```
"I'm eating lunch."              나는 지금 점심을 먹고 있다.
"She's studying English."        그녀는 영어를 공부하고 있다.
"They're playing soccer."        그들은 축구를 하고 있다.
"Look! It's raining."            봐! 비가 오고 있어.
"Be quiet. The baby is sleeping." 조용히 해. 아기가 자고 있어.
```

### 2.2 Temporary Situations (Around Now)

```
"I'm living in Seoul this month."    나는 이번 달에 서울에 살고 있다.
(temporary — not my permanent home)

"She's working at a cafe this summer." 그녀는 이번 여름에 카페에서 일하고 있다.
(temporary job)

"He's reading a great book."        그는 좋은 책을 읽고 있다.
(currently in the middle of reading it)
```

### 2.3 Changing/Developing Situations

```
"The weather is getting colder."     날씨가 점점 추워지고 있다.
"Prices are rising."                 물가가 오르고 있다.
"My English is improving."           내 영어가 나아지고 있다.
"The population is growing."        인구가 증가하고 있다.
```

### 2.4 Annoying Repeated Actions (with always/constantly)

```
"She's always complaining!"          그녀는 항상 불평해! (짜증)
"He's constantly losing his keys."   그는 계속 열쇠를 잃어버려!
"They're always making noise."       그들은 항상 시끄럽게 해!

※ "always" + present continuous = annoyance/irritation
   "always" + simple present = neutral habit
   "She always complains." (fact, neutral)
   "She's always complaining!" (annoying! 짜증 표현)
```

## 3. State Verbs vs Action Verbs

### 3.1 State Verbs (NOT used with -ing)

```
┌─────────────────────────────────────────────────────────────────┐
│   These verbs describe STATES, not actions.                     │
│   They are NOT normally used in present continuous.             │
│                                                                 │
│   Mental states:                                                │
│   know, believe, think (=believe), understand, remember,        │
│   forget, imagine, suppose, mean, realize, recognize            │
│                                                                 │
│   Emotions:                                                     │
│   like, love, hate, want, need, prefer, wish, care              │
│                                                                 │
│   Senses (involuntary):                                         │
│   see, hear, smell, taste, feel (=have the feeling)             │
│                                                                 │
│   Possession:                                                   │
│   have (=possess), own, belong to, possess, contain, include    │
│                                                                 │
│   Other:                                                        │
│   be, seem, appear, cost, weigh, measure, exist, matter,       │
│   depend, deserve, fit, owe                                     │
│                                                                 │
│   ✓ "I know the answer."       (simple present)                │
│   ✗ "I'm knowing the answer."  (WRONG)                         │
│                                                                 │
│   ✓ "She likes coffee."        (simple present)                │
│   ✗ "She's liking coffee."     (WRONG)                         │
│                                                                 │
│   ✓ "I have a car."            (simple present — possession)   │
│   ✗ "I'm having a car."        (WRONG)                         │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Verbs with BOTH State and Action Meanings

```
think:
  "I think it's good."        (= believe, state)     ✓ simple
  "I'm thinking about it."    (= considering, action) ✓ continuous

have:
  "I have a car."             (= possess, state)      ✓ simple
  "I'm having lunch."         (= eating, action)       ✓ continuous
  "I'm having a great time."  (= experiencing, action) ✓ continuous

see:
  "I see the mountain."       (= perceive, state)     ✓ simple
  "I'm seeing the doctor."    (= visiting, action)     ✓ continuous

taste:
  "This cake tastes good."    (= has the flavor, state) ✓ simple
  "I'm tasting the soup."     (= actively trying, action) ✓ continuous

look:
  "She looks happy."          (= appears, state)       ✓ simple
  "She's looking at me."      (= directing eyes, action) ✓ continuous

weigh:
  "I weigh 70 kg."            (= my weight is, state)  ✓ simple
  "I'm weighing the flour."   (= measuring, action)    ✓ continuous

※ Korean 생각하다 can always use -고 있다: "생각하고 있다" ✓
  English "think" depends on meaning:
  "thinking about" (considering) ✓ continuous
  "think that..." (believing) ✗ NOT continuous
```

## 4. Present Continuous vs Simple Present

### 4.1 The Key Distinction

```
┌─────────────────────────────────────────────────────────────────┐
│   Simple Present          vs    Present Continuous               │
│   ──────────────                ────────────────────             │
│   Habits/routines               Actions happening NOW           │
│   "I eat lunch at noon."       "I'm eating lunch (right now)."  │
│                                                                 │
│   Permanent situations          Temporary situations            │
│   "I live in Seoul."           "I'm living in Seoul (this month)│
│                                                                 │
│   General facts                 Current changes                 │
│   "Water boils at 100°C."     "The water is boiling!"          │
│                                                                 │
│   Scheduled events              Annoying habits                 │
│   "The train leaves at 9."    "She's always leaving late!"     │
│                                                                 │
│   ※ Korean often doesn't distinguish these clearly:            │
│     "나는 밥을 먹는다" = both "I eat rice" (habit) AND           │
│                         "I'm eating rice" (now)                 │
│     English REQUIRES the distinction!                           │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Examples Side by Side

```
Simple Present:                   Present Continuous:
"I work at a bank."              "I'm working from home today."
(permanent job)                   (temporary, just today)

"She speaks French."              "She's speaking French right now."
(ability/skill)                   (at this moment)

"It rains a lot in summer."      "It's raining outside."
(general fact)                    (happening now)

"He plays tennis on Sundays."    "He's playing tennis right now."
(regular habit)                   (at this moment)
```

## 5. Future with will

### 5.1 Formation

```
Positive: subject + will + base verb
  I will go.        (I'll go.)
  She will come.    (She'll come.)

Negative: subject + will not + base verb
  I will not go.    (I won't go.)
  She will not come. (She won't come.)

Question: Will + subject + base verb?
  Will you go?      Will she come?

  ※ "will" is the same for ALL subjects (no changes!)
    I will, you will, he will, she will, we will, they will
```

### 5.2 When to Use will

```
1. Spontaneous decisions (즉석 결정 — decided at the moment of speaking):
   (Phone rings) "I'll answer it."       내가 받을게.
   "It's cold. I'll close the window."   내가 창문 닫을게.
   "I'm hungry. I'll make a sandwich."   샌드위치 만들어야겠다.

2. Predictions (예측 — what you think will happen):
   "It will rain tomorrow."              내일 비가 올 거야.
   "She'll be a great doctor."           그녀는 훌륭한 의사가 될 거야.
   "I think he will win."                그가 이길 것 같아.

3. Promises (약속):
   "I'll call you tomorrow."             내일 전화할게.
   "I won't tell anyone."                아무에게도 안 말할게.
   "I'll always love you."               항상 사랑할게.

4. Offers and requests (제안과 요청):
   "I'll help you with that."            내가 도와줄게.
   "Will you help me?"                   도와줄래?
   "Shall I open the door?"              문 열어줄까?

5. Threats and warnings:
   "I'll tell your mother!"              엄마한테 말할 거야!
   "Be careful or you'll fall."          조심해 안 그러면 넘어질 거야.
```

## 6. Future with going to

### 6.1 Formation

```
Positive: subject + am/is/are + going to + base verb
  I am going to eat.       (I'm going to eat.)
  She is going to study.   (She's going to study.)
  They are going to leave. (They're going to leave.)

Negative: subject + am/is/are + not + going to + base verb
  I'm not going to eat.    She isn't going to study.

Question: Am/Is/Are + subject + going to + base verb?
  Are you going to eat?    Is she going to study?

  ※ In spoken English: "going to" → "gonna" /ˈɡʌnə/
    "I'm gonna eat." (very informal, never in writing)
```

### 6.2 When to Use going to

```
1. Pre-planned intentions (이미 결정한 계획):
   "I'm going to study medicine."        나는 의학을 공부할 거야. (이미 결정)
   "We're going to visit Korea next year." 내년에 한국 방문할 거야. (계획 있음)
   "She's going to quit her job."         그녀는 직장을 그만둘 거야. (결심 완료)

2. Predictions with evidence (증거 기반 예측):
   "Look at those clouds. It's going to rain." 저 구름 봐. 비 올 거야.
   "She's studied hard. She's going to pass."  열심히 했으니 합격할 거야.
   "Be careful! You're going to fall!"         조심해! 넘어지겠어!

   ※ "going to" predictions = you can SEE evidence
     "will" predictions = you THINK/BELIEVE it will happen
```

## 7. Future with Present Continuous

### 7.1 Present Continuous for Future Arrangements

```
Present continuous can express FIXED future arrangements:

  "I'm meeting Tom tomorrow."         내일 톰을 만나기로 했어.
  "She's flying to London next week." 다음 주에 런던에 가기로 했어.
  "We're having dinner at 7."         7시에 저녁 먹기로 했어.
  "They're getting married in June."  6월에 결혼해.

  ※ These are ARRANGEMENTS — already organized with other people
    (time, place, and people are decided)
    Similar to Korean ~기로 했다 / ~하기로 되어 있다
```

### 7.2 Present Simple for Future Timetables

```
For SCHEDULED events (timetables, programs):

  "The train leaves at 9 AM."        기차는 9시에 출발한다.
  "The movie starts at 7:30."        영화는 7시 30분에 시작한다.
  "School begins in September."      학교는 9월에 시작한다.
  "The store closes at 10 PM."       가게는 밤 10시에 닫는다.

  ※ Use simple present for fixed schedules/timetables
    (things that won't change — official schedules)
```

## 8. Comparing the Three Future Forms

### 8.1 Decision Chart

```
┌─────────────────────────────────────────────────────────────────┐
│              FUTURE FORM DECISION CHART                         │
│                                                                 │
│   Is it a scheduled timetable?                                  │
│   YES → Simple Present: "The train leaves at 9."               │
│                                                                 │
│   Is it already arranged (with people, places set)?             │
│   YES → Present Continuous: "I'm meeting Tom at 3."            │
│                                                                 │
│   Is it a pre-planned intention?                                │
│   YES → going to: "I'm going to study abroad."                 │
│                                                                 │
│   Is it a spontaneous decision (decided NOW)?                   │
│   YES → will: "I'll have the pasta." (at a restaurant)         │
│                                                                 │
│   Is it a prediction based on evidence?                         │
│   YES → going to: "Look! It's going to rain."                  │
│                                                                 │
│   Is it a prediction based on opinion/belief?                   │
│   YES → will: "I think she'll win."                             │
│                                                                 │
│   Is it a promise, offer, or threat?                            │
│   YES → will: "I'll help you." / "I won't forget."             │
└─────────────────────────────────────────────────────────────────┘
```

### 8.2 Same Situation, Different Nuances

```
"I'll visit Korea."
→ Spontaneous decision (just decided now)
  "That sounds great! I'll visit Korea too!"

"I'm going to visit Korea."
→ Pre-planned intention (decided before this moment)
  "I've been planning it for months. I'm going to visit Korea."

"I'm visiting Korea next month."
→ Fixed arrangement (flights booked, plans confirmed)
  "I already bought the tickets. I'm visiting Korea next month."
```

### 8.3 Korean Comparison

```
┌─────────────────────────────────────────────────────────────────┐
│   Korean future:                                                │
│   ~할 것이다    (formal intention/prediction)                     │
│   ~할 거야     (informal intention/prediction)                   │
│   ~하겠다      (intention/willingness)                           │
│   ~할 예정이다  (scheduled/planned)                               │
│   ~하기로 했다  (arranged/decided)                                │
│                                                                 │
│   English future:                                               │
│   will = spontaneous decision, prediction, promise              │
│   going to = pre-planned intention, evidence-based prediction  │
│   present continuous = fixed arrangement                        │
│   simple present = timetable/schedule                           │
│                                                                 │
│   ※ Korean speakers tend to overuse "will" for everything.    │
│     Learn to use "going to" and present continuous too!        │
└─────────────────────────────────────────────────────────────────┘
```

## 9. Future Time Expressions

### 9.1 Common Expressions

```
tomorrow                  내일
tonight                   오늘 밤
next week/month/year      다음 주/달/해
this weekend              이번 주말
in two days / in a week   이틀 후 / 일주일 후
soon                      곧
later                     나중에
someday / one day          언젠가
the day after tomorrow    모레

※ No preposition with next/this:
  "next Monday" ✓  "on next Monday" ✗
  "this weekend" ✓  "in this weekend" ✗
```

## Quick Reference

```
┌─────────────────────────────────────────────────────────────────┐
│         Present Continuous & Future Quick Guide                  │
│                                                                 │
│   Present Continuous: am/is/are + verb-ing                      │
│   Uses: happening now, temporary, changing, annoying (+always)  │
│   ※ NOT with state verbs: know, like, want, have (possess)    │
│                                                                 │
│   Future forms:                                                 │
│   will + base verb:                                             │
│     → spontaneous decision, prediction (opinion), promise       │
│   be + going to + base verb:                                    │
│     → pre-planned intention, prediction (evidence)              │
│   present continuous (be + -ing):                               │
│     → fixed arrangement (people, time, place decided)           │
│   simple present:                                               │
│     → timetable/schedule (train leaves at 9)                    │
│                                                                 │
│   ※ Korean speakers overuse "will" — practice "going to"      │
│     and present continuous for planned/arranged futures!        │
└─────────────────────────────────────────────────────────────────┘
```
