## 1. 가족

### 1.1 직계 가족

```
┌─────────────────────────────────────────────────────────────────┐
│   영어               한국어        비고                          │
│   ─────────         ──────        ─────                         │
│   father / dad      아버지 / 아빠  "dad" = 비격식                │
│   mother / mom      어머니 / 엄마  "mom" (미국식), "mum" (영국식) │
│   parents           부모님                                       │
│   son               아들                                        │
│   daughter           딸                                         │
│   brother            형/오빠/남동생  ※ 한국어에는 4개 단어!       │
│   sister             언니/누나/여동생 ※ 한국어에는 4개 단어!       │
│   husband            남편                                       │
│   wife               아내                                       │
│   children           자녀/아이들                                  │
│                                                                 │
│   ※ 한국어는 나이 많고 적음과 화자의 성별을 구분합니다:           │
│     형 (손윗 남자 형제, 남자 화자)                                │
│     오빠 (손윗 남자 형제, 여자 화자)                               │
│     누나 (손윗 여자 형제, 남자 화자)                               │
│     언니 (손윗 여자 형제, 여자 화자)                               │
│     남동생 (손아랫 남자 형제)                                     │
│     여동생 (손아랫 여자 형제)                                     │
│                                                                 │
│   영어에는 brother, sister만 있음 (+ older/younger/big/little)   │
│     "my older brother" 또는 "my big brother"                    │
│     "my younger sister" 또는 "my little sister"                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 친척

```
grandfather / grandpa    할아버지         grandmother / grandma  할머니
grandparents             조부모님         grandchildren          손주들
grandson                 손자             granddaughter          손녀
uncle                    삼촌/외삼촌       aunt                   이모/고모
nephew                   조카 (남)         niece                  조카 (여)
cousin                   사촌
father-in-law            장인/시아버지     mother-in-law          장모/시어머니
brother-in-law           처남/매형/시동생   sister-in-law         시누이/올케/형수
son-in-law               사위             daughter-in-law        며느리
stepfather               새아버지         stepmother             새어머니
half-brother             이복형제         half-sister            이복자매

※ 한국어는 영어보다 더 구체적인 가족 호칭이 있습니다:
  한국어 삼촌 = 아버지의 형제, 외삼촌 = 어머니의 형제
  영어는 둘 다 "uncle"!
  한국어 이모 = 어머니의 자매, 고모 = 아버지의 자매
  영어는 둘 다 "aunt"!
```

### 1.3 가족 관련 표현

```
"How many siblings do you have?"     형제자매가 몇 명이에요?
"I'm an only child."                 저는 외동이에요.
"She's married."                     그녀는 결혼했어요.
"He's single."                       그는 미혼이에요.
"They got divorced."                 그들은 이혼했어요.
"She's pregnant."                    그녀는 임신 중이에요.
"I'm the eldest/oldest."            저는 맏이예요.
"I'm the youngest."                 저는 막내예요.
```

## 2. 음식과 요리

### 2.1 기본 식품 분류

```
고기 (Meat):           chicken, beef, pork, lamb, fish, seafood
채소 (Vegetables):      carrot, onion, potato, tomato, lettuce, broccoli,
                       pepper, garlic, cucumber, spinach, mushroom
과일 (Fruits):          apple, banana, orange, grape, strawberry, watermelon,
                       peach, mango, pineapple, lemon, cherry, pear
곡물 (Grains):          rice, bread, pasta, noodles, cereal, oatmeal
유제품 (Dairy):         milk, cheese, butter, yogurt, cream, eggs
음료 (Drinks):          water, coffee, tea, juice, soda, beer, wine
```

### 2.2 요리 동사

```
boil     끓이다     "Boil the water."          물을 끓여라.
fry      튀기다     "Fry the egg."             달걀을 부쳐라.
bake     굽다(오븐)  "Bake a cake."            케이크를 구워라.
grill    굽다(불)   "Grill the meat."          고기를 구워라.
steam    찌다       "Steam the vegetables."    야채를 쪄라.
roast    구워지다    "Roast the chicken."       닭을 구워라.
stir     젓다       "Stir the soup."           수프를 저어라.
chop     자르다     "Chop the onions."         양파를 썰어라.
slice    얇게 썰다  "Slice the bread."         빵을 썰어라.
peel     껍질을 벗기다 "Peel the potato."       감자 껍질을 벗겨라.
mix      섞다       "Mix the ingredients."     재료를 섞어라.
pour     붓다       "Pour the water."          물을 부어라.
```

### 2.3 맛과 식감

```
맛 (Taste):
  sweet (달다)      sour (시다)       salty (짜다)
  bitter (쓰다)     spicy/hot (맵다)   bland (싱겁다)
  savory (고소하다)  delicious (맛있다) terrible (맛없다)
  rich (진하다)      mild (순하다)

식감 (Texture):
  crispy (바삭한)    soft (부드러운)    chewy (쫄깃한)
  crunchy (아삭한)   tender (연한)      tough (질긴)
  juicy (즙이 많은)  creamy (크리미한)   sticky (끈적한)
```

### 2.4 식당 영어

```
"A table for two, please."          2인 테이블 부탁합니다.
"Can I see the menu?"               메뉴판 볼 수 있을까요?
"I'd like the steak, please."       스테이크를 주세요.
"How would you like your steak?"    스테이크 어떻게 구워 드릴까요?
  → rare / medium-rare / medium / well-done
"Could I have the check/bill?"      계산서 주세요.
"Is the tip included?"              팁이 포함되어 있나요?

※ 한국어 서비스 (service) = 영어 "free/complimentary"
  한국어 셀프 (self) = 영어 "self-service"
```

## 3. 의류

### 3.1 일반적인 의류 품목

```
상의:     shirt, T-shirt, blouse, sweater, hoodie, jacket, coat
하의:     pants (미국식) / trousers (영국식), jeans, shorts, skirt, dress
신발:     shoes, sneakers, boots, sandals, heels, slippers
액세서리: hat, cap, scarf, gloves, belt, tie, glasses, watch
속옷:     underwear, socks, bra

※ "pants" = 바지 (미국식) 하지만 속옷 (영국식)!
  미국식 "pants" = 영국식 "trousers"
  영국식 "pants" = 미국식 "underwear"
```

### 3.2 의류 동사와 표현

```
put on (입다/착용하다):     "Put on your coat."       코트를 입어.
take off (벗다):            "Take off your shoes."    신발을 벗어.
wear (입고 있다 — 상태):    "She's wearing a dress."  그녀는 원피스를 입고 있다.
try on (입어보다):          "Can I try this on?"      이거 입어봐도 돼요?
fit (맞다):                 "Does it fit?"            사이즈가 맞나요?
match (어울리다):           "This shirt matches."     이 셔츠가 어울린다.

※ "wear" vs "put on":
  "I'm wearing a hat." = 지금 쓰고 있다 (상태)
  "I'm putting on a hat." = 지금 쓰는 중이다 (동작)
```

### 3.3 사이즈와 쇼핑

```
사이즈: XS (extra small), S (small), M (medium), L (large), XL (extra large)

"What size are you?"                 사이즈가 어떻게 되세요?
"Do you have this in a medium?"      이거 미디엄 사이즈 있어요?
"It's too big/small/tight/loose."    너무 커요/작아요/꽉 끼어요/헐렁해요.
"How much is this?"                  이거 얼마예요?
"I'll take it."                      이걸로 할게요.
"Can I return this?"                 반품할 수 있나요?
```

### 6.2 교통 동사

```
┌─────────────────────────────────────────────────────────────────┐
│   drive a car        차를 운전하다    (자가용)                    │
│   ride a bicycle     자전거를 타다    (올라타는 것)                │
│   take the bus       버스를 타다      (대중교통)                  │
│   get on / get off   타다 / 내리다    (대중교통)                  │
│   get in / get out   타다 / 내리다    (택시, 승용차)              │
│   catch a train      기차를 타다      (시간에 맞춰)               │

│                                                                 │
│   ※ get ON/OFF = 큰 교통수단 (bus, train, plane, ship)          │
│     get IN/OUT OF = 작은 교통수단 (car, taxi)                    │
│     "Get on the bus." ✓    "Get in the bus." ✗                  │
│     "Get in the car." ✓    "Get on the car." ✗                  │
└─────────────────────────────────────────────────────────────────┘
```

## 7. 날씨 표현

### 7.1 날씨 어휘

```
sunny    화창한       cloudy    흐린        rainy     비 오는
windy    바람 부는    snowy     눈 오는     foggy     안개 낀
stormy   폭풍우의    humid     습한        dry       건조한
hot      더운        warm      따뜻한      cool      시원한
cold     추운        freezing  몹시 추운

"It's sunny today."          오늘 화창하다.
"It's raining."              비가 온다.
"It's snowing."              눈이 온다.
"It's windy."                바람이 분다.
```

### 7.2 기온

```
"It's hot."          덥다.
"It's warm."         따뜻하다.
"It's cool."         시원하다 / 선선하다.
"It's cold."         춥다.
"It's freezing."     몹시 춥다 / 꽁꽁 얼 정도로 춥다.
"It's 30 degrees."   30도이다.
"It's below zero."   영하이다.

※ 영어는 날씨에 가주어 "it"을 사용합니다:
  "It is raining." (비가 온다 — "it"은 실제 의미 없음)
  한국어: 비가 온다 (비 = rain, 주어)
  영어: "Rains." ✗ → "It rains." ✓ ("it"이 반드시 필요)
```

### 7.3 날씨 관련 표현

```
"What's the weather like?"     날씨가 어때?
"What's the forecast?"         일기 예보가 뭐야?
"It looks like rain."          비가 올 것 같다.
"The weather is getting better." 날씨가 좋아지고 있다.
"It's pouring."                비가 쏟아진다 / 폭우가 온다.
"It's drizzling."              이슬비가 온다.
"There's a thunderstorm."      천둥번개가 친다.
"It cleared up."               날이 개었다.
```

│   (결과물이 있는 것: 만들다)                                      │
│   make breakfast (아침을 만들다)                                  │
│   make a decision (결정을 만들다)                                 │
│   make money (돈을 만들다/벌다)                                   │
│                                                                 │
│   DO = 활동/작업을 수행                                           │
│   (활동을 수행하는 것: 하다)                                      │
│   do homework (숙제를 하다)                                      │
│   do the dishes (설거지를 하다)                                   │
│   do exercise (운동을 하다)                                      │
│                                                                 │
│   ※ "make homework" ✗ → "do homework" ✓                       │
│   ※ "do a mistake" ✗ → "make a mistake" ✓                     │
│   ※ "make the dishes" ✗ → "do the dishes" ✓                   │
│   이것들은 외워야 합니다 — 완벽한 규칙은 없습니다!                │
└─────────────────────────────────────────────────────────────────┘
```

## 9. 일과 어휘

### 9.1 아침 일과

```
wake up          일어나다       get up            (침대에서) 일어나다
take a shower    샤워하다       brush my teeth    양치하다
get dressed      옷을 입다      have breakfast    아침을 먹다
leave home       집을 나서다    go to work/school 출근/등교하다
```

### 9.2 낮 활동

```
arrive at work   직장에 도착하다   check email       이메일을 확인하다
have a meeting   회의를 하다      eat lunch         점심을 먹다
take a break     휴식을 취하다    finish work       퇴근하다
go home          집에 가다
```

### 9.3 저녁 일과

```
come home        집에 돌아오다    cook dinner       저녁을 요리하다
eat dinner       저녁을 먹다     watch TV          TV를 보다
do homework      숙제를 하다     take a bath       목욕하다
read a book      책을 읽다       go to bed         잠자리에 들다
fall asleep      잠이 들다       set the alarm     알람을 맞추다
```

### 연습 3: 날씨

5. It's cloudy.
```

</details>
