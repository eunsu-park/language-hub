## 1. 구동사란?

### 1.1 정의

구동사는 **동사 + 불변화사**(부사 또는 전치사)의 결합으로, 개별 단어와는 다른 새로운 의미를 만들어 냅니다.

```
┌─────────────────────────────────────────────────────────────────┐
│   동사 단독:  "look" = 보다                                      │
│   구동사:     "look up" = (사전에서) 찾아보다                       │
│               "look after" = 돌보다                               │
│               "look forward to" = 기대하다                        │
│               "look into" = 조사하다                               │
│               "look out" = 조심하다                                │
│                                                                 │
│   불변화사가 의미를 완전히 바꿉니다!                                 │
│   개별 단어로는 의미를 추측할 수 없습니다.                           │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 구동사의 유형

```
1. 동사 + 부사:          "give up" (포기하다)
2. 동사 + 전치사:        "look after" (돌보다)
3. 동사 + 부사 + 전치사: "look forward to" (기대하다)
```

### 1.3 왜 한국어 화자에게 어려운가?

```
한국어는 영어에서 구동사를 쓰는 곳에 단일 동사를 사용합니다:
  포기하다 → give up          찾다 → find out / look up
  연기하다 → put off          참다 → put up with
  돌보다  → look after       제출하다 → hand in

한국어 화자는 격식적인 단일 단어 동의어를 사용하는 경향이 있습니다:
  "postpone" 대신 "put off"
  "tolerate" 대신 "put up with"
  "investigate" 대신 "look into"

격식적 동의어도 맞지만, 구동사가 일상 구어체에서
더 자연스럽게 들립니다.
```

## 2. 분리 가능 vs 분리 불가

### 2.1 분리 가능 구동사

목적어가 동사와 불변화사 **사이**에 오거나 불변화사 **뒤**에 올 수 있습니다.

```
┌─────────────────────────────────────────────────────────────────┐
│   분리 가능: 목적어가 중간이나 끝에 올 수 있음                      │
│                                                                 │
│   "Turn off the light."  ✓  (목적어가 불변화사 뒤)                │
│   "Turn the light off."  ✓  (목적어가 중간)                      │
│                                                                 │
│   단, 목적어가 대명사이면 반드시 중간에 와야 합니다!                  │
│   "Turn it off."         ✓                                      │
│   "Turn off it."         ✗                                      │
│                                                                 │
│   추가 예:                                                       │
│   "Pick up the kids." / "Pick the kids up." / "Pick them up."   │
│   "Put on your coat." / "Put your coat on." / "Put it on."      │
│   "Turn down the music." / "Turn the music down." / "Turn it down."│
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 분리 불가 구동사

목적어는 반드시 불변화사 **뒤**에 와야 합니다. 분리할 수 없습니다.

```
┌─────────────────────────────────────────────────────────────────┐
│   분리 불가: 목적어는 항상 불변화사 뒤에                             │
│                                                                 │
│   "Look after the children."    ✓                               │
│   "Look the children after."    ✗                               │
│   "Look after them."            ✓                               │
│                                                                 │
│   추가 예:                                                       │
│   "I ran into an old friend."    (우연히 만나다)                   │
│   "She takes after her mother."  (닮다)                          │
│   "He got over the illness."     (~에서 회복하다)                  │
│   "We came across a problem."    (우연히 발견하다)                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 3단어 구동사 (항상 분리 불가)

```
"I look forward to the meeting."     ✓
"I look the meeting forward to."     ✗

"She puts up with his behavior."     ✓
"We ran out of milk."                ✓
"He came up with a great idea."      ✓
"I need to catch up with my work."   ✓
```

## 11. 연습 문제

### 연습 1: 분리 가능인가, 분리 불가인가?

구동사가 분리 가능(S)인지 분리 불가(I)인지 판별하고, 대명사를 사용하여 다시 쓰세요.

1. Turn off the computer. → Turn ___ off.
2. Look after the baby. → Look after ___.
3. Pick up the package. → Pick ___ up.
4. Run into a friend. → Run into ___.
5. Put on your jacket. → Put ___ on.

<details>
<summary>정답 보기</summary>

```
1. 분리 가능 (S) → Turn it off.
2. 분리 불가 (I) → Look after him/her.
3. 분리 가능 (S) → Pick it up.
4. 분리 불가 (I) → Run into him/her.
5. 분리 가능 (S) → Put it on.
```

</details>

### 연습 2: 불변화사 채우기

올바른 불변화사를 선택하세요: up, down, out, off, on.

1. I need to figure ___ this math problem.
2. The meeting was called ___ due to bad weather.
3. She grew ___ in a small town.
4. Can you turn ___ the volume? It's too loud.
5. We ran ___ of time during the exam.
6. He turned ___ the job offer because the salary was too low.
7. Keep ___ trying! You'll succeed eventually.
8. The fire broke ___ at midnight.
9. I need to catch ___ with my reading.
10. Please take ___ your shoes before entering.

<details>
<summary>정답 보기</summary>

```
1. out (figure out = 이해하다/해결하다)
2. off (call off = 취소하다)
3. up (grow up = 성장하다)
4. down (turn down = 음량을 줄이다)
5. out (run out of = 다 떨어지다)
6. down (turn down = 거절하다)
7. on (keep on = 계속하다)
8. out (break out = 갑자기 발생하다)
9. up (catch up = 따라잡다)
10. off (take off = 벗다)
```

</details>

### 연습 3: 구동사로 바꾸기

밑줄 친 단어를 구동사로 바꾸세요.

1. I need to **discover** who took my pen. →
2. She **resembles** her grandmother. →
3. They had to **postpone** the wedding. →
4. Please **complete** this form. →
5. I can't **tolerate** this noise anymore. →
6. She **rejected** his proposal. →
7. We need to **investigate** this matter. →
8. He **invented** a crazy excuse. →

<details>
<summary>정답 보기</summary>

```
1. find out (발견하다)
2. takes after (닮다)
3. put off (연기하다)
4. fill out / fill in (양식을 작성하다)
5. put up with (참다)
6. turned down (거절하다)
7. look into (조사하다)
8. made up (지어내다/꾸며내다)
```

</details>

### 연습 4: 오류 수정하기

1. I need to look up it in the dictionary.
2. She takes after to her mother.
3. Turn off it before you leave.
4. He gave up to smoke last year.
5. I ran into with my old classmate.

<details>
<summary>정답 보기</summary>

```
1. I need to look it up in the dictionary.
   (분리 가능 구동사에서 대명사는 동사와 불변화사 사이에 위치)
2. She takes after her mother.
   ("take after"에는 "to"를 사용하지 않음)
3. Turn it off before you leave.
   (대명사는 동사와 불변화사 사이에 위치)
4. He gave up smoking last year.
   ("give up" + 동명사, "to + 부정사" 아님)
5. I ran into my old classmate.
   ("run into"에는 "with"를 사용하지 않음)
```

</details>

### 연습 5: 관용구 연결하기

관용구를 의미와 연결하세요.

| 관용구 | 의미 |
|--------|------|
| 1. Break the ice | a. 아주 쉬운 |
| 2. A piece of cake | b. 비밀을 누설하다 |
| 3. Hit the nail on the head | c. 분위기를 편하게 하다 |
| 4. Let the cat out of the bag | d. 몸이 좀 안 좋다 |
| 5. Under the weather | e. 정확히 맞는 말을 하다 |

<details>
<summary>정답 보기</summary>

```
1 — c (Break the ice = 분위기를 편하게 하다)
2 — a (A piece of cake = 아주 쉬운)
3 — e (Hit the nail on the head = 정확히 맞는 말을 하다)
4 — b (Let the cat out of the bag = 비밀을 누설하다)
5 — d (Under the weather = 몸이 좀 안 좋다)
```

</details>
