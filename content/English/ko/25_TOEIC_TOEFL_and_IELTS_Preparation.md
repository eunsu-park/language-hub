## 2. TOEIC: 형식과 구조

### 2.1 듣기 섹션 (45분, 100문항)

```
Part 1: 사진 묘사 (6문항)
  → 사진에 대한 4개의 진술을 듣고 가장 적절한 것 선택
  → 팁: 음성 재생 전 사진을 살펴보세요;
    사람, 행동, 물체, 장소에 집중

Part 2: 질의응답 (25문항)
  → 질문 + 3개의 가능한 응답을 듣기
  → 팁: 질문의 첫 번째 단어에 집중
    (Who/What/Where/When/Why/How — 답변 유형 결정)
  → 비슷하게 들리는 오답에 주의

Part 3: 대화 (39문항 = 13개 대화 x 3문항)
  → 2-3명 간의 대화 듣기
  → 대화당 3문항 답변
  → 팁: 음성 재생 전에 문제를 먼저 읽으세요;
    무엇을 들어야 하는지 예측

Part 4: 설명문 (30문항 = 10개 담화 x 3문항)
  → 짧은 독백 듣기 (안내 방송, 음성 메시지 등)
  → 담화당 3문항 답변
  → 팁: Part 3과 동일 — 문제를 미리 읽기
```

### 2.2 읽기 섹션 (75분, 100문항)

```
Part 5: 불완전한 문장 (30문항)
  → 문장을 완성할 올바른 단어 선택
  → 테스트: 문법, 어휘, 어형
  → 팁: 무엇이 테스트되는지 파악 (문법 vs 어휘);
    빈칸 앞뒤의 단어를 살펴보세요

Part 6: 지문 완성 (16문항 = 4개 지문 x 4문항)
  → 지문을 완성할 단어나 문장 선택
  → 테스트: 문법, 어휘, 문맥
  → 팁: 빈칸이 있는 문장만이 아니라
    전체 지문을 읽고 문맥 파악

Part 7: 독해 (54문항)
  → 단일 지문 (29문항): 이메일, 광고, 기사
  → 복수 지문 (25문항): 2-3개의 관련 텍스트
  → 팁: 문제를 먼저 읽고 답을 찾아 스캔;
    시간 관리를 신중히 (이 섹션이 가장 김)
```

## 3. TOEIC: 듣기 전략

### 3.1 Part 1 (사진)

```
전략:
  1. 음성 시작 전 5초 동안 사진을 살펴보기
  2. 파악하기: 누가 (사람), 무엇 (행동), 어디서 (장소)
  3. 사진을 정확히 묘사하는 진술을 듣기
  4. 주의할 점:
     - 비슷하게 들리지만 의미가 다른 단어
     - 사진의 일부만 묘사하는 진술
     - 사진에 나타나지 않는 것을 가정하는 진술

함정 예시:
  사진: 한 여성이 노트북으로 타이핑하는 모습
  ✗ "The woman is talking on the phone." (잘못된 행동)
  ✗ "The woman is working in a garden." (잘못된 장소)
  ✗ "The laptop is being repaired." (나타나지 않음)
  ✓ "The woman is using a computer."
```

### 3.2 Part 2 (질의응답)

```
핵심 전략: 첫 번째 단어에 집중

  Who  → 답변에 사람/이름 예상
  What → 사물/행동 예상
  Where → 장소 예상
  When → 시간 예상
  Why  → 이유 예상 (Because...)
  How  → 방법이나 정도 예상

  함정:
  - 비슷한 소리의 답변 (예: "meet" vs "meat")
  - 질문의 단어를 반복하는 답변 (대개 오답!)
  - Yes/No 질문: 답변이 yes/no로 시작하지 않을 수 있음
    Q: "Didn't you attend the meeting?"
    A: "I had another appointment." (yes/no 없이도 유효한 답변)
```

### 3.3 Parts 3-4 (대화와 담화)

```
1. 미리 읽기: 음성 재생 전 3개 문제를 읽기
2. 예측하기: 대화/담화의 주제 추측
3. 능동적 듣기: 관련 정보를 들으며 각 문제에 답변
4. 기다리지 말기: 답을 들으면 즉시 표시;
   음성 끝까지 기다리지 말기
5. 넘어가기: 답을 놓치면 붙잡지 말고;
   다음 문제에 집중

일반적인 문제 유형:
  - 대화의 주제/목적은?
  - 대화는 어디서 이루어지는가?
  - 화자가 제안/추천하는 것은?
  - 청자는 다음에 무엇을 할 가능성이 높은가?
  - 도표를 보세요 — 어떤 [선택지]가 맞는가?
```

## 4. TOEIC: 읽기 전략

### 4.1 Part 5 (불완전한 문장) — 목표: 10분

```
문법 문제 — 단서를 찾으세요:
  1. 어형: 어떤 품사가 필요한가?
     "The presentation was very ___."
     → 형용사 필요 (impressive, impress/impression 아님)

  2. 동사 시제: 시간 표현 확인
     "She ___ the report yesterday."
     → 과거 시제 필요 (submitted)

  3. 전치사: 일반적인 연어 학습
     "interested ___" → in
     "depend ___" → on
     "responsible ___" → for

  4. 접속사: 논리적 관계 확인
     "___ the rain, they continued." → Despite/In spite of

어휘 문제 — 문맥 활용:
  "The company will ___ the event next month."
  → host/hold (저녁 만찬이 아님)
```

### 4.2 Part 6 (지문 완성) — 목표: 8분

```
1. 문맥 파악을 위해 전체 지문을 먼저 읽기
2. 어휘 빈칸: 주변 문맥 활용
3. 문장 삽입: 논리적 흐름 확인
   - 문장이 앞뒤 내용과 연결되는가?
   - 어조와 주제가 일치하는가?
4. 문법 빈칸: Part 5와 동일한 전략
```

### 4.3 Part 7 (독해) — 목표: 57분

```
시간 관리가 핵심!

단일 지문: 문제당 ~1분
복수 지문: 문제당 ~1.5분

전략:
1. 문제를 먼저 읽기 (무엇을 찾아야 하는지 파악)
2. 지문을 훑어서 주요 아이디어 파악
3. 구체적 답변을 찾아 스캔
4. "NOT" 문제: 언급된 3개를 찾기
5. 추론 문제: 텍스트의 단서 찾기
6. 문맥 속 어휘: 해당 문장을 주의깊게 다시 읽기

일반적인 텍스트 유형:
  이메일, 편지, 광고, 공지, 기사, 양식,
  일정표, 리뷰, 문자 메시지 대화
```

## 5. TOEFL iBT: 형식과 구조

### 5.1 네 개 섹션

> 읽기 (READING) (35분)
> • 2개 지문 (각 ~700단어)
> • 지문당 10문항 = 총 20문항
> • 학술 주제: 과학, 역사, 사회과학, 예술
> ├────────────────────────────────────────────────────────────────┤
> 듣기 (LISTENING) (36분)
> • 3개 강의 (각 6문항) = 18문항
> • 2개 대화 (각 5문항) = 10문항
> • 총: 28문항
> • 메모 허용
> ├────────────────────────────────────────────────────────────────┤
> 말하기 (SPEAKING) (16분)
> • 4개 과제
> • 과제 1: 독립형 (의견 제시) — 45초
> • 과제 2-4: 통합형 (읽기/듣기 + 말하기) — 각 60초
> • 채점 0-30
> ├────────────────────────────────────────────────────────────────┤
> 쓰기 (WRITING) (29분)
> • 과제 1: 통합형 (읽기 + 듣기 + 쓰기) — 20분
> • 과제 2: 학술 토론 — 10분
> • 채점 0-30

```

총점: 0-120 (섹션당 30점)
총 시간: ~2시간
```

## 6. TOEFL: 섹션별 전략

### 6.1 읽기

```
문제 유형:
  1. 사실 정보 (지문에서 X에 대해 뭐라고 하는가?)
  2. 부정 사실 (언급되지 않은 것은?)
  3. 추론 (무엇을 추론할 수 있는가?)
  4. 문맥 속 어휘 (X라는 단어의 의미는...)
  5. 지시어 ("they"가 가리키는 것은?)
  6. 문장 삽입 (이 문장은 어디에 들어가야 하는가?)
  7. 요약 (지문을 요약하는 3개 문장 선택)

전략:
  • 지문을 먼저 훑어보기 (2분)
  • 각 문단의 주제문 읽기
  • 어휘 문제는 문맥 단서 활용
  • 요약 문제는 사소한 세부사항 제거
  • 문제당 1.5분 이상 소비하지 않기
```

### 6.2 듣기

```
전략:
  • 메모하기 — 이것이 필수!
  • 메모할 것: 주제, 핵심 세부사항, 예시, 화자의 태도
  • 신호 단어 경청:
    "The important thing is..." (핵심 포인트 등장)
    "For example..." (뒷받침 세부사항)
    "However..." (대조/방향 전환)
    "In other words..." (핵심 아이디어 재진술)
  • 모든 것을 쓰려 하지 말기 — 핵심 단어만 포착
  • 화자의 어조와 강조에 주의
```

### 6.3 말하기

```
과제 1 (독립형 — 준비 15초, 말하기 45초):
  템플릿:
    "I believe that [의견]. First, [이유 1 + 예시].
     Second, [이유 2 + 예시]. That's why I think [재진술]."

과제 2-4 (통합형 — 준비 30초, 말하기 60초):
  읽기 + 듣기 템플릿:
    "The reading passage discusses [주제]. The lecture
     [agrees/disagrees] with this by explaining [요점].
     Specifically, the professor mentions [세부사항 1] and
     [세부사항 2]."

일반 팁:
  • 명확하고 자연스러운 속도로 말하기
  • 억양 걱정하지 말기 — 명확성이 더 중요
  • 전환어 사용 (first, second, however, therefore)
  • 타이머로 연습 — 45초와 60초는 빠르게 지나감!
  • 생각을 위해 잠시 멈추는 것은 괜찮음 (1-2초)
```

### 6.4 쓰기

```
과제 1 (통합형 — 20분):
  • 지문 읽기 (3분), 강의 듣기
  • 강의가 읽기 지문과 어떻게 관련되는지 쓰기
  • 250-300단어 목표
  • 템플릿:
    "The reading argues that [X]. However, the lecturer
     challenges this by arguing that [Y]. First, while the
     reading states [요점 1], the lecturer counters that
     [반박 1]. Second,..."

과제 2 (학술 토론 — 10분):
  • 교수의 질문과 두 학생의 응답 읽기
  • 토론에 자신의 기여를 작성
  • 100단어 이상 목표
  • 의견을 표현하고 뒷받침하며, 최소 한 학생의
    요점에 대해 언급
```

## 7. IELTS: 형식과 구조

### 7.1 두 가지 유형

```
IELTS Academic:  대학 입학용
IELTS General:   이민 및 취업용

둘 다 동일한 듣기와 말하기 섹션을 가짐.
읽기와 쓰기 섹션이 다름.
```

### 7.2 네 개 섹션

> 듣기 (LISTENING) (30분 + 10분 답안 옮기기 시간)
> • 4개 섹션, 총 40문항
> • 섹션 1: 사회적 대화 (2명의 화자)
> • 섹션 2: 사회적 독백 (1명의 화자)
> • 섹션 3: 학술적 대화 (2-4명의 화자)
> • 섹션 4: 학술 강의 (1명의 화자)
> • 각 녹음은 한 번만 재생
> ├────────────────────────────────────────────────────────────────┤
> 읽기 (READING) (60분)
> • 3개 지문, 총 40문항
> • Academic: 학술지, 잡지, 서적
> • General: 일상 텍스트, 업무 관련, 일반 관심사
> ├────────────────────────────────────────────────────────────────┤
> 쓰기 (WRITING) (60분)
> • 과제 1 (20분): Academic = 차트/그래프/도표 설명
> General = 편지 쓰기
> • 과제 2 (40분): 둘 다 = 에세이 (250단어 이상)
> ├────────────────────────────────────────────────────────────────┤
> 말하기 (SPEAKING) (11-14분, 시험관과 대면)
> • Part 1: 소개 + 일반 질문 (4-5분)
> • Part 2: 긴 발언 — 주어진 주제에 대해 2분 말하기 (3-4분)
> • Part 3: 토론 — 추상적/복잡한 질문 (4-5분)

```

밴드 점수: 0-9 (각 섹션 + 종합 밴드)
```

## 8. IELTS: 섹션별 전략

### 8.1 듣기

```
전략:
  • 각 섹션 재생 전 문제를 미리 읽기
  • 답변 유형 예측 (숫자? 이름? 형용사?)
  • 동의어 경청 — 답변이 문제의 정확한 단어를
    거의 사용하지 않음
  • 정보를 들으면서 답변 작성
  • 단어 수 제한 확인 ("세 단어 이하")
  • 철자가 중요! 자주 틀리는 단어 연습

문제 유형:
  객관식, 매칭, 지도/도면 라벨링,
  양식 완성, 노트 완성, 문장 완성,
  요약 완성, 표 완성
```

### 8.2 읽기

```
전략:
  • 지문을 단어 하나하나 읽지 마세요
  • 제목, 첫 문장, 굵은 글씨를 훑어보기
  • 문제를 읽고 답을 찾아 스캔
  • True/False/Not Given:
    TRUE = 정보가 지문과 일치
    FALSE = 정보가 지문과 모순
    NOT GIVEN = 정보가 지문에 없음
  • 제목 매칭: 각 문단의 첫 번째와
    마지막 문장 읽기
  • 시간: 지문당 ~20분

흔한 함정: "NOT GIVEN"은 "FALSE"와 같지 않음
  FALSE = 지문이 반대를 말함
  NOT GIVEN = 지문이 전혀 언급하지 않음
```

### 8.3 쓰기

```
과제 1 — Academic (20분, 150단어 이상):
  차트, 그래프, 표, 도표, 또는 과정을 설명
  • 시각 자료가 보여주는 것의 바꿔 말하기로 시작
  • 주요 추세, 비교, 주목할 만한 특징 파악
  • 데이터로 관찰을 뒷받침
  • 의견을 제시하지 말 것 (설명만 할 것)
  • 핵심 표현:
    "The graph shows that..."
    "It is noticeable that..."
    "The most significant change was..."
    "Between 2010 and 2020, the figure rose from X to Y."

과제 2 — 에세이 (40분, 250단어 이상):
  프롬프트에 응답하는 에세이 작성
  구조:
    서론: 질문 바꿔 말하기 + 입장 진술
    본론 1: 첫 번째 주요 요점 + 증거
    본론 2: 두 번째 주요 요점 + 증거
    (선택적 본론 3 또는 반론)
    결론: 입장 재진술 + 최종 생각

  밴드 7+ 팁:
  • 다양한 어휘 사용 (같은 단어 반복 피하기)
  • 복잡한 문장 구조 사용
  • 각 요점을 예시와 함께 충분히 전개
  • 주제에 맞게 — 질문에 직접 답변
  • 교정을 위해 3-5분 남기기
```

### 8.4 말하기

```
Part 1 (일반 질문 — 4-5분):
  • 2-3문장으로 답변, 단순히 "yes" 또는 "no"만 하지 않기
  • 이유와 예시로 답변 확장
  Q: "Do you enjoy reading?"
  약한 답: "Yes, I do."
  좋은 답: "Yes, I really enjoy reading, especially novels.
   I try to read for at least 30 minutes before bed.
   Recently, I've been reading mystery novels by Korean
   authors."

Part 2 (긴 발언 — 2분 동안 말하기):
  • 준비 1분 + 메모 작성
  • 메모를 가이드로 사용, 대본이 아님
  • 구조: 소개, 세부사항, 개인적 성찰
  • 계속 말하세요! 침묵이 작은 실수보다 나쁨
  • 타이머로 연습

Part 3 (토론 — 4-5분):
  • 자세하고 사려 깊은 답변 제공
  • 추상적 주제를 토론할 수 있음을 보여주기
  • 다음 표현 사용:
    "I think..." / "In my opinion..."
    "On the one hand... on the other hand..."
    "It depends on..." / "Generally speaking..."
    "That's an interesting question. I think..."

채점 기준:
  1. 유창성과 일관성 (Fluency and coherence)
  2. 어휘 범위 (Lexical resource)
  3. 문법 범위와 정확성 (Grammatical range and accuracy)
  4. 발음 (Pronunciation)
```

## 10. 학습 계획과 팁

### 10.1 일반 학습 계획

> **12주 학습 계획**
>
> 1-2주:   진단 테스트 + 약점 파악
> 3-4주:   가장 약한 섹션에 집중 (집중 연습)
> 5-6주:   어휘 쌓기 + 문법 복습
> 7-8주:   전체 모의 시험 + 시간제한 섹션 연습
> 9-10주:  시험 전략에 집중
> 11-12주: 시험 조건과 동일한 전체 모의고사
> 일일 루틴 (1.5-2시간):
> • 30분: 어휘 학습 (플래시카드, 앱)
> • 30분: 섹션 연습 (매일 교대)
> • 30분: 실수 복습 및 학습
> • 주말: 전체 모의 시험 (2주마다)

### 10.2 범용 팁

```
1. 먼저 진단 테스트를 보세요 — 현재 수준을 파악
2. 가장 약한 섹션에 집중 — 가장 많은 점수를 올릴 수 있는 곳
3. 시간 제한 조건에서 연습 — 시간 압박은 실제로 존재
4. 실수를 복습 — 왜 틀렸는지 이해
5. 매일 어휘 쌓기 — 하루 10단어라도
6. 영어에 몰입 — 영어 팟캐스트, 뉴스, 영화
7. 빈칸을 남기지 마세요 — 찍어도 감점 없음
   (TOEIC, TOEFL)
8. 시험 형식에 익숙해지기 — 시험 날 놀라지 않게
9. 전날 밤 잘 자기 — 피로는 성적을 떨어뜨림
10. 안내문을 주의깊게 읽기 — 매번
```

## 11. 연습 문제

### 연습 1: TOEIC Part 5 스타일

올바른 답을 선택하세요.

1. The annual report ___ to all shareholders next week.
   a) will send  b) will be sent  c) sends  d) is sending

2. Ms. Kim is ___ for the marketing department's budget.
   a) responsibly  b) respond  c) responsible  d) responsibility

3. ___ the heavy rain, the outdoor event proceeded as planned.
   a) Although  b) Despite  c) Because  d) However

4. The meeting has been ___ until further notice.
   a) postponing  b) postpone  c) postponed  d) postpones

5. All employees must ___ with the new safety regulations.
   a) comply  b) apply  c) supply  d) reply

<details>
<summary>정답 보기</summary>

```
1. b) will be sent (수동태 — 보고서가 스스로 보내지지 않음)
2. c) responsible (형용사 필요: "responsible for")
3. b) Despite (전치사 + 명사구 "the heavy rain")
4. c) postponed (과거분사: "has been postponed")
5. a) comply ("comply with" = 규정을 따르다/준수하다)
```

</details>

### 연습 2: TOEFL 읽기 스타일

지문을 읽고 질문에 답하세요.

> The concept of neuroplasticity has revolutionized our understanding of the brain. Previously, scientists believed that the adult brain was fixed and unchangeable. However, research over the past few decades has demonstrated that the brain continues to form new neural connections throughout life. This ability to reorganize itself is particularly evident in cases of brain injury, where undamaged areas can sometimes take over functions previously managed by damaged regions. Moreover, activities such as learning a new language, playing a musical instrument, or even regular physical exercise have been shown to promote neuroplasticity and enhance cognitive function.

1. 이 지문의 핵심 아이디어는?
   a) 뇌는 어린 시절 이후 변할 수 없다.
   b) 뇌 손상은 항상 영구적인 피해를 일으킨다.
   c) 뇌는 평생에 걸쳐 재조직하고 새로운 연결을 형성할 수 있다.
   d) 신체 운동이 뇌 기능을 향상시키는 최선의 방법이다.

2. 지문에서 "evident"와 의미가 가장 가까운 단어는:
   a) surprising  b) clear  c) rare  d) controversial

3. 지문에 따르면, 다음 중 신경가소성을 촉진하는 것은?
   a) 더 많이 자기  b) 건강한 음식 먹기  c) 새로운 언어 배우기  d) 텔레비전 시청

<details>
<summary>정답 보기</summary>

```
1. c) 뇌는 평생에 걸쳐 재조직하고 새로운 연결을 형성할 수
   있다. (신경가소성의 핵심 아이디어)

2. b) clear ("evident" = 분명한, 명백한, 확실한)

3. c) 새로운 언어 배우기 (마지막 문장에서 악기 연주,
   운동과 함께 구체적으로 언급됨)
```

</details>

### 연습 3: IELTS Writing Task 2 — 개요 작성

다음 IELTS 프롬프트에 대한 개요(전체 에세이가 아님)를 작성하세요:

"Some people believe that university education should be free for everyone. Others argue that students should pay for their education. Discuss both views and give your own opinion."

<details>
<summary>정답 보기</summary>

```
서론:
  - 질문 바꿔 말하기
  - 진술: 양쪽 견해에 모두 장점이 있지만, [나의 입장]

본론 1 — 무상 교육 견해:
  - 모든 사람이 교육에 대한 동등한 접근 권리
  - 불평등과 사회적 이동성 장벽 감소
  - 경제에 이로움 (더 교육받은 노동력)
  - 예시: 무상 교육을 시행하는 국가 (독일, 노르웨이)

본론 2 — 학생이 비용을 부담해야 하는 견해:
  - 대학은 양질의 교육을 위한 재원이 필요
  - 투자할 때 학생들이 교육을 더 가치있게 여김
  - 납세자가 타인의 개인적 발전에 비용을 대면 안 됨
  - 학자금 대출과 장학금이 대안 제공

본론 3 — 나의 의견:
  - 균형 잡힌 접근: 소득 기반 졸업 후 상환과
    함께 보조금 지원 교육
  - 정부가 기본 비용을 부담; 학생이 공정하게 기여
  - 접근성과 질을 동시에 보장

결론:
  - 입장 재진술
  - 접근성과 질 모두 중요
  - 균형 잡힌 시스템이 사회에 가장 도움이 됨
```

</details>
