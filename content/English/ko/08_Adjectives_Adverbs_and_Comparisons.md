## 1. 영어 형용사 vs 한국어 형용사

### 1.1 핵심 차이점

```
┌─────────────────────────────────────────────────────────────────┐
│   한국어 형용사는 동사처럼 행동합니다:                              │
│   크다 (to be big), 작다 (to be small), 빠르다 (to be fast)     │
│   활용됨: 크다, 큰, 컸다, 클 것이다                                │
│                                                                 │
│   영어 형용사는 별개의 단어입니다 (활용 없음):                      │
│   big, small, fast — 시제에 따라 절대 형태가 변하지 않음            │
│   "She is big." "She was big." "She will be big."              │
│   → "big"은 모든 시제에서 "big"!                                  │
│                                                                 │
│   한국어: 큰 집 (형용사가 ㄴ/은으로 명사를 수식)                    │
│   영어: a big house (형용사가 명사 바로 앞에)                      │
│                                                                 │
│   한국어: 그 집은 크다 (형용사가 서술어 역할)                       │
│   영어: The house IS big. (형용사에 BE 동사 필요)                  │
│   ※ "The house big." ✗ — 연결동사가 반드시 있어야 합니다!          │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 형용사 위치

### 2.1 명사 앞 (한정적 용법)

```
형용사 + 명사:
  a tall man                  키 큰 남자
  a beautiful day             아름다운 날
  an expensive car            비싼 차
  a cold drink                차가운 음료
  an interesting book         재미있는 책

  ※ 한국어: 큰 + 집 (형용사 어간 + ㄴ + 명사)
    영어: big + house (형용사 + 명사, 사이에 아무것도 없음)
```

### 2.2 연결동사 뒤 (서술적 용법)

```
주어 + 연결동사 + 형용사:
  She is tall.                그녀는 키가 크다.
  The food smells delicious.  음식 냄새가 맛있다.
  He looks tired.             그는 피곤해 보인다.
  The music sounds beautiful. 음악이 아름답게 들린다.
  This feels soft.            이것은 부드럽게 느껴진다.

연결동사: be, seem, appear, look, sound, smell, taste, feel, become, get

  ※ 연결동사 뒤에는 형용사를 사용합니다 (부사가 아님):
    "She looks beautiful." ✓  (look 뒤에 형용사)
    "She looks beautifully." ✗ (이것은 "눈을 아름답게 사용한다"는 의미)
```

### 2.3 한 위치에서만 사용 가능한 형용사

```
명사 앞에서만 사용 가능 (한정적 전용):
  main, chief, principal, former, latter, elder, mere, utter, only, sole
  "the main reason" ✓     "The reason is main." ✗
  "the former president" ✓ "The president is former." ✗

동사 뒤에서만 사용 가능 (서술적 전용):
  afraid, alive, alone, asleep, awake, aware, glad, ill (영국식), well
  "She is afraid." ✓      "an afraid girl" ✗ → "a frightened girl" ✓
  "He is asleep." ✓       "an asleep baby" ✗ → "a sleeping baby" ✓
  "She is alone." ✓       "an alone person" ✗ → "a lonely person" ✓
```

## 3. 여러 형용사의 순서

### 3.1 OSASCOMP 규칙

```
┌─────────────────────────────────────────────────────────────────┐
│   명사 앞에 여러 형용사가 올 때 특정 순서를 따릅니다               │
│   (원어민은 직감적으로 이 순서를 지킵니다):                        │
│                                                                 │
│   O - 의견 (Opinion)     (beautiful, nice, ugly, delicious)     │
│   S - 크기 (Size)        (big, small, tall, short, long)        │
│   A - 나이 (Age)         (old, young, new, ancient)             │
│   S - 모양 (Shape)       (round, square, flat, wide)            │
│   C - 색깔 (Color)       (red, blue, green, dark)               │
│   O - 출신 (Origin)      (Korean, American, Japanese, French)   │
│   M - 재료 (Material)    (wooden, metal, cotton, plastic)       │
│   P - 목적 (Purpose)     (sleeping [bag], running [shoes])      │
│                                                                 │
│   예시:                                                         │
│   "a beautiful big old round red Korean wooden dining table"    │
│    의견    크기 나이 모양  색깔 출신   재료    목적    명사       │
│                                                                 │
│   실제로는 2-3개 이상의 형용사를 잘 쓰지 않습니다:                 │
│   "a beautiful old house"      (의견 + 나이)                     │
│   "a small red car"            (크기 + 색깔)                     │
│   "a tall young Korean man"    (크기 + 나이 + 출신)               │
│                                                                 │
│   ※ 한국어: 아름다운 큰 오래된 빨간 한국의 나무 식탁               │
│     한국어 순서는 더 유연하지만, 영어 순서는 고정!                  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 빠른 예시

```
"a nice little old town"          (의견 + 크기 + 나이)            ✓
"a little nice old town"          (크기 + 의견 + 나이)            ✗

"a big black leather bag"         (크기 + 색깔 + 재료)            ✓
"a black big leather bag"         (색깔 + 크기 + 재료)            ✗

"an expensive Japanese car"       (의견 + 출신)                    ✓
"a Japanese expensive car"        (출신 + 의견)                    ✗
```

## 4. 비교급 (-er / more)

### 4.1 비교급 만들기 규칙

```
┌─────────────────────────────────────────────────────────────────┐
│   1음절 형용사 → -er 추가:                                       │
│   tall → taller     old → older     fast → faster               │
│   cheap → cheaper   long → longer   young → younger             │
│                                                                 │
│   1음절, CVC로 끝나면 → 자음 두 배 + -er:                        │
│   big → bigger      hot → hotter    thin → thinner              │
│   fat → fatter      wet → wetter    sad → sadder                │
│                                                                 │
│   2음절, -y로 끝나면 → y를 -ier로 변경:                           │
│   happy → happier   easy → easier   busy → busier               │
│   early → earlier   heavy → heavier  funny → funnier            │
│                                                                 │
│   2음절 이상 형용사 → more + 형용사:                              │
│   beautiful → more beautiful                                    │
│   expensive → more expensive                                    │
│   important → more important                                    │
│   interesting → more interesting                                │
│   comfortable → more comfortable                                │
│                                                                 │
│   ※ 절대 이중 사용 금지: "more bigger" ✗ "more taller" ✗        │
│     -er 또는 more 중 하나만, 절대 둘 다 쓰지 마세요!              │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 비교급 사용하기

```
패턴: 형용사-er / more + 형용사 + than

  Seoul is bigger than Busan.
  서울은 부산보다 더 크다.

  English is more difficult than Spanish.
  영어는 스페인어보다 더 어렵다.

  She is taller than me/I.
  그녀는 나보다 더 키가 크다.

  This book is more interesting than that one.
  이 책은 저 책보다 더 재미있다.

  ※ 한국어: ~보다 더 (보다는 비교 대상 뒤에)
    영어: more/~er ... THAN (than은 형용사 뒤에)
    한국어: 서울은 부산보다 크다     (부산보다 → 크다)
    영어: Seoul is bigger than Busan  (bigger → than Busan)
```

### 4.3 비교급 수식어

```
much / far / a lot + 비교급 (훨씬):
  "Seoul is MUCH bigger than Busan."
  "This is FAR more expensive."
  "She is A LOT taller than him."

a little / a bit / slightly + 비교급 (약간):
  "She is A LITTLE taller than me."
  "This is SLIGHTLY more expensive."

even + 비교급 (심지어, 더욱):
  "Today is EVEN colder than yesterday."

  ※ "very bigger" ✗ — 비교급에 "very"를 쓸 수 없습니다!
    대신 "much"를 사용: "much bigger" ✓
```

## 5. 최상급 (-est / most)

### 5.1 최상급 만들기 규칙

```
1음절 → the + 형용사-est:
  tall → the tallest     old → the oldest     fast → the fastest
  big → the biggest      hot → the hottest

2음절, -y로 끝나면 → the + 형용사-iest:
  happy → the happiest   easy → the easiest   busy → the busiest

2음절 이상 → the most + 형용사:
  beautiful → the most beautiful
  important → the most important
  interesting → the most interesting

  ※ 최상급에는 항상 "the"를 사용:
    "She is the tallest."    ✓
    "She is tallest."        ✗ ("the" 필요)
```

### 5.2 최상급 사용하기

```
패턴: the + 형용사-est / the most + 형용사 + (in/of)

  Seoul is the biggest city in Korea.
  서울은 한국에서 가장 큰 도시이다.

  She is the smartest student in the class.
  그녀는 반에서 가장 똑똑한 학생이다.

  This is the most expensive restaurant in the city.
  이것은 도시에서 가장 비싼 식당이다.

  He is the tallest of the three brothers.
  그는 세 형제 중 가장 키가 크다.

  ※ "in" → 장소/그룹: in the class, in the world, in Korea
    "of" → 특정 집합: of the three, of all students
```

## 6. 불규칙 비교급과 최상급

### 6.1 불규칙 형태

```
┌─────────────────────────────────────────────────────────────────┐
│   형용사       비교급          최상급                              │
│   ─────────   ───────────    ───────────                        │
│   good        better         the best                           │
│   bad         worse          the worst                          │
│   far         farther/further the farthest/furthest              │
│   little      less           the least                          │
│   much/many   more           the most                           │
│   old         older/elder    the oldest/eldest                   │
│                                                                 │
│   ※ "gooder" ✗  "badder" ✗  "more good" ✗  "more bad" ✗      │
│     이것들은 항상 불규칙 — 외우세요!                              │
│                                                                 │
│   farther = 물리적 거리: "Seoul is farther than Suwon."         │
│   further = 추가적/더: "We need further information."           │
│   elder = 가족: "my elder brother" (형/언니)                     │
│   older = 일반적: "She is older than me."                       │
└─────────────────────────────────────────────────────────────────┘
```

## 7. as...as 비교

### 7.1 동등 비교: as + 형용사 + as

```
긍정 (같은 정도):
  She is as tall as her mother.
  그녀는 어머니만큼 키가 크다.

  This book is as interesting as that one.
  이 책은 저 책만큼 재미있다.

  He runs as fast as his brother.
  그는 형만큼 빨리 달린다.

부정 (다른 정도):
  She is not as tall as her mother.
  그녀는 어머니만큼 키가 크지 않다.

  English is not as easy as Spanish.
  영어는 스페인어만큼 쉽지 않다.

  ※ 한국어: ~만큼 (만큼은 비교 대상 뒤에)
    영어: as...as (비교 대상은 두 번째 "as" 뒤에)
```

### 7.2 흔한 as...as 표현

```
as soon as possible (ASAP)     가능한 한 빨리
as much as possible            가능한 한 많이
as far as I know               내가 아는 한
as long as                     ~하는 한
as well as                     ~뿐만 아니라
as good as new                 새것이나 다름없는
as easy as pie                 매우 쉬운
as cold as ice                 얼음처럼 차가운
```

## 8. 부사

### 8.1 부사 만들기: -ly

```
대부분의 형용사 → -ly를 추가하여 부사 형성:
  slow → slowly         quick → quickly       careful → carefully
  quiet → quietly       loud → loudly         beautiful → beautifully
  happy → happily       easy → easily         lucky → luckily
  comfortable → comfortably                   terrible → terribly

철자 변화:
  -le 어미 → -le을 -ly로: simple → simply, gentle → gently
  -y 어미 → -y를 -ily로: happy → happily, easy → easily
  -ic 어미 → -ally 추가: basic → basically, automatic → automatically
  -ll 어미 → -y 추가: full → fully
  -ue 어미 → -e 삭제, -ly 추가: true → truly

  ※ 모든 -ly 단어가 부사는 아닙니다!
    friendly, lovely, lonely, silly, ugly, lively = 형용사
    "She is friendly." ✓ (형용사)
    "She acted in a friendly way." ✓ ("friendlily"가 아님)
```

### 8.2 불규칙 부사

```
┌─────────────────────────────────────────────────────────────────┐
│   형용사 → 부사 (동일 형태 또는 불규칙):                          │
│                                                                 │
│   good → well         "She speaks English well."                │
│   fast → fast         "He runs fast."  ("fastly"가 아님)        │
│   hard → hard         "She works hard." ("hardly"가 아님)       │
│   late → late         "He arrived late."                        │
│   early → early       "She woke up early."                     │
│   straight → straight "Go straight."                           │
│                                                                 │
│   ※ hard vs hardly:                                            │
│     "She works hard."    = 열심히 일한다 (성실하게)               │
│     "She hardly works."  = 거의 일 안 한다 (겨우)                │
│     이 두 가지는 반대 의미!                                      │
│                                                                 │
│   ※ late vs lately:                                            │
│     "He arrived late."   = 늦게 도착했다 (정시가 아님)            │
│     "I've been busy lately." = 최근에 바빴다 (요즘)              │
│     다른 의미!                                                   │
└─────────────────────────────────────────────────────────────────┘
```

### 8.3 부사 위치

```
방법 부사 (how) → 동사/목적어 뒤:
  "She sings beautifully."            (동사 뒤)
  "He speaks English fluently."       (목적어 뒤)
  ※ "She beautifully sings." 아님 (문학적 스타일에서는 가능, 보통 아님)

빈도 부사 → 일반 동사 앞, BE 뒤:
  "I always eat breakfast."           (일반 동사 앞)
  "She is always happy."              (BE 뒤)

정도 부사 → 형용사/부사 앞:
  "She is very tall."                 (형용사 앞)
  "He runs very fast."                (부사 앞)
  "It was really interesting."        (형용사 앞)
  "She quite likes it."               (동사 앞 — 영국식)
```

### 8.4 부사의 비교급과 최상급

```
짧은 부사: -er / -est
  fast → faster → fastest
  hard → harder → hardest
  early → earlier → earliest

긴 부사 (-ly): more / most
  quickly → more quickly → most quickly
  carefully → more carefully → most carefully
  beautifully → more beautifully → most beautifully

불규칙:
  well → better → best
  badly → worse → worst
  far → farther/further → farthest/furthest
  much → more → most
  little → less → least
```

## 9. enough와 too

### 9.1 enough

```
enough + 명사 (충분한 + 명사):
  "I have enough money."          나는 돈이 충분하다.
  "There isn't enough time."      시간이 충분하지 않다.

형용사/부사 + enough (형용사/부사 + 충분히):
  "She is old enough to drive."   그녀는 운전할 만큼 나이가 든다.
  "He runs fast enough."          그는 충분히 빨리 달린다.

  ※ enough는 형용사/부사 뒤에, 명사 앞에:
    "old enough" ✓   "enough old" ✗
    "enough money" ✓  "money enough" ✗

enough + to + 동사:
  "This bag is big enough to carry all my books."
  이 가방은 내 책을 다 넣을 만큼 충분히 크다.
```

### 9.2 too

```
too + 형용사/부사 (너무 ~한):
  "This coffee is too hot."       이 커피는 너무 뜨겁다.
  "He speaks too quickly."        그는 너무 빨리 말한다.

too + 형용사 + to + 동사:
  "She is too young to drive."    그녀는 운전하기에 너무 어리다.
  "It's too cold to swim."        수영하기에 너무 춥다.

too much + 불가산명사:
  "I ate too much food."          나는 음식을 너무 많이 먹었다.

too many + 가산명사:
  "There are too many people."    사람이 너무 많다.

  ※ too vs very:
    "very" = 중립적 강조 (매우): "She is very tall." (단순 사실)
    "too" = 지나치게, 문제가 됨 (너무): "She is too tall." (문제임)
    한국어 너무는 긍정적으로도 쓰임: "너무 예쁘다!" (매우 예쁘다)
    영어 "too"는 항상 부정적: "too pretty" = 문제가 될 정도로 예쁜
```

## 10. 연습 문제

올바른 형태를 채우세요.

```
1. Seoul is ___ (big) than Busan.
2. She is the ___ (tall) girl in the class.
3. This test is ___ (difficult) than the last one.
4. He is the ___ (good) player on the team.
5. Today is ___ (cold) than yesterday.
6. This is the ___ (interesting) book I've ever read.
7. My car is ___ (expensive) than yours.
8. She is the ___ (beautiful) woman I know.
9. His grades are ___ (bad) than mine.
10. This is the ___ (easy) exercise in the book.
```

<details>
<summary>정답 보기</summary>

```
1. bigger                  6. most interesting
2. tallest                 7. more expensive
3. more difficult          8. most beautiful
4. best (불규칙)            9. worse (불규칙)
5. colder                  10. easiest
```

</details>

### 연습 2: 형용사인가 부사인가?

올바른 형태를 고르세요.

```
1. She sings (beautiful / beautifully).
2. He is a (careful / carefully) driver.
3. The food smells (delicious / deliciously).
4. She speaks English (fluent / fluently).
5. He runs very (fast / fastly).
6. She works (hard / hardly) every day.
7. The test was (easy / easily).
8. He arrived (late / lately).
9. She looks (happy / happily).
10. He did the work (good / well).
```

<details>
<summary>정답 보기</summary>

```
1. beautifully  (부사 — 동사 "sings" 수식)
2. careful      (형용사 — 명사 "driver" 수식)
3. delicious    (형용사 — 연결동사 "smells" 뒤)
4. fluently     (부사 — 동사 "speaks" 수식)
5. fast         (부사 — 형용사와 같은 형태)
6. hard         (부사 — "hardly" = 거의 안, 다른 의미!)
7. easy         (형용사 — 연결동사 "was" 뒤)
8. late         (부사 — "lately" = 최근에, 다른 의미!)
9. happy        (형용사 — 연결동사 "looks" 뒤)
10. well        (부사 — "good"의 불규칙 부사형)
```

</details>

### 연습 3: as...as

as...as로 완성하세요.

```
1. He is ___ tall ___ his father.              (같은 키)
2. English is not ___ easy ___ Spanish.        (영어가 더 어려움)
3. Please come ___ soon ___ possible.          (가능한 빨리)
4. This book is ___ interesting ___ that one.  (동등)
5. She doesn't run ___ fast ___ her sister.    (언니가 더 빠름)
```

<details>
<summary>정답 보기</summary>

```
1. He is as tall as his father.
2. English is not as easy as Spanish.
3. Please come as soon as possible.
4. This book is as interesting as that one.
5. She doesn't run as fast as her sister.
```

</details>

### 연습 4: enough 또는 too?

```
1. This coffee is ___ hot to drink.
2. She is old ___ to vote. (그녀는 20살.)
3. I have ___ money to buy a car.    (돈이 있고 충분함.)
4. There are ___ many people in this room.
5. He isn't tall ___ to reach the shelf.
```

<details>
<summary>정답 보기</summary>

```
1. too (지나치게 뜨거움 — 마실 수 없음)
2. enough (충분한 나이 — 투표 가능)
3. enough (충분한 돈)
4. too (지나치게 많음 — 문제)
5. enough (충분한 키가 아님)
```

</details>

### 연습 5: 형용사 순서

형용사를 올바른 순서로 배열하세요.

```
1. a ___ car (blue, small)
2. a ___ house (old, beautiful)
3. a ___ bag (leather, big, black)
4. ___ shoes (running, new, white)
5. a ___ man (young, tall, Korean)
```

<details>
<summary>정답 보기</summary>

```
1. a small blue car                    (크기 + 색깔)
2. a beautiful old house               (의견 + 나이)
3. a big black leather bag             (크기 + 색깔 + 재료)
4. new white running shoes             (나이 + 색깔 + 목적)
5. a tall young Korean man             (크기 + 나이 + 출신)
```

</details>

## 빠른 참조

```
┌─────────────────────────────────────────────────────────────────┐
│          형용사 & 부사 빠른 가이드                                │
│                                                                 │
│   형용사 위치: 명사 앞 / 연결동사 뒤                              │
│   복수 형용사 순서: 의견-크기-나이-모양-색깔-출신-                  │
│                    재료-목적 (OSASCOMP)                          │
│                                                                 │
│   비교급: 짧은 형 → -er + than / 긴 형 → more + than            │
│   최상급: 짧은 형 → the -est / 긴 형 → the most                 │
│   불규칙: good-better-best, bad-worse-worst                     │
│   ※ 절대 "more better"나 "most best" 쓰지 마세요!               │
│                                                                 │
│   부사: 형용사 + -ly (slowly, carefully, beautifully)            │
│   불규칙: good→well, fast→fast, hard→hard                       │
│   ※ hard ≠ hardly, late ≠ lately                               │
│                                                                 │
│   as...as = 같은 정도    not as...as = 덜한 정도                 │
│   enough: 형 + enough / enough + 명사                           │
│   too: too + 형 (지나침, 부정적 의미)                             │
│   ※ 영어 "too" ≠ 한국어 너무 (너무는 종종 긍정적)                │
└─────────────────────────────────────────────────────────────────┘
```
