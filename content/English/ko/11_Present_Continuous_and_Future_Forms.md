## 1. 현재진행형 형성

### 1.1 구조

```
┌─────────────────────────────────────────────────────────────────┐
│   현재진행형 = BE (am/is/are) + 동사-ING                         │
│                                                                 │
│   I          am eating      (I'm eating)                        │
│   you        are eating     (you're eating)                     │
│   he/she/it  is eating      (he's/she's/it's eating)           │
│   we         are eating     (we're eating)                      │
│   they       are eating     (they're eating)                    │
│                                                                 │
│   부정문: am/is/are + NOT + 동사-ing                             │
│   I'm not eating.    She isn't eating.    They aren't eating.  │
│                                                                 │
│   의문문: Am/Is/Are + 주어 + 동사-ing?                           │
│   Are you eating?    Is she eating?    What are they eating?   │
│                                                                 │
│   ※ 한국어에는 -고 있다 (하고 있다, 먹고 있다)가 진행형이지만,   │
│     한국어 화자는 영어보다 덜 자주 사용합니다.                    │
│     영어에서는 진행 중인 행동에 현재진행형이 반드시 필요합니다.    │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 -ing 철자 규칙

```
규칙 1: 대부분의 동사 → -ing 추가
  eat → eating      play → playing     read → reading
  go → going        study → studying   work → working

규칙 2: -e로 끝나는 동사 → -e를 빼고 -ing 추가
  make → making     write → writing    come → coming
  live → living     dance → dancing    have → having
  ※ 예외: be → being, see → seeing, agree → agreeing

규칙 3: 짧은 CVC 동사 → 마지막 자음을 겹치고 -ing 추가
  sit → sitting     run → running      swim → swimming
  get → getting     cut → cutting      stop → stopping
  put → putting     begin → beginning  plan → planning
  ※ w, x, y는 겹치지 않음 (show → showing, fix → fixing)

규칙 4: -ie로 끝나는 동사 → -ying으로 변환
  die → dying       lie → lying        tie → tying

규칙 5: -c로 끝나는 동사 → -king 추가
  panic → panicking   picnic → picnicking
```

## 2. 현재진행형 용법

### 2.1 바로 지금 일어나고 있는 행동

```
"I'm eating lunch."              나는 지금 점심을 먹고 있다.
"She's studying English."        그녀는 영어를 공부하고 있다.
"They're playing soccer."        그들은 축구를 하고 있다.
"Look! It's raining."            봐! 비가 오고 있어.
"Be quiet. The baby is sleeping." 조용히 해. 아기가 자고 있어.
```

### 2.2 일시적인 상황 (현재 즈음)

```
"I'm living in Seoul this month."    나는 이번 달에 서울에 살고 있다.
(일시적 — 영구적인 집이 아님)

"She's working at a cafe this summer." 그녀는 이번 여름에 카페에서 일하고 있다.
(임시 직업)

"He's reading a great book."        그는 좋은 책을 읽고 있다.
(현재 그 책을 읽는 중)
```

### 2.3 변화하는/발전하는 상황

```
"The weather is getting colder."     날씨가 점점 추워지고 있다.
"Prices are rising."                 물가가 오르고 있다.
"My English is improving."           내 영어가 나아지고 있다.
"The population is growing."        인구가 증가하고 있다.
```

### 2.4 짜증나는 반복 행동 (always/constantly와 함께)

```
"She's always complaining!"          그녀는 항상 불평해! (짜증)
"He's constantly losing his keys."   그는 계속 열쇠를 잃어버려!
"They're always making noise."       그들은 항상 시끄럽게 해!

※ "always" + 현재진행형 = 짜증/불만 표현
   "always" + 단순현재 = 중립적 습관
   "She always complains." (사실, 중립적)
   "She's always complaining!" (짜증나! 불만 표현)
```

## 3. 상태 동사 vs 동작 동사

### 3.1 상태 동사 (-ing와 함께 사용하지 않음)

```
┌─────────────────────────────────────────────────────────────────┐
│   이 동사들은 행동이 아닌 상태를 나타냅니다.                      │
│   보통 현재진행형으로 사용하지 않습니다.                          │
│                                                                 │
│   정신 상태:                                                     │
│   know, believe, think (=believe), understand, remember,        │
│   forget, imagine, suppose, mean, realize, recognize            │
│                                                                 │
│   감정:                                                          │
│   like, love, hate, want, need, prefer, wish, care              │
│                                                                 │
│   감각 (비자발적):                                                │
│   see, hear, smell, taste, feel (=느낌이 있다)                   │
│                                                                 │
│   소유:                                                          │
│   have (=소유하다), own, belong to, possess, contain, include    │
│                                                                 │
│   기타:                                                          │
│   be, seem, appear, cost, weigh, measure, exist, matter,       │
│   depend, deserve, fit, owe                                     │
│                                                                 │
│   ✓ "I know the answer."       (단순현재)                       │
│   ✗ "I'm knowing the answer."  (틀림)                           │
│                                                                 │
│   ✓ "She likes coffee."        (단순현재)                       │
│   ✗ "She's liking coffee."     (틀림)                           │
│                                                                 │
│   ✓ "I have a car."            (단순현재 — 소유)                │
│   ✗ "I'm having a car."        (틀림)                           │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 상태 의미와 동작 의미를 모두 가진 동사

```
think:
  "I think it's good."        (= believe, 상태)      ✓ 단순현재
  "I'm thinking about it."    (= considering, 동작)   ✓ 진행형

have:
  "I have a car."             (= possess, 상태)       ✓ 단순현재
  "I'm having lunch."         (= eating, 동작)         ✓ 진행형
  "I'm having a great time."  (= experiencing, 동작)   ✓ 진행형

see:
  "I see the mountain."       (= perceive, 상태)      ✓ 단순현재
  "I'm seeing the doctor."    (= visiting, 동작)       ✓ 진행형

taste:
  "This cake tastes good."    (= has the flavor, 상태) ✓ 단순현재
  "I'm tasting the soup."     (= actively trying, 동작) ✓ 진행형

look:
  "She looks happy."          (= appears, 상태)        ✓ 단순현재
  "She's looking at me."      (= directing eyes, 동작)  ✓ 진행형

weigh:
  "I weigh 70 kg."            (= my weight is, 상태)   ✓ 단순현재
  "I'm weighing the flour."   (= measuring, 동작)      ✓ 진행형

※ 한국어 생각하다는 항상 -고 있다를 쓸 수 있음: "생각하고 있다" ✓
  영어 "think"는 의미에 따라 다름:
  "thinking about" (considering) ✓ 진행형
  "think that..." (believing) ✗ 진행형 불가
```

## 4. 현재진행형 vs 단순현재

### 4.1 핵심 차이

```
┌─────────────────────────────────────────────────────────────────┐
│   단순현재                  vs    현재진행형                       │
│   ──────────────                ────────────────────             │
│   습관/일상                      지금 일어나는 행동               │
│   "I eat lunch at noon."       "I'm eating lunch (right now)."  │
│                                                                 │
│   영구적 상황                    일시적 상황                      │
│   "I live in Seoul."           "I'm living in Seoul (this month)│
│                                                                 │
│   일반적 사실                    현재 변화                        │
│   "Water boils at 100°C."     "The water is boiling!"          │
│                                                                 │
│   예정된 일정                    짜증나는 습관                    │
│   "The train leaves at 9."    "She's always leaving late!"     │
│                                                                 │
│   ※ 한국어에서는 이 구분이 명확하지 않은 경우가 많습니다:        │
│     "나는 밥을 먹는다" = "I eat rice" (습관)과                   │
│                         "I'm eating rice" (지금) 둘 다          │
│     영어에서는 이 구분이 반드시 필요합니다!                       │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 비교 예시

```
단순현재:                          현재진행형:
"I work at a bank."              "I'm working from home today."
(영구적 직업)                     (일시적, 오늘만)

"She speaks French."              "She's speaking French right now."
(능력/기술)                       (이 순간)

"It rains a lot in summer."      "It's raining outside."
(일반적 사실)                     (지금 일어나는 일)

"He plays tennis on Sundays."    "He's playing tennis right now."
(규칙적 습관)                     (이 순간)
```

## 5. will을 사용한 미래

### 5.1 형성

```
긍정문: 주어 + will + 동사원형
  I will go.        (I'll go.)
  She will come.    (She'll come.)

부정문: 주어 + will not + 동사원형
  I will not go.    (I won't go.)
  She will not come. (She won't come.)

의문문: Will + 주어 + 동사원형?
  Will you go?      Will she come?

  ※ "will"은 모든 주어에 동일 (변화 없음!)
    I will, you will, he will, she will, we will, they will
```

### 5.2 will을 사용하는 경우

```
1. 즉석 결정 (말하는 순간에 결정):
   (전화벨 울림) "I'll answer it."       내가 받을게.
   "It's cold. I'll close the window."   내가 창문 닫을게.
   "I'm hungry. I'll make a sandwich."   샌드위치 만들어야겠다.

2. 예측 (일어날 것이라고 생각하는 것):
   "It will rain tomorrow."              내일 비가 올 거야.
   "She'll be a great doctor."           그녀는 훌륭한 의사가 될 거야.
   "I think he will win."                그가 이길 것 같아.

3. 약속:
   "I'll call you tomorrow."             내일 전화할게.
   "I won't tell anyone."                아무에게도 안 말할게.
   "I'll always love you."               항상 사랑할게.

4. 제안과 요청:
   "I'll help you with that."            내가 도와줄게.
   "Will you help me?"                   도와줄래?
   "Shall I open the door?"              문 열어줄까?

5. 위협과 경고:
   "I'll tell your mother!"              엄마한테 말할 거야!
   "Be careful or you'll fall."          조심해 안 그러면 넘어질 거야.
```

## 6. going to를 사용한 미래

### 6.1 형성

```
긍정문: 주어 + am/is/are + going to + 동사원형
  I am going to eat.       (I'm going to eat.)
  She is going to study.   (She's going to study.)
  They are going to leave. (They're going to leave.)

부정문: 주어 + am/is/are + not + going to + 동사원형
  I'm not going to eat.    She isn't going to study.

의문문: Am/Is/Are + 주어 + going to + 동사원형?
  Are you going to eat?    Is she going to study?

  ※ 구어체에서: "going to" → "gonna" /ˈɡʌnə/
    "I'm gonna eat." (매우 비격식, 글에서는 사용하지 않음)
```

### 6.2 going to를 사용하는 경우

```
1. 미리 계획한 의도 (이미 결정한 계획):
   "I'm going to study medicine."        나는 의학을 공부할 거야. (이미 결정)
   "We're going to visit Korea next year." 내년에 한국 방문할 거야. (계획 있음)
   "She's going to quit her job."         그녀는 직장을 그만둘 거야. (결심 완료)

2. 증거 기반 예측 (눈에 보이는 증거가 있을 때):
   "Look at those clouds. It's going to rain." 저 구름 봐. 비 올 거야.
   "She's studied hard. She's going to pass."  열심히 했으니 합격할 거야.
   "Be careful! You're going to fall!"         조심해! 넘어지겠어!

   ※ "going to" 예측 = 증거를 볼 수 있는 경우
     "will" 예측 = 일어날 것이라고 생각/믿는 경우
```

## 7. 현재진행형을 사용한 미래

### 7.1 미래 약속/예정에 현재진행형 사용

```
현재진행형은 확정된 미래 약속/예정을 표현할 수 있습니다:

  "I'm meeting Tom tomorrow."         내일 톰을 만나기로 했어.
  "She's flying to London next week." 다음 주에 런던에 가기로 했어.
  "We're having dinner at 7."         7시에 저녁 먹기로 했어.
  "They're getting married in June."  6월에 결혼해.

  ※ 이것들은 약속/예정 — 이미 다른 사람과 조율됨
    (시간, 장소, 사람이 정해진 상태)
    한국어 ~기로 했다 / ~하기로 되어 있다와 유사
```

### 7.2 미래 시간표에 단순현재 사용

```
예정된 일정(시간표, 프로그램)에는 단순현재를 사용:

  "The train leaves at 9 AM."        기차는 9시에 출발한다.
  "The movie starts at 7:30."        영화는 7시 30분에 시작한다.
  "School begins in September."      학교는 9월에 시작한다.
  "The store closes at 10 PM."       가게는 밤 10시에 닫는다.

  ※ 고정된 일정/시간표에는 단순현재를 사용
    (변하지 않는 것 — 공식 일정)
```

## 8. 세 가지 미래 형태 비교

### 8.1 결정 차트

```
┌─────────────────────────────────────────────────────────────────┐
│              미래 형태 결정 차트                                   │
│                                                                 │
│   예정된 시간표인가?                                              │
│   예 → 단순현재: "The train leaves at 9."                        │
│                                                                 │
│   이미 약속/예정된 것인가 (사람, 장소가 정해진)?                   │
│   예 → 현재진행형: "I'm meeting Tom at 3."                       │
│                                                                 │
│   미리 계획한 의도인가?                                           │
│   예 → going to: "I'm going to study abroad."                   │
│                                                                 │
│   즉석 결정인가 (지금 결정한)?                                    │
│   예 → will: "I'll have the pasta." (식당에서)                   │
│                                                                 │
│   증거에 기반한 예측인가?                                         │
│   예 → going to: "Look! It's going to rain."                    │
│                                                                 │
│   의견/믿음에 기반한 예측인가?                                    │
│   예 → will: "I think she'll win."                               │
│                                                                 │
│   약속, 제안, 또는 위협인가?                                      │
│   예 → will: "I'll help you." / "I won't forget."               │
└─────────────────────────────────────────────────────────────────┘
```

### 8.2 같은 상황, 다른 뉘앙스

```
"I'll visit Korea."
→ 즉석 결정 (방금 결정)
  "That sounds great! I'll visit Korea too!"

"I'm going to visit Korea."
→ 미리 계획한 의도 (이 순간 전에 결정)
  "I've been planning it for months. I'm going to visit Korea."

"I'm visiting Korea next month."
→ 확정된 약속 (항공편 예약, 계획 확정)
  "I already bought the tickets. I'm visiting Korea next month."
```

### 8.3 한국어 비교

```
┌─────────────────────────────────────────────────────────────────┐
│   한국어 미래:                                                    │
│   ~할 것이다    (격식적 의도/예측)                                 │
│   ~할 거야     (비격식 의도/예측)                                  │
│   ~하겠다      (의도/의지)                                        │
│   ~할 예정이다  (예정/계획된)                                      │
│   ~하기로 했다  (약속/결정된)                                      │
│                                                                 │
│   영어 미래:                                                      │
│   will = 즉석 결정, 예측, 약속                                    │
│   going to = 미리 계획한 의도, 증거 기반 예측                     │
│   현재진행형 = 확정된 약속/예정                                    │
│   단순현재 = 시간표/일정                                          │
│                                                                 │
│   ※ 한국어 화자는 모든 것에 "will"을 과도하게 사용하는 경향.      │
│     "going to"와 현재진행형도 사용하는 법을 배우세요!              │
└─────────────────────────────────────────────────────────────────┘
```

## 9. 미래 시간 표현

### 9.1 일반적인 표현

```
tomorrow                  내일
tonight                   오늘 밤
next week/month/year      다음 주/달/해
this weekend              이번 주말
in two days / in a week   이틀 후 / 일주일 후
soon                      곧
later                     나중에
someday / one day          언젠가
the day after tomorrow    모레

※ next/this와 전치사를 함께 쓰지 않음:
  "next Monday" ✓  "on next Monday" ✗
  "this weekend" ✓  "in this weekend" ✗
```

## 10. 연습 문제

### 연습 1: 현재진행형

이 사람들이 지금 하고 있는 것에 대해 문장을 쓰세요.

```
1. She / study / English
2. They / play / soccer
3. He / not / work / today
4. We / have / lunch
5. The baby / sleep
```

<details>
<summary>정답 보기</summary>

```
1. She is studying English. / She's studying English.
2. They are playing soccer. / They're playing soccer.
3. He is not working today. / He isn't working today.
4. We are having lunch. / We're having lunch.
5. The baby is sleeping. / The baby's sleeping.
```

</details>

### 연습 2: 단순현재 또는 현재진행형?

올바른 형태를 고르세요.

```
1. I (eat / am eating) lunch every day at noon.
2. She (reads / is reading) a book right now.
3. Water (boils / is boiling) at 100°C.
4. Look! It (rains / is raining)!
5. He (works / is working) at a bank. (영구적 직업)
6. She (works / is working) at a cafe this summer. (일시적)
7. I (know / am knowing) the answer.
8. I (think / am thinking) about the problem. (고려하는 중)
9. He (has / is having) two brothers. (소유)
10. We (have / are having) dinner. (먹는 중)
```

<details>
<summary>정답 보기</summary>

```
1. eat (습관 → 단순현재)
2. is reading (지금 → 진행형)
3. boils (사실 → 단순현재)
4. is raining (지금 일어나는 일 → 진행형)
5. works (영구적 → 단순현재)
6. is working (일시적 → 진행형)
7. know (상태 동사 → 단순현재)
8. am thinking (고려하는 중 → 동작 의미 → 진행형 가능)
9. has (소유 → 상태 → 단순현재)
10. are having (먹는 중 → 동작 의미 → 진행형 가능)
```

</details>

### 연습 3: will, going to, 또는 현재진행형?

```
1. I've decided. I ___ (study) abroad next year. (미리 계획)
2. "The phone is ringing!" "I ___ (answer) it." (즉석 결정)
3. Look at those clouds. It ___ (rain). (증거)
4. I ___ (meet) Sarah at 3 PM tomorrow. (약속)
5. I think Korea ___ (win) the match. (예측/의견)
6. Don't worry. I ___ (help) you. (약속)
7. She ___ (have) a baby in March. (예정)
8. I ___ (not tell) anyone. I promise. (약속)
```

<details>
<summary>정답 보기</summary>

```
1. I'm going to study abroad next year. (미리 계획한 의도)
2. I'll answer it. (즉석 결정)
3. It's going to rain. (증거 기반 예측)
4. I'm meeting Sarah at 3 PM tomorrow. (확정된 약속)
5. I think Korea will win the match. (의견 기반 예측)
6. I'll help you. (약속/제안)
7. She's having a baby in March. (예정)
8. I won't tell anyone. (약속)
```

</details>

### 연습 4: 한국어 → 영어

```
1. 나는 지금 저녁을 요리하고 있다.
2. 비가 올 것 같아. (구름을 보면서)
3. 내일 톰을 만나기로 했어.
4. 다음 달에 한국에 갈 거야. (이미 계획)
5. 걱정 마. 내가 도와줄게. (약속)
6. 나는 매일 아침 운동한다. (습관)
7. 날씨가 점점 따뜻해지고 있다.
```

<details>
<summary>정답 보기</summary>

```
1. I'm cooking dinner right now.
2. It's going to rain. (증거 → going to)
3. I'm meeting Tom tomorrow. (약속 → 현재진행형)
4. I'm going to go to Korea next month. (미리 계획 → going to)
5. Don't worry. I'll help you. (약속 → will)
6. I exercise every morning. (습관 → 단순현재)
7. The weather is getting warmer. (변화하는 상황 → 진행형)
```

</details>
