## 1. 영어 소리 소개

### 1.1 세계 공용어로서의 영어

```
┌─────────────────────────────────────────────────────────────────┐
│                  English: A Global Language                      │
│                                                                 │
│   원어민 수: ~4억 명                                             │
│   전체 사용자: ~15억 명 (L2 포함)                                │
│   공용어 지정: 67개국                                            │
│   UN 공식 언어: 예 (6개 중 하나)                                 │
│                                                                 │
│   어족: 인도유럽어 → 게르만어 → 서게르만어                        │
│   주요 변종: 미국식(AmE), 영국식(BrE), 호주식                    │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 한국어 화자에게 영어 발음이 어려운 이유

스페인어나 이탈리아어와 달리, 영어 철자는 발음과 일치하지 않습니다. 같은 글자가 여러 소리를 내고, 같은 소리가 여러 철자로 쓰일 수 있습니다.

```
"ough"라는 철자의 7가지 이상의 발음:
  through  /θruː/     (우 소리)
  though   /ðoʊ/      (오우 소리)
  tough    /tʌf/      (어프 소리)
  cough    /kɒf/      (오프 소리)
  thought  /θɔːt/     (오트 소리)
  bough    /baʊ/      (아우 소리)
  hiccough /ˈhɪkʌp/   (업 소리)
```

> **한국어 화자 참고**: 한글은 글자와 소리가 거의 완벽하게 일치합니다. 영어는 그렇지 않습니다. 각 단어의 발음을 개별적으로 배워야 합니다 — 철자만으로 발음을 정확히 추측하기 어렵습니다.

### 1.3 IPA: 발음 가이드

IPA(국제 음성 기호)는 각 소리에 정확히 하나의 기호를 부여합니다. 이 강좌 전체에서 IPA 표기를 사용합니다.

```
IPA가 한국어 화자에게 중요한 이유:
  한국어 철자: 읽다 → 일다 (철자 ≠ 발음)
  영어 철자: knight → /naɪt/ (k, g, h 모두 묵음!)

  IPA는 모호함을 제거합니다. IPA를 읽을 줄 알면
  사전에서 항상 정확한 발음을 알 수 있습니다.
```

### 3.4 슈와 /ə/ — 영어에서 가장 중요한 소리

```
┌─────────────────────────────────────────────────────────────────┐
│   슈와 /ə/ — 영어에서 가장 많이 나오는 소리                       │
│                                                                 │
│   무엇인가: 짧고, 약하고, 중립적인 "어" 소리                     │
│   언제 나오는가: 강세 없는 음절에서                                │
│   왜 중요한가: 거의 모든 영어 단어에 등장                          │
│                                                                 │
│   예시 (슈와 표시):                                              │
│   a-BOUT   /ə.ˈbaʊt/     (첫 음절이 약하게 "어")                 │
│   ba-NA-na  /bə.ˈnæ.nə/  (1번째, 3번째 음절 모두 슈와)           │
│   TO-day    /tə.ˈdeɪ/     (첫 음절이 약하게 "터")                 │
│   po-LICE   /pə.ˈliːs/   (첫 음절이 약하게 "퍼")                 │
│                                                                 │
│   ※ 한국어 화자는 모든 음절을 똑같이 또렷하게 발음하는 경향이     │
│     있습니다. 영어에서는 강세 없는 음절이 슈와로 축약됩니다.       │
│     이것이 자연스러운 영어 발음의 핵심입니다.                      │
└─────────────────────────────────────────────────────────────────┘
```

## 4. 모음 체계: 이중모음

### 4.1 8개의 이중모음

이중모음은 한 음절 안에서 하나의 소리에서 다른 소리로 미끄러지는 모음입니다.

| IPA | 예시 | 한국어 근사음 | 움직임 |
|-----|------|-------------|--------|
| /eɪ/ | d**ay**, m**a**ke, r**ai**n | 에이 | e → ɪ |
| /aɪ/ | t**i**me, b**uy**, cr**y** | 아이 | a → ɪ |
| /ɔɪ/ | b**oy**, n**oi**se, t**oy** | 오이 | ɔ → ɪ |
| /aʊ/ | h**ou**se, n**ow**, c**ow** | 아우 | a → ʊ |
| /oʊ/ | g**o**, h**o**me, kn**ow** | 오우 | o → ʊ |
| /ɪə/ | h**ere**, n**ear**, b**eer** (영국식) | 이어 | ɪ → ə |
| /eə/ | th**ere**, c**are**, h**air** (영국식) | 에어 | e → ə |
| /ʊə/ | p**ure**, t**our**, c**ure** (영국식) | 우어 | ʊ → ə |

> **참고**: 미국식 영어(AmE)에서 /ɪə/, /eə/, /ʊə/는 보통 모음 + /r/로 대체됩니다: /ɪr/, /er/, /ʊr/.

### 4.2 혼동하기 쉬운 모음 쌍

```
한국어 화자가 혼동하는 쌍:

  /æ/ vs /e/           bat /bæt/ vs bet /bet/
  /ʌ/ vs /ɑː/         cut /kʌt/ vs cart /kɑːrt/
  /ɪ/ vs /iː/         ship /ʃɪp/ vs sheep /ʃiːp/
  /ʊ/ vs /uː/         full /fʊl/ vs fool /fuːl/
  /ɔː/ vs /oʊ/        caught /kɔːt/ vs coat /koʊt/

  ※ 모음 길이가 영어에서는 의미를 바꿉니다!
  한국어의 모음 장단 구분 (말 vs 말:)은 사라지고 있어서,
  한국어 화자는 영어 모음 길이를 무시하는 경향이 있습니다.
```

영어에는 24개의 자음 음소가 있습니다. 대부분은 한국어와 잘 대응되지만, 몇 가지는 그렇지 않습니다.

```
│   파찰음:     /tʃ/ /dʒ/                                         │
│   비음:       /m/ /n/ /ŋ/                                        │
│   유음:       /l/ /r/                                            │
│   활음:       /w/ /j/                                            │
│                                                                 │
```

### 5.2 파열음 (폐쇄음)

| IPA | 한국어 유사음 | 예시 | 참고사항 |
|-----|-------------|------|---------|
| /p/ | ㅍ (격음) / ㅂ (평음) | **p**en, sto**p** | 어두에서: ㅍ처럼 유기음 |
| /b/ | ㅂ (유성) | **b**ig, ca**b** | 유성음 — 성대를 진동 |
| /t/ | ㅌ (격음) / ㄷ (평음) | **t**en, ca**t** | 어두에서: ㅌ처럼 유기음 |
| /d/ | ㄷ (유성) | **d**og, be**d** | 유성음 — 성대를 진동 |
| /k/ | ㅋ (격음) / ㄱ (평음) | **c**at, ba**ck** | 어두에서: ㅋ처럼 유기음 |
| /g/ | ㄱ (유성) | **g**et, bi**g** | 유성음 — 성대를 진동 |

> **한국어 화자 팁**: 영어는 유성음 vs 무성음 (/b/ vs /p/, /d/ vs /t/, /g/ vs /k/)을 구분합니다. 한국어는 평음 vs 격음 vs 경음 (ㅂ/ㅍ/ㅃ)을 구분합니다. 이 둘은 다른 체계입니다! /b/, /d/, /g/의 유성음 (성대 진동) 연습이 필요합니다.

## 6. 한국어에 없는 소리

### 6.1 R vs L — 1순위 난관 (R/L 구분)

```
┌─────────────────────────────────────────────────────────────────┐
│              R /r/ vs L /l/ — 핵심 구분                          │
│                                                                 │
│   한국어 ㄹ = 하나의 소리 (탄설음, 스페인어 "r"과 유사)            │
│   영어 = 완전히 다른 두 개의 소리                                 │
│                                                                 │
│   /l/ (설측음):                                                  │
│   • 혀끝이 윗니 뒤의 치조에 닿음                                  │
│   • 공기가 혀의 옆으로 흐름                                      │
│   • "light" /laɪt/    "bell" /bel/    "play" /pleɪ/            │
│                                                                 │
│   /r/ (접근음):                                                  │
│   • 혀끝이 뒤로 말림, 아무 데도 닿지 않음                         │
│   • 혀 옆이 위 어금니에 닿음                                     │
│   • "right" /raɪt/    "car" /kɑːr/    "pray" /preɪ/            │
│                                                                 │
│   최소 대립쌍:                                                   │
│   right /raɪt/  vs  light /laɪt/                                │
│   road /roʊd/   vs  load /loʊd/                                 │
│   read /riːd/   vs  lead /liːd/                                 │
│   row /roʊ/     vs  low /loʊ/                                   │
│   rock /rɒk/    vs  lock /lɒk/                                  │
│   pray /preɪ/   vs  play /pleɪ/                                 │
│   fry /fraɪ/    vs  fly /flaɪ/                                  │
│   grow /groʊ/   vs  glow /gloʊ/                                 │
│                                                                 │
│   ※ 연습: "la la la" 반복 (혀가 치조에 닿음)                    │
│     그런 다음 "ra ra ra" 반복 (혀를 뒤로 말림, 아무 데도 안 닿음) │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 /θ/와 /ð/ — "th" 소리

```
┌─────────────────────────────────────────────────────────────────┐
│   무성 th /θ/:  think, three, bath, math, tooth                  │
│   • 혀끝을 이 사이에 넣고 공기를 불음 (성대 진동 없음)            │
│   • 한국어 화자는 ㅅ로 대체 → "sink" 대신 "think"이 되어야 함    │
│                                                                 │
│   유성 th /ð/:     this, that, the, mother, breathe              │
│   • 혀끝을 이 사이에 넣고 성대를 진동시키며 공기를 불음            │
│   • 한국어 화자는 ㄷ으로 대체 → "dis" 대신 "this"가 되어야 함    │
│                                                                 │
│   최소 대립쌍:                                                   │
│   think /θɪŋk/ vs sink /sɪŋk/ (th vs s)                        │
│   three /θriː/ vs tree /triː/ (th vs t)                        │
│   path /pæθ/   vs pass /pæs/  (th vs s)                         │
│   then /ðen/   vs den /den/   (th vs d)                         │
│   they /ðeɪ/   vs day /deɪ/   (th vs d)                         │
│                                                                 │
│   ※ 연습: 혀를 이 사이로 살짝 내밀어보세요.                     │
│     혀 위로 공기가 흐르는 것을 느껴보세요. 이 소리는 한국어에     │
│     없으므로 집중적인 연습이 필요합니다.                          │
└─────────────────────────────────────────────────────────────────┘
```

### 6.3 /f/와 /v/ — 순치 마찰음

```
/f/ (무성 순치 마찰음):
  • 윗니가 아랫입술에 닿고 공기를 불음
  • fish, five, life, off, phone
  • 한국어 화자는 ㅍ (/p/)로 대체 → "pish"가 아니라 "fish"
  • ※ ㅍ은 두 입술을 사용. /f/는 이 + 입술을 사용. 다릅니다!

/v/ (유성 순치 마찰음):
  • /f/와 같은 위치 + 성대 진동
  • very, live, van, love, video
  • 한국어 화자는 ㅂ (/b/)로 대체 → "bery"가 아니라 "very"
  • ※ ㅂ은 두 입술을 사용. /v/는 이 + 입술을 사용. 다릅니다!

최소 대립쌍:
  fan /fæn/   vs  pan /pæn/   (f vs p)
  fine /faɪn/ vs  pine /paɪn/ (f vs p)
  vine /vaɪn/ vs  wine /waɪn/ (v vs w)
  vest /vest/ vs  best /best/ (v vs b)
  very /ˈveri/ vs berry /ˈberi/ (v vs b)
```

### 6.4 /z/ — 유성 S

```
/z/ (유성 치조 마찰음):
  • /s/와 같은 입 위치 + 성대 진동
  • zoo, buzz, zero, please, dogs
  • 한국어에는 /z/ 소리가 없음
  • 한국어 화자는 ㅈ (/dʒ/)로 대체 → "joo"가 아니라 "zoo"

최소 대립쌍:
  sip /sɪp/   vs  zip /zɪp/
  bus /bʌs/   vs  buzz /bʌz/
  price /praɪs/ vs prize /praɪz/
  ice /aɪs/   vs  eyes /aɪz/

※ 많은 영어 단어에서 "s" 철자가 /z/로 발음됩니다:
  dogs /dɒgz/    is /ɪz/      has /hæz/
  boys /bɔɪz/    was /wɒz/    his /hɪz/
```

## 7. 단어 강세

### 7.1 단어 강세란?

```
┌─────────────────────────────────────────────────────────────────┐
│              영어 단어 강세 — 핵심 개념                           │
│                                                                 │
│   2음절 이상의 모든 영어 단어에는 하나의 강세 음절이 있음          │
│   강세 음절은: 더 크게, 더 길게, 더 높은 음높이로 발음             │
│                                                                 │
│   ba-NA-na    /bə.ˈnæ.nə/     (2번째 음절 강세)                 │
│   COM-pu-ter  /kəm.ˈpjuː.tər/ (2번째 음절 강세)                 │
│   BEAU-ti-ful /ˈbjuː.tɪ.fəl/  (1번째 음절 강세)                │
│                                                                 │
│   ※ 한국어는 음절 박자 언어(syllable-timed)로 각 음절이          │
│     거의 같은 비중을 가집니다. 영어는 강세 박자 언어               │
│     (stress-timed)로 강세 음절은 크고/길고, 비강세 음절은          │
│     압축됩니다. 이것이 영어가 한국어 귀에 "고르지 않게"           │
│     들리는 이유입니다.                                            │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 품사별 강세 패턴

```
2음절 명사 — 보통 첫 번째 음절에 강세:
  TA-ble    WA-ter    DOC-tor    MU-sic    PIC-ture

2음절 동사 — 보통 두 번째 음절에 강세:
  be-GIN    for-GET   ar-RIVE    de-CIDE   re-PEAT

같은 단어, 다른 강세 = 다른 의미:
  RE-cord (명사: 기록)    vs  re-CORD (동사: 기록하다)
  PRE-sent (명사: 선물)   vs  pre-SENT (동사: 제시하다)
  OB-ject (명사: 물체)    vs  ob-JECT (동사: 반대하다)
  CON-tract (명사: 계약)  vs  con-TRACT (동사: 수축하다)
  PRO-duce (명사: 농산물)  vs  pro-DUCE (동사: 생산하다)
```

### 7.3 긴 단어의 강세

```
강세를 끌어당기는 접미사 (접미사 또는 바로 앞 음절에 강세):
  -tion / -sion    → edu-CA-tion, de-CI-sion
  -ic              → fan-TAS-tic, e-co-NOM-ic
  -ity             → u-ni-VER-si-ty, e-lec-TRI-ci-ty
  -ial / -ious     → of-FI-cial, am-BI-tious
  -graphy          → pho-TOG-ra-phy, ge-OG-ra-phy

강세를 바꾸지 않는 접미사:
  -ment            → de-VE-lop → de-VE-lop-ment
  -ness            → HAP-py → HAP-pi-ness
  -ful             → BEAU-ty → BEAU-ti-ful
  -less            → CARE → CARE-less
  -ly              → QUICK → QUICK-ly
```

## 8. 문장 강세와 리듬

### 8.1 내용어 vs 기능어

```
┌─────────────────────────────────────────────────────────────────┐
│   내용어 (강세 있음) — 의미를 전달:                               │
│   명사:      DOG, TABLE, KOREA                                   │
│   본동사:    EAT, STUDY, WANT                                    │
│   형용사:    BIG, HAPPY, FAST                                    │
│   부사:      QUICKLY, ALWAYS, VERY                               │
│   부정어:    NOT, NEVER, NO                                      │
│   의문사:    WHAT, WHERE, WHO                                    │
│                                                                 │
│   기능어 (강세 없음) — 문법적 접착제:                              │
│   관사:      a, an, the                                          │
│   대명사:    I, you, he, she, it, we, they                       │
│   전치사:    in, on, at, to, for, of                              │
│   조동사:    am, is, are, was, do, have, can, will               │
│   접속사:    and, but, or, that                                   │
│                                                                 │
│   예시:                                                          │
│   "I want to GO to the STORE and BUY some BREAD."               │
│    약   약약   약  강   약  강    약약   약   강                    │
└─────────────────────────────────────────────────────────────────┘
```

### 8.2 강세 박자 리듬

```
영어는 강세 박자 언어(STRESS-TIMED):
  강세 음절 사이의 시간이 대략 일정하며,
  사이에 비강세 음절이 몇 개든 상관없습니다.

  DOGS    CHASE    CATS           (강세 3개, ~1초)
  The DOGS   are CHAS-ing  the CATS   (강세 3개, ~1초)
  The BIG DOGS  have been CHAS-ing  the SMALL CATS  (강세 사이 여전히 ~1초)

  ※ 한국어는 음절 박자 언어(SYLLABLE-TIMED): 각 음절이 거의 같은 시간.
    그래서 한국어 화자의 영어가 "로봇처럼" 들리는 것입니다 —
    모든 음절에 같은 비중을 주기 때문입니다.
    해결책: 내용어에 강세를, 기능어는 압축하세요.
```

## 9. 연음 현상

### 9.1 연결 (Linking)

자연스럽게 말할 때, 영어 단어는 따로따로 발음되지 않습니다. 서로 연결됩니다.

```
자음 → 모음 연결:
  "turn off"     → "tur-NOFF"     /tɜːr.ˈnɒf/
  "pick it up"   → "pi-KI-TUP"   /pɪ.kɪ.ˈtʌp/
  "an apple"     → "a-NA-pple"    /ə.ˈnæ.pəl/

모음 → 모음 연결:
  "go away"      → "go-WA-way"   (/w/ 삽입)
  "she asked"    → "she-YASKED"  (/j/ 삽입)
  "to eat"       → "to-WEAT"     (/w/ 삽입)

자음 → 자음 연결:
  "good day"     → "goo-DAY"     (첫 /d/ 미파열)
  "hot tea"      → "ho-TEA"      (첫 /t/ 합류)
  "big game"     → "bi-GAME"     (첫 /g/ 미파열)
```

### 9.2 축약 (Reduction)

비강세 기능어는 빠른 말에서 크게 변합니다.

```
단어        완전형        축약형          예시
and         /ænd/        /ən/ or /n/     bread 'n butter
to          /tuː/        /tə/            want to → wanna /ˈwɒnə/
for         /fɔːr/       /fər/           "what's it for?"
of          /ɒv/         /əv/ or /ə/     cup of tea → cuppa tea
you         /juː/        /jə/            "did you" → "didja"
can         /kæn/        /kən/           "I can go" → /aɪ kən goʊ/
have        /hæv/        /əv/ or /ə/     "should have" → "shoulda"
```

### 9.3 탈락 (Elision)

```
흔히 탈락되는 소리:
  "probably"    → "probly"      /ˈprɒbli/     (중간 음절 탈락)
  "comfortable" → "comfterble"  /ˈkʌmftəbəl/  (음절 탈락)
  "Wednesday"   → "Wensday"    /ˈwenzdeɪ/     (/d/ 탈락)
  "next day"    → "nex day"    /ˈneksdeɪ/     (/t/ 탈락)
  "must be"     → "mus be"     /ˈmʌsbi/       (/t/ 탈락)
```

## 10. 억양 패턴

### 10.1 하강 억양 (↘)

```
사용: 서술문, 의문사 의문문, 명령문

  서술문:
    "I live in Seoul."
    ──────────╲
              ↓ (끝에서 하강)

  의문사 의문문:
    "Where do you live?"
    ╱──────────╲
    ↑ ("where"에서 높게)  ↓ (끝에서 하강)

  명령문:
    "Close the door."
    ──────╲
          ↓
```

### 10.2 상승 억양 (↗)

```
사용: 예/아니오 의문문, 놀람 표현, 나열 (마지막 항목 제외)

  예/아니오 의문문:
    "Do you speak English?"
    ──────────────╱
                  ↑ (끝에서 상승)

  놀람:
    "Really?"
    ────╱
        ↑

  나열 (마지막 전까지):
    "I bought apples↗, oranges↗, and bananas↘."
```

### 10.3 하강-상승 억양 (↘↗)

```
사용: 불확실함, 공손함, 대조, "하지만..."

  "I like the color... (but not the size)"
  ────╲──╱
      ↓  ↑

  "It's okay..."  (유보를 암시)
  ───╲──╱
```

> **한국어 화자 참고**: 한국어 억양은 영어에 비해 상대적으로 평탄합니다. 한국어는 조사(까, 니)로 의문을 표시하지만 영어는 억양 + 어순을 사용합니다. 음높이 변화를 과장해서 연습하세요 — 한국어에서 "너무 과장스럽게" 느껴지는 것이 영어에서는 정상입니다.

## 11. 한국어 화자의 흔한 발음 실수

### 11.1 체계적 오류 패턴

| 오류 | 한국어 화자 발음 | 올바른 발음 | 교정 방법 |
|------|----------------|------------|----------|
| /f/ → /p/ | "pish" | "fish" /fɪʃ/ | 윗니를 아랫입술에 |
| /v/ → /b/ | "bery" | "very" /ˈveri/ | 윗니를 아랫입술에 + 성대 진동 |
| /θ/ → /s/ | "sink" | "think" /θɪŋk/ | 혀를 이 사이에 |
| /ð/ → /d/ | "dis" | "this" /ðɪs/ | 혀를 이 사이에 + 성대 진동 |
| /z/ → /dʒ/ | "joo" | "zoo" /zuː/ | /s/와 같은 위치 + 성대 진동 |

| /l/ → /r/ | "rice" | "lice" /laɪs/ | 혀끝이 치조에 닿게 |
| 모음 삽입 | "seh-tuh-rike" | "strike" /straɪk/ | 자음 사이에 모음 넣지 않기 |

### 11.2 종성 자음 파열

```
┌─────────────────────────────────────────────────────────────────┐
│   한국어 종성(받침) 규칙:                                        │
│   • 7개 소리만 가능: ㄱ,ㄴ,ㄷ,ㄹ,ㅁ,ㅂ,ㅇ                       │
│   • 자음이 미파열 (공기 분출 없음)                                │
│                                                                 │
│   영어 단어 끝 자음 규칙:                                        │
│   • 어떤 자음이든 단어를 끝낼 수 있음                             │
│   • 자음이 파열될 수 있음 (공기 분출 있음)                        │
│   • 자음군(consonant cluster)이 흔함: texts /teksts/             │
│                                                                 │
│   한국어 화자의 흔한 오류:                                       │
│   "dog" → "dok" (종성 /g/를 /k/로 무성화)                       │
│   "bed" → "bet" (종성 /d/를 /t/로 무성화)                       │
│   "five" → "paibeu" (모음 추가, /f/를 /p/로 변환)               │
│   "bus" → "beoseu" (모음 추가)                                   │
│                                                                 │
│   ※ 종성 자음 파열 연습:                                        │
│     "cat" — /t/가 공기와 함께 터져 나오게                        │
│     "dog" — /g/를 끝까지 유성으로                                │
│     "love" — /v/ (윗니+입술)로 끝내고, 으를 추가하지 않기         │
└─────────────────────────────────────────────────────────────────┘
```

### 11.3 자음군 (Consonant Clusters)

```
한국어는 음절 시작에 자음 최대 1개, 끝에도 1개만 허용합니다.
영어는 시작에 최대 3개, 끝에 4개 이상 허용합니다.

한국어 화자가 어려워하는 영어 자음군:
  어두:     str- (strike), spr- (spring), skr- (scream)
  중간:     -nstr- (construct), -mpl- (simple)
  어말:     -lk (milk), -nks (thanks), -ksts (texts)

  ※ 자음 사이에 모음을 삽입하지 마세요!
  "strike" ≠ "suh-tuh-rai-kuh"  →  /straɪk/ (1음절!)
  "spring" ≠ "suh-puh-ring"     →  /sprɪŋ/  (1음절!)
  "milk"   ≠ "mil-kuh"          →  /mɪlk/   (1음절!)
```

```
3. cat /kæt/ (단모음)     vs  cart /kɑːrt/ (장모음)     → æ vs ɑː
4. cut /kʌt/ (단모음)     vs  cart /kɑːrt/ (장모음)     → ʌ vs ɑː
5. cot /kɒt/ (단모음)     vs  caught /kɔːt/ (장모음)    → ɒ vs ɔː
6. sit /sɪt/ (단모음)     vs  seat /siːt/ (장모음)      → ɪ vs iː
7. pull /pʊl/ (단모음)    vs  pool /puːl/ (장모음)      → ʊ vs uː
8. hot /hɒt/ (단모음)     vs  heart /hɑːrt/ (장모음)    → ɒ vs ɑː
```

</details>

### 연습 4: 단어 강세

각 단어의 강세 위치를 표시하세요. 어느 음절에 주강세가 있나요?

<details>
<summary>정답 보기</summary>

```
1. computer    → com-PU-ter     (2번째 음절)
2. banana      → ba-NA-na       (2번째 음절)
3. beautiful   → BEAU-ti-ful    (1번째 음절)
4. important   → im-POR-tant    (2번째 음절)
5. university  → u-ni-VER-si-ty (3번째 음절)
6. photograph  → PHO-to-graph   (1번째 음절)
7. photography → pho-TOG-ra-phy (2번째 음절) ← 강세가 이동!
8. information → in-for-MA-tion (3번째 음절)
9. record (명사) → RE-cord      (1번째 음절)
10. record (동사) → re-CORD     (2번째 음절)
```

### 연습 5: 연음

이 구문들을 자연스럽게 읽으며 단어들을 연결하세요.

<details>
<summary>정답 보기</summary>

```
1. "turn off"      → tur-NOFF       (자음이 모음에 연결)
2. "pick it up"    → pi-KI-TUP     (모든 단어가 연결)
3. "an apple"      → a-NA-pple     (/n/이 /æ/에 연결)
4. "good evening"  → goo-DEE-vning (/d/가 /iː/에 연결)
5. "not at all"    → no-TA-TALL    (모두 연결)
6. "I want to go"  → I-WANNA-go    (빠른 말에서 want to → wanna)
7. "cup of tea"    → CU-pə-TEA     (of가 /ə/로 축약)
8. "bread and butter" → BRE-dən-BUTTER (and → 'n)
```

### 연습 6: f/v 연습

각 단어를 말할 때, 윗니가 아랫입술에 닿는지 확인하세요.

```
/f/ 단어:  fish, five, friend, off, laugh, phone, coffee
/v/ 단어:  very, visit, video, love, live, move, over

대립쌍:  fan vs van    fine vs vine    few vs view
        safe vs save   leaf vs leave   half vs have
```
