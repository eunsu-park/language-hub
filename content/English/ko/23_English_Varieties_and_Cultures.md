## 1. 영어 변종 소개

### 1.1 세계 언어로서의 영어

```
┌─────────────────────────────────────────────────────────────────┐
│                    세계의 영어                                    │
│                                                                 │
│   원어민 화자:      ~4억 명                                       │
│   제2언어 화자:     ~7억 5천만 명                                  │
│   제3언어+ 화자:    ~6억 명                                       │
│   합계:            전 세계 ~17억 5천만 명의 화자                     │
│                                                                 │
│   공식 언어:       ~60개국                                        │
│   UN 공식 언어:    예 (6개 중 하나)                                │
│                                                                 │
│   주요 변종:                                                     │
│   • 미국 영어 (AmE) — ~3억 3천만 명                               │
│   • 영국 영어 (BrE) — ~6,700만 명                                 │
│   • 호주 영어 (AuE) — ~2,600만 명                                 │
│   • 캐나다 영어 (CaE) — ~3,800만 명                               │
│   • 인도 영어 (InE) — ~1억 2,500만 명 (L2)                        │
│   • 남아공 영어 — ~500만 명                                       │
│   • 필리핀 영어 — ~7,000만 명 (L2)                                │
│   • 나이지리아 영어 — ~8,000만 명 (L2)                             │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 "올바른" 영어는 없다

```
중요한 원칙:
  어떤 영어 변종도 다른 것보다 "더 올바른" 것은 없습니다.
  미국식, 영국식, 호주식, 인도식 영어 — 모두 유효합니다.

그러나:
  • 하나의 문서 내에서는 일관성을 유지하세요
    (미국식과 영국식 철자를 섞지 마세요)
  • 독자를 파악하세요
    (영국 학술 논문은 영국식 영어 사용)
  • 대부분의 국제적 맥락에서는 미국식 영어 사용
    (미디어, 기술, 비즈니스의 영향)
  • 한국 표준화 시험 (TOEIC, TOEFL)은 미국식 영어 사용
  • IELTS는 미국식과 영국식 영어 모두 허용
```

## 4. 미국식 vs 영국식: 문법

### 4.1 주요 문법 차이

```
┌─────────────────────────────────────────────────────────────────┐
│   1. 현재완료 vs 단순과거                                         │
│                                                                 │
│   BrE: "I've just eaten." / "Have you seen the news?"          │
│   AmE: "I just ate."      / "Did you see the news?"            │
│                                                                 │
│   미국 영어는 영국 영어가 현재완료를 쓰는 곳에서                    │
│   단순과거를 더 자주 사용합니다.                                   │
│                                                                 │
│   2. 집합명사                                                    │
│                                                                 │
│   BrE: "The team are playing well." (복수 동사 가능)             │
│   AmE: "The team is playing well." (단수 동사)                   │
│                                                                 │
│   영국 영어는 집합명사를 복수로 취급할 수 있습니다.                  │
│   미국 영어는 단수로 취급합니다.                                   │
│                                                                 │
│   3. "GOT" vs "GOTTEN"                                         │
│                                                                 │
│   BrE: "I've got a car." (소유)                                 │
│   AmE: "I've gotten better." (되다/얻다)                         │
│   AmE: "I've got a car." (소유에도 사용)                          │
│                                                                 │
│   "Gotten"은 미국식만 해당. 영국 영어는 모든 의미에                │
│   "got"을 사용합니다.                                             │
│                                                                 │
│   4. "SHALL" vs "WILL"                                         │
│                                                                 │
│   BrE: "Shall we go?" (제안 — 흔히 사용)                         │
│   AmE: "Should we go?" 또는 "Do you want to go?"               │
│         (미국 영어에서 "shall"은 드물게 사용)                      │
│                                                                 │
│   5. 본동사로서의 "HAVE"                                          │
│                                                                 │
│   BrE: "Have you got a pen?"   / "I haven't got time."         │
│   AmE: "Do you have a pen?"    / "I don't have time."          │
│                                                                 │
│   6. 전치사                                                      │
│                                                                 │
│   AmE: "on the weekend"        BrE: "at the weekend"           │
│   AmE: "Monday through Friday" BrE: "Monday to Friday"         │
│   AmE: "different from/than"   BrE: "different from/to"        │
│   AmE: "write (someone)"      BrE: "write to (someone)"       │
└─────────────────────────────────────────────────────────────────┘
```

## 5. 미국식 vs 영국식: 발음

### 5.1 주요 발음 차이

```
1. R 발음 (rhotic vs non-rhotic):
   AmE: 모든 위치에서 /r/ 발음 (rhotic)
        "car" /kɑːr/, "water" /wɔːtər/, "here" /hɪr/
   BrE: 모음 앞을 제외하고 /r/ 생략 (non-rhotic)
        "car" /kɑː/, "water" /wɔːtə/, "here" /hɪə/

2. /æ/ vs /ɑː/ (bath 계열 단어):
   AmE: "bath" /bæθ/, "can't" /kænt/, "dance" /dæns/
   BrE: "bath" /bɑːθ/, "can't" /kɑːnt/, "dance" /dɑːns/

3. T 탄설음화 (미국식):
   AmE: "water" /wɔːdər/, "better" /bedər/, "city" /sɪdi/
   BrE: "water" /wɔːtə/, "better" /betə/, "city" /sɪti/
   (미국 영어에서 모음 사이의 /t/는 빠른 /d/ 소리)

4. /ɒ/ vs /ɑː/ (lot 계열 단어):
   AmE: "hot" /hɑːt/, "not" /nɑːt/, "lot" /lɑːt/
   BrE: "hot" /hɒt/, "not" /nɒt/, "lot" /lɒt/

5. 강세 차이:
   AmE: AD-dress (명사)        BrE: a-DRESS (둘 다)
   AmE: ga-RAGE                BrE: GA-rage
   AmE: week-END               BrE: WEEK-end
```

## 6. 호주 영어

### 6.1 주요 특징

```
발음:
  • "day" → /daɪ/ 처럼 발음
  • 평서문 끝에서 올라가는 억양
    (질문처럼 들림)
  • "i" 소리 → /ɔɪ/: "right"가 "roight"처럼 들림

어휘:
  arvo = afternoon (오후)       "See you this arvo."
  barbie = barbecue (바베큐)    "Let's have a barbie."
  brekkie = breakfast (아침)    "What's for brekkie?"
  cheers = thanks (고마워)      "Cheers, mate."
  mate = friend (친구)          "G'day, mate."
  no worries = no problem       "No worries, mate."
         (문제없어)
  reckon = think (생각하다)     "I reckon it'll rain."
  thongs = flip-flops           "Grab your thongs for the beach."
         (슬리퍼)
  ute = pickup truck (픽업트럭) "Load it into the ute."

축약어 (매우 흔함):
  afternoon → arvo              university → uni
  breakfast → brekkie           barbecue → barbie
  mosquito → mozzie             sunglasses → sunnies
  Australian → Aussie           documentation → doco
```

## 7. 인도 영어

### 7.1 주요 특징

```
어휘 (인도 영어 고유):
  lakh = 100,000 (10만)         "The car costs 5 lakhs."
  crore = 10,000,000 (천만)     "Revenue of 2 crores."
  prepone = 앞당기다            "Let's prepone the meeting."
    (postpone의 반대 — 다른 변종에서는 사용하지 않음)
  do the needful = 필요한       "Please do the needful."
    조치를 취하다
  passed out = 졸업하다         "I passed out of IIT."
    (다른 변종에서는: 기절하다)
  revert = 회신하다             "Please revert at the earliest."
    (다른 변종에서는: revert = 되돌아가다)

문법 특징:
  • 상태동사에 진행형 사용:
    "I am having a car." (표준: "I have a car.")
  • "isn't it?" 또는 "no?"로 부가의문문:
    "You're coming, isn't it?" / "He's late, no?"
  • 강조를 위한 "itself":
    "Today itself we should finish." (= 오늘, 나중이 아니라)
  • 강조를 위한 반복:
    "Come come, sit sit." / "It was small small pieces."
```

## 8. 기타 변종

### 8.1 캐나다 영어

```
  • 철자: 영국식과 미국식 혼합
    (colour이지만 organize, centre이지만 realize)
  • 부가의문문으로 "eh?" 사용: "Nice day, eh?"
  • "about" → 비캐나다인에게 "aboot"으로 들림
  • "sorry" → "sore-ee"에 가깝게 발음
  • toque = 니트 모자
  • loonie/toonie = 1/2 달러 동전
```

### 8.2 남아프리카 영어

```
  • "just now" = 잠시 후에 (즉시가 아님!)
  • "now now" = 곧
  • "robot" = 신호등
  • "braai" = 바베큐
  • "howzit" = 안녕 / 어떻게 지내
  • 아프리칸스어와 아프리카 언어의 영향
```

### 8.3 싱가포르 영어 (Singlish)

```
  • 문장 끝에 "lah" 조사: "Can lah!"
  • "can" 또는 "cannot"을 완전한 대답으로 사용
  • 중국어, 말레이어, 타밀어의 영향
  • "Where got?" = "그건 사실이 아니야"
  • "steady lah" = 잘했어
```

## 9. 문화적 의사소통 규범

### 9.1 고맥락 vs 저맥락 의사소통

```
┌─────────────────────────────────────────────────────────────────┐
│   고맥락 (한국, 일본, 중국):                                      │
│   • 의미가 직접 진술되지 않고 암시됨                                │
│   • 맥락, 어조, 관계가 말보다 중요                                 │
│   • "눈치"가 중요                                                 │
│   • 간접적 거절: "그건 좀 어려울 수도..."                           │
│                                                                 │
│   저맥락 (미국, 독일, 호주):                                       │
│   • 의미가 명시적으로 진술됨                                       │
│   • 말이 직접 메시지를 전달                                        │
│   • "Say what you mean, mean what you say"                      │
│   • 직접적 거절: "No, I can't do that."                           │
│                                                                 │
│   중간 맥락 (영국):                                                │
│   • 미국보다 간접적이지만 한국보다는 직접적                          │
│   • 공손 전략과 완곡 표현 사용                                     │
│   • "That's an interesting idea"는 "동의하지 않아"일 수 있음        │
└─────────────────────────────────────────────────────────────────┘
```

### 9.2 국가별 문화 팁

```
미국:
  • 스몰토크가 중요 (날씨, 스포츠, 주말 계획)
  • 비즈니스에서도 빠르게 이름(first name) 사용
  • "How are you?"는 인사이지 진짜 질문이 아님
    답변: "Good, thanks!" (긴 답변 아님)
  • 직접적 표현을 중시 — 할 말을 하세요
  • 긍정적 피드백이 흔함 ("Great job!")

영국:
  • 완곡 표현과 아이러니가 흔함
  • "Not bad"는 종종 "꽤 좋다"를 의미
  • "I'll bear it in mind"는 "안 할 거야"를 의미할 수 있음
  • 줄 서기(queuing)는 신성시됨
  • 차(tea) 관련 사교 의례가 중요

호주:
  • 매우 캐주얼하고 비격식적인 문화
  • Tall poppy syndrome (잘난 체하지 않기)
  • 자기 비하 유머가 흔함
  • "How are you going?" = "잘 지내?"
  • Mateship (우정, 충성)이 중시됨
```

## 10. 직접적 표현 vs 간접적 표현

### 10.1 영국식 완곡 표현

```
영국인이 하는 말           vs    실제 의미:
─────────────────────────────────────────────────────
"That's quite good."               "훌륭하다."
"That's interesting."              "동의하지 않아 / 이상하네."
"I hear what you say."             "완전히 동의하지 않아."
"With the greatest respect..."     "틀렸다고 생각해."
"I'll bear it in mind."           "이미 잊어버렸어."
"That's a very brave proposal."    "미쳤군."
"I'm sure it's my fault."         "분명 네 잘못이야."
"I almost agree."                  "전혀 동의하지 않아."
"Could we consider alternatives?"  "네 아이디어는 안 돼."
"Not bad."                         "꽤 좋아 / 훌륭해."
```

### 10.2 영어에서의 한국식 의사소통 스타일

```
한국어 화자가 알아야 할 점:

1. "네"라고 하면서 "글쎄요"나 "아니요"를 의미하면
   영어 화자가 혼란스러워합니다. 더 명확하게:
   대신: "Yes..." (글쎄요의 의미로)
   이렇게: "I'll think about it." / "I'm not sure yet."

2. "아니요"를 직접 말하지 않으면 오해가 생길 수 있습니다:
   대신: 침묵이나 모호한 대답
   이렇게: "I'm afraid I can't." / "Unfortunately, that won't work."

3. 자기 비하가 오해될 수 있습니다:
   한국식: "My English is not good" (겸손)
   영어 화자는 이를 그대로 받아들입니다.
   더 나은 표현: "I'm still working on improving my English."

4. 지나친 사과는 진심이 없어 보일 수 있습니다:
   한국어 화자는 "죄송합니다"를 매우 자주 씁니다.
   영어에서 과도한 사과는 그 말의 무게를 약화시킵니다.
   적절할 때 "excuse me"나 "thank you"를 대신 사용하세요:
   대신: "Sorry for being late."
   이렇게: "Thank you for waiting."
```

### 연습 1: 미국식인가 영국식인가?

각 단어가 미국 영어(AmE)인지 영국 영어(BrE)인지 구별하세요.

1. colour

```
1. colour — BrE (AmE: color)
2. apartment — AmE (BrE: flat)
3. lorry — BrE (AmE: truck)
4. sidewalk — AmE (BrE: pavement)
5. holiday — BrE (AmE: vacation)
6. theater — AmE (BrE: theatre)
7. queue — BrE (AmE: line)
8. elevator — AmE (BrE: lift)
```

</details>

### 연습 2: AmE를 BrE로 (또는 그 반대로) 변환

1. I put the groceries in the trunk. (→ BrE)
2. She lives in a flat in London. (→ AmE)
3. Let's take the subway. (→ BrE)
4. He's wearing his new trainers. (→ AmE)
5. Do you want crisps or sweets? (→ AmE)

<details>
<summary>정답 보기</summary>

```
1. I put the groceries in the boot.
2. She lives in an apartment in London.
3. Let's take the underground / tube.
4. He's wearing his new sneakers.
5. Do you want chips or candy?
```

</details>

### 연습 3: 영국식 완곡 표현

영국인이 실제로 의미하는 바는 무엇일까요?

1. "That's a very interesting suggestion."
2. "I'm not entirely convinced."
3. "With the greatest respect, I think there may be another approach."
4. "Not bad at all."
5. "I'll bear that in mind."

<details>
<summary>정답 보기</summary>

```
1. "당신의 제안에 동의하지 않아요." / "이상하네요."
2. "강하게 반대합니다."
3. "당신이 틀렸고, 제가 더 나은 아이디어가 있습니다."
4. "사실 꽤 좋습니다."
5. "아마 그렇게 하지 않을 겁니다." / "잊어버릴 거예요."
```

</details>

### 연습 4: 문화적 인식

문화적 맥락에 가장 적절한 대응을 선택하세요.

1. 미국인 동료: "How are you?" — 어떻게 해야 할까요:
   a) 건강에 대해 자세히 대답한다
   b) "Good, thanks! And you?"라고 말한다
   c) 아무 말도 하지 않는다

2. 영국인 동료가 당신의 아이디어에 대해 "That's quite brave of you"라고 말합니다:
   a) "Thank you!"라고 열정적으로 말한다
   b) 위험하거나 어리석다고 생각할 수 있다고 고려한다
   c) 무시한다

3. 호주인 동료의 제안에 동의하지 않습니다:
   a) 아무 말도 하지 않는다 (한국식)
   b) "I think that's a bad idea"라고 말한다
   c) "That's a good point, but have you considered...?"라고 말한다

<details>
<summary>정답 보기</summary>

```
1. b) "Good, thanks! And you?" — 이것은 인사이지
   진짜 질문이 아닙니다. 짧고 긍정적으로 유지하세요.

2. b) 위험하거나 어리석다고 생각할 수 있다고 고려합니다.
   영국식 "brave"는 이 맥락에서 종종 "위험한" 또는
   "현명하지 못한"을 의미합니다.

3. c) "That's a good point, but have you considered...?"
   상대의 아이디어를 인정하면서 자신의 견해를 표현합니다.
   대부분의 영어권 업무 환경에서는 존중하면서
   직접적으로 말하는 것이 적절합니다.
```

</details>
