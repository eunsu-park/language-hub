## 1. 현재완료가 한국어 화자에게 어려운 이유

```
┌─────────────────────────────────────────────────────────────────┐
│   한국어 화자에게 가장 어려운 시제                                 │
│                                                                 │
│   한국어에는 현재완료 시제가 없습니다.                             │
│   한국어 과거시제 (-었-)가 두 가지를 모두 포함합니다:              │
│     "I ate lunch." (단순과거)                                    │
│     "I have eaten lunch." (현재완료)                              │
│   둘 다 한국어로 나는 점심을 먹었다!                              │
│                                                                 │
│   하지만 영어에서는 다릅니다:                                     │
│                                                                 │
│   "I ate lunch."          (단순과거)                              │
│   → 과거 사건에 초점, 완료됨, 구체적인 과거 시점                  │
│   → 나는 점심을 먹었다 (과거 사건에 초점)                         │
│                                                                 │
│   "I have eaten lunch."   (현재완료)                              │
│   → 과거 행동의 현재 결과에 초점                                  │
│   → 나는 점심을 먹은 상태이다 (현재 결과에 초점 — 배가 안 고프다) │
│                                                                 │
│   핵심 개념: 현재완료는 과거와 현재를 연결합니다.                  │
│   행동은 과거에 일어났지만, 그 결과나 관련성이 현재에 있습니다.    │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 현재완료 형성

### 2.1 구조

```
현재완료 = have/has + 과거분사

I          have eaten     (I've eaten)
you        have eaten     (you've eaten)
he/she/it  has eaten      (he's/she's/it's eaten)
we         have eaten     (we've eaten)
they       have eaten     (they've eaten)

부정문:
I have not (haven't) eaten.
She has not (hasn't) eaten.

의문문:
Have you eaten?
Has she eaten?
What have you done?

※ 과거분사 = 불규칙 동사표의 3번째 열
  규칙 동사: 과거분사 = 과거형과 동일 (-ed)
    walk → walked → walked
    play → played → played
  불규칙 동사: 종종 다름
    eat → ate → eaten
    go → went → gone
    see → saw → seen
    do → did → done
    write → wrote → written
```

### 2.2 주요 불규칙 과거분사

```
원형         과거형       과거분사

do          did         done
drink       drank       drunk

wear        wore        worn
write       wrote       written
```

## 3. 현재완료 용법

### 3.1 인생 경험 (경험)

```
"Have you ever been to Japan?"      일본에 가 본 적 있어?
"I have visited 10 countries."      나는 10개국을 방문한 적이 있다.
"She has never eaten sushi."        그녀는 초밥을 먹어 본 적이 없다.
"I have seen that movie three times." 나는 그 영화를 세 번 봤다.

※ 초점: 인생에서 지금까지 어느 시점에든
  언제인지는 중요하지 않음 — 경험이 중요합니다.
  "I've been to Japan." ✓ (경험이 존재)
  "I went to Japan in 2019." ✓ (구체적 과거 시점 → 단순과거)
  "I've been to Japan in 2019." ✗ (구체적 시점 → 단순과거를 사용!)
```

### 3.2 최근 완료 + 현재 결과

```
"I've finished my homework."        숙제를 끝냈다. (→ 지금 자유다)
"She's lost her keys."              열쇠를 잃어버렸다. (→ 지금 없다)
"He has broken his leg."            다리가 부러졌다. (→ 지금 깁스 중)
"I've eaten too much."              너무 많이 먹었다. (→ 지금 배부르다)
"The taxi has arrived."             택시가 도착했다. (→ 지금 밖에 있다)

※ 핵심: 과거 행동이 현재에 눈에 보이는 결과를 가짐.
  "I lost my keys." = 과거 사건, 그 후에 찾았을 수도 있음
  "I've lost my keys." = 지금도 없다 (현재 결과!)
```

### 3.3 아직 끝나지 않은 기간

```
"I have eaten three meals TODAY."    오늘 세 끼를 먹었다. (오늘 아직 안 끝남)
"She has written two books THIS YEAR." 올해 책을 두 권 썼다. (올해 아직 안 끝남)
"We haven't had any problems SO FAR." 지금까지 문제없었다.

※ "today", "this week", "this month", "this year" = 끝나지 않은 기간
  기간이 아직 진행 중이므로 현재완료를 사용합니다.

비교:
  "I ate three meals today." ✗ (오늘이 끝났다는 의미)
  "I've eaten three meals today." ✓ (오늘이 아직 진행 중)

  "She wrote two books last year." ✓ (작년 = 끝난 기간)
  "She has written two books this year." ✓ (올해 = 끝나지 않은 기간)
```

### 3.4 지금까지의 기간: since/for

```
"I have lived in Seoul FOR 5 years."   나는 서울에 5년 동안 살았다. (지금도 산다)
"She has worked here SINCE 2020."      그녀는 2020년부터 여기서 일했다. (지금도 일한다)
"We have been friends SINCE childhood." 우리는 어린 시절부터 친구였다. (지금도 친구)
"He has known her FOR a long time."    그는 그녀를 오래 알았다. (지금도 안다)

※ 현재완료 + since/for = 과거에 시작해서 지금도 계속
  "I lived in Seoul for 5 years." = 더 이상 그곳에 살지 않음 (끝남)
  "I have lived in Seoul for 5 years." = 아직도 그곳에 산다 (계속)
  이 구분은 한국어에는 존재하지 않습니다!
```

## 4. 현재완료 vs 단순과거

### 4.1 핵심 구분

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   단순과거                       현재완료                        │
│   ───────────                    ────────────────                │
│   완료된 과거 시간               현재와의 연결                    │
│   yesterday, last week, in 2020  today, this week, ever, never  │
│                                                                 │
│   "I ate lunch."                 "I've eaten lunch."            │
│   (언제? 아마 정오에)            (결과: 지금 배부르다)            │
│                                                                 │
│   "She went to Paris."           "She's been to Paris."         │
│   (과거의 구체적 여행)            (경험: 파리를 알고 있다)        │
│                                                                 │
│   "I lost my keys."              "I've lost my keys."           │
│   (과거 사건, 찾았을 수도)       (결과: 지금 없다)               │
│                                                                 │
│   "He lived in Seoul for 3 years." "He has lived in Seoul for   │
│   (더 이상 살지 않음)              3 years." (아직도 산다)       │
│                                                                 │
│   구체적 시간 → 단순과거          비구체적 시간 → 현재완료        │
│   yesterday, last week,          ever, never, already, yet,    │
│   in 2020, when I was young      just, so far, recently        │
│                                                                 │
│   ※ 현재완료를 구체적 과거 시간과 절대 함께 쓰지 마세요:        │
│     "I have eaten lunch yesterday." ✗✗✗                        │
│     "I ate lunch yesterday." ✓                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 시간 표지 규칙

```
단순과거 표지 (구체적이고 완료된 시간):
  yesterday, last night/week/month/year, ago, in 2020,
  on Monday (과거), when I was young, at 3 PM

현재완료 표지 (비구체적이거나 끝나지 않은 시간):
  ever, never, already, yet, just, recently, lately,
  so far, today, this week/month/year, since, for,
  before (인생에서), how long

  "I have been to Japan." ✓ (구체적 시간 없음 — 경험)
  "I went to Japan last year." ✓ (구체적 시간 — 과거)
  "I have been to Japan last year." ✗✗✗ (혼합 — 절대 이렇게 쓰지 마세요!)
```

### 4.3 대화 흐름: 완료 → 과거

```
자연스러운 대화는 종종 현재완료(일반)로 시작하고
단순과거(세부사항)로 전환합니다:

A: "Have you ever been to Korea?"     (현재완료 — 경험)
B: "Yes, I have."
A: "When did you go?"                 (단순과거 — 구체적 시간)
B: "I went last summer."              (단순과거 — 구체적 시간)
A: "What did you do there?"           (단순과거 — 구체적 과거 사건)
B: "I visited Seoul and Busan."       (단순과거)

※ 첫 질문: 일반적 경험 → 현재완료
  후속 세부사항: 언제, 어디서, 무엇을 → 단순과거
```

## 5. since vs for

### 5.1 차이점

```
┌─────────────────────────────────────────────────────────────────┐
│   since + 시점                     for + 기간                    │
│   ──────────────────────────       ─────────────────────         │
│   since 2020                       for 4 years                  │
│   since Monday                     for 3 days                   │
│   since January                    for 6 months                 │
│   since 9 AM                       for 5 hours                  │
│   since I was a child              for a long time              │
│   since she arrived                for ages                     │
│   since last summer                for 2 weeks                  │
│                                                                 │
│   since = 시작 시점 (부터)          for = 기간 길이 (동안)        │
│                                                                 │
│   "I've lived here since 2020."    = 2020년부터                 │
│   "I've lived here for 4 years."   = 4년 동안                   │
│   → 같은 의미, 다른 표현!                                       │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 흔한 오류

```
"I live here since 2020." ✗ → "I have lived here since 2020." ✓
(since/for + 기간 → 현재완료 또는 현재완료진행형을 반드시 사용!)

"I have lived here since 4 years." ✗ → "...for 4 years." ✓
(4 years = 기간 → for, since가 아님)

"I have lived here for 2020." ✗ → "...since 2020." ✓
(2020 = 시점 → since, for가 아님)
```

## 6. already, yet, just, still

### 6.1 already (이미/벌써)

```
위치: have와 과거분사 사이, 또는 문장 끝

"I've already eaten."                이미 먹었어.
"She has already finished."          그녀는 이미 끝냈어.
"He's already left."                 그는 벌써 떠났어.
"I've done it already."              이미 했어. (문장 끝 위치)

※ "already" = 예상보다 빠른, 완료를 강조
  긍정문에서 사용.
```

### 6.2 yet (아직)

```
위치: 문장 끝
사용: 부정문과 의문문

부정문:
"I haven't eaten yet."              아직 안 먹었어.
"She hasn't finished yet."          아직 안 끝냈어.
"He hasn't called yet."             아직 전화 안 했어.

의문문:
"Have you eaten yet?"               벌써 먹었어? / 아직 안 먹었어?
"Has she finished yet?"             벌써 끝났어?

※ "yet" = 이 순간까지, 하지만 일어날 것으로 예상
  "I didn't eat yet." ✗ (미국식 비격식은 가능하나, 문법적으로: 현재완료)
  "I haven't eaten yet." ✓
```

### 6.3 just (방금, 막)

```
위치: have와 과거분사 사이

"I've just eaten."                   방금 먹었어.
"She has just arrived."              그녀가 방금 도착했어.
"He's just left."                    그가 막 떠났어.
"We've just finished the meeting."   회의가 방금 끝났어.

※ "just" = 아주 짧은 시간 전에 (방금 전에)
  미국식에서는 "I just ate." (단순과거 + just)도 허용
  영국식에서는 "I've just eaten." (현재완료 + just)를 강하게 선호
```

### 6.4 still (아직도, 여전히)

```
위치: 조동사 앞 (haven't 앞)

"I still haven't finished."         아직도 안 끝냈어.
"She still hasn't called."          아직도 전화 안 했어.
"He still hasn't found a job."      아직도 일자리를 못 찾았어.

※ "still"은 예상했던 일이 아직 일어나지 않았음을 강조
  놀라움이나 좌절감을 나타냅니다.
  "I haven't finished yet." (중립적)
  "I still haven't finished." (너무 오래 걸렸다! 좌절감)
```

## 7. ever와 never

### 7.1 ever (경험 질문 — ~해 본 적 있니?)

```
위치: have와 과거분사 사이
사용: 의문문 (및 일부 특별한 패턴)

"Have you ever been to Japan?"       일본에 가 본 적 있어?
"Has she ever eaten Korean food?"    한국 음식을 먹어 본 적 있어?
"Have you ever seen a whale?"        고래를 본 적 있어?
"Is this the best movie you've ever seen?"  지금까지 본 최고의 영화야?

※ "ever" = 인생에서 어느 때든 (살면서 한 번이라도)
  의문문과 최상급 패턴에서만 사용, 일반 긍정문에서는 사용하지 않음.
  "I have ever been to Japan." ✗
  "I have been to Japan." ✓
```

### 7.2 never (절대 ~한 적 없다)

```
위치: have와 과거분사 사이

"I have never been to Africa."       아프리카에 가 본 적 없어.
"She has never eaten sushi."         초밥을 먹어 본 적 없어.
"He has never driven a car."         차를 운전해 본 적 없어.
"We've never met before."           전에 만난 적 없어요.

※ "never"는 그 자체로 부정 — "not"을 추가하지 마세요:
  "I haven't never been." ✗ (이중 부정)
  "I have never been." ✓
```

## 8. 과거완료: had + 과거분사

### 8.1 형성

```
과거완료 = had + 과거분사 (모든 주어 동일)

I had eaten           (I'd eaten)
she had eaten         (she'd eaten)
they had eaten        (they'd eaten)

부정문: had not (hadn't) + 과거분사
의문문: Had + 주어 + 과거분사?
```

### 8.2 과거완료를 사용하는 경우

```
┌─────────────────────────────────────────────────────────────────┐
│   과거완료 = "과거의 과거"                                        │
│   과거에 대해 이야기하면서 그보다 더 이전에                        │
│   일어난 일을 언급할 때 사용합니다.                               │
│                                                                 │
│   시간순:                                                        │
│   ──────────────X──────────────X──────────── 현재               │
│              과거완료          단순과거                            │
│              (더 이전)         (더 나중)                          │
│                                                                 │
│   "When I arrived, the movie had already started."              │
│   (내가 도착했을 때, 영화는 이미 시작한 상태였다.)                │
│   1번째: 영화 시작 (과거완료 — 더 이전)                          │
│   2번째: 내가 도착 (단순과거 — 더 나중)                          │
│                                                                 │
│   "She told me she had been to Korea."                          │
│   (그녀는 한국에 가 본 적 있다고 나에게 말했다.)                  │
│   1번째: 한국에 감 (과거완료 — 더 이전)                          │
│   2번째: 나에게 말함 (단순과거 — 더 나중)                        │
└─────────────────────────────────────────────────────────────────┘
```

### 8.3 과거완료 예시

```
1. 순서:
   "I had eaten before she arrived."
   (그녀가 도착하기 전에 이미 먹었다.)

2. 과거 특정 시점까지의 경험:
   "By 2020, I had visited 15 countries."
   (2020년까지 15개국을 방문한 상태였다.)

3. 과거 상황의 원인:
   "She was tired because she had worked all day."
   (하루 종일 일했기 때문에 피곤했다.)

4. 간접 화법:
   "He said he had finished."
   (그는 끝냈다고 말했다.)

5. 3차 조건문 (if + 과거완료 → would have):
   "If I had known, I would have helped."
   (알았더라면 도와줬을 텐데.)
```

### 8.4 과거완료 vs 단순과거

```
순서가 명확한 경우 (before/after 사용 시) 과거완료는 선택 사항:

  "She left before I arrived."           ✓ (순서가 명확)
  "She had left before I arrived."       ✓ (순서를 강조)

  "After he ate, he watched TV."         ✓ (순서가 명확)
  "After he had eaten, he watched TV."   ✓ (강조)

순서가 모호할 수 있는 경우 과거완료가 도움:

  "When I arrived, she left."
  → 내가 도착했을 때 떠났나? 아니면 그 전에?

  "When I arrived, she had left."
  → 명확: 내가 도착하기 전에 이미 떠난 상태 (내가 도착했을 때 이미 떠난 상태)
```

## 9. 한국어 화자의 흔한 실수

### 9.1 오류 차트

```
┌─────────────────────────────────────────────────────────────────┐
│   오류                             올바른 표현                    │
│   ─────                          ───────                        │
│   "I have been to Japan          "I went to Japan               │
│    last year."                    last year."                   │
│   (구체적 시간 → 단순과거)                                       │
│                                                                 │
│   "I lost my keys."              "I've lost my keys."           │
│   (현재 결과가 중요할 때)          (아직 없다)                    │
│                                                                 │
│   "I live here since 2020."      "I've lived here since 2020." │
│   (since/for → 현재완료)                                         │
│                                                                 │
│   "Did you ever eat sushi?"      "Have you ever eaten sushi?"  │
│   (인생 경험 → 현재완료)                                         │
│                                                                 │
│   "I have eaten lunch             "I ate lunch yesterday."      │
│    yesterday."                    (구체적 시간 → 단순과거)       │
│   ✗✗✗ 가장 흔한 오류                                            │
│                                                                 │
│   "I didn't eat yet."            "I haven't eaten yet."         │
│   (yet → 현재완료)                (표준/격식)                    │
│                                                                 │
│   "I have lived in Seoul          "I lived in Seoul             │
│    for 3 years."                   for 3 years."               │
│   (더 이상 살지 않는 경우)         (더 이상 살지 않음)            │
│   ※ 아직도 그곳에 산다면: "I have lived" ✓                      │
│     더 이상 살지 않는다면: "I lived" ✓                           │
└─────────────────────────────────────────────────────────────────┘
```

## 10. 연습 문제

### 연습 1: 현재완료 또는 단순과거?

올바른 시제를 고르세요.

```
1. I (went / have been) to Paris in 2019.
2. She (ate / has eaten) lunch already.
3. (Did you ever visit / Have you ever visited) Korea?
4. He (lost / has lost) his wallet. He can't find it.
5. I (lived / have lived) here since 2020. (아직도 여기에)
6. They (got / have got) married last June.
7. I (didn't finish / haven't finished) yet.
8. She (worked / has worked) at this company for 10 years. (아직도 일함)
9. We (met / have met) at a party in 2018.
10. This is the best book I (read / have ever read).
```

<details>
<summary>정답 보기</summary>

```
1. went (구체적 시간: in 2019)
2. has eaten (already → 현재완료)
3. Have you ever visited (인생 경험 → 현재완료)
4. has lost (현재 결과: 찾을 수 없다 → 현재완료)
5. have lived (since 2020, 아직도 여기에 → 현재완료)
6. got (구체적 시간: last June → 단순과거)
7. haven't finished (yet → 현재완료)
8. has worked (for 10 years, 아직도 일함 → 현재완료)
9. met (구체적 시간: in 2018 → 단순과거)
10. have ever read (최상급 + ever → 현재완료)
```

</details>

### 연습 3: already, yet, just, 또는 still?

```
1. Have you finished your homework ___?

4. He's ___ arrived — he walked in one minute ago.
5. I haven't seen that movie ___.
```

<details>
<summary>정답 보기</summary>

```
1. yet (의문문 → yet)
2. already (완료됨 → already)

4. just (매우 최근 → just)

```

</details>

### 연습 4: 과거완료

단순과거 또는 과거완료로 완성하세요.

```
1. When I arrived, the party ___ (already / start).
2. She ___ (never / see) snow before she ___ (move) to Canada.
3. After he ___ (finish) dinner, he ___ (watch) TV.
4. I ___ (not / eat) sushi before I ___ (go) to Japan.
5. By the time we ___ (arrive), the train ___ (already / leave).
```

<details>
<summary>정답 보기</summary>

```
1. When I arrived, the party had already started.
2. She had never seen snow before she moved to Canada.
3. After he had finished dinner, he watched TV.
4. I had not eaten sushi before I went to Japan.
5. By the time we arrived, the train had already left.
```

</details>

## 빠른 참조

```
┌─────────────────────────────────────────────────────────────────┐
│         현재완료 & 과거완료 빠른 가이드                            │
│                                                                 │
│   현재완료: have/has + 과거분사                                   │
│   용법: 경험(ever/never), 결과(아직 관련 있음),                  │
│         끝나지 않은 기간(today/this year), 기간(since/for)       │
│                                                                 │
│   ※ 구체적 과거 시간과 절대 함께 쓰지 않음 (yesterday, in 2020, ago) │
│   ※ 단순과거 = 구체적 시간의 완료된 과거 사건                    │
│   ※ 현재완료 = 현재와의 연결                                     │
│                                                                 │
│   since + 시점:  since 2020, since Monday, since 3 PM           │
│   for + 기간:    for 5 years, for 3 days, for a long time       │
│                                                                 │
│   already (이미) → 긍정: "I've already finished."                │
│   yet (아직) → 부정/의문: "Haven't finished yet."               │
│   just (방금) → 최근: "I've just eaten."                         │
│   still (아직도) → 좌절감: "I still haven't finished."           │
│   ever (한번이라도) → 의문: "Have you ever been?"                │
│   never (한번도 없다) → 부정: "I've never been."                 │
│                                                                 │
│   과거완료: had + 과거분사                                        │
│   = 두 과거 사건 중 더 이전의 것                                  │
│   "When I arrived, she had already left."                       │
└─────────────────────────────────────────────────────────────────┘
```
