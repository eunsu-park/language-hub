## 1. 영어 시제 개요

### 1.1 12시제 격자표

영어에는 **12가지 시제**가 있으며, 시간(과거/현재/미래)과 양상(단순/진행/완료/완료진행)으로 구분됩니다.

```
┌──────────────────┬───────────────┬───────────────┬──────────────────┐
│                  │    과거       │   현재        │    미래           │
│                  │    PAST       │   PRESENT     │    FUTURE        │
├──────────────────┼───────────────┼───────────────┼──────────────────┤
│ 단순             │ worked        │ work(s)       │ will work        │
│ SIMPLE           │ ate           │ eat(s)        │ will eat         │
├──────────────────┼───────────────┼───────────────┼──────────────────┤
│ 진행             │ was working   │ am/is/are     │ will be          │
│ CONTINUOUS       │ were eating   │ working       │ working          │
│                  │               │ am/is/are     │ will be          │
│                  │               │ eating        │ eating           │
├──────────────────┼───────────────┼───────────────┼──────────────────┤
│ 완료             │ had worked    │ have/has      │ will have        │
│ PERFECT          │ had eaten     │ worked        │ worked           │
│                  │               │ have/has      │ will have        │
│                  │               │ eaten         │ eaten            │
├──────────────────┼───────────────┼───────────────┼──────────────────┤
│ 완료진행         │ had been      │ have/has been │ will have been   │
│ PERFECT          │ working       │ working       │ working          │
│ CONTINUOUS       │ had been      │ have/has been │ will have been   │
│                  │ eating        │ eating        │ eating           │
└──────────────────┴───────────────┴───────────────┴──────────────────┘
```

### 1.2 양상(Aspect) 이해

```
단순 (SIMPLE):          사실이나 습관으로서의 행동
                        "I eat breakfast."

진행 (CONTINUOUS):      진행 중인 행동 (일시적, 계속되는)
                        "I am eating breakfast."

완료 (PERFECT):         기준 시점 이전에 완료된 행동으로,
                        해당 시점과의 관련성을 나타냄
                        "I have eaten breakfast." (그래서 지금 배가 안 고프다)

완료진행               기준 시점 이전에 시작되어
(PERFECT CONTINUOUS):  그 시점까지(또는 가까이) 계속된 행동으로,
                        지속 기간을 강조
                        "I have been eating for 30 minutes."
```

## 2. 현재 시제

### 2.1 단순 현재 (Simple Present)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: 기본형 동사 (he/she/it에는 +s/es)                        │
│                                                                 │
│   I work          We work                                       │
│   You work        You work                                      │
│   He/She works    They work                                     │
│                                                                 │
│   부정:  do not (don't) + 기본형 동사                             │
│          does not (doesn't) + 기본형 동사 (he/she/it)            │
│   의문:  Do/Does + 주어 + 기본형 동사?                            │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 습관과 일과:             "I drink coffee every morning."
  2. 사실과 일반적 진리:       "Water boils at 100°C."
  3. 예정된 일정:             "The train leaves at 9:00 AM."
  4. 상태와 조건:             "She lives in Seoul."
  5. 실황 중계 (스포츠 등):   "He shoots! He scores!"

신호 단어: always, usually, often, sometimes, rarely, never,
            every day/week/month, on Mondays
```

### 2.2 현재 진행 (Present Continuous)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: am/is/are + 동사-ing                                     │
│                                                                 │
│   I am working         We are working                           │
│   You are working      You are working                          │
│   He/She is working    They are working                         │
│                                                                 │
│   부정:  am/is/are + not + 동사-ing                              │
│   의문:  Am/Is/Are + 주어 + 동사-ing?                            │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 지금 일어나고 있는 행동:      "I am reading a book."
  2. 일시적인 상황:               "She is staying with friends."
  3. 추세와 변화:                 "Prices are rising."
  4. 미래의 약속/계획:            "We are meeting tomorrow at 3."
  5. 짜증나는 습관 (always와 함께): "He is always complaining!"

신호 단어: now, right now, at the moment, currently, today,
            this week, these days, Look!, Listen!

비진행 동사 (상태 동사) — 진행형으로 사용 불가:
  ✗ I am knowing the answer.     ✓ I know the answer.
  ✗ She is believing in ghosts.  ✓ She believes in ghosts.
  ✗ I am wanting coffee.         ✓ I want coffee.

  상태 동사 목록: know, believe, want, need, prefer, like, love,
  hate, understand, remember, forget, belong, own, possess,
  seem, appear, mean, consist, contain, deserve
```

### 2.3 현재 완료 (Present Perfect)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: have/has + 과거분사                                      │
│                                                                 │
│   I have worked         We have worked                          │
│   You have worked       You have worked                         │
│   He/She has worked     They have worked                        │
│                                                                 │
│   부정:  have/has + not + 과거분사                                │
│   의문:  Have/Has + 주어 + 과거분사?                              │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 인생 경험 (ever/never):
     "Have you ever been to Paris?"

  2. 불특정 시점에 완료된 행동:
     "I have read that book." (언제인지는 중요하지 않음)

  3. 최근에 완료된 행동:
     "She has just finished her homework."

  4. 과거부터 현재까지 이어지는 상황:
     "I have lived here for 10 years." (여전히 여기 살고 있음)

  5. 현재 보이는 결과:
     "Someone has broken the window!" (지금 깨져 있음)

신호 단어: already, yet, just, ever, never, recently, so far,
            since, for, up to now, still (부정), lately
```

### 2.4 현재 완료 진행 (Present Perfect Continuous)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: have/has + been + 동사-ing                               │
│                                                                 │
│   I have been working         We have been working              │
│   You have been working       You have been working             │
│   He/She has been working     They have been working            │
│                                                                 │
│   부정:  have/has + not + been + 동사-ing                        │
│   의문:  Have/Has + 주어 + been + 동사-ing?                      │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 진행 중인 활동의 기간 (얼마나 오래 했는지 강조):
     "I have been studying English for 3 years."

  2. 최근 활동의 가시적 결과:
     "You're sweating. Have you been running?"

  3. 현재까지의 기간 동안 반복된 행동:
     "She has been calling you all morning."

신호 단어: for, since, all day/morning/week, how long,
            recently, lately

현재 완료 vs 현재 완료 진행:
  "I have written 3 emails."     (결과에 초점: 3개 완성)
  "I have been writing emails."  (활동에 초점: 아직 하는 중)
```

## 3. 과거 시제

### 3.1 단순 과거 (Simple Past)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: 동사 + -ed (규칙) 또는 불규칙 형태                        │
│                                                                 │
│   I worked / ate        We worked / ate                         │
│   You worked / ate      You worked / ate                        │
│   He/She worked / ate   They worked / ate                       │
│                                                                 │
│   부정:  did not (didn't) + 기본형 동사                           │
│   의문:  Did + 주어 + 기본형 동사?                                │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 특정 시점에 완료된 행동:
     "I visited Jeju Island last summer."

  2. 연속된 완료 행동:
     "I woke up, brushed my teeth, and ate breakfast."

  3. 과거의 습관 (used to):
     "When I was young, I played soccer every day."

  4. 과거의 상태:
     "She was a teacher for 20 years."

신호 단어: yesterday, last week/month/year, ago, in 2020,
            when I was young, at that time, then
```

### 3.2 과거 진행 (Past Continuous)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: was/were + 동사-ing                                      │
│                                                                 │
│   I was working         We were working                         │
│   You were working      You were working                        │
│   He/She was working    They were working                       │
│                                                                 │
│   부정:  was/were + not + 동사-ing                               │
│   의문:  Was/Were + 주어 + 동사-ing?                             │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 과거 특정 시점에 진행 중이던 행동:
     "At 8 PM, I was watching TV."

  2. 중단된 행동 (배경):
     "I was sleeping when the phone rang."

  3. 과거의 두 가지 동시 행동:
     "While she was cooking, he was cleaning."

  4. 이야기에서 배경 묘사:
     "The sun was shining and birds were singing."

신호 단어: while, when, as, at that time, at 3 o'clock,
            all day yesterday
```

### 3.3 과거 완료 (Past Perfect)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: had + 과거분사                                           │
│                                                                 │
│   I had worked         We had worked                            │
│   You had worked       You had worked                           │
│   He/She had worked    They had worked                          │
│                                                                 │
│   부정:  had not (hadn't) + 과거분사                              │
│   의문:  Had + 주어 + 과거분사?                                   │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 다른 과거 행동보다 먼저 완료된 행동:
     "When I arrived, the movie had already started."

  2. 과거 시점 이전의 경험:
     "I had never seen snow before I went to Canada."

  3. 제3 조건문 (비현실 과거):
     "If I had known, I would have helped."

  4. 간접화법 (시제 후퇴):
     "He said he had finished the project."

신호 단어: before, after, by the time, already, just,
            never (before), until, when
```

### 3.4 과거 완료 진행 (Past Perfect Continuous)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: had + been + 동사-ing                                    │
│                                                                 │
│   I had been working         We had been working                │
│   You had been working       You had been working               │
│   He/She had been working    They had been working              │
│                                                                 │
│   부정:  had not (hadn't) + been + 동사-ing                      │
│   의문:  Had + 주어 + been + 동사-ing?                           │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 다른 과거 행동 이전의 활동 기간:
     "I had been waiting for 2 hours when the bus finally came."

  2. 과거 상태의 원인 (눈에 보이는 증거):
     "She was tired because she had been working all night."

  3. 과거 시점 이전의 반복 행동:
     "He had been calling her for days before she answered."

신호 단어: for, since, all day, before, when, by the time,
            how long
```

## 4. 미래 시제

### 4.1 단순 미래 (Simple Future)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태:                                                         │
│   (a) will + 기본형 동사 (예측, 결정, 약속)                       │
│   (b) am/is/are going to + 기본형 동사 (계획, 증거가 있는 예측)   │
│                                                                 │
│   will:                                                        │
│   I will work         We will work                              │
│   부정: will not (won't) + 기본형 동사                            │
│   의문: Will + 주어 + 기본형 동사?                                │
│                                                                 │
│   going to:                                                    │
│   I am going to work  We are going to work                     │
│   부정: am/is/are + not + going to + 기본형 동사                  │
│   의문: Am/Is/Are + 주어 + going to + 기본형 동사?                │
└─────────────────────────────────────────────────────────────────┘

will vs going to:
  즉흥적 결정:        "I'll have the pasta." (지금 결정)
  계획된 의도:        "I'm going to visit my parents." (이미 계획)
  예측 (의견):        "I think it will rain tomorrow."
  예측 (증거):        "Look at those clouds. It's going to rain."
  약속:               "I will always love you."
  제안:               "I'll carry that for you."

신호 단어: tomorrow, next week/month/year, in the future,
            soon, later, tonight
```

### 4.2 미래 진행 (Future Continuous)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: will be + 동사-ing                                       │
│                                                                 │
│   I will be working         We will be working                  │
│   You will be working       You will be working                 │
│   He/She will be working    They will be working                │
│                                                                 │
│   부정:  will not (won't) be + 동사-ing                          │
│   의문:  Will + 주어 + be + 동사-ing?                            │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 미래의 특정 시점에 진행 중인 행동:
     "At 10 PM tonight, I will be studying."

  2. 당연히 일어날 미래 행동:
     "I'll be seeing John tomorrow. (이미 정해진 것)"

  3. 공손한 문의:
     "Will you be using the car tonight?"

신호 단어: at this time tomorrow, at 8 PM, all day tomorrow,
            this time next week
```

### 4.3 미래 완료 (Future Perfect)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: will have + 과거분사                                     │
│                                                                 │
│   I will have worked         We will have worked                │
│   You will have worked       You will have worked               │
│   He/She will have worked    They will have worked              │
│                                                                 │
│   부정:  will not (won't) have + 과거분사                        │
│   의문:  Will + 주어 + have + 과거분사?                          │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 미래 시점 이전에 완료될 행동:
     "By next June, I will have graduated."

  2. 미래 시점까지의 기간:
     "By December, I will have lived here for 5 years."

신호 단어: by (+ 미래 시점), by the time, before,
            by next week/month/year
```

### 4.4 미래 완료 진행 (Future Perfect Continuous)

```
┌─────────────────────────────────────────────────────────────────┐
│   형태: will have been + 동사-ing                                │
│                                                                 │
│   I will have been working        We will have been working     │
│   You will have been working      You will have been working    │
│   He/She will have been working   They will have been working   │
│                                                                 │
│   부정:  will not have been + 동사-ing                           │
│   의문:  Will + 주어 + have been + 동사-ing?                     │
└─────────────────────────────────────────────────────────────────┘

용법:
  1. 미래 시점까지의 활동 기간 (얼마나 오래 했는지 강조):
     "By 2027, I will have been working here for 10 years."

  2. 미래 상태의 원인:
     "She will be tired because she will have been traveling
      for 20 hours."

신호 단어: by, by the time, for, since (미래 참조와 함께)
```

## 5. 시제 타임라인 다이어그램

### 5.1 현재 시제 타임라인

```
                        현재(NOW)
      과거 ─────────────|───────────── 미래
                        │
  단순 현재:             ─────────────────────── (습관/사실)
  "I work every day."   ▲  ▲  ▲  ▲  ▲  ▲  ▲

  현재 진행:                  ████              (지금 진행 중)
  "I am working."            └NOW┘

  현재 완료:            ═══════════►│          (과거부터 현재까지
  "I have worked here       시작  NOW          연결)
   for 5 years."

  현재 완료 진행:       ≈≈≈≈≈≈≈≈≈≈≈►│         (현재까지의 기간)
  "I have been working          시작  NOW
   for 3 hours."
```

### 5.2 과거 시제 타임라인

```
                        현재(NOW)
      과거 ─────────────|───────────── 미래
              │
  단순 과거:       ■                           (완료된 사건)
  "I worked         │
   yesterday."    어제

  과거 진행:       ████■                      (진행 중에
  "I was working         │                     중단됨)
   when he called."   전화 올 때

  과거 완료:    ═══►│  │                      (다른 과거 사건
  "I had finished    │  │                      이전)
   before he came." 끝남 옴

  과거 완료     ≈≈≈≈►│ │                     (다른 과거 사건
  진행:              │  │                     이전의 기간)

### 5.3 미래 시제 타임라인

## 6. 시제 선택 가이드

### 6.1 의사결정 흐름도

```
┌─ 행동이 과거, 현재, 미래 중 언제인가? ──────────────────────────┐
│                                                                 │
├─ 현재 ────────────────────────────────────────────────────────┤
│  ├─ 습관/사실/일과?            → 단순 현재                     │
│  ├─ 지금 일어나고 있는 중?      → 현재 진행                     │
│  ├─ 과거 행동이 현재와 연결?    → 현재 완료                     │
│  │  (결과가 중요)                                              │
│  └─ 현재까지의 기간?           → 현재 완료 진행                 │
│     (얼마나 오래?)                                              │
│                                                                 │
├─ 과거 ────────────────────────────────────────────────────────┤
│  ├─ 특정 과거 시점에 완료?      → 단순 과거                     │
│  ├─ 과거 시점에 진행 중?        → 과거 진행                     │
│  ├─ 다른 과거 사건 이전?        → 과거 완료                     │
│  └─ 다른 과거 사건 이전의       → 과거 완료 진행                │
│     기간?                                                      │
│                                                                 │
├─ 미래 ────────────────────────────────────────────────────────┤
│  ├─ 예측/계획/약속?            → 단순 미래                     │
│  ├─ 미래 시점에 진행 중?        → 미래 진행                     │
│  ├─ 미래 마감 이전에 완료?      → 미래 완료                     │
│  └─ 미래 시점까지의 기간?       → 미래 완료 진행                │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 빠른 참조 질문표

| 자문해 보기... | 사용할 시제 |
|---------------|------------|
| 규칙적인 습관이나 사실인가? | 단순 현재 (Simple Present) |
| 지금 일어나고 있는가? | 현재 진행 (Present Continuous) |
| 일어났는데 (언제인지 모르거나 상관없는가)? | 현재 완료 (Present Perfect) |
| 얼마나 오래 계속되고 있는가? | 현재 완료 진행 (Present Perfect Continuous) |
| 특정 과거 시점에 일어났는가? | 단순 과거 (Simple Past) |
| 다른 일이 발생했을 때 진행 중이었는가? | 과거 진행 (Past Continuous) |
| 다른 과거 사건 이전에 일어났는가? | 과거 완료 (Past Perfect) |
| 그 전에 얼마나 오래 진행되고 있었는가? | 과거 완료 진행 (Past Perfect Continuous) |
| 일어날 것인가? | 단순 미래 (Simple Future) |
| 미래의 어떤 시점에 진행 중일 것인가? | 미래 진행 (Future Continuous) |
| 미래의 마감 전에 완료될 것인가? | 미래 완료 (Future Perfect) |
| 그때까지 얼마나 오래 진행되고 있을 것인가? | 미래 완료 진행 (Future Perfect Continuous) |

## 7. 시제 대조

### 7.1 단순 현재 vs 현재 진행

```
단순 현재 (영구적/습관적):
  "I live in Seoul."        (영구적인 집)
  "She works at a bank."    (정규직)
  "I drink coffee."         (일반적인 습관)

현재 진행 (일시적/현재):
  "I'm living in Seoul."    (일시적, 이사할 수도 있음)
  "She's working at a bank." (일시적, 이번 여름)
  "I'm drinking coffee."    (바로 지금, 이 순간)
```

### 7.2 단순 과거 vs 현재 완료

```
단순 과거 (특정 시점, 완료):
  "I visited Paris in 2019."          (특정 시점: 2019)
  "I ate lunch."                       (완료, 오늘 일찍)
  "She lived in London for 3 years."   (지금은 거기 안 삶)

현재 완료 (불특정 시점 또는 계속):
  "I have visited Paris."              (인생에서 어느 시점에)
  "I have eaten lunch."                (결과: 배가 안 고프다)
  "She has lived in London for 3 years." (아직도 거기 살고 있음)
```

### 7.3 현재 완료 vs 현재 완료 진행

```
현재 완료 (결과/완료에 초점):
  "I have written 5 emails."    (이메일 5개 완성 — 결과)
  "She has read the book."      (다 읽었다)

현재 완료 진행 (활동/기간에 초점):
  "I have been writing emails." (활동 진행 중, 아직 안 끝남)
  "She has been reading."       (활동, 기간 강조)
```

### 7.4 Will vs Going To

```
Will (즉흥적, 예측, 약속):
  "I'll answer the phone."     (즉흥적 결정)
  "It will probably rain."     (의견에 기반한 예측)
  "I'll help you."             (약속/제안)

Going to (계획, 증거에 기반한 예측):
  "I'm going to study abroad." (계획된 결정)
  "Look! It's going to rain."  (증거에 기반한 예측)
  "We're going to move."       (의도, 이미 결정됨)
```

### 7.5 단순 과거 vs 과거 진행

```
단순 과거 (완료된 행동, 주요 사건):
  "I ate dinner at 7 PM."          (완료됨)
  "The phone rang."                (단일 사건)

과거 진행 (배경, 중단된 행동):
  "I was eating dinner at 7 PM."   (그 시간에 진행 중)
  "I was cooking when the phone rang." (배경 행동)
```

## 9. 한국어 화자의 흔한 오류

### 9.1 한국어에는 영어와 같은 시제 표지가 없다

```
┌─────────────────────────────────────────────────────────────────┐
│   한국어 시제 체계는 훨씬 단순합니다:                               │
│                                                                 │
│   한국어는 3가지 기본 시제만 있습니다:                               │
│   • 현재: -ㄴ다/-는다  (간다, 먹는다)                               │
│   • 과거: -았/-었        (갔다, 먹었다)                             │
│   • 미래: -겠-/-ㄹ 것이다 (가겠다, 먹을 것이다)                     │
│                                                                 │
│   영어에는 12가지 시제가 있습니다!                                   │
│   한국어에 없는 것들:                                               │
│   • 진행 양상 (am/is/are + -ing)                                   │
│   • 완료 양상 (have/has + 과거분사)                                 │
│   • 완료 진행                                                      │
│                                                                 │
│   이것이 한국어 화자들이 영어에서 종종 단순 시제만 사용하고,           │
│   진행형과 완료형을 무시하는 이유입니다.                               │
└─────────────────────────────────────────────────────────────────┘
```

### 9.2 가장 흔한 10가지 오류

| # | 오류 | 올바른 표현 | 설명 |
|---|------|------------|------|
| 1 | ✗ I live here since 2020. | ✓ I **have lived** here since 2020. | "since"는 현재 완료 필요 |
| 2 | ✗ I am knowing the answer. | ✓ I **know** the answer. | "know"는 상태 동사 |
| 3 | ✗ I have been to Paris last year. | ✓ I **went** to Paris last year. | 특정 시점 = 단순 과거 |
| 4 | ✗ When I will arrive, I call you. | ✓ When I **arrive**, I **will call** you. | 시간절에는 "will" 불가 |
| 5 | ✗ I am work now. | ✓ I **am working** now. | 진행형에는 -ing 필요 |
| 6 | ✗ Did you ever eat sushi? | ✓ **Have** you **ever eaten** sushi? | 인생 경험 = 현재 완료 |
| 7 | ✗ I will going to study. | ✓ I **am going to** study. / I **will** study. | will + going to 혼용 금지 |
| 8 | ✗ Before he came, I finish. | ✓ Before he came, I **had finished**. | 더 앞선 과거 = 과거 완료 |
| 9 | ✗ She is study English for 3 years. | ✓ She **has been studying** English for 3 years. | 현재까지의 기간 = 현재 완료 진행 |
| 10 | ✗ I eat breakfast already. | ✓ I **have already eaten** breakfast. | "already"는 현재 완료 신호 |

### 9.3 "Since/For" 문제

```
한국어 화자들은 "since"와 "for"에 종종 단순 현재를 사용합니다:

  ✗ I study English for 5 years.
  ✓ I have studied English for 5 years.
  ✓ I have been studying English for 5 years.

  ✗ She lives here since 2018.
  ✓ She has lived here since 2018.

규칙:
  for + 기간    (for 3 hours, for 2 years, for a long time)
  since + 시점  (since Monday, since 2020, since I was born)
  상황이 현재까지 계속될 때, 둘 다 현재 완료 또는 현재 완료 진행이 필요
```

### 연습 2: 올바른 시제 선택하기

1. I (live / have lived) in Seoul since 2015.
2. She (reads / is reading) a book right now.
3. We (went / have gone) to Busan last weekend.
4. By the time you arrive, I (will finish / will have finished) cooking.
5. They (were watching / watched) TV when the earthquake (happened / was happening).

4. will have finished (by the time + 미래 완료)
5. were watching / happened
   (배경 행동 = 과거 진행; 주요 사건 = 단순 과거)
6. works (정규직 = 단순 현재)
7. have never seen (인생 경험 = 현재 완료)

### 연습 3: 오류 수정하기

각 문장에 시제 오류가 있습니다. 수정하세요.

1. I am knowing the answer to this question.
2. She works here since last January.
3. When I will get home, I will call you.
4. They have went to the park yesterday.
5. I am study English every day.
6. He had been living in London since he was born. He still lives there.
7. By next month, I will be working here for 10 years.

<details>
<summary>정답 보기</summary>

```
1. I know the answer to this question.
   ("know"는 상태 동사 — 진행형 불가)

2. She has worked / has been working here since last January.
   ("since"는 현재 완료 필요)

3. When I get home, I will call you.
   ("when" 시간절에는 "will" 사용 불가)

4. They went to the park yesterday.
   ("yesterday" = 특정 시점 = 단순 과거, 현재 완료 아님)

5. I study English every day.
   ("every day" = 단순 현재, 진행형 아님; "am study"도 문법적으로
    오류 — 진행형이면 "am studying"이어야 함)

6. He has been living in London since he was born. He still lives there.
   (현재까지 유효 = 현재 완료 진행, 과거 완료 아님)
