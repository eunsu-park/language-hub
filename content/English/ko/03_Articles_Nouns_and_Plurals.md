## 1. 왜 관사가 한국어 화자에게 어려운가

```
┌─────────────────────────────────────────────────────────────────┐
│   한국어 화자에게 가장 어려운 문법 포인트 1위                      │
│                                                                 │
│   한국어에는 관사가 전혀 없습니다:                                │
│   "나는 사과를 먹는다" → I eat apple? an apple? the apple?         │
│                                                                 │
│   한국어에서 사과는 항상 그냥 사과입니다.                          │
│   영어에서는 반드시 선택해야 합니다:                               │
│                                                                 │
│   a/an  → "I eat AN apple."    (아무 사과, 처음 언급)             │
│   the   → "I eat THE apple."   (특정 사과, 어떤 것인지 앎)         │
│   (없음)→ "I eat apples."      (사과 일반)                        │
│                                                                 │
│   영어 원어민은 이 선택을 하루에 수백 번 자동으로 합니다.            │
│   한국어 화자는 이것을 배워야 합니다.                              │
│                                                                 │
│   ※ 고급 수준의 한국어 화자도 관사 실수를 합니다.                 │
│     이 레슨은 정확도를 위한 가장 중요한 레슨입니다.                │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 부정관사: a/an

### 2.1 a/an을 사용하는 경우

```
a/an을 사용하는 경우:

1. 처음 언급 — 새로운 것을 소개할 때:
   "I saw A dog in the park."     (듣는 사람이 어떤 개인지 아직 모름)
   처음 언급할 때 사용

2. 여러 개 중 하나 — 그룹의 아무 멤버:
   "She is A teacher."            (세상에 많은 선생님 중 한 명)
   여러 개 중 하나를 말할 때

3. 분류 — 종류/유형을 말할 때:
   "This is A smartphone."        (물건을 분류)
   종류를 말할 때

4. ~당의 의미:
   "I exercise three times A week."  (= per week, 주당)
   "$5 A kilogram"                   (= per kilogram, 킬로당)
```

### 2.2 a vs an

```
┌─────────────────────────────────────────────────────────────────┐
│   a  → 자음 소리 앞에                                           │
│   an → 모음 소리 앞에                                           │
│                                                                 │
│   ※ 철자가 아니라 소리에 따라 결정!                              │
│                                                                 │
│   a book        (b = 자음 소리)                                  │
│   a car         (c = 자음 소리)                                  │
│   a dog         (d = 자음 소리)                                  │
│   a university  (u = /juː/ 자음 /j/ 소리로 시작!)                │
│   a European    (E = /jʊ/ 자음 /j/ 소리로 시작!)                 │
│   a one-way     (o = /wʌ/ 자음 /w/ 소리로 시작!)                 │
│                                                                 │
│   an apple      (a = 모음 소리)                                  │
│   an egg        (e = 모음 소리)                                  │
│   an hour       (h가 묵음! /aʊər/ 모음으로 시작)                  │
│   an honest man (h가 묵음! /ˈɒnɪst/ 모음으로 시작)               │
│   an MBA        (M = /em/ 모음 /e/ 소리로 시작!)                  │
│   an X-ray      (X = /eks/ 모음 /e/ 소리로 시작!)                 │
│                                                                 │
│   ※ 한국어 화자의 흔한 실수:                                     │
│     "a umbrella" ✗ → "an umbrella" ✓                            │
│     "an university" ✗ → "a university" ✓                        │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 직업과 신분에 a/an 사용

```
영어에서 단수 직업명 앞에는 a/an이 필수입니다:
  한국어: 나는 학생이다.        (관사 불필요)
  영어: I am A student.         (관사 필수)

  She is A doctor.             ✓
  She is doctor.               ✗ (오류 — "a"가 있어야 함)
  He is AN engineer.           ✓
  He is engineer.              ✗ (오류)

  ※ 이것은 한국어 화자가 가장 자주 하는 실수 중 하나입니다.
    BE 동사 뒤에 단수 가산명사가 올 때는 항상 a/an을 사용하세요.
```

## 3. 정관사: the

### 3.1 the를 사용하는 경우

```
화자와 청자 모두 어떤 것인지 알 때 "the"를 사용합니다:

1. 두 번째 언급 — 이미 소개된 것:
   "I saw a dog. THE dog was big."
   (두 번째 언급 — 아까 말한 그 개)

2. 유일한 것 — 하나뿐인 것:
   "The sun is bright."        (태양은 하나뿐)
   "The moon is full tonight." (달도 하나뿐)
   "The President of Korea"    (대통령은 한 명)

3. 공유된 지식 — 둘 다 어떤 것인지 아는 경우:
   "Can you close THE door?"   (이 방의 문 — 둘 다 앎)
   "Where is THE bathroom?"    (가장 가까운 화장실)

4. 최상급과 서수:
   "She is THE best student."  (최고의 학생)
   "This is THE first lesson." (첫 번째 수업)

5. of/in/that절로 한정된 경우:
   "The capital OF Korea"      (한국의 수도)
   "The book THAT you gave me" (네가 준 그 책)
```

### 3.2 "the"의 발음

```
자음 소리 앞: "the" = /ðə/ (약하게, '더'처럼)
  the book /ðə bʊk/
  the car /ðə kɑːr/
  the dog /ðə dɒg/

모음 소리 앞: "the" = /ðiː/ (강하게, '디'처럼)
  the apple /ðiː ˈæpəl/
  the egg /ðiː eg/
  the other /ðiː ˈʌðər/

강조할 때: "the" = /ðiː/ 항상
  "This is THE /ðiː/ restaurant." (이게 바로 그 식당이야!)
```

### 3.3 지명과 고유명사에 the 사용

```
"the"를 사용하는 경우:              "the"를 사용하지 않는 경우:
  the United States                   Korea (the Korea가 아님)
  the United Kingdom                  Japan
  the Philippines                     France
  the Netherlands                     Seoul
  the Pacific Ocean                   Mount Everest
  the Han River                       Lake Michigan
  the Sahara Desert                   Central Park
  the Alps (산맥)                     Broadway
  the New York Times (신문)            Harvard University

기억하기 쉬운 규칙:
  복수 국가명 → the (the Philippines, the Netherlands)
  "Republic/Kingdom/States" → the (the Republic of Korea)
  대양, 바다, 강 → the
  개별 산, 호수 → the 없음
  대학교 → 보통 the 없음 (Harvard, Seoul National)
```

## 4. 무관사 (관사 없음)

### 4.1 관사를 사용하지 않는 경우

```
1. 복수 명사의 일반적 의미:
   "Dogs are loyal."           (개는 충성스럽다 — 일반적인 개)
   "Books are expensive."      (책은 비싸다 — 일반적인 책)
   ※ "The dogs are loyal"이 아님 (특정 개가 아닌 한)

2. 불가산명사의 일반적 의미:
   "Water is essential."       (물은 필수적이다 — 일반적인 물)
   "Music makes me happy."     (음악은 나를 행복하게 한다)
   "Information is power."     (정보는 힘이다)

3. 식사:
   "I eat breakfast at 7."     (아침을 7시에 먹는다)
   "Let's have lunch."         (점심 먹자)
   ※ 단, "The breakfast was delicious." (특정 식사)

4. 언어:
   "I speak Korean."           (나는 한국어를 한다)
   "English is important."     (영어는 중요하다)

5. 스포츠:
   "I play soccer."            (나는 축구를 한다)
   "She likes tennis."         (그녀는 테니스를 좋아한다)

6. 학과목:
   "I study mathematics."      (나는 수학을 공부한다)
   "History is interesting."   (역사는 재미있다)

7. 요일, 월:
   "I work on Monday."         (나는 월요일에 일한다)
   "It's cold in January."     (1월에는 춥다)
```

## 5. 관사 결정 트리

### 5.1 마스터 결정 트리

```
┌─────────────────────────────────────────────────────────────────┐
│                    관사 결정 트리                                  │
│                                                                 │
│   명사가 가산명사인가?                                            │
│   │                                                             │
│   ├── 예 (가산)                                                  │
│   │   │                                                         │
│   │   ├── 단수?                                                  │
│   │   │   │                                                     │
│   │   │   ├── 청자가 아는가? ──예──→ THE book                    │
│   │   │   │                                                     │
│   │   │   └── 모르거나 처음 언급? ──→ A/AN book                  │
│   │   │                                                         │
│   │   └── 복수?                                                  │
│   │       │                                                     │
│   │       ├── 특정 그룹? ──예──→ THE books (on the table)        │
│   │       │                                                     │
│   │       └── 일반적/전체? ──→ ∅ books (Books are great.)        │
│   │                                                             │
│   └── 아니오 (불가산)                                            │
│       │                                                         │
│       ├── 특정? ──예──→ THE water (in the glass)                 │
│       │                                                         │
│       └── 일반적? ──→ ∅ water (Water is essential.)              │
│                                                                 │
│   절대 안 됨: a/an + 불가산   (a water ✗, an information ✗)      │
│   절대 안 됨: a/an + 복수     (a books ✗, an apples ✗)           │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 빠른 예시

| 상황 | 관사 | 예시 | 한국어 |
|------|------|------|--------|
| 처음 언급, 단수 | a/an | "I bought **a** book." | 책을 한 권 샀다 |
| 알려진/특정 | the | "I read **the** book." | 그 책을 읽었다 |
| 일반적, 복수 | (없음) | "I like **∅** books." | 나는 책을 좋아한다 |
| 일반적, 불가산 | (없음) | "I drink **∅** water." | 나는 물을 마신다 |
| 특정, 불가산 | the | "**The** water is cold." | 그 물이 차갑다 |
| 직업명, 단수 | a/an | "She is **a** nurse." | 그녀는 간호사이다 |

## 6. 가산명사 vs 불가산명사

### 6.1 개념

```
┌─────────────────────────────────────────────────────────────────┐
│   가산명사 (셀 수 있는 명사):                                     │
│   셀 수 있음: one book, two books, three books                   │
│   단수형과 복수형 있음                                            │
│   a/an 사용 가능: a book, an apple                               │
│                                                                 │
│   불가산명사 (셀 수 없는 명사):                                   │
│   개별적으로 셀 수 없음                                           │
│   복수형 없음                                                    │
│   a/an 사용 불가                                                 │
│                                                                 │
│   ※ 한국어에는 이 구분이 없습니다!                               │
│     물은 물 한 잔, 물 두 잔 (수량사 '잔' 사용)으로 셀 수 있지만    │
│     영어: water (불가산) — "two waters" 불가                      │
│     대신: "two glasses of water"                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 흔한 불가산명사

```
액체 & 음식:       water, milk, coffee, tea, bread, rice, sugar, meat
재료:              wood, metal, glass, paper, gold, plastic
추상 개념:          information, advice, knowledge, news, happiness,
                   love, music, art, work, money, time, weather
학문:              mathematics, physics, economics, research
기타:              furniture, luggage, equipment, traffic, homework

※ 이것들은 영어에서 모두 단수입니다:
  "The news IS good."        ("are"가 아님)    뉴스는 좋다
  "The information IS useful." ("are"가 아님)   정보는 유용하다
  "The furniture IS new."    ("are"가 아님)     가구는 새것이다
```

### 6.3 불가산명사 세기

```
"수량사" 표현을 사용합니다 (한국어의 수량사 개, 잔, 장 등과 비슷):

a glass of water           물 한 잔
a cup of coffee            커피 한 잔
a piece of information     정보 한 가지
a piece of advice          조언 한 가지
a slice of bread           빵 한 조각
a piece of furniture       가구 한 점
a piece of luggage         짐 한 개
a bar of chocolate         초콜릿 한 개
a sheet of paper           종이 한 장

Two glasses of water       물 두 잔
Three pieces of advice     조언 세 가지
```

### 6.4 가산과 불가산 모두 가능한 명사

```
일부 명사는 가산/불가산에 따라 의미가 바뀝니다:

불가산 (재료/일반)                    가산 (특정 항목)
───────────────────────────────   ──────────────────────────
glass (유리)                      a glass (유리잔)
paper (종이)                      a paper (논문/신문)
light (빛)                        a light (조명기구)
room (공간)                       a room (방)
time (시간)                       a time (횟수) "three times"
chicken (닭고기)                   a chicken (닭 한 마리)
coffee (커피 — 물질)               a coffee (커피 한 잔 — 비격식)
experience (경험 — 추상)           an experience (경험 — 사건)
hair (머리카락 — 전체)              a hair (머리카락 한 올)
```

## 7. 규칙 복수형

### 7.1 기본 규칙

```
┌─────────────────────────────────────────────────────────────────┐
│                  복수형 만들기 규칙                                │
│                                                                 │
│   규칙 1: 대부분의 명사 → -s 추가                                │
│   book → books    cat → cats    dog → dogs    table → tables    │
│                                                                 │
│   규칙 2: -s, -ss, -sh, -ch, -x, -z로 끝나면 → -es 추가        │
│   bus → buses     class → classes    dish → dishes              │
│   watch → watches  box → boxes       quiz → quizzes             │
│                                                                 │
│   규칙 3: 자음 + y로 끝나면 → y를 -ies로 변경                    │
│   city → cities   baby → babies    story → stories              │
│   ※ 모음 + y → 그냥 -s 추가: boy → boys, day → days             │
│                                                                 │
│   규칙 4: -f/-fe로 끝나면 → -ves로 변경 (예외 많음)              │
│   knife → knives   wife → wives    leaf → leaves                │
│   life → lives     wolf → wolves   half → halves                │
│   ※ 예외: roof → roofs, chief → chiefs, belief → beliefs        │
│                                                                 │
│   규칙 5: -o로 끝나면 → -es 추가(일부) 또는 -s 추가(일부)        │
│   tomato → tomatoes  potato → potatoes  hero → heroes           │
│   ※ 음악/외래어: piano → pianos, photo → photos                 │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 -s/-es 복수형의 발음

```
복수형 어미는 3가지 발음이 있습니다:

/s/ — 무성 자음 뒤: /p/, /t/, /k/, /f/, /θ/
  books /bʊks/    cats /kæts/     cups /kʌps/
  roofs /ruːfs/   months /mʌnθs/

/z/ — 유성 자음과 모음 뒤: /b/, /d/, /g/, /v/, /m/, /n/, /l/ 등
  dogs /dɒgz/     beds /bedz/     rooms /ruːmz/
  days /deɪz/     trees /triːz/   boys /bɔɪz/

/ɪz/ — 치찰음 뒤: /s/, /z/, /ʃ/, /ʒ/, /tʃ/, /dʒ/
  buses /ˈbʌsɪz/     boxes /ˈbɒksɪz/     dishes /ˈdɪʃɪz/
  watches /ˈwɒtʃɪz/  judges /ˈdʒʌdʒɪz/   roses /ˈroʊzɪz/

※ 한국어 화자는 모든 복수형에 /ɪ/를 추가하는 경향이 있습니다: "books" ≠ "bookseuh"
  주의해서 들으세요: "cats"는 1음절 /kæts/이지, 2음절이 아닙니다!
```

## 8. 불규칙 복수형

### 8.1 흔한 불규칙 복수형

```
모음 변화:
  man → men           woman → women         foot → feet
  tooth → teeth       goose → geese         mouse → mice
  louse → lice

-en 어미:
  child → children    ox → oxen

단복수 동형 (단수와 복수가 같은 형태):
  sheep → sheep       fish → fish           deer → deer
  aircraft → aircraft series → series       species → species
  means → means

외래어 복수형:
  criterion → criteria     phenomenon → phenomena
  analysis → analyses      thesis → theses
  crisis → crises          datum → data
  cactus → cacti/cactuses  focus → foci/focuses
  fungus → fungi           alumnus → alumni
  formula → formulae/formulas

-is → -es:
  basis → bases        diagnosis → diagnoses
  oasis → oases        hypothesis → hypotheses
```

### 8.2 항상 복수형인 명사

```
이 명사들은 항상 복수형입니다 ("is"가 아니라 "are" 사용):

쌍으로 된 물건:
  scissors (가위)      glasses (안경)       pants (바지)
  jeans (청바지)        shorts (반바지)      trousers (바지, 영국식)
  binoculars (쌍안경)   headphones (헤드폰)

  "These scissors ARE sharp."  ("is"가 아님)
  "My pants ARE new."          ("is"가 아님)

  셀 때: "a pair of scissors", "two pairs of pants"

기타 항상 복수형:
  clothes (옷)         stairs (계단)        goods (상품)
  belongings (소지품)   savings (저축)       surroundings (주변환경)
```

## 9. 지시대명사: this/that/these/those

### 9.1 네 가지 지시대명사

```
┌─────────────────────────────────────────────────────────────────┐
│              지시대명사                                           │
│                                                                 │
│              단수              복수                               │
│   가까이     this (이것)       these (이것들)                     │
│   멀리       that (저것)       those (저것들)                     │
│                                                                 │
│   "This" / "These" = 화자에게 가까이                              │
│   "That" / "Those" = 화자에서 멀리                                │
│                                                                 │
│   대명사로 사용:                                                  │
│   "This is my book."           이것은 내 책이다                   │
│   "That is your bag."          저것은 네 가방이다                  │
│   "These are my friends."      이 사람들은 내 친구들이다           │
│   "Those are expensive."       저것들은 비싸다                    │
│                                                                 │
│   형용사로 사용 (명사 앞):                                        │
│   "This book is mine."         이 책은 내 것이다                  │
│   "That car is fast."          저 차는 빠르다                    │
│   "These shoes are new."       이 신발들은 새것이다               │
│   "Those people are Korean."   저 사람들은 한국인이다              │
└─────────────────────────────────────────────────────────────────┘
```

### 9.2 시간에서의 this/that

```
this = 현재 또는 다가오는:
  "this morning"    (오늘 아침)
  "this week"       (이번 주)
  "this year"       (올해)
  "this Friday"     (이번 금요일)

that = 과거 또는 먼:
  "that day"        (그 날)
  "that time"       (그때)
  "in those days"   (그 시절에)
```

### 9.3 전화 통화에서의 this/that

```
전화에서:
  "Hello, this is Minjun."      (자신을 밝힐 때)
  "Is that Sarah?"              (상대방이 누구인지 물을 때)

  ※ "This" = 나 (화자)
    "That" = 당신 (청자)

직접 소개할 때:
  "This is my friend Sujin."    (옆에 있는 사람)
  "That is Professor Lee."      (멀리 있는 사람)
```

### 9.4 한국어와의 비교

```
한국어 지시대명사:
  이 (this) — 화자에게 가까이
  그 (that) — 청자에게 가까이 또는 이전에 언급됨
  저 (that over there) — 둘 다에게 멀리

영어 지시대명사:
  this/these — 화자에게 가까이
  that/those — 화자에서 멀리 (한국어의 그와 저를 모두 포함)

  ※ 영어에는 한국어의 '그' (청자에게 가까운 것)에 해당하는 것이 없습니다.
    "That"이 그와 저를 모두 커버합니다.

  한국어 그 책 (언급한 그 책) → 영어: "that book" 또는 "the book"
  한국어 저 산 (저 멀리 산) → 영어: "that mountain"
```

## 10. 연습 문제

### 연습 1: a, an, the, 또는 ∅ (없음)?

빈칸에 올바른 관사를 채우세요 (무관사인 경우 빈칸으로 두세요).

```
1. I am ___ student.
2. ___ sun is very hot today.
3. She likes ___ music.
4. I bought ___ umbrella.
5. ___ water is essential for life.
6. He is ___ honest man.
7. I go to ___ school every day.
8. ___ books on the table are mine.
9. She is ___ best player on the team.
10. I had ___ breakfast at 7 AM.
11. Can you pass me ___ salt?
12. ___ dogs are friendly animals.
13. I saw ___ movie last night. ___ movie was great.
14. She plays ___ piano very well.
15. He works at ___ university.
```

<details>
<summary>정답 보기</summary>

```
1. I am A student.                     (직업명, 단수 가산명사)
2. THE sun is very hot today.          (유일한 것 — 태양은 하나뿐)
3. She likes ∅ music.                  (일반적, 불가산)
4. I bought AN umbrella.              (처음 언급, 모음 소리)
5. ∅ Water is essential for life.      (일반적, 불가산)
6. He is AN honest man.               (h가 묵음 → 모음 소리 /ˈɒnɪst/)
7. I go to ∅ school every day.         (기관 일반)
8. THE books on the table are mine.    (특정 — "on the table"로 한정)
9. She is THE best player on the team. (최상급 → 항상 "the")
10. I had ∅ breakfast at 7 AM.         (식사 — 일반적)
11. Can you pass me THE salt?          (특정 — 이 테이블의 소금)
12. ∅ Dogs are friendly animals.       (일반적 범주)
13. I saw A movie last night. THE movie was great. (처음 언급 → 두 번째 언급)
14. She plays THE piano very well.     (악기에는 "the" 사용)
15. He works at A university.          (u = /juː/ 자음 소리 → "a")
```

</details>

### 연습 5: 오류 수정

다음 문장에서 관사 오류를 찾아 수정하세요.

```
1. She is teacher.
2. I need a information.
3. The dogs are great pets. (의미: 개 일반)
4. I bought a umbrella.
5. He goes to the school every day. (의미: 그는 학생)
6. I drink the water every morning. (의미: 물 일반)
7. She has the long hair.
8. I ate the breakfast at 8 AM. (의미: 일상적 습관)
```

<details>
<summary>정답 보기</summary>

```
1. She is A teacher.                    (단수 직업명 앞에 a/an 필요)
2. I need ∅ information. / I need A PIECE OF information.
                                        (information은 불가산 — "a" 불가)
3. ∅ Dogs are great pets.              (일반적 → "the" 불필요)
4. I bought AN umbrella.               (모음 소리 → "an")
5. He goes to ∅ school every day.      (기관 → "the" 불필요)
6. I drink ∅ water every morning.      (일반적 → "the" 불필요)
7. She has ∅ long hair.                (일반적 특성 → "the" 불필요)
8. I ate ∅ breakfast at 8 AM.          (식사 → "the" 불필요)
```

</details>

```
┌─────────────────────────────────────────────────────────────────┐
│            관사 & 복수형 빠른 가이드                               │
│                                                                 │
│   a/an  = 처음 언급, 아무 하나, 분류                              │
│           a + 자음 소리 / an + 모음 소리                          │

│   불가산: 셀 수 없음 (water, information, furniture)               │

│   복수형: -s (books), -es (boxes), -ies (cities)                 │

│   지시대명사:                                                     │
│   가까이: this (단수) / these (복수)                               │
│   멀리:   that (단수) / those (복수)                               │
│                                                                 │
│   ※ 한국어 화자 오류 1위: 관사를 완전히 빠뜨리기                   │

```
