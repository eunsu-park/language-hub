## 1. 어휘 구축 소개

### 1.1 체계적 어휘 학습이 중요한 이유

```
┌─────────────────────────────────────────────────────────────────┐
│   영어 원어민은 약 20,000~35,000개의 단어군을 알고 있습니다.         │
│                                                                 │
│   효과적인 의사소통을 위해 학습자에게 필요한 것:                      │
│   • 일상 대화를 위한 약 3,000개 단어군                              │
│   • 뉴스와 미디어 이해를 위한 약 5,000~6,000개                      │
│   • 원본 텍스트 읽기를 위한 약 8,000~9,000개                        │
│   • 학술 및 전문적 맥락을 위한 약 10,000개 이상                      │
│                                                                 │
│   전략: 고립된 단어-번역 쌍을 외우지 말고                            │
│   문맥과 단어군을 통해 학습하세요.                                   │
│                                                                 │
│   단어군 예시:                                                    │
│   employ → employer, employee, employment, unemployment,         │
│           employable, unemployable, self-employed                 │
│   하나의 어근 = 7개 이상의 단어 학습!                                │
└─────────────────────────────────────────────────────────────────┘
```

## 6. 단어 형성: 접두사

### 6.1 부정 접두사

```
un-    (아닌, 반대의)
  happy → unhappy        usual → unusual
  fair → unfair          do → undo

in-    (아닌) — 특정 글자 앞에서 il-, im-, ir-로 변함
  active → inactive      visible → invisible
  legal → illegal        mature → immature
  regular → irregular    responsible → irresponsible
  complete → incomplete  patient → impatient

dis-   (아닌, 반대의, 분리)
  agree → disagree       appear → disappear
  connect → disconnect   advantage → disadvantage
  honest → dishonest     like → dislike

non-   (아닌)
  fiction → nonfiction   sense → nonsense
  profit → nonprofit    violent → nonviolent

mis-   (잘못, 나쁘게)
  understand → misunderstand    lead → mislead
  spell → misspell              use → misuse
  judge → misjudge              behave → misbehave
```

### 6.2 긍정 및 방향 접두사

```
re-    (다시, 되돌려)
  do → redo              write → rewrite
  build → rebuild        start → restart
  cycle → recycle        use → reuse

pre-   (이전에)
  view → preview         heat → preheat
  historic → prehistoric war → prewar
  pay → prepay           school → preschool

post-  (이후에)
  war → postwar          modern → postmodern
  graduate → postgraduate

over-  (너무 많이)
  eat → overeat          work → overwork
  load → overload        react → overreact
  sleep → oversleep      estimate → overestimate

under- (너무 적게, 아래에)
  estimate → underestimate      pay → underpay
  value → undervalue            line → underline
  ground → underground          age → underage

out-   (이상으로, 넘어서)
  do → outdo             perform → outperform
  number → outnumber     grow → outgrow
  run → outrun           live → outlive
```

### 6.3 크기와 정도 접두사

```
super- (위에, 넘어서)    superhero, supernatural, supervise
sub-   (아래에)          subway, subtitle, subconscious
semi-  (절반, 부분적)     semicircle, semifinal, semicolon
multi- (많은)            multilingual, multimedia, multinational
mono-  (하나, 단일)       monolingual, monopoly, monotone
bi-    (둘)              bilingual, bicycle, biweekly
tri-   (셋)              triangle, trilingual, trilogy
inter- (사이에)          international, interact, internet
trans- (가로질러)        transport, translate, transform
co-    (함께)            cooperate, coexist, coworker
anti-  (반대)            antibody, antisocial, antibiotic
auto-  (스스로)          automatic, autobiography, autonomous
```

## 7. 단어 형성: 접미사

### 7.1 명사 접미사

```
-tion / -sion (행동, 상태)
  educate → education          decide → decision
  inform → information         express → expression
  communicate → communication  conclude → conclusion

-ment (결과, 행동)
  develop → development        achieve → achievement
  manage → management          agree → agreement
  improve → improvement        govern → government

-ness (상태, 질)
  happy → happiness           dark → darkness
  kind → kindness             aware → awareness
  sad → sadness               weak → weakness

-ity / -ty (상태, 질)
  real → reality              creative → creativity
  electric → electricity      able → ability
  diverse → diversity         secure → security

-er / -or (~하는 사람)
  teach → teacher             act → actor
  drive → driver              invent → inventor
  lead → leader               visit → visitor

-ist (신념/기술을 가진 사람)
  science → scientist         art → artist
  tour → tourist              piano → pianist
  journal → journalist        novel → novelist

-ance / -ence (상태, 질)
  perform → performance       exist → existence
  assist → assistance         differ → difference
  import → importance         depend → dependence
```

### 7.2 형용사 접미사

```
-able / -ible (~할 수 있는)
  afford → affordable         access → accessible
  rely → reliable             flex → flexible
  comfort → comfortable       response → responsible

-ful (~로 가득 찬)
  hope → hopeful              care → careful
  beauty → beautiful          success → successful
  power → powerful            peace → peaceful

-less (~이 없는)
  hope → hopeless             care → careless
  home → homeless             use → useless
  end → endless               meaning → meaningless

-ous / -ious / -eous (~의 성질을 가진)
  danger → dangerous          ambition → ambitious
  courage → courageous        religion → religious
  fame → famous               mystery → mysterious

-al / -ial (~에 관련된)
  nature → natural            finance → financial
  music → musical             industry → industrial
  person → personal           commerce → commercial

-ive (~의 특성을 가진)
  act → active                create → creative
  attract → attractive        effect → effective
  impress → impressive        expense → expensive
```

### 7.3 동사 접미사

```
-ize / -ise (~하게 하다, ~이 되다)
  modern → modernize          organ → organize
  standard → standardize      real → realize
  special → specialize        custom → customize

-ify (~하게 하다, ~를 유발하다)
  simple → simplify           intense → intensify
  class → classify            beauty → beautify
  just → justify              identity → identify

-en (~하게 하다, ~이 되다)
  wide → widen                deep → deepen
  strength → strengthen       short → shorten
  dark → darken               bright → brighten
```

### 7.4 부사 접미사

```
-ly (~한 방식으로)
  quick → quickly             careful → carefully
  happy → happily             serious → seriously
  unfortunate → unfortunately easy → easily
```

### 8.3 모르는 단어 해독하는 법

```
예: "telecommunication"
  tele (먼) + com (함께) + munic (공유) + ation (명사)
  = 먼 거리에서 정보를 공유함

예: "autobiography"
  auto (스스로) + bio (생명) + graph (쓰다) + y (명사)
  = 자신의 삶에 대해 쓰는 것

예: "unprecedented"
  un (아닌) + pre (이전에) + ced (가다) + ent (형용사) + ed
  = 이전에 간 적이 없는 = 이전에 일어난 적이 없는

예: "biodegradable"
  bio (생명) + de (아래로) + grad (단계) + able (~할 수 있는)
  = 살아있는 유기체에 의해 분해될 수 있는
```

## 9. 기타 단어 구성 전략

### 9.1 합성 (Compounding)

```
두 단어를 결합하여 새로운 단어 형성:

명사 + 명사:     airport, bedroom, newspaper, bookshelf
형용사 + 명사:   blackboard, greenhouse, highway, software
명사 + 동사:     sunrise, rainfall, headache, earthquake
동사 + 명사:     washing machine, swimming pool, driving license
형용사 + 형용사: bittersweet, well-known, red-hot
```

### 9.2 전환 (Zero Derivation)

```
형태를 바꾸지 않고 단어를 다른 품사로 사용:

명사 → 동사:     "Email me." / "Google it." / "Bookmark this page."
동사 → 명사:     "a run" / "a swim" / "a try" / "a look"
형용사 → 동사:   "clean the room" / "dry your hair" / "empty the box"
형용사 → 명사:   "the poor" / "the rich" / "the elderly"
```

### 9.3 혼성어 (Blending)

```
두 단어의 일부분을 결합:

brunch    = breakfast + lunch
smog      = smoke + fog
internet  = inter(national) + net(work)
email     = electronic + mail
blog      = web + log
podcast   = iPod + broadcast
frenemy   = friend + enemy
```

### 9.4 절단어 (Clipping)

```
긴 단어를 줄이기:

exam      = examination
lab       = laboratory
ad        = advertisement
flu       = influenza
phone     = telephone
gym       = gymnasium
fridge    = refrigerator
info      = information
memo      = memorandum
photo     = photograph
```

## 10. 한국어 화자를 위한 참고사항

### 10.1 동족어와 가짜 동족어 (Cognates and False Friends)

```
많은 영어 단어가 한국어에 외래어(콩글리시)로 존재하지만,
일부는 의미가 변했습니다:

  아르바이트 (arbeit) → 파트타임 일
    영어: "part-time job" 또는 "gig"

  핸들 (handle) → 운전대
    영어: "steering wheel" (handle은 문/가방 손잡이)

  노트북 (notebook) → 노트북 컴퓨터
    영어: "laptop" (notebook은 글쓰기용 종이 공책)

  컨닝 (cunning) → 시험 부정행위
    영어: "cheating" (cunning은 교활한/영리한이라는 뜻)

  서비스 (service) → 무료 증정
    영어: "complimentary" 또는 "on the house"

  원룸 (one room) → 원룸 아파트
    영어: "studio apartment" 또는 "studio"

  오바이트 (overeat) → 구토
    영어: "throw up" 또는 "vomit" (overeat = 과식하다)
```

### 10.2 단어 형성의 차이

```
한국어는 영어가 라틴어/그리스어 어근을 사용하는 곳에서
한자어 합성어를 자주 사용합니다:

  교육 (教育) = education (educ + ation)
  경제 (經濟) = economy (eco + nomy)
  민주 (民主) = democracy (demo + cracy)
  과학 (科學) = science (scient + ia)

이 대응 관계를 이해하면 도움이 됩니다:
  한국어 한자어 = 영어 라틴어/그리스어 어근
  둘 다 의미 있는 구성 요소로 복잡한 단어를 만듭니다!
```

### 연습 4: 단어 해독하기

어근, 접두사, 접미사에 대한 지식을 사용하여 의미를 파악하세요.

1. biodegradable
2. international
3. mispronunciation
4. unprecedented
5. telecommunication

<details>
<summary>정답 보기</summary>

```
1. biodegradable: bio (생명) + de (아래) + grad (단계) + able
   = 살아있는 유기체에 의해 분해될 수 있는

2. international: inter (사이) + nation + al (~에 관련된)
   = 국가 간의 활동에 관련된

3. mispronunciation: mis (잘못) + pro (앞으로) + nunci (알리다) + ation
   = 잘못된 발음

4. unprecedented: un (아닌) + pre (이전에) + ced (가다) + ent + ed
   = 이전에 일어나거나 존재한 적이 없는

5. telecommunication: tele (먼) + com (함께) + munic (공유) + ation
   = 먼 거리에서의 통신
```

</details>
