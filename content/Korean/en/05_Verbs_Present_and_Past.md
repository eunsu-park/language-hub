## 1. Korean Verb Basics

### 1.1 Dictionary Form (-다)

Every Korean verb in its base (dictionary) form ends in **-다** (da). To conjugate a verb, you remove -다 to get the **verb stem**, then add the appropriate ending.

> **Korean Verb Structure**
>
> Dictionary Form = Verb Stem + 다
>
> - 가다 (to go) → 가 (stem) + 다
> - 먹다 (to eat) → 먹 (stem) + 다
> - 공부하다 (to study) → 공부하 (stem) + 다
> - 읽다 (to read) → 읽 (stem) + 다
>
> Conjugation = Stem + Ending
>
> - 가 + -아요 → 가요 (go -- polite)
> - 먹 + -어요 → 먹어요 (eat -- polite)
> - 가 + -ㅂ니다 → 갑니다 (go -- formal)
> - 먹 + -습니다 → 먹습니다 (eat -- formal)

### 1.2 Key Properties of Korean Verbs

1. **No conjugation for person or number**: 가요 means "I go," "you go," "he/she goes," "we go," "they go" — all the same
2. **Tense is marked on the verb**: 가요 (go/present), 갔어요 (went/past), 갈 거예요 (will go/future)
3. **Politeness is marked on the verb**: 가 (casual), 가요 (polite), 갑니다 (formal)
4. **Verbs always come last** in the sentence
5. **No articles needed**: 책을 읽어요 = "I read a/the book"

### 1.3 Action Verbs vs Descriptive Verbs

Korean has two types of "verbs":

| Type | Korean Term | Examples | English Equivalent |
|------|-----------|---------|-------------------|
| Action verbs (동사) | 동작 동사 | 가다, 먹다, 하다 | Verbs (go, eat, do) |
| Descriptive verbs (형용사) | 상태 동사 | 크다, 좋다, 예쁘다 | Adjectives (big, good, pretty) |

> **Important**: In Korean, adjectives function as verbs. "크다" literally means "to be big." They conjugate like verbs: 커요 (is big), 컸어요 (was big). This is covered fully in Lesson 06.

## 2. Vowel Harmony

### 2.1 The Rule

Vowel harmony (모음 조화, moeum joha) determines whether you add **-아** or **-어** endings. Look at the **last vowel** of the verb stem:

> **Vowel Harmony Rule**
>
> If the last vowel of the stem is ㅏ or ㅗ (bright/양성모음):
> → Use -아 endings
>
> If the last vowel is ANYTHING ELSE (dark/음성모음):
> → Use -어 endings
>
> Special case: 하다 verbs
> → 하 + 여 → 해 (하다 contracts to 해)
>
> **Summary:**
> - ㅏ, ㅗ → -아 (가다 → 가 + 아요, 오다 → 오 + 아요)
> - Everything else → -어 (먹다 → 먹 + 어요, 마시다 → 마시 + 어요)
> - 하다 → -해 (공부하다 → 공부해요)

### 2.2 Contraction Rules

When the stem vowel and the ending vowel meet, they often **contract**:

| Stem Ending | + Ending | Result | Example |
|------------|---------|--------|---------|
| ㅏ | + 아 | ㅏ (no change — they merge) | 가 + 아요 → 가요 |
| ㅗ | + 아 | ㅘ | 오 + 아요 → 와요 |
| ㅜ | + 어 | ㅝ | 배우 + 어요 → 배워요 |
| ㅣ | + 어 | ㅕ | 마시 + 어요 → 마셔요 |
| ㅡ | + 어 | depends on preceding vowel | 쓰 + 어요 → 써요 |
| ㅓ | + 어 | ㅓ (merge) | 서 + 어요 → 서요 |
| ㅐ | + 어 | ㅐ (merge) | 보내 + 어요 → 보내요 |

> **Tip**: Contraction is about making pronunciation easier. Say the vowels quickly and listen to how they naturally merge.

## 3. Present Tense Informal Polite (-아/어요)

### 3.1 Formation

The **해요체** (informal polite) is the most commonly used speech level in everyday Korean.

```
Stem (last vowel ㅏ/ㅗ) + 아요
Stem (other vowels)     + 어요
하다 stems              → 해요
```

### 3.2 Conjugation Examples

| Dictionary Form | Stem | Last Vowel | Ending | Conjugated | Meaning |
|----------------|------|-----------|--------|-----------|---------|
| 가다 (to go) | 가 | ㅏ | + 아요 | **가요** (가+아요 contracts) | go |
| 오다 (to come) | 오 | ㅗ | + 아요 | **와요** (오+아요 contracts) | come |
| 보다 (to see) | 보 | ㅗ | + 아요 | **봐요** (보+아요 contracts) | see |
| 먹다 (to eat) | 먹 | ㅓ | + 어요 | **먹어요** | eat |
| 읽다 (to read) | 읽 | ㅣ | + 어요 | **읽어요** | read |
| 마시다 (to drink) | 마시 | ㅣ | + 어요 | **마셔요** (시+어→셔) | drink |
| 쓰다 (to write) | 쓰 | ㅡ | + 어요 | **써요** (ㅡ drops, + 어요) | write |
| 자다 (to sleep) | 자 | ㅏ | + 아요 | **자요** (자+아요 contracts) | sleep |
| 일어나다 (to get up) | 일어나 | ㅏ | + 아요 | **일어나요** | get up |
| 하다 (to do) | 하 | — | → 해요 | **해요** | do |
| 공부하다 (to study) | 공부하 | — | → 해요 | **공부해요** | study |

### 3.3 Step-by-Step Process

```
How to conjugate 마시다 (to drink):

Step 1: Remove -다 → 마시 (stem)
Step 2: Last vowel of stem = ㅣ (not ㅏ/ㅗ) → use -어요
Step 3: 마시 + 어요
Step 4: Contract: 시 + 어 → 셔
Step 5: Result = 마셔요
```

## 4. Present Tense Formal (-ㅂ니다/-습니다)

### 4.1 Formation

The **합쇼체** (formal polite) is used in formal situations, news, presentations, and with strangers.

> **Formal Polite Present Tense (-ㅂ니다 / -습니다)**
>
> If stem ends in a **VOWEL**: stem + ㅂ니다
> - 가 → 갑니다 (ㅂ attaches to 가 → 갑)
> - 하 → 합니다
> - 마시 → 마십니다
>
> If stem ends in a **CONSONANT**: stem + 습니다
> - 먹 → 먹습니다
> - 읽 → 읽습니다
> - 있 → 있습니다

### 4.2 Conjugation Examples

| Dictionary Form | Stem | Ends In | Formal Present |
|----------------|------|---------|---------------|
| 가다 | 가 | Vowel | 갑니다 |
| 오다 | 오 | Vowel | 옵니다 |
| 보다 | 보 | Vowel | 봅니다 |
| 하다 | 하 | Vowel | 합니다 |
| 마시다 | 마시 | Vowel | 마십니다 |
| 먹다 | 먹 | Consonant | 먹습니다 |
| 읽다 | 읽 | Consonant | 읽습니다 |
| 자다 | 자 | Vowel | 잡니다 |
| 있다 | 있 | Consonant | 있습니다 |
| 쓰다 | 쓰 | Vowel | 씁니다 |

### 4.3 Formal Questions: -ㅂ니까? / -습니까?

To make a formal question, change -다 to -까:

| Statement | Question |
|-----------|----------|
| 갑니다 (go) | 갑니까? (do you go?) |
| 먹습니다 (eat) | 먹습니까? (do you eat?) |
| 합니다 (do) | 합니까? (do you do?) |
| 있습니다 (exist) | 있습니까? (is there?) |

## 5. Past Tense Informal Polite (-았/었어요)

### 5.1 Formation

The past tense is formed by inserting **-았/었-** between the stem and the ending:

```
Stem (last vowel ㅏ/ㅗ) + 았어요
Stem (other vowels)     + 었어요
하다 stems              → 했어요
```

### 5.2 Conjugation Examples

| Dictionary Form | Stem | Past Tense (해요체) | Process |
|----------------|------|-------------------|---------|
| 가다 (to go) | 가 | **갔어요** | 가 + 았어요 → 갔어요 |
| 오다 (to come) | 오 | **왔어요** | 오 + 았어요 → 왔어요 |
| 보다 (to see) | 보 | **봤어요** | 보 + 았어요 → 봤어요 |
| 먹다 (to eat) | 먹 | **먹었어요** | 먹 + 었어요 |
| 읽다 (to read) | 읽 | **읽었어요** | 읽 + 었어요 |
| 마시다 (to drink) | 마시 | **마셨어요** | 마시 + 었어요 → 마셨어요 |
| 쓰다 (to write) | 쓰 | **썼어요** | 쓰 + 었어요 → 썼어요 |
| 자다 (to sleep) | 자 | **잤어요** | 자 + 았어요 → 잤어요 |
| 일어나다 (to get up) | 일어나 | **일어났어요** | 일어나 + 았어요 → 일어났어요 |
| 하다 (to do) | 하 | **했어요** | 하 + 였어요 → 했어요 |
| 공부하다 (to study) | 공부하 | **공부했어요** | 공부하 + 였어요 → 공부했어요 |

### 5.3 Contraction Patterns in Past Tense

The same contraction rules apply as in present tense:

| Pattern | Process | Example |
|---------|---------|---------|
| ㅏ + 았 → ㅏㅆ | 가 + 았 → 갔 | 갔어요 (went) |
| ㅗ + 았 → ㅘㅆ | 오 + 았 → 왔 | 왔어요 (came) |
| ㅜ + 었 → ㅝㅆ | 배우 + 었 → 배웠 | 배웠어요 (learned) |
| ㅣ + 었 → ㅕㅆ | 마시 + 었 → 마셨 | 마셨어요 (drank) |
| ㅡ + 었 → contracted | 쓰 + 었 → 썼 | 썼어요 (wrote) |
| 하 + 였 → 했 | 하 + 였 → 했 | 했어요 (did) |

## 6. Past Tense Formal (-았/었습니다)

### 6.1 Formation

Combine the past tense marker with the formal ending:

```
Stem + 았/었 + 습니다

가다  →  갔습니다      (went — formal)
먹다  →  먹었습니다    (ate — formal)
하다  →  했습니다      (did — formal)
```

### 6.2 Conjugation Table

| Dictionary Form | Past Formal | Past Formal Question |
|----------------|------------|---------------------|
| 가다 | 갔습니다 | 갔습니까? |
| 오다 | 왔습니다 | 왔습니까? |
| 먹다 | 먹었습니다 | 먹었습니까? |
| 마시다 | 마셨습니다 | 마셨습니까? |
| 하다 | 했습니다 | 했습니까? |
| 읽다 | 읽었습니다 | 읽었습니까? |
| 보다 | 봤습니다 | 봤습니까? |
| 자다 | 잤습니다 | 잤습니까? |

### 6.3 Complete Tense/Formality Chart

> **Complete Conjugation Chart: 먹다 (to eat)**
>
> |  | Present | Past |
> |---|---------|------|
> | Formal | 먹습니다 | 먹었습니다 |
> | Polite | 먹어요 | 먹었어요 |
> | Casual | 먹어 | 먹었어 |
>
> **Complete Conjugation Chart: 가다 (to go)**
>
> |  | Present | Past |
> |---|---------|------|
> | Formal | 갑니다 | 갔습니다 |
> | Polite | 가요 | 갔어요 |
> | Casual | 가 | 갔어 |

> **Complete Conjugation Chart: 하다 (to do)**
>
> |  | Present | Past |

> | Casual | 해 | 했어 |

## 7. Negative Forms

### 7.1 Short Negation: 안 + Verb

Place **안** (an) directly before the verb:

```
안 + verb

가요 → 안 가요        (don't go)
먹어요 → 안 먹어요    (don't eat)
와요 → 안 와요        (don't come)
읽어요 → 안 읽어요    (don't read)

For 하다 compound verbs:
공부해요 → 공부 안 해요   (don't study) ← 안 goes before 해요
운동해요 → 운동 안 해요   (don't exercise)
```

> **Note**: For 하다 compound verbs, 안 is placed between the noun part and 하다 (공부 **안** 해요), NOT before the whole verb (**X** 안 공부해요).

### 7.2 Long Negation: Verb + -지 않다

Attach **-지 않다** to the verb stem. This is more formal/emphatic:

```
Stem + 지 않다   →   Stem + 지 않아요 (polite)
                     Stem + 지 않습니다 (formal)

가다   → 가지 않아요      (don't go)
먹다   → 먹지 않아요      (don't eat)
하다   → 하지 않아요      (don't do)
공부하다 → 공부하지 않아요 (don't study)
```

### 7.3 Short vs Long Negation

| Short (안) | Long (-지 않다) | Meaning |
|-----------|----------------|---------|
| 안 가요 | 가지 않아요 | don't go |
| 안 먹어요 | 먹지 않아요 | don't eat |
| 안 했어요 | 하지 않았어요 | didn't do |
| 공부 안 해요 | 공부하지 않아요 | don't study |

> **When to use which?**
> - 안 is more common in casual speech (shorter, quicker)
> - -지 않다 is more common in formal writing and when emphasizing the negation

### 7.4 Past Tense Negation

```
Short: 안 + past tense verb
  안 갔어요        (didn't go)
  안 먹었어요      (didn't eat)

Long: stem + 지 않았어요
  가지 않았어요    (didn't go)
  먹지 않았어요    (didn't eat)
```

### 7.5 Special Negation: 못 (Cannot)

**못** (mot) means "cannot" — indicating inability rather than choice:

```
못 + verb

가요 → 못 가요        (can't go)
먹어요 → 못 먹어요    (can't eat)
해요 → 못 해요        (can't do)

Long form: stem + -지 못하다
  가지 못해요          (can't go)
  먹지 못해요          (can't eat)
```

| 안 (don't) | 못 (can't) |
|-----------|-----------|
| 안 가요 (I don't go / I choose not to go) | 못 가요 (I can't go / I'm unable to go) |
| 안 먹어요 (I don't eat it) | 못 먹어요 (I can't eat it) |
