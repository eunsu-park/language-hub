## Prerequisites

- Hangul reading and writing ability (Lesson 01)
- Basic sentence structure (Lesson 03)
- Present and past tense verb conjugation, vowel harmony rules (Lesson 05)
- Adjective conjugation (Lesson 06)

## Learning Objectives

- Form propositive ("Let's...") expressions with -자 and -(으)ㅂ시다
- Use imperative ("Do...") forms: -(으)세요 and -아/어라
- Express desire with -고 싶다
- Suggest trying something with -아/어 보다
- Describe ongoing actions with -고 있다 (progressive)
- Ask and give permission with -아/어도 되다
- Express prohibition with -(으)면 안 되다
- Talk about ability with -(으)ㄹ 수 있다/없다

## 1. Propositive: "Let's..."

Korean has two main ways to say "Let's..." depending on formality.

### 1.1 Informal: -자

Attach **-자** directly to the verb stem. Used among close friends, peers, or those younger.

| Verb | Stem | Propositive | Meaning |
|------|------|-----------|---------|
| 가다 (go) | 가 | 가자! | Let's go! |
| 먹다 (eat) | 먹 | 먹자! | Let's eat! |
| 하다 (do) | 하 | 하자! | Let's do it! |
| 놀다 (play) | 놀 | 놀자! | Let's hang out! |
| 쉬다 (rest) | 쉬 | 쉬자! | Let's rest! |

```
같이 가자!
gachi gaja!
Let's go together!

점심 먹자.
jeomsim meokja.
Let's eat lunch.

내일 만나자.
naeil mannaja.
Let's meet tomorrow.
```

### 1.2 Formal Polite: -(으)ㅂ시다

Used in formal or polite situations — meetings, with seniors, or public contexts.

| Stem Ending | Rule | Example |
|------------|------|---------|
| Vowel | + ㅂ시다 | 가 → 갑시다 |
| Consonant | + 읍시다 | 먹 → 먹읍시다 |
| ㄹ (drop ㄹ) | + ㅂ시다 | 놀 → 놉시다 |

| Korean | Translation |
|--------|------------|
| 같이 갑시다. | Let's go together. |
| 시작합시다. | Let's begin. |
| 회의를 시작합시다. | Let's start the meeting. |
| 건배합시다! | Let's toast! (cheers) |

### 1.3 Polite Informal: -(으)ㄹ까요?

A softer proposal in question form — "Shall we...?"

| Korean | Translation |
|--------|------------|
| 같이 갈까요? | Shall we go together? |
| 뭐 먹을까요? | What shall we eat? |
| 커피 마실까요? | Shall we have coffee? |

## 2. Imperative: Commands and Requests

```
┌──────────────────────────────────────────────────────────────┐
│              Imperative Forms (명령형)                          │
│                                                               │
│  Politeness Level     Form              Example               │
│  ─────────────────    ──────────        ────────              │
│  Polite request       -(으)세요          앉으세요 (Please sit)  │
│  Casual/Plain         -아/어라           앉아라 (Sit down)      │
│  Soft/Friendly        -아/어             앉아 (Sit)             │
│  Very formal          -(으)십시오        앉으십시오 (Please sit) │
│                                                               │
│  Most commonly used in daily life: -(으)세요                   │
└──────────────────────────────────────────────────────────────┘
```

### 2.1 Polite Request: -(으)세요

This is the most versatile and frequently used imperative. It is polite and respectful.

| Stem Ending | Rule | Example |
|------------|------|---------|
| Vowel | + 세요 | 보다 → 보세요 (Please look) |
| Consonant | + 으세요 | 읽다 → 읽으세요 (Please read) |
| ㄹ (drop ㄹ) | + 세요 | 만들다 → 만드세요 (Please make) |

| Korean | Translation |
|--------|------------|
| 앉으세요. | Please sit down. |
| 여기 보세요. | Please look here. |
| 천천히 말씀해 주세요. | Please speak slowly. |
| 드세요. | Please eat. (honorific of 먹다) |
| 조심하세요. | Please be careful. |

### 2.2 Casual Command: -아/어라

Used by parents to children, among very close friends, or in written directives.

| Korean | Translation |
|--------|------------|
| 빨리 와라! | Come quickly! |
| 여기 앉아라. | Sit here. |
| 공부해라. | Study! |
| 조용히 해라. | Be quiet! |

> **Warning**: -아/어라 sounds authoritative or even rude if used with the wrong person. When in doubt, use -(으)세요.

### 2.3 Negative Imperative: -지 마세요 / -지 마

| Polite | Casual | Meaning |
|--------|--------|---------|
| 가지 마세요. | 가지 마. | Don't go. |
| 걱정하지 마세요. | 걱정하지 마. | Don't worry. |
| 만지지 마세요. | 만지지 마. | Don't touch. |

## 3. Want and Desire: -고 싶다

Attach **-고 싶다** to the verb stem to express "want to [verb]."

### 3.1 Formation

```
Verb stem + 고 싶다  →  "want to [verb]"

Present: -고 싶어요     (I want to...)
Past:    -고 싶었어요    (I wanted to...)
```

### 3.2 Examples

| Korean | Translation |
|--------|------------|
| 한국에 가고 싶어요. | I want to go to Korea. |
| 뭐 하고 싶어요? | What do you want to do? |
| 커피 마시고 싶어요. | I want to drink coffee. |
| 한국어를 잘 하고 싶어요. | I want to be good at Korean. |
| 어제 쉬고 싶었어요. | I wanted to rest yesterday. |

### 3.3 Third Person: -고 싶어하다

When talking about **someone else's** desire, use -고 싶어하다 instead:

| Korean | Translation |
|--------|------------|
| 동생이 아이스크림을 먹고 싶어해요. | My younger sibling wants to eat ice cream. |
| 친구가 한국에 가고 싶어해요. | My friend wants to go to Korea. |

> **Why the distinction?** Korean treats your own feelings (-고 싶다) differently from observed feelings of others (-고 싶어하다). This reflects the cultural principle that you cannot directly know another person's inner state.

## 4. Try Doing: -아/어 보다

Attach **-아/어 보다** to a verb to mean "try doing" something.

### 4.1 Formation

Apply the same vowel harmony rules as 해요체:

| Verb | -아/어 보다 | Polite | Meaning |
|------|----------|--------|---------|
| 먹다 | 먹어 보다 | 먹어 보세요 | Please try eating |
| 가다 | 가 보다 | 가 보세요 | Please try going |
| 하다 | 해 보다 | 해 보세요 | Please try doing |
| 입다 (wear) | 입어 보다 | 입어 보세요 | Please try wearing |
| 읽다 (read) | 읽어 보다 | 읽어 보세요 | Please try reading |

### 4.2 Examples

| Korean | Translation |
|--------|------------|
| 이것 먹어 보세요. | Please try eating this. |
| 한번 해 보세요. | Please try doing it once. |
| 한국에 가 봤어요? | Have you been to Korea? (tried going) |
| 이 책 읽어 봤어요. | I've tried reading this book. |
| 김치를 먹어 봤어요? | Have you tried kimchi? |

> **Past tense nuance**: -아/어 봤어요 means "have tried/experienced" — it indicates life experience, similar to "Have you ever...?" in English.

## 5. Progressive: -고 있다

Attach **-고 있다** to the verb stem to express an action in progress ("am/is/are ~ing").

### 5.1 Formation

```
Verb stem + 고 있다  →  "is [verb]-ing"

Present: -고 있어요      (am doing)
Past:    -고 있었어요     (was doing)
```

### 5.2 Examples

| Korean | Translation |
|--------|------------|
| 지금 밥을 먹고 있어요. | I'm eating right now. |
| 음악을 듣고 있어요. | I'm listening to music. |
| 친구를 기다리고 있어요. | I'm waiting for a friend. |
| 한국어를 공부하고 있어요. | I'm studying Korean. |
| 뭐 하고 있어요? | What are you doing? |
| 그때 자고 있었어요. | I was sleeping at that time. |

### 5.3 Wearing/Carrying (State)

-고 있다 also describes a continuing **state** for verbs like wearing and carrying:

| Korean | Translation |
|--------|------------|
| 안경을 쓰고 있어요. | I'm wearing glasses. |
| 모자를 쓰고 있어요. | I'm wearing a hat. |
| 가방을 들고 있어요. | I'm carrying a bag. |

## 6. Permission: -아/어도 되다

Use **-아/어도 되다** to ask for or grant permission ("May I...?" / "It's okay to...").

### 6.1 Formation

```
Verb stem + 아/어도 되다  →  "may [verb]" / "it's okay to [verb]"

Question:  -아/어도 돼요?     (May I...?)
Answer:    네, -아/어도 돼요.  (Yes, you may.)
```

### 6.2 Examples

| Korean | Translation |
|--------|------------|
| 여기 앉아도 돼요? | May I sit here? |
| 사진 찍어도 돼요? | May I take a photo? |
| 들어가도 돼요? | May I come in? |
| 이거 먹어도 돼요? | May I eat this? |
| 창문 열어도 돼요? | May I open the window? |

### 6.3 Granting and Denying Permission

| Response | Korean | Translation |
|----------|--------|------------|
| Yes | 네, -아/어도 돼요. | Yes, you may. |
| Of course | 그럼요, -아/어도 돼요. | Of course, you may. |
| No | 아니요, -면 안 돼요. | No, you must not. |

```
A: 여기서 전화해도 돼요?     (May I make a phone call here?)
B: 네, 괜찮아요. 해도 돼요.   (Yes, it's fine. You may.)

A: 사진 찍어도 돼요?         (May I take a photo?)
B: 아니요, 찍으면 안 돼요.    (No, you must not take photos.)
```

## 7. Prohibition: -(으)면 안 되다

Use **-(으)면 안 되다** to express prohibition ("must not," "may not").

### 7.1 Formation

| Stem Ending | Rule | Example |
|------------|------|---------|
| Vowel | + 면 안 되다 | 가 → 가면 안 돼요 |
| Consonant | + 으면 안 되다 | 먹 → 먹으면 안 돼요 |

### 7.2 Examples

| Korean | Translation |
|--------|------------|
| 여기서 담배를 피우면 안 돼요. | You must not smoke here. |
| 수업 시간에 핸드폰을 쓰면 안 돼요. | You must not use your phone during class. |
| 거짓말하면 안 돼요. | You must not lie. |
| 늦으면 안 돼요. | You must not be late. |
| 여기에 주차하면 안 돼요. | You must not park here. |

### 7.3 Common Prohibition Signs

```
┌─────────────────────────────────────────────────────────┐
│         Common Prohibition Expressions                    │
│                                                           │
│  촬영 금지          No photography                        │
│  주차 금지          No parking                            │
│  흡연 금지          No smoking                            │
│  출입 금지          No entry                              │
│  음식물 반입 금지    No outside food                       │
│  만지지 마세요      Please do not touch                    │
│  뛰지 마세요        Please do not run                     │
└─────────────────────────────────────────────────────────┘
```

## 8. Ability: -(으)ㄹ 수 있다/없다

Express ability ("can") or inability ("cannot") with **-(으)ㄹ 수 있다/없다**.

### 8.1 Formation

| Stem Ending | Rule | Example |
|------------|------|---------|
| Vowel | + ㄹ 수 있다 | 가 → 갈 수 있어요 (can go) |
| Consonant | + 을 수 있다 | 먹 → 먹을 수 있어요 (can eat) |
| ㄹ | + 수 있다 (ㄹ stays) | 만들 → 만들 수 있어요 (can make) |

### 8.2 Can / Cannot

| Korean | Translation |
|--------|------------|
| 한국어를 할 수 있어요. | I can speak Korean. |
| 한국어를 할 수 없어요. | I cannot speak Korean. |
| 운전할 수 있어요? | Can you drive? |
| 매운 음식을 먹을 수 있어요. | I can eat spicy food. |
| 내일 올 수 있어요? | Can you come tomorrow? |
| 수영할 수 있어요? | Can you swim? |

### 8.3 Alternative: 못 + verb

A shorter way to say "cannot" is **못** placed before the verb:

| Long Form | Short Form | Meaning |
|----------|-----------|---------|
| 갈 수 없어요 | 못 가요 | can't go |
| 먹을 수 없어요 | 못 먹어요 | can't eat |
| 할 수 없어요 | 못 해요 | can't do |

> **Nuance**: 못 implies inability due to circumstances or lack of skill. -(으)ㄹ 수 없다 is more general and slightly more formal.

```
매운 것을 못 먹어요.
maeun geos-eul mot meogeoyo.
I can't eat spicy things. (I don't have the ability/tolerance)

오늘 못 가요.
oneul mot gayo.
I can't go today. (circumstances prevent me)
```

## 9. Verb Ending Summary

```
┌──────────────────────────────────────────────────────────────────┐
│                 Verb Ending Quick Reference                       │
│                                                                   │
│  CATEGORY          FORM                  EXAMPLE                  │
│  ────────          ──────                ────────                  │
│  Propositive       -자 (casual)          가자! (Let's go!)         │
│  (Let's...)        -(으)ㅂ시다 (formal)   갑시다 (Let's go)         │
│                    -(으)ㄹ까요? (soft)    갈까요? (Shall we go?)    │
│                                                                   │
│  Imperative        -(으)세요 (polite)     가세요 (Please go)        │
│  (Do...)           -아/어라 (casual)      가라 (Go!)               │
│                    -지 마세요 (don't)      가지 마세요 (Don't go)    │
│                                                                   │
│  Desire            -고 싶다              가고 싶어요 (want to go)    │
│                    -고 싶어하다           가고 싶어해요 (wants to go) │
│                                                                   │
│  Try               -아/어 보다            가 보세요 (try going)      │
│                                                                   │
│  Progressive       -고 있다              가고 있어요 (is going)      │
│                                                                   │
│  Permission        -아/어도 되다          가도 돼요 (may go)         │
│                                                                   │
│  Prohibition       -(으)면 안 되다        가면 안 돼요 (must not go) │
│                                                                   │
│  Ability           -(으)ㄹ 수 있다        갈 수 있어요 (can go)      │
│  Inability         -(으)ㄹ 수 없다        갈 수 없어요 (can't go)   │
│                    못 + verb             못 가요 (can't go)         │
└──────────────────────────────────────────────────────────────────┘
```

## 10. Practice Exercises

### Exercise 1: Propositive Forms

Convert to "Let's..." form in both casual and formal.

| Verb | Casual (-자) | Formal (-(으)ㅂ시다) |
|------|------------|-------------------|
| 먹다 (eat) | ___ | ___ |
| 가다 (go) | ___ | ___ |
| 시작하다 (start) | ___ | ___ |
| 쉬다 (rest) | ___ | ___ |
| 만들다 (make) | ___ | ___ |

### Exercise 2: Make Polite Requests (-(으)세요)

1. 읽다 (read) → ___
2. 앉다 (sit) → ___
3. 기다리다 (wait) → ___
4. 만들다 (make) → ___
5. 조심하다 (be careful) → ___

### Exercise 3: Express Desires (-고 싶다)

Translate to Korean:

1. I want to eat Korean food.
2. I want to go to Jeju Island.
3. What do you want to do this weekend?
4. I wanted to rest yesterday.

### Exercise 4: Permission and Prohibition

Complete the dialogues:

1. A: 여기 앉아도 ___? (May I sit here?)
   B: 네, 앉아도 ___. (Yes, you may.)

2. A: 사진 찍어도 ___? (May I take a photo?)
   B: 아니요, 찍___ 안 ___. (No, you must not.)

3. A: 한국어를 할 수 ___? (Can you speak Korean?)
   B: 네, 조금 할 수 ___. (Yes, I can a little.)

### Exercise 5: Translate to Korean

1. Let's eat lunch together.
2. Please speak slowly.
3. I'm studying Korean right now.
4. Have you tried kimchi?
5. You must not smoke here.
6. I can swim.
