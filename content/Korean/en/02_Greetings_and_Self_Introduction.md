## 1. Essential Korean Greetings

### 1.1 The Universal Greeting: 안녕하세요

```
┌──────────────────────────────────────────────────────────────────┐
│                     안녕하세요                                     │
│               (annyeonghaseyo)                                   │
│                                                                  │
│   Literal meaning: "Are you at peace?"                           │
│   Usage: Hello / Hi / Good morning / Good afternoon / Good       │
│          evening — used at ANY time of day                       │
│   Formality: Polite (존댓말) — safe for nearly all situations    │
│                                                                  │
│   Breakdown:                                                     │
│     안녕 (annyeong) = peace, well-being                          │
│     하 (ha)         = to do/be (verb stem of 하다)               │
│     세요 (seyo)     = honorific polite ending                    │
│                                                                  │
│   Literally: "Do you do peace?" = "Are you well?"               │
└──────────────────────────────────────────────────────────────────┘
```

| Situation | Expression | Romanization | Formality |
|-----------|-----------|-------------|-----------|
| Any greeting (standard) | 안녕하세요 | annyeonghaseyo | Polite |
| Any greeting (casual) | 안녕 | annyeong | Casual (friends/kids) |
| Any greeting (very formal) | 안녕하십니까 | annyeonghasimnikka | Formal (speeches, military, news) |

> **Key point**: Korean does NOT have separate greetings for morning, afternoon, and evening in everyday speech. 안녕하세요 covers all times of day.

### 1.2 Thank You

| Expression | Romanization | Usage |
|-----------|-------------|-------|
| 감사합니다 | gamsahamnida | Formal — standard polite "thank you" |
| 고마워요 | gomawoyo | Informal polite — to friends, people your age |
| 고마워 | gomawo | Casual — close friends only |
| 감사드립니다 | gamsadeurimnida | Very formal/humble — to superiors, in business |

> **감사 vs 고맙다**: 감사 is Sino-Korean (from Chinese 感謝) and feels more formal. 고맙다 is native Korean and feels warmer/more personal.

### 1.3 I'm Sorry / Excuse Me

| Expression | Romanization | Usage |
|-----------|-------------|-------|
| 죄송합니다 | joesonghamnida | Formal apology — "I'm sorry" (to elders, strangers, superiors) |
| 미안합니다 | mianhamnida | Polite apology — "I'm sorry" (slightly less formal) |
| 미안해요 | mianhaeyo | Informal polite — "I'm sorry" (to peers) |
| 미안해 | mianhae | Casual — "Sorry" (close friends) |
| 실례합니다 | sillyehamnida | "Excuse me" — to get attention or pass by someone |

### 1.4 Greeting at Meals

| Expression | Romanization | Meaning | When |
|-----------|-------------|---------|------|
| 잘 먹겠습니다 | jal meokgesseumnida | "I will eat well" | Before eating (to the host/cook) |
| 잘 먹었습니다 | jal meogeosseumnida | "I ate well" | After eating (to the host/cook) |

## 2. Self-Introduction Pattern

### 2.1 The Basic Template

```
┌──────────────────────────────────────────────────────────────────┐
│              Korean Self-Introduction Template                    │
│                                                                  │
│   안녕하세요.                        Hello.                       │
│   저는 [Name]입니다.                 I am [Name].                 │
│   [Country]에서 왔습니다.            I came from [Country].       │
│   [Job/Occupation]입니다.            I am a [Job].                │
│   만나서 반갑습니다.                 Nice to meet you.             │
│                                                                  │
│   Example:                                                       │
│   안녕하세요.                                                     │
│   저는 마이클입니다.                 I am Michael.                │
│   미국에서 왔습니다.                 I came from America.          │
│   학생입니다.                        I am a student.              │
│   만나서 반갑습니다.                                              │
└──────────────────────────────────────────────────────────────────┘
```

### 2.3 Grammar Breakdown

```
저는 [Name]입니다.

저   = I (humble/polite form)
는   = topic marker (marks "I" as the topic)
입니다 = "is/am" (formal polite copula)

[Country]에서 왔습니다.

에서  = from (particle indicating origin/source)
왔습니다 = came (past tense formal polite of 오다, "to come")

[Job]입니다.

입니다 = is/am (copula — subject "I" is omitted because it's obvious)
```

## 3. Basic Politeness: 합쇼체 Introduction

### 3.1 What is 합쇼체?

Korean has multiple speech levels. The **합쇼체** (hapssyoche) is the **formal polite** level — the highest level of politeness used in everyday conversation.

```
┌──────────────────────────────────────────────────────────────────┐
│           Speech Level Quick Reference (Preview)                  │
│                                                                  │
│   합쇼체 (Formal polite):  -ㅂ니다 / -습니다                     │
│     → News, business, military, first meetings, public speech    │
│     → "I am a student" = 학생입니다                               │
│                                                                  │
│   해요체 (Informal polite): -아/어요                              │
│     → Everyday conversation with strangers, coworkers, shops     │
│     → "I am a student" = 학생이에요                               │
│                                                                  │
│   해체 (Casual):           -아/어 (no 요)                        │
│     → Close friends of same age, children, intimate relationships│
│     → "I am a student" = 학생이야                                 │
│                                                                  │
│   For beginners: Master 합쇼체 and 해요체 first.                  │
│   Using 합쇼체 is NEVER wrong — it's always safe.                 │
└──────────────────────────────────────────────────────────────────┘
```

### 3.2 합쇼체 Patterns for Self-Introduction

| Pattern | Structure | Example |
|---------|----------|---------|
| "I am [noun]" | [noun]입니다 | 학생입니다 (I am a student) |
| "I am not [noun]" | [noun]이/가 아닙니다 | 학생이 아닙니다 (I am not a student) |
| "I came from [place]" | [place]에서 왔습니다 | 미국에서 왔습니다 (I came from the US) |
| "[Noun] is [noun]" | [noun]은/는 [noun]입니다 | 이것은 책입니다 (This is a book) |

## 4. Meeting Someone New

### 4.1 First-Meeting Expressions

| Korean | Romanization | English | Usage |
|--------|-------------|---------|-------|
| 만나서 반갑습니다 | mannaseo bangapseumnida | Nice to meet you | Formal polite — standard |
| 만나서 반가워요 | mannaseo bangawoyo | Nice to meet you | Informal polite |
| 처음 뵙겠습니다 | cheoeum boepgesseumnida | "I see you for the first time" | Very formal/humble — to superiors |
| 성함이 어떻게 되세요? | seongham-i eotteoke doeseyo? | What is your name? (honorific) | Polite way to ask a name |
| 이름이 뭐예요? | ireum-i mwoyeyo? | What is your name? | Standard polite |

### 4.2 Exchanging Information

| Question | Answer Template |
|----------|----------------|
| 이름이 뭐예요? (What is your name?) | 저는 [name]입니다. |
| 어디에서 왔어요? (Where are you from?) | [country/city]에서 왔습니다. |
| 직업이 뭐예요? (What is your job?) | [job]입니다. |
| 나이가 어떻게 되세요? (How old are you?) | [number]살입니다. |
| 취미가 뭐예요? (What is your hobby?) | 제 취미는 [hobby]입니다. |

### 4.3 Sample Conversation

```
A: 안녕하세요.                          Hello.
B: 안녕하세요.                          Hello.

A: 저는 김민수입니다.                    I'm Kim Minsu.
   만나서 반갑습니다.                    Nice to meet you.

B: 저는 사라입니다.                      I'm Sarah.
   미국에서 왔습니다.                    I came from America.
   만나서 반갑습니다.                    Nice to meet you.

A: 직업이 뭐예요?                       What's your job?
B: 학생입니다. 한국어를 공부합니다.       I'm a student. I study Korean.

A: 아, 그렇군요! 화이팅!                Oh, I see! Fighting! (Go for it!)
B: 감사합니다.                          Thank you.
```

## 5. Goodbye Expressions and Context

Korean has different goodbye expressions depending on **who is leaving and who is staying**.

```
┌──────────────────────────────────────────────────────────────────┐
│                  Korean Goodbye Expressions                       │
│                                                                  │
│   Situation 1: You are LEAVING, the other person STAYS           │
│   → 안녕히 계세요. (annyeonghi gyeseyo)                          │
│     Literal: "Stay in peace." (Please be well while staying.)    │
│                                                                  │
│   Situation 2: The other person is LEAVING, you STAY             │
│   → 안녕히 가세요. (annyeonghi gaseyo)                           │
│     Literal: "Go in peace." (Please travel well.)                │
│                                                                  │
│   Situation 3: BOTH people are leaving                           │
│   → 안녕히 가세요. (both say this to each other)                  │
│                                                                  │
│   Casual (friends):                                              │
│   → 잘 가. (jal ga) = "Go well."                                │
│   → 잘 있어. (jal isseo) = "Stay well."                         │
│   → 안녕. (annyeong) = Bye. (same word as casual "hello")       │
└──────────────────────────────────────────────────────────────────┘
```

### 5.1 Other Parting Expressions

| Korean | Romanization | Meaning | Context |
|--------|-------------|---------|---------|
| 내일 봐요 | naeil bwayo | See you tomorrow | Informal polite |
| 또 만나요 | tto mannayo | Let's meet again | Informal polite |
| 조심히 가세요 | josimhi gaseyo | Go carefully / Take care | Polite, caring |
| 수고하셨습니다 | sugohasyeosseumnida | You've worked hard | After work/meetings |
| 먼저 가겠습니다 | meonjeo gagetseumnida | I'll go first | Formal — leaving before others |

## 6. Common Daily Phrases

### 6.1 Yes and No

| Korean | Romanization | Meaning | Formality |
|--------|-------------|---------|-----------|
| 네 | ne | Yes | Polite (standard) |
| 예 | ye | Yes | More formal |
| 아니요 | aniyo | No | Polite |
| 아니 | ani | No | Casual |

> **Important**: Korean "yes" (네) confirms the **statement structure**, not the content. If someone asks "Don't you like it?" (안 좋아요?), answering 네 means "Yes, (it's correct that) I don't like it." This is the opposite of English.

### 6.2 Essential Daily Expressions

| Korean | Romanization | English |
|--------|-------------|---------|
| 괜찮아요 | gwaenchanayo | It's okay / I'm fine / No worries |
| 잠시만요 | jamsimanyo | Just a moment, please |
| 실례합니다 | sillyehamnida | Excuse me (politely interrupting) |
| 저기요 | jeogiyo | Excuse me (calling for attention — at a restaurant, etc.) |
| 여기요 | yeogiyo | Over here, please (calling a waiter) |
| 알겠습니다 | algetseumnida | I understand (formal) |
| 모르겠어요 | moreugesseoyo | I don't understand / I don't know |
| 다시 한번 말씀해 주세요 | dasi hanbeon malsseumhae juseyo | Please say it one more time |
| 천천히 말해 주세요 | cheoncheonhi malhae juseyo | Please speak slowly |
| 한국어를 조금 해요 | hangugeoreul jogeum haeyo | I speak a little Korean |

### 6.3 Phone/Contact Basics

| Korean | Romanization | English |

## 7. Cultural Notes

### 7.1 Bowing (인사)

```
┌──────────────────────────────────────────────────────────────────┐
│                    Korean Bowing Etiquette                         │
│                                                                  │
│   Casual nod (목례, mongnye):                                    │
│     ~15° angle — casual greetings, peers                         │
│                                                                  │
│   Standard bow (보통례, botongnyе):                               │
│     ~30° angle — general polite greeting                         │
│                                                                  │
│   Deep bow (정중례, jeongjongnye):                                │
│     ~45° angle — showing great respect (elders, first meeting    │
│     with someone important)                                      │
│                                                                  │
│   Full bow (큰절, keunjeol):                                     │
│     Kneeling, forehead to floor — formal occasions (New Year's,  │
│     weddings, to grandparents)                                   │
│                                                                  │
│   Note: When greeting, say 안녕하세요 while bowing. Do NOT bow   │
│   while shaking hands — do one or the other (or bow slightly     │
│   while shaking hands in business settings).                     │
└──────────────────────────────────────────────────────────────────┘
```

### 7.2 Titles and Address

In Korea, you almost never call someone by their name alone, especially if they are older or in a higher position:

| Title | Usage |
|-------|-------|
| [Name]씨 (ssi) | Mr./Ms. — polite but peers-level. Use full name + 씨, NOT last name + 씨 |
| [Name/Title]님 (nim) | Honorific suffix — added to titles or roles (선생님, 사장님) |
| 선배 (seonbae) | Senior (at school/work) |
| 후배 (hubae) | Junior (at school/work) |
| 오빠 (oppa) | Older brother (used by females) — also for older male friends |
| 형 (hyeong) | Older brother (used by males) |
| 언니 (eonni) | Older sister (used by females) |
| 누나 (nuna) | Older sister (used by males) |
| 아저씨 (ajeossi) | Middle-aged man (uncle-age) |
| 아주머니 (ajumeoni) | Middle-aged woman (aunt-age) |

### 7.3 Age Culture (나이 문화)

Korean culture places great importance on age. It determines:
- Which **speech level** you use (formal vs. casual)
- Which **titles** you use (형/오빠/언니/누나 etc.)
- Who **pours drinks** first (younger person pours for older)
- **Seating** and **eating order** (elders served first)

One of the first questions Koreans ask new acquaintances is: **몇 살이에요?** (How old are you?) or the more polite **나이가 어떻게 되세요?** This is not considered rude — it establishes the social dynamic for proper speech.

> **Korean age vs. international age**: Korea traditionally used a system where you are 1 at birth and everyone ages on January 1st. Since 2023, South Korea officially adopted the international age system for legal and administrative purposes, but the traditional counting system persists socially.

### 7.4 Using Two Hands

When giving or receiving objects (business cards, gifts, drinks), use **two hands** or your right hand supported by your left hand holding your right forearm. This shows respect.
