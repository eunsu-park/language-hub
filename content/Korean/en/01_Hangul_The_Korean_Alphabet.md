## 1. History of Hangul

### 1.1 The Creation of Hangul

```
┌──────────────────────────────────────────────────────────────────┐
│                  The Birth of Hangul                              │
│                                                                  │
│   Creator:  King Sejong the Great (세종대왕)                      │
│   Year:     1443 (invented), 1446 (promulgated)                  │
│   Document: Hunminjeongeum (훈민정음, "Correct Sounds for         │
│             the Instruction of the People")                      │
│                                                                  │
│   Purpose:  Enable all Korean people — including commoners —     │
│             to read and write, replacing reliance on Chinese      │
│             characters (漢字/한자) that only the elite knew       │
│                                                                  │
│   UNESCO:   Hunminjeongeum registered as Memory of the World     │
│             (1997). Hangul Day: October 9 (한글날)                │
└──────────────────────────────────────────────────────────────────┘
```

Before Hangul, Koreans used Chinese characters (Hanja) for writing. Since Chinese characters are logographic and their grammar differs fundamentally from Korean, only the aristocratic class (양반, yangban) could read and write. The vast majority of the population was illiterate.

King Sejong, the fourth king of the Joseon Dynasty, recognized this injustice. In the preface to the Hunminjeongeum, he wrote:

> "The sounds of our language differ from those of Chinese and are not easily conveyed using Chinese characters. Many among the common people who wish to express their thoughts cannot do so. I am deeply saddened by this and have newly created twenty-eight letters. I wish only that everyone may learn them easily and use them conveniently in daily life."

### 1.2 Design Principles

Hangul is often praised by linguists as one of the most scientifically designed writing systems in history. Its design is based on:

1. **Consonants model the speech organs**: The basic consonant shapes depict the position of the tongue, lips, and throat when producing each sound
2. **Vowels model the universe**: Based on three elements — Heaven (ㆍ, dot), Earth (ㅡ, horizontal line), and Human (ㅣ, vertical line)
3. **Systematic expansion**: Related sounds are derived by adding strokes to base shapes
4. **Featural alphabet**: Consonants that share the same place of articulation share the same base shape

```
┌──────────────────────────────────────────────────────────────────┐
│           Consonant Design: Articulatory Basis                    │
│                                                                  │
│   ㄱ (g/k) — shape of the tongue touching the back of the mouth │
│   ㄴ (n)   — shape of the tongue touching the front palate      │
│   ㅁ (m)   — shape of the mouth (lips closed)                   │
│   ㅅ (s)   — shape of a tooth                                   │
│   ㅇ (ng)  — shape of the throat (circle = open)                │
│                                                                  │
│   These 5 base shapes generate all other consonants by adding   │
│   strokes:  ㄱ → ㅋ (add stroke)  →  ㄲ (double)                │
│             ㄴ → ㄷ → ㅌ (add strokes)                           │
│             ㅁ → ㅂ → ㅍ (add strokes)                           │
│             ㅅ → ㅈ → ㅊ (add strokes)                           │
│             ㅇ → ㅎ (add strokes)                                │
└──────────────────────────────────────────────────────────────────┘
```

## 2. 14 Basic Consonants

Korean has 14 basic consonants (자음, jaeum). Each consonant has an initial sound (at the beginning of a syllable) and a final sound (at the end of a syllable, as 받침).

| Letter | Name | Romanization | IPA (Initial) | IPA (Final) | Pronunciation Guide |
|--------|------|-------------|---------------|-------------|-------------------|
| ㄱ | 기역 (giyeok) | g/k | /k/ or /g/ | /k̚/ | Between "g" and "k" — unaspirated. Like "g" in "go" but with less voicing |
| ㄴ | 니은 (nieun) | n | /n/ | /n/ | Like "n" in "no" |
| ㄷ | 디귿 (digeut) | d/t | /t/ or /d/ | /t̚/ | Between "d" and "t" — unaspirated. Like "d" in "do" but with less voicing |
| ㄹ | 리을 (rieul) | r/l | /ɾ/ | /l/ | Initial: flap like "r" in Spanish "pero". Final: "l" as in "pool" |
| ㅁ | 미음 (mieum) | m | /m/ | /m/ | Like "m" in "mom" |
| ㅂ | 비읍 (bieup) | b/p | /p/ or /b/ | /p̚/ | Between "b" and "p" — unaspirated. Like "b" in "boy" but with less voicing |
| ㅅ | 시옷 (siot) | s | /s/ or /ɕ/ | /t̚/ | Like "s" in "sun". Before ㅣ: "sh" sound |
| ㅇ | 이응 (ieung) | -/ng | silent | /ŋ/ | Initial: silent placeholder. Final: "ng" as in "sing" |
| ㅈ | 지읒 (jieut) | j | /tɕ/ | /t̚/ | Like "j" in "judge" — unaspirated |
| ㅊ | 치읓 (chieut) | ch | /tɕʰ/ | /t̚/ | Like "ch" in "church" — aspirated |
| ㅋ | 키읔 (kieuk) | k | /kʰ/ | /k̚/ | Like "k" in "kite" — strongly aspirated |
| ㅌ | 티읕 (tieut) | t | /tʰ/ | /t̚/ | Like "t" in "top" — strongly aspirated |
| ㅍ | 피읖 (pieup) | p | /pʰ/ | /p̚/ | Like "p" in "pop" — strongly aspirated |
| ㅎ | 히읗 (hieut) | h | /h/ | /t̚/ | Like "h" in "hat" |

### 2.1 Three Types of Stops

A key concept in Korean consonants is the **three-way contrast** among stop consonants. English has two (voiced/voiceless), but Korean has three:

```
┌──────────────────────────────────────────────────────────────────┐
│              Korean Three-Way Stop Contrast                       │
│                                                                  │
│   Plain (Lax)     Aspirated         Tense (Double)               │
│   ───────────     ─────────         ───────────────              │
│   ㄱ /k/          ㅋ /kʰ/           ㄲ /k͈/                      │
│   ㄷ /t/          ㅌ /tʰ/           ㄸ /t͈/                      │
│   ㅂ /p/          ㅍ /pʰ/           ㅃ /p͈/                      │
│   ㅈ /tɕ/         ㅊ /tɕʰ/          ㅉ /t͈ɕ/                     │
│   ㅅ /s/          —                 ㅆ /s͈/                      │
│                                                                  │
│   Plain:     Low aspiration, can voice between voiced sounds     │
│   Aspirated: Strong puff of air (hold paper in front of mouth)  │
│   Tense:     Stiff vocal cords, no aspiration, "pressed" sound  │
└──────────────────────────────────────────────────────────────────┘
```

> **Tip**: Place your hand in front of your mouth. For ㅋ (aspirated), you should feel a strong puff of air. For ㄱ (plain) and ㄲ (tense), you should feel little to no air. The difference between plain and tense is in throat tension — tense consonants feel "squeezed."

## 3. 5 Double Consonants (Tense Consonants)

The five double (쌍, ssang) consonants are written by doubling a basic consonant. They are pronounced with a tense, glottalized quality — tighten the muscles in your throat.

| Letter | Name | Romanization | IPA | Pronunciation Guide |
|--------|------|-------------|-----|-------------------|
| ㄲ | 쌍기역 (ssang-giyeok) | kk | /k͈/ | Like "k" in "sky" — tense, no aspiration |
| ㄸ | 쌍디귿 (ssang-digeut) | tt | /t͈/ | Like "t" in "stun" — tense, no aspiration |
| ㅃ | 쌍비읍 (ssang-bieup) | pp | /p͈/ | Like "p" in "spy" — tense, no aspiration |
| ㅆ | 쌍시옷 (ssang-siot) | ss | /s͈/ | Like "ss" in "boss" — tense, sharp |
| ㅉ | 쌍지읒 (ssang-jieut) | jj | /t͈ɕ/ | Like "j" in "budge" — tense, no aspiration |

### 3.1 Examples

| Word | Romanization | Meaning |
|------|-------------|---------|
| 까치 | kkachi | magpie |
| 뚜껑 | ttukkeoong | lid |
| 빠르다 | ppareuda | to be fast |
| 쓰다 | sseuda | to write / to be bitter |
| 짜다 | jjada | to be salty / to squeeze |

## 4. 10 Basic Vowels

Korean has 10 basic vowels (모음, moeum). They are organized around two principles:

1. **Dot/stroke position** relative to the vertical or horizontal base line
2. **Yin-Yang contrast**: bright vowels (양성모음: ㅏ, ㅗ, ㅑ, ㅛ) vs. dark vowels (음성모음: ㅓ, ㅜ, ㅕ, ㅠ) vs. neutral (ㅡ, ㅣ)

```
┌──────────────────────────────────────────────────────────────────┐
│                  Hangul Vowel Design Principle                    │
│                                                                  │
│   Three cosmological elements:                                   │
│                                                                  │
│     ● (dot/short stroke) = Heaven (하늘, ㆍ — archaic vowel)    │
│     ─ (horizontal line)  = Earth (땅, ㅡ)                        │
│     │ (vertical line)     = Human (사람, ㅣ)                      │
│                                                                  │
│   Combinations of ● with ─ or │ produce the basic vowels:       │
│                                                                  │
│     Bright (양성):           Dark (음성):                         │
│     ㅏ  (│+ ● right)        ㅓ  (● left + │)                     │
│     ㅗ  (● above + ─)       ㅜ  (─ + ● below)                    │
│     ㅑ  (│+ ●● right)       ㅕ  (●● left + │)                    │
│     ㅛ  (●● above + ─)      ㅠ  (─ + ●● below)                   │
│                                                                  │
│   Neutral:                                                       │
│     ㅡ  (─)                 ㅣ  (│)                               │
└──────────────────────────────────────────────────────────────────┘
```

| Letter | Name | Romanization | IPA | Pronunciation Guide |
|--------|------|-------------|-----|-------------------|
| ㅏ | 아 (a) | a | /a/ | Like "a" in "father" |
| ㅑ | 야 (ya) | ya | /ja/ | Like "ya" in "yard" |
| ㅓ | 어 (eo) | eo | /ʌ/ | Like "u" in "bus" — open your mouth wider than for "o" |
| ㅕ | 여 (yeo) | yeo | /jʌ/ | Like "yo" in "young" |
| ㅗ | 오 (o) | o | /o/ | Like "o" in "go" — lips rounded, no glide |
| ㅛ | 요 (yo) | yo | /jo/ | Like "yo" in "yoga" |
| ㅜ | 우 (u) | u | /u/ | Like "oo" in "moon" |
| ㅠ | 유 (yu) | yu | /ju/ | Like "you" |
| ㅡ | 으 (eu) | eu | /ɯ/ | Like "oo" in "book" but with lips spread (unrounded). Smile and say "oo" |
| ㅣ | 이 (i) | i | /i/ | Like "ee" in "see" |

### 4.1 Bright vs. Dark Vowels (Vowel Harmony)

This distinction matters for verb conjugation (covered in Lesson 05):

```
Bright vowels (양성모음):  ㅏ  ㅗ  ㅑ  ㅛ    → pair with -아
Dark vowels (음성모음):    ㅓ  ㅜ  ㅕ  ㅠ    → pair with -어
Neutral vowels:           ㅡ  ㅣ             → pair with -어
```

## 5. 11 Compound Vowels

Compound vowels (이중 모음, ijung moeum) are formed by combining basic vowels.

| Letter | Name | Romanization | IPA | Composition | Pronunciation Guide |
|--------|------|-------------|-----|------------|-------------------|
| ㅐ | 애 (ae) | ae | /ɛ/ | ㅏ + ㅣ | Like "e" in "bed" |
| ㅒ | 얘 (yae) | yae | /jɛ/ | ㅑ + ㅣ | Like "ye" in "yes" |
| ㅔ | 에 (e) | e | /e/ | ㅓ + ㅣ | Like "e" in "hey" (slightly higher than ㅐ) |
| ㅖ | 예 (ye) | ye | /je/ | ㅕ + ㅣ | Like "ye" in "yet" |
| ㅘ | 와 (wa) | wa | /wa/ | ㅗ + ㅏ | Like "wa" in "water" |
| ㅙ | 왜 (wae) | wae | /wɛ/ | ㅗ + ㅐ | Like "we" in "wet" |
| ㅚ | 외 (oe) | oe | /we/ | ㅗ + ㅣ | Like "we" in "way" (in modern Seoul dialect) |
| ㅝ | 워 (wo) | wo | /wʌ/ | ㅜ + ㅓ | Like "wo" in "won" |
| ㅞ | 웨 (we) | we | /we/ | ㅜ + ㅔ | Like "we" in "wet" |
| ㅟ | 위 (wi) | wi | /wi/ | ㅜ + ㅣ | Like "wee" in "week" |
| ㅢ | 의 (ui) | ui | /ɰi/ | ㅡ + ㅣ | Start with ㅡ, glide to ㅣ. No English equivalent |

> **Modern note**: In contemporary Seoul Korean, ㅐ and ㅔ have merged for most speakers. Both sound approximately like /e/. Similarly, ㅙ, ㅚ, and ㅞ are often pronounced identically as /we/.

## 6. Syllable Block Structure

### 6.1 How Syllable Blocks Work

Unlike alphabets where letters are written linearly (e.g., "han"), Korean groups letters into **syllable blocks** (음절 블록). Each block represents one syllable and occupies one character space.

```
┌──────────────────────────────────────────────────────────────────┐
│                  Syllable Block Patterns                          │
│                                                                  │
│   Type 1: CV (Consonant + Vowel)                                 │
│   ┌───┐    ┌───┬───┐    Example: 가 (ga)                        │
│   │ C V│    │ ㄱ│ ㅏ│    ㄱ (g) + ㅏ (a) = 가                   │
│   └───┘    └───┴───┘                                             │
│                                                                  │
│   Type 2: CVC (Consonant + Vowel + Consonant)                   │
│   ┌───┐    ┌───┬───┐    Example: 한 (han)                       │
│   │ C V│    │ ㅎ│ ㅏ│    ㅎ (h) + ㅏ (a) + ㄴ (n) = 한          │
│   │ C  │    │  ㄴ  │                                             │
│   └───┘    └─────┘                                               │
│                                                                  │
│   Type 3: CVCC (Consonant + Vowel + 2 Consonants)               │
│   ┌───┐    ┌───┬───┐    Example: 삶 (salm → sam)                │
│   │ C V│    │ ㅅ│ ㅏ│    ㅅ (s) + ㅏ (a) + ㄹㅁ (lm) = 삶       │
│   │ CC │    │ ㄹㅁ │                                             │
│   └───┘    └─────┘                                               │
└──────────────────────────────────────────────────────────────────┘
```

### 6.2 Block Layout Rules

The layout depends on the vowel shape:

```
Vertical vowels (ㅏ ㅑ ㅓ ㅕ ㅣ ㅐ ㅒ ㅔ ㅖ):
  ┌───┬───┐        ┌───┬───┐
  │ C │ V │   or   │ C │ V │     C = initial consonant
  └───┴───┘        ├───┴───┤     V = vowel
                   │  C(C) │     C(C) = final consonant(s)
                   └───────┘

Horizontal vowels (ㅗ ㅛ ㅜ ㅠ ㅡ):
  ┌───────┐        ┌───────┐
  │   C   │   or   │   C   │
  ├───────┤        ├───────┤
  │   V   │        │   V   │
  └───────┘        ├───────┤
                   │ C(C)  │
                   └───────┘

Compound vowels (ㅘ ㅙ ㅚ ㅝ ㅞ ㅟ ㅢ):
  ┌───┬───┐        ┌───┬───┐
  │ C │   │   or   │ C │   │
  ├───│ V │        ├───│ V │
  │   │   │        │   │   │
  └───┴───┘        ├───┴───┤
                   │ C(C)  │
                   └───────┘
```

### 6.3 Important Rules

1. **Every syllable block MUST begin with a consonant.** If a syllable starts with a vowel sound, use ㅇ as a silent placeholder: 아 (a) = ㅇ + ㅏ
2. **Every syllable block MUST contain exactly one vowel.**
3. **A syllable block can have 0, 1, or 2 final consonants** (받침).

### 6.4 Reading Examples

| Word | Blocks | Breakdown | Meaning |
|------|--------|-----------|---------|
| 한국 | 한 + 국 | ㅎ+ㅏ+ㄴ, ㄱ+ㅜ+ㄱ | Korea |
| 서울 | 서 + 울 | ㅅ+ㅓ, ㅇ+ㅜ+ㄹ | Seoul |
| 학교 | 학 + 교 | ㅎ+ㅏ+ㄱ, ㄱ+ㅛ | school |
| 사람 | 사 + 람 | ㅅ+ㅏ, ㄹ+ㅏ+ㅁ | person |
| 감사 | 감 + 사 | ㄱ+ㅏ+ㅁ, ㅅ+ㅏ | gratitude |

## 7. 받침 (Batchim) — Final Consonants

### 7.1 What is 받침?

받침 (batchim, literally "support") refers to the consonant(s) at the bottom of a syllable block. While 19 consonants and 11 consonant clusters can appear as 받침, they are all pronounced as one of **only 7 representative sounds**:

```
┌──────────────────────────────────────────────────────────────────┐
│              The 7 받침 Representative Sounds                     │
│                                                                  │
│   Sound [k̚]:  ㄱ ㅋ ㄲ ㄳ ㄺ                                    │
│   Sound [n]:   ㄴ ㄵ ㄶ                                          │
│   Sound [t̚]:  ㄷ ㅌ ㅅ ㅆ ㅈ ㅊ ㅎ                               │
│   Sound [l]:   ㄹ ㄼ ㄽ ㄾ ㅀ                                    │
│   Sound [m]:   ㅁ ㄻ                                             │
│   Sound [p̚]:  ㅂ ㅍ ㄿ ㅄ                                       │
│   Sound [ŋ]:   ㅇ                                                │
│                                                                  │
│   ̚ = unreleased (the sound stops but air is not released)       │
└──────────────────────────────────────────────────────────────────┘
```

### 7.2 Double 받침 (겹받침)

When two consonants appear in the 받침 position and no vowel follows, only one is pronounced. The general rule:

- **Left consonant is usually pronounced**: ㄳ→[k̚], ㄵ→[n], ㄼ→[l], ㄽ→[l], ㄾ→[l], ㅄ→[p̚]
- **Right consonant is pronounced for**: ㄺ→[k̚], ㄻ→[m], ㄿ→[p̚]

| 겹받침 | Pronounced as | Example | Pronunciation |
|--------|--------------|---------|---------------|
| ㄳ | [k̚] | 몫 (share) | [mok̚] |
| ㄵ | [n] | 앉다 (to sit) | [an.tta] |
| ㄶ | [n] | 많다 (many) | [man.ta] |
| ㄺ | [k̚] | 읽다 (to read) | [ik̚.tta] |
| ㄻ | [m] | 삶 (life) | [sam] |
| ㄼ | [l] | 여덟 (eight) | [jʌ.dʌl] |

> **Important**: When a vowel follows a double 받침, both consonants are pronounced — the first stays in the 받침, and the second moves to the next syllable's initial position.

## 8. Pronunciation Rules

### 8.1 연음 (Yeonum) — Linking

When a syllable ending in a consonant is followed by a syllable beginning with ㅇ (silent), the final consonant "links" to become the initial consonant of the next syllable.

| Written | Linked Pronunciation | Meaning |
|---------|---------------------|---------|
| 한국어 | [한구거] (han-gu-geo) | Korean language |
| 음악 | [으막] (eu-mak) | music |
| 먹어요 | [머거요] (meo-geo-yo) | eat (polite) |
| 있어요 | [이써요] (i-sseo-yo) | there is (polite) |
| 읽어요 | [일거요] (il-geo-yo) | read (polite) |

### 8.2 비음화 (Bieumhwa) — Nasalization

When a stop consonant (ㄱ, ㄷ, ㅂ) in 받침 is followed by a nasal consonant (ㄴ, ㅁ), the stop becomes nasal:

```
ㄱ [k̚] + ㄴ/ㅁ → ㅇ [ŋ]    (velar stop → velar nasal)
ㄷ [t̚] + ㄴ/ㅁ → ㄴ [n]    (alveolar stop → alveolar nasal)
ㅂ [p̚] + ㄴ/ㅁ → ㅁ [m]    (bilabial stop → bilabial nasal)
```

| Written | Pronounced | Rule Applied | Meaning |
|---------|-----------|-------------|---------|
| 학년 | [항년] | ㄱ → ㅇ before ㄴ | school year |
| 작년 | [장년] | ㄱ → ㅇ before ㄴ | last year |
| 받는 | [반는] | ㄷ → ㄴ before ㄴ | receiving |
| 합니다 | [함니다] | ㅂ → ㅁ before ㄴ | do (formal) |
| 십만 | [심만] | ㅂ → ㅁ before ㅁ | 100,000 |

### 8.3 유기음화 (Yugieumhwa) — Aspiration

When ㅎ meets a plain stop consonant (ㄱ, ㄷ, ㅂ, ㅈ), they combine into the aspirated version:

```
ㅎ + ㄱ → ㅋ      ㄱ + ㅎ → ㅋ
ㅎ + ㄷ → ㅌ      ㄷ + ㅎ → ㅌ
ㅎ + ㅂ → ㅍ      ㅂ + ㅎ → ㅍ
ㅎ + ㅈ → ㅊ      ㅈ + ㅎ → ㅊ
```

| Written | Pronounced | Rule Applied | Meaning |
|---------|-----------|-------------|---------|
| 좋다 | [조타] | ㅎ + ㄷ → ㅌ | to be good |
| 놓고 | [노코] | ㅎ + ㄱ → ㅋ | putting down |
| 입학 | [이팍] | ㅂ + ㅎ → ㅍ | enrollment |
| 급히 | [그피] | ㅂ + ㅎ → ㅍ | urgently |

### 8.4 구개음화 (Gugaeumhwa) — Palatalization

When ㄷ or ㅌ in 받침 is followed by the vowel ㅣ (specifically in the suffixes 이 and 히), they become ㅈ and ㅊ respectively:

| Written | Pronounced | Rule | Meaning |
|---------|-----------|------|---------|
| 같이 | [가치] | ㅌ + 이 → ㅊ + 이 | together |
| 굳이 | [구지] | ㄷ + 이 → ㅈ + 이 | obstinately |
| 해돋이 | [해도지] | ㄷ + 이 → ㅈ + 이 | sunrise |

## 9. Romanization (Revised Romanization System)

The **Revised Romanization of Korean** (국어의 로마자 표기법) was adopted by the South Korean government in 2000. It replaced the older McCune-Reischauer system.

### 9.1 Key Rules

| Korean | Romanization | Note |
|--------|-------------|------|
| ㄱ | g (initial) / k (final) | Voiced between vowels |
| ㄷ | d (initial) / t (final) | |
| ㅂ | b (initial) / p (final) | |
| ㄹ | r (initial) / l (final) | |
| ㅓ | eo | NOT "o" — this is the biggest mistake English speakers make |
| ㅡ | eu | NOT "u" |
| ㅐ | ae | |
| ㅔ | e | |
| ㅢ | ui | |

### 9.2 Examples

| Korean | Romanization | Note |
|--------|-------------|------|
| 부산 | Busan | City name |
| 인천 | Incheon | City name |
| 독립 | dongnip | ㄱ + ㄹ → [ŋ + n], pronounced "dongnip" |
| 한라산 | Hallasan | Mountain name — ㄴ + ㄹ → [l + l] |
| 종로 | Jongno | District in Seoul |

> **Caution**: Romanization is a crutch. Learn to read Hangul directly as soon as possible. Romanization can be misleading (e.g., "eo" does NOT sound like English "eo").
