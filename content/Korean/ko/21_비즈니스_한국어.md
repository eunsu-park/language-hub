## 1. 기업 위계와 직급

### 1.1 한국 기업 직급 (직급)

한국의 직장은 엄격한 위계 체계로 운영됩니다. 직급은 매우 중요하며, 잘못된 직급을 사용하면 실례가 되거나 당혹스러운 상황을 초래할 수 있습니다.

| 직급 | 한국어 | 높임 형태 | 대략적 영어 대응 |
|---|---|---|---|
| 회장 | 회장 | 회장님 | Chairman / Chairperson |
| 사장 / CEO | 사장 | 사장님 | President / CEO |
| 부사장 | 부사장 | 부사장님 | Vice President |
| 전무이사 | 전무 | 전무님 | Managing Director |
| 상무이사 | 상무 | 상무님 | Director |
| 부장 | 부장 | 부장님 | General Manager |
| 차장 | 차장 | 차장님 | Deputy General Manager |
| 과장 | 과장 | 과장님 | Manager |
| 대리 | 대리 | 대리님 | Assistant Manager |
| 사원 | 사원 | — | Staff / Associate |
| 인턴 | 인턴 | — | Intern |

> **참고**: 님은 높임 접미사입니다. 상대방의 직급을 부를 때 항상 붙이며, 자신을 소개할 때는 절대 사용하지 않습니다.

### 1.2 동료 호칭법

한국 직장에서 상대방을 부르는 기본 공식:

**[성] + [직급]님**

| 상황 | 예시 | 비고 |
|---|---|---|
| 김씨 성의 과장을 부를 때 | **김 과장님** | 가장 일반적인 패턴 |
| 부장을 부를 때 | **부장님** | 높은 직급에는 직급만 사용 가능 |
| 동료를 이름으로 부를 때 | **김민수 씨** | 동기나 후배에게만; 씨는 중립적 표현 |
| CEO를 부를 때 | **사장님** | 항상 직급만 사용; 이름은 절대 사용하지 않음 |

**규칙**:
- 윗사람을 이름만으로 부르지 마세요 — 항상 직급 + 님을 사용합니다.
- 동기 사이에서는 이름 + 씨를 사용할 수 있습니다 (예: 민수 씨).
- 자기소개 시: 저는 마케팅부 김민수 대리입니다. (저는 마케팅부 김민수 대리입니다.)

### 1.3 직장 어휘

| 한국어 | 영어 |
|---|---|
| 회사 | company |
| 사무실 | office |
| 부서 | department |

## 2. 격식체 비즈니스 화법

### 2.1 격식체 표현 (격식체)

비즈니스 상황에서 한국어는 표준 공손 화법(해요체)을 넘어서는 표현을 사용합니다. 이러한 격식체 패턴은 하십시오체와 특별한 겸양/존경 어휘를 사용합니다.

| 일반 공손체 | 격식체 비즈니스 표현 | 의미 |
|---|---|---|
| 말할게요 | **말씀드리겠습니다** | 말씀드리겠습니다 (겸양) |
| 보고할게요 | **보고드리겠습니다** | 보고드리겠습니다 (겸양) |
| 부탁할게요 | **부탁드리겠습니다** | 부탁드리겠습니다 (겸양) |
| 확인할게요 | **확인해 드리겠습니다** | 확인해 드리겠습니다 (겸양) |
| 알려 줄게요 | **알려 드리겠습니다** | 알려 드리겠습니다 (겸양) |
| 전달할게요 | **전달해 드리겠습니다** | 전달해 드리겠습니다 (겸양) |

> **패턴**: **동사 어간 + 드리겠습니다**는 윗사람에게 행위/서비스를 제공하는 겸양 표현입니다. 드리다(겸양의 "주다")에 -겠-(의지)과 -습니다(격식체 어미)를 결합한 형태입니다.

### 2.2 회의 표현 (회의 표현)

**회의 시작**:
- 지금부터 회의를 시작하겠습니다. (지금부터 회의를 시작하겠습니다.)
- 오늘 회의 안건은 ~입니다. (오늘 회의 안건은 ~입니다.)
- 먼저 지난 회의 내용을 간단히 정리하겠습니다. (먼저 지난 회의 내용을 간단히 정리하겠습니다.)

**회의 진행 중**:
- 의견 있으시면 말씀해 주십시오. (의견이 있으시면 말씀해 주십시오.)
- 이 점에 대해서 어떻게 생각하십니까? (이 점에 대해서 어떻게 생각하십니까?)
- 한 가지 제안이 있습니다. (한 가지 제안이 있습니다.)
- 동의합니다. / 동의하기 어렵습니다. (동의합니다. / 동의하기 어렵습니다.)
- 보충 설명을 드리겠습니다. (보충 설명을 드리겠습니다.)
- 자료를 보시면 아시겠지만... (자료를 보시면 아시겠지만...)

**회의 마무리**:
- 오늘 회의는 여기까지 하겠습니다. (오늘 회의는 여기까지 하겠습니다.)
- 다음 회의는 ~에 예정되어 있습니다. (다음 회의는 ~에 예정되어 있습니다.)
- 회의록은 이메일로 공유드리겠습니다. (회의록은 이메일로 공유드리겠습니다.)

### 2.3 윗사람에게 보고하기 (보고 표현)

| 표현 | 의미 | 사용 상황 |
|---|---|---|
| ~에 대해 보고드리겠습니다 | ~에 대해 보고드리겠습니다 | 보고 시작 |
| 현재 상황은 ~입니다 | 현재 상황은 ~입니다 | 현황 보고 |
| ~한 결과, ~인 것으로 나타났습니다 | ~한 결과, ~인 것으로 나타났습니다 | 결과 제시 |
| 문제점은 ~이고, 대안은 ~입니다 | 문제점은 ~이고, 대안은 ~입니다 | 문제와 해결책 |
| 승인해 주시면 감사하겠습니다 | 승인해 주시면 감사하겠습니다 | 승인 요청 |
| 검토해 주시기 바랍니다 | 검토해 주시기 바랍니다 | 검토 요청 |

**보고 시작 예시**:

> 부장님, 신규 프로젝트 진행 상황에 대해 보고드리겠습니다. 현재 1단계는 완료되었고, 2단계는 다음 주부터 시작할 예정입니다. 예산 관련 문제점이 있어 검토해 주시기 바랍니다.

## 3. 업무 이메일 작성

### 3.1 이메일 구조

한국어 업무 이메일은 명확하고 정형화된 구조를 따릅니다:

```
수신 (To):        김영수 부장님
참조 (CC):        이지은 과장님
제목 (Subject):   [마케팅부] 3월 월간 보고서 제출의 건

본문 (Body):

김영수 부장님께,

안녕하십니까. 마케팅부 박민호 대리입니다.

[이메일 본문 내용]

검토해 주시면 감사하겠습니다.
잘 부탁드립니다.

마케팅부 대리 박민호 드림
```

### 3.2 이메일 구성 요소

**인사말**:
| 표현 | 사용법 |
|---|---|
| [이름] [직급]님께, | 일반적인 시작 (예: 김부장님께,) |
| 안녕하십니까. | 격식체 인사 |
| [부서] [직급] [이름]입니다. | 자기소개 |

**제목 작성 규칙**:
- 부서명은 대괄호 사용: [마케팅부]
- ~의 건으로 마무리: 회의 일정 변경의 건 (회의 일정 변경의 건)
- 구체적이고 간결하게 작성

**본문에서 자주 사용하는 표현**:

| 한국어 | 영어 |
|---|---|
| 첨부 파일을 확인해 주시기 바랍니다 | Please check the attached file |
| 회신 부탁드립니다 | Please reply |
| 검토해 주시기 바랍니다 | Please review |
| 확인 부탁드립니다 | Please confirm |
| 아래 내용을 공유드립니다 | I am sharing the following information |
| 양해 부탁드립니다 | I ask for your understanding |
| 불편을 드려 죄송합니다 | I apologize for the inconvenience |
| ~까지 제출해 주시면 감사하겠습니다 | I would appreciate if you submit by ~ |

**마무리 인사**:
| 한국어 | 사용법 |
|---|---|
| 감사합니다. | 일반적인 마무리 |
| 잘 부탁드립니다. | 협조를 부탁하는 표현 |
| 좋은 하루 되십시오. | 좋은 하루 되십시오. (매우 격식체) |
| [부서] [직급] [이름] 드림 | 서명 (드림 = "올림," 겸양) |

### 3.3 업무 이메일 예시

**이메일 1: 보고서 제출**

```
제목: [영업부] 2분기 실적 보고서 제출의 건

김영수 부장님께,

안녕하십니까. 영업부 이지은 과장입니다.

2분기 실적 보고서를 첨부 파일로 제출드립니다.
주요 내용을 간단히 요약드리면 다음과 같습니다.

1. 매출 목표 대비 112% 달성
2. 신규 고객 15건 확보
3. 3분기 전략 제안

검토해 주시면 감사하겠습니다.
궁금하신 점이 있으시면 말씀해 주십시오.

영업부 과장 이지은 드림
```

**이메일 2: 회의 요청**

```
제목: 프로젝트 관련 회의 요청의 건

박수진 차장님께,

안녕하십니까. 기획부 최동훈 대리입니다.

신규 프로젝트 관련하여 회의를 요청드립니다.
아래 일정 중 가능하신 시간을 알려 주시면 감사하겠습니다.

- 3월 15일 (수) 오전 10시
- 3월 16일 (목) 오후 2시
- 3월 17일 (금) 오전 11시

회신 부탁드립니다.
감사합니다.

기획부 대리 최동훈 드림
```

## 4. 발표 표현

### 4.1 발표 시작하기

| 한국어 | 영어 |
|---|---|
| 안녕하십니까. [이름]입니다. | Hello. I am [Name]. |
| 지금부터 발표를 시작하겠습니다. | I will now begin the presentation. |
| 오늘 발표 주제는 ~입니다. | Today's presentation topic is ~. |
| 발표 순서는 다음과 같습니다. | The presentation order is as follows. |
| 총 ~분 정도 소요될 예정입니다. | It is expected to take about ~ minutes. |

**시작 예시**:

> 안녕하십니까. 마케팅부 김수진입니다. 지금부터 "2024년 마케팅 전략"에 대해 발표를 시작하겠습니다. 발표 순서는 다음과 같습니다. 첫째, 시장 분석. 둘째, 전략 제안. 셋째, 예산 계획입니다. 총 20분 정도 소요될 예정이며, 질문은 발표 후에 받겠습니다.

### 4.2 전환 표현

| 한국어 | 영어 | 사용법 |
|---|---|---|
| 먼저 | First | 첫 번째 항목 시작 |
| 다음으로 | Next | 다음 항목으로 이동 |
| 그다음에 | After that | 순서 계속 |
| 한편 | Meanwhile / On the other hand | 대조적 관점 |
| 이와 관련하여 | In relation to this | 이전 내용과 연결 |
| 이 그래프에서 보시다시피 | As you can see in this graph | 시각 자료 참조 |
| 마지막으로 | Lastly | 마지막 항목 |
| 정리하면 | To summarize | 요약 시작 |

### 4.3 마무리와 질의응답

| 한국어 | 영어 |
|---|---|
| 이상으로 발표를 마치겠습니다. | This concludes my presentation. |
| 지금까지 들어 주셔서 감사합니다. | Thank you for listening. |
| 질문 있으시면 말씀해 주십시오. | If you have questions, please speak up. |
| 좋은 질문 감사합니다. | Thank you for the good question. |
| 말씀하신 부분에 대해 답변드리겠습니다. | I will answer regarding the point you mentioned. |
| 추후 자료로 공유드리겠습니다. | I will share the materials later. |

## 5. 전화 예절

### 5.1 전화 받기와 걸기

**전화 받기**:
| 한국어 | 영어 |
|---|---|
| 여보세요. [회사] [부서] [이름]입니다. | Hello. This is [Name] from [Department] at [Company]. |
| 네, 말씀하세요. | Yes, please go ahead. |

**전화 걸기**:
| 한국어 | 영어 |
|---|---|
| 여보세요. 저는 [회사] [이름]입니다. | Hello. I am [Name] from [Company]. |
| [이름] [직급]님 계시면 바꿔 주시겠습니까? | Could you transfer me to [Name] [Title] if they are available? |
| ~건으로 전화드렸습니다. | I am calling regarding ~. |

### 5.2 전화 통화 주요 표현

| 한국어 | 영어 |
|---|---|
| 잠시만 기다려 주십시오. | Please wait a moment. |
| 지금 자리에 안 계십니다. | They are not at their desk right now. |
| 메모 남겨 드릴까요? | Shall I take a message? |
| 전화번호를 남겨 주시겠습니까? | Could you leave your phone number? |
| 나중에 다시 전화 드리겠습니다. | I will call again later. |
| 돌아오시면 전화 부탁드린다고 전해 주십시오. | Please tell them to call me when they return. |
| 전화 잘못 거셨습니다. | You have the wrong number. |

### 5.3 전화 통화 예시

> **A**: 여보세요. 한국전자 영업부 박준혁입니다.
>
> **B**: 안녕하십니까. 대한무역 김소영입니다. 이정민 과장님 계시면 바꿔 주시겠습니까?
>
> **A**: 죄송합니다만, 지금 회의 중이십니다. 메모 남겨 드릴까요?
>
> **B**: 네, 회의 끝나시면 전화 부탁드린다고 전해 주십시오. 제 번호는 010-1234-5678입니다.
>
> **A**: 네, 전달해 드리겠습니다. 감사합니다.

## 6. 연습 문제

### 연습 1: 직급 채우기

각 상황에 맞는 올바른 직급을 고르세요.

1. 마케팅부의 부서장 → 마케팅부 ______님
2. 새로 입사한 직원 → ______
3. 회사 대표이사 → ______님
4. 같은 직급의 동료 이수진 → ______

<details>
<summary>정답</summary>

1. 마케팅부 **부장**님
2. **사원** (또는 신입 사원)
3. **사장**님
4. 이수진 **씨** (또는 수진 씨)

</details>

### 연습 2: 격식체 비즈니스 표현으로 변환

다음 비격식 문장을 격식체 비즈니스 표현으로 바꾸세요.

1. 내일 보고할게. → _______________
2. 이메일 보낼게. → _______________
3. 확인해 볼게. → _______________
4. 회의 시작할게. → _______________

<details>
<summary>정답</summary>

1. 내일 보고**드리겠습니다**.
2. 이메일 보내**드리겠습니다**.
3. 확인**해 드리겠습니다**.
4. 회의를 시작**하겠습니다**.

</details>

### 연습 3: 업무 이메일 작성

다음 상황에 맞는 업무 이메일을 작성하세요:

- 본인: 인사부 박지영 대리
- 수신자: 최영호 부장님
- 주제: 다음 달 사내 워크숍 안내
- 포함 내용: 일시 (4월 5일), 장소 (본사 대회의실), 부서별 참석자 명단 요청

<details>
<summary>모범 답안</summary>

```
제목: [인사부] 사내 워크숍 안내의 건

최영호 부장님께,

안녕하십니까. 인사부 박지영 대리입니다.

다음 달 사내 워크숍에 대해 안내드립니다.

- 일시: 4월 5일 (금) 오전 10시
- 장소: 본사 대회의실

부서별 참석자 명단을 3월 25일까지 회신해 주시면 감사하겠습니다.

궁금하신 점이 있으시면 말씀해 주십시오.
감사합니다.

인사부 대리 박지영 드림
```

</details>

### 연습 4: 전화 역할극

시나리오를 읽고 적절한 한국어 대화를 작성하세요.

**시나리오**: 삼성전자에 전화하여 한지민 대리와 통화하려고 합니다. 안내원이 출장 중이며 다음 주에 돌아올 예정이라고 알려 줍니다.

<details>
<summary>모범 답안</summary>

> **나**: 여보세요. 저는 현대기업 김태호입니다. 한지민 대리님 계시면 바꿔 주시겠습니까?
>
> **안내원**: 죄송합니다만, 한 대리님은 현재 출장 중이십니다. 다음 주 월요일에 복귀하실 예정입니다.
>
> **나**: 그렇습니까. 그러면 돌아오시면 전화 부탁드린다고 전해 주십시오. 제 번호는 010-9876-5432입니다.
>
> **안내원**: 네, 전달해 드리겠습니다.
>
> **나**: 감사합니다. 좋은 하루 되십시오.

</details>

## 핵심 정리

1. **직급은 매우 중요합니다** — 한국 비즈니스 문화에서 윗사람에게는 항상 [성] + [직급]님을 사용합니다
2. **드리다**는 비즈니스의 핵심 겸양 동사입니다: 말씀드리다, 보고드리다, 전달드리다
3. **업무 이메일**은 엄격한 형식을 따릅니다: 인사 → 자기소개 → 본문 → 마무리
4. **-겠습니다**는 정중한 의지를 전달하며 격식체 비즈니스 화법의 핵심입니다
5. **전화 예절**은 정형화된 표현을 사용합니다 — 자연스러운 대화를 위해 핵심 표현을 암기하세요
