## 1. 청유형: "~하자"

한국어에는 격식 수준에 따라 "~하자"를 표현하는 두 가지 주요 방법이 있습니다.

### 1.1 비격식: -자

동사 어간에 **-자**를 직접 붙입니다. 친한 친구, 또래, 손아랫사람에게 사용합니다.

| 동사 | 어간 | 청유형 | 뜻 |
|------|------|--------|-----|
| 가다 (가다) | 가 | 가자! | 가자! |
| 먹다 (먹다) | 먹 | 먹자! | 먹자! |
| 하다 (하다) | 하 | 하자! | 하자! |
| 놀다 (놀다) | 놀 | 놀자! | 놀자! |
| 쉬다 (쉬다) | 쉬 | 쉬자! | 쉬자! |

```
같이 가자!
gachi gaja!
같이 가자!

점심 먹자.
jeomsim meokja.
점심 먹자.

내일 만나자.
naeil mannaja.
내일 만나자.
```

### 1.2 격식 존댓말: -(으)ㅂ시다

격식이나 존댓말 상황 — 회의, 윗사람에게, 공적인 상황에서 사용합니다.

| 어간 끝 | 규칙 | 예시 |
|---------|------|------|
| 모음 | + ㅂ시다 | 가 → 갑시다 |
| 자음 | + 읍시다 | 먹 → 먹읍시다 |
| ㄹ (ㄹ 탈락) | + ㅂ시다 | 놀 → 놉시다 |

| 한국어 | 영어 번역 |
|--------|----------|
| 같이 갑시다. | Let's go together. |
| 시작합시다. | Let's begin. |
| 회의를 시작합시다. | Let's start the meeting. |
| 건배합시다! | Let's toast! (cheers) |

### 1.3 부드러운 존댓말: -(으)ㄹ까요?

의문문 형태의 부드러운 제안 — "~할까요?"

| 한국어 | 영어 번역 |
|--------|----------|
| 같이 갈까요? | Shall we go together? |
| 뭐 먹을까요? | What shall we eat? |
| 커피 마실까요? | Shall we have coffee? |

## 2. 명령형: 명령과 요청

> **명령형 (명령형)**
>
> | 존댓말 수준 | 형태 | 예시 |
> |------------|------|------|
> | 정중한 요청 | -(으)세요 | 앉으세요 (앉으세요) |
> | 일상/반말 | -아/어라 | 앉아라 (앉아라) |
> | 부드러운/친근 | -아/어 | 앉아 (앉아) |
> | 아주 격식적 | -(으)십시오 | 앉으십시오 (앉으십시오) |
>
> 일상생활에서 가장 많이 쓰는 형태: **-(으)세요**

### 2.1 정중한 요청: -(으)세요

가장 다양하게 쓰이고 자주 사용되는 명령형입니다. 정중하고 예의 바릅니다.

| 어간 끝 | 규칙 | 예시 |
|---------|------|------|
| 모음 | + 세요 | 보다 → 보세요 (보세요) |
| 자음 | + 으세요 | 읽다 → 읽으세요 (읽으세요) |
| ㄹ (ㄹ 탈락) | + 세요 | 만들다 → 만드세요 (만드세요) |

| 한국어 | 영어 번역 |
|--------|----------|
| 앉으세요. | Please sit down. |
| 여기 보세요. | Please look here. |
| 천천히 말씀해 주세요. | Please speak slowly. |
| 드세요. | Please eat. (먹다의 높임말) |
| 조심하세요. | Please be careful. |

### 2.2 일상 명령: -아/어라

부모가 자녀에게, 아주 친한 사이에서, 또는 서면 지시문에 사용합니다.

| 한국어 | 영어 번역 |
|--------|----------|
| 빨리 와라! | Come quickly! |
| 여기 앉아라. | Sit here. |
| 공부해라. | Study! |
| 조용히 해라. | Be quiet! |

> **주의**: -아/어라는 상대에게 따라 권위적이거나 무례하게 들릴 수 있습니다. 확신이 없을 때는 -(으)세요를 사용하세요.

### 2.3 부정 명령: -지 마세요 / -지 마

| 존댓말 | 반말 | 뜻 |
|--------|------|-----|
| 가지 마세요. | 가지 마. | 가지 마세요. |
| 걱정하지 마세요. | 걱정하지 마. | 걱정하지 마세요. |
| 만지지 마세요. | 만지지 마. | 만지지 마세요. |

## 3. 희망 표현: -고 싶다

동사 어간에 **-고 싶다**를 붙여 "~하고 싶다"를 표현합니다.

### 3.1 만드는 법

```
동사 어간 + 고 싶다  →  "~하고 싶다"

현재: -고 싶어요     (~하고 싶어요)
과거: -고 싶었어요    (~하고 싶었어요)
```

### 3.2 예문

| 한국어 | 영어 번역 |
|--------|----------|
| 한국에 가고 싶어요. | I want to go to Korea. |
| 뭐 하고 싶어요? | What do you want to do? |
| 커피 마시고 싶어요. | I want to drink coffee. |
| 한국어를 잘 하고 싶어요. | I want to be good at Korean. |
| 어제 쉬고 싶었어요. | I wanted to rest yesterday. |

### 3.3 3인칭: -고 싶어하다

**다른 사람**의 희망을 말할 때는 -고 싶어하다를 사용합니다:

| 한국어 | 영어 번역 |
|--------|----------|
| 동생이 아이스크림을 먹고 싶어해요. | My younger sibling wants to eat ice cream. |
| 친구가 한국에 가고 싶어해요. | My friend wants to go to Korea. |

> **왜 구별할까요?** 한국어에서는 자신의 감정(-고 싶다)과 다른 사람의 관찰된 감정(-고 싶어하다)을 구별합니다. 이는 타인의 내면 상태를 직접 알 수 없다는 문화적 원칙을 반영합니다.

## 4. 시도 표현: -아/어 보다

동사에 **-아/어 보다**를 붙여 "~해 보다"를 표현합니다.

### 4.1 만드는 법

해요체와 같은 모음조화 규칙을 적용합니다:

| 동사 | -아/어 보다 | 존댓말 | 뜻 |
|------|-----------|--------|-----|
| 먹다 | 먹어 보다 | 먹어 보세요 | 먹어 보세요 |
| 가다 | 가 보다 | 가 보세요 | 가 보세요 |
| 하다 | 해 보다 | 해 보세요 | 해 보세요 |
| 입다 (입다) | 입어 보다 | 입어 보세요 | 입어 보세요 |
| 읽다 (읽다) | 읽어 보다 | 읽어 보세요 | 읽어 보세요 |

### 4.2 예문

| 한국어 | 영어 번역 |
|--------|----------|
| 이것 먹어 보세요. | Please try eating this. |
| 한번 해 보세요. | Please try doing it once. |
| 한국에 가 봤어요? | Have you been to Korea? (가 본 적 있어요?) |
| 이 책 읽어 봤어요. | I've tried reading this book. |
| 김치를 먹어 봤어요? | Have you tried kimchi? |

> **과거형 뉘앙스**: -아/어 봤어요는 "~해 본 적이 있다"라는 뜻입니다. 생애 경험을 나타내며, 영어의 "Have you ever...?"와 비슷합니다.

## 5. 진행형: -고 있다

동사 어간에 **-고 있다**를 붙여 진행 중인 행위("~하고 있다")를 표현합니다.

### 5.1 만드는 법

```
동사 어간 + 고 있다  →  "~하고 있다"

현재: -고 있어요      (~하고 있어요)
과거: -고 있었어요     (~하고 있었어요)
```

### 5.2 예문

| 한국어 | 영어 번역 |
|--------|----------|
| 지금 밥을 먹고 있어요. | I'm eating right now. |
| 음악을 듣고 있어요. | I'm listening to music. |
| 친구를 기다리고 있어요. | I'm waiting for a friend. |
| 한국어를 공부하고 있어요. | I'm studying Korean. |
| 뭐 하고 있어요? | What are you doing? |
| 그때 자고 있었어요. | I was sleeping at that time. |

### 5.3 착용/소지 (상태)

-고 있다는 착용이나 소지와 같은 **지속 상태**를 나타내기도 합니다:

| 한국어 | 영어 번역 |
|--------|----------|
| 안경을 쓰고 있어요. | I'm wearing glasses. |
| 모자를 쓰고 있어요. | I'm wearing a hat. |
| 가방을 들고 있어요. | I'm carrying a bag. |

## 6. 허가 표현: -아/어도 되다

**-아/어도 되다**를 사용하여 허가를 구하거나 허가를 줍니다 ("~해도 돼요?" / "~해도 돼요.").

### 6.1 만드는 법

```
동사 어간 + 아/어도 되다  →  "~해도 되다"

질문:  -아/어도 돼요?     (~해도 돼요?)
대답:  네, -아/어도 돼요.  (네, ~해도 돼요.)
```

### 6.2 예문

| 한국어 | 영어 번역 |
|--------|----------|
| 여기 앉아도 돼요? | May I sit here? |
| 사진 찍어도 돼요? | May I take a photo? |
| 들어가도 돼요? | May I come in? |
| 이거 먹어도 돼요? | May I eat this? |
| 창문 열어도 돼요? | May I open the window? |

### 6.3 허가와 거절

| 응답 | 한국어 | 영어 번역 |
|------|--------|----------|
| 네 | 네, -아/어도 돼요. | Yes, you may. |
| 물론 | 그럼요, -아/어도 돼요. | Of course, you may. |
| 아니요 | 아니요, -면 안 돼요. | No, you must not. |

```
A: 여기서 전화해도 돼요?     (여기서 전화해도 돼요?)
B: 네, 괜찮아요. 해도 돼요.   (네, 괜찮아요. 해도 돼요.)

A: 사진 찍어도 돼요?         (사진 찍어도 돼요?)
B: 아니요, 찍으면 안 돼요.    (아니요, 찍으면 안 돼요.)
```

## 7. 금지 표현: -(으)면 안 되다

**-(으)면 안 되다**를 사용하여 금지를 표현합니다 ("~하면 안 돼요").

### 7.1 만드는 법

| 어간 끝 | 규칙 | 예시 |
|---------|------|------|
| 모음 | + 면 안 되다 | 가 → 가면 안 돼요 |
| 자음 | + 으면 안 되다 | 먹 → 먹으면 안 돼요 |

### 7.2 예문

| 한국어 | 영어 번역 |
|--------|----------|
| 여기서 담배를 피우면 안 돼요. | You must not smoke here. |
| 수업 시간에 핸드폰을 쓰면 안 돼요. | You must not use your phone during class. |
| 거짓말하면 안 돼요. | You must not lie. |
| 늦으면 안 돼요. | You must not be late. |
| 여기에 주차하면 안 돼요. | You must not park here. |

### 7.3 자주 보는 금지 표현

> **자주 쓰는 금지 표현**
>
> - **촬영 금지** — 촬영 금지
> - **주차 금지** — 주차 금지
> - **흡연 금지** — 흡연 금지
> - **출입 금지** — 출입 금지
> - **음식물 반입 금지** — 음식물 반입 금지
> - **만지지 마세요** — 만지지 마세요
> - **뛰지 마세요** — 뛰지 마세요

## 8. 능력 표현: -(으)ㄹ 수 있다/없다

**-(으)ㄹ 수 있다/없다**를 사용하여 능력("~할 수 있다")이나 불능("~할 수 없다")을 표현합니다.

### 8.1 만드는 법

| 어간 끝 | 규칙 | 예시 |
|---------|------|------|
| 모음 | + ㄹ 수 있다 | 가 → 갈 수 있어요 (갈 수 있어요) |
| 자음 | + 을 수 있다 | 먹 → 먹을 수 있어요 (먹을 수 있어요) |
| ㄹ | + 수 있다 (ㄹ 유지) | 만들 → 만들 수 있어요 (만들 수 있어요) |

### 8.2 가능 / 불가능

| 한국어 | 영어 번역 |
|--------|----------|
| 한국어를 할 수 있어요. | I can speak Korean. |
| 한국어를 할 수 없어요. | I cannot speak Korean. |
| 운전할 수 있어요? | Can you drive? |
| 매운 음식을 먹을 수 있어요. | I can eat spicy food. |
| 내일 올 수 있어요? | Can you come tomorrow? |
| 수영할 수 있어요? | Can you swim? |

### 8.3 약식 표현: 못 + 동사

"~할 수 없다"를 더 짧게 표현하는 방법은 동사 앞에 **못**을 놓는 것입니다:

| 긴 형태 | 짧은 형태 | 뜻 |
|---------|----------|-----|
| 갈 수 없어요 | 못 가요 | 못 가요 |
| 먹을 수 없어요 | 못 먹어요 | 못 먹어요 |
| 할 수 없어요 | 못 해요 | 못 해요 |

> **뉘앙스**: 못은 상황이나 능력 부족으로 인한 불가능을 암시합니다. -(으)ㄹ 수 없다는 더 일반적이고 약간 더 격식적입니다.

```
매운 것을 못 먹어요.
maeun geos-eul mot meogeoyo.
매운 것을 못 먹어요. (능력/내성이 없어서)

오늘 못 가요.
oneul mot gayo.
오늘 못 가요. (상황이 안 돼서)
```
