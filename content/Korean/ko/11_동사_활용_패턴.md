## 1. 규칙 활용 복습

불규칙 활용을 배우기 전에 규칙 활용 규칙을 정리합시다.

### 모음 조화 (모음 조화)

한국어에서 아/어로 시작하는 어미는 어간의 **마지막 모음**에 따라 모음 조화를 따릅니다:

| 어간의 마지막 모음 | 어미 | 예시 |
|---|---|---|
| ㅏ 또는 ㅗ (양성 모음) | -아 | 가다 → 가 + 아요 → 가요 |
| 그 외 모음 (음성 모음) | -어 | 먹다 → 먹 + 어요 → 먹어요 |
| 하 | -여 (→ 해) | 하다 → 하 + 여요 → 해요 |

### 축약 규칙

어간의 끝 모음이 어미의 모음과 만나면 축약이 일어납니다:

| 어간 끝 | + 아/어 | 결과 | 예시 |
|---|---|---|---|
| ㅏ + 아 | ㅏ (합침) | 가다 → 가 + 아요 → 가요 |
| ㅗ + 아 | ㅘ | 오다 → 오 + 아요 → 와요 |
| ㅜ + 어 | ㅝ | 배우다 → 배우 + 어요 → 배워요 |
| ㅣ + 어 | ㅕ | 마시다 → 마시 + 어요 → 마셔요 |
| ㅡ + 어 | 상황에 따라 | 쓰다 → 쓰 + 어요 → 써요 |
| ㅐ + 어 | ㅐ (합침) | 보내다 → 보내 + 어요 → 보내요 |

> **핵심 포인트**: 이 축약 규칙은 **모든** -아/어 어미에 적용됩니다: -아/어요, -아/어서, -아/어도, -았/었다 등.

### 다양한 어미에서의 규칙 활용

먹다 (먹다)를 규칙 동사 예시로 사용합니다:

| 어미 유형 | 형태 | 예문 |
|---|---|---|
| -아/어요 (해요체 현재) | 먹어요 | 밥을 먹어요. (밥을 먹어요.) |
| -았/었어요 (해요체 과거) | 먹었어요 | 밥을 먹었어요. (밥을 먹었어요.) |
| -ㅂ니다/습니다 (합쇼체) | 먹습니다 | 밥을 먹습니다. |
| -(으)세요 (존경) | 드세요 | 많이 드세요. (많이 드세요.) |
| -(으)면 (조건) | 먹으면 | 많이 먹으면 배불러요. |
| -고 (나열) | 먹고 | 밥을 먹고 나갔어요. |

## 2. ㅂ 불규칙

**규칙**: 어간이 ㅂ으로 끝나면, 모음으로 시작하는 어미 앞에서 ㅂ이 **우**로 바뀝니다.

> 한국어에서 가장 흔한 불규칙 유형입니다.

### 패턴

| 사전형 | 모음 앞 | -아/어요 | -으면 | -(으)ㄴ |
|---|---|---|---|---|
| 돕다 (돕다) | ㅂ → 우 | 도와요 | 도우면 | 도운 |
| 춥다 (춥다) | ㅂ → 우 | 추워요 | 추우면 | 추운 |
| 가깝다 (가깝다) | ㅂ → 우 | 가까워요 | 가까우면 | 가까운 |
| 어렵다 (어렵다) | ㅂ → 우 | 어려워요 | 어려우면 | 어려운 |
| 쉽다 (쉽다) | ㅂ → 우 | 쉬워요 | 쉬우면 | 쉬운 |
| 덥다 (덥다) | ㅂ → 우 | 더워요 | 더우면 | 더운 |
| 아름답다 (아름답다) | ㅂ → 우 | 아름다워요 | 아름다우면 | 아름다운 |
| 맵다 (맵다) | ㅂ → 우 | 매워요 | 매우면 | 매운 |
| 귀엽다 (귀엽다) | ㅂ → 우 | 귀여워요 | 귀여우면 | 귀여운 |

> **기억법**: ㅂ이 따뜻한 모음을 만나면 "녹아서" 우가 된다고 생각하세요.

### 단계별 과정

1. 춥다 → 다 제거 → 어간: 춥
2. -어요 앞: ㅂ → 우 → 추우 + 어요
3. 축약: 추우 + 어요 → 추워요

### 예문

- 오늘 날씨가 **추워요**. (오늘 날씨가 춥습니다.)
- 그 식당이 집에서 **가까워요**. (그 식당이 집에서 가깝습니다.)
- 한국어가 **어려워요**? (한국어가 어렵습니까?)
- 이 음식이 너무 **매워요**. (이 음식이 너무 맵습니다.)

### ⚠️ 예외: 규칙 ㅂ 동사

이 동사들은 ㅂ으로 끝나지만 **규칙적으로** 활용합니다 (ㅂ이 변하지 않습니다):

| 동사 | -아/어요 | 비고 |
|---|---|---|
| 잡다 (잡다) | 잡아요 | 규칙 |
| 좁다 (좁다) | 좁아요 | 규칙 |
| 씹다 (씹다) | 씹어요 | 규칙 |
| 입다 (입다) | 입어요 | 규칙 |
| 집다 (집다) | 집어요 | 규칙 |
| 업다 (업다) | 업어요 | 규칙 |

> **팁**: 대부분의 ㅂ으로 끝나는 **형용사(서술용언)**는 불규칙입니다. 대부분의 ㅂ으로 끝나는 **동작 동사**는 규칙입니다. 주요 예외는 돕다 (동작 동사이지만 불규칙)입니다.

## 3. ㄷ 불규칙

**규칙**: 어간이 ㄷ으로 끝나면, 모음으로 시작하는 어미 앞에서 ㄷ이 **ㄹ**로 바뀝니다.

### 패턴

| 사전형 | 모음 앞 | -아/어요 | -으면 | -(으)ㄴ |
|---|---|---|---|---|
| 듣다 (듣다) | ㄷ → ㄹ | 들어요 | 들으면 | 들은 |
| 걷다 (걷다) | ㄷ → ㄹ | 걸어요 | 걸으면 | 걸은 |
| 묻다 (묻다, 물어보다) | ㄷ → ㄹ | 물어요 | 물으면 | 물은 |
| 싣다 (싣다) | ㄷ → ㄹ | 실어요 | 실으면 | 실은 |
| 깨닫다 (깨닫다) | ㄷ → ㄹ | 깨달아요 | 깨달으면 | 깨달은 |

### 과정

1. 듣다 → 다 제거 → 어간: 듣
2. -어요 앞: ㄷ → ㄹ → 들 + 어요 → 들어요
3. -고 앞 (자음): 변화 없음 → 듣고

### 예문

- 음악을 **들어요**. (음악을 듣습니다.)
- 공원에서 **걸어요**. (공원에서 걷습니다.)
- 선생님한테 **물어요**. (선생님에게 물어봅니다.)
- 그때 비로소 **깨달았어요**. (그때 비로소 깨달았습니다.)

### 자음으로 시작하는 어미 앞 (변화 없음)

| 어미 | 듣다 | 걷다 |
|---|---|---|
| -고 | 듣고 | 걷고 |
| -지만 | 듣지만 | 걷지만 |
| -습니다 | 듣습니다 | 걷습니다 |

### ⚠️ 예외: 규칙 ㄷ 동사

| 동사 | -아/어요 | 비고 |
|---|---|---|
| 닫다 (닫다) | 닫아요 | 규칙 |
| 받다 (받다) | 받아요 | 규칙 |
| 믿다 (믿다) | 믿어요 | 규칙 |
| 얻다 (얻다) | 얻어요 | 규칙 |
| 묻다 (파묻다) | 묻어요 | 규칙 ("질문하다"의 묻다와 다릅니다!) |

> **주의**: 묻다는 두 가지 뜻이 있습니다. "질문하다"는 불규칙 (물어요)이고, "파묻다"는 규칙 (묻어요)입니다. 문맥으로 구별합니다.

## 4. ㅅ 불규칙

**규칙**: 어간이 ㅅ으로 끝나면, 모음으로 시작하는 어미 앞에서 ㅅ이 **탈락**합니다.

### 패턴

| 사전형 | 모음 앞 | -아/어요 | -으면 | -(으)ㄴ |
|---|---|---|---|---|
| 짓다 (짓다) | ㅅ 탈락 | 지어요 | 지으면 | 지은 |
| 낫다 (낫다) | ㅅ 탈락 | 나아요 | 나으면 | 나은 |
| 잇다 (잇다) | ㅅ 탈락 | 이어요 | 이으면 | 이은 |
| 젓다 (젓다) | ㅅ 탈락 | 저어요 | 저으면 | 저은 |
| 붓다 (붓다) | ㅅ 탈락 | 부어요 | 부으면 | 부은 |

### 과정

1. 짓다 → 다 제거 → 어간: 짓
2. -어요 앞: ㅅ 탈락 → 지 + 어요 → 지어요
3. -고 앞: 변화 없음 → 짓고

### 예문

- 집을 **지어요**. (집을 짓습니다.)
- 감기가 **나았어요**. (감기가 나았습니다.)
- 이야기를 **이어요**. (이야기를 잇습니다.)
- 눈이 **부었어요**. (눈이 부었습니다.)

### ⚠️ 예외: 규칙 ㅅ 동사

| 동사 | -아/어요 | 비고 |
|---|---|---|
| 벗다 (벗다) | 벗어요 | 규칙 |
| 씻다 (씻다) | 씻어요 | 규칙 |
| 빗다 (빗다) | 빗어요 | 규칙 |
| 웃다 (웃다) | 웃어요 | 규칙 |

## 5. ㄹ 불규칙

**규칙**: 어간이 ㄹ로 끝나면, **ㄴ, ㅂ, ㅅ**으로 시작하는 어미 앞에서 ㄹ이 **탈락**합니다.

> 어간 자체가 변하는 것이 아니라 특정 어미만 ㄹ 탈락을 유발하므로 "반불규칙"이라고도 합니다.

### 패턴

| 사전형 | -ㅂ니다 | -(으)세요 | -ㄴ/는 | -아/어요 |
|---|---|---|---|---|
| 만들다 (만들다) | 만듭니다 | 만드세요 | 만드는 | 만들어요 |
| 살다 (살다) | 삽니다 | 사세요 | 사는 | 살아요 |
| 알다 (알다) | 압니다 | 아세요 | 아는 | 알아요 |
| 팔다 (팔다) | 팝니다 | 파세요 | 파는 | 팔아요 |
| 열다 (열다) | 엽니다 | 여세요 | 여는 | 열어요 |
| 놀다 (놀다) | 놉니다 | 노세요 | 노는 | 놀아요 |
| 울다 (울다) | 웁니다 | 우세요 | 우는 | 울어요 |

### 탈락을 유발하는 세 자음: ㄴ, ㅂ, ㅅ

| 어미 | 유발 자음 | 살다 예시 |
|---|---|---|
| -ㅂ니다 (합쇼체) | ㅂ | 삽니다 |
| -(으)세요 (존경) | ㅅ | 사세요 |
| -는 (현재 관형) | ㄴ | 사는 |
| -(으)ㄴ (과거 관형) | ㄴ | 산 |
| -(으)ㄹ (미래 관형) | — | 살 |

### 다른 자음 앞에서는 변화 없음

| 어미 | 살다 |
|---|---|
| -고 | 살고 |
| -지만 | 살지만 |
| -아/어요 | 살아요 |
| -았/었어요 | 살았어요 |

### 예문

- 서울에 **삽니다**. (서울에 삽니다.) — 합쇼체
- 어디에 **사세요**? (어디에 사세요?) — 존경
- 제가 **사는** 곳은 서울이에요. (제가 사는 곳은 서울입니다.) — 관형
- 그 사실을 **압니다**. (그 사실을 압니다.)
- 문을 **여세요**. (문을 여세요.)

## 6. ㅎ 불규칙

**규칙**: 어간이 ㅎ으로 끝나면, -아/어 어미를 만날 때 ㅎ이 탈락하고 모음이 변합니다. -(으)ㄴ 앞에서도 ㅎ이 탈락합니다.

> 이 규칙은 주로 색깔과 일부 서술용언에 적용됩니다.

### 패턴

| 사전형 | -아/어요 | -(으)ㄴ (관형) | -(으)면 |
|---|---|---|---|
| 파랗다 (파랗다) | 파래요 | 파란 | 파라면 |
| 하얗다 (하얗다) | 하얘요 | 하얀 | 하야면 |
| 빨갛다 (빨갛다) | 빨개요 | 빨간 | 빨가면 |
| 노랗다 (노랗다) | 노래요 | 노란 | 노라면 |
| 까맣다 (까맣다) | 까매요 | 까만 | 까마면 |
| 그렇다 (그렇다) | 그래요 | 그런 | 그러면 |
| 이렇다 (이렇다) | 이래요 | 이런 | 이러면 |
| 어떻다 (어떻다) | 어때요 | 어떤 | — |

### 과정

1. 파랗다 → 다 제거 → 어간: 파랗
2. -아요 앞: ㅎ 탈락 → 파라 + 아요 → 모음 합침 → 파래요
3. -(으)ㄴ 앞: ㅎ 탈락 → 파라 + ㄴ → 파란

### 예문

- 하늘이 **파래요**. (하늘이 파랗습니다.)
- **하얀** 눈이 내려요. (하얀 눈이 내립니다.)
- **빨간** 사과를 샀어요. (빨간 사과를 샀습니다.)
- **그러면** 어떻게 해요? (그러면 어떻게 합니까?)
- **이런** 경우에는 어떻게 해야 돼요? (이런 경우에는 어떻게 해야 합니까?)

## 7. 르 불규칙

**규칙**: 어간이 르로 끝나면, 르가 분리됩니다 — ㄹ이 앞 음절의 받침으로 붙고, 모음 조화에 따라 라 또는 러가 뒤따릅니다.

### 패턴

| 사전형 | 분리 결과 | -아/어요 |
|---|---|---|
| 빠르다 (빠르다) | 빠 + ㄹ + 라 | 빨라요 |
| 모르다 (모르다) | 모 + ㄹ + 라 | 몰라요 |
| 부르다 (부르다) | 부 + ㄹ + 러 | 불러요 |
| 자르다 (자르다) | 자 + ㄹ + 라 | 잘라요 |
| 다르다 (다르다) | 다 + ㄹ + 라 | 달라요 |
| 고르다 (고르다) | 고 + ㄹ + 라 | 골라요 |
| 기르다 (기르다) | 기 + ㄹ + 러 | 길러요 |
| 마르다 (마르다) | 마 + ㄹ + 라 | 말라요 |
| 서두르다 (서두르다) | 서두 + ㄹ + 러 | 서둘러요 |

### 과정

1. 모르다 → 다 제거 → 어간: 모르
2. -아요 앞: 르 → ㄹ라 → 모 + ㄹ + 라요 → 몰라요
3. ㄹ이 모의 받침이 됨 → 몰 + 라요

### 자음으로 시작하는 어미 앞 (변화 없음)

| 어미 | 모르다 |
|---|---|
| -고 | 모르고 |
| -지만 | 모르지만 |
| -ㅂ니다 | 모릅니다 (ㄹ 불규칙도 적용) |

### 예문

- 저는 **몰라요**. (저는 모릅니다.)
- 너무 **빨라요**! (너무 빠릅니다!)
- 이름을 **불러요**. (이름을 부릅니다.)
- 종이를 **잘라요**. (종이를 자릅니다.)
- 고양이를 **길러요**. (고양이를 기릅니다.)

## 8. 피동사

한국어에는 **단형 피동** (파생적)과 **장형 피동** (우언적) 두 가지 피동 구문이 있습니다.

### 단형 피동: 접미사 -이/히/리/기

동사 어간에 접미사를 직접 붙여 피동사를 만듭니다. 어떤 접미사를 쓸지는 규칙이 없으므로 암기해야 합니다.

| 접미사 | 능동 → 피동 | 의미 |
|---|---|---|
| -이 | 보다 → **보이다** | 보다 → 보이다 |
| -이 | 쓰다 → **쓰이다** | 쓰다 → 쓰이다 |
| -이 | 놓다 → **놓이다** | 놓다 → 놓이다 |
| -히 | 잡다 → **잡히다** | 잡다 → 잡히다 |
| -히 | 읽다 → **읽히다** | 읽다 → 읽히다 |
| -히 | 먹다 → **먹히다** | 먹다 → 먹히다 |
| -히 | 닫다 → **닫히다** | 닫다 → 닫히다 |
| -리 | 열다 → **열리다** | 열다 → 열리다 |
| -리 | 팔다 → **팔리다** | 팔다 → 팔리다 |
| -리 | 듣다 → **들리다** | 듣다 → 들리다 |
| -기 | 끊다 → **끊기다** | 끊다 → 끊기다 |
| -기 | 안다 → **안기다** | 안다 → 안기다 |
| -기 | 빼다 → **빼기다** | 빼다 → 빼기다 |

### 단형 피동 예문

- 산이 **보여요**. (산이 보입니다.)
- 이 책이 많이 **읽혀요**. (이 책이 많이 읽힙니다.)
- 문이 **열렸어요**. (문이 열렸습니다.)
- 소리가 **들려요**. (소리가 들립니다.)
- 전화가 **끊겼어요**. (전화가 끊겼습니다.)

### 장형 피동: -아/어지다

이 구문은 거의 모든 동사에 사용할 수 있으며, 특히 서술용언(형용사)과 함께 상태 변화를 나타낼 때 많이 쓰입니다.

| 능동 / 서술용언 | 피동 / 변화 | 의미 |
|---|---|---|
| 만들다 → **만들어지다** | 만들어지다 | 빵이 만들어졌어요. (빵이 만들어졌습니다.) |
| 자르다 → **잘리다** / **잘라지다** | 잘리다 | 나무가 잘렸어요. (나무가 잘렸습니다.) |
| 깨끗하다 → **깨끗해지다** | 깨끗해지다 | 방이 깨끗해졌어요. (방이 깨끗해졌습니다.) |
| 예쁘다 → **예뻐지다** | 예뻐지다 | 점점 예뻐졌어요. (점점 예뻐졌습니다.) |
| 따뜻하다 → **따뜻해지다** | 따뜻해지다 | 날씨가 따뜻해졌어요. (날씨가 따뜻해졌습니다.) |

> **중요**: 단형 피동과 장형 피동을 함께 사용하면 안 됩니다. ~~열려지다~~는 틀린 표현이며 — 열리다를 사용해야 합니다.

## 9. 사동사

한국어 사동사는 "누군가에게 무엇을 하게 하다"를 표현합니다. 피동과 마찬가지로 두 가지 구문이 있습니다.

### 단형 사동: 접미사 -이/히/리/기/우/추

| 접미사 | 기본형 → 사동형 | 의미 |
|---|---|---|
| -이 | 먹다 → **먹이다** | 먹다 → 먹이다 |
| -이 | 보다 → **보이다** | 보다 → 보이다 |
| -이 | 죽다 → **죽이다** | 죽다 → 죽이다 |
| -이 | 녹다 → **녹이다** | 녹다(자동) → 녹이다(타동) |
| -히 | 읽다 → **읽히다** | 읽다 → 읽히다 |
| -히 | 앉다 → **앉히다** | 앉다 → 앉히다 |
| -히 | 입다 → **입히다** | 입다 → 입히다 |
| -리 | 살다 → **살리다** | 살다 → 살리다 |
| -리 | 알다 → **알리다** | 알다 → 알리다 |
| -리 | 울다 → **울리다** | 울다 → 울리다 |
| -기 | 웃다 → **웃기다** | 웃다 → 웃기다 |
| -기 | 벗다 → **벗기다** | 벗다 → 벗기다 |
| -우 | 자다 → **재우다** | 자다 → 재우다 |
| -우 | 타다 → **태우다** | 타다 → 태우다 |
| -우 | 깨다 → **깨우다** | 깨다 → 깨우다 |
| -추 | 맞다 → **맞추다** | 맞다 → 맞추다 |
| -추 | 낮다 → **낮추다** | 낮다 → 낮추다 |

### 단형 사동 예문

- 아이에게 밥을 **먹여요**. (아이에게 밥을 먹입니다.)
- 의사가 환자를 **살렸어요**. (의사가 환자를 살렸습니다.)
- 아기를 **재워요**. (아기를 재웁니다.)
- 그 사람이 나를 **웃겨요**. (그 사람이 저를 웃깁니다.)
- 소리를 **낮추세요**. (소리를 낮추세요.)

### 장형 사동: -게 하다

이 구문은 거의 모든 동사에 사용할 수 있으며 단형보다 생산적입니다.

| 기본 동사 | 사동 | 예문 |
|---|---|---|
| 공부하다 (공부하다) | 공부하**게 하다** | 아이를 공부하게 했어요. (아이에게 공부를 시켰습니다.) |
| 가다 (가다) | 가**게 하다** | 친구를 먼저 가게 했어요. (친구를 먼저 가게 했습니다.) |
| 기다리다 (기다리다) | 기다리**게 하다** | 오래 기다리게 해서 미안해요. (오래 기다리게 해서 미안합니다.) |

### 단형 사동 vs. 장형 사동

| 단형 | 장형 | 뉘앙스 |
|---|---|---|
| 읽히다 | 읽게 하다 | 단형이 더 직접적/강제적 |
| 먹이다 | 먹게 하다 | 단형: 물리적으로 먹임; 장형: 먹게 허락/유도 |
| 웃기다 | 웃게 하다 | 단형: 본질적으로 웃김; 장형: 의도적으로 웃게 함 |

> **참고**: 일부 동사는 **같은 접미사**로 피동과 사동 형태를 모두 가집니다 (예: 읽히다 = "읽히다" 또는 "읽게 하다"). 문맥으로 의미를 결정합니다.

주어진 어미로 활용하세요.

1. 살다 + -ㅂ니다 →
2. 알다 + -(으)세요 →
3. 만들다 + -는 →
4. 놀다 + -아/어요 →
5. 팔다 + -(으)ㄴ (과거 관형) →

<details>

### 연습 4: 피동과 사동

올바른 피동 또는 사동 형태를 고르세요.

1. 문을 닫았어요. → 문이 _______. (문이 닫혔습니다.)
2. 아기가 자요. → 아기를 _______. (아기를 재웁니다.)
3. 소리를 들어요. → 소리가 _______. (소리가 들립니다.)
4. 아이가 밥을 먹어요. → 아이에게 밥을 _______. (아이에게 밥을 먹입니다.)
5. 그 책을 많이 읽어요. → 그 책이 많이 _______. (그 책이 많이 읽힙니다.)

<details>
<summary>정답</summary>

1. **닫혔어요** (피동 -히: 닫히다)
2. **재워요** (사동 -우: 재우다)
3. **들려요** (피동 -리: 들리다)
4. **먹여요** (사동 -이: 먹이다)
5. **읽혀요** (피동 -히: 읽히다)

</details>

### 연습 5: 문장 완성

괄호 안의 동사를 올바른 형태로 빈칸에 채우세요.

1. 날씨가 _______ 집에 있었어요. (춥다, -아/어서)
2. 한국 음식이 _______ 자주 먹어요. (맛있다, -(으)ㄴ데)
3. _______ 음식을 주문하세요. (맵다, -(으)ㄴ 부정: 안)
4. 그 노래를 _______? (듣다, -았/었어요)
5. 서울에서 _______ 지 5년 됐어요. (살다, -(으)ㄴ)

<details>
<summary>정답</summary>

1. **추워서** (ㅂ 불규칙 + -아/어서)
2. **맛있는데** (규칙 — 있다는 -는데 앞에서 규칙)
3. **안 매운** (ㅂ 불규칙 관형: 맵다 → 매운, 부정: 안 매운)
4. **들었어요** (ㄷ 불규칙 + 과거)
5. **산** (ㄴ 앞에서 ㄹ 탈락: 살다 → 산)

</details>

## 불규칙 활용 요약표

| 유형 | 규칙 | 예시 | 주요 예외 |
|---|---|---|---|
| ㅂ | 모음 앞에서 ㅂ → 우 | 춥다 → 추워요 | 잡다, 좁다, 씹다, 입다 |
| ㄷ | 모음 앞에서 ㄷ → ㄹ | 듣다 → 들어요 | 닫다, 받다, 믿다 |
| ㅅ | 모음 앞에서 ㅅ 탈락 | 짓다 → 지어요 | 벗다, 씻다, 빗다 |
| ㄹ | ㄴ, ㅂ, ㅅ 앞에서 ㄹ 탈락 | 살다 → 삽니다 | (예외 없음) |
| ㅎ | ㅎ 탈락, 모음 변화 | 파랗다 → 파래요 | (예외 없음) |
| 르 | 르 → ㄹ + 라/러 | 모르다 → 몰라요 | (예외 없음) |
