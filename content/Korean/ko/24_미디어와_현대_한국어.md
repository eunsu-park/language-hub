## 1. 뉴스 한국어 (뉴스 한국어)

한국어 뉴스 언어(보도문)는 일상 대화 한국어와 상당히 다른 독특한 격식 문체를 사용합니다.

### 1.1 헤드라인 관습

한국어 뉴스 헤드라인은 특정 패턴을 따릅니다:

| 관습 | 예시 | 설명 |
|---|---|---|
| 동사 축약 | 정부, 새 정책 **발표** | 동사가 명사형으로 축약 (발표하다 → 발표) |
| 인용 부호 | "경제 회복 **순항**"...전문가 전망 | 강조를 위한 키워드 인용 |
| 말줄임표 | 물가 상승...국민 부담 **가중** | 지속 또는 결과를 함축 |
| 의문형 | 부동산 시장, **회복세** 이어질까? | 기사의 핵심 질문 제기 |
| 조사 생략 | 서울시 대중교통 요금 인상 | 헤드라인에서는 조사를 자주 생략 |

**실제 스타일 헤드라인**:
- 정부, 내년 최저임금 인상안 확정 (정부, 내년 최저임금 인상안 확정)
- "한류 열풍" 동남아 한국어 학습자 급증 ("한류 열풍" 동남아 한국어 학습자 급증)
- 수출 호조에 경제성장률 상향 조정 (수출 호조에 경제성장률 상향 조정)

### 1.2 뉴스 주요 어휘

| 한국어 | 영어 | 분야 |
|---|---|---|
| 보도 | report, coverage | 미디어 |
| 발표 | announcement | 공식 |
| 기자 | journalist, reporter | 미디어 |
| 취재 | news gathering, coverage | 미디어 |
| 논란 | controversy | 사회 |
| 대책 | countermeasure, plan | 정책 |
| 사건 | incident, case | 범죄/법률 |
| 사고 | accident | 범죄/법률 |
| 피해 | damage, harm | 재난 |
| 시행 | implementation, enforcement | 정책 |
| 합의 | agreement, settlement | 외교 |
| 전망 | outlook, forecast | 경제 |
| 급등 | sharp rise | 경제 |
| 급락 | sharp drop | 경제 |
| 논의 | discussion, deliberation | 정치 |
| 촉구 | urging, calling for | 외교 |
| 규탄 | condemnation | 정치 |
| 선고 | sentencing | 법률 |

### 1.3 뉴스 문장 패턴

한국어 뉴스 기사는 일상 대화에서는 거의 나타나지 않는 특정 문법 패턴을 사용합니다.

| 패턴 | 의미 | 예시 |
|---|---|---|
| ~한 것으로 나타났다 | ~인 것으로 밝혀졌다/확인되었다 | 경제 성장률이 3%인 것으로 나타났다. |
| ~할 전망이다 | ~할 것으로 예상된다 | 내년에는 경기가 회복될 전망이다. |
| ~한 것으로 알려졌다 | ~인 것으로 알려졌다 | 용의자가 체포된 것으로 알려졌다. |
| ~에 따르면 | ~에 따르면 | 정부 발표에 따르면, 실업률이 감소했다. |
| ~할 방침이다 | ~할 방침/계획이다 | 정부는 세금을 인하할 방침이다. |
| ~에 나섰다 | ~에 착수했다 | 경찰이 수사에 나섰다. |
| ~을/를 촉구했다 | ~을/를 촉구했다 | 시민단체가 대책 마련을 촉구했다. |
| ~(으)로 밝혀졌다 | ~(으)로 밝혀졌다/확인되었다 | 원인은 시스템 오류로 밝혀졌다. |

### 1.4 뉴스 기사 예시

> **서울시, 대중교통 요금 내년부터 인상**
>
> 서울시는 내년 1월부터 버스와 지하철 요금을 각각 150원씩 인상한다고 발표했다. 이번 인상은 10년 만에 이루어지는 것으로, 물가 상승과 운영 적자가 주요 원인인 것으로 나타났다.
>
> 서울시 관계자에 따르면, 현재 1,250원인 기본 요금이 1,400원으로 조정될 전망이다. 이에 대해 시민들 사이에서는 찬반 논란이 일고 있다.
>
> 한편, 서울시는 저소득층에 대한 교통비 지원 대책을 마련할 방침이라고 밝혔다.

## 2. 인터넷과 SNS 용어 (인터넷/SNS 용어)

한국어 인터넷 언어는 창의적이고, 빠르게 진화하며, 대폭 축약됩니다. 온라인 콘텐츠를 읽고 젊은 한국인과 소통하는 데 필수적입니다.

### 2.1 초성 줄임말 (초성 줄임말)

한국 인터넷 사용자들은 빠른 타이핑을 위해 단어의 초성만 입력하는 경우가 많습니다.

| 줄임말 | 원래 형태 | 의미 |
|---|---|---|
| ㅋㅋㅋ | 크크크 | 웃음 ("ㅎㅎ"과 유사). ㅋ이 많을수록 더 큰 웃음 |
| ㅎㅎ | 하하 | 가벼운 웃음 |
| ㅇㅋ | 오케이 | OK |
| ㄴㄴ | 노노 | No no |
| ㅇㅇ | 응응 | 응응 / 그래 |
| ㄱㄱ | 고고 | 가자 (let's go) |
| ㄷㄷ | 덜덜 | 떨림 (충격/놀람) |
| ㅈㅅ | 죄송 | 미안 |
| ㄱㅅ | 감사 | 고마워 |
| ㅊㅋ | 축하 | 축하해 |
| ㅂㅂ | 바이바이 | 잘 가 |
| ㅁㅊ | 미친 | 미친 (비속어, 주의해서 사용) |

### 2.2 줄임말 (줄임말)

| 줄임말 | 원래 형태 | 의미 |
|---|---|---|
| 갑분싸 | 갑자기 분위기 싸해짐 | 갑자기 분위기가 싸해질 때 |
| 별다줄 | 별걸 다 줄인다 | "별걸 다 줄인다" (메타 코멘트) |
| 인싸 | 인사이더 (insider) | 인기 있고 사교적인 사람 |
| 아싸 | 아웃사이더 (outsider) | 혼자인 사람, 비사교적인 사람 |
| 갑통알 | 갑자기 통장을 보니 알바를 해야겠다 | "갑자기 통장을 보니 알바를 해야겠다" |
| 점메추 | 점심 메뉴 추천 | "점심 메뉴 추천해 줘" |
| 저메추 | 저녁 메뉴 추천 | "저녁 메뉴 추천해 줘" |
| 맛점 | 맛있는 점심 | "맛있는 점심 드세요" |
| 좋댓구알 | 좋아요 댓글 구독 알림 | "좋아요, 댓글, 구독, 알림" (유튜브) |
| 카공 | 카페에서 공부 | 카페에서 공부하기 |
| 혼밥 | 혼자 밥 먹기 | 혼자 밥 먹기 |
| 혼술 | 혼자 술 마시기 | 혼자 술 마시기 |

### 2.3 이모티콘과 표현 (이모티콘)

한국에는 고유한 텍스트 이모티콘 스타일이 있습니다:

| 이모티콘 | 의미 |
|---|---|
| ^^ | 미소 (눈이 위로 올라간 모양) |
| ㅠㅠ / ㅜㅜ | 울음 (눈물이 흐르는 모양) |
| -_- | 무감동, 지루함 |
| @_@ | 혼란, 압도됨 |
| >.< | 좌절 또는 당혹 |
| ~~ | 귀엽고 장난스러운 느낌 추가 |

### 2.4 채팅 맞춤법 관습

| 관습 | 예시 | 표준 | 비고 |
|---|---|---|---|
| 하다에서 ㅎ 탈락 | 뭐 해 → 뭐 해 / 뭐해 | 동일 | 문자에서 흔함 |
| ㅋ 어미 | 맞아ㅋㅋ | 맞아 | 웃음 추가 |
| ㅇ을 추임새로 사용 | ㅇㅇ 맞아 | 응, 맞아 | 빠른 동의 |
| 요 → 용/욘 | 감사해용~ | 감사해요 | 귀여운 어미 |
| 의도적 오기 | 머해? | 뭐 해? | 편한/귀여운 스타일 |

## 3. 신조어 (신조어)

한국어 속어는 빠르게 진화합니다. 다음은 주류로 편입된 널리 사용되는 용어입니다.

### 3.1 주요 속어

| 속어 | 의미 | 유래/용법 |
|---|---|---|
| 대박 (대박!) | 대단하다, 놀랍다, 대성공 | 범용 감탄사: 대박! 진짜? |
| 헐 | 세상에, 말도 안 돼, 충격 | 불신 표현: 헐, 말도 안 돼. |
| 꿀잼 | 매우 재미있음, 흥미진진 | 꿀(꿀) + 재미(fun): 이 영화 꿀잼이야! |
| 노잼 | 재미없음, 지루함 | NO + 재미: 오늘 수업 완전 노잼... |
| 갓 | 신급, 최고 | 영어 "god"에서: 갓세븐 |
| 레전드 | 전설적, 대단한 | 영어 "legend"에서: 그 장면 레전드야. |
| TMI | 너무 많은 정보 | 영어와 동일: TMI인데, 나 오늘 양말 안 신었어. |
| JMT | 존맛탱 (정말 맛있다) | 정말 맛있다: 이거 JMT야! |
| 고수 | 전문가, 달인 | 문자 그대로 "높은 손": 그 사람 진짜 고수야. |
| 찐 | 진짜, 진정한 | 찐이야. / 찐 맛집 |

### 3.2 유행 표현

| 표현 | 의미 | 예시 |
|---|---|---|
| ~할 자신 있음? | ~할 자신 있어? | (상대방에게 도전할 때) |
| 어쩔티비 | "그래서 어쩔? TV나 봐" | 무시하는 대꾸 (어쩔 + TV) |
| 킹받다 | 짜증나다/화나다 | 킹(king, 강조어) + 받다: 진짜 킹받아. |
| 점메추 좀 | 점심 메뉴 추천해 줘 | 점심 메뉴 추천의 줄임말 |
| 가성비 | 가격 대비 성능 | 가격 대비 성능비: 이 식당 가성비 좋아. |
| 가심비 | 가격 대비 심리적 만족 | 가격 대비 심리적 만족: 가심비가 높다 |
| 소확행 | 작지만 확실한 행복 | 小確幸: 카페에서 커피 마시는 소확행 |
| 워라밸 | 일과 삶의 균형 | 영어에서: 워라밸이 좋은 회사 |

### 3.3 세대별 언어 특성

| 세대 | 특징 | 예시 |
|---|---|---|
| MZ세대 (밀레니얼 + Z세대) | 과도한 축약, 영어 혼합 | 인싸, 아싸, TMI, MBTI |
| 90년대생 | 인터넷 문화 선구자 | 대박, 헐, ㅋㅋ |
| 00년대생 | 초압축, 밈(meme) 중심 | 어쩔티비, 킹받다, 찐 |

## 4. 콩글리시 (콩글리시)

콩글리시는 영어 차용어가 한국어에서 원래 영어와 다른 의미, 발음, 용법을 갖게 된 것을 말합니다.

### 4.1 의미가 변화된 콩글리시

| 콩글리시 | 한국어에서의 의미 | 실제 영어 표현 |
|---|---|---|
| 핸드폰 | 휴대전화 | Cell phone / Smartphone ("hand phone"이 아님) |
| 아이쇼핑 | 구경하기, 윈도쇼핑 | Window shopping ("eye shopping"이 아님) |
| 셀카 | 셀피 | Selfie (셀프 카메라 → 셀카) |
| 파이팅 | 힘내! / 화이팅! | "Fighting"은 영어에서 이런 의미 없음; "Go for it!" 사용 |
| 에어컨 | 에어컨디셔너 | Air conditioner (에어컨디셔너 축약) |
| 오피스텔 | 원룸형 아파트 | Office + Hotel → studio apartment |
| 원룸 | 원룸형 아파트 | "One room" — 영어에서는 이렇게 사용하지 않음 |
| 미팅 | 소개팅 (단체 미팅) | Meeting (영어에서 "meeting" = 회의) |
| 서비스 | 무료 제공 / 덤 | Service (영어에서 "service" = 서비스/봉사) |
| 사인 | 사인(유명인 서명) | Sign (영어에서 "sign" = 표지판/서명) |
| 올드미스 | 미혼 연상 여성 | 영어에서 사용하지 않음 (구식/불쾌한 표현) |
| 노트북 | 노트북 컴퓨터 | Notebook (영어에서 "notebook" = 공책) |
| 리모컨 | 리모트 컨트롤 | Remote control (리모트 컨트롤 축약) |
| 컨닝 | 시험 부정행위 | Cunning (영어에서는 다른 의미) |
| 백미러 | 후사경 | Back mirror (표준 영어가 아님; "rearview mirror") |
| 비닐봉지 | 비닐봉지 | Vinyl bag (영어에서 사용하지 않음) |

### 4.2 콩글리시 동사 패턴

많은 콩글리시 명사가 하다와 결합하여 동사를 형성합니다:

| 콩글리시 + 하다 | 의미 | 예시 |
|---|---|---|
| 다이어트하다 | 다이어트하다 | 요즘 다이어트하고 있어요. |
| 드라이브하다 | 드라이브하다 | 주말에 드라이브하러 가자. |
| 스트레스받다 | 스트레스받다 | 시험 때문에 스트레스받아요. |
| 업데이트하다 | 업데이트하다 | 앱을 업데이트해야 해요. |
| 로그인하다 | 로그인하다 | 로그인해 주세요. |
| 캔슬하다 | 취소하다 | 예약을 캔슬했어요. |

### 4.3 영어 사용자를 위한 콩글리시 주의사항

| 한국인이 하는 말 | 의미 | 영어 사용자가 듣는 것 |
|---|---|---|
| 서비스예요! | 무료예요! / 서비스입니다! | It's a service? |
| 미팅이 있어요. | 소개팅이 있어요. | I have a meeting. |
| 핸드폰 번호 좀 알려 주세요. | 전화번호를 알려 주세요. | Hand phone? |
| 사인해 주세요. | 사인(서명) 좀 해 주세요. | Please sign (a document)? |
| 원샷! | 한 번에 다 마셔! | One shot? |

## 5. 격식체 vs 비격식체 비교

격차를 이해하는 것은 다양한 상황에서 적절하게 소통하는 데 매우 중요합니다.

### 5.1 같은 메시지의 격차별 표현

**메시지**: "회의 시간이 오후 3시로 변경되었습니다."

| 격차 | 표현 |
|---|---|
| **뉴스** (보도문) | 회의 시간이 오후 3시로 변경된 것으로 알려졌다. |
| **격식 비즈니스** (격식체) | 회의 시간이 오후 3시로 변경되었음을 알려 드립니다. |
| **공손체** (합니다 체) | 회의 시간이 오후 3시로 바뀌었습니다. |
| **일상 공손체** (해요 체) | 회의 시간이 3시로 바뀌었어요. |
| **비격식** (반말) | 회의 시간 3시로 바뀌었어. |
| **문자/채팅** | 회의 3시로 변경ㅇㅇ |

### 5.2 글쓰기에서의 격차

| 맥락 | 문체 | 예시 문장 |
|---|---|---|
| 학술 논문 | 이다 체 | 이러한 현상은 사회적 요인에 기인한다. |
| 뉴스 기사 | 보도문체 | ~한 것으로 나타났다 / ~할 전망이다 |
| 업무 이메일 | 합니다 체 | 검토해 주시면 감사하겠습니다. |
| 블로그 글 | 해요 체 | 오늘은 제가 좋아하는 카페를 소개할게요. |
| 소셜 미디어 | 반말/줄임말 | 여기 분위기 완전 꿀잼ㅋㅋ |
| 문자 메시지 | 줄임말 | ㄱㅅ! ㅇㅇ 3시에 봐 |

### 5.3 상황별 적절한 격차

| 상황 | 적절한 격차 | 피해야 할 것 |
|---|---|---|
| 취업 면접 | 합니다 체 + 존칭 | 모든 속어와 줄임말 |
| 카카오톡 (친구) | 해요 체 또는 반말 | 합니다 체 (너무 딱딱함) |
| TOPIK 쓰기 | 이다 체 (문어체) | 해요 체, 속어 |
| 교수님과 대화 | 합니다 체 + 존칭 | 해요 체 |
| 온라인 댓글 (유튜브) | 해요 체 또는 반말 | 지나치게 격식적인 표현 |

### 연습 2: 콩글리시 수정

4. 오늘 미팅이 있어서 못 만나.

<details>
<summary>정답</summary>

### 연습 3: 격차 변환

다음 문장을 5가지 격차로 바꾸세요.

2. 격식 비즈니스 (합니다 체) →

4. 비격식 (반말) →
5. 인터넷/채팅 →

<details>
<summary>정답</summary>

3. **일상 공손**: 새로 생긴 식당에서 점심 먹었는데 맛있었어요.
4. **비격식**: 새로 생긴 식당에서 점심 먹었는데 맛있었어.

</details>

### 연습 4: 뉴스 기사 독해

다음 뉴스 스타일 지문을 읽고 질문에 답하세요.

> 최근 조사에 따르면, 한국의 MZ세대(1981-2010년생)의 80%가 워라밸을 가장 중요한 직장 선택 기준으로 꼽은 것으로 나타났다. 이는 이전 세대가 연봉과 고용 안정성을 중시했던 것과는 대조적이다. 전문가들은 이러한 변화가 앞으로 기업 문화에 큰 영향을 미칠 전망이라고 분석했다.

**질문**:
1. MZ세대의 80%가 가장 중요하게 여기는 것은?
2. 이전 세대는 무엇을 중시했는가?
3. 전문가들의 전망은?

<details>
<summary>정답</summary>

1. **워라밸** (일과 삶의 균형) — 직장 선택의 가장 중요한 기준으로 꼽았다.
2. **연봉과 고용 안정성**을 중시했다.
3. 이러한 변화가 앞으로 **기업 문화에 큰 영향을 미칠 전망**이라고 분석했다.

## 핵심 정리

1. **뉴스 한국어**는 ~것으로 나타났다, ~할 전망이다, ~에 따르면 등 격식 패턴을 사용합니다 — TOPIK 독해에 필수
2. **인터넷 줄임말** (초성 줄임말)은 매우 보편적입니다: ㅋㅋ, ㅇㅇ, ㄱㄱ, ㅂㅂ — 실제 한국어 온라인 콘텐츠를 이해하려면 반드시 학습
3. **속어**는 빠르게 진화합니다: 대박, 꿀잼, 킹받다, 찐 — 최신 트렌드를 따라가되 사용하면 안 되는 상황도 알아야 합니다
4. **콩글리시**는 함정이 될 수 있습니다: 핸드폰 ≠ hand phone, 서비스 ≠ service, 미팅 ≠ meeting
5. **격차 인식**은 어휘만큼 중요합니다 — 같은 생각을 맥락에 따라 5가지 이상의 방식으로 표현할 수 있습니다
