## 1. 필수 한국어 인사

### 1.1 범용 인사: 안녕하세요

> **안녕하세요 (annyeonghaseyo)**
>
> - **직역**: "평안하십니까?"
> - **용도**: 안녕하세요 / 좋은 아침 / 좋은 오후 / 좋은 저녁 — 하루 중 어느 때나 사용
> - **격식**: 존댓말(공손한 표현) — 거의 모든 상황에서 안전
>
> - **분석**:
>   - 안녕 (annyeong) = 평안, 안녕
>   - 하 (ha) = 하다(동사 어간)
>   - 세요 (seyo) = 존경 표현의 공손 어미
>
> 직역: "평안을 하십니까?" = "잘 지내십니까?"

| 상황 | 표현 | 로마자 표기 | 격식 수준 |
|------|------|-----------|----------|
| 일반 인사(표준) | 안녕하세요 | annyeonghaseyo | 존댓말 |
| 일반 인사(반말) | 안녕 | annyeong | 반말(친구/아이) |
| 일반 인사(격식 높음) | 안녕하십니까 | annyeonghasimnikka | 격식체(연설, 군대, 뉴스) |

> **핵심 포인트**: 한국어에는 영어처럼 아침, 오후, 저녁을 구분하는 인사가 일상 대화에 없습니다. 안녕하세요 하나로 하루 중 모든 시간에 사용합니다.

### 1.2 감사 표현

| 표현 | 로마자 표기 | 용도 |
|------|-----------|------|
| 감사합니다 | gamsahamnida | 격식 — 표준 공손 "감사합니다" |
| 고마워요 | gomawoyo | 비격식 존댓말 — 친구, 또래에게 |
| 고마워 | gomawo | 반말 — 가까운 친구에게만 |
| 감사드립니다 | gamsadeurimnida | 매우 격식/겸양 — 윗사람에게, 비즈니스에서 |

> **감사 vs 고맙다**: 감사는 한자어(感謝)로 더 격식적인 느낌입니다. 고맙다는 고유어로 더 따뜻하고 개인적인 느낌입니다.

## 3. 기본 존댓말: 합쇼체 소개

### 3.1 합쇼체란?

한국어에는 여러 높임법이 있습니다. **합쇼체**(hapssyoche)는 **격식 높임** 단계로, 일상 대화에서 사용되는 가장 높은 공손 수준입니다.

> **높임법 간편 참고표(미리보기)**
>
> **합쇼체 (격식 존댓말)**: -ㅂ니다 / -습니다
> → 뉴스, 비즈니스, 군대, 첫 만남, 공식 연설
> → "학생입니다" = 저는 학생입니다
>
> **해요체 (비격식 존댓말)**: -아/어요
> → 낯선 사람, 동료, 가게에서의 일상 대화
> → "학생이에요" = 저는 학생이에요
>
> **해체 (반말)**: -아/어 (요 없이)
> → 같은 나이의 가까운 친구, 어린이, 친밀한 관계
> → "학생이야" = 나는 학생이야
>
> 초보자 권장: 합쇼체와 해요체를 먼저 익히세요.
> 합쇼체를 사용하면 절대 실례가 되지 않습니다 — 항상 안전합니다.

### 3.2 합쇼체 자기소개 패턴

| 패턴 | 구조 | 예시 |
|------|------|------|
| "저는 [명사]입니다" | [명사]입니다 | 학생입니다 (학생입니다) |
| "저는 [명사]가 아닙니다" | [명사]이/가 아닙니다 | 학생이 아닙니다 (학생이 아닙니다) |
| "[장소]에서 왔습니다" | [장소]에서 왔습니다 | 미국에서 왔습니다 (미국에서 왔습니다) |
| "[명사]는 [명사]입니다" | [명사]은/는 [명사]입니다 | 이것은 책입니다 (이것은 책입니다) |

## 5. 작별 인사와 상황별 표현

한국어는 **누가 떠나고 누가 남느냐**에 따라 다른 작별 인사를 사용합니다.

> **한국어 작별 인사**
>
> **상황 1**: 내가 떠나고, 상대방이 남을 때
> → 안녕히 계세요. (annyeonghi gyeseyo)
> 직역: "편안히 계세요." (남아 있는 동안 잘 지내세요.)
>
> **상황 2**: 상대방이 떠나고, 내가 남을 때
> → 안녕히 가세요. (annyeonghi gaseyo)
> 직역: "편안히 가세요." (잘 가세요.)
>
> **상황 3**: 둘 다 떠날 때
> → 안녕히 가세요. (서로에게)
>
> **반말 (친구)**:
> → 잘 가. (jal ga) = "잘 가."
> → 잘 있어. (jal isseo) = "잘 있어."
> → 안녕. (annyeong) = 안녕. (반말 "안녕"과 같은 단어)

## 7. 문화 노트

### 7.1 인사(절)

> **한국의 인사 예절**
>
> - **목례(목례, mongnye)**: ~15도 각도 — 가벼운 인사, 또래 사이
> - **보통례(보통례, botongnyе)**: ~30도 각도 — 일반적인 공손한 인사
> - **정중례(정중례, jeongjongnye)**: ~45도 각도 — 큰 존경을 표할 때(어른, 중요한 사람과의 첫 만남)
> - **큰절(큰절, keunjeol)**: 무릎을 꿇고 이마를 바닥에 대기 — 격식 행사(설날, 결혼식, 조부모님께)
>
> 참고: 인사할 때 절하면서 안녕하세요를 말합니다. 악수하면서 절하지 마세요 — 둘 중 하나를 하세요(비즈니스에서는 악수하면서 가볍게 고개를 숙이기도 합니다).

### 7.2 호칭과 칭호

한국에서는 상대방의 이름만 부르는 일이 거의 없습니다. 특히 나이가 많거나 지위가 높은 사람에게는 더욱 그렇습니다:

| 호칭 | 용도 |
|------|------|
| [이름]씨 (ssi) | ~씨 — 공손하지만 동등한 위치. 성만이 아닌 전체 이름 + 씨를 사용 |
| [이름/직함]님 (nim) | 존칭 접미사 — 직함이나 역할에 붙임(선생님, 사장님) |
| 선배 (seonbae) | 선배(학교/직장) |
| 후배 (hubae) | 후배(학교/직장) |
| 오빠 (oppa) | 오빠(여성이 사용) — 약간 나이 많은 남성 친구에게도 사용 |
| 형 (hyeong) | 형(남성이 사용) |
| 언니 (eonni) | 언니(여성이 사용) |
| 누나 (nuna) | 누나(남성이 사용) |
| 아저씨 (ajeossi) | 아저씨(중년 남성) |
| 아주머니 (ajumeoni) | 아주머니(중년 여성) |

### 7.3 나이 문화

한국 문화에서 나이는 매우 중요합니다. 나이에 따라 결정되는 것들:
- 어떤 **높임법**을 사용하는지(격식 vs. 반말)
- 어떤 **호칭**을 사용하는지(형/오빠/언니/누나 등)
- 누가 **먼저 술을 따르는지**(어린 사람이 윗사람에게)
- **자리 배치**와 **식사 순서**(어른이 먼저)

한국인이 새로운 사람을 만날 때 자주 묻는 질문 중 하나가 **몇 살이에요?** 또는 더 공손하게 **나이가 어떻게 되세요?**입니다. 이것은 무례한 것이 아니라, 적절한 대화를 위한 사회적 관계를 정립하는 것입니다.

> **한국 나이 vs. 만 나이**: 한국은 전통적으로 태어나자마자 1살이고, 매년 1월 1일에 한 살씩 더하는 체계를 사용했습니다. 2023년부터 대한민국은 법적·행정적 목적으로 만 나이(국제 나이)를 공식 채택했지만, 전통적 방식은 사회적으로 여전히 사용됩니다.

### 7.4 두 손 사용

물건을 주고받을 때(명함, 선물, 음료), **두 손**을 사용하거나 오른손으로 주면서 왼손으로 오른팔을 받칩니다. 이는 존경의 표시입니다.

## 8. 연습 문제

### 연습 1: 상황에 맞는 표현 고르기

각 상황에 맞는 올바른 표현을 고르세요:

```
상황:
a. 식당에 들어갈 때    b. 누군가를 부딪혔을 때
c. 전화를 받을 때      d. 다른 사람보다 먼저 퇴근할 때
e. 다른 사람의 집에서 식사를 마친 후

표현:
1. 죄송합니다
2. 안녕하세요
3. 여보세요
4. 먼저 가겠습니다
5. 잘 먹었습니다
```

### 연습 2: 자기소개 작성

다음 템플릿을 활용하여 한국어로 자기소개를 작성하세요:

```
안녕하세요.
저는 ________입니다.         (이름)
________에서 왔습니다.        (나라)
________입니다.               (직업/학생)
만나서 반갑습니다.
```

### 연습 3: 올바른 작별 인사 고르기

각 상황에 맞는 적절한 작별 인사를 고르세요:

```
1. 친구 집을 떠날 때. 내가 하는 말: ___
   a) 안녕히 계세요   b) 안녕히 가세요

2. 친구가 내 집을 떠날 때. 내가 하는 말: ___
   a) 안녕히 계세요   b) 안녕히 가세요

3. 동료와 함께 사무실을 나설 때. 서로 하는 말: ___
   a) 안녕히 계세요   b) 안녕히 가세요

4. 친구가 떠날 때(반말). 내가 하는 말: ___
   a) 잘 가   b) 잘 있어
```

### 연습 4: 한국어로 번역하기

```
1. 안녕하세요, 만나서 반갑습니다.
2. 저는 캐나다에서 온 학생입니다.

4. 죄송합니다.
5. 천천히 말해 주세요.
6. 모르겠어요.
```

### 연습 5: 대화 완성

빈칸을 채우세요:

```
A: 안녕하세요. 이름이 뭐예요?
B: 저는 _________입니다. ________에서 왔습니다.

A: 아, 그래요? 직업이 뭐예요?
B: _________입니다.

A: _________. (만나서 반갑습니다.)
B: 저도 만나서 반갑습니다. (저도 만나서 반갑습니다.)
```
