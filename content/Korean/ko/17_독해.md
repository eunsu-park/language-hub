## 1. 한국어 텍스트의 유형

TOPIK II-4 수준에서는 다양한 실제 한국어 텍스트를 읽고 이해해야 한다. 각 유형에는 고유한 관습, 어휘 수준, 구조적 패턴이 있다.

### 1.1 신문 기사 (Newspaper Articles)

한국어 신문 기사는 관습을 알면 쉽게 탐색할 수 있는 예측 가능한 구조를 따른다.

**제목 관습**:

| 특징 | 관습 | 예시 |
|---|---|---|
| 조사 생략 | 주격/목적격 조사 탈락 | 정부 새 정책 발표 (정부가 새 정책을 발표) |
| 서술어 생략 | 이다/하다 탈락 | 서울 올해 가장 더워 (서울이 올해 가장 더워) |
| 명사 종결 | 동사가 아닌 명사로 끝남 | 총리 방한 예정 (총리의 방한이 예정) |
| 인용 부호 | 직접 인용에 「 」 사용 | 「경제 회복 신호」전문가 분석 |
| 한자어 사용 | 격식적인 한자어 어휘 | 발표(announce), 실시(implement), 추진(promote) |

**기사 본문 구조**:

```
제목:   정부, 청년 일자리 대책 발표
리드:   핵심 내용 요약 (누가, 무엇을, 언제, 어디서, 왜)
본문:   세부 내용, 배경, 인용문
결론:   전망, 향후 계획
```

**흔한 신문 어휘**:

| 한국어 | 영어 | 용례 |
|---|---|---|
| 발표하다 | to announce | 정부가 새 정책을 발표했다 |
| 실시하다 | to implement | 내년부터 새 제도를 실시한다 |
| 추진하다 | to promote/push forward | 사업을 적극 추진하고 있다 |
| 전망하다 | to forecast/predict | 경제 성장을 전망했다 |
| 지적하다 | to point out | 전문가들이 문제점을 지적했다 |
| 강조하다 | to emphasize | 대통령이 협력을 강조했다 |
| 우려하다 | to worry/be concerned | 부작용을 우려하는 목소리 |
| 논란이 되다 | to become controversial | 이 정책이 논란이 되고 있다 |

**뉴스 문장 종결 표현**:
- **-다** (해라체): 표준 뉴스 문체. 정부가 발표**했다**. (정부가 발표했다.)
- **-(으)ㄴ 것으로 나타났다**: ~한 것으로 나타났다. 경제가 성장한 것으로 **나타났다**.
- **-(으)ㄹ 예정이다**: ~할 예정이다. 내년에 실시**할 예정이다**.
- **-(으)ㄴ 것으로 알려졌다**: ~한 것으로 알려졌다. 피해가 큰 것으로 **알려졌다**.

### 1.2 에세이 / 수필 (Personal Essays)

한국어 수필은 TOPIK II 읽기 영역에서 흔한 장르이다. 성찰적이고 개인적이며, 부드럽고 문학적인 어조를 사용하는 경향이 있다.

**특징**:
- 1인칭 시점 (나는, 저는)
- 격식체와 일상체의 혼합
- 삶, 자연, 문화에 대한 성찰적 관찰
- 흔히 도덕적 깨달음이나 교훈으로 마무리
- 묘사적 언어와 비유 사용

**흔한 에세이 어휘**:

| 한국어 | 영어 |
|---|---|
| 깨닫다 | to realize |
| 돌이켜 보다 | to look back on |
| 되새기다 | to reflect upon |
| 소중하다 | to be precious/valuable |
| 그리워하다 | to miss/long for |
| 감사하다 | to be grateful |
| 여유 | leisure, relaxation |
| 인생 | life |

### 1.3 편지 / 이메일 (Letters and Emails)

한국어 편지와 이메일에는 영어와 다른 특정 개시 및 종결 관습이 있다.

**격식 편지 구조**:

```
받는 사람: 김민수 부장님 귀하          (수신자 + 직함 + 귀하)
보내는 사람: 이지영                    (발신자)

부장님께,                              (인사말: ~께)

안녕하십니까?                           (격식 인사)
다름이 아니라...                        (도입구: "다름이 아니라...")

[본문 내용]

바쁘신 중에 읽어 주셔서 감사합니다.      (바쁘신 중에 읽어 주셔서 감사합니다)
건강하시기 바랍니다.                     (건강을 기원합니다)

2026년 3월 12일                        (날짜)
이지영 드림                             (이름 + 드림 = "올림")
```

**비격식 이메일/편지**:

```
민수야,                                 (이름 + 야/아 — 반말)

안녕? 잘 지내지?                        (반말 인사)

[본문 내용]

또 연락할게. 안녕!                      (반말 마무리)
지영이가                                (보내는 사람)
```

**주요 편지 표현**:

| 한국어 | 문체 | 의미 |
|---|---|---|
| ~님께 | 격식 | ~에게 (존칭) |
| 다름이 아니라 | 격식 | 다름이 아니라... (용건 도입) |
| ~드림 | 격식 | ~올림 (발신자 이름 뒤에) |
| 귀하 | 격식 | 귀하 (봉투에) |
| 건강하시기 바랍니다 | 격식 | 건강을 기원합니다 |
| ~에게 / ~한테 | 비격식 | ~에게 |
| ~가/이 / ~올림 | 비격식/반격식 | ~가/이 (발신자 이름 뒤에) |

### 1.4 공고 / 안내문 (Public Notices and Announcements)

공고문은 간결하고 격식적인 언어와 특정 서식 관습을 사용한다.

**특징**:
- 굵은 글씨 또는 큰 글씨의 제목 (공고, 안내, 알림)
- 번호 또는 글머리 기호 정보
- 핵심 정보: 날짜, 시간, 장소, 자격
- 명사화 어미를 포함한 격식적 어휘
- 끝에 연락처 정보

**흔한 공고 어휘**:

| 한국어 | 영어 |
|---|---|
| 공고 | public notice |
| 안내 | guide/information |
| 알림 | notification |
| 대상 | target/eligibility |
| 기간 | period/duration |
| 장소 | location/venue |
| 신청 | application |
| 문의 | inquiry |
| 접수 | reception/submission |
| 마감 | deadline |
| 참고 | reference/note |

**전형적인 공고 구조**:

```
[제목] 2026년 한국어 능력 시험 안내

1. 시험 일시: 2026년 4월 12일 (일) 오전 10시
2. 시험 장소: 서울대학교 시험 센터
3. 접수 기간: 3월 1일~3월 15일
4. 접수 방법: 온라인 신청 (www.topik.go.kr)
5. 대상: 외국인 및 재외동포
6. 문의: 02-1234-5678

※ 자세한 사항은 홈페이지를 참고하시기 바랍니다.
```

## 2. 읽기 전략

### 2.1 훑어 읽기 (Skimming) — 전체 파악

**목적**: 모든 단어를 읽지 않고 빠르게 주요 주제와 전체 구조를 파악한다.

**한국어 텍스트 훑어 읽기 방법**:

| 단계 | 행동 | 찾아볼 것 |
|---|---|---|
| 1 | 제목 읽기 | 주요 주제 |
| 2 | 각 문단의 첫 문장 읽기 | 주제문 |
| 3 | 반복되는 핵심 단어 찾기 | 중심 주제 어휘 |
| 4 | 마지막 문단 읽기 | 결론/핵심 요점 |

**연습 팁**: 1~2분 타이머를 설정하고 "이 글은 무엇에 대한 것인가?"와 "저자의 주요 주장은 무엇인가?"에 답해 보세요.

### 2.2 찾아 읽기 (Scanning) — 특정 정보 찾기

**목적**: 전체 텍스트를 읽지 않고 특정 사실(날짜, 이름, 숫자, 장소)을 찾는다.

**한국어 텍스트 찾아 읽기 방법**:

| 대상 | 찾아볼 것 |
|---|---|
| 날짜 | 년, 월, 일, 요일 패턴 |
| 숫자 | 숫자, %, 명, 원, 개 |
| 이름 | 씨, 님, 교수, 박사 |
| 장소 | 시, 구, 동, 대학교, 센터 |
| 핵심 명사 | 질문과 일치하는 단어 |

### 2.3 문맥 추론 (Inferring Meaning from Context)

모르는 단어를 만났을 때, 주변 문맥을 사용하여 의미를 추론한다.

**전략**:

1. **한자어 어근**: 합성어를 한자어 형태소로 분해
   - 환경오염 → 환경(environment) + 오염(pollution) = 환경오염
   - 국제회의 → 국제(international) + 회의(meeting) = 국제회의

2. **문장 수준 단서**: 앞뒤를 살펴본다
   - "그 약을 먹은 후에 **증상**이 나아졌다." → 증상은 "symptoms"을 의미해야 한다 (약을 먹은 후 나아진 것)

3. **대구 구조**: 한국어는 종종 대구적 표현을 사용한다
   - "장점도 있지만 **단점**도 있다." → 장점 = 장점이면, 단점 = 단점

4. **접속사 단서**: 담화 표지는 관계를 드러낸다
   - 그러나/하지만 → 대조 (다음 부분은 반대를 말함)
   - 따라서/그러므로 → 결론 (다음 부분은 결과)
   - 즉/다시 말해 → 바꿔 말하기 (다음 부분은 같은 내용을 다르게 설명)

## 3. 한국어 텍스트 구조 관습

### 3.1 문단 구성: 서론-본론-결론

한국어 학술적·격식적 텍스트는 영어와 유사한 구조를 따른다:

| 부분 | 한국어 | 기능 | 표지 |
|---|---|---|---|
| 서론 | **서론** | 주제, 논지, 배경 제시 | 최근, 요즘, ~에 대해, 이 글에서는 |
| 본론 | **본론** | 논거, 증거, 세부 내용 | 먼저, 다음으로, 또한, 한편 |
| 결론 | **결론** | 요약, 의견, 함의 | 결론적으로, 요약하면, 따라서 |

### 3.2 격식 문어체 표지

문어체 한국어는 구어체 한국어와 상당히 다르다. 이러한 표지를 인식하는 것은 독해에 필수적이다.

**-이다 체 (문어체 서술 문체)**:

| 구어 (해요체) | 문어 (-이다 체) | 영어 |
|---|---|---|
| 중요해요 | 중요하**다** | is important |
| 필요해요 | 필요하**다** | is necessary |
| 있었어요 | 있**었다** | there was |
| 할 거예요 | 할 것이**다** | will do |
| 했어요 | 하**였다** / 했**다** | did |

**-(으)ㅁ 명사화 — 문어체 텍스트에서**:

격식적 텍스트는 구어체보다 절의 명사화를 더 많이 사용한다:
- 이 연구의 목적은 한국어 교육의 문제점을 분석하**는 데** 있다. (이 연구의 목적은 한국어 교육의 문제점을 분석하는 데 있다.)
- 경제가 성장하**였음**을 보여 준다. (경제가 성장하였음을 보여 준다.)

### 3.3 흔한 문어체 접속사 (접속 부사)

이러한 담화 표지는 격식적 한국어 텍스트에 자주 등장한다. 이를 숙달하는 것은 독해에 필수적이다.

| 한국어 | 영어 | 기능 | 예시 |
|---|---|---|---|
| **그러나** | However | 대조 | 경제는 성장했다. **그러나** 실업률도 높아졌다. |
| **하지만** | But | 대조 (약간 덜 격식적) | 노력했다. **하지만** 결과는 좋지 않았다. |
| **따라서** | Therefore | 결론 | 증거가 충분하다. **따라서** 유죄이다. |
| **그러므로** | Therefore (격식) | 결론 | 준비가 필요하다. **그러므로** 미리 계획하자. |
| **그래서** | So | 결과 | 비가 왔다. **그래서** 집에 있었다. |
| **한편** | Meanwhile / On the other hand | 첨가/전환 | 서울은 더웠다. **한편** 부산은 비가 왔다. |
| **반면에** | On the other hand | 대조 | 수출은 늘었다. **반면에** 내수는 줄었다. |
| **또한** | Also / Furthermore | 첨가 | 건강에 좋다. **또한** 맛도 좋다. |
| **게다가** | Moreover | 강조 첨가 | 비싸다. **게다가** 맛도 없다. |
| **즉** | In other words / That is | 바꿔 말하기 | 공휴일, **즉** 쉬는 날이다. |
| **다시 말해** | In other words | 바꿔 말하기 | 낙제했다. **다시 말해** 시험에 떨어졌다. |
| **예를 들어** | For example | 예시 | 과일을 좋아한다. **예를 들어** 사과, 딸기 등. |
| **결국** | In the end / After all | 최종 결과 | 노력했지만 **결국** 실패했다. |
| **오히려** | Rather / On the contrary | 뜻밖의 대조 | 도움이 안 되었다. **오히려** 방해가 되었다. |

## 4. 예문 텍스트와 독해 문제

### 예문 1: 신문 기사 — 한국 경제

> **한국 경제, 올해 2.5% 성장 전망**
>
> 한국은행은 12일 올해 한국 경제가 2.5% 성장할 것으로 전망했다. 이는 지난해 2.1% 성장률보다 높은 수치이다.
>
> 한국은행 총재는 "수출이 회복되고 내수도 점차 개선되고 있다"고 밝혔다. 특히 반도체 산업의 수출 증가가 경제 성장을 이끌고 있는 것으로 분석됐다.
>
> 그러나 일부 전문가들은 글로벌 경기 둔화와 물가 상승이 경제 성장에 걸림돌이 될 수 있다고 우려했다. 한편, 고용 시장은 서비스업을 중심으로 점진적인 개선세를 보이고 있다.
>
> 한국은행은 하반기에 금리 인하를 검토할 수 있다고 밝혔으며, 향후 경제 상황을 면밀히 모니터링할 계획이라고 전했다.

**어휘 안내**:

| 한국어 | 영어 |
|---|---|
| 한국은행 | Bank of Korea |
| 성장 | growth |
| 전망하다 | to forecast |
| 수치 | figure, number |
| 총재 | governor (of a bank) |
| 수출 | exports |
| 내수 | domestic demand |
| 반도체 | semiconductor |
| 경기 둔화 | economic slowdown |

## 5. 문맥을 통한 어휘 학습

### 5.1 한자어 어근을 활용한 미지 단어 해독

많은 한국어 단어가 공통된 한자 형태소를 공유한다. 이러한 어근을 인식하면 새로운 단어를 해독할 수 있다:

| 어근 | 의미 | 이 어근을 사용하는 단어 |
|---|---|---|
| 경(經) | 지나다, 관리하다 | **경**제(economy), **경**험(experience), **경**영(management) |
| 교(敎) | 가르치다 | **교**육(education), **교**사(teacher), **교**실(classroom) |
| 사(社) | 사회 | **사**회(society), 회**사**(company), **사**장(president) |
| 학(學) | 배우다 | **학**교(school), **학**생(student), 과**학**(science) |
| 발(發) | 내보내다 | **발**표(announcement), **발**전(development), **발**견(discovery) |
| 정(政/正) | 정치, 바르다 | **정**부(government), **정**치(politics), **정**확(accurate) |

## 핵심 정리

1. **텍스트 유형을 파악하라**: 신문 기사, 에세이, 편지, 공고문은 각각 고유한 관습과 어휘가 있다
2. **읽기 전략을 활용하라**: 훑어 읽기로 전체를 파악하고, 찾아 읽기로 세부 정보를 찾고, 문맥에서 미지 단어를 추론한다
3. **문어체 한국어는 구어체와 다르다**: -다 체 어미, 격식 접속사, 명사화 구조를 인식한다
4. **접속사는 길 안내판이다**: 그러나 (대조), 따라서 (결론), 한편 (전환), 즉 (바꿔 말하기) — 이것들은 텍스트의 방향을 알려준다
5. **한자어 어근이 최고의 도구다**: 흔한 한자어 형태소를 배우면 수천 개의 단어를 해독할 수 있다
