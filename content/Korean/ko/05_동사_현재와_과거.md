## 1. 한국어 동사 기본

### 1.1 기본형 (-다)

모든 한국어 동사의 기본형(사전형)은 **-다**(da)로 끝납니다. 동사를 활용하려면 -다를 떼어 **어간**을 구한 후, 적절한 어미를 붙입니다.

```
┌──────────────────────────────────────────────────────────────────┐
│                한국어 동사 구조                                     │
│                                                                  │
│   기본형 = 어간 + 다                                              │
│                                                                  │
│   가다 (가다)       →  가 (어간) + 다                              │
│   먹다 (먹다)       →  먹 (어간) + 다                              │
│   공부하다 (공부하다) →  공부하 (어간) + 다                         │
│   읽다 (읽다)       →  읽 (어간) + 다                              │
│                                                                  │
│   활용 = 어간 + 어미                                              │
│                                                                  │
│   가 + -아요 → 가요     (가다 — 존댓말)                            │
│   먹 + -어요 → 먹어요   (먹다 — 존댓말)                           │
│   가 + -ㅂ니다 → 갑니다  (가다 — 격식)                             │
│   먹 + -습니다 → 먹습니다 (먹다 — 격식)                           │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 한국어 동사의 핵심 특성

1. **인칭이나 수에 따른 활용이 없음**: 가요는 "나는 가요", "너는 가요", "그/그녀는 가요", "우리는 가요", "그들은 가요" — 모두 동일
2. **시제는 동사에 표시됨**: 가요(현재), 갔어요(과거), 갈 거예요(미래)
3. **높임법은 동사에 표시됨**: 가(반말), 가요(존댓말), 갑니다(격식)
4. **동사는 항상 문장 끝에 옴**
5. **관사 불필요**: 책을 읽어요 = "책을 읽어요"

### 1.3 동작 동사 vs 상태 동사(형용사)

한국어에는 두 종류의 "동사"가 있습니다:

| 유형 | 한국어 용어 | 예시 | 영어 대응 |
|------|-----------|------|----------|
| 동작 동사(동사) | 동작 동사 | 가다, 먹다, 하다 | 동사(가다, 먹다, 하다) |
| 상태 동사(형용사) | 상태 동사 | 크다, 좋다, 예쁘다 | 형용사(크다, 좋다, 예쁘다) |

> **중요**: 한국어에서 형용사는 동사처럼 기능합니다. "크다"는 문자 그대로 "크다(상태)"를 의미합니다. 동사처럼 활용됩니다: 커요(커요), 컸어요(컸어요). 이것은 6과에서 자세히 다룹니다.

## 2. 모음 조화

### 2.1 규칙

모음 조화(모음 조화, moeum joha)는 **-아** 또는 **-어** 어미를 붙일지를 결정합니다. 어간의 **마지막 모음**을 확인하세요:

```
┌──────────────────────────────────────────────────────────────────┐
│                    모음 조화 규칙                                   │
│                                                                  │
│   어간의 마지막 모음이 ㅏ 또는 ㅗ (양성모음)이면:                  │
│     → -아 어미 사용                                               │
│                                                                  │
│   마지막 모음이 그 외(음성모음)이면:                               │
│     → -어 어미 사용                                               │
│                                                                  │
│   특수: 하다 동사                                                  │
│     → 하 + 여 → 해 (하다가 해로 축약)                             │
│                                                                  │
│   요약:                                                           │
│     ㅏ, ㅗ        → -아  (가다 → 가 + 아요, 오다 → 오 + 아요)     │
│     그 외 전부    → -어  (먹다 → 먹 + 어요, 마시다 → 마시 + 어요) │
│     하다          → -해  (공부하다 → 공부해요)                     │
└──────────────────────────────────────────────────────────────────┘
```

### 2.2 축약 규칙

어간 모음과 어미 모음이 만나면 자주 **축약**됩니다:

| 어간 끝 | + 어미 | 결과 | 예시 |
|---------|--------|------|------|
| ㅏ | + 아 | ㅏ (합침) | 가 + 아요 → 가요 |
| ㅗ | + 아 | ㅘ | 오 + 아요 → 와요 |
| ㅜ | + 어 | ㅝ | 배우 + 어요 → 배워요 |
| ㅣ | + 어 | ㅕ | 마시 + 어요 → 마셔요 |
| ㅡ | + 어 | 앞 모음에 따라 | 쓰 + 어요 → 써요 |
| ㅓ | + 어 | ㅓ (합침) | 서 + 어요 → 서요 |
| ㅐ | + 어 | ㅐ (합침) | 보내 + 어요 → 보내요 |

> **팁**: 축약은 발음을 편하게 하기 위한 것입니다. 모음을 빠르게 발음하고 자연스럽게 합쳐지는 소리를 들어보세요.

## 3. 현재시제 비격식 존댓말 (-아/어요)

### 3.1 형성

**해요체**(비격식 존댓말)는 일상 한국어에서 가장 많이 사용되는 높임법입니다.

```
어간 (마지막 모음 ㅏ/ㅗ) + 아요
어간 (기타 모음)         + 어요
하다 어간               → 해요
```

### 3.2 활용 예시

| 기본형 | 어간 | 마지막 모음 | 어미 | 활용형 | 의미 |
|--------|------|-----------|------|--------|------|
| 가다 (가다) | 가 | ㅏ | + 아요 | **가요** (가+아요 축약) | 가다 |
| 오다 (오다) | 오 | ㅗ | + 아요 | **와요** (오+아요 축약) | 오다 |
| 보다 (보다) | 보 | ㅗ | + 아요 | **봐요** (보+아요 축약) | 보다 |
| 먹다 (먹다) | 먹 | ㅓ | + 어요 | **먹어요** | 먹다 |
| 읽다 (읽다) | 읽 | ㅣ | + 어요 | **읽어요** | 읽다 |
| 마시다 (마시다) | 마시 | ㅣ | + 어요 | **마셔요** (시+어→셔) | 마시다 |
| 쓰다 (쓰다) | 쓰 | ㅡ | + 어요 | **써요** (ㅡ 탈락, + 어요) | 쓰다 |
| 자다 (자다) | 자 | ㅏ | + 아요 | **자요** (자+아요 축약) | 자다 |
| 일어나다 (일어나다) | 일어나 | ㅏ | + 아요 | **일어나요** | 일어나다 |
| 하다 (하다) | 하 | — | → 해요 | **해요** | 하다 |
| 공부하다 (공부하다) | 공부하 | — | → 해요 | **공부해요** | 공부하다 |

### 3.3 단계별 과정

```
마시다 (마시다) 활용 방법:

1단계: -다 제거 → 마시 (어간)
2단계: 어간의 마지막 모음 = ㅣ (ㅏ/ㅗ 아님) → -어요 사용
3단계: 마시 + 어요
4단계: 축약: 시 + 어 → 셔
5단계: 결과 = 마셔요
```

## 4. 현재시제 격식체 (-ㅂ니다/-습니다)

### 4.1 형성

**합쇼체**(격식 존댓말)는 격식 상황, 뉴스, 발표, 낯선 사람에게 사용됩니다.

```
┌──────────────────────────────────────────────────────────────────┐
│           격식 존댓말 현재시제 (-ㅂ니다 / -습니다)                  │
│                                                                  │
│   어간이 모음으로 끝나면:   어간 + ㅂ니다                          │
│     가 → 갑니다     (ㅂ이 가에 붙어 → 갑)                         │
│     하 → 합니다                                                  │
│     마시 → 마십니다                                               │
│                                                                  │
│   어간이 자음으로 끝나면:   어간 + 습니다                          │
│     먹 → 먹습니다                                                │
│     읽 → 읽습니다                                                │
│     있 → 있습니다                                                │
└──────────────────────────────────────────────────────────────────┘
```

### 4.2 활용 예시

| 기본형 | 어간 | 끝음절 | 격식 현재 |
|--------|------|--------|----------|
| 가다 | 가 | 모음 | 갑니다 |
| 오다 | 오 | 모음 | 옵니다 |
| 보다 | 보 | 모음 | 봅니다 |
| 하다 | 하 | 모음 | 합니다 |
| 마시다 | 마시 | 모음 | 마십니다 |
| 먹다 | 먹 | 자음 | 먹습니다 |
| 읽다 | 읽 | 자음 | 읽습니다 |
| 자다 | 자 | 모음 | 잡니다 |
| 있다 | 있 | 자음 | 있습니다 |
| 쓰다 | 쓰 | 모음 | 씁니다 |

### 4.3 격식 의문문: -ㅂ니까? / -습니까?

격식 의문문은 -다를 -까로 바꿉니다:

| 서술문 | 의문문 |
|--------|--------|
| 갑니다 (가다) | 갑니까? (갑니까?) |
| 먹습니다 (먹다) | 먹습니까? (먹습니까?) |
| 합니다 (하다) | 합니까? (합니까?) |
| 있습니다 (있다) | 있습니까? (있습니까?) |

## 5. 과거시제 비격식 존댓말 (-았/었어요)

### 5.1 형성

과거시제는 어간과 어미 사이에 **-았/었-**을 삽입하여 만듭니다:

```
어간 (마지막 모음 ㅏ/ㅗ) + 았어요
어간 (기타 모음)         + 었어요
하다 어간               → 했어요
```

### 5.2 활용 예시

| 기본형 | 어간 | 과거시제(해요체) | 과정 |
|--------|------|----------------|------|
| 가다 (가다) | 가 | **갔어요** | 가 + 았어요 → 갔어요 |
| 오다 (오다) | 오 | **왔어요** | 오 + 았어요 → 왔어요 |
| 보다 (보다) | 보 | **봤어요** | 보 + 았어요 → 봤어요 |
| 먹다 (먹다) | 먹 | **먹었어요** | 먹 + 었어요 |
| 읽다 (읽다) | 읽 | **읽었어요** | 읽 + 었어요 |
| 마시다 (마시다) | 마시 | **마셨어요** | 마시 + 었어요 → 마셨어요 |
| 쓰다 (쓰다) | 쓰 | **썼어요** | 쓰 + 었어요 → 썼어요 |
| 자다 (자다) | 자 | **잤어요** | 자 + 았어요 → 잤어요 |
| 일어나다 (일어나다) | 일어나 | **일어났어요** | 일어나 + 았어요 → 일어났어요 |
| 하다 (하다) | 하 | **했어요** | 하 + 였어요 → 했어요 |
| 공부하다 (공부하다) | 공부하 | **공부했어요** | 공부하 + 였어요 → 공부했어요 |

### 5.3 과거시제 축약 패턴

현재시제와 동일한 축약 규칙이 적용됩니다:

| 패턴 | 과정 | 예시 |
|------|------|------|
| ㅏ + 았 → ㅏㅆ | 가 + 았 → 갔 | 갔어요 (갔어요) |
| ㅗ + 았 → ㅘㅆ | 오 + 았 → 왔 | 왔어요 (왔어요) |
| ㅜ + 었 → ㅝㅆ | 배우 + 었 → 배웠 | 배웠어요 (배웠어요) |
| ㅣ + 었 → ㅕㅆ | 마시 + 었 → 마셨 | 마셨어요 (마셨어요) |
| ㅡ + 었 → 축약 | 쓰 + 었 → 썼 | 썼어요 (썼어요) |
| 하 + 였 → 했 | 하 + 였 → 했 | 했어요 (했어요) |

## 7. 부정형

### 7.1 짧은 부정: 안 + 동사

동사 바로 앞에 **안**(an)을 놓습니다:

```
안 + 동사

가요 → 안 가요        (안 가요)
먹어요 → 안 먹어요    (안 먹어요)
와요 → 안 와요        (안 와요)
읽어요 → 안 읽어요    (안 읽어요)

하다 합성 동사의 경우:
공부해요 → 공부 안 해요   (공부 안 해요) ← 안이 해요 앞에 옴
운동해요 → 운동 안 해요   (운동 안 해요)
```

> **참고**: 하다 합성 동사에서 안은 명사와 하다 사이에 놓입니다(공부 **안** 해요), 전체 동사 앞이 아닙니다(**X** 안 공부해요).

### 7.2 긴 부정: 어간 + -지 않다

어간에 **-지 않다**를 붙입니다. 더 격식적/강조적입니다:

```
어간 + 지 않다   →   어간 + 지 않아요 (존댓말)
                     어간 + 지 않습니다 (격식)

가다   → 가지 않아요      (가지 않아요)
먹다   → 먹지 않아요      (먹지 않아요)
하다   → 하지 않아요      (하지 않아요)
공부하다 → 공부하지 않아요 (공부하지 않아요)
```

### 7.3 짧은 부정 vs 긴 부정

| 짧은(안) | 긴(-지 않다) | 의미 |
|---------|-------------|------|
| 안 가요 | 가지 않아요 | 안 가요 |
| 안 먹어요 | 먹지 않아요 | 안 먹어요 |
| 안 했어요 | 하지 않았어요 | 안 했어요 |
| 공부 안 해요 | 공부하지 않아요 | 공부 안 해요 |

> **언제 어떤 것을 사용하나요?**
> - 안은 일상 대화에서 더 흔함(짧고 빠름)
> - -지 않다는 격식적 글이나 부정을 강조할 때 더 흔함

### 7.4 과거시제 부정

```
짧은: 안 + 과거시제 동사
  안 갔어요        (안 갔어요)
  안 먹었어요      (안 먹었어요)

긴: 어간 + 지 않았어요
  가지 않았어요    (가지 않았어요)
  먹지 않았어요    (먹지 않았어요)
```

### 7.5 특수 부정: 못 (못하다)

**못**(mot)은 "못하다"를 의미합니다 — 선택이 아닌 능력의 부재를 나타냅니다:

```
못 + 동사

가요 → 못 가요        (못 가요)
먹어요 → 못 먹어요    (못 먹어요)
해요 → 못 해요        (못 해요)

긴 형태: 어간 + -지 못하다
  가지 못해요          (가지 못해요)
  먹지 못해요          (먹지 못해요)
```

| 안 (안 하다) | 못 (못 하다) |
|-------------|-------------|
| 안 가요 (안 가요 / 가지 않기로 함) | 못 가요 (못 가요 / 갈 수 없음) |
| 안 먹어요 (안 먹어요) | 못 먹어요 (못 먹어요) |
