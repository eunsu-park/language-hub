## 1. 한국어 형용사 = 상태 동사

영어에서 형용사가 별도의 품사인 것(예: "The house is **big**")과 달리, 한국어 형용사는 **상태 동사**(형용사, hyeong-yong-sa)라는 동사의 일종입니다. 동작 동사와 마찬가지로 활용됩니다.

> **한국어 형용사 vs. 영어 형용사**
>
> - **영어:** "The house is big." — 명사 + be동사 + 형용사
> - **한국어:** "집이 커요." (jib-i keo-yo) — 명사 + 조사 + 상태 동사
>
> 핵심: 크다 (keuda, "크다")가 바로 형용사입니다.
> 별도의 "이다"가 필요 없음 -- 형용사가 이미 이를 포함합니다.
>
> - 기본형: 크다 (크다)
> - 활용형: 커요 (커요 -- 존댓말)
> - 관형형: 큰 + 명사 (큰 [명사])

이것이 의미하는 바:

- 형용사와 함께 **이다**(서술격 조사 "~이다")를 사용할 필요 **없음**
- 형용사는 문장 **끝에** 옴 (SOV 어순의 동사 위치)
- 형용사는 동사처럼 **시제, 높임법, 서법**에 따라 활용됨

| 영어 | 한국어(기본형) | 한국어(존댓말) | 직역 |
|------|-------------|-------------|------|
| The room is small. | 방이 작다. | 방이 작아요. | 방이 작다. |
| The weather is good. | 날씨가 좋다. | 날씨가 좋아요. | 날씨가 좋다. |
| The food is delicious. | 음식이 맛있다. | 음식이 맛있어요. | 음식이 맛있다. |

## 3. 해요체 형용사 활용

한국어 형용사는 비격식 존댓말(해요체) 활용 시 동작 동사와 동일한 모음 조화 규칙을 따릅니다.

### 3.1 기본 활용 규칙

> **형용사 활용 규칙 (해요체)**
>
> 1. 기본형에서 다를 떼어 → 어간 구하기
> 2. 어간의 마지막 모음 확인
> 3. 모음 조화 적용:
>    - 마지막 모음이 ㅏ 또는 ㅗ → 어간 + 아요
>    - 마지막 모음이 그 외 → 어간 + 어요
>    - 어간이 하로 끝남 → 어간 + 여요 (= 해요)

### 3.2 활용 예시

| 기본형 | 어간 | 마지막 모음 | 규칙 | 해요체 |
|--------|------|-----------|------|--------|
| 크다 (크다) | 크 | ㅡ | + 어요 | 커요 (ㅡ+ㅓ 축약) |
| 작다 (작다) | 작 | ㅏ | + 아요 | 작아요 |
| 좋다 (좋다) | 좋 | ㅗ | + 아요 | 좋아요 |
| 나쁘다 (나쁘다) | 나쁘 | ㅡ | + 어요 | 나빠요 (ㅡ 탈락, ㅃ) |
| 많다 (많다) | 많 | ㅏ | + 아요 | 많아요 |
| 비싸다 (비싸다) | 비싸 | ㅏ | + 아요 | 비싸요 (ㅏ+ㅏ 축약) |
| 예쁘다 (예쁘다) | 예쁘 | ㅡ | + 어요 | 예뻐요 (ㅡ 탈락) |
| 덥다 (덥다) | 덥 | ㅓ | + 어요 | 더워요 (ㅂ 불규칙) |
| 춥다 (춥다) | 춥 | ㅜ | + 어요 | 추워요 (ㅂ 불규칙) |

> **ㅂ-불규칙 형용사 참고**: ㅂ으로 끝나는 많은 형용사는 불규칙입니다. 모음 어미 앞에서 ㅂ이 우로 바뀝니다: 덥다 → 더워요, 춥다 → 추워요, 새롭다 → 새로워요.

### 3.3 과거시제

과거시제는 요를 ㅆ어요로 바꿉니다:

| 현재(해요체) | 과거(해요체) | 의미 |
|------------|------------|------|
| 커요 | 컸어요 | 컸어요 |
| 작아요 | 작았어요 | 작았어요 |
| 좋아요 | 좋았어요 | 좋았어요 |
| 더워요 | 더웠어요 | 더웠어요 |
| 추워요 | 추웠어요 | 추웠어요 |
| 예뻐요 | 예뻤어요 | 예뻤어요 |

## 4. 관형형: 명사 앞 형용사

형용사가 명사를 직접 수식할 때(예: "큰 집"), 한국어는 **관형형** -ㄴ/은을 사용합니다.

### 4.1 형성 규칙

> **형용사의 관형형 (-ㄴ/은)**
>
> - 어간이 **모음**으로 끝나면 → 어간 + ㄴ: 크 → 큰, 싸 → 싼, 예쁘 → 예쁜
> - 어간이 **자음**으로 끝나면 → 어간 + 은: 작 → 작은, 많 → 많은, 좋 → 좋은
> - **ㅂ-불규칙**: ㅂ → 우 + ㄴ = 운: 덥 → 더운, 춥 → 추운, 새롭 → 새로운

### 4.2 예시

| 형용사 | 관형형 | + 명사 | 의미 |
|--------|--------|--------|------|
| 크다 (크다) | 큰 | 큰 집 | 큰 집 |
| 작다 (작다) | 작은 | 작은 고양이 | 작은 고양이 |
| 좋다 (좋다) | 좋은 | 좋은 날씨 | 좋은 날씨 |
| 나쁘다 (나쁘다) | 나쁜 | 나쁜 사람 | 나쁜 사람 |
| 비싸다 (비싸다) | 비싼 | 비싼 옷 | 비싼 옷 |
| 싸다 (싸다) | 싼 | 싼 가격 | 싼 가격 |
| 새롭다 (새롭다) | 새로운 | 새로운 시작 | 새로운 시작 |
| 덥다 (덥다) | 더운 | 더운 날 | 더운 날 |
| 예쁘다 (예쁘다) | 예쁜 | 예쁜 꽃 | 예쁜 꽃 |
| 맛있다 (맛있다) | 맛있는* | 맛있는 음식 | 맛있는 음식 |

> **중요**: 있다/없다 합성어(맛있다, 재미있다, 멋있다)는 순수 형용사가 아닌 있다(존재 동사)를 포함하므로 관형형에 **-는**을 사용합니다(-ㄴ/은 아님).

### 4.3 관형형 형용사 문장 사용

```
큰 집에서 살고 싶어요.
큰 집에서 살고 싶어요.

작은 가방을 주세요.
작은 가방을 주세요.

좋은 아이디어예요!
좋은 아이디어예요!

맛있는 한국 음식을 먹었어요.
맛있는 한국 음식을 먹었어요.
```

## 5. 비교 표현

한국어 비교급은 **더 (deo, "더") + 형용사 + 보다 (boda, "보다")** 구조를 사용합니다.

### 5.1 기본 비교: A가 B보다 더 [형용사]

> **비교 구조**
>
> A가/이 B보다 더 [형용사]
> (주격) (보다) (더) (상태 동사)
>
> 예시: 사과가 배보다 더 커요.
> (사과가 배보다 더 크다)

### 5.2 비교 예시

| 한국어 | 해석 |
|--------|------|
| 한국어가 일본어보다 더 쉬워요. | 한국어가 일본어보다 더 쉬워요. |
| 여름이 겨울보다 더 더워요. | 여름이 겨울보다 더 더워요. |
| 이 식당이 그 식당보다 더 맛있어요. | 이 식당이 그 식당보다 더 맛있어요. |
| 오빠가 저보다 더 키가 커요. | 오빠가 저보다 더 키가 커요. |

> **참고**: 보다가 이미 비교를 나타내므로 더를 생략할 수 있습니다: 사과가 배보다 커요도 맞습니다.

### 5.3 "덜" 비교: 덜

"덜"을 말하려면 더 대신 **덜(deol)**을 사용합니다:

| 한국어 | 해석 |
|--------|------|
| 오늘이 어제보다 덜 추워요. | 오늘이 어제보다 덜 추워요. |
| 이것이 그것보다 덜 비싸요. | 이것이 그것보다 덜 비싸요. |

## 6. 최상급 표현

한국어에는 "가장"을 나타내는 두 단어가 있습니다: **가장(gajang)**(표준/문어)과 **제일(jeil)**(구어/일상).

### 6.1 구조: 가장/제일 + [형용사]

| 한국어 | 해석 |
|--------|------|
| 가장 큰 나라 | 가장 큰 나라 |
| 제일 맛있는 음식 | 제일 맛있는 음식 |
| 한국에서 가장 높은 산은 한라산이에요. | 한국에서 가장 높은 산은 한라산이에요. |
| 반에서 제일 똑똑한 학생이에요. | 반에서 제일 똑똑한 학생이에요. |
| 이것이 가장 좋아요. | 이것이 가장 좋아요. |

### 6.2 가장 vs. 제일

| 가장 | 제일 |
|------|------|
| 표준, 문어체 | 구어, 일상적 |
| 격식 글에서 사용 | 일상 대화에서 사용 |
| 가장 중요한 일 | 제일 중요한 일 |

대부분의 상황에서 둘 다 맞습니다. 에세이나 격식 상황에는 가장을, 일상 대화에는 제일을 사용하세요.

## 7. 형용사 부정

한국어에는 동사 부정과 동일한 두 가지 형용사 부정 방법이 있습니다.

### 7.1 짧은 부정: 안 + 형용사

활용된 형용사 바로 앞에 **안**을 놓습니다:

| 긍정 | 부정 | 의미 |
|------|------|------|
| 커요 | 안 커요 | 안 커요 |
| 좋아요 | 안 좋아요 | 안 좋아요 |
| 비싸요 | 안 비싸요 | 안 비싸요 |
| 예뻐요 | 안 예뻐요 | 안 예뻐요 |

### 7.2 긴 부정: 어간 + -지 않다

형용사 어간에 **-지 않다**를 붙이고, 않다를 활용합니다:

| 긍정 | 부정(긴) | 의미 |
|------|---------|------|
| 크다 | 크지 않아요 | 크지 않아요 |
| 좋다 | 좋지 않아요 | 좋지 않아요 |
| 비싸다 | 비싸지 않아요 | 비싸지 않아요 |
| 덥다 | 덥지 않아요 | 덥지 않아요 |

> **뉘앙스**: 긴 형태(-지 않다)는 약간 더 부드럽고 격식적입니다. 짧은 형태(안)는 일상 대화에서 더 흔합니다.

### 7.3 문맥 속 예시

```
오늘 날씨가 안 추워요.
오늘 날씨가 안 추워요.

이 영화가 재미있지 않았어요.
이 영화가 재미있지 않았어요.

그 식당은 안 비싸요. 괜찮아요.
그 식당은 안 비싸요. 괜찮아요.
```

## 8. 형용사 짝과 반의어

형용사를 짝으로 배우면 암기에 도움이 되고 어휘가 넓어집니다.

### 8.1 물리적 속성

| 한국어 | 의미 | 반의어 | 의미 |
|--------|------|--------|------|
| 크다 | 크다 | 작다 | 작다 |
| 길다 | 길다 | 짧다 | 짧다 |
| 높다 | 높다 | 낮다 | 낮다 |
| 넓다 | 넓다 | 좁다 | 좁다 |
| 무겁다 | 무겁다 | 가볍다 | 가볍다 |
| 두껍다 | 두껍다 | 얇다 | 얇다 |

### 8.2 품질과 가치

| 한국어 | 의미 | 반의어 | 의미 |
|--------|------|--------|------|
| 좋다 | 좋다 | 나쁘다 | 나쁘다 |
| 새롭다 | 새롭다 | 오래되다 | 오래되다(사물) |
| 비싸다 | 비싸다 | 싸다 | 싸다 |
| 깨끗하다 | 깨끗하다 | 더럽다 | 더럽다 |
| 쉽다 | 쉽다 | 어렵다 | 어렵다 |

### 8.3 감정과 감각

| 한국어 | 의미 | 반의어 | 의미 |
|--------|------|--------|------|
| 덥다 | 덥다(날씨) | 춥다 | 춥다(날씨) |
| 뜨겁다 | 뜨겁다(물체) | 차갑다 | 차갑다(물체) |
| 기쁘다 | 기쁘다 | 슬프다 | 슬프다 |
| 재미있다 | 재미있다 | 재미없다 | 재미없다 |
| 바쁘다 | 바쁘다 | 한가하다 | 한가하다 |

## 9. 특수 형용사 패턴

### 9.1 있다/없다 합성 형용사

여러 자주 쓰는 형용사가 명사 + 있다(있다/존재하다) 또는 없다(없다/존재하지 않다)의 합성어입니다:

| 합성어 | 구성 | 의미 |
|--------|------|------|
| 맛있다 | 맛 (맛) + 있다 | 맛있다 (직역 "맛이 있다") |
| 맛없다 | 맛 (맛) + 없다 | 맛없다 (직역 "맛이 없다") |
| 재미있다 | 재미 (재미) + 있다 | 재미있다 |
| 재미없다 | 재미 (재미) + 없다 | 재미없다 |
| 멋있다 | 멋 (멋) + 있다 | 멋있다 |
| 멋없다 | 멋 (멋) + 없다 | 멋없다 |

> **기억하세요**: 이들은 관형형에서 **-는**을 사용합니다(-ㄴ/은 아님): 맛있**는** 음식, 재미있**는** 영화.

### 9.2 하다 형용사

많은 한국어 형용사는 **명사 + 하다** 형태로 구성됩니다:

| 한국어 | 의미 | 관형형 |
|--------|------|--------|
| 행복하다 | 행복하다 | 행복한 |
| 건강하다 | 건강하다 | 건강한 |
| 조용하다 | 조용하다 | 조용한 |
| 깨끗하다 | 깨끗하다 | 깨끗한 |
| 유명하다 | 유명하다 | 유명한 |
| 중요하다 | 중요하다 | 중요한 |
| 필요하다 | 필요하다 | 필요한 |
| 특별하다 | 특별하다 | 특별한 |

이들은 활용이 쉽습니다: 하다를 동사로 취급하면 됩니다.

- 행복하다 → 행복해요 (행복해요) → 행복했어요 (행복했어요)
- 조용하다 → 조용해요 (조용해요) → 조용한 방 (조용한 방)

## 10. 연습 문제

### 연습 1: 해요체로 활용하기

다음 형용사를 기본형에서 존댓말 현재시제로 바꾸세요.

1. 작다 → ___
2. 좋다 → ___
3. 예쁘다 → ___
4. 덥다 → ___
5. 새롭다 → ___
6. 깨끗하다 → ___

### 연습 2: 관형형

빈칸에 올바른 관형형을 넣으세요.

1. ___ 집 (크다 → 큰 집)
2. ___ 음식 (맛있다 → 맛있는 음식)
3. ___ 날씨 (좋다 → 좋은 날씨)
4. ___ 날 (덥다 → 더운 날)
5. ___ 사람 (유명하다 → 유명한 사람)
6. ___ 방 (조용하다 → 조용한 방)

### 연습 3: 비교문 만들기

주어진 단어로 비교 문장을 만드세요.

1. 서울 / 부산 / 크다 (서울이 부산보다 더 커요)
2. 여름 / 봄 / 덥다 (여름이 봄보다 더 더워요)
3. 한국어 / 영어 / 어렵다 (한국어가 영어보다 더 어려워요)

1. 이 가방이 비싸요.
2. 어제 날씨가 추웠어요.
3. 그것이 가장 맛있는 음식이에요.
4. 이 방이 안 커요.
5. 조용한 집에서 살고 싶어요.
