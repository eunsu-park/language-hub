## 1. SOV 어순

### 1.1 근본적 차이

영어는 **SVO**(주어-동사-목적어) 어순을 따릅니다. 한국어는 **SOV**(주어-목적어-동사) 어순을 따릅니다. 동사는 항상 문장 끝에 옵니다.

```
┌──────────────────────────────────────────────────────────────────┐
│              영어 (SVO) vs 한국어 (SOV)                            │
│                                                                  │
│   영어:    I       eat     an apple.                             │
│            [주]    [동]    [목]                                   │
│                                                                  │
│   한국어:  저는    사과를    먹어요.                                │
│            [주]    [목]     [동]                                  │
│            나      사과를    먹다                                  │
│                                                                  │
│   영어:    She     studies  Korean.                              │
│            [주]    [동]     [목]                                  │
│                                                                  │
│   한국어:  그녀는   한국어를   공부해요.                             │
│            [주]     [목]      [동]                                │
│            그녀는   한국어를   공부하다                              │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 한국어 어순의 핵심 원칙

1. **동사는 항상 마지막에 옵니다.** 이것은 유일한 절대 규칙입니다.
2. **다른 요소는 유연합니다.** 주어, 목적어, 부사는 강조를 위해 순서를 바꿀 수 있습니다.
3. **조사가 역할을 표시합니다.** 조사(은/는, 이/가, 을/를 등)가 있으므로 어순이 유연합니다 — 조사가 위치에 관계없이 각 단어의 역할을 알려줍니다.
4. **주어/목적어는 문맥에서 명확할 때 자주 생략됩니다.**

```
다음 문장 모두 "나는 사과를 먹어요"라는 의미:

저는 사과를 먹어요.        (표준: 주-목-동)
사과를 저는 먹어요.        ("나"를 강조: 목-주-동)
사과를 먹어요.             (주어 생략: 목-동)
먹어요.                    (둘 다 생략: 동 — 문맥으로 파악)
```

### 1.3 영어 문장 패턴과의 비교

| 영어 | 한국어 | 패턴 |
|------|--------|------|
| I go. | 저는 가요. | 주-동 |
| I eat rice. | 저는 밥을 먹어요. | 주-목-동 |
| I give a friend a book. | 저는 친구에게 책을 줘요. | 주-간접목-목-동 |
| I study Korean at school. | 저는 학교에서 한국어를 공부해요. | 주-장소-목-동 |

## 2. 주제 표지 은/는

### 2.1 주제란?

주제(주제, juje)는 문장이 "~에 대해" 말하는 대상입니다. 한국어는 조사 **은/는**으로 주제를 명시적으로 표시합니다:

- **은** 받침이 있는 단어 뒤: 학생**은** (생이 ㅇ 받침으로 끝남)
- **는** 받침이 없는 단어 뒤: 저**는** (저가 ㅓ 모음으로 끝남)

```
┌──────────────────────────────────────────────────────────────────┐
│                    주제 표지 은/는                                 │
│                                                                  │
│   규칙:  받침 있는 단어 + 은     저는 → 불가(저는 모음으로         │
│          받침 없는 단어 + 는     끝나므로 는)                      │
│                                                                  │
│   받침 있는 단어:                  받침 없는 단어:                  │
│     학생 → 학생은 (학생)           저 → 저는 (나)                  │
│     선생님 → 선생님은 (선생님)     사과 → 사과는 (사과)            │
│     한국 → 한국은 (한국)           커피 → 커피는 (커피)            │
└──────────────────────────────────────────────────────────────────┘
```

### 2.2 은/는의 기능

| 기능 | 예시 | 해석 |
|------|------|------|
| 주제 설정 | 저**는** 학생입니다. | 저에 대해 말하면, 학생입니다. |
| 대조 | 커피**는** 좋아요. 차**는** 싫어요. | 커피는 좋아요. 차는 싫어요. |
| 일반 진술 | 한국어**는** 재미있어요. | 한국어는(일반적으로) 재미있어요. |

### 2.3 예시

| 한국어 | 직역 | 자연스러운 해석 |
|--------|------|----------------|
| 날씨는 좋아요. | 날씨에 대해 말하면, 좋아요. | 날씨가 좋아요. |
| 이것은 책입니다. | 이것에 대해 말하면, 책입니다. | 이것은 책입니다. |
| 저는 미국 사람입니다. | 저에 대해 말하면, 미국 사람입니다. | 저는 미국 사람입니다. |
| 한국 음식은 맛있어요. | 한국 음식에 대해 말하면, 맛있어요. | 한국 음식은 맛있어요. |

## 3. 주격 조사 이/가

### 3.1 주격 조사

주격 조사 **이/가**는 문장의 주어를 표시하며, 주로 **새로운 정보**를 소개하거나 "누가/무엇이"에 대한 답변을 나타냅니다:

- **이** 받침이 있는 단어 뒤: 학생**이** (생이 ㅇ으로 끝남)
- **가** 받침이 없는 단어 뒤: 누구**가** (구가 ㅜ로 끝남)

### 3.2 은/는 vs 이/가 — 핵심 차이

```
┌──────────────────────────────────────────────────────────────────┐
│               은/는 (주제) vs 이/가 (주격)                         │
│                                                                  │
│   은/는은 알려진 정보(논의 중인 주제)를 표시:                      │
│     "~에 대해 말하면..."                                          │
│                                                                  │
│   이/가는 새로운 정보(주장되는 내용)를 표시:                       │
│     "~인 것은 바로..."                                            │
│                                                                  │
│   대화 예시:                                                      │
│   질문: 누가 학생이에요?      "누가 학생이에요?"                   │
│   답변: 민수가 학생이에요.    "민수가 학생이에요." (새 정보)        │
│                                                                  │
│   질문: 민수는 뭐예요?       "민수는 뭐예요?"                     │
│   답변: 민수는 학생이에요.   "민수는 학생이에요." (주제=민수)       │
│                                                                  │
│   같은 문장, 다른 조사, 다른 뉘앙스:                              │
│     민수가 학생이에요 = 학생인 것은 바로 민수                      │
│                        (누구인지에 초점)                           │
│     민수는 학생이에요 = 민수에 대해 말하면, 학생이에요              │
│                        (주제가 이미 정해짐)                        │
└──────────────────────────────────────────────────────────────────┘
```

### 3.3 이/가의 특수 용법

| 용법 | 예시 | 해석 |
|------|------|------|
| 새 정보 / "누가/무엇이"에 대한 답 | **누가** 왔어요? 민수**가** 왔어요. | 누가 왔어요? 민수가 왔어요. |
| 좋다/싫다 뒤 | 저는 커피**가** 좋아요. | 저는 커피가 좋아요. (직역: 커피가 나에게 좋다.) |
| 있다/없다 뒤 | 시간**이** 있어요. | 시간이 있어요. (직역: 시간이 존재한다.) |
| 되다 뒤 | 의사**가** 되었어요. | 의사가 되었어요. |
| 방금 일어난 일 묘사 | 비**가** 와요. | 비가 와요. (비가 오고 있다.) |

## 4. 서술격 조사 이다/아니다

### 4.1 이다 — "~이다"

서술격 조사 이다(ida)는 "~이다"를 의미하며 명사에 직접 붙습니다(띄어쓰기 없이):

```
[명사]이다  →  [명사]이다

학생이다    →  학생이다
한국 사람이다 → 한국 사람이다
```

**활용형**:

| 높임법 | 받침 있는 단어 뒤 | 받침 없는 단어 뒤 | 예시 |
|--------|-----------------|-----------------|------|
| 격식체 (합쇼체) | -입니다 | -입니다 | 학생입니다 / 의사입니다 |
| 비격식 존댓말 (해요체) | -이에요 | -예요 | 학생이에요 / 의사예요 |
| 반말 (해체) | -이야 | -야 | 학생이야 / 의사야 |

| 한국어 | 해석 |
|--------|------|
| 저는 학생입니다. | 저는 학생입니다. (격식) |
| 이것은 책이에요. | 이것은 책이에요. (존댓말) |
| 오늘은 월요일이에요. | 오늘은 월요일이에요. (존댓말) |

### 4.2 아니다 — "~가 아니다"

아니다(anida)는 부정 서술격 조사입니다. 이다와 달리, 명사에 주격 조사 이/가를 붙인 후 아니다를 씁니다:

```
[명사]이/가 아니다  →  [명사]가 아니다

학생이 아니다      →  학생이 아니다
```

**활용형**:

| 높임법 | 형태 | 예시 |
|--------|------|------|
| 격식체 | 아닙니다 | 학생이 아닙니다 (학생이 아닙니다) |
| 비격식 존댓말 | 아니에요 | 학생이 아니에요 (학생이 아니에요) |
| 반말 | 아니야 | 학생이 아니야 (학생이 아니야) |

### 4.3 요약 비교

```
┌──────────────────────────────────────────────────────────────────┐
│            이다 vs 아니다 — 간편 참고표                            │
│                                                                  │
│   긍정:                                                          │
│     저는 학생입니다.            저는 학생입니다.                    │
│     이것은 펜이에요.            이것은 펜이에요.                    │
│                                                                  │
│   부정:                                                          │
│     저는 학생이 아닙니다.       저는 학생이 아닙니다.              │
│     이것은 펜이 아니에요.       이것은 펜이 아니에요.              │
│                                                                  │
│   구조적 차이 참고:                                               │
│     이다: [명사]입니다         (명사 + 서술격 조사, 붙여 씀)       │
│     아니다: [명사]이/가 아닙니다 (명사 + 이/가 + 부정 서술격 조사) │
└──────────────────────────────────────────────────────────────────┘
```

## 5. 지시 대명사 이/그/저

### 5.1 삼원 구분

한국어는 화자와 청자로부터의 거리에 따른 삼원 지시 대명사 체계를 가지고 있습니다:

| 지시사 | 의미 | 거리 |
|--------|------|------|
| 이 (i) | 이 | **화자**에게 가까움 |
| 그 (geu) | 그 | **청자**에게 가까움 / 알려진/언급된 것 |
| 저 (jeo) | 저 (저기) | **화자와 청자 모두**에게서 멀리 |

```
┌──────────────────────────────────────────────────────────────────┐
│              삼원 지시 대명사 체계                                  │
│                                                                  │
│                                                                  │
│    화자              청자              멀리                       │
│      ●                ●                                          │
│     [이]             [그]              [저]                       │
│     이것              그것              저것                       │
│   (나에게 가까움)   (너에게 가까움 /  (둘 다에게서                │
│                    이미 언급된 것)     멀리)                      │
│                                                                  │
│   이 책 = 이 책 (내가 들고 있는)                                  │
│   그 책 = 그 책 (네가 들고 있는 / 우리가 얘기한)                  │
│   저 책 = 저 책 (저기 선반 위에 있는)                             │
└──────────────────────────────────────────────────────────────────┘
```

### 5.2 지시사 계열

| 접두사 | + 것 (사물) | + 곳/데 (장소) | + 사람 (사람) | + 때 (시간) |
|--------|-----------|---------------|-------------|------------|
| 이 | 이것 (이것) | 이곳/여기 (여기) | 이 사람 (이 사람) | 이때 (이때) |
| 그 | 그것 (그것) | 그곳/거기 (거기) | 그 사람 (그 사람) | 그때 (그때) |
| 저 | 저것 (저것) | 저곳/저기 (저기) | 저 사람 (저 사람) | — |

**축약형** (대화에서 매우 흔함):

| 원형 | 축약형 | 의미 |
|------|--------|------|
| 이것 | 이거 | 이것 |
| 그것 | 그거 | 그것 |
| 저것 | 저거 | 저것 |
| 이것은 | 이건 | 이것은 |
| 그것은 | 그건 | 그것은 |

### 5.3 예시

| 한국어 | 해석 |
|--------|------|
| 이것은 뭐예요? | 이것은 뭐예요? |
| 그것은 가방이에요. | 그것은 가방이에요. |
| 저 건물은 학교예요. | 저 건물은 학교예요. |
| 이 사람은 제 친구예요. | 이 사람은 제 친구예요. |
| 그 영화는 재미있었어요. | 그 영화는(우리가 얘기한) 재미있었어요. |

## 6. 있다/없다 — 존재와 소유

### 6.1 두 가지 중요한 동사

**있다**(itta)와 **없다**(eoptta)는 한국어에서 가장 많이 사용되는 단어입니다. 두 가지 주요 의미가 있습니다:

```
┌──────────────────────────────────────────────────────────────────┐
│                   있다 / 없다                                     │
│                                                                  │
│   의미 1: 존재                                                    │
│     있다 = 존재하다, ~이 있다                                     │
│     없다 = 존재하지 않다, ~이 없다                                │
│                                                                  │
│     고양이가 있어요.       고양이가 있어요.                        │
│     고양이가 없어요.       고양이가 없어요.                        │
│                                                                  │
│   의미 2: 소유 (가지다 / 가지지 않다)                             │
│     있다 = 가지고 있다                                            │
│     없다 = 가지고 있지 않다                                       │
│                                                                  │
│     저는 차가 있어요.      차가 있어요.                            │
│     저는 차가 없어요.      차가 없어요.                            │
│                                                                  │
│   참고: 존재하거나 소유된 대상은 이/가(을/를 아님)를 사용          │
└──────────────────────────────────────────────────────────────────┘
```

### 6.2 활용

| 높임법 | 있다 | 없다 |
|--------|------|------|
| 격식체 | 있습니다 | 없습니다 |
| 비격식 존댓말 | 있어요 | 없어요 |
| 반말 | 있어 | 없어 |

### 6.3 장소 + 에 + 있다/없다

어떤 **장소에** 무엇이 있다고 말하려면, 조사 **에**를 사용합니다:

```
[장소]에 [사물]이/가 있어요.    [장소]에 [사물]이 있어요.
[사물]은/는 [장소]에 있어요.    [사물]은 [장소]에 있어요.
```

| 한국어 | 해석 |
|--------|------|
| 책상 위에 책이 있어요. | 책상 위에 책이 있어요. |
| 냉장고에 우유가 없어요. | 냉장고에 우유가 없어요. |
| 화장실은 2층에 있어요. | 화장실은 2층에 있어요. |
| 여기에 뭐가 있어요? | 여기에 뭐가 있어요? |

### 6.4 자주 쓰는 위치 단어

| 한국어 | 뜻 | 한국어 | 뜻 |

## 7. 의문사

### 7.1 필수 의문사 (의문사)

| 한국어 | 로마자 표기 | 의미 | 예시 |
|--------|-----------|------|------|
| 뭐 (무엇) | mwo (mueot) | 무엇 | 이것은 뭐예요? (이것은 뭐예요?) |
| 누구 | nugu | 누구 | 누가 왔어요? (누가 왔어요?) |
| 어디 | eodi | 어디 | 어디에 가요? (어디에 가요?) |
| 언제 | eonje | 언제 | 언제 와요? (언제 와요?) |
| 왜 | wae | 왜 | 왜 안 가요? (왜 안 가요?) |
| 어떻게 | eotteoke | 어떻게 | 어떻게 해요? (어떻게 해요?) |
| 몇 | myeot | 몇 / 몇 번째 | 몇 시예요? (몇 시예요?) |
| 얼마 | eolma | 얼마(가격) | 이것은 얼마예요? (이것은 얼마예요?) |
| 어떤 | eotteon | 어떤 종류의 | 어떤 음악을 좋아해요? (어떤 음악을 좋아해요?) |
| 어느 | eoneu | 어느 | 어느 나라에서 왔어요? (어느 나라에서 왔어요?) |

### 7.2 누구의 특수 용법

누구(누구)가 주어로 쓰일 때는 **누가**(누구가 아님)가 됩니다:

```
누구 + 가 → 누가    (불규칙 축약)

누가 왔어요?       누가 왔어요?
누가 했어요?       누가 했어요?
이것은 누구의 것이에요?   이것은 누구의 것이에요? (누구 + 의 = 누구의)
```

### 7.3 뭐 vs 무엇

뭐는 무엇의 구어/일상적 형태입니다:

| 상황 | 형태 | 예시 |
|------|------|------|
| 구어/일상 | 뭐 | 뭐 먹어요? (뭐 먹어요?) |
| 문어/격식 | 무엇 | 무엇을 하십니까? (무엇을 하십니까?) |
| 조사와 함께 | 무엇 / 뭐 | 뭐가 있어요? / 무엇이 있어요? (뭐가 있어요?) |

## 8. 기본 의문문 만들기

### 8.1 한국어 의문문 만들기

한국어의 의문문은 매우 간단합니다 — 영어보다 훨씬 쉽습니다(do/does/did 불필요, 주어-동사 도치 불필요).

```
┌──────────────────────────────────────────────────────────────────┐
│              한국어 의문문 만들기                                   │
│                                                                  │
│   방법 1: 상승 억양 (비격식 존댓말 — 해요체)                      │
│     서술: 학생이에요.↘     (학생이에요)                            │
│     의문: 학생이에요?↗     (학생이에요?)                           │
│     → 끝을 올려 말하기만 하면 됩니다!                              │
│                                                                  │
│   방법 2: 격식 의문 어미 (합쇼체)                                  │
│     서술: 학생입니다.      (학생입니다)                             │
│     의문: 학생입니까?      (학생입니까?)                            │
│     → -ㅂ니다를 -ㅂ니까로 변경                                     │
│                                                                  │
│   방법 3: 의문사 + 문장                                           │
│     뭐예요?               뭐예요?                                 │
│     어디에 가요?           어디에 가요?                             │
│     언제 와요?             언제 와요?                              │
│     → 어순 변경 불필요 — 의문사만 추가                             │
└──────────────────────────────────────────────────────────────────┘
```

### 8.2 예/아니요 의문문

| 서술문 | 의문문 | 해석 |
|--------|--------|------|
| 학생입니다. | 학생입니까? | 학생입니까? (격식) |
| 학생이에요. | 학생이에요? | 학생이에요? (존댓말) |
| 한국 사람이에요. | 한국 사람이에요? | 한국 사람이에요? |
| 커피 있어요. | 커피 있어요? | 커피 있어요? |

### 8.3 정보 의문문 (의문사 의문문)

| 한국어 | 해석 | 패턴 |
|--------|------|------|
| 이것은 뭐예요? | 이것은 뭐예요? | [주제] + 뭐 + 이다 |
| 누가 학생이에요? | 누가 학생이에요? | 누가 + [명사] + 이다 |
| 어디에 가요? | 어디에 가요? | 어디 + 에 + 동사 |
| 언제 와요? | 언제 와요? | 언제 + 동사 |
| 왜 안 먹어요? | 왜 안 먹어요? | 왜 + 안 + 동사 |
| 어떻게 가요? | 어떻게 가요? | 어떻게 + 동사 |
| 몇 살이에요? | 몇 살이에요? | 몇 + 단위 + 이다 |
| 이거 얼마예요? | 이거 얼마예요? | [사물] + 얼마 + 이다 |

## 9. 연습 문제

### 연습 1: 올바른 조사 넣기 (은/는 또는 이/가)

```
1. 저___ 학생입니다.              (저는 학생입니다.)
2. 날씨___ 좋아요.               (날씨가 좋아요.)
3. 누구___ 선생님이에요?          (누가 선생님이에요?)
4. 시간___ 없어요.               (시간이 없어요.)
5. 한국어___ 재미있어요.          (한국어는 재미있어요.)
6. 물___ 있어요?                  (물이 있어요?)
```

### 연습 2: 한국어로 번역하기

```
1. 이것은 책이에요.
2. 그것은 학교가 아니에요.
3. 저는 선생님이 아닙니다.
4. 방에 고양이가 있어요.
5. 이것은 뭐예요?
6. 화장실은 어디에 있어요?
```

### 연습 3: 이다 또는 아니다 고르기

올바른 서술격 조사 형태(비격식 존댓말)를 넣으세요:

```
1. 이것은 가방___. (이것은 가방이에요.)
2. 저는 의사가 ___. (저는 의사가 아니에요.)
3. 오늘은 금요일___. (오늘은 금요일이에요.)
4. 그것은 물이 ___. (그것은 물이 아니에요.)
5. 저 사람은 선생님___. (저 사람은 선생님이에요.)
```

### 연습 4: 의문문 만들기

각 서술문을 의문문으로 바꾸세요:

```
1. 학생입니다.        → ___ (격식 의문문)
2. 한국 사람이에요.    → ___ (존댓말 의문문)
3. 이것은 펜이에요.    → ___ (뭐 사용)
4. 학교에 가요.        → ___ (어디 사용)
5. 민수 씨가 했어요.   → ___ (누가 사용)
```

### 연습 5: 있다/없다로 빈칸 채우기

```
1. 냉장고에 우유가 ___. (냉장고에 우유가 있어요.)
2. 저는 돈이 ___. (저는 돈이 없어요.)
3. 책상 위에 뭐가 ___? (책상 위에 뭐가 있어요?)
4. 여기에 화장실이 ___? (여기에 화장실이 있어요?)
5. 시간이 ___. (시간이 없어요.)
```

### 연습 6: 한국어를 해석하기

```
1. 이것은 뭐예요?
2. 저는 미국 사람이 아닙니다.
3. 그 사람은 누구예요?
4. 학교는 어디에 있어요?
5. 왜 안 먹어요?
6. 저 건물은 병원이에요.
```
