## 1. 한자어 수

한자어 수(한자어 수, hanja-eo su)는 한자에서 유래한 숫자입니다. 큰 숫자, 날짜, 금액, 전화번호, 주소, 수학 관련 맥락에서 사용됩니다.

### 1.1 기본 한자어 수

| 숫자 | 한국어 | 로마자 표기 | 한자 |
|------|--------|-----------|------|
| 0 | 영/공 | yeong/gong | 零/空 |
| 1 | 일 | il | 一 |
| 2 | 이 | i | 二 |
| 3 | 삼 | sam | 三 |
| 4 | 사 | sa | 四 |
| 5 | 오 | o | 五 |
| 6 | 육 | yuk | 六 |
| 7 | 칠 | chil | 七 |
| 8 | 팔 | pal | 八 |
| 9 | 구 | gu | 九 |
| 10 | 십 | sip | 十 |

### 1.2 큰 숫자 만들기

한자어 수는 자릿값 표지를 사용하여 체계적으로 만들어집니다:

| 숫자 | 한국어 | 구성 |
|------|--------|------|
| 11 | 십일 | 10 + 1 |
| 15 | 십오 | 10 + 5 |
| 20 | 이십 | 2 × 10 |
| 25 | 이십오 | 2 × 10 + 5 |
| 30 | 삼십 | 3 × 10 |
| 99 | 구십구 | 9 × 10 + 9 |
| 100 | 백 | 百 |
| 200 | 이백 | 2 × 100 |
| 1,000 | 천 | 千 |
| 3,500 | 삼천오백 | 3 × 1000 + 5 × 100 |
| 10,000 | 만 | 萬 |
| 50,000 | 오만 | 5 × 10000 |
| 100,000 | 십만 | 10 × 10000 |
| 1,000,000 | 백만 | 100 × 10000 |

> **한국어 수 체계: 만(10,000) 단위**
>
> 한국어는 숫자를 1,000이 아니라 10,000(만) 단위로 묶습니다.
> 이것이 영어 세기 방식과 가장 큰 차이입니다.
>
> - 10 → 십
> - 100 → 백
> - 1,000 → 천
> - 10,000 → 만 (새 단위, 영어에 대응 없음)
> - 100,000 → 십만 (10 x 만)
> - 1,000,000 → 백만 (100 x 만)
> - 10,000,000 → 천만 (1000 x 만)
> - 100,000,000 → 억 (새 단위)
>
> 예: 350,000 = 35 x 만 = 삼십오만 (삼백오십천 아님)

### 1.3 전화번호

전화번호는 한자어 수로 한 자리씩 읽습니다. 0은 공(gong)을 사용합니다:

```
010-1234-5678
→ 공일공 - 일이삼사 - 오육칠팔
  (gong-il-gong  il-i-sam-sa  o-yuk-chil-pal)

전화번호가 뭐예요?        전화번호가 뭐예요?
공일공 이삼사오 육칠팔구예요.  공일공 이삼사오 육칠팔구예요.
```

## 2. 고유어 수

고유어 수(고유어 수, goyueo su)는 원래의 한국어 수 체계입니다. 물건을 셀 때, 시간(시), 나이를 표현할 때, 대부분의 단위 명사와 함께 사용됩니다.

### 2.1 기본 고유어 수 (1–99)

| 숫자 | 한국어 | 로마자 표기 | 단위 명사 앞 형태 |
|------|--------|-----------|-----------------|
| 1 | 하나 | hana | 한 |
| 2 | 둘 | dul | 두 |
| 3 | 셋 | set | 세 |
| 4 | 넷 | net | 네 |
| 5 | 다섯 | daseot | 다섯 |
| 6 | 여섯 | yeoseot | 여섯 |
| 7 | 일곱 | ilgop | 일곱 |
| 8 | 여덟 | yeodeol | 여덟 |
| 9 | 아홉 | ahop | 아홉 |
| 10 | 열 | yeol | 열 |

| 숫자 | 한국어 | 로마자 표기 |
|------|--------|-----------|
| 11 | 열하나 | yeolhana |
| 12 | 열둘 | yeoldul |
| 20 | 스물 | seumul |
| 21 | 스물하나 | seumulhana |
| 30 | 서른 | seoreun |
| 40 | 마흔 | maheun |
| 50 | 쉰 | swin |
| 60 | 예순 | yesun |
| 70 | 일흔 | ilheun |
| 80 | 여든 | yeodeun |
| 90 | 아흔 | aheun |
| 99 | 아흔아홉 | aheunahop |

> **중요**: 고유어 수는 99까지만 있습니다. 100 이상은 항상 한자어 수를 사용합니다.

### 2.2 단위 명사 앞 축약형

하나, 둘, 셋, 넷, 스물은 **단위 명사 앞에서** 축약됩니다:

```
하나 → 한     하나 + 개 → 한 개 (하나)
둘  → 두      둘 + 명 → 두 명 (둘)
셋  → 세      셋 + 시 → 세 시 (세 시)
넷  → 네      넷 + 잔 → 네 잔 (네 잔)
스물 → 스무    스물 + 살 → 스무 살 (스무 살)
```

## 3. 어떤 체계를 언제 사용하는가

> **한자어 수(일, 이, 삼) vs 고유어 수(하나, 둘, 셋)**
>
> **한자어 수:**
> - 날짜 (년/월/일), 금액 (원), 전화번호
> - 분(분)과 초(초), 주소, 월 (월)
> - 100 이상의 숫자, 수학/과학/통계
> - 층수 (층), 연도 (년)
>
> **고유어 수:**
> - 물건 세기 (+ 단위 명사), 나이 (살)
> - 시간(시), 수량 (1-99), 수량 표현
>
> **혼합 -- 시각 표현:**
> - [고유어] 시 [한자어] 분
> - 두 시 삼십 분 = 2:30

## 4. 분류사/단위 명사

한국어는 물건을 셀 때 **분류사**(분류사/단위 명사)를 사용합니다 — 영어에서 "a sheet of paper"나 "a cup of coffee"라고 하는 것과 비슷합니다. 구조:

```
[고유어 수] + [단위 명사]

사과 세 개        사과 세 개       (세 = 3, 개 = 일반 단위)
학생 두 명        학생 두 명       (두 = 2, 명 = 사람 단위)
```

### 4.1 필수 단위 명사

| 단위 명사 | 대상 | 예시 |
|----------|------|------|
| 개 (gae) | 일반 사물 | 사과 두 개 (사과 두 개) |
| 명 (myeong) | 사람 (공손) | 학생 세 명 (학생 세 명) |
| 사람 (saram) | 사람 (일상) | 세 사람 (세 사람) |
| 분 (bun) | 사람 (존경) | 두 분 (두 분 — 존칭) |
| 마리 (mari) | 동물 | 고양이 한 마리 (고양이 한 마리) |
| 잔 (jan) | 컵/잔(음료) | 커피 한 잔 (커피 한 잔) |
| 권 (gwon) | 책, 권 | 책 다섯 권 (책 다섯 권) |
| 장 (jang) | 납작한 것(종이, 표) | 종이 열 장 (종이 열 장) |
| 대 (dae) | 탈것, 기계 | 차 한 대 (차 한 대) |
| 벌 (beol) | 옷(세트) | 옷 두 벌 (옷 두 벌) |
| 그루 (geuru) | 나무 | 나무 세 그루 (나무 세 그루) |
| 병 (byeong) | 병 | 물 한 병 (물 한 병) |
| 송이 (songi) | 꽃(송이/줄기) | 장미 열 송이 (장미 열 송이) |
| 살 (sal) | 나이(세) | 스무 살 (스무 살) |
| 번 (beon) | 횟수(한자어) | 세 번 (세 번) |
| 층 (cheung) | 층(한자어) | 삼 층 (삼 층) |

### 4.2 단위 명사와의 어순

두 가지 일반 패턴:

```
패턴 1: [명사] [숫자] [단위 명사]     ← 더 흔함
  사과 세 개                           사과 세 개

패턴 2: [숫자] [단위 명사]의 [명사]    ← 격식/문어
  세 개의 사과                          사과 세 개
```

### 4.3 문장 속 예시

| 한국어 | 해석 |
|--------|------|
| 커피 두 잔 주세요. | 커피 두 잔 주세요. |
| 강아지가 세 마리 있어요. | 강아지가 세 마리 있어요. |
| 책을 한 권 읽었어요. | 책을 한 권 읽었어요. |
| 사과 몇 개 있어요? | 사과 몇 개 있어요? |
| 학생이 스무 명이에요. | 학생이 스무 명이에요. |

## 5. 시각 표현

### 5.1 혼합 체계

한국어는 두 수 체계를 **혼합**하여 시각을 말합니다:
- **시**: 고유어 수 + 시 (si)
- **분**: 한자어 수 + 분 (bun)
- **초**: 한자어 수 + 초 (cho)

> **한국어 시각 표현**
>
> [고유어] 시 [한자어] 분 [한자어] 초
> (시) (분) (초)
>
> - 3:30 → 세 시 삼십 분 (se si samsip bun)
> - 7:15 → 일곱 시 십오 분 (ilgop si sibo bun)
> - 12:00 → 열두 시 (yeoldu si)
> - 1:45 → 한 시 사십오 분 (han si sasibo bun)
>
> - "지금 몇 시예요?" → 지금 몇 시예요?
> - "세 시예요." → 세 시예요.
>
> - 오전 → 오전 (ojeon)
> - 오후 → 오후 (ohu)
> - 오후 두 시 = 오후 2:00

### 5.2 시 (시)

| 시각 | 한국어 |
|------|--------|
| 1:00 | 한 시 |
| 2:00 | 두 시 |
| 3:00 | 세 시 |
| 4:00 | 네 시 |
| 5:00 | 다섯 시 |
| 6:00 | 여섯 시 |
| 7:00 | 일곱 시 |
| 8:00 | 여덟 시 |
| 9:00 | 아홉 시 |
| 10:00 | 열 시 |
| 11:00 | 열한 시 |
| 12:00 | 열두 시 |

### 5.3 반과 15분

| 표현 | 한국어 | 의미 |
|------|--------|------|
| 반 (ban) | 세 시 반 | 3:30 (3시 반) |
| 십오 분 | 세 시 십오 분 | 3:15 (3시 15분) |
| 사십오 분 | 세 시 사십오 분 | 3:45 (3시 45분) |

### 5.4 시간 관련 어휘

| 한국어 | 뜻 | 한국어 | 뜻 |
|--------|-----|--------|-----|
| 지금 | 지금 | 아까 | 아까 |
| 오늘 | 오늘 | 내일 | 내일 |
| 어제 | 어제 | 모레 | 모레 |

| 오후 | 오후 | 저녁 | 저녁 |
| 밤 | 밤 | 새벽 | 새벽 |

## 6. 요일, 월, 날짜

### 6.1 요일 (요일)

모든 요일은 요일(yoil, "요일")로 끝납니다. 접두어는 한자어입니다:

| 한국어 | 한자 | 요소 | 뜻 |
|--------|------|------|-----|
| 월요일 | 月 | 달 | 월요일 |
| 화요일 | 火 | 불 | 화요일 |
| 수요일 | 水 | 물 | 수요일 |
| 목요일 | 木 | 나무 | 목요일 |
| 금요일 | 金 | 쇠/금 | 금요일 |
| 토요일 | 土 | 흙 | 토요일 |
| 일요일 | 日 | 해 | 일요일 |

```
오늘 무슨 요일이에요?     오늘 무슨 요일이에요?
오늘은 수요일이에요.       오늘은 수요일이에요.
```

### 6.2 월 (월)

월은 한자어 수 + 월(wol, "월")을 사용합니다:

| 월 | 한국어 | 월 | 한국어 |
|-----|--------|-----|--------|
| 1월 | 일월 | 7월 | 칠월 |
| 2월 | 이월 | 8월 | 팔월 |
| 3월 | 삼월 | 9월 | 구월 |
| 4월 | 사월 | 10월 | 시월* |
| 5월 | 오월 | 11월 | 십일월 |
| 6월 | 유월* | 12월 | 십이월 |

> ***불규칙 발음**: 유월(육월 아님)과 시월(십월 아님). 발음을 쉽게 하기 위해 ㄱ과 ㅂ이 탈락합니다.

### 6.3 날짜

날짜는 한자어 수 + 일(il, "일")을 사용합니다:

```
[연도]년 [월]월 [일]일

2024년 3월 15일  →  이천이십사년 삼월 십오일
                    (2024년 3월 15일)

오늘 며칠이에요?        오늘 며칠이에요?
삼월 십오일이에요.       삼월 십오일이에요.
```

### 6.4 기간

| 표현 | 단위 명사 | 수 체계 | 예시 |
|------|----------|--------|------|
| ___ 년 | 년 (nyeon) | 한자어 | 삼 년 (3년) |
| ___ 개월 | 개월 (gaewol) | 한자어 | 육 개월 (6개월) |
| ___ 주 | 주 (ju) | 한자어 | 이 주 (2주) |
| ___ 일 | 일 (il) | 한자어 | 오 일 (5일) |
| ___ 시간 | 시간 (sigan) | 고유어 | 세 시간 (3시간) |
| ___ 분 | 분 (bun) | 한자어 | 삼십 분 (30분) |

## 7. 나이

### 7.1 나이 표현

나이는 **고유어 수** + 살(sal)을 사용합니다:

| 나이 | 한국어 | 나이 | 한국어 |
|------|--------|------|--------|
| 1살 | 한 살 | 15 | 열다섯 살 |
| 2 | 두 살 | 20 | 스무 살 |
| 3 | 세 살 | 25 | 스물다섯 살 |
| 5 | 다섯 살 | 30 | 서른 살 |
| 10 | 열 살 | 50 | 쉰 살 |

> **100살 이상**: 한자어를 사용: 백 살, 백한 살 등.

### 7.2 나이 묻기

| 표현 | 격식 수준 | 대상 |
|------|----------|------|
| 몇 살이에요? | 비격식 존댓말 | 어린이, 또래 |
| 나이가 어떻게 되세요? | 존댓말/존경 | 성인, 나이 많은 분 |
| 연세가 어떻게 되세요? | 매우 존경 | 어르신 |

```
A: 나이가 어떻게 되세요?     나이가 어떻게 되세요?
B: 스물다섯 살이에요.         스물다섯 살이에요.
```

## 8. 금액

|------|--------|-----------|
| ₩100 | 백 원 | baek won |
| ₩500 | 오백 원 | obaek won |
| ₩1,000 | 천 원 | cheon won |
| ₩5,000 | 오천 원 | ocheon won |
| ₩10,000 | 만 원 | man won |
| ₩50,000 | 오만 원 | oman won |
| ₩100,000 | 십만 원 | simman won |
| ₩1,000,000 | 백만 원 | baengman won |

### 8.2 쇼핑 표현

| 한국어 | 해석 |
|--------|------|
| 이거 얼마예요? | 이거 얼마예요? |
| 삼천 원이에요. | 삼천 원이에요. |
| 너무 비싸요. | 너무 비싸요. |

| 영수증 주세요. | 영수증 주세요. |

> **한국 화폐 간편 참고표**
>
> - **동전:** ₩10, ₩50, ₩100, ₩500
> - **지폐:** ₩1,000 (천 원), ₩5,000 (오천 원), ₩10,000 (만 원), ₩50,000 (오만 원)
>
> 대략적 환율 (참고용): ₩1,000 ≈ $0.75 USD
>
> - 커피 한 잔: 4,000-6,000원 (사천~육천 원)
> - 식사 한 끼: 8,000-15,000원 (팔천~만오천 원)
> - 지하철 요금: 1,400원 (천사백 원)
