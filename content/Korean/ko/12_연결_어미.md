# 12. 연결 어미 (Connective Endings)

## 학습 목표

- 이유, 대조, 조건, 목적, 나열의 주요 연결 어미를 습득한다
- 유사한 연결 어미 간의 차이를 구별한다 (예: -(으)니까 vs. -아/어서)
- 각 어미의 문법적 제약을 이해한다 (예: 명령문과 함께 쓸 수 있는 어미)
- 적절한 연결 어미를 사용하여 절을 자연스럽게 연결한다

## 1. 이유와 원인

한국어에는 세 가지 주요 이유/원인 연결어미가 있습니다. 올바른 것을 선택하는 것은 TOPIK에서 가장 많이 시험되는 항목 중 하나입니다.

### 1.1 -(으)니까 — 주관적 이유

**형성**: 동사 어간 + -(으)니까

| 어간 끝 | 결합 | 예시 |
|---|---|---|
| 모음 또는 ㄹ | -니까 | 가다 → 가니까, 살다 → 사니까 |
| 자음 | -으니까 | 먹다 → 먹으니까, 읽다 → 읽으니까 |
| 과거 시제 | -았/었으니까 | 갔으니까, 먹었으니까 |

**주요 특징**:
- 화자의 **주관적 이유나 발견**을 표현
- 명령형 (-세요), 청유형 (-ㅂ시다/자), 요청문 뒤에 **올 수 있음**
- 조언, 명령, 제안을 할 때 자주 사용

**예문**:
- 비가 오**니까** 우산을 가져가세요. (비가 오니까 우산을 가져가세요.)
- 시간이 없**으니까** 빨리 갑시다. (시간이 없으니까 빨리 갑시다.)
- 배고프**니까** 밥 먹자. (배고프니까 밥 먹자.)
- 직접 가 보**니까** 정말 좋았어요. (직접 가 보니까 정말 좋았어요.)

### 1.2 -아/어서 — 객관적/자연적 원인

**형성**: 동사 어간 + -아/어서 (모음 조화에 따라)

| 마지막 모음 | 결합 | 예시 |
|---|---|---|
| ㅏ 또는 ㅗ | -아서 | 만나다 → 만나서, 오다 → 와서 |
| 그 외 | -어서 | 먹다 → 먹어서, 읽다 → 읽어서 |
| 하다 | -해서 | 공부하다 → 공부해서 |

**주요 특징**:
- **자연적, 객관적, 사실적 인과 관계**를 표현
- 명령형이나 청유형 뒤에 **올 수 없음**
- -아/어서 절에 과거 시제를 붙일 수 없음 (~~갔어서~~는 틀림 — 가서를 사용)

**예문**:
- 배가 아파**서** 병원에 갔어요. (배가 아파서 병원에 갔어요.)
- 만나**서** 반갑습니다. (만나서 반갑습니다.) — *직역: "만나서 기쁩니다."*
- 늦어**서** 죄송합니다. (늦어서 죄송합니다.)
- 비가 와**서** 못 갔어요. (비가 와서 못 갔어요.)

> **흔한 실수**: ~~배가 아파서 약을 드세요.~~ → 이것은 틀린 표현입니다. -아/어서 뒤에 명령형이 올 수 없기 때문입니다. 올바른 표현: 배가 아프**니까** 약을 드세요.

### 1.3 -기 때문에 — 격식적/서면적 이유

**형성**: 동사 어간 + -기 때문에

**주요 특징**:
- -(으)니까나 -아/어서보다 더 격식적이고 명시적
- 글, 뉴스, 학술 문맥에서 흔히 사용
- 과거 시제 가능: -았/었기 때문에

**예문**:
- 시간이 없**기 때문에** 참석할 수 없습니다. (시간이 없기 때문에 참석할 수 없습니다.)
- 한국어를 공부했**기 때문에** 한국 드라마를 이해할 수 있어요. (한국어를 공부했기 때문에 한국 드라마를 이해할 수 있어요.)
- 교통사고가 났**기 때문에** 길이 막혔습니다. (교통사고가 났기 때문에 길이 막혔습니다.)

### ⚠️ 핵심 비교: -(으)니까 vs. -아/어서

이 구분은 중급 한국어에서 가장 중요한 문법 포인트 중 하나입니다.

| 특징 | -(으)니까 | -아/어서 |
|---|---|---|
| 뉘앙스 | 주관적 이유, 발견 | 자연적/객관적 원인 |
| + 명령형 (명령) | ✅ 가능 | ❌ 불가 |
| + 청유형 (제안) | ✅ 가능 | ❌ 불가 |
| 절 내 과거 시제 | ✅ -았/었으니까 | ❌ 과거 시제 불가 |
| 격식 | 중립 | 중립 |
| 전형적 사용 | 조언, 명령, 정당화 | 사실 설명, 사과 |

**비교 예문**:

| 상황 | 올바른 표현 | 틀린 표현 |
|---|---|---|
| 조언하기 | 추우**니까** 코트를 입으세요. | ~~추워서 코트를 입으세요.~~ |
| 제안하기 | 날씨가 좋**으니까** 산책합시다. | ~~날씨가 좋아서 산책합시다.~~ |
| 사실 설명 | 비가 와**서** 못 갔어요. | (비가 오니까 못 갔어요도 가능) |
| 사과하기 | 늦어**서** 죄송합니다. | (늦었으니까 죄송합니다 — 덜 자연스러움) |

## 2. 대조와 양보

### 2.1 -지만 — 단순 대조 ("~지만")

**형성**: 동사 어간 + -지만 (모음 조화 불필요, 직접 결합)

**주요 특징**:
- 가장 직접적인 "~지만" 연결어미
- 모든 동사/형용사, 모든 시제에 결합 가능
- 중립적 어감

**예문**:
- 비싸**지만** 좋아요. (비싸지만 좋아요.)
- 한국어가 어렵**지만** 재미있어요. (한국어가 어렵지만 재미있어요.)
- 어제 공부했**지만** 시험을 못 봤어요. (어제 공부했지만 시험을 못 봤어요.)
- 먹었**지만** 아직 배고파요. (먹었지만 아직 배고파요.)

### 2.2 -(으)ㄴ데/는데 — 배경/완화된 대조

**형성**:

| 유형 | 현재 | 과거 |
|---|---|---|
| 동작 동사 | -는데 | -(으)ㄴ데 → -았/었는데 |
| 서술용언 (형용사) | -(으)ㄴ데 | -았/었는데 |
| 명사 + 이다 | -인데 | -이었는데/였는데 |

**주요 특징**:
- 진술, 질문, 요청 전에 **배경 정보**를 제공
- -지만보다 부드러운 대조
- 구어 한국어에서 매우 다양하고 흔하게 사용
- 장면을 설정하거나 질문 전에 주제를 도입할 때 자주 사용

**예문**:
- 날씨가 좋**은데** 산책할까요? (날씨가 좋은데 산책할까요?)
- 시간이 있**는데** 영화 볼래요? (시간이 있는데 영화 볼래요?)
- 한국어를 공부하**는데** 너무 어려워요. (한국어를 공부하는데 너무 어려워요.)
- 배가 고프**ㄴ데** 뭐 먹을까요? (배가 고픈데 뭐 먹을까요?)
- 학생**인데** 할인 받을 수 있어요? (학생인데 할인 받을 수 있어요?)

### 2.3 -아/어도 — 양보 ("~해도/~하더라도")

**형성**: 동사 어간 + -아/어도 (모음 조화에 따라)

**주요 특징**:
- "~해도" 또는 "~하더라도"를 표현
- 조건에 관계없이 결과가 발생
- 과거 시제 가능: -았/었어도

**예문**:
- 비가 와**도** 갈 거예요. (비가 와도 갈 거예요.)
- 많이 먹어**도** 안 살쪄요. (많이 먹어도 안 살쪄요.)
- 바빠**도** 운동해야 돼요. (바빠도 운동해야 돼요.)
- 아무리 공부해**도** 어려워요. (아무리 공부해도 어려워요.)

> **패턴**: 아무리 + V-아/어도 = "아무리 ~해도..."

## 3. 조건

### 3.1 -(으)면 — 일반 조건 ("~면/~하면")

**형성**: 동사 어간 + -(으)면

| 어간 끝 | 결합 | 예시 |
|---|---|---|
| 모음 또는 ㄹ | -면 | 가다 → 가면, 살다 → 살면 |
| 자음 | -으면 | 먹다 → 먹으면, 읽다 → 읽으면 |

**예문**:
- 시간이 있**으면** 같이 가요. (시간이 있으면 같이 가요.)
- 봄이 **오면** 꽃이 펴요. (봄이 오면 꽃이 펴요.)
- 열심히 공부하**면** 합격할 수 있어요. (열심히 공부하면 합격할 수 있어요.)
- 모르**면** 물어보세요. (모르면 물어보세요.)

### 3.2 -다면 — 가정 조건 ("~라면/~한다면")

**형성**: 동사 어간 + -다면 (또는 동작 동사 -ㄴ다면, 명사 -라면)

**주요 특징**:
- -(으)면보다 더 가정적이거나 반사실적
- 상상의 상황이나 격식적 조건에 자주 사용

**예문**:
- 백만 원이 있**다면** 뭘 할 거예요? (백만 원이 있다면 뭘 할 거예요?)
- 시간을 되돌릴 수 있**다면** 뭘 바꾸고 싶어요? (시간을 되돌릴 수 있다면 뭘 바꾸고 싶어요?)
- 내가 새**라면** 하늘을 날 거예요. (내가 새라면 하늘을 날 거예요.)
- 그게 사실이**라면** 큰일이에요. (그게 사실이라면 큰일이에요.)

### 3.3 -(으)려면 — 조건적 목적 ("~하려면")

**형성**: 동사 어간 + -(으)려면

**주요 특징**:
- 의도 (-(으)려고)와 조건 (-(으)면)의 결합
- "~하고 싶으면 / ~하려면..."
- 흔히 조언이나 필요한 행동이 뒤따름

**예문**:
- 한국어를 잘하**려면** 매일 연습하세요. (한국어를 잘하려면 매일 연습하세요.)
- 건강하**려면** 운동을 해야 돼요. (건강하려면 운동을 해야 돼요.)
- 의사가 되**려면** 오래 공부해야 해요. (의사가 되려면 오래 공부해야 해요.)
- 여기 오**려면** 몇 시간 걸려요? (여기 오려면 몇 시간 걸려요?)

## 4. 목적

### 4.1 -(으)러 — 하러 가다/오다

**형성**: 동사 어간 + -(으)러

**주요 특징**:
- 이동 동사**만** 함께 사용: 가다, 오다, 다니다 등
- 주절의 동사가 반드시 이동을 나타내야 함

**예문**:
- 밥 먹**으러** 가요. (밥 먹으러 가요.)
- 친구를 만나**러** 왔어요. (친구를 만나러 왔어요.)
- 운동하**러** 헬스장에 가요. (운동하러 헬스장에 가요.)
- 한국어를 배우**러** 한국에 왔어요. (한국어를 배우러 한국에 왔어요.)

> **제약**: ~~밥 먹으러 공부해요~~는 공부하다가 이동 동사가 아니므로 틀린 표현입니다.

### 4.2 -(으)려고 — 의도

**형성**: 동사 어간 + -(으)려고

**주요 특징**:
- 의도나 계획을 표현
- 뒤따르는 동사에 제약 없음
- 명령형이나 청유형 뒤에 올 수 없음 (그 경우 -(으)려면 사용)

**예문**:
- 한국어를 배우**려고** 한국에 왔어요. (한국어를 배우려고 한국에 왔어요.)
- 돈을 모으**려고** 아르바이트를 해요. (돈을 모으려고 아르바이트를 해요.)
- 일찍 출발하**려고** 알람을 맞췄어요. (일찍 출발하려고 알람을 맞췄어요.)

### 4.3 -기 위해(서) — ~하기 위하여 (격식)

**형성**: 동사 어간 + -기 위해(서)

**주요 특징**:
- -(으)려고보다 더 격식적이고 의도적
- 서면 한국어, 연설, 격식적 문맥에서 흔히 사용
- 두 절의 주어가 같아야 함

**예문**:
- 건강을 **위해서** 운동해요. (건강을 위해서 운동해요.) — *명사 + 위해서*
- 합격하**기 위해** 열심히 공부했어요. (합격하기 위해 열심히 공부했어요.)
- 환경을 보호하**기 위해서** 분리수거를 해야 합니다. (환경을 보호하기 위해서 분리수거를 해야 합니다.)

### 목적 어미 비교

| 어미 | 제약 | 어감 | 예시 |
|---|---|---|---|
| -(으)러 | 이동 동사만 | 일상/중립 | 밥 먹으러 가요. |
| -(으)려고 | 명령형 불가 | 중립 | 밥 먹으려고 식당에 갔어요. |
| -기 위해(서) | 같은 주어 | 격식 | 건강을 위해 운동합니다. |

## 5. 나열과 부가

### 5.1 -고 — "~하고" / "~하고 나서"

**형성**: 동사 어간 + -고

**주요 특징**:
- 동작이나 상태를 나열하거나 순서를 나타냄
- 시제는 마지막 동사에만 표시
- 한국어에서 가장 기본적인 연결어미

**예문 (나열)**:
- 그 사람은 키가 크**고** 잘생겼어요. (그 사람은 키가 크고 잘생겼어요.)
- 커피도 마시**고** 케이크도 먹었어요. (커피도 마시고 케이크도 먹었어요.)

**예문 (순서)**:
- 손을 씻**고** 밥을 먹었어요. (손을 씻고 밥을 먹었어요.)
- 숙제를 하**고** 잤어요. (숙제를 하고 잤어요.)

### 5.2 -아/어서 — "~하고 나서" (인과적 순서)

**형성**: 동사 어간 + -아/어서

**주요 특징**:
- 순서에 사용할 때 (이유가 아닌), 첫 번째 동작이 두 번째의 **전제 조건**임을 함의
- 시간적으로 분리 불가 — 동작이 밀접하게 연결됨

**예문**:
- 백화점에 가**서** 옷을 샀어요. (백화점에 가서 옷을 샀어요.)
- 택시를 타**서** 갔어요. (택시를 타서 갔어요.)
- 앉아**서** 기다리세요. (앉아서 기다리세요.)

> **-고 vs. -아/어서 순서 비교**:
> - 집에 가**고** 공부했어요. = 집에 가고, (따로) 공부했어요.
> - 집에 가**서** 공부했어요. = 집에 가서 (거기서) 공부했어요. (인과적 연결)

### 5.3 -(으)며 — "~하며" / "~하면서" (격식)

**형성**: 동사 어간 + -(으)며

**주요 특징**:
- -고 (나열)나 -(으)면서 (동시)의 격식/문어체 대응
- 서면 한국어, 뉴스, 격식 발표에서 흔히 사용

**예문**:
- 그는 학생이**며** 운동선수입니다. (그는 학생이며 운동선수입니다.)
- 이 제품은 가볍**으며** 튼튼합니다. (이 제품은 가볍으며 튼튼합니다.)

---

## 6. 동시 동작

### -(으)면서 — ~하면서

**형성**: 동사 어간 + -(으)면서

| 어간 끝 | 결합 | 예시 |
|---|---|---|
| 모음 또는 ㄹ | -면서 | 가다 → 가면서, 만들다 → 만들면서 |
| 자음 | -으면서 | 먹다 → 먹으면서, 읽다 → 읽으면서 |

**주요 특징**:
- **같은 주어**의 두 동작이 동시에 일어남
- 각 절의 주어가 다를 수 없음

**예문**:
- 음악을 들**으면서** 공부해요. (음악을 들으면서 공부해요.)
- 커피를 마시**면서** 이야기했어요. (커피를 마시면서 이야기했어요.)
- 걸**으면서** 전화하지 마세요. (걸으면서 전화하지 마세요.)
- TV를 보**면서** 밥을 먹어요. (TV를 보면서 밥을 먹어요.)

### -(으)면서의 대조 용법

-(으)면서는 모순을 나타낼 수도 있습니다 (영어의 "~하면서"와 유사):

- 돈이 있**으면서** 안 쓰는 사람이에요. (돈이 있으면서 안 쓰는 사람이에요.)
- 알**면서** 왜 안 알려 줬어요? (알면서 왜 안 알려 줬어요?)

---

## 7. 연결 어미 요약표

| 기능 | 어미 | 핵심 특징 | 예시 |
|---|---|---|---|
| **이유** | -(으)니까 | + 명령형 가능 | 추우니까 코트 입으세요. |
| | -아/어서 | 객관적 원인, 명령형 불가 | 늦어서 죄송합니다. |
| | -기 때문에 | 격식/서면 | 시간이 없기 때문에... |
| **대조** | -지만 | 단순 "~지만" | 비싸지만 좋아요. |
| | -(으)ㄴ데/는데 | 배경 + 완화된 대조 | 날씨가 좋은데 나갈까요? |
| | -아/어도 | "~해도/~하더라도" | 비가 와도 갈 거예요. |
| **조건** | -(으)면 | 일반 "~면/~하면" | 시간이 있으면 가요. |
| | -다면 | 가정 "~라면" | 새라면 날 거예요. |
| | -(으)려면 | "~하려면" | 잘하려면 연습하세요. |
| **목적** | -(으)러 | + 이동 동사만 | 먹으러 가요. |
| | -(으)려고 | 의도 | 배우려고 왔어요. |
| | -기 위해(서) | 격식적 목적 | 합격하기 위해 공부했어요. |
| **나열** | -고 | "~하고 / ~하고 나서" | 씻고 잤어요. |
| | -아/어서 | 전제적 순서 | 가서 샀어요. |
| | -(으)며 | 격식 "~하며" | 가볍으며 튼튼합니다. |
| **동시** | -(으)면서 | 같은 주어, 동시 | 들으면서 공부해요. |

## 8. 연습 문제

### 연습 1: -(으)니까 vs. -아/어서

올바른 어미를 선택하세요. 일부 문장은 하나만 정답일 수 있습니다.

1. 비가 _______ 우산을 가져가세요. (오다)
2. 만나_______ 반갑습니다. (만나다)
3. 늦_______ 미안해요. (늦다)
4. 날씨가 좋_______ 산책합시다. (좋다)
5. 너무 피곤해_______ 일찍 잤어요. (피곤하다)

<details>
<summary>정답</summary>

1. **오니까** (명령형이 뒤따르므로 → -(으)니까 사용 필수)
2. **서** → 만나서 (고정 표현: 만나서 반갑습니다)
3. **어서** → 늦어서 (사과, 자연적 원인 → -아/어서)
4. **으니까** (청유형이 뒤따르므로 → -(으)니까 사용 필수)
5. **서** → 피곤해서 (사실적 인과 → -아/어서가 자연스러움; 피곤하니까도 문법적으로 가능)

</details>

### 연습 2: 올바른 연결어미 선택

빈칸에 가장 적절한 연결 어미를 채우세요.

1. 한국어가 어렵_______ 재미있어요. (~지만)
2. 돈을 모으_______ 아르바이트를 해요. (~하려고)
3. 음악을 들_______ 공부해요. (~하면서)
4. 밥 먹_______ 식당에 가요. (~하러)
5. 비가 와_______ 갈 거예요. (~해도)
6. 시간이 있_______ 같이 가요. (~하면)

<details>
<summary>정답</summary>

1. **지만** (단순 대조)
2. **으려고** (의도: -(으)려고)
3. **으면서** (동시: -(으)면서)
4. **으러** (목적 + 이동: -(으)러)
5. **도** (양보: -아/어도)
6. **으면** (조건: -(으)면)

</details>

### 연습 3: 문장 결합

각 쌍을 괄호 안에 표시된 연결 어미를 사용하여 결합하세요.

1. 배가 아프다 + 병원에 갔어요. (-아/어서)
2. 날씨가 좋다 + 산책할까요? (-(으)ㄴ데)
3. 돈이 없다 + 여행을 못 가요. (-기 때문에)
4. 손을 씻다 + 밥을 먹어요. (-고)
5. 한국에 오다 + 한국어를 배우다. (-(으)려고)

<details>
<summary>정답</summary>

1. 배가 **아파서** 병원에 갔어요. (배가 아파서 병원에 갔어요.)
2. 날씨가 **좋은데** 산책할까요? (날씨가 좋은데 산책할까요?)
3. 돈이 **없기 때문에** 여행을 못 가요. (돈이 없기 때문에 여행을 못 가요.)
4. 손을 **씻고** 밥을 먹어요. (손을 씻고 밥을 먹어요.)
5. 한국어를 **배우려고** 한국에 왔어요. (한국어를 배우려고 한국에 왔어요.)

</details>

### 연습 4: 오류 수정

각 문장에서 오류를 찾아 수정하세요.

1. 비가 와서 우산을 가져가세요.
2. 어제 늦었어서 미안합니다.
3. 커피를 마시면서 갔어요. (주어가 다른 경우: 나 / 그)
4. 영화를 보러 공부해요.

<details>
<summary>정답</summary>

1. 비가 **오니까** 우산을 가져가세요. (-아/어서는 명령형 앞에 올 수 없음)
2. 어제 **늦어서** 미안합니다. (-아/어서 절에 과거 시제를 붙일 수 없음)
3. 커피를 **마시고** 갔어요. (-(으)면서는 같은 주어가 필요; -고는 다른 주어나 순서 표현 가능)
4. 영화를 **보러** 극장에 **가요**. (-(으)러는 이동 동사가 필요; 공부하다는 이동 동사가 아님)

</details>

### 연습 5: 번역

지정된 어미를 사용하여 한국어로 번역하세요.

1. 한국어를 잘하고 싶으면 매일 연습해야 돼요. (-(으)려면)
2. 한국어를 공부하러 한국에 왔어요. (-(으)러)
3. 비쌌지만 샀어요. (-아/어도)
4. 저는 학생인데 할인 있어요? (-(으)ㄴ데)
5. 아무리 공부해도 어려워요. (아무리 + -아/어도)

<details>
<summary>정답</summary>

1. 한국어를 잘하**려면** 매일 연습해야 돼요.
2. 한국어를 공부하**러** 한국에 왔어요.
3. 비싸**도** 샀어요.
4. 학생**인데** 할인 있어요?
5. 아무리 공부해**도** 어려워요.

</details>
