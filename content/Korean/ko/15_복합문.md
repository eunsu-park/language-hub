## 1. 관형사형 — 한국어 관계절

영어에서 관계절은 명사 **뒤에** 옵니다: "the book **that I read**." 한국어에서는 수식절이 명사 **앞에** 옵니다: "**내가 읽은** 책." 이것은 영어와 한국어 사이의 가장 중요한 구조적 차이 중 하나입니다.

### 1.1 현재 시제 관형사형

형태는 **동작 동사**인지 **서술용언(형용사)**인지에 따라 다릅니다.

#### 동작 동사: -는

| 동사 | + -는 + 명사 | 의미 |
|---|---|---|
| 읽다 | **읽는** 책 | 읽고 있는 책 |
| 먹다 | **먹는** 음식 | 먹고 있는 음식 |
| 살다 | **사는** 곳 | 사는 곳 |
| 일하다 | **일하는** 사람 | 일하는 사람 |
| 가다 | **가는** 길 | 가는 길 |

> **ㄹ 어간 참고**: -는 앞에서 ㄹ 탈락: 살다 → 사는, 만들다 → 만드는, 알다 → 아는

**예문**:
- 지금 **읽는** 책이 뭐예요? (지금 읽는 책이 뭐예요?)
- 저기에서 **일하는** 사람이 제 친구예요. (저기에서 일하는 사람이 제 친구예요.)
- 서울에 **사는** 사람이 많아요. (서울에 사는 사람이 많아요.)

#### 서술용언 (형용사): -(으)ㄴ

| 동사 | + -(으)ㄴ + 명사 | 의미 |
|---|---|---|
| 예쁘다 | **예쁜** 꽃 | 예쁜 꽃 |
| 크다 | **큰** 집 | 큰 집 |
| 작다 | **작은** 가방 | 작은 가방 |
| 좋다 | **좋은** 날씨 | 좋은 날씨 |
| 재미있다 | **재미있는** 영화 | 재미있는 영화 |

> **참고**: 있다/없다는 서술용언이지만 **-는**을 취합니다: 재미있**는**, 맛없**는**

**예문**:
- **예쁜** 꽃을 샀어요. (예쁜 꽃을 샀어요.)
- **큰** 도시에 살고 싶어요. (큰 도시에 살고 싶어요.)
- **재미있는** 이야기를 해 주세요. (재미있는 이야기를 해 주세요.)

### 1.2 과거 시제 관형사형: -(으)ㄴ

**동작 동사**에서 -(으)ㄴ은 완료된 행위를 나타냅니다.

| 동사 | + -(으)ㄴ + 명사 | 의미 |
|---|---|---|
| 읽다 | **읽은** 책 | 읽은 책 |
| 먹다 | **먹은** 음식 | 먹은 음식 |
| 만나다 | **만난** 사람 | 만난 사람 |
| 가다 | **간** 곳 | 간 곳 |
| 만들다 | **만든** 케이크 | 만든 케이크 |

**예문**:
- 어제 **읽은** 책이 재미있었어요. (어제 읽은 책이 재미있었어요.)
- 지난주에 **만난** 사람이 기억나요? (지난주에 만난 사람이 기억나요?)
- 엄마가 **만든** 음식이 제일 맛있어요. (엄마가 만든 음식이 제일 맛있어요.)

### 1.3 미래 시제 관형사형: -(으)ㄹ

**앞으로 일어날** 일이나 **의도/예정**을 나타냅니다.

| 동사 | + -(으)ㄹ + 명사 | 의미 |
|---|---|---|
| 읽다 | **읽을** 책 | 읽을 책 |
| 먹다 | **먹을** 것 | 먹을 것 |
| 가다 | **갈** 곳 | 갈 곳 |
| 하다 | **할** 일 | 할 일 |
| 살다 | **살** 집 | 살 집 |

**예문**:
- **읽을** 책이 많아요. (읽을 책이 많아요.)
- **먹을** 것이 없어요. (먹을 것이 없어요.)
- 내일 **할** 일이 많아요. (내일 할 일이 많아요.)
- **살** 집을 찾고 있어요. (살 집을 찾고 있어요.)

### 1.4 과거 습관/미완료: -던

**과거에 반복적으로 일어났던** 행위나 **진행 중이었으나 완료되지 않은** 상태를 나타냅니다.

| 동사 | + -던 + 명사 | 의미 |
|---|---|---|
| 살다 | **살던** 곳 | 살던 곳 |
| 다니다 | **다니던** 학교 | 다니던 학교 |
| 좋아하다 | **좋아하던** 노래 | 좋아하던 노래 |
| 읽다 | **읽던** 책 | 읽던 책 (미완료) |
| 먹다 | **먹던** 음식 | 먹던 / 먹곤 하던 음식 |

**예문**:
- 어렸을 때 **살던** 곳이 그리워요. (어렸을 때 살던 곳이 그리워요.)
- 대학교 때 **다니던** 카페가 아직 있어요. (대학교 때 다니던 카페가 아직 있어요.)
- **읽던** 책을 다시 펼쳤어요. (읽던 책을 다시 펼쳤어요.)

### -(으)ㄴ vs. -던 (과거)

| 형태 | 뉘앙스 | 예시 |
|---|---|---|
| -(으)ㄴ | 완료된 사실 | **간** 곳 = 간 곳 (완료된 방문) |
| -던 | 습관적/미완료 | **가던** 곳 = 가곤 하던 / 가고 있던 곳 |

- **만난** 사람 = 만난 사람 (구체적 완료된 만남)
- **만나던** 사람 = 만나곤 하던 / 만나고 있던 사람 (지속적 또는 습관적)

### 1.5 관형사형 요약표

| | 동작 동사 | 서술용언 |
|---|---|---|
| **현재** | -는 (먹는 음식) | -(으)ㄴ (맛있는 음식*) |
| **과거** | -(으)ㄴ (먹은 음식) | — |
| **미래** | -(으)ㄹ (먹을 음식) | -(으)ㄹ (맛있을 음식) |
| **과거 습관** | -던 (먹던 음식) | -던 (맛있던 음식) |

> *있다/없다는 특수: 상태형이지만 현재형에서 -는을 취합니다.

## 2. 명사화

명사화는 동사나 절을 명사로 만드는 것입니다. 한국어에는 세 가지 주요 명사화 방법이 있습니다.

### 2.1 -기 명사화

**형성**: 동사 어간 + -기

추상적 명사구를 만들 때 사용합니다. 특정 고정 패턴에서 흔합니다.

| 동사 | 명사화 | 의미 |
|---|---|---|
| 수영하다 | **수영하기** | 수영 |
| 읽다 | **읽기** | 읽기 |
| 쓰다 | **쓰기** | 쓰기 |
| 듣다 | **듣기** | 듣기 |
| 말하다 | **말하기** | 말하기 |

**-기와 함께 쓰이는 주요 패턴**:

| 패턴 | 의미 | 예문 |
|---|---|---|
| -기 쉽다 | ~하기 쉽다 | 한국어는 배우**기** 쉬워요. (한국어는 배우기 쉬워요.) |
| -기 어렵다 | ~하기 어렵다 | 발음하**기** 어려워요. (발음하기 어려워요.) |
| -기 시작하다 | ~하기 시작하다 | 비가 오**기** 시작했어요. (비가 오기 시작했어요.) |
| -기 전에 | ~하기 전에 | 먹**기** 전에 손을 씻으세요. (먹기 전에 손을 씻으세요.) |
| -기로 하다 | ~하기로 결정하다 | 한국에 가**기로** 했어요. (한국에 가기로 했어요.) |
| -기 바라다 | ~하기를 바라다 | 건강하**기** 바랍니다. (건강하기 바랍니다.) |
| -기 때문에 | ~이기 때문에 | 바쁘**기** 때문에 못 가요. (바쁘기 때문에 못 가요.) |
| -기 위해(서) | ~하기 위하여 | 합격하**기** 위해 공부해요. (합격하기 위해 공부해요.) |

### 2.2 -(으)ㅁ 명사화

**형성**: 동사 어간 + -(으)ㅁ

-기보다 더 격식적이고 문어체입니다. 서면 공지, 공식 문서, 격식 표현에서 흔합니다.

| 동사 | 명사화 | 의미 |
|---|---|---|
| 있다 | **있음** | 있음 |
| 없다 | **없음** | 없음 |
| 알다 | **앎** | 앎 |
| 다르다 | **다름** | 다름 |
| 감사하다 | **감사함** | 감사함 |

**주요 용법**:

| 문맥 | 예시 | 의미 |
|---|---|---|
| 서면 공지 | 주차 금지**임** | 주차 금지임 |
| 서식/문서 | 이상 **없음** | 이상 없음 |
| 서면 종결 | 위와 같**음** | 위와 같음 |
| 격식 표현 | 다름이 아니라... | 다름이 아니라... (격식 서두) |

### 2.3 -는 것 명사화

**형성**: 관형사형 + 것 (것)

가장 다양하고 흔히 사용되는 명사화입니다. 거의 모든 문맥에서 사용 가능합니다.

| 시제 | 형성 | 예시 | 의미 |
|---|---|---|---|
| 현재 (동작) | V-는 것 | **먹는 것** | 먹는 것 |
| 현재 (서술) | A-(으)ㄴ 것 | **좋은 것** | 좋은 것 |
| 과거 | V-(으)ㄴ 것 | **먹은 것** | 먹은 것 |
| 미래 | V-(으)ㄹ 것 | **먹을 것** | 먹을 것 |

> **구어 축약**: 것 → 거 (비격식)
> - 먹는 것 → 먹는 **거**, 좋은 것 → 좋은 **거**

**예문**:
- **먹는 것**을 좋아해요. (먹는 것을 좋아해요.)
- 한국어를 **배우는 것**이 재미있어요. (한국어를 배우는 것이 재미있어요.)
- 어제 **한 것**을 말해 주세요. (어제 한 것을 말해 주세요.)
- **읽은 것**을 이해했어요? (읽은 것을 이해했어요?)
- 내일 **할 것**이 많아요. (내일 할 것이 많아요.)

### -기 vs. -는 것

| 특징 | -기 | -는 것 |
|---|---|---|
| 격식도 | 약간 격식적 | 범용적 (구어 + 문어) |
| 주어로 사용 | ✅ (덜 흔함) | ✅ (매우 흔함) |
| 목적어로 사용 | ⚠️ 제한된 패턴 | ✅ 광범위 |
| 고정 표현 | 많음 (-기 시작하다, -기 전에) | 많음 (-는 것 같다, -는 것이 좋다) |
| "~하는 것을 좋아하다" | 수영하기를 좋아해요 (괜찮지만 딱딱함) | 수영하는 것을 좋아해요 / 수영하는 걸 좋아해요 (자연스러움) |

## 3. 종결어미 패턴

이 어미들은 문장 끝에 나타나며 단순한 진술이나 질문을 넘어서는 뉘앙스, 태도, 대인 관계적 의미를 더합니다.

### 3.1 -거든요 — 이유 / 배경 제공

**형성**: 동사 어간 + -거든요

**의미**: "있잖아요..." / "그게요..." — 배경 설명이나 정당화를 제공합니다. 청자가 이유를 모를 때 자주 사용합니다.

**예문**:
- 오늘 못 만나요. 내일 시험이 있**거든요**. (오늘 못 만나요. 내일 시험이 있거든요.)
- 택시를 탔어요. 버스가 안 오**거든요**. (택시를 탔어요. 버스가 안 오거든요.)
- 한국 음식을 좋아해요. 매운 음식을 좋아하**거든요**. (한국 음식을 좋아해요. 매운 음식을 좋아하거든요.)

> **사용법**: -거든요는 보통 **두 번째** 문장에 나와 **첫 번째** 문장을 설명합니다. 암묵적인 "왜?"에 답합니다.

### 3.2 -잖아요 — 알려진 정보 환기

**형성**: 동사 어간 + -잖아요

**의미**: "알잖아요..." / "알다시피..." — 청자가 이미 알아야 할 것을 상기시킵니다. 약간 나무라는 뉘앙스가 있을 수 있습니다.

**예문**:
- 제가 말했**잖아요**. (제가 말했잖아요. / 말했잖아요!)
- 내일 시험이**잖아요**. 공부하세요. (내일 시험이잖아요. 공부하세요.)
- 저도 학생이**잖아요**. 돈이 없어요. (저도 학생이잖아요. 돈이 없어요.)
- 약속했**잖아요**. 왜 안 왔어요? (약속했잖아요. 왜 안 왔어요?)

### 3.3 -네요 — 가벼운 놀라움 / 깨달음

**형성**: 동사 어간 + -네요

**의미**: "아, 그렇군요..." / "와..." — 화자가 새로운 것을 발견했을 때의 깨달음이나 가벼운 놀라움을 표현합니다.

**예문**:
- 한국어를 잘하시**네요**! (한국어를 잘하시네요!)
- 날씨가 정말 좋**네요**. (날씨가 정말 좋네요.)
- 여기 사람이 많**네요**. (여기 사람이 많네요.)
- 벌써 12시**네요**. (벌써 12시네요.)
- 이 케이크 맛있**네요**. (이 케이크 맛있네요.)

### 3.4 -군요 / -구나 — 감탄적 발견

**형성**:
- 공손: V/A-군요, N-이군요
- 반말: V/A-구나, N-이구나

**의미**: "아! 그랬군요!" — -네요보다 더 강한 깨달음이나 발견. 새로운 정보를 알게 되었을 때 자주 사용합니다.

**예문**:
- 여기가 서울이**군요**! (여기가 서울이군요!)
- 한국어를 공부하는**군요**. (한국어를 공부하는군요.)
- 정말 맛있**군요**! (정말 맛있군요!)
- 그렇**구나**. (그렇구나.) — 반말
- 벌써 졸업했**구나**! (벌써 졸업했구나!) — 반말

### 3.5 -(으)ㄹ게요 — 약속 / 화자의 결정

**형성**: 동사 어간 + -(으)ㄹ게요

**의미**: "~할게요..." — 화자가 결정하거나 약속합니다. **1인칭** (나/우리)에만 사용합니다.

**예문**:
- 내일 전화**할게요**. (내일 전화할게요.)
- 제가 **할게요**. (제가 할게요.)
- 내일 일찍 **갈게요**. (내일 일찍 갈게요.)
- 다음에 꼭 **올게요**. (다음에 꼭 올게요.)
- 열심히 공부**할게요**. (열심히 공부할게요.)

> **-(으)ㄹ게요 vs. -(으)ㄹ 거예요**:
> - -(으)ㄹ게요: 지금 내린 약속/결정, 청자에게 향함
> - -(으)ㄹ 거예요: 기존 계획이나 예측, 누구에 대해서든 가능

### 3.6 -(으)ㄹ까요? — 제안 / 추측

**형성**: 동사 어간 + -(으)ㄹ까요?

**의미**: 두 가지 용법 — (1) 제안하기, (2) 추측하기.

**제안 (~할까요?)**:
- 같이 **갈까요**? (같이 갈까요?)
- 뭐 먹**을까요**? (뭐 먹을까요?)
- 창문을 열**까요**? (창문을 열까요?)

**추측 (~일까요?)**:
- 내일 비가 올**까요**? (내일 비가 올까요?)
- 그 사람이 올**까요**? (그 사람이 올까요?)
- 이게 맞을**까요**? (이게 맞을까요?)

### 종결어미 패턴 요약

| 패턴 | 기능 | 예시 |
|---|---|---|
| -거든요 | 배경 이유 | 내일 시험이 있거든요. |
| -잖아요 | 알려진 정보 환기 | 제가 말했잖아요. |
| -네요 | 가벼운 놀라움 | 한국어를 잘하시네요! |
| -군요/-구나 | 감탄적 발견 | 여기가 서울이군요! |
| -(으)ㄹ게요 | 약속/결정 (1인칭) | 내일 전화할게요. |
| -(으)ㄹ까요? | 제안 / 추측 | 같이 갈까요? |

## 4. 복합 패턴 결합

실제 한국어 문장은 종종 여러 문법 패턴을 겹겹이 쌓습니다. 이 과와 이전 과에서 배운 패턴들이 어떻게 결합되는지 보여주는 예시입니다.

### 패턴 겹쳐 쓰기 예시

**관형사형 + 명사화**:
- 어제 **읽은 것**이 기억나요? (어제 읽은 것이 기억나요?)
- 제가 **만든 것**을 드세요. (제가 만든 것을 드세요.)

**관형사형 + 종결어미**:
- 지금 **읽는** 책이 정말 재미있**네요**. (지금 읽는 책이 정말 재미있네요.)
- 어제 **만난** 사람이 민수**였군요**! (어제 만난 사람이 민수였군요!)

**연결어미 + 관형사형**:
- 한국에서 **사는** 것이 재미있**지만** 가끔 힘들어요. (한국에서 사는 것이 재미있지만 가끔 힘들어요.)
- **읽을** 책이 많**아서** 바빠요. (읽을 책이 많아서 바빠요.)

**복합 패턴 한 문장에**:
- 어제 **만난** 사람이 한국어를 정말 잘하**는** 사람이**었거든요**. (어제 만난 사람이 한국어를 정말 잘하는 사람이었거든요.)
- 내일 **할** 일이 많**으니까** 오늘 일찍 자**ㄹ게요**. (내일 할 일이 많으니까 오늘 일찍 잘게요.)
- 제가 **만든** 음식을 **드시는** 것을 보**니까** 기분이 좋**네요**. (제가 만든 음식을 드시는 것을 보니까 기분이 좋네요.)

### 복합문 단계별 구성

"친구가 추천한 식당에서 먹은 한국 음식이 정말 맛있었어요."를 구성해 봅시다.

1. 핵심부터: 한국 음식이 정말 맛있었어요. (한국 음식이 정말 맛있었어요.)
2. "먹은" 추가: **먹은** 한국 음식이 정말 맛있었어요.
3. "식당에서" 추가: 식당에서 **먹은** 한국 음식이 정말 맛있었어요.
4. "친구가 추천한" 추가: 친구가 **추천한** 식당에서 **먹은** 한국 음식이 정말 맛있었어요.

완성: **친구가 추천한 식당에서 먹은 한국 음식이 정말 맛있었어요.**

## 5. 연습 문제

### 연습 1: 관형사형

빈칸에 올바른 관형사형을 채우세요.

1. 지금 _______ 영화가 재미있어요. (보다, 현재)
2. 어제 _______ 책이 재미있었어요. (읽다, 과거)
3. 내일 _______ 일이 많아요. (하다, 미래)
4. _______ 날씨에 산책하고 싶어요. (좋다, 현재 서술)
5. 대학교 때 _______ 노래를 들었어요. (좋아하다, 과거 습관)

<details>
<summary>정답</summary>

1. 지금 **보는** 영화가 재미있어요. (현재 동작 동사 -는)
2. 어제 **읽은** 책이 재미있었어요. (과거 동작 동사 -(으)ㄴ)
3. 내일 **할** 일이 많아요. (미래 -(으)ㄹ)
4. **좋은** 날씨에 산책하고 싶어요. (현재 서술용언 -(으)ㄴ)
5. 대학교 때 **좋아하던** 노래를 들었어요. (과거 습관 -던)

</details>

### 연습 2: 명사화

가장 적절한 명사화 형태(-기, -(으)ㅁ, 또는 -는 것)를 선택하세요.

1. 한국어를 _______ 재미있어요. (배우다 + 명사화 — "한국어를 배우는 것이 재미있어요")
2. 이상 _______. (없다 — 서면 서식/문서에서)
3. _______ 전에 손을 씻으세요. (먹다 — "먹기 전에")
4. 한국 음식을 _______ 좋아해요. (먹다 — "한국 음식 먹는 것을 좋아해요")
5. 일찍 일어나_______ 힘들어요. ("일찍 일어나기가 힘들어요")

<details>
<summary>정답</summary>

1. 한국어를 **배우는 것이** / **배우기가** 재미있어요. (둘 다 가능; -는 것이 구어에서 더 자연스러움)
2. 이상 **없음**. (서면 공지 형식에서 -(으)ㅁ)
3. **먹기** 전에 손을 씻으세요. (-기 전에는 고정 패턴)
4. 한국 음식을 **먹는 것을** / **먹는 걸** 좋아해요. (-는 것이 좋아하다의 목적어로 자연스러움)
5. 일찍 일어나**기가** 힘들어요. (-기 + 힘들다/어렵다는 고정 패턴)

</details>

### 연습 3: 종결어미 패턴

다음 중에서 올바른 어미를 선택하세요: -거든요, -잖아요, -네요, -군요, -(으)ㄹ게요, -(으)ㄹ까요

1. 오늘 못 만나요. 내일 시험이 있_______. (이유 제공)
2. 한국어를 잘하시_______! (능력에 대한 놀라움)
3. 같이 _______? (함께 가자는 제안 — 가다)
4. 제가 말했_______. 왜 안 들었어요? (상기시키기)
5. 내일 꼭 전화_______. (약속)
6. 아, 여기가 유명한 식당이_______! (발견)

<details>
<summary>정답</summary>

1. 있**거든요** (배경 이유)
2. 잘하시**네요** (가벼운 놀라움/칭찬)
3. 같이 **갈까요**? (제안)
4. 말했**잖아요** (알려진 정보 환기)
5. 전화**할게요** (약속, 1인칭)
6. 식당이**군요** (감탄적 발견)

</details>

### 연습 4: 복합문 구성

주어진 요소를 하나의 한국어 문장으로 결합하세요.

1. [어제 온 사람] + [제 친구예요]
2. [먹고 싶은 것] + [비빔밥이에요]
3. [영화를 보는 것을] + [좋아해요]
4. [내일 할 일이 많으니까] + [오늘 일찍 잘게요]

<details>
<summary>정답</summary>

1. 어제 **온** 사람이 제 친구예요. (어제 온 사람이 제 친구예요.)
2. **먹고 싶은 것**은 비빔밥이에요. (먹고 싶은 것은 비빔밥이에요.)
3. 영화를 **보는 것을** 좋아해요. (영화를 보는 것을 좋아해요.)
4. 내일 **할** 일이 **많으니까** 오늘 일찍 **잘게요**. (내일 할 일이 많으니까 오늘 일찍 잘게요.)

</details>

### 연습 5: 번역

배운 패턴을 사용하여 한국어로 번역하세요.

1. 살던 곳은 부산이었어요.
2. 아침에 일찍 일어나기가 힘들어요.
3. 이 커피 정말 맛있네요!
4. 내일 휴일이잖아요. 쉽시다.
5. 오늘 못 가요. 아프거든요.

<details>
<summary>정답</summary>

1. 제가 **살던** 곳은 부산이었어요.
2. 아침에 일찍 일어나**기**가 힘들어요.
3. 이 커피 정말 맛있**네요**!
4. 내일 휴일이**잖아요**. 쉽시다. / 쉬어요.
5. 오늘 못 가요. 아프**거든요**.

</details>

### 연습 6: 관형사형 시제 구분

각 문맥에 맞는 올바른 관형사형을 선택하세요.

1. 지금 _______ 사람이 누구예요? (노래하다 — "지금 노래하는 사람")
2. 아까 _______ 사람이 누구였어요? (노래하다 — "아까 노래한 사람")
3. 다음에 _______ 사람은 김민수 씨입니다. (노래하다 — "다음에 노래할 사람")
4. 학교 다닐 때 항상 _______ 사람이 그리워요. (노래하다 — "항상 노래하던 사람")

<details>
<summary>정답</summary>

1. 지금 **노래하는** 사람 (-는 현재 동작)
2. 아까 **노래한** 사람 (-(으)ㄴ 과거 동작)
3. 다음에 **노래할** 사람 (-(으)ㄹ 미래)
4. 항상 **노래하던** 사람 (-던 과거 습관)

</details>

## 핵심 정리

1. **관형사형**은 한국어의 관계절에 해당합니다 — 수식어가 명사 **앞에** 옵니다
2. **-는** (동작, 현재), **-(으)ㄴ** (서술, 현재 / 동작, 과거), **-(으)ㄹ** (미래), **-던** (과거 습관)
3. **-는 것**은 일상 구어에서 가장 범용적인 명사화입니다
4. **-기**는 고정 패턴에서 선호됩니다 (-기 전에, -기 시작하다, -기 쉽다/어렵다)
5. **종결어미 패턴**은 중요한 뉘앙스를 더합니다 — -거든요 (설명), -잖아요 (환기), -네요 (놀라움), -군요 (발견) 사용법을 익히세요
