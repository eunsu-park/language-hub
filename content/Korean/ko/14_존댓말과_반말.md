## 1. 한국어 경어법 개요

한국어에는 **7가지 전통 경어법 단계**가 있지만, 현대 한국어에서는 **4가지만 흔히** 사용됩니다. 경어법은 모든 문장 끝의 **어미**에 반영됩니다.

### 7가지 경어법

| 단계 | 이름 | 격식 | 현대 사용 |
|---|---|---|---|
| 1 | **합쇼체** (하십시오체) | 격식 존대 | ✅ 매우 흔함 |
| 2 | **해요체** | 비격식 존대 | ✅ 가장 흔함 |
| 3 | **하오체** | 격식 중립 | ❌ 고어 (사극에서 사용) |
| 4 | **하게체** | 준격식 | ⚠️ 제한적 사용 |
| 5 | **해라체** | 격식 평서 | ⚠️ 글, 인용, 일기 |
| 6 | **해체** (반말) | 비격식 | ✅ 매우 흔함 |
| 7 | **해체** (친밀) | 친밀 | (해체에 통합) |

일상생활에서는 주로 **합쇼체, 해요체, 해체**가 필요합니다. 하게체는 특정 상황에서 나타납니다.

### 가다 (가다)로 빠른 비교

| 경어법 | 현재 | 과거 | 미래 |
|---|---|---|---|
| 합쇼체 (격식 존대) | 갑니다 | 갔습니다 | 갈 겁니다 |
| 해요체 (비격식 존대) | 가요 | 갔어요 | 갈 거예요 |
| 해체 (반말) | 가 | 갔어 | 갈 거야 |
| 하게체 (준격식) | 가네 | 갔네 | 갈 걸세 |

## 2. 합쇼체 — 격식 존대

가장 높은 수준의 경어법으로, 강한 존경과 격식을 나타냅니다.

### 형성

| 문장 유형 | 모음 어간 | 자음 어간 | 예시 |
|---|---|---|---|
| 평서문 | -ㅂ니다 | -습니다 | 갑니다 / 먹습니다 |
| 의문문 | -ㅂ니까? | -습니까? | 갑니까? / 먹습니까? |
| 청유문 | -ㅂ시다 | -읍시다 | 갑시다 / 먹읍시다 |
| 명령문 | -(으)십시오 | -(으)십시오 | 가십시오 / 드십시오 |

### 합쇼체 사용 상황

- **뉴스 방송 및 안내**: 다음 뉴스를 전해 드리겠습니다. (다음 뉴스를 전해 드리겠습니다.)
- **격식 발표**: 지금부터 발표를 시작하겠습니다. (지금부터 발표를 시작하겠습니다.)
- **군대 및 경찰**: 충성! 보고합니다! (충성! 보고합니다!)
- **어른과의 첫 만남**: 처음 뵙겠습니다. (처음 뵙겠습니다.)
- **고객 응대**: 어서 오십시오. (어서 오십시오.)
- **공식 연설**: 국민 여러분, 감사합니다. (국민 여러분, 감사합니다.)

### 예문

- 저는 김민수**입니다**. (저는 김민수입니다.)
- 한국어를 공부**합니다**. (한국어를 공부합니다.)
- 어디에서 오셨**습니까**? (어디에서 오셨습니까?)
- 같이 **갑시다**. (같이 갑시다.)
- 여기 앉으**십시오**. (여기 앉으십시오.)

## 3. 해요체 — 비격식 존대

대부분의 일상 대화에서 사용되는 **기본** 경어법입니다. 공손하지만 딱딱하지 않습니다.

### 형성

동사 어간 + -아/어요 (제11과의 모음 조화 규칙에 따름)

| 문장 유형 | 형성 | 예시 |
|---|---|---|
| 평서문 | -아/어요 | 가요 / 먹어요 |
| 의문문 | -아/어요? (올림 억양) | 가요? / 먹어요? |
| 청유문 | -아/어요 (또는 -(으)ㄹ까요?) | 가요 / 갈까요? |
| 명령문 | -아/어요 (또는 -(으)세요) | 가요 / 가세요 |

### 해요체 사용 상황

- **대부분의 일상 대화**: 지인, 동료, 낯선 사람과
- **쇼핑과 식당**: 이거 주세요. (이거 주세요.)
- **처음 만나는 동갑**: 안전한 기본 선택
- **시댁/처가**: 존중하면서도 따뜻하게
- **직장**: 동료 간 가장 흔히 사용

### 예문

- 오늘 날씨가 좋**아요**. (오늘 날씨가 좋아요.)
- 뭐 먹을 거**예요**? (뭐 먹을 거예요?)
- 저는 학생이**에요**. (저는 학생이에요.)
- 같이 가**요**. (같이 가요.)
- 천천히 말해 주**세요**. (천천히 말해 주세요.)

### 합쇼체 vs. 해요체

| 상황 | 합쇼체 | 해요체 |
|---|---|---|
| 취업 면접 | ✅ 선호 | ⚠️ 수용 가능 |
| 동료와 대화 | ❌ 너무 딱딱함 | ✅ 자연스러움 |
| 뉴스 방송 | ✅ 표준 | ❌ 너무 편함 |
| 친구 부모님과 대화 | ⚠️ 매우 격식적 | ✅ 자연스러움 |
| 결혼식 축사 | ✅ 기대됨 | ⚠️ 너무 편함 |

## 4. 해체 — 비격식 / 반말

친한 친구, 아이들, 또는 확실히 어린 사람에게 사용합니다. 잘못된 상대에게 반말을 쓰면 심각한 사회적 결례입니다.

### 형성

해요체에서 **요만 빼면** 됩니다.

| 해요체 | 해체 (반말) | 의미 |
|---|---|---|
| 가요 | 가 | 가다 |
| 먹어요 | 먹어 | 먹다 |
| 했어요 | 했어 | 했다 |
| 좋아요 | 좋아 | 좋다 |
| 갈 거예요 | 갈 거야 | 갈 것이다 |
| 뭐예요? | 뭐야? | 뭐이다? |
| -이에요/예요 | -이야/야 | ~이다 (서술격 조사) |

### 반말 사용 상황

- **같은 나이의 친한 친구** 사이 (또는 명시적 허락 후)
- **어린아이**에게 말할 때
- **확실히 어린** 사람에게 (주의 필요 — 나이 차이가 중요)
- **연인** 사이 (관계가 확립된 후)
- **자기 자신에게** 혼잣말할 때

### 예문

- 어디 **가**? (어디 가?)
- 밥 **먹었어**? (밥 먹었어?)
- 이거 진짜 맛있**어**! (이거 진짜 맛있어!)
- 내일 뭐 **할 거야**? (내일 뭐 할 거야?)
- 나 학생**이야**. (나 학생이야.)

### ⚠️ 반말 사용 규칙

| 상황 | 적절? |
|---|---|
| 같은 나이 친한 친구에게 | ✅ 네 |
| 같은 나이 낯선 사람에게 | ❌ 아니요 (해요체 사용) |
| 어린아이에게 | ✅ 네 |
| 어른에게 | ❌ 절대 안 됨 (상대가 허락하지 않는 한) |
| 반말 허락 후 | ✅ 네 |

## 5. 하게체 — 준격식

해요체와 반말 사이의 중간 단계입니다. 덜 흔하지만 여전히 만날 수 있습니다.

### 형성

| 문장 유형 | 어미 | 예시 |
|---|---|---|
| 평서문 | -네 / -(으)ㄹ세 | 가네 / 갈세 |
| 의문문 | -나? / -(으)ㄴ가? | 가나? / 먹었는가? |
| 명령문 | -게 | 가게 / 앉게 |
| 청유문 | -세 | 가세 / 앉으세 |

### 하게체 사용 상황

- **어른이 아는 젊은 성인에게** (예: 교수가 아는 학생에게)
- **장인이 사위에게** (전통적)
- **문학 및 학술 글** (예전 문체)
- **일부 방언** (특정 지역에서 더 흔함)

### 예문

- 자네, 요즘 어떻**나**? (요즘 어떻나?) — 어른이 젊은 성인에게
- 그래, 잘 먹었**네**. (그래, 잘 먹었네.)
- 어서 앉**게**. (어서 앉게.)
- 이리 와 보**게**. (이리 와 보게.)

> **학습자를 위한 안내**: 하게체를 적극적으로 사용할 필요는 없지만, 특히 나이 든 한국인이나 한국 드라마에서 들을 때 인식할 수 있어야 합니다.

## 6. 주체 존경 -(으)시-

**주체 존경** 접미사 -(으)시-는 문장의 **주어** (행위를 하는 사람)에 대한 존경을 나타내기 위해 동사에 삽입됩니다. 경어법 단계와는 독립적입니다.

### 형성

| 어간 끝 | 존경 접미사 | 예시 |
|---|---|---|
| 모음 | -시- | 가다 → 가**시**다 |
| 자음 | -으시- | 읽다 → 읽**으시**다 |
| ㄹ 끝 | -시- (ㄹ 탈락) | 살다 → 사**시**다 |

### 경어법과 결합

| 기본형 | 합쇼체 | 해요체 | 과거 |
|---|---|---|---|
| 가다 | 가십니다 | 가세요 | 가셨어요 |
| 먹다 | 드십니다 | 드세요 | 드셨어요 |
| 읽다 | 읽으십니다 | 읽으세요 | 읽으셨어요 |
| 살다 | 사십니다 | 사세요 | 사셨어요 |
| 있다 | 계십니다 | 계세요 | 계셨어요 |

### -(으)시- 사용 대상

| 주어 | -(으)시- 사용? | 예시 |
|---|---|---|
| 선생님, 교수님 | ✅ 네 | 선생님이 오**셨**어요. (선생님이 오셨어요.) |
| 부모님, 조부모님 | ✅ 네 | 어머니가 요리를 하**세요**. (어머니가 요리를 하세요.) |
| 상사, 선배 | ✅ 네 | 부장님이 말씀하**셨**어요. (부장님이 말씀하셨어요.) |
| 친구 (동갑) | ❌ 아니요 | 민수가 왔어요. (민수가 왔어요.) |
| 자기 자신 | ❌ 절대 안 됨 | ~~제가 가셨어요~~ → 제가 갔어요 |
| 손님 (서비스업) | ✅ 네 | 주문하**시**겠어요? (주문하시겠어요?) |

### 흔한 -(으)시- 패턴

- **-(으)세요**: 존경 명령/질문 — 앉으세요. (앉으세요.)
- **-(으)셨어요**: 과거 존경 — 잘 주무셨어요? (잘 주무셨어요?)
- **-(으)실 거예요**: 미래 존경 — 언제 오실 거예요? (언제 오실 거예요?)
- **-(으)시겠어요**: 존경 의향/제안 — 뭐 드시겠어요? (뭐 드시겠어요?)

## 7. 존경어 쌍

한국어에는 존경하는 주어일 때 사용하는 특별한 **존경 어휘**가 있습니다. -(으)시-를 붙이는 것만큼이나 중요합니다.

### 동사 쌍

| 일반 | 존경 | 의미 |
|---|---|---|
| 먹다 | **드시다** / **잡수시다** | 먹다 |
| 마시다 | **드시다** | 마시다 |
| 자다 | **주무시다** | 자다 |
| 있다 (존재) | **계시다** | 있다 (장소) |
| 없다 | **안 계시다** | 없다 (장소) |
| 죽다 | **돌아가시다** | 돌아가시다 |
| 말하다 | **말씀하시다** | 말씀하시다 |
| 아프다 | **편찮으시다** | 편찮으시다 |
| 주다 | **(주시다)** | 주다 (-시- 추가) |

### 명사 쌍

| 일반 | 존경 | 의미 |
|---|---|---|
| 나이 | **연세** | 나이 |
| 밥 | **진지** | 식사 |
| 집 | **댁** | 댁 |
| 이름 | **성함** | 성함 |
| 말 | **말씀** | 말씀 |
| 생일 | **생신** | 생신 |
| 사람 | **분** | 분 |

### 조사 쌍

| 일반 | 존경 | 역할 |
|---|---|---|
| 이/가 | **께서** | 주격 조사 |
| 에게/한테 | **께** | ~에게 |

### 예문

- 할머니, **진지** 드셨어요? (할머니, 진지 드셨어요?) — *~~밥 먹었어요~~가 아님*
- 선생님**께서** 교실에 **계세요**. (선생님께서 교실에 계세요.)
- **성함**이 어떻게 되세요? (성함이 어떻게 되세요?) — *~~이름이 뭐예요~~가 아님*
- 할아버지 **연세**가 어떻게 되세요? (할아버지 연세가 어떻게 되세요?)
- 아버지가 작년에 **돌아가셨어요**. (아버지가 작년에 돌아가셨어요.) — *~~죽었어요~~가 아님*
- 사장님**께** 말씀 **드리겠습니다**. (사장님께 말씀 드리겠습니다.)

## 8. 겸양어 / 낮춤 형태

높은 지위의 사람에게 말할 때, 겸양형을 사용하여 **자신을 낮춥니다**. 이는 존경법 체계를 보완합니다.

### 대명사 변화

| 일반 | 겸양 | 사용 |
|---|---|---|
| 나 | **저** | 나/저 (공손) |
| 우리 | **저희** | 우리/저희 (공손) |
| 내 (나의) | **제 (저의)** | 나의/저의 (공손) |

### 겸양 동사 형태

| 일반 | 겸양 | 의미 |
|---|---|---|
| 주다 | **드리다** | 주다 (윗사람에게) |
| 묻다/물어보다 | **여쭙다/여쭤보다** | 여쭙다 (윗사람에게) |
| 만나다 | **뵙다/뵈다** | 뵙다 (윗사람을) |
| 데리다 | **모시다** | 모시다/수행하다 |
| 보다 | **뵙다** | 뵙다 (윗사람을) |
| 말하다 | **말씀드리다** | 말씀드리다 (윗사람에게) |
| 있다 | **있겠습니다** (겸양 문맥) | 있다/존재하다 |

### 겸양어와 존경어가 함께 쓰이는 경우

한 문장에서 상대를 높이면서 동시에 자신을 낮출 수 있습니다:

| 문장 | 분석 |
|---|---|
| **제**가 선생님**께** 말씀 **드리겠습니다**. | 제 (겸양 나) + 께 (존경 조사) + 드리다 (겸양 주다/말하다) |
| **처음 뵙겠습니다**. | 뵙다 (겸양 만나다) — "처음 뵙겠습니다" |
| 선생님을 **모시고** 왔습니다. | 모시다 (겸양 수행하다) — "선생님을 (공손히) 모시고 왔습니다" |
| **여쭤볼** 것이 있습니다. | 여쭤보다 (겸양 물어보다) — "여쭤볼 것이 있습니다" |

### 흔한 겸양 표현

- **처음 뵙겠습니다.** (처음 뵙겠습니다.) — 매우 격식적인 첫 만남
- **말씀 드릴 게 있습니다.** (말씀드릴 것이 있습니다.) — 윗사람에게
- **제가 모시겠습니다.** (제가 모시겠습니다.)
- **감사 드립니다.** (감사드립니다.) — 겸양 + 격식

## 9. 경어법 전환 시기

### 경어법을 결정하는 요인

| 요인 | 높은 단계 (존댓말) | 낮은 단계 (반말) |
|---|---|---|
| **나이** | 나보다 나이가 많음 | 나보다 어림 |
| **사회적 지위** | 상사, 선생님, 고객 | 부하, 학생 |
| **관계** | 지인, 낯선 사람 | 친한 친구, 가족 |
| **상황** | 격식적 (사무실, 면접) | 비격식적 (집, 술자리) |
| **첫 만남** | 항상 존댓말 | 절대 반말 아님 |

### 나이 요인

한국 문화에서는 **1년**의 나이 차이도 경어법에 영향을 줄 수 있습니다:

- **동갑**: 존댓말로 시작, 이후 반말 협의
- **1-2살 위 (형/누나/오빠/언니)**: 보통 친하지 않으면 존댓말
- **3살 이상 위**: 존댓말 필수
- **나보다 어림**: 교감이 형성된 후 반말 가능

### 흔한 전환 상황

| 상황 | 전형적 단계 | 비고 |
|---|---|---|
| 처음 만나는 사람 | 해요체 | 안전한 기본값 |
| 취업 면접 | 합쇼체 | 최대한 존경 |
| 함께 술자리 후 | 해요체 → 반말로 전환 가능 | 술이 격식을 풀어줌 |
| 온라인 (같은 또래) | 반말 흔함 | 인터넷 문화가 덜 격식적 |
| 직장 상사 | 합쇼체 또는 해요체 + -(으)시- | 회사 문화에 따라 다름 |

## 10. 문화적 참고사항

### 반말 허락 — 비격식 말투 사용 허가 받기

한국 문화에서는 단순히 반말을 시작할 수 없습니다. 이 전환은 협의가 필요한 **사회적 이정표**입니다.

**과정**:
1. 두 사람이 만나서 존댓말 (공손한 말투)을 사용합니다.
2. 한 사람 (보통 나이가 많은 쪽)이 말합니다: "우리 말 놓을까요?" 또는 "반말 할까요?" (격식을 풀까요?)
3. 상대가 동의합니다: "네, 좋아요!" (네, 좋아요!)
4. 둘 다 반말로 전환합니다 (또는 어린 쪽이 해요체를 유지하고 나이 많은 쪽이 반말을 사용).

**중요**: 보통 **나이가 많거나** **지위가 높은** 사람이 먼저 제안합니다. 어린 사람이 먼저 제안하면 건방져 보일 수 있습니다.

### 나이 묻기 — 나이에 대해 물어보기

한국인들은 관계 초기에 나이를 물어봅니다 — 무례함이 아니라, 올바른 경어법을 결정하기 위해서입니다.

흔한 질문 방법:
- 몇 살이에요? (몇 살이에요?) — 일상적 공손
- 나이가 어떻게 되세요? (나이가 어떻게 되세요?) — 존중적
- 연세가 어떻게 되세요? (연세가 어떻게 되세요?) — 어른에게
- 몇 년생이에요? (몇 년생이에요?) — 젊은 사람들 사이에서 흔함

### 경어법 혼용은 실수

잘못된 경어법을 사용하면 심각한 사회적 어색함을 초래할 수 있습니다:

| 실수 | 인식 |
|---|---|
| 나이 많은 낯선 사람에게 반말 | 매우 무례, 모욕적일 수 있음 |
| 친한 친구에게 합쇼체 | 차갑고, 감정적 거리감 조성 |
| 상사에게 반말 | 사회생활을 끝낼 수 있는 실수 |
| 어린아이에게 존댓말 | 이상하고, 과도하게 격식적 |

## 11. 연습 문제

### 연습 1: 경어법 전환

문장을 지정된 경어법으로 전환하세요.

1. 저는 학생이에요. → 합쇼체: _______
2. 한국어를 공부합니다. → 해체 (반말): _______
3. 어디 가? → 해요체: _______
4. 밥을 먹었어요. → 합쇼체: _______
5. 내일 옵니까? → 해요체: _______

<details>
<summary>정답</summary>

1. 저는 학생**입니다**.
2. 한국어를 공부**해**.
3. 어디 **가요**?
4. 밥을 **먹었습니다**.
5. 내일 **와요**? / 내일 **올 거예요**?

</details>

### 연습 2: 존경어 적용

적절한 존경 형태 (-(으)시- 및/또는 존경 어휘)를 사용하여 문장을 다시 쓰세요.

1. 할머니가 밥을 먹었어요. →
2. 선생님이 학교에 있어요. →
3. 사장님이 말했어요. →
4. 할아버지가 잤어요. →
5. 이름이 뭐예요? (어른에게 물을 때) →

<details>
<summary>정답</summary>

1. 할머니**께서** **진지**를 **드셨어요**.
2. 선생님**께서** 학교에 **계세요**.
3. 사장님**께서** **말씀하셨어요**.
4. 할아버지**께서** **주무셨어요**.
5. **성함**이 어떻게 **되세요**?

</details>

### 연습 3: 겸양 형태

빈칸에 적절한 겸양 표현을 채우세요.

1. 선생님을 _______ 왔습니다. (만나러 오다 — 겸양)
2. 한 가지 _______ 것이 있습니다. (물어볼 — 겸양)
3. 선생님께 선물을 _______. (주다 — 겸양, 과거)
4. _______ 소개하겠습니다. (나 — 겸양 + 자기소개)
5. _______ 회사에서 일하고 있습니다. (우리 — 겸양)

<details>
<summary>정답</summary>

1. 선생님을 **뵈러** / **뵙러** 왔습니다. (선생님을 뵈러 왔습니다.)
2. 한 가지 **여쭤볼** 것이 있습니다. (한 가지 여쭤볼 것이 있습니다.)
3. 선생님께 선물을 **드렸어요**. (선생님께 선물을 드렸어요.)
4. **제가** 소개하겠습니다. (제가 소개하겠습니다.)
5. **저희** 회사에서 일하고 있습니다. (저희 회사에서 일하고 있습니다.)

</details>

### 연습 4: 적절한 경어법 선택

각 상황에서 어떤 경어법을 사용할까요?

1. 학회에서 발표하기
2. 친한 친구와 커피 마시며 대화하기
3. 친구 어머니를 처음 만나서 대화하기
4. 식당에서 주문하기
5. 할아버지가 성인 손주에게 말하기

<details>
<summary>정답</summary>

1. **합쇼체** (격식 존대 — 공식/전문적 상황)
2. **해체 / 반말** (비격식 — 친한 친구)
3. **해요체** + -(으)시- (비격식 존대 + 존경)
4. **해요체** (비격식 존대 — 서비스 상황 표준)
5. **해체 / 반말** 또는 **하게체** (비격식 또는 준격식으로 어린 가족에게)

</details>

### 연습 5: 대화 완성

관계를 고려하여 적절한 형태로 빈칸을 채우세요.

**상황**: 대학생(당신)이 교수님을 처음 만남.

당신: 안녕하세요. _______ 뵙겠습니다. (처음 만남 — 겸양/격식)
교수님: 아, 네. 반가워요. 이름이 어떻게 돼요?
당신: _______ 김민수입니다. (저는 김민수입니다 — 겸양)
교수님: 한국어 수업을 듣는 학생이에요?
당신: 네, 선생님 수업을 _______ 왔습니다. (수업을 들으러 왔습니다 — 겸양)

<details>
<summary>정답</summary>

당신: 안녕하세요. **처음** 뵙겠습니다.
당신: **저는** 김민수입니다.
당신: 네, 선생님 수업을 **들으러** 왔습니다. (또는 **듣고 싶어서** 왔습니다)

</details>
