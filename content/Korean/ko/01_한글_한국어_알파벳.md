## 1. 한글의 역사

### 1.1 한글의 창제

```
┌──────────────────────────────────────────────────────────────────┐
│                    한글의 탄생                                     │
│                                                                  │
│   창제자:  세종대왕(世宗大王)                                      │
│   연도:    1443년(창제), 1446년(반포)                              │
│   문헌:    훈민정음(訓民正音, "백성을 가르치는 바른 소리")           │
│                                                                  │
│   목적:  양반 계급만 알던 한자(漢字) 의존에서 벗어나                │
│          모든 백성이 읽고 쓸 수 있도록 하기 위함                    │
│                                                                  │
│   유네스코:  훈민정음이 세계기록유산으로 등재(1997)                  │
│             한글날: 10월 9일(한글날)                                │
└──────────────────────────────────────────────────────────────────┘
```

한글이 만들어지기 전, 한국인들은 한자(漢字)를 사용하여 글을 썼습니다. 한자는 표의 문자이고 그 문법이 한국어와 근본적으로 달랐기 때문에 양반(兩班) 계급만이 읽고 쓸 수 있었습니다. 대다수의 백성은 문맹이었습니다.

조선 제4대 왕인 세종대왕은 이러한 불평등을 인식하였습니다. 훈민정음 서문에서 다음과 같이 밝혔습니다:

> "나랏말이 중국과 달라 한자와 서로 통하지 아니하므로, 어리석은 백성이 말하고자 하는 바가 있어도 마침내 제 뜻을 펴지 못하는 사람이 많다. 내가 이를 가엾이 여겨 새로 스물여덟 글자를 만드니, 사람마다 쉽게 익혀 날마다 쓰기에 편하게 하고자 할 따름이다."

### 1.2 설계 원리

한글은 언어학자들로부터 역사상 가장 과학적으로 설계된 문자 체계 중 하나로 칭송받고 있습니다. 설계 원리는 다음과 같습니다:

1. **자음은 발음 기관을 본뜸**: 기본 자음 모양은 각 소리를 낼 때 혀, 입술, 목의 위치를 묘사
2. **모음은 우주를 본뜸**: 천(天, ㆍ 점), 지(地, ㅡ 가로획), 인(人, ㅣ 세로획) 세 요소 기반
3. **체계적 확장**: 관련 소리는 기본 모양에 획을 추가하여 파생
4. **자질 문자**: 같은 조음 위치의 자음은 같은 기본 모양을 공유

```
┌──────────────────────────────────────────────────────────────────┐
│           자음 설계: 조음 기관 기반                                │
│                                                                  │
│   ㄱ (g/k) — 혀뿌리가 목 뒤쪽에 닿는 모양                        │
│   ㄴ (n)   — 혀끝이 윗잇몸에 닿는 모양                            │
│   ㅁ (m)   — 입 모양(입술을 다문 형태)                            │
│   ㅅ (s)   — 이(치아)의 모양                                     │
│   ㅇ (ng)  — 목구멍의 모양(원형 = 열린 상태)                      │
│                                                                  │
│   이 5개 기본 모양에서 획을 추가하여 다른 자음을 만듦:             │
│     ㄱ → ㅋ (획 추가) → ㄲ (쌍자음)                               │
│     ㄴ → ㄷ → ㅌ (획 추가)                                        │
│     ㅁ → ㅂ → ㅍ (획 추가)                                        │
│     ㅅ → ㅈ → ㅊ (획 추가)                                        │
│     ㅇ → ㅎ (획 추가)                                             │
└──────────────────────────────────────────────────────────────────┘
```

## 2. 14개 기본 자음

한국어에는 14개의 기본 자음(자음, jaeum)이 있습니다. 각 자음은 초성(음절 시작)과 종성(음절 끝의 받침)에서 다른 소리를 냅니다.

| 글자 | 이름 | 로마자 표기 | IPA(초성) | IPA(종성) | 발음 안내 |
|------|------|-----------|-----------|-----------|----------|
| ㄱ | 기역 | g/k | /k/ 또는 /g/ | /k̚/ | "ㄱ"과 "ㅋ" 사이 — 무기음. 영어 "g"보다 성대 진동이 약함 |
| ㄴ | 니은 | n | /n/ | /n/ | 영어 "n"과 유사 |
| ㄷ | 디귿 | d/t | /t/ 또는 /d/ | /t̚/ | "ㄷ"과 "ㅌ" 사이 — 무기음 |
| ㄹ | 리을 | r/l | /ɾ/ | /l/ | 초성: 스페인어 "r"과 유사한 탄설음. 종성: 영어 "l" |
| ㅁ | 미음 | m | /m/ | /m/ | 영어 "m"과 유사 |
| ㅂ | 비읍 | b/p | /p/ 또는 /b/ | /p̚/ | "ㅂ"과 "ㅍ" 사이 — 무기음 |
| ㅅ | 시옷 | s | /s/ 또는 /ɕ/ | /t̚/ | 영어 "s"와 유사. ㅣ 앞에서: "sh" 소리 |
| ㅇ | 이응 | -/ng | 무음 | /ŋ/ | 초성: 무음(자리 채움). 종성: 영어 "sing"의 "ng" |
| ㅈ | 지읒 | j | /tɕ/ | /t̚/ | 영어 "j"와 유사 — 무기음 |
| ㅊ | 치읓 | ch | /tɕʰ/ | /t̚/ | 영어 "ch"와 유사 — 유기음(거센소리) |
| ㅋ | 키읔 | k | /kʰ/ | /k̚/ | 영어 "k"와 유사 — 강한 유기음 |
| ㅌ | 티읕 | t | /tʰ/ | /t̚/ | 영어 "t"와 유사 — 강한 유기음 |
| ㅍ | 피읖 | p | /pʰ/ | /p̚/ | 영어 "p"와 유사 — 강한 유기음 |
| ㅎ | 히읗 | h | /h/ | /t̚/ | 영어 "h"와 유사 |

### 2.1 세 가지 유형의 폐쇄음

한국어 자음의 핵심 개념은 폐쇄음의 **삼중 대립**입니다. 영어에는 두 가지(유성/무성)가 있지만, 한국어에는 세 가지가 있습니다:

```
┌──────────────────────────────────────────────────────────────────┐
│              한국어 폐쇄음의 삼중 대립                              │
│                                                                  │
│   평음(예사소리)    유기음(거센소리)    경음(된소리)                 │
│   ───────────     ─────────         ───────────                  │
│   ㄱ /k/          ㅋ /kʰ/           ㄲ /k͈/                      │
│   ㄷ /t/          ㅌ /tʰ/           ㄸ /t͈/                      │
│   ㅂ /p/          ㅍ /pʰ/           ㅃ /p͈/                      │
│   ㅈ /tɕ/         ㅊ /tɕʰ/          ㅉ /t͈ɕ/                     │
│   ㅅ /s/          —                 ㅆ /s͈/                      │
│                                                                  │
│   평음:     기식이 약하고, 유성음 사이에서 유성화 가능              │
│   유기음:   강한 날숨(입 앞에 종이를 대면 흔들림)                  │
│   경음:     성대를 긴장시키고, 기식 없이 "짜는" 느낌의 소리         │
└──────────────────────────────────────────────────────────────────┘
```

> **팁**: 입 앞에 손을 대 보세요. ㅋ(유기음)에서는 강한 공기를 느낄 수 있고, ㄱ(평음)과 ㄲ(경음)에서는 공기가 거의 느껴지지 않습니다. 평음과 경음의 차이는 목의 긴장도에 있습니다 — 경음은 "조이는" 느낌입니다.

## 3. 5개 쌍자음(된소리)

5개의 쌍(쌍, ssang) 자음은 기본 자음을 두 번 써서 표기합니다. 목의 근육을 긴장시킨 성문화된 소리로 발음합니다.

| 글자 | 이름 | 로마자 표기 | IPA | 발음 안내 |
|------|------|-----------|-----|----------|
| ㄲ | 쌍기역 | kk | /k͈/ | 영어 "sky"의 "k"처럼 — 긴장, 무기 |
| ㄸ | 쌍디귿 | tt | /t͈/ | 영어 "stun"의 "t"처럼 — 긴장, 무기 |
| ㅃ | 쌍비읍 | pp | /p͈/ | 영어 "spy"의 "p"처럼 — 긴장, 무기 |
| ㅆ | 쌍시옷 | ss | /s͈/ | 영어 "boss"의 "ss"처럼 — 긴장, 날카로움 |
| ㅉ | 쌍지읒 | jj | /t͈ɕ/ | 영어 "budge"의 "j"처럼 — 긴장, 무기 |

### 3.1 예시

| 단어 | 로마자 표기 | 의미 |
|------|-----------|------|
| 까치 | kkachi | 까치 |
| 뚜껑 | ttukkeoong | 뚜껑 |
| 빠르다 | ppareuda | 빠르다 |
| 쓰다 | sseuda | 쓰다 / 쓴맛 |
| 짜다 | jjada | 짜다 / 짜내다 |

## 4. 10개 기본 모음

한국어에는 10개의 기본 모음(모음, moeum)이 있습니다. 두 가지 원리로 구성됩니다:

1. **점/획의 위치**: 세로 또는 가로 기본선에 대한 상대적 위치
2. **양성/음성 대립**: 양성모음(ㅏ, ㅗ, ㅑ, ㅛ) vs. 음성모음(ㅓ, ㅜ, ㅕ, ㅠ) vs. 중성(ㅡ, ㅣ)

```
┌──────────────────────────────────────────────────────────────────┐
│                  한글 모음 설계 원리                                │
│                                                                  │
│   세 가지 우주론적 요소:                                          │
│                                                                  │
│     ● (점/짧은 획) = 하늘(天, ㆍ — 옛 모음)                      │
│     ─ (가로선)      = 땅(地, ㅡ)                                  │
│     │ (세로선)       = 사람(人, ㅣ)                                │
│                                                                  │
│   ●과 ─ 또는 │의 조합으로 기본 모음을 생성:                       │
│                                                                  │
│     양성(양성모음):           음성(음성모음):                       │
│     ㅏ  (│+ ● 오른쪽)        ㅓ  (● 왼쪽 + │)                     │
│     ㅗ  (● 위 + ─)           ㅜ  (─ + ● 아래)                     │
│     ㅑ  (│+ ●● 오른쪽)       ㅕ  (●● 왼쪽 + │)                    │
│     ㅛ  (●● 위 + ─)          ㅠ  (─ + ●● 아래)                    │
│                                                                  │
│   중성:                                                           │
│     ㅡ  (─)                  ㅣ  (│)                               │
└──────────────────────────────────────────────────────────────────┘
```

| 글자 | 이름 | 로마자 표기 | IPA | 발음 안내 |
|------|------|-----------|-----|----------|
| ㅏ | 아 | a | /a/ | 영어 "father"의 "a" |
| ㅑ | 야 | ya | /ja/ | 영어 "yard"의 "ya" |
| ㅓ | 어 | eo | /ʌ/ | 영어 "bus"의 "u" — "o"보다 입을 더 크게 벌림 |
| ㅕ | 여 | yeo | /jʌ/ | 영어 "young"의 "yo" |
| ㅗ | 오 | o | /o/ | 영어 "go"의 "o" — 입술을 둥글게, 활음 없이 |
| ㅛ | 요 | yo | /jo/ | 영어 "yoga"의 "yo" |
| ㅜ | 우 | u | /u/ | 영어 "moon"의 "oo" |
| ㅠ | 유 | yu | /ju/ | 영어 "you" |
| ㅡ | 으 | eu | /ɯ/ | 영어 "book"의 "oo"처럼 하되 입술을 옆으로 벌림(비원순). 미소 짓고 "우"라고 말하기 |
| ㅣ | 이 | i | /i/ | 영어 "see"의 "ee" |

### 4.1 양성모음 vs. 음성모음(모음 조화)

이 구분은 동사 활용에서 중요합니다(5과에서 자세히 다룸):

```
양성모음(양성모음):  ㅏ  ㅗ  ㅑ  ㅛ    → -아와 결합
음성모음(음성모음):  ㅓ  ㅜ  ㅕ  ㅠ    → -어와 결합
중성모음:           ㅡ  ㅣ             → -어와 결합
```

## 5. 11개 복합 모음

복합 모음(이중 모음, ijung moeum)은 기본 모음을 결합하여 만듭니다.

| 글자 | 이름 | 로마자 표기 | IPA | 구성 | 발음 안내 |
|------|------|-----------|-----|------|----------|
| ㅐ | 애 | ae | /ɛ/ | ㅏ + ㅣ | 영어 "bed"의 "e" |
| ㅒ | 얘 | yae | /jɛ/ | ㅑ + ㅣ | 영어 "yes"의 "ye" |
| ㅔ | 에 | e | /e/ | ㅓ + ㅣ | 영어 "hey"의 "e"(ㅐ보다 약간 높음) |
| ㅖ | 예 | ye | /je/ | ㅕ + ㅣ | 영어 "yet"의 "ye" |
| ㅘ | 와 | wa | /wa/ | ㅗ + ㅏ | 영어 "water"의 "wa" |
| ㅙ | 왜 | wae | /wɛ/ | ㅗ + ㅐ | 영어 "wet"의 "we" |
| ㅚ | 외 | oe | /we/ | ㅗ + ㅣ | 영어 "way"의 "we"(현대 서울 방언) |
| ㅝ | 워 | wo | /wʌ/ | ㅜ + ㅓ | 영어 "won"의 "wo" |
| ㅞ | 웨 | we | /we/ | ㅜ + ㅔ | 영어 "wet"의 "we" |
| ㅟ | 위 | wi | /wi/ | ㅜ + ㅣ | 영어 "week"의 "wee" |
| ㅢ | 의 | ui | /ɰi/ | ㅡ + ㅣ | ㅡ에서 ㅣ로 활음. 영어에 대응 없음 |

> **현대어 참고**: 현대 서울말에서 ㅐ와 ㅔ는 대부분의 화자에게 합류하여 거의 /e/로 발음됩니다. 마찬가지로 ㅙ, ㅚ, ㅞ도 /we/로 동일하게 발음되는 경우가 많습니다.

## 6. 음절 블록 구조

### 6.1 음절 블록의 원리

글자를 일렬로 쓰는 알파벳(예: "han")과 달리, 한국어는 글자를 **음절 블록**(음절 블록)으로 묶습니다. 각 블록은 하나의 음절을 나타내며 한 글자 공간을 차지합니다.

```
┌──────────────────────────────────────────────────────────────────┐
│                  음절 블록 유형                                    │
│                                                                  │
│   유형 1: CV (자음 + 모음)                                        │
│   ┌───┐    ┌───┬───┐    예: 가 (ga)                              │
│   │ C V│    │ ㄱ│ ㅏ│    ㄱ (g) + ㅏ (a) = 가                    │
│   └───┘    └───┴───┘                                             │
│                                                                  │
│   유형 2: CVC (자음 + 모음 + 자음)                                │
│   ┌───┐    ┌───┬───┐    예: 한 (han)                             │
│   │ C V│    │ ㅎ│ ㅏ│    ㅎ (h) + ㅏ (a) + ㄴ (n) = 한           │
│   │ C  │    │  ㄴ  │                                             │
│   └───┘    └─────┘                                               │
│                                                                  │
│   유형 3: CVCC (자음 + 모음 + 자음 2개)                           │
│   ┌───┐    ┌───┬───┐    예: 삶 (salm → sam)                     │
│   │ C V│    │ ㅅ│ ㅏ│    ㅅ (s) + ㅏ (a) + ㄹㅁ (lm) = 삶        │
│   │ CC │    │ ㄹㅁ │                                             │
│   └───┘    └─────┘                                               │
└──────────────────────────────────────────────────────────────────┘
```

### 6.2 블록 배치 규칙

배치는 모음의 모양에 따라 달라집니다:

```
세로 모음 (ㅏ ㅑ ㅓ ㅕ ㅣ ㅐ ㅒ ㅔ ㅖ):
  ┌───┬───┐        ┌───┬───┐
  │ C │ V │   또는  │ C │ V │     C = 초성 자음
  └───┴───┘        ├───┴───┤     V = 모음
                   │  C(C) │     C(C) = 종성 자음
                   └───────┘

가로 모음 (ㅗ ㅛ ㅜ ㅠ ㅡ):
  ┌───────┐        ┌───────┐
  │   C   │   또는  │   C   │
  ├───────┤        ├───────┤
  │   V   │        │   V   │
  └───────┘        ├───────┤
                   │ C(C)  │
                   └───────┘

복합 모음 (ㅘ ㅙ ㅚ ㅝ ㅞ ㅟ ㅢ):
  ┌───┬───┐        ┌───┬───┐
  │ C │   │   또는  │ C │   │
  ├───│ V │        ├───│ V │
  │   │   │        │   │   │
  └───┴───┘        ├───┴───┤
                   │ C(C)  │
                   └───────┘
```

### 6.3 중요한 규칙

1. **모든 음절 블록은 반드시 자음으로 시작해야 합니다.** 모음 소리로 시작하는 음절은 ㅇ을 무음 자리 채움 글자로 사용합니다: 아 (a) = ㅇ + ㅏ
2. **모든 음절 블록에는 정확히 하나의 모음이 있어야 합니다.**
3. **음절 블록에는 0, 1, 또는 2개의 종성 자음(받침)**이 올 수 있습니다.

### 6.4 읽기 예시

| 단어 | 블록 | 분해 | 의미 |
|------|------|------|------|
| 한국 | 한 + 국 | ㅎ+ㅏ+ㄴ, ㄱ+ㅜ+ㄱ | 한국 |
| 서울 | 서 + 울 | ㅅ+ㅓ, ㅇ+ㅜ+ㄹ | 서울 |
| 학교 | 학 + 교 | ㅎ+ㅏ+ㄱ, ㄱ+ㅛ | 학교 |
| 사람 | 사 + 람 | ㅅ+ㅏ, ㄹ+ㅏ+ㅁ | 사람 |
| 감사 | 감 + 사 | ㄱ+ㅏ+ㅁ, ㅅ+ㅏ | 감사 |

## 7. 받침 — 종성

### 7.1 받침이란?

받침(batchim, 문자 그대로 "받쳐 주는 것")은 음절 블록 하단에 오는 자음을 가리킵니다. 19개의 자음과 11개의 겹자음이 받침으로 올 수 있지만, 모두 **7가지 대표음** 중 하나로 발음됩니다:

```
┌──────────────────────────────────────────────────────────────────┐
│              7가지 받침 대표음                                     │
│                                                                  │
│   소리 [k̚]:  ㄱ ㅋ ㄲ ㄳ ㄺ                                     │
│   소리 [n]:   ㄴ ㄵ ㄶ                                           │
│   소리 [t̚]:  ㄷ ㅌ ㅅ ㅆ ㅈ ㅊ ㅎ                                │
│   소리 [l]:   ㄹ ㄼ ㄽ ㄾ ㅀ                                     │
│   소리 [m]:   ㅁ ㄻ                                              │
│   소리 [p̚]:  ㅂ ㅍ ㄿ ㅄ                                        │
│   소리 [ŋ]:   ㅇ                                                 │
│                                                                  │
│   ̚ = 불파음(소리가 멈추지만 공기가 방출되지 않음)                  │
└──────────────────────────────────────────────────────────────────┘
```

### 7.2 겹받침

두 자음이 받침 위치에 오고 뒤에 모음이 없는 경우, 하나만 발음됩니다. 일반 규칙:

- **왼쪽 자음이 주로 발음됨**: ㄳ→[k̚], ㄵ→[n], ㄼ→[l], ㄽ→[l], ㄾ→[l], ㅄ→[p̚]
- **오른쪽 자음이 발음되는 경우**: ㄺ→[k̚], ㄻ→[m], ㄿ→[p̚]

| 겹받침 | 발음 | 예시 | 발음 표기 |
|--------|------|------|----------|
| ㄳ | [k̚] | 몫 (몫) | [mok̚] |
| ㄵ | [n] | 앉다 (앉다) | [an.tta] |
| ㄶ | [n] | 많다 (많다) | [man.ta] |
| ㄺ | [k̚] | 읽다 (읽다) | [ik̚.tta] |
| ㄻ | [m] | 삶 (삶) | [sam] |
| ㄼ | [l] | 여덟 (여덟) | [jʌ.dʌl] |

> **중요**: 겹받침 뒤에 모음이 오면 두 자음 모두 발음됩니다 — 첫 번째는 받침에 남고, 두 번째는 다음 음절의 초성으로 넘어갑니다.

## 8. 발음 규칙

### 8.1 연음 (Linking)

받침이 있는 음절 뒤에 ㅇ(무음)으로 시작하는 음절이 오면, 받침이 다음 음절의 초성으로 "연결"됩니다.

| 표기 | 연음 발음 | 의미 |
|------|----------|------|
| 한국어 | [한구거] (han-gu-geo) | 한국어 |
| 음악 | [으막] (eu-mak) | 음악 |
| 먹어요 | [머거요] (meo-geo-yo) | 먹어요(공손) |
| 있어요 | [이써요] (i-sseo-yo) | 있어요(공손) |
| 읽어요 | [일거요] (il-geo-yo) | 읽어요(공손) |

### 8.2 비음화

받침의 폐쇄음(ㄱ, ㄷ, ㅂ) 뒤에 비음(ㄴ, ㅁ)이 오면, 폐쇄음이 비음으로 변합니다:

```
ㄱ [k̚] + ㄴ/ㅁ → ㅇ [ŋ]    (연구개 폐쇄음 → 연구개 비음)
ㄷ [t̚] + ㄴ/ㅁ → ㄴ [n]    (치경 폐쇄음 → 치경 비음)
ㅂ [p̚] + ㄴ/ㅁ → ㅁ [m]    (양순 폐쇄음 → 양순 비음)
```

| 표기 | 발음 | 적용 규칙 | 의미 |
|------|------|----------|------|
| 학년 | [항년] | ㄱ → ㅇ (ㄴ 앞) | 학년 |
| 작년 | [장년] | ㄱ → ㅇ (ㄴ 앞) | 작년 |
| 받는 | [반는] | ㄷ → ㄴ (ㄴ 앞) | 받는 |
| 합니다 | [함니다] | ㅂ → ㅁ (ㄴ 앞) | 합니다(격식) |
| 십만 | [심만] | ㅂ → ㅁ (ㅁ 앞) | 십만 |

### 8.3 유기음화 (Aspiration)

ㅎ이 평음 폐쇄음(ㄱ, ㄷ, ㅂ, ㅈ)을 만나면, 유기음(거센소리)으로 합쳐집니다:

```
ㅎ + ㄱ → ㅋ      ㄱ + ㅎ → ㅋ
ㅎ + ㄷ → ㅌ      ㄷ + ㅎ → ㅌ
ㅎ + ㅂ → ㅍ      ㅂ + ㅎ → ㅍ
ㅎ + ㅈ → ㅊ      ㅈ + ㅎ → ㅊ
```

| 표기 | 발음 | 적용 규칙 | 의미 |
|------|------|----------|------|
| 좋다 | [조타] | ㅎ + ㄷ → ㅌ | 좋다 |
| 놓고 | [노코] | ㅎ + ㄱ → ㅋ | 놓고 |
| 입학 | [이팍] | ㅂ + ㅎ → ㅍ | 입학 |
| 급히 | [그피] | ㅂ + ㅎ → ㅍ | 급히 |

### 8.4 구개음화 (Palatalization)

받침 ㄷ 또는 ㅌ 뒤에 모음 ㅣ(특히 접미사 이, 히)가 오면, 각각 ㅈ과 ㅊ으로 변합니다:

| 표기 | 발음 | 규칙 | 의미 |
|------|------|------|------|
| 같이 | [가치] | ㅌ + 이 → ㅊ + 이 | 같이 |
| 굳이 | [구지] | ㄷ + 이 → ㅈ + 이 | 굳이 |
| 해돋이 | [해도지] | ㄷ + 이 → ㅈ + 이 | 해돋이 |

## 9. 로마자 표기법(국어의 로마자 표기법)

**국어의 로마자 표기법**은 2000년에 대한민국 정부가 채택하였습니다. 기존의 매큔-라이샤워 표기법을 대체하였습니다.

### 9.1 주요 규칙

| 한글 | 로마자 표기 | 참고 |
|------|-----------|------|
| ㄱ | g (초성) / k (종성) | 모음 사이에서 유성화 |
| ㄷ | d (초성) / t (종성) | |
| ㅂ | b (초성) / p (종성) | |
| ㄹ | r (초성) / l (종성) | |
| ㅓ | eo | "o"가 아님 — 영어권 학습자가 가장 많이 틀리는 부분 |
| ㅡ | eu | "u"가 아님 |
| ㅐ | ae | |
| ㅔ | e | |
| ㅢ | ui | |

### 9.2 예시

| 한글 | 로마자 표기 | 참고 |
|------|-----------|------|
| 부산 | Busan | 도시 이름 |
| 인천 | Incheon | 도시 이름 |
| 독립 | dongnip | ㄱ + ㄹ → [ŋ + n], "dongnip"으로 발음 |
| 한라산 | Hallasan | 산 이름 — ㄴ + ㄹ → [l + l] |
| 종로 | Jongno | 서울의 행정구역 |

> **주의**: 로마자 표기는 보조 수단일 뿐입니다. 가능한 한 빨리 한글을 직접 읽는 법을 배우세요. 로마자 표기는 오해를 불러일으킬 수 있습니다(예: "eo"는 영어의 "eo" 소리가 아닙니다).
