## 1. How Korea Reads Classical Chinese

### 1.1 The Korean Reading Tradition

Korea developed a unique system for reading Classical Chinese that adapts the Chinese text to Korean grammar. This system is called 현토 (懸吐, hyeonto) — literally "hanging" Korean grammatical markers onto Chinese characters.

> **Korean Methods of Reading Classical Chinese**
>
> - **Method 1: 음독 (音讀)** — Read each character's Korean sound
>   天高 → 천고 (cheon-go)
>
> - **Method 2: 현토 (懸吐)** — Add Korean particles to Chinese text
>   天이 高하니 → 하늘이 높으니 (The sky being high...)
>
> - **Method 3: 석독 (釋讀)** — Full Korean translation
>   天高 → 하늘이 높다 (The sky is high)
>
> Historical context:
> Before Hangul (1443), Koreans read Chinese texts by
> mentally converting to Korean word order. After Hangul,
> 현토 became a formal written system used in education.

### 1.2 Why Learn Korean Reading Methods?

- **Academic requirement**: Korean 한문 education requires 현토 and 석독 skills
- **Exam preparation**: 한자능력검정시험 includes 현토 questions
- **Cultural literacy**: Understanding how Koreans historically engaged with Classical Chinese
- **Translation skill**: 현토 is the bridge between Classical Chinese and Korean translation

## 2. 훈독 vs 음독 Review

### 2.1 Two Readings for Every Character

Every Hanja character has two Korean readings:

| Reading Type | Korean | Definition | Example with 山 |
|-------------|--------|-----------|-----------------|
| 음독 (音讀) | Sound reading | Korean pronunciation derived from Chinese | 산 (san) |
| 훈독 (訓讀) | Meaning reading | Korean word for the meaning | 메/뫼 (mountain) |

### 2.2 When to Use Each

| Context | Reading Used | Example |
|---------|-------------|---------|
| Reading compounds aloud | 음독 | 山水 → 산수 (san-su) |
| Dictionary citation | 훈 + 음 | 山 = 메 산 (me san) |
| 현토 reading | 음독 for characters, Korean for particles | 山이 高하니 |
| 석독 (translation) | 훈독 converted to full Korean | 산이 높다 → 뫼가 높다 |

### 2.3 Full Citation Format

When citing a character's reading in Korean, the format is: **훈독 + 음독**

| Character | 훈독 (Meaning) | 음독 (Sound) | Citation |
|-----------|---------------|-------------|----------|
| 天 | 하늘 | 천 | 하늘 천 |
| 地 | 땅 | 지 | 땅 지 |
| 人 | 사람 | 인 | 사람 인 |
| 學 | 배울 | 학 | 배울 학 |
| 善 | 착할 | 선 | 착할 선 |
| 大 | 큰 | 대 | 큰 대 |
| 讀 | 읽을 | 독 | 읽을 독 |
| 書 | 글 | 서 | 글 서 |

## 3. 현토 (懸吐): Adding Korean Particles

### 3.1 What is 현토?

현토 (懸吐) is the practice of inserting Korean grammatical particles (토/吐) between Chinese characters to make the text readable in Korean word order.

> **현토 (懸吐) System**
>
> - **Original Classical Chinese**: 天高地厚
>
> With 현토 added:
>   天이 高하고 地가 厚하다
>   천이 고하고 지가 후하다
>
> Korean translation:
>   하늘이 높고 땅이 두껍다
>   The sky is high and the earth is thick.
>
> The 현토 particles:
> - **이** = subject marker (이/가)
> - **하고** = "and" (conjunctive)
> - **가** = subject marker
> - **하다** = verb ending

### 3.2 Historical Development

| Period | Development |
|--------|-------------|
| Before Hangul (pre-1443) | 이두 (吏讀) system used Chinese characters for Korean particles |
| After Hangul (post-1443) | 현토 written in Hangul became standard for education |
| Joseon Dynasty | 현토 widely used in Confucian academies (서원/書院) |
| Modern education | 현토 taught as part of 한문 curriculum |

## 4. Common 토 (吐) Markers

### 4.1 Subject and Topic Markers

| 토 | Function | Korean Equivalent | When Used |
|----|----------|------------------|-----------|
| 이/가 | Subject marker | 이/가 | After subject noun |
| 은/는 | Topic marker | 은/는 | After topic noun |
| 도 | "Also" | 도 | Adding "also" meaning |

### 4.2 Object Markers

| 토 | Function | Korean Equivalent | When Used |
|----|----------|------------------|-----------|
| 을/를 | Object marker | 을/를 | After direct object |
| 에/에서 | Location | 에/에서 | After location |
| 에게 | Indirect object | 에게 | After recipient |

### 4.3 Verb Endings and Conjunctions

| 토 | Function | Korean Equivalent | When Used |
|----|----------|------------------|-----------|
| 하다 | Declarative ending | -하다 | End of declarative sentence |
| 하니 | Causal/sequential | -하니 | "because/when" |
| 하고 | Conjunctive | -하고 | "and" (connecting clauses) |
| 하면 | Conditional | -하면 | "if" |
| 하여 | Resultative | -하여 | "and so, thereby" |
| 하라 | Imperative | -하라 | Command ending |
| 한가 / 하는가 | Interrogative | -한가 | Question ending |
| 하도다 | Exclamatory | -하도다 | Exclamation ending |

### 4.4 Connective Particles

| 토 | Function | Used For |
|----|----------|----------|
| 而 → -고/며 | "and/and then" | Connecting sequential actions |
| 則 → -면/즉 | "then/if...then" | Conditional result |
| 之 → 의/를 | Possessive/object | Connecting nouns or marking objects |
| 者 → -는 것은 | Nominalizer | Creating noun clauses |
| 也 → -이다/이니라 | Copula | Equational endings |

## 5. Korean Word Order Reading

### 5.1 The Word Order Problem

Classical Chinese and Korean have different word orders:

```
Classical Chinese: S + V + O  (Subject-Verb-Object)
Korean:           S + O + V  (Subject-Object-Verb)

Example:
CC:     孔子 學 禮       (Confucius study rites)
Korean: 공자가 예를 배우다  (Confucius rites study)
```

### 5.2 Reading Order Rules

When reading Classical Chinese in Korean order, follow these steps:

> **Korean Reading Order Rules**
>
> 1. Identify the Subject → add 이/가 or 은/는
> 2. Identify the Object → add 을/를, move before verb
> 3. Identify the Verb → add Korean ending (-다/-하다)
> 4. Read in Korean S-O-V order
>
> CC original: 王 愛 民
> - Step 1: 王이 (subject + marker)
> - Step 2: 民을 (object + marker, moved before verb)
> - Step 3: 愛하다 (verb + ending)
> - Result: 왕이 민을 애하다 → 왕이 백성을 사랑하다
>   The king loves the people

### 5.3 Reading Order Examples

| Original CC | Reading Order | 현토 Reading | Korean Translation |
|-------------|--------------|-------------|-------------------|
| 天高 | 天이 高하다 | 천이 고하다 | 하늘이 높다 |
| 王愛民 | 王이 民을 愛하다 | 왕이 민을 애하다 | 왕이 백성을 사랑하다 |
| 孔子學禮 | 孔子가 禮를 學하다 | 공자가 례를 학하다 | 공자가 예를 배우다 |
| 人讀書 | 人이 書를 讀하다 | 인이 서를 독하다 | 사람이 책을 읽다 |
| 民耕田 | 民이 田을 耕하다 | 민이 전을 경하다 | 백성이 밭을 갈다 |

### 5.4 Special Reading Patterns

| CC Pattern | 현토 Pattern | Example |
|-----------|-------------|---------|
| A者B也 | A는 B이다 | 學者는 覺也이다 (학이란 것은 깨달음이다) |
| A之B | A의 B | 人之心 = 인의 심 (사람의 마음) |
| 不A | A하지 않다 | 不知 = 알지 못하다 |
| 有A | A가 있다 | 有道 = 도가 있다 |
| 無A | A가 없다 | 無道 = 도가 없다 |

## 6. 석독 (釋讀): Full Translation Reading

### 6.1 What is 석독?

석독 (釋讀, seokdok) is the complete Korean translation of Classical Chinese, converting all characters into native Korean expressions where possible.

### 6.2 Three Levels of Reading

| Level | Method | Example (天高地厚) |
|-------|--------|-------------------|
| 음독 (Sound) | Read sounds only | 천고지후 |
| 현토 (Annotated) | Add Korean particles | 천이 고하고 지가 후하다 |
| 석독 (Translation) | Full Korean | 하늘이 높고 땅이 두껍다 |

### 6.3 석독 Examples

| CC Original | 현토 | 석독 (Full Korean) |
|-------------|------|-------------------|
| 日出 | 일이 출하다 | 해가 뜨다 |
| 月明 | 월이 명하다 | 달이 밝다 |
| 山高水清 | 산이 고하고 수가 청하다 | 산이 높고 물이 맑다 |
| 人不學則不知 | 인이 학하지 아니하면 즉 지하지 못하다 | 사람이 배우지 않으면 알지 못한다 |
| 王愛民 | 왕이 민을 애하다 | 왕이 백성을 사랑하다 |
| 學而時習之 | 학하고 때때로 습하다 | 배우고 때때로 익히다 |

## 8. 현토 Practice: Classical Passages

### 8.1 Analects 1.1 (論語 學而篇)

**Original:**
```
子曰 學而時習之 不亦說乎
有朋自遠方來 不亦樂乎
人不知而不慍 不亦君子乎
```

**현토:**
```
子ㅣ 曰하시되 學하고 時로 之를 習하면 不亦 說하지 아니한가
朋이 有하여 遠方으로부터 來하면 不亦 樂하지 아니한가
人이 知하지 아니하여도 慍하지 아니하면 不亦 君子가 아닌가
```

**석독:**
```
공자께서 말씀하시기를, 배우고 때때로 그것을 익히면 또한 기쁘지 아니한가.
벗이 있어 먼 곳에서 찾아오면 또한 즐겁지 아니한가.
남이 알아주지 아니하여도 원망하지 아니하면 또한 군자가 아닌가.
```

### 8.2 Great Learning Opening (大學)

**Original:**
```
大學之道 在明明德 在親民 在止於至善
```

**현토:**
```
大學의 道는 明德을 明하는 데 在하며
民을 親하는 데 在하며
至善에 止하는 데 在하다
```

**석독:**
```
대학의 도는 밝은 덕을 밝히는 데 있으며,
백성을 새롭게 하는 데 있으며,
지극한 선에 머무는 데 있다.
```

## 9. Reading Strategies and Tips

### 9.1 Step-by-Step Reading Method

> **5-Step Korean Reading Method**
>
> 1. Read once for 음독 — get the sounds
>    孔子學禮 → 공자학례
>
> 2. Identify grammar — Subject? Object? Verb?
>    孔子(S) 學(V) 禮(O)
>
> 3. Add 현토 — insert Korean particles
>    孔子가 禮를 學하다
>
> 4. Rearrange to Korean order
>    S + O + V → 공자가 례를 학하다
>
> 5. 석독 — full Korean translation
>    공자가 예를 배우다
>    (Confucius studied the rites)

### 9.2 Key Patterns to Recognize

| CC Pattern | Korean Reading | Translation Pattern |
|-----------|---------------|-------------------|
| A + Adj | A이/가 + Adj하다 | A is [adjective] |
| S + V + O | S이/가 + O을/를 + V하다 | S does V to O |
| A者B也 | A는 B이다 | A is B |
| A之B | A의 B | A's B |
| 不 + V | V하지 않다 | Does not V |
| 有 + N | N이 있다 | There is N |
| 無 + N | N이 없다 | There is no N |
| V乎 | V하는가 | Does [one] V? |

### 9.3 Common Mistakes to Avoid

| Mistake | Why It Is Wrong | Correct Approach |
|---------|----------------|-----------------|
| Reading SVO as SOV | CC is SVO, Korean is SOV | Rearrange object before verb |
| Missing 之 as possessive | 之 is often "의" | Check if 之 connects two nouns |
| Ignoring 而 | 而 marks clause boundaries | Translate as "and" or "but" |
| Treating 也 as a word | 也 is a sentence-ender | Translate as copula "-이다" |
