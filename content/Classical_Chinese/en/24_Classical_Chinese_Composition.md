## 1. Principles of Composition

### The Five Qualities of Good Classical Chinese Writing

| Quality | Hanja | Korean | Description |
|---------|-------|--------|-------------|
| Conciseness | 簡 | 간 (gan) | Say much with few words |
| Clarity | 明 | 명 (myeong) | Meaning should be immediately clear |
| Elegance | 雅 | 아 (a) | Use refined vocabulary and structure |
| Force | 勁 | 경 (gyeong) | Writing should have energy and momentum |
| Depth | 深 | 심 (sim) | Layers of meaning beneath the surface |

### Key Maxim

```
辭達而已矣
사달이이의
```

"Words need only convey their meaning" — 공자/孔子 (Confucius), from 논어/論語.

| Character | Reading | Meaning |
|-----------|---------|---------|
| 辭 | 사 (sa) | words, diction |
| 達 | 달 (dal) | to reach, convey |
| 而已 | 이이 (ii) | that is all |
| 矣 | 의 (ui) | (final particle) |

This does not mean writing should be plain — it means every word should serve the purpose of communication. Ornament without meaning is empty.

### The Writer's Toolbox

| Tool | Description | Example |
|------|-------------|---------|
| Concrete nouns | Use specific, vivid words | 松 (pine) not just 木 (tree) |
| Active verbs | Prefer dynamic verbs | 奔 (rush) not 去 (go) |
| Parallel structure | Balance phrases | A↔B pairing |
| Function words | Control rhythm and logic | 而, 則, 所, 之 |
| Allusions | Layer meaning with references | 전고(典故) |

## 2. Building Blocks: Words and Phrases

### Word Formation Patterns

Classical Chinese words are predominantly monosyllabic, but compound words follow specific patterns:

| Pattern | Structure | Example | Reading | Meaning |
|---------|-----------|---------|---------|---------|
| Synonym compound | A+A | 道路 | 도로 | road |
| Antonym compound | A+B | 大小 | 대소 | size (big-small) |
| Modifier + Head | Adj+N | 明月 | 명월 | bright moon |
| Verb + Object | V+O | 讀書 | 독서 | to read |
| Subject + Verb | S+V | 日出 | 일출 | sunrise |
| Verb + Complement | V+C | 看見 | 간견 | to see (look-perceive) |

### Common Predicate Types

| Type | Pattern | Example | Translation |
|------|---------|---------|-------------|
| Action | S + V + O | 人讀書 (인독서) | A person reads a book |
| Description | S + Adj | 山高 (산고) | The mountain is high |
| Judgment | S + 也 | 仁者人也 (인자인야) | The benevolent person is human |
| Existence | 有 + N | 有人 (유인) | There is a person |
| Negation | 無 + N | 無人 (무인) | There is no one |

### Useful Linking Words

| Word | Reading | Function | Example |
|------|---------|----------|---------|
| 而 | 이 | And/but | 學而知 (learn and know) |
| 則 | 즉 | Then | 學則知 (if learn, then know) |
| 故 | 고 | Therefore | 故曰 (therefore it is said) |
| 然 | 연 | However | 然而 (however) |
| 蓋 | 개 | For/because | 蓋因 (this is because) |
| 夫 | 부 | Now/well | 夫學者 (now, as for scholars) |
| 且 | 차 | Moreover | 且夫 (moreover) |
| 雖 | 수 | Although | 雖然 (although it is so) |

## 3. Sentence Construction

### Level 1: Simple Sentences

Start with the most basic patterns:

| Pattern | Formula | Example | Translation |
|---------|---------|---------|-------------|
| S + V | Subject + Verb | 鳥飛 (조비) | Birds fly |
| S + V + O | Subject + Verb + Object | 人讀書 (인독서) | A person reads a book |
| S + Adj | Subject + Adjective | 天高 (천고) | The sky is high |
| 有 + N | Existence | 山有木 (산유목) | The mountain has trees |
| 無 + N | Non-existence | 路無人 (로무인) | The road has no people |

### Level 2: Modified Sentences

Add modifiers and function words:

| Pattern | Example | Translation |
|---------|---------|-------------|
| Adj + N + V | 白鳥飛 (백조비) | The white bird flies |
| S + Adv + V | 鳥高飛 (조고비) | The bird flies high |
| S + V + Adj + O | 人讀古書 (인독고서) | A person reads old books |
| S + Neg + V | 人不知 (인부지) | People do not know |
| N之N | 天之道 (천지도) | The way of heaven |

### Level 3: Complex Sentences

Combine clauses using function words:

| Connector | Example | Translation |
|-----------|---------|-------------|
| 而 (sequential) | 起而行 (기이행) | Rise and walk |
| 而 (adversative) | 學而不思 (학이불사) | Study but not think |
| 則 (conditional) | 學則知 (학즉지) | If one studies, one knows |
| 故 (causal) | 不學 故不知 (불학 고부지) | Does not study, therefore does not know |
| 雖...然 | 雖貧然樂 (수빈연락) | Although poor, yet happy |

### Level 4: Multi-Clause Sentences

| Type | Example | Translation |
|------|---------|-------------|
| Parallel | 學而知 知而行 (학이지 지이행) | Study and know; know and act |
| Progressive | 修身而後齊家 齊家而後治國 (수신이후제가 제가이후치국) | Cultivate self, then regulate family; regulate family, then govern state |
| Concessive-adversative | 雖不能至 然心嚮往之 (수불능지 연심향왕지) | Although I cannot reach it, my heart yearns toward it |

## 4. Qichengzhuanjie Essay Structure

### 기승전결(起承轉結) in Prose

This four-part structure works for essays and arguments as well as poetry.

| Phase | Hanja | Function | Prose Application |
|-------|-------|----------|-------------------|
| 기 (起) | Opening | State the topic or question | Present the issue |
| 승 (承) | Development | Elaborate with evidence or examples | Build the argument |
| 전 (轉) | Turn | Introduce a contrast or new angle | Address objections or shift perspective |
| 결 (結) | Conclusion | Resolve and summarize | State final position |

### Model Essay Structure

**Topic**: 學之道 (The Way of Learning)

**起 (Opening)**:
```
夫學者 所以明道也
부학자 소이명도야
```
"Now learning — it is the means by which one illuminates the Way."

**承 (Development)**:
```
古之聖賢 皆以學成德
고지성현 개이학성덕

不學而能者 未之有也
불학이능자 미지유야
```
"The sages and worthies of old all achieved virtue through learning. Those who can succeed without learning — there have never been such people."

**轉 (Turn)**:
```
或曰 學不必讀書
혹왈 학불필독서

然不讀書 何以知古今之理乎
연불독서 하이지고금지리호
```
"Some say learning does not require reading books. But without reading, how can one know the principles of past and present?"

**結 (Conclusion)**:
```
是故 爲學之要 在於日新
시고 위학지요 재어일신

不可一日而廢也
불가일일이폐야
```
"For this reason, the key to learning lies in daily renewal. One must not abandon it for even a single day."

## 5. Common Composition Patterns

### Pattern 1: Definition

**Formula**: X者 Y也 (X is Y)

| Example | Reading | Translation |
|---------|---------|-------------|
| 仁者愛人也 | 인자애인야 | Benevolence is loving others |
| 義者宜也 | 의자의야 | Righteousness is what is fitting |
| 學者覺也 | 학자각야 | Learning is awakening |

### Pattern 2: Comparison

**Formula**: A不如B (A is not as good as B)

| Example | Reading | Translation |
|---------|---------|-------------|
| 聞不如見 | 문불여견 | Hearing is not as good as seeing |
| 知不如行 | 지불여행 | Knowing is not as good as doing |
| 獨學不如共學 | 독학불여공학 | Studying alone is not as good as studying together |

### Pattern 3: Conditional

**Formula**: A則B (If A, then B)

| Example | Reading | Translation |
|---------|---------|-------------|
| 學則知 | 학즉지 | If one studies, one knows |
| 勤則成 | 근즉성 | If diligent, one succeeds |
| 不學則愚 | 불학즉우 | If one does not study, one is foolish |

### Pattern 4: Rhetorical Question

**Formula**: 豈不...乎 / 何以...乎

| Example | Reading | Translation |
|---------|---------|-------------|
| 豈不痛哉 | 기불통재 | Is this not painful? |
| 何以知之 | 하이지지 | How can one know this? |
| 不亦樂乎 | 불역락호 | Is this not also joyful? |

### Pattern 5: Progressive Argument

**Formula**: A而後B, B而後C

| Example | Reading | Translation |
|---------|---------|-------------|
| 知止而後有定 定而後能靜 | 지지이후유정 정이후능정 | Know where to stop, then have stability; have stability, then be able to be calm |

## 6. Writing Parallel Prose

### 대구문(對句文) — Parallel Prose Construction

Parallel prose (변려문/駢儷文) pairs sentences of equal length with matching grammatical structure.

### Step-by-Step Construction

**Step 1**: Choose a theme and two contrasting elements

Theme: The seasons
Elements: 春 (spring) and 秋 (autumn)

**Step 2**: Build noun phrases

| Position | Element A | Element B |
|----------|-----------|-----------|
| Season | 春 (춘, spring) | 秋 (추, autumn) |
| Image | 花 (화, flower) | 月 (월, moon) |

**Step 3**: Add verbs

| Position | Element A | Element B |
|----------|-----------|-----------|
| Action | 開 (개, bloom) | 明 (명, shine) |

**Step 4**: Combine

```
春花開    춘화개    Spring flowers bloom
秋月明    추월명    Autumn moon shines
```

**Step 5**: Expand to 5 or 7 characters

```
春來百花開    춘래백화개    When spring comes, a hundred flowers bloom
秋至明月明    추지명월명    When autumn arrives, the bright moon shines
```

### Practice Examples

Build parallel pairs from these elements:

| Theme | Element A | Element B |
|-------|-----------|-----------|
| Landscape | 山 (mountain) | 水 (water) |
| Time | 古 (ancient) | 今 (present) |
| Direction | 東 (east) | 西 (west) |
| Quality | 剛 (hard) | 柔 (soft) |

**Sample results**:

```
山高而蒼翠    산고이창취    Mountains are high and verdant
水深而淸澈    수심이청철    Waters are deep and clear

古人勤學    고인근학    Ancients were diligent in study
今人怠惰    금인태타    Moderns are lazy

東方日出    동방일출    In the east, the sun rises
西方月落    서방월락    In the west, the moon sets
```

## 7. Writing Simple Verse

### Composing a 오언절구(五言絕句)

Follow this step-by-step process:

**Step 1**: Choose a theme and emotion

- Theme: 秋夜 (autumn night)
- Emotion: 孤獨 (solitude)

**Step 2**: Establish the rhyme

Choose rhyme characters from the same 운부. Using the 眞韻(진운) group:
- 人(인), 春(춘), 新(신), 塵(진)

**Step 3**: Write the 기(起) line — set the scene

```
秋夜月光寒    추야월광한
```
"On the autumn night, the moonlight is cold"

**Step 4**: Write the 승(承) line — develop (must rhyme)

```
獨坐聽秋蟬    독좌청추선
```
"Sitting alone, listening to autumn cicadas"

Wait — 蟬(선) is in a different rhyme group. Revise:

```
獨坐思故人    독좌사고인
```
"Sitting alone, thinking of an old friend" — 人(인) rhymes

**Step 5**: Write the 전(轉) line — shift perspective (no rhyme)

```
遙望千里外    요망천리외
```
"Gazing far off beyond a thousand li"

**Step 6**: Write the 결(結) line — resolve (must rhyme)

```
何處是陽春    하처시양춘
```
"Where can warm spring be found?" — 春(춘) rhymes

**Complete poem**:

```
秋夜月光寒    추야월광한
獨坐思故人    독좌사고인
遙望千里外    요망천리외
何處是陽春    하처시양춘
```

## 8. Common Errors and How to Avoid Them

### Error 1: Wrong Word Order

| Error | Correct | Issue |
|-------|---------|-------|
| 書讀 (서독) | 讀書 (독서) | Verb must precede object in Classical Chinese |
| 高山 is fine as modifier | But 山高 for "mountain is high" | Adjective predicates follow the subject |

### Error 2: Missing Function Words

| Error | Correct | Missing |
|-------|---------|---------|
| 人善善也 (인선선야) | 仁者善也 (인자선야) | Need 者 to nominalize |
| 學知 (학지) | 學而知 (학이지) | Need 而 to connect clauses |

### Error 3: Incorrect Negation

| Error | Correct | Rule |
|-------|---------|------|
| 無學 (무학) for "did not study" | 不學 (불학) | 不 negates verbs; 無 negates existence |
| 不人 (불인) for "no person" | 無人 (무인) | 無 means "there is no..." |
| 非好 (비호) for "not good" | 不好 (불호) | 非 negates identity ("is not X"), 不 negates quality |

### Error 4: Misusing 之 and 者

| Error | Correct | Rule |
|-------|---------|------|
| 學者之人 | 學者 or 好學之人 | 者 already nominalizes; don't add 之人 |
| 天者道 | 天之道 | 之 needed for possessive |

### Error 5: Anachronistic Vocabulary

| Modern Term | Classical Equivalent | Note |
|-------------|---------------------|------|
| 問題 (문제, problem) | 難/疑 (난/의) | 問題 is modern coinage |
| 社會 (사회, society) | 天下/國 (천하/국) | 社會 is modern |
| 經濟 (경제, economy) | 經世濟民 (경세제민) | Classical uses the full phrase |

### Error 6: Overlong Sentences

Classical Chinese favors short, punchy sentences. Break long thoughts into multiple clauses:

| Too long | Better |
|----------|--------|
| 人之所以不能學而不能知者以其不勤也 | 人不學不知 何也 不勤也 |
| (One long run-on) | (Three short, punchy sentences) |

## 9. Guided Composition Exercises

### Exercise A: Write a Definition

Define 友(friend) using the pattern X者Y也:

**Model**: 友者 同志之人也 (우자 동지지인야)
"A friend is a person of shared purpose."

Now define:
1. 師 (teacher)
2. 學 (learning)
3. 德 (virtue)

### Exercise B: Write a Comparison

Using the pattern A不如B:

**Model**: 言不如行 (언불여행) — "Words are not as good as actions"

Now write:
1. Compare 富 (wealth) and 德 (virtue)
2. Compare 速 (speed) and 慎 (caution)

### Exercise C: Write a Parallel Couplet

Match each first line with a composed second line:

1. 春來花開 (춘래화개) → ___________
2. 讀書明理 (독서명리) → ___________
3. 天高雲淡 (천고운담) → ___________

### Exercise D: Write a Short Argument

Topic: 學之必要 (The Necessity of Learning)

Write 4-6 lines following 기승전결 structure. Use at least:
- One 不...則... conditional
- One rhetorical question with 豈 or 何
- One parallel pair
