## 1. Introduction to Comparative Reading

Comparative reading (비교독해/比較讀解) is the practice of examining multiple texts on the same subject to understand how different authors, periods, or traditions present the same material. This is essential for advanced Classical Chinese study because:

- No single text tells the complete story
- Authors had different agendas, sources, and audiences
- Comparing versions reveals biases, omissions, and embellishments
- Textual errors accumulate over centuries of copying

### Key Terms

| Term | Hanja | Reading | Meaning |
|------|-------|---------|---------|
| Comparison | 比較 | 비교 (bigyo) | Comparing texts |
| Parallel passage | 並行文 | 병행문 (byeonghaengmun) | Same content in different texts |
| Variant reading | 異文 | 이문 (imun) | Different versions of same passage |
| Textual criticism | 校勘 | 교감 (gyogam) | Examining and correcting texts |
| Interpolation | 竄入 | 찬입 (chanip) | Later insertion into a text |
| Collation | 對校 | 대교 (daegyo) | Comparing editions side by side |
| Original text | 原文 | 원문 (wonmun) | The original version |
| Commentary | 注 / 註 | 주 (ju) | Explanatory notes |

## 3. Parallel Accounts: Comparing Sources

### Case Study 1: The Story of 항우(項羽, Xiang Yu)

The same events are recorded in both 사기(史記) and 한서(漢書), but with different emphases.

**사기/史記 — 項羽本紀**:

```
項籍少時 學書不成 去
항적소시 학서불성 거

學劍 又不成
학검 우불성

項梁怒之
항량노지

籍曰 書足以記名姓而已
적왈 서족이기명성이이

劍一人敵 不足學
검일인적 부족학

學萬人敵
학만인적
```

| Line | Translation |
|------|-------------|
| 項籍少時 學書不成 去 | When Xiang Ji was young, he studied writing but did not master it; he abandoned it |
| 學劍 又不成 | He studied swordsmanship and also did not master it |
| 項梁怒之 | Xiang Liang was angry at him |
| 籍曰 書足以記名姓而已 | Ji said: "Writing is enough to record one's name, that is all" |
| 劍一人敵 不足學 | "Swordsmanship defeats only one man — not worth studying" |
| 學萬人敵 | "I want to study [the art of] defeating ten thousand men" |

**한서/漢書 — 項籍傳**: The same episode appears but with slightly different wording:

```
項籍字羽 下相人也
항적자우 하상인야

少時學書不成 去學劍 又不成
소시학서불성 거학검 우불성
```

### Comparison

| Element | 史記 | 漢書 |
|---------|------|------|
| Opening | Starts directly | Adds biographical details (字羽, 下相人也) |
| Dialogue | Quoted in full | Abbreviated |
| Tone | Dramatic, vivid | More formal, concise |
| Purpose | Character development | Historical record |

### Case Study 2: Korean Founding Myths

Compare the founding myth of Goguryeo (고구려) in two Korean sources:

**삼국사기/三國史記** (official history by 김부식):

```
始祖東明聖王 姓高氏
시조동명성왕 성고씨

諱朱蒙
휘주몽
```

**삼국유사/三國遺事** (folklore collection by 일연):

```
朱蒙 一云鄒牟
주몽 일운추모

又云衆解
우운중해
```

| Feature | 三國史記 | 三國遺事 |
|---------|---------|---------|
| Approach | Official, factual | Includes legends and variants |
| Alternate names | One name only | Lists multiple variants (추모, 중해) |
| Supernatural elements | Minimized | Preserved and detailed |
| Purpose | Political legitimacy | Cultural preservation |

## 4. Textual Criticism Basics

### What is 교감(校勘)?

교감(校勘, textual criticism) is the systematic comparison of different editions or manuscripts of the same text to identify errors, variations, and the most reliable version.

### Types of Textual Errors

| Error Type | Hanja | Korean | Description | Example |
|-----------|-------|--------|-------------|---------|
| Omission | 脫文 | 탈문 (talmun) | Missing characters | Text skips a line |
| Addition | 衍文 | 연문 (yeonmun) | Extra characters inserted | Margin note absorbed into text |
| Transposition | 倒文 | 도문 (domun) | Characters in wrong order | AB → BA |
| Substitution | 訛文 | 와문 (wamun) | Wrong character | Similar-looking char replaces original |
| Corruption | 錯簡 | 착간 (chakgan) | Bamboo strips in wrong order | Paragraphs shuffled |

### How Errors Occur

| Cause | Description |
|-------|-------------|
| Visual similarity | 己(기) confused with 已(이) or 巳(사) |
| Sound similarity | Homophonous characters exchanged |
| Scribal fatigue | Long copying sessions cause skipping |
| Intentional editing | Political censorship or "improvement" |
| Physical damage | Bamboo/silk decay, fire, water |

### Example: Visually Similar Characters

| Group | Characters | Readings | Risk |
|-------|-----------|----------|------|
| 1 | 己, 已, 巳 | 기, 이, 사 | Differ by one stroke |
| 2 | 戊, 戌, 戍, 戎 | 무, 술, 수, 융 | Differ by dot/stroke position |
| 3 | 未, 末 | 미, 말 | Differ by stroke length |
| 4 | 曰, 日 | 왈, 일 | Differ by width |
| 5 | 土, 士 | 토, 사 | Differ by horizontal stroke length |

## 5. Variant Readings and Interpolation

### Identifying Variant Readings (異文/이문)

When two editions of the same text use different characters, scholars must determine which is original.

### Criteria for Evaluating Variants

| Criterion | Principle |
|-----------|-----------|
| Older is better | Earlier manuscripts are generally more reliable |
| Harder is better | Scribes tend to simplify, not complicate (lectio difficilior) |
| Context | Which reading makes better grammatical/semantic sense? |
| Parallel evidence | Do other texts quote the passage? Which version? |
| Author's style | Which reading matches the author's known style? |
| Rhyme/meter | In poetry, which reading fits the tonal pattern? |

### Example: Variant in 도덕경/道德經

Chapter 1 has a famous textual variant:

**Received text (전래본/傳來本)**:
```
故常無欲 以觀其妙
고상무욕 이관기묘

常有欲 以觀其徼
상유욕 이관기요
```

**Mawangdui silk text (마왕퇴 백서본/馬王堆帛書本)**:
```
故恒無欲也 以觀其眇
고항무욕야 이관기묘

恒有欲也 以觀其所噭
항유욕야 이관기소요
```

| Difference | Received | Mawangdui | Significance |
|-----------|----------|-----------|-------------|
| "Constant" | 常 (상) | 恒 (항) | 恒 was taboo (Emperor Wen's name) |
| "Subtlety" | 妙 (묘) | 眇 (묘) | Different characters, similar meaning |
| Particle | (none) | 也 (야) | Mawangdui adds emphasis |
| Punctuation | 常+無欲 | 恒+無+欲也 | Affects grammatical interpretation |

### Interpolation (竄入/찬입)

Interpolation occurs when later material is inserted into an original text. Signs of interpolation:

| Sign | Description |
|------|-------------|
| Anachronism | Text mentions something that did not exist at the time of writing |
| Style break | Passage uses vocabulary or grammar inconsistent with the rest |
| External evidence | Other sources quote the passage differently or not at all |
| Logical interruption | Passage breaks the flow of argument |

## 6. Korean vs. Chinese Classical Chinese

### Distinctive Features of Korean Hanmun

| Feature | Chinese Classical Chinese | Korean Hanmun |
|---------|--------------------------|---------------|
| Vocabulary | Standard literary terms | Includes Korea-specific compounds |
| Place names | Chinese toponyms | Korean toponyms in hanja |
| Personal names | Chinese naming conventions | Korean naming conventions |
| Official titles | Chinese bureaucratic titles | Korean-adapted titles (판서, 영의정) |
| Grammar | Pure Classical Chinese syntax | Occasional Korean-influenced syntax |
| Reading tradition | Direct reading (直讀/직독) | 현토(懸吐) with Korean particles |

### Korean-Specific Vocabulary Examples

| Korean Hanmun | Reading | Standard Chinese | Meaning |
|---------------|---------|-----------------|---------|
| 殿下 | 전하 | 陛下 | Your Majesty (King) |
| 兩班 | 양반 | — | Yangban aristocracy |
| 科擧 | 과거 | 科擧 | Civil service exam (same term, different system) |
| 書堂 | 서당 | 私塾 | Village school |
| 鄕校 | 향교 | — | Local Confucian school |
| 成均館 | 성균관 | 太學 | National Academy |

### Korean Syntactic Influence

Occasionally, Korean hanmun shows Korean word order (SOV instead of SVO):

| Pattern | Korean-influenced | Standard Chinese |
|---------|-------------------|-----------------|
| "See the moon" | 月을 見하다 influence → 月見 | 見月 |
| "Read a book" | 書를 讀하다 influence → 書讀 | 讀書 |

> Note: These inversions are relatively rare in formal hanmun but appear in less formal texts.

## 7. Japanese Kanbun Conventions

### 漢文訓讀 (한문훈독, Kanbun Kundoku)

Japanese scholars developed a system of reading Classical Chinese in Japanese word order, analogous to Korean 현토.

### Key Differences

| Feature | Korean Method | Japanese Method |
|---------|--------------|-----------------|
| Term | 현토 (懸吐) | 訓讀 (kunten) |
| Markers | Korean particles added | 返り点 (kaeriten) reading marks |
| Word order | Korean SOV adaptation | Japanese SOV adaptation |
| Pronunciation | Korean음독/훈독 | On'yomi/Kun'yomi |
| Writing | Original text preserved | Text annotated with marks |

### Japanese Reading Marks (返り点/반점)

Japanese uses numerical and sequential marks to indicate reading order:

| Mark | Function |
|------|----------|
| レ (re-ten) | Read the character below before the one above |
| 一...二 | Read in numbered order |
| 上...下 | Read bottom-to-top within the marked range |

### Example: Reading 學而時習之

| Language | Reading Order | Result |
|----------|--------------|--------|
| Chinese | 學而時習之 | Xué ér shí xí zhī |
| Korean | 學하고 時로 習하여 之를 | Bae-u-go ttae-ro ik-hi-yeo i-reul |
| Japanese | 學びて時に之を習ふ | Manabite toki ni kore wo narau |

### Shared Classical Chinese Across East Asia

| Feature | China | Korea | Japan | Vietnam |
|---------|-------|-------|-------|---------|
| Script | 漢字 (hanzi) | 漢字/한자 (hanja) | 漢字 (kanji) | 漢字 (chu Han) |
| Pronunciation | Chinese readings | Korean readings | Japanese readings | Vietnamese readings |
| Grammar adaptation | Native | 현토 | 訓讀 | Adapted |
| Period of use | Ancient - present | ~4th c. - 1894 | ~5th c. - 1945 | ~2nd c. - early 20th c. |

## 8. Cross-Cultural Text Analysis

### Case Study: The Analects Across Cultures

The opening line of the 논어/論語 is read differently across East Asia:

```
學而時習之 不亦說乎
```

| Culture | Reading | Translation |
|---------|---------|-------------|
| Chinese | Xué ér shí xí zhī, bù yì yuè hū? | "To study and regularly practice, is that not a pleasure?" |
| Korean | 학이시습지 불역열호 | 배우고 때때로 익히면 또한 기쁘지 아니한가 |
| Japanese | まなびてときにこれをならう またよろこばしからずや | 学びて時にこれを習う、亦悦ばしからずや |

### Different Interpretive Traditions

| Tradition | Key Feature |
|-----------|-------------|
| Chinese (中國) | Focus on Zhu Xi's (주희) Neo-Confucian interpretation |
| Korean (韓國) | Emphasis on 이기론(理氣論) — principle and material force |
| Japanese (日本) | 古學派(고학파) — return to original meaning movement |

### Comparative Commentary Traditions

| Commentary | Hanja | Tradition | Period |
|-----------|-------|-----------|--------|
| 주자집주 | 朱子集注 | Chinese Neo-Confucian | 12th c. |
| 퇴계 사서석의 | 退溪 四書釋義 | Korean Neo-Confucian | 16th c. |
| 이토 진사이 | 伊藤仁斎 | Japanese Ancient Learning | 17th c. |
| 오규 소라이 | 荻生徂徠 | Japanese Philological | 18th c. |

## 9. Critical Reading Methodology

### Five-Step Method for Comparative Reading

| Step | Action | Tools |
|------|--------|-------|
| 1. Read independently | Read each text on its own first | Dictionary, grammar knowledge |
| 2. Align parallel passages | Place matching sections side by side | Table format |
| 3. Mark differences | Note every variation in wording | Color coding |
| 4. Evaluate variants | Determine which is likely original | Criteria from Section 5 |
| 5. Synthesize | Reconstruct the most probable version | Historical context |

### Questions to Ask When Comparing Texts

| Question | Purpose |
|----------|---------|
| Who wrote each version? | Understand biases |
| When was each written? | Earlier may be more reliable |
| What was the purpose? | Political, literary, religious agenda? |
| What sources did each use? | Shared or independent sources? |
| Where do they agree? | Probably reliable information |
| Where do they disagree? | Requires further investigation |
| What does each omit? | Omissions can be as telling as additions |

### Example: Analyzing a Discrepancy

If 사기(史記) says a battle involved 30,000 troops and 한서(漢書) says 300,000:

| Possibility | Likelihood | How to evaluate |
|-------------|-----------|-----------------|
| Scribal error (脫字) | High | One zero dropped or added? |
| Different sources | Medium | Each author used different informants |
| Political bias | Medium | Larger numbers glorify one's own side |
| Later interpolation | Low | Check oldest manuscripts |

1. 己 (self) written as 已 (already)

### Exercise 3: Evaluate Variant Readings

### Exercise 4: Cross-Cultural Comparison

### Exercise 5: Vocabulary Comparison

| Korean Hanmun | Chinese Term |
|---------------|-------------|
| 1. 殿下 | a. 太學 |
| 2. 書堂 | b. 陛下 |

### Exercise 6: Source Analysis

You find the same historical event described in 삼국사기 and 삼국유사. The 삼국사기 version is factual and brief; the 삼국유사 version includes supernatural elements. Which would you trust more for historical facts, and why? Which is more valuable for understanding cultural beliefs?

### Exercise 7: Textual Criticism Practice

The following passage has a suspected error. Identify it and propose a correction:

```
天下莫柔弱於火
천하막유약어화
```

(Hint: Compare with 도덕경 Chapter 78)

<details>
<summary>Answers</summary>

**Exercise 1**: 1) 史記 uses 陳涉 while 漢書 uses 陳勝字涉 (adds courtesy name) 2) 漢書 adds 陽城人也 (place of origin) 3) 史記 leads with the story immediately; 漢書 starts with biographical format

**Exercise 2**: 1) 訛文 (substitution — similar-looking character) 2) 脫文 (omission) 3) 訛文 (substitution) 4) 倒文 (transposition) 5) 衍文 (addition/interpolation)

**Exercise 3**: 恒(항) is likely original. The character 常(상) was substituted because Emperor Wen of Han (漢文帝) was named 劉恒 — his personal name became taboo (避諱/피휘), so all texts were changed to avoid using 恒. The Mawangdui text predates this taboo, preserving the original.

**Exercise 4**: All three cultures read the same Classical Chinese text but adapt it to their native grammar. Chinese reads in original SVO order. Korean and Japanese both rearrange into SOV order and add native particles (之를 in Korean, これを in Japanese). This shows that Classical Chinese served as a shared written medium across cultures with very different spoken languages.

**Exercise 5**: 1-b, 2-c, 3-a

**Exercise 6**: For historical facts, 삼국사기 is generally more reliable as it was compiled as an official history following Chinese historiographical standards, with critical evaluation of sources. However, 삼국유사 is more valuable for understanding religious beliefs, folklore, and cultural traditions that the official history deliberately excluded. Both are essential for a complete picture.

**Exercise 7**: 火(화, fire) should be 水(수, water). The original passage is 天下莫柔弱於水 — "In all the world nothing is softer and weaker than water." Fire is neither soft nor weak. This is a 訛文(와문) error — the scribe confused 水 with 火, which look similar in certain handwriting styles.
