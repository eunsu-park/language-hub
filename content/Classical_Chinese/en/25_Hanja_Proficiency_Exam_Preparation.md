## 1. Exam Overview

### 한자능력검정시험 (漢字能力檢定試驗)

The Hanja Proficiency Test is the standard certification exam for Classical Chinese character knowledge in Korea. It is administered by the 한국어문회(韓國語文會) and other organizations.

### Key Facts

| Feature | Detail |
|---------|--------|
| Administering body | 한국어문회 (韓國語文會) |
| Frequency | Multiple times per year (approximately 4-6 sessions) |
| Format | Written exam (paper-based) |
| Duration | 50-90 minutes depending on level |
| Passing score | 70% (most levels) |
| Validity | Permanent (no expiration) |
| Levels | 8급 (easiest) to 특급 (most difficult) |
| Recognition | Academic credit, employment, university admissions |

### Exam Structure Overview

| Component | Content |
|-----------|---------|
| Section 1 | 읽기(讀) — Reading hanja characters |
| Section 2 | 쓰기(書) — Writing hanja from Korean |
| Section 3 | 뜻(訓) — Meaning of characters |
| Section 4 | 음(音) — Pronunciation/reading of characters |
| Section 5 | 어휘(語彙) — Compound words and usage |
| Section 6 | 한문(漢文) — Classical Chinese passages (higher levels) |

## 3. Question Types: Reading

### Type 1: 음독(音讀) — Read the Character

Given a hanja character, provide its Korean reading (pronunciation).

**Format**: 다음 한자의 음(音)을 쓰시오.

| Question | Answer |
|----------|--------|
| 山 | 산 (san) |
| 水 | 수 (su) |
| 學 | 학 (hak) |
| 德 | 덕 (deok) |
| 義 | 의 (ui) |

### Type 2: 훈독(訓讀) — Meaning of the Character

Given a hanja character, provide its Korean meaning.

**Format**: 다음 한자의 훈(訓)을 쓰시오.

| Question | Answer |
|----------|--------|
| 山 | 메/뫼 산 (mountain) |
| 水 | 물 수 (water) |
| 大 | 큰 대 (big) |
| 小 | 작을 소 (small) |
| 人 | 사람 인 (person) |

### Type 3: 음훈(音訓) — Full Reading

Provide both the meaning (훈) and pronunciation (음) together.

**Format**: 다음 한자의 음과 훈을 쓰시오.

| Character | 훈 (Meaning) | 음 (Reading) |
|-----------|--------------|-------------|
| 天 | 하늘 | 천 |
| 地 | 땅 | 지 |
| 日 | 날 | 일 |
| 月 | 달 | 월 |
| 花 | 꽃 | 화 |

### Strategy Tips for Reading Questions

| Tip | Description |
|-----|-------------|
| Learn radicals | Radicals often hint at meaning category |
| Group by sound | Characters with same phonetic component often share readings |
| Use flashcards | Daily review of character cards |
| Write by hand | Physical writing reinforces memory |

## 4. Question Types: Writing

### Type 4: 한자쓰기 — Write the Character

Given the Korean reading and/or meaning, write the correct hanja character.

**Format**: 다음의 음(또는 훈)에 해당하는 한자를 쓰시오.

| Reading/Meaning | Answer |
|-----------------|--------|
| 산 (mountain) | 山 |
| 물 수 (water) | 水 |
| 배울 학 (to study) | 學 |
| 글 문 (writing) | 文 |
| 나라 국 (country) | 國 |

### Type 5: 한자어 쓰기 — Write Compound Words

Given a Korean word, write it in hanja.

**Format**: 다음 한자어를 한자로 쓰시오.

| Korean | Hanja | Meaning |
|--------|-------|---------|
| 학교 | 學校 | school |
| 국가 | 國家 | nation |
| 대학 | 大學 | university |
| 자연 | 自然 | nature |
| 문화 | 文化 | culture |

### Common Pitfalls in Writing

| Pitfall | Example | Correct |
|---------|---------|---------|
| Missing strokes | 天 written without final stroke | 天 (4 strokes) |
| Wrong stroke order | 必 written left-to-right | 必 (heart-dot first) |
| Similar characters confused | 土/士, 未/末 | Check stroke lengths |
| Simplified vs. traditional | 学 (simplified) | 學 (traditional — required) |

## 5. Question Types: Meaning and Usage

### Type 6: 문맥 뜻 — Meaning in Context

Given a sentence with a highlighted hanja, choose or write the correct meaning.

**Format**: 밑줄 친 한자의 뜻으로 알맞은 것을 고르시오.

**Example**:
문: 讀**書**를 좋아합니다.
- ① 글 ② 편지 ③ 쓰다 ④ 사무실

**Answer**: ① 글 (writing, book)

### Type 7: 반의어/유의어 — Antonyms and Synonyms

**Antonym (反意語/반의어) examples**:

| Character | Reading | Antonym | Reading |
|-----------|---------|---------|---------|
| 大 | 대 (big) | 小 | 소 (small) |
| 上 | 상 (up) | 下 | 하 (down) |
| 善 | 선 (good) | 惡 | 악 (evil) |
| 生 | 생 (life) | 死 | 사 (death) |
| 古 | 고 (old) | 今 | 금 (present) |
| 明 | 명 (bright) | 暗 | 암 (dark) |
| 強 | 강 (strong) | 弱 | 약 (weak) |
| 長 | 장 (long) | 短 | 단 (short) |

**Synonym (類義語/유의어) examples**:

| Character | Reading | Synonym | Reading |
|-----------|---------|---------|---------|
| 家 | 가 (house) | 宅 | 택 (residence) |
| 道 | 도 (road) | 路 | 로 (path) |
| 始 | 시 (begin) | 初 | 초 (first) |
| 言 | 언 (speak) | 語 | 어 (word) |
| 美 | 미 (beautiful) | 麗 | 려 (lovely) |

### Type 8: 부수(部首) — Radical Identification

Identify the radical of a given character.

| Character | Radical | Radical Name | Radical Meaning |
|-----------|---------|-------------|-----------------|
| 河 | 氵(水) | 삼수변 | water |
| 林 | 木 | 나무목 | tree |
| 語 | 言 | 말씀언 | speech |
| 銅 | 金 | 쇠금 | metal |
| 花 | 艹(草) | 초두머리 | grass/plant |

## 6. Question Types: Compounds and Idioms

### Type 9: 한자어 구성 — Compound Word Formation

Identify the structure of compound words.

| Structure | Pattern | Example | Reading |
|-----------|---------|---------|---------|
| Similar meaning | A+A | 道路 | 도로 (road) |
| Opposite meaning | A+B | 大小 | 대소 (size) |
| Modifier + Head | Adj+N | 高山 | 고산 (high mountain) |
| Verb + Object | V+O | 登山 | 등산 (mountain climbing) |
| Subject + Verb | S+V | 地震 | 지진 (earthquake) |

### Type 10: 사자성어(四字成語) — Four-Character Idioms

A major component of higher-level exams.

**Format A**: Complete the idiom

| Given | Missing | Complete | Meaning |
|-------|---------|----------|---------|
| 溫故_新 | 知 | 溫故知新 | Review old, learn new |
| 臥薪_膽 | 嘗 | 臥薪嘗膽 | Endure hardship for revenge |
| _苦同樂 | 同 | 同苦同樂 | Share hardship and joy |
| 以心_心 | 傳 | 以心傳心 | Heart-to-heart communication |

**Format B**: Match idiom to meaning

| Idiom | Reading | Meaning |
|-------|---------|---------|
| 切磋琢磨 | 절차탁마 | Mutual refinement through study |
| 塞翁之馬 | 새옹지마 | Misfortune may be a blessing |
| 百折不屈 | 백절불굴 | Indomitable spirit |
| 螢雪之功 | 형설지공 | Studying under hardship |

### Type 11: 한문 독해 — Classical Chinese Passage Reading

(Higher levels: 3급 and above)

**Format**: Read the passage and answer questions.

```
子曰 學而時習之 不亦說乎
자왈 학이시습지 불역열호
```

Questions:
1. Who is speaking? (공자/孔子)
2. What does 說 mean here? (기쁘다, 즐겁다 — pleasure)
3. What is the main message? (Learning and practice bring joy)

## 7. Study Strategies

### Strategy 1: Systematic Character Learning

| Method | Description | Best For |
|--------|-------------|---------|
| Radical-based | Learn characters grouped by radical | Understanding structure |
| Frequency-based | Learn most common characters first | Practical reading ability |
| Level-based | Follow the exam level character lists | Exam preparation |
| Thematic | Learn by topic (nature, body, etc.) | Vocabulary building |

### Strategy 2: The 3-3-3 Method

| Session | Activity | Duration |
|---------|----------|----------|
| Morning | Learn 10 new characters | 30 min |
| Afternoon | Review today's + yesterday's characters | 30 min |
| Evening | Write practice + compound words | 30 min |

### Strategy 3: Active Recall vs. Passive Review

| Method | Type | Effectiveness |
|--------|------|---------------|
| Re-reading character lists | Passive | Low |
| Covering answers and recalling | Active | High |
| Writing from memory | Active | Very high |
| Teaching someone else | Active | Highest |

### Strategy 4: Spaced Repetition Schedule

| Day | Review |
|-----|--------|
| Day 1 | Learn new characters |
| Day 2 | Review Day 1 |
| Day 4 | Review Day 1-2 |
| Day 7 | Review Day 1-4 |
| Day 14 | Review Day 1-7 |
| Day 30 | Review all |

### Strategy 5: Character Components Analysis

Break complex characters into familiar components:

| Character | Components | Memory Aid |
|-----------|-----------|-----------|
| 想 (생각 상) | 相 + 心 | Appearance (相) in the heart (心) = thought |
| 休 (쉴 휴) | 人 + 木 | Person (人) against a tree (木) = rest |
| 明 (밝을 명) | 日 + 月 | Sun (日) and moon (月) = bright |
| 林 (수풀 림) | 木 + 木 | Two trees (木木) = forest |
| 好 (좋을 호) | 女 + 子 | Woman (女) and child (子) = good |

## 8. Sample Questions by Level

### 8급 Sample (50 characters)

**Q1**: 다음 한자의 음을 쓰시오.
- 一 → ( )  二 → ( )  三 → ( )

**Q2**: 다음 한자의 훈과 음을 쓰시오.
- 山 → ( )  水 → ( )  火 → ( )

**Q3**: 다음 중 뜻이 바른 것을 고르시오.
- 大: ① 크다 ② 작다 ③ 많다 ④ 적다

### 5급 Sample (500 characters)

**Q1**: 다음 한자어의 음을 쓰시오.
- 學校 → ( )  自然 → ( )  歷史 → ( )

**Q2**: 다음 빈칸에 알맞은 한자를 쓰시오.
- 東西南( ) → 北  大同小( ) → 異

**Q3**: 다음 한자의 반의어를 쓰시오.
- 善 ↔ ( )  生 ↔ ( )  古 ↔ ( )

### 3급 Sample (1,817 characters)

**Q1**: 다음 사자성어의 빈칸을 채우시오.
- 溫( )知( )  切磋( )磨

**Q2**: 다음 한문을 해석하시오.
```
學而不思則罔 思而不學則殆
```

**Q3**: 다음 중 部首가 같은 한자끼리 묶은 것을 고르시오.
- ① 河, 海, 湖  ② 河, 花, 語  ③ 山, 海, 河  ④ 木, 林, 海

### 1급 Sample (5,978 characters)

**Q1**: 다음 한문을 읽고 물음에 답하시오.
```
天下莫柔弱於水 而攻堅強者莫之能勝
```
- 이 글의 출전은? (道德經)
- 이 글의 주제는? (柔弱이 剛強을 이김)

**Q2**: 다음 한자의 음과 뜻이 다른 것을 고르시오.
- 樂: ① 풍류 악 ② 즐길 락 ③ 좋아할 요 ④ 밝을 명

## 9. Time Management and Exam Day

### Time Allocation by Level

**For 50-question exams (50 minutes)**:

| Phase | Time | Activity |
|-------|------|----------|
| Phase 1 | 20 min | Easy questions — do all you know immediately |
| Phase 2 | 20 min | Medium questions — work through carefully |
| Phase 3 | 10 min | Hard questions + review all answers |

**For 100+ question exams (90 minutes)**:

| Phase | Time | Activity |
|-------|------|----------|
| Phase 1 | 30 min | Scan all questions, answer easy ones (40-50%) |
| Phase 2 | 35 min | Work through medium-difficulty questions |
| Phase 3 | 15 min | Attempt remaining difficult questions |
| Phase 4 | 10 min | Review and check all answers |

### Exam Day Checklist

| Item | Note |
|------|------|
| Admission ticket | Print or save digitally |
| ID | Government-issued photo ID |
| Writing implements | Black pen (for writing sections), pencil (for multiple choice) |
| Watch | For time management (no smart watches) |
| Eraser | For corrections |
| Water | Stay hydrated |
| Arrive early | 30 minutes before start time |

### Common Mistakes to Avoid on Exam Day

| Mistake | Prevention |
|---------|-----------|
| Misreading questions | Read each question twice |
| Wrong answer location | Check question numbers match answer sheet |
| Incomplete characters | Write clearly with all strokes |
| Time mismanagement | Set checkpoints at 25%, 50%, 75% of time |
| Leaving blanks | Always guess — no penalty for wrong answers |
| Mixing 음 and 훈 | Double-check whether 음 or 훈 is requested |

## Final Words

Preparing for the 한자능력검정시험 is not just about passing an exam — it is about connecting with the vast literary and philosophical heritage of East Asia. Every character you learn opens a door to centuries of accumulated wisdom, from the moral teachings of Confucius to the paradoxes of Zhuangzi, from the beauty of Tang poetry to the governance principles of the Joseon Dynasty.

The key to success is consistent daily practice. As the ancients said:

```
不積跬步 無以至千里
불적규보 무이지천리

不積小流 無以成江海
불적소류 무이성강해
```

"Without accumulating small steps, one cannot travel a thousand li. Without accumulating small streams, one cannot form rivers and seas." — 순자/荀子
