## 1. Old-Style vs. Regulated Verse

### 고체시(古體詩) — Old-Style Verse

Old-style verse developed before the Tang Dynasty and follows relatively free rules.

| Feature | Description |
|---------|-------------|
| Line length | Variable (4, 5, 7, or irregular) |
| Number of lines | No fixed limit |
| Tonal rules | None strictly required |
| Parallelism | Optional |
| Rhyme | Required but flexible |
| Period | Pre-Tang and continuing |

### 근체시(近體詩) — Regulated (New-Style) Verse

Regulated verse crystallized during the Tang Dynasty with strict formal requirements.

| Feature | Description |
|---------|-------------|
| Line length | Fixed: 5 or 7 characters |
| Number of lines | Fixed: 4 (절구) or 8 (율시) |
| Tonal rules | Strict 평측 patterns |
| Parallelism | Required in couplets 2 and 3 (율시) |
| Rhyme | Strict, from approved rhyme tables |
| Period | Tang Dynasty onward |

### Comparison Chart

| Aspect | 고체시 (古體詩) | 근체시 (近體詩) |
|--------|-----------------|-----------------|
| Korean term | 고체시 (old form) | 근체시 (new form) |
| Freedom | High | Low |
| Difficulty to compose | Lower | Higher |
| Examples | 詩經, 楚辭, 古樂府 | 절구, 율시 |
| Notable practitioners | 도연명(陶淵明), 이백(李白) | 두보(杜甫), 왕유(王維) |

## 2. Lüshi: Eight-Line Regulated Verse

### 율시(律詩) Structure

A 율시 consists of four couplets (聯/련), each with a specific function:

| Couplet | Name | Hanja | Function |
|---------|------|-------|----------|
| 1st (lines 1-2) | 수련 | 首聯 | Opening couplet: introduces theme/setting |
| 2nd (lines 3-4) | 함련 | 頷聯 | Chin couplet: develops with parallelism |
| 3rd (lines 5-6) | 경련 | 頸聯 | Neck couplet: develops with parallelism |
| 4th (lines 7-8) | 미련 | 尾聯 | Tail couplet: conclusion/reflection |

### Rules of 율시

| Rule | Requirement |
|------|-------------|
| Line length | All lines must be 5 or 7 characters |
| Total lines | Exactly 8 |
| Parallelism | 함련(頷聯) and 경련(頸聯) MUST be parallel |
| Rhyme | Lines 2, 4, 6, 8 must rhyme; line 1 may optionally rhyme |
| Tonal pattern | Must follow 평측 rules |
| No repeated rhyme | Each rhyming character must be different |

### Parallelism Requirements

In the 함련 and 경련, corresponding positions must match:

| Position | Line A | Line B | Match Type |
|----------|--------|--------|-----------|
| 1st char | Noun | Noun | Same word class |
| 2nd char | Noun | Noun | Same word class |
| 3rd char | Verb | Verb | Same word class |
| 4th char | Adj | Adj | Same word class |
| 5th char | Noun | Noun | Same word class |
| Tone | 평/측 | 측/평 | Opposite |

## 3. Advanced Pingze Patterns

### 오언율시 (五言律詩) Tonal Pattern

Using ○ for 평(level) and ● for 측(oblique):

```
Type A:
Line 1: ○○●●○  (rhyme optional)
Line 2: ●●●○○  (rhyme)
Line 3: ●●○○●
Line 4: ○○●●○  (rhyme)
Line 5: ○○○●●
Line 6: ●●●○○  (rhyme)
Line 7: ●●○○●
Line 8: ○○●●○  (rhyme)
```

### 칠언율시 (七言律詩) Tonal Pattern

```
Type A:
Line 1: ●●○○●●○  (rhyme optional)
Line 2: ○○●●●○○  (rhyme)
Line 3: ○○●●○○●
Line 4: ●●○○●●○  (rhyme)
Line 5: ●●○○○●●
Line 6: ○○●●●○○  (rhyme)
Line 7: ○○●●○○●
Line 8: ●●○○●●○  (rhyme)
```

### Key Rules

| Rule | Hanja | Reading | Description |
|------|-------|---------|-------------|
| 1-3-5 may vary | 一三五不論 | 일삼오불론 | Positions 1, 3, 5 have tonal flexibility |
| 2-4-6 must match | 二四六分明 | 이사육분명 | Positions 2, 4, 6 must follow the pattern |
| No four same tones | 孤平 | 고평 | Avoid isolated level tones |
| Contrast between lines | 對 | 대 | Adjacent lines in a couplet contrast tones |
| Continuation across couplets | 粘 | 점 | First line of next couplet matches tone of previous couplet's last line |

## 4. Rhyme Schemes: Yayun

### 압운(押韻) — Rhyming

In regulated verse, the rhyme (韻/운) follows strict conventions.

| Rule | Description |
|------|-------------|
| Rhyme position | End of lines 2, 4, 6, 8 (and optionally line 1) |
| Rhyme category | Must be from the same 운부(韻部, rhyme group) |
| Tone of rhyme | Must be 평성(平聲, level tone) in regulated verse |
| No repetition | Each rhyming character must be unique |

### Major Rhyme Groups (운부/韻部)

| Group | Example Characters | Korean Readings |
|-------|-------------------|-----------------|
| 東韻 (동운) | 東, 風, 中, 空, 紅 | 동, 풍, 중, 공, 홍 |
| 陽韻 (양운) | 陽, 光, 鄉, 霜, 長 | 양, 광, 향, 상, 장 |
| 眞韻 (진운) | 眞, 人, 春, 新, 塵 | 진, 인, 춘, 신, 진 |
| 先韻 (선운) | 天, 年, 前, 煙, 然 | 천, 년, 전, 연, 연 |
| 魚韻 (어운) | 魚, 書, 居, 車, 初 | 어, 서, 거, 거, 초 |
| 尤韻 (우운) | 樓, 流, 秋, 愁, 舟 | 루, 류, 추, 수, 주 |

## 5. Du Fu: Climbing High

### 두보(杜甫): 登高 (등고, Climbing High)

This poem is considered by many to be the greatest 칠언율시 ever written.

```
風急天高猿嘯哀    풍급천고원소애
渚清沙白鳥飛回    저청사백조비회
無邊落木蕭蕭下    무변낙목소소하
不盡長江滾滾來    부진장강곤곤래
萬里悲秋常作客    만리비추상작객
百年多病獨登臺    백년다병독등대
艱難苦恨繁霜鬢    간난고한번상빈
潦倒新停濁酒杯    료도신정탁주배
```

### Couplet-by-Couplet Analysis

**수련(首聯): Lines 1-2 — Setting the scene**

| Line | Characters | Translation |
|------|-----------|-------------|
| 風急天高猿嘯哀 | 풍급천고원소애 | Wind sharp, sky high, apes wail in sorrow |
| 渚清沙白鳥飛回 | 저청사백조비회 | Islet clear, sand white, birds circle in flight |

Key vocabulary:

| Character | Reading | Meaning |
|-----------|---------|---------|
| 急 | 급 (geup) | sharp, urgent |
| 猿 | 원 (won) | ape, gibbon |
| 嘯 | 소 (so) | to howl, wail |
| 渚 | 저 (jeo) | islet, sandbar |
| 回 | 회 (hoe) | to circle, return |

**함련(頷聯): Lines 3-4 — Parallel development**

| Line | Characters | Translation |
|------|-----------|-------------|
| 無邊落木蕭蕭下 | 무변낙목소소하 | Boundless falling leaves rustle down |
| 不盡長江滾滾來 | 부진장강곤곤래 | The endless Yangtze rolls and rolls on |

Parallelism analysis:

| Position | Line 3 | Line 4 |
|----------|--------|--------|
| 1-2 | 無邊 (무변, boundless) | 不盡 (부진, endless) |
| 3-4 | 落木 (낙목, falling leaves) | 長江 (장강, Yangtze River) |
| 5-6 | 蕭蕭 (소소, rustling) | 滾滾 (곤곤, rolling) |
| 7 | 下 (하, down) | 來 (래, coming) |

**경련(頸聯): Lines 5-6 — Personal reflection**

| Line | Characters | Translation |
|------|-----------|-------------|
| 萬里悲秋常作客 | 만리비추상작객 | Ten thousand li away, mourning autumn, always a guest |
| 百年多病獨登臺 | 백년다병독등대 | A hundred years, much illness, alone climbing the terrace |

| Position | Line 5 | Line 6 |
|----------|--------|--------|
| 1-2 | 萬里 (만리, vast distance) | 百年 (백년, lifetime) |
| 3-4 | 悲秋 (비추, mourn autumn) | 多病 (다병, much illness) |
| 5-6 | 常作 (상작, always being) | 獨登 (독등, alone climbing) |
| 7 | 客 (객, guest) | 臺 (대, terrace) |

**미련(尾聯): Lines 7-8 — Conclusion**

| Line | Characters | Translation |
|------|-----------|-------------|
| 艱難苦恨繁霜鬢 | 간난고한번상빈 | Hardships and bitter regrets multiply the frost in my temples |
| 潦倒新停濁酒杯 | 료도신정탁주배 | Worn and weary, I newly cease the cup of turbid wine |

## 6. Li Bai: Bring in the Wine

### 이백(李白): 將進酒 (장진주, Bring in the Wine)

This is a 고체시(古體詩) — old-style verse with irregular structure, showcasing Li Bai's exuberant, uninhibited spirit.

```
君不見黃河之水天上來    군불견황하지수천상래
奔流到海不復回          분류도해불복회
君不見高堂明鏡悲白髮    군불견고당명경비백발
朝如靑絲暮成雪          조여청사모성설
```

### Analysis of Opening Lines

**Line 1: 君不見黃河之水天上來**

| Character | Reading | Meaning |
|-----------|---------|---------|
| 君 | 군 (gun) | you (respectful) |
| 不見 | 불견 (bulgyeon) | do you not see |
| 黃河 | 황하 (hwangha) | Yellow River |
| 之 | 지 (ji) | (possessive) |
| 水 | 수 (su) | water |
| 天上 | 천상 (cheonsang) | from heaven above |
| 來 | 래 (rae) | coming |

"Do you not see the waters of the Yellow River, descending from heaven above?"

**Line 2: 奔流到海不復回**

| Character | Reading | Meaning |
|-----------|---------|---------|
| 奔流 | 분류 (bunryu) | rushing, racing flow |
| 到 | 도 (do) | reaching, arriving at |
| 海 | 해 (hae) | sea |
| 不復 | 불복 (bulbok) | never again |
| 回 | 회 (hoe) | to return |

"Racing to the sea, never to return again!"

**Lines 3-4**:

"Do you not see, in the high hall's bright mirror, the sorrow of white hair — / In the morning like blue-black silk, by evening turned to snow."

### Key Themes

| Theme | Expression | Meaning |
|-------|-----------|---------|
| Time's passage | 朝如靑絲暮成雪 | Youth to old age in a single day |
| Carpe diem | 人生得意須盡歡 | When life goes well, exhaust your joy |
| Defiance of fate | 天生我材必有用 | Heaven bore my talents; they must be used |
| Transience | 黃河之水天上來 | Even the great river flows away |

### Later Famous Lines

```
天生我材必有用    천생아재필유용
千金散盡還復來    천금산진환복래
```

"Heaven bore my talents — they will surely be of use; / A thousand gold pieces scattered will all come back again."

| Character | Reading | Meaning |
|-----------|---------|---------|
| 天生 | 천생 (cheonsaeng) | heaven-born |
| 材 | 재 (jae) | talent, material |
| 必 | 필 (pil) | certainly, must |
| 千金 | 천금 (cheongeum) | a thousand gold |
| 散 | 산 (san) | to scatter |
| 盡 | 진 (jin) | completely |
| 還 | 환 (hwan) | again |

## 7. Bai Juyi: Song of Everlasting Sorrow

### 백거이(白居易): 長恨歌 (장한가, Song of Everlasting Sorrow)

This narrative poem of 840 characters tells the tragic love story of Emperor Xuanzong (현종/玄宗) and Yang Guifei (양귀비/楊貴妃).

### Famous Opening

```
漢皇重色思傾國    한황중색사경국
御宇多年求不得    어우다년구부득
```

| Line | Translation |
|------|-------------|
| 漢皇重色思傾國 | The Han emperor prized beauty, longing for a kingdom-toppling beauty |
| 御宇多年求不得 | He ruled the realm for many years, seeking but never finding |

Key vocabulary:

| Character | Reading | Meaning |
|-----------|---------|---------|
| 皇 | 황 (hwang) | emperor |
| 重 | 중 (jung) | to value, prize |
| 色 | 색 (saek) | beauty, complexion |
| 傾國 | 경국 (gyeongguk) | to topple a nation (extreme beauty) |
| 御 | 어 (eo) | to rule, imperial |
| 宇 | 우 (u) | realm, universe |

### The Most Quoted Lines

```
在天願作比翼鳥    재천원작비익조
在地願爲連理枝    재지원위련리지
```

| Line | Translation |
|------|-------------|
| 在天願作比翼鳥 | In heaven, we wish to be birds flying wing to wing |
| 在地願爲連理枝 | On earth, we wish to be trees with intertwined branches |

| Character | Reading | Meaning |
|-----------|---------|---------|
| 願 | 원 (won) | to wish |
| 比翼 | 비익 (biik) | wing to wing |
| 連理 | 련리 (ryeonri) | intertwined branches |
| 枝 | 지 (ji) | branch |

### Closing Lines

```
天長地久有時盡    천장지구유시진
此恨綿綿無絕期    차한면면무절기
```

"Heaven and earth may someday end; / This sorrow stretches on and on, without end."

## 8. The Ci Lyric Form

### 사(詞, ci) — Lyric Poetry

The 사(詞) developed during the late Tang and flourished in the Song Dynasty (960-1279). Unlike regulated verse, 사 are written to specific musical tune patterns.

### Characteristics of 사(詞)

| Feature | Description |
|---------|-------------|
| Alternate name | 장단구(長短句, lines of varying length) |
| Music | Originally set to specific tunes (詞牌/사패) |
| Line length | Irregular — lines of 1 to 11 characters |
| Structure | Divided into stanzas (闋/궐) |
| Rhyme | Follows the tune pattern |
| Subjects | Love, nature, history, philosophy |

### 사패(詞牌) — Tune Titles

| Tune Title | Hanja | Reading | Lines | Style |
|-----------|-------|---------|-------|-------|
| 如夢令 | 여몽령 | yeomonryeong | 33 chars | Short, lyrical |
| 菩薩蠻 | 보살만 | bosalman | 44 chars | Romantic |
| 水調歌頭 | 수조가두 | sujogadu | 95 chars | Grand, philosophical |
| 念奴嬌 | 념노교 | nyeomnogyo | 100 chars | Extended, dramatic |
| 滿江紅 | 만강홍 | manganghong | 93 chars | Heroic, patriotic |

### Example: 소식(蘇軾) — 水調歌頭 (수조가두)

```
明月幾時有    명월기시유
把酒問靑天    파주문청천
不知天上宮闕    부지천상궁궐
今夕是何年    금석시하년
```

| Line | Translation |
|------|-------------|
| 明月幾時有 | When did the bright moon first appear? |
| 把酒問靑天 | Holding wine, I ask the blue sky |
| 不知天上宮闕 | I do not know in the heavenly palace |
| 今夕是何年 | What year tonight might be |

### Famous closing:

```
但願人長久    단원인장구
千里共嬋娟    천리공선연
```

"I only wish that we may long endure / And share this lovely moonlight, though a thousand li apart."

## 9. Qichengzhuanjie: Poetic Structure

### 기승전결(起承轉結) — The Four-Phase Structure

This structural principle governs not only poetry but also essays and narratives across East Asian literature.

| Phase | Hanja | Reading | Function | Poetic Role |
|-------|-------|---------|----------|-------------|
| Opening | 起 | 기 (gi) | Introduce | Set scene, present topic |
| Development | 承 | 승 (seung) | Continue | Elaborate, develop opening |
| Turn | 轉 | 전 (jeon) | Shift | Change perspective, create tension |
| Conclusion | 結 | 결 (gyeol) | Resolve | Emotional resolution, reflection |

### Application to 靜夜思 (정야사)

| Phase | Line | Function |
|-------|------|----------|
| 起 | 床前明月光 | Sets the scene: moonlight before the bed |
| 承 | 疑是地上霜 | Develops: could it be frost? (visual similarity) |
| 轉 | 舉頭望明月 | Turn: raises head, sees it IS the moon |
| 結 | 低頭思故鄉 | Resolution: lowers head, thinks of home |

### Application to 登鸛雀樓

| Phase | Line | Function |
|-------|------|----------|
| 起 | 白日依山盡 | Scene: sun sinking behind mountains |
| 承 | 黃河入海流 | Development: river flowing to sea |
| 轉 | 欲窮千里目 | Turn: desire to see further |
| 結 | 更上一層樓 | Resolution: climb higher (metaphor for aspiration) |

### Emotional Arc

```
起 (Introduction)  →  승 (Development)  →  轉 (Turn)  →  結 (Resolution)
   Calm/Neutral         Building              Surprise        Deepened emotion
```

**Exercise 6**: 猿(원, ape), 渚(저, islet), 嘯(소, howl), 潦(료, flooded/worn), 濁(탁, turbid), 鬢(빈, temples/sideburns), 闕(궐, palace gate), 嬋(선, beautiful)

**Exercise 7**: "In heaven, we wish to be birds flying wing to wing; / On earth, we wish to be trees with intertwined branches."
