## 1. 사람 한자: 사회적 역할

### 1.1 기본 사람 한자

| 한자 | 음독 | 훈독 | 획수 | 형성 원리 | 비고 |
|------|------|------|------|-----------|------|
| 人 | 인 (in) | 사람 인 | 2 | 象形: 걸어가는 사람의 옆모습 | 가장 기본적인 "사람" 한자 |
| 女 | 여 (yeo) | 계집 녀 | 3 | 象形: 팔을 교차하고 무릎 꿇은 사람 | 많은 한자의 부수로 사용 |
| 子 | 자 (ja) | 아들 자 | 3 | 象形: 포대기에 싸인 팔을 뻗은 아기 | "스승"의 뜻도 있음 (孔子, 孟子) |
| 男 | 남 (nam) | 사내 남 | 7 | 會意: 田 (밭) + 力 (힘) | "밭에서 힘을 쓰는 사람" |
| 夫 | 부 (bu) | 지아비 부 | 4 | 指事: 大 (사람) + 一 (비녀) | 비녀를 꽂은 성인 남자 (성인식) |
| 婦 | 부 (bu) | 며느리 부 | 11 | 會意: 女 (여자) + 帚 (빗자루) | 결혼한 여성 |

### 1.2 人 한자의 변형

人 한자는 위치에 따라 여러 형태로 나타납니다:

| 형태 | 위치 | 명칭 | 예시 |
|------|------|------|------|
| 人 | 독립 또는 하단 | 사람인 | 人, 大人 |
| 亻 | 좌측 (부수) | 사람인변 | 休 (쉬다), 仁 (어짐) |
| 入 | — | 들입 | 다른 글자: "들어가다" |

### 2.3 효도 (孝)

孝 (효, hyo) — 효도 — 는 老 (늙은이)가 위에, 子 (자식)가 아래에 위치하여 자식이 어른을 받드는 모습을 상징합니다. 이 개념은 유교 윤리의 핵심입니다.

> **孝 (효) — 효도**
>
> - **구조**: 老 (늙은이/어른)가 위 + 子 (자식)가 아래
> - **의미**: 자식이 어른을 받들고/부양하는 모습
>
> 주요 숙어:
> - 孝道 (효도) — 효도의 도리
> - 孝子 (효자) — 효도하는 아들; 효성스러운 자녀
> - 孝女 (효녀) — 효도하는 딸
> - 不孝 (불효) — 불효; 효도를 다하지 못함
>
> 고전 인용:
> 百善孝爲先 (백선효위선)
> "모든 선행 중 효도가 으뜸이다."

### 5.2 심(心) — 마음의 연결

한문 사상에서 心 (심)은 물리적 심장과 정신/의식을 모두 나타냅니다. 이는 心/忄을 부수로 사용하는 많은 감정·인지 한자에 반영되어 있습니다:

| 한자 | 음독 | 훈독 | 범주 | 心과의 관계 |
|------|------|------|------|------------|
| 思 | 사 (sa) | 생각할 사 | 인지 | 사고 — 心의 활동 |
| 想 | 상 (sang) | 생각할 상 | 인지 | 상상 — 心 + 相 (모습) |
| 情 | 정 (jeong) | 뜻 정 | 감정 | 감정 — 心의 내적 상태 |
| 性 | 성 (seong) | 성품 성 | 성격 | 본성 — 心의 타고난 자질 |
| 愛 | 애 (ae) | 사랑 애 | 감정 | 사랑 — 글자 안에 心이 있음 |
| 忠 | 충 (chung) | 충성 충 | 덕목 | 충성 — 中 (가운데) + 心 (마음) |
| 怒 | 노 (no) | 성낼 노 | 감정 | 분노 — 奴 (성부) + 心 |
| 悲 | 비 (bi) | 슬플 비 | 감정 | 슬픔 — 非 (아니다) + 心 |
| 恐 | 공 (gong) | 두려울 공 | 감정 | 공포 — 巩 (성부) + 心 |

| 書 | 서 (seo) | 글 서 | 쓰다, 책 | 종이 위에 붓을 든 손 |
| 讀 | 독 (dok) | 읽을 독 | 읽다 | 言 (말) + 賣 (성부) |
| 問 | 문 (mun) | 물을 문 | 묻다, 질문 | 口 (입)가 門 (문) 옆 — 문에서 묻다 |
| 答 | 답 (dap) | 대답할 답 | 대답하다 | 竹 (대나무) + 合 (합하다) |
| 知 | 지 (ji) | 알 지 | 알다 | 矢 (화살) + 口 (입) — 말이 화살처럼 빠름 |
| 敎 | 교 (gyo) | 가르칠 교 | 가르치다 | 복잡한 구성: 손 + 아이 + 동작 = 가르치는 모습 |
| 話 | 화 (hwa) | 말씀 화 | 말, 이야기 | 言 (말) + 舌 (혀) |

### 8.2 소통 숙어

| 숙어 | 독음 | 의미 | 분석 |
|------|------|------|------|
| 言語 | 언어 | 언어 | 言 (말) + 語 (어) |
| 問答 | 문답 | 문답; 대화 | 問 (묻다) + 答 (대답하다) |
| 讀書 | 독서 | 독서 (책 읽기) | 讀 (읽다) + 書 (책) |
| 見聞 | 견문 | 견문; 보고 듣는 것 | 見 (보다) + 聞 (듣다) |
| 敎育 | 교육 | 교육 | 敎 (가르치다) + 育 (기르다) |
| 知識 | 지식 | 지식 | 知 (알다) + 識 (분별하다) |
| 書生 | 서생 | 서생; 학자 (고전적 표현) | 書 (책) + 生 (사람) |
| 問學 | 문학 | 물어서 배움 | 問 (묻다) + 學 (배우다) |

### 8.3 言 (말씀 부수)의 힘

言/訁 부수를 가진 한자는 가장 크고 중요한 한자 가족 중 하나를 이룹니다:

| 한자 | 음독 | 훈독 | 의미 |
|------|------|------|------|
| 語 | 어 (eo) | 말씀 어 | 언어, 말 |
| 話 | 화 (hwa) | 말씀 화 | 이야기, 말 |
| 記 | 기 (gi) | 기록할 기 | 기록하다 |
| 論 | 논 (non) | 논할 논 | 논하다, 이론 |
| 說 | 설 (seol) | 말씀 설 | 설명하다, 이론 |
| 詩 | 시 (si) | 시 시 | 시, 시가 |
| 認 | 인 (in) | 알 인 | 인정하다 |
| 誠 | 성 (seong) | 정성 성 | 성실, 진실 |
| 謝 | 사 (sa) | 사례할 사 | 감사하다, 사과하다 |

## 9. 사람, 신체, 동작 숙어

### 9.1 인간 활동 숙어

| 숙어 | 독음 | 직역 | 의미 |
|------|------|------|------|
| 人生 | 인생 | 사람 + 삶 | 인생 |
| 人心 | 인심 | 사람 + 마음 | 인심; 인정 |
| 民心 | 민심 | 백성 + 마음 | 민심 |
| 王子 | 왕자 | 왕 + 아들 | 왕자 |
| 女王 | 여왕 | 여자 + 왕 | 여왕 |
| 手書 | 수서 | 손 + 쓰다 | 수서; 친필 |
| 口頭 | 구두 | 입 + 머리 | 구두; 구두로 |
| 目下 | 목하 | 눈 + 아래 | 목하; 현재 |
| 心身 | 심신 | 마음 + 몸 | 심신 |
| 耳目 | 이목 | 귀 + 눈 | 이목; 관심 |

### 9.2 사자성어 (四字成語)

| 성어 | 독음 | 직역 | 의미 |
|------|------|------|------|
| 人山人海 | 인산인해 | 사람 산, 사람 바다 | 인파가 가득한 모습 |
| 手不釋卷 | 수불석권 | 손에서 책을 놓지 않다 | 책을 손에서 놓지 않음; 독서에 열중 |
| 口是心非 | 구시심비 | 입은 맞다, 마음은 아니다 | 겉과 속이 다름; 언행불일치 |
| 一目了然 | 일목요연 | 한 눈에 분명히 | 한눈에 알 수 있음 |
| 言行一致 | 언행일치 | 말과 행동이 하나 | 말과 행동이 일치함 |
| 見物生心 | 견물생심 | 물건을 보면 마음이 생김 | 보면 욕심이 남 |
| 父子有親 | 부자유친 | 부자 간에 친함이 있다 | 유교의 오륜: 부자간의 친함 |

| 문장 | 독음 | 축자 분석 | 번역 |

| 學生讀書 | 학생독서 | 배우다-사람-읽다-책 | 학생이 책을 읽는다 |

## 10. 연습 문제

### 연습 1: 사람 한자 인식

각 한자의 한국어 독음과 의미를 쓰시오:

```
1. 人
2. 女
3. 子
4. 父
5. 母
6. 兄
7. 弟
8. 王
9. 臣
10. 民
```

<details>
<summary>정답 확인</summary>

```
1. 人 = 인 (in), 사람 인 — 사람
2. 女 = 여 (yeo), 계집 녀 — 여자
3. 子 = 자 (ja), 아들 자 — 아이, 아들
4. 父 = 부 (bu), 아비 부 — 아버지
5. 母 = 모 (mo), 어미 모 — 어머니
6. 兄 = 형 (hyeong), 맏 형 — 형/오빠
7. 弟 = 제 (je), 아우 제 — 아우/동생
8. 王 = 왕 (wang), 임금 왕 — 왕
9. 臣 = 신 (sin), 신하 신 — 신하, 관리
10. 民 = 민 (min), 백성 민 — 백성, 시민
```

</details>

### 연습 2: 신체와 동작 한자

각 한자를 의미와 연결하시오:

```
A열 (한자)               B열 (의미)
1. 口                    a. 보다
2. 手                    b. 마음
3. 足                    c. 글/책
4. 目                    d. 입
5. 耳                    e. 귀
6. 心                    f. 발
7. 見                    g. 손
8. 聞                    h. 눈
9. 言                    i. 듣다
10. 書                   j. 말/말씀
```

<details>
<summary>정답 확인</summary>

```
1-d: 口 = 입 (입 구)
2-g: 手 = 손 (손 수)
3-f: 足 = 발 (발 족)
4-h: 目 = 눈 (눈 목)
5-e: 耳 = 귀 (귀 이)
6-b: 心 = 마음 (마음 심)
7-a: 見 = 보다 (볼 견)
8-i: 聞 = 듣다 (들을 문)
9-j: 言 = 말/말씀 (말씀 언)
10-c: 書 = 글/책 (글 서)
```

</details>

### 연습 3: 숙어 형성

한자를 결합하여 올바른 숙어를 만들고 의미를 쓰시오:

```
1. 父 + 母 = ?
2. 兄 + 弟 = ?
3. 學 + 生 = ?
4. 讀 + 書 = ?
5. 問 + 答 = ?
6. 出 + 口 = ?
7. 人 + 生 = ?
8. 食 + 事 = ?
9. 子 + 女 = ?
10. 生 + 死 = ?
```

<details>
<summary>정답 확인</summary>

```
1. 父母 (부모) — 부모
2. 兄弟 (형제) — 형제; 동기
3. 學生 (학생) — 학생
4. 讀書 (독서) — 독서 (책 읽기)
5. 問答 (문답) — 문답; 대화
6. 出口 (출구) — 출구
7. 人生 (인생) — 인생
8. 食事 (식사) — 식사
9. 子女 (자녀) — 자녀 (아들과 딸)
10. 生死 (생사) — 생사
```

</details>

### 연습 4: 이동 동사

빈칸에 알맞은 한자를 채우시오:

```
1. ___入 (출입, 드나듦): 出 + ?
2. ___校 (입교, 학교에 들어감): ? + 校
3. ___國 (귀국, 나라로 돌아감): ? + 國
4. ___立 (독립, 홀로 섬): 獨 + ?
5. ___行 (보행, 걸어감): 步 + ?
```

<details>
<summary>정답 확인</summary>

```
1. 出入 (출입): 出 + 入 (나가다 + 들어가다 = 드나듦)
2. 入校 (입교): 入 + 校 (들어가다 + 학교 = 학교에 들어감)
3. 歸國 (귀국): 歸 + 國 (돌아가다 + 나라 = 나라로 돌아감)
4. 獨立 (독립): 獨 + 立 (홀로 + 서다 = 독립)
5. 步行 (보행): 步 + 行 (걸음 + 가다 = 걸어감)
```

</details>

### 연습 5: 사자성어 해석

각 성어의 의미를 설명하시오:

```
1. 人山人海 (인산인해)
2. 言行一致 (언행일치)
3. 父子有親 (부자유친)
4. 手不釋卷 (수불석권)
5. 口是心非 (구시심비)
```

<details>
<summary>정답 확인</summary>

```
1. 人山人海 (인산인해)
   직역: 사람-산, 사람-바다
   의미: 산과 바다처럼 많은 사람의 무리; 엄청난 인파

2. 言行一致 (언행일치)
   직역: 말-행동-하나-일치
   의미: 말과 행동이 같음; 언행이 한결같음

3. 父子有親 (부자유친)
   직역: 아버지-아들-있다-친함
   의미: 부모와 자식 사이에 친함이 있어야 함
   (유교 오륜 중 하나)

4. 手不釋卷 (수불석권)
   직역: 손-아니다-놓다-책
   의미: 늘 책을 읽음; 독서에 몰두하는 사람

5. 口是心非 (구시심비)
   직역: 입-맞다-마음-아니다
   의미: 입으로는 그렇다 하면서 마음은 다름; 위선
```

</details>

### 연습 6: 간단한 문장 읽기

다음 한문 구절을 번역하시오:

```
1. 子曰 (자왈)
2. 人皆有心 (인개유심)
3. 王愛民 (왕애민)
4. 兄弟有信 (형제유신)
5. 學而不止 (학이불지)
```

<details>
<summary>정답 확인</summary>

```
1. 子曰 (자왈) — 공자께서 말씀하셨다
   子 = 스승/선생님, 曰 = 말씀하다

2. 人皆有心 (인개유심) — 모든 사람에게 마음이 있다
   人 = 사람, 皆 = 모두, 有 = 있다, 心 = 마음

3. 王愛民 (왕애민) — 왕이 백성을 사랑한다
   王 = 왕, 愛 = 사랑하다, 民 = 백성

4. 兄弟有信 (형제유신) — 형제 사이에 믿음이 있다
   兄弟 = 형제, 有 = 있다, 信 = 믿음

5. 學而不止 (학이불지) — 배우되 멈추지 않는다
   學 = 배우다, 而 = 그리고/그러나, 不 = 아니다, 止 = 멈추다
```

</details>
