## 1. 작문의 원리

### 좋은 한문 글쓰기의 다섯 가지 덕목

| 덕목 | 한자 | 한국어 | 설명 |
|------|------|--------|------|
| 간결함 | 簡 | 간 (gan) | 적은 말로 많은 뜻을 전함 |
| 명확함 | 明 | 명 (myeong) | 뜻이 바로 드러나야 함 |
| 우아함 | 雅 | 아 (a) | 세련된 어휘와 구조 사용 |
| 힘 | 勁 | 경 (gyeong) | 글에 기운과 추동력이 있어야 함 |
| 깊이 | 深 | 심 (sim) | 표면 아래에 의미의 층위 |

### 핵심 격언

```
辭達而已矣
사달이이의
```

"말은 뜻을 전달하면 족하다" — 공자/孔子, 논어/論語.

| 글자 | 독음 | 의미 |
|------|------|------|
| 辭 | 사 (sa) | 말, 문사 |
| 達 | 달 (dal) | 이르다, 전달하다 |
| 而已 | 이이 (ii) | 그뿐이다 |
| 矣 | 의 (ui) | (종결 어조사) |

이것은 글이 꾸밈없이 소박해야 한다는 뜻이 아닙니다 — 모든 말이 전달이라는 목적에 부합해야 한다는 뜻입니다. 의미 없는 장식은 공허합니다.

### 작문의 도구

| 도구 | 설명 | 예시 |
|------|------|------|
| 구체적 명사 | 구체적이고 생생한 말을 사용 | 松 (소나무), 木 (나무) 대신 |
| 활동적 동사 | 역동적 동사 선호 | 奔 (달리다), 去 (가다) 대신 |
| 대구 구조 | 구절의 균형 맞추기 | A↔B 짝짓기 |
| 허사 | 리듬과 논리를 조절 | 而, 則, 所, 之 |
| 전고 | 인용으로 의미에 층위를 더함 | 전고(典故) |

## 2. 구성 요소: 어휘와 구

### 단어 형성 패턴

한문 단어는 주로 단음절이지만, 합성어는 특정 패턴을 따릅니다:

| 패턴 | 구조 | 예시 | 독음 | 의미 |
|------|------|------|------|------|
| 유의어 합성 | A+A | 道路 | 도로 | 길 |
| 반의어 합성 | A+B | 大小 | 대소 | 크기 (큼-작음) |
| 수식어 + 핵 | 형+명 | 明月 | 명월 | 밝은 달 |
| 동사 + 목적어 | 동+목 | 讀書 | 독서 | 읽다 |
| 주어 + 동사 | 주+동 | 日出 | 일출 | 해돋이 |
| 동사 + 보어 | 동+보 | 看見 | 간견 | 보다 (살펴-인식) |

### 주요 서술 유형

| 유형 | 패턴 | 예시 | 해석 |
|------|------|------|------|
| 동작 | 주 + 동 + 목 | 人讀書 (인독서) | 사람이 책을 읽다 |
| 묘사 | 주 + 형 | 山高 (산고) | 산이 높다 |
| 판단 | 주 + 也 | 仁者人也 (인자인야) | 어진 사람은 사람이다 |
| 존재 | 有 + 명 | 有人 (유인) | 사람이 있다 |
| 부정 | 無 + 명 | 無人 (무인) | 사람이 없다 |

### 유용한 접속사

| 접속사 | 독음 | 기능 | 예시 |
|--------|------|------|------|
| 而 | 이 | 그리고/그러나 | 學而知 (배우고 알다) |
| 則 | 즉 | 그러면 | 學則知 (배우면 알다) |
| 故 | 고 | 그러므로 | 故曰 (그러므로 이르기를) |
| 然 | 연 | 그러나 | 然而 (그러나) |
| 蓋 | 개 | 대저/왜냐하면 | 蓋因 (이는 ~때문이다) |
| 夫 | 부 | 대저 | 夫學者 (대저 배우는 자는) |
| 且 | 차 | 게다가 | 且夫 (더구나) |
| 雖 | 수 | 비록 | 雖然 (비록 그러하나) |

## 3. 문장 구성

### 수준 1: 단순 문장

가장 기본적인 패턴으로 시작합니다:

| 패턴 | 공식 | 예시 | 해석 |
|------|------|------|------|
| 주 + 동 | 주어 + 동사 | 鳥飛 (조비) | 새가 난다 |
| 주 + 동 + 목 | 주어 + 동사 + 목적어 | 人讀書 (인독서) | 사람이 책을 읽다 |
| 주 + 형 | 주어 + 형용사 | 天高 (천고) | 하늘이 높다 |
| 有 + 명 | 존재 | 山有木 (산유목) | 산에 나무가 있다 |
| 無 + 명 | 비존재 | 路無人 (로무인) | 길에 사람이 없다 |

### 수준 2: 수식 문장

수식어와 허사를 추가합니다:

| 패턴 | 예시 | 해석 |
|------|------|------|
| 형 + 명 + 동 | 白鳥飛 (백조비) | 흰 새가 난다 |
| 주 + 부 + 동 | 鳥高飛 (조고비) | 새가 높이 난다 |
| 주 + 동 + 형 + 목 | 人讀古書 (인독고서) | 사람이 옛 책을 읽다 |
| 주 + 부정 + 동 | 人不知 (인부지) | 사람들이 알지 못한다 |
| 명之명 | 天之道 (천지도) | 하늘의 도 |

### 수준 3: 복합 문장

허사를 사용하여 절을 결합합니다:

| 접속 | 예시 | 해석 |
|------|------|------|
| 而 (순접) | 起而行 (기이행) | 일어나서 행하다 |
| 而 (역접) | 學而不思 (학이불사) | 배우되 생각하지 않다 |
| 則 (조건) | 學則知 (학즉지) | 배우면 알다 |
| 故 (인과) | 不學 故不知 (불학 고부지) | 배우지 않으니, 그러므로 알지 못하다 |
| 雖...然 | 雖貧然樂 (수빈연락) | 비록 가난하나 즐겁다 |

### 수준 4: 다절 문장

| 유형 | 예시 | 해석 |
|------|------|------|
| 대구 | 學而知 知而行 (학이지 지이행) | 배우고 알며, 알고 행한다 |
| 점진 | 修身而後齊家 齊家而後治國 (수신이후제가 제가이후치국) | 자신을 닦은 뒤에 집안을 가지런히 하고, 집안을 가지런히 한 뒤에 나라를 다스린다 |
| 양보-역접 | 雖不能至 然心嚮往之 (수불능지 연심향왕지) | 비록 이르지 못하나, 마음은 그리로 향한다 |

## 4. 기승전결 논술 구조

### 기승전결(起承轉結) 산문에서의 활용

이 사부 구조는 시뿐 아니라 논설과 서사에도 적용됩니다.

| 단계 | 한자 | 기능 | 산문 적용 |
|------|------|------|----------|
| 기 (起) | 시작 | 주제나 물음을 제시 | 쟁점 제시 |
| 승 (承) | 전개 | 증거나 예시로 정교화 | 논증 구축 |
| 전 (轉) | 전환 | 대비나 새로운 시각 도입 | 반론 제기나 관점 전환 |
| 결 (結) | 결론 | 해결하고 요약 | 최종 입장 표명 |

### 모범 논술 구조

**주제**: 學之道 (배움의 도)

**기 (서두)**:
```
夫學者 所以明道也
부학자 소이명도야
```
"대저 배움이란, 도를 밝히는 수단이다."

**승 (전개)**:
```
古之聖賢 皆以學成德
고지성현 개이학성덕

不學而能者 未之有也
불학이능자 미지유야
```
"옛 성현들은 모두 배움으로 덕을 이루었다. 배우지 않고도 능한 사람은 있어본 적이 없다."

**전 (전환)**:
```
或曰 學不必讀書
혹왈 학불필독서

然不讀書 何以知古今之理乎
연불독서 하이지고금지리호
```
"혹자는 배움에 반드시 책을 읽을 필요가 없다고 말한다. 그러나 책을 읽지 않으면 어찌 고금의 이치를 알 수 있겠는가?"

**결 (결론)**:
```
是故 爲學之要 在於日新
시고 위학지요 재어일신

不可一日而廢也
불가일일이폐야
```
"이런 까닭에 배움의 핵심은 날마다 새로워지는 데 있다. 하루도 폐할 수 없는 것이다."

## 5. 주요 작문 패턴

### 패턴 1: 정의

**공식**: X者 Y也 (X는 Y이다)

| 예시 | 독음 | 해석 |
|------|------|------|
| 仁者愛人也 | 인자애인야 | 인(仁)이란 사람을 사랑하는 것이다 |
| 義者宜也 | 의자의야 | 의(義)란 마땅한 것이다 |
| 學者覺也 | 학자각야 | 배움이란 깨달음이다 |

### 패턴 2: 비교

**공식**: A不如B (A는 B만 못하다)

| 예시 | 독음 | 해석 |
|------|------|------|
| 聞不如見 | 문불여견 | 듣는 것은 보는 것만 못하다 |
| 知不如行 | 지불여행 | 아는 것은 행하는 것만 못하다 |
| 獨學不如共學 | 독학불여공학 | 홀로 배우는 것은 함께 배우는 것만 못하다 |

### 패턴 3: 조건

**공식**: A則B (~하면 ~하다)

| 예시 | 독음 | 해석 |
|------|------|------|
| 學則知 | 학즉지 | 배우면 안다 |
| 勤則成 | 근즉성 | 부지런하면 이룬다 |
| 不學則愚 | 불학즉우 | 배우지 않으면 어리석다 |

### 패턴 4: 수사적 의문

**공식**: 豈不...乎 / 何以...乎

| 예시 | 독음 | 해석 |
|------|------|------|
| 豈不痛哉 | 기불통재 | 어찌 아프지 않으랴? |
| 何以知之 | 하이지지 | 어찌 이를 알겠는가? |
| 不亦樂乎 | 불역락호 | 또한 즐겁지 아니한가? |

### 패턴 5: 점진적 논증

**공식**: A而後B, B而後C

| 예시 | 독음 | 해석 |
|------|------|------|
| 知止而後有定 定而後能靜 | 지지이후유정 정이후능정 | 그칠 곳을 알아야 확고해지고, 확고해야 고요해질 수 있다 |

## 6. 대구문 작성

### 대구문(對句文) — 대구 산문의 구성

대구 산문(변려문/駢儷文)은 같은 길이와 같은 문법 구조의 문장을 짝지어 구성합니다.

### 단계별 구성

**1단계**: 주제와 대비 요소 두 가지를 선택

주제: 계절
요소: 春 (봄)과 秋 (가을)

**2단계**: 명사구 구성

| 위치 | 요소 A | 요소 B |
|------|--------|--------|
| 계절 | 春 (춘, 봄) | 秋 (추, 가을) |
| 이미지 | 花 (화, 꽃) | 月 (월, 달) |

**3단계**: 동사 추가

| 위치 | 요소 A | 요소 B |
|------|--------|--------|
| 동작 | 開 (개, 피다) | 明 (명, 밝다) |

**4단계**: 결합

```
春花開    춘화개    봄꽃이 피고
秋月明    추월명    가을 달이 밝다
```

**5단계**: 5자 또는 7자로 확장

```
春來百花開    춘래백화개    봄이 오면 온갖 꽃이 피고
秋至明月明    추지명월명    가을이 이르면 밝은 달이 빛난다
```

### 연습 예시

다음 요소로 대구 쌍을 구성하세요:

| 주제 | 요소 A | 요소 B |
|------|--------|--------|
| 풍경 | 山 (산) | 水 (물) |
| 시간 | 古 (옛날) | 今 (지금) |
| 방향 | 東 (동쪽) | 西 (서쪽) |
| 성질 | 剛 (강함) | 柔 (부드러움) |

**예시 결과**:

```
山高而蒼翠    산고이창취    산은 높고 푸르고
水深而淸澈    수심이청철    물은 깊고 맑다

古人勤學    고인근학    옛사람은 부지런히 배우고
今人怠惰    금인태타    지금 사람은 게으르다

東方日出    동방일출    동쪽에서 해가 뜨고
西方月落    서방월락    서쪽에서 달이 진다
```

## 7. 간단한 운문 짓기

### 오언절구(五言絕句) 짓기

다음 단계를 따릅니다:

**1단계**: 주제와 감정 선택

- 주제: 秋夜 (가을 밤)
- 감정: 孤獨 (고독)

**2단계**: 운 정하기

같은 운부에서 운자를 선택합니다. 眞韻(진운) 사용:
- 人(인), 春(춘), 新(신), 塵(진)

**3단계**: 기(起) 구 쓰기 — 배경 설정

```
秋夜月光寒    추야월광한
```
"가을 밤에 달빛이 차갑다"

**4단계**: 승(承) 구 쓰기 — 전개 (반드시 압운)

```
獨坐思故人    독좌사고인
```
"홀로 앉아 옛 벗을 그리다" — 人(인)이 운에 맞음

**5단계**: 전(轉) 구 쓰기 — 관점 전환 (비압운)

```
遙望千里外    요망천리외
```
"멀리 천 리 밖을 바라보다"

**6단계**: 결(結) 구 쓰기 — 마무리 (반드시 압운)

```
何處是陽春    하처시양춘
```
"어디에 따스한 봄이 있는가?" — 春(춘)이 운에 맞음

**완성된 시**:

```
秋夜月光寒    추야월광한
獨坐思故人    독좌사고인
遙望千里外    요망천리외
何處是陽春    하처시양춘
```

## 8. 흔한 오류와 회피법

### 오류 1: 잘못된 어순

| 오류 | 올바른 표현 | 문제점 |
|------|-----------|--------|
| 書讀 (서독) | 讀書 (독서) | 한문에서 동사가 목적어 앞에 와야 함 |
| 高山은 수식어로 적합 | 그러나 "산이 높다"는 山高 | 형용사 서술어는 주어 뒤에 옴 |

### 오류 2: 허사 누락

| 오류 | 올바른 표현 | 누락 |
|------|-----------|------|
| 人善善也 (인선선야) | 仁者善也 (인자선야) | 명사화에 者가 필요 |
| 學知 (학지) | 學而知 (학이지) | 절 연결에 而가 필요 |

### 오류 3: 부정사 오용

| 오류 | 올바른 표현 | 규칙 |
|------|-----------|------|
| 無學 (무학) "배우지 않았다" | 不學 (불학) | 不은 동사를 부정; 無는 존재를 부정 |
| 不人 (불인) "사람이 없다" | 無人 (무인) | 無는 "~이 없다" |
| 非好 (비호) "좋지 않다" | 不好 (불호) | 非는 정체성을 부정("~이 아니다"), 不은 성질을 부정 |

### 오류 4: 之와 者의 오용

| 오류 | 올바른 표현 | 규칙 |
|------|-----------|------|
| 學者之人 | 學者 또는 好學之人 | 者가 이미 명사화; 之人을 추가하지 말 것 |
| 天者道 | 天之道 | 소유격에 之가 필요 |

### 오류 5: 시대착오적 어휘

| 현대어 | 고전 대응어 | 참고 |
|--------|-----------|------|
| 問題 (문제, problem) | 難/疑 (난/의) | 問題는 근대 조어 |
| 社會 (사회, society) | 天下/國 (천하/국) | 社會는 근대어 |
| 經濟 (경제, economy) | 經世濟民 (경세제민) | 고전에서는 전체 구절을 사용 |

### 오류 6: 과도하게 긴 문장

한문은 짧고 힘 있는 문장을 선호합니다. 긴 생각은 여러 절로 나누세요:

| 너무 긴 문장 | 개선된 표현 |
|-------------|-----------|
| 人之所以不能學而不能知者以其不勤也 | 人不學不知 何也 不勤也 |
| (하나의 긴 연속) | (짧고 힘 있는 세 문장) |

## 9. 안내된 작문 연습

### 연습 A: 정의 쓰기

X者Y也 패턴을 사용하여 友(벗)를 정의하세요:

**모범**: 友者 同志之人也 (우자 동지지인야)
"벗이란 뜻을 같이하는 사람이다."

이제 다음을 정의하세요:
1. 師 (스승)

이제 다음을 쓰세요:
1. 富 (부)와 德 (덕) 비교

## 10. 연습 문제

### 연습 1: 문장 구성

다음 한국어/영어를 한문으로 쓰세요:

1. "지혜로운 사람은 배움을 좋아한다" (지혜=智, 좋아하다=好, 배움=學)
2. "산은 높고 물은 길다" (산=山, 높다=高, 물=水, 길다=長)
3. "배우지 않으면 알지 못한다" (배우다=學, 알다=知, 아니다=不, 그러면=則)
4. "산에 사람이 없다" (없다=無, 사람=人, 산=山)

### 연습 2: 오류 교정

다음 문장의 오류를 고치세요:

1. 書讀不好 (의도: "글을 잘 읽지 못한다")
2. 無學故無知 (의도: "배우지 않으니 알지 못한다")
3. 人善人也 (의도: "어진 사람은 착한 사람이다")
4. 天者道好 (의도: "하늘의 도는 좋다")

### 연습 3: 패턴 적용

각 패턴에 따라 문장을 쓰세요:

1. X者Y也 (정의)
2. A不如B (비교)
3. A則B (조건)
4. 豈不...乎 (수사적 의문)

### 연습 4: 대구 구성

각 대구의 짝을 완성하세요:

1. 日出東方 → ___________
2. 山明水秀 → ___________
3. 鳥飛天上 → ___________

### 연습 5: 짧은 논술

다음 주제 가운데 하나를 골라 기승전결 논술(최소 4행)을 쓰세요:
