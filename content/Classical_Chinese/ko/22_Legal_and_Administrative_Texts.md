## 1. 행정 문서 개요

한문으로 된 행정 및 법률 문서는 형식, 어휘, 수사 구조에서 엄격한 관습을 따릅니다. 이러한 문서는 2천여 년 동안 동아시아 국가의 실제 운영을 지배했습니다.

### 행정 문서의 유형

| 유형 | 한자 | 한국어 | 방향 | 목적 |
|------|------|--------|------|------|
| 상소 | 上疏 | 상소 (sangso) | 관리 → 왕 | 건의, 청원, 항의 |
| 조칙 | 詔勅 | 조칙 (jochik) | 왕 → 관리 | 왕명 |
| 격문 | 檄文 | 격문 (gyeokmun) | 장수/지도자 → 대중 | 규합, 규탄 |
| 계 | 啓 | 계 (gye) | 관리 → 상관 | 상황 보고 |
| 서 | 書 | 서 (seo) | 동급 → 동급 | 외교 서신 |
| 령 | 令 | 령 (ryeong) | 상급 → 하급 | 구체적 명령 |
| 교지 | 敎旨 | 교지 (gyoji) | 왕 → 관리 | 관직 임명 |
| 판 | 判 | 판 (pan) | 관리 → 사건 | 법적 판결 |

### 행정 문체의 특징

| 특징 | 설명 |
|------|------|
| 격식 | 극도의 공경, 자기 낮춤 |
| 정확성 | 정확한 용어, 모호함 배제 |
| 구조 | 정해진 절차에 따른 엄격한 형식 |
| 어조 | 엄숙하고 절제되며 권위적 |
| 어휘 | 전문 관료 용어 |
| 길이 | 매우 길 수 있음 (수백 자) |

## 2. 상소: 임금에게 올리는 글

### 상소문/上疏文 — 완전한 구조

상소문은 정해진 순서의 절차를 따릅니다:

| 절차 | 시작 정형구 | 한국어 | 기능 |
|------|------------|--------|------|
| 1. 부복 | 伏以 | 복이 | 엎드려 아뢰옵건대... |
| 2. 자기 낮춤 | 臣不佞 | 신불녕 | 불초한 신하... |
| 3. 배경 | 竊惟 | 절유 | 삼가 생각하옵건대... |
| 4. 본론 | 臣以爲 | 신이위 | 신이 생각하기에... |
| 5. 근거 | 古者...今者... | 고자...금자... | 옛날에는...지금은... |
| 6. 청원 | 伏望殿下 | 복망전하 | 엎드려 바라옵건대 전하께서... |
| 7. 결어 | 臣不勝懇切屏營之至 | 신불승간절병영지지 | 신은 간절함을 이기지 못하옵나이다... |

### 모범 상소문: 이이(李珥) — 만언봉사/萬言封事

이이(李珥, 율곡)의 "만언봉사"는 한국 역사에서 가장 유명한 상소 가운데 하나입니다.

**서두**:

```
臣伏惟 殿下以聰明之資
신복유 전하이총명지자

嗣大統 臨御以來
사대통 임어이래

勵精求治 日新又新
려정구치 일신우신
```

### 상세 분석

| 구절 | 독음 | 해석 |
|------|------|------|
| 臣伏惟 | 신복유 | 신이 엎드려 생각하옵건대 |
| 殿下 | 전하 | 전하께서 |
| 以聰明之資 | 이총명지자 | 총명한 자질로 |
| 嗣大統 | 사대통 | 대통(왕위)을 이으시어 |
| 臨御以來 | 임어이래 | 나라를 다스리신 이래로 |
| 勵精求治 | 려정구치 | 정력을 다해 선정(善政)을 구하시어 |
| 日新又新 | 일신우신 | 날마다 새롭고 또 새로워지셨나이다 |

주요 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 臣 | 신 (sin) | 신하 (왕에 대한 1인칭) |
| 伏 | 복 (bok) | 엎드리다 |
| 惟 | 유 (yu) | 생각하다 |
| 聰明 | 총명 (chongmyeong) | 총명하다, 영명하다 |
| 資 | 자 (ja) | 자질 |
| 嗣 | 사 (sa) | 잇다, 계승하다 |
| 統 | 통 (tong) | 계통, 왕통 |
| 勵 | 려 (ryeo) | 힘쓰다, 분발하다 |
| 精 | 정 (jeong) | 정력, 정신 |

## 3. 조칙: 왕명

### 조칙/詔勅 — 왕명 형식

왕이 내리는 조칙은 고유한 엄격한 형식을 따릅니다:

| 절차 | 정형구 | 기능 |
|------|--------|------|
| 시작 | 王若曰 (왕약왈) | "왕이 이르기를:" |
| 대상 | 咨爾群臣 (자이군신) | "너희 신하들에게" |
| 이유 | 朕惟 (짐유) | "짐이 생각하건대..." |
| 명령 | 宜...可... | "마땅히...해야 한다..." |
| 경고 | 其勿違 (기물위) | "어기지 말라" |
| 결어 | 故玆詔示 (고자조시) | "이에 이 조서를 내린다" |

### 조칙 핵심 어휘

| 용어 | 한자 | 독음 | 의미 |
|------|------|------|------|
| 짐 (왕의 1인칭) | 朕 | 짐 (jim) | 왕의 자칭 |
| 이르다 | 若曰 | 약왈 (yakwal) | "이르시기를" |
| 너희에게 | 咨爾 | 자이 (jai) | 관리들에게 고함 |
| 마땅히 | 宜 | 의 (ui) | 적절하다 |
| ~해야 | 可 | 가 (ga) | ~할 수 있다, ~해야 한다 |
| 선포하다 | 詔示 | 조시 (josi) | 조서로 공포하다 |
| 어기지 말라 | 勿違 | 물위 (mulwi) | 반드시 따르라 |
| 영구히 | 永爲 | 영위 (yeongwi) | 영구히 정하다 |

### 조칙 예문

```
王若曰
왕약왈

朕惟治國之道 在於得人
짐유치국지도 재어득인

今有某 才德兼備
금유모 재덕겸비

可授某官 其勿辭
가수모관 기물사

故玆詔示
고자조시
```

| 행 | 해석 |
|----|------|
| 王若曰 | 왕이 이르기를: |
| 朕惟治國之道 在於得人 | 짐이 생각하건대 나라를 다스리는 도는 인재를 얻는 데 있다 |
| 今有某 才德兼備 | 지금 아무개가 있으니, 재주와 덕을 겸비하였다 |
| 可授某官 其勿辭 | 아무 관직을 수여할 것이니 사양하지 말라 |
| 故玆詔示 | 이에 이 조서를 내린다 |

## 4. 격문: 포고문

### 격문/檄文 — 군사 및 공개 포고문

격문은 지지를 모으거나, 적을 규탄하거나, 대의를 선포하는 공개 문서입니다.

### 구조

| 절차 | 기능 | 수사적 목표 |
|------|------|-----------|
| 서두 | 불의를 밝힘 | 분노 |
| 증거 | 적의 죄상 나열 | 격분 |
| 덕에 호소 | 의로움을 내세움 | 도덕적 명분 |
| 행동 촉구 | 호응을 명함 | 동기 부여 |
| 경고 | 불복종자를 위협 | 긴박감 |

### 이순신(李舜臣): 격문 발췌

```
嗟爾倭賊 犯我疆土
차이왜적 범아강토

殺我百姓 焚我宗廟
살아백성 분아종묘

天地鬼神之所共怒也
천지귀신지소공노야
```

### 분석

| 구절 | 독음 | 해석 |
|------|------|------|
| 嗟爾倭賊 | 차이왜적 | 이 흉악한 왜적들아! |
| 犯我疆土 | 범아강토 | 우리 강토를 침범하고 |
| 殺我百姓 | 살아백성 | 우리 백성을 죽이고 |
| 焚我宗廟 | 분아종묘 | 우리 종묘를 불태웠으니 |
| 天地鬼神之所共怒也 | 천지귀신지소공노야 | 천지와 귀신이 함께 분노하는 바이다 |

주요 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 嗟 | 차 (cha) | 탄식 (경멸의 감탄사) |
| 倭 | 왜 (wae) | 왜(일본, 역사적·비하적) |
| 賊 | 적 (jeok) | 도적, 적 |
| 犯 | 범 (beom) | 침범하다 |
| 疆土 | 강토 (gangto) | 영토 |
| 焚 | 분 (bun) | 불태우다 |
| 宗廟 | 종묘 (jongmyo) | 종묘(왕실 사당) |
| 怒 | 노 (no) | 분노 |

### 수사적 분석

| 수사법 | 사례 | 효과 |
|--------|------|------|
| 대구 | 犯我疆土 / 殺我百姓 / 焚我宗廟 | 율격적 점층 |
| 반복 | 我 (우리)의 반복 | 집단적 정체성 |
| 과장 | 天地鬼神之所共怒 | 우주적 의분 |
| 직접 호칭 | 嗟爾倭賊 | 대결적 어조 |

## 7. 법전 체계

### 한국의 주요 법전

| 법전 | 한자 | 독음 | 시기 | 목적 |
|------|------|------|------|------|
| 경국대전 | 經國大典 | 경국대전 (gyeongguk daejeon) | 1469 | 조선의 종합 법전 |
| 속대전 | 續大典 | 속대전 (sok daejeon) | 1746 | 경국대전 보충 |
| 대전통편 | 大典通編 | 대전통편 (daejeon tongpyeon) | 1785 | 통합 법전 |
| 대전회통 | 大典會通 | 대전회통 (daejeon hoetong) | 1865 | 최종 종합 법전 |

### 경국대전/經國大典의 구성

법전은 육조(六曹)에 따라 편성되었습니다:

| 편 | 해당 조 | 내용 |
|----|---------|------|
| 이전 (吏典) | 인사 | 관리, 임명, 품계 |
| 호전 (戶典) | 재정 | 조세, 토지, 인구 |
| 예전 (禮典) | 의례 | 의식, 교육, 과거 |
| 병전 (兵典) | 군사 | 군대, 방위, 역참 |
| 형전 (刑典) | 사법 | 법률, 형벌, 재판 |
| 공전 (工典) | 공사 | 건축, 공예, 노비 |

### 법률 조문 예시

```
凡官吏受賕者 計贓科斷
범관리수뢰자 계장과단
```

| 구절 | 독음 | 해석 |
|------|------|------|
| 凡 | 범 (beom) | 무릇 |
| 官吏 | 관리 (gwalli) | 관원과 이속 |
| 受賕者 | 수뢰자 (suroe-ja) | 뇌물을 받은 자 |
| 計贓 | 계장 (gyejang) | 장물의 액수를 계산하여 |
| 科斷 | 과단 (gwadan) | 형벌을 정한다 |

## 8. 주석 달린 예문

### 상소문 발췌 전문 주석

조선 시대 정책에 항의하는 상소문에서:

```
臣聞 治世之道 在得其人
신문 치세지도 재득기인

得其人則治 失其人則亂
득기인즉치 실기인즉란

此古今之通論也
차고금지통론야

今殿下所任用者 非其人也
금전하소임용자 비기인야

臣不勝憂慮之至
신불승우려지지
```

### 행별 분석

| 행 | 독음 | 해석 |
|----|------|------|
| 臣聞 | 신문 | 신이 듣건대 |
| 治世之道 在得其人 | 치세지도 재득기인 | 세상을 다스리는 도는 적임자를 얻는 데 있다 하옵니다 |
| 得其人則治 | 득기인즉치 | 적임자를 얻으면 다스려지고 |
| 失其人則亂 | 실기인즉란 | 적임자를 잃으면 어지러워지니 |
| 此古今之通論也 | 차고금지통론야 | 이것은 고금에 통하는 원리입니다 |
| 今殿下所任用者 | 금전하소임용자 | 지금 전하께서 임용하신 자들은 |
| 非其人也 | 비기인야 | 적임자가 아닙니다 |
| 臣不勝憂慮之至 | 신불승우려지지 | 신은 근심을 이기지 못하겠나이다 |

### 문법 주석

| 요소 | 기능 |
|------|------|
| 臣聞 | 정형구: "신이 듣건대" — 일반 원칙을 도입 |
| 則 | 조건 표지: "~하면 ~한다" |
| 此...也 | 강조 판단: "이것은 ~이다" |
| 今 | 시간 전환: "지금" / "현재" |
| 所任用者 | 所+동사+者: "임용된 사람들" |
| 非...也 | 부정 판단: "~이 아니다" |
| 不勝...之至 | 정형구: "~의 극에 이르러 이기지 못하다" |
