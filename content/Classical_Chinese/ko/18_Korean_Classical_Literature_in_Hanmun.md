## 1. 한국 한문학 개요

한국 한문학(漢文學)이란 한국인 저자가 한문으로 쓴 문학 총체를 말합니다. 삼국시대부터 조선 말기까지 한문은 한국 학술, 정치, 고급 문학의 주된 문어(文語)였습니다.

### 한국 한문학 연표

| 시기 | 시대 | 주요 발전 |
|------|------|----------|
| 삼국시대 | ~기원전 57 - 668 | 한자 수용; 초기 금석문 |
| 통일신라 | 668-935 | 최초의 주요 한문 작가 (최치원) |
| 고려 | 918-1392 | 불교 문학; 과거 시험 문장 |
| 조선 전기 | 1392-1592 | 성리학; 철학 논문 |
| 조선 후기 | 1592-1897 | 실학(實學); 문학 비평 |

### 한국 한문학을 공부해야 하는 이유

| 이유 | 설명 |
|------|------|
| 역사 기록 | 한국사는 한문으로 기록되었다 |
| 철학 | 한국 유학 사상은 한문으로 표현되었다 |
| 문학 | 한문으로 된 시, 산문, 소설이 방대한 자산을 이룬다 |
| 법률 문서 | 법률, 칙령, 상소문이 한문으로 되어 있다 |
| 문화 정체성 | 한문 이해는 한국 문화 유산과 연결된다 |

## 2. 최치원: 한국 최초의 문학 대가

### 최치원(崔致遠, 857-?) — 학자, 시인, 외교관

최치원은 18세에 당나라 과거에 급제하고 당 조정에서 관직을 지낸 후 신라로 돌아왔습니다. 한국 최고의 한문 작가로 평가받습니다.

### 토황소격문/討黃巢檄文 (황소를 토벌하는 격문)

이 격문은 당나라 장수를 대신하여 반란군 황소를 규탄하며 쓴 것으로, 군사 수사의 걸작으로 꼽힙니다. 황소가 읽고 자리에서 떨어졌다는 일화가 전해집니다.

**유명한 발췌**:

```
不惟天下之人 皆欲殺汝
불유천하지인 개욕살여

抑亦天下之鬼 皆欲殺汝
억역천하지귀 개욕살여
```

### 글자 분석

| 글자 | 독음 | 의미 |
|------|------|------|
| 不惟 | 불유 (bulyu) | ~뿐만 아니라 |
| 天下 | 천하 (cheonha) | 천하 |
| 之 | 지 (ji) | (관형격) |
| 人 | 인 (in) | 사람 |
| 皆 | 개 (gae) | 모두 |
| 欲 | 욕 (yok) | ~하고자 하다 |
| 殺 | 살 (sal) | 죽이다 |
| 汝 | 여 (yeo) | 너 |
| 抑 | 억 (eok) | 더욱이 |
| 亦 | 역 (yeok) | 또한 |
| 鬼 | 귀 (gwi) | 귀신 |

**번역**: "천하의 사람들만 너를 죽이고자 하는 것이 아니라, 더욱이 천하의 귀신들도 모두 너를 죽이고자 한다."

### 수사 분석

| 기법 | 사례 |
|------|------|
| 대구 | 天下之人 ↔ 天下之鬼 |
| 과장 | 귀신조차 너를 죽이고 싶어한다 |
| 반복 | 皆欲殺汝 반복 |
| 점층 | 사람에서 귀신으로 — 규탄의 고조 |

### 최치원의 시

**秋夜雨中 (추야우중, 가을 비 내리는 밤에)**

```
秋風唯苦吟    추풍유고음
世路少知音    세로소지음
窗外三更雨    창외삼경우
燈前萬里心    등전만리심
```

| 행 | 번역 |
|----|------|
| 秋風唯苦吟 | 가을바람에 오직 쓰라린 읊조림뿐 |
| 世路少知音 | 세상길에 지음(知音)이 드무네 |
| 窗外三更雨 | 창 밖엔 삼경의 비 |
| 燈前萬里心 | 등불 앞에 만 리의 마음이여 |

핵심 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 秋 | 추 (chu) | 가을 |
| 苦 | 고 (go) | 쓰라린 |
| 吟 | 음 (eum) | 읊다, 읊조리다 |
| 知音 | 지음 (jieum) | 나를 아는 이, 벗 |
| 三更 | 삼경 (samgyeong) | 삼경 (밤 11시-1시) |
| 萬里 | 만리 (malli) | 만 리 (매우 먼) |

## 3. 정약용: 조선 르네상스의 학자

### 정약용(丁若鏞, 1762-1836) — 다산, 박학다식한 개혁가

정약용, 호 다산(茶山)은 실학(實學) 운동의 선도적 인물입니다. 18년간의 유배 기간 중 통치, 법률, 의학, 공학, 시에 걸친 500여 권의 저술을 남겼습니다.

### 선정 시(詩)

**竹蘭詩 (죽란시, 대나무와 난초의 시)**

```
竹有節而虛    죽유절이허
蘭有香而幽    란유향이유
君子取其德    군자취기덕
不以色爲求    불이색위구
```

### 행별 분석

| 행 | 독음 | 번역 |
|----|------|------|
| 竹有節而虛 | 죽유절이허 | 대나무는 마디가 있되 속이 비고 |
| 蘭有香而幽 | 란유향이유 | 난초는 향기가 있되 은은하네 |
| 君子取其德 | 군자취기덕 | 군자는 그 덕을 취하되 |
| 不以色爲求 | 불이색위구 | 겉모습으로 구하지 않는다 |

핵심 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 竹 | 죽 (juk) | 대나무 |
| 節 | 절 (jeol) | 마디, 절개 |
| 虛 | 허 (heo) | 비어 있는, 겸허한 |
| 蘭 | 란 (ran) | 난초 |
| 香 | 향 (hyang) | 향기 |
| 幽 | 유 (yu) | 은은한, 그윽한 |
| 德 | 덕 (deok) | 덕 |
| 色 | 색 (saek) | 색, 외모 |

### 목민심서/牧民心書에서 (백성을 다스림에 대한 훈계)

```
牧之義 養也
목지의 양야

牧民者 養民也
목민자 양민야

非以民養己也
비이민양기야
```

| 행 | 번역 |
|----|------|
| 牧之義 養也 | "목(牧)"의 뜻은 "기름"이라 |
| 牧民者 養民也 | 백성을 목하는 자는 백성을 기르는 것이니 |
| 非以民養己也 | 백성을 가지고 자기를 기르는 것이 아니다 |

## 4. 이황: 한국의 유학 성인

### 이황(李滉, 1501-1570) — 퇴계, 성리학의 대가

이황, 호 퇴계(退溪)는 한국 최대의 유학 철학자로 꼽힙니다. 이(理)와 기(氣)의 본질에 대한 이이(李珥, 율곡)와의 논쟁은 한국 철학 사상의 정점을 대표합니다.

### 퇴계집/退溪集 — 학문에 대하여

```
爲學之要 在於窮理
위학지요 재어궁리

窮理之要 在於讀書
궁리지요 재어독서

讀書之法 必先端心
독서지법 필선단심

正坐緩讀
정좌완독
```

### 분석

| 행 | 독음 | 번역 |
|----|------|------|
| 爲學之要 在於窮理 | 위학지요 재어궁리 | 학문의 요체는 이치를 궁구하는 데 있고 |
| 窮理之要 在於讀書 | 궁리지요 재어독서 | 이치를 궁구하는 요체는 독서에 있으며 |
| 讀書之法 必先端心 | 독서지법 필선단심 | 독서의 방법은 반드시 먼저 마음을 바로잡아야 하니 |
| 正坐緩讀 | 정좌완독 | 바르게 앉아 천천히 읽어야 한다 |

핵심 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 爲 | 위 (wi) | 하다, 행하다 |
| 要 | 요 (yo) | 요체, 핵심 |
| 窮 | 궁 (gung) | 다하다, 궁구하다 |
| 理 | 리 (ri) | 이치, 원리 |
| 端 | 단 (dan) | 바르게 하다 |
| 正 | 정 (jeong) | 바른, 올바른 |
| 緩 | 완 (wan) | 느긋한, 천천히 |

### 도산십이곡/陶山十二曲 (도산십이곡)

본래 한국어 가사로 지어졌지만, 퇴계의 서문은 한문으로 쓰여 그의 시론을 설명합니다:

```
古人之作歌 蓋出於性情之正
고인지작가 개출어성정지정
```

"옛사람의 노래 짓기는 모두 성정의 바름에서 나온 것이다."

## 5. 한국 특유의 용법 패턴

### 표준 한문과의 차이

한국 한문은 중국인이 쓴 한문과 구별되는 독특한 특성을 발전시켰습니다.

| 특징 | 표준 한문 | 한국 한문 |
|------|----------|----------|
| 어순 | 중국어 어법에 가까움 | 간혹 한국어 SOV 어순 영향 |
| 어휘 | 표준 문학 한문 | 한국 고유 용어 포함 |
| 고유명사 | 중국 이름 | 한자로 쓴 한국 이름 |
| 행정 용어 | 중국 관료 용어 | 한국식으로 변용된 용어 |
| 독법 | 직독 | 현토(懸吐) — 한국어 조사 추가 |

### 한국 특유의 어휘

| 용어 | 한자 | 독음 | 의미 |
|------|------|------|------|
| 한국 | 朝鮮 / 東國 | 조선/동국 | 조선 / 동방의 나라 |
| 임금 | 殿下 | 전하 | 전하 (한국식 용법) |
| 수도 | 漢城 / 漢陽 | 한성/한양 | 한성/한양 (서울) |
| 도(道) | 道 | 도 | 도 단위 |
| 군(郡) | 郡 | 군 | 군 단위 |
| 관복 | 冠帶 | 관대 | 조복 |
| 과거 | 科擧 | 과거 | 국가 시험 |
| 유생 | 儒生 | 유생 | 유학자 |
| 서원 | 書院 | 서원 | 사설 유학 교육 기관 |

### 현토/懸吐 (독법 표기)

한국 학자들은 한문을 한국어 어순으로 읽기 위해 한국어 문법 조사를 추가하는 체계를 발전시켰습니다:

**예**: 學而時習之

- 원문: 學而時習之
- 현토 적용: 學하고 而 時로 習하야 之를
- 한국어 독법: 배우고 때로 익히어 이를

## 6. 상소문 형식

### 상소문/上疏文 (왕에게 올리는 상소)

상소문은 관리가 왕에게 올리는 공식 문서로, 한국 한문의 중요한 문체입니다.

### 표준 구조

| 부분 | 한자 | 독음 | 기능 |
|------|------|------|------|
| 인사 | 伏以 | 복이 | "엎드려 아뢰옵건대..." |
| 배경 | 竊惟 | 절유 | "삼가 생각하옵건대..." |
| 본론 | 臣以爲 | 신이위 | "신이 생각하옵기에..." |
| 요청 | 伏望 | 복망 | "엎드려 바라옵건대..." |
| 맺음 | 臣不勝 | 신불승 | "신은 이기지 못하겠사옵니다..." |

### 주요 관용 표현

| 표현 | 한자 | 독음 | 의미 |
|------|------|------|------|
| 신(臣) | 臣 | 신 (sin) | (왕에 대한) 1인칭 |
| 삼가 | 竊 | 절 (jeol) | 겸손한 자기 지칭 |
| 엎드림 | 伏 | 복 (bok) | 극도의 공경 표시 |
| 전하 | 殿下 | 전하 (jeonha) | 왕에 대한 호칭 |
| 만사 | 萬死 | 만사 (mansa) | "만 번 죽음을 무릅쓰고" (관용적) |

### 서두 예

```
臣某 誠惶誠恐 頓首頓首
신모 성황성공 돈수돈수
```

| 구 | 독음 | 번역 |
|---|------|------|
| 臣某 | 신모 | 신 아무개 |
| 誠惶誠恐 | 성황성공 | 진실로 황공하고 두렵사옵니다 |
| 頓首頓首 | 돈수돈수 | 머리를 거듭 조아리옵니다 |

이 관용적 서두는 왕 앞에서 글쓴이의 겸손한 위치를 표현합니다.

## 7. 기타 주요 한국 한문학 작가

### 주요 작가 개관

| 작가 | 한자 | 시기 | 주요 저작 |
|------|------|------|----------|
| 김부식 | 金富軾 | 1075-1151 | 삼국사기(三國史記), 정사 |
| 일연 | 一然 | 1206-1289 | 삼국유사(三國遺事), 설화와 전설 |
| 이규보 | 李奎報 | 1168-1241 | 동국이상국집(東國李相國集), 시 |
| 정도전 | 鄭道傳 | 1342-1398 | 조선경국전(朝鮮經國典), 국정론 |
| 김시습 | 金時習 | 1435-1493 | 금오신화(金鰲新話), 최초의 한문 소설 |
| 허균 | 許筠 | 1569-1618 | 홍길동전(洪吉童傳), 사회 비판 |
| 박지원 | 朴趾源 | 1737-1805 | 열하일기(熱河日記), 기행문학 |

### 김시습(金時習): 금오신화/金鰲新話에서

금오신화는 한문으로 쓰인 최초의 한국 소설로 평가됩니다. 각 이야기는 사랑, 죽음, 초자연적 세계를 탐구합니다.

**만복사저포기/萬福寺樗蒲記의 서두**:

```
南原府有萬福寺
남원부유만복사

寺有古殿
사유고전

梁某秀才 棲遲其中
양모수재 서지기중
```

| 행 | 번역 |
|----|------|
| 南原府有萬福寺 | 남원부에 만복사가 있었다 |
| 寺有古殿 | 절에는 오래된 전각이 있었다 |

### 이규보(李奎報): 新春 (신춘)

```
去年今日花猶在    거년금일화유재
今年今日花又開    금년금일화우개
花開花落年年事    화개화락연연사
人去人來日日催    인거인래일일최
```

| 행 | 번역 |
|----|------|
| 去年今日花猶在 | 지난해 오늘 꽃은 아직 있었는데 |
| 今年今日花又開 | 올해 오늘 꽃이 또 피었네 |
| 花開花落年年事 | 꽃 피고 지는 것은 해마다의 일이요 |
| 人去人來日日催 | 사람 가고 오는 것은 나날이 재촉되네 |

### 조선왕조실록/朝鮮王朝實錄

한국 한문으로 된 가장 방대한 역사 기록으로, 조선 왕조 472년(1392-1863)을 다루고 있습니다. 1997년 유네스코 세계기록유산으로 지정되었습니다.

**기록 형식 예**:

```
[연도] 月 [일]
[관리] 上疏曰...
上曰...
```

| 요소 | 의미 |
|------|------|
| 月 (월) | 월 |
| 上疏曰 (상소왈) | 상소를 올려 말하기를... |
| 上曰 (상왈) | 임금이 말하기를... |

### 삼국사기/三國史記 — 서두

```
始祖姓朴氏 諱赫居世
시조성박씨 휘혁거세
```

"시조의 성은 박씨이고, 휘는 혁거세이다."

| 글자 | 독음 | 의미 |
|------|------|------|
| 始祖 | 시조 (sijo) | 시조 |
| 姓 | 성 (seong) | 성(姓) |
| 諱 | 휘 (hwi) | 휘 (시호/이름) |
