## 1. 한문 부정 개요

### 1.1 "아니다"를 표현하는 여섯 가지 방법

한문에는 여러 부정사가 있으며, 각각 고유한 뉘앙스를 가지고 있습니다. 적절한 부정사를 선택하는 것이 정확한 표현에 필수적입니다.

> **부정사 6자 한눈에 보기**
>
> - **不 (불/부)**: "않다" (동사/형용사의 일반 부정)
> - **非 (비)**: "아니다" (동일성/등식의 부정)
> - **無 (무)**: "없다" (존재의 부정)
> - **未 (미)**: "아직 않다" (시제: 아직 일어나지 않음)
> - **莫 (막)**: "아무도/아무것도 않다" (전칭 부정)
> - **勿 (물)**: "하지 마라" (금지/명령)
>
> 한국어 대응:
> - **不** = -지 않다 / 안-, **非** = -이 아니다
> - **無** = 없다, **未** = 아직 -지 않다
> - **莫** = 아무도 -지 않다, **勿** = -하지 마라

### 1.2 부정사 비교

| 부정사 | 유형 | 부정 대상 | 예문 | 번역 |
|--------|------|----------|------|------|
| 不 | 일반 | 동사, 형용사 | 不知 | 알지 못하다 |
| 非 | 동일성 | "A는 B이다" 등식 | 非人 | 사람이 아니다 |
| 無 | 존재 | "X가 있다" | 無道 | 도가 없다 |
| 未 | 시제 | 아직 행해지지 않은 행위 | 未學 | 아직 배우지 못하다 |
| 莫 | 전칭 | 아무도, 아무것도 | 莫知 | 아무도 알지 못하다 |
| 勿 | 금지 | 명령형 부정 | 勿言 | 말하지 마라 |

## 2. 不 — 일반 부정

### 2.1 기본 용법

不 (불/부, bul/bu)는 가장 일반적인 부정사입니다. 동사와 형용사를 부정합니다.

| 한자 | 음독 | 훈독 | 기능 |
|------|------|------|------|
| 不 | 불/부 (bul/bu) | 아닐 불 | 동사와 형용사의 일반 부정 |

> **발음 참고**: 不는 보통 불(bul)로 읽지만, 특정 음절 앞에서 부(bu)로 변합니다. 예: 不當 → 부당.

### 2.2 不 + 동사

| 문장 | 음독 | 번역 |
|------|------|------|
| 不知 | 부지 | 알지 못하다 |
| 不學 | 불학 | 배우지 않다 |
| 不行 | 불행 | 가지 않다; 행하지 않다 |
| 不言 | 불언 | 말하지 않다 |
| 不食 | 불식 | 먹지 않다 |
| 不見 | 불견 | 보지 못하다 |

### 2.3 不 + 형용사

| 문장 | 음독 | 번역 |
|------|------|------|
| 不善 | 불선 | 착하지 않다 |
| 不義 | 불의 | 의롭지 않다 |
| 不仁 | 불인 | 어질지 않다 |
| 不足 | 부족 | 충분하지 않다; 모자라다 |
| 不遠 | 불원 | 멀지 않다 |

### 2.4 不가 포함된 완전한 문장

| 문장 | 음독 | 번역 |
|------|------|------|
| 人不學 | 인불학 | 사람이 배우지 않다 |
| 王不仁 | 왕불인 | 왕이 어질지 않다 |
| 天不言 | 천불언 | 하늘이 말하지 않다 |
| 子不食肉 | 자불식육 | 아이가 고기를 먹지 않다 |
| 人不知而不慍 | 인부지이불온 | 남이 알아주지 않아도 원망하지 않다 |

### 2.5 한국어에서 흔히 쓰이는 不 복합어

| 복합어 | 음독 | 의미 |
|--------|------|------|
| 不可 | 불가 | 불가능한; 용납할 수 없는 |
| 不足 | 부족 | 부족한; 모자라는 |
| 不當 | 부당 | 부당한; 부적절한 |
| 不正 | 부정 | 부정직한; 부정한 |
| 不幸 | 불행 | 불행한; 불운한 |
| 不安 | 불안 | 불안한; 초조한 |
| 不義 | 불의 | 불의한; 부정한 |
| 不信 | 불신 | 불신; 믿지 않음 |

## 3. 非 — 동일성 부정

### 3.1 기본 용법

非 (비, bi)는 동일성을 부정합니다. 즉 "X는 Y가 아니다"라고 말합니다. 동사나 형용사가 아닌 계사(是/也)를 부정합니다.

| 한자 | 음독 | 훈독 | 기능 |
|------|------|------|------|
| 非 | 비 (bi) | 아닐 비 | 동일성 부정: "-이 아니다" |

### 3.2 非 vs 不

| 부정사 | 부정 대상 | 예문 | 번역 |
|--------|----------|------|------|
| 不 | 동사/형용사 | 不善 | 착하지 않다 (성질) |
| 非 | 동일성/등식 | 非善也 | [이것은] 선(善)이 아니다 (동일성) |

### 3.3 非 예문

| 문장 | 음독 | 번역 |
|------|------|------|
| 非人也 | 비인야 | [이것은] 사람이 아니다 / 인간답지 않다 |
| 非禮也 | 비례야 | [이것은] 예가 아니다 |
| 非吾所知也 | 비오소지야 | [이것은] 내가 아는 바가 아니다 |
| 非學者之道也 | 비학자지도야 | [이것은] 학자의 도가 아니다 |
| 此非義也 | 차비의야 | 이것은 의가 아니다 |

### 3.4 非...也 패턴

非는 동일성 부정 완전문에서 흔히 也와 짝을 이룹니다:

| 문장 | 음독 | 번역 |
|------|------|------|
| 此非禮也 | 차비례야 | 이것은 예가 아니다 |
| 彼非君子也 | 피비군자야 | 그는 군자가 아니다 |
| 戰非善事也 | 전비선사야 | 전쟁은 좋은 일이 아니다 |

### 3.5 한국어의 非 복합어

| 복합어 | 음독 | 의미 |
|--------|------|------|
| 非常 | 비상 | 비상한; 비상사태 |
| 是非 | 시비 | 시비; 옳고 그름 |
| 非難 | 비난 | 비난; 비판 |
| 非法 | 비법 | 불법의 |

## 4. 無 — 존재 부정

### 4.1 기본 용법

無 (무, mu)는 존재를 부정합니다. 즉 "X가 없다" 또는 "X가 존재하지 않는다"라고 말합니다. 有 (유, "있다")의 반대입니다.

| 한자 | 음독 | 훈독 | 기능 |
|------|------|------|------|
| 無 | 무 (mu) | 없을 무 | 존재 부정: "없다; -이 없이" |

### 4.2 有 vs 無

| 긍정 | 부정 | 음독 | 번역 |
|------|------|------|------|
| 有道 | 無道 | 유도 / 무도 | 도가 있다 / 도가 없다 |
| 有人 | 無人 | 유인 / 무인 | 사람이 있다 / 사람이 없다 |
| 有心 | 無心 | 유심 / 무심 | 뜻이 있다 / 뜻이 없다 |
| 有禮 | 無禮 | 유례 / 무례 | 예가 있다 / 무례하다 |

### 4.3 無가 포함된 완전한 문장

| 문장 | 음독 | 번역 |
|------|------|------|
| 天下無道 | 천하무도 | 천하에 도가 없다 |
| 國無賢臣 | 국무현신 | 나라에 어진 신하가 없다 |
| 人無信不立 | 인무신불립 | 신의가 없으면 사람이 설 수 없다 |
| 無父無君 | 무부무군 | 아버지도 없고 임금도 없다 (극도의 불경) |
| 吾無隱乎爾 | 오무은호이 | 나는 너에게 숨기는 것이 없다 |

### 4.4 한국어의 無 복합어

| 복합어 | 음독 | 의미 |
|--------|------|------|
| 無限 | 무한 | 무한한; 끝없는 |
| 無理 | 무리 | 무리한; 지나친 |
| 無知 | 무지 | 무지한 |
| 無名 | 무명 | 무명의; 알려지지 않은 |
| 無禮 | 무례 | 무례한; 예의 없는 |
| 無事 | 무사 | 무사한; 안전한 |
| 無心 | 무심 | 무심한; 의도 없는 |
| 無用 | 무용 | 쓸모없는 |

## 5. 未 — 시제 부정

### 5.1 기본 용법

未 (미, mi)는 "아직 ~하지 않다"를 뜻합니다. 행위가 아직 일어나지 않았지만 앞으로 일어날 수 있음을 나타냅니다.

| 한자 | 음독 | 훈독 | 기능 |
|------|------|------|------|
| 未 | 미 (mi) | 아직 미 | 시제 부정: "아직 ~하지 않다" |

### 5.2 未 vs 不

| 부정사 | 의미 | 예문 | 번역 |
|--------|------|------|------|
| 不知 | 알지 못하다 (일반) | 人不知 | 사람들이 알지 못하다 |
| 未知 | 아직 알지 못하다 (시제) | 人未知 | 사람들이 아직 알지 못하다 |

### 5.3 未 예문

| 문장 | 음독 | 번역 |
|------|------|------|
| 未學 | 미학 | 아직 배우지 못하다 |
| 未知 | 미지 | 아직 알지 못하다; 미지의 |
| 未見 | 미견 | 아직 보지 못하다 |
| 未聞 | 미문 | 아직 듣지 못하다; 전례 없는 |
| 吾未見仁者 | 오미견인자 | 나는 아직 참으로 어진 사람을 만나지 못하였다 |
| 道未行也 | 도미행야 | 도가 아직 행해지지 않았다 |

### 5.4 한국어의 未 복합어

| 복합어 | 음독 | 의미 |
|--------|------|------|
| 未來 | 미래 | 미래 ("아직 오지 않은 것") |
| 未知 | 미지 | 미지의; 알려지지 않은 |
| 未定 | 미정 | 미정의; 결정되지 않은 |
| 未滿 | 미만 | 미만; 이하 |
| 未成年 | 미성년 | 미성년; 성인이 되지 않은 |

## 6. 莫 — 전칭 부정

### 6.1 기본 용법

莫 (막, mak)는 "아무도 ~하지 않다" 또는 "아무것도 ~하지 않다"를 의미합니다. 모든 가능한 주어나 대상을 전칭적으로 부정합니다.

| 한자 | 음독 | 훈독 | 기능 |
|------|------|------|------|
| 莫 | 막 (mak) | 없을 막 | 전칭 부정: "아무도; 아무것도; ~할 수 없다" |

### 6.2 莫 — "아무도 ~하지 않다"

| 문장 | 음독 | 번역 |
|------|------|------|
| 莫知 | 막지 | 아무도 알지 못하다 |
| 莫敢言 | 막감언 | 아무도 감히 말하지 못하다 |
| 莫能行 | 막능행 | 아무도 행할 수 없다 |
| 天下莫不知 | 천하막부지 | 천하에 알지 못하는 이가 없다 (= 모두 알다) |

### 6.3 莫 — "아무것도 ~하지 않다"

| 문장 | 음독 | 번역 |
|------|------|------|
| 莫大於天 | 막대어천 | 하늘보다 큰 것은 없다 |
| 莫善於仁 | 막선어인 | 인보다 좋은 것은 없다 |
| 莫重於義 | 막중어의 | 의보다 무거운 것은 없다 |

### 6.4 莫의 금지 용법 (문어체)

일부 맥락에서 莫는 勿과 유사한 금지의 기능을 합니다:

| 문장 | 음독 | 번역 |
|------|------|------|
| 莫言 | 막언 | 말하지 마라 / 아무것도 말하지 마라 |
| 莫忘 | 막망 | 잊지 마라 |

## 7. 勿 — 금지

### 7.1 기본 용법

勿 (물, mul)은 금지에 사용됩니다. "~하지 마라!"라는 명령입니다.

| 한자 | 음독 | 훈독 | 기능 |
|------|------|------|------|
| 勿 | 물 (mul) | 말 물 | 금지: "~하지 마라; ~해서는 안 된다" |

### 7.2 勿 예문

| 문장 | 음독 | 번역 |
|------|------|------|
| 勿言 | 물언 | 말하지 마라 |
| 勿忘 | 물망 | 잊지 마라 |
| 勿食 | 물식 | 먹지 마라 |
| 勿行 | 물행 | 가지 마라 |

### 7.3 고전 인용문 속 勿

| 문장 | 음독 | 번역 | 출전 |
|------|------|------|------|
| 非禮勿視 | 비례물시 | 예가 아니면 보지 마라 | 論語 |
| 非禮勿聽 | 비례물청 | 예가 아니면 듣지 마라 | 論語 |
| 非禮勿言 | 비례물언 | 예가 아니면 말하지 마라 | 論語 |
| 非禮勿動 | 비례물동 | 예가 아니면 행하지 마라 | 論語 |
| 己所不欲 勿施於人 | 기소불욕 물시어인 | 자기가 원하지 않는 것을 남에게 베풀지 마라 | 論語 |

### 7.4 기타 금지 표현

| 어휘 | 음독 | 의미 | 예문 | 번역 |
|------|------|------|------|------|
| 勿 | 물 | ~하지 마라 (표준) | 勿言 | 말하지 마라 |
| 毋 | 무 | ~하지 마라 (격식) | 毋忘 | 잊지 마라 |
| 莫 | 막 | ~하지 마라 (문어) | 莫言 | 말하지 마라 |
| 不可 | 불가 | ~해서는 안 된다; 불가하다 | 不可行 | 행해서는 안 된다 |

## 8. 이중 부정 패턴

### 8.1 이중 부정 = 강조

한문에서 두 번의 부정은 강한 긍정을 만듭니다:

> **이중 부정 패턴**
>
> - **不 + 不** = 강한 긍정
> - **莫不** = 모두/예외 없이
> - **無不** = ~하지 않는 것이 없다; 모두
> - **未嘗不** = 항상 (한 번도 아닌 적이 없다)
> - **不可不** = 반드시 ~해야 한다 (안 할 수 없다)
>
> - 예시: 莫不知 = 알지 못하는 이가 없다 = 모두 알다
> - 예시: 不可不學 = 배우지 않을 수 없다 = 반드시 배워야 한다

### 8.2 주요 이중 부정 표현

| 표현 | 음독 | 직역 | 의미 |
|------|------|------|------|
| 莫不知 | 막부지 | 알지 않는 이가 없다 | 모두 알다 |
| 無不善 | 무불선 | 착하지 않은 것이 없다 | 모두 착하다 |
| 不可不學 | 불가불학 | 배우지 않을 수 없다 | 반드시 배워야 한다 |
| 不可不知 | 불가부지 | 알지 않을 수 없다 | 반드시 알아야 한다 |
| 不亦樂乎 | 불역락호 | 또한 즐겁지 않은가? | 참으로 즐겁다! |
| 不亦說乎 | 불역열호 | 또한 기쁘지 않은가? | 참으로 기쁘다! |
| 無不為 | 무불위 | 하지 않는 것이 없다 | 모든 것을 하다 |
| 未嘗不然 | 미상불연 | 그렇지 않은 적이 없다 | 항상 그러하다 |

### 8.3 不亦...乎 패턴 (수사적 이중 부정)

이 패턴은 수사적 의문을 통해 강한 긍정을 만듭니다:

| 문장 | 음독 | 직역 | 의미 |
|------|------|------|------|
| 不亦善乎 | 불역선호 | 또한 좋지 않은가? | 참으로 좋다! |
| 不亦美乎 | 불역미호 | 또한 아름답지 않은가? | 참으로 아름답다! |
| 不亦樂乎 | 불역락호 | 또한 즐겁지 않은가? | 정말 즐겁다! |

---

## 9. 고전 텍스트의 부정문

### 9.1 논어(論語)에서

```
子曰 知之者 不如好之者 好之者 不如樂之者
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 知之者 | 지지자 | 그것을 아는 사람은 |
| 不如好之者 | 불여호지자 | 그것을 좋아하는 사람만 못하고 |
| 好之者 | 호지자 | 그것을 좋아하는 사람은 |
| 不如樂之者 | 불여락지자 | 그것을 즐기는 사람만 못하다 |

> 不如 (불여) = "~만 못하다; ~에 미치지 못하다"

### 9.2 맹자(孟子)에서

```
無惻隱之心 非人也
無羞惡之心 非人也
無辭讓之心 非人也
無是非之心 非人也
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 無惻隱之心 | 무측은지심 | 측은히 여기는 마음이 없으면 |
| 非人也 | 비인야 | [참된] 사람이 아니다 |
| 無羞惡之心 | 무수오지심 | 부끄러워하는 마음이 없으면 |
| 非人也 | 비인야 | [참된] 사람이 아니다 |
| 無辭讓之心 | 무사양지심 | 사양하는 마음이 없으면 |
| 非人也 | 비인야 | [참된] 사람이 아니다 |
| 無是非之心 | 무시비지심 | 시비를 가리는 마음이 없으면 |
| 非人也 | 비인야 | [참된] 사람이 아니다 |

> 이 구절은 無(존재 부정)와 非(동일성 부정)를 함께 사용하고 있습니다.

## 10. 연습 문제

### 연습 1: 올바른 부정사 선택

不, 非, 無, 未, 莫, 勿 중에서 선택하세요:

```
1. ___ 言 (말하지 마라! — 금지)
2. ___ 知 (아직 알지 못하다 — 시제)
3. ___ 善 (착하지 않다 — 일반)
4. ___ 人也 (사람이 아니다 — 동일성)
5. ___ 道 (도가 없다 — 존재)
6. ___ 敢言 (아무도 감히 말하지 못하다 — 전칭)
```

<details>
<summary>정답 확인</summary>

```
1. 勿言 (물언) — 말하지 마라 (금지)
2. 未知 (미지) — 아직 알지 못하다 (시제)
3. 不善 (불선) — 착하지 않다 (일반)
4. 非人也 (비인야) — 사람이 아니다 (동일성 부정)
5. 無道 (무도) — 도가 없다 (존재 부정)
6. 莫敢言 (막감언) — 아무도 감히 말하지 못하다 (전칭)
```

</details>

### 연습 2: 부정문 번역

```
1. 人不學則不知
2. 天下無道
3. 吾未見仁者
4. 此非義也
5. 莫大於天
6. 非禮勿言
7. 不可不學
8. 人無信不立
```

<details>
<summary>정답 확인</summary>

```
1. 人不學則不知 (인불학즉부지)
   사람이 배우지 않으면 알지 못한다.

2. 天下無道 (천하무도)
   천하에 도가 없다.

3. 吾未見仁者 (오미견인자)
   나는 아직 어진 사람을 만나지 못하였다.

4. 此非義也 (차비의야)
   이것은 의가 아니다.

5. 莫大於天 (막대어천)
   하늘보다 큰 것은 없다.

6. 非禮勿言 (비례물언)
   예가 아닌 것은 말하지 마라.

7. 不可不學 (불가불학)
   배우지 않을 수 없다 = 반드시 배워야 한다.

8. 人無信不立 (인무신불립)
   신의가 없으면 사람이 설 수 없다.
```

</details>

### 연습 3: 不 vs 非 vs 無

각 부정사가 선택된 이유를 설명하세요:

```
1. 不知 vs 非知也 vs 無知
2. 不仁 vs 非仁也 vs 無仁
```

<details>
<summary>정답 확인</summary>

```
1a. 不知 (부지) = 알지 못하다 (동사 "알다"를 부정 — 일반적 사실)
1b. 非知也 (비지야) = 지식이 아니다 (동일성 부정 — "이것은 지식 자체가 아니다")
1c. 無知 (무지) = 지식이 없다; 무지 (지식의 존재를 부정)

2a. 不仁 (불인) = 어질지 않다 (성질 부정 — 어진 자질이 없는 사람)
2b. 非仁也 (비인야) = 인이 아니다 (동일성 부정 — "이 행위는 인 자체가 아니다")
2c. 無仁 (무인) = 인이 없다 (존재 부정 — 인이 부재함)
```

</details>

### 연습 4: 이중 부정

이중 부정 표현을 긍정문으로 바꾸세요:

```
1. 莫不知
2. 不可不學
3. 無不善
4. 不亦樂乎
5. 未嘗不然
```

<details>
<summary>정답 확인</summary>

```
1. 莫不知 (막부지) = 알지 않는 이가 없다 → 모두 알다
2. 不可不學 (불가불학) = 배우지 않을 수 없다 → 반드시 배워야 한다
3. 無不善 (무불선) = 착하지 않은 것이 없다 → 모두 착하다
4. 不亦樂乎 (불역락호) = 즐겁지 않은가? → 참으로 즐겁다!
5. 未嘗不然 (미상불연) = 그렇지 않은 적이 없다 → 항상 그러하다
```

</details>

### 연습 5: 부정문 작문

다음을 한문으로 쓰세요:

```
1. "왕이 알지 못하다" (일반 부정)
2. "이것은 도가 아니다" (동일성 부정)
3. "현명한 사람이 없다" (존재 부정)
4. "나는 아직 듣지 못하였다" (시제 부정)
5. "잊지 마라!" (금지)
```

<details>
<summary>정답 확인</summary>

```
1. 王不知 (왕부지) — 왕이 알지 못하다
2. 此非道也 (차비도야) — 이것은 도가 아니다
3. 無賢人 (무현인) — 현명한 사람이 없다
4. 吾未聞之 (오미문지) — 나는 아직 그것을 듣지 못하였다
5. 勿忘 (물망) — 잊지 마라!
```

</details>
