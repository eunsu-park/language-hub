## 1. 한국의 한문 읽기 방식

### 1.1 한국의 한문 독해 전통

한국은 한문 텍스트를 한국어 문법에 맞추어 읽는 독자적인 체계를 발전시켰습니다. 이 체계를 현토(懸吐, hyeonto)라 합니다. 글자 그대로 한문 글자에 한국어 문법 표지를 "매다는" 것입니다.

> **한문을 읽는 한국식 방법**
>
> - **방법 1: 음독 (音讀)** — 각 한자의 한국어 음을 읽기
>   天高 → 천고 (cheon-go)
>
> - **방법 2: 현토 (懸吐)** — 한문에 한국어 조사 달기
>   天이 高하니 → 하늘이 높으니 (하늘이 높으니...)
>
> - **방법 3: 석독 (釋讀)** — 완전한 한국어 번역
>   天高 → 하늘이 높다 (하늘이 높다)
>
> 역사적 맥락:
> 한글 이전(1443년), 한국인들은 한문 텍스트를 머릿속으로
> 한국어 어순으로 변환하며 읽었습니다. 한글 이후,
> 현토는 교육에 사용되는 공식 서기 체계가 되었습니다.

### 1.2 한국식 독법을 배워야 하는 이유

- **학업 필수**: 한국의 한문 교육에서 현토와 석독 능력 필수
- **시험 대비**: 한자능력검정시험에 현토 문제 포함
- **문화적 소양**: 한국인이 역사적으로 한문과 상호작용한 방식의 이해
- **번역 기술**: 현토는 한문과 한국어 번역의 가교 역할

## 2. 훈독 vs 음독 복습

### 2.1 모든 한자의 두 가지 읽기

모든 한자에는 두 가지 한국어 읽기가 있습니다:

| 읽기 유형 | 한국어 | 정의 | 山의 예 |
|----------|--------|------|---------|
| 음독 (音讀) | 소리 읽기 | 중국어에서 유래한 한국어 발음 | 산 (san) |
| 훈독 (訓讀) | 뜻 읽기 | 뜻에 해당하는 한국어 단어 | 메/뫼 (산) |

### 2.2 사용 시점

| 맥락 | 사용하는 읽기 | 예시 |
|------|-------------|------|
| 복합어를 소리 내어 읽기 | 음독 | 山水 → 산수 (san-su) |
| 사전 표제어 | 훈 + 음 | 山 = 메 산 (me san) |
| 현토 읽기 | 한자는 음독, 조사는 한국어 | 山이 高하니 |
| 석독 (번역) | 훈독을 완전한 한국어로 변환 | 산이 높다 → 뫼가 높다 |

### 2.3 완전한 인용 형식

한국어에서 한자의 읽기를 인용할 때의 형식: **훈독 + 음독**

| 한자 | 훈독 (뜻) | 음독 (소리) | 인용 |
|------|----------|------------|------|
| 天 | 하늘 | 천 | 하늘 천 |
| 地 | 땅 | 지 | 땅 지 |
| 人 | 사람 | 인 | 사람 인 |
| 學 | 배울 | 학 | 배울 학 |
| 善 | 착할 | 선 | 착할 선 |
| 大 | 큰 | 대 | 큰 대 |
| 讀 | 읽을 | 독 | 읽을 독 |
| 書 | 글 | 서 | 글 서 |

## 3. 현토 (懸吐): 한국어 조사 달기

### 3.1 현토란?

현토(懸吐)는 한문 글자 사이에 한국어 문법 조사(토/吐)를 삽입하여 한국어 어순으로 읽을 수 있게 하는 방법입니다.

> **현토 (懸吐) 체계**
>
> - **원래 한문**: 天高地厚
>
> 현토 적용:
>   天이 高하고 地가 厚하다
>   천이 고하고 지가 후하다
>
> 한국어 번역:
>   하늘이 높고 땅이 두껍다
>   하늘이 높고 땅이 두껍다.
>
> 현토 조사:
> - **이** = 주격 조사 (이/가)
> - **하고** = "~하고" (접속)
> - **가** = 주격 조사
> - **하다** = 용언 어미

### 3.2 역사적 발전

| 시기 | 발전 |
|------|------|
| 한글 이전 (1443년 이전) | 이두(吏讀) 체계로 한자를 빌려 한국어 조사를 표기 |
| 한글 이후 (1443년 이후) | 한글로 쓴 현토가 교육의 표준이 됨 |
| 조선시대 | 유교 서원(書院)에서 현토가 널리 사용 |
| 현대 교육 | 현토가 한문 교육 과정의 일부로 교육됨 |

## 4. 주요 토 (吐) 표지

### 4.1 주격 및 화제 표지

| 토 | 기능 | 한국어 대응 | 사용 시점 |
|----|------|-----------|----------|
| 이/가 | 주격 조사 | 이/가 | 주어 명사 뒤 |
| 은/는 | 화제 조사 | 은/는 | 화제 명사 뒤 |
| 도 | "또한" | 도 | "또한"의 의미 추가 |

### 4.2 목적격 표지

| 토 | 기능 | 한국어 대응 | 사용 시점 |
|----|------|-----------|----------|
| 을/를 | 목적격 조사 | 을/를 | 직접 목적어 뒤 |
| 에/에서 | 처소격 | 에/에서 | 장소 뒤 |
| 에게 | 여격 | 에게 | 수여 대상 뒤 |

### 4.3 용언 어미와 접속

| 토 | 기능 | 한국어 대응 | 사용 시점 |
|----|------|-----------|----------|
| 하다 | 서술형 어미 | -하다 | 서술문 끝 |
| 하니 | 원인/순접 | -하니 | "~하므로/~할 때" |
| 하고 | 접속형 | -하고 | "~하고" (절 연결) |
| 하면 | 조건형 | -하면 | "~하면" |
| 하여 | 결과형 | -하여 | "~하여, 그리하여" |
| 하라 | 명령형 | -하라 | 명령 어미 |
| 한가 / 하는가 | 의문형 | -한가 | 의문 어미 |
| 하도다 | 감탄형 | -하도다 | 감탄 어미 |

### 4.4 연결 조사

| 토 | 기능 | 용도 |
|----|------|------|
| 而 → -고/며 | "~하고/~하며" | 순차적 행위 연결 |
| 則 → -면/즉 | "~하면/곧" | 조건의 결과 |
| 之 → 의/를 | 소유격/목적격 | 명사 연결 또는 목적어 표시 |
| 者 → -는 것은 | 명사화 | 명사절 만들기 |
| 也 → -이다/이니라 | 계사 | 등가문 어미 |

## 5. 한국어 어순 읽기

### 5.1 어순 문제

한문과 한국어는 어순이 다릅니다:

```
한문:     S + V + O  (주어-동사-목적어)
한국어:   S + O + V  (주어-목적어-동사)

예시:
한문:     孔子 學 禮       (공자 배우다 예)
한국어:   공자가 예를 배우다  (공자 예 배우다)
```

### 5.2 어순 읽기 규칙

한문을 한국어 어순으로 읽을 때 다음 단계를 따릅니다:

> **한국어 어순 읽기 규칙**
>
> 1. 주어 파악 → 이/가 또는 은/는 추가
> 2. 목적어 파악 → 을/를 추가, 동사 앞으로 이동
> 3. 동사 파악 → 한국어 어미 추가 (-다/-하다)
> 4. 한국어 S-O-V 어순으로 읽기
>
> 한문 원문: 王 愛 民
> - 단계 1: 王이 (주어 + 조사)
> - 단계 2: 民을 (목적어 + 조사, 동사 앞으로 이동)
> - 단계 3: 愛하다 (동사 + 어미)
> - 결과: 왕이 민을 애하다 → 왕이 백성을 사랑하다

### 5.3 어순 읽기 예시

| 한문 원문 | 읽기 순서 | 현토 읽기 | 한국어 번역 |
|----------|----------|----------|------------|
| 天高 | 天이 高하다 | 천이 고하다 | 하늘이 높다 |
| 王愛民 | 王이 民을 愛하다 | 왕이 민을 애하다 | 왕이 백성을 사랑하다 |
| 孔子學禮 | 孔子가 禮를 學하다 | 공자가 례를 학하다 | 공자가 예를 배우다 |
| 人讀書 | 人이 書를 讀하다 | 인이 서를 독하다 | 사람이 책을 읽다 |
| 民耕田 | 民이 田을 耕하다 | 민이 전을 경하다 | 백성이 밭을 갈다 |

### 5.4 특수 읽기 패턴

| 한문 패턴 | 현토 패턴 | 예시 |
|----------|----------|------|
| A者B也 | A는 B이다 | 學者는 覺也이다 (학이란 것은 깨달음이다) |
| A之B | A의 B | 人之心 = 인의 심 (사람의 마음) |
| 不A | A하지 않다 | 不知 = 알지 못하다 |
| 有A | A가 있다 | 有道 = 도가 있다 |
| 無A | A가 없다 | 無道 = 도가 없다 |

## 6. 석독 (釋讀): 완전한 번역 읽기

### 6.1 석독이란?

석독(釋讀, seokdok)은 한문을 완전한 한국어로 번역하는 것으로, 모든 한자를 가능한 한 고유 한국어 표현으로 변환합니다.

### 6.2 세 단계 읽기

| 단계 | 방법 | 예시 (天高地厚) |
|------|------|----------------|
| 음독 (소리) | 소리만 읽기 | 천고지후 |
| 현토 (주석) | 한국어 조사 추가 | 천이 고하고 지가 후하다 |
| 석독 (번역) | 완전한 한국어 | 하늘이 높고 땅이 두껍다 |

### 6.3 석독 예시

| 한문 원문 | 현토 | 석독 (완전한 한국어) |
|----------|------|---------------------|
| 日出 | 일이 출하다 | 해가 뜨다 |
| 月明 | 월이 명하다 | 달이 밝다 |
| 山高水清 | 산이 고하고 수가 청하다 | 산이 높고 물이 맑다 |
| 人不學則不知 | 인이 학하지 아니하면 즉 지하지 못하다 | 사람이 배우지 않으면 알지 못한다 |
| 王愛民 | 왕이 민을 애하다 | 왕이 백성을 사랑하다 |
| 學而時習之 | 학하고 때때로 습하다 | 배우고 때때로 익히다 |

## 7. 현토 연습: 단문

### 7.1 주술 문장

다음 주술 문장에 현토를 달아 보세요:

| 한문 | 현토 | 석독 |
|------|------|------|
| 天高 | 天이 高하다 | 하늘이 높다 |
| 地厚 | 地가 厚하다 | 땅이 두껍다 |

| 人善 | 人이 善하다 | 사람이 착하다 |
| 道遠 | 道가 遠하다 | 길이 멀다 |

### 7.2 주동빈 문장

| 한문 | 현토 | 석독 |
|------|------|------|
| 人讀書 | 人이 書를 讀하다 | 사람이 책을 읽다 |
| 王愛民 | 王이 民을 愛하다 | 왕이 백성을 사랑하다 |
| 子食飯 | 子가 飯을 食하다 | 아이가 밥을 먹다 |
| 師敎學 | 師가 學을 敎하다 | 스승이 학문을 가르치다 |

### 7.3 복합 문장

| 한문 | 현토 | 석독 |
|------|------|------|
| 學而時習之 | 學하고 時로 之를 習하다 | 배우고 때때로 그것을 익히다 |

| 溫故而知新 | 故를 溫하고 新을 知하다 | 옛것을 익히고 새것을 알다 |

## 8. 현토 연습: 고전 구절

### 8.1 논어 1.1 (論語 學而篇)

**원문:**
```
子曰 學而時習之 不亦說乎
有朋自遠方來 不亦樂乎
人不知而不慍 不亦君子乎
```

**현토:**
```
子ㅣ 曰하시되 學하고 時로 之를 習하면 不亦 說하지 아니한가
朋이 有하여 遠方으로부터 來하면 不亦 樂하지 아니한가
人이 知하지 아니하여도 慍하지 아니하면 不亦 君子가 아닌가
```

**석독:**
```
공자께서 말씀하시기를, 배우고 때때로 그것을 익히면 또한 기쁘지 아니한가.
벗이 있어 먼 곳에서 찾아오면 또한 즐겁지 아니한가.
남이 알아주지 아니하여도 원망하지 아니하면 또한 군자가 아닌가.
```

### 8.2 대학 서문 (大學)

**원문:**
```
大學之道 在明明德 在親民 在止於至善
```

**현토:**
```
大學의 道는 明德을 明하는 데 在하며
民을 親하는 데 在하며
至善에 止하는 데 在하다
```

**석독:**
```
대학의 도는 밝은 덕을 밝히는 데 있으며,
백성을 새롭게 하는 데 있으며,
지극한 선에 머무는 데 있다.
```

## 9. 독해 전략과 요령

### 9.1 단계별 독해법

> **5단계 한국식 독해법**
>
> 1. 음독으로 먼저 읽기 — 소리 파악
>    孔子學禮 → 공자학례
>
> 2. 문법 파악 — 주어? 목적어? 동사?
>    孔子(S) 學(V) 禮(O)
>
> 3. 현토 달기 — 한국어 조사 삽입
>    孔子가 禮를 學하다
>
> 4. 한국어 어순으로 재배열
>    S + O + V → 공자가 례를 학하다
>
> 5. 석독 — 완전한 한국어 번역
>    공자가 예를 배우다
>    (공자가 예를 배웠다)

### 9.2 알아두어야 할 핵심 패턴

| 한문 패턴 | 한국어 읽기 | 번역 패턴 |
|----------|-----------|----------|
| A + 형용사 | A이/가 + 형용사하다 | A는 [형용사]이다 |
| S + V + O | S이/가 + O을/를 + V하다 | S가 O를 V하다 |
| A者B也 | A는 B이다 | A는 B이다 |
| A之B | A의 B | A의 B |
| 不 + V | V하지 않다 | V하지 않다 |
| 有 + N | N이 있다 | N이 있다 |
| 無 + N | N이 없다 | N이 없다 |
| V乎 | V하는가 | V하는가? |

### 9.3 흔한 실수 방지

| 실수 | 틀린 이유 | 올바른 접근 |
|------|----------|-----------|
| SVO를 SOV로 읽기 | 한문은 SVO, 한국어는 SOV | 목적어를 동사 앞으로 재배열 |
| 之를 소유격으로 놓치기 | 之는 흔히 "의" | 之가 두 명사를 연결하는지 확인 |
| 而를 무시하기 | 而는 절의 경계를 표시 | "~하고" 또는 "~하나"로 번역 |
| 也를 독립된 단어로 취급 | 也는 문말 어기사 | 계사 "-이다"로 번역 |

## 10. 연습 문제

### 연습 1: 현토 달기

각 문장에 한국어 조사(현토)를 달아 보세요:

```
1. 天高
2. 王愛民
3. 人讀書
4. 花開
5. 學者覺也
```

<details>
<summary>정답 확인</summary>

```
1. 天이 高하다 (하늘이 높다)
2. 王이 民을 愛하다 (왕이 백성을 사랑하다)
3. 人이 書를 讀하다 (사람이 책을 읽다)
4. 花가 開하다 (꽃이 피다)
5. 學이라 하는 者는 覺이다 (배움이란 것은 깨달음이다)
```

</details>

### 연습 2: 현토에서 석독으로

현토 읽기를 완전한 석독(한국어 번역)으로 변환하세요:

```
1. 天이 高하고 地가 厚하다
2. 人이 書를 讀하면 知가 明하다
3. 王이 民을 愛하면 民이 王을 敬하다
4. 學하고 時로 之를 習하다
5. 仁者는 人을 愛하다
```

<details>
<summary>정답 확인</summary>

```
1. 하늘이 높고 땅이 두껍다
   (하늘이 높고 땅이 두껍다)

2. 사람이 책을 읽으면 지혜가 밝아진다
   (사람이 책을 읽으면 지혜가 밝아진다)

3. 왕이 백성을 사랑하면 백성이 왕을 공경한다
   (왕이 백성을 사랑하면 백성이 왕을 공경한다)

4. 배우고 때때로 그것을 익히다
   (배우고 때때로 그것을 익히다)

5. 어진 사람은 사람을 사랑한다
   (어진 사람은 사람을 사랑한다)
```

</details>

### 연습 3: 한국어 어순으로 재배열

각 한문 문장을 한국어(SOV) 어순으로 재배열하고 조사를 달아 보세요:

```
1. 孔子學禮 (S-V-O → S-O-V)
2. 師敎子 (S-V-O → S-O-V)
3. 民耕田 (S-V-O → S-O-V)
4. 臣事君 (S-V-O → S-O-V)
5. 父敎子禮 (S-V-IO-DO → S-IO-DO-V)
```

<details>
<summary>정답 확인</summary>

```
1. 孔子學禮 → 孔子가 禮를 學하다 (공자가 예를 배우다)
2. 師敎子 → 師가 子를 敎하다 (스승이 아이를 가르치다)
3. 民耕田 → 民이 田을 耕하다 (백성이 밭을 갈다)
4. 臣事君 → 臣이 君을 事하다 (신하가 임금을 섬기다)
5. 父敎子禮 → 父가 子에게 禮를 敎하다 (아버지가 아들에게 예를 가르치다)
```

</details>

### 연습 4: 완전한 3단계 읽기

각 문장에 대해 세 단계(음독 / 현토 / 석독) 모두를 제시하세요:

```
1. 人不學則不知
2. 溫故而知新
3. 有朋自遠方來
```

<details>
<summary>정답 확인</summary>

```
1. 人不學則不知
   음독: 인불학즉부지
   현토: 人이 學하지 아니하면 則 知하지 못하다
   석독: 사람이 배우지 않으면 곧 알지 못한다

2. 溫故而知新
   음독: 온고이지신
   현토: 故를 溫하고 新을 知하다
   석독: 옛것을 익히고 새것을 알다

3. 有朋自遠方來
   음독: 유붕자원방래
   현토: 朋이 有하여 遠方으로부터 來하다
   석독: 벗이 있어 먼 곳에서 찾아오다
```

</details>

### 연습 5: 구절 읽기

다음 구절을 세 단계로 모두 읽어 보세요:

```
子曰 吾十有五而志于學 三十而立 四十而不惑
```

<details>
<summary>정답 확인</summary>

```
음독:
자왈 오십유오이지우학 삼십이립 사십이불혹

현토:
子ㅣ 曰하시되 吾가 十有五에 學에 志하고
三十에 立하고 四十에 惑하지 아니하다

석독:
공자께서 말씀하시기를,
나는 열다섯 살에 학문에 뜻을 두었고,
서른 살에 (뜻이) 확립되었고,
마흔 살에 미혹되지 않았다.

영어:
The Master said: "At fifteen, I set my heart on learning.
At thirty, I stood firm.
At forty, I had no doubts."
```

</details>
