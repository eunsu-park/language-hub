## 1. 맹자(孟子) 소개

### 1.1 맹자는 누구인가?

> **孟子 (맹자) — 맹자**
>
> - **이름**: 孟軻 (맹가), 자(字) 子輿 (자여)
> - **출생**: 기원전 372년경, 추(鄒)나라
> - **사망**: 기원전 289년경
> - **칭호**: 亞聖 (아성, "아성" — 공자 다음가는 성인)
>
> 주요 공헌:
>   - 性善說 (성선설) — 인간의 본성이 선하다는 이론
>   - 四端 (사단) — 도덕의 네 가지 실마리
>   - 王道政治 (왕도정치) — 왕도를 통한 정치
>   - 民本思想 (민본사상) — 백성이 나라의 근본
>
> 맹자(孟子)는 7편이며, 각 편이 상하로 나뉩니다 (총 14편).
> 논어에 비해 보다 논쟁적이고 대화가 풍부한 문체입니다.

### 1.2 맹자 vs 공자: 상호 보완적 사상가

| 항목 | 공자(孔子) | 맹자(孟子) |
|------|-----------|-----------|
| 시대 | 기원전 551-479년 | 기원전 372-289년경 |
| 칭호 | 至聖 (지성) | 亞聖 (아성) |
| 핵심 | 禮(예절), 仁(인) | 義(의로움), 性善(선한 본성) |
| 문체 | 간결한 어록 (語錄體) | 확장된 논변 (論辯體) |
| 정치관 | 주나라 예법 질서 회복 | 인정(仁政)으로 다스리기 |
| 인간 본성 | 명시적으로 논하지 않음 | 명시적으로 선하다고 주장 (性善說) |

## 2. 사서(四書)

### 2.1 개요

사서(四書/사서)는 주희(朱熹/주희, 1130-1200)가 유교 핵심 교육 과정으로 확립한 것입니다. 조선시대 내내 과거시험(科擧試驗/과거시험)의 주요 교재였습니다.

| 서명 | 한국어 | 시기 | 핵심 주제 | 분량 |
|------|--------|------|----------|------|
| 大學 | 대학 (대학) | 선진시대 | 수신 → 치국 | 짧음 (1편) |
| 論語 | 논어 (논어) | 기원전 5~3세기경 | 공자의 어록 | 20편 |
| 孟子 | 맹자 (맹자) | 기원전 3세기경 | 맹자의 논변 | 7편 × 2 |
| 中庸 | 중용 (중용) | 선진시대 | 균형과 조화 | 짧음 (33절) |

### 2.2 대학(大學) — 서두 구절

```
大學之道 在明明德 在親民 在止於至善
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 大學之道 | 대학지도 | 대학의 도는 |
| 在明明德 | 재명명덕 | 밝은 덕을 밝히는 데 있으며 |
| 在親民 | 재친민 | 백성을 새롭게 하는 데 있으며 |
| 在止於至善 | 재지어지선 | 지극한 선에 머무는 데 있다 |

### 2.3 중용(中庸) — 서두

```
天命之謂性 率性之謂道 修道之謂敎
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 天命之謂性 | 천명지위성 | 하늘이 명한 것을 성(性)이라 한다 |
| 率性之謂道 | 솔성지위도 | 성을 따르는 것을 도(道)라 한다 |
| 修道之謂敎 | 수도지위교 | 도를 닦는 것을 교(敎)라 한다 |

## 3. 性善說: 인간의 본성은 선하다

### 3.1 핵심 논거

맹자의 가장 유명한 철학적 입장은 인간의 본성(性/성)이 본래 선(善/선)하다는 것입니다.

### 3.2 원문: 물의 비유

```
孟子曰 人性之善也 猶水之就下也
人無有不善 水無有不下
```

### 3.3 글자별 주석

| 한자 | 음독 | 훈독 | 문맥상 의미 |
|------|------|------|------------|
| 人 | 인 | 사람 인 | 사람, 인간 |
| 性 | 성 | 성품 성 | 본성, 타고난 성품 |
| 善 | 선 | 착할 선 | 선하다, 착하다 |
| 猶 | 유 | 오히려 유 | 마치 ~와 같다 |
| 水 | 수 | 물 수 | 물 |
| 就 | 취 | 나아갈 취 | 나아가다, 향하다 |
| 下 | 하 | 아래 하 | 아래로 |
| 無有 | 무유 | — | ~한 경우가 없다 |
| 不善 | 불선 | — | 선하지 않은 |
| 不下 | 불하 | — | 아래로 [흐르지] 않는 |

### 3.4 현토와 석독

**현토:**
```
孟子ㅣ 曰하시되 人의 性이 善한 것은 猶 水가 下로 就함과 같다
人이 善하지 않음이 없고 水가 下로 [흐르지] 않음이 없다
```

**석독:**
```
맹자께서 말씀하셨다.
"사람의 본성이 선한 것은, 물이 아래로 흐르는 것과 같다.
사람치고 선하지 않은 이가 없고, 물치고 아래로 흐르지 않는 것이 없다."
```

### 3.5 문법 분석

| 패턴 | 예시 | 설명 |
|------|------|------|
| A之B也 猶C之D也 | 人性之善也 猶水之就下也 | "A가 B한 것은 C가 D한 것과 같다" — 유추 패턴 |
| 無有不 | 無有不善 | "선하지 않은 것이 없다" → 모두 선하다 |

## 4. 四端: 네 가지 실마리

### 4.1 사단이란?

맹자는 모든 사람이 네 가지 타고난 도덕적 경향(四端/사단)을 가지고 태어나며, 이것이 네 가지 기본 덕목의 "실마리" 또는 "싹"이라고 주장했습니다:

| 단(실마리/싹) | 음독 | 타고난 감정 | 이르는 덕목 |
|--------------|------|-----------|------------|
| 惻隱之心 | 측은지심 | 측은히 여기는 마음 | 仁 (인) |
| 羞惡之心 | 수오지심 | 부끄러워하는 마음 | 義 (의로움) |
| 辭讓之心 | 사양지심 | 사양하는 마음 | 禮 (예절) |
| 是非之心 | 시비지심 | 옳고 그름을 가리는 마음 | 智 (지혜) |

### 4.2 원문

```
惻隱之心 仁之端也
羞惡之心 義之端也
辭讓之心 禮之端也
是非之心 智之端也
```

### 4.3 석독

```
측은히 여기는 마음은 인(仁)의 실마리이고,
부끄러워하는 마음은 의(義)의 실마리이며,
사양하는 마음은 예(禮)의 실마리이고,
옳고 그름을 가리는 마음은 지(智)의 실마리이다.
```

### 4.4 상세 어휘

| 용어 | 음독 | 의미 |
|------|------|------|
| 惻隱 | 측은 | 동정, 연민; 불쌍히 여김 |
| 羞惡 | 수오 | 수치심과 미워함; 도덕적 혐오 |
| 辭讓 | 사양 | 사양, 양보; 겸양 |
| 是非 | 시비 | 옳고 그름; 도덕적 판단 |
| 端 | 단 | 실마리, 싹, 단서 |

## 5. 惻隱之心: 측은히 여기는 마음

### 5.1 우물가의 어린아이

맹자의 가장 유명한 사유 실험:

```
人皆有不忍人之心
所以謂人皆有不忍人之心者
今人乍見孺子將入於井
皆有怵惕惻隱之心
非所以內交於孺子之父母也
非所以要譽於鄉黨朋友也
非惡其聲而然也
```

### 5.2 핵심 구절 주석

| 구절 | 음독 | 번역 |
|------|------|------|
| 人皆有不忍人之心 | 인개유불인인지심 | 사람은 누구나 남의 [고통을] 차마 보지 못하는 마음을 가지고 있다 |
| 今人乍見孺子 | 금인사견유자 | 지금 어떤 사람이 갑자기 어린아이를 보고 |
| 將入於井 | 장입어정 | 우물에 빠지려 한다면 |
| 皆有怵惕惻隱之心 | 개유출척측은지심 | 누구나 놀라고 측은한 마음이 생길 것이니 |
| 非所以內交於孺子之父母也 | 비소이내교어유자지부모야 | 그 아이의 부모와 사귀려고 해서가 아니며 |
| 非所以要譽於鄉黨朋友也 | 비소이요예어향당붕우야 | 이웃과 친구들에게 칭찬을 받으려고 해서가 아니고 |
| 非惡其聲而然也 | 비오기성이연야 | (그 아이의) 울음소리가 싫어서 그런 것도 아니다 |

### 5.3 석독

```
사람은 누구나 남의 고통을 차마 보지 못하는 마음을 가지고 있다.
내가 사람에게 누구나 차마 그러지 못하는 마음이 있다고 말하는 이유는 이러하다.
지금 어떤 사람이 갑자기 어린아이가 우물에 빠지려는 것을 보면,
누구나 놀라고 측은한 마음이 생기리니,
이것은 그 아이의 부모와 사귀려고 해서가 아니며,
이웃과 친구들에게 칭찬을 받으려고 해서가 아니고,
(그 아이의) 울음소리가 싫어서 그런 것도 아니다.
```

## 6. 五倫: 다섯 가지 인간관계

### 6.1 원문

```
父子有親 君臣有義 夫婦有別 長幼有序 朋友有信
```

### 6.2 상세 분석

| 관계 | 음독 | 양 당사자 | 지배하는 덕목 |
|------|------|----------|-------------|
| 父子有親 | 부자유친 | 아버지(父)와 아들(子) | 親 (친함) |
| 君臣有義 | 군신유의 | 임금(君)과 신하(臣) | 義 (의로움) |
| 夫婦有別 | 부부유별 | 남편(夫)과 아내(婦) | 別 (구별) |
| 長幼有序 | 장유유서 | 어른(長)과 어린이(幼) | 序 (차례) |
| 朋友有信 | 붕우유신 | 벗(朋)과 벗(友) | 信 (믿음) |

### 6.3 석독

```
아버지와 아들 사이에는 친함이 있고,
임금과 신하 사이에는 의로움이 있으며,
남편과 아내 사이에는 구별이 있고,
어른과 어린이 사이에는 차례가 있으며,
벗과 벗 사이에는 믿음이 있다.
```

### 6.4 한국 문화에서의 오륜

| 관계 | 한국어 표현 | 문화적 영향 |
|------|-----------|------------|
| 父子有親 | 부자유친 | 한국 가족 구조의 토대 (효도) |
| 君臣有義 | 군신유의 | 통치자/조직에 대한 충성 (충성) |
| 夫婦有別 | 부부유별 | 성별 역할 (역사적으로 엄격한 구분) |
| 長幼有序 | 장유유서 | 나이 기반 위계질서 (존댓말 체계) |
| 朋友有信 | 붕우유신 | 우정 윤리 (의리) |

## 8. 맹자 추가 선독

### 8.1 백성이 가장 귀하다

```
孟子曰 民為貴 社稷次之 君為輕
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 民為貴 | 민위귀 | 백성이 가장 귀하고 |
| 社稷次之 | 사직차지 | 사직이 그다음이며 |
| 君為輕 | 군위경 | 임금이 가장 가볍다 |

**석독:**
```
맹자께서 말씀하셨다.
"백성이 가장 귀하고, 사직이 그다음이며, 임금이 가장 가볍다."
```

> 이 구절은 고대에 민주적 정신을 보여주는 점에서 주목할 만합니다. 民本思想(민본사상, "백성이 근본이라는 사상")은 맹자의 가장 급진적인 정치적 공헌이었습니다.

### 8.2 인정(仁政)

```
老吾老以及人之老 幼吾幼以及人之幼
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 老吾老 | 노오노 | 나의 어른을 공경하여 |
| 以及人之老 | 이급인지노 | 남의 어른에게까지 미치고 |
| 幼吾幼 | 유오유 | 나의 어린이를 사랑하여 |
| 以及人之幼 | 이급인지유 | 남의 어린이에게까지 미친다 |

**석독:**
```
나의 어른을 공경하여 남의 어른에게까지 미치고,
나의 어린이를 사랑하여 남의 어린이에게까지 미친다.
```

### 8.3 물고기와 의로움

```
生亦我所欲也 義亦我所欲也
二者不可得兼 舍生而取義者也
```

**석독:**
```
삶도 내가 원하는 바이요, 의로움도 내가 원하는 바이다.
둘을 다 얻을 수 없으면, 삶을 버리고 의를 취하리라.
```

---

## 9. 맹자의 문법 패턴

### 9.1 유추 패턴: A猶B

猶 (유, "~와 같다")로 유추를 만듭니다:

| 예시 | 음독 | 번역 |
|------|------|------|
| 人性之善也 猶水之就下也 | 인성지선야 유수지취하야 | 인간의 본성이 선한 것은 물이 아래로 흐르는 것과 같다 |

### 9.2 전칭 패턴: 皆有

皆有 (개유, "모두 가지고 있다")로 보편적 소유를 진술합니다:

| 예시 | 음독 | 번역 |
|------|------|------|
| 人皆有不忍人之心 | 인개유불인인지심 | 사람은 누구나 차마 [남의 고통을] 보지 못하는 마음을 가지고 있다 |

### 9.3 A亦B也 패턴

亦 (역, "또한")으로 병렬 단언을 만듭니다:

| 예시 | 음독 | 번역 |
|------|------|------|
| 生亦我所欲也 | 생역아소욕야 | 삶도 내가 원하는 바이다 |
| 義亦我所欲也 | 의역아소욕야 | 의로움도 내가 원하는 바이다 |

### 9.4 非所以...也 패턴

非所以...也 (비소이...야)는 "~하려고 해서가 아니다"를 의미합니다:

| 예시 | 음독 | 번역 |
|------|------|------|
| 非所以內交於孺子之父母也 | 비소이내교어유자지부모야 | 그 아이의 부모와 사귀려고 해서가 아니다 |

## 10. 연습 문제

### 연습 1: 사단 짝짓기

각 마음(心)에 해당하는 덕목을 짝지으세요:

```
A열 (마음)               B열 (덕목)
1. 惻隱之心               a. 智 (지혜)
2. 羞惡之心               b. 仁 (인)
3. 辭讓之心               c. 義 (의로움)
4. 是非之心               d. 禮 (예절)
```

<details>
<summary>정답 확인</summary>

```
1-b: 惻隱之心 (측은지심, 연민) → 仁 (인, 어짊)
2-c: 羞惡之心 (수오지심, 수치심) → 義 (의, 의로움)
3-d: 辭讓之心 (사양지심, 겸양) → 禮 (례, 예절)
4-a: 是非之心 (시비지심, 옳고 그름) → 智 (지, 지혜)
```

</details>

### 연습 2: 오륜

각 관계에 지배하는 덕목을 채우세요:

```
1. 父子有___
2. 君臣有___
3. 夫婦有___
4. 長幼有___
5. 朋友有___
```

<details>
<summary>정답 확인</summary>

```
1. 父子有親 (부자유친) — 부모와 자식 사이의 친함
2. 君臣有義 (군신유의) — 임금과 신하 사이의 의로움
3. 夫婦有別 (부부유별) — 남편과 아내 사이의 구별
4. 長幼有序 (장유유서) — 어른과 아이 사이의 차례
5. 朋友有信 (붕우유신) — 벗과 벗 사이의 믿음
```

</details>

### 연습 3: 번역

다음 맹자 구절을 번역하세요:

```
1. 人性之善也 猶水之就下也
2. 惻隱之心 仁之端也
3. 民為貴 社稷次之 君為輕
4. 老吾老以及人之老
5. 舍生而取義者也
```

<details>
<summary>정답 확인</summary>

```
1. 人性之善也 猶水之就下也 (인성지선야 유수지취하야)
   인간 본성의 선함은 물이 아래로 흐르는 것과 같다.

2. 惻隱之心 仁之端也 (측은지심 인지단야)
   측은히 여기는 마음은 인(仁)의 실마리이다.

3. 民為貴 社稷次之 君為輕 (민위귀 사직차지 군위경)
   백성이 가장 귀하고, 사직이 그다음이며,
   임금이 가장 가볍다.

4. 老吾老以及人之老 (노오노이급인지노)
   나의 어른을 공경하여 남의 어른에게까지 미친다.

5. 舍生而取義者也 (사생이취의자야)
   삶을 버리고 의를 취하리라.
```

</details>

### 연습 4: 오상 어휘

오상(五常) 각각의 한자, 음독, 의미를 쓰세요:

```
1. 인(어짊): ___ (___) ___
2. 의(의로움): ___ (___) ___
3. 예(예절): ___ (___) ___
4. 지(지혜): ___ (___) ___
5. 신(믿음): ___ (___) ___
```

<details>
<summary>정답 확인</summary>

```
1. 인(어짊): 仁 (인) 어질 인
2. 의(의로움): 義 (의) 옳을 의
3. 예(예절): 禮 (례) 예도 례
4. 지(지혜): 智 (지) 지혜 지
5. 신(믿음): 信 (신) 믿을 신
```

</details>

### 연습 5: 현토 달기

다음 맹자 구절에 현토를 달아 보세요:

```
1. 人皆有不忍人之心
```

```
   (측은히 여기는 마음은 인의 실마리이다)

3. 民이 貴하다
   (백성이 귀하다)
```

</details>
