## 1. 장자(莊子) 개요

장자/莊子(Zhuangzi)는 도가 사상의 두 근본 경전 가운데 하나로, 철학자 장주/莊周(기원전 약 369-286년)의 저작으로 전해집니다. 간결하고 수수께끼 같은 도덕경과 달리, 장자는 풍부한 서사, 대화, 우화, 그리고 유머를 통해 철학적 비전을 전달합니다.

### 장자의 구성

| 구분 | 한자 | 한국어 | 편수 | 저작 추정 |
|------|------|--------|------|----------|
| 내편 | 內篇 | 내편 (naepyeon) | 1-7 | 장자 자신의 저작으로 추정 |
| 외편 | 外篇 | 외편 (oepyeon) | 8-22 | 제자 및 후대 편집자 |
| 잡편 | 雜篇 | 잡편 (jappyeon) | 23-33 | 다수의 저자 |

### 주요 편목

| 편명 | 한자 | 한국어 | 주제 |
|------|------|--------|------|
| 소요유 | 逍遙遊 | 소요유 | 절대적 자유 |
| 제물론 | 齊物論 | 제물론 | 관점의 상대성 |
| 양생주 | 養生主 | 양생주 | 본성의 보존 |
| 인간세 | 人間世 | 인간세 | 세상 속에서의 삶 |
| 덕충부 | 德充符 | 덕충부 | 내면의 덕 |
| 대종사 | 大宗師 | 대종사 | 궁극적 실재 |
| 응제왕 | 應帝王 | 응제왕 | 통치 |

## 2. 소요유: 자유로운 거닐음

### 소요유/逍遙遊 — 장자 제1편

첫 번째 편은 거대한 붕새의 우화를 통해 장자가 그리는 절대적 자유의 비전을 제시합니다.

### 제목 분석

| 글자 | 독음 | 의미 |
|------|------|------|
| 逍 | 소 (so) | 거닐다 |
| 遙 | 요 (yo) | 멀리 |
| 遊 | 유 (yu) | 노닐다, 거닐다 |

소요유는 문자 그대로 "자유롭고 거리낌 없는 거닐음" — 구속, 의존, 걱정 없이 삶을 헤쳐 나가는 경지를 뜻합니다.

## 3. 붕새: 무한한 시야

### 서두 구절

```
北冥有魚 其名爲鯤
북명유어 기명위곤

鯤之大 不知其幾千里也
곤지대 부지기기천리야

化而爲鳥 其名爲鵬
화이위조 기명위붕

鵬之背 不知其幾千里也
붕지배 부지기기천리야
```

### 글자별 분석

**제1행: 北冥有魚 其名爲鯤**

| 글자 | 독음 | 의미 |
|------|------|------|
| 北 | 북 (buk) | 북쪽 |
| 冥 | 명 (myeong) | 어둡다, 깊은 바다 |
| 有 | 유 (yu) | 있다 |
| 魚 | 어 (eo) | 물고기 |
| 其 | 기 (gi) | 그 |
| 名 | 명 (myeong) | 이름 |
| 爲 | 위 (wi) | ~이라 한다 |
| 鯤 | 곤 (gon) | 곤 (거대한 물고기) |

해석: "어두운 북쪽 바다에 물고기가 있으니, 그 이름을 곤이라 한다."

**제2행: 鯤之大 不知其幾千里也**

| 글자 | 독음 | 의미 |
|------|------|------|
| 之 | 지 (ji) | (~의) |
| 大 | 대 (dae) | 크다 |
| 不知 | 부지 (buji) | 알지 못하다 |
| 幾 | 기 (gi) | 몇 |
| 千里 | 천리 (cheonri) | 천 리 |
| 也 | 야 (ya) | (강조) |

해석: "곤의 크기는 몇 천 리인지 알 수 없다."

**제3행: 化而爲鳥 其名爲鵬**

| 글자 | 독음 | 의미 |
|------|------|------|
| 化 | 화 (hwa) | 변하다 |
| 而 | 이 (i) | 그리하여 |
| 鳥 | 조 (jo) | 새 |
| 鵬 | 붕 (bung) | 붕 (거대한 새) |

해석: "변하여 새가 되니, 그 이름을 붕이라 한다."

### 이어지는 구절

```
怒而飛 其翼若垂天之雲
노이비 기익약수천지운

是鳥也 海運則將徙於南冥
시조야 해운즉장사어남명
```

| 행 | 해석 |
|----|------|
| 怒而飛 | 힘을 떨쳐 날아오르면 |
| 其翼若垂天之雲 | 그 날개는 하늘에 드리운 구름과 같다 |
| 是鳥也 | 이 새는 |
| 海運則將徙於南冥 | 바다가 요동치면 남쪽 어두운 바다로 옮겨가려 한다 |

주요 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 怒 | 노 (no) | 힘을 떨치다, 기운을 쏟다 |
| 翼 | 익 (ik) | 날개 |
| 若 | 약 (yak) | ~같다, ~과 같이 |
| 垂 | 수 (su) | 드리우다 |
| 雲 | 운 (un) | 구름 |
| 徙 | 사 (sa) | 옮겨가다, 이동하다 |

### 작은 것과 큰 것

이 편은 계속해서 거대한 붕새를 작은 생물과 대비합니다:

```
蜩與學鳩笑之曰
조여학구소지왈

我決起而飛 搶榆枋而止
아결기이비 창유방이지

時則不至 而控於地而已矣
시즉불지 이공어지이이의
```

"매미와 어린 비둘기가 [붕새를] 비웃으며 말했다: '우리는 훌쩍 날아올라 느릅나무나 박달나무에 이르러 내려앉지. 때로는 거기에도 닿지 못하고 그냥 땅에 내려앉을 뿐이야.'"

**철학적 논점**: 작은 생물은 붕새의 여행을 이해할 수 없습니다. 그들 자신의 경험이 너무 제한되어 있기 때문입니다. 마찬가지로 좁은 시야를 가진 사람은 넓은 시야를 가진 사람을 이해할 수 없습니다.

## 4. 제물론: 만물의 평등

### 제물론/齊物論 — 장자 제2편

이 편은 모든 구별 — 옳고 그름, 아름다움과 추함, 큼과 작음 — 이 상대적이며 궁극적으로 허상임을 논합니다.

### 핵심 구절: 천뢰(天籟)

```
天地與我並生
천지여아병생

而萬物與我爲一
이만물여아위일
```

| 글자 | 독음 | 의미 |
|------|------|------|
| 天地 | 천지 (cheonji) | 하늘과 땅 |
| 與 | 여 (yeo) | ~와 함께 |
| 我 | 아 (a) | 나 |
| 並 | 병 (byeong) | 함께 |
| 生 | 생 (saeng) | 나다, 생겨나다 |
| 萬物 | 만물 (manmul) | 만물 (모든 것) |
| 爲一 | 위일 (wiil) | 하나가 되다 |

해석: "천지는 나와 함께 생겨났고, 만물은 나와 하나이다."

### 관점의 상대성에 대하여

```
方生方死 方死方生
방생방사 방사방생

方可方不可 方不可方可
방가방불가 방불가방가
```

| 구절 | 해석 |
|------|------|
| 方生方死 | 태어나는 순간 이미 죽음이 있고 |
| 方死方生 | 죽는 순간 이미 삶이 있다 |
| 方可方不可 | 옳은 것이 동시에 그른 것이고 |
| 方不可方可 | 그른 것이 동시에 옳은 것이다 |

주요 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 方 | 방 (bang) | 바로 그 순간에, 동시에 |
| 可 | 가 (ga) | 옳다, 긍정할 수 있다 |

## 5. 호접지몽

### 장주몽접/莊周夢蝶 — 장자의 나비 꿈

중국 철학에서 가장 유명한 구절입니다:

```
昔者莊周夢爲胡蝶
석자장주몽위호접

栩栩然胡蝶也
허허연호접야

自喩適志與
자유적지여

不知周也
부지주야

俄然覺 則蘧蘧然周也
아연각 즉거거연주야

不知周之夢爲胡蝶與
부지주지몽위호접여

胡蝶之夢爲周與
호접지몽위주여
```

### 상세 분석

**제1행: 昔者莊周夢爲胡蝶**

| 글자 | 독음 | 의미 |
|------|------|------|
| 昔者 | 석자 (seokja) | 옛날, 전에 |
| 莊周 | 장주 (jangju) | 장주 (장자) |
| 夢 | 몽 (mong) | 꿈을 꾸다 |
| 爲 | 위 (wi) | ~이 되다 |
| 胡蝶 | 호접 (hojeop) | 나비 |

"옛날에 장주가 꿈에 나비가 되었다."

**제2행: 栩栩然胡蝶也**

| 글자 | 독음 | 의미 |
|------|------|------|
| 栩栩然 | 허허연 (heoheoyeon) | 팔랑팔랑, 생생하게 |
| 也 | 야 (ya) | (강조: 참으로!) |

"팔랑팔랑 나는 나비였다, 참으로!"

**제3행: 自喩適志與**

| 글자 | 독음 | 의미 |
|------|------|------|
| 自 | 자 (ja) | 스스로 |
| 喩 | 유 (yu) | 느끼다, 깨닫다 |
| 適 | 적 (jeok) | 편안하다, 만족하다 |
| 志 | 지 (ji) | 뜻, 마음 |
| 與 | 여 (yeo) | (감탄) |

"스스로 만족하고 편안함을 느꼈다!"

**제4행: 不知周也**

"자신이 장주인 줄 몰랐다."

**제5행: 俄然覺 則蘧蘧然周也**

| 글자 | 독음 | 의미 |
|------|------|------|
| 俄然 | 아연 (ayeon) | 갑자기 |
| 覺 | 각 (gak) | 깨다 |
| 蘧蘧然 | 거거연 (geogeo-yeon) | 확실하게, 분명히 |

"갑자기 잠에서 깨니, 확실하고 분명하게 장주였다."

**제6-7행**: 철학적 핵심:

"장주가 꿈에 나비가 된 것인지, 나비가 꿈에 장주가 된 것인지 알 수 없었다."

### 철학적 의의

| 개념 | 해석 |
|------|------|
| 물화(物化) | "만물의 변화" — 자아와 타자, 꿈과 현실의 경계는 유동적 |
| 정체성 | 우리가 누구인지는 고정되지 않고 끊임없이 변화함 |
| 현실 | 꿈과 깨어 있음을 궁극적으로 구별할 수 없음 |
| 자유 | 진정한 자유는 고정된 정체성에 대한 집착에서 벗어남에서 옴 |

## 6. 노자: 고급 구절

### 도덕경/道德經 제42장 — 만물의 기원

```
道生一    도생일
一生二    일생이
二生三    이생삼
三生萬物  삼생만물
```

| 행 | 해석 |
|----|------|
| 道生一 | 도(道)가 하나를 낳고 |
| 一生二 | 하나가 둘을 낳고 |
| 二生三 | 둘이 셋을 낳고 |
| 三生萬物 | 셋이 만물을 낳는다 |

주요 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 生 | 생 (saeng) | 낳다, 생산하다 |
| 一 | 일 (il) | 하나 (태초의 통일체) |
| 二 | 이 (i) | 둘 (음과 양) |
| 三 | 삼 (sam) | 셋 (음, 양, 그리고 그 상호작용) |

### 제81장 — 마지막 장

```
信言不美    신언불미
美言不信    미언불신

善者不辯    선자불변
辯者不善    변자불선

知者不博    지자불박
博者不知    박자불지
```

| 대구 | 해석 |
|------|------|
| 信言不美 / 美言不信 | 참된 말은 아름답지 않고, 아름다운 말은 참되지 않다 |
| 善者不辯 / 辯者不善 | 선한 사람은 말로 다투지 않고, 말로 다투는 사람은 선하지 않다 |
| 知者不博 / 博者不知 | 아는 사람은 박학하지 않고, 박학한 사람은 참으로 알지 못한다 |

## 7. 물과 유약함

### 도덕경/道德經 제78장

```
天下莫柔弱於水
천하막유약어수

而攻堅強者莫之能勝
이공견강자막지능승

以其無以易之
이기무이역지
```

### 상세 분석

**제1행: 天下莫柔弱於水 (천하막유약어수)**

| 글자 | 독음 | 의미 |
|------|------|------|
| 天下 | 천하 (cheonha) | 천하, 온 세상에 |
| 莫 | 막 (mak) | 없다 |
| 柔弱 | 유약 (yuyak) | 부드럽고 약하다 |
| 於 | 어 (eo) | ~보다 (비교) |
| 水 | 수 (su) | 물 |

해석: "온 세상에 물보다 부드럽고 약한 것은 없다."

**제2행: 而攻堅強者莫之能勝 (이공견강자막지능승)**

| 글자 | 독음 | 의미 |
|------|------|------|
| 攻 | 공 (gong) | 공격하다 |
| 堅 | 견 (gyeon) | 단단하다 |
| 強 | 강 (gang) | 강하다 |
| 者 | 자 (ja) | ~하는 것 |
| 莫之能勝 | 막지능승 | 능히 이길 수 없다 |

해석: "그러나 단단하고 강한 것을 공격하는 데 그것을 이길 수 있는 것은 없다."

### 유약함의 원리

| 개념 | 한자 | 독음 | 의미 |
|------|------|------|------|
| 부드러움이 강함을 이김 | 柔勝剛 | 유승강 | 부드러운 것이 강한 것을 이긴다 |
| 약함이 강함을 이김 | 弱勝強 | 약승강 | 약한 것이 강한 것을 이긴다 |
| 가장 낮은 위치가 이로움 | 下善利 | 하선리 | 낮은 위치가 이로움을 가져옴 |

## 8. 앎과 자기 인식

### 도덕경/道德經 제33장

```
知人者智    지인자지
自知者明    자지자명

勝人者有力    승인자유력
自勝者強      자승자강
```

### 분석

| 행 | 독음 | 해석 |
|----|------|------|
| 知人者智 | 지인자지 | 남을 아는 자는 지혜롭고 |
| 自知者明 | 자지자명 | 자신을 아는 자는 밝다 |
| 勝人者有力 | 승인자유력 | 남을 이기는 자는 힘이 있고 |
| 自勝者強 | 자승자강 | 자신을 이기는 자는 참으로 강하다 |

주요 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 智 | 지 (ji) | 지혜롭다, 슬기롭다 |
| 明 | 명 (myeong) | 밝다, 깨달음 |
| 勝 | 승 (seung) | 이기다, 극복하다 |
| 力 | 력 (ryeok) | 힘 |
| 強 | 강 (gang) | 참으로 강하다 |

### 철학적 대비

| 외적 | 내적 | 평가 |
|------|------|------|
| 知人 (남을 앎) | 自知 (자기를 앎) | 자기 인식이 더 높음 |
| 勝人 (남을 이김) | 自勝 (자기를 이김) | 자기 극복이 더 높음 |
| 智 (슬기) | 明 (깨달음) | 깨달음이 슬기를 초월함 |
| 有力 (힘이 있음) | 強 (참된 강함) | 내면의 힘이 외면의 힘을 초월함 |

이 구절은 외적인 것에서 내적인 것으로 나아가며, 참된 지혜와 참된 힘은 타인을 정복하는 데 있지 않고 자기 자신을 다스리는 데 있음을 논합니다.

**연습 2**: 호접지몽은 꿈과 깨어 있음을 참으로 구별할 수 있는지, 어느 쪽이 "진짜"인지 하는 물음을 제기합니다. 장자가 나비였을 때 그는 장자인 줄 전혀 몰랐습니다 — 나비의 현실은 완전했습니다. 깨어난 뒤에도 자신이 나비 꿈을 꾼 장주인지, 장주 꿈을 꾸고 있는 나비인지 판단할 수 없었습니다. 이것은 물화(物化) — 자아와 타자 사이의 유동적 변환 — 의 개념을 보여주며, 고정된 정체성에 대한 우리의 전제에 도전합니다.

**연습 6**: 노자는 知人(남을 앎)과 自知(자기를 앎), 勝人(남을 이김)과 自勝(자기를 이김)을 대비합니다. 각 쌍에서 내적인 것이 더 높이 평가됩니다. 남을 아는 것은 단지 슬기(智)이지만, 자기를 아는 것은 깨달음(明)입니다. 남을 이기는 것은 힘(力)이 필요하지만, 자기를 이기는 것은 참된 강함(強)이 필요합니다. 이것은 외적 성취보다 내면의 수양을 중시하는 도가의 강조점을 반영합니다.
