## 1. 고체시와 근체시

### 고체시(古體詩) — 고풍시

고체시는 당나라 이전에 발전한 것으로, 비교적 자유로운 규칙을 따릅니다.

| 특징 | 설명 |
|------|------|
| 구의 길이 | 유동적 (4, 5, 7자 또는 불규칙) |
| 행의 수 | 제한 없음 |
| 성률 규칙 | 엄격히 요구되지 않음 |
| 대구 | 선택 사항 |
| 압운 | 필수이나 유연함 |
| 시기 | 당나라 이전 및 이후 지속 |

### 근체시(近體詩) — 금체시(율시)

근체시는 당나라 시기에 엄격한 형식 요건을 갖추어 정립되었습니다.

| 특징 | 설명 |
|------|------|
| 구의 길이 | 고정: 5자 또는 7자 |
| 행의 수 | 고정: 4행 (절구) 또는 8행 (율시) |
| 성률 규칙 | 엄격한 평측 패턴 |
| 대구 | 2련과 3련에서 필수 (율시) |
| 압운 | 엄격, 공인된 운서(韻書)에 따름 |
| 시기 | 당나라 이후 |

### 비교표

| 측면 | 고체시 (古體詩) | 근체시 (近體詩) |
|------|----------------|----------------|
| 한국어 명칭 | 고체시 (고풍) | 근체시 (금체) |
| 자유도 | 높음 | 낮음 |
| 작시 난이도 | 상대적으로 낮음 | 높음 |
| 예시 | 詩經, 楚辭, 古樂府 | 절구, 율시 |
| 대표 시인 | 도연명(陶淵明), 이백(李白) | 두보(杜甫), 왕유(王維) |

## 2. 율시: 8행 근체시

### 율시(律詩) 구조

율시는 네 개의 련(聯/련)으로 구성되며, 각 련은 특정 기능을 담당합니다:

| 련 | 이름 | 한자 | 기능 |
|----|------|------|------|
| 제1련 (1-2행) | 수련 | 首聯 | 개시련: 주제/배경 제시 |
| 제2련 (3-4행) | 함련 | 頷聯 | 턱련: 대구로 전개 |
| 제3련 (5-6행) | 경련 | 頸聯 | 목련: 대구로 전개 |
| 제4련 (7-8행) | 미련 | 尾聯 | 꼬리련: 결론/성찰 |

### 율시의 규칙

| 규칙 | 요건 |
|------|------|
| 구의 길이 | 모든 행은 5자 또는 7자 |
| 총 행 수 | 정확히 8행 |
| 대구 | 함련(頷聯)과 경련(頸聯)은 반드시 대구 |
| 압운 | 2, 4, 6, 8행은 반드시 압운; 1행은 선택적 |
| 성률 패턴 | 반드시 평측 규칙을 따름 |
| 운자 중복 금지 | 각 운자는 서로 달라야 함 |

### 대구 요건

함련과 경련에서 대응하는 위치의 글자는 일치해야 합니다:

| 위치 | A행 | B행 | 대응 유형 |
|------|-----|-----|----------|
| 제1자 | 명사 | 명사 | 같은 품사 |
| 제2자 | 명사 | 명사 | 같은 품사 |
| 제3자 | 동사 | 동사 | 같은 품사 |
| 제4자 | 형용사 | 형용사 | 같은 품사 |
| 제5자 | 명사 | 명사 | 같은 품사 |
| 성조 | 평/측 | 측/평 | 반대 |

## 3. 고급 평측 패턴

### 오언율시 (五言律詩) 성률 패턴

○은 평(平)성, ●은 측(仄)성을 나타냅니다:

```
유형 A:
제1행: ○○●●○  (압운 선택)
제2행: ●●●○○  (압운)
제3행: ●●○○●
제4행: ○○●●○  (압운)
제5행: ○○○●●
제6행: ●●●○○  (압운)
제7행: ●●○○●
제8행: ○○●●○  (압운)
```

### 칠언율시 (七言律詩) 성률 패턴

```
유형 A:
제1행: ●●○○●●○  (압운 선택)
제2행: ○○●●●○○  (압운)
제3행: ○○●●○○●
제4행: ●●○○●●○  (압운)
제5행: ●●○○○●●
제6행: ○○●●●○○  (압운)
제7행: ○○●●○○●
제8행: ●●○○●●○  (압운)
```

### 핵심 규칙

| 규칙 | 한자 | 독음 | 설명 |
|------|------|------|------|
| 1·3·5 불론 | 一三五不論 | 일삼오불론 | 1, 3, 5번째 위치는 성조 유동적 |
| 2·4·6 분명 | 二四六分明 | 이사육분명 | 2, 4, 6번째 위치는 반드시 패턴을 따름 |
| 고평 회피 | 孤平 | 고평 | 평성이 고립되는 것을 피함 |
| 행간 대비 | 對 | 대 | 한 련 내 인접한 행은 성조가 대비됨 |
| 련간 연속 | 粘 | 점 | 다음 련의 첫 행은 이전 련 마지막 행의 성조를 따름 |

## 4. 압운 체계: 압운

### 압운(押韻) — 운율 맞추기

근체시에서 운(韻/운)은 엄격한 규범을 따릅니다.

| 규칙 | 설명 |
|------|------|
| 운자 위치 | 2, 4, 6, 8행 끝 (1행은 선택) |
| 운부 | 같은 운부(韻部, 운의 범주)에서 |
| 운의 성조 | 근체시에서는 반드시 평성(平聲) |
| 중복 금지 | 각 운자는 유일해야 함 |

### 주요 운부(韻部)

| 운부 | 예시 글자 | 한국어 독음 |
|------|----------|------------|
| 東韻 (동운) | 東, 風, 中, 空, 紅 | 동, 풍, 중, 공, 홍 |
| 陽韻 (양운) | 陽, 光, 鄉, 霜, 長 | 양, 광, 향, 상, 장 |
| 眞韻 (진운) | 眞, 人, 春, 新, 塵 | 진, 인, 춘, 신, 진 |
| 先韻 (선운) | 天, 年, 前, 煙, 然 | 천, 년, 전, 연, 연 |
| 魚韻 (어운) | 魚, 書, 居, 車, 初 | 어, 서, 거, 거, 초 |
| 尤韻 (우운) | 樓, 流, 秋, 愁, 舟 | 루, 류, 추, 수, 주 |

## 5. 두보: 등고(登高)

### 두보(杜甫): 登高 (등고, 높이 오르다)

이 시는 역대 최고의 칠언율시로 평가받는 작품입니다.

```
風急天高猿嘯哀    풍급천고원소애
渚清沙白鳥飛回    저청사백조비회
無邊落木蕭蕭下    무변낙목소소하
不盡長江滾滾來    부진장강곤곤래
萬里悲秋常作客    만리비추상작객
百年多病獨登臺    백년다병독등대
艱難苦恨繁霜鬢    간난고한번상빈
潦倒新停濁酒杯    료도신정탁주배
```

### 련별 분석

**수련(首聯): 1-2행 — 배경 묘사**

| 행 | 원문 | 해석 |
|----|------|------|
| 風急天高猿嘯哀 | 풍급천고원소애 | 바람은 매섭고 하늘은 높은데 원숭이가 슬피 울고 |
| 渚清沙白鳥飛回 | 저청사백조비회 | 물가는 맑고 모래는 흰데 새들이 빙빙 날아든다 |

주요 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 急 | 급 (geup) | 급하다, 매섭다 |
| 猿 | 원 (won) | 원숭이, 긴팔원숭이 |
| 嘯 | 소 (so) | 울부짖다 |
| 渚 | 저 (jeo) | 물가, 모래섬 |
| 回 | 회 (hoe) | 돌다, 돌아오다 |

**함련(頷聯): 3-4행 — 대구 전개**

| 행 | 원문 | 해석 |
|----|------|------|
| 無邊落木蕭蕭下 | 무변낙목소소하 | 끝없이 낙엽이 쓸쓸히 떨어지고 |
| 不盡長江滾滾來 | 부진장강곤곤래 | 끝없는 장강이 도도히 흘러온다 |

대구 분석:

| 위치 | 제3행 | 제4행 |
|------|-------|-------|
| 1-2 | 無邊 (무변, 끝없는) | 不盡 (부진, 다함없는) |
| 3-4 | 落木 (낙목, 지는 잎) | 長江 (장강, 양쯔강) |
| 5-6 | 蕭蕭 (소소, 쓸쓸히) | 滾滾 (곤곤, 도도히) |
| 7 | 下 (하, 내려오다) | 來 (래, 다가오다) |

**경련(頸聯): 5-6행 — 개인적 성찰**

| 행 | 원문 | 해석 |
|----|------|------|
| 萬里悲秋常作客 | 만리비추상작객 | 만 리 타향에서 가을을 슬퍼하며 늘 나그네로 지내고 |
| 百年多病獨登臺 | 백년다병독등대 | 한평생 병이 많아 홀로 높은 곳에 오른다 |

| 위치 | 제5행 | 제6행 |
|------|-------|-------|
| 1-2 | 萬里 (만리, 먼 거리) | 百年 (백년, 한평생) |
| 3-4 | 悲秋 (비추, 가을을 슬퍼함) | 多病 (다병, 병이 많음) |
| 5-6 | 常作 (상작, 늘 ~이 됨) | 獨登 (독등, 홀로 오름) |
| 7 | 客 (객, 나그네) | 臺 (대, 대) |

**미련(尾聯): 7-8행 — 결어**

| 행 | 원문 | 해석 |
|----|------|------|
| 艱難苦恨繁霜鬢 | 간난고한번상빈 | 고난과 괴로운 한으로 서리 낀 귀밑머리가 늘어나고 |
| 潦倒新停濁酒杯 | 료도신정탁주배 | 초라한 몸으로 탁주잔을 새로이 놓는다 |

## 6. 이백: 장진주(將進酒)

### 이백(李白): 將進酒 (장진주, 술을 권하며)

이 시는 고체시(古體詩)로, 불규칙한 구조를 지니며 이백의 호방하고 자유분방한 기질을 잘 보여줍니다.

```
君不見黃河之水天上來    군불견황하지수천상래
奔流到海不復回          분류도해불복회
君不見高堂明鏡悲白髮    군불견고당명경비백발
朝如靑絲暮成雪          조여청사모성설
```

### 서두 분석

**제1행: 君不見黃河之水天上來**

| 글자 | 독음 | 의미 |
|------|------|------|
| 君 | 군 (gun) | 그대 (존칭) |
| 不見 | 불견 (bulgyeon) | 보지 못하는가 |
| 黃河 | 황하 (hwangha) | 황하 |
| 之 | 지 (ji) | (~의) |
| 水 | 수 (su) | 물 |
| 天上 | 천상 (cheonsang) | 하늘 위에서 |
| 來 | 래 (rae) | 오다 |

"그대여, 보지 못하는가? 황하의 물이 하늘 위에서 내려오는 것을."

**제2행: 奔流到海不復回**

| 글자 | 독음 | 의미 |
|------|------|------|
| 奔流 | 분류 (bunryu) | 세차게 흐르다 |
| 到 | 도 (do) | 이르다 |
| 海 | 해 (hae) | 바다 |
| 不復 | 불복 (bulbok) | 다시는 ~않다 |
| 回 | 회 (hoe) | 돌아오다 |

"바다로 치달려 가면 다시는 돌아오지 않는 것을!"

**제3-4행**:

"그대여, 보지 못하는가? 높은 집 밝은 거울에 비친 흰 머리카락의 슬픔을 — / 아침에는 푸른 실 같더니 저녁이면 눈이 되었구나."

### 핵심 주제

| 주제 | 표현 | 의미 |
|------|------|------|
| 세월의 흐름 | 朝如靑絲暮成雪 | 하루 사이에 젊음에서 노년으로 |
| 현재를 즐김 | 人生得意須盡歡 | 인생이 뜻대로 될 때 즐거움을 다하라 |
| 운명에 대한 저항 | 天生我材必有用 | 하늘이 내 재능을 낳았으니 반드시 쓸 데가 있으리 |
| 덧없음 | 黃河之水天上來 | 거대한 강물도 흘러가 버림 |

### 후반부의 명구

```
天生我材必有用    천생아재필유용
千金散盡還復來    천금산진환복래
```

"하늘이 내 재주를 낳았으니 반드시 쓸 데가 있을 것이요, / 천금을 다 흩뿌려도 다시 돌아오리라."

| 글자 | 독음 | 의미 |
|------|------|------|
| 天生 | 천생 (cheonsaeng) | 하늘이 낳다 |
| 材 | 재 (jae) | 재주, 재능 |
| 必 | 필 (pil) | 반드시 |
| 千金 | 천금 (cheongeum) | 천금 |
| 散 | 산 (san) | 흩뿌리다 |
| 盡 | 진 (jin) | 다하다 |
| 還 | 환 (hwan) | 다시 |

## 7. 백거이: 장한가(長恨歌)

### 백거이(白居易): 長恨歌 (장한가, 긴 한의 노래)

이 서사시는 840자로 이루어져 있으며, 당 현종(玄宗)과 양귀비(楊貴妃)의 비극적인 사랑 이야기를 들려줍니다.

### 유명한 서두

```
漢皇重色思傾國    한황중색사경국
御宇多年求不得    어우다년구부득
```

| 행 | 해석 |
|----|------|
| 漢皇重色思傾國 | 한나라 황제가 미색을 중히 여겨, 나라를 기울일 미인을 그리워하였으나 |
| 御宇多年求不得 | 천하를 다스린 지 여러 해에 구하였으나 얻지 못했다 |

주요 어휘:

| 글자 | 독음 | 의미 |
|------|------|------|
| 皇 | 황 (hwang) | 황제 |
| 重 | 중 (jung) | 중히 여기다 |
| 色 | 색 (saek) | 아름다움, 미색 |
| 傾國 | 경국 (gyeongguk) | 나라를 기울이다 (절세미인) |
| 御 | 어 (eo) | 다스리다, 어(御) |
| 宇 | 우 (u) | 천하, 우주 |

### 가장 많이 인용되는 구절

```
在天願作比翼鳥    재천원작비익조
在地願爲連理枝    재지원위련리지
```

| 행 | 해석 |
|----|------|
| 在天願作比翼鳥 | 하늘에서는 나란히 나는 비익조가 되기를 원하고 |
| 在地願爲連理枝 | 땅에서는 가지가 얽힌 연리지가 되기를 원한다 |

| 글자 | 독음 | 의미 |
|------|------|------|
| 願 | 원 (won) | 원하다 |
| 比翼 | 비익 (biik) | 나란히 나는 날개 |
| 連理 | 련리 (ryeonri) | 얽힌 나뭇가지 |
| 枝 | 지 (ji) | 가지 |

### 마지막 구절

```
天長地久有時盡    천장지구유시진
此恨綿綿無絕期    차한면면무절기
```

"천지는 언젠가 다할 때가 있어도, / 이 한은 면면히 이어져 끝날 기약이 없으리."

## 8. 사(詞) 형식

### 사(詞, ci) — 서정시

사(詞)는 만당(晩唐) 시기에 발전하여 송나라(960-1279)에 꽃을 피운 시 형식입니다. 근체시와 달리 사(詞)는 특정 악곡 패턴에 맞추어 쓰입니다.

### 사(詞)의 특징

| 특징 | 설명 |
|------|------|
| 별칭 | 장단구(長短句, 길이가 다른 구) |
| 음악 | 본래 특정 악곡(詞牌/사패)에 맞춤 |
| 구의 길이 | 불규칙 — 1자에서 11자까지 |
| 구성 | 궐(闋)로 나뉨 |
| 압운 | 악곡 패턴에 따름 |
| 소재 | 사랑, 자연, 역사, 철학 |

### 사패(詞牌) — 곡조명

| 곡조명 | 한자 | 독음 | 글자 수 | 풍격 |
|--------|------|------|---------|------|
| 如夢令 | 여몽령 | yeomonryeong | 33자 | 짧고 서정적 |
| 菩薩蠻 | 보살만 | bosalman | 44자 | 낭만적 |
| 水調歌頭 | 수조가두 | sujogadu | 95자 | 웅대하고 철학적 |
| 念奴嬌 | 념노교 | nyeomnogyo | 100자 | 장편, 극적 |
| 滿江紅 | 만강홍 | manganghong | 93자 | 영웅적, 우국적 |

### 예시: 소식(蘇軾) — 水調歌頭 (수조가두)

```
明月幾時有    명월기시유
把酒問靑天    파주문청천
不知天上宮闕    부지천상궁궐
今夕是何年    금석시하년
```

| 행 | 해석 |
|----|------|
| 明月幾時有 | 밝은 달은 언제부터 있었던가? |
| 把酒問靑天 | 술잔을 들고 푸른 하늘에 묻노라 |
| 不知天上宮闕 | 하늘 위 궁궐에서는 알 수 없으니 |
| 今夕是何年 | 오늘 밤이 어느 해인지 |

### 유명한 결구:

```
但願人長久    단원인장구
千里共嬋娟    천리공선연
```

"다만 바라노니 사람이 오래 살아, / 천 리 밖에서도 함께 이 아름다운 달빛을 나누기를."

## 9. 기승전결: 시의 구조

### 기승전결(起承轉結) — 4단 구조

이 구조 원리는 시뿐만 아니라 동아시아 문학의 산문과 서사에서도 광범위하게 쓰입니다.

| 단계 | 한자 | 독음 | 기능 | 시적 역할 |
|------|------|------|------|----------|
| 시작 | 起 | 기 (gi) | 도입 | 장면 설정, 주제 제시 |
| 전개 | 承 | 승 (seung) | 계속 | 시작 부분의 정교화, 발전 |
| 전환 | 轉 | 전 (jeon) | 변화 | 관점 전환, 긴장 조성 |
| 결말 | 結 | 결 (gyeol) | 마무리 | 감정적 해결, 성찰 |

### 정야사(靜夜思)에의 적용

| 단계 | 행 | 기능 |
|------|-----|------|
| 起 | 床前明月光 | 배경: 침상 앞의 달빛 |
| 承 | 疑是地上霜 | 전개: 서리가 아닌가? (시각적 유사) |
| 轉 | 舉頭望明月 | 전환: 고개를 들어 보니 달이었음 |
| 結 | 低頭思故鄉 | 결말: 고개를 숙이고 고향을 생각함 |

### 등관작루(登鸛雀樓)에의 적용

| 단계 | 행 | 기능 |
|------|-----|------|
| 起 | 白日依山盡 | 배경: 해가 산 너머로 지고 |
| 承 | 黃河入海流 | 전개: 황하가 바다로 흘러들고 |
| 轉 | 欲窮千里目 | 전환: 천 리 너머까지 보고 싶은 마음 |
| 結 | 更上一層樓 | 결말: 한 층 더 올라가라 (포부의 비유) |

### 감정의 흐름

```
起 (도입)  →  承 (전개)  →  轉 (전환)  →  結 (결말)
  평온/중립       고조         반전        심화된 감정
```
