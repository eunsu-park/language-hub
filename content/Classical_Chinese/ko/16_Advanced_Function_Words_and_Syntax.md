## 1. 所 — 관계절 표지어

### 기본 기능

所(소, so)는 동사를 명사구로 변환하여 "~되는 것" 또는 "~하는 바"의 의미를 만듭니다.

**공식**: 所 + 동사 = "~되는 것 / ~하는 바"

### 핵심 예

| 표현 | 독음 | 직역 | 자연스러운 번역 |
|------|------|------|----------------|
| 所見 | 소견 (sogyeon) | 보이는 바 | 견해, 의견 |
| 所聞 | 소문 (somun) | 들리는 바 | 소문, 듣는 것 |
| 所言 | 소언 (soeon) | 말해지는 바 | 말한 것 |
| 所學 | 소학 (sohak) | 배우는 바 | 배운 것 |
| 所爲 | 소위 (sowi) | 행해지는 바 | 하는 것, 소위 |
| 所知 | 소지 (soji) | 아는 바 | 아는 것 |
| 所有 | 소유 (soyu) | 가지는 바 | 소유물, 가진 것 |
| 所望 | 소망 (somang) | 바라는 바 | 소망, 희망 |

### 所와 행위자

행위자를 명시할 필요가 있을 때는 다음 패턴을 사용합니다:

**공식**: A + 所 + 동사 = "A가 ~하는 바"

| 문장 | 독음 | 번역 |
|------|------|------|
| 吾所欲 | 오소욕 | 내가 바라는 바 |
| 人所不知 | 인소부지 | 사람들이 알지 못하는 바 |
| 天下所歸 | 천하소귀 | 천하가 돌아가는 바 |

### 所以 — "~하는 까닭" / "~하는 수단"

所以(소이)는 두 가지 뚜렷한 의미를 지닌 중요한 복합어입니다:

| 맥락 | 의미 | 예 |
|------|------|-----|
| 이유 | ~하는 까닭 | 所以然 (소이연) — 그러한 까닭 |
| 수단 | ~하는 수단 | 所以治國 (소이치국) — 나라를 다스리는 수단 |

**맹자/孟子에서의 예**:

```
人之所以異於禽獸者幾希
인지소이이어금수자기희
```

"사람이 짐승과 다른 바는 거의 없다."

| 부분 | 독음 | 기능 |
|------|------|------|
| 人 | 인 | 주어: 사람 |
| 之 | 지 | 관형격 조사 |
| 所以 | 소이 | ~하는 바 |
| 異 | 이 | 다르다 |
| 於 | 어 | ~와 (전치사) |
| 禽獸 | 금수 | 짐승 |
| 者 | 자 | 명사화 어미 |
| 幾希 | 기희 | 매우 적다 |

## 2. 者 — 고급 용법

### 단순 명사화를 넘어서

者(자, ja)는 동사나 형용사를 명사로 바꾸는 기본 기능 외에 여러 가지 기능을 합니다.

### 용법 1: 명사화 어미 — "~하는 자 / ~인 것"

| 표현 | 독음 | 번역 |
|------|------|------|
| 仁者 | 인자 | 어진 자 |
| 知者 | 지자 | 지혜로운 자 |
| 勇者 | 용자 | 용감한 자 |
| 學者 | 학자 | 배우는 자, 학자 |
| 強者 | 강자 | 강한 자 |
| 弱者 | 약자 | 약한 자 |

### 용법 2: 주제 표지어 — 판단의 주어를 한정

**패턴**: X者, Y也 = "X에 대해 말하면, 곧 Y이다" / "X는 Y이다"

| 문장 | 독음 | 번역 |
|------|------|------|
| 仁者愛人 | 인자애인 | 어진 자는 사람을 사랑한다 |
| 學者不必爲師 | 학자불필위사 | 배우는 자가 반드시 스승이 되어야 하는 것은 아니다 |
| 廉者不受嗟來之食 | 렴자불수차래지식 | 청렴한 자는 모욕적으로 주는 음식을 받지 않는다 |

### 용법 3: 휴지/열거 표지어

者가 절 뒤에 나타나면, 주된 술어 전에 휴지를 만듭니다:

```
富與貴 是人之所欲也
부여귀 시인지소욕야
```
"부와 귀함은 사람이 바라는 바이다."

### 용법 4: 시간적 — "~할 때 / ~하던 때에"

```
昔者聖王之治天下也
석자성왕지치천하야
```
"옛날 성왕이 천하를 다스리던 때에..."

## 3. 之 — 다용도 조사

之(지, ji)는 한문에서 가장 흔한 허사로, 최소 네 가지 뚜렷한 용법이 있습니다.

### 용법 1: 관형격 — "~의"

| 표현 | 독음 | 번역 |
|------|------|------|
| 天之道 | 천지도 | 하늘의 도 |
| 人之心 | 인지심 | 사람의 마음 |
| 國之本 | 국지본 | 나라의 근본 |
| 學之道 | 학지도 | 배움의 도 |
| 仁義之人 | 인의지인 | 인의의 사람 |

### 용법 2: 대명사 — "그것, 그를, 그들"

| 문장 | 독음 | 번역 |
|------|------|------|
| 知之 | 지지 | 그것을 알다 |
| 取之 | 취지 | 그것을 취하다 |
| 愛之 | 애지 | 그를 사랑하다 |
| 學而時習之 | 학이시습지 | 배우고 때때로 그것을 익히다 |

### 용법 3: 구조적 조사 — 수식어와 명사를 연결

之가 절과 명사를 연결할 때 관계대명사처럼 기능합니다:

| 표현 | 독음 | 번역 |
|------|------|------|
| 所謂之 | 소위지 | 이른바... |
| 可歎之事 | 가탄지사 | 탄식할 만한 일 |
| 不可不知之理 | 불가부지지리 | 반드시 알아야 할 이치 |

### 용법 4: 리듬 충전어

때로 之는 순전히 리듬의 균형을 위해 삽입되며, 문법적 의미는 없습니다:

```
頃之 (경지) — 잠시 후 (之는 리듬적)
久之 (구지) — 오랜 후 (之는 리듬적)
```

## 4. 以 — 수단과 원인

以(이, i)는 "~으로써", "~때문에", "~하기 위하여", 또는 "~을 사용하여"의 뜻을 가진 다용도 전치사/접속사입니다.

### 핵심 의미

| 기능 | 패턴 | 예 | 번역 |
|------|------|-----|------|
| 수단 | 以 + 명사 + 동사 | 以劍殺之 (이검살지) | 칼로 그를 죽이다 |
| 이유 | 以 + 절 | 以其無禮 (이기무례) | 그가 무례했기 때문에 |
| 목적 | 동사 + 以 + 동사 | 學以致用 (학이치용) | 실용에 이르고자 배우다 |
| 지위 | 以 + 명사 | 以師禮待之 (이사례대지) | 스승의 예로 그를 대하다 |
| 기준 | 以 + 명사 + 爲 | 以仁爲本 (이인위본) | 인을 근본으로 삼다 |

### 以를 포함하는 주요 복합어

| 복합어 | 독음 | 의미 |
|--------|------|------|
| 以爲 | 이위 | ~으로 여기다, ~라고 생각하다 |
| 以來 | 이래 | ~이래로 |
| 以上 | 이상 | 이상, ~보다 많은 |
| 以下 | 이하 | 이하, ~보다 적은 |
| 以外 | 이외 | 이외에, ~밖에 |
| 所以 | 소이 | ~하는 까닭; ~하는 수단 |
| 可以 | 가이 | ~할 수 있다 |
| 足以 | 족이 | ~에 충분하다 |

### 예문

| 문장 | 독음 | 번역 |
|------|------|------|
| 以德報怨 | 이덕보원 | 덕으로써 원한을 갚다 |
| 不以物喜 不以己悲 | 불이물희 불이기비 | 외물로 인해 기뻐하지 않고, 자신으로 인해 슬퍼하지 않는다 |
| 何以知之 | 하이지지 | 무엇으로써 이를 아는가? |

## 5. 於/于 — 장소와 비교의 전치사

於(어, eo)와 于(우, u)는 장소, 방향, 비교, 또는 피동의 행위자를 표시하는 호환 가능한 전치사입니다.

### 기능

| 기능 | 패턴 | 예 | 번역 |
|------|------|-----|------|
| 장소 | 동사 + 於 + 장소 | 生於亂世 (생어난세) | 난세에 태어나다 |
| 방향 | 동사 + 於 + 대상 | 問於師 (문어사) | 스승에게 묻다 |
| 비교 | A + 於 + B | 賢於人 (현어인) | 사람보다 어질다 |
| 출처 | 동사 + 於 + 원천 | 取之於民 (취지어민) | 백성에게서 취하다 |
| 피동 행위자 | 동사 + 於 + 행위자 | 見笑於大方 (견소어대방) | 지혜로운 이에게 웃음을 당하다 |
| 시간 | 於 + 시간 | 於是 (어시) | 이에, 그래서 |

### 비교 패턴

X + 於 + Y 패턴은 "X가 Y보다"의 의미입니다:

| 문장 | 독음 | 번역 |
|------|------|------|
| 靑出於藍 | 청출어람 | 쪽에서 나온 청색이 쪽보다 푸르다 (제자가 스승을 능가하다) |
| 甚於水火 | 심어수화 | 수화(水火)보다 심하다 |
| 人固有一死 或重於泰山 或輕於鴻毛 | 인고유일사 혹중어태산 혹경어홍모 | 사람은 반드시 한 번 죽으니, 어떤 죽음은 태산보다 무겁고 어떤 죽음은 기러기 깃털보다 가볍다 |

## 6. 而 — 위대한 접속사

而(이, i)는 한문에서 가장 흔한 접속사로, 다양한 논리적 관계로 절을 연결합니다.

### 기능

| 기능 | 의미 | 예 | 번역 |
|------|------|-----|------|
| 순차 | 그리고, 그런 다음 | 學而知之 (학이지지) | 배워서 그것을 알다 |
| 역접 | 그러나 | 學而不思 (학이불사) | 배우되 생각하지 않다 |
| 양보 | 비록 ~이지만 | 貧而無諂 (빈이무첨) | 가난하되 아첨하지 않다 |
| 조건 | 만일 ~이면 | 人而無信 (인이무신) | 사람이 신의가 없다면 |
| 묘사 | 그리고 (방식) | 默而識之 (묵이지지) | 묵묵히 그것을 기억하다 |

### 而의 의미 구별

문맥이 而의 의미를 결정합니다. 주요 단서:

| 단서 | 가능한 의미 | 예 |
|------|------------|-----|
| 긍정 + 긍정 | 순차 "그리고" | 仁而勇 (인이용) — 어질고 용감한 |
| 긍정 + 부정 | 역접 "그러나" | 學而不厭 (학이불염) — 배우되 싫증내지 않다 |
| 부정 + 긍정 | 양보 "그러나" | 不怒而威 (불노이위) — 노하지 않으되 위엄 있다 |
| 동작 + 동작 | 순차 "그런 다음" | 起而行 (기이행) — 일어나서 걷다 |

## 7. 則 — 조건과 순차

則(즉, jeuk)은 결과, 조건, 또는 시간적 순서를 표시합니다 — "그러면", "이에", "그 경우에".

### 기능

| 기능 | 패턴 | 예 | 번역 |
|------|------|-----|------|
| 조건 | A則B | 學則知 (학즉지) | 배우면 곧 안다 |
| 순차 | A則B | 天明則起 (천명즉기) | 날이 밝으면 곧 일어난다 |
| 대조 | A則B, C則D | 水則載舟 亦則覆舟 (수즉재주 역즉복주) | 물은 배를 띄울 수도, 뒤집을 수도 있다 |
| 강조 | 則 단독 | 非我則誰 (비아즉수) | 내가 아니면 누구인가? |

### 중요한 패턴

| 패턴 | 독음 | 의미 |
|------|------|------|
| A者B也 C者D也 | ...자...야 ...자...야 | A에 대해서 말하면 B이고, C에 대해서 말하면 D이다 |
| 非...則... | 비...즉... | ~이 아니면 곧 ~이다 |
| 不...則... | 불...즉... | ~하지 않으면 곧 ~이다 |

### 논어/論語에서의 예

```
君子不重則不威 學則不固
군자불중즉불위 학즉불고
```

"군자가 무겁지 않으면 위엄이 없고, 배우면 (그 학문이) 굳건하지 않게 되지 않는다."

## 8. 허사의 결합

한문의 진정한 복잡성은 여러 허사가 함께 나타날 때 드러납니다.

### 주요 결합

| 결합 | 독음 | 의미 | 예 |
|------|------|------|-----|
| 所以 | 소이 | ~하는 까닭 / ~하는 수단 | 所以然者 (소이연자) — 그러한 까닭 |
| 所謂 | 소위 | 이른바 | 所謂仁 (소위인) — 이른바 인(仁) |
| 之所以 | 지소이 | (~이) ~하는 까닭 | 人之所以爲人 (인지소이위인) |
| 以...爲 | 이...위 | ~을 ~으로 여기다 | 以德爲本 (이덕위본) — 덕을 근본으로 삼다 |
| 以...而 | 이...이 | ~으로써 ~하다 | 以禮而行 (이례이행) — 예로써 행하다 |
| 之於 | 지어 | (주제) ~에 대하여 | 君之於民 (군지어민) — 군주가 백성에 대하여 |

### 복잡한 문장 예

**예 1**:
```
所謂修身在正其心者
소위수신재정기심자
```

분석:
| 부분 | 독음 | 기능 |
|------|------|------|
| 所謂 | 소위 | 이른바 |
| 修身 | 수신 | 몸을 닦다 |
| 在 | 재 | ~에 있다 |
| 正 | 정 | 바로잡다 |
| 其 | 기 | 그 |
| 心 | 심 | 마음 |
| 者 | 자 | (명사화/주제) |

번역: "'몸을 닦는 것은 그 마음을 바로잡는 데 있다'라 하는 것은..."

**예 2**:
```
天下之所以治者 以有法也
천하지소이치자 이유법야
```

분석:
| 부분 | 기능 |
|------|------|
| 天下 | 주어: 천하 |
| 之 | 관형격 |
| 所以 | ~하는 까닭 |
| 治 | 다스려지다 |
| 者 | (명사화) |
| 以 | ~때문에 |
| 有 | 있다 |
| 法 | 법 |
| 也 | (단정) |

번역: "천하가 다스려지는 까닭은 법이 있기 때문이다."

## 9. 복잡한 문장 분석

### 분석 전략

복잡한 문장을 만났을 때:

1. **주동사 찾기** — 구조의 중심
2. **허사 찾기** — 구 경계를 표시
3. **절로 나누기** — 而, 則, 以가 흔히 절을 구분
4. **관계 파악** — 원인, 조건, 순차, 대비
5. **바깥쪽으로 번역** — 주동사에서 시작하여 수식어를 추가

### 안내식 분석 연습

**문장**: 不以物喜 不以己悲 (범중엄/范仲淹: 岳陽樓記에서)

| 단계 | 작업 | 결과 |
|------|------|------|
| 1 | 주동사 찾기 | 喜(희, 기뻐하다), 悲(비, 슬퍼하다) |
| 2 | 허사 찾기 | 不(불, ~않다), 以(이, ~때문에) |
| 3 | 제1절 분석 | 不 + 以 + 物 + 喜 = 외물 때문에 기뻐하지 않다 |
| 4 | 제2절 분석 | 不 + 以 + 己 + 悲 = 자신 때문에 슬퍼하지 않다 |
| 5 | 번역 | "외물로 인해 기뻐하지 않고, 자신으로 인해 슬퍼하지 않는다" |

**문장**: 學而不思則罔 思而不學則殆

| 단계 | 작업 | 결과 |
|------|------|------|
| 1 | 주동사 | 罔(망, 어리석음), 殆(태, 위태로움) |
| 2 | 허사 | 而(이, 그러나), 不(불, ~않다), 則(즉, 그러면) |
| 3 | 제1절 | 學 而 不思 則 罔 |
| 4 | 구조 | [배우다] 그러나 [생각하지 않다] → 그러면 → [어리석다] |
| 5 | 제2절 | 思 而 不學 則 殆 |
| 6 | 구조 | [생각하다] 그러나 [배우지 않다] → 그러면 → [위태롭다] |

## 10. 연습 문제

### 연습 1: 所 구문

다음을 번역하시오:

1. 吾之所見 (오지소견)
2. 人之所欲 (인지소욕)
3. 所以然者何 (소이연자하)
4. 學者所以明道 (학자소이명도)
5. 人之所以異於禽獸 (인지소이이어금수)

### 연습 2: 之의 기능 식별

之가 관형격, 대명사, 구조적, 리듬적 중 어느 용법인지 쓰시오:

1. 天之道 (천지도)
2. 取之 (취지)
3. 不可不知之理 (불가부지지리)
4. 久之 (구지)
5. 學而時習之 (학이시습지)

### 연습 3: 복잡한 문장 분석

각 문장을 구성 요소로 나누고 번역하시오:

1. 以德報怨 (이덕보원)
2. 生於憂患 死於安樂 (생어우환 사어안락)
3. 學而不厭 誨人不倦 (학이불염 회인불권)

### 연습 4: 而의 기능 식별

而가 "그리고", "그러나", "비록~이지만", "만일~이면" 중 어느 의미인지 판별하시오:

1. 學而知之 (학이지지)
2. 學而不思 (학이불사)
3. 貧而無諂 (빈이무첨)
4. 人而無信 不知其可 (인이무신 부지기가)

### 연습 5: 전체 문장 번역

다음 문장을 번역하고 모든 허사를 식별하시오:

```
天時不如地利
천시불여지리
地利不如人和
지리불여인화
```

**연습 2**: 1) 관형격 2) 대명사 (그것) 3) 구조적 (절과 명사를 연결) 4) 리듬 충전어 5) 대명사 (그것)
