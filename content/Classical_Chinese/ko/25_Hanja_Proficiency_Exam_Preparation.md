## 1. 시험 개요

### 한자능력검정시험 (漢字能力檢定試驗)

한자능력검정시험은 한국에서 한문 한자 지식을 검증하는 표준 자격 시험입니다. 한국어문회(韓國語文會) 및 기타 기관에서 시행합니다.

### 주요 사항

| 항목 | 내용 |
|------|------|
| 시행 기관 | 한국어문회 (韓國語文會) |
| 시행 횟수 | 연간 수회 (약 4-6회) |
| 시험 형식 | 필기시험 (지필) |
| 시험 시간 | 급수에 따라 50-90분 |
| 합격 기준 | 70% (대부분 급수) |
| 유효 기간 | 영구 (만료 없음) |
| 급수 | 8급 (가장 쉬움) ~ 특급 (가장 어려움) |
| 인정 범위 | 학점, 취업, 대학 입시 |

### 시험 구성 개요

| 영역 | 내용 |
|------|------|
| 제1영역 | 읽기(讀) — 한자 읽기 |
| 제2영역 | 쓰기(書) — 한글에서 한자 쓰기 |
| 제3영역 | 뜻(訓) — 한자의 의미 |
| 제4영역 | 음(音) — 한자의 발음/독음 |
| 제5영역 | 어휘(語彙) — 한자어와 활용 |
| 제6영역 | 한문(漢文) — 한문 문장 독해 (상위 급수) |

## 3. 문제 유형: 읽기

### 유형 1: 음독(音讀) — 한자 읽기

한자를 보고 한국어 독음(발음)을 쓰는 문제입니다.

**형식**: 다음 한자의 음(音)을 쓰시오.

| 문제 | 정답 |
|------|------|
| 山 | 산 (san) |
| 水 | 수 (su) |
| 學 | 학 (hak) |
| 德 | 덕 (deok) |
| 義 | 의 (ui) |

### 유형 2: 훈독(訓讀) — 한자의 뜻

한자를 보고 한국어 뜻(훈)을 쓰는 문제입니다.

**형식**: 다음 한자의 훈(訓)을 쓰시오.

| 문제 | 정답 |
|------|------|
| 山 | 메/뫼 산 (산) |
| 水 | 물 수 (물) |
| 大 | 큰 대 (크다) |
| 小 | 작을 소 (작다) |
| 人 | 사람 인 (사람) |

### 유형 3: 음훈(音訓) — 완전한 읽기

뜻(훈)과 발음(음)을 함께 쓰는 문제입니다.

**형식**: 다음 한자의 음과 훈을 쓰시오.

| 글자 | 훈 (뜻) | 음 (읽기) |
|------|---------|----------|
| 天 | 하늘 | 천 |
| 地 | 땅 | 지 |
| 日 | 날 | 일 |
| 月 | 달 | 월 |
| 花 | 꽃 | 화 |

### 읽기 문제 공략 요령

| 요령 | 설명 |
|------|------|
| 부수 학습 | 부수가 의미 범주를 암시하는 경우가 많음 |
| 음성 그룹 | 같은 음부(音符)를 가진 한자는 독음이 비슷한 경우가 많음 |
| 플래시카드 | 매일 한자 카드 복습 |
| 손글씨 | 직접 쓰기가 기억을 강화 |

## 4. 문제 유형: 쓰기

### 유형 4: 한자쓰기 — 한자 쓰기

한국어 독음이나 뜻을 보고 올바른 한자를 쓰는 문제입니다.

**형식**: 다음의 음(또는 훈)에 해당하는 한자를 쓰시오.

| 독음/뜻 | 정답 |
|---------|------|
| 산 (산) | 山 |
| 물 수 (물) | 水 |
| 배울 학 (배우다) | 學 |
| 글 문 (글) | 文 |
| 나라 국 (나라) | 國 |

### 유형 5: 한자어 쓰기 — 합성어 쓰기

한국어 단어를 보고 한자로 쓰는 문제입니다.

**형식**: 다음 한자어를 한자로 쓰시오.

| 한국어 | 한자 | 의미 |
|--------|------|------|
| 학교 | 學校 | 학교 |
| 국가 | 國家 | 나라 |
| 대학 | 大學 | 대학교 |
| 자연 | 自然 | 자연 |
| 문화 | 文化 | 문화 |

### 쓰기의 함정

| 함정 | 예시 | 올바른 형태 |
|------|------|-----------|
| 획 누락 | 天을 마지막 획 없이 씀 | 天 (4획) |
| 잘못된 필순 | 必를 좌에서 우로 씀 | 必 (심(心)의 점 먼저) |
| 유사 한자 혼동 | 土/士, 未/末 | 획 길이 확인 |
| 간체자 vs 정체자 | 学 (간체) | 學 (정체자 — 필수) |

## 5. 문제 유형: 뜻과 활용

### 유형 6: 문맥 속 뜻 — 문맥에서의 의미

문장 속 밑줄 친 한자의 올바른 뜻을 고르거나 쓰는 문제입니다.

**형식**: 밑줄 친 한자의 뜻으로 알맞은 것을 고르시오.

**예시**:
문: 讀**書**를 좋아합니다.
- ① 글 ② 편지 ③ 쓰다 ④ 사무실

**정답**: ① 글 (글, 책)

### 유형 7: 반의어/유의어 — 반의어와 유의어

**반의어(反意語) 예시**:

| 글자 | 독음 | 반의어 | 독음 |
|------|------|--------|------|
| 大 | 대 (크다) | 小 | 소 (작다) |
| 上 | 상 (위) | 下 | 하 (아래) |
| 善 | 선 (착하다) | 惡 | 악 (악하다) |
| 生 | 생 (삶) | 死 | 사 (죽음) |
| 古 | 고 (옛) | 今 | 금 (지금) |
| 明 | 명 (밝다) | 暗 | 암 (어둡다) |
| 強 | 강 (강하다) | 弱 | 약 (약하다) |
| 長 | 장 (길다) | 短 | 단 (짧다) |

**유의어(類義語) 예시**:

| 글자 | 독음 | 유의어 | 독음 |
|------|------|--------|------|
| 家 | 가 (집) | 宅 | 택 (집) |
| 道 | 도 (길) | 路 | 로 (길) |
| 始 | 시 (시작) | 初 | 초 (처음) |
| 言 | 언 (말) | 語 | 어 (말) |
| 美 | 미 (아름답다) | 麗 | 려 (곱다) |

### 유형 8: 부수(部首) — 부수 식별

한자의 부수를 찾는 문제입니다.

| 글자 | 부수 | 부수 이름 | 부수 뜻 |
|------|------|----------|---------|
| 河 | 氵(水) | 삼수변 | 물 |
| 林 | 木 | 나무목 | 나무 |
| 語 | 言 | 말씀언 | 말 |
| 銅 | 金 | 쇠금 | 쇠 |
| 花 | 艹(草) | 초두머리 | 풀 |

## 6. 문제 유형: 한자어와 성어

### 유형 9: 한자어 구성 — 합성어 형성

합성어의 구조를 파악하는 문제입니다.

| 구조 | 유형 | 예시 | 독음 |
|------|------|------|------|
| 유의 합성 | A+A | 道路 | 도로 (길) |
| 반의 합성 | A+B | 大小 | 대소 (크기) |
| 수식+핵 | 형+명 | 高山 | 고산 (높은 산) |
| 동사+목적어 | 동+목 | 登山 | 등산 (산에 오름) |
| 주어+동사 | 주+동 | 地震 | 지진 (땅의 흔들림) |

### 유형 10: 사자성어(四字成語)

상위 급수 시험의 주요 영역입니다.

**형식 A**: 성어 완성

| 제시 | 빠진 글자 | 완성 | 의미 |
|------|----------|------|------|
| 溫故_新 | 知 | 溫故知新 | 옛것을 익혀 새것을 앎 |
| 臥薪_膽 | 嘗 | 臥薪嘗膽 | 복수를 위해 고난을 참음 |
| _苦同樂 | 同 | 同苦同樂 | 고락을 함께 나눔 |
| 以心_心 | 傳 | 以心傳心 | 마음에서 마음으로 전함 |

**형식 B**: 성어와 의미 연결

| 성어 | 독음 | 의미 |
|------|------|------|
| 切磋琢磨 | 절차탁마 | 서로 갈고 닦아 연마함 |
| 塞翁之馬 | 새옹지마 | 불행이 행운이 될 수 있음 |
| 百折不屈 | 백절불굴 | 불굴의 정신 |
| 螢雪之功 | 형설지공 | 어려운 환경에서의 학업 |

### 유형 11: 한문 독해

(상위 급수: 3급 이상)

**형식**: 다음 한문을 읽고 물음에 답하시오.

```
子曰 學而時習之 不亦說乎
자왈 학이시습지 불역열호
```

문제:
1. 누가 말했는가? (공자/孔子)
2. 說은 여기서 무슨 뜻인가? (기쁘다, 즐겁다 — 기쁨)
3. 핵심 메시지는 무엇인가? (배우고 익히는 것이 기쁨을 가져다줌)

### 전략 5: 한자 구성 분석

| 글자 | 구성 요소 | 기억 도움 |

| 林 (수풀 림) | 木 + 木 | 나무(木) 두 개 = 수풀 |
| 好 (좋을 호) | 女 + 子 | 여자(女)와 아이(子) = 좋다 |

- 一 → ( )  二 → ( )  三 → ( )

**문3**: 다음 중 뜻이 바른 것을 고르시오.

**문1**: 다음 한자어의 음을 쓰시오.

- 善 ↔ ( )  生 ↔ ( )  古 ↔ ( )

**문1**: 다음 사자성어의 빈칸을 채우시오.
- 溫( )知( )  切磋( )磨

**문2**: 다음 한문을 해석하시오.

```
學而不思則罔 思而不學則殆
```

### 1급 예시 (5,978자)

**문1**: 다음 한문을 읽고 물음에 답하시오.
```
天下莫柔弱於水 而攻堅強者莫之能勝
```
- 이 글의 출전은? (道德經)
- 이 글의 주제는? (柔弱이 剛強을 이김)

**문2**: 다음 한자의 음과 뜻이 다른 것을 고르시오.
- 樂: ① 풍류 악 ② 즐길 락 ③ 좋아할 요 ④ 밝을 명

## 9. 시간 관리와 시험 당일

### 급수별 시간 배분

**50문항 시험 (50분)**:

| 단계 | 시간 | 활동 |
|------|------|------|
| 제1단계 | 20분 | 쉬운 문제 — 아는 것 모두 즉시 풀기 |
| 제2단계 | 20분 | 중간 난이도 문제 — 신중하게 풀기 |
| 제3단계 | 10분 | 어려운 문제 + 전체 답안 검토 |

**100문항 이상 시험 (90분)**:

| 단계 | 시간 | 활동 |
|------|------|------|
| 제1단계 | 30분 | 전체 문제 훑고, 쉬운 것 먼저 (40-50%) |
| 제2단계 | 35분 | 중간 난이도 문제 풀기 |
| 제3단계 | 15분 | 남은 어려운 문제 도전 |
| 제4단계 | 10분 | 전체 답안 검토 및 확인 |

### 시험 당일 점검 목록

| 준비물 | 참고 |
|--------|------|
| 수험표 | 인쇄 또는 디지털 저장 |
| 신분증 | 사진이 있는 공적 신분증 |
| 필기도구 | 검정 펜 (서술형), 연필 (객관식) |
| 시계 | 시간 관리용 (스마트워치 불가) |
| 지우개 | 수정용 |
| 물 | 수분 보충 |
| 일찍 도착 | 시작 30분 전 |

### 시험 당일 흔한 실수

| 실수 | 예방법 |
|------|--------|
| 문제 오독 | 각 문제를 두 번 읽기 |
| 답안 위치 오류 | 문항 번호와 답안지 번호 일치 확인 |
| 불완전한 한자 | 모든 획을 정확히 써서 또렷하게 |
| 시간 관리 실패 | 시간의 25%, 50%, 75% 지점에 점검 |
| 공란 남기기 | 항상 추측이라도 답을 쓰기 — 오답 감점 없음 |
| 음과 훈 혼동 | 음을 묻는지 훈을 묻는지 재확인 |

## 맺음말

한자능력검정시험 준비는 단순히 시험에 합격하기 위한 것이 아닙니다 — 동아시아의 방대한 문학적·철학적 유산과 연결되는 일입니다. 배우는 한자 하나하나가 수백 년간 축적된 지혜의 문을 열어줍니다. 공자의 도덕적 가르침에서 장자의 역설까지, 당시의 아름다움에서 조선의 통치 원리까지.

성공의 열쇠는 매일의 꾸준한 연습입니다. 옛사람이 이르기를:

```
不積跬步 無以至千里
불적규보 무이지천리

不積小流 無以成江海
불적소류 무이성강해
```

"반 걸음을 쌓지 않으면 천 리에 이를 수 없고, 작은 물줄기를 모으지 않으면 강과 바다를 이룰 수 없다." — 순자/荀子
