## 1. 실사와 허사

### 1.1 두 가지 범주

한문의 모든 단어는 두 가지 대범주 중 하나에 속합니다:

> **실사 (實詞/실사) vs 허사 (虛詞/허사)**
>
> 實詞 (실사) 실사:
>   - 독립적 의미를 가짐
>   - 명사, 동사, 형용사
>   - 예: 人(사람), 學(배우다), 大(크다), 山(산)
>
> 虛詞 (허사) 허사:
>   - 독립적 어휘 의미 없음
>   - 문법적 관계를 표시
>   - 한국어의 은/는, 이/가, 을/를, -다와 유사
>   - 예: 之, 乎, 者, 也, 矣, 焉, 哉, 夫
>
> 한문의 상용 허사는 약 20-30개에 불과하지만,
> 거의 모든 문장에 등장합니다. 이것을 숙달하는 것이
> 독해 유창성의 핵심입니다.

### 1.2 8가지 핵심 허사 개요

| 한자 | 음독 | 훈독 | 주요 기능 | 한국어 대응 |
|------|------|------|----------|-----------|
| 之 | 지 (ji) | 갈 지 | 소유격; 대명사; 구조 조사 | 의; 그것을; — |
| 乎 | 호 (ho) | 어조사 호 | 의문 표지; 감탄 | -ㄴ가?; -도다 |
| 者 | 자 (ja) | 놈 자 | 명사화; 주제 표지 | -는 것; -는 자 |
| 也 | 야 (ya) | 어조사 야 | 계사; 단정 | -이다; -이니라 |
| 矣 | 의 (ui) | 어조사 의 | 완료; 상태 변화 | -했다; -게 되었다 |
| 焉 | 언 (eon) | 어조사 언 | 장소; 수사 표지 | 거기에; -는가 |
| 哉 | 재 (jae) | 어조사 재 | 감탄 (강함) | -도다!; -로다! |
| 夫 | 부 (bu) | 지아비 부 / 어조사 부 | 담화 개시; 주제 | 무릇; 대저 |

## 2. 之 — 가장 다기능적인 조사

之는 한문 텍스트에서 가장 빈번하게 출현하는 글자입니다. 세 가지 주요 기능이 있습니다:

### 2.1 기능 1: 소유격 조사 (소유격 조사, "의")

두 명사 사이에 之가 오면 소유 관계를 만듭니다. 한국어 의와 같습니다:

| 예시 | 독음 | 번역 | 구조 |
|------|------|------|------|
| 人之心 | 인지심 | 사람의 마음 | A 之 B = A의 B |
| 天之道 | 천지도 | 하늘의 도 | 天의 道 |
| 國之本 | 국지본 | 나라의 근본 | 國의 本 |
| 仁之道 | 인지도 | 인의 도 | 仁의 道 |
| 父之敎 | 부지교 | 아버지의 가르침 | 父의 敎 |
| 民之心 | 민지심 | 백성의 마음 | 民의 心 |
| 學之始 | 학지시 | 배움의 시작 | 學의 始 |
| 天地之間 | 천지지간 | 천지 사이 | 天地의 間 |

### 2.2 기능 2: 대명사 (대명사, "그것을")

之는 앞서 언급된 명사를 대신하는 대명사로 쓰입니다. "그것" 또는 "그들"에 해당합니다:

| 예시 | 독음 | 번역 | 之가 가리키는 것 |
|------|------|------|----------------|
| 學而時習之 | 학이시습지 | 배우고 그것을 익히다 | 배운 것 |
| 知之 | 지지 | 그것을 알다 | 해당 주제 |
| 愛之 | 애지 | 그것을/그들을 사랑하다 | 사랑의 대상 |
| 取之 | 취지 | 그것을 취하다 | 언급된 것 |
| 觀之 | 관지 | 그것을 관찰하다 | 논의 중인 것 |

### 2.3 기능 3: 구조 조사 (구조조사)

之는 수식절과 명사 사이에 나타나기도 합니다. 직접적 대응어가 없습니다:

| 예시 | 독음 | 번역 | 기능 |
|------|------|------|------|
| 不義之財 | 불의지재 | 불의한 재물 | 수식어와 명사 연결 |
| 可愛之人 | 가애지인 | 사랑스러운 사람 | 수식어와 명사 연결 |
| 千里之行 | 천리지행 | 천리의 여행 | 수식어와 명사 연결 |

### 2.4 之 기능 요약

> **之 한눈에 보기**
>
> - ① 소유격 (의): 人之心 = 사람의 마음
> - ② 대명사 (그것을): 學而習之 = 배워서 그것을 익힌다
> - ③ 구조 조사: 千里之行 = 천리의 행 (천 리의 여행)
>
> 구별 방법:
>   - 명사 之 명사 → 소유격
>   - 동사 之 → 대명사 (동사의 목적어)
>   - 수식어 之 명사 → 구조 조사

## 3. 乎 — 의문과 감탄

### 3.1 기능 1: 의문 표지 (의문 조사, "-ㄴ가?")

문장 끝의 乎는 의문문을 만듭니다. 한국어 -ㄴ가? 또는 -습니까?와 같습니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 可乎 | 가호 | 괜찮은가? |
| 有乎 | 유호 | 있는가? |
| 知之乎 | 지지호 | 그것을 아는가? |
| 學而時習之 不亦說乎 | 학이시습지 불역열호 | 또한 기쁘지 않은가? |
| 有朋自遠方來 不亦樂乎 | 유붕자원방래 불역락호 | 또한 즐겁지 않은가? |

### 3.2 기능 2: 감탄 (감탄, "-도다!")

乎는 감탄이나 경탄을 표현하기도 합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 大哉乎 | 대재호 | 크도다! |
| 善乎 | 선호 | 좋도다! |
| 美乎 | 미호 | 아름답도다! |

### 3.3 기능 3: 전치사 ("~에서, ~에" = 於의 대용)

일부 텍스트에서 乎는 전치사 於 (어, "~에서/~에")처럼 기능합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 生乎吾前 | 생호오전 | 나보다 먼저 태어나다 |
| 出乎其外 | 출호기외 | 그것을 넘어서다 |

## 4. 者 — 명사화 조사

### 4.1 기능 1: 명사화 ("-는 것/사람")

者는 앞의 동사나 형용사를 명사구로 만듭니다:

| 예시 | 독음 | 번역 | 구조 |
|------|------|------|------|
| 學者 | 학자 | 배우는 사람; 학자 | 동사 + 者 = ~하는 자 |
| 仁者 | 인자 | 어진 사람 | 형용사 + 者 = ~한 자 |
| 知者 | 지자 | 지혜로운 사람 | 동사 + 者 = 아는 자 |
| 勇者 | 용자 | 용감한 사람 | 형용사 + 者 = 용감한 자 |
| 言者 | 언자 | 말하는 사람 | 동사 + 者 = 말하는 자 |
| 善者 | 선자 | 착한 사람 | 형용사 + 者 = 착한 자 |

### 4.2 기능 2: 者...也 패턴의 주제 표지

者는 동일 문장의 주제를 표시합니다 (05과 참조):

| 예시 | 독음 | 번역 |
|------|------|------|
| 學者 覺也 | 학자 각야 | 배움이란 깨달음이다 |
| 仁者 愛人也 | 인자 애인야 | 인이란 사람을 사랑하는 것이다 |
| 民者 國之本也 | 민자 국지본야 | 백성은 나라의 근본이다 |

### 4.3 기능 3: 멈춤/열거

者는 복잡한 문장에서 멈춤을 만들어 조건이나 범주를 나열합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 生者必死 | 생자필사 | 살아있는 자는 반드시 죽는다 |
| 知者樂水 仁者樂山 | 지자요수 인자요산 | 지혜로운 자는 물을 좋아하고 어진 자는 산을 좋아한다 |

## 5. 也 — 계사와 단정

### 5.1 기능 1: 계사 ("-이다")

문장 끝의 也는 주어가 서술어와 같음을 단정합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 善也 | 선야 | 선하다 |
| 然也 | 연야 | 그러하다 |
| 是也 | 시야 | 그렇다; 맞다 |
| 非也 | 비야 | 그렇지 않다 |
| 孔子聖人也 | 공자성인야 | 공자는 성인이다 |

### 5.2 기능 2: 강조/설명 ("-이니라")

也는 강조적 단정이나 설명적 뉘앙스를 더합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 天命也 | 천명야 | 하늘의 명이니라 |
| 此仁也 | 차인야 | 이것이 참으로 인이다 |

### 5.3 기능 3: 문중 멈춤

也는 때때로 문장 중간에서 멈춤이나 전환을 표시합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 大學之道也 在明明德 | 대학지도야 재명명덕 | 대학의 도란 — 밝은 덕을 밝히는 데 있다 |

## 6. 矣 — 완료와 변화

### 6.1 기능 1: 완료 ("-했다, -되었다")

矣는 행위나 변화가 완료되었거나 새로운 상황이 발생했음을 나타냅니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 知矣 | 지의 | (이제) 알겠다 |
| 已矣 | 이의 | 끝났다; 그만이다 |
| 足矣 | 족의 | (이제) 충분하다 |
| 可矣 | 가의 | (이제) 괜찮다 |
| 死矣 | 사의 | 죽었다 |

### 6.2 기능 2: 상태 변화 ("-게 되었다")

矣는 이전 상태에서의 변화를 나타냅니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 吾老矣 | 오노의 | 내가 늙었다 |
| 天下平矣 | 천하평의 | 천하가 평화로워졌다 |
| 國亡矣 | 국망의 | 나라가 망했다 |

### 6.3 也 vs 矣

| 조사 | 기능 | 뉘앙스 | 예시 |
|------|------|--------|------|
| 也 | 계사/단정 | 불변의 진리, 정의 | 仁者愛人也 (인이란 사람을 사랑하는 것이다 — 항상 참) |
| 矣 | 완료/변화 | 새로운 상황, 방금 발생 | 吾知矣 (이제 알겠다 — 방금 깨달음) |

## 7. 焉 — 장소와 수사 표지

### 7.1 기능 1: 장소 ("거기에, 그 안에" = 於此/어차)

焉은 "거기에" 또는 "이것 안에"를 의미합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 心不在焉 | 심부재언 | 마음이 (거기에) 있지 않다 |
| 有子存焉 | 유자존언 | 아들이 거기에 살고 있다 |

### 7.2 기능 2: 문말 조사

문장 끝의 焉은 완곡한 단정이나 수사적 어조를 더합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 三人行必有我師焉 | 삼인행필유아사언 | 세 사람이 길을 가면 반드시 그 가운데 나의 스승이 있다 |

## 8. 哉 — 감탄

### 8.1 강한 감탄 ("-도다! -구나!")

哉는 강한 감정 — 감탄, 놀라움, 탄식 — 을 표현합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 大哉 | 대재 | 크도다! |
| 善哉 | 선재 | 훌륭하도다! 좋도다! |
| 悲哉 | 비재 | 슬프도다! 아! |
| 壯哉 | 장재 | 장하도다! |
| 危哉 | 위재 | 위태롭도다! |

### 8.2 哉를 사용한 수사 의문

哉가 의문사와 결합하면 강조적 수사 의문을 만듭니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 何哉 | 하재 | 이 무엇인가?! |
| 豈不痛哉 | 기불통재 | 어찌 아프지 않겠는가?! |
| 何足道哉 | 하족도재 | 어찌 말할 것이 있겠는가?! |

## 9. 夫 — 담화 개시사

### 9.1 기능 1: 주제 도입 ("무릇, 대저")

夫가 문장 시작에 나타나면 (명사 부/남편이 아닌 어조사 부로 읽음), 논의할 주제를 도입합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 夫仁者 | 부인자 | 무릇, 어진 자는... |
| 夫天地者 | 부천지자 | 무릇, 천지란... |
| 夫學者 | 부학자 | 무릇, 학자란... |

### 9.2 기능 2: 지시사 ("저, 그")

夫는 지시 대명사로도 기능합니다:

| 예시 | 독음 | 번역 |
|------|------|------|
| 夫人 | 부인 | 저 사람; 부인 |
| 夫子 | 부자 | 저 스승; (공자) |

## 10. 비교 용법표

### 10.1 문말 조사 비교

| 조사 | 어조 | 사용 시점 | 예시 |
|------|------|----------|------|
| 也 | 서술적, 차분 | 정의, 불변의 진리 | 學者覺也 (배움이란 깨달음이다) |
| 矣 | 단정적, 새로운 | 완료된 행위, 변화 | 吾知矣 (이제 알겠다) |
| 乎 | 의문적 | 질문 | 知之乎 (그것을 아는가?) |
| 哉 | 감탄적 | 강한 감정 | 善哉 (좋도다!) |
| 焉 | 완곡한 단정 | 완화된 서술, 장소 | 有師焉 (거기에 스승이 있다) |

### 10.2 다기능 요약

| 한자 | 기능 1 | 기능 2 | 기능 3 |
|------|--------|--------|--------|
| 之 | 소유격 (의) | 대명사 (그것을) | 구조 조사 |
| 乎 | 의문 (-ㄴ가?) | 감탄 (-도다) | 전치사 (에서) |
| 者 | 명사화 (-는 자) | 주제 표지 (者...也) | 멈춤/열거 |
| 也 | 계사 (-이다) | 강조 (-이니라) | 문중 멈춤 |

## 11. 연습 문제

### 연습 1: 허사 식별

각 문장에서 허사를 찾아 이름을 말하시오:

```
1. 人之心善也
2. 知之乎
3. 學者覺也
4. 善哉
5. 吾老矣
```

<details>
<summary>정답 확인</summary>

```
1. 人[之]心善[也]
   之 = 소유격 (사람의 마음), 也 = 계사 (선하다)
   번역: 사람의 마음은 착하다.

2. 知[之][乎]
   之 = 대명사 (그것을), 乎 = 의문 표지
   번역: 그것을 아는가?

3. 學[者]覺[也]
   者 = 주제 표지, 也 = 계사
   번역: 배움이란 깨달음이다.

4. 善[哉]
   哉 = 감탄
   번역: 좋도다!

5. 吾老[矣]
   矣 = 상태 변화
   번역: 내가 늙었다.
```

</details>

### 연습 2: 之 기능 식별

각 구절에서 之의 기능을 식별하시오 (소유격 / 대명사 / 구조 조사):

```
1. 天之道
2. 學而習之
3. 千里之行
4. 國之本
5. 知之者
6. 民之心
7. 可愛之人
8. 取之
```

<details>
<summary>정답 확인</summary>

```
1. 天之道 → 소유격: 하늘의 도 (천의 도)
2. 學而習之 → 대명사: 배우고 그것을 익히다 (그것을)
3. 千里之行 → 구조 조사: 천 리의 여행
4. 國之本 → 소유격: 나라의 근본 (국의 본)
5. 知之者 → 대명사: 그것을 아는 자 (그것을 아는 자)
6. 民之心 → 소유격: 백성의 마음 (민의 심)
7. 可愛之人 → 구조 조사: 사랑스러운 사람
8. 取之 → 대명사: 그것을 취하다 (그것을)
```

</details>

### 연습 3: 문말 조사 선택

올바른 문말 조사를 선택하시오 (也, 矣, 乎, 哉):

```
1. 仁者愛人___  (인의 정의)
2. 吾知___  (방금 깨달음 — 이제 알게 됨)
3. 善___  (좋도다! — 감탄)
4. 子知之___  (그것을 아는가? — 질문)
5. 天下平___  (천하가 평화로워짐 — 변화)
```

<details>
<summary>정답 확인</summary>

```
1. 仁者愛人也 — 정의/불변의 진리 → 也
2. 吾知矣 — 방금 깨달음/상태 변화 → 矣
3. 善哉 — 강한 감탄 → 哉
4. 子知之乎 — 질문 → 乎
5. 天下平矣 — 상태 변화 → 矣
```

</details>

### 연습 4: 허사에 주의한 번역

허사에 주의하여 각 문장을 번역하시오:

```
1. 天之道遠矣
2. 學者不知乎
3. 夫仁者 人之本也
4. 大哉 天地之德
5. 三人行 必有我師焉
```

<details>
<summary>정답 확인</summary>

```
1. 天之道遠矣 (천지도원의)
   하늘의 도가 멀어졌다.
   之 = 소유격, 矣 = 상태 변화

2. 學者不知乎 (학자부지호)
   학자가 알지 못하는가?
   者 = 명사화, 乎 = 의문

3. 夫仁者 人之本也 (부인자 인지본야)
   무릇 인이란 — 사람의 근본이다.
   夫 = 주제 개시, 者 = 주제 표지, 之 = 소유격, 也 = 계사

4. 大哉 天地之德 (대재 천지지덕)
   크도다, 천지의 덕이여!
   哉 = 감탄, 之 = 소유격

5. 三人行 必有我師焉 (삼인행 필유아사언)
   세 사람이 길을 가면 반드시 그 가운데 나의 스승이 있다.
   焉 = "거기에/그 가운데" (장소)
```

</details>

### 연습 5: 문장 짓기

배운 허사를 사용하여 한문 문장을 지으시오:

```
1. "배움의 길은 어렵다." (之와 也 사용)
2. "그것을 아는가?" (之와 乎 사용)
3. "훌륭하도다!" (哉 사용)
4. "이제 알겠다." (矣 사용)
5. "지혜로운 자는 물을 좋아한다." (者 사용)
```

<details>
<summary>정답 확인</summary>

```
1. 學之道難也 (학지도난야) — 배움의 길은 어렵다.
2. 知之乎 (지지호) — 그것을 아는가?
3. 善哉 (선재) — 훌륭하도다!
4. 吾知矣 (오지의) — 이제 알겠다.
5. 知者樂水 (지자요수) — 지혜로운 자는 물을 좋아한다.
```

</details>
