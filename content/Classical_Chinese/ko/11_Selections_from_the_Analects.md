## 1. 논어(論語) 소개

### 1.1 논어란?

> **論語 (논어) — 논어**
>
> - **저자**: 공자(孔子, 공자)의 제자들이 편찬
> - **시기**: 기원전 5~3세기경 (공자 사후 편찬)
> - **구조**: 20편(篇), 512장
> - **내용**: 공자와 그 제자들의 어록과 대화
> - **지위**: 유교 사서(四書) 중 하나
>
> 공자(孔子):
> - **이름**: 孔丘 (공구), 자(字) 仲尼 (중니)
> - **출생**: 기원전 551년, 노(魯)나라
> - **사망**: 기원전 479년
> - **칭호**: 至聖先師 (지성선사, "지극한 성인이자 으뜸 스승")
>
> 논어는 거의 모든 장이 子曰(자왈)로 시작됩니다.
> "선생님께서 말씀하셨다"라는 뜻입니다.

### 1.2 사서(四書/사서)

| 서명 | 한국어 | 저자/편찬자 | 내용 |
|------|--------|-----------|------|
| 大學 | 대학 | 증자(曾子)에 귀속 | 대학의 도 |
| 論語 | 논어 | 공자의 제자들 | 공자의 어록 |
| 孟子 | 맹자 | 맹자 | 맹자의 가르침 |
| 中庸 | 중용 | 자사(子思)에 귀속 | 중용의 도 |

## 2. 學而篇 선독 1: 세 가지 즐거움

### 2.1 원문

```
子曰 學而時習之 不亦說乎
有朋自遠方來 不亦樂乎
人不知而不慍 不亦君子乎
```

### 2.2 글자별 주석

**제1절: 子曰 學而時習之 不亦說乎**

| 한자 | 음독 | 훈독 | 문법적 역할 | 문맥상 의미 |
|------|------|------|-----------|------------|
| 子 | 자 | 아들 자 | 주어 | 선생님 (공자) |
| 曰 | 왈 | 가로되 왈 | 동사 | 말씀하셨다 |
| 學 | 학 | 배울 학 | 동사 | 배우다, 공부하다 |
| 而 | 이 | 말이을 이 | 접속사 | 그리고 나서 |
| 時 | 시 | 때 시 | 부사 | 때때로 |
| 習 | 습 | 익힐 습 | 동사 | 익히다, 복습하다 |
| 之 | 지 | 갈 지 | 대명사 | 그것 (배운 것) |
| 不 | 불 | 아닐 불 | 부정 | ~아니다 |
| 亦 | 역 | 또 역 | 부사 | 또한 |
| 說 | 열 | 기쁠 열 | 형용사 | 기쁘다 (=悅) |
| 乎 | 호 | 어조사 호 | 어기사 | 의문 표지 |

**제2절: 有朋自遠方來 不亦樂乎**

| 한자 | 음독 | 훈독 | 문법적 역할 | 문맥상 의미 |
|------|------|------|-----------|------------|
| 有 | 유 | 있을 유 | 동사 | 있다; 가지다 |
| 朋 | 붕 | 벗 붕 | 명사 | 벗 |
| 自 | 자 | 스스로 자 | 전치사 | ~로부터 |
| 遠 | 원 | 멀 원 | 형용사 | 먼, 먼 곳의 |
| 方 | 방 | 모 방 | 명사 | 곳, 지방 |
| 來 | 래 | 올 래 | 동사 | 오다 |
| 樂 | 락 | 즐길 락 | 형용사 | 즐겁다 |

**제3절: 人不知而不慍 不亦君子乎**

| 한자 | 음독 | 훈독 | 문법적 역할 | 문맥상 의미 |
|------|------|------|-----------|------------|
| 人 | 인 | 사람 인 | 주어 | 남, 사람들 |
| 知 | 지 | 알 지 | 동사 | 알아주다, 인정하다 |
| 慍 | 온 | 성낼 온 | 형용사 | 원망하다, 노여워하다 |
| 君 | 군 | 임금 군 | 명사 | 임금 (君子에서: 군자) |
| 子 | 자 | 아들 자 | 명사 | 사람 (君子에서: 군자) |

### 2.3 현토 읽기

```
子ㅣ 曰하시되
學하고 時로 之를 習하면 不亦 說하지 아니한가
朋이 有하여 遠方으로부터 來하면 不亦 樂하지 아니한가
人이 知하지 아니하여도 慍하지 아니하면 不亦 君子가 아닌가
```

### 2.4 석독 (완전한 한국어 번역)

```
공자께서 말씀하셨다.
"배우고 때때로 그것을 익히면, 또한 기쁘지 아니한가.
벗이 먼 곳에서 찾아오면, 또한 즐겁지 아니한가.
남이 알아주지 않아도 원망하지 않으면, 또한 군자가 아닌가."
```

### 2.5 문법 분석

| 패턴 | 예시 | 설명 |
|------|------|------|
| 而 (순접) | 學而時習之 | "배우고 나서 익히다" |
| 不亦...乎 (수사적) | 不亦說乎 | "또한 기쁘지 아니한가?" → 그렇다! |
| 自 (~로부터) | 自遠方 | "먼 곳으로부터" |
| 而 (양보적) | 不知而不慍 | "알아주지 않아도 원망하지 않다" |

## 3. 學而篇 선독 2: 날마다의 자기 성찰

### 3.1 원문

```
曾子曰 吾日三省吾身
為人謀而不忠乎
與朋友交而不信乎
傳不習乎
```

### 3.2 글자별 주석

| 한자 | 음독 | 훈독 | 문맥상 의미 |
|------|------|------|------------|
| 曾 | 증 | 일찍 증 | 성(曾子 = 증자) |
| 吾 | 오 | 나 오 | 나, 나의 |
| 日 | 일 | 날 일 | 매일 |
| 三 | 삼 | 석 삼 | 세 (번) |
| 省 | 성 | 살필 성 | 돌아보다, 성찰하다 |
| 身 | 신 | 몸 신 | 자기 자신, 행실 |
| 為 | 위 | 할 위 | ~을 위해, ~를 대신하여 |
| 謀 | 모 | 꾀 모 | 도모하다, 계책을 세우다 |
| 忠 | 충 | 충성 충 | 충성스러운, 성실한 |
| 與 | 여 | 더불 여 | ~과 함께 |
| 交 | 교 | 사귈 교 | 사귀다, 교류하다 |
| 信 | 신 | 믿을 신 | 믿음직한, 성실한 |
| 傳 | 전 | 전할 전 | 전수하다; 전수받은 것(가르침) |

### 3.3 석독

```
증자가 말하였다.
"나는 날마다 세 가지로 내 몸을 돌아본다.
남을 위하여 일을 도모하면서 충성스럽지 못하지 않았는가?
벗과 사귀면서 믿음직하지 못하지 않았는가?
(스승에게서) 전수받은 것을 익히지 않았는가?"
```

## 4. 學而篇 선독 3: 덕으로 다스리기

### 4.1 원문

```
子曰 道千乘之國 敬事而信 節用而愛人 使民以時
```

### 4.2 주석

| 구절 | 음독 | 번역 |
|------|------|------|
| 道千乘之國 | 도천승지국 | 천 대의 전차를 가진 나라를 다스리되 |
| 敬事而信 | 경사이신 | 일을 공경히 하고 믿음이 있으며 |
| 節用而愛人 | 절용이애인 | 쓰임을 절약하고 사람을 사랑하며 |
| 使民以時 | 사민이시 | 백성을 부리되 때에 맞게 하라 |

### 4.3 석독

```
공자께서 말씀하셨다.
"천 대의 전차를 가진 나라를 다스리되,
일을 공경히 하고 믿음이 있으며,
쓰임을 절약하고 사람을 사랑하며,
백성을 부리되 때에 맞게 하라."
```

## 5. 為政篇 선독 1: 인생의 단계

### 5.1 원문

```
子曰 吾十有五而志于學 三十而立 四十而不惑
五十而知天命 六十而耳順 七十而從心所欲不踰矩
```

### 5.2 글자별 주석

| 구절 | 음독 | 핵심 한자 | 번역 |
|------|------|----------|------|
| 吾十有五 | 오십유오 | 吾(나), 十有五(열다섯) | 나는 열다섯에 |
| 而志于學 | 이지우학 | 志(뜻을 두다), 于(~에), 學(배움) | 학문에 뜻을 두었다 |
| 三十而立 | 삼십이립 | 三十(서른), 立(서다) | 서른에 확립되었다 |
| 四十而不惑 | 사십이불혹 | 不惑(미혹되지 않다) | 마흔에 미혹되지 않았다 |
| 五十而知天命 | 오십이지천명 | 天命(하늘의 명) | 쉰에 하늘의 명을 알았다 |
| 六十而耳順 | 육십이이순 | 耳(귀), 順(순하다) | 예순에 귀가 순해졌다 |
| 七十而從心所欲 | 칠십이종심소욕 | 從(따르다), 心(마음), 所欲(원하는 바) | 일흔에 마음이 원하는 바를 따라도 |
| 不踰矩 | 불유구 | 踰(넘다), 矩(법도) | 법도를 넘지 않았다 |

### 5.3 석독

```
공자께서 말씀하셨다.
"나는 열다섯에 학문에 뜻을 두었고,
서른에 (뜻이) 확립되었으며,
마흔에 미혹되지 않았고,
쉰에 하늘의 명을 알았으며,
예순에 귀가 순해졌고,
일흔에 마음이 원하는 바를 따라도 법도를 넘지 않았다."
```

### 5.4 공자의 나이 (한국 문화적 맥락)

이 구절은 한국 문화에 여러 표준적 표현을 남겼습니다:

| 나이 | 한문 | 한국어 표현 | 의미 |
|------|------|-----------|------|
| 15 | 志于學 | 지학 (志學) | 학문에 뜻을 세우는 나이 |
| 30 | 而立 | 이립 (而立) | 확립/독립의 나이 |
| 40 | 不惑 | 불혹 (不惑) | 미혹되지 않는 나이 |
| 50 | 知天命 | 지천명 (知天命) | 하늘의 뜻을 아는 나이 |
| 60 | 耳順 | 이순 (耳順) | 어떤 말도 동요 없이 듣는 나이 |
| 70 | 從心 | 종심 (從心) | 마음을 따르는 나이 |

## 6. 為政篇 선독 2: 온고지신

### 6.1 원문

```
子曰 溫故而知新 可以為師矣
```

### 6.2 주석

| 한자 | 음독 | 훈독 | 의미 |
|------|------|------|------|
| 溫 | 온 | 따뜻할 온 | 따뜻하게 하다; 복습하다 |
| 故 | 고 | 옛 고 | 옛것, 지난 것 |
| 知 | 지 | 알 지 | 알다; 이해하다 |
| 新 | 신 | 새 신 | 새것 |
| 可 | 가 | 옳을 가 | ~할 수 있다 |
| 以 | 이 | 써 이 | ~로써, 그리하여 |
| 為 | 위 | 할 위 | 되다, ~역할을 하다 |
| 師 | 사 | 스승 사 | 스승 |
| 矣 | 의 | 어조사 의 | 완료 표지 |

### 6.3 현토와 석독

**현토:**
```
子ㅣ 曰하시되 故를 溫하여 新을 知하면 可히 以하여 師가 될 수 있다
```

**석독:**
```
공자께서 말씀하셨다.
"옛것을 익히고 새것을 알면, 가히 스승이 될 수 있다."
```

### 6.4 문화적 영향

溫故知新(온고지신)은 한국에서 가장 유명한 사자성어(四字成語) 중 하나입니다:
- 교육에서 기본 복습의 중요성을 강조할 때 사용
- 한국 일상생활에서 흔히 사용되는 표현
- 한국 시험에 자주 출제

## 7. 為政篇 선독 3: 배움과 생각

### 7.1 원문

```
子曰 學而不思則罔 思而不學則殆
```

### 7.2 주석

| 한자 | 음독 | 훈독 | 의미 |
|------|------|------|------|
| 學 | 학 | 배울 학 | 배우다, 공부하다 |
| 思 | 사 | 생각할 사 | 생각하다, 사색하다 |
| 則 | 즉 | 곧 즉 | ~하면 곧 (결과) |
| 罔 | 망 | 없을 망 | 어둡다, 어리석다, 방향을 잃다 |
| 殆 | 태 | 위태할 태 | 위태롭다, 위험하다 |

### 7.3 현토와 석독

**현토:**
```
子ㅣ 曰하시되 學하고 思하지 아니하면 則 罔하고
思하고 學하지 아니하면 則 殆하다
```

**석독:**
```
공자께서 말씀하셨다.
"배우기만 하고 생각하지 않으면 어둡고,
생각하기만 하고 배우지 않으면 위태롭다."
```

### 7.4 문법 분석

| 패턴 | 용례 | 기능 |
|------|------|------|
| A而不B則C | 學而不思則罔 | A하되 B하지 않으면 C하다 |
| 而 | 접속사 | "~하고" / "~하나" |
| 不 | 부정 | "~하지 않다" |
| 則 | 결과 | "~하면 곧" |

## 8. 핵심 유교 어휘

### 8.1 핵심 덕목

| 한자 | 음독 | 훈독 | 의미 | 중요성 |

| 義 | 의 (ui) | 옳을 의 | 의로움, 정의 | 올바른 행동 |
| 禮 | 례 (rye) | 예도 례 | 예절, 의례 | 사회적 조화 |

| 忠 | 충 (chung) | 충성 충 | 충성, 성실 | 직분에 대한 헌신 |
| 孝 | 효 (hyo) | 효도 효 | 효도 | 부모에 대한 공경 |

## 9. 논어의 문법 패턴

### 9.1 주요 패턴

| 패턴 | 구조 | 예시 | 번역 |
|------|------|------|------|
| 子曰... | 도입 | 子曰 學而時習之 | 선생님께서 말씀하셨다: "배우고..." |
| 不亦...乎 | 수사적 의문 | 不亦說乎 | 기쁘지 아니한가? |
| A而B | 순접/대조 | 學而不思 | 배우되 생각하지 않다 |
| A則B | 조건 결과 | 不學則不知 | 배우지 않으면 알지 못하다 |
| A者B也 | 정의 | 學者覺也 | 배움이란 깨달음이다 |

### 9.2 논어의 문말 어기사

| 어기사 | 빈도 | 기능 | 예시 |
|--------|------|------|------|
| 乎 | 매우 빈번 | 의문 | 不亦說乎 |
| 也 | 매우 빈번 | 단정 | 君子也 |
| 矣 | 빈번 | 완료 | 可以為師矣 |
| 哉 | 비교적 적음 | 감탄 | 善哉 |

## 10. 연습 문제

### 연습 1: 글자 식별

이 구절에서 각 글자의 의미와 문법적 역할을 파악하세요:

```
子曰 溫故而知新 可以為師矣
```

<details>
<summary>정답 확인</summary>

```
子 (자) = 선생님 — 주어
曰 (왈) = 말씀하셨다 — 동사
溫 (온) = 복습하다/따뜻하게 하다 — 동사
故 (고) = 옛것 — 목적어
而 (이) = 그리고 (나서) — 접속사
知 (지) = 알다 — 동사
新 (신) = 새것 — 목적어
可 (가) = ~할 수 있다 — 보조
以 (이) = 그리하여 — 전치사
為 (위) = 되다 — 동사
師 (사) = 스승 — 보어
矣 (의) = (완료) — 어기사
```

</details>

### 연습 2: 현토 달기

다음 구절에 현토를 달아 보세요:

```
1. 學而不思則罔
2. 人不知而不慍
3. 有朋自遠方來
```

<details>
<summary>정답 확인</summary>

```
1. 學하고 思하지 아니하면 則 罔하다
2. 人이 知하지 아니하여도 慍하지 아니하다
3. 朋이 有하여 遠方으로부터 來하다
```

</details>

### 연습 3: 번역

영어로 번역하세요:

```
1. 子曰 學而時習之 不亦說乎
2. 子曰 吾十有五而志于學
3. 子曰 溫故而知新 可以為師矣
4. 子曰 學而不思則罔 思而不學則殆
```

<details>
<summary>정답 확인</summary>

```
1. 선생님께서 말씀하셨다: "배우고 때때로 그것을 익히면,
   또한 기쁘지 아니한가?"

2. 선생님께서 말씀하셨다: "열다섯에 학문에 뜻을 두었다."

3. 선생님께서 말씀하셨다: "옛것을 익히고 새것을 알면,
   가히 스승이 될 수 있다."

4. 선생님께서 말씀하셨다: "배우기만 하고 생각하지 않으면 어둡고,
   생각하기만 하고 배우지 않으면 위태롭다."
```

</details>

### 연습 4: 문법 패턴 식별

각 구절에 사용된 문법 패턴을 식별하세요:

```
1. 不亦說乎
2. 學而不思
3. 不思則罔
4. 可以為師矣
5. 人不知而不慍
```

<details>
<summary>정답 확인</summary>

```
1. 不亦說乎 → 不亦...乎 패턴 (수사적 의문: "또한 ~하지 아니한가?")
2. 學而不思 → A而不B 패턴 (A하되 B하지 않다: "배우되 생각하지 않다")
3. 不思則罔 → 不A則B 패턴 (A하지 않으면 B하다: "생각하지 않으면 어둡다")
4. 可以為師矣 → 可以...矣 패턴 (가히 ~할 수 있다 — 완료)
5. 人不知而不慍 → A而不B 패턴, 양보적 而 ("알아주지 않아도 원망하지 않다")
```

</details>

### 연습 5: 나이 표현

각 나이에 해당하는 논어의 표현과 그 의미를 쓰세요:

```
1. 15세: ___
2. 30세: ___
3. 40세: ___
4. 50세: ___
5. 60세: ___
6. 70세: ___
```

<details>
<summary>정답 확인</summary>

```
1. 15세: 志于學 (지우학) — 학문에 뜻을 세우다 (志學/지학)
2. 30세: 而立 (이립) — 확립되다; 독립하다
3. 40세: 不惑 (불혹) — 미혹되지 않다; 의심이 없다
4. 50세: 知天命 (지천명) — 하늘의 명을 알다
5. 60세: 耳順 (이순) — 귀가 순해지다; 어떤 말도 편히 듣다
6. 70세: 從心所欲不踰矩 (종심소욕불유구) — 마음을 따라도 법도를 넘지 않다
```

</details>
