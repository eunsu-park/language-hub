## 1. 역사 서술 소개

### 1.1 한문 역사 서술 전통

> **한문 역사 서술 전통**
>
> 중국의 역사 서술(史學/사학)은 세계에서 가장 오래된
> 연속적 역사 기록 전통 중 하나입니다.
>
> 주요 특징:
>   - 역사에서 도덕적 교훈 강조
>   - 전기적 접근 (개인의 이야기로 시대를 그림)
>   - 공식 궁정 사관이 실시간으로 사건을 기록
>   - "포폄" (褒貶/포폄) 전통 — 칭찬과 비판
>   - 한국 역사 서술에 영향 (삼국사기, 고려사 등)
>
> 주요 중국 역사서:
> - 春秋 (춘추) — 춘추 (孔子에 귀속)
> - 史記 (사기) — 사기 (司馬遷)
> - 漢書 (한서) — 한서 (班固)
> - 三國志 (삼국지) — 삼국지 (陳壽)
> - 資治通鑑 (자치통감) — 자치통감 (司馬光)

## 2. 사기(史記)

### 2.1 사기에 관하여

| 항목 | 상세 |
|------|------|
| 제목 | 史記 (사기, Shiji) — 사기 |
| 저자 | 司馬遷 (사마천, Sima Qian), 기원전 145-86년경 |
| 범위 | 전설상의 황제(黃帝)부터 한 무제까지 (약 2,500년) |
| 분량 | 130편(篇), 약 526,500자 |
| 지위 | 중국 전기체 역사 서술의 시조 |
| 문체 | 대화를 포함한 서사 산문; 생동감 있는 인물 묘사 |

### 2.2 사마천(司馬遷)에 관하여

> **司馬遷 (사마천, Sima Qian)**
>
> - **출생**: 기원전 145년경
> - **사망**: 기원전 86년경
> - **직위**: 太史令 (태사령, 한나라 궁정의 태사령)
>
> 생애:
>   - 부친 司馬談(사마담)이 사기 집필을 시작
>   - 사마천이 부친 사후 집필을 이어감
>   - 장군 李陵(이릉)을 변호하다 무제의 노여움을 삼
>   - 궁형(宮刑/궁형)에 처해짐 — 죽기보다 치욕을 감내하고 사기를 완성하기로 결심
>   - 그의 인내는 진실에 대한 헌신의 상징이 됨
>
> 유명한 구절 (임안에게 보내는 편지에서):
> 人固有一死 或重於泰山 或輕於鴻毛
> "사람은 본래 한 번 죽기 마련이니,
> 어떤 죽음은 태산보다 무겁고 어떤 죽음은 깃털보다 가볍다."

## 3. 사기의 구조

### 3.1 다섯 부분

사기는 이후 모든 중국 정사의 표준이 된 "기전체"(紀傳體/기전체) 형식을 창시했습니다:

| 부분 | 한국어 | 편수 | 내용 | 목적 |
|------|--------|------|------|------|
| 本紀 | 본기 | 12 | 제왕 전기 | 통치자/황제의 연대기 |
| 世家 | 세가 | 30 | 세가 | 제후와 그 가문의 이야기 |
| 列傳 | 열전 | 70 | 열전 | 주목할 만한 인물의 전기 |
| 表 | 표 | 10 | 연표 | 시간 순서표와 도표 |
| 書 | 서 | 8 | 지(志) | 주제별: 예악, 역법, 경제 등 |

### 3.2 부분 명칭 독해

| 한자 | 음독 | 훈독 | 문맥상 의미 |
|------|------|------|------------|
| 本 | 본 (bon) | 근본 본 | 근본, 기본 — 황제 자체의 기록 |
| 紀 | 기 (gi) | 기록할 기 | 기록, 기(紀) |
| 世 | 세 (se) | 대 세 | 대(代), 세습적 |
| 家 | 가 (ga) | 집 가 | 집, 가문 |
| 列 | 렬 (ryeol) | 벌일 렬 | 차례로 벌여 놓은; 일련의 |
| 傳 | 전 (jeon) | 전할 전 | 전기; 전하다 |
| 表 | 표 (pyo) | 겉 표 | 표, 도표 |
| 書 | 서 (seo) | 글 서 | 지(志), 서술 기록 |

## 5. 항우본기 발췌: 항우의 등장

### 5.1 배경

項羽(항우, Xiang Yu, 기원전 232-202년)는 진나라 멸망기 가장 강력한 군사 지도자였습니다. 사기에 실린 그의 전기는 한문 문학 전체에서 가장 극적이고 문학적인 구절 중 하나입니다.

### 5.2 원문: 젊은 항우

```
項籍少時 學書不成 去 學劍 又不成
項梁怒之 籍曰 書足以記名姓而已
劍一人敵 不足學 學萬人敵
於是項梁乃敎籍兵法 籍大喜
```

### 5.3 글자별 주석

| 한자 | 음독 | 문맥상 의미 |
|------|------|------------|
| 項籍 | 항적 | 항적 (항우의 본명) |
| 少時 | 소시 | 어릴 때; 젊었을 때 |
| 學 | 학 | 배웠다 |
| 書 | 서 | 글; 서예 |
| 不成 | 불성 | 이루지 못하다; 포기하다 |
| 去 | 거 | 그만두다; 떠나다 |
| 劍 | 검 | 검술 |
| 又 | 우 | 또; 다시 |
| 項梁 | 항량 | 항량 (그의 삼촌) |
| 怒 | 노 | 화를 내다 |
| 足以 | 족이 | ~하기에 충분하다 |
| 記 | 기 | 기록하다 |
| 名姓 | 명성 | 이름과 성 |
| 而已 | 이이 | 그뿐이다; 그만이다 |
| 一人敵 | 일인적 | 한 사람의 적수 |
| 不足 | 부족 | ~할 가치가 없다 |
| 萬人敵 | 만인적 | 만 사람의 적수 |
| 於是 | 어시 | 이에; 그리하여 |
| 乃 | 내 | 곧; 이에 |
| 兵法 | 병법 | 병법; 군사 전략 |
| 大喜 | 대희 | 크게 기뻐하다 |

### 5.4 석독

```
항적(항우)은 어릴 때 글을 배웠으나 이루지 못하고 그만두었다.
검술을 배웠으나 또한 이루지 못하였다.
항량(삼촌)이 그를 꾸짖자, 항적이 말하였다.
"글이란 이름과 성을 쓸 수 있으면 그만이고,
검은 한 사람을 상대하는 것이니 배울 것이 못 됩니다.
만 명을 상대하는 법을 배우겠습니다."
이에 항량이 항적에게 병법을 가르치니, 항적은 크게 기뻐하였다.
```

### 5.5 문법 분석

| 패턴 | 예시 | 기능 |
|------|------|------|
| 不成 | 學書不成 | 동사 + 不成 = "~에 성공하지 못하다" |
| 足以 | 足以記名姓 | "~하기에 충분하다" |
| 而已 | 記名姓而已 | "그뿐이다; 그만이다" (문말 한정) |
| 於是...乃 | 於是項梁乃敎 | "이에...곧" (서사적 전환) |

## 6. 항우본기 발췌: 최후의 전투

### 6.1 배경

해하(垓下/해하)에서 항우의 최후의 패배와 유명한 "우미인에 대한 이별"은 중국 문학에서 가장 비장한 장면 중 하나입니다.

### 6.2 원문: 해하의 포위

```
項王軍壁垓下 兵少食盡
漢軍及諸侯兵圍之數重
夜聞漢軍四面皆楚歌
項王乃大驚曰 漢皆已得楚乎
是何楚人之多也
```

### 6.3 주석

| 구절 | 음독 | 번역 |
|------|------|------|
| 項王軍壁垓下 | 항왕군벽해하 | 항왕의 군대가 해하에 진을 치고 있었다 |
| 兵少食盡 | 병소식진 | 병사는 적고 양식이 다하였다 |
| 漢軍及諸侯兵 | 한군급제후병 | 한나라 군대와 제후들의 군대가 |
| 圍之數重 | 위지수중 | 그를 몇 겹으로 포위하였다 |
| 夜聞漢軍 | 야문한군 | 밤에 한나라 군대에서 |
| 四面皆楚歌 | 사면개초가 | 사방에서 모두 초나라 노래를 부르는 것을 들으니 |
| 項王乃大驚曰 | 항왕내대경왈 | 항왕이 크게 놀라 말하였다: |
| 漢皆已得楚乎 | 한개이득초호 | "한나라가 이미 모두 초나라를 차지하였는가?" |
| 是何楚人之多也 | 시하초인지다야 | "어찌 초나라 사람이 이리도 많은가!" |

### 6.4 석독

```
항왕의 군대는 해하에 진을 치고 있었는데,
병사는 적고 양식이 다하였다.
한나라 군대와 제후들의 군대가 그를 몇 겹으로 포위하였다.
밤에 한나라 군대 사방에서 모두 초나라 노래를 부르는 것을 들으니,
항왕이 크게 놀라 말하였다.
"한나라가 이미 모두 초나라를 차지하였는가?
어찌 초나라 사람이 이리도 많은가!"
```

### 6.5 문화적 유산

이 구절에서 유명한 표현 四面楚歌(사면초가)가 탄생하였습니다:

| 표현 | 음독 | 직역 | 의미 |
|------|------|------|------|
| 四面楚歌 | 사면초가 | 사방에서 초나라 노래 | 사방이 모두 적에게 둘러싸임; 절망적 고립 |

> 이것은 현대 한국어에서 가장 자주 쓰이는 사자성어 중 하나로, 사방이 역경에 둘러싸인 상황을 묘사하는 데 사용됩니다.

## 7. 열전 서두 구절

### 7.1 伯夷列傳 — 백이열전 (서두)

사기 최초의 열전은 이후 모든 열전의 철학적 기조를 세웁니다:

```
伯夷叔齊 孤竹君之二子也
父欲立叔齊 及父卒 叔齊讓伯夷
伯夷曰 父命也 遂逃去
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 伯夷叔齊 | 백이숙제 | 백이와 숙제는 |
| 孤竹君之二子也 | 고죽군지이자야 | 고죽군의 두 아들이었다 |
| 父欲立叔齊 | 부욕립숙제 | 아버지가 숙제를 세우려 하였다 |
| 及父卒 | 급부졸 | 아버지가 돌아가시자 |
| 叔齊讓伯夷 | 숙제양백이 | 숙제가 백이에게 양보하였다 |
| 伯夷曰 | 백이왈 | 백이가 말하였다: |
| 父命也 | 부명야 | "아버지의 명이다" |
| 遂逃去 | 수도거 | 그리하여 도망쳐 떠났다 |

### 7.2 李將軍列傳 — 이장군열전 (서두)

```
李將軍廣者 隴西成紀人也
其先曰李信 秦時為將
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 李將軍廣者 | 이장군광자 | 이장군 광(廣)은 |
| 隴西成紀人也 | 농서성기인야 | 농서 성기 사람이다 |
| 其先曰李信 | 기선왈이신 | 그 선조는 이신(李信)이니 |
| 秦時為將 | 진시위장 | 진나라 때 장군을 지냈다 |

### 7.3 열전 서두의 정형 구문

열전 전기의 표준적인 서두 패턴에 주목하세요:

```
패턴: [이름]者 [지명]人也
"[이름]은 [지명] 사람이다"

예시:
孔子者 魯人也 — 공자는 노나라 사람이다
司馬遷者 夏陽人也 — 사마천은 하양 사람이다
```

---

## 8. 서사 산문 기법

### 8.1 대화 도입

역사 텍스트에는 직접 화법이 빈번하게 등장합니다:

| 형식 | 음독 | 의미 |
|------|------|------|
| A曰 | A왈 | A가 말하였다: |
| A謂B曰 | A위B왈 | A가 B에게 말하였다: |
| A問B曰 | A문B왈 | A가 B에게 물었다: |
| A對曰 | A대왈 | A가 대답하였다: |
| A怒曰 | A노왈 | A가 화를 내며 말하였다: |
| A歎曰 | A탄왈 | A가 탄식하며 말하였다: |

### 8.2 시간 표지

| 표지 | 음독 | 의미 |
|------|------|------|
| 初 | 초 | 처음에; 당초 |
| 已 | 이 | 이미 |
| 及 | 급 | ~에 이르러; ~할 무렵 |
| 後 | 후 | 이후에; 뒤에 |
| 遂 | 수 | 그리하여; 마침내 |
| 乃 | 내 | 곧; 이에 |
| 於是 | 어시 | 이때; 이에 |
| 卒 | 졸 | 마침내; 결국; 돌아가다 |

### 8.3 서사적 접속

| 패턴 | 음독 | 기능 | 예시 |
|------|------|------|------|
| 於是 | 어시 | 전환: "이에" | 於是項梁乃敎籍兵法 |
| 遂 | 수 | 결과: "그리하여" | 遂逃去 (그리하여 도망쳤다) |
| 乃 | 내 | 순차: "곧" | 乃大驚 (곧 크게 놀랐다) |
| 而 | 이 | 연결: "~하고/~하나" | 學書不成 去 (배웠으나 이루지 못하고) |
| 及 | 급 | 시간: "~에 이르러" | 及父卒 (아버지가 돌아가시자) |

## 9. 한문으로 쓴 한국의 역사 서술

### 9.1 한문으로 쓴 한국 역사서

한국의 역사가들은 천 년 이상 한문으로 역사를 기록했습니다:

| 저작 | 한국어 | 저자 | 시기 | 내용 |
|------|--------|------|------|------|
| 三國史記 | 삼국사기 | 金富軾 (김부식) | 1145 | 삼국의 역사 |
| 三國遺事 | 삼국유사 | 一然 (일연) | 1281년경 | 삼국의 설화와 역사 |
| 高麗史 | 고려사 | 鄭麟趾 (정인지) 외 | 1451 | 고려 왕조의 역사 |
| 朝鮮王朝實錄 | 조선왕조실록 | 궁정 사관 | 1413-1865 | 조선 왕조 연대기 |

### 9.2 삼국사기 예시

```
始祖姓朴氏 諱赫居世
前漢孝宣帝五鳳元年甲子 四月丙辰
```

| 구절 | 음독 | 번역 |
|------|------|------|
| 始祖姓朴氏 | 시조성박씨 | 시조의 성은 박씨이다 |
| 諱赫居世 | 휘혁거세 | 휘(이름)는 혁거세이다 |
| 前漢孝宣帝五鳳元年甲子 | 전한효선제오봉원년갑자 | 전한 효선제 오봉 원년 갑자 (기원전 57년) |

## 10. 연습 문제

### 연습 1: 역사 어휘

각 역사 용어의 의미를 쓰세요:

```
1. 王 (왕)
2. 帝 (제)
3. 臣 (신)
4. 將 (장)
5. 兵 (병)
6. 戰 (전)
7. 勝 (승)
8. 敗 (패)
9. 攻 (공)
10. 守 (수)
```

<details>
<summary>정답 확인</summary>

```
1. 王 (왕) = 왕; 통치자
2. 帝 (제) = 황제
3. 臣 (신) = 신하; 신민
4. 將 (장) = 장군 (군사)
5. 兵 (병) = 병사; 무기; 군사
6. 戰 (전) = 전투; 전쟁
7. 勝 (승) = 승리; 이기다
8. 敗 (패) = 패배; 지다
9. 攻 (공) = 공격하다
10. 守 (수) = 지키다; 방어하다
```

</details>

### 연습 2: 항우 구절 번역

각 구절을 번역하세요:

```
1. 項籍少時
2. 學書不成
3. 書足以記名姓而已
4. 學萬人敵
5. 籍大喜
```

<details>
<summary>정답 확인</summary>

```
1. 項籍少時 (항적소시) — 항적이 어릴 때
2. 學書不成 (학서불성) — 글을 배웠으나 이루지 못하다
3. 書足以記名姓而已 (서족이기명성이이) — 글이란 이름과 성을 기록할 수 있으면 그만이다
4. 學萬人敵 (학만인적) — 만 명을 상대하는 [법을] 배우겠다
5. 籍大喜 (적대희) — 항적이 크게 기뻐하였다
```

</details>

### 연습 3: 사면초가 독해

사면초가 구절을 번역하세요:

```
夜聞漢軍四面皆楚歌
項王乃大驚曰 漢皆已得楚乎
```

<details>
<summary>정답 확인</summary>

```
夜聞漢軍四面皆楚歌 (야문한군사면개초가)
밤에 한나라 군대 사방에서 모두 초나라 노래를 부르는 것을 들었다.

項王乃大驚曰 (항왕내대경왈)
항왕이 곧 크게 놀라 말하였다:

漢皆已得楚乎 (한개이득초호)
"한나라가 이미 모두 초나라를 차지하였는가?"
```

</details>

### 연습 4: 열전 서두 패턴

A者B人也 패턴으로 열전 서두를 작성하세요:

```
1. 공자, 노(魯)나라 사람
2. 맹자, 추(鄒)나라 사람
3. 사마천(司馬遷), 하양(夏陽) 사람
```

<details>
<summary>정답 확인</summary>

```
1. 孔子者 魯人也 (공자자 노인야) — 공자는 노나라 사람이다
2. 孟子者 鄒人也 (맹자자 추인야) — 맹자는 추나라 사람이다
3. 司馬遷者 夏陽人也 (사마천자 하양인야) — 사마천은 하양 사람이다
```

</details>

### 연습 5: 사기의 구조

각 부분과 그 내용을 짝지으세요:

```
A열 (부분)             B열 (내용)
1. 本紀                 a. 주목할 만한 인물의 전기
2. 世家                 b. 연표
3. 列傳                 c. 제왕 연대기
4. 表                   d. 주제별 지(志)
5. 書                   e. 세습 가문의 역사
```

<details>
<summary>정답 확인</summary>

```
1-c: 本紀 (본기) = 제왕 연대기 (통치자 전기)
2-e: 世家 (세가) = 세습 가문의 역사 (제후)
3-a: 列傳 (열전) = 주목할 만한 인물의 전기
4-b: 表 (표) = 연표
5-d: 書 (서) = 주제별 지(志) (예악, 음악, 경제 등)
```

</details>

### 연습 6: 서사 기법 식별

각 구절의 서사 기법을 식별하세요:

```
1. 項王曰 (항왕왈)
2. 於是項梁乃敎籍兵法 (어시항량내교적병법)
3. 及父卒 (급부졸)
4. 遂逃去 (수도거)
5. 項王乃大驚曰 (항왕내대경왈)
```

<details>
<summary>정답 확인</summary>

```
1. 項王曰 → 대화 도입 (A曰 = "A가 말하였다")
2. 於是項梁乃敎籍兵法 → 서사적 전환 (於是...乃 = "이에...곧")
3. 及父卒 → 시간 표지 (及 = "~에 이르러")
4. 遂逃去 → 결과 (遂 = "그리하여")
5. 項王乃大驚曰 → 순차적 행동 + 대화 (乃...曰 = "곧...말하였다")
```

</details>
