## 1. 한국어 속의 한자: 60%의 기반

### 1.1 한자가 한국어에 중요한 이유

한국어 어휘의 약 60-70%가 한자어(漢字語) — 한자에서 유래한 단어 — 로 구성되어 있습니다. 한자를 이해하면 대부분의 격식체, 학술, 전문 한국어 어휘의 의미를 파악할 수 있습니다.

> **한국어 어휘 구성**
>
> - **한자어 (漢字語)**: ~60-70%
> - **고유어 (固有語)**: ~25-30%
> - **외래어 (外來語)**: ~5-10%
>
> 예시: 도서관 (library)
>   - 도(圖) = 그림, 도면
>   - 서(書) = 책, 글
>   - 관(館) = 건물, 관
>   - → "책과 그림이 있는 건물" = 도서관
>
> 한자를 알면 한국어 단어의 의미가 보입니다.

### 1.2 한자가 한국어가 되는 과정

각 한자에는 고정된 한국어 발음(음독/音讀)이 있습니다. 한자가 결합하여 숙어가 되면 이 발음들이 연결됩니다:

| 한자 | 개별 독음 | 숙어 | 한국어 단어 |
|------|----------|------|------------|
| 學 + 校 | 학 + 교 | 學校 | 학교 (학교) |
| 先 + 生 | 선 + 생 | 先生 | 선생 (선생님) |
| 大 + 學 | 대 + 학 | 大學 | 대학 (대학교) |
| 圖 + 書 + 館 | 도 + 서 + 관 | 圖書館 | 도서관 (도서관) |

## 2. 2글자 숙어 형성 규칙

### 2.1 구조적 유형

대부분의 한자어는 2글자 숙어입니다. 예측 가능한 구조적 유형을 따릅니다:

| 유형 | 구조 | 예시 | 독음 | 의미 |
|------|------|------|------|------|
| **수식어 + 피수식어** | 형용사 + 명사 | 大學 | 대학 | 큰 + 배움 = 대학교 |
| **동사 + 목적어** | 동사 + 명사 | 讀書 | 독서 | 읽다 + 책 = 독서 |
| **주어 + 서술어** | 명사 + 동사 | 日出 | 일출 | 해 + 나오다 = 일출 |
| **병렬** | 명사 + 명사 (같은 범주) | 父母 | 부모 | 아버지 + 어머니 = 부모 |
| **반의 결합** | 반의어 + 반의어 | 上下 | 상하 | 위 + 아래 = 상하 |
| **동사 + 보어** | 동사 + 결과 | 說明 | 설명 | 말하다 + 밝다 = 설명하다 |

### 2.2 수식어 + 피수식어 (修飾語 + 被修飾語)

첫 번째 한자가 두 번째를 수식(설명)합니다:

| 숙어 | 독음 | 분석 | 의미 |
|------|------|------|------|
| 新聞 | 신문 | 新 (새로운) + 聞 (듣다) | 신문 (새로운 소식) |
| 白馬 | 백마 | 白 (흰) + 馬 (말) | 백마 |
| 高山 | 고산 | 高 (높은) + 山 (산) | 높은 산 |
| 古文 | 고문 | 古 (옛) + 文 (글) | 고문; 옛 글 |
| 長男 | 장남 | 長 (맏) + 男 (남자) | 장남 |
| 小說 | 소설 | 小 (작은) + 說 (이야기) | 소설 ("작은 이야기") |

### 2.3 동사 + 목적어 (動詞 + 目的語)

첫 번째 한자(동사)가 두 번째(목적어)에 작용합니다:

| 숙어 | 독음 | 분석 | 의미 |
|------|------|------|------|
| 讀書 | 독서 | 讀 (읽다) + 書 (책) | 독서 |
| 登山 | 등산 | 登 (오르다) + 山 (산) | 등산 |
| 作文 | 작문 | 作 (만들다) + 文 (글) | 작문 |
| 入學 | 입학 | 入 (들다) + 學 (학교) | 입학 |
| 出國 | 출국 | 出 (나가다) + 國 (나라) | 출국 |
| 開門 | 개문 | 開 (열다) + 門 (문) | 문을 열다 |

### 2.4 병렬 숙어 (竝列 構造)

동등한 지위의 두 한자가 결합합니다:

| 숙어 | 독음 | 분석 | 의미 |
|------|------|------|------|
| 山川 | 산천 | 山 (산) + 川 (내) | 산천; 산과 강 |
| 兄弟 | 형제 | 兄 (형) + 弟 (아우) | 형제 |
| 飮食 | 음식 | 飮 (마시다) + 食 (먹다) | 음식 |
| 言語 | 언어 | 言 (말) + 語 (말씀) | 언어 |
| 道路 | 도로 | 道 (길) + 路 (길) | 도로 |

### 2.5 반의 결합 숙어 (反義 構造)

두 반의어가 결합하여 더 넓은 개념을 표현합니다:

| 숙어 | 독음 | 분석 | 의미 |
|------|------|------|------|
| 上下 | 상하 | 上 (위) + 下 (아래) | 상하; 위아래 |
| 大小 | 대소 | 大 (큰) + 小 (작은) | 크기; 대소 |
| 長短 | 장단 | 長 (긴) + 短 (짧은) | 길이; 장단점 |
| 生死 | 생사 | 生 (살다) + 死 (죽다) | 생사 |
| 出入 | 출입 | 出 (나가다) + 入 (들다) | 출입 |
| 男女 | 남녀 | 男 (남자) + 女 (여자) | 남녀 |
| 左右 | 좌우 | 左 (왼쪽) + 右 (오른쪽) | 좌우; 영향 |
| 古今 | 고금 | 古 (옛) + 今 (지금) | 고금; 예와 지금 |

## 7. 한자어 수 체계

### 7.1 한국어의 두 가지 수 체계

한국어에는 두 가지 수 체계가 있습니다. 한자어 수 체계는 한자를 기반으로 합니다:

| 한자어 수 | 고유어 수 | 사용처 |
|----------|----------|--------|
| 일(一), 이(二), 삼(三)... | 하나, 둘, 셋... | 한자어 수: 날짜, 돈, 전화번호, 수학, 주소 |
| 일(一), 이(二), 삼(三)... | 하나, 둘, 셋... | 고유어 수: 나이 (살), 셈 (고유어 단위) |

### 7.2 일상에서의 한자어 수

| 용도 | 예시 | 한자 | 한국어 |
|------|------|------|--------|
| 날짜 | 1월 1일 | 一月 一日 | 일월 일일 |
| 돈 | 5,000원 | 五千원 | 오천 원 |
| 전화 | 02-123-4567 | — | 공이-일이삼-사오육칠 |
| 층수 | 3층 | 三層 | 삼층 |
| 연도 | 2026년 | 二千二十六年 | 이천이십육 년 |
| 월 | 3월 | 三月 | 삼월 |
| 백분율 | 50% | 五十% | 오십 퍼센트 |

### 7.3 특수 수 표현

| 표현 | 독음 | 직역 | 의미 |
|------|------|------|------|
| 第一 | 제일 | 차례 + 하나 | 제일; 첫째 |
| 十分 | 십분 | 열 + 나누다 | 충분히; 완전히 |
| 百分率 | 백분율 | 백 + 나누다 + 비율 | 백분율 |
| 千萬 | 천만 | 천 + 만 | 천만; "부디" |

| 민주주의 | 民主主義 | 民(백성) + 主(주인) + 主(주인) + 義(의리) | 민주주의 |

### 연습 2: 한국어 단어 분해

각 구성 요소의 한자와 의미를 쓰시오:

```
1. 학교 (학교)
2. 병원 (병원)
3. 도서관 (도서관)
4. 대한민국 (대한민국)
5. 자연 (자연)
```

<details>
<summary>정답 확인</summary>

```
1. 학교 = 學校
   學 (배울 학, 배우다) + 校 (학교 교, 학교 건물)
   → "배움의 장소"

2. 병원 = 病院
   病 (병 병, 병) + 院 (집 원, 기관)
   → "병을 다루는 기관" = 병원

3. 도서관 = 圖書館
   圖 (그림 도, 그림) + 書 (글 서, 책) + 館 (집 관, 건물)
   → "그림과 책이 있는 건물" = 도서관

4. 대한민국 = 大韓民國
   大 (큰 대, 큰) + 韓 (나라이름 한, 한국) + 民 (백성 민, 백성) + 國 (나라 국, 나라)
   → "큰 한국 백성의 나라" = 대한민국

5. 자연 = 自然
   自 (스스로 자, 스스로) + 然 (그러할 연, 그러하다)
   → "스스로 그러함" = 자연
```

</details>

### 연습 3: 구성 요소 확장

주어진 기본 한자를 사용하여 아는 숙어 5개를 쓰시오:

1. 學 (학, 배우다): _____, _____, _____, _____, _____

</details>

### 연습 4: 일상 속 한자 간판

다음 표지의 의미를 말하고 한자를 분석하시오:

```
1. 出口
2. 禁煙
3. 男 / 女
4. 非常口
5. 駐車禁止
```

<details>
<summary>정답 확인</summary>

```
1. 出口 (출구) = 출구
   出 (나갈 출, 나가다) + 口 (입 구, 구멍) → 나가는 곳

2. 禁煙 (금연) = 금연
   禁 (금할 금, 금하다) + 煙 (연기 연, 연기) → 흡연 금지

3. 男 / 女 (남 / 여) = 남성 / 여성 (화장실 표시)
   男 (사내 남, 남성) / 女 (계집 녀, 여성)

4. 非常口 (비상구) = 비상구
   非 (아닐 비, 아니다) + 常 (항상 상, 평상) + 口 (입 구, 구멍) → 비상시 출구

5. 駐車禁止 (주차금지) = 주차금지
   駐 (머무를 주, 머무르다) + 車 (수레 차, 차) + 禁 (금할 금, 금하다) + 止 (그칠 지, 멈추다)
```

</details>

### 연습 5: 수 읽기

각 수 표현을 한자와 한자어 독음으로 쓰시오:

```
1. 365
2. 2,026
3. 10,000
4. 50%
5. 3월 15일
```

<details>
<summary>정답 확인</summary>

```
1. 365 = 三百六十五 (삼백육십오)
2. 2,026 = 二千二十六 (이천이십육)
3. 10,000 = 一萬 (일만)
4. 50% = 五十百分率 (오십 퍼센트) / 五十% (오십 프로)
5. 3월 15일 = 三月 十五日 (삼월 십오일)
```

</details>
