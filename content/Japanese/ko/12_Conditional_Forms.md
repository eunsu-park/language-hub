## 1. 개요: "만약"을 표현하는 네 가지 방법

일본어에는 각기 다른 뉘앙스를 가진 네 가지 주요 조건 표현이 있습니다:

```
+------------------------------------------------------------------+
|  조건 표현           핵심 의미                 주요 특징             |
+------------------------------------------------------------------+
|  ば (ba)             일반적 "만약"             가정적               |
|  たら (tara)         "만약/~하면...그러면"      완료된 조건          |
|  なら (nara)         "그런 경우라면"            화제/문맥 기반       |
|  と (to)             "~하면 항상"              자연적 결과          |
+------------------------------------------------------------------+
```

> **중요**: 네 가지 모두 한국어로 "만약"이나 "~하면"으로 번역될 수 있지만, 서로 **호환되지 않습니다**. 각각 가장 자연스러운 특정 맥락이 있습니다.

## 2. ば 조건형 (仮定形)

ば형은 **일반적 조건**과 **가정 상황**을 표현합니다: "X라면, Y이다."

### 활용 규칙

| 유형 | 규칙 | 예시 |
|------|------|---------|
| 1그룹 동사 | 마지막 -u를 -e + ば로 변경 | 書く → 書**けば**, 読む → 読**めば**, 行く → 行**けば** |
| 2그룹 동사 | -る을 빼고 -れば 추가 | 食べる → 食べ**れば**, 見る → 見**れば** |
| する | すれば | する → **すれば** |
| 来る | 来れば (くれば) | 来る → **来れば** |
| い형용사 | -い를 빼고 -ければ 추가 | 高い → 高**ければ**, 安い → 安**ければ** |
| な형용사 | 어간 + であれば / なら(ば) | 静かであれば, 元気であれば |
| 명사 | 명사 + であれば | 学生であれば |
| 동사 부정 | ない → なければ | 行かない → 行か**なければ** |

### 1그룹 동사 ば형: 모든 어미

| 끝 가나 | 변화 | 예시 |
|-----------|--------|---------|
| く → けば | 書く → 書けば | |
| す → せば | 話す → 話せば | |
| つ → てば | 待つ → 待てば | |
| ぬ → ねば | 死ぬ → 死ねば | |
| ぶ → べば | 遊ぶ → 遊べば | |
| む → めば | 読む → 読めば | |
| る → れば | 帰る → 帰れば | |
| う → えば | 買う → 買えば | |
| ぐ → げば | 泳ぐ → 泳げば | |

### 용법과 뉘앙스

#### A. 일반적 / 가정 조건

> **安ければ、買います。** — 싸면 사겠습니다.
> **時間があれば、映画を見たい。** — 시간이 있으면 영화를 보고 싶다.

#### B. 속담과 보편적 진리

> **練習すれば、上手になる。** — 연습하면 잘하게 된다.
> **急がば回れ。** — 급할수록 돌아가라. (직역: 서두르면 돌아가라.)

#### C. 조언 (〜ばいい)

> **先生に聞けばいい。** — 선생님에게 물어보면 된다.
> **早く寝ればいいのに。** — 일찍 자면 좋을 텐데. (약간의 핀잔 뉘앙스)

### 제약: ば와 의지적 행위

> **핵심 규칙**: ば를 사용할 때 주절은 일반적으로 의지적 행위(명령, 요청, 권유)가 되어서는 안 됩니다. 단, 조건절이 상태나 형용사인 경우는 예외입니다.
>
> ✗ 東京に行けば、寿司を食べましょう。 (부자연스러움)
> ○ 東京に行ったら、寿司を食べましょう。 (たら를 대신 사용)
>
> ○ 天気がよければ、散歩しましょう。 (OK — 조건이 행위가 아닌 상태)

### 예문

1. **薬を飲めば、すぐ治る。** — 약을 먹으면 금방 낫는다.
2. **もっと勉強すれば、合格できる。** — 더 공부하면 합격할 수 있다.
3. **彼に会えば、伝えておきます。** — 그를 만나면 전해 놓겠습니다.
4. **この道をまっすぐ行けば、駅に着きます。** — 이 길을 곧장 가면 역에 도착합니다.
5. **暇であれば、手伝ってくれませんか。** — 한가하시면 도와주시겠습니까?

## 3. たら 조건형

たら형은 조건이 **완료된 후에** 결과가 일어나는 것을 표현합니다: "만약/~하면, 그러면 Y이다." 가장 다용도적이고 흔하게 사용되는 조건형입니다.

### 활용 규칙

**보통체 과거형**(た형)에 ら를 붙이기만 하면 됩니다:

| 유형 | 과거형 | たら형 |
|------|-----------|----------|
| 1그룹 동사 | 書いた | 書い**たら** |
| 1그룹 동사 | 行った | 行っ**たら** |
| 1그룹 동사 | 読んだ | 読ん**だら** |
| 2그룹 동사 | 食べた | 食べ**たら** |
| する | した | し**たら** |
| 来る | 来た (きた) | 来**たら** (きたら) |
| い형용사 | 高かった | 高かっ**たら** |
| な형용사 | 静かだった | 静かだっ**たら** |
| 명사 | 学生だった | 学生だっ**たら** |
| 동사 부정 | 行かなかった | 行かなかっ**たら** |

> **만드는 법은 간단합니다**: 과거형 + ら. 그것뿐입니다!

### 용법과 뉘앙스

#### A. 완료 조건 ("~하면/~하고 나서")

> **家に帰ったら、電話してね。** — 집에 돌아가면 전화해.
> **宿題が終わったら、遊びに行こう。** — 숙제가 끝나면 놀러 가자.

#### B. 가정 조건 ("만약 X가 일어난다면")

> **宝くじに当たったら、家を買う。** — 복권에 당첨되면 집을 살 것이다.
> **もし明日雨だったら、試合は中止です。** — 만약 내일 비가 오면 경기는 취소입니다.

#### C. 발견된 결과 ("~했더니, ~이었다")

> **窓を開けたら、猫がいた。** — 창문을 열었더니 고양이가 있었다.
> **駅に着いたら、電車がもう出ていた。** — 역에 도착했더니 전철이 이미 떠난 뒤였다.

#### D. 조언과 제안 (〜たらどう？)

> **医者に行ったらどうですか。** — 의사에게 가보는 게 어떻습니까?
> **もっと休んだら？** — 좀 더 쉬는 게 어때? (반말)

### もし 접두사

もし(만약)는 어떤 조건형 앞에든 강조를 위해 붙일 수 있지만, たら와 가장 자연스럽게 어울립니다:

> **もし時間があったら、来てください。** — 혹시 시간이 있으면 와 주세요.
> **もし私だったら、そうはしない。** — 만약 내가 였다면, 그렇게 하지 않을 것이다.

### 예문

1. **日本に行ったら、温泉に入りたい。** — 일본에 가면 온천에 들어가고 싶다.
2. **もし100万円あったら、何をする？** — 만약 100만 엔이 있다면 뭘 할 거야?
3. **大人になったら、パイロットになりたい。** — 어른이 되면 파일럿이 되고 싶다.
4. **道を曲がったら、コンビニが見えた。** — 길을 돌았더니 편의점이 보였다.
5. **分からなかったら、聞いてください。** — 모르면 물어보세요.

## 4. なら 조건형

なら형은 **화제 기반**입니다: 상대방이 말한 것이나 알려진 상황에 대응합니다. "~라는 경우라면" 또는 "~에 대해서라면"이라는 의미입니다.

### 활용 규칙

| 유형 | なら형 |
|------|----------|
| 동사 (보통체) | 行く**なら** |
| い형용사 | 高い**なら** (또는 高いの**なら**) |
| な형용사 | 静か**なら** (또는 静かな**のなら**) |
| 명사 | 学生**なら** |

> なら는 **보통체(사전형)**에 바로 붙입니다. 활용 변화가 필요 없습니다.

### 용법과 뉘앙스

#### A. 정보에 대한 반응 ("그런 경우라면...")

> A: 「明日は暇だ。」(내일은 한가해.)
> B: 「暇**なら**、映画を見に行かない？」(한가하다면, 영화 보러 가지 않을래?)

> A: 「日本語を勉強している。」(일본어를 공부하고 있어.)
> B: 「日本語を勉強している**なら**、この本がいいよ。」(일본어를 공부하고 있다면, 이 책이 좋아.)

#### B. 문맥에 기반한 조언

> **日本に行くなら、秋がいいですよ。** — 일본에 간다면 가을이 좋습니다.
> **買うなら、この店が安い。** — 산다면 이 가게가 싸다.

#### C. 조건부 한정

> **遊ぶなら、宿題をしてからにしなさい。** — 놀려면 숙제를 하고 나서 해라.
> **お酒を飲むなら、運転してはいけない。** — 술을 마실 거라면 운전해서는 안 된다.

### 주요 특징: 결과가 조건보다 먼저 일어날 수 있음

다른 조건형과 달리 なら에서는 결과 행위가 조건보다 **먼저** 일어날 수 있습니다:

> **スーパーに行くなら、牛乳を買ってきて。**
> 슈퍼에 갈 거라면, 우유를 사다 줘.
> (우유 구매가 가는 도중에, 도착 전에 일어남.)

たら와 비교:
> **スーパーに行ったら、牛乳を買ってきて。**
> 슈퍼에 가면, 우유를 사다 줘.
> (도착한 후에 구매가 일어남.)

### 예문

1. **北海道に行くなら、ラーメンを食べたほうがいい。** — 홋카이도에 간다면 라멘을 먹는 게 좋다.
2. **彼が来るなら、私も行く。** — 그가 온다면 나도 간다.
3. **学生なら、この割引が使えます。** — 학생이라면 이 할인을 사용할 수 있습니다.
4. **暑いなら、窓を開けてもいい。** — 덥다면 창문을 열어도 된다.
5. **田中さんのことなら、よく知っていますよ。** — 다나카 씨에 대해서라면, 잘 알고 있습니다.

## 5. と 조건형

と 조건형은 **자연적 결과**, **습관적 결과**, **발견**을 표현합니다: "X하면 항상 Y" 또는 "X하면 Y가 따라온다."

### 활용 규칙

| 유형 | と형 |
|------|--------|
| 동사 (보통체 비과거) | 押す**と** |
| い형용사 | 高い**と** |
| な형용사 | 静かだ**と** |
| 명사 | 春だ**と** |
| 부정 | 行かない**と** |

> と는 항상 **보통체 비과거형**에 붙입니다.

### 용법과 뉘앙스

#### A. 자연적 / 자동적 결과

> **ボタンを押すと、ドアが開く。** — 버튼을 누르면 문이 열린다.
> **春になると、桜が咲く。** — 봄이 되면 벚꽃이 핀다.

#### B. 습관적 행위

> **毎朝起きると、コーヒーを飲む。** — 매일 아침 일어나면 커피를 마신다.
> **この道をまっすぐ行くと、駅がある。** — 이 길을 곧장 가면 역이 있다.

#### C. 발견 (과거 시제)

> **窓を開けると、雪が降っていた。** — 창문을 열었더니 눈이 내리고 있었다.
> **家に帰ると、手紙が届いていた。** — 집에 돌아왔더니 편지가 도착해 있었다.

### 제약: 주절에 의지적 행위 불가

と의 주절에는 일반적으로 요청, 명령, 의도, 제안이 올 수 없습니다:

```
✗ ボタンを押すと、中に入ってください。   (부자연스러움)
○ ボタンを押したら、中に入ってください。 (たら를 사용)

✗ 6時になると、出かけましょう。          (부자연스러움)
○ 6時になったら、出かけましょう。        (たら를 사용)
```

> **규칙**: と는 결과가 **자동으로** 일어난다는 것을 암시하므로, 듣는 사람의 의도적 선택이 올 수 없습니다.

### 예문

1. **この薬を飲むと、眠くなる。** — 이 약을 먹으면 졸려진다.
2. **右に曲がると、銀行があります。** — 오른쪽으로 돌면 은행이 있습니다.
3. **冬になると、雪がたくさん降る。** — 겨울이 되면 눈이 많이 온다.
4. **ドアを開けると、猫が飛び出してきた。** — 문을 열었더니 고양이가 뛰어나왔다.
5. **食べ過ぎると、お腹が痛くなる。** — 너무 많이 먹으면 배가 아파진다.

## 6. 비교 도표: 올바른 조건형 선택하기

### 요약표

| 특징 | ば | たら | なら | と |
|---------|---|------|------|---|
| **핵심 의미** | 일반적 가정 | 완료 조건 | 화제/문맥 기반 | 자연적 결과 |
| **의지적 결과** | 제한적 | 가능 | 가능 | 불가 |
| **과거 사건** | 불가 | 가능 | 불가 | 가능 (발견) |
| **조언** | ばいい | たらどう | 직접 조언 | 불가 |
| **습관** | 가능 | 드묾 | 불가 | 가능 |
| **격식** | 약간 격식적 | 중립 (가장 흔함) | 중립 | 중립 |
| **もし와의 조합** | 가능 | 가장 자연스러움 | 가능 | 드묾 |

### 선택 가이드

```
+---------------------------------------------------+
|  "내일 비가 오면..."                                |
|                                                   |
|  일반적:     雨が降れば...     (ば)                |
|  구체적:     雨が降ったら...   (たら)              |
|  맥락적:     雨なら...         (なら)              |
|  습관적:     雨が降ると...     (と) — 항상         |
+---------------------------------------------------+
```

### 실용 비교 예시

| 상황 | 가장 적합한 형태 | 문장 | 이유 |
|-----------|-----------|----------|-----|
| 길 안내 | と | まっすぐ行くと、駅がある。 | 자동적 결과 |
| 계획 세우기 | たら | 雨が止んだら、出かけよう。 | 의지적 결과 |
| 정보에 반응 | なら | 東京に行くなら、案内する。 | 화제에 반응 |
| 일반적 진리 | ば | 努力すれば、成功する。 | 가정 |
| 발견 상황 | たら / と | ドアを開けたら/開けると、猫がいた。 | 과거 발견 |
| 조언 | ば / たら | 聞けばいい / 聞いたらどう | 제안 |

## 7. 자주 쓰이는 패턴과 관용 표현

### ば 패턴

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| 〜ば〜ほど | ~하면 할수록 | 練習すればするほど上手になる。 |
| 〜ばいい | ~하면 된다 | 先生に聞けばいい。 |
| 〜ばよかった | ~했으면 좋았을 텐데 (후회) | もっと勉強すればよかった。 |
| 〜なければならない | ~해야 한다 | 行かなければならない。 |

### たら 패턴

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| 〜たらどうですか | ~하는 게 어떻습니까? | 休んだらどうですか。 |
| 〜たらいいですか | ~하면 됩니까? | 何を持って行ったらいいですか。 |
| もし〜たら | 만약 ~한다면 | もし宝くじが当たったら... |

### なら 패턴

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| 〜のなら | ~이라면 (강조) | 行くのなら、早くしなさい。 |
| 명사 + なら | ~에 대해서라면 / ~이라면 | 寿司なら、あの店がいい。 |

### と 패턴

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| 〜と思う | ~라고 생각하다 | 美味しいと思う。 |
| 〜ないといけない | ~해야 한다 | 行かないといけない。 |
| 〜とき | ~할 때 | 日本にいたとき |

## 8. 흔한 실수와 주의점

### 실수 1: と와 명령·요청의 조합

```
✗ 駅に着くと、電話してください。
○ 駅に着いたら、電話してください。
```

> と의 결과절에는 의지적 행위(명령, 요청, 제안)가 올 수 없습니다.

### 실수 2: 일회성 과거 사건에 ば 사용

```
✗ 昨日、家に帰れば、母がいた。
○ 昨日、家に帰ったら、母がいた。
```

> ば는 일반적으로 특정 과거 사건에 사용하지 않습니다. "~했더니, Y가 일어났다"에는 たら를 사용하세요.

### 실수 3: なら의 시간 관계 혼동

```
スーパーに行くなら、牛乳を買ってきて。
  = 가는 길에 우유를 사다 줘 (요청이 가는 도중에 해당)

スーパーに行ったら、牛乳を買ってきて。
  = 슈퍼에 가면 우유를 사다 줘 (도착한 후에 해당)
```

### 실수 4: たら의 과도한 사용

たら가 가장 다용도적이지만, 모든 곳에 사용하면 부자연스럽습니다:

```
△ 春になったら、桜が咲く。 (허용되지만 덜 자연스러움)
○ 春になると、桜が咲く。   (습관적/자연적 사건에 더 자연스러움)
```

## 9. 연습 문제

### 연습 1: ば 조건형 만들기

ば형으로 변환하세요.

1. 食べる → ________
2. 安い → ________
3. 行く → ________
4. する → ________
5. 学生 → ________

<details>
<summary>정답</summary>

1. 食べ**れば** (2그룹: る을 빼고 れば 추가)
2. 安**ければ** (い형용사: い를 빼고 ければ 추가)
3. 行**けば** (1그룹: く → けば)
4. **すれば** (불규칙)
5. 学生**であれば** (명사 + であれば)

</details>

### 연습 2: たら 조건형 만들기

たら형으로 변환하세요.

1. 読む → ________
2. 来る (くる) → ________
3. いい → ________
4. 暇だ → ________
5. 食べない → ________

<details>
<summary>정답</summary>

1. 読ん**だら** (과거: 読んだ + ら)
2. 来**たら** (きたら) (과거: 来た + ら)
3. よかっ**たら** (いい의 과거: よかった + ら)
4. 暇だっ**たら** (과거: 暇だった + ら)
5. 食べなかっ**たら** (과거 부정: 食べなかった + ら)

</details>

### 연습 3: 가장 적합한 조건형 선택

가장 자연스러운 조건형(ば, たら, なら, と)으로 빈칸을 채우세요.

1. ボタンを押す______、ドアが開きます。(자동적 결과)
2. もし明日雨だっ______、ピクニックは中止にしよう。(계획 변경)
3. 日本に行く______、着物を買ったらどう？(상대방의 계획에 반응)
4. もっと練習すれ______、上手になりますよ。(일반적 진리)
5. 冬にな______、雪がたくさん降る。(습관적/계절적)

<details>
<summary>정답</summary>

1. ボタンを押す**と** (자동적/자연적 결과)
2. もし明日雨だっ**たら** (가정 + 의지적 결과)
3. 日本に行く**なら** (상대방이 말한 계획에 반응)
4. もっと練習すれ**ば** (일반적 진리/가정)
5. 冬にな**ると** (습관적 계절 사건)

</details>

### 연습 4: 조건형을 사용하여 번역

가장 적합한 조건형을 사용하여 일본어로 번역하세요.

1. 한가하면 노래방에 가자. (상대방이 말한 것에 반응)
2. 문을 열었더니 아무도 없었다. (과거 발견)
3. 매일 공부하면 일본어가 늘어난다. (일반적 진리)
4. 봄이 되면 따뜻해진다. (자연적 결과)
5. 컴퓨터를 살 거라면, 저 가게가 좋다. (문맥에 기반한 조언)

<details>
<summary>정답</summary>

1. 暇**なら**、カラオケに行こう。(なら — 정보에 반응)
2. ドアを開け**たら** / 開ける**と**、誰もいなかった。(たら/と — 과거 발견)
3. 毎日勉強すれ**ば**、日本語が上手になる。(ば — 일반적 진리)
4. 春になる**と**、暖かくなる。(と — 자연적 결과)
5. パソコンを買う**なら**、あの店がいい。(なら — 맥락에 기반한 조언)

</details>

### 연습 5: 오류 수정

각 문장에서 오류를 찾아 수정하세요.

1. ✗ まっすぐ行くと、右に曲がってください。
2. ✗ 昨日家に帰れば、手紙が届いていた。
3. ✗ 春になったら、桜が咲く。(자연적 계절 사건)
4. ✗ この薬を飲めば、眠くなる。(자동적 신체 효과)

<details>
<summary>정답</summary>

1. ○ まっすぐ行っ**たら**、右に曲がってください。(と의 결과에 명령 불가)
2. ○ 昨日家に帰っ**たら**、手紙が届いていた。(ば는 특정 과거 발견에 사용하지 않음)
3. ○ 春にな**ると**、桜が咲く。(습관적/자연적 사건에는 と가 더 자연스러움)
4. ○ この薬を飲む**と**、眠くなる。(자동적 결과에는 と가 더 자연스러움)

</details>
