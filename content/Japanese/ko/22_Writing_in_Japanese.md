## 1. 문자 체계와 격식

### 1.1 적절한 문자 선택

일본어 글쓰기는 세 가지 문자 체계를 동시에 사용하며, 각각은 고유한 어감을 가지고 있습니다:

| 문자 | 글쓰기에서의 용도 | 어감 |
|--------|-----------------|------|
| 漢字 (한자) | 내용어, 격식적 용어 | 성숙한, 교양 있는 |
| ひらがな (히라가나) | 문법 조사, 표준 한자가 없는 고유어 | 부드러운, 친근한 |
| カタカナ (가타카나) | 외래어, 강조, 전문 용어 | 현대적인, 날카로운 |

### 1.2 문어체 vs 구어체

일본어에는 구어(口語 / 話し言葉)와 다른 별개의 문어(文語 / 書き言葉)가 있습니다. 이 둘을 혼용하는 것은 흔한 오류입니다.

| 구어 (話し言葉) | 문어 (書き言葉) | 한국어 |
|------------------|-------------------|---------|
| ～じゃない | ～ではない | ~이/가 아니다 |
| ～だけど | ～だが / ～であるが | ~이지만 |
| ～してる | ～している | ~하고 있다 |
| ～なんだ | ～なのである / ～のだ | ~인 것이다 |
| すごく | 非常に / 極めて | 매우 |
| ちゃんと | きちんと / 正確に | 제대로 |
| やっぱり | やはり | 역시 |
| ～みたいだ | ～のようだ | ~인 것 같다 |
| ～っていう | ～という | ~라고 하는 |
| いろんな | さまざまな / 種々の | 다양한 |

### 1.3 격식체 글쓰기 표지

だ/である체(常体)는 에세이와 학술 글쓰기에 사용되고, です/ます체(敬体)는 편지와 이메일에 사용됩니다. 같은 문서 내에서 절대 혼용하지 않습니다.

```
+----------------------------------------------------------+
|         WRITING STYLE SELECTION                          |
+----------------------------------------------------------+
|                                                          |
|  敬体 (です/ます체):                                      |
|  • 편지 (手紙)                                           |
|  • 이메일 (メール)                                        |
|  • 비즈니스 문서                                          |
|  • 자기소개 에세이                                        |
|                                                          |
|  常体 (だ/である체):                                      |
|  • 학술 에세이 (論文)                                     |
|  • 보고서 (レポート)                                      |
|  • 신문 기사                                              |
|  • 일기 (日記)                                            |
|                                                          |
|  한 문서 내에서 절대 두 문체를 혼용하지 않습니다.             |
+----------------------------------------------------------+
```

## 2. 구두점과 서식

### 2.1 일본어 구두점

| 기호 | 명칭 | 용도 |
|------|------|-------|
| 。 | 句点（くてん） | 마침표 — 문장을 끝냄 |
| 、 | 読点（とうてん） | 쉼표 — 절을 구분함 |
| 「」 | 鉤括弧（かぎかっこ） | 인용 부호 |
| 『』 | 二重鉤括弧 | 인용 안의 인용, 책/영화 제목 |
| ・ | 中黒（なかぐろ） | 항목 구분자 (글머리 기호 같은 역할) |
| ～ | 波ダッシュ | 범위 표시 (9時～5時) |
| … | 三点リーダー | 줄임표 (항상 두 개: ……) |
| ― | ダッシュ | 대시 (항상 두 개: ――) |
| （） | 丸括弧 | 괄호 |
| ※ | 米印（こめじるし） | 주의/참고 표시 |

### 2.2 원고용지 규칙 (原稿用紙)

전통적인 일본어 글쓰기에서 原稿用紙 (겐코우요우시 — 400자 원고용지)는 엄격한 규칙을 따릅니다:

```
+----------------------------------------------------------+
|         原稿用紙 RULES                                    |
+----------------------------------------------------------+
|                                                          |
|  1. 제목: 1행, 2-3칸 비우고 제목 시작                      |
|  2. 이름: 2행, 마지막 칸 앞에서 끝나도록                    |
|  3. 새 문단: 1칸 들여쓰기 (一字下げ)                       |
|  4. 구두점: 절대 행의 시작에 오지 않음                      |
|     → 이전 행의 끝으로 이동                                |
|     → 또는 이전 문자와 마지막 칸을 공유                     |
|  5. 작은 가나 (っ, ゃ, ゅ, ょ):                            |
|     → 행의 시작에 올 수 있음 (구두점과 달리)                |
|  6. 숫자:                                                 |
|     → 세로쓰기: 漢数字 (一, 二, 三)                        |
|     → 가로쓰기: 아라비아 숫자 (1, 2, 3)                    |
|  7. 「 여는 괄호: 행의 시작에 옴                            |
|  8. 대화: 새로운 화자마다 새 행 시작                        |
+----------------------------------------------------------+
```

### 2.3 가로쓰기 vs 세로쓰기

| 특징 | 가로쓰기 (横書き) | 세로쓰기 (縦書き) |
|---------|---------------------|-------------------|
| 방향 | 왼쪽에서 오른쪽 | 위에서 아래, 오른쪽에서 왼쪽 |
| 사용처 | 웹, 과학, 비즈니스 | 문학, 신문, 격식체 편지 |
| 숫자 | 아라비아 숫자 (1, 2, 3) | 한자 (一、二、三) |
| 구두점 | 표준 배치 | 동일한 규칙이 회전하여 적용 |

## 3. 에세이 구조 (序論・本論・結論)

### 3.1 세 부분 구조

일본어 에세이는 명확한 세 부분 패턴을 따릅니다:

```
+----------------------------------------------------------+
|         JAPANESE ESSAY STRUCTURE                         |
+----------------------------------------------------------+
|                                                          |
|  序論 (じょろん) — 서론                                    |
|  ├── 주제/문제 제시                                       |
|  ├── 간단한 배경 제공                                     |
|  └── 논지/주장 제시                                       |
|      표현: ～について論じる / ～を考察する                   |
|                                                          |
|  本論 (ほんろん) — 본론                                    |
|  ├── 논점 1 + 근거 / 예시                                 |
|  ├── 논점 2 + 근거 / 예시                                 |
|  ├── 반론 + 반박 (선택 사항)                               |
|  └── 표현: まず... / 次に... / さらに...                   |
|                                                          |
|  結論 (けつろん) — 결론                                    |
|  ├── 주요 논점 요약                                       |
|  ├── 논지 재진술                                          |
|  └── 최종 견해 / 전망                                     |
|      표현: 以上のことから... / 結論として...                 |
|                                                          |
+----------------------------------------------------------+
```

### 3.2 대안적 구조: 起承転結

짧은 글에서 자주 사용되는 전통적인 일본식 구조입니다:

| 부분 | 한자 | 역할 | 한국어 |
|------|-------|------|---------|
| 起 | き | 도입 | 주제 소개 |
| 承 | しょう | 전개 | 주제 확장 |
| 転 | てん | 전환 / 반전 | 다른 관점이나 반론 제시 |
| 結 | けつ | 결말 | 전체를 하나로 모으기 |

### 3.3 에세이 서두 패턴

| 패턴 | 예시 |
|---------|---------|
| 질문 | 「なぜ日本語を学ぶ外国人が増えているのだろうか。」 |
| 통계 | 「2024年、日本語能力試験の受験者数は過去最高を記録した。」 |
| 정의 | 「グローバル化とは、国境を越えた交流が活発になる現象である。」 |
| 인용 | 「『言葉は文化の鏡である』と言われている。」 |
| 일반적 진술 | 「現代社会において、外国語学習の重要性はますます高まっている。」 |

### 3.4 에세이 결말 패턴

| 패턴 | 예시 |
|---------|---------|
| 요약 | 「以上のことから、○○であると結論づけられる。」 |
| 재진술 | 「本稿で論じたように、○○は○○にとって不可欠である。」 |
| 전망 | 「今後、○○がさらに発展することが期待される。」 |
| 행동 촉구 | 「私たち一人一人が○○について考える必要があるのではないだろうか。」 |
| 열린 질문 | 「この問題に対する答えは、まだ見つかっていない。」 |

### 3.5 에세이 문단 예시 (常体)

```
  現代社会において、人工知能（AI）の発展は目覚ましいものがある。
特に、自然言語処理の分野では、大規模言語モデルの登場により、
機械が人間のような文章を生成できるようになった。しかし、
これは言語学習にどのような影響を与えるのだろうか。

  まず、AI翻訳ツールの精度向上により、外国語を学ぶ必要性を
疑問視する声がある。確かに、日常的なコミュニケーションにおいて、
翻訳アプリは便利なツールとなりつつある。しかしながら、言語学習の
意義は単なる意思疎通にとどまらない。言語を学ぶことは、その文化を
理解し、異なる思考方法を身につけることでもあるのだ。
```

## 4. 격식체 편지 작성 (手紙の書き方)

### 4.1 편지 구조

일본어 격식체 편지는 매우 정해진 형식을 따릅니다:

```
+----------------------------------------------------------+
|        FORMAL LETTER STRUCTURE (手紙の構成)               |
+----------------------------------------------------------+
|                                                          |
|  ① 前文 (서두)                                           |
|     ├── 頭語 (시작 인사): 拝啓                            |
|     ├── 時候の挨拶 (계절 인사)                             |
|     └── 安否の挨拶 (안부 인사)                             |
|                                                          |
|  ② 主文 (본문)                                            |
|     ├── 起こし言葉: さて、 / ところで、                    |
|     └── 주요 내용                                         |
|                                                          |
|  ③ 末文 (맺음 부분)                                       |
|     ├── 結びの挨拶 (맺음 인사)                             |
|     └── 結語 (끝맺음 인사): 敬具                           |
|                                                          |
|  ④ 後付け (부기 정보)                                      |
|     ├── 日付 (날짜)                                       |
|     ├── 差出人名 (발신자 이름)                             |
|     └── 宛名 (수신자 이름 + 様)                            |
|                                                          |
+----------------------------------------------------------+
```

### 4.2 시작 인사와 끝맺음 인사 쌍 (頭語 / 結語)

| 유형 | 시작 (頭語) | 끝맺음 (結語) | 용도 |
|------|---------------|---------------|-------|
| 일반 | 拝啓 (はいけい) | 敬具 (けいぐ) | 표준 격식 |
| 더 격식적 | 謹啓 (きんけい) | 謹白 (きんぱく) | 매우 격식적인 경우 |
| 답장 | 拝復 (はいふく) | 敬具 (けいぐ) | 편지에 답장할 때 |
| 긴급 | 急啓 (きゅうけい) | 草々 (そうそう) | 긴급한 사안 |
| 비격식 편지 | 前略 (ぜんりゃく) | 草々 (そうそう) | 형식을 생략할 때 |

> **규칙**: 頭語와 結語는 반드시 짝을 맞추어 사용해야 합니다. 혼용하면 부적절합니다.

### 4.3 계절 인사 (時候の挨拶)

각 달에는 전통적인 계절 인사가 있습니다:

| 월 | 인사 | 번역 |
|-------|----------|-------------|
| 1월 | 新春の候 / 厳寒の候 | 새봄의 계절 / 엄한의 계절 |
| 2월 | 立春の候 / 余寒の候 | 입춘의 계절 / 여한의 계절 |
| 3월 | 早春の候 / 春暖の候 | 이른 봄의 계절 / 봄 따뜻함의 계절 |
| 4월 | 陽春の候 / 桜花の候 | 따뜻한 봄의 계절 / 벚꽃의 계절 |
| 5월 | 新緑の候 / 薫風の候 | 신록의 계절 / 향기로운 바람의 계절 |
| 6월 | 入梅の候 / 初夏の候 | 장마의 계절 / 초여름의 계절 |
| 7월 | 盛夏の候 / 猛暑の候 | 한여름의 계절 / 맹서의 계절 |
| 8월 | 残暑の候 / 晩夏の候 | 잔서의 계절 / 늦여름의 계절 |
| 9월 | 初秋の候 / 秋涼の候 | 초가을의 계절 / 가을 서늘함의 계절 |
| 10월 | 秋冷の候 / 紅葉の候 | 가을 추위의 계절 / 단풍의 계절 |
| 11월 | 晩秋の候 / 向寒の候 | 늦가을의 계절 / 추위를 향하는 계절 |
| 12월 | 初冬の候 / 師走の候 | 초겨울의 계절 / 연말 분주함의 계절 |

### 4.4 격식체 편지 예시

```
                                          拝啓

  陽春の候、○○様におかれましては
ますますご清祥のこととお慶び申し上げます。
  平素は格別のご高配を賜り、厚く御礼申し上げます。

  さて、このたびは○○の件で
ご連絡させていただきました。
  ○○につきまして、ご検討いただけますと幸いです。

  末筆ながら、皆様のご健康と
ご多幸をお祈り申し上げます。

                                          敬具

  令和八年四月十日
                        山田太郎

  鈴木花子様
```

## 5. 이력서 형식 (履歴書)

### 5.1 일본식 이력서 구조

일본의 履歴書는 서양식 이력서와 매우 다른, 극도로 표준화된 형식을 따릅니다:

```
+----------------------------------------------------------+
|         日本式履歴書 (JAPANESE RESUME)                     |
+----------------------------------------------------------+
|                                                          |
|  DATE: 令和○年○月○日現在                                 |
|                                                          |
|  PHOTO: 증명사진, 3cm x 4cm, 우측 상단에 부착              |
|                                                          |
|  ┌──────────────────────────────────────────┐            |
|  │ ふりがな:  やまだ たろう                    │            |
|  │ 氏名:     山田 太郎                        │            |
|  │ 生年月日:  令和○年○月○日生（○歳）          │            |
|  │ 性別:     男                               │            |
|  │ 住所:     〒___-____ ○○県○○市...          │            |
|  │ 電話:     090-XXXX-XXXX                   │            |
|  │ E-mail:   xxxxxx@xxx.com                  │            |
|  └──────────────────────────────────────────┘            |
|                                                          |
|  学歴・職歴 (학력 및 경력)                                 |
|  ├── 시간순으로 나열                                      |
|  ├── 学歴 헤더, 학력 항목                                 |
|  ├── 職歴 헤더, 경력 항목                                 |
|  └── 以上 (이상)으로 끝맺음                                |
|                                                          |
|  免許・資格 (면허 및 자격)                                  |
|  ├── JLPT, 운전면허 등                                   |
|  └── 최신순 또는 관련성 순                                 |
|                                                          |
|  志望動機 (지원 동기)                                      |
|  └── 3-5문장으로 이 회사를 선택한 이유 설명                 |
|                                                          |
|  本人希望記入欄 (희망 사항 기입란)                           |
|  └── 특별한 요청이 없으면 「貴社の規定に従います」           |
|                                                          |
+----------------------------------------------------------+
```

### 5.2 서양식 이력서와의 주요 차이점

| 항목 | 일본식 履歴書 | 서양식 이력서 |
|---------|----------------|----------------|
| 사진 | 필수 (격식 사진) | 보통 포함하지 않음 |
| 생년월일 | 필수 | 보통 생략 |
| 성별 | 기재 | 보통 생략 |
| 형식 | 표준화된 템플릿 | 자유로운 디자인 |
| 길이 | 1-2페이지 (고정 형식) | 다양함 |
| 필기 | 전통적으로 손글씨 (현재는 타이핑도 가능) | 항상 타이핑 |

| 자기소개 | 志望動機 (필수) | 선택적 커버레터 |

## 6. 보고서 작성 (レポート)

### 6.1 보고서 구조

학술 및 업무 보고서는 다음 구조를 따릅니다:

```
+----------------------------------------------------------+
|         REPORT STRUCTURE (レポートの構成)                  |
+----------------------------------------------------------+
|                                                          |
|  表紙 (표지)                                              |
|  ├── 제목                                                |
|  ├── 과목명/부서명                                        |
|  ├── 저자명 및 학번/사번                                   |
|  └── 날짜                                                |
|                                                          |
|  目次 (목차)                                              |
|  └── 섹션 제목과 페이지 번호                               |
|                                                          |
|  はじめに / 序論 (서론)                                    |
|  ├── 배경 및 목적                                         |
|  └── 보고서의 범위                                        |
|                                                          |
|  本文 (본문)                                              |
|  ├── 第1章: 첫 번째 주요 섹션                              |
|  ├── 第2章: 두 번째 주요 섹션                              |
|  └── 근거, 데이터, 분석                                   |
|                                                          |
|  結論 / おわりに (결론)                                    |
|  ├── 발견사항 요약                                        |
|  └── 향후 과제                                            |
|                                                          |
|  参考文献 (참고 문헌)                                      |
|  └── 인용 순서대로 나열                                   |
|                                                          |
+----------------------------------------------------------+
```

### 6.2 보고서 문체

보고서는 常体 (だ/である체)를 사용하며, 특별히 요청받지 않는 한 개인적 의견을 피합니다.

| 적절한 표현 | 부적절한 표현 |
|----|-------|
| ～と考えられる (~라고 생각할 수 있다) | ～と思う (나는 ~라고 생각한다) |
| ～であると言える (~라고 말할 수 있다) | ～だと感じた (나는 ~라고 느꼈다) |
| ～ことが明らかになった (~이 분명해졌다) | ～のは面白い (~은 흥미롭다) |
| データによれば (데이터에 따르면) | たぶん (아마) |
| ～と推測される (~라고 추측된다) | ～かもしれない (~일지도 모른다) |

### 6.3 일본어 참고 문헌 표기

| 인용 유형 | 형식 |
|---------------|--------|
| 서적 | 著者名（出版年）『書名』出版社 |
| 학술지 논문 | 著者名（発表年）「論文タイトル」『雑誌名』巻号, ページ |
| 웹사이트 | 著者名（更新年）「ページタイトル」URL（閲覧日: ○年○月○日） |
| 본문 내 인용 | ○○（2024）によれば… / ○○（2024: 45）は…と述べている |

### 6.4 유용한 보고서 표현

| 기능 | 표현 |
|----------|--------|
| 목적 진술 | 本レポートでは○○について考察する。 |
| 배경 | ○○は近年注目されている問題である。 |
| 정의 | ○○とは、○○のことを指す。 |
| 데이터 제시 | 図1に示すように、○○である。 |
| 비교 | ○○と比較すると、○○であることがわかる。 |
| 인과관계 | ○○の結果、○○が生じた。 |
| 결론 도출 | 以上の分析から、○○であると結論づけられる。 |
| 한계점 | 本研究の限界として、○○が挙げられる。 |

## 7. 이메일 작성 (メールの書き方)

### 7.1 비격식 vs 격식 이메일

21과에서 비즈니스 이메일을 상세히 다루었지만, 이 섹션에서는 보다 넓은 범위의 이메일 작성을 다룹니다.

| 요소 | 비격식 (친구에게) | 반격식 (선생님에게) | 격식 (회사에) |
|---------|-------------------|------------------------|-------------------|
| 인사 | ○○ちゃん！ / やっほー | ○○先生、こんにちは | ○○様、お世話になっております |
| 어조 | タメ口 (반말) | です/ます | ビジネス敬語 |
| 맺음말 | じゃあね！ / またね | よろしくお願いします | 何卒よろしくお願い申し上げます |
| 이모지 | OK ✓ | 최소한으로 | 절대 안 됨 ✗ |

### 7.2 교수/선생님에게 보내는 이메일

```
件名：○○授業のレポート提出について

○○先生

こんにちは。○○学部○年の山田太郎です。

○月○日締め切りのレポートについてご質問がございます。
参考文献の数に指定はありますでしょうか。

お忙しいところ恐れ入りますが、
ご回答いただけますと幸いです。

よろしくお願いいたします。

○○学部○年
山田太郎
学籍番号：12345678
```

### 7.3 회사에 보내는 문의 이메일

```
件名：商品○○についてのお問い合わせ

○○株式会社
カスタマーサポート部 御中

初めてメールをお送りいたします。
○○と申します。

貴社の商品「○○」について、
いくつかお伺いしたい点がございます。

1. ○○の仕様について
2. 納期について
3. 大量注文時の割引について

ご多忙のところ恐縮ですが、
ご回答いただけますと幸いです。

何卒よろしくお願い申し上げます。

山田太郎
TEL: 090-XXXX-XXXX
E-mail: taro.yamada@xxx.com
```

## 8. 흔한 글쓰기 실수

### 8.1 비원어민 학습자의 실수

| 실수 | 부적절한 예 | 올바른 예 | 설명 |
|---------|-----------|---------|-------------|
| 문체 혼용 | ～です。～である。 | 하나를 선택하여 일관되게 사용 | 敬体와 常体를 절대 혼용하지 않음 |
| は vs が 혼동 | 私が学生です。 | 私は学生です。 | は는 주제, が는 새로운 정보/주어 |
| 조사 생략 | 東京行く | 東京に行く | 글쓰기에서는 조사를 생략할 수 없음 |
| 구어체 사용 | ～ちゃった / ～じゃない | ～てしまった / ～ではない | 격식적 글쓰기에서는 문어체 사용 |
| 지나치게 긴 문장 | ～て、～て、～て、～た。 | 짧은 문장으로 분리 | て형 연결이 너무 많음 |
| 잘못된 조수사 | 人が3つ | 人が3人 | 적절한 조수사 사용 |
| 한자 과용 | 出来る事を全て遣る | できることをすべてやる | 일부 단어는 관례상 히라가나로 표기 |
| 문단 들여쓰기 누락 | 들여쓰기 없음 | 一字下げ (1칸 들여쓰기) | 새 문단에서는 항상 들여쓰기 |

### 8.2 관례상 히라가나로 쓰는 단어 (한자 사용 안 함)

다음 단어들은 한자가 있지만, 관례상 히라가나로 표기합니다:

| 한자 형태 | 히라가나 형태 | 의미 |
|-----------|---------------|---------|
| 出来る | できる | 할 수 있다 |
| 事 | こと | 것 (추상적) |
| 物 | もの | 것 (구체적) |
| 所 | ところ | 장소 / 측면 |
| 時 | とき | ~할 때 |
| 為 | ため | ~을 위해 / ~이기 때문에 |
| 通り | とおり | ~대로 |
| 下さい | ください | ~해 주세요 |
| 頂く | いただく | 받다 (겸양) |
| 致す | いたす | 하다 (겸양) |

## 9. 유용한 접속사와 전환 표현

### 9.1 기능별 접속사

**추가**:
| 일본어 | 사용 수준 | 한국어 |
|----------|-------------|---------|
| また | 문어/구어 | 또한, 게다가 |
| さらに | 문어/구어 | 더 나아가 |
| 加えて | 문어 | 더불어 |
| そのうえ | 문어/구어 | 그 위에 |
| なおかつ | 문어 | 게다가 (격식) |

**대조**:
| 일본어 | 사용 수준 | 한국어 |
|----------|-------------|---------|
| しかし | 문어/구어 | 그러나 |
| しかしながら | 문어 (격식) | 그럼에도 불구하고 |
| 一方（で） | 문어/구어 | 한편 |
| それに対して | 문어 | 그에 반해 |
| ～にもかかわらず | 문어 | ~에도 불구하고 |

**인과관계**:
| 일본어 | 사용 수준 | 한국어 |
|----------|-------------|---------|
| したがって | 문어 | 따라서 |
| そのため | 문어/구어 | 그 때문에 |
| その結果 | 문어 | 그 결과 |
| ～ことから | 문어 | ~라는 사실에서 |
| ゆえに | 문어 (격식) | 고로 |

**예시와 설명**:
| 일본어 | 사용 수준 | 한국어 |
|----------|-------------|---------|
| 例えば | 문어/구어 | 예를 들어 |
| 具体的には | 문어 | 구체적으로는 |
| すなわち | 문어 | 즉, 다시 말하면 |
| つまり | 문어/구어 | 요컨대 |
| 言い換えれば | 문어 | 달리 말하면 |

**결론**:
| 일본어 | 사용 수준 | 한국어 |
|----------|-------------|---------|
| 以上のことから | 문어 | 이상의 내용에서 |
| 結論として | 문어 | 결론적으로 |
| 要するに | 문어/구어 | 요약하면 |
| このように | 문어 | 이와 같이 |
| 総じて | 문어 (격식) | 전체적으로 |

## 10. 연습 문제

### 연습 1: 문체 교정

다음 구어체 문장을 문어체(常体)로 변환하십시오:

1. すごくいい天気だったんで、外で食べちゃった。
2. やっぱり日本語ってむずかしいんだよね。
3. いろんな人がいるけど、みんなちゃんとやってる。
4. 何でこうなっちゃったのか、全然わかんない。

<details>
<summary>정답</summary>

1. 非常に天気が良かったため、屋外で食事をした。
2. やはり日本語は難しいものである。
3. さまざまな人がいるが、全員がきちんと取り組んでいる。
4. なぜこのようになったのか、まったくわからない。

</details>

### 연습 2: 계절 인사

각 달에 맞는 격식체 편지의 첫 두 줄을 작성하십시오:

1. 7월
2. 10월
3. 1월

<details>
<summary>정답</summary>

1. 7월:
拝啓　盛夏の候、○○様におかれましてはますますご清祥のこととお慶び申し上げます。

2. 10월:
拝啓　秋冷の候、○○様におかれましてはますますご健勝のこととお慶び申し上げます。

3. 1월:
拝啓　新春の候、○○様におかれましてはますますご繁栄のこととお慶び申し上げます。

</details>

### 연습 3: 에세이 서론 작성

다음 주제에 대해 常体로 3-4문장의 서론(序論)을 작성하십시오:
「外国語学習にAI翻訳ツールは必要か」(외국어 학습에 AI 번역 도구는 필요한가)

<details>
<summary>예시 답안</summary>

近年、AI技術の急速な発展に伴い、翻訳ツールの精度が飛躍的に向上している。Google翻訳やDeepLなどのサービスは、日常的なコミュニケーションにおいて十分な品質を提供するようになった。このような状況の中で、外国語学習におけるAI翻訳ツールの役割について、さまざまな議論がなされている。本稿では、AI翻訳ツールが外国語学習にとって有益であるかどうかについて、その利点と課題を考察する。

</details>

### 연습 4: 이메일 작성

일본어 선생님에게 과제 제출 기한을 일주일 연장해 달라고 요청하는 이메일을 작성하십시오. 다음 내용을 포함하세요:
- 적절한 제목
- 적절한 인사
- 요청 사유
- 정중한 맺음말

<details>
<summary>예시 답안</summary>

```
件名：宿題提出期限の延長のお願い

○○先生

こんにちは。日本語クラスの○○です。
いつもご指導いただきありがとうございます。

来週月曜日が締め切りの作文課題についてご相談がございます。
現在、他の科目の期末試験が重なっており、
十分な時間を確保することが難しい状況です。

大変恐縮ですが、提出期限を1週間
延長していただくことは可能でしょうか。

ご迷惑をおかけして申し訳ございませんが、
ご検討いただけますと幸いです。

よろしくお願いいたします。

○○（学籍番号：XXXXXXXX）
```

</details>

### 연습 5: 이력서 작성

다음 경력을 가진 사람의 学歴 (학력) 부분을 작성하십시오:
- 2016년 4월 동경도립고등학교 입학
- 2019년 3월 졸업
- 2019년 4월 와세다대학 문학부 입학
- 현재 재학 중

<details>
<summary>정답</summary>

```
            学歴
平成28年 4月  東京都立○○高等学校 入学
平成31年 3月  東京都立○○高等学校 卒業
平成31年 4月  早稲田大学 文学部 入学
                          現在に至る
```

참고: 平成28年 = 2016, 平成31年 = 2019. 일본식 이력서는 전통적으로 연호(元号)를 사용하지만, 서력 사용도 점점 허용되고 있습니다.

</details>
