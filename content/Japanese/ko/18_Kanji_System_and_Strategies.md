## 1. 한자의 작동 원리

한자는 중국에서 차용한 표의 문자입니다. 각 글자는 **의미**와 **소리**를 담고 있으며, 이들이 결합하여 단어를 형성합니다.

### 1.1 기원에 따른 한자의 유형

| 유형 | 일본어 | 설명 | 예시 |
|------|----------|-------------|---------|
| 상형문자 | 象形文字 (しょうけいもじ) | 사물의 양식화된 그림 | 山 (산), 川 (강), 日 (해) |
| 지사문자 | 指事文字 (しじもじ) | 추상적 개념을 시각적으로 표현 | 上 (위), 下 (아래), 一 (하나) |
| 회의문자 | 会意文字 (かいいもじ) | 2개 이상의 요소를 결합하여 의미 표현 | 森 (숲) = 木+木+木 (나무+나무+나무) |
| 형성문자 | 形声文字 (けいせいもじ) | 의미 부분 + 소리 부분 | 清 (맑을) = 氵(물) + 青 (파랑/소리) |

> **전체 한자의 약 80%가 형성문자입니다.** 즉, 대부분의 한자는 다음을 가지고 있습니다:
> 1. **의미 범주**를 암시하는 **부수** (部首)
> 2. **발음**을 암시하는 **음부**(音符)

### 1.2 한자의 구성 요소

```
┌─────────────────────────────────────────┐
│              漢字 (한자)                  │
│                                         │
│    ┌──────────┐    ┌──────────────┐     │
│    │  部首     │    │  音符         │     │
│    │ (부수)    │    │ (음부)       │     │
│    │ = 의미    │    │ = 소리 힌트  │     │
│    │  범주     │    │              │     │
│    └──────────┘    └──────────────┘     │
│                                         │
│    예시: 語 (언어/말)                    │
│    言 (말 부수) + 吾 (음부)              │
└─────────────────────────────────────────┘
```

## 2. 부수 (部首): 구성 요소

모든 한자에는 정확히 **하나의 공식 부수**(部首)가 있으며, 이는 사전 분류에 사용됩니다. 다만 한자에는 여러 개의 인식 가능한 구성 요소가 포함될 수 있습니다.

### 2.1 부수의 위치

부수는 한자 내 특정 위치에 나타납니다:

```
┌───────────────────────────────────────────┐
│  위치           명칭          예시         │
├───────────────────────────────────────────┤
│  왼쪽          へん (偏)    氵(水)→ 海    │
│  오른쪽        つくり (旁)  刂(刀)→ 割    │
│  위쪽          かんむり (冠) 宀 → 家      │
│  아래쪽        あし (脚)    灬(火)→ 熱    │
│  감싸는 형태   かまえ (構)  門 → 間       │
│  왼쪽+아래     にょう (繞)  辶 → 道       │
│  왼쪽 위       たれ (垂)    广 → 店       │
└───────────────────────────────────────────┘
```

## 4. 음독 vs. 훈독

각 한자에는 여러 읽기가 있을 수 있습니다. 두 가지 주요 범주는 다음과 같습니다:

### 4.1 음독 (音読み) — 한자음 읽기

- 중국어 발음에서 유래
- 주로 **한자 복합어** (두 개 이상의 한자가 함께)에서 사용
- 사전에서 **가타카나**로 표기

### 4.2 훈독 (訓読み) — 일본 고유어 읽기

- 한자에 대응시킨 원래의 일본어 단어
- 한자가 **단독으로** 사용되거나 **오쿠리가나**(히라가나 어미)가 뒤따를 때 사용
- 사전에서 **히라가나**로 표기

### 4.3 예시

| 한자 | 음독 | 훈독 | 복합어 (음독) | 단독 (훈독) |
|-------|---------|----------|---------------|-----------------|
| 山 | サン | やま | 山脈 (さんみゃく, 산맥) | 山 (やま, 산) |
| 水 | スイ | みず | 水曜 (すいよう, 수요일) | 水 (みず, 물) |
| 人 | ジン・ニン | ひと | 日本人 (にほんじん, 일본인) | 人 (ひと, 사람) |
| 大 | ダイ・タイ | おお(きい) | 大学 (だいがく, 대학) | 大きい (おおきい, 큰) |
| 食 | ショク | た(べる) | 食事 (しょくじ, 식사) | 食べる (たべる, 먹다) |
| 生 | セイ・ショウ | い(きる)・う(まれる)・なま | 学生 (がくせい, 학생) | 生きる (いきる, 살다) |

> **生**은 일반적인 한자 중 가장 많은 읽기를 가지고 있습니다:
> 음독: セイ, ショウ | 훈독: い(きる), う(まれる), お(う), は(える), なま, き
> 이것은 예외적인 경우이며 — 대부분의 한자는 음독 1~2개, 훈독 1~2개입니다.

## 5. 읽기 선택 규칙

### 5.1 일반 지침

```
+----------------------------------------------------------+
|           어떤 읽기를 사용할 것인가?                        |
+----------------------------------------------------------+
|                                                          |
|  복합어 (한자 + 한자):                                    |
|  → 보통 음독                                              |
|  예) 学校 (がっこう), 電話 (でんわ)                        |
|                                                          |
|  단독 (한자만 또는 오쿠리가나 포함):                        |
|  → 보통 훈독                                              |
|  예) 花 (はな), 食べる (たべる)                            |
|                                                          |
|  예외 (암기 필요):                                        |
|  → 일부 복합어는 훈독 사용:                               |
|    手紙 (てがみ, 편지), 場所 (ばしょ, 장소)               |
|  → 일부 단독은 음독 사용:                                 |
|    本 (ほん, 책), 駅 (えき, 역)                           |
+----------------------------------------------------------+
```

### 5.2 특수 읽기 유형

| 유형 | 일본어 | 설명 | 예시 |
|------|----------|-------------|---------|
| 숙자훈 | 熟字訓 | 복합어를 하나의 고유어로 읽음 | 大人 = おとな (だいじん이 아님) |
| 나노리 | 名乗り | 이름에 사용되는 특수 읽기 | 美 = よし (이름에서) |
| 아테지 | 当て字 | 의미가 아닌 소리에 맞춰 한자를 선택 | 寿司 = すし |

### 5.3 일반적인 숙자훈 (특수 복합어 읽기)

| 한자 | 예상 읽기 | 실제 읽기 | 의미 |
|-------|-----------------|---------------|---------|
| 大人 | だいじん | **おとな** | 어른 |
| 今日 | こんにち | **きょう** | 오늘 |
| 明日 | めいにち | **あした** | 내일 |
| 昨日 | さくにち | **きのう** | 어제 |
| 一人 | いちにん | **ひとり** | 한 명 |
| 二人 | にじん | **ふたり** | 두 명 |
| 友達 | ゆうたつ | **ともだち** | 친구 |
| 果物 | かぶつ | **くだもの** | 과일 |
| 眼鏡 | がんきょう | **めがね** | 안경 |
| 部屋 | ぶおく | **へや** | 방 |

## 6. 한자 복합어 (熟語) 패턴

한자가 어떻게 결합되는지 이해하면 사전을 찾지 않고도 새로운 단어의 의미를 예측할 수 있습니다.

### 6.1 구성 패턴

| 패턴 | 구조 | 예시 |
|---------|-----------|---------|
| **유사 의미** | A + B (비슷한 뜻) | 道路 (길+길 = 도로), 森林 (숲+수풀 = 삼림) |
| **반대 의미** | A + B (반대말 결합) | 大小 (크다+작다 = 대소), 売買 (팔다+사다 = 매매) |
| **수식어 + 피수식어** | 형/부 + 명/동 | 新聞 (새+듣다 = 신문), 高速 (높+속도 = 고속) |
| **동사 + 목적어** | 행위 + 대상 | 読書 (읽다+책 = 독서), 登山 (오르다+산 = 등산) |
| **주어 + 동사** | 행위자/사물 + 행위 | 地震 (땅+흔들 = 지진), 日没 (해+지다 = 일몰) |
| **부정** | 不/無/非/未 + 단어 | 不可能 (아니+가능 = 불가능), 無料 (없+요금 = 무료) |

### 6.2 일반적인 접두사

| 접두사 | 의미 | 예시 |
|--------|---------|---------|
| 不 (ふ/ぶ) | 아닌, 불- | 不安 (불안), 不便 (불편), 不足 (부족) |
| 無 (む/ぶ) | 없는, 무- | 無料 (무료), 無理 (무리), 無視 (무시) |
| 非 (ひ) | 아닌, 비- | 非常 (비상), 非公式 (비공식) |
| 未 (み) | 아직 ~않은 | 未来 (미래), 未定 (미정), 未満 (미만) |
| 再 (さい) | 다시, 재- | 再開 (재개), 再生 (재생), 再利用 (재활용) |
| 超 (ちょう) | 초-, 매우 | 超人 (초인), 超高速 (초고속) |
| 最 (さい) | 가장 | 最高 (최고), 最初 (최초), 最新 (최신) |
| 各 (かく) | 각각 | 各国 (각국), 各自 (각자) |

### 6.3 일반적인 접미사

| 접미사 | 의미 | 예시 |
|--------|---------|---------|
| 的 (てき) | ~적 | 国際的 (국제적), 具体的 (구체적) |
| 化 (か) | ~화 | 国際化 (국제화), 近代化 (근대화) |
| 性 (せい) | ~성 | 可能性 (가능성), 重要性 (중요성) |
| 者 (しゃ) | ~하는 사람 | 研究者 (연구자), 記者 (기자) |
| 学 (がく) | ~학 | 科学 (과학), 文学 (문학), 医学 (의학) |
| 力 (りょく) | ~력 | 能力 (능력), 努力 (노력), 体力 (체력) |
| 感 (かん) | ~감 | 安心感 (안심감), 達成感 (달성감) |
| 式 (しき) | ~식, 의식 | 日本式 (일본식), 入学式 (입학식) |

## 8. 학습 전략

### 8.1 부수법 (Radical Method)

**전략:** 부수를 먼저 배우고, 구성 요소로부터 한자를 구축합니다.

```
1단계: 부수 학습           氵= 물
2단계: 한자에서 확인        海 (바다), 泳 (헤엄), 洗 (씻다), 湖 (호수)
3단계: 연결 만들기          "물 관련 한자에는 모두 氵이 있다"
4단계: 새 한자 예측         汚 (더럽다) → 물 + ? → 오염된 물?
                           실제로: 더러운, 오염된 ✓
```

**장점:**
- 2,000자 이상의 한자를 약 200개의 구성 요소로 줄임
- 모르는 한자를 분석 가능하게 만듦
- 읽기를 잊어도 의미를 기억하는 데 도움

### 8.2 간격 반복 (間隔反復)

**전략:** 한자를 점점 늘어나는 간격으로 복습하여 장기 기억으로 옮깁니다.

```
+---------------------------------------------------+
|  간격 반복 스케줄                                    |
+---------------------------------------------------+
|                                                   |
|  1일차:  새 한자 10자 학습                          |
|  2일차:  1일차 복습 + 새 10자 학습                   |
|  4일차:  1~2일차 복습 + 새 10자 학습                 |
|  7일차:  1~4일차 복습 + 새 10자 학습                 |
|  14일차: 1~7일차 복습 + 새 10자 학습                 |
|  30일차: 1~14일차 복습 + 새 10자 학습                |
|                                                   |
|  도구: Anki, WaniKani, Kanji Study 앱              |
+---------------------------------------------------+
```

### 8.3 문맥 학습

**전략:** 한자를 단독이 아닌 어휘를 통해 학습합니다.

| 접근 방식 | 방법 | 예시 |
|----------|--------|---------|
| 나쁨 | 한자만 외우기 | 食 = 먹다 |
| 좋음 | 복합어로 학습 | 食事 (しょくじ, 식사), 食堂 (しょくどう, 식당) |
| 가장 좋음 | 문장 속에서 학습 | 毎日食堂で食事をする。(매일 식당에서 식사를 한다.) |

### 8.4 이야기/연상법

**전략:** 한자 구성 요소와 의미를 연결하는 기억에 남는 이야기를 만듭니다.

| 한자 | 구성 요소 | 연상법 |
|-------|-----------|----------|
| 休 | 亻(사람) + 木 (나무) | **사람**이 **나무**에 기대어 쉬는 모습 = 쉬다 |
| 好 | 女 (여자) + 子 (아이) | **여자**와 **아이** = 좋아하다/사랑하다 |
| 明 | 日 (해) + 月 (달) | **해**와 **달**이 함께 = 밝다 |
| 岩 | 山 (산) + 石 (돌) | **산** 위의 **돌** = 바위 |
| 鳴 | 口 (입) + 鳥 (새) | **새**가 **입**을 벌리는 = 울다/짖다 |

### 8.5 한자 일지법

**전략:** 새 한자를 배울 때마다 다음 정보를 함께 기록합니다:

```
┌──────────────────────────────────────┐
│  한자: 経                            │
│  부수: 糸 (실)                       │
│  음독: ケイ / キョウ                  │
│  훈독: へ(る) / た(つ)               │
│  의미: 지나다, 경험하다              │
│  복합어: 経験 (경험)                  │
│          経済 (경제)                  │
│          経営 (경영)                  │
│  예문: この経験を通じて学んだ。       │
│  (이 경험을 통해 배웠다)             │
│  연상법: 실(糸)이 베틀을 통과하는    │
│  모습 — 지나다                       │
└──────────────────────────────────────┘
```

## 10. 연습 문제

### 연습 1: 부수 식별

각 한자에서 부수를 찾고 의미 범주를 추측하십시오.

1. 海 — 부수: _______ → 범주: _______
2. 話 — 부수: _______ → 범주: _______
3. 持 — 부수: _______ → 범주: _______
4. 銀 — 부수: _______ → 범주: _______
5. 雪 — 부수: _______ → 범주: _______
6. 聞 — 부수: _______ → 범주: _______
7. 飲 — 부수: _______ → 범주: _______
8. 道 — 부수: _______ → 범주: _______

<details>
<summary>정답</summary>

1. **氵** (さんずい) → **물** — 海 = 바다
2. **言/訁** (ごんべん) → **말** — 話 = 이야기
3. **扌** (てへん) → **손** — 持 = 들다
4. **釒** (かねへん) → **쇠** — 銀 = 은
5. **雨** (あめかんむり) → **비/날씨** — 雪 = 눈
6. **耳** (みみ) → **귀** — 聞 = 듣다 (門 안에 耳)
7. **飠** (しょくへん) → **음식** — 飲 = 마시다
8. **辶** (しんにょう) → **이동/길** — 道 = 길

</details>

### 연습 2: 음독인가, 훈독인가?

각 단어에서 한자가 음독을 사용하는지 훈독을 사용하는지 판별하십시오.

1. 山 (やま) — 산
2. 登山 (とざん) — 등산
3. 食べる (たべる) — 먹다
4. 食事 (しょくじ) — 식사
5. 大きい (おおきい) — 큰
6. 大学 (だいがく) — 대학
7. 花 (はな) — 꽃
8. 花瓶 (かびん) — 꽃병

<details>
<summary>정답</summary>

1. **훈독** — 단독 한자: やま
2. **음독** — 한자 복합어: とざん (ト + ザン)
3. **훈독** — 오쿠리가나 포함: た(べる)
4. **음독** — 한자 복합어: ショク + ジ
5. **훈독** — 오쿠리가나 포함: おお(きい)
6. **음독** — 한자 복합어: ダイ + ガク
7. **훈독** — 단독: はな
8. **음독** — 한자 복합어: カ + ビン

</details>

### 연습 3: 복합어 의미 예측

복합어 구성 패턴을 활용하여 각 단어의 의미를 한자 구성 요소로부터 예측하십시오.

1. 火山 (불 + 산) = ?
2. 入口 (들어가다 + 입/입구) = ?
3. 出口 (나가다 + 입/입구) = ?
4. 不安 (아닌 + 편안) = ?
5. 再開 (다시 + 열다) = ?
6. 国際 (나라 + 경계) = ?
7. 人口 (사람 + 입) = ?
8. 年金 (해 + 돈) = ?

<details>
<summary>정답</summary>

1. **火山** (かざん) = **화산** (불 + 산)
2. **入口** (いりぐち) = **입구** (들어가다 + 입구)
3. **出口** (でぐち) = **출구** (나가다 + 입구)
4. **不安** (ふあん) = **불안** (아닌 + 편안)
5. **再開** (さいかい) = **재개** (다시 + 열다)
6. **国際** (こくさい) = **국제** (나라 + 경계)
7. **人口** (じんこう) = **인구** (사람 + 입 — 먹여 살릴 사람들)
8. **年金** (ねんきん) = **연금** (해 + 돈 — 연간 지급금)

</details>

### 연습 4: 접미사와 접두사 연습

올바른 접두사나 접미사를 추가하여 목표 의미를 만드십시오.

1. 可能 + _______ = 가능성 (性/化/的)
2. _______ + 料 = 무료 (不/無/非)
3. 国際 + _______ = 국제적 (형용사) (性/化/的)
4. 研究 + _______ = 연구자 (者/学/力)
5. _______ + 定 = 미정 (不/無/未)
6. 近代 + _______ = 근대화 (性/化/的)

<details>
<summary>정답</summary>

1. 可能**性** (かのうせい) = 가능성 (-성 접미사)
2. **無**料 (むりょう) = 무료 (없+요금)
3. 国際**的** (こくさいてき) = 국제적 (-적 접미사)
4. 研究**者** (けんきゅうしゃ) = 연구자 (~하는 사람 접미사)
5. **未**定 (みてい) = 미정 (아직+결정)
6. 近代**化** (きんだいか) = 근대화 (-화 접미사)

</details>

### 연습 5: 한자 이야기 만들기

각 한자의 구성 요소를 활용하여 연상 이야기를 만드십시오. 정답은 없습니다 — 기억에 남을수록 좋습니다.

1. 鳴 (口 입 + 鳥 새) = 울다, 짖다
2. 畑 (火 불 + 田 논) = 밭
3. 閉 (門 문 + 才 빗장) = 닫다
4. 男 (田 논 + 力 힘) = 남자
5. 安 (宀 지붕 + 女 여자) = 싼, 안전한, 평화로운

<details>
<summary>모범 답안</summary>

1. **鳴**: **새**가 **입**을 열어 노래한다 — 짹짹!
2. **畑**: **불**로 개간한 **논** — 마른 밭 (물 댄 논과 대비)
3. **閉**: **문**에 **빗장**이 걸린 — 닫힌!
4. **男**: **논**에서 **힘**을 쓰는 사람 — 땅을 경작하는 남자
5. **安**: **지붕** 아래의 **여자** — 집에서 안전하고 평화롭게

(이것은 개인적인 연상법입니다 — 자신에게 기억이 잘 되는 것을 만드십시오!)

</details>
