## 1. 숫자 1-10

### 1.1 두 가지 숫자 체계

일본어에는 고유 일본어 체계 (和語 / wago)와 중국어에서 유래한 체계 (漢語 / kango) 두 가지 세는 방법이 있습니다. 1-10까지는 둘 다 알아야 합니다:

| 숫자 | 한자 | 한어 (중국식) | 와어 (일본식) |
|------|------|--------------|--------------|
| 1 | 一 | いち (ichi) | ひとつ (hitotsu) |
| 2 | 二 | に (ni) | ふたつ (futatsu) |
| 3 | 三 | さん (san) | みっつ (mittsu) |
| 4 | 四 | し / よん (shi / yon) | よっつ (yottsu) |
| 5 | 五 | ご (go) | いつつ (itsutsu) |
| 6 | 六 | ろく (roku) | むっつ (muttsu) |
| 7 | 七 | しち / なな (shichi / nana) | ななつ (nanatsu) |
| 8 | 八 | はち (hachi) | やっつ (yattsu) |
| 9 | 九 | きゅう / く (kyuu / ku) | ここのつ (kokonotsu) |
| 10 | 十 | じゅう (juu) | とお (too) |

```
┌──────────────────────────────────────────────────────────────────┐
│              어떤 체계를 사용할까                                   │
│                                                                  │
│   한어 (いち、に、さん...):                                        │
│     • 특정 조수사와 함께 (一本、二人、三台)                          │
│     • 전화번호, 주소, 수학                                         │
│     • 10 이상의 숫자                                               │
│                                                                  │
│   와어 (ひとつ、ふたつ、みっつ...):                                  │
│     • 특정 조수사 없이 사물을 셀 때                                 │
│     • 어린 아이의 나이 (비격식)                                     │
│     • 10까지만 사용 가능                                           │
│                                                                  │
│   4, 7, 9에 대한 참고:                                             │
│     4 = し (shi)는 死 (죽음)와 발음이 같음 → よん (yon) 선호        │
│     7 = しち (shichi) → なな (nana)가 자주 선호됨                  │
│     9 = く (ku)는 苦 (고통)와 발음이 같음 → きゅう 선호             │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 영(0)

| 읽기 | 로마자 | 용도 |
|------|--------|------|
| ゼロ | zero | 일반, 현대적 (영어에서 유래) |
| れい | rei | 격식, 전화번호 |
| まる | maru | "동그라미" — 숫자를 하나씩 읽을 때 사용 |

## 3. 숫자 100-10,000

### 3.1 백 단위 (100-900)

300, 600, 800에서의 음 변화에 주의하세요:

| 숫자 | 일본어 | 로마자 | 참고 |
|------|--------|--------|------|
| 100 | ひゃく | hyaku | |
| 200 | にひゃく | ni-hyaku | |
| 300 | **さんびゃく** | san-**b**yaku | 음 변화! |
| 400 | よんひゃく | yon-hyaku | |
| 500 | ごひゃく | go-hyaku | |
| 600 | **ろっぴゃく** | ro**pp**yaku | 음 변화! |
| 700 | ななひゃく | nana-hyaku | |
| 800 | **はっぴゃく** | ha**pp**yaku | 음 변화! |
| 900 | きゅうひゃく | kyuu-hyaku | |

### 3.2 천 단위 (1,000-9,000)

3,000과 8,000에서의 음 변화에 주의하세요:

| 숫자 | 일본어 | 로마자 | 참고 |
|------|--------|--------|------|
| 1,000 | せん | sen | |
| 2,000 | にせん | ni-sen | |
| 3,000 | **さんぜん** | san-**z**en | 음 변화! |
| 4,000 | よんせん | yon-sen | |
| 5,000 | ごせん | go-sen | |
| 6,000 | ろくせん | roku-sen | |
| 7,000 | ななせん | nana-sen | |
| 8,000 | **はっせん** | ha**ss**en | 음 변화! |
| 9,000 | きゅうせん | kyuu-sen | |

### 3.3 만 단위

| 숫자 | 일본어 | 로마자 |
|------|--------|--------|
| 10,000 | いちまん | ichi-man |

```
┌──────────────────────────────────────────────────────────────────┐
│              일본어 큰 숫자 단위                                   │
│                                                                  │
│   영어는 3자리씩 묶음: thousand, million, billion                  │
│   일본어는 4자리씩 묶음: 万 (man), 億 (oku), 兆 (chou)            │
│                                                                  │
│   1,0000 → いちまん (ichiman)         = 10,000                   │
│   10,0000 → じゅうまん (juuman)        = 100,000                  │
│   100,0000 → ひゃくまん (hyakuman)     = 1,000,000               │
│   1000,0000 → いっせんまん (issenman)  = 10,000,000              │
│   1,0000,0000 → いちおく (ichioku)     = 100,000,000             │
│                                                                  │
│   이 4자리 묶음 체계는 영어 화자에게                                │
│   일본어 숫자에서 가장 까다로운 부분 중 하나입니다.                   │
└──────────────────────────────────────────────────────────────────┘
```

### 3.4 복합 예시

| 숫자 | 일본어 | 로마자 |
|------|--------|--------|
| 150 | ひゃくごじゅう | hyaku-gojuu |
| 256 | にひゃくごじゅうろく | nihyaku-gojuu-roku |
| 1,234 | せんにひゃくさんじゅうよん | sen-nihyaku-sanjuu-yon |
| 3,500 | さんぜんごひゃく | sanzen-gohyaku |
| 10,000 | いちまん | ichiman |

## 4. 일본어 조수사 체계

### 4.1 조수사란?

```
┌──────────────────────────────────────────────────────────────────┐
│              일본어 조수사 체계                                     │
│                                                                  │
│   한국어: "책 3권", "고양이 2마리", "커피 1잔"                      │
│   일본어: 숫자와 명사 사이에 반드시 조수사를                         │
│           넣어야 합니다.                                           │
│                                                                  │
│   한국어:  "책 3권"                                                │
│   일본어:  本を 三冊 ください。                                      │
│            Hon o san-satsu kudasai.                               │
│            "책 3권 주세요."                                        │
│            (三 = 3, 冊 = 책을 세는 조수사)                          │
│                                                                  │
│   일본어에는 100개 이상의 조수사가 있지만,                           │
│   일상 대화에서는 약 10-15개만 있으면 됩니다.                        │
│                                                                  │
│   적절한 조수사를 모르면 범용 일본어 조수사를 사용하세요:              │
│   ひとつ、ふたつ、みっつ...                                         │
└──────────────────────────────────────────────────────────────────┘
```

### 4.2 조수사 어순

조수사는 보통 명사 바로 뒤에 (또는 조사 뒤에) 옵니다:

```
패턴 1: [명사]を [숫자+조수사] [동사]
  りんごを 三つ ください。
  Ringo o mittsu kudasai.
  "사과 3개 주세요."

패턴 2: [숫자+조수사]の [명사]
  三冊の本
  San-satsu no hon
  "3권의 책"
```

## 5. 주요 조수사

### 5.1 つ (tsu) — 범용 조수사

특정 조수사를 모를 때 사용하는 와어 조수사입니다. 10까지만 사용 가능합니다:

| 개수 | 일본어 | 로마자 |
|------|--------|--------|
| 1 | ひとつ | hitotsu |
| 2 | ふたつ | futatsu |
| 3 | みっつ | mittsu |
| 4 | よっつ | yottsu |
| 5 | いつつ | itsutsu |
| 6 | むっつ | muttsu |
| 7 | ななつ | nanatsu |
| 8 | やっつ | yattsu |
| 9 | ここのつ | kokonotsu |
| 10 | とお | too |
| 몇 개? | いくつ | ikutsu |

### 5.2 人 (nin/ri) — 사람

| 개수 | 일본어 | 로마자 | 참고 |
|------|--------|--------|------|
| 1명 | **ひとり** | hito**ri** | 불규칙! |
| 2명 | **ふたり** | futa**ri** | 불규칙! |
| 3명 | さんにん | san-nin | 여기부터 규칙적 |
| 4명 | **よにん** | yo-nin | よん이 아닌 よ |
| 5명 | ごにん | go-nin | |
| 6명 | ろくにん | roku-nin | |
| 7명 | ななにん / しちにん | nana-nin / shichi-nin | |
| 8명 | はちにん | hachi-nin | |
| 9명 | きゅうにん / くにん | kyuu-nin / ku-nin | |
| 10명 | じゅうにん | juu-nin | |
| 몇 명? | なんにん | nan-nin | |

> **중요**: 1명 = ひとり, 2명 = ふたり. いちにん이나 ににん이 아닙니다!

## 6. 시간 말하기

### 6.1 시: 〜時 (ji)

| 시 | 일본어 | 로마자 | 참고 |
|----|--------|--------|------|
| 1:00 | いちじ | ichi-ji | |
| 2:00 | にじ | ni-ji | |
| 3:00 | さんじ | san-ji | |
| 4:00 | **よじ** | **yo**-ji | よんじ가 아님! |
| 5:00 | ごじ | go-ji | |
| 6:00 | ろくじ | roku-ji | |
| 7:00 | **しちじ** | **shichi**-ji | ななじ가 아님! |
| 8:00 | はちじ | hachi-ji | |
| 9:00 | **くじ** | **ku**-ji | きゅうじ가 아님! |
| 10:00 | じゅうじ | juu-ji | |
| 11:00 | じゅういちじ | juuichi-ji | |
| 12:00 | じゅうにじ | juuni-ji | |
| 몇 시? | なんじ | nan-ji | |

> **주의**: 4:00 = よじ, 7:00 = しちじ, 9:00 = くじ. 이들은 불규칙입니다!

| 4분 | よん**ぷん** | yon-**p**un | 음 변화! |
| 5분 | ごふん | go-fun | |
| 6분 | ろ**っぷん** | ro**ppun** | 음 변화! |
| 7분 | ななふん | nana-fun | |
| 8분 | は**っぷん** | ha**ppun** | 음 변화! (또는 はちふん) |
| 9분 | きゅうふん | kyuu-fun | |
| 10분 | じゅ**っぷん** | ju**ppun** | 음 변화! |
| 15분 | じゅうごふん | juugo-fun | |
| 30분 | さんじゅ**っぷん** | sanjuppun | 또는 はん (han) = 반 |
| 45분 | よんじゅうごふん | yonjuugo-fun | |
| 몇 분? | なん**ぷん** | nan-**p**un | |

### 6.3 조합하기

```
[시]時 [분]分 です。
[Hour]-ji [Minute]-fun desu.
```

| 시간 | 일본어 | 로마자 |
|------|--------|--------|
| 3:15 | さんじ じゅうごふん | san-ji juugo-fun |
| 7:30 | しちじ さんじゅっぷん / しちじはん | shichi-ji sanjuppun / shichi-ji han |
| 9:45 | くじ よんじゅうごふん | ku-ji yonjuugo-fun |
| 12:00 | じゅうにじ | juuni-ji |

## 7. 요일

### 7.1 일곱 요일

각 요일은 천체나 원소의 이름 뒤에 曜日 (youbi)가 붙습니다:

| 요일 | 한자 | 히라가나 | 로마자 | 원소 |
|------|------|----------|--------|------|
| 월요일 | 月曜日 | げつようび | getsu-youbi | 달 (月) |
| 화요일 | 火曜日 | かようび | ka-youbi | 불 (火) |
| 수요일 | 水曜日 | すいようび | sui-youbi | 물 (水) |
| 목요일 | 木曜日 | もくようび | moku-youbi | 나무 (木) |
| 금요일 | 金曜日 | きんようび | kin-youbi | 금/금속 (金) |
| 토요일 | 土曜日 | どようび | do-youbi | 흙 (土) |
| 일요일 | 日曜日 | にちようび | nichi-youbi | 해 (日) |
| 무슨 요일? | 何曜日 | なんようび | nan-youbi | |

```
기억법: 원소는 여러 언어에서 행성과 같은 패턴을 따릅니다:
  月 달 → 월요일     (프랑스어: lundi)
  火 화성 → 화요일    (프랑스어: mardi)
  水 수성 → 수요일    (프랑스어: mercredi)
  木 목성 → 목요일    (프랑스어: jeudi)
  金 금성 → 금요일    (프랑스어: vendredi)
  土 토성 → 토요일    (영어: Saturday)
  日 태양 → 일요일    (영어: Sunday)
```

### 7.2 요일 관련 표현

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| 今日 (きょう) | kyou | 오늘 |
| 明日 (あした) | ashita | 내일 |
| 昨日 (きのう) | kinou | 어제 |
| 毎日 (まいにち) | mainichi | 매일 |
| 今週 (こんしゅう) | konshuu | 이번 주 |
| 来週 (らいしゅう) | raishuu | 다음 주 |
| 先週 (せんしゅう) | senshuu | 지난주 |
| 週末 (しゅうまつ) | shuumatsu | 주말 |

## 8. 월

### 8.1 12개월

월은 단순히 숫자 + 月 (gatsu)입니다:

| 월 | 한자 | 히라가나 | 로마자 |
|----|------|----------|--------|
| 1월 | 一月 | いちがつ | ichi-gatsu |
| 2월 | 二月 | にがつ | ni-gatsu |
| 3월 | 三月 | さんがつ | san-gatsu |
| 4월 | **四月** | **しがつ** | **shi**-gatsu |
| 5월 | 五月 | ごがつ | go-gatsu |
| 6월 | 六月 | ろくがつ | roku-gatsu |
| 7월 | **七月** | **しちがつ** | **shichi**-gatsu |
| 8월 | 八月 | はちがつ | hachi-gatsu |
| 9월 | **九月** | **くがつ** | **ku**-gatsu |
| 10월 | 十月 | じゅうがつ | juu-gatsu |
| 11월 | 十一月 | じゅういちがつ | juuichi-gatsu |
| 12월 | 十二月 | じゅうにがつ | juuni-gatsu |
| 몇 월? | 何月 | なんがつ | nan-gatsu |

> **주의**: 4월 = **し**がつ (よんがつ가 아님), 7월 = **しち**がつ (なながつ가 아님), 9월 = **く**がつ (きゅうがつ가 아님).

## 9. 날짜

### 9.1 매월 날짜

1일~10일은 특별한 일본 고유 읽기를 사용합니다. 11일 이후는 규칙적입니다:

| 일 | 일본어 | 로마자 | 참고 |
|----|--------|--------|------|
| 1일 | **ついたち** | tsuitachi | 매우 불규칙! |
| 2일 | **ふつか** | futsuka | 불규칙 |
| 3일 | **みっか** | mikka | 불규칙 |
| 4일 | **よっか** | yokka | 불규칙 |
| 5일 | **いつか** | itsuka | 불규칙 |
| 6일 | **むいか** | muika | 불규칙 |
| 7일 | **なのか** | nanoka | 불규칙 |
| 8일 | **ようか** | youka | 불규칙 |
| 9일 | **ここのか** | kokonoka | 불규칙 |
| 10일 | **とおか** | tooka | 불규칙 |
| 11일 | じゅういちにち | juuichi-nichi | 여기부터 규칙적 |
| 14일 | **じゅうよっか** | juu-yokka | 불규칙 |
| 20일 | **はつか** | hatsuka | 불규칙! |
| 24일 | **にじゅうよっか** | nijuu-yokka | 불규칙 |
| 기타 | [숫자]にち | -nichi | 규칙적 |
| 몇 일? | なんにち | nan-nichi | |

> **반드시 암기**: 1일~10일은 모두 불규칙입니다. 14일, 20일, 24일도 불규칙입니다. 나머지는 [숫자]にち 패턴을 따릅니다.

### 9.2 완전한 날짜 표현

```
[연도]年 [월]月 [일]日

Nisen-nijuugo-nen san-gatsu juugo-nichi
"2025년 3월 15일"
```

## 10. 연도와 기간

### 10.1 연도

연도는 조수사 年 (nen)을 사용합니다:

| 연도 | 일본어 | 로마자 |
|------|--------|--------|
| 2020 | にせんにじゅうねん | nisen-nijuu-nen |
| 2025 | にせんにじゅうごねん | nisen-nijuugo-nen |
| 2026 | にせんにじゅうろくねん | nisen-nijuuroku-nen |

### 10.2 기간 조수사

| 기간 | 조수사 | 예시 | 로마자 |
|------|--------|------|--------|
| ___분간 | 〜分間 (funkan) | 三十分間 | sanjuppunkan (30분간) |
| ___시간 | 〜時間 (jikan) | 二時間 | ni-jikan (2시간) |
| ___일간 | 〜日間 (nichikan) | 三日間 | mikka-kan (3일간) |
| ___주간 | 〜週間 (shuukan) | 一週間 | isshuukan (1주간) |
| ___개월 | 〜か月 (kagetsu) | 六か月 | rokkagetsu (6개월) |
| ___년간 | 〜年間 (nenkan) | 一年間 | ichinen-kan (1년간) |

### 10.3 기간에 대한 질문

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| どのくらいかかりますか。 | Dono kurai kakarimasu ka. | 얼마나 걸립니까? |
| 三十分かかります。 | Sanjuppun kakarimasu. | 30분 걸립니다. |
| 何時間勉強しますか。 | Nanjikan benkyou shimasu ka. | 몇 시간 공부합니까? |
| 二時間勉強します。 | Ni-jikan benkyou shimasu. | 2시간 공부합니다. |

## 11. 연습 문제

### 연습 1: 숫자 읽기

다음 숫자를 로마자로 쓰세요:

```
1. 十五
2. 三十八
3. 百二十
4. 三百
5. 千五百
6. 四千二百
7. 八百
8. 三千
9. 一万
10. 六千七百
```

<details>
<summary>정답 보기</summary>

```
1. 十五 = じゅうご (juugo) = 15
2. 三十八 = さんじゅうはち (sanjuu-hachi) = 38
3. 百二十 = ひゃくにじゅう (hyaku-nijuu) = 120
4. 三百 = さんびゃく (sanbyaku) = 300 — 음 변화!
5. 千五百 = せんごひゃく (sen-gohyaku) = 1,500
6. 四千二百 = よんせんにひゃく (yonsen-nihyaku) = 4,200
7. 八百 = はっぴゃく (happyaku) = 800 — 음 변화!
8. 三千 = さんぜん (sanzen) = 3,000 — 음 변화!
9. 一万 = いちまん (ichiman) = 10,000
10. 六千七百 = ろくせんななひゃく (rokusen-nanahyaku) = 6,700
```

</details>

### 연습 2: 올바른 조수사 선택

각 항목에 적절한 조수사를 선택하세요:

```
조수사: つ、人、本、枚、匹、台、冊、杯

1. 학생 2명: 学生 二___
2. 연필 3자루: えんぴつ 三___
3. 종이 5장: かみ 五___
4. 고양이 1마리: ねこ 一___
5. 커피 2잔: コーヒー 二___
6. 책 4권: 本 四___
7. 자동차 3대: 車 三___
8. 사과 6개 (범용): りんご 六___
```

<details>
<summary>정답 보기</summary>

```
1. 学生 二人 (ふたり, futari) — 사람 조수사
2. えんぴつ 三本 (さんぼん, sanbon) — 길고 가는 물건
3. かみ 五枚 (ごまい, gomai) — 납작하고 얇은 물건
4. ねこ 一匹 (いっぴき, ippiki) — 작은 동물
5. コーヒー 二杯 (にはい, nihai) — 컵/잔
6. 本 四冊 (よんさつ, yonsatsu) — 책/제본물
7. 車 三台 (さんだい, sandai) — 기계/탈것
8. りんご 六つ (むっつ, muttsu) — 범용 조수사
```

</details>

### 연습 3: 시간 말하기

다음 시간을 일본어로 쓰세요:

```
1. 3:00
2. 7:30
3. 9:15
4. 12:45
5. 오전 4:00
6. 오후 8:00
```

<details>
<summary>정답 보기</summary>

```
1. さんじ (san-ji)
2. しちじはん (shichi-ji han) 또는 しちじさんじゅっぷん (shichi-ji sanjuppun)
3. くじじゅうごふん (ku-ji juugo-fun)
4. じゅうにじよんじゅうごふん (juuni-ji yonjuugo-fun)
5. ごぜんよじ (gozen yo-ji)
6. ごごはちじ (gogo hachi-ji)
```

</details>

### 연습 4: 요일과 날짜

일본어로 답하세요:

```
1. 일본어 수업은 무슨 요일입니까? (월요일)
2. 지금 몇 월입니까? (3월)
3. 오늘 날짜는? (3월 6일)
4. 생일이 언제입니까? (자신의 것을 사용하거나 만들어 보세요)
5. 20일은 어떻게 읽습니까?
```

<details>
<summary>정답 보기</summary>

```
1. 月曜日です。 (Getsuyoubi desu.)
2. 三月です。 (San-gatsu desu.)
3. 三月むいかです。 (San-gatsu muika desu.) — 6일은 불규칙 읽기!
4. (예시) 私の誕生日は五月ついたちです。
   (Watashi no tanjoubi wa go-gatsu tsuitachi desu.)
   "제 생일은 5월 1일입니다."
5. はつか (hatsuka) — 20일의 불규칙 읽기!
```

</details>

### 연습 5: 일본어로 번역하기

```
1. 고양이가 3마리 있습니다.
2. 커피 2잔 주세요.
3. 매일 2시간 공부합니다.
4. 6시 반에 일어납니다.
5. 월요일에 학교에 갑니다.
6. 학생이 5명 있습니다.
7. 이것은 얼마입니까? — 3,500엔입니다.
8. 지금 몇 시입니까? — 오후 4시 15분입니다.
```

<details>
<summary>정답 보기</summary>

```
1. ねこが三匹います。 (Neko ga sanbiki imasu.)
2. コーヒーを二杯ください。 (Koohii o nihai kudasai.)
3. 毎日二時間勉強します。 (Mainichi ni-jikan benkyou shimasu.)
4. 六時半に起きます。 (Roku-ji han ni okimasu.)
5. 月曜日に学校に行きます。 (Getsuyoubi ni gakkou ni ikimasu.)
6. 学生が五人います。 (Gakusei ga gonin imasu.)
7. これはいくらですか。— 三千五百円です。
   (Kore wa ikura desu ka. — Sanzen-gohyaku-en desu.)
8. 今何時ですか。— 午後四時十五分です。
   (Ima nanji desu ka. — Gogo yo-ji juugo-fun desu.)
```

</details>

### 연습 6: 음 변화 퀴즈

다음 조수사 조합의 올바른 읽기를 쓰세요:

```
1. 一 + 本 = ?
2. 三 + 百 = ?
3. 六 + 百 = ?
4. 八 + 千 = ?
5. 一 + 匹 = ?
6. 三 + 杯 = ?
7. 一 + 分 = ?
8. 十 + 本 = ?
```

<details>
<summary>정답 보기</summary>

```
1. 一本 = いっぽん (ippon) — いちほん이 아님
2. 三百 = さんびゃく (sanbyaku) — さんひゃく이 아님
3. 六百 = ろっぴゃく (roppyaku) — ろくひゃく이 아님
4. 八千 = はっせん (hassen) — はちせん이 아님
5. 一匹 = いっぴき (ippiki) — いちひき가 아님
6. 三杯 = さんばい (sanbai) — さんはい가 아님
7. 一分 = いっぷん (ippun) — いちふん이 아님
8. 十本 = じゅっぽん (juppon) — じゅうほん이 아님
```

</details>
