## 1. 일본어 문장 구조: SOV

### 1.1 영어 vs 일본어 어순

```
┌──────────────────────────────────────────────────────────────────┐
│              영어 (SVO) vs 일본어 (SOV)                            │
│                                                                  │
│   영어:    I     eat    sushi.                                   │
│            S     V      O                                        │
│                                                                  │
│   일본어:  私は   すしを  食べます。                                 │
│            S(は)  O(を)   V                                       │
│            Watashi wa  sushi wo  tabemasu.                        │
│                                                                  │
│   핵심 규칙: 동사는 항상 문장 끝에 옵니다.                          │
│   동사만 마지막에 오면, 나머지 요소의 순서는                         │
│   어느 정도 유연합니다.                                             │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 핵심 규칙

| 규칙 | 설명 | 예시 |
|------|------|------|
| 동사 후치 | 동사는 항상 문장 끝에 | 食べます (tabemasu)가 마지막에 옴 |
| 조사가 역할 표시 | は, が, を, に, で가 각 단어의 역할을 알려줌 | 私**は** すし**を** 食べます |
| 주어 생략 | 명백한 주어는 종종 생략됨 | すしを食べます。(나는 초밥을 먹습니다 — "나는"이 생략됨) |
| 중간 순서 유연 | 동사 이외의 요소는 순서 변경 가능 | すしを私は食べます도 문법적으로 유효 |

### 1.3 기본 문장 패턴

```
패턴 1:  [주제]は [명사]です。        — A는 B입니다
패턴 2:  [주제]は [목적어]を [동사]。  — A가 B에 [동사]를 합니다
패턴 3:  [주제]は [장소]に います。    — A가 [장소]에 있습니다
패턴 4:  [주제]は [장소]で [동사]。   — A가 [장소]에서 [동사]를 합니다
```

## 2. AはBです 패턴

### 2.1 가장 기본적인 일본어 문장

일본어에서 가장 단순하고 가장 흔한 패턴입니다:

```
A は B です。
A wa B desu.
"A는 B입니다."
```

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| 私は学生です。 | Watashi wa gakusei desu. | 저는 학생입니다. |
| これは本です。 | Kore wa hon desu. | 이것은 책입니다. |
| 田中さんは先生です。 | Tanaka-san wa sensei desu. | 다나카 씨는 선생님입니다. |
| 今日は月曜日です。 | Kyou wa getsuyoubi desu. | 오늘은 월요일입니다. |

### 2.2 부정형: AはBではありません

```
A は B ではありません。
A wa B dewa arimasen.
"A는 B가 아닙니다."
```

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| 私は学生ではありません。 | Watashi wa gakusei dewa arimasen. | 저는 학생이 아닙니다. |
| これは本ではありません。 | Kore wa hon dewa arimasen. | 이것은 책이 아닙니다. |
| 田中さんは先生じゃありません。 | Tanaka-san wa sensei ja arimasen. | 다나카 씨는 선생님이 아닙니다. |

> **구어 줄임말**: ではありません → じゃありません (ja arimasen). 둘 다 정중하지만, じゃ가 구어에서 더 일반적입니다.

### 2.3 의문형: AはBですか

```
A は B ですか。
A wa B desu ka.
"A는 B입니까?"
```

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| あなたは学生ですか。 | Anata wa gakusei desu ka. | 당신은 학생입니까? |
| これは本ですか。 | Kore wa hon desu ka. | 이것은 책입니까? |
| 田中さんは先生ですか。 | Tanaka-san wa sensei desu ka. | 다나카 씨는 선생님입니까? |

> **참고**: 일본어에서는 의문문을 만들 때 어순을 바꾸지 않습니다. 문장 끝에 か를 붙이기만 하면 됩니다. 엄밀히 물음표는 필요하지 않지만 (か 자체가 의문을 나타냄), 비격식 글쓰기에서는 물음표를 많이 사용합니다.

### 2.4 요약 표: AはBです 활용

| 형태 | 구조 | 예시 |
|------|------|------|
| 긍정 | AはBです | 私は学生です。 |
| 부정 | AはBではありません | 私は学生ではありません。 |
| 의문 | AはBですか | あなたは学生ですか。 |
| 부정 의문 | AはBではありませんか | あなたは学生ではありませんか。 |
| 과거 긍정 | AはBでした | 私は学生でした。 |
| 과거 부정 | AはBではありませんでした | 私は学生ではありませんでした。 |

## 3. 조사 は (wa): 주제 표시

### 3.1 は의 기능

```
┌──────────────────────────────────────────────────────────────────┐
│              は — 주제 표시 조사 (topic marker)                     │
│                                                                  │
│   표기: は (히라가나 "ha")                                        │
│   읽기: wa (조사로 사용될 때)                                      │
│   기능: 문장의 주제를 표시                                         │
│          "~에 대해서 말하면..."                                     │
│                                                                  │
│   私は田中です。                                                   │
│   Watashi wa Tanaka desu.                                        │
│   "나에 대해 말하면, 다나카입니다." → "저는 다나카입니다."            │
│                                                                  │
│   東京は大きいです。                                                │
│   Toukyou wa ookii desu.                                         │
│   "도쿄에 대해 말하면, 큽니다." → "도쿄는 큽니다."                 │
└──────────────────────────────────────────────────────────────────┘
```

### 3.2 주제 vs 주어

주제 (は)는 문장이 **무엇에 대한 것**인지를 나타냅니다. 항상 문법적 주어와 같지는 않습니다:

| 일본어 | 직역 | 자연스러운 한국어 |
|--------|------|-----------------|
| 私はさかなです。 | "나에 대해 말하면, 생선입니다." | "저는 생선으로 할게요." (레스토랑에서 주문할 때) |
| 象は鼻が長いです。 | "코끼리에 대해 말하면, 코가 깁니다." | "코끼리는 코가 깁니다." |

> **핵심 포인트**: は는 장면을 설정합니다 — "지금 X에 대해 이야기합니다." 술어의 실제 주어는 다를 수 있습니다.

### 3.3 は의 대조 용법

は는 **대조**를 나타내는 데도 사용됩니다:

| 일본어 | 로마자 | 의미 |
|--------|--------|------|
| 肉は食べますが、魚は食べません。 | Niku wa tabemasu ga, sakana wa tabemasen. | 고기는 먹지만, 생선은 먹지 않습니다. |
| 日本語は話しますが、中国語は話しません。 | Nihongo wa hanashimasu ga, chuugokugo wa hanashimasen. | 일본어는 하지만, 중국어는 하지 않습니다. |

## 4. 조사 が (ga): 주어 표시

### 4.1 が의 기능

```
┌──────────────────────────────────────────────────────────────────┐
│              が — 주어 표시 조사 (subject marker)                   │
│                                                                  │
│   기능: 문법적 주어를 표시                                         │
│   주요 용도:                                                      │
│     1. "누가/무엇이" 질문에 대한 답변                               │
│     2. 새로운 정보 도입                                            │
│     3. 특정 동사/형용사와 함께 사용                                  │
│     4. 관계절에서 주어 표시                                         │
└──────────────────────────────────────────────────────────────────┘
```

### 4.2 は vs が: 핵심 차이

| 조사 | 기능 | 예시 | 맥락 |
|------|------|------|------|
| は (wa) | 주제 — 이미 알고 있는 정보 | 田中さん**は**学生です。 | "다나카 씨는 (알다시피) 학생입니다." |
| が (ga) | 주어 — 새로운 정보 | 田中さん**が**学生です。 | "다나카 씨가 학생입니다." ("누가 학생이야?"에 대한 답) |

**이렇게 생각하세요**:
- は는 이야기의 주제를 표시 (기존/알려진 정보)
- が는 강조하는 대상을 표시 (새로운/중요한 정보)

### 4.3 존재 및 능력에서의 が

특정 동사와 형용사는 が를 필요로 합니다:

| 패턴 | 일본어 | 로마자 | 한국어 |
|------|--------|--------|--------|
| 존재 | ねこ**が**います。 | Neko ga imasu. | 고양이가 있습니다. |
| 좋아함 | すし**が**好きです。 | Sushi ga suki desu. | 초밥을 좋아합니다. |
| 능력 | 日本語**が**わかります。 | Nihongo ga wakarimasu. | 일본어를 알아듣습니다. |
| 원함 | 水**が**ほしいです。 | Mizu ga hoshii desu. | 물을 원합니다. |
| 잘함 | テニス**が**上手です。 | Tenisu ga jouzu desu. | 테니스를 잘합니다. |

### 4.4 의문사 + が

의문사 (누가, 무엇이)가 주어일 때는 항상 が를 사용합니다:

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| だれ**が**来ましたか。 | Dare ga kimashita ka. | 누가 왔습니까? |
| 何**が**ありますか。 | Nani ga arimasu ka. | 무엇이 있습니까? |
| どれ**が**いいですか。 | Dore ga ii desu ka. | 어느 것이 좋습니까? |

## 5. 조사 を (wo): 목적어 표시

### 5.1 を의 기능

```
┌──────────────────────────────────────────────────────────────────┐
│              を — 목적어 표시 조사 (object marker)                   │
│                                                                  │
│   표기: を (히라가나 "wo")                                        │
│   읽기: o (현대 일본어에서 "w"는 묵음)                              │
│   기능: 동사의 직접 목적어를 표시                                   │
│          (행위의 대상)                                             │
│                                                                  │
│   すし を 食べます。                                               │
│   Sushi o tabemasu.                                              │
│   "초밥을 먹습니다." (초밥 = 먹히는 대상)                           │
└──────────────────────────────────────────────────────────────────┘
```

### 5.2 예시

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| 水**を**飲みます。 | Mizu o nomimasu. | 물을 마십니다. |
| 本**を**読みます。 | Hon o yomimasu. | 책을 읽습니다. |
| テレビ**を**見ます。 | Terebi o mimasu. | 텔레비전을 봅니다. |
| 日本語**を**勉強します。 | Nihongo o benkyou shimasu. | 일본어를 공부합니다. |
| 手紙**を**書きます。 | Tegami o kakimasu. | 편지를 씁니다. |
| 音楽**を**聞きます。 | Ongaku o kikimasu. | 음악을 듣습니다. |

### 5.3 이동 동사와 함께 쓰는 を

を는 이동이 통과하는 공간을 표시하는 데도 사용됩니다:

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| 公園**を**散歩します。 | Kouen o sanpo shimasu. | 공원을 산책합니다. |
| 道**を**渡ります。 | Michi o watarimasu. | 길을 건넙니다. |
| 橋**を**渡ります。 | Hashi o watarimasu. | 다리를 건넙니다. |

## 6. 조사 に (ni): 방향, 시간, 장소

### 6.1 に의 여러 역할

に는 가장 다양하게 사용되는 조사 중 하나입니다. N5 수준에서는 다음 세 가지 용도에 집중하세요:

```
┌──────────────────────────────────────────────────────────────────┐
│              に — 방향 / 시간 / 장소                                │
│                                                                  │
│   1. 방향/목표: "~에/~으로" (장소로)                               │
│      学校に行きます。 (Gakkou ni ikimasu.)                         │
│      "학교에 갑니다."                                              │
│                                                                  │
│   2. 시간: "~에" (특정 시간에)                                     │
│      七時に起きます。 (Shichiji ni okimasu.)                       │
│      "7시에 일어납니다."                                           │
│                                                                  │
│   3. 존재 장소: "~에" (いる/ある와 함께)                            │
│      部屋にいます。 (Heya ni imasu.)                               │
│      "방에 있습니다."                                              │
└──────────────────────────────────────────────────────────────────┘
```

### 6.2 방향의 に ("~에/~으로")

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| 日本**に**行きます。 | Nihon ni ikimasu. | 일본에 갑니다. |
| うち**に**帰ります。 | Uchi ni kaerimasu. | 집에 돌아갑니다. |
| 友達**に**会います。 | Tomodachi ni aimasu. | 친구를 만납니다. |

### 6.3 시간의 に ("~에")

に는 특정 시간 (시각, 요일, 날짜)에 사용되지만, 상대적 시간에는 사용하지 않습니다:

| に 사용 | に 미사용 |
|---------|----------|
| 七時**に** (~7시에) | 今日 (오늘) — に 없음 |
| 月曜日**に** (~월요일에) | 明日 (내일) — に 없음 |
| 三月**に** (~3월에) | 毎日 (매일) — に 없음 |
| 二〇二五年**に** (~2025년에) | 今 (지금) — に 없음 |

### 6.4 존재 장소의 に

いる (생물)와 ある (무생물)과 함께 사용됩니다:

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| 猫が庭**に**います。 | Neko ga niwa ni imasu. | 고양이가 정원에 있습니다. |
| 本が机の上**に**あります。 | Hon ga tsukue no ue ni arimasu. | 책이 책상 위에 있습니다. |

## 7. 조사 で (de): 행위 장소, 수단

### 7.1 で의 두 가지 주요 용도

```
┌──────────────────────────────────────────────────────────────────┐
│              で — 행위 장소 / 수단                                  │
│                                                                  │
│   1. 행위 장소: "~에서" (행동이 일어나는 장소)                      │
│      図書館で勉強します。 (Toshokan de benkyou shimasu.)           │
│      "도서관에서 공부합니다."                                      │
│                                                                  │
│   2. 수단/방법: "~로/~으로"                                       │
│      バスで行きます。 (Basu de ikimasu.)                           │
│      "버스로 갑니다."                                              │
└──────────────────────────────────────────────────────────────────┘
```

### 7.2 행위 장소의 で

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| レストラン**で**食べます。 | Resutoran de tabemasu. | 레스토랑에서 먹습니다. |
| 学校**で**勉強します。 | Gakkou de benkyou shimasu. | 학교에서 공부합니다. |
| 公園**で**遊びます。 | Kouen de asobimasu. | 공원에서 놉니다. |

### 7.3 수단/방법의 で

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| 電車**で**行きます。 | Densha de ikimasu. | 전철로 갑니다. |
| 日本語**で**話します。 | Nihongo de hanashimasu. | 일본어로 말합니다. |
| はし**で**食べます。 | Hashi de tabemasu. | 젓가락으로 먹습니다. |
| ペン**で**書きます。 | Pen de kakimasu. | 펜으로 씁니다. |

### 7.4 に vs で: 장소 혼동

초보자가 가장 흔히 하는 실수 중 하나입니다:

| 조사 | 용도 | 예시 | 의미 |
|------|------|------|------|
| **に** | 존재하는 장소 | 図書館**に**います。 | 도서관에 있습니다. (그냥 거기 있음) |
| **で** | 행위가 일어나는 장소 | 図書館**で**勉強します。 | 도서관에서 공부합니다. (무언가를 하고 있음) |

```
이렇게 생각하세요:
  に = 정적 위치 (어딘가에 있음)
  で = 동적 위치 (어딘가에서 무언가를 함)

  公園にいます。    → 공원에 있습니다. (거기 존재)
  公園で走ります。  → 공원에서 달립니다. (행동을 수행)
```

## 8. か (ka)를 사용한 의문문

### 8.1 의문문 만들기

```
┌──────────────────────────────────────────────────────────────────┐
│              か — 의문 조사                                        │
│                                                                  │
│   규칙: 문장 끝에 か를 붙이면 의문문이 됩니다                       │
│   어순 변경이 필요 없습니다!                                       │
│                                                                  │
│   서술문: これは本です。    (Kore wa hon desu.)                    │
│           "이것은 책입니다."                                       │
│                                                                  │
│   의문문: これは本ですか。  (Kore wa hon desu ka.)                 │
│           "이것은 책입니까?"                                       │
└──────────────────────────────────────────────────────────────────┘
```

### 8.2 예/아니오 의문문

| 서술문 | 의문문 |
|--------|--------|
| 日本人です。(일본인입니다.) | 日本人ですか。(일본인입니까?) |
| 学生です。(학생입니다.) | 学生ですか。(학생입니까?) |
| おいしいです。(맛있습니다.) | おいしいですか。(맛있습니까?) |

**예/아니오 의문문 답하기**:

| 답변 | 일본어 | 로마자 |
|------|--------|--------|
| 네 | はい、そうです。 | Hai, sou desu. |
| 아니오 | いいえ、ちがいます。 | Iie, chigaimasu. |

### 8.3 의문사 (疑問詞 / ぎもんし)

| 의문사 | 로마자 | 의미 | 예시 |
|--------|--------|------|------|
| 何 / なに / なん | nani / nan | 무엇 | これは何ですか。(이것은 무엇입니까?) |
| だれ | dare | 누구 | あの人はだれですか。(저 사람은 누구입니까?) |
| どこ | doko | 어디 | トイレはどこですか。(화장실은 어디입니까?) |
| いつ | itsu | 언제 | いつ行きますか。(언제 갑니까?) |
| どう | dou | 어떻게 | 日本はどうですか。(일본은 어떻습니까?) |
| どうして / なぜ | doushite / naze | 왜 | どうして日本語を勉強しますか。(왜 일본어를 공부합니까?) |
| いくら | ikura | 얼마 | これはいくらですか。(이것은 얼마입니까?) |
| どれ | dore | 어느 것 (3개 이상) | どれがいいですか。(어느 것이 좋습니까?) |
| どちら | dochira | 어느 쪽 (2개) | どちらがいいですか。(어느 쪽이 좋습니까?) |

### 8.4 何 (nani) vs 何 (nan)

何에는 뒤에 오는 것에 따라 두 가지 읽기가 있습니다:

| 읽기 | 뒤에 오는 것 | 예시 |
|------|-------------|------|
| なん (nan) | です, の, 조수사 | これは**なん**ですか。(이것은 무엇입니까?) |
| なに (nani) | を, が, も, 조사 | **なに**を食べますか。(무엇을 먹습니까?) |

## 9. こそあど 지시어

### 9.1 こそあど 체계

일본어 지시어는 화자로부터의 거리에 따라 체계적으로 구성됩니다:

```
┌──────────────────────────────────────────────────────────────────┐
│              こそあど — 거리 체계                                   │
│                                                                  │
│   こ (ko-) = 화자 가까이 ("이")                                    │
│   そ (so-) = 청자 가까이 ("그")                                    │
│   あ (a-)  = 둘 다에서 멀리 ("저")                                 │
│   ど (do-) = 의문 ("어느?")                                       │
│                                                                  │
│        화자              청자              멀리                    │
│         ↓                 ↓                ↓                      │
│        こ (ko)           そ (so)          あ (a)                  │
│        "이"              "그"            "저"                      │
│                                                                  │
│                        ど (do) = "어느?"                           │
└──────────────────────────────────────────────────────────────────┘
```

### 9.2 완전한 こそあど 표

| 범주 | こ (내 가까이) | そ (네 가까이) | あ (멀리) | ど (의문) |
|------|--------------|---------------|---------|---------|
| 사물 | これ (kore) 이것 | それ (sore) 그것 | あれ (are) 저것 | どれ (dore) 어느 것 |
| 사물 + 명사 | この (kono) 이~ | その (sono) 그~ | あの (ano) 저~ | どの (dono) 어느~ |
| 장소 | ここ (koko) 여기 | そこ (soko) 거기 | あそこ (asoko) 저기 | どこ (doko) 어디 |
| 방향 | こちら (kochira) 이쪽 | そちら (sochira) 그쪽 | あちら (achira) 저쪽 | どちら (dochira) 어느 쪽 |
| 종류 | こんな (konna) 이런 | そんな (sonna) 그런 | あんな (anna) 저런 | どんな (donna) 어떤 |

### 9.3 これ/それ/あれ (대명사: 이것/그것/저것)

단독으로 사용됩니다 (명사 없이):

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| **これ**は何ですか。 | Kore wa nan desu ka. | 이것은 무엇입니까? |
| **それ**は本です。 | Sore wa hon desu. | 그것은 책입니다. |
| **あれ**は富士山です。 | Are wa Fuji-san desu. | 저것은 후지산입니다. |
| **どれ**がいいですか。 | Dore ga ii desu ka. | 어느 것이 좋습니까? |

### 9.4 この/その/あの (형용사: 이/그/저 + 명사)

명사 앞에 붙여서 수식합니다:

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| **この**本は面白いです。 | Kono hon wa omoshiroi desu. | 이 책은 재미있습니다. |
| **その**ペンは私のです。 | Sono pen wa watashi no desu. | 그 펜은 제 것입니다. |
| **あの**人はだれですか。 | Ano hito wa dare desu ka. | 저 사람은 누구입니까? |
| **どの**本がいいですか。 | Dono hon ga ii desu ka. | 어느 책이 좋습니까? |

> **중요**: これ, それ, あれ는 단독으로 사용됩니다. この, その, あの는 반드시 명사가 뒤따라야 합니다.
> - 올바름: **この本** (이 책)
> - 틀림: ~~この~~ (이 — 불완전, 명사가 필요)

### 9.5 ここ/そこ/あそこ (장소: 여기/거기/저기)

| 일본어 | 로마자 | 한국어 |
|--------|--------|--------|
| **ここ**はどこですか。 | Koko wa doko desu ka. | 여기는 어디입니까? / 여기가 어디예요? |
| トイレは**そこ**です。 | Toire wa soko desu. | 화장실은 거기입니다. |
| 駅は**あそこ**です。 | Eki wa asoko desu. | 역은 저기입니다. |

## 10. 종합 정리

### 10.1 조사 요약

| 조사 | 읽기 | 기능 | 예시 |
|------|------|------|------|
| は | wa | 주제 표시 | 私**は**学生です。(저는 학생입니다.) |
| が | ga | 주어 표시 | 猫**が**います。(고양이가 있습니다.) |
| を | o | 목적어 표시 | すし**を**食べます。(초밥을 먹습니다.) |
| に | ni | 방향 / 시간 / 존재 장소 | 学校**に**行きます。(학교에 갑니다.) |
| で | de | 행위 장소 / 수단 | 図書館**で**勉強します。(도서관에서 공부합니다.) |
| か | ka | 의문 | 学生です**か**。(학생입니까?) |
| の | no | 소유 / 연결 | 私**の**本 (내 책) |
| も | mo | ~도 | 私**も**学生です。(저도 학생입니다.) |

### 10.2 더 긴 문장 만들기

```
기본:
  すしを食べます。
  Sushi o tabemasu.
  "초밥을 먹습니다."

주제 추가:
  私はすしを食べます。
  Watashi wa sushi o tabemasu.
  "저는 초밥을 먹습니다."

장소 추가:
  私はレストランですしを食べます。
  Watashi wa resutoran de sushi o tabemasu.
  "저는 레스토랑에서 초밥을 먹습니다."

시간 추가:
  私は毎日レストランですしを食べます。
  Watashi wa mainichi resutoran de sushi o tabemasu.
  "저는 매일 레스토랑에서 초밥을 먹습니다."

전체 패턴: [주제]は [시간] [장소]で [목적어]を [동사]
```

### 10.3 일반적인 어순 템플릿

```
[주제]は [시간(に)] [장소(で/に)] [목적어(を)] [동사]

私は     毎朝      カフェで     コーヒーを   飲みます。
Watashi wa  maiasa   kafe de    koohii o    nomimasu.
"저는 매일 아침 카페에서 커피를 마십니다."
```

```
4. 図書館___勉強します。

6. これは本です___。

```

<details>
<summary>정답 보기</summary>

```
1. 私は学生です。 (は — 주제 표시)
2. すしを食べます。 (を — 목적어 표시)
3. 学校に行きます。 (に — 방향 "~에")
4. 図書館で勉強します。 (で — 행위 장소)
5. 猫がいます。 (が — 존재 동사의 주어)
6. これは本ですか。 (か — 의문)
7. 何を食べますか。 (を — "먹다"의 목적어)
8. 七時に起きます。 (に — 특정 시간)
9. あの人はだれですか。 (は — 주제 표시)
10. ペンで書きます。 (で — 수단/도구)
```

</details>

### 연습 2: 올바른 지시어 선택

これ/それ/あれ 또는 この/その/あの를 넣으세요:

```
1. ___は何ですか。 (내가 들고 있는 것을 가리키며)
2. ___本は面白いです。 (청자 가까이에 있는 책)
3. ___は富士山です。 (멀리 있는 산)
4. ___ペンをください。 (화자 가까이에 있는 펜)
5. ___人はだれですか。 (두 사람 모두에게서 먼 사람)
```

<details>
<summary>정답 보기</summary>

```
1. これは何ですか。 (これ — 화자 가까이, 단독 사용)
2. その本は面白いです。 (その — 명사 수식, 청자 가까이)
3. あれは富士山です。 (あれ — 둘 다에게서 멀리, 단독 사용)
4. このペンをください。 (この — 명사 수식, 화자 가까이)
5. あの人はだれですか。 (あの — 명사 수식, 둘 다에게서 멀리)
```

</details>

### 연습 3: に vs で

に 또는 で를 선택하세요:

```
1. 公園___走ります。 (공원에서 달립니다)
2. 公園___います。 (공원에 있습니다)
3. レストラン___食べます。 (레스토랑에서 먹습니다)
4. 部屋___猫がいます。 (방에 고양이가 있습니다)
5. 日本___行きます。 (일본에 갑니다)
6. 電車___行きます。 (전철로 갑니다)
```

<details>
<summary>정답 보기</summary>

```
1. 公園で走ります。 (で — 행위가 일어나는 장소)
2. 公園にいます。 (に — 존재 장소)
3. レストランで食べます。 (で — 행위가 일어나는 장소)
4. 部屋に猫がいます。 (に — いる와 함께 존재 장소)
5. 日本に行きます。 (に — 방향/목적지)
6. 電車で行きます。 (で — 교통 수단)
```

</details>

### 연습 4: 일본어로 번역하기

다음 문장을 정중체로 일본어로 번역하세요:

```
1. 이것은 펜입니다.
2. 저는 선생님이 아닙니다.
3. 역은 어디입니까?
4. 학교에서 일본어를 공부합니다.
5. 전철로 도쿄에 갑니다.
6. 그것은 무엇입니까? (청자 가까이에 있는 것)
7. 저 사람은 누구입니까?
8. 7시에 아침밥을 먹습니다.
```

<details>
<summary>정답 보기</summary>

```
1. これはペンです。 (Kore wa pen desu.)
2. 私は先生ではありません。 (Watashi wa sensei dewa arimasen.)
3. 駅はどこですか。 (Eki wa doko desu ka.)
4. 学校で日本語を勉強します。 (Gakkou de nihongo o benkyou shimasu.)
5. 電車で東京に行きます。 (Densha de Toukyou ni ikimasu.)
6. それは何ですか。 (Sore wa nan desu ka.)
7. あの人はだれですか。 (Ano hito wa dare desu ka.)
8. 七時に朝ごはんを食べます。 (Shichiji ni asagohan o tabemasu.)
```

</details>

### 연습 5: は vs が

は 또는 が를 선택하세요:

```
1. 私___田中です。 (저는 다나카입니다 — 자기소개)
2. だれ___来ましたか。 (누가 왔습니까?)
3. 日本語___わかります。 (일본어를 알아듣습니다)
4. あの人___先生です。 (저 사람은 선생님입니다 — 주제)
5. 田中さん___来ました。 (다나카 씨가 왔습니다 — "누가 왔어?"에 대한 답)
6. すし___好きです。 (초밥을 좋아합니다)
```

<details>
<summary>정답 보기</summary>

```
1. 私は田中です。 (は — 주제, 자기소개)
2. だれが来ましたか。 (が — 의문사가 주어)
3. 日本語がわかります。 (が — 능력 동사 わかる는 が를 요구)
4. あの人は先生です。 (は — 주제, 알려진 정보)
5. 田中さんが来ました。 (が — 새로운 정보, "누가"에 대한 답)
6. すしが好きです。 (が — 형용사 好き는 が를 요구)
```

</details>

### 연습 6: 완전한 문장 만들기

주어진 단어들을 올바른 일본어 문장으로 배열하세요:

```
1. [食べます / を / 私は / すし]
2. [行きます / 学校に / で / 電車]
3. [ですか / これ / 何 / は]
4. [で / します / 勉強 / を / 図書館 / 日本語]
5. [います / に / 猫が / 庭]
```

<details>
<summary>정답 보기</summary>

```
1. 私はすしを食べます。
   (Watashi wa sushi o tabemasu. — 저는 초밥을 먹습니다.)

2. 電車で学校に行きます。
   (Densha de gakkou ni ikimasu. — 전철로 학교에 갑니다.)

3. これは何ですか。
   (Kore wa nan desu ka. — 이것은 무엇입니까?)

4. 図書館で日本語を勉強します。
   (Toshokan de nihongo o benkyou shimasu. — 도서관에서 일본어를 공부합니다.)

5. 庭に猫がいます。
   (Niwa ni neko ga imasu. — 정원에 고양이가 있습니다.)
```

</details>
