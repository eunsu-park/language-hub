## 1. 일본어 동사 기초

### 1.1 일본어 동사가 영어와 다른 점

```
┌──────────────────────────────────────────────────────────────────┐
│              일본어 동사 핵심 사항                                  │
│                                                                  │
│   • 동사는 항상 문장 끝에 옵니다                                   │
│   • 동사는 인칭/수에 따라 변하지 않습니다                           │
│     ("I eat" vs "he eats" 같은 구별 없음)                         │
│   • 동사는 시제, 공손함, 서법에 따라 활용됩니다                     │
│   • 사전형은 항상 -u 소리로 끝납니다                                │
│   • 두 가지 공손함 수준: 평어(반말)와 정중체(です/ます)              │
│                                                                  │
│   食べる (taberu) = "먹다"                                        │
│   모든 인칭에 같은 형태: 내가, 너가, 그가, 우리가, 그들이 먹다      │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 사전형 (辞書形 / じしょけい)

**사전형** (평어 현재 긍정이라고도 함)은 사전에서 찾을 수 있는 기본 형태입니다. 항상 -u 모음 소리로 끝납니다:

| 어미 | 예시 |
|------|------|
| -る (-ru) | 食べる (taberu), 見る (miru), 起きる (okiru) |
| -う (-u) | 買う (kau), 会う (au) |
| -く (-ku) | 書く (kaku), 聞く (kiku) |
| -ぐ (-gu) | 泳ぐ (oyogu) |
| -す (-su) | 話す (hanasu), 出す (dasu) |
| -つ (-tsu) | 待つ (matsu), 持つ (motsu) |
| -ぬ (-nu) | 死ぬ (shinu) — 유일한 하나! |
| -ぶ (-bu) | 遊ぶ (asobu), 飛ぶ (tobu) |
| -む (-mu) | 読む (yomu), 飲む (nomu) |

## 2. 세 가지 동사 그룹

### 2.1 개요

```
┌──────────────────────────────────────────────────────────────────┐
│              세 가지 동사 그룹                                     │
│                                                                  │
│   그룹 1: 이치단 동사 (一段動詞) — "る동사"                        │
│     -iru 또는 -eru로 끝남                                        │
│     る를 떼고 활용 어미를 붙임                                     │
│     예: 食べる, 見る, 起きる                                       │
│                                                                  │
│   그룹 2: 고단 동사 (五段動詞) — "う동사"                          │
│     -u, -ku, -gu, -su, -tsu, -nu, -bu, -mu, -ru로 끝남           │
│     마지막 모음을 변경한 후 어미를 붙임                             │
│     예: 書く, 話す, 読む, 買う                                     │
│                                                                  │
│   그룹 3: 불규칙 동사 (不規則動詞)                                  │
│     단 두 개:                                                     │
│     する (suru, 하다)와 来る (kuru, 오다)                           │
│     개별적으로 외워야 함                                            │
└──────────────────────────────────────────────────────────────────┘
```

### 2.2 그룹 1: 이치단 동사 (る동사)

이치단은 "한 단계"라는 뜻으로, 활용에 한 단계만 필요합니다: る를 떼고 새 어미를 붙입니다.

**식별 방법**: 동사가 -**i**ru 또는 -**e**ru로 끝납니다:

| 동사 | 로마자 | 의미 | 어간 (る 제거) |
|------|--------|------|---------------|
| 食べる | taberu | 먹다 | 食べ (tabe) |
| 見る | miru | 보다 | 見 (mi) |
| 起きる | okiru | 일어나다 | 起き (oki) |
| 寝る | neru | 자다 | 寝 (ne) |
| 着る | kiru | 입다 | 着 (ki) |
| 出る | deru | 나가다 | 出 (de) |
| 開ける | akeru | 열다 | 開け (ake) |
| 閉める | shimeru | 닫다 | 閉め (shime) |
| 教える | oshieru | 가르치다 | 教え (oshie) |
| 答える | kotaeru | 대답하다 | 答え (kotae) |

### 2.3 그룹 2: 고단 동사 (う동사)

고단은 "다섯 단계"라는 뜻으로, 마지막 자음이 다양한 활용에서 5개의 서로 다른 모음 (a, i, u, e, o)과 결합합니다.

| 동사 | 로마자 | 의미 | 어미 |
|------|--------|------|------|
| 書く | kaku | 쓰다 | -ku |
| 聞く | kiku | 듣다 | -ku |
| 泳ぐ | oyogu | 수영하다 | -gu |
| 話す | hanasu | 말하다 | -su |
| 待つ | matsu | 기다리다 | -tsu |
| 死ぬ | shinu | 죽다 | -nu |
| 遊ぶ | asobu | 놀다 | -bu |
| 読む | yomu | 읽다 | -mu |
| 飲む | nomu | 마시다 | -mu |
| 買う | kau | 사다 | -u |
| 会う | au | 만나다 | -u |
| 作る | tsukuru | 만들다 | -ru |
| 乗る | noru | 타다 | -ru |
| 帰る | kaeru | 돌아가다 | -ru |
| 走る | hashiru | 달리다 | -ru |

### 2.4 주의해야 할 る로 끝나는 고단 동사

일부 동사는 -iru나 -eru로 끝나지만 실제로는 이치단(그룹 1)이 아닌 고단(그룹 2)입니다. 반드시 외워야 합니다:

| 동사 | 로마자 | 의미 | 그룹 | 주의 이유 |
|------|--------|------|------|-----------|
| 帰る | kaeru | 돌아가다 | 고단 | -eru로 끝남 |
| 走る | hashiru | 달리다 | 고단 | -iru로 끝남 |
| 切る | kiru | 자르다 | 고단 | -iru로 끝남 |
| 知る | shiru | 알다 | 고단 | -iru로 끝남 |
| 入る | hairu | 들어가다 | 고단 | -iru로 끝남 |
| 要る | iru | 필요하다 | 고단 | -iru로 끝남 |

> **팁**: 이를 구별할 확실한 규칙은 없습니다. 새 동사를 배울 때 항상 어떤 그룹인지 함께 배우세요.

### 2.5 그룹 3: 불규칙 동사

진정한 불규칙 동사는 단 두 개뿐입니다:

| 동사 | 로마자 | 의미 |
|------|--------|------|
| する | suru | 하다 |
| 来る (くる) | kuru | 오다 |

する는 명사와 결합하여 복합 동사를 만들기도 합니다:

| 복합어 | 로마자 | 의미 |
|--------|--------|------|
| 勉強する | benkyou suru | 공부하다 |
| 料理する | ryouri suru | 요리하다 |
| 運動する | undou suru | 운동하다 |
| 散歩する | sanpo suru | 산책하다 |
| 買い物する | kaimono suru | 쇼핑하다 |
| 電話する | denwa suru | 전화하다 |

## 3. ます형: 정중한 현재/미래

### 3.1 형성 규칙

ます형은 표준 정중체입니다. 현재 시제, 미래 시제, 습관적 행위를 표현합니다:

```
┌──────────────────────────────────────────────────────────────────┐
│              ます형 만드는 방법                                     │
│                                                                  │
│   그룹 1 (이치단): る 떼기 → ます 붙이기                            │
│     食べる → 食べ + ます → 食べます                                  │
│                                                                  │
│   그룹 2 (고단): 마지막 -u를 -i로 변경 → ます 붙이기                │
│     書く → 書き + ます → 書きます  (ku → ki)                       │
│     話す → 話し + ます → 話します  (su → shi)                      │
│     読む → 読み + ます → 読みます  (mu → mi)                      │
│                                                                  │
│   그룹 3 (불규칙): 암기                                            │
│     する → します                                                  │
│     来る → 来ます (きます)                                         │
└──────────────────────────────────────────────────────────────────┘
```

### 3.2 그룹 2 (고단) 어간 변화

마지막 가나가 모음 전환(u → i)에 따라 변합니다:

| 어미 | 변화 | 사전형 → ます | 예시 |
|------|------|--------------|------|
| -う (u) | う → い | 買**う** → 買**い**ます | kau → kaimasu |
| -く (ku) | く → き | 書**く** → 書**き**ます | kaku → kakimasu |
| -ぐ (gu) | ぐ → ぎ | 泳**ぐ** → 泳**ぎ**ます | oyogu → oyogimasu |
| -す (su) | す → し | 話**す** → 話**し**ます | hanasu → hanashimasu |
| -つ (tsu) | つ → ち | 待**つ** → 待**ち**ます | matsu → machimasu |
| -ぬ (nu) | ぬ → に | 死**ぬ** → 死**に**ます | shinu → shinimasu |
| -ぶ (bu) | ぶ → び | 遊**ぶ** → 遊**び**ます | asobu → asobimasu |
| -む (mu) | む → み | 読**む** → 読**み**ます | yomu → yomimasu |
| -る (ru) | る → り | 帰**る** → 帰**り**ます | kaeru → kaerimasu |

### 3.3 ます형 활용 표

| 형태 | 어미 | 의미 | 예시 (食べる) | 예시 (書く) |
|------|------|------|-------------|------------|
| 현재 긍정 | ます | 먹다/먹을 것이다 | 食べます | 書きます |
| 현재 부정 | ません | 먹지 않다 | 食べません | 書きません |
| 과거 긍정 | ました | 먹었다 | 食べました | 書きました |
| 과거 부정 | ませんでした | 먹지 않았다 | 食べませんでした | 書きませんでした |

## 5. て형: 만능 활용형

### 5.1 て형이 중요한 이유

```
┌──────────────────────────────────────────────────────────────────┐
│              て형 — 가장 다재다능한 활용형                           │
│                                                                  │
│   て형의 용도:                                                    │
│   • 문장 연결 ("먹고 마시다")                                      │
│   • 요청 (食べてください — "드세요")                                │
│   • 진행 (食べている — "먹고 있다")                                 │
│   • 허가 (食べてもいい — "먹어도 된다")                              │
│   • 금지 (食べてはいけない — "먹어서는 안 된다")                     │
│   • 행위의 순서 ("가서 먹었다")                                    │
│                                                                  │
│   て형은 시제가 아닙니다 — 연결 형태입니다.                          │
│   て형을 배우는 것은 일본어 문법에서                                 │
│   가장 중요한 이정표 중 하나입니다.                                  │
└──────────────────────────────────────────────────────────────────┘
```

### 5.2 그룹별 て형 규칙

**그룹 1 (이치단)**: る 떼기 → て 붙이기

| 사전형 | て형 | 로마자 |
|--------|------|--------|
| 食べる | 食べて | tabete |
| 見る | 見て | mite |
| 起きる | 起きて | okite |

**그룹 3 (불규칙)**: 암기

| 사전형 | て형 | 로마자 |
|--------|------|--------|
| する | して | shite |
| 来る | 来て (きて) | kite |

### 5.3 고단 て형: 노래 방법

많은 학습자들이 て형 규칙을 노래로 외웁니다 ("Santa Claus Is Coming to Town" 멜로디):

```
┌──────────────────────────────────────────────────────────────────┐
│              고단 동사 て형 노래                                    │
│                                                                  │
│   う、つ、る → って      (u, tsu, ru → tte)                      │
│   む、ぶ、ぬ → んで     (mu, bu, nu → nde)                      │
│   く → いて              (ku → ite)                              │
│   ぐ → いで              (gu → ide)                              │
│   す → して              (su → shite)                            │
│                                                                  │
│   예외: 行く → 行って (itte, いいて가 아님)                        │
└──────────────────────────────────────────────────────────────────┘
```

### 5.4 고단 て형 상세 표

| 어미 | て형 규칙 | 사전형 | て형 | 로마자 |
|------|-----------|--------|------|--------|
| -う | → -って | 買**う** | 買**って** | katte |
| -つ | → -って | 待**つ** | 待**って** | matte |
| -る | → -って | 帰**る** | 帰**って** | kaette |
| -む | → -んで | 読**む** | 読**んで** | yonde |
| -ぶ | → -んで | 遊**ぶ** | 遊**んで** | asonde |
| -ぬ | → -んで | 死**ぬ** | 死**んで** | shinde |
| -く | → -いて | 書**く** | 書**いて** | kaite |
| -ぐ | → -いで | 泳**ぐ** | 泳**いで** | oyoide |
| -す | → -して | 話**す** | 話**して** | hanashite |

> **예외**: 行く (iku, 가다) → 行**って** (itte), 行いて가 아닙니다. 이것은 -く 동사 중 유일한 예외입니다.

### 5.5 て형 용도 (미리보기)

| 용도 | 패턴 | 예시 | 의미 |
|------|------|------|------|
| 요청 | て + ください | 食べてください | 드세요 |
| 진행 | て + いる/います | 食べています | 먹고 있습니다 |
| 연결 | て + 다음 행위 | 食べて、飲みました | 먹고, 마셨습니다 |
| 허가 | て + もいいですか | 食べてもいいですか | 먹어도 됩니까? |

## 6. た형: 과거 시제

### 6.1 た형 = て형에서 た/だ

평어 과거형은 て형과 정확히 같은 패턴을 따르지만, て/で 대신 た/だ를 사용합니다:

```
규칙: て형에서 て가 있는 곳 → た형에서는 た
     て형에서 で가 있는 곳 → た형에서는 だ
```

### 6.2 た형 표

**그룹 1 (이치단)**: る 떼기 → た 붙이기

| 사전형 | た형 | 로마자 |
|--------|------|--------|
| 食べる | 食べた | tabeta |
| 見る | 見た | mita |

**그룹 3 (불규칙)**:

| 사전형 | た형 | 로마자 |
|--------|------|--------|
| する | した | shita |
| 来る | 来た (きた) | kita |

**그룹 2 (고단)**:

| 어미 | て형 | た형 | 예시: 사전형 → た |
|------|------|------|------------------|
| -う, -つ, -る | -って | -った | 買う → 買った (katta) |
| -む, -ぶ, -ぬ | -んで | -んだ | 読む → 読んだ (yonda) |
| -く | -いて | -いた | 書く → 書いた (kaita) |
| -ぐ | -いで | -いだ | 泳ぐ → 泳いだ (oyoida) |
| -す | -して | -した | 話す → 話した (hanashita) |
| 行く (예외) | 行って | 行った | 行く → 行った (itta) |

### 6.3 정중한 과거 시제

정중한 과거형은 단순히 ました를 사용합니다:

| 사전형 | 정중 과거 | 로마자 | 의미 |
|--------|-----------|--------|------|
| 食べる | 食べました | tabemashita | 먹었습니다 |
| 行く | 行きました | ikimashita | 갔습니다 |
| する | しました | shimashita | 했습니다 |
| 来る | 来ました (きました) | kimashita | 왔습니다 |
| 読む | 読みました | yomimashita | 읽었습니다 |
| 買う | 買いました | kaimashita | 샀습니다 |

## 7. 부정형: ません과 ない

### 7.1 정중 부정: ません

ます를 ません으로 바꾸면 됩니다:

| 긍정 | 부정 | 의미 |
|------|------|------|
| 食べます | 食べません | 먹지 않습니다 |
| 行きます | 行きません | 가지 않습니다 |
| 書きます | 書きません | 쓰지 않습니다 |
| します | しません | 하지 않습니다 |
| 来ます | 来ません (きません) | 오지 않습니다 |

### 7.2 정중 부정 과거: ませんでした

| 긍정 과거 | 부정 과거 | 의미 |
|-----------|-----------|------|
| 食べました | 食べませんでした | 먹지 않았습니다 |
| 行きました | 行きませんでした | 가지 않았습니다 |
| しました | しませんでした | 하지 않았습니다 |

### 7.3 평어 부정: ない형

평어 부정형은 각 그룹마다 다르게 형성됩니다:

```
┌──────────────────────────────────────────────────────────────────┐
│              ない형 (평어 부정)                                     │
│                                                                  │
│   그룹 1 (이치단): る 떼기 → ない 붙이기                            │
│     食べる → 食べない (tabenai)                                    │
│                                                                  │
│   그룹 2 (고단): -u를 -a로 변경 → ない 붙이기                      │
│     書く → 書かない (kakanai)  (ku → ka)                          │
│     話す → 話さない (hanasanai) (su → sa)                         │
│     読む → 読まない (yomanai) (mu → ma)                           │
│                                                                  │
│     특수: -う 동사 → -わない (NOT -あない)                          │
│     買う → 買わない (kawanai)                                      │
│                                                                  │
│   그룹 3 (불규칙): 암기                                            │
│     する → しない (shinai)                                        │
│     来る → 来ない (こない, konai)                                  │
│                                                                  │
│   특수: ある → ない (あらない가 아님)                               │
└──────────────────────────────────────────────────────────────────┘
```

### 7.4 고단 ない형: 모음 전환 (u → a)

| 어미 | 변화 | 사전형 → ない | 로마자 |
|------|------|--------------|--------|
| -う (u) | → -わない | 買**う** → 買**わ**ない | kawanai |
| -く (ku) | → -かない | 書**く** → 書**か**ない | kakanai |
| -ぐ (gu) | → -がない | 泳**ぐ** → 泳**が**ない | oyoganai |
| -す (su) | → -さない | 話**す** → 話**さ**ない | hanasanai |
| -つ (tsu) | → -たない | 待**つ** → 待**た**ない | matanai |
| -ぬ (nu) | → -なない | 死**ぬ** → 死**な**ない | shinanai |
| -ぶ (bu) | → -ばない | 遊**ぶ** → 遊**ば**ない | asobanai |
| -む (mu) | → -まない | 読**む** → 読**ま**ない | yomanai |
| -る (ru) | → -らない | 帰**る** → 帰**ら**ない | kaeranai |

> **중요한 예외**: -う로 끝나는 동사는 -**わ**ない를 사용하며, -あない가 아닙니다. 이 위치의 역사적 가나가 わ였기 때문입니다.

### 7.5 평어 부정 과거: なかった

ない를 なかった로 변경합니다:

| 평어 부정 | 평어 부정 과거 | 의미 |
|-----------|---------------|------|
| 食べない | 食べなかった | 먹지 않았다 |
| 行かない | 行かなかった | 가지 않았다 |
| しない | しなかった | 하지 않았다 |
| 来ない (こない) | 来なかった (こなかった) | 오지 않았다 |

## 8. 전체 활용 요약

### 8.1 마스터 표: 그룹 1 (이치단) — 食べる (taberu, 먹다)

| 형태 | 정중체 | 평어 |
|------|--------|------|
| 현재 긍정 | 食べます (tabemasu) | 食べる (taberu) |
| 현재 부정 | 食べません (tabemasen) | 食べない (tabenai) |
| 과거 긍정 | 食べました (tabemashita) | 食べた (tabeta) |
| 과거 부정 | 食べませんでした (tabemasen deshita) | 食べなかった (tabenakatta) |
| て형 | — | 食べて (tabete) |

### 8.2 마스터 표: 그룹 2 (고단) — 書く (kaku, 쓰다)

| 형태 | 정중체 | 평어 |
|------|--------|------|
| 현재 긍정 | 書きます (kakimasu) | 書く (kaku) |
| 현재 부정 | 書きません (kakimasen) | 書かない (kakanai) |
| 과거 긍정 | 書きました (kakimashita) | 書いた (kaita) |
| 과거 부정 | 書きませんでした (kakimasen deshita) | 書かなかった (kakanakatta) |
| て형 | — | 書いて (kaite) |

### 8.3 마스터 표: 그룹 2 (고단) — 読む (yomu, 읽다)

| 형태 | 정중체 | 평어 |
|------|--------|------|
| 현재 긍정 | 読みます (yomimasu) | 読む (yomu) |
| 현재 부정 | 読みません (yomimasen) | 読まない (yomanai) |
| 과거 긍정 | 読みました (yomimashita) | 読んだ (yonda) |
| 과거 부정 | 読みませんでした (yomimasen deshita) | 読まなかった (yomanakatta) |
| て형 | — | 読んで (yonde) |

### 8.4 마스터 표: 불규칙 — する (suru, 하다) & 来る (kuru, 오다)

| 형태 | する (정중체) | する (평어) | 来る (정중체) | 来る (평어) |
|------|-------------|-----------|-------------|-----------|
| 현재 긍정 | します | する | きます | くる |
| 현재 부정 | しません | しない | きません | こない |
| 과거 긍정 | しました | した | きました | きた |
| 과거 부정 | しませんでした | しなかった | きませんでした | こなかった |
| て형 | — | して | — | きて |

## 9. 문장에서 동사 사용하기

### 9.1 기본 문장 패턴

```
패턴 1: [주제]は [목적어]を [동사]ます。
  私はすしを食べます。
  Watashi wa sushi o tabemasu.
  "저는 초밥을 먹습니다."

패턴 2: [주제]は [장소]に [동사]ます。
  私は学校に行きます。
  Watashi wa gakkou ni ikimasu.
  "저는 학교에 갑니다."

패턴 3: [주제]は [장소]で [목적어]を [동사]ます。
  私は図書館で本を読みます。
  Watashi wa toshokan de hon o yomimasu.
  "저는 도서관에서 책을 읽습니다."

패턴 4: [주제]は [시간]に [동사]ます。
  私は七時に起きます。
  Watashi wa shichi-ji ni okimasu.
  "저는 7시에 일어납니다."
```

### 9.2 て형으로 행위 연결하기

```
朝、起きて、シャワーを浴びて、朝ごはんを食べます。
Asa, okite, shawaa o abite, asagohan o tabemasu.
"아침에 일어나서, 샤워를 하고, 아침밥을 먹습니다."
```

### 9.3 의문문

| 질문 | 대답 |
|------|------|
| 何を食べますか。(무엇을 먹습니까?) | すしを食べます。(초밥을 먹습니다.) |
| どこに行きますか。(어디에 갑니까?) | 学校に行きます。(학교에 갑니다.) |
| いつ来ましたか。(언제 왔습니까?) | 昨日来ました。(어제 왔습니다.) |
| 日本語を話しますか。(일본어를 합니까?) | はい、少し話します。(네, 조금 합니다.) |

### 9.4 일과 표현하기

```
私は毎日六時半に起きます。
Watashi wa mainichi roku-ji han ni okimasu.
"저는 매일 6시 반에 일어납니다."

シャワーを浴びて、朝ごはんを食べます。
Shawaa o abite, asagohan o tabemasu.
"샤워를 하고, 아침밥을 먹습니다."

八時に学校に行きます。
Hachi-ji ni gakkou ni ikimasu.
"8시에 학교에 갑니다."

学校で日本語を勉強します。
Gakkou de nihongo o benkyou shimasu.
"학교에서 일본어를 공부합니다."

五時に家に帰ります。
Go-ji ni ie ni kaerimasu.
"5시에 집에 돌아갑니다."

夜、本を読んで、十一時に寝ます。
Yoru, hon o yonde, juuichi-ji ni nemasu.
"밤에 책을 읽고, 11시에 잡니다."
```

6. 来る — 그룹 3 (불규칙)
7. 飲む — 그룹 2 (고단) — -mu로 끝남
8. 起きる — 그룹 1 (이치단) — -iru로 끝남, る 제거
9. 走る — 그룹 2 (고단) — 주의! -iru로 끝나지만 고단
10. 勉強する — 그룹 3 (불규칙) — する의 복합 동사
