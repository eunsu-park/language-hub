## 선수 학습

- 히라가나와 가타카나를 유창하게 읽는 능력
- 약 500자 이상의 한자 지식
- 기본 및 중급 문법 숙지 (1~16과)
- 동사 활용형 이해 (사전형, ます형, て형, ない형, た형, 조건형)

## 학습 목표

- 일본어 텍스트에 특화된 효과적인 독해 전략 적용하기

- 일본어 논설문(論説文)을 파악하고 해석하기
- 비즈니스 문서(ビジネス文書) 읽기 및 해석하기
- 훑어읽기(스캐닝)와 정독 기법 구별하기
- JLPT 및 학술 환경에서 사용되는 주요 독해 문제 유형 인식하기
- 실제 일본어 자료 읽기에 자신감 갖기

## 1. 일본어 텍스트 독해 전략

일본어 독해에는 독특한 도전 과제가 있습니다: 세 가지 문자 체계, 단어 사이에 공백 없음, 정보 밀도가 높은 한자 복합어. 다음 전략들이 이러한 도전 과제를 극복하는 데 도움이 됩니다.

```
+----------------------------------------------------------+
|         일본어 텍스트 독해 전략                             |
+----------------------------------------------------------+
|                                                          |
|  읽기 전 (読む前に):                                      |
|  1. 제목과 소제목 확인                                    |
|  2. 텍스트 유형 파악 (기사? 이메일? 에세이?)               |
|  3. 아는 한자 훑어보기                                    |
|  4. 전체 길이와 구조 파악                                 |
|  5. 주제에 관한 배경 지식 활성화                           |
|                                                          |
|  읽는 중 (読みながら):                                    |
|  6. 모르는 한자마다 멈추지 않기                            |
|  7. 문맥으로 의미 유추하기                                |
|  8. 주제 표시 조사(は)와 주어 표시 조사(が) 주시하기        |
|  9. 각 문장 끝의 동사 찾기                                |
|  10. 접속사와 연결어에 주의하기                            |
|  11. 경어 수준으로 문맥 단서 파악하기                      |
|                                                          |
|  읽은 후 (読んだ後で):                                    |
|  12. 핵심 내용을 한 문장으로 요약                          |
|  13. 저자의 입장이나 목적 파악                             |
|  14. 모르는 어휘를 문맥 속에서 복습                        |
|  15. 새로운 이해를 바탕으로 어려운 부분 다시 읽기           |
+----------------------------------------------------------+
```

### 1.1 "동사 후치(後置)" 원칙

일본어에서는 **동사가 항상 절의 끝**에 옵니다. 이는 가장 중요한 정보(행동, 결론, 부정)가 맨 끝에 있다는 의미입니다. 다음을 훈련하십시오:

1. 문두에서 주제/주어 파악 (は나 が로 표시)
2. 중간에서 목적어와 수식어 훑어보기
3. **문말 동사**에 집중하여 핵심 의미 파악
4. 맨 끝에서 부정(ない, ません) 확인

```
┌─────────┬─────────────────┬──────────────┐
│ 주제/   │    목적어,       │  동사        │
│ 주어    │    수식어        │  (핵심 정보)  │
│ は/が   │    を/に/で/と   │  ⬅ 여기 주목 │
└─────────┴─────────────────┴──────────────┘
```

### 1.2 접속사 (接続詞)

접속사는 문장 간의 관계를 나타냅니다. 이를 인식하면 다음에 올 내용을 예측하는 데 도움이 됩니다.

| 접속사 | 의미 | 신호 |
|-----------|---------|--------|
| しかし / だが / けれども | 그러나 | 대비되는 내용이 옴 |
| したがって / そのため / だから | 따라서 | 결론이 뒤따름 |
| また / さらに / その上 | 게다가 | 추가 정보 |
| 一方 / 他方 / 反面 | 한편 | 다른 관점 |
| つまり / すなわち / 要するに | 즉 | 바꿔 말하기 / 요약 |
| 例えば / 具体的には | 예를 들어 | 예시가 뒤따름 |
| ところで / さて | 그런데 | 화제 전환 |
| なぜなら / というのは | 왜냐하면 | 설명이 뒤따름 |

## 2. 독해에서의 세 가지 문자 체계

세 가지 문자가 텍스트에서 어떻게 기능하는지 이해하면 읽기 속도와 이해력 향상에 도움이 됩니다.

### 2.1 연속 텍스트에서의 문자 기능

| 문자 | 기능 | 예시 |
|--------|----------|---------|
| **漢字** (한자) | 내용어: 명사, 동사/형용사 어간 | 学生 (학생), 食べる (먹다) |
| **ひらがな** (히라가나) | 문법: 조사, 어미, 고유어 | は, が, を, の, する, できる |
| **カタカナ** (가타카나) | 외래어, 강조, 의성어/의태어 | コンピューター, アメリカ, データ |

### 2.2 읽기 속도 향상 팁

```
+------------------------------------------+
|  시각적 분석 전략                          |
+------------------------------------------+
|                                          |
|  한자 복합어 → 핵심 개념                  |
|  (핵심 아이디어 파악을 위해 먼저 훑기)     |
|                                          |
|  한자 사이의 히라가나 → 문법적 연결       |
|  (조사가 누가 무엇을 했는지 알려줌)        |
|                                          |
|  가타카나 블록 → 외래어                   |
|  (영어/포르투갈어로 소리 내어 읽기)        |
|                                          |
|  문말 です/ます → 정중체                  |
|  문말 だ/である → 보통체/문어체            |
+------------------------------------------+
```

> **팁:** 훑어읽기를 할 때 **한자 복합어**에 집중하십시오. 핵심 의미를 담고 있습니다.
> 한자 사이의 히라가나는 문법적 관계를 나타냅니다.
> 예: 「環境問題に関する国際会議が開催された」
> 핵심 한자: 環境問題 (환경 문제) → 国際会議 (국제 회의) → 開催 (개최)
> → "환경 문제에 관한 국제 회의가 개최되었다."

## 3. 신문 기사 (新聞記事)

### 3.1 일본어 뉴스 기사의 구조

일본어 신문 기사는 서양 저널리즘과 유사한 **역피라미드** 구조를 따르지만, 특유의 어휘 관행이 있습니다.

```
┌──────────────────────────────────────┐
│           見出し (헤드라인)            │  ← 짧고 한자 중심, 조사 생략이 잦음
├──────────────────────────────────────┤
│         リード文 (리드 문장)           │  ← 누가, 무엇을, 언제, 어디서를 1~2문장으로
├──────────────────────────────────────┤
│         本文 (본문)                   │  ← 세부 사항, 배경, 인용문
├──────────────────────────────────────┤
│         背景・補足 (배경·보충)         │  ← 추가 맥락, 관련 정보
└──────────────────────────────────────┘
```

### 3.2 헤드라인 어휘

신문 헤드라인은 특별한 압축 문체를 사용합니다. 일반적인 패턴은 다음과 같습니다:

| 헤드라인 스타일 | 완전한 형태 | 의미 |
|----------------|-----------|---------|
| 首相、訪米へ | 首相は訪米する予定だ | 총리, 방미 예정 |
| 新法成立 | 新しい法律が成立した | 신법 성립 |
| 景気回復の兆し | 景気が回復する兆しがある | 경기 회복 조짐 |
| ～か | ～かもしれない | ~일 수도 |
| ～見通し | ～見通しである | ~전망 / ~예상 |

### 3.3 일반적인 뉴스 어휘

| 일본어 | 읽기 | 한국어 |
|----------|---------|---------|
| 政府 | せいふ | 정부 |
| 首相 | しゅしょう | 총리 |
| 発表 | はっぴょう | 발표 |
| 会談 | かいだん | 회담 |
| 対策 | たいさく | 대책 |
| 被害 | ひがい | 피해 |
| 増加 | ぞうか | 증가 |
| 減少 | げんしょう | 감소 |
| 検討 | けんとう | 검토 |
| 実施 | じっし | 실시 |
| 影響 | えいきょう | 영향 |
| 回復 | かいふく | 회복 |
| 経済 | けいざい | 경제 |
| 調査 | ちょうさ | 조사 |
| 批判 | ひはん | 비판 |
| 合意 | ごうい | 합의 |
| 方針 | ほうしん | 방침 |
| 見込み | みこみ | 전망 |

### 3.4 뉴스에서의 수동태

일본어 뉴스는 객관성을 유지하기 위해 **수동태**(受身形)를 많이 사용합니다:

| 수동태 (뉴스 스타일) | 한국어 |
|---------------------|---------|
| 計画が発表**された**。 | 계획이 발표**되었다**. |
| 新法が施行**される**見通しだ。 | 신법이 시행**될** 전망이다. |
| 容疑者が逮捕**された**。 | 용의자가 체포**되었다**. |
| 被害が確認**されている**。 | 피해가 확인**되고 있다**. |

## 4. 논설문 (論説文)

### 4.1 일본어 에세이의 구조

일본어 에세이(특히 논설문)는 **起承転結**(きしょうてんけつ)이라는 전통적인 4단계 구조를 따릅니다:

| 부분 | 명칭 | 역할 |
|------|------|------|
| 起 (き) | 도입 | 주제 소개, 맥락 제공 |
| 承 (しょう) | 전개 | 주제 확장, 근거 제시 |
| 転 (てん) | 전환 | 다른 각도, 반론, 또는 더 깊은 논점 제시 |
| 結 (けつ) | 결론 | 핵심 주장 제시, 마무리 |

> **중요:** 転(전환) 부분이 매우 중요합니다. 일본어 에세이는 결론에 도달하기 전에 예상치 못한
> 각도나 반론을 제시하는 경우가 많습니다. 전개 부분에 저자의 최종 의견이
> 들어 있다고 가정하지 마십시오.

### 4.2 저자의 의견 파악하기

다음과 같은 의견 표지(markers)를 찾으십시오:

| 패턴 | 의미 | 강도 |
|---------|---------|----------|
| ～と思う / ～と考える | ~라고 생각하다 | 직접적 의견 |
| ～べきだ / ～べきである | ~해야 한다 | 강한 권고 |
| ～ではないだろうか | ~이 아닌가? | 수사적 질문 (= 그렇다고 생각함) |
| ～のではないか | ~이 아닌가? | 동의를 유도 |
| ～に違いない | ~임에 틀림없다 | 강한 확신 |
| ～と言えるだろう | ~라고 할 수 있을 것이다 | 온건한 주장 |
| ～必要がある | ~할 필요가 있다 | 행동 촉구 |
| ～わけにはいかない | ~할 수는 없다 | 강한 필요성 |

### 4.3 일반적인 논설문 어휘

| 일본어 | 읽기 | 한국어 |
|----------|---------|---------|
| 主張 | しゅちょう | 주장 |
| 根拠 | こんきょ | 근거 |
| 傾向 | けいこう | 경향 |
| 現状 | げんじょう | 현 상황 |
| 背景 | はいけい | 배경 |
| 課題 | かだい | 과제 |
| 観点 | かんてん | 관점 |
| 矛盾 | むじゅん | 모순 |
| 具体的 | ぐたいてき | 구체적 |
| 抽象的 | ちゅうしょうてき | 추상적 |

### 4.4 논리 전개 표지

| 표지 | 기능 | 사용 예 |
|--------|----------|-------------|
| まず / 第一に | 우선 | 첫 번째 논점 도입 |
| 次に / 第二に | 다음으로 / 둘째 | 두 번째 논점 도입 |
| 最後に / 結論として | 마지막으로 / 결론적으로 | 마무리 |
| 確かに～。しかし～。 | 확실히 ~. 그러나 ~. | 인정 후 반박 |
| ～という意見もある。しかし～。 | ~라는 의견도 있다. 그러나 ~. | 반대 의견 제시 |

## 5. 비즈니스 문서 (ビジネス文書)

### 5.1 비즈니스 문서의 유형

| 문서 유형 | 일본어 | 용도 |
|--------------|----------|---------|
| 비즈니스 이메일 | ビジネスメール | 일상 업무 커뮤니케이션 |
| 사내 메모 | 社内メモ | 사내 커뮤니케이션 |
| 회의록 | 議事録 (ぎじろく) | 회의 기록 |
| 기획서 | 企画書 (きかくしょ) | 프로젝트 제안 |
| 보고서 | 報告書 (ほうこくしょ) | 상황 및 결과 보고 |
| 통지서 | 通知 (つうち) | 공식 안내 |

### 5.2 비즈니스 이메일 구조

```
┌──────────────────────────────────────────┐
│  宛名: ○○株式会社 △△部 □□様            │  ← 수신인 (회사, 부서, 이름 + 様)
│                                          │
│  件名: ご注文の確認について              │  ← 제목
│                                          │
│  ○○様                                   │  ← 인사
│                                          │
│  いつもお世話になっております。           │  ← 정형화된 인사말
│  ○○株式会社の△△でございます。           │  ← 자기소개
│                                          │
│  本文 (본문)                             │  ← 주요 내용
│                                          │
│  ご確認のほど、よろしくお願いいたします。 │  ← 마무리 요청
│                                          │
│  敬具                                    │  ← 격식체 맺음말
│  ○○株式会社 △△部                       │  ← 서명란
│  氏名                                    │
│  電話: xxx-xxxx-xxxx                     │
│  メール: name@company.co.jp              │
└──────────────────────────────────────────┘
```

### 5.3 필수 비즈니스 문구

| 일본어 | 한국어 | 용도 |
|----------|---------|-------|
| いつもお世話になっております。 | 항상 신세를 지고 있습니다. | 이메일 서두 |
| ご連絡いただきありがとうございます。 | 연락 주셔서 감사합니다. | 답장 서두 |
| ご確認のほど、よろしくお願いいたします。 | 확인 부탁드립니다. | 요청 |
| ご検討のほど、よろしくお願いいたします。 | 검토 부탁드립니다. | 제안 |
| 以上、ご報告申し上げます。 | 이상 보고드립니다. | 보고서 맺음 |
| お忙しいところ恐れ入りますが... | 바쁘신 와중에 죄송합니다만... | 공손한 서두 |
| ご不明な点がございましたら... | 불분명한 점이 있으시면... | 도움 제안 |
| 何卒よろしくお願い申し上げます。 | 무엇이든 잘 부탁드립니다. | 매우 격식적 맺음말 |

### 5.4 비즈니스 문서에서의 경어 (敬語)

비즈니스 문서에서는 세 가지 수준의 경어(敬語)를 사용합니다:

| 유형 | 일본어 | 기능 | 예시 |
|------|----------|----------|---------|
| 尊敬語 (존경어) | そんけいご | 상대방을 높임 | ご覧になる (~을 보시다) |
| 謙譲語 (겸양어) | けんじょうご | 자신을 낮춤 | 拝見する (~을 뵙다/보다) |
| 丁寧語 (정중어) | ていねいご | 정중한 표현 | ございます (~이 있습니다) |

## 6. 훑어읽기 vs. 정독

### 6.1 훑어읽기 (拾い読み — ひろいよみ)

**목적:** 특정 정보를 빠르게 찾기.

**사용 시점:**
- 날짜, 이름, 숫자, 키워드를 찾을 때
- 구체적인 사실 관련 질문에 답할 때
- 긴 문서를 검색할 때

**일본어 텍스트 훑어읽기 방법:**

1. 찾고자 하는 정보를 특정합니다 (날짜? 이름? 숫자?)
2. 시각적 단서를 찾습니다:
   - **숫자:** 数字나 漢数字가 시각적으로 눈에 띕니다
   - **이름:** 인명은 보통 한자로 쓰이며 후리가나가 있는 경우가 많습니다
   - **가타카나:** 외래어와 고유명사가 즉시 눈에 들어옵니다
   - **인용부호:** 「」가 직접 인용과 핵심 용어를 표시합니다
3. 텍스트를 **Z패턴**으로 시선을 이동합니다
4. 모든 단어를 읽지 말고 — 한자 묶음 사이로 시선을 이동하십시오

### 6.2 정독 (精読 — せいどく)

**목적:** 문장의 모든 세부 사항을 이해하기.

**사용 시점:**
- 저자의 논지를 분석할 때
- 문맥 속에서 문법을 학습할 때
- 독해 시험을 준비할 때
- 계약서나 법률 문서를 읽을 때

**일본어 텍스트 정독 방법:**

1. 문장 처음부터 동사 끝까지 전체를 읽습니다
2. 문법 구조를 분석합니다: 주제 + [수식절] + 주술부
3. 조사 사용법을 확인합니다 (は vs. が, に vs. で 등)
4. 모든 접속사와 그 논리적 기능을 파악합니다
5. 완전히 이해한 후에 다음 문장으로 넘어갑니다

### 6.3 전략 선택하기

```
+-----------------------------------------------------+
|  질문 유형            →  전략                         |
+-----------------------------------------------------+
|  "X는 언제 일어났나?"  →  날짜/時를 훑어읽기          |
|  "누가 X라고 했나?"    →  「」와 이름 훑어읽기        |
|  "핵심 내용은?"        →  서론과 결론 정독             |
|  "저자가 왜 X를       →  転(전환) 부분 정독            |
|   주장하는가?"                                        |
|  "암시된 내용은?"      →  문맥 단서 정독               |
+-----------------------------------------------------+
```

## 7. 주요 독해 문제 유형

JLPT 및 학술 독해 시험에서 일반적으로 사용되는 문제 유형입니다:

### 7.1 주제 파악 문제 (主旨問題)

**표지 문구:** 「この文章の主旨は何ですか」「筆者が最も言いたいことは何ですか」

전략: 결론(結) 부분에 집중합니다. 저자의 핵심 주장은 보통 글의 끝 부분에서 진술되거나 재진술됩니다.

### 7.2 세부 사항 문제 (詳細問題)

**표지 문구:** 「～について、正しいものはどれですか」「～とは何ですか」

전략: 관련 키워드를 훑어읽기로 찾은 후, 주변 문장을 정독합니다.

### 7.3 추론 문제 (推論問題)

**표지 문구:** 「～から何がわかりますか」「筆者はどう考えていますか」

전략: 의견 표지(と思う, べきだ, ではないだろうか)를 찾습니다. 직접 진술되지 않은 암시된 내용을 고려합니다.

### 7.4 이유 문제 (理由問題)

**표지 문구:** 「なぜ～ですか」「～の理由は何ですか」

전략: 언급된 내용 근처에서 인과 접속사(なぜなら, から, ため, ので)를 찾습니다.

### 7.5 지시어 문제 (指示語問題)

**표지 문구:** 「『それ』は何を指していますか」「『この問題』とは何ですか」

전략: 지시어(それ, これ, この, その) 바로 앞 문장을 살펴봅니다. 지시 대상은 거의 항상 앞선 문맥에 있습니다.

### 7.6 문맥 속 어휘 의미 문제 (語彙問題)

**표지 문구:** 「～はどういう意味ですか」「～に最も近い意味は...」

전략: 해당 단어가 포함된 전체 문장을 읽습니다. 주변 문맥과 한자 구성 요소를 활용하여 의미를 유추합니다.

## 8. 연습 텍스트 1: 뉴스 기사

다음 신문 기사를 읽고 질문에 답하십시오.

---

**AI技術の発展と社会への影響**

人工知能（AI）技術の急速な発展が、私たちの社会に大きな変化をもたらしている。総務省が発表した最新の報告書によると、日本国内でAIを導入している企業の割合は、2023年の35%から2025年には58%に増加した。

特に注目されているのは、医療分野でのAIの活用である。画像診断においては、AIの正確性が人間の医師と同等、あるいはそれ以上の成績を示すケースも報告されている。しかし、最終的な判断は必ず医師が行うべきだという意見も根強い。

一方、AI技術の普及に伴い、雇用への影響を懸念する声も高まっている。専門家は、「AIに代替されやすい職種がある一方で、AIを活用できる人材の需要は今後も増え続けるだろう」と指摘している。

---

### 질문:

1. 日本国内でAIを導入している企業の割合は、2年間でどれくらい増えましたか。
2. 医療分野でのAI活用について、どのような議論がありますか。
3. 雇用への影響について、専門家はどのように考えていますか。

<details>
<summary>정답</summary>

1. 2023年の35%から2025年の58%に増え、**23포인트 증가**했습니다.
2. AI의 화상 진단 정확성은 의사와 동등하거나 그 이상의 성적을 보이는 사례도 있지만, **최종적인 판단은 의사가 해야 한다**는 의견도 뿌리 깊습니다. 즉, AI의 능력은 인정되지만 인간의 판단의 중요성도 주장되고 있습니다.
3. 전문가는 AI에 의해 대체되기 쉬운 직종이 있는 한편, **AI를 활용할 수 있는 인재의 수요는 앞으로도 계속 늘어날 것**이라고 보고 있습니다. 즉, 일부 직종은 영향을 받지만 새로운 수요도 생긴다는 견해입니다.

</details>

## 9. 연습 텍스트 2: 논설문

다음 논설문 발췌를 읽고 질문에 답하십시오.

---

**「読書離れ」は本当に問題か**

近年、「若者の読書離れ」が社会問題として取り上げられることが多い。確かに、紙の書籍の売り上げは年々減少している。しかし、これをもって「読書離れ」と結論づけるのは早計ではないだろうか。

現代の若者は、スマートフォンやタブレットを通じて、かつてないほど多くのテキストに触れている。SNSの投稿、ニュースアプリの記事、電子書籍など、文字を読む機会はむしろ増えているとも言える。問題は「読む量」ではなく、「何を、どのように読むか」という質の問題なのではないか。

つまり、「読書離れ」という表現自体が、旧来の読書形態にとらわれた見方であり、デジタル時代における新しい「読書」の定義を考え直す必要があるのではないだろうか。

---

### 질문:

1. 「読書離れ」の根拠として挙げられている事実は何ですか。
2. 筆者は「読書離れ」についてどのような立場ですか。
3. 「ではないだろうか」が2回使われていますが、これはどのような効果がありますか。

<details>
<summary>정답</summary>

1. **종이 서적의 매출이 해마다 감소하고 있다는 것**이 근거로 제시되어 있습니다.
2. 필자는 「読書離れ」라고 단정할 수 없다는 입장입니다. 디지털 매체를 통한 독서 기회는 오히려 늘어나고 있으며, **문제는 양이 아니라 질이라고** 주장하고 있습니다. 또한 **「読書」의 정의 자체를 재검토해야 한다고** 제안하고 있습니다.
3. 「ではないだろうか」는 **수사적 의문문**(반어)으로, 「～と思う」보다 부드럽게 자신의 의견을 제시하는 효과가 있습니다. 직접적인 주장을 피하면서도 독자에게 동의를 유도하는 일본어 특유의 논설 기법입니다.

</details>

## 10. 연습 텍스트 3: 비즈니스 이메일

다음 비즈니스 이메일을 읽고 질문에 답하십시오.

---

件名: 来月の研修プログラムについて

山田部長

いつもお世話になっております。
人事部の佐藤でございます。

来月開催予定の新入社員研修プログラムについてご連絡いたします。

日時: 4月10日（月）～ 4月14日（金） 9:00～17:00
場所: 本社5階 大会議室
対象: 2026年度新入社員（15名）
講師: 外部講師2名 + 社内講師3名

つきましては、山田部長に4月12日（水）の午後の講義（14:00～16:00）をご担当いただけないかと存じます。テーマは「プロジェクト管理の基礎」を予定しております。

ご多忙のところ恐れ入りますが、3月15日までにご回答いただけますと幸いです。ご不明な点がございましたら、お気軽にお問い合わせください。

何卒よろしくお願い申し上げます。

人事部 佐藤花子
内線: 3456
メール: sato.hanako@company.co.jp

---

### 질문:

1. この文書の目的は何ですか。
2. 山田部長に依頼されている具体的な内容は何ですか。
3. 「ご担当いただけないかと存じます」はどのような表現ですか。なぜ直接的に「担当してください」と言わないのですか。

<details>
<summary>정답</summary>

1. 신입 사원 연수 프로그램의 안내와 **야마다 부장에게 강의 담당을 의뢰하는 것**이 목적입니다.
2. **4월 12일(수) 14:00~16:00에 「프로젝트 관리의 기초」라는 주제로 강의를 진행하는 것**이 의뢰되고 있습니다.
3. 「ご担当いただけないかと存じます」는 **겸양어를 사용한 간접적 의뢰 표현**입니다. 「いただけないか」는 「もらえないか」의 겸양어이며, 「と存じます」(「と思います」의 겸양어)를 더함으로써 명령이 아닌 의뢰의 뉘앙스를 나타내고 있습니다. 일본의 비즈니스 문화에서는 윗사람에게 의뢰할 때 직접적인 표현을 피하고 상대방에게 거절할 여지를 남기는 것이 예의입니다.

</details>

## 11. 연습 문제

### 연습 1: 접속사 식별

각 문장에서 접속사를 찾고 어떤 관계를 나타내는지 설명하십시오.

1. 日本語は難しい。**しかし**、とても面白い言語だ。
2. 気温が上がった。**そのため**、桜が早く咲いた。
3. 彼は成績が良い。**さらに**、スポーツも得意だ。
4. 東京は便利だ。**一方**、家賃が高い。
5. **つまり**、この計画は現実的ではない。

<details>
<summary>정답</summary>

1. **しかし** — 대비: 두 번째 문장이 반대되는 내용을 제시
2. **そのため** — 인과: 두 번째 문장이 첫 번째의 결과
3. **さらに** — 추가: 두 번째 문장이 추가 정보를 제공
4. **一方** — 비교: 다른 측면을 제시
5. **つまり** — 요약/바꿔 말하기: 결론을 간단하게 재진술

</details>

### 연습 2: 문제 유형 분류

다음 각각은 어떤 유형의 독해 문제입니까?

1. この文章で筆者が最も言いたいことは何ですか。
2. 「その問題」とは何を指していますか。
3. なぜ著者は反対の立場をとっていますか。
4. 2025年の売上高はいくらですか。
5. 「画期的」に最も近い意味はどれですか。

<details>
<summary>정답</summary>

1. **主旨問題** (주제 파악 문제) — 「最も言いたいこと」가 핵심 주장을 묻는 표지
2. **指示語問題** (지시어 문제) — 「何を指していますか」가 지시어의 지시 대상을 묻고 있음
3. **理由問題** (이유 문제) — 「なぜ」가 이유를 묻고 있음
4. **詳細問題** (세부 사항 문제) — 구체적인 사실 정보를 묻고 있음
5. **語彙問題** (문맥 속 어휘 의미 문제) — 「最も近い意味」가 가장 가까운 의미를 묻고 있음

</details>

### 연습 3: 훑어읽기 연습

다음 텍스트에는 구체적인 정보가 포함되어 있습니다. 훑어읽기 기법을 사용하여 가능한 한 빠르게 각 질문에 답하십시오 — 모든 단어를 읽을 필요는 없습니다.

---

東京スカイツリーは2012年5月22日に開業した。高さは634メートルで、自立式電波塔としては世界一である。建設費は約650億円で、工期は約3年8ヶ月であった。年間来場者数は約600万人に達し、展望台は350メートルと450メートルの2カ所にある。最寄り駅は東武スカイツリーライン「とうきょうスカイツリー駅」と、半蔵門線「押上駅」である。

---

1. スカイツリーの高さは？
2. 開業日はいつ？
3. 展望台は何メートルの場所にある？
4. 最寄り駅の路線名を2つ挙げてください。

<details>
<summary>정답</summary>

1. **634メートル**
2. **2012年5月22日**
3. **350メートルと450メートル**
4. **東武スカイツリーライン**과 **半蔵門線**

</details>

### 연습 4: 뉴스 헤드라인 해석

다음 축약된 신문 헤드라인을 완전한 문장으로 전개하십시오.

1. 首相、来月訪中へ
2. 大型台風接近、交通に影響も
3. 新薬承認、年内にも
4. GDP3期連続プラス

<details>
<summary>정답</summary>

1. 首相**は**来月**に**中国**を**訪問**する予定だ**。 (총리는 다음 달 중국을 방문할 예정이다.)
2. 大型**の**台風**が**接近**しており**、交通**に**影響**が出る可能性もある**。 (대형 태풍이 접근하고 있으며, 교통에 영향이 나올 가능성도 있다.)
3. 新薬**が**承認**される見通しで**、年内**にも承認される可能性がある**。 (신약이 승인될 전망이며, 연내에도 승인될 가능성이 있다.)
4. GDP**は**3期連続**で**プラス**成長を記録した**。 (GDP는 3분기 연속 플러스 성장을 기록했다.)

</details>
