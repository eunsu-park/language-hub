## 2. 레스토랑에서

### 2.1 필수 레스토랑 표현

```
いらっしゃいませ。
Irasshaimase.
어서 오세요! (직원이 하는 말)

何名様ですか。
Nan-mei-sama desu ka.
몇 분이십니까?

二人です。
Futari desu.
두 명입니다.

メニューをお願いします。
Menyū o onegai shimasu.
메뉴 부탁합니다.

注文をお願いします。
Chūmon o onegai shimasu.
주문 부탁합니다.

これをください。
Kore o kudasai.
이것으로 주세요.

お会計をお願いします。
Okaikei o onegai shimasu.
계산 부탁합니다.
```

### 2.2 주문 어휘

| 일본어 | 로마자 | 의미 |
|----------|--------|---------|
| 注文 | chūmon | 주문 |
| メニュー | menyū | 메뉴 |
| 定食 | teishoku | 정식 (세트 메뉴) |
| おすすめ | osusume | 추천 |
| 大盛り | ōmori | 곱빼기 |
| お会計 | okaikei | 계산서 |
| お持ち帰り | omochikaeri | 포장 |
| 別々に | betsubetsu ni | 따로따로 (각자 계산) |

### 2.3 대화 예시

```
店員: いらっしゃいませ。何名様ですか。
      Irasshaimase. Nan-mei-sama desu ka.

客:   二人です。
      Futari desu.

店員: こちらへどうぞ。メニューです。
      Kochira e dōzo. Menyū desu.

客:   すみません、注文をお願いします。
      Sumimasen, chūmon o onegai shimasu.
      カレーを一つと、ラーメンを一つください。
      Karē o hitotsu to, rāmen o hitotsu kudasai.

店員: お飲み物は？
      Onomimono wa?

客:   お茶を二つお願いします。
      Ocha o futatsu onegai shimasu.

店員: かしこまりました。
      Kashikomarimashita. (알겠습니다.)
```

## 5. 가족 호칭

일본어에는 두 세트의 가족 호칭이 있습니다: **자기 가족**용(겸양)과 **상대방 가족**용(존경). 이 구별은 예의 바른 대화에 필수적입니다.

### 5.1 가족 호칭 비교표

| 관계 | 자기 가족 (겸양) | 상대방 가족 (존경) |
|-------------|---------------------|-------------------------------|
| 아버지 | 父 (chichi) | お父さん (otōsan) |
| 어머니 | 母 (haha) | お母さん (okāsan) |
| 형/오빠 | 兄 (ani) | お兄さん (oniisan) |
| 누나/언니 | 姉 (ane) | お姉さん (onēsan) |
| 남동생 | 弟 (otōto) | 弟さん (otōto-san) |
| 여동생 | 妹 (imōto) | 妹さん (imōto-san) |
| 남편 | 夫 / 主人 (otto / shujin) | ご主人 (goshujin) |
| 아내 | 妻 / 家内 (tsuma / kanai) | 奥さん (okusan) |
| 아이 | 子供 (kodomo) | お子さん (okosan) |
| 조부모 | 祖父/祖母 (sofu/sobo) | おじいさん/おばあさん |
| 가족 | 家族 (kazoku) | ご家族 (gokazoku) |

### 5.2 언제 어떤 것을 사용하는가

> **자기 가족 vs 상대방 가족**
>
> 자기 가족에 대해 다른 사람에게 말할 때:
> 「母は先生です。」
> Haha wa sensei desu.
> 저희 어머니는 선생님입니다.
>
> 상대방 가족에 대해 물을 때:
> 「お母さんはお元気ですか。」
> Okāsan wa ogenki desu ka.
> 어머님은 건강하십니까?
>
> 자기 가족에게 직접 말할 때 (집에서):
> 「お母さん、ご飯まだ？」
> Okāsan, gohan mada?
> 엄마, 밥 아직이야?
> (직접 부를 때는 존경형을 사용)
>
> - **규칙**: 외부인에게 자기 가족 이야기 → 겸양형.
> 상대방 가족을 가리킬 때 → 존경형.
> 자기 가족에게 직접 부를 때 → 존경형.

### 5.3 예문

```
私の家族は四人です。父と母と姉と私です。
Watashi no kazoku wa yonin desu. Chichi to haha to ane to watashi desu.
저희 가족은 네 명입니다. 아버지, 어머니, 누나, 그리고 저입니다.

ご家族は何人ですか。
Gokazoku wa nannin desu ka.
가족이 몇 분이십니까?

兄はアメリカに住んでいます。
Ani wa Amerika ni sunde imasu.
형은 미국에 살고 있습니다.

お姉さんは何をしていますか。
Onēsan wa nani o shite imasu ka.
누나(언니)는 무엇을 하고 계십니까?
```

```
六時に帰ります。
Roku-ji ni kaerimasu.
6시에 돌아옵니다.

晩ご飯を作ります。
Bangohan o tsukurimasu.
저녁밥을 만듭니다.

お風呂に入ります。
Ofuro ni hairimasu.
목욕합니다.

本を読みます。
Hon o yomimasu.
책을 읽습니다.

十一時に寝ます。
Jūichi-ji ni nemasu.
11시에 잡니다.
```

### 7.3 て형을 활용한 하루 전체

```
朝七時に起きて、顔を洗って、朝ご飯を食べて、八時に家を出ます。
電車で会社に行って、五時まで働いて、六時に帰ります。
晩ご飯を食べて、お風呂に入って、テレビを見て、十一時に寝ます。

Asa shichi-ji ni okite, kao o aratte, asagohan o tabete, hachi-ji ni ie o demasu.
Densha de kaisha ni itte, go-ji made hataraite, roku-ji ni kaerimasu.
Bangohan o tabete, ofuro ni haitte, terebi o mite, jūichi-ji ni nemasu.
```

## 8. 일상적으로 자주 쓰이는 표현

| いただきます | itadakimasu | 잘 먹겠습니다 | 식사 전 |
| ごちそうさまでした | gochisōsama deshita | 잘 먹었습니다 | 식사 후 |
| いってきます | ittekimasu | 다녀오겠습니다 | 집을 나설 때 |

| ただいま | tadaima | 다녀왔습니다 | 집에 도착할 때 |
| おかえりなさい | okaerinasai | 어서 오세요 (대답) | 돌아온 사람에게 |
| おやすみなさい | oyasuminasai | 안녕히 주무세요 | 잠자리에 들기 전 |
| お疲れさまです | otsukaresama desu | 수고하셨습니다 | 일/수고 후 |

### 8.2 유용한 일상 표현

## 9. 일상생활의 조수사

일본어는 세는 대상에 따라 특정한 **조수사**를 사용합니다. 일상 상황에서 가장 필수적인 것들을 소개합니다.

### 9.1 일반 사물 조수사 (つ)

특정 조수사가 없을 때 일반 사물에 사용합니다.

| 숫자 | 조수사 | 로마자 |
|--------|---------|--------|
| 1 | 一つ | hitotsu |
| 2 | 二つ | futatsu |
| 3 | 三つ | mittsu |
| 4 | 四つ | yottsu |
| 5 | 五つ | itsutsu |
| 6 | 六つ | muttsu |
| 7 | 七つ | nanatsu |
| 8 | 八つ | yattsu |
| 9 | 九つ | kokonotsu |
| 10 | 十 | tō |

### 9.2 사람 조수사 (人)

| 숫자 | 조수사 | 로마자 |
|--------|---------|--------|
| 1명 | 一人 | hitori |
| 2명 | 二人 | futari |
| 3명 | 三人 | sannin |
| 4명 | 四人 | yonin |
| 5명 | 五人 | gonin |

### 9.3 기타 자주 쓰이는 조수사

| 조수사 | 용도 | 예시 |
|---------|----------|---------|
| 〜個 (ko) | 작은 물건 | りんごを三個 (사과 3개) |
| 〜本 (hon/bon/pon) | 길고 가는 물건, 병 | ペンを二本 (펜 2자루) |
| 〜枚 (mai) | 평평한 물건 | 紙を五枚 (종이 5장) |
| 〜杯 (hai/bai/pai) | 컵, 그릇 | コーヒーを一杯 (커피 1잔) |
| 〜台 (dai) | 기계, 탈것 | 車が三台 (자동차 3대) |
| 〜匹 (hiki/biki/piki) | 작은 동물 | 猫が二匹 (고양이 2마리) |
| 〜冊 (satsu) | 책, 잡지 | 本を四冊 (책 4권) |
| 〜円 (en) | 엔 (통화) | 五百円 (500엔) |

### 9.4 本/杯/匹의 음변 규칙

> **조수사 음변 규칙 (本・杯・匹)**
>
> 숫자 │  本 (hon)  │  杯 (hai)  │  匹 (hiki)
> ─────┼────────────┼────────────┼────────────
> 1    │ いっぽん   │ いっぱい   │ いっぴき
> 2    │ にほん     │ にはい     │ にひき
> 3    │ さんぼん   │ さんばい   │ さんびき
> 4    │ よんほん   │ よんはい   │ よんひき
> 5    │ ごほん     │ ごはい     │ ごひき
> 6    │ ろっぽん   │ ろっぱい   │ ろっぴき
> 7    │ ななほん   │ ななはい   │ ななひき
> 8    │ はっぽん   │ はっぱい   │ はっぴき
> 9    │ きゅうほん │ きゅうはい │ きゅうひき
> 10   │ じゅっぽん │ じゅっぱい │ じゅっぴき
>
> - **패턴**: 1, 6, 8, 10 → 촉음 + ぱ행
> 3 → ば행
> 나머지 → は행 (변화 없음)

## 10. 연습 문제

### 연습 1: 어휘 매칭

일본어 단어를 한국어 의미와 연결하세요.

```
1. 牛乳          a. 역
2. 冷蔵庫        b. 편의점
3. 駅            c. 우유
4. コンビニ      d. 세탁기
5. 洗濯機        e. 냉장고
6. 地下鉄        f. 형 (겸양형)
7. 兄            g. 지하철
8. 掃除          h. 청소
```

<details>
<summary>정답 보기</summary>

```
1-c (牛乳 = 우유)
2-e (冷蔵庫 = 냉장고)
3-a (駅 = 역)
4-b (コンビニ = 편의점)
5-d (洗濯機 = 세탁기)
6-g (地下鉄 = 지하철)
7-f (兄 = 형, 겸양형)
8-h (掃除 = 청소)
```

</details>

### 연습 2: 가족 호칭 — 올바른 형태 선택

상황에 맞게 겸양형 또는 존경형을 넣으세요.

```
1. (자기 어머니에 대해 직장 동료에게 말할 때)
   _________は看護師です。 (저희 어머니는 간호사입니다.)

2. (상대방 아버지에 대해 물을 때)
   _________はお元気ですか。 (아버님은 건강하십니까?)

3. (자기 누나/언니에 대해 말할 때)
   _________は大阪に住んでいます。 (누나는 오사카에 살고 있습니다.)

4. (상대방 자녀에 대해 물을 때)
   _________は何歳ですか。 (아이는 몇 살입니까?)
```

<details>
<summary>정답 보기</summary>

```
1. 母 (haha) — 자기 가족에 대해 이야기 → 겸양형
2. お父さん (otōsan) — 상대방 가족에 대해 물음 → 존경형
3. 姉 (ane) — 자기 가족에 대해 이야기 → 겸양형
4. お子さん (okosan) — 상대방 가족에 대해 물음 → 존경형
```

</details>

### 연습 3: 레스토랑에서

다음 레스토랑 대화를 올바른 순서로 배열하세요.

```
a. カレーを一つください。
b. いらっしゃいませ。何名様ですか。
c. お会計をお願いします。
d. 一人です。
e. メニューをお願いします。
f. こちらへどうぞ。
```

<details>
<summary>정답 보기</summary>

```
올바른 순서: b → d → f → e → a → c

b. いらっしゃいませ。何名様ですか。 (어서 오세요! 몇 분이십니까?)
d. 一人です。 (한 명입니다.)
f. こちらへどうぞ。 (이쪽으로 오세요.)
e. メニューをお願いします。 (메뉴 부탁합니다.)
a. カレーを一つください。 (카레 하나 주세요.)
c. お会計をお願いします。 (계산 부탁합니다.)
```

</details>

### 연습 4: 조수사

각 항목에 올바른 조수사를 넣으세요.

```
1. コーヒーを二___ください。 (커피 2잔)
2. りんごを三___買いました。 (사과 3개)
3. ペンが五___あります。 (펜 5자루)
4. 猫が二___います。 (고양이 2마리)
5. 本を四___借りました。 (책 4권)
6. 家族は五___です。 (가족 5명)
```

<details>
<summary>정답 보기</summary>

```
1. コーヒーを二杯(にはい)ください。 (杯 — 잔)
2. りんごを三個(さんこ)買いました。 (個 — 작은 물건)
3. ペンが五本(ごほん)あります。 (本 — 길고 가는 물건)
4. 猫が二匹(にひき)います。 (匹 — 작은 동물)
5. 本を四冊(よんさつ)借りました。 (冊 — 책)
6. 家族は五人(ごにん)です。 (人 — 사람)
```

</details>

### 연습 5: 일과 — 일본어로 번역

존댓말을 사용하여 하루 일과를 묘사하세요.

```
1. I wake up at 7:00.
2. I eat breakfast and leave home at 8:00.
3. I go to work by train.
4. I work from 9:00 to 5:00.
5. I come home and cook dinner.
6. I take a bath and go to sleep at 11:00.
```

<details>
<summary>정답 보기</summary>

```
1. 七時に起きます。
   (Shichi-ji ni okimasu.)

2. 朝ご飯を食べて、八時に家を出ます。
   (Asagohan o tabete, hachi-ji ni ie o demasu.)

3. 電車で会社に行きます。
   (Densha de kaisha ni ikimasu.)

4. 九時から五時まで働きます。
   (Ku-ji kara go-ji made hatarakimasu.)

5. 帰って、晩ご飯を作ります。
   (Kaette, bangohan o tsukurimasu.)

6. お風呂に入って、十一時に寝ます。
   (Ofuro ni haitte, jūichi-ji ni nemasu.)
```

</details>

### 연습 6: 쇼핑 역할극

다음 쇼핑 대화를 일본어로 번역하세요.

```
You:    Excuse me, how much is this?
Clerk:  That is 1,500 yen.
You:    It's a little expensive. Is there a cheaper one?
Clerk:  This one is 800 yen.
You:    Then, I'll take this one please.
Clerk:  Do you need a bag?
You:    No, I'm fine. Thank you.
```

<details>
<summary>정답 보기</summary>

```
You:    すみません、これはいくらですか。
        (Sumimasen, kore wa ikura desu ka.)
Clerk:  それは千五百円です。
        (Sore wa sen gohyaku en desu.)
You:    ちょっと高いですね。もっと安いのはありますか。
        (Chotto takai desu ne. Motto yasui no wa arimasu ka.)
Clerk:  こちらは八百円です。
        (Kochira wa happyaku en desu.)
You:    じゃあ、これをください。
        (Jā, kore o kudasai.)
Clerk:  袋はいりますか。
        (Fukuro wa irimasu ka.)
You:    いいえ、大丈夫です。ありがとうございます。
        (Iie, daijōbu desu. Arigatō gozaimasu.)
```

</details>
