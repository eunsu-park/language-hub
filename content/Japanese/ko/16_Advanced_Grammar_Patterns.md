## 1. ～ばかり — "방금 ~했다" / "~뿐"

～ばかり는 접속하는 형태에 따라 두 가지 다른 의미를 가집니다.

### 1.1 의미 A: "방금 ~했다" (たばかり)

어떤 행동이 **최근에 완료**되었음을 나타냅니다. 그 결과나 상태가 아직 생생합니다.

**접속 방법:**

```
동사 (た형) + ばかり + だ / です
```

| 일본어 | 한국어 |
|----------|---------|
| 日本に来た**ばかり**です。 | 일본에 막 왔습니다. |
| さっき食べた**ばかり**なのに、もうお腹がすいた。 | 방금 먹었는데 벌써 배가 고프다. |
| 新しいパソコンを買った**ばかり**だ。 | 새 컴퓨터를 막 샀다. |
| 起きた**ばかり**で、まだ眠い。 | 방금 일어나서 아직 졸립다. |

> **참고:** 행동과 현재 사이의 시간 간격은 주관적입니다. 몇 시간이나 며칠 전의 일이라도
> 화자가 최근이라고 느끼면 たばかり로 표현할 수 있습니다.

### 1.2 의미 B: "~뿐 / ~만" (ばかり)

한 가지만 하거나 한 가지만 있는 것을 강조합니다 — **~만 하다**, **~뿐이다**라는 배타성이나 과도함을 나타냅니다.

**접속 방법:**

```
동사 (て형) + ばかりいる         → ~만 하고 있다
명사 + ばかり                   → ~뿐
동사 (사전형) + ばかり           → ~만 하다
```

| 일본어 | 한국어 |
|----------|---------|
| 彼はゲームをして**ばかり**いる。 | 그는 게임만 하고 있다. |
| 甘いもの**ばかり**食べている。 | 단것만 먹고 있다. |
| 文句**ばかり**言うな。 | 불평만 하지 마. |
| 最近、悪いこと**ばかり**起きる。 | 요즘 나쁜 일만 일어난다. |

### 1.3 핵심 구별

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| Vた + ばかり | "방금 ~했다" | 始めたばかり (방금 시작했다) |
| Vて + ばかりいる | "~만 하다" | 遊んでばかりいる (놀기만 한다) |
| 명사 + ばかり | "~뿐 / ~만" | 嘘ばかり (거짓말뿐) |

## 2. ～っぽい — "~같은 / ~스러운"

명사, 동사, 형용사에 접속하여 경향, 유사성, 인상을 나타냅니다.

**접속 방법:**

```
명사 + っぽい      → "~같은", "~스러운"
동사 어간 + っぽい  → "~하는 경향이 있다"
い형용사 어간 + っぽい → "약간 ~한"
```

> **활용:** っぽい는 い형용사로 활용됩니다:
> っぽくない (부정), っぽかった (과거), っぽくて (て형)

### 2.1 명사와 함께 — "~같은, ~스러운"

| 일본어 | 한국어 |
|----------|---------|
| 子供**っぽい** | 유치한, 어린애 같은 |
| 男**っぽい** | 남자다운 |
| 女**っぽい** | 여성스러운 |
| 水**っぽい** | 물 같은 (예: 너무 묽은 국) |
| 埃（ほこり）**っぽい** | 먼지투성이인 |
| 安**っぽい** | 싸구려 같은 |

### 2.2 동사 어간과 함께 — "~하는 경향이 있다"

| 일본어 | 한국어 |
|----------|---------|
| 忘れ**っぽい** | 잘 잊어버리는 (건망증이 있는) |
| 怒り**っぽい** | 화를 잘 내는 (성질이 급한) |
| 飽き**っぽい** | 싫증을 잘 내는 (변덕스러운) |
| 惚れ**っぽい** | 쉽게 반하는 |

### 2.3 형용사 어간과 함께

| 일본어 | 한국어 |
|----------|---------|
| 白**っぽい** | 허얘 보이는 |
| 黒**っぽい** | 거무스름한 |
| 赤**っぽい** | 불그스름한 |

### 2.4 っぽい vs. らしい vs. みたい

| 패턴 | 뉘앙스 | 예시 |
|---------|--------|---------|
| っぽい | 해당 성질을 가진; 주관적 인상 | 子供っぽい (유치한 — 부정적) |
| らしい | ~에 어울리는, ~다운; 또는 전문(傳聞) | 子供らしい (아이다운 — 긍정적) |
| みたい | ~처럼 보이는, ~인 것 같은 | 子供みたい (아이 같은 — 중립적) |

> **子供っぽい** = 미숙하게 행동하는 (부정적 뉘앙스).
> **子供らしい** = 아이답게 자연스럽게 행동하는 (긍정적 뉘앙스).
> **子供みたい** = 아이와 닮은 (중립적 관찰).

## 3. ～がち — "~하기 쉽다"

**부정적 경향**이나 **바람직하지 않게 자주 일어나는 일**을 나타냅니다.

**접속 방법:**

```
동사 (ます 어간) + がち
명사 + がち
```

> **활용:** がち는 な형용사처럼 활용됩니다:
> がちな + 명사, がちに + 동사, がちだ / がちです

### 3.1 일반적인 예시

| 일본어 | 한국어 |
|----------|---------|
| 休み**がち** | 결석하기 쉬운 |
| 忘れ**がち** | 잊기 쉬운 |
| 遅れ**がち** | 늦기 쉬운 |
| 曇り**がち** | 흐리기 쉬운 |
| 病気**がち** | 병에 걸리기 쉬운 |
| 留守**がち** | 집을 자주 비우는 |

### 3.2 예문

| 일본어 | 한국어 |
|----------|---------|
| 最近、彼は学校を休み**がち**だ。 | 요즘 그는 학교를 자주 결석한다. |
| 冬は風邪をひき**がち**です。 | 겨울에는 감기에 걸리기 쉽습니다. |
| 忙しいと、食事を抜き**がち**になる。 | 바쁘면 식사를 거르기 쉬워진다. |
| この地域は曇り**がち**な天気が続く。 | 이 지역은 흐린 날씨가 계속되기 쉽다. |
| 一人暮らしは栄養が偏り**がち**だ。 | 혼자 살면 영양이 편향되기 쉽다. |

### 3.3 がち vs. っぽい

| 패턴 | 뉘앙스 |
|---------|--------|
| がち | 습관적인 부정적 경향; 객관적 관찰 |
| っぽい | 성질 / 인상; 주관적 느낌 |

- 忘れ**がち** = 잊는 경향이 있다 (실제로 자주 일어남)
- 忘れ**っぽい** = 건망증이 있다 (성격적 특성 / 인상)

## 4. ～気味 — "약간 / 다소"

무언가의 **가벼운 정도**를 나타내며, 보통 바람직하지 않은 상태에 사용됩니다.

**접속 방법:**

```
동사 (ます 어간) + 気味（ぎみ）
명사 + 気味
```

> 気味는 접미사로 사용될 때 **ぎみ** (きみ가 아님)로 읽습니다.

### 4.1 일반적인 예시

| 일본어 | 한국어 |
|----------|---------|
| 風邪**気味** | 감기 기운이 있는 |
| 太り**気味** | 약간 살이 찐 |
| 疲れ**気味** | 약간 피곤한 |
| 遅れ**気味** | 다소 늦어지고 있는 |
| 下がり**気味** | 약간 하락 추세인 |

### 4.2 예문

| 일본어 | 한국어 |
|----------|---------|
| 今日は風邪**気味**なので、早く帰ります。 | 오늘은 감기 기운이 있어서 일찍 들어갑니다. |
| 最近、太り**気味**だから、運動を始めた。 | 요즘 약간 살이 쪄서 운동을 시작했다. |
| プロジェクトが遅れ**気味**です。 | 프로젝트가 다소 지연되고 있습니다. |
| 彼は最近、疲れ**気味**に見える。 | 그는 요즘 약간 피곤해 보인다. |
| 株価は下がり**気味**だ。 | 주가가 약간 하락 추세이다. |

### 4.3 気味 vs. がち

| 패턴 | 정도 | 예시 |
|---------|--------|---------|
| 気味 | 가볍고 약한 | 風邪気味 (감기 기운) |
| がち | 습관적 경향 | 病気がち (병에 자주 걸리는) |

## 5. ～ずに — "~하지 않고"

다른 행동을 **하지 않고** 무언가를 하는 것을 나타냅니다. ～ないで의 문어체/격식체 표현입니다.

**접속 방법:**

```
동사 (ない 어간) + ずに
예외: する → せずに
```

### 5.1 활용

| 사전형 | ない형 | ～ずに형 |
|----------------|-----------|------------|
| 食べる (먹다) | 食べない | 食べ**ずに** |
| 行く (가다) | 行かない | 行か**ずに** |
| 読む (읽다) | 読まない | 読ま**ずに** |
| する (하다) | しない | **せずに** |
| 見る (보다) | 見ない | 見**ずに** |
| 寝る (자다) | 寝ない | 寝**ずに** |

> **중요:** する는 **せずに**가 되며, しずに가 아닙니다. 가장 흔한 불규칙 변화입니다.

### 5.2 예문

| 일본어 | 한국어 |
|----------|---------|
| 朝ご飯を食べ**ずに**学校へ行った。 | 아침을 먹지 않고 학교에 갔다. |
| 辞書を使わ**ずに**読めた。 | 사전을 사용하지 않고 읽을 수 있었다. |
| 何も言わ**ずに**帰った。 | 아무 말도 하지 않고 돌아갔다. |
| 予習**せずに**授業に出た。 | 예습하지 않고 수업에 나갔다. |
| 諦め**ずに**頑張ろう。 | 포기하지 않고 힘내자. |

### 5.3 ～ずに vs. ～ないで

| 패턴 | 문체 | 예시 |
|---------|----------|---------|
| ～ずに | 문어체 / 격식체 | 食べずに出かけた |
| ～ないで | 구어체 / 비격식체 | 食べないで出かけた |

둘 다 같은 의미이지만, ～ずに는 문어체, 신문, 격식체 연설에서 선호됩니다.

## 6. ～にもかかわらず — "~에도 불구하고"

**"~에도 불구하고"**를 의미하는 격식체 표현입니다. 두 사실 사이의 **모순**을 강조합니다.

**접속 방법:**

```
동사 (보통형)         + にもかかわらず
い형용사              + にもかかわらず
な형용사 + な/である   + にもかかわらず
명사 + である/な       + にもかかわらず
```

### 6.1 예문

| 일본어 | 한국어 |
|----------|---------|
| 雨が降っている**にもかかわらず**、試合は続行された。 | 비가 내리고 있음에도 불구하고 경기는 계속되었다. |
| 何度も練習した**にもかかわらず**、うまくできなかった。 | 여러 번 연습했음에도 불구하고 잘하지 못했다. |
| 彼は若い**にもかかわらず**、とても落ち着いている。 | 그는 젊음에도 불구하고 매우 침착하다. |
| 高価である**にもかかわらず**、よく売れている。 | 고가임에도 불구하고 잘 팔리고 있다. |
| 休日**にもかかわらず**、出社しなければならない。 | 휴일임에도 불구하고 출근해야 한다. |

### 6.2 관련 표현

| 패턴 | 격식도 | 뉘앙스 |
|---------|-----------|--------|
| にもかかわらず | 매우 격식체 / 문어체 | 강한 대비 |
| のに | 중립 / 구어체 | 실망이나 놀라움 |
| にしても | 보통 | "~을 고려하더라도" |
| くせに | 비격식체 / 비판적 | "~인 주제에 (비판적 어조)" |

> **にもかかわらず**는 주로 문어체에서 사용됩니다: 신문, 에세이, 보고서.
> 대화에서는 **のに**가 훨씬 자연스럽습니다.

## 7. ～に関して — "~에 관하여"

**"~에 관하여", "~에 대해"**를 의미하는 격식체 표현입니다. 격식적인 맥락에서 주제를 도입할 때 사용합니다.

**접속 방법:**

```
명사 + に関して          → 부사적 ("~에 관하여")
명사 + に関する + 명사   → 연체 수식 ("~에 관한 ~")
```

### 7.1 예문

| 일본어 | 한국어 |
|----------|---------|
| この問題**に関して**、ご質問はありますか。 | 이 문제에 관하여 질문이 있으십니까? |
| 環境問題**に関する**研究が増えている。 | 환경 문제에 관한 연구가 늘어나고 있다. |
| 日本文化**に関して**は、彼が詳しい。 | 일본 문화에 관해서는 그가 잘 알고 있다. |
| 個人情報**に関する**法律が改正された。 | 개인정보에 관한 법률이 개정되었다. |
| この件**に関して**は、後日連絡します。 | 이 건에 관해서는 추후 연락드리겠습니다. |

### 7.2 관련 주제 제시 표현

| 패턴 | 격식도 | 용법 |
|---------|-----------|-------|
| について | 일반적 | 일반적인 주제 도입 |
| に関して | 격식체 | 학술, 비즈니스, 공식적 |
| に対して | 격식체 | "~에 대해" — 행동이나 태도의 방향 |
| をめぐって | 격식체 | "~을 둘러싸고" — 논쟁적 주제 |

비교 예시:

| 일본어 | 한국어 |
|----------|---------|
| 日本の教育**について**話しましょう。 | 일본의 교육에 대해 이야기합시다. (중립적) |
| 日本の教育**に関して**調査した。 | 일본의 교육에 관하여 조사했다. (격식체) |
| 日本の教育**に対して**批判がある。 | 일본의 교육에 대한 비판이 있다. (방향적) |
| 日本の教育**をめぐって**議論が起きている。 | 일본의 교육을 둘러싸고 논의가 일어나고 있다. (논쟁적) |

## 8. ～を通じて — "~을 통해 / ~내내"

**수단** ("~을 통해") 또는 **기간** ("~내내")을 나타냅니다.

**접속 방법:**

```
명사 + を通じて / を通して
```

> を通じて와 を通して는 대체로 교환 가능하지만, を通じて가 약간 더 격식적입니다.

### 8.1 의미 A: 수단 / 매개

| 일본어 | 한국어 |
|----------|---------|
| インターネット**を通じて**情報を集めた。 | 인터넷을 통해 정보를 모았다. |
| 友人**を通じて**彼と知り合った。 | 친구를 통해 그와 알게 되었다. |
| 留学**を通じて**多くのことを学んだ。 | 유학을 통해 많은 것을 배웠다. |
| この経験**を通じて**成長できた。 | 이 경험을 통해 성장할 수 있었다. |

### 8.2 의미 B: 기간 / ~내내

| 일본어 | 한국어 |
|----------|---------|
| 一年**を通じて**温暖な気候だ。 | 1년 내내 온난한 기후이다. |
| 生涯**を通じて**研究を続けた。 | 일생을 통해 연구를 계속했다. |
| 四季**を通じて**楽しめる公園です。 | 사계절 내내 즐길 수 있는 공원입니다. |

### 8.3 を通じて vs. によって

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| を通じて | ~을 통해 (매개/기간) | 友人を通じて知った (친구를 통해 알았다) |
| によって | ~에 의해 / ~에 따라 | 方法によって結果が変わる (방법에 따라 결과가 달라진다) |

## 9. 비교 및 용법 정리

```
+--------------------------------------------------------------+
|          고급 문법 패턴 용법 정리                              |
+--------------------------------------------------------------+
|                                                              |
|  경향 / 성질:                                                |
|  ・っぽい → 인상, 성질 (~스러운)                               |
|  ・がち  → 부정적 습관적 경향                                  |
|  ・気味  → 가벼운/약한 정도                                    |
|                                                              |
|  대비 / 양보:                                                |
|  ・にもかかわらず → ~에도 불구하고 (격식체, 문어체)              |
|  ・のに          → ~인데도 (구어체, 감정적)                    |
|                                                              |
|  "~하지 않고":                                               |
|  ・ずに   → ~하지 않고 (문어체/격식체)                         |
|  ・ないで → ~하지 않고 (구어체/비격식체)                        |
|                                                              |
|  주제 제시:                                                   |
|  ・に関して → ~에 관하여 (격식체)                              |
|  ・について → ~에 대해 (중립적)                                |
|                                                              |
|  수단 / 기간:                                                |
|  ・を通じて → ~을 통해 / ~내내 (격식체)                        |
|  ・で      → ~으로 (중립적)                                   |
|                                                              |
|  최근성 / 배타성:                                             |
|  ・たばかり         → 방금 ~했다                               |
|  ・てばかりいる     → ~만 하다                                 |
+--------------------------------------------------------------+
```

## 10. 연습 문제

### 연습 1: 올바른 패턴 선택

빈칸에 적절한 문법 패턴을 넣으십시오.

1. 彼女はいつも遅刻して、怒り______人だ。 (그녀는 쉽게 화내는 타입이다)
2. 最近、疲れ______なので、休みを取りたい。 (요즘 좀 피곤하다)
3. このスープは水______。 (이 수프는 묽다)
4. 駅前の事故______、電車が遅れている。 (역 앞 사고에도 불구하고)
5. 朝ご飯を食べ______出かけた。 (아침을 먹지 않고 나갔다)

<details>
<summary>정답</summary>

1. **っぽい** — 怒りっぽい (화를 잘 내는; 성격적 특성)
2. **気味** — 疲れ気味 (약간 피곤한; 가벼운 정도)
3. **っぽい** — 水っぽい (물 같은; 성질/인상)
4. **にもかかわらず** 가 여기서 적절하지 않습니다; 이 문장은 대비가 아닌 원인을 설명합니다. 재해석하면: 事故**に関して**情報はありますか (사고에 관한 정보가 있습니까). 그러나 "사고 때문에"라는 맥락이라면 다른 패턴을 사용합니다. 만약 의도된 의미가 "사고에도 불구하고"라면: 事故**にもかかわらず** (사고에도 불구하고 전철이 지연되고 있다 — 운행될 것이라고 예상할 수 있다는 대비).
5. **ずに** — 食べずに (먹지 않고; 문어체)

</details>

### 연습 2: がち vs. 気味 vs. っぽい

각 문장에 가장 자연스러운 선택지를 고르십시오.

1. 冬は風邪をひき（がち / 気味 / っぽい）だ。 — 겨울에 습관적인 경향
2. 少し風邪（がち / 気味 / っぽい）です。 — 지금 약간 감기 기운
3. 彼は飽き（がち / 気味 / っぽい）性格だ。 — 성격적 특성
4. 最近、仕事が遅れ（がち / 気味 / っぽい）です。 — 다소 그런 상태
5. 彼は子供（がち / 気味 / っぽい）。 — 유치하게 행동한다

<details>
<summary>정답</summary>

1. **がち** — 습관적 경향: 風邪をひきがち
2. **気味** — 현재의 가벼운 정도: 風邪気味
3. **っぽい** — 성격적 인상: 飽きっぽい
4. **気味** — 현재의 약간의 상태: 遅れ気味
5. **っぽい** — 인상/성질: 子供っぽい

</details>

### 연습 3: ～ずに를 사용하여 다시 쓰기

비격식체 ～ないで 문장을 격식체 ～ずに 형태로 변환하십시오.

1. 何も食べないで寝た。
2. 辞書を使わないで翻訳した。
3. 準備しないで試験を受けた。
4. 誰にも相談しないで決めた。
5. 休まないで働き続けた。

<details>
<summary>정답</summary>

1. 何も食べ**ずに**寝た。 (아무것도 먹지 않고 잤다.)
2. 辞書を使わ**ずに**翻訳した。 (사전을 사용하지 않고 번역했다.)
3. 準備**せずに**試験を受けた。 (준비하지 않고 시험을 봤다.) — する → せずに
4. 誰にも相談**せずに**決めた。 (누구와도 상담하지 않고 결정했다.) — 相談する → 相談せずに
5. 休ま**ずに**働き続けた。 (쉬지 않고 계속 일했다.)

</details>

### 연습 4: 일본어로 번역하기

이 과에서 배운 문법 패턴을 사용하십시오.

1. 비쌈에도 불구하고 이 레스토랑은 항상 붐빈다.
2. 책을 통해 일본 역사에 대해 배웠다.
3. 일정 변경에 관해서는 사무실에 문의해 주십시오.
4. 그는 최근 도쿄로 이사했다. (최근에)
5. 그녀는 잘 잊어버리는 편이다.

<details>
<summary>정답</summary>

1. 高い**にもかかわらず**、このレストランはいつも混んでいる。
2. 本**を通じて**日本の歴史について学んだ。
3. スケジュールの変更**に関して**は、事務所にお問い合わせください。
4. 彼は東京に引っ越した**ばかり**だ。
5. 彼女は忘れ**っぽい**。 / 彼女は忘れ**がち**だ。

</details>

### 연습 5: 문장 작성

바쁜 직장인에 대한 짧은 문단(3~4문장)을 작성하되, 이 과에서 배운 패턴 중 최소 4가지를 포함하십시오. ばかり, がち, 気味, ずに, にもかかわらず를 활용해 보십시오.

<details>
<summary>모범 답안</summary>

田中さんは先月転職したばかりだ。新しい仕事が忙しくて、最近は疲れ気味に見える。昼ご飯を食べずに働くことが多く、食事を抜きがちだ。体調が悪いにもかかわらず、毎日遅くまで残業している。

(다나카 씨는 지난달 막 이직했다. 새 직장이 바빠서 요즘 약간 피곤해 보인다. 점심을 먹지 않고 일하는 경우가 많아 식사를 거르기 쉽다. 몸 상태가 좋지 않음에도 불구하고 매일 늦게까지 야근하고 있다.)

</details>
