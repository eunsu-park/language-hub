## 1. と思う — "~라고 생각하다"

### 접속 방법

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (보통체) | + と思う | 行く**と思う** (~가 갈 거라고 생각하다) |
| い형용사 | + と思う | 高い**と思う** (비싸다고 생각하다) |
| な형용사 | **だ** + と思う | 便利**だと思う** (편리하다고 생각하다) |
| 명사 | **だ** + と思う | 学生**だと思う** (학생이라고 생각하다) |
| 부정 | ないと思う | 来ない**と思う** (오지 않을 거라고 생각하다) |

### 주요 패턴

#### A. と思う vs. と思っている

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| と思う | 순간적 생각 / 의견 | 明日は雨だ**と思う**。(내일 비가 올 거라고 생각한다.) |
| と思っている | 지속적 신념 / 계속되는 의견 | 日本語は面白い**と思っている**。(일본어가 재미있다고 생각하고 있다. — 지속적 신념) |

> **중요**: 3인칭에는 と思っている를 사용합니다:
> ○ 田中さんは来ない**と思っている**。(다나카 씨는 오지 않을 거라고 생각하고 있다.)
> ✗ 田中さんは来ない**と思う**。(다나카가 올지에 대한 자신의 생각처럼 들림.)

#### B. と思う + 의지형 (계획/의도)

> **来年、日本に行こうと思う。** — 내년에 일본에 가려고 생각한다.
> **新しい車を買おうと思っている。** — 새 차를 사려고 생각하고 있다.

#### C. 의견 묻기: どう思いますか

> **この映画についてどう思いますか。** — 이 영화에 대해 어떻게 생각하십니까?
> **日本の食べ物はどう思う？** — 일본 음식에 대해 어떻게 생각해?

### 뉘앙스: と思う vs. と思わない

"X가 일어나지 않을 거라고 생각한다"는 영어와 다르게 표현됩니다:

```
영어:    I don't think he will come.
일본어:  彼は来ないと思う。        (그가 오지 않을 거라고 생각한다.)
         NOT: 彼は来ると思わない。   (덜 자연스러움)
```

> 일본어에서는 "생각하다" 자체를 부정하지 않고, 생각의 **내용**을 부정합니다.

### 예문

1. **あの人は日本人だと思う。** — 저 사람은 일본인이라고 생각한다.
2. **この問題は難しいと思います。** — 이 문제는 어렵다고 생각합니다.
3. **彼女はもう帰ったと思う。** — 그녀는 이미 돌아갔다고 생각한다.
4. **将来、教師になろうと思っています。** — 장래에 교사가 되려고 생각하고 있습니다.
5. **日本の電車は便利だと思いませんか。** — 일본 전철은 편리하다고 생각하지 않으십니까?

## 2. ようだ — "~인 것 같다 / ~처럼 보이다"

ようだ는 화자가 관찰한 **증거에 기반한 판단**을 표현합니다. 비교적 격식적입니다.

### 접속 방법

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (보통체) | + ようだ | 雨が降る**ようだ** |
| い형용사 | + ようだ | 難しい**ようだ** |
| な형용사 | **な** + ようだ | 静か**なようだ** |
| 명사 | **の** + ようだ | 風邪**のようだ** |

### 용법

#### A. 관찰에 기반한 판단

> **外は寒いようだ。**（窓の外を見て）
> 밖은 추운 것 같다. (창 밖을 보면서.)

> **彼は忙しいようです。** — 그는 바쁜 것 같습니다.

#### B. 비유 / 비교 ("~같은", "마치 ~처럼")

ような(명사 앞) 또는 ように(동사 앞)를 사용합니다:

> **雪のような白い肌。** — 눈 같이 하얀 피부.
> **子供のように笑った。** — 아이처럼 웃었다.
> **夢のような話だ。** — 꿈 같은 이야기다.

#### C. ように + 동사 (목적 / 양태)

> **聞こえるように大きな声で話した。** — 들리도록 큰 소리로 말했다.
> **忘れないようにメモした。** — 잊지 않도록 메모했다.

### ようだ의 활용

| 형태 | 일본어 |
|------|----------|
| 보통체 | ようだ |
| 정중체 | ようです |
| 부정 | ようではない |
| 과거 | ようだった |
| 부사적 (동사 앞) | ように |
| 연체형 (명사 앞) | ような |

### 예문

1. **道に迷ったようだ。** — 길을 잃은 것 같다.
2. **彼女は怒っているようです。** — 그녀는 화가 난 것 같습니다.
3. **まるで映画のような景色だ。** — 마치 영화 같은 풍경이다.
4. **日本語が上手になるように毎日練習している。** — 일본어가 잘 되도록 매일 연습하고 있다.
5. **試験に合格したようだ。** — 시험에 합격한 것 같다.

## 3. みたいだ — "~인 것 같다" (구어체)

みたいだ는 ようだ의 **구어체 표현**입니다. 같은 의미를 가지지만 일상 대화에서 사용됩니다.

### 접속 방법

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (보통체) | + みたいだ | 帰った**みたいだ** |
| い형용사 | + みたいだ | 寒い**みたいだ** |
| な형용사 | + みたいだ | 元気**みたいだ** |
| 명사 | + みたいだ | 学生**みたいだ** |

> **ようだ와의 주요 차이**: みたいだ는 바로 접속합니다 — の나 な 접속사가 필요 없습니다.

### 용법

#### A. 증거에 기반한 판단 (ようだ와 동일하지만 구어체)

> **雨が降っているみたいだ。** — 비가 오고 있는 것 같다.
> **彼はもう寝たみたいだよ。** — 그는 이미 잔 것 같아.

#### B. 비유 / 비교 ("~같은")

> **子供みたいに笑う。** — 아이처럼 웃다.
> **夢みたいだ！** — 꿈 같다!
> **本当のことみたいに聞こえる。** — 진짜인 것처럼 들린다.

### みたいだ의 활용

| 형태 | 일본어 |
|------|----------|
| 보통체 | みたいだ |
| 정중체 | みたいです |
| 부정 | みたいではない / みたいじゃない |
| 과거 | みたいだった |
| 동사 앞 | みたいに |
| 명사 앞 | みたいな |

### ようだ vs. みたいだ

| 특징 | ようだ | みたいだ |
|---------|--------|---------|
| 격식 | 격식적 / 서면체 | 구어적 / 회화체 |
| な형용사 접속 | 静かな + ようだ | 静か + みたいだ |
| 명사 접속 | 学生の + ようだ | 学生 + みたいだ |
| 의미 | 동일 | 동일 |

### 예문

1. **電車が遅れているみたいだ。** — 전철이 늦어지고 있는 것 같다.
2. **この料理、ちょっと辛いみたいだね。** — 이 요리, 좀 매운 것 같네.
3. **お姫様みたいなドレスだね。** — 공주님 같은 드레스네.
4. **田中さん、風邪みたいだよ。** — 다나카 씨, 감기인 것 같아.
5. **子供みたいにはしゃいでいた。** — 아이처럼 들떠 있었다.

## 4. らしい — "~인 것 같다 / ~라고 하다"

らしい는 화자가 **다른 사람에게서 들은** 정보나 **간접적 증거**를 표현합니다. 화자가 직접 관찰하는 것이 아니라 보고하는 것입니다.

### 접속 방법

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (보통체) | + らしい | 来る**らしい** |
| い형용사 | + らしい | 高い**らしい** |
| な형용사 | + らしい | 大変**らしい** |
| 명사 | + らしい | 本当**らしい** |

> みたいだ처럼 らしい도 모든 품사에 바로 접속합니다.

### 용법

#### A. 전문 / 간접 정보

> **明日は雨が降るらしい。** — 내일은 비가 온다고 한다. (누군가에게서 들었거나 / 일기 예보를 봤다.)
> **田中さんは来月結婚するらしい。** — 다나카 씨는 다음 달에 결혼한다고 한다.
> **あの店はおいしいらしいよ。** — 저 가게는 맛있다고 해.

#### B. 전형성: 명사 + らしい ("~답다", "~다운")

らしい가 명사에 직접 붙으면 "~에 어울리는" 또는 "~다운"이라는 의미입니다:

> **男らしい** — 남자답다
> **子供らしい** — 아이답다 (긍정적 의미)
> **春らしい天気** — 봄다운 날씨 (전형적인 봄 날씨)
> **学生らしくない** — 학생답지 않다 (학생으로서 마땅한 행동을 하지 않다)

> **みたいだ와의 대조**:
> 子供**らしい** = 아이답다 (긍정적: 아이에게 자연스러운, 기대되는 행동)
> 子供**みたいだ** = 아이 같다 (중립/부정적: 유치하다는 의미일 수 있음)

### らしい의 활용

| 형태 | 일본어 |
|------|----------|
| 보통체 | らしい |
| 정중체 | らしいです |
| 부정 | らしくない |
| 과거 | らしかった |

### 예문

1. **来週テストがあるらしい。** — 다음 주에 시험이 있는 것 같다.
2. **彼女は有名な歌手らしい。** — 그녀는 유명한 가수라고 한다.
3. **今日は春らしい天気ですね。** — 오늘은 봄다운 날씨네요.
4. **あの映画は面白くないらしいよ。** — 그 영화는 재미없다고 해.
5. **社長は厳しい人らしい。** — 사장님은 엄격한 사람이라고 한다.

## 5. そうだ — 두 가지 다른 의미

そうだ는 접속 방법에 따라 **완전히 다른 두 가지 의미**를 가집니다.

### 의미 1: 외관 (〜そうだ) — "~할 것 같다 / ~해 보이다"

**시각적 인상**이나 **직감**에 기반합니다 — 화자가 무언가를 보고 판단을 내립니다.

#### 접속 방법 (외관)

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 ます형 어간 | + そうだ | 降り**そうだ** (비가 올 것 같다) |
| い형용사 (い 제거) | + そうだ | おいし**そうだ** (맛있어 보인다) |
| な형용사 (어간) | + そうだ | 元気**そうだ** (건강해 보인다) |

> **특수한 경우**:
> いい → **よさそうだ** (좋아 보인다) — *いそうだ가 아님
> ない → **なさそうだ** (~없어 보인다) — *なそうだ가 아님

#### 예문 (외관)

> **このケーキはおいしそうだ。** — 이 케이크는 맛있어 보인다.
> **雨が降りそうだ。** — 비가 올 것 같다.
> **彼は元気そうだ。** — 그는 건강해 보인다.
> **難しそうな本だね。** — 어려워 보이는 책이네.
> **あの人は優しそうだ。** — 저 사람은 상냥해 보인다.

### 의미 2: 전문 (〜そうだ) — "~라고 하다"

화자가 출처(뉴스, 다른 사람 등)로부터 받은 **정보를 전달**합니다.

#### 접속 방법 (전문)

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (보통체) | + そうだ | 来る**そうだ** (온다고 한다) |
| い형용사 | + そうだ | 高い**そうだ** (비싸다고 한다) |
| な형용사 + **だ** | + そうだ | 便利**だそうだ** (편리하다고 한다) |
| 명사 + **だ** | + そうだ | 先生**だそうだ** (선생님이라고 한다) |

#### 예문 (전문)

> **明日は雨だそうだ。** — 내일은 비라고 한다.
> **あの映画は面白いそうです。** — 그 영화는 재미있다고 합니다.
> **田中さんは来月転勤するそうだ。** — 다나카 씨는 다음 달에 전근한다고 한다.
> **このレストランは有名だそうだ。** — 이 레스토랑은 유명하다고 한다.

### 핵심 구분

```
+------------------------------------------------------------------+
|  외관 vs. 전문 — 주요 차이                                        |
+------------------------------------------------------------------+
|                                                                  |
|  외관 (~해 보이다):   동사 ます형 어간 + そうだ                    |
|                      형용사 어간 + そうだ                         |
|                                                                  |
|  전문 (~라고 하다):   보통체 + そうだ                              |
|                                                                  |
+------------------------------------------------------------------+
|                                                                  |
|  おいしそうだ = 맛있어 보인다. (음식을 볼 수 있다.)               |
|  おいしいそうだ = 맛있다고 한다. (누군가가 말해줬다.)             |
|                                                                  |
|  降りそうだ = 비가 올 것 같다. (먹구름이 보인다.)                 |
|  降るそうだ = 비가 온다고 한다. (일기 예보를 봤다.)               |
|                                                                  |
+------------------------------------------------------------------+
```

### そうだ의 활용 (외관)

| 형태 | 일본어 | 예시 |
|------|----------|---------|
| 보통체 | そうだ | おいしそうだ |
| 정중체 | そうです | おいしそうです |
| 부정 | そうにない / そうもない | 降りそうにない |
| 명사 앞 | そうな | おいしそうなケーキ |
| 동사 앞 | そうに | おいしそうに食べる |

> **전문 そうだ**는 활용하지 않습니다 — 항상 そうだ/そうです입니다.

## 6. はずだ — "~일 것이다 / ~해야 한다"

はずだ는 추론이나 증거에 기반한 화자의 **논리적 기대**나 **강한 확신**을 표현합니다.

### 접속 방법

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (보통체) | + はずだ | 来る**はずだ** (올 것이다) |
| い형용사 | + はずだ | 高い**はずだ** (비쌀 것이다) |
| な형용사 + **な** | + はずだ | 簡単**なはずだ** (쉬울 것이다) |
| 명사 + **の** | + はずだ | 休み**のはずだ** (쉬는 날일 것이다) |
| 부정 | ないはずだ | 来ない**はずだ** (오지 않을 것이다) |

### 용법

#### A. 논리적 기대

> **もう届いたはずだ。**（先週送ったから）
> 이미 도착했을 것이다. (지난주에 보냈으니까.)

> **彼は知っているはずだ。** — 그는 알고 있을 것이다. (들었으니까.)

#### B. "~일 리가 없다"

> **こんなに高いはずがない！** — 이렇게 비쌀 리가 없다!
> **彼が嘘をつくはずがない。** — 그가 거짓말할 리가 없다.

> **はずがない**는 강한 부정입니다: "~일 리가 없다..."

#### C. 현실과의 대조

> **簡単なはずだったのに、難しかった。**
> 쉬울 거였는데 어려웠다.

### はずだ vs. 기타 표현

| 표현 | 확신 수준 | 근거 |
|-----------|----------------|----------|
| はずだ | 높음 (논리적) | 추론 / 증거 |
| ようだ | 중간 | 직접 관찰 |
| らしい | 중간 | 전문 / 간접적 |
| かもしれない | 낮음 (아마도) | 가능성 |

### 예문

1. **電車は8時に来るはずです。** — 전철은 8시에 올 것입니다.
2. **彼女はもう帰ったはずだ。** — 그녀는 이미 돌아갔을 것이다.
3. **そんなはずはない！** — 그럴 리가 없다!
4. **明日は休みのはずだけど。** — 내일은 쉬는 날일 텐데...
5. **このレストランは安いはずだよ。** — 이 레스토랑은 쌀 것이야. (리뷰를 읽었으니까.)

## 7. ことがある — "~할 때가 있다"

이 패턴은 무언가가 **가끔 일어나는** 것이나 **경험이 있는** 것을 표현합니다.

### 접속 방법

| 용법 | 접속 방법 | 예시 |
|-----|-----------|---------|
| 가끔 일어나는 일 | 동사 (사전형) + ことがある | 忘れる**ことがある** |
| 과거 경험 | 동사 (た형) + ことがある | 行った**ことがある** |

### A. 가끔 일어나는 일 (사전형 + ことがある)

때때로 일어나는 일을 묘사합니다:

> **朝ご飯を食べないことがある。** — 아침밥을 먹지 않을 때가 있다.
> **電車が遅れることがある。** — 전철이 늦을 때가 있다.
> **夜中に目が覚めることがある。** — 한밤중에 잠이 깰 때가 있다.

### B. 과거 경험 (た형 + ことがある)

특정 경험이 있는지를 묘사합니다:

> **日本に行ったことがある。** — 일본에 간 적이 있다. (그 경험이 있다.)
> **納豆を食べたことがない。** — 낫토를 먹은 적이 없다.
> **富士山に登ったことがありますか。** — 후지산에 오른 적이 있습니까?

### 경험 묻기

> Q: **寿司を作ったことがありますか。** — 초밥을 만든 적이 있습니까?
> A: **はい、一度だけ作ったことがあります。** — 네, 한 번만 만든 적이 있습니다.
> A: **いいえ、一度もありません。** — 아니요, 한 번도 없습니다.

### 예문

1. **彼に会ったことがある。** — 그를 만난 적이 있다.
2. **朝寝坊することがある。** — 늦잠을 잘 때가 있다.
3. **この映画を見たことがありますか。** — 이 영화를 본 적이 있습니까?
4. **日本語で夢を見たことがある。** — 일본어로 꿈을 꾼 적이 있다.
5. **授業に遅刻することがある。** — 수업에 지각할 때가 있다.

## 8. 비교 도표

### 증거 vs. 정보 출처

| 표현 | 표현하는 것 | 증거 출처 | 확신도 |
|-----------|-------------------|----------------|-----------|
| と思う | 개인 의견 | 화자 자신의 생각 | 화자의 신념 |
| ようだ | 추론 | 직접 관찰 (격식적) | 중간 |
| みたいだ | 추론 | 직접 관찰 (구어적) | 중간 |
| らしい | 전문/추론 | 타인에게서 들음 | 중간 |
| そうだ (외관) | 시각적 인상 | 무언가를 봄 | 중간 |
| そうだ (전문) | 보고 | 특정 출처 | 중간 |
| はずだ | 논리적 기대 | 추론/지식 | 높음 |

### 빠른 참조: "비가 올 것 같다"

| 일본어 | 뉘앙스 |
|----------|--------|
| 雨が降る**と思う** | 비가 올 거라고 생각한다. (내 의견) |
| 雨が降る**ようだ** | 비가 올 것 같다. (먹구름이 보인다 — 격식적) |
| 雨が降る**みたいだ** | 비가 올 것 같다. (먹구름이 보인다 — 구어적) |
| 雨が降る**らしい** | 비가 온다고 한다. (누군가가 말해줬다 / 간접적) |
| 雨が降り**そうだ** | 비가 올 것 같다. (하늘이 위협적으로 보인다) |
| 雨が降る**そうだ** | 비가 온다고 한다. (일기 예보에서 말했다) |
| 雨が降る**はずだ** | 비가 올 것이다. (예보에서 90% 확률이라고 했다) |

## 9. 흔한 실수와 주의점

### 실수 1: そうだ의 외관과 전문 혼동

```
✗ おいしいそうだ = 맛있다고 한다 (전문)
  おいしそうだ  = 맛있어 보인다 (외관)

기억하세요: ます형 어간/형용사 어간 + そうだ = 외관
           보통체 + そうだ = 전문
```

### 실수 2: 3인칭에 ている 없이 と思う 사용

```
✗ 田中さんはそう思う。 (자신의 생각처럼 들림)
○ 田中さんはそうと思っている。 (다나카 씨가 그렇게 생각하고 있다 — 그의 지속적 신념)
```

### 실수 3: いい와 ない의 외관 そうだ 접속

```
✗ いそうだ / なそうだ
○ よさそうだ (いい → よさそう)
○ なさそうだ (ない → なさそう)
```

### 실수 4: らしい(전문)와 らしい(전형성) 혼동

```
彼は学生らしい。
  = 그는 학생인 것 같다. (전문)
  또는
  = 그는 학생답다. (전형적 행동)
문맥으로 구분합니다.

彼は学生らしい学生だ。 = 그는 학생다운 학생이다.
                        (전형성 — 두 의미 모두 활용)
```

## 10. 연습 문제

### 연습 1: と思う 연습

と思う를 사용하여 다음 의견을 표현하세요.

1. 일본어는 재미있다. (나는 ~라고 생각한다...)
2. 그는 내일 오지 않을 것이다. (나는 ~라고 생각한다...)
3. 내년에 유학하려고 생각하고 있다. (지속적 계획)
4. 이 책에 대해 어떻게 생각합니까? (의견 묻기)

<details>
<summary>정답</summary>

1. 日本語は面白い**と思う / と思います**。
2. 彼は明日来ない**と思う**。(と思う가 아닌 내용을 부정)
3. 来年留学しよう**と思っている / と思っています**。(의지형 + と思っている으로 지속적 계획 표현)
4. この本についてどう**思いますか**。

</details>

### 연습 2: 외관 vs. 전문 そうだ

올바른 そうだ 형태를 선택하세요.

1. (케이크를 보면서) 이 케이크 맛있어 보인다!
2. (누군가가 말해줬다) 저 레스토랑은 맛있다고 한다.
3. (하늘을 보면서) 비가 올 것 같다.
4. (뉴스에서 읽었다) 지진이 있었다고 한다.

<details>
<summary>정답</summary>

1. このケーキはおいし**そうだ**。(외관: おいし + そう)
2. あのレストランはおいしい**そうだ**。(전문: おいしい + そうだ)
3. 雨が降り**そうだ**。(외관: 降り + そう)
4. 地震があった**そうだ**。(전문: あった + そうだ)

</details>

### 연습 3: ようだ / みたいだ / らしい

가장 적합한 표현으로 빈칸을 채우세요.

1. (떨고 있는 사람을 보면서) 彼は寒い ________ 。(직접 관찰, 구어체)
2. (동료가 말해줬다) 来週会議がある ________ 。(전문)
3. (젖은 우산을 보면서) 外は雨が降っている ________ 。(직접 관찰, 격식적)
4. 今日は春 ________ 天気ですね。(봄다운)

<details>
<summary>정답</summary>

1. 彼は寒い**みたいだ**。(직접 관찰, 구어적 = みたいだ)
2. 来週会議がある**らしい**。(전문 = らしい)
3. 外は雨が降っている**ようだ**。(직접 관찰, 격식적 = ようだ)
4. 今日は春**らしい**天気ですね。(~다운 = 명사 + らしい)

</details>

### 연습 4: はずだ

はずだ를 사용하여 기대를 표현하세요.

1. 가게는 열려 있을 것이다. (아직 영업시간이니까.)
2. 그녀는 알고 있을 것이다. (회의에 참석했으니까.)
3. 이것이 틀렸을 리가 없다! (강한 부정)
4. 쉬울 거였는데 어려웠다.

<details>
<summary>정답</summary>

1. 店は開いている**はずだ**。(まだ営業時間だから。)
2. 彼女は知っている**はずだ**。(会議にいたから。)
3. これが間違っている**はずがない**！(はずがない = ~일 리가 없다)
4. 簡単な**はずだったのに**、難しかった。(はずだった = ~일 거였다)

</details>

### 연습 5: ことがある

ことがある를 사용하여 완성하세요.

1. 교토에 간 적이 있습니까?
2. 가끔 숙제를 잊어버린다.
3. 생선회를 먹은 적이 없다.
4. 밤에 잠이 안 올 때가 있다.

<details>
<summary>정답</summary>

1. 京都に行った**ことがありますか**。(과거 경험: た + ことがある)
2. 宿題を忘れる**ことがある**。(가끔: 사전형 + ことがある)
3. 生の魚を食べた**ことがない**。(경험 없음: た + ことがない)
4. 夜眠れない**ことがある**。(가끔: 사전형 + ことがある)

</details>

### 연습 6: 혼합 — 가장 적합한 표현 선택

각 맥락에 가장 자연스러운 표현을 선택하세요.

1. (먹구름이 보인다) 雨が ________ 。
   a. 降るそうだ b. 降りそうだ c. 降るはずだ d. 降るらしい

2. (친구가 레스토랑에 대해 말해줬다) あの店は安い ________ 。
   a. みたいだ b. はずだ c. らしい d. そうだ (외관)

3. (시간표를 알고 있다) 電車は3時に来る ________ 。
   a. らしい b. はずだ c. みたいだ d. と思う

4. (개인 의견을 말하기) この本は面白い ________ 。
   a. ようだ b. そうだ c. と思う d. らしい

<details>
<summary>정답</summary>

1. **b. 降りそうだ** (하늘의 시각적 인상 = 외관 そうだ)
2. **c. らしい** (친구로부터의 정보 = 전문)
3. **b. はずだ** (알고 있는 시간표에 기반 = 논리적 기대)
4. **c. と思う** (개인 의견 표현)

</details>
