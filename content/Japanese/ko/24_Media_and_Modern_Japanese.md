## 2. 뉴스 기사 읽기

### 2.1 뉴스 문체

일본어 뉴스 기사는 일상 언어와 다른 독특한 문체를 사용합니다:

```
+----------------------------------------------------------+
|         NEWS JAPANESE CHARACTERISTICS                    |
+----------------------------------------------------------+
|                                                          |
|  1. 헤드라인 압축:                                        |
|     • 조사가 자주 생략됨                                   |
|     • 동사는 사전형 (ます형이 아님)                         |
|     • 예시: 「首相、増税見送り表明」                         |
|       전문: 首相が増税の見送りを表明した                     |
|                                                          |
|  2. 격식적 문어체:                                        |
|     • ～である / ～だ (～です/～ます가 아님)                 |
|     • ～とのこと / ～という                                |
|     • ～と見られている (~라고 보여지고 있다)                 |
|                                                          |
|  3. 출처 표시 패턴:                                       |
|     • ～によると (~에 따르면)                               |
|     • ～と述べた (~라고 밝혔다)                             |
|     • 関係者によれば (관계자에 따르면)                       |
|                                                          |
|  4. 수동태가 매우 빈번:                                    |
|     • ～が発表された (~가 발표되었다)                       |
|     • ～が指摘されている (~가 지적되고 있다)                 |
|     • ～ことが明らかになった (~이 밝혀졌다)                  |
+----------------------------------------------------------+
```

### 2.2 주석이 달린 뉴스 기사 예시

```
【見出し】
政府、来年度予算案を閣議決定 ← 헤드라인: 조사 생략

【本文】
政府は20日、来年度の一般会計予算案を閣議決定した。
         ↑ 날짜 패턴: ○日
総額は約115兆円で、過去最大となった。
              ↑ 약: 대략
防衛費は前年度比で16%増の7兆9000億円に上り、
                          ↑ 큰 숫자
3年連続で過去最高を更新した。

財務大臣は記者会見で「持続可能な財政運営に
                              ↑ 인용
全力を尽くす」と述べた。
             ↑ 출처 표시

一方、野党からは「国民生活への配慮が不十分だ」
↑ 대조/전환
との批判の声が上がっている。
     ↑ 수동 패턴
```

### 2.3 자주 쓰이는 뉴스 패턴

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| ～と見られている | ~라고 보여지고 있다 | 犯人は20代と見られている |
| ～ことが分かった | ~임이 밝혀졌다 | 原因は人為的ミスであることが分かった |
| ～の見通し | 전망 | 景気は回復する見通しだ |
| ～に踏み切る | 결정적인 조치를 취하다 | 政府は増税に踏み切った |
| ～を受けて | ~을 받아 / ~에 대응하여 | 批判を受けて、計画を修正した |
| ～に先立ち | ~에 앞서 | 会談に先立ち、外相が声明を出した |
| ～の一環として | ~의 일환으로 | 改革の一環として新制度を導入する |

### 3.1 흔한 SNS 표현

## 4. 젊은이 속어 (若者言葉)

### 4.1 현재 젊은이 표현

젊은이 속어(若者言葉 / わかものことば)는 빠르게 변화합니다. 다음은 지속적으로 사용되고 있는 정착된 표현들입니다:

| 속어 | 의미 | 용례 |
|-------|---------|-------|
| やばい / ヤバい | 대박 / 최악 (맥락에 따라) | 「このラーメン、やばい！」(이 라멘, 대박이야!) |
| エモい | 감성적인 / 그리운 / 감동적인 | 「この曲エモいわ」(이 곡 감성적이야) |
| ガチ | 진짜 / 진심 | 「ガチで言ってる？」(진심으로 말하는 거야?) |
| 推し (おし) | 최애 (아이돌, 캐릭터) | 「推しが尊い」(최애가 존귀해) |
| 尊い (とうとい) | 존귀한 / 너무 좋은 | 사랑하는 캐릭터/아이돌에 대해 사용 |
| 沼 (ぬま) | 빠져듦 / 깊은 몰입 | 「BTS沼にハマった」(BTS 늪에 빠졌다) |
| 陽キャ (ようきゃ) | 외향적인 사람 | 陽 = 밝음 + キャラ = 캐릭터 |
| 陰キャ (いんきゃ) | 내향적인 사람 | 陰 = 그림자 + キャラ = 캐릭터 |
| それな | "완전 맞아" / 동의 | 보편적 동의 표현 |
| ワンチャン | 혹시 / 어쩌면 | ワンチャンス (one chance) |

### 4.2 의미 변화 단어

일부 단어들은 젊은이 사용에서 의미가 크게 변화했습니다:

| 단어 | 원래 의미 | 젊은이 의미 |
|------|-----------------|---------------|
| やばい | 위험한 / 나쁜 | 대박 (긍정 또는 부정) |
| 全然 (ぜんぜん) | 전혀 ~않다 (+ 부정) | 완전 / 확실히 (+ 긍정) |
| 普通に (ふつうに) | 보통으로 | 꽤 / 진심으로 |
| 微妙 (びみょう) | 미묘한 / 섬세한 | 별로 / 그저 그런 |
| 適当 (てきとう) | 적절한 / 적합한 | 대충 / 건성으로 |
| 大丈夫 (だいじょうぶ) | 괜찮은 / 안전한 | 아니요, 됐어요 (정중한 거절) |

> **大丈夫를 거절로 사용하는 예**:
> 店員：「レジ袋はいりますか。」(비닐봉지 필요하세요?)
> 客：「大丈夫です。」(아뇨, 괜찮아요 / 됐어요.)

### 4.3 속어의 단어 형성 패턴

| 패턴 | 예시 | 작동 원리 |
|---------|----------|-------------|
| ～み (명사화) | つらみ, わかりみ, やばみ | 형용사/동사 어간 + み = 추상명사 |
| ～活 (かつ = 활동) | 婚活, 就活, 推し活, 朝活 | 약어 + 活 = "활동" |
| 약어 | スタバ, ファミマ, バイト | 외래어 축약 |
| ～い (형용사화) | エモい, エグい, キモい | 명사/외래어 + い = 형용사 |
| 逆○○ (ぎゃく) | 逆に好き | "오히려 좋아" = 예상 밖의 긍정 |

### 4.4 세대별 속어 연표

| 시대 | 대표적 속어 |
|-----|---------------------|
| 1990년대 | チョベリグ (초좋음), チョベリバ (초나쁨), マジ卍 |
| 2000년대 | KY (空気読めない = 분위기 파악 못함), ウケる (웃겨) |

## 9. 연습 문제

### 연습 1: 뉴스 헤드라인 해독

다음 압축된 헤드라인을 완전한 문장으로 확장하십시오:

1. 首相、来月訪米で調整
2. 新幹線、大雪で運休
3. 今年の漢字「金」に決定
4. AI活用、教育現場に広がる

<details>
<summary>정답</summary>

1. 首相は来月に訪米することで調整している。
   (총리는 다음 달 방미를 조율하고 있다.)

2. 新幹線が大雪の影響で運休となった。
   (신칸센이 폭설 영향으로 운행이 중단되었다.)

3. 今年の漢字が「金」に決定した。
   (올해의 한자가 「金」(금/돈)으로 결정되었다.)

4. AIの活用が教育現場に広がっている。
   (AI 활용이 교육 현장에 확산되고 있다.)

</details>

### 연습 2: SNS 번역

다음 SNS 메시지를 표준 일본어로 번역하십시오:

1. 今日のライブまじ神wwww 推しが尊すぎて無理😭
2. あのカフェ行ってきた！めっちゃ映える✨ おすすめ！
3. 新作ゲーム、ガチでやばい。沼にハマりそう orz
4. 明日の予定 り。朝活してからスタバ行く

<details>
<summary>정답</summary>

1. 표준어: 今日のライブは本当に素晴らしかった。好きなアイドルがとても魅力的で感動した。
   (오늘 라이브는 정말 훌륭했다. 좋아하는 아이돌이 너무 매력적이어서 감동받았다.)

2. 표준어: あのカフェに行ってきました。写真映えするお店でした。おすすめです。
   (그 카페에 다녀왔습니다. 사진이 잘 나오는 가게였습니다. 추천합니다.)

3. 표준어: 新作のゲームは本当にすごい。夢中になりそうだ。
   (신작 게임은 정말 대단하다. 빠져들 것 같다.)

4. 표준어: 明日の予定、了解しました。朝の活動をしてからスターバックスに行きます。
   (내일 일정, 알겠습니다. 아침 활동을 한 후에 스타벅스에 갑니다.)

</details>

### 연습 3: 간판 읽기

다음 간판은 무슨 뜻입니까? 어디에서 볼 수 있습니까?

1. 関係者以外立入禁止
2. 食べ歩き禁止
3. 只今満席です。しばらくお待ちください。
4. 本日貸切

<details>
<summary>정답</summary>

1. 関係者以外立入禁止 (かんけいしゃいがいたちいりきんし)
   "관계자 외 출입 금지." 공사 현장, 무대 뒤, 사무실 건물에서 볼 수 있습니다.

2. 食べ歩き禁止 (たべあるききんし)
   "먹으면서 걷기 금지." 전통 상점가, 사찰, 관광지에서 볼 수 있습니다.

3. 只今満席です。しばらくお待ちください。(ただいままんせきです)
   "현재 만석입니다. 잠시만 기다려 주십시오." 식당에서 볼 수 있습니다.

4. 本日貸切 (ほんじつかしきり)
   "오늘은 대관 행사입니다." 일반에 공개되지 않는 식당이나 행사장에서 볼 수 있습니다.

</details>

### 연습 4: 젊은이 속어 문맥

문맥에 따라 밑줄 친 단어의 올바른 의미를 선택하십시오:

1. 「このケーキ、**やばい**！超おいしい！」
   a) 위험한  b) 끔찍한  c) 대박인

2. 「あの映画、**全然**よかったよ。」
   a) 전혀 좋지 않았다  b) 꽤 좋았다  c) 조금 좋았다

3. 「バイト大変だったけど、**ワンチャン**来月休めるかも。」
   a) 한 번의 기회  b) 어쩌면  c) 확실히

4. 「新しいドラマの主人公、**推せる**。」
   a) 밀 수 있다  b) 응원할 만하다/좋다  c) 이상하다

<details>
<summary>정답</summary>

1. c) 대박인 — 이어지는 「超おいしい」가 긍정적 의미를 확인해 줍니다.
2. b) 꽤 좋았다 — 젊은이들이 全然을 긍정적 형용사와 함께 사용하면 "완전/꽤"라는 뜻입니다.
3. b) 어쩌면 — ワンチャン (one chance)은 캐주얼한 말에서 "어쩌면/혹시"의 의미로 사용됩니다.
4. b) 응원할 만하다/좋다 — 推せる (推し가 될 수 있다)는 그 캐릭터가 최애가 될 만한 가치가 있다는 뜻입니다.

</details>

### 연습 5: 뉴스 기사 독해

다음 짧은 뉴스 발췌문을 읽고 질문에 답하십시오:

```
政府は5日、来年度の教育予算案を閣議決定した。
総額は約5兆8000億円で、前年度比3%増となった。
文部科学大臣は記者会見で「デジタル教育の推進に
重点を置く」と述べ、全国の小中学校にAI教材を
導入する方針を明らかにした。
一方、教育関係者からは「教員の負担軽減が
先決ではないか」との声も上がっている。
```

1. 내각에서 무엇을 결정했습니까?
2. 예산 총액은 얼마입니까?
3. 문부과학대신은 무엇을 강조했습니까?
4. 교육 관계자들은 어떤 우려를 표명했습니까?

<details>
<summary>정답</summary>

1. 내각은 다음 연도 교육 예산안을 각의 결정했습니다 (来年度の教育予算案を閣議決定した).

2. 약 5조 8000억 엔 (約5兆8000億円)으로, 전년도 대비 3% 증가했습니다.

3. 대신은 디지털 교육 추진에 중점을 둔다 (デジタル教育の推進に重点を置く)고 강조하고, 전국 초중학교에 AI 교재를 도입하는 방침을 밝혔습니다.

4. 교육 관계자들은 교원의 부담 경감이 선결 과제가 아닌가 (教員の負担軽減が先決ではないか)라는 우려를 표명했습니다.

</details>
