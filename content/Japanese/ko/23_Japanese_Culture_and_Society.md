## 1. 전통 예술 (伝統芸術)

### 1.1 다도 (茶道 / さどう)

다도는 茶の湯 (ちゃのゆ)라고도 불리며, 一期一会 (いちごいちえ — "일생에 한 번뿐인 만남")의 미학 철학을 구현합니다. 모든 만남을 일생에 한 번뿐인 사건으로 여기는 것입니다.

```
+----------------------------------------------------------+
|         TEA CEREMONY OVERVIEW (茶道)                      |
+----------------------------------------------------------+
|                                                          |
|  철학: 和敬清寂 (わけいせいじゃく)                          |
|  ├── 和 (wa) — 조화                                      |
|  ├── 敬 (kei) — 경의                                     |
|  ├── 清 (sei) — 청정                                     |
|  └── 寂 (jaku) — 고요                                    |
|                                                          |
|  주요 유파:                                               |
|  ├── 表千家 (おもてせんけ) — 오모테센케                     |
|  ├── 裏千家 (うらせんけ) — 우라센케 (가장 보편적)           |
|  └── 武者小路千家 (むしゃこうじせんけ)                      |
|                                                          |
|  주요 도구:                                               |
|  ├── 茶碗 (ちゃわん) — 찻잔                               |
|  ├── 茶筅 (ちゃせん) — 대나무 거품기                       |
|  ├── 茶杓 (ちゃしゃく) — 차 떠는 수저                      |
|  ├── 棗 (なつめ) — 차통                                   |
|  └── 釜 (かま) — 철 주전자                                |
|                                                          |
|  차의 종류:                                               |
|  ├── 薄茶 (うすちゃ) — 연한 차 (일상용)                    |
|  └── 濃茶 (こいちゃ) — 진한 차 (격식용)                    |
+----------------------------------------------------------+
```

**손님을 위한 기본 다도 예절**:

| 순서 | 행동 | 일본어 |
|------|--------|----------|
| 1 | 방에 들어갈 때 인사하기 | お辞儀をして入室する |
| 2 | 족자(掛軸) 감상하기 | 掛軸を拝見する |
| 3 | 적절한 순서대로 앉기 | 席順に座る |
| 4 | 과자를 받으면 감사 인사하기 | 「お先に」「頂戴いたします」 |
| 5 | 차가 오기 전에 과자 먹기 | お菓子を先にいただく |
| 6 | 차가 앞에 놓이면 | 「お点前、頂戴いたします」 |
| 7 | 찻잔을 시계 방향으로 두 번 돌리기 | 茶碗を右に2回まわす |
| 8 | 몇 모금에 마시기 | 数口で飲む |
| 9 | 입을 댄 부분 닦기 | 飲み口を拭く |
| 10 | 찻잔을 반시계 방향으로 두 번 돌려 되돌리기 | 左に2回まわして戻す |

### 1.2 꽃꽂이 (華道 / かどう)

生け花 (いけばな)라고도 불리는 꽃꽂이는 세심하게 구성된 배열을 통해 자연의 아름다움을 표현하는 예술입니다.

| 유파 | 일본어 | 특징 |
|--------|----------|----------------|
| 이케노보 | 池坊（いけのぼう） | 가장 오래된 유파 (550년 이상), 격식적, 고전적 |
| 오하라 | 小原流（おはらりゅう） | 모리바나 (평면 용기) 스타일로 유명 |
| 소게츠 | 草月流（そうげつりゅう） | 현대적, 전위적, 비식물 재료도 사용 |

**핵심 원칙**:
- 天・地・人 (てん・ち・じん) — 천, 지, 인: 우주적 질서를 나타내는 세 주요 줄기
- 間 (ま) — 여백은 꽃 자체만큼이나 중요합니다
- 계절 꽃은 현재 계절을 반영해야 합니다

### 1.3 서예 (書道 / しょどう)

```
+----------------------------------------------------------+
|         CALLIGRAPHY OVERVIEW (書道)                       |
+----------------------------------------------------------+
|                                                          |
|  도구 (文房四宝 — ぶんぼうしほう):                          |
|  ├── 筆 (ふで) — 붓                                      |
|  ├── 墨 (すみ) — 먹                                      |
|  ├── 硯 (すずり) — 벼루                                   |
|  └── 紙 (かみ) — 종이 (和紙 / 와시)                       |
|                                                          |
|  서체:                                                    |
|  ├── 楷書 (かいしょ) — 해서/정자체                         |
|  ├── 行書 (ぎょうしょ) — 행서/반흘림체                      |
|  ├── 草書 (そうしょ) — 초서/흘림체                         |
|  └── 仮名 (かな) — 일본 가나 서예                          |
|                                                          |
|  주요 활용 상황:                                           |
|  ├── 年賀状 (ねんがじょう) — 연하장                         |
|  ├── 書き初め (かきぞめ) — 새해 첫 글씨                    |
|  └── 학교에서 8세부터 교과목으로 학습                       |
+----------------------------------------------------------+
```

### 1.4 전통 공연 예술

| 예술 형태 | 일본어 | 설명 |
|----------|----------|-------------|
| 能 | のう | 노 — 가면을 쓰고 하는 무용극, 가장 오래된 주요 연극 형태 (600년 이상) |
| 歌舞伎 | かぶき | 가부키 — 화려하고 극적인 연극, 전원 남성 출연 |
| 文楽 | ぶんらく | 분라쿠 — 인형극, 낭송자와 샤미센 연주자가 함께 공연 |
| 落語 | らくご | 라쿠고 — 1인 코미디 이야기, 방석에 앉아 공연 |
| 雅楽 | ががく | 가가쿠 — 고대 궁정 음악과 무용 |

```
+----------------------------------------------------------+
```

```
|  • 직급이 낮은 사람이 더 깊고 오래 인사                     |
+----------------------------------------------------------+
```

### 4.2 선물 문화 (贈り物 / おくりもの)

일본의 선물 문화는 매우 의례적이며, 계절에 따른 구체적인 의무가 있습니다:

| 상황 | 일본어 | 시기 | 선물 |
|----------|----------|------|------|
| 여름 선물 | お中元 (おちゅうげん) | 7-8월 | 음식, 음료, 생활용품 |
| 연말 선물 | お歳暮 (おせいぼ) | 12월 | 음식, 음료, 생활용품 |
| 기념품 | お土産 (おみやげ) | 여행 후 | 방문 지역의 특산물 |
| 병문안 | お見舞い (おみまい) | 아플 때 | 꽃 (화분은 안 됨!), 과일 |
| 집들이 | 新築祝い (しんちくいわい) | 이사 시 | 실용적인 물건 |
| 축의금 | ご祝儀 (ごしゅうぎ) | 결혼식 | 특별 봉투에 넣은 현금 |
| 부의금 | 香典 (こうでん) | 장례식 | 흑백 봉투에 넣은 현금 |

**선물 규칙**:

| 규칙 | 설명 |
|------|-------------|
| 포장이 중요 | 風呂敷 (후로시키) 또는 적절한 포장지 사용 |
| 양손으로 건네기 | 양손으로 건네며 존중 표시 |
| 한두 번 사양하기 | 받는 사람이 처음에 거절할 수 있음 — 다시 권하기 |
| 「つまらないものですが」라고 말하기 | "변변치 않은 것입니다만..." (겸손한 표현) |
| 4개짜리 세트 피하기 | 四 (し) = 죽음 (死). 대신 3개 또는 5개짜리 세트 선물 |
| 즉시 열지 않기 | 준 사람이 떠난 후에 열기 (서양 문화와 다름) |
| 답례하기 (お返し) | 대략 절반 가치로 답례 (半返し) |

## 5. 계절과 연중행사 (年中行事)

### 5.1 일본 문화에서의 사계절

일본의 문화적 정체성은 사계절(四季 / しき)과 깊이 결부되어 있습니다. 계절 감각(季節感 / きせつかん)은 언어, 음식, 패션, 일상 생활에 스며들어 있습니다.

```
+----------------------------------------------------------+
|         THE JAPANESE YEAR                                |
+----------------------------------------------------------+
|                                                          |
|  春 (はる) — 봄 (3월 - 5월)                               |
```

## 6. 지역 차이 (地方の違い)

### 6.1 간토 vs 간사이 (関東 vs 関西)

도쿄 지역(関東)과 오사카/교토 지역(関西) 사이의 문화적 차이는 일본 사회의 특징적인 요소입니다.

| 특징 | 関東 (간토) | 関西 (간사이) |
|---------|-------------|--------------|
| 주요 도시 | 東京 (도쿄) | 大阪 (오사카) |
| 방언 | 標準語 (표준어) | 関西弁 (간사이 사투리) |
| 성격 | 절제된, 격식적 | 외향적, 유머러스 |
| 에스컬레이터 | 왼쪽에 서기 | 오른쪽에 서기 |
| 간장 | 濃口 (진간장) | 薄口 (연간장) |
| 우동 국물 | 진한 색 (간장 베이스) | 연한 색 (다시 베이스) |
| 전기 | 50Hz | 60Hz |
| 코미디 스타일 | ツッコミ寄り (츳코미 중심) | ボケ寄り (보케 중심) |
| 떡 모양 | 四角い (네모) | 丸い (동그라미) |

### 6.2 간사이 사투리 기초 (関西弁)

| いくら | なんぼ | 얼마 |
| 捨てる | ほかす | 버리다 |
| 違う | ちゃう | 다른 / 틀린 |

## 7. 종교와 철학

### 7.1 신도와 불교의 공존

일본은 독특하게 두 가지 주요 종교 전통을 융합합니다:

```
+----------------------------------------------------------+
|       SHINTO (神道) vs BUDDHISM (仏教)                    |
+----------------------------------------------------------+
|                                                          |
|  神道 (신도):                                             |
|  ├── 일본 토착 종교                                       |
|  ├── 핵심: 자연 정령 (神 / かみ)                          |
|  ├── 예배 장소: 神社 (じんじゃ / 신사)                     |
|  ├── 문: 鳥居 (とりい / 도리이)                            |
|  ├── 의식: 祭り, お祓い, 初詣                              |
|  └── 생애 의식: 출생, 결혼, 축제                           |
|                                                          |
|  仏教 (불교):                                             |
|  ├── 중국/한국에서 전래 (6세기)                            |
|  ├── 핵심: 깨달음, 윤회                                   |
|  ├── 예배 장소: 寺 (てら / 절)                             |
|  ├── 문: 山門 (さんもん)                                  |
|  ├── 의식: 명상, 독경                                     |
|  └── 생애 의식: 장례, 추도 법사                            |
|                                                          |
|  흔한 말: 「日本人は生まれたときは神道、                     |
|  死んだときは仏教」                                       |
|  (일본인은 태어날 때는 신도, 죽을 때는 불교.)               |
+----------------------------------------------------------+
```

### 7.2 신사 vs 절 에티켓

| 행동 | 신사 (神社) | 절 (寺) |
|--------|-------------|-----------|
| 입장 | 鳥居를 통과하며 인사 | 山門을 통해 입장 |
| 정화 | 手水舎: 왼손, 오른손, 입 순서로 씻기 | 동일 (있는 경우) |
| 참배 | 二拝二拍手一拝 (2번 절, 2번 박수, 1번 절) | 절하고, 두 손을 합장 (박수 치지 않음) |
| 봉납 | 동전 던지기, 방울 울리기 | 동전 던지기, 방울 울리기 |
| 운세 | おみくじ (운세 제비) | おみくじ |

### 8.1 식사 에티켓

### 8.2 젓가락 금기 (箸のマナー違反)

| 금기 | 일본어 | 이유 |

| 음식 위에서 머뭇거리며 젓가락 움직이기 | 迷い箸（まよいばし） | 우유부단함을 보여줌 |
| 젓가락 핥기 | ねぶり箸 | 비위생적, 무례함 |
| 젓가락으로 그릇 끌어당기기 | 寄せ箸（よせばし） | 손으로 그릇을 옮기기 |

## 9. 연습 문제

### 연습 1: 문화 어휘 매칭

각 용어를 올바른 설명과 연결하십시오:

| 용어 | 설명 선택지 |
|------|-------------------|
| 1. 一期一会 | a. 종이에 소원을 적기 |
| 2. 七夕 | b. 섣달 그믐의 108번 타종 |
| 3. 除夜の鐘 | c. 모든 만남을 일생에 한 번뿐으로 여기기 |
| 4. 風呂敷 | d. 새해 첫 신사 참배 |
| 5. 初詣 | e. 전통 보자기 |

<details>
<summary>정답</summary>

1 → c (一期一会 — 일생에 한 번뿐인 만남)
2 → a (七夕 — 칠석, 단자쿠에 소원 적기)
3 → b (除夜の鐘 — 섣달 그믐 자정에 108번 타종)
4 → e (風呂敷 — 선물과 물건을 싸는 전통 보자기)
5 → d (初詣 — 새해 첫 신사 참배)

</details>

### 연습 2: 신사 아니면 절?

각 활동에 대해 神社 (신사)에 가는지 寺 (절)에 가는지 표시하십시오:

1. 初詣（はつもうで）
2. お葬式（おそうしき — 장례식）
3. 七五三（しちごさん）
4. 法事（ほうじ — 추도 법사）
5. 結婚式（けっこんしき — 결혼식）

<details>
<summary>정답</summary>

1. 神社 (신사) — 가장 일반적이지만, 일부 절에서도 初詣를 합니다
2. 寺 (절) — 장례식은 불교 의식입니다
3. 神社 (신사) — 아이 성장 축하 행사는 신도 행사입니다
4. 寺 (절) — 추도 법사는 불교 의식입니다
5. 둘 다 가능하지만, 전통적으로는 神社 (신사)에서 신도식 결혼식을 합니다. 현대 일본에서는 기독교식(教会) 결혼식도 매우 인기가 있습니다.

</details>

### 연습 3: 간사이 사투리 번역

다음 간사이벤 문장을 표준 일본어로 변환하십시오:

1. それ、めっちゃおもろいやん。
2. なんぼするん？
3. そんなんあかんやろ。
4. ほんまにちゃうで。

<details>
<summary>정답</summary>

1. それ、すごく面白いじゃない。(그거, 정말 재미있잖아.)
2. いくらするの？(얼마야?)
3. そんなのだめでしょう。(그런 건 안 되지.)
4. 本当に違うよ。(진짜로 달라.)

</details>

### 연습 4: 선물 상황

각 상황에서 적절한 행동은 무엇입니까?

1. 일본인 동료의 집을 처음 방문합니다. 무엇을 가져가며 뭐라고 말합니까?
2. 약 5,000엔 상당의 お中元을 받았습니다. 어떻게 해야 합니까?
3. 일본인 친구가 입원해 있습니다. 무엇을 가져가면 안 됩니까?
4. 일본식 결혼식에 참석합니다. 무엇을 가져갑니까?

<details>
<summary>정답</summary>

1. 작고 예쁘게 포장한 선물(과자, 과일 또는 고향 특산물)을 가져갑니다. 「つまらないものですが、どうぞ」 또는 「ほんの気持ちですが」라고 말합니다. 양손으로 건넵니다.

2. 감사 편지(お礼状)를 보내고, 절반 가치의 답례 선물(お返し)을 보내는 것을 고려합니다 (半返し — 약 2,500엔).

3. 화분 식물(鉢植え — 병원에 "뿌리를 내린다"는 의미, 즉 오래 입원한다는 뜻)을 피합니다. 또한 향이 강한 꽃과 4개짜리 세트(四 = 死 = 죽음)를 피합니다.

4. ご祝儀 (축의금)를 특별한 祝儀袋 (축의 봉투, 빨강과 금색 장식)에 넣어 가져갑니다. 금액은 홀수(奇数)여야 합니다: 친구는 30,000엔이 표준, 가까운 친척은 50,000엔. 40,000엔(四 = 죽음)이나 20,000엔(짝수 = 나눌 수 있음 = 이별)은 절대 안 됩니다.

</details>

### 연습 5: 문화 에세이

일본 문화의 관심 있는 한 가지 측면에 대해 日本語로 5-6문장의 문단을 작성하십시오 (常体 또는 敬体). 이 과에서 배운 어휘를 최소 3개 사용하십시오.

<details>
<summary>예시 답안</summary>

日本の食文化の中で、特に興味深いのは季節感との深い結びつきである。
日本では、春には桜餅や筍を楽しみ、夏にはそうめんやかき氷で涼を取る。
秋には栗やきのこが食卓に並び、冬には鍋料理で体を温める。
このように、食を通じて四季の移り変わりを感じることは、
日本文化の大きな特徴の一つである。
「いただきます」という食前の挨拶も、
自然の恵みへの感謝を表す日本独特の習慣だと言える。

</details>
