## 선수 학습

- 약 500자 이상의 한자 지식
- 기본 및 중급 문법 숙지 (1~16과)
- 동사 활용 및 형용사 형태에 대한 이해
- 히라가나, 가타카나, 기본 한자 복합어 읽기 능력

## 학습 목표

- 신체 부위별(目, 口, 手, 足 등) 일본어 관용구 학습하기
- 의성어·의태어(オノマトペ)를 이해하고 자연스러운 일본어에서 사용하기
- 일반적인 속담(ことわざ)과 그 문화적 의미 파악하기
- 정형 표현과 연어(collocations)를 대화와 글쓰기에 적용하기
- 관용적 의미와 문자적 의미를 구별하기
- 독해 맥락에서 관용구 인식하기

## 1. 신체 관용구 (体の慣用句)

일본어에는 신체 부위를 사용하는 관용구가 풍부합니다. 이를 体の慣用句 (からだのかんようく)라고 합니다. 신체 부위가 대체적인 의미 범주에 대한 단서를 제공합니다.

```
+----------------------------------------------------------+
|          신체 관용구 범주                                   |
+----------------------------------------------------------+
|                                                          |
|  目 (눈)   → 인식, 판단, 주의                             |
|  口 (입)   → 말, 성격, 식욕                               |
|  手 (손)   → 노력, 기술, 방법                             |
|  足 (발)   → 이동, 방해, 기반                             |
|  耳 (귀)   → 청각, 경청, 순종                             |
|  頭 (머리) → 지능, 사고                                   |
|  首 (목)   → 해고, 기대                                   |
|  顔 (얼굴) → 평판, 체면, 친숙함                           |
|  腹 (배)   → 감정, 숨겨진 속마음                          |
|  肩 (어깨) → 부담, 지위                                   |
+----------------------------------------------------------+
```

## 2. 눈 관용구 (目)

| 관용구 | 읽기 | 문자적 의미 | 실제 의미 |
|-------|---------|----------------|---------------|
| 目が高い | めがたかい | 눈이 높다 | 안목이 좋다 / 감식안이 있다 |
| 目が回る | めがまわる | 눈이 돌다 | 매우 바쁘다 / 어지럽다 |
| 目を通す | めをとおす | 눈을 통과시키다 | 훑어보다, 대충 읽다 |
| 目に入る | めにはいる | 눈에 들어오다 | 눈에 띄다 |
| 目をつぶる | めをつぶる | 눈을 감다 | 모르는 척하다, 눈감아 주다 |
| 目を引く | めをひく | 눈을 끌다 | 시선을 끌다, 눈길을 끌다 |
| 目の前 | めのまえ | 눈앞 | 바로 눈앞 |
| 目が覚める | めがさめる | 눈이 깨다 | 잠에서 깨다 / 깨달음을 얻다 |
| 目を丸くする | めをまるくする | 눈을 동그랗게 하다 | 놀라서 눈이 휘둥그레지다 |
| 白い目で見る | しろいめでみる | 흰 눈으로 보다 | 불쾌하게 쳐다보다 |

### 예문

- さすが、**目が高い**ですね。この絵は本物です。
  (역시 안목이 높으시군요. 이 그림은 진품입니다.)
- 年末は仕事で**目が回る**ほど忙しい。
  (연말에는 일 때문에 눈이 돌 정도로 바쁩니다.)
- この書類に**目を通して**いただけますか。
  (이 서류를 훑어봐 주시겠습니까?)
- 彼の無礼な態度に**目をつぶる**わけにはいかない。
  (그의 무례한 태도에 눈을 감아 줄 수는 없습니다.)

## 3. 입 관용구 (口)

| 관용구 | 읽기 | 문자적 의미 | 실제 의미 |
|-------|---------|----------------|---------------|
| 口が軽い | くちがかるい | 입이 가볍다 | 입이 가벼운, 비밀을 못 지키는 |
| 口が堅い | くちがかたい | 입이 단단하다 | 입이 무거운, 비밀을 잘 지키는 |
| 口が悪い | くちがわるい | 입이 나쁘다 | 말이 거친, 독설가인 |
| 口を出す | くちをだす | 입을 내다 | 참견하다, 말참견하다 |
| 口をそろえる | くちをそろえる | 입을 맞추다 | 이구동성으로 말하다, 만장일치로 동의하다 |
| 口に合う | くちにあう | 입에 맞다 | 입맛에 맞다 (음식) |
| 口がうまい | くちがうまい | 입이 능숙하다 | 말재주가 좋은, 달변인 |
| 口を滑らせる | くちをすべらせる | 입이 미끄러지다 | 실언하다, 무심코 말하다 |

### 예문

- 彼女は**口が軽い**から、秘密を話さない方がいい。
  (그녀는 입이 가벼우니 비밀을 말하지 않는 게 좋습니다.)
- 人の問題に**口を出す**のはやめなさい。
  (남의 문제에 참견하지 마십시오.)
- この料理は**口に合い**ますか。
  (이 요리가 입맛에 맞으십니까?)
- うっかり**口を滑らせて**、サプライズパーティーのことを言ってしまった。
  (무심코 실언해서 서프라이즈 파티에 대해 말해 버렸습니다.)

## 4. 손 관용구 (手)

| 관용구 | 읽기 | 문자적 의미 | 실제 의미 |
|-------|---------|----------------|---------------|
| 手を抜く | てをぬく | 손을 빼다 | 대충 하다, 손을 빼다 |
| 手を貸す | てをかす | 손을 빌려주다 | 도와주다, 거들다 |
| 手に入る | てにはいる | 손에 들어오다 | 손에 넣다, 입수하다 |
| 手がかかる | てがかかる | 손이 가다 | 손이 많이 가다 / 번거롭다 |
| 手が出ない | てがでない | 손이 나가지 않다 | 살 여유가 없다 / 손이 닿지 않다 |
| 手を打つ | てをうつ | 손을 치다 | 대책을 세우다 / 합의에 이르다 |
| 手を焼く | てをやく | 손을 태우다 | 다루기 어렵다, 골치 아프다 |
| 手に負えない | てにおえない | 손으로 감당할 수 없다 | 감당할 수 없다, 통제 불능이다 |
| 手も足も出ない | てもあしもでない | 손도 발도 나오지 않다 | 꼼짝달싹 못하다 |
| 手をつける | てをつける | 손을 대다 | 착수하다 / 손대다 |

### 예문

- この仕事は**手を抜かない**でください。
  (이 일은 대충 하지 마십시오.)
- 引っ越しの時、友達が**手を貸して**くれた。
  (이사할 때 친구가 도와주었습니다.)
- あのブランドのバッグは高くて**手が出ない**。
  (그 브랜드의 가방은 비싸서 살 여유가 없습니다.)
- この子は本当に**手がかかる**。
  (이 아이는 정말 손이 많이 갑니다.)

## 5. 발/다리 관용구 (足)

| 관용구 | 읽기 | 문자적 의미 | 실제 의미 |
|-------|---------|----------------|---------------|
| 足を引っ張る | あしをひっぱる | 다리를 잡아당기다 | 발목을 잡다, 방해하다 |
| 足が出る | あしがでる | 발이 나오다 | 예산을 초과하다 |
| 足を運ぶ | あしをはこぶ | 발을 나르다 | 발걸음을 하다, 방문하다 (존경) |
| 足が棒になる | あしがぼうになる | 다리가 막대기가 되다 | 걸어서 다리가 뻣뻣해지다 |
| 足元を見る | あしもとをみる | 발밑을 보다 | 약점을 이용하다 |
| 二の足を踏む | にのあしをふむ | 두 번째 발을 내딛다 | 망설이다, 주저하다 |

### 예문

- チームの**足を引っ張ら**ないように頑張る。
  (팀의 발목을 잡지 않도록 노력하겠습니다.)
- 予算から**足が出て**しまった。
  (예산을 초과해 버렸습니다.)
- ぜひ一度**足を運んで**みてください。
  (꼭 한번 방문해 보십시오.)
- 一日中歩いて、**足が棒になった**。
  (하루 종일 걸어서 다리가 뻣뻣해졌습니다.)

## 6. 기타 신체 관용구

### 머리 (頭)

| 관용구 | 읽기 | 의미 |
|-------|---------|---------|
| 頭が固い | あたまがかたい | 완고한, 융통성 없는 |
| 頭を抱える | あたまをかかえる | 머리를 쥐어짜다, 난감해하다 |
| 頭が下がる | あたまがさがる | 감탄하다, 존경하다 |
| 頭に来る | あたまにくる | 화가 나다, 분노하다 |

### 배 (腹)

| 관용구 | 읽기 | 의미 |
|-------|---------|---------|
| 腹が立つ | はらがたつ | 화가 나다 |
| 腹を割る | はらをわる | 속마음을 털어놓다, 솔직하게 말하다 |
| 腹が黒い | はらがくろい | 속이 검은, 뒤가 구린 |
| 腹を決める | はらをきめる | 결심하다, 각오를 다지다 |

### 어깨 (肩)

| 관용구 | 읽기 | 의미 |
|-------|---------|---------|
| 肩を持つ | かたをもつ | 편을 들다 |
| 肩の荷が下りる | かたのにがおりる | 짐을 덜다, 마음이 놓이다 |
| 肩身が狭い | かたみがせまい | 어깨가 좁은, 체면이 없는 |

### 얼굴 (顔)

| 관용구 | 읽기 | 의미 |
|-------|---------|---------|
| 顔が広い | かおがひろい | 얼굴이 넓은, 인맥이 넓은 |
| 顔を立てる | かおをたてる | 체면을 세워주다 |
| 顔が利く | かおがきく | 영향력이 있다 / 안면이 넓다 |
| 顔を出す | かおをだす | 얼굴을 내밀다, 잠깐 들르다 |

### 예문

- 彼はいつも**頭が固く**て、新しいアイデアを受け入れない。
  (그는 항상 완고해서 새로운 아이디어를 받아들이지 않습니다.)
- 今度のプロジェクトが成功して、**肩の荷が下りた**。
  (이번 프로젝트가 성공해서 짐을 덜었습니다.)
- 社長は**顔が広い**ので、色々な人を紹介してくれる。
  (사장은 인맥이 넓어서 여러 사람을 소개해 줍니다.)
- 彼女は本当に**腹黒い**人だ。表面上は優しいが...
  (그녀는 정말 속이 검은 사람입니다. 겉으로는 상냥하지만...)

## 7. 의성어·의태어 (オノマトペ)

일본어에는 매우 풍부한 의성어·의태어 체계가 있습니다. 세 가지 유형이 있습니다:

| 유형 | 일본어 | 설명 | 예시 |
|------|----------|-------------|---------|
| 擬音語 (기온고) | ぎおんご | 실제 소리를 흉내 냄 | ワンワン (개 짖는 소리), ザーザー (폭우) |
| 擬態語 (기타이고) | ぎたいご | 상태/조건을 묘사 | きらきら (반짝반짝), ぐちゃぐちゃ (엉망진창) |
| 擬情語 (기조고) | ぎじょうご | 감정을 묘사 | わくわく (설레는), いらいら (짜증나는) |

### 7.1 말과 의사소통

| 의성어·의태어 | 의미 | 예시 |
|-------------|---------|---------|
| ぺらぺら | 유창하게 (말하기) | 日本語が**ぺらぺら**だ。 (일본어가 유창하다.) |
| ぶつぶつ | 중얼중얼, 불평 | 何を**ぶつぶつ**言っているの？ (뭘 중얼중얼 거리는 거야?) |
| べらべら | 쉴 새 없이 지껄이는 | 秘密を**べらべら**話すな。 (비밀을 함부로 떠벌리지 마.) |
| ぼそぼそ | 작은 목소리로 중얼거리는 | **ぼそぼそ**話さないで。 (중얼거리지 마.) |
| はきはき | 또렷하게 말하는 | **はきはき**答えてください。 (또렷하게 대답해 주십시오.) |

### 7.2 감정과 느낌

| 의성어·의태어 | 의미 | 예시 |
|-------------|---------|---------|
| わくわく | 설레는, 두근거리는 | 旅行が**わくわく**する。 (여행이 설렌다.) |
| どきどき | 두근두근 (긴장/흥분) | 試験前に**どきどき**する。 (시험 전에 두근거린다.) |
| いらいら | 짜증나는, 초조한 | 渋滞で**いらいら**する。 (교통 체증에 짜증난다.) |
| うきうき | 즐거운, 들뜬 | 春になると**うきうき**する。 (봄이 되면 들뜬다.) |
| びくびく | 겁먹은, 초조한 | 上司の前で**びくびく**している。 (상사 앞에서 전전긍긍한다.) |
| うんざり | 질리는, 지겨운 | 同じ話に**うんざり**した。 (같은 이야기에 질렸다.) |
| しょんぼり | 풀이 죽은, 우울한 | 試合に負けて**しょんぼり**している。 (시합에 져서 풀이 죽어 있다.) |
| にこにこ | 방긋방긋 웃는 | 子供が**にこにこ**笑っている。 (아이가 방긋방긋 웃고 있다.) |

### 7.3 상태와 조건

| 의성어·의태어 | 의미 | 예시 |
|-------------|---------|---------|
| ぐずぐず | 꾸물거리는, 미적대는 | **ぐずぐず**しないで！ (꾸물거리지 마!) |
| ぴかぴか | 반짝반짝, 번쩍번쩍 | 床が**ぴかぴか**だ。 (바닥이 반짝반짝하다.) |
| ぼろぼろ | 낡은, 너덜너덜한 | 靴が**ぼろぼろ**になった。 (신발이 너덜너덜해졌다.) |
| ふわふわ | 폭신폭신, 부드러운 | **ふわふわ**のタオル。 (폭신한 수건.) |
| ぎりぎり | 간신히, 아슬아슬하게 | **ぎりぎり**間に合った。 (간신히 시간에 맞았다.) |
| ばらばら | 뿔뿔이, 제각각 | 意見が**ばらばら**だ。 (의견이 제각각이다.) |
| じめじめ | 축축한, 눅눅한 | 梅雨は**じめじめ**する。 (장마철은 축축하다.) |

### 7.4 행동과 움직임

| 의성어·의태어 | 의미 | 예시 |
|-------------|---------|---------|
| のろのろ | 느릿느릿 | **のろのろ**運転は危ない。 (느릿느릿 운전은 위험하다.) |
| すらすら | 술술, 막힘없이 | 日本語を**すらすら**読める。 (일본어를 술술 읽을 수 있다.) |
| ごろごろ | 뒹굴뒹굴 / 우르릉 | 休日は家で**ごろごろ**する。 (휴일에는 집에서 뒹굴뒹굴한다.) |
| うろうろ | 어슬렁어슬렁 | 知らない町を**うろうろ**した。 (낯선 마을을 어슬렁거렸다.) |
| きちんと | 깔끔하게, 단정히 | 部屋を**きちんと**片付ける。 (방을 깔끔하게 정리한다.) |
| ぐっすり | 푹 (잠자기) | 昨夜は**ぐっすり**眠れた。 (어젯밤에는 푹 잤다.) |

### 7.5 의성어·의태어의 문법

| 용법 | 패턴 | 예시 |
|-------|---------|---------|
| 부사 | 의성어·의태어 + 동사 | どきどき**する** (두근거리다) |
| な형용사 | 의성어·의태어 + な + 명사 | ふわふわ**な**ケーキ (폭신한 케이크) |
| の형용사 | 의성어·의태어 + の + 명사 | ぴかぴか**の**車 (반짝반짝한 차) |
| と부사 | 의성어·의태어 + と + 동사 | にこにこ**と**笑う (방긋방긋 웃다) |

## 8. 속담 (ことわざ)

### 8.1 인내와 노력

| 속담 | 읽기 | 의미 | 한국어 대응 표현 |
|---------|---------|---------|-------------------|
| 七転び八起き | ななころびやおき | 일곱 번 넘어져도 여덟 번 일어난다 | 칠전팔기 |
| 石の上にも三年 | いしのうえにもさんねん | 돌 위에도 삼 년 | 꾸준히 하면 성과가 있다 |
| 継続は力なり | けいぞくはちからなり | 계속은 힘이다 | 계속이 힘이다 |
| 塵も積もれば山となる | ちりもつもればやまとなる | 티끌 모아 태산 | 티끌 모아 태산 |
| 千里の道も一歩から | せんりのみちもいっぽから | 천 리 길도 한 걸음부터 | 천 리 길도 한 걸음부터 |

### 8.2 지혜와 신중

| 속담 | 읽기 | 의미 | 한국어 대응 표현 |
|---------|---------|---------|-------------------|
| 猿も木から落ちる | さるもきからおちる | 원숭이도 나무에서 떨어진다 | 원숭이도 나무에서 떨어진다 |
| 能ある鷹は爪を隠す | のうあるたかはつめをかくす | 능력 있는 매는 발톱을 숨긴다 | 빈 수레가 요란하다 (반대 표현) |
| 急がば回れ | いそがばまわれ | 급하면 돌아가라 | 급할수록 돌아가라 |
| 転ばぬ先の杖 | ころばぬさきのつえ | 넘어지기 전의 지팡이 | 유비무환 |
| 百聞は一見に如かず | ひゃくぶんはいっけんにしかず | 백 번 듣는 것은 한 번 보는 것만 못하다 | 백문이 불여일견 |

### 8.3 관계와 사람

| 속담 | 읽기 | 의미 | 한국어 대응 표현 |
|---------|---------|---------|-------------------|
| 類は友を呼ぶ | るいはともをよぶ | 유유상종 | 유유상종 |
| 人の振り見て我が振り直せ | ひとのふりみてわがふりなおせ | 남의 행동을 보고 내 행동을 고쳐라 | 타산지석 |
| 井の中の蛙大海を知らず | いのなかのかわずたいかいをしらず | 우물 안 개구리 큰 바다를 모른다 | 우물 안 개구리 |
| 郷に入っては郷に従え | ごうにいってはごうにしたがえ | 마을에 들어가면 마을 풍습을 따르라 | 로마에 가면 로마법을 따르라 |
| 親しき仲にも礼儀あり | したしきなかにもれいぎあり | 친한 사이에도 예의가 있다 | 친한 사이에도 예의가 있다 |

### 8.4 기회와 시기

| 속담 | 읽기 | 의미 | 한국어 대응 표현 |
|---------|---------|---------|-------------------|
| 鉄は熱いうちに打て | てつはあついうちにうて | 쇠는 뜨거울 때 두드려라 | 쇠뿔도 단김에 빼라 |
| 善は急げ | ぜんはいそげ | 좋은 일은 서둘러라 | 선은 급히 행해야 한다 |
| 時は金なり | ときはかねなり | 시간은 돈이다 | 시간은 금이다 |
| 花より団子 | はなよりだんご | 꽃보다 경단 | 금강산도 식후경 |

### 8.5 문화적 맥락

많은 일본어 속담은 핵심 문화 가치를 반영합니다:

| 가치 | 속담 | 교훈 |
|-------|----------|--------|
| 인내 | 石の上にも三年 / 七転び八起き | 끈기와 인내는 미덕이다 |
| 겸손 | 能ある鷹は爪を隠す | 진정한 실력은 자랑하지 않는다 |
| 화합 | 郷に入っては郷に従え / 親しき仲にも礼儀あり | 동조와 예의가 집단의 화합을 유지한다 |
| 신중 | 急がば回れ / 転ばぬ先の杖 | 신중한 계획이 문제를 방지한다 |
| 자기 개선 | 人の振り見て我が振り直せ | 타인을 관찰하여 자신을 개선한다 |

## 10. 연습 문제

### 연습 1: 신체 관용구 짝 맞추기

각 관용구를 그 의미와 짝 맞추십시오.

| | 관용구 | | 의미 |
|-|-------|-|---------|
| 1 | 目が高い | A | 대충 하다 |
| 2 | 口が軽い | B | 발목을 잡다 |
| 3 | 手を抜く | C | 안목이 좋다 |
| 4 | 足を引っ張る | D | 입이 가벼운 |
| 5 | 腹が立つ | E | 화가 나다 |

<details>
<summary>정답</summary>

1 — **C** (目が高い = 안목이 좋다 / 감식안이 있다)
2 — **D** (口が軽い = 입이 가벼운)
3 — **A** (手を抜く = 대충 하다)
4 — **B** (足を引っ張る = 발목을 잡다)
5 — **E** (腹が立つ = 화가 나다)

</details>

### 연습 2: 의성어·의태어 빈칸 채우기

각 문장에 맞는 의성어·의태어를 고르십시오.

보기: ぺらぺら, ぐずぐず, わくわく, どきどき, ぐっすり, にこにこ

1. 明日から旅行だ！______する。 (설렌다!)
2. 彼女は英語が______だ。 (그녀는 영어가 유창하다.)
3. ______しないで、早く準備して！ (꾸물거리지 말고 빨리 준비해!)
4. 発表の前は______する。 (발표 전에 두근거린다.)
5. 昨夜は______眠れた。 (어젯밤에 푹 잤다.)
6. 赤ちゃんが______笑っている。 (아기가 방긋방긋 웃고 있다.)

<details>
<summary>정답</summary>

1. **わくわく** (설레는, 두근거리는)
2. **ぺらぺら** (유창하게)
3. **ぐずぐず** (꾸물거리는)
4. **どきどき** (두근두근, 긴장)
5. **ぐっすり** (푹)
6. **にこにこ** (방긋방긋)

</details>

### 연습 3: 속담 완성하기

기억을 더듬어 각 속담을 완성하십시오.

1. 七転び ________
2. 猿も木から ________
3. 急がば ________
4. 百聞は一見に ________
5. 石の上にも ________
6. 類は友を ________

<details>
<summary>정답</summary>

1. 七転び**八起き** (일곱 번 넘어져도 여덟 번 일어난다)
2. 猿も木から**落ちる** (원숭이도 나무에서 떨어진다)
3. 急がば**回れ** (급하면 돌아가라)
4. 百聞は一見に**如かず** (백 번 듣는 것은 한 번 보는 것만 못하다)
5. 石の上にも**三年** (돌 위에도 삼 년 = 인내)
6. 類は友を**呼ぶ** (유유상종)

</details>

### 연습 4: 상황별 표현

각 상황에서 어떤 관용구나 표현을 사용하겠습니까?

1. 동료가 항상 회사의 기밀 정보를 외부인에게 이야기합니다.
2. 하루 종일 공부하고 서 있어서 다리가 지쳤습니다.
3. 과거의 앙금을 잊겠다고 말하고 싶습니다.
4. 동료가 항상 다른 사람이 필요한 것을 눈치채고 먼저 도와줍니다.
5. 항상 늦고 시간 약속을 지키지 못하는 친구를 묘사하고 싶습니다.

<details>
<summary>정답</summary>

1. **口が軽い** (입이 가벼운) — 彼は口が軽いから、秘密を言わない方がいい。
2. **足が棒になる** (다리가 뻣뻣해지다) — 一日中立っていて、足が棒になった。
3. **水に流す** (과거를 잊다) — 過去のことは水に流しましょう。
4. **気が利く** (눈치가 빠른/배려하는) — 彼女は本当に気が利く人だ。
5. **ぐずぐず** (꾸물거리는) — 彼はいつもぐずぐずしていて、約束に遅れる。

</details>

### 연습 5: 의성어·의태어 분류

각 의성어·의태어를 擬音語 (실제 소리), 擬態語 (상태/조건), 擬情語 (감정) 중 하나로 분류하십시오.

1. わくわく — 설레는
2. ワンワン — 개 짖는 소리
3. ぴかぴか — 반짝이는
4. どきどき — 두근두근
5. ザーザー — 폭우 소리
6. ふわふわ — 폭신한
7. いらいら — 짜증나는
8. ごろごろ — 천둥소리 / 뒹굴뒹굴

<details>
<summary>정답</summary>

1. **擬情語** (감정) — 설렘이라는 감정을 묘사
2. **擬音語** (소리) — 개가 짖는 소리를 흉내 냄
3. **擬態語** (상태) — 반짝이는 시각적 상태를 묘사
4. **擬情語** (감정) — 심장이 두근거리는 느낌을 묘사
5. **擬音語** (소리) — 폭우 소리를 흉내 냄
6. **擬態語** (상태) — 폭신한 물리적 성질을 묘사
7. **擬情語** (감정) — 짜증이라는 감정을 묘사
8. **擬音語** 또는 **擬態語** — 문맥에 따라 다름: 우르릉 소리 (擬音語) 또는 뒹굴뒹굴 (擬態語)

</details>

### 연습 6: 관용구를 사용하여 일본어로 번역하기

이 과에서 배운 관용구와 표현을 사용하여 번역하십시오.

1. 전문가도 때때로 실수한다.
2. 이 보고서를 내일까지 훑어봐 주십시오.
3. 저 차는 비싸서 살 여유가 없다.
4. 로마에 가면 로마법을 따르라.
5. 그녀는 세 개 국어에 유창하다.

<details>
<summary>정답</summary>

1. **猿も木から落ちる**。専門家でも時々間違える。
2. 明日までにこのレポートに**目を通して**ください。
3. あの車は高くて**手が出ない**。
4. **郷に入っては郷に従え**。
5. 彼女は三か国語が**ぺらぺら**だ。

</details>
