## 1. 일본어 문자 체계 개요

### 1.1 세 가지 문자, 하나의 언어

일본어는 주요 세계 언어 중에서 세 가지 서로 다른 문자 체계를 동시에 사용하는 유일한 언어입니다. 하나의 문장에 세 가지 문자가 모두 포함될 수 있습니다:

> **세 가지 일본어 문자 체계**
>
> ひらがな  Hiragana   — 일본 고유어, 문법
> カタカナ  Katakana   — 외래어, 강조, 이름
> 漢字      Kanji      — 중국 기원의 뜻글자
>
> 세 가지를 모두 사용하는 예문:
> 田中さんは コーヒー を 飲みました。
> (Tanaka-san wa koohii o nomimashita.)
> "다나카 씨는 커피를 마셨습니다."
>
> 田中 = 한자 (이름)
> さんは...を...ました = 히라가나 (문법/조사)
> - **コーヒー = 가타카나 (외래어**: "coffee")
> - **飲み = 한자 (동사 어간**: "마시다")

### 1.2 각 문자의 사용 시기

| 문자 | 용도 | 예시 |
|------|------|------|
| 히라가나 (ひらがな) | 고유어, 조사, 동사/형용사 어미, 아동용 텍스트 | たべる (taberu, 먹다), は (wa, 주제 표시) |
| 가타카나 (カタカナ) | 외래어, 의성어, 강조, 학명 | コンピュータ (konpyuuta, 컴퓨터), ドイツ (Doitsu, 독일) |
| 한자 (漢字) | 명사, 동사/형용사 어간, 이름, 의미 밀도가 높은 내용 | 食べる (taberu, 먹다), 日本 (Nihon, 일본) |
| 로마자 (ローマ字) | 라틴 알파벳 — 간판, 입력 방식, 외국인 학습자용 | Tokyo, sushi, karate |

### 1.3 학습 순서

```
1단계: 히라가나 익히기 (1-2주)         ← 현재 단계
2단계: 가타카나 배우기 (1-2주)
3단계: 기초 한자 시작 (지속적)
4단계: 한자 지식을 꾸준히 확장
```

> **중요**: 히라가나는 절대적으로 필수입니다. 히라가나 없이는 일본어 학습을 진행할 수 없습니다. 먼저 배우고 완벽히 익히세요.

## 2. 히라가나: 기초

### 2.1 히라가나란?

히라가나는 **음절 문자(syllabary)** 입니다 — 영어처럼 개별 소리가 아니라, 각 문자가 하나의 음절(자음 + 모음 조합)을 나타냅니다. 기본 문자는 46개입니다.

> **히라가나 핵심 사항**
>
> - **문자 수**: 기본 46 + 탁음 25 + 결합음 33 = 총 104
> - **모양**: 둥글고 유려한 필기체 스타일
> - **기원**: 헤이안 시대 여성들이 한자를 간소화하여 만듦
> - **용도**: 문법, 고유어, 후리가나 (한자 읽기 표시)
>
> 각 문자 = 하나의 음절 (모라)
> - **예**: さくら (sa-ku-ra) = 3문자 = 3음절

### 2.2 일본어 음운 체계

일본어에는 5개의 모음과 14개의 자음이 있으며, 이들이 결합하여 음절을 형성합니다:

| 모음 | 로마자 | IPA | 발음 |
|------|--------|-----|------|
| あ | a | /a/ | "아" — "father"의 "a"와 유사 |
| い | i | /i/ | "이" — "feet"의 "ee"와 유사 |
| う | u | /ɯ/ | "우" — "food"의 "oo"와 유사 (입술을 둥글게 하지 않음) |
| え | e | /e/ | "에" — "pet"의 "e"와 유사 |
| お | o | /o/ | "오" — "go"의 "o"와 유사 (짧고, 활음 없이) |

> **팁**: 일본어 모음은 순수합니다 — 영어 모음처럼 활음(glide)이 없습니다. 짧고 선명하게 발음하세요.

## 3. 히라가나 표: 46개 기본 문자

46개의 기본 히라가나는 **고주온** (五十音, "50음") 표로 정리됩니다. 전통적으로는 위에서 아래로, 오른쪽에서 왼쪽으로 읽지만, 아래 표는 왼쪽에서 오른쪽으로 정리했습니다.

### 3.1 고주온 표

| | a | i | u | e | o |
|---|---|---|---|---|---|
| **(모음)** | あ a | い i | う u | え e | お o |
| **k-** | か ka | き ki | く ku | け ke | こ ko |
| **s-** | さ sa | し shi* | す su | せ se | そ so |
| **t-** | た ta | ち chi* | つ tsu* | て te | と to |
| **n-** | な na | に ni | ぬ nu | ね ne | の no |
| **h-** | は ha | ひ hi | ふ fu* | へ he | ほ ho |
| **m-** | ま ma | み mi | む mu | め me | も mo |
| **y-** | や ya | — | ゆ yu | — | よ yo |
| **r-** | ら ra | り ri | る ru | れ re | ろ ro |
| **w-** | わ wa | — | — | — | を wo* |
| **n** | ん n | | | | |

> ***불규칙 발음**: し = "shi" ("si"가 아님), ち = "chi" ("ti"가 아님), つ = "tsu" ("tu"가 아님), ふ = "fu" ("hu"가 아님), を = "wo" (조사로만 사용되며 "o"로 발음)

### 3.2 학습 그룹

46개 문자를 관리하기 쉬운 그룹으로 나누세요:

```
그룹 1 (모음):     あ い う え お
그룹 2 (K행):      か き く け こ
그룹 3 (S행):      さ し す せ そ
그룹 4 (T행):      た ち つ て と
그룹 5 (N행):      な に ぬ ね の
그룹 6 (H행):      は ひ ふ へ ほ
그룹 7 (M행):      ま み む め も
그룹 8 (Y행):      や    ゆ    よ
그룹 9 (R행):      ら り る れ ろ
그룹 10 (W+N):     わ          を ん

전략: 하루에 1그룹씩 배우고, 이전 그룹을 매일 복습
```

### 3.3 주요 기억법

어려운 문자들을 위한 기억 보조 수단입니다:

| 문자 | 로마자 | 기억법 |
|------|--------|--------|
| あ a | a | 물구나무를 서는 **곡예사(a**crobat)처럼 보임 |
| き ki | ki | **열쇠(key)** 모양을 닮음 |
| さ sa | sa | **안장(sa**ddle)에 앉은 사람처럼 보임 |
| す su | su | **그네(su**ing/swing)를 타는 사람처럼 보임 |
| な na | na | 매**듭(kn**ot)을 묶은 모양 |
| ね ne | ne | 뱀이 있는 **둥지(ne**st)처럼 보임 |
| は ha | ha | 누군가 **하하(ha-ha)** 웃는 모양 |
| ほ ho | ho | 깃발이 달린 **성스러운(Ho**ly) 십자가 모양 |
| め me | me | 눈처럼 보임 ("look at **me**"의 me) |
| り ri | ri | **강(ri**ver)이 흐르는 모양 |
| れ re | re | 누군가 **손을 뻗는(re**aching) 모양 |
| ん n | n | 글자 "n"과 비슷한 모양 |

## 4. 탁점, 반탁점, 결합 문자

### 4.1 탁점 (゛) — 유성 자음

특정 히라가나에 두 개의 작은 획(゛)을 추가하면 자음이 **유성음화** 됩니다:

| 기본 | + 탁점 | 음 변화 |
|------|--------|---------|
| か ka → | が ga | k → g |
| き ki → | ぎ gi | k → g |
| く ku → | ぐ gu | k → g |
| け ke → | げ ge | k → g |
| こ ko → | ご go | k → g |
| さ sa → | ざ za | s → z |
| し shi → | じ ji | sh → j |
| す su → | ず zu | s → z |
| せ se → | ぜ ze | s → z |
| そ so → | ぞ zo | s → z |
| た ta → | だ da | t → d |
| ち chi → | ぢ ji* | ch → j |
| つ tsu → | づ zu* | ts → z |
| て te → | で de | t → d |
| と to → | ど do | t → d |
| は ha → | ば ba | h → b |
| ひ hi → | び bi | h → b |
| ふ fu → | ぶ bu | f → b |
| へ he → | べ be | h → b |
| ほ ho → | ぼ bo | h → b |

> ***참고**: ぢ (ji)와 づ (zu)는 거의 사용되지 않습니다. じ (ji)와 ず (zu)가 표준입니다.

### 4.2 반탁점 (゜) — P 소리

H행에 작은 동그라미(゜)를 추가하면 P 소리가 됩니다:

| 기본 | + 반탁점 | 음 변화 |
|------|----------|---------|
| は ha → | ぱ pa | h → p |
| ひ hi → | ぴ pi | h → p |
| ふ fu → | ぷ pu | f → p |
| へ he → | ぺ pe | h → p |
| ほ ho → | ぽ po | h → p |

### 4.3 요음 (拗音) — 결합 문자

특정 문자 뒤에 작은 や, ゆ, よ를 붙이면 결합음이 만들어집니다:

## 5. 가타카나: 외래어 문자

### 5.1 가타카나란?

가타카나는 두 번째 음절 문자입니다. 히라가나와 **완전히 같은 소리**를 나타내지만, 다른 맥락에서 사용됩니다.

> **가타카나 핵심 사항**
>
> - **문자 수**: 기본 46개 (히라가나와 같은 소리)
> - **모양**: 각지고 날카로운 기하학적 형태
> - **기원**: 승려들이 한자의 일부를 간소화하여 만듦
> 용도:
> - 외래어 (コンピュータ = 컴퓨터)
> - 외국 이름 (マイケル = 마이클)
> - 국가 이름 (アメリカ = 미국)
> - 의성어/의태어 (ドキドキ = 두근두근)
> - 강조 (영어의 대문자나 이탤릭체와 유사)
> - 과학/기술 용어
> - 회사/브랜드 이름

### 5.2 히라가나 vs 가타카나 비교

| 히라가나 | 가타카나 | 로마자 | 참고 |
|----------|----------|--------|------|
| あ | ア | a | 같은 소리, 다른 모양 |
| か | カ | ka | 가타카나가 더 각진 형태 |
| さ | サ | sa | |
| た | タ | ta | |
| な | ナ | na | |
| は | ハ | ha | |

### 6.2 헷갈리기 쉬운 가타카나 쌍

이 쌍들은 모양이 비슷하여 자주 실수가 발생합니다:

| 쌍 | 문자 | 구별 방법 |
|------|------|-----------|
| ア / マ | a / ma | ア는 가로 획이 있음; マ는 두 개의 아래로 향하는 획이 있음 |
| シ / ツ | shi / tsu | シ 획은 위쪽으로 향함 (↗); ツ 획은 아래쪽으로 향함 (↘) |
| ソ / ン | so / n | ソ 획은 아래쪽으로 향함 (↘); ン 획은 위쪽으로 향함 (↗) |
| ク / タ | ku / ta | タ에는 추가 가로 획이 있음 |
| ヌ / ス | nu / su | ヌ에는 교차 부분이 있음; ス에는 없음 |
| ウ / ワ | u / wa | ウ에는 위쪽 획이 있음; ワ는 위가 열려 있음 |

> **중요**: シ/ツ와 ソ/ン이 가장 많이 혼동됩니다. 즉시 구별할 수 있을 때까지 연습하세요.

### 6.3 외래 음성을 위한 확장 가타카나

현대 일본어는 일본어 고유 음에 존재하지 않는 소리를 위해 특별한 가타카나 조합을 사용합니다:

| 가타카나 | 로마자 | 용도 | 예시 |
|----------|--------|------|------|
| ティ | ti | 영어 "ti" | パーティー (paatii, 파티) |
| ディ | di | 영어 "di" | ディスク (disuku, 디스크) |
| ファ | fa | 영어 "fa" | ファイル (fairu, 파일) |
| フィ | fi | 영어 "fi" | フィルム (firumu, 필름) |
| フェ | fe | 영어 "fe" | カフェ (kafe, 카페) |
| フォ | fo | 영어 "fo" | フォーク (fooku, 포크) |

## 7. 한자 입문

### 7.1 한자란?

> **한자 핵심 사항**
>
> - **기원**: 중국 문자에서 차용
> - **총 문자 수**: ~50,000개 이상
> - **상용한자 (常用漢字)**: 2,136자 (공식 "일상 사용" 세트)
> - **JLPT N5 한자**: ~100자
> - **JLPT N1 한자**: ~2,000자
>
> 각 한자에는 다음이 있습니다:
> - 의미
> - 음독 (音読み) — 중국어에서 유래한 읽기
> - 훈독 (訓読み) — 일본 고유 읽기
> - 필순 — 정해진 순서대로 써야 함
> - 부수 — 구성 요소

### 7.2 기본 한자 예시

| 한자 | 의미 | 음독 | 훈독 | 획수 | 예시 단어 |
|------|------|------|------|------|-----------|
| 一 | 하나 | イチ (ichi) | ひと (hito) | 1 | 一つ (hitotsu, 한 개) |
| 二 | 둘 | ニ (ni) | ふた (futa) | 2 | 二つ (futatsu, 두 개) |
| 三 | 셋 | サン (san) | み (mi) | 3 | 三つ (mittsu, 세 개) |
| 人 | 사람 | ジン (jin) | ひと (hito) | 2 | 日本人 (nihonjin, 일본인) |
| 日 | 날/해 | ニチ (nichi) | ひ (hi) | 4 | 日曜日 (nichiyoubi, 일요일) |
| 本 | 책/근본 | ホン (hon) | もと (moto) | 5 | 日本 (nihon, 일본) |
| 山 | 산 | サン (san) | やま (yama) | 3 | 山田 (Yamada, 성) |
| 川 | 강 | セン (sen) | かわ (kawa) | 3 | 川 (kawa, 강) |
| 大 | 크다 | ダイ (dai) | おお (oo) | 3 | 大きい (ookii, 큰) |
| 小 | 작다 | ショウ (shou) | ちい (chii) | 3 | 小さい (chiisai, 작은) |

### 7.3 일본어에 한자가 존재하는 이유

히라가나로 모든 것을 쓸 수 있는데 왜 한자를 사용할까? 세 가지 실용적 이유가 있습니다:

1. **동음이의어 구별**: 일본어에는 동음이의어가 많습니다. 橋 (hashi, 다리) vs 箸 (hashi, 젓가락) vs 端 (hashi, 끝)
2. **읽기 속도**: 한자는 시각적으로 밀도가 높습니다. 東京都千代田区는 とうきょうとちよだく보다 빠르게 읽힙니다
3. **간결함**: 한자는 의미를 적은 문자로 압축합니다. 学校 (gakkou, 학교) = 2문자 vs がっこう = 4문자

## 8. 부수: 한자의 구성 요소

### 8.1 부수란?

부수 (部首, bushu)는 한자를 구성하는 요소입니다. 레고 블록처럼 생각하면 됩니다. 부수를 배우면 다음에 도움이 됩니다:

- **한자 기억** — 복잡한 문자를 익숙한 부분으로 나누기
- **한자 찾기** — 사전은 부수별로 정리되어 있음
- **의미 추측** — 부수는 종종 의미 범주를 암시함

### 8.2 주요 부수

| 부수 | 이름 | 의미 | 포함되는 한자 |
|------|------|------|--------------|
| 人 (亻) | にんべん | 사람 | 休 (쉬다), 体 (몸), 何 (무엇) |
| 口 | くちへん | 입 | 食 (먹다), 言 (말하다), 名 (이름) |
| 水 (氵) | さんずい | 물 | 海 (바다), 泳 (수영), 酒 (술) |
| 木 | きへん | 나무 | 森 (숲), 本 (책), 校 (학교) |
| 日 | にちへん | 해/날 | 時 (시간), 明 (밝다), 曜 (요일) |
| 火 (灬) | ひへん | 불 | 焼 (태우다), 点 (점), 熱 (열) |
| 土 | つちへん | 흙 | 地 (땅), 場 (장소) |
| 女 | おんなへん | 여자 | 好 (좋아하다), 妹 (여동생) |
| 金 | かねへん | 금속/금 | 銀 (은), 鉄 (철) |
| 言 (訁) | ごんべん | 말 | 話 (이야기하다), 読 (읽다), 語 (언어) |

### 8.3 부수의 위치

부수는 한자 내에서 특정 위치에 나타납니다:

> **한자에서의 부수 위치**
>
> 偏 (hen)  — 왼쪽       亻 in 休
> 旁 (tsukuri) — 오른쪽  力 in 動
> 冠 (kanmuri) — 위쪽    雨 in 雪
> 脚 (ashi) — 아래쪽     心 in 思
> 構 (kamae) — 둘러싸기  門 in 間
> 垂 (tare) — 왼쪽 위 드리움 广 in 店
> 繞 (nyou) — 왼쪽 아래 감싸기 辶 in 道

## 9. 음독과 훈독: 두 가지 읽기 체계

### 9.1 이중 읽기 문제

모든 한자에는 최소 두 가지 읽는 방법이 있으며, 때로는 그 이상인 경우도 있습니다. 이것이 한자에서 가장 어려운 부분입니다.

> **음독 vs 훈독**
>
> 음독 (音読み) — "소리 읽기"
> - 중국어에서 유래한 발음
> - 복합어 (숙어)에서 사용
> - 사전에서 가타카나로 표기
> - **• 예**: 山 = サン (SAN) → 山脈 (sanmyaku, 산맥)
>
> 훈독 (訓読み) — "뜻 읽기"
> - 일본 고유의 발음
> - 한자가 단독으로 사용되거나 히라가나 어미가 붙을 때 사용
> - 사전에서 히라가나로 표기
> - **• 예**: 山 = やま (yama) → 山 (yama, 산)

### 9.2 어떤 읽기를 사용할지

| 상황 | 읽기 | 예시 |
|------|------|------|
| 한자 단독 (또는 + 히라가나) | 훈독 | 山 (yama), 食べる (taberu) |
| 2개 이상의 한자 복합어 | 음독 | 火山 (kazan, 화산), 食事 (shokuji, 식사) |
| 이름 | 둘 다 가능 | 山田 (Yamada), 山本 (Yamamoto) |

### 9.3 예시: 한자 食

| 읽기 | 유형 | 상황 | 단어 |
|------|------|------|------|
| ショク (shoku) | 음독 | 복합어에서 | 食事 (shokuji, 식사), 食堂 (shokudou, 식당) |
| た.べる (taberu) | 훈독 | 히라가나 어미와 함께 | 食べる (taberu, 먹다) |
| く.う (kuu) | 훈독 | 구어체/남성어 | 食う (kuu, 먹다) |

> **초보자를 위한 팁**: 읽기를 단독으로 외우려 하지 마세요. 어휘를 통해 읽기를 배우세요. 食べる (taberu)라는 단어를 배우면, 자연스럽게 食의 읽기가 た라는 것을 알게 됩니다.

## 10. 쓰기 연습 안내

### 10.1 필순 규칙

일본어 문자는 정해진 필순으로 써야 합니다. 기본 규칙은 다음과 같습니다:

```
규칙 1: 위에서 아래로              규칙 5: 왼쪽 빗금 → 오른쪽 빗금
        一 (가로 획 먼저)                  人 (왼쪽 \ → 오른쪽 /)

규칙 2: 왼쪽에서 오른쪽으로         규칙 6: 중심 → 양쪽 날개
        川 (왼쪽 획 먼저)                  小 (중심 | → 왼쪽, 오른쪽)

규칙 3: 가로 → 세로                규칙 7: 둘러싼 부분 → 안쪽 내용
        十 (— → |)                        国 (바깥 口 → 안쪽 玉)

규칙 4: 바깥 → 안쪽                규칙 8: 아래 닫는 획은 마지막
        月 (바깥 획 먼저)                  日 (아래 — 이 상자를 닫음)
```

### 10.2 연습 전략

> **권장 학습 일정**
>
> - **1-2주차**: 히라가나
> - 하루에 5-10자씩 배우기
> - 각 문자를 10회 이상 쓰기
> - 플래시카드로 인식 연습
> - 간단한 단어 읽기 연습
>
> - **3-4주차**: 가타카나
> - 히라가나와 같은 방법 적용
> - 알고 있는 외래어로 연습
> - 비슷한 모양의 쌍 비교
>
> - **5주차 이후**: 한자 시작
> - 하루에 3-5자의 새 한자 배우기
> - 항상 문맥(어휘)과 함께 배우기
> - 간격 반복법으로 복습
> - 부수에 주의하기
>
> - **도구**: 원고용지 (genkouyoushi 原稿用紙), 플래시카드 앱
> (Anki), 쓰기 연습 시트

### 10.3 초보자가 흔히 하는 실수

| 실수 | 문제 | 해결 방법 |
|------|------|-----------|
| 필순 무시 | 문자가 이상하게 보이고, 읽기 어려움 | 처음부터 올바른 순서를 배우기 |
| 히라가나 건너뛰기 | 아무것도 읽을 수 없고, 로마자에 의존 | 히라가나를 다른 무엇보다 먼저 완벽히 익히기 |
| 단어 없이 한자 배우기 | 읽기를 외우지만 사용할 수 없음 | 항상 어휘를 통해 한자 배우기 |
| 비슷한 가나 혼동 | は/ほ, わ/れ/ね, シ/ツ, ソ/ン | 쌍을 나란히 놓고 연습하기 |
| 로마자를 보조 수단 이상으로 사용 | 진정한 읽기 능력이 발달하지 않음 | 가능한 빨리 가나만으로 전환하기 |
