## 1. ので — "~이므로" (정중)

ので는 **이유나 원인**을 정중하고 객관적인 어조로 표현합니다. から보다 부드럽고 배려 있는 표현입니다.

### 접속 방법

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (보통체) | + ので | 行く**ので** |
| い형용사 | + ので | 高い**ので** |
| な형용사 | **な** + ので | 静か**なので** |
| 명사 | **な** + ので | 学生**なので** |
| 과거 | た + ので | 行った**ので** |
| 부정 | ない + ので | 行かない**ので** |

> **핵심**: な형용사와 명사는 ので 앞에 **な**를 사용합니다 (だ가 아님).

### ので vs. から

| 특징 | ので | から |
|---------|------|------|
| 어조 | 부드러움, 객관적, 정중 | 직접적, 주관적, 단정적 |
| 격식 | 더 격식적 | 더 비격식적 |
| 명령과 함께 | 자연스러움 | 자연스러움 |
| 요청과 함께 | 선호됨 (더 부드러움) | 괜찮지만 강압적으로 들릴 수 있음 |
| 뉘앙스 | "~인 만큼..." | "~이니까...!" |

```
○ 雨が降っているので、傘を持って行ったほうがいいですよ。
  비가 오고 있으므로, 우산을 가져가는 게 좋겠습니다. (부드러운 조언)

○ 雨が降っているから、傘を持って行け！
  비가 오니까 우산 가져가! (직접적 명령)
```

### ので를 사용하는 경우

- 정중하거나 격식적인 상황
- 윗사람이나 모르는 사람에게 이유를 말할 때
- 변명이나 부드러운 설명을 할 때
- 서면 일본어에서

### 예문

1. **明日テストがあるので、今日は早く寝ます。** — 내일 시험이 있으므로 오늘은 일찍 자겠습니다.
2. **道が混んでいたので、遅れました。** — 길이 막혀서 늦었습니다.
3. **日本語が好きなので、毎日勉強しています。** — 일본어를 좋아하므로 매일 공부하고 있습니다.
4. **学生なので、割引がありますか。** — 학생이므로, 할인이 있습니까?
5. **体調が悪いので、今日は休ませてください。** — 컨디션이 안 좋으므로 오늘 쉬게 해주세요.

## 2. のに — "~인데도 / ~에도 불구하고"

のに는 **대조 또는 좌절**을 표현합니다 — 조건을 고려하면 결과가 예상 밖이거나 실망스러운 경우입니다.

### 접속 방법

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (보통체) | + のに | 勉強した**のに** |
| い형용사 | + のに | 高い**のに** |
| な형용사 | **な** + のに | 静か**なのに** |
| 명사 | **な** + のに | 日曜日**なのに** |

> ので와 같은 접속 규칙입니다: な형용사와 명사는 のに 앞에 **な**를 사용합니다.

### 뉘앙스: 좌절과 놀라움

のに는 기대대로 되지 않은 것에 대한 **불만**, **후회**, **놀라움**의 감정을 담는 경우가 많습니다:

```
せっかく作ったのに、誰も食べなかった。
모처럼 만들었는데, 아무도 먹지 않았다. (좌절)

たくさん勉強したのに、テストに落ちた。
많이 공부했는데도 시험에 떨어졌다. (실망)
```

### のに vs. けど / が

| 표현 | 어조 | 예시 |
|-----------|------|---------|
| のに | 좌절 / 감정적 대조 | 約束したのに、来なかった。(약속했는데 안 왔어! — 짜증) |
| けど / けれども | 중립적 대조 | 約束したけど、来なかった。(약속했지만, 오지 않았다. — 중립) |
| が | 격식적 중립 대조 | 連絡しましたが、返事がありませんでした。(연락했지만, 답이 없었습니다.) |

### 문말 のに

のに로 문장을 끝내면, 여운을 남기며 좌절감을 암시합니다:

> **せっかく来たのに...** — 모처럼 왔는데... (암시된 실망)
> **教えてくれればよかったのに。** — 알려줬으면 좋았을 텐데. (알려주지 않아서 아쉬움.)

### 예문

1. **薬を飲んだのに、まだ頭が痛い。** — 약을 먹었는데도 아직 머리가 아프다.
2. **日曜日なのに、仕事がある。** — 일요일인데도 일이 있다.
3. **高いのに、あまりおいしくない。** — 비싼데도 별로 맛없다.
4. **何度も練習したのに、上手にできなかった。** — 여러 번 연습했는데도 잘하지 못했다.
5. **せっかくの休みなのに、雨だ。** — 모처럼의 휴일인데 비가 온다.

## 3. ために — "~하기 위해 / ~때문에"

ために는 앞에 오는 형태에 따라 **두 가지 다른 의미**를 가집니다.

### 의미 1: 목적 — "~하기 위해"

**사전형 동사** 또는 **명사 + の**가 앞에 올 때:

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (사전형) | + ために | 合格する**ために** (~에 합격하기 위해) |
| 명사 + の | + ために | 試験の**ために** (시험을 위해) |

#### 예문 (목적)

> **日本語を勉強するために、日本に来ました。** — 일본어를 공부하기 위해 일본에 왔습니다.
> **健康のために、毎日運動しています。** — 건강을 위해 매일 운동하고 있습니다.
> **将来のために、お金を貯めている。** — 미래를 위해 돈을 모으고 있다.

### 의미 2: 원인/이유 — "~때문에"

**과거형 동사**, **い형용사**, **な형용사 + な**, **명사 + の**가 앞에 올 때:

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (과거) | + ために | 雨が降った**ために** (비가 왔기 때문에) |
| い형용사 | + ために | 道が狭い**ために** (길이 좁기 때문에) |
| な형용사 + **な** | + ために | 不便**なために** (불편하기 때문에) |
| 명사 + **の** | + ために | 台風の**ために** (태풍 때문에) |

> **원인의 ために**는 ので보다 더 격식적이며, 뉴스와 서면 일본어에서 흔합니다.

#### 예문 (원인)

> **事故があったために、電車が遅れている。** — 사고가 있었기 때문에 전철이 늦어지고 있다.
> **台風のために、飛行機が欠航になった。** — 태풍 때문에 비행기가 결항되었다.
> **風邪を引いたために、学校を休んだ。** — 감기에 걸렸기 때문에 학교를 쉬었다.

### 목적과 원인 구분

```
+------------------------------------------------------------------+
|  목적 (~하기 위해):   사전형 + ために                              |
|  → 日本語を話すために (일본어를 말하기 위해)                      |
|                                                                  |
|  원인 (~때문에):       과거형 / 상태 + ために                      |
|  → 日本語を話したために (일본어를 말했기 때문에)                  |
+------------------------------------------------------------------+
```

### 예문 (혼합)

1. **試験に合格するために、毎日3時間勉強している。** — 시험에 합격하기 위해 매일 3시간 공부하고 있다. (목적)
2. **大雪のために、学校が休みになった。** — 폭설 때문에 학교가 휴교가 되었다. (원인)
3. **家族のために、一生懸命働いている。** — 가족을 위해 열심히 일하고 있다. (목적)
4. **工事のために、この道は通れません。** — 공사 때문에 이 길은 통행할 수 없습니다. (원인)
5. **夢を叶えるために、日本に来た。** — 꿈을 이루기 위해 일본에 왔다. (목적)

## 4. ように — "~하도록 / ~하게"

ように는 **목표 상태**나 **행위의 양태**를 표현합니다. ために와 달리, ように의 목표는 일반적으로 화자가 직접 통제할 수 없는 것입니다.

### 접속 방법

| 유형 | 접속 | 예시 |
|------|-----------|---------|
| 동사 (사전형) | + ように | 見える**ように** (보이도록) |
| 동사 (가능형) | + ように | 話せる**ように** (말할 수 있도록) |
| 동사 (ない) | + ように | 忘れない**ように** (잊지 않도록) |

### 용법

#### A. 목표 / 소원 ("~하도록")

비의지적 동사(가능형, 자동사, 상태를 표현하는 동사)와 함께 사용됩니다:

> **日本語が話せるように、毎日練習しています。**
> 일본어를 말할 수 있도록 매일 연습하고 있습니다.

> **遅刻しないように、早く起きた。**
> 지각하지 않도록 일찍 일어났다.

> **試験に受かるように、頑張る。**
> 시험에 붙도록 열심히 하겠다.

#### B. 양태 ("~하게 / ~처럼")

> **子供にも分かるように説明した。**
> 아이도 알 수 있게 설명했다.

> **聞こえるように大きな声で話した。**
> 들리도록 큰 소리로 말했다.

#### C. ようにする — "~하도록 노력하다 / ~하게 하다"

> **毎日野菜を食べるようにしている。** — 매일 채소를 먹도록 노력하고 있다.
> **遅刻しないようにしてください。** — 지각하지 않도록 해주세요.

#### D. ようになる — "~하게 되다 / ~할 수 있게 되다"

**상태 변화**나 **능력 습득**을 표현합니다:

> **日本語が話せるようになった。** — 일본어를 말할 수 있게 되었다.
> **納豆が食べられるようになった。** — 낫토를 먹을 수 있게 되었다.
> **毎朝走るようになった。** — 매일 아침 달리게 되었다.

### ために vs. ように

| 특징 | ために | ように |
|---------|--------|--------|
| 동사 유형 | 의지적 (통제 가능) | 비의지적 (가능, 부정, 상태) |
| 주어 | 양쪽 절의 주어 동일 | 다른 주어 가능 |
| 뉘앙스 | 직접적 목적 | 목표 상태 / 소원 |
| 예시 | 合格する**ために**勉強する | 合格できる**ように**祈る |
| 번역 | 합격**하기 위해** 공부하다 | 합격할 수 있**도록** 기도하다 |

```
○ 日本語を勉強するために日本に来た。  (직접적 목적 — 공부는 내가 통제 가능)
○ 日本語が話せるようになるために勉強する。 (상태 변화를 향한 목적)
○ 日本語が上手になるように毎日練習する。  (목표 상태 — 덜 직접적인 통제)
```

### 예문

1. **風邪を引かないように気をつけている。** — 감기에 걸리지 않도록 조심하고 있다.
2. **忘れないようにメモした。** — 잊지 않도록 메모했다.
3. **ピアノが弾けるようになりたい。** — 피아노를 칠 수 있게 되고 싶다.
4. **みんなに見えるように大きく書いた。** — 모두에게 보이도록 크게 썼다.
5. **健康に気をつけるようにしています。** — 건강에 신경 쓰도록 하고 있습니다.

## 5. ながら — "~하면서"

ながら는 **같은 주어**가 수행하는 **두 가지 동시 행위**를 표현합니다.

### 접속 방법

| 유형 | 규칙 | 예시 |
|------|------|---------|
| 동사 | ます형 어간 + ながら | 食べ**ながら**, 歩き**ながら**, 聞き**ながら** |

> ます형에서 ます를 빼고(ます형 어간) ながら를 붙이면 됩니다.

### 구조

```
[부차적 행위] ながら、[주된 행위]。

주된 행위가 두 번째 절에 옵니다.
ながら절은 동시에 하는 것을 묘사합니다.
```

> **音楽を聞きながら、勉強する。**
> 음악을 들으면서(부차적) 공부한다(주된).

### 핵심 규칙: 같은 주어

양쪽 행위의 주어가 반드시 **같은 사람**이어야 합니다:

```
○ 私は歩きながら、電話で話した。 (나는 걸으면서 전화로 말했다.)
✗ 私が歩きながら、彼が走った。  (주어가 다름 — 間/間に를 대신 사용)
```

### 예문

1. **テレビを見ながら、ご飯を食べる。** — 텔레비전을 보면서 밥을 먹는다.
2. **コーヒーを飲みながら、新聞を読んだ。** — 커피를 마시면서 신문을 읽었다.
3. **歩きながらスマホを見るのは危ない。** — 걸으면서 스마트폰을 보는 것은 위험하다.
4. **笑いながら話す。** — 웃으면서 말한다.
5. **音楽を聴きながら寝るのが好きだ。** — 음악을 들으면서 자는 것을 좋아한다.

### 확장 용법: ながら(も) — "~이면서도" (양보)

ながら는 대조적 뉘앙스로 "~이면서도" 또는 "~에도 불구하고"의 의미로도 쓰입니다. 이것은 더 문어적인 용법입니다:

> **知っていながら、何も言わなかった。** — 알고 있으면서도 아무 말도 하지 않았다.
> **残念ながら、参加できません。** — 유감이지만 참가할 수 없습니다.
> **小さいながらも、立派な家だ。** — 작으면서도 훌륭한 집이다.

## 6. たり...たりする — "~하기도 하고 ~하기도 하다"

たり...たりする는 모든 것을 나열하지 않고 **대표적 행위**를 나열합니다 — "X도 하고 Y도 하고 한다."

### 접속 방법

| 유형 | 규칙 | 예시 |
|------|------|---------|
| 동사 | た형에서 た→たり | 食べ**たり**, 読ん**だり**, 行っ**たり** |
| い형용사 | かったり | 高かっ**たり**, 安かっ**たり** |
| な형용사 | だったり | 静かだっ**たり**, にぎやかだっ**たり** |
| 명사 | だったり | 学生だっ**たり**, 先生だっ**たり** |

### 구조

```
[행위 1]たり、[행위 2]たり する。

끝의 する는 시제와 정중도에 따라 활용합니다:
  します (정중체), した (과거), している (진행)
```

### 용법

#### A. 대표적 행위 나열

> **週末は映画を見たり、買い物をしたりする。**
> 주말에는 영화를 보거나 쇼핑을 하거나 한다.

> **休みの日は本を読んだり、音楽を聴いたりしています。**
> 쉬는 날에는 책을 읽거나 음악을 듣거나 하고 있습니다.

#### B. 변동 / 교대 표현

> **天気が良かったり悪かったりする。** — 날씨가 좋기도 하고 나쁘기도 한다.
> **彼の気分は変わったりする。** — 그의 기분은 변하기도 한다.

#### C. 단일 たり (대표적 행위 하나)

구어체에서는 たり 하나만 쓰는 것이 흔합니다:

> **昨日は友達と遊んだりした。** — 어제는 친구와 놀기도 했다.

### 주요 포인트

- 나열은 **전부가 아닙니다** — たり는 "등등"을 암시합니다
- 격식적 용법에서는 최소 두 개의 たり 항목이 표준입니다
- 끝에 する가 필수입니다 (매우 비격식적인 말에서는 생략되기도 함)

### 예문

1. **日本では寿司を食べたり、温泉に入ったりしたい。** — 일본에서는 초밥을 먹기도 하고 온천에 들어가기도 하고 싶다.
2. **毎日、走ったり泳いだりして運動している。** — 매일 달리기도 하고 수영하기도 하면서 운동하고 있다.
3. **彼は怒ったり笑ったり、忙しい人だ。** — 그는 화내기도 하고 웃기도 하고, 바쁜 사람이다.
4. **このクラスは簡単だったり難しかったりする。** — 이 수업은 쉽기도 하고 어렵기도 하다.
5. **子供たちは公園で遊んだり、絵を描いたりしていた。** — 아이들은 공원에서 놀기도 하고 그림을 그리기도 하고 있었다.

## 7. 추가 절 연결 표현

### し — 이유 나열 (누적)

し는 결론을 뒷받침하는 **여러 이유**를 나열합니다:

| 접속 | 예시 |
|-----------|---------|
| 보통체 + し | 安いし、おいしいし |

> **この店は安いし、おいしいし、最高だ。**
> 이 가게는 싸고, 맛있고, 최고다.

> **雨だし、寒いし、家にいよう。**
> 비도 오고, 춥기도 하니까 집에 있자.

### くせに — "~인 주제에" (비판)

くせに는 비판이나 비난을 표현합니다 — のに보다 더 강하고 부정적입니다:

> **知っているくせに、教えてくれない。**
> 알고 있는 주제에 알려주지 않는다. (짜증!)

> **日本人のくせに、敬語が使えない。**
> 일본인인 주제에 경어를 사용할 줄 모른다. (비판적 어조)

### ものの — "~이기는 하지만" (격식/문어)

のに보다 더 격식적이며, 서면 일본어에서 사용됩니다:

> **申し込んだものの、まだ返事が来ない。**
> 신청하기는 했지만, 아직 답이 오지 않는다.

### うちに — "~하는 동안에 / ~하기 전에 (상황이 바뀌기 전에)"

상황이 변하기 전에 무언가를 하는 것을 암시합니다:

> **暗くならないうちに帰ろう。** — 어두워지기 전에 돌아가자.
> **若いうちに旅行したほうがいい。** — 젊을 때 여행하는 게 좋다.
> **日本にいるうちに、日本語を上達させたい。** — 일본에 있는 동안 일본어를 향상시키고 싶다.

### 上で — "~한 다음에 / ~한 후에"

한 행위가 완료된 후에 다음 행위가 이어짐을 나타냅니다:

> **よく考えた上で、決めます。** — 잘 생각한 후에 결정하겠습니다.
> **説明を聞いた上で、質問があればどうぞ。** — 설명을 들은 후에, 질문이 있으시면 말씀해 주세요.

## 8. 비교 도표: 이유와 목적

### 이유 표현

| 표현 | 어조/격식 | 예시 |
|-----------|--------------|---------|
| から | 비격식, 주관적 | 暑い**から**、窓を開けた。 |
| ので | 정중, 객관적 | 暑い**ので**、窓を開けました。 |
| ために | 격식, 서면체 | 暑かった**ために**、体調を崩した。 |
| し | 누적 이유 | 暑い**し**、湿気もある**し**。 |

### 목적 표현

| 표현 | 사용 대상 | 예시 |
|-----------|-----------|---------|
| ために | 의지적 동사 (직접 목적) | 留学する**ために**勉強する。 |
| ように | 비의지적 동사 (목표 상태) | 話せる**ように**練習する。 |
| のに | "~하는 데" (용도) | この道具は料理する**のに**便利だ。 |

### 대조 표현

| 표현 | 어조 | 예시 |
|-----------|------|---------|
| けど / けれども | 중립 | 勉強した**けど**、落ちた。 |
| のに | 좌절/감정적 | 勉強した**のに**、落ちた。 |
| が | 격식적 중립 | 勉強しました**が**、落ちました。 |
| くせに | 비판/비난 | 勉強した**くせに**、落ちた。 |
| ものの | 격식/문어 | 勉強した**ものの**、落ちた。 |

## 9. 흔한 실수와 주의점

### 실수 1: な형용사/명사와 ので의 잘못된 접속

```
✗ 静かだので / 学生だので
○ 静かなので / 学生なので  (ので 앞에 だ가 아닌 な 사용)
```

### 실수 2: ために의 목적과 원인 혼동

```
목적:  試験に合格するために勉強する。 (사전형 = 목적)
원인:  試験に合格したために喜んでいる。 (과거형 = 원인)

✗ 試験に合格したために勉強する。 (과거 + ために는 목적이 될 수 없음)
```

### 실수 3: 비의지적 동사에 ために를 목적으로 사용

```
✗ 日本語が話せるために勉強する。
○ 日本語が話せるように勉強する。  (가능 동사 = ように 사용)

규칙: ために (목적)에는 의지적 동사가 필요합니다.
      ように는 비의지적 동사(가능형, 자동사, 부정형)에 사용합니다.
```

### 실수 4: ながら에서 주어가 다른 경우

```
✗ 母が料理しながら、父がテレビを見ている。 (주어가 다름)
○ 母が料理している間、父はテレビを見ている。 (주어가 다르면 間 사용)
```

### 실수 5: たり에서 する 누락

```
✗ 週末は映画を見たり、買い物をしたり。     (する 누락)
○ 週末は映画を見たり、買い物をしたりする。 (する 필수)
```

> 매우 비격식적인 말에서는 する가 생략될 수 있지만, 표준/서면 일본어에서는 포함합니다.

## 10. 연습 문제

### 연습 1: ので 연습

ので를 사용하여 완성하세요.

1. 明日早い ________ 、今日は早く寝ます。(내일 이르므로)
2. 日本語が好き ________ 、毎日勉強しています。(일본어를 좋아하므로)
3. 雨が降っている ________ 、タクシーで行きましょう。(비가 오고 있으므로)
4. 学生 ________ 、お金がありません。(학생이므로)

<details>
<summary>정답</summary>

1. 明日早い**ので** (い형용사: 早い + ので)
2. 日本語が好き**なので** (な형용사: 好き + なので)
3. 雨が降っている**ので** (동사: 降っている + ので)
4. 学生**なので** (명사: 学生 + なので)

</details>

### 연습 2: のに 연습

のに를 사용하여 좌절감을 표현하세요.

1. 많이 공부했는데 시험에 떨어졌다.
2. 일요일인데도 일을 해야 한다.
3. 초대했는데 오지 않았다.
4. 비쌌는데도 쉽게 부서졌다.

<details>
<summary>정답</summary>

1. たくさん勉強した**のに**、テストに落ちた。
2. 日曜日**なのに**、仕事をしなければならない。
3. 誘った**のに**、来なかった。
4. 高かった**のに**、すぐ壊れた。

</details>

### 연습 3: ために — 목적인가, 원인인가?

ために가 목적인지 원인인지 식별하고 번역하세요.

1. 試験に合格するために、毎日勉強している。
2. 台風のために、学校が休みになった。
3. 日本語を学ぶために、日本に来ました。
4. 事故があったために、電車が止まっている。

<details>
<summary>정답</summary>

1. **목적** — 시험에 합격하기 위해 매일 공부하고 있다.
2. **원인** — 태풍 때문에 학교가 휴교가 되었다.
3. **목적** — 일본어를 배우기 위해 일본에 왔습니다.
4. **원인** — 사고가 있었기 때문에 전철이 멈춰 있다.

</details>

### 연습 4: ために vs. ように

ために 또는 ように로 빈칸을 채우세요.

1. 日本語が上手になる ________ 、毎日練習している。(실력이 늘다 — 상태)
2. 新しい車を買う ________ 、お金を貯めている。(사다 — 의지적)
3. 遅刻しない ________ 、早く起きた。(지각하지 않다 — 부정)
4. 留学する ________ 、奨学金に申し込んだ。(유학하다 — 의지적)

<details>
<summary>정답</summary>

1. 日本語が上手になる**ように** (상태 변화 / 비의지적 → ように)
2. 新しい車を買う**ために** (의지적 행위 → ために)
3. 遅刻しない**ように** (부정형 → ように)
4. 留学する**ために** (의지적 행위 → ために)

</details>

### 연습 5: ながら 연습

ながら를 사용하여 하나의 문장으로 합치세요.

1. 音楽を聴く + 料理する = ________
2. コーヒーを飲む + 仕事をする = ________
3. テレビを見る + 宿題をする = ________
4. 歩く + 考える = ________

<details>
<summary>정답</summary>

1. 音楽を聴き**ながら**、料理する。(음악을 들으면서 요리한다.)
2. コーヒーを飲み**ながら**、仕事をする。(커피를 마시면서 일한다.)
3. テレビを見**ながら**、宿題をする。(텔레비전을 보면서 숙제를 한다.)
4. 歩き**ながら**、考える。(걸으면서 생각한다.)

</details>

### 연습 6: たり...たりする 연습

たり...たりする를 사용하여 활동을 묘사하세요.

1. 주말에: 영화를 보다, 책을 읽다
2. 일본에서: 초밥을 먹다, 절을 방문하다
3. 집에서: 청소하다, 빨래하다
4. 날씨: 때때로 덥다, 때때로 춥다

<details>
<summary>정답</summary>

1. 週末は映画を見**たり**、本を読ん**だりする**。
2. 日本では寿司を食べ**たり**、お寺を訪れ**たりした**。
3. 家では掃除し**たり**、洗濯し**たりする**。
4. 天気は暑かっ**たり**、寒かっ**たりする**。

</details>

### 연습 7: 혼합 — 가장 적합한 접속사 선택

가장 적절한 접속사를 선택하세요.

1. 天気がいい ________ 、散歩に行きましょう。
   a. ので b. のに c. ながら d. ために

2. 薬を飲んだ ________ 、まだ頭が痛い。
   a. ので b. のに c. し d. ために

3. 忘れない ________ メモしておこう。
   a. ために b. ように c. のに d. ながら

4. 歩き ________ 食べるのはマナーが悪い。
   a. たり b. ので c. ながら d. のに

5. 休みの日は本を読ん ________ 映画を見 ________ する。
   a. で...て b. たり...たり c. ながら...ながら d. ので...ので

<details>
<summary>정답</summary>

1. **a. ので** (정중한 이유: "날씨가 좋으므로")
2. **b. のに** (좌절: "약을 먹었는데도")
3. **b. ように** (부정형 동사의 목적: "잊지 않도록")
4. **c. ながら** (동시 행위: "걸으면서")
5. **b. たり...たり** (대표적 활동 나열)

</details>

### 연습 8: 번역

지시된 패턴을 사용하여 일본어로 번역하세요.

1. 비가 오고 있으므로 택시를 탑시다. (ので)
2. 그는 부자인데도 행복하지 않다. (のに)
3. 여행하기 위해 돈을 모으고 있다. (ために)
4. 피아노를 칠 수 있도록 매일 연습하고 있다. (ように)
5. 책을 읽다가 잠들어 버렸다. (ながら)
6. 주말에는 쇼핑도 하고 친구도 만나고 한다. (たり)

<details>
<summary>정답</summary>

1. 雨が降っている**ので**、タクシーに乗りましょう。
2. お金持ちな**のに**、幸せではない。
3. 旅行する**ために**、お金を貯めている。
4. ピアノが弾ける**ように**、毎日練習している。
5. 本を読み**ながら**、寝てしまった。
6. 週末は買い物をし**たり**、友達に会っ**たりする**。

</details>
