- 조사 で의 장소, 수단, 이유 용법을 완벽히 익힌다

## 1. 조사 개요 및 복습

일본어 조사(助詞 / joshi)는 명사, 동사 또는 구 뒤에 붙어 문법적 기능을 나타내는 짧은 단어입니다. 각 단어가 문장의 나머지 부분과 어떤 관계인지를 알려주는 이정표라고 생각하면 됩니다.

### 1.1 기본 조사 빠른 복습

| 조사 | 기능 | 예시 |
|----------|----------|---------|
| は (wa) | 주제 표시 | 私**は**学生です。(저는 학생입니다.) |
| が | 주어 표시 | 猫**が**いる。(고양이가 있다.) |
| を (o) | 직접 목적어 | 本**を**読む。(책을 읽다.) |
| に | 방향 / 시간 / 존재 장소 | 学校**に**行く。(학교에 가다.) |
| へ (e) | 방향 (이동에서 に와 교환 가능) | 東京**へ**行く。(도쿄에 가다.) |

### 1.2 조사 기능 맵

> **조사 기능 맵 (PARTICLE FUNCTION MAP)**
>
> 누가/무엇을
> は  주제 ("~에 관해서는...")
> が  주어 (새로운 정보 / 초점 / 존재)
> を  직접 목적어
>
> 어디서 / 언제
> に  시점, 대상, 존재의 장소
> で  동작의 장소, 수단, 재료
> へ  이동의 방향
>
> 범위
> から  시작점 (~에서, ~부터)
> まで  끝점 (~까지)
>
> 연결
> と  완전 나열 "~와", "~과 함께", 인용
> や  비완전 나열 "~와 ~등"
> も  "~도", "~조차"
>
> 수식
> の  소유, 명사 수식, 명사화
>

## 2. で — 장소, 수단, 이유

조사 で (de)는 가장 다재다능한 조사 중 하나입니다. 세 가지 주요 용법이 있습니다.

### 2.1 동작의 장소

で는 **동작**이 이루어지는 장소를 표시합니다. 이것은 무언가가 **존재**하는 장소를 표시하는 に와 다릅니다.

```
図書館で勉強する。
Toshokan de benkyō suru.
도서관에서 공부한다.

レストランで食べる。
Resutoran de taberu.
레스토랑에서 먹는다.

公園で遊ぶ。
Kōen de asobu.
공원에서 논다.
```

**に와 で의 비교:**

> **に (존재/상태)  vs  で (동작)**
>
> 図書館に いる。     도서관에 있다. (존재)
> 図書館で 勉強する。 도서관에서 공부한다. (동작)
>
> 家に ある。         집에 있다. (존재)
> 家で 作る。         집에서 만든다. (동작)
>
> - **규칙**: いる/ある → に
> 동작 동사 → で

### 2.2 수단, 도구, 방법

で는 무언가를 하는 데 쓰이는 도구, 방법, 수단을 나타냅니다.

```
バスで行く。
Basu de iku.
버스로 간다.

はしで食べる。
Hashi de taberu.
젓가락으로 먹는다.

日本語で話す。
Nihongo de hanasu.
일본어로 말한다.

パソコンで仕事をする。
Pasokon de shigoto o suru.
컴퓨터로 일한다.
```

### 2.3 이유 또는 원인

で는 특히 사건이나 상황의 이유나 원인을 나타낼 수 있습니다.

```
病気で学校を休んだ。
Byōki de gakkō o yasunda.
병으로 학교를 쉬었다.

台風で電車が止まった。
Taifū de densha ga tomatta.
태풍으로 전차가 멈추었다.

仕事で東京に行きます。
Shigoto de Tōkyō ni ikimasu.
일 때문에 도쿄에 갑니다.
```

### 2.4 재료

で는 무언가를 만드는 재료를 나타낼 수 있습니다.

```
木で作る。
Ki de tsukuru.
나무로 만든다.

紙で飛行機を作った。
Kami de hikōki o tsukutta.
종이로 비행기를 만들었다.
```

## 3. から와 まで — ~에서와 ~까지

### 3.1 から (kara) — ~에서, ~부터

から는 시간, 공간, 순서에서 시작점을 표시합니다.

**시간:**

```
九時から仕事をする。
Ku-ji kara shigoto o suru.
9시부터 일한다.

月曜日から金曜日まで学校に行く。
Getsuyōbi kara kinyōbi made gakkō ni iku.
월요일부터 금요일까지 학교에 간다.

来年から日本語を勉強する。
Rainen kara nihongo o benkyō suru.
내년부터 일본어를 공부한다.
```

**장소:**

```
東京から来ました。
Tōkyō kara kimashita.
도쿄에서 왔습니다.

ここから駅まで歩いて十分です。
Koko kara eki made aruite juppun desu.
여기서 역까지 걸어서 10분입니다.
```

**이유 (~때문에):**

```
暑いから、窓を開けてください。
Atsui kara, mado o akete kudasai.
더우니까 창문을 열어 주세요.

時間がないから、タクシーで行きます。
Jikan ga nai kara, takushii de ikimasu.
시간이 없으니까 택시로 갑니다.
```

> **참고**: "~때문에"를 뜻하는 から는 절(문장 + から) **뒤에** 옵니다. 이것은 공간적·시간적 용법과 다릅니다.

### 3.2 まで (made) — ~까지

まで는 끝점을 표시합니다.

**시간:**

```
五時まで働く。
Go-ji made hataraku.
5시까지 일한다.

金曜日まで待ってください。
Kinyōbi made matte kudasai.
금요일까지 기다려 주세요.
```

**장소:**

```
駅まで歩いた。
Eki made aruita.
역까지 걸었다.

東京から大阪まで新幹線で行く。
Tōkyō kara Ōsaka made shinkansen de iku.
도쿄에서 오사카까지 신칸센으로 간다.
```

### 3.3 までに — ~까지(기한)

までに는 기한을 나타냅니다 — 특정 시간 **안에** 무언가를 해야 합니다.

> **まで (~까지) vs までに (~까지는)**
>
> 五時まで働く。
> 5시까지 일한다. (5시까지 계속되는 동작)
>
> 五時までに帰る。
> 5시까지 돌아온다. (5시라는 기한 전에)
>
> まで = 해당 시점까지 동작이 계속됨
> までに = 해당 시점 전에 동작이 일어남

## 4. と — ~와, ~과 함께, 인용

### 4.1 완전 나열의 "~와"

と는 항목을 완전히 나열합니다 — 언급된 항목**만** 해당되고 다른 것은 없습니다.

```
コーヒーとお茶を飲みます。
Kōhii to ocha o nomimasu.
커피와 차를 마십니다. (이 둘만)

犬と猫がいます。
Inu to neko ga imasu.
개와 고양이가 있습니다. (정확히 이것들)

日本語と英語を話す。
Nihongo to eigo o hanasu.
일본어와 영어를 한다.
```

### 4.2 "~과 함께"

と는 동반을 나타냅니다 — 누군가와 함께 무언가를 합니다.

```
友達と映画を見た。
Tomodachi to eiga o mita.
친구와 영화를 보았다.

田中さんと話した。
Tanaka-san to hanashita.
다나카 씨와 이야기했다.

家族と旅行に行く。
Kazoku to ryokō ni iku.
가족과 여행을 간다.
```

### 4.3 인용 표시

と는 인용된 말이나 생각을 표시합니다. 言う (iu — 말하다), 思う (omou — 생각하다), 聞く (kiku — 듣다/묻다) 같은 동사 앞에 사용됩니다.

```
「おはよう」と言った。
"Ohayō" to itta.
"좋은 아침"이라고 말했다.

難しいと思う。
Muzukashii to omou.
어렵다고 생각한다.

田中さんが来ると聞いた。
Tanaka-san ga kuru to kiita.
다나카 씨가 온다고 들었다.

日本語は面白いと思います。
Nihongo wa omoshiroi to omoimasu.
일본어는 재미있다고 생각합니다.
```

### 4.4 조건의 と (소개)

と는 "~하면/~할 때"라는 조건절(자동적·자연적 결과)을 만들 수도 있습니다. 이것은 N4/N3 문법 항목으로, 여기서는 소개 차원에서 다룹니다.

```
春になると、桜が咲く。
Haru ni naru to, sakura ga saku.
봄이 되면 벚꽃이 핀다.

このボタンを押すと、ドアが開く。
Kono botan o osu to, doa ga aku.
이 버튼을 누르면 문이 열린다.
```

## 5. も — ~도, ~조차, 강조

### 5.1 "~도" — は, が, を를 대체

も는 다른 조사를 대체하여 "~도", "~역시"를 의미합니다.

```
田中さんは学生です。山田さんも学生です。
Tanaka-san wa gakusei desu. Yamada-san mo gakusei desu.
다나카 씨는 학생입니다. 야마다 씨도 학생입니다.

コーヒーを飲みました。お茶も飲みました。
Kōhii o nomimashita. Ocha mo nomimashita.
커피를 마셨습니다. 차도 마셨습니다.
```

> **참고**: も는 は, が, を를 **대체**합니다. 함께 쓰지 않습니다.
> - ✗ 田中さん**はも** → ✓ 田中さん**も**
> - ✗ お茶**をも** → ✓ お茶**も**
> - 단, に, で 등은 **유지**됩니다: 東京**にも**行った (도쿄에도 갔다)

### 5.2 다른 조사와 함께

も가 장소/방향 조사와 결합할 때, 원래 조사는 유지됩니다.

```
東京にも行った。
Tōkyō ni mo itta.
도쿄에도 갔다.

図書館でも勉強する。
Toshokan de mo benkyō suru.
도서관에서도 공부한다.

友達とも話した。
Tomodachi to mo hanashita.
친구와도 이야기했다.
```

### 5.3 강조와 "~조차"

```
子供でも分かる。
Kodomo de mo wakaru.
아이라도 안다.

百円も持っていない。
Hyaku-en mo motte inai.
100엔조차 가지고 있지 않다.
```

### 5.4 의문사 + も (부정)

부정문에서 의문사와 결합하면 も는 "아무것도/아무도/어디에도"라는 의미를 만듭니다.

> **의문사 + も + 부정**
>
> 何 (nani) + も + 부정     = 아무것도 ~않다
> 何も食べなかった。  아무것도 먹지 않았다.
>
> 誰 (dare) + も + 부정    = 아무도 ~않다
> 誰もいなかった。    아무도 없었다.
>
> どこ (doko) + も + 부정  = 어디에도 ~않다
> どこにも行かなかった。어디에도 가지 않았다.
>
> いつ (itsu) + も        = 항상 (긍정)
> いつも元気です。    항상 건강합니다.

## 6. や — 비완전 나열

### 6.1 기본 사용법

や는 항목을 나열하면서 언급되지 않은 **다른 항목이 더 있음**을 암시합니다. 완전 나열인 と와 대비됩니다.

```
りんごやみかんを買った。
Ringo ya mikan o katta.
사과나 귤 등을 샀다.

本や雑誌を読む。
Hon ya zasshi o yomu.
책이나 잡지 등을 읽는다.
```

### 6.2 と vs や

> **と (완전 나열) vs や (비완전 나열)**
>
> 犬と猫がいる。
> 개와 고양이가 있다. (이 둘만)
>
> 犬や猫がいる。
> 개나 고양이 등이 있다. (다른 동물도 있음을 암시)
>
> コーヒーとお茶を飲む。
> 커피와 차를 마신다. (이 둘만)
>
> コーヒーやお茶を飲む。
> 커피나 차 등을 마신다. (다른 것도 있음을 암시)

### 6.3 や...など 패턴

비완전 나열의 의미를 더욱 명확하게 하기 위해 など (nado — ~등)이 자주 추가됩니다.

```
りんごやみかんなどを買った。
Ringo ya mikan nado o katta.
사과나 귤 등을 샀다.

テニスやサッカーなどのスポーツが好きです。
Tenisu ya sakkaa nado no supōtsu ga suki desu.
테니스나 축구 같은 스포츠를 좋아합니다.
```

## 7. の — 소유를 넘어서

### 7.1 소유 (복습)

```
私の本      watashi no hon        내 책
田中さんの車 Tanaka-san no kuruma  다나카 씨의 차
日本の料理   Nihon no ryōri        일본 요리
```

### 7.2 명사 수식 (동격)

の는 두 명사를 연결하여 첫 번째 명사가 두 번째 명사를 수식하거나 설명합니다.

```
英語の先生     eigo no sensei         영어 선생님
大学の友達     daigaku no tomodachi   대학 친구
東京の天気     Tōkyō no tenki        도쿄의 날씨
三年生の学生   san-nensei no gakusei  3학년 학생
```

### 7.3 명사화 (동사를 명사로 만들기)

の는 동사구를 명사구로 바꿀 수 있으며, 영어의 "...ing"이나 "to..."와 유사합니다.

```
本を読むのが好きです。
Hon o yomu no ga suki desu.
책 읽는 것을 좋아합니다. (직역: 책을 읽는 것이 좋다.)

泳ぐのは楽しい。
Oyogu no wa tanoshii.
수영하는 것은 즐겁다.

日本語を話すのは難しい。
Nihongo o hanasu no wa muzukashii.
일본어를 말하는 것은 어렵다.
```

### 7.4 설명의 の (んです / のです)

문장 끝에 오는 の(구어에서는 ん으로 축약)는 설명적 또는 감정적 뉘앙스를 더합니다 — 설명을 구하거나 제공합니다.

```
どうしたんですか。
Dō shita n desu ka.
무슨 일이 있었어요? (설명을 구함)

頭が痛いんです。
Atama ga itai n desu.
(사실은) 머리가 아파요.

明日テストがあるんです。
Ashita tesuto ga aru n desu.
(있잖아요,) 내일 시험이 있어요.
```

> **일반문 vs 설명문**
>
> 行きます。      갑니다. (단순 진술)
> 行くんです。    가는 겁니다. (이유/배경이 있음)
>
> 食べません。    안 먹습니다. (단순 진술)
> 食べないんです。안 먹는 겁니다. (이유를 설명...)
>
> 사용 상황:
> - 이유를 설명할 때
> - "무슨 일이야?"라고 물을 때
> - 배경 정보를 제공할 때
> - 감정적 관여를 나타낼 때

## 8. 조사 조합

### 8.1 자주 쓰이는 조합

| 조합 | 의미 | 예시 |
|-------------|---------|---------|
| には | ~에는 (강조) / ~에(주제) | 日本**には**四季がある。(일본에는 사계절이 있다.) |
| では | ~에서는 (주제) / 그러면 | 日本**では**有名です。(일본에서는 유명합니다.) |
| にも | ~에도 | 大阪**にも**行った。(오사카에도 갔다.) |
| でも | ~에서도 / 하지만 | 家**でも**仕事をする。(집에서도 일한다.) |
| からの | ~에서 온 (명사 수식) | 母**からの**手紙 (어머니로부터의 편지) |
| までの | ~까지의 (명사 수식) | 駅**までの**道 (역까지의 길) |
| との | ~와의 (명사 수식) | 友達**との**約束 (친구와의 약속) |

### 8.2 조사 결합 규칙

> **조사 결합 규칙**
>
> は, が, を  →  も로 대체됨
> 私は → 私も   (나도)
> 本を → 本も   (책도)
>
> に, で, と, から, まで  →  は/も 앞에 유지됨
> 学校に → 学校にも  (학교에도)
> バスで → バスでは  (버스로는)
> 友達と → 友達とも  (친구와도)
> 東京から → 東京からも (도쿄에서도)
>
> 결합 불가:
> ✗ はが  ✗ がは  ✗ をが  ✗ はを

## 9. 흔한 조사 실수

### 9.1 に vs で (장소)

```
✗ 図書館に勉強する。     (に는 존재용, 동작에는 사용 불가)
✓ 図書館で勉強する。     도서관에서 공부한다.

✗ 部屋でいる。           (で는 동작용, 존재에는 사용 불가)
✓ 部屋にいる。           방에 있다.
```

### 9.2 は vs が

```
새로운 정보 소개 → が
  あそこに猫がいます。     저기에 고양이가 있습니다. (새로운 정보)

알려진 주제 → は
  猫は黒いです。           그 고양이는 검정입니다. (어떤 고양이인지 알고 있음)

의문사에는 항상 が:
  誰が来ましたか。         누가 왔습니까?
  何がありますか。         무엇이 있습니까?
```

### 9.3 と vs や

```
모든 항목 나열 → と
  ペンと鉛筆を買った。     펜과 연필을 샀다. (이것들만)

일부 예시 나열 → や
  ペンや鉛筆を買った。     펜이나 연필 등을 샀다. (다른 것도 있음을 암시)
```

### 9.4 に vs へ (방향)

```
둘 다 방향에 사용 가능:
  学校に行く。 = 学校へ行く。  학교에 간다.

に만 가능한 경우:
  - 시간: 三時に会う。 (3시에 만난다.)
  - 간접 목적어: 友達にあげる。 (친구에게 준다.)
  - 존재: ここにある。 (여기에 있다.)
```

## 10. 연습 문제

### 연습 1: 올바른 조사 선택

빈칸에 적절한 조사(で, に, を, が, は, と, も, や, から, まで)를 넣으세요.

```
1. 図書館 ___ 勉強します。(도서관에서 공부합니다)
2. 九時 ___ 五時 ___ 働きます。(9시부터 5시까지 일합니다)
3. 友達 ___ 映画を見ました。(친구와 영화를 보았습니다)
4. りんご ___ バナナ ___ 買いました。(사과나 바나나 등을 샀습니다)
5. 日本語 ___ 話してください。(일본어로 말해 주세요)
6. 猫 ___ 部屋にいます。(고양이가 방에 있습니다)
7. 私 ___ 学生です。山田さん ___ 学生です。(저는 학생입니다. 야마다 씨도 학생입니다)
8. 病気 ___ 学校を休みました。(병으로 학교를 쉬었습니다)
```

<details>
<summary>정답 보기</summary>

```
1. 図書館 で 勉強します。 (동작의 장소 → で)
2. 九時 から 五時 まで 働きます。 (~부터/~까지 → から/まで)
3. 友達 と 映画を見ました。 (~과 함께 → と)
4. りんご や バナナ を 買いました。 (비완전 나열 → や)
5. 日本語 で 話してください。 (수단/방법 → で)
6. 猫 が 部屋にいます。 (존재, 새로운 정보 → が)
7. 私 は 学生です。山田さん も 学生です。 (주제 → は, ~도 → も)
8. 病気 で 学校を休みました。 (이유/원인 → で)
```

</details>

### 연습 2: に vs で

각 빈칸에 に 또는 で를 넣으세요.

```
1. 公園 ___ 子供が遊んでいる。(공원에서 아이들이 놀고 있다.)
2. 椅子の上 ___ 猫がいる。(의자 위에 고양이가 있다.)
3. レストラン ___ 昼ご飯を食べた。(레스토랑에서 점심을 먹었다.)
4. 東京 ___ 住んでいます。(도쿄에 살고 있습니다.)
5. 駅の前 ___ バスが止まっている。(역 앞에 버스가 서 있다.)
6. 家 ___ テレビを見る。(집에서 텔레비전을 본다.)
```

<details>
<summary>정답 보기</summary>

```
1. 公園 で (놀다 = 동작 → で)
2. 椅子の上 に (고양이의 존재 → に)
3. レストラン で (먹다 = 동작 → で)
4. 東京 に (住む는 거주 장소에 に를 사용)
5. 駅の前 に (止まっている = 멈춰 있는 상태 → に)
6. 家 で (텔레비전 시청 = 동작 → で)
```

</details>

### 연습 3: と vs や

각 상황에 と 또는 や를 선택하세요.

```
1. I have exactly two pets: a dog and a cat.
   犬 ___ 猫を飼っています。

2. My hobbies include reading, running, and other things.
   読書 ___ ランニングが好きです。

3. Please buy milk and eggs (just these two).
   牛乳 ___ 卵を買ってください。

4. I study subjects like math, science, etc.
   数学 ___ 理科を勉強します。
```

<details>
<summary>정답 보기</summary>

```
1. 犬 と 猫を飼っています。 (완전 나열 — 정확히 이 둘 → と)
2. 読書 や ランニングが好きです。 (비완전 나열 — 다른 것도 있음 → や)
3. 牛乳 と 卵を買ってください。 (완전 나열 — 이것들만 → と)
4. 数学 や 理科を勉強します。 (비완전 나열 — 더 있음을 암시 → や)
```

</details>

### 연습 4: 일본어로 번역

존댓말(ます/です) 형태를 사용하세요.

```
1. I go to school from Monday to Friday.
2. I came from Osaka.
3. I ate sushi with my friend at a restaurant.
4. Nobody came yesterday.
5. I think Japanese is interesting.
6. I like reading books.
7. There is a letter from my mother.
8. Please speak more slowly because it's difficult.
```

<details>
<summary>정답 보기</summary>

```
1. 月曜日から金曜日まで学校に行きます。
   (Getsuyōbi kara kinyōbi made gakkō ni ikimasu.)

2. 大阪から来ました。
   (Ōsaka kara kimashita.)

3. レストランで友達とすしを食べました。
   (Resutoran de tomodachi to sushi o tabemashita.)

4. 昨日は誰も来ませんでした。
   (Kinō wa dare mo kimasen deshita.)

5. 日本語は面白いと思います。
   (Nihongo wa omoshiroi to omoimasu.)

6. 本を読むのが好きです。
   (Hon o yomu no ga suki desu.)

7. 母からの手紙があります。
   (Haha kara no tegami ga arimasu.)

8. 難しいから、もっとゆっくり話してください。
   (Muzukashii kara, motto yukkuri hanashite kudasai.)
```

</details>

### 연습 5: 오류 교정

각 문장에서 조사 오류를 찾아 교정하세요.

```
1. 図書館に勉強しました。
2. 友達を映画を見ました。
3. 田中さんはも来ました。
4. バスに学校に行きます。
5. 何を食べなかった。
```

<details>
<summary>정답 보기</summary>

```
1. 図書館で勉強しました。 (장소에서의 동작 → で, に가 아님)
2. 友達と映画を見ました。 (~과 함께 → と, を가 아님)
3. 田中さんも来ました。 (も는 は를 대체함, 겹쳐 쓰지 않음)
4. バスで学校に行きます。 (교통수단 → で, に가 아님)
5. 何も食べなかった。 (의문사 + も + 부정 = 아무것도 ~않다)
```

</details>

## 빠른 참조

> **조사 요약표 (助詞まとめ)**
> ├──────────────────────────────────────────────────────────────┤
>
> - **で  동작의 장소**: 図書館で勉強する
> - **수단/도구**: バスで行く
> - **이유**: 病気で休む
> - **재료**: 木で作る
>
> - **から 시작점**: 九時から (9시부터)
> - **이유 (절)**: 暑いから (더우니까)
> - **まで 끝점**: 五時まで (5시까지)
> - **までに 기한**: 五時までに (5시까지는)
>
> - **と  완전 나열**: 犬と猫 (개와 고양이 — 이것들만)
> - **"~과 함께"**: 友達と行く (친구와 가다)
> - **인용**: 面白いと思う (재미있다고 생각하다)
>
> - **も  ~도**: 私も (나도)
> - **~조차**: 子供でも (아이라도)
> - **의문사 + 부정**: 何も〜ない (아무것도 ~않다)
>
> - **や  비완전 나열**: りんごやみかん (사과나 귤 등)
> - **자주 など과 함께**: 〜や〜など (~나 ~ 등)
>
> - **の  소유**: 私の本 (내 책)
> - **수식**: 英語の先生 (영어 선생님)
> - **명사화**: 読むのが好き (읽는 것을 좋아하다)
> - **설명문**: 〜んです (있잖아요, 사실은...)
>
> - **결합**: は/が/を → も로 대체
> に/で/と/から/まで → は/も 앞에 유지
>
