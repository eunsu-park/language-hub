## 1. 비즈니스 경어 (ビジネス敬語)

비즈니스 일본어는 14과에서 다룬 교과서 수준의 경어를 훨씬 뛰어넘는 높은 수준의 공손함을 요구합니다. 기업 세계에서 잘못된 경어 사용은 관계를 손상시키고 무능하다는 인상을 줄 수 있습니다.

```
+----------------------------------------------------------+
|          BUSINESS KEIGO HIERARCHY                         |
+----------------------------------------------------------+
|                                                          |
|  Level 1: 丁寧語 (정중어)                                  |
|  └── です / ます 형태                                      |
|  └── 동료나 친한 직장 동료에게 사용                          |
|                                                          |
|  Level 2: 尊敬語 (존경어)                                  |
|  └── 상대방의 행위를 높임                                   |
|  └── 상사, 고객, 외부 담당자에게 사용                        |
|                                                          |
|  Level 3: 謙譲語 (겸양어)                                  |
|  └── 자신의 행위를 낮춤                                     |
|  └── 상사에게 자신에 대해 말할 때 사용                       |
|                                                          |
|  Level 4: ビジネス敬語 (비즈니스 경어)                       |
|  └── 고정된 정형 표현과 초격식적 표현                        |
|  └── 고객 대면 커뮤니케이션에 필수                           |
+----------------------------------------------------------+
```

### 1.1 일반 동사를 비즈니스 수준으로 격상하기

비즈니스 환경에서는 표준 경어만으로는 부족한 경우가 많습니다. 다음은 자주 사용되는 동사의 비즈니스 등급 업그레이드입니다:

| 보통체 | 표준 경어 | 비즈니스 수준 |
|-------|---------------|----------------|
| する (하다) | なさる / いたす | させていただく |
| 言う (말하다) | おっしゃる / 申す | 申し上げる |
| 行く (가다) | いらっしゃる / 参る | お伺いする / 参上する |
| 来る (오다) | いらっしゃる / 参る | お越しになる / お見えになる |
| 見る (보다) | ご覧になる / 拝見する | 拝見させていただく |
| 知る (알다) | ご存じ / 存じる | 存じ上げる |
| 食べる (먹다) | 召し上がる / いただく | 頂戴する |
| もらう (받다) | いただく | 頂戴する / 賜る（たまわる） |
| あげる (주다) | 差し上げる | 進呈する（しんていする） |
| 聞く (듣다/묻다) | お聞きになる / 伺う | 拝聴する（はいちょうする） |

### 1.2 ～させていただく 구문

이것은 현대 비즈니스 일본어의 가장 특징적인 표현입니다. 문자 그대로 "겸손하게 ~하는 것을 허락해 주십시오"라는 뜻으로, 매우 광범위하게 사용됩니다:

```
구조: 동사 (させ형) + ていただく

예문:
  確認させていただきます。
  → 확인하겠습니다. (겸손하게 확인하는 것을 허락해 주십시오)

  説明させていただきます。
  → 설명드리겠습니다.

  ご連絡させていただきました。
  → 연락드리게 되었습니다.

  担当させていただいております。
  → 담당하고 있습니다. (담당하는 영광을 얻었습니다)
```

> **주의사항**: ～させていただく의 남용은 흔한 실수입니다. 다음과 같은 경우에 사용합니다:
> 1. 실제로 누군가의 허락을 받는 경우
> 2. 그 행위를 통해 혜택을 받는 경우
>
> 부적절: ✗ 「帰らせていただきます」 (정시 퇴근 시 — 허락이 필요 없음)
> 적절: ○ 「お先に失礼いたします」 (표준 퇴근 인사)

### 1.3 이중 경어 오류 (二重敬語)

비즈니스 환경에서 흔한 실수는 경어를 중복으로 사용하는 것입니다:

| 부적절 (二重敬語) | 올바른 표현 |
|---------------------|---------|
| ✗ お召し上がりになられる | ○ 召し上がる |
| ✗ おっしゃられる | ○ おっしゃる |
| ✗ ご覧になられる | ○ ご覧になる |
| ✗ お伺いさせていただく | ○ 伺う / お伺いする |

## 2. 업무 환경의 정형 표현

### 2.1 인사말과 일상 표현

| 상황 | 표현 | 의미 |
|-----------|--------|---------|
| 아침 인사 | おはようございます | 안녕하세요 (오전) |
| 먼저 퇴근할 때 | お先に失礼いたします | 먼저 실례하겠습니다 |
| 퇴근하는 사람에게 | お疲れ様でした（おつかれさまでした） | 수고하셨습니다 |
| 업무 시작 시 | 本日もよろしくお願いいたします | 오늘도 잘 부탁드립니다 |
| 사무실로 돌아왔을 때 | 戻りました | 돌아왔습니다 |
| 돌아온 사람에게 | お帰りなさい / お疲れ様です | 돌아오셨군요 / 수고하셨습니다 |
| 외출할 때 | 行ってまいります | 다녀오겠습니다 |
| 외출하는 사람에게 | 行ってらっしゃい(ませ) | 다녀오세요 |

### 2.2 사과 표현 (격식 순서)

```
+----------------------------------------------------------+
|           APOLOGY FORMALITY SCALE                        |
+----------------------------------------------------------+
|                                                          |
|  비격식 ──────────────────────────────── 초격식           |
|                                                          |
|  すみません                                               |
|    ↓                                                     |
|  申し訳ありません                                          |
|    ↓                                                     |
|  申し訳ございません                                        |
|    ↓                                                     |
|  大変申し訳ございません                                     |
|    ↓                                                     |
|  誠に申し訳ございませんでした                                |
|    ↓                                                     |
|  深くお詫び申し上げます                                     |
|                                                          |
+----------------------------------------------------------+
```

### 2.3 요청과 거절

**요청하기**:
| 격식 수준 | 표현 |
|-----------|------------|
| 표준 정중 | ～していただけますか |
| 더 정중 | ～していただけないでしょうか |
| 매우 격식적 | ～していただけますと幸いです |
| 초격식적 | ～賜りますようお願い申し上げます |

**정중하게 거절하기**:
| 상황 | 표현 |
|-----------|------------|
| 일반적 거절 | 申し訳ございませんが、ちょっと難しい状況でして... |
| 일정 충돌 | あいにく、その日は先約がございまして... |
| 능력 밖의 일 | 力不足で申し訳ございませんが... |
| 사내 규정 | 社内規定により、お受けいたしかねます |

> **문화적 참고**: 일본 비즈니스 문화에서는 직접적인 거절을 피합니다. "아니요"라고 말하는 대신 ちょっと... (좀...) 또는 難しい (어렵다)와 같은 완화 표현을 사용하여 대립 없이 거절을 암시합니다.

```

## 5. 전화 예절

### 5.1 전화 받기와 걸기

일본어 전화 에티켓에는 매우 구체적인 패턴이 있습니다. 여기서의 실수는 즉시 눈에 띄며 비즈니스 관계를 손상시킬 수 있습니다.

```
+----------------------------------------------------------+
|        ANSWERING A BUSINESS CALL                         |
+----------------------------------------------------------+
|                                                          |
|  따르릉...                                                |
|                                                          |
|  「はい、○○株式会社でございます。」                         |
|  (네, ○○주식회사입니다.)                                   |
|                                                          |
|  주의: 비즈니스 전화는 もしもし로 시작하지 않습니다.          |
|  もしもし는 개인 통화 전용입니다.                            |
|                                                          |
+----------------------------------------------------------+
|        MAKING A BUSINESS CALL                            |
+----------------------------------------------------------+
|                                                          |
|  「お忙しいところ恐れ入ります。                             |
|    ○○株式会社の○○と申します。                              |
|    ○○部の○○様はいらっしゃいますでしょうか。」                |
|                                                          |
|  (바쁘신 중에 실례합니다.                                   |
|   ○○주식회사의 ○○라고 합니다.                              |
|   ○○부의 ○○님 계십니까?)                                  |
|                                                          |
+----------------------------------------------------------+
```

### 5.2 전화 표현

| 상황 | 표현 |
|-----------|--------|
| 전화 받기 | ○○株式会社でございます。 |
| 자기 소개 | ○○会社の○○と申します。 |
| 상대 찾기 | ○○様はいらっしゃいますでしょうか。 |
| 보류 요청 | 少々お待ちください(ませ)。 |
| 연결하기 | ○○におつなぎいたします。 |
| 부재 시 | あいにく、○○はただいま外出しております。 |
| 통화 중 | ○○はただいま別の電話に出ております。 |
| 콜백 제안 | 折り返しお電話させていただいてもよろしいでしょうか。 |
| 메시지 접수 | ご伝言を承ります。 |
| 정보 확인 | 念のため、お電話番号を確認させていただけますか。 |
| 통화 종료 | 失礼いたします。 |

### 5.3 잘 안 들릴 때

| 상황 | 표현 |
|-----------|------------|
| 통화 상태가 안 좋을 때 | 恐れ入りますが、お電話が少々遠いようでして... |
| 다시 말해달라고 할 때 | もう一度おっしゃっていただけますでしょうか。 |
| 이름 확인 시 | お名前をもう一度お伺いしてもよろしいでしょうか。 |
| 한자 확인 시 | 漢字ではどのように書かれますか。 |

### 5.4 전화 대화 예시

```

Caller:       お忙しいところ恐れ入ります。

## 6. 명함 교환

### 6.1 名刺交換 (메이시 코우칸) 프로토콜

명함 교환은 일본 비즈니스 문화에서 가장 의례화된 측면 중 하나입니다. 명함은 그 사람을 대표하며, 명함을 부적절하게 다루는 것은 그 사람을 무시하는 것과 동일합니다.

```
+----------------------------------------------------------+
|       BUSINESS CARD EXCHANGE STEPS                       |
+----------------------------------------------------------+
|                                                          |
|  준비:                                                    |
|  • 명함은 名刺入れ (명함 케이스)에 넣기, 지갑은 안 됨       |
|  • 서 있는 상태, 테이블을 사이에 두지 않기                   |
|  • 연소자 또는 방문자가 먼저 건넴                           |
|                                                          |
|  건넬 때:                                                  |
|  1. 명함을 양손으로 잡기                                    |
|  2. 상대방이 읽을 수 있도록 방향을 맞추기                    |
|  3. 가슴 높이에서 잡기                                     |
|  4. 자기소개하기:                                          |
|     「○○会社の○○と申します。                              |
|      よろしくお願いいたします。」                           |
|                                                          |
|  받을 때:                                                  |
|  1. 양손으로 받기                                          |
|  2. 「頂戴いたします」라고 말하기 (겸손히 받겠습니다)         |
|  3. 명함을 주의 깊게 읽기 (절대 건너뛰지 않기)              |
|  4. 이름 읽기가 불확실하면 정중하게 묻기:                    |
|     「お名前はどのようにお読みすればよろしいでしょうか」      |
|                                                          |
|  동시 교환:                                                |
|  • 양쪽이 동시에 명함을 건냄                                |
|  • 오른손으로 자기 명함 건네기                               |
|  • 왼손으로 상대 명함 받기                                  |
|  • 그 후 양손으로 받은 명함을 잡기                           |
|                                                          |
|  교환 후:                                                  |
|  • 회의 중에는 명함을 테이블 위에 놓기                       |
|  • 가장 높은 직급의 명함은 명함 케이스 위에 놓기              |
|  • 나머지는 테이블 위에 줄지어 놓기                          |
|  • 회의 중 절대 명함에 메모하지 않기                         |
|  • 나중에 명함 케이스에 소중히 보관하기                      |
+----------------------------------------------------------+
```

### 6.2 피해야 할 흔한 실수

| 실수 | 이유 |
|---------|---------------|
| 한 손으로 사용하기 | 존중이 부족함을 보여줌 |
| 명함을 뒷주머니에 넣기 | 그 사람 위에 앉는 것과 동일 |
| 상대 앞에서 명함에 쓰기 | 상대의 정체성을 훼손하는 행위 |
| 테이블 위로 명함을 밀기 | 게으르고 비인격적 |
| 명함을 읽지 않기 | 상대에 대한 무관심을 보여줌 |
| 명함 케이스를 잊기 | 준비 부족, 비전문적 |
| 회의 중 명함을 만지작거리기 | 무례한 행동 |

## 7. 직장 내 위계와 커뮤니케이션

### 7.1 직함과 호칭

| 직함 | 일본어 | 용법 |
|-------|----------|-------|
| 사장 / CEO | 社長（しゃちょう） | ○○社長 |
| 부사장 | 副社長（ふくしゃちょう） | ○○副社長 |
| 이사 | 取締役（とりしまりやく） | ○○取締役 |
| 부장 | 部長（ぶちょう） | ○○部長 |
| 과장 | 課長（かちょう） | ○○課長 |
| 계장 | 係長（かかりちょう） | ○○係長 |
| 주임 | 主任（しゅにん） | ○○主任 |
| 일반 사원 | 社員（しゃいん） | ○○さん |

> **중요 규칙**: 외부인에게 자기 회사 사람을 언급할 때는 경칭을 빼고 이름만 사용합니다:
>
> ✗ 「部長の田中様はただいま外出しております」
> ○ 「田中はただいま外出しております」
>
> 자기 회사 사람은 "우치" (内 = 내부)이므로, 자기 쪽을 낮춥니다.

### 7.2 内 (우치) vs 外 (소토) — 비즈니스에서의 적용

```
+----------------------------------------------------------+
|       UCHI / SOTO DYNAMIC IN BUSINESS                    |
+----------------------------------------------------------+
|                                                          |
|  외부인에게 자기 회사에 대해 말할 때:                       |
|  ┌───────────────────────────────────────────────┐       |
|  │  자기 회사 = UCHI (겸양)                       │       |
|  │  • 弊社 (へいしゃ) = 저희 회사 (겸양)           │       |
|  │  • 자기 사람의 직함 생략                        │       |
|  │  • 자기 회사의 행동에는 謙譲語 사용              │       |
|  │                                               │       |
|  │  상대 회사 = SOTO (존경)                       │       |
|  │  • 御社 (おんしゃ) = 귀사 (구어)                │       |
|  │  • 貴社 (きしゃ) = 귀사 (문어)                  │       |
|  │  • 상대 회사의 행동에는 尊敬語 사용              │       |
|  └───────────────────────────────────────────────┘       |
|                                                          |
|  자기 상사에게 말할 때:                                    |
|  ┌───────────────────────────────────────────────┐       |
|  │  표준 경어 (尊敬語/謙譲語) 사용                  │       |
|  │  직함으로 호칭: ○○部長, ○○課長                  │       |
|  └───────────────────────────────────────────────┘       |
+----------------------------------------------------------+
```

### 7.3 報告・連絡・相談 (호우렌소우)

일본 직장 커뮤니케이션의 기본은 ほうれんそう입니다:

| 요소 | 일본어 | 의미 | 시기 |
|-----------|----------|---------|------|
| 報告 | ほうこく | 보고 | 업무 완료 후 또는 중간 단계에서 |
| 連絡 | れんらく | 연락 / 통지 | 변경사항이나 새로운 정보가 있을 때 |
| 相談 | そうだん | 상담 | 의사결정 전이나 불확실할 때 |

```
예문:

報告: 「先日ご依頼いただいた件について、ご報告いたします。」
       (며칠 전 의뢰하신 건에 대해 보고드리겠습니다.)

連絡: 「○○の件で、一点ご連絡がございます。」
       (○○의 건으로 한 가지 연락드릴 사항이 있습니다.)

相談: 「お忙しいところ恐れ入りますが、
        一点ご相談させていただいてもよろしいでしょうか。」
       (바쁘신 중에 죄송합니다만,
        한 가지 상담드려도 될까요?)
```

## 8. 자주 쓰이는 비즈니스 표현

### 8.1 비즈니스에서의 사자성어 (四字熟語)

| 사자성어 | 읽기 | 의미 | 비즈니스 활용 |
|-------|---------|---------|-------------|
| 一期一会 | いちごいちえ | 일생에 한 번뿐인 만남 | 각 고객과의 만남을 소중히 |
| 以心伝心 | いしんでんしん | 이심전심 / 말없는 이해 | 분위기 파악 |
| 臨機応変 | りんきおうへん | 임기응변 | 상황에 맞게 대응 |
| 前代未聞 | ぜんだいみもん | 전대미문 | 전례 없는 상황 설명 |
| 試行錯誤 | しこうさくご | 시행착오 | 개발 과정 설명 |
| 一石二鳥 | いっせきにちょう | 일석이조 | 효율적인 해결책 |

### 8.2 유용한 비즈니스 정형 표현

| 일본어 | 한국어 |
|----------|---------|
| ご検討のほど、よろしくお願いいたします | 검토해 주시길 부탁드립니다 |
| 前向きに検討させていただきます | 전향적으로 검토하겠습니다 |
| 善処いたします | 최선을 다해 처리하겠습니다 |
| 社内で検討の上、改めてご連絡いたします | 사내에서 검토 후 다시 연락드리겠습니다 |
| 大変恐縮ですが... | 대단히 죄송합니다만... |
| お手数をおかけして申し訳ございません | 수고를 끼쳐드려 죄송합니다 |
| ご足労いただきありがとうございます | 직접 왕림해 주셔서 감사합니다 |
| 今後ともよろしくお願い申し上げます | 앞으로도 잘 부탁드립니다 |

> **행간 읽기**: 일부 비즈니스 표현은 간접적인 거절입니다:
> - 「前向きに検討させていただきます」는 종종 "하지 않겠다"는 뜻입니다
> - 「善処いたします」는 종종 "기대하지 마세요"라는 뜻입니다
> - 「ちょっと難しいかもしれません」는 종종 "아니요"라는 뜻입니다

## 9. 연습 문제

### 연습 1: 경어 교정

다음 문장의 경어 오류를 수정하십시오.

1. お客様がおっしゃられた通りです。
2. 社長が申していました。（외부 고객에게 자기 사장에 대해 말할 때）
3. 先生、こちらの資料を拝見してください。
4. 部長にお会いしたいのですが。（자기 부장에 대해 안내데스크에 말할 때）

<details>
<summary>정답</summary>

1. ✗ おっしゃられた → ○ おっしゃった (이중 경어: おっしゃる + られる)
2. 이것은 실제로 올바릅니다. 외부인에게 자기 사장에 대해 말할 때는 겸양어(申す)를 사용하고 직함을 생략합니다(社長 → 성만 사용). 다만, "社長が"는 "○○（성）が申しておりました。"로 해야 합니다.
3. ✗ 拝見してください → ○ ご覧ください (拝見は은 겸양어이므로 — 상대방에게 겸양하라고 할 수 없음)
4. 안내데스크에서 자기 회사 부장에 대해 물을 때는 부장에게 경칭을 쓰지 않습니다: ✗ 「部長にお会いしたい」 → ○ 「○○に会いたいのですが」 (단, 다른 회사 사람을 만나려고 할 때는 尊敬語를 사용합니다.)

</details>

### 연습 4: 명함 교환

다음 행동 중 名刺交換 시 올바른 것은 어느 것입니까? 각각 ○ (올바름) 또는 ✗ (잘못됨)로 표시하십시오.

1. 인사하면서 한 손으로 명함 받기
2. 받은 명함을 테이블 위 명함 케이스에 놓기
3. 회의 중 명함 뒷면에 메모 쓰기
4. 명함에 있는 어려운 이름의 읽기 방법 묻기
5. 윗사람이 먼저 명함을 건네기
6. 가장 높은 직급의 명함을 명함 케이스 위에 놓기

<details>
<summary>정답</summary>

1. ✗ — 반드시 양손으로 받아야 합니다
2. ○ — 회의 중 가장 높은 직급의 명함을 명함 케이스 위에 놓는 것은 올바릅니다
3. ✗ — 절대 상대방 앞에서 명함에 메모하면 안 됩니다
4. ○ — 정중하며 당연한 것입니다: 「お名前はどのようにお読みすればよろしいでしょうか」
5. ✗ — 연소자 또는 방문자가 먼저 건넵니다
6. ○ — 가장 높은 직급의 명함은 명함 케이스 위에, 나머지는 테이블 위에 줄지어 놓습니다

</details>

### 연습 5: 비즈니스 일본어로 변환

다음 캐주얼한 발언을 적절한 비즈니스 일본어로 변환하십시오:

1. "I'll check it." (고객에게)
2. "I don't know." (상사에게)
3. "Can you send it by email?" (고객에게)
4. "Sorry for being late." (회의에 늦게 도착했을 때)
5. "Let me think about it." (제안하는 고객에게)

<details>
<summary>정답</summary>

1. 確認させていただきます。/ 確認いたします。
2. 申し訳ございません、存じ上げません。/ 勉強不足で申し訳ございません。確認いたします。
3. メールにてお送りいただけますでしょうか。
4. 遅くなりまして、大変申し訳ございません。
5. 社内で検討の上、改めてご連絡させていただきます。

</details>
