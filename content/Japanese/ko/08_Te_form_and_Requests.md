## 학습 목표

- 모든 동사 그룹에 대해 て형(te-form)을 이해하고 만든다
- て + ください로 정중한 요청을 한다
- て + いる로 진행 중인 동작과 상태를 묘사한다
- て + もいい로 허가를 구하고 허락한다
- て + はいけない로 금지를 표현한다
- て + ある로 결과 상태를 묘사한다
- て형을 사용하여 여러 동작을 연결한다

## 1. て형이란?

て형(te-kei)은 일본어에서 가장 다재다능하고 자주 사용되는 동사 형태 중 하나입니다. て형 자체로는 시제나 공손함을 나타내지 않습니다. 대신 보조 동사나 표현에 붙어 새로운 의미를 만드는 **접속 형태**로 기능합니다.

```
┌──────────────────────────────────────────────────────────────┐
│           て형은 만능 도구입니다                                │
│                                                              │
│   て + ください      → ~해 주세요                            │
│   て + いる          → ~하고 있다 / ~한 상태이다             │
│   て + もいい        → ~해도 되나요? / ~해도 괜찮다          │
│   て + はいけない    → ~하면 안 된다                         │
│   て + ある          → ~해 놓은 상태이다                     │
│   て + て + て...    → X하고 Y하고 Z하고...                  │
│                                                              │
│   て형은 중급 일본어의 문을 여는 핵심 열쇠입니다.            │
│   이것을 완벽히 익히면 수십 가지 문법 패턴을 사용할 수       │
│   있게 됩니다.                                               │
└──────────────────────────────────────────────────────────────┘
```

## 2. て형 만드는 규칙

て형은 동사의 사전형에서 만들어집니다. 규칙은 동사 그룹에 따라 다릅니다.

### 2.1 2그룹: 1단 동사 (一段動詞) — 쉬운 것들

마지막 る를 떼고 て를 붙입니다.

| 사전형 | 의미 | て형 |
|----------------|---------|---------|
| 食べる (taberu) | 먹다 | 食べ**て** |
| 見る (miru) | 보다 | 見**て** |
| 起きる (okiru) | 일어나다 | 起き**て** |
| 寝る (neru) | 자다 | 寝**て** |
| 教える (oshieru) | 가르치다 | 教え**て** |

### 2.2 1그룹: 5단 동사 (五段動詞) — 까다로운 것들

사전형의 마지막 가나에 따라 어미가 변합니다. 유명한 **て형 노래**가 기억에 도움이 됩니다.

| 사전형 어미 | 변화 | 예시 | て형 |
|-------------------|--------|---------|---------|
| う, つ, る | → って | 買う (kau) | 買**って** |
| | | 待つ (matsu) | 待**って** |
| | | 帰る (kaeru) | 帰**って** |
| む, ぶ, ぬ | → んで | 読む (yomu) | 読**んで** |
| | | 遊ぶ (asobu) | 遊**んで** |
| | | 死ぬ (shinu) | 死**んで** |
| く | → いて | 書く (kaku) | 書**いて** |
| ぐ | → いで | 泳ぐ (oyogu) | 泳**いで** |
| す | → して | 話す (hanasu) | 話**して** |

### 2.3 て형 노래 (て形の歌)

5단 동사 て형 규칙을 기억하기 위한 암기법입니다:

```
┌──────────────────────────────────────────────────────────────┐
│               て형 노래                                        │
│                                                              │
│   ♪ う、つ、る → って  って  って                             │
│     (u, tsu, ru → tte)                                       │
│                                                              │
│   ♪ む、ぶ、ぬ → んで  んで  んで                             │
│     (mu, bu, nu → nde)                                       │
│                                                              │
│   ♪ く → いて     ぐ → いで                                  │
│     (ku → ite)    (gu → ide)                                 │
│                                                              │
│   ♪ す → して                                                │
│     (su → shite)                                             │
│                                                              │
│   노래로 부르고, 암기하고, 좋아하세요.                       │
└──────────────────────────────────────────────────────────────┘
```

### 2.4 3그룹: 불규칙 동사

| 사전형 | 의미 | て형 |
|----------------|---------|---------|
| する (suru) | 하다 | **して** |
| 来る (kuru) | 오다 | **来て** (kite) |

### 2.5 중요한 예외

行く (iku — 가다)는 く로 끝나는 동사 중 예외입니다:

```
行く → 行って  (行いて가 아님)
iku → itte     (iite가 아님)
```

### 2.6 완전한 참조표

```
┌──────────────────────────────────────────────────────────────┐
│                  て형 완전표                                    │
├──────────────┬────────────────┬───────────────────────────────┤
│ 그룹         │ 규칙           │ 예시                          │
├──────────────┼────────────────┼───────────────────────────────┤
│ 1단 (G2)     │ る → て        │ 食べる → 食べて              │
├──────────────┼────────────────┼───────────────────────────────┤
│ 5단 う/つ/る │ → って         │ 買う → 買って               │
│ 5단 む/ぶ/ぬ │ → んで        │ 読む → 読んで               │
│ 5단 く       │ → いて         │ 書く → 書いて               │
│ 5단 ぐ       │ → いで         │ 泳ぐ → 泳いで               │
│ 5단 す       │ → して         │ 話す → 話して               │
├──────────────┼────────────────┼───────────────────────────────┤
│ 예외         │ 行く → 行って  │ (行いて가 아님)              │
├──────────────┼────────────────┼───────────────────────────────┤
│ 불규칙       │ する → して    │ 勉強する → 勉強して         │
│              │ 来る → 来て    │ (kuru → kite)                │
└──────────────┴────────────────┴───────────────────────────────┘
```

## 3. て + ください — 정중한 요청

### 3.1 기본 사용법

て + ください (kudasai)는 "~해 주세요"라는 정중한 요청을 만듭니다.

```
待ってください。
Matte kudasai.
기다려 주세요.

ここに座ってください。
Koko ni suwatte kudasai.
여기에 앉아 주세요.

名前を書いてください。
Namae o kaite kudasai.
이름을 써 주세요.

もう一度言ってください。
Mō ichido itte kudasai.
한 번 더 말해 주세요.

ゆっくり話してください。
Yukkuri hanashite kudasai.
천천히 말해 주세요.
```

### 3.2 부정 요청

무언가를 하지 말라고 부탁할 때는 ないで + ください를 사용합니다.

```
触らないでください。
Sawaranaide kudasai.
만지지 말아 주세요.

心配しないでください。
Shinpai shinaide kudasai.
걱정하지 말아 주세요.

ここで写真を撮らないでください。
Koko de shashin o toranaide kudasai.
여기서 사진을 찍지 말아 주세요.
```

### 3.3 공손함의 단계

```
┌──────────────────────────────────────────────────────────────┐
│          요청의 공손함 단계                                     │
│                                                              │
│   待って。                매우 격식 없음 (친한 친구 사이)     │
│   待ってください。        정중함 (표준)                       │
│   待っていただけますか。  매우 정중함 / 겸양                  │
│   お待ちください。        격식체 / 존경                       │
│                                                              │
│   지금은 ください를 완벽히 익히세요. 나머지는 제14과에서     │
│   다룹니다.                                                  │
└──────────────────────────────────────────────────────────────┘
```

## 4. て + いる — 진행과 상태

て + いる (iru)는 일본어에서 가장 많이 사용되는 구조 중 하나입니다. 동사의 유형에 따라 여러 의미를 가집니다.

### 4.1 진행 중인 동작 (진행형)

동작 동사와 함께 쓰면, て + いる는 현재 진행 중인 동작을 나타냅니다. 영어의 "-ing"과 같습니다.

```
今、本を読んでいます。
Ima, hon o yonde imasu.
지금 책을 읽고 있습니다.

田中さんは電話で話している。
Tanaka-san wa denwa de hanashite iru.
다나카 씨는 전화로 이야기하고 있다.

子供たちが公園で遊んでいます。
Kodomo-tachi ga kōen de asonde imasu.
아이들이 공원에서 놀고 있습니다.
```

### 4.2 결과 상태

특정 동사(상태 변화 동사)와 함께 쓰면, て + いる는 완료된 동작의 **현재 상태**를 묘사합니다.

```
結婚している。
Kekkon shite iru.
결혼한 상태이다. ("결혼하는 중"이 아님)

知っている。
Shitte iru.
알고 있다. ("알아가는 중"이 아님)

窓が開いている。
Mado ga aite iru.
창문이 열려 있다. (열린 결과)

田中さんはもう来ている。
Tanaka-san wa mō kite iru.
다나카 씨는 이미 와 있다. (이미 여기 있다)
```

### 4.3 습관적 동작

て + いる는 습관적이거나 규칙적인 동작도 묘사할 수 있습니다.

```
毎日コーヒーを飲んでいます。
Mainichi kōhii o nonde imasu.
매일 커피를 마십니다. (습관)

東京に住んでいます。
Tōkyō ni sunde imasu.
도쿄에 살고 있습니다. (지속 상태)

銀行で働いている。
Ginkō de hataraite iru.
은행에서 일하고 있다. (규칙적 직업)
```

### 4.4 ている의 활용

| 형태 | 반말 | 존댓말 |
|------|--------|--------|
| 현재 (+) | 食べている | 食べています |
| 현재 (-) | 食べていない | 食べていません |
| 과거 (+) | 食べていた | 食べていました |
| 과거 (-) | 食べていなかった | 食べていませんでした |

> **구어 축약**: ている는 일상적인 대화에서 종종 てる로 줄여 씁니다.
> - 食べている → 食べてる
> - 読んでいる → 読んでる

### 4.5 知っている vs 知らない

중요한 특수 경우입니다: 知っている (알다)의 부정형은 知**ら**ない이지, 知っていない이 아닙니다.

```
知っています。     알고 있습니다.
知りません。       모릅니다. (知っていません이 아님)
```

## 5. て + もいい — 허가

### 5.1 허가 구하기

て + もいいですか는 "~해도 됩니까?" 또는 "~해도 괜찮습니까?"를 묻습니다.

```
ここに座ってもいいですか。
Koko ni suwatte mo ii desu ka.
여기에 앉아도 됩니까?

写真を撮ってもいいですか。
Shashin o totte mo ii desu ka.
사진을 찍어도 됩니까?

窓を開けてもいいですか。
Mado o akete mo ii desu ka.
창문을 열어도 됩니까?

この本を借りてもいいですか。
Kono hon o karite mo ii desu ka.
이 책을 빌려도 됩니까?
```

### 5.2 허가하기

```
はい、いいですよ。           네, 괜찮습니다.
はい、どうぞ。               네, 어서 하세요.
もちろん。                   물론이죠.
食べてもいいですよ。         드셔도 됩니다.
```

### 5.3 허가 거부하기

허가를 거부할 때는 て + は + いけない (제6절 참조)을 사용하거나, 더 부드러운 대안을 씁니다.

```
すみません、ちょっと...       죄송합니다, 좀... (부드러운 거절)
ここでは撮れません。         여기서는 (사진을) 찍을 수 없습니다. (간접적)
```

## 6. て + はいけない — 금지

### 6.1 기본 사용법

て + はいけない (wa ikenai) 또는 て + はいけません (존댓말)은 "~하면 안 됩니다" 또는 "~하는 것은 허용되지 않습니다"를 의미합니다.

```
ここでタバコを吸ってはいけません。
Koko de tabako o sutte wa ikemasen.
여기서 담배를 피우면 안 됩니다.

授業中に携帯を使ってはいけない。
Jugyō-chū ni keitai o tsukatte wa ikenai.
수업 중에 휴대폰을 사용하면 안 된다.

ここで走ってはいけません。
Koko de hashitte wa ikemasen.
여기서 뛰면 안 됩니다.

嘘をついてはいけない。
Uso o tsuite wa ikenai.
거짓말을 하면 안 된다.
```

### 6.2 허가 vs 금지 요약

```
┌──────────────────────────────────────────────────────────────┐
│        허가 vs 금지                                            │
│                                                              │
│   て + もいいですか         ~해도 됩니까?                    │
│   て + もいいですよ         ~해도 됩니다 / ~해도 괜찮습니다  │
│                                                              │
│   て + はいけません         ~하면 안 됩니다                   │
│   て + はだめです           ~하면 안 됩니다 (약간 격식 없음)  │
│                                                              │
│   예시 쌍:                                                   │
│   Q: ここで食べてもいいですか。  여기서 먹어도 됩니까?       │
│   A: はい、食べてもいいですよ。  네, 드셔도 됩니다.         │
│   A: いいえ、食べてはいけません。아니요, 먹으면 안 됩니다.  │
│                                                              │
│   ※ だめ (dame)는 금지의 いけない에 대한                    │
│     구어체 대안입니다.                                       │
│     食べてはだめです。= 먹으면 안 됩니다.                    │
└──────────────────────────────────────────────────────────────┘
```

## 7. て + ある — 결과 상태

### 7.1 기본 사용법

て + ある는 **누군가의 의도적인 행동에서 비롯된 상태**를 묘사합니다. 동작 자체가 아닌 현재 상태에 초점을 맞춥니다.

```
窓が開けてある。
Mado ga akete aru.
창문이 열어져 있다. (누군가가 의도적으로 열었고, 아직 열려 있다)

テーブルの上に本が置いてある。
Tēburu no ue ni hon ga oite aru.
테이블 위에 책이 놓여 있다. (누군가가 놓았다)

壁に絵が掛けてある。
Kabe ni e ga kakete aru.
벽에 그림이 걸려 있다.
```

### 7.2 ている vs てある

```
┌──────────────────────────────────────────────────────────────┐
│           ている vs てある                                    │
│                                                              │
│   窓が開いている。                                           │
│   창문이 열려 있다. (상태를 묘사, 행위자 없음)               │
│   자동사 사용: 開く (aku — 스스로 열리다)                    │
│                                                              │
│   窓が開けてある。                                           │
│   창문이 열어져 있다. (누군가가 의도적으로 열었다)           │
│   타동사 사용: 開ける (akeru — 무언가를 열다)                │
│                                                              │
│   핵심 차이:                                                 │
│   ている = 상태 (자연적이든 아니든)                          │
│   てある = 누군가의 의도적 행동에서 비롯된 상태              │
└──────────────────────────────────────────────────────────────┘
```

### 7.3 추가 예문

```
ドアが閉めてある。         문이 (누군가에 의해) 닫혀 있다.
名前が書いてある。         이름이 (누군가에 의해) 적혀 있다.
料理が作ってある。         요리가 준비되어 있다.
ホテルが予約してある。     호텔이 예약되어 있다.
```

## 8. て로 동작 연결하기

### 8.1 순차적 동작

て형은 동작을 순서대로 연결합니다. 영어의 "and" 또는 "then"과 유사합니다.

```
朝起きて、顔を洗って、朝ご飯を食べます。
Asa okite, kao o aratte, asagohan o tabemasu.
아침에 일어나서, 세수하고, 아침밥을 먹습니다.

図書館に行って、本を借りました。
Toshokan ni itte, hon o karimashita.
도서관에 가서, 책을 빌렸습니다.

電車に乗って、会社に行きます。
Densha ni notte, kaisha ni ikimasu.
전차를 타고, 회사에 갑니다.
```

> **참고**: **마지막** 동사만 시제와 공손함을 나타냅니다. 앞의 모든 동사는 て형을 사용합니다.

### 8.2 방법이나 수단

て형은 동작이 이루어지는 방식을 묘사할 수 있습니다.

```
歩いて学校に行く。
Aruite gakkō ni iku.
걸어서 학교에 간다. (걸어서 가다)

座って食べてください。
Suwatte tabete kudasai.
앉아서 드세요.

急いで帰った。
Isoide kaetta.
서둘러서 돌아갔다. (급하게 돌아가다)
```

### 8.3 원인과 결과

```
風邪をひいて、学校を休んだ。
Kaze o hiite, gakkō o yasunda.
감기에 걸려서 학교를 쉬었다.

お金がなくて、買えなかった。
Okane ga nakute, kaenakatta.
돈이 없어서 살 수 없었다.
```

### 8.4 연결 길이

て형을 연결할 수 있는 횟수에 엄격한 제한은 없지만, 보통 3-4개가 적절합니다. 그 이상은 번거로워집니다.

```
朝起きて、シャワーを浴びて、朝ご飯を食べて、学校に行きました。
Asa okite, shawā o abite, asagohan o tabete, gakkō ni ikimashita.
아침에 일어나서, 샤워하고, 아침밥을 먹고, 학교에 갔습니다.
```

## 9. 형용사와 명사의 て형

형용사와 명사도 て형이 있으며, 연결(제6과 복습)이나 일부 문법 패턴에 사용됩니다.

### 9.1 い형용사

い를 떼고 くて를 붙입니다:

```
大きい → 大きくて   (크고...)
おいしい → おいしくて (맛있고...)
いい → よくて (좋고...) ← 불규칙
```

### 9.2 な형용사

어간 뒤에 で를 붙입니다:

```
静か → 静かで (조용하고...)
きれい → きれいで (예쁘고...)
元気 → 元気で (건강하고...)
```

### 9.3 명사

명사 뒤에 で를 붙입니다:

```
学生 → 学生で (학생이고...)
日本人 → 日本人で (일본인이고...)
```

### 9.4 문장에서의 예시

```
この部屋は広くて、明るいです。
Kono heya wa hirokute, akarui desu.
이 방은 넓고 밝습니다.

田中さんは日本人で、東京に住んでいます。
Tanaka-san wa nihonjin de, Tōkyō ni sunde imasu.
다나카 씨는 일본인이고, 도쿄에 살고 있습니다.

あの人は親切で、面白い人です。
Ano hito wa shinsetsu de, omoshiroi hito desu.
그 사람은 친절하고 재미있는 사람입니다.
```

## 10. 연습 문제

### 연습 1: て형으로 변환

각 동사의 て형을 쓰세요.

```
1. 食べる (taberu — 먹다)
2. 書く (kaku — 쓰다)
3. 読む (yomu — 읽다)
4. 行く (iku — 가다)
5. 話す (hanasu — 말하다)
6. 待つ (matsu — 기다리다)
7. 泳ぐ (oyogu — 수영하다)
8. 遊ぶ (asobu — 놀다)
9. する (suru — 하다)
10. 来る (kuru — 오다)
11. 帰る (kaeru — 돌아가다)
12. 飲む (nomu — 마시다)
```

<details>
<summary>정답 보기</summary>

```
1.  食べる → 食べて   (1단: る → て)
2.  書く  → 書いて   (5단 く → いて)
3.  読む  → 読んで   (5단 む → んで)
4.  行く  → 行って   (예외: く → って, いて가 아님)
5.  話す  → 話して   (5단 す → して)
6.  待つ  → 待って   (5단 つ → って)
7.  泳ぐ  → 泳いで   (5단 ぐ → いで)
8.  遊ぶ  → 遊んで   (5단 ぶ → んで)
9.  する  → して     (불규칙)
10. 来る  → 来て     (불규칙, 읽기: きて)
11. 帰る  → 帰って   (5단 る → って)
12. 飲む  → 飲んで   (5단 む → んで)
```

</details>

### 연습 2: 요청 만들기

각 지시를 てください를 사용하여 정중한 요청으로 바꾸세요.

```
1. Please open the window. (窓を開ける)
2. Please write your name. (名前を書く)
3. Please wait a moment. (ちょっと待つ)
4. Please speak in Japanese. (日本語で話す)
5. Please don't smoke here. (ここでタバコを吸う → 부정 요청)
6. Please come to school tomorrow. (明日学校に来る)
```

<details>
<summary>정답 보기</summary>

```
1. 窓を開けてください。
   (Mado o akete kudasai.)

2. 名前を書いてください。
   (Namae o kaite kudasai.)

3. ちょっと待ってください。
   (Chotto matte kudasai.)

4. 日本語で話してください。
   (Nihongo de hanashite kudasai.)

5. ここでタバコを吸わないでください。
   (Koko de tabako o suwanaide kudasai.)

6. 明日学校に来てください。
   (Ashita gakkō ni kite kudasai.)
```

</details>

### 연습 3: ている — 무엇을 의미하나요?

각 문장에서 ている가 (A) 진행 중인 동작, (B) 결과 상태, (C) 습관적 동작 중 무엇을 나타내는지 판별하세요.

```
1. 今、ご飯を食べている。 (지금 밥을 먹고 있다.)
2. 結婚している。 (결혼한 상태이다.)
3. 毎朝ジョギングをしている。 (매일 아침 조깅을 한다.)
4. 田中さんは東京に住んでいる。 (다나카 씨는 도쿄에 살고 있다.)
5. 子供が泣いている。 (아이가 울고 있다.)
6. あの人を知っている。 (그 사람을 알고 있다.)
```

<details>
<summary>정답 보기</summary>

```
1. (A) 진행 중인 동작 — 현재 먹고 있음
2. (B) 결과 상태 — 결혼해서 여전히 결혼 상태
3. (C) 습관적 동작 — 규칙적인 아침 습관
4. (C) 습관적 동작 / 지속 상태 — 그곳에 규칙적으로 거주
5. (A) 진행 중인 동작 — 현재 울고 있음
6. (B) 결과 상태 — 알게 되어 여전히 알고 있음
```

</details>

### 연습 4: 허가와 금지

각 문장을 일본어로 번역하세요.

```
1. May I use this pen?
2. May I take a photo here?
3. You must not run in the hallway.
4. You must not use your phone during class.
5. Yes, you may sit here.
```

<details>
<summary>정답 보기</summary>

```
1. このペンを使ってもいいですか。
   (Kono pen o tsukatte mo ii desu ka.)

2. ここで写真を撮ってもいいですか。
   (Koko de shashin o totte mo ii desu ka.)

3. 廊下で走ってはいけません。
   (Rōka de hashitte wa ikemasen.)

4. 授業中に携帯を使ってはいけません。
   (Jugyō-chū ni keitai o tsukatte wa ikemasen.)

5. はい、ここに座ってもいいですよ。
   (Hai, koko ni suwatte mo ii desu yo.)
```

</details>

### 연습 5: 동작 연결

동작들을 て형을 사용하여 하나의 문장으로 결합하세요. 마지막 동사는 존댓말(ます) 형태를 사용하세요.

```
1. wake up → eat breakfast → go to school
   (起きる → 朝ご飯を食べる → 学校に行く)

2. go to the library → borrow a book → go home
   (図書館に行く → 本を借りる → 家に帰る)

3. take the train → walk → arrive at the company
   (電車に乗る → 歩く → 会社に着く)

4. buy vegetables → make dinner → eat
   (野菜を買う → 晩ご飯を作る → 食べる)
```

<details>
<summary>정답 보기</summary>

```
1. 起きて、朝ご飯を食べて、学校に行きます。
   (Okite, asagohan o tabete, gakkō ni ikimasu.)

2. 図書館に行って、本を借りて、家に帰ります。
   (Toshokan ni itte, hon o karite, ie ni kaerimasu.)

3. 電車に乗って、歩いて、会社に着きます。
   (Densha ni notte, aruite, kaisha ni tsukimasu.)

4. 野菜を買って、晩ご飯を作って、食べます。
   (Yasai o katte, bangohan o tsukutte, tabemasu.)
```

</details>

### 연습 6: 일본어로 번역

각 문장에 적절한 て형 문법을 사용하세요.

```
1. Tanaka-san is reading a newspaper now.
2. I wake up, take a shower, and go to work.
3. A name has been written on the wall.
4. May I open this box?
5. Please don't forget your homework.
6. The children are playing in the park.
```

<details>
<summary>정답 보기</summary>

```
1. 田中さんは今新聞を読んでいます。
   (Tanaka-san wa ima shinbun o yonde imasu.)

2. 起きて、シャワーを浴びて、仕事に行きます。
   (Okite, shawā o abite, shigoto ni ikimasu.)

3. 壁に名前が書いてあります。
   (Kabe ni namae ga kaite arimasu.)

4. この箱を開けてもいいですか。
   (Kono hako o akete mo ii desu ka.)

5. 宿題を忘れないでください。
   (Shukudai o wasurenaide kudasai.)

6. 子供たちが公園で遊んでいます。
   (Kodomo-tachi ga kōen de asonde imasu.)
```

</details>
