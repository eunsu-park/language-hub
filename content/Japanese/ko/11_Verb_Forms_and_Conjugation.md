## 1. 일본어 동사 그룹 복습

고급 활용을 배우기 전에 세 가지 동사 그룹을 확실히 구분할 수 있어야 합니다.

| 그룹 | 명칭 | 구분 방법 | 예시 |
|-------|------|---------------|----------|
| 1그룹 | 五段動詞 (Godan) | U동사: 사전형이 -u행 가나로 끝남 (-iru/-eru 제외) | 書く, 話す, 読む, 飲む, 行く |
| 2그룹 | 一段動詞 (Ichidan) | Ru동사: 사전형이 -iru 또는 -eru로 끝남 | 食べる, 見る, 起きる, 寝る |
| 3그룹 | 不規則動詞 (불규칙) | 두 개의 동사만 해당 | する, 来る (くる) |

> **주의**: 2그룹처럼 보이지만 실제로는 1그룹인 동사가 있습니다:
> 帰る (かえる — 돌아가다), 知る (しる — 알다), 走る (はしる — 달리다),
> 切る (きる — 자르다), 入る (はいる — 들어가다).
> 이런 동사를 "위장 고단 동사(disguised Godan verbs)"라고 부릅니다.

## 2. 가능형 (可能形)

가능형은 **능력**을 표현합니다 — "~할 수 있다"라는 의미입니다.

### 활용 규칙

| 그룹 | 규칙 | 예시 |
|-------|------|---------|
| 1그룹 (Godan) | 마지막 -u를 -e + る로 변경 | 書く → 書**ける**, 読む → 読**める**, 話す → 話**せる** |
| 2그룹 (Ichidan) | -る을 빼고 -られる 추가 | 食べる → 食べ**られる**, 見る → 見**られる** |
| 3그룹 (불규칙) | 암기 | する → **できる**, 来る → **来られる** (こられる) |

### 전체 활용: 1그룹 (書く → 書ける)

| 형태 | 일본어 | 로마자 |
|------|----------|--------|
| 사전형 | 書ける | kakeru |
| 부정형 | 書けない | kakenai |
| 정중형 | 書けます | kakemasu |
| 과거형 | 書けた | kaketa |
| 과거 부정형 | 書けなかった | kakenakatta |

### 전체 활용: 2그룹 (食べる → 食べられる)

| 형태 | 일본어 | 로마자 |
|------|----------|--------|
| 사전형 | 食べられる | taberareru |
| 부정형 | 食べられない | taberarenai |
| 정중형 | 食べられます | taberaremasu |
| 과거형 | 食べられた | taberareta |
| 과거 부정형 | 食べられなかった | taberarenakatta |

### 조사 변경: を → が

가능문에서는 목적어 조사 を가 일반적으로 が로 바뀝니다:

```
일반문:    日本語を話す。        (나는 일본어를 말한다.)
가능문:    日本語が話せる。      (나는 일본어를 말할 수 있다.)
```

> **참고**: 일상 회화에서는 を를 그대로 사용하기도 하지만, 문법적으로는 が가 표준입니다.

### ら抜き言葉 (라 빼기 표현) — 구어체 축약형

일상적인 구어체 일본어에서 2그룹 동사의 가능형은 종종 ら를 생략합니다:

| 표준형 | 구어체 (ら抜き) | 의미 |
|----------|-------------------|---------|
| 食べ**ら**れる | 食べれる | 먹을 수 있다 |
| 見**ら**れる | 見れる | 볼 수 있다 |
| 起き**ら**れる | 起きれる | 일어날 수 있다 |

> 이 표현은 일상 회화에서 매우 흔하며, 특히 젊은 세대에서 많이 사용됩니다.
> 그러나 공식적인 글이나 JLPT 시험에서는 비표준으로 간주됩니다.

### 예문

1. **漢字が読めますか。** — 한자를 읽을 수 있습니까?
2. **明日は来られません。** — 내일은 올 수 없습니다.
3. **この店では日本語が話せる。** — 이 가게에서는 일본어를 말할 수 있다.
4. **子供の時、泳げなかった。** — 어렸을 때 수영할 수 없었다.
5. **一人でできると思う。** — 혼자서 할 수 있다고 생각한다.

## 3. 의지형 (意志形)

의지형은 **의도, 제안, 또는 권유**를 표현합니다 — "~하자", "~할까", "~하겠다"라는 의미입니다.

### 활용 규칙

| 그룹 | 규칙 | 예시 |
|-------|------|---------|
| 1그룹 (Godan) | 마지막 -u를 -o + う로 변경 | 書く → 書**こう**, 読む → 読**もう**, 行く → 行**こう** |
| 2그룹 (Ichidan) | -る을 빼고 -よう 추가 | 食べる → 食べ**よう**, 見る → 見**よう** |
| 3그룹 (불규칙) | 암기 | する → **しよう**, 来る → **来よう** (こよう) |

### 1그룹 의지형: 모든 어미

| 끝 가나 | 변화 | 예시 |
|-----------|--------|---------|
| く → こう | 書く → 書こう | |
| す → そう | 話す → 話そう | |
| つ → とう | 待つ → 待とう | |
| ぬ → のう | 死ぬ → 死のう | |
| ぶ → ぼう | 遊ぶ → 遊ぼう | |
| む → もう | 読む → 読もう | |
| る → ろう | 帰る → 帰ろう | |
| う → おう | 買う → 買おう | |
| ぐ → ごう | 泳ぐ → 泳ごう | |

### 정중한 의지형

ます형 어간에 ましょう를 붙입니다:

| 반말 | 존댓말 | 의미 |
|--------|--------|---------|
| 行こう | 行きましょう | 가자 / 갑시다 |
| 食べよう | 食べましょう | 먹자 / 먹읍시다 |
| しよう | しましょう | 하자 / 합시다 |

### 의지형의 용법

#### A. 제안 / 권유 ("~하자...")

> **映画を見よう。** — 영화를 보자.
> **一緒に勉強しましょう。** — 함께 공부합시다.

#### B. 개인적 의지 ("~하겠다...")

> **明日から早く起きよう。** — 내일부터 일찍 일어나겠다.
> **もっと頑張ろう。** — 더 열심히 하겠다.

#### C. 의지형 + と思う ("~하려고 생각하다...")

계획을 표현하는 매우 일반적인 패턴입니다:

> **来年、日本に行こうと思っている。** — 내년에 일본에 가려고 생각하고 있다.
> **新しい仕事を探そうと思う。** — 새로운 일을 찾으려고 생각한다.

> **문법 포인트**: 지속적인 의도에는 と思う(순간적인 생각)가 아닌 と思っている(지속적인 생각/계획)를 사용합니다.

### 예문

1. **今日は早く寝よう。** — 오늘은 일찍 자겠다.
2. **週末に旅行しようか。** — 주말에 여행할까?
3. **何を食べましょうか。** — 무엇을 먹을까요?
4. **日本語をもっと勉強しようと思っています。** — 일본어를 더 공부하려고 생각하고 있습니다.
5. **帰ろう。もう遅いよ。** — 돌아가자. 이미 늦었어.

## 4. 수동형 (受身形)

수동형은 주어가 동작을 **받는** 것을 표현합니다 — "~되다" 또는 "~을 당하다"라는 의미입니다.

### 활용 규칙

| 그룹 | 규칙 | 예시 |
|-------|------|---------|
| 1그룹 (Godan) | 마지막 -u를 -a + れる로 변경 | 書く → 書**かれる**, 読む → 読**まれる**, 話す → 話**される** |
| 2그룹 (Ichidan) | -る을 빼고 -られる 추가 | 食べる → 食べ**られる**, 見る → 見**られる** |
| 3그룹 (불규칙) | 암기 | する → **される**, 来る → **来られる** (こられる) |

> **참고**: 2그룹 수동형과 가능형은 형태가 동일합니다(食べられる).
> 문맥으로 구분합니다.

### 1그룹 수동형: 모든 어미

| 끝 가나 | 변화 | 예시 |
|-----------|--------|---------|
| く → かれる | 書く → 書かれる | |
| す → される | 話す → 話される | |
| つ → たれる | 待つ → 待たれる | |
| ぬ → なれる | 死ぬ → 死なれる | |
| ぶ → ばれる | 呼ぶ → 呼ばれる | |
| む → まれる | 読む → 読まれる | |
| る → られる | 取る → 取られる | |
| う → われる | 言う → 言われる | |
| ぐ → がれる | 脱ぐ → 脱がれる | |

### 수동문의 조사 구조

```
능동문:  先生が     学生を    褒めた。
         선생님이   학생을    칭찬했다.
         (선생님이 학생을 칭찬했다.)

수동문:  学生が     先生に    褒められた。
         학생이     선생님에게  칭찬받았다.
         (학생이 선생님에게 칭찬받았다.)
```

주요 조사 변화:
- 동작을 받는 사람이 주어(が)가 됩니다
- 동작을 하는 사람에게 に를 붙입니다

### 일본어 수동형의 종류

#### A. 직접 수동 (표준)

주어가 직접 동작을 받습니다 — 영어의 수동태와 동일합니다:

> **この本は多くの人に読まれている。** — 이 책은 많은 사람들에게 읽히고 있다.
> **新しい学校が建てられた。** — 새로운 학교가 지어졌다.

#### B. 간접 수동 (피해의 수동 / 迷惑の受身)

일본어 특유의 표현입니다. 누군가가 어떤 행위에 의해 **부정적인 영향을 받았음**을 나타냅니다:

> **電車で足を踏まれた。** — 전철에서 발을 밟혔다.
> **雨に降られた。** — 비를 맞았다. (비가 내려서 곤란했다!)
> **隣の人にたばこを吸われた。** — 옆 사람이 담배를 피웠다. (그래서 불쾌했다!)

> **핵심 포인트**: 간접 수동은 화자가 그 행위의 **피해자**임을 나타내는 경우가 많습니다. 이것은 영어에 직접적인 대응이 없는 표현입니다.

### 예문

1. **先生に名前を呼ばれた。** — 선생님에게 이름이 불렸다.
2. **この歌は世界中で歌われている。** — 이 노래는 전 세계에서 불리고 있다.
3. **友達に秘密を言われた。** — 친구가 비밀을 말해버렸다. (그래서 기분이 좋지 않다.)
4. **彼は皆に尊敬されている。** — 그는 모두에게 존경받고 있다.
5. **泥棒に財布を盗まれた。** — 도둑에게 지갑을 도둑맞았다.

## 5. 사역형 (使役形)

사역형은 누군가에게 무언가를 **시키거나** **허락하는** 것을 표현합니다.

### 활용 규칙

| 그룹 | 규칙 | 예시 |
|-------|------|---------|
| 1그룹 (Godan) | 마지막 -u를 -a + せる로 변경 | 書く → 書**かせる**, 読む → 読**ませる**, 行く → 行**かせる** |
| 2그룹 (Ichidan) | -る을 빼고 -させる 추가 | 食べる → 食べ**させる**, 見る → 見**させる** |
| 3그룹 (불규칙) | 암기 | する → **させる**, 来る → **来させる** (こさせる) |

### 1그룹 사역형: 모든 어미

| 끝 가나 | 변화 | 예시 |
|-----------|--------|---------|
| く → かせる | 書く → 書かせる | |
| す → させる | 話す → 話させる | |
| つ → たせる | 待つ → 待たせる | |
| ぬ → なせる | 死ぬ → 死なせる | |
| ぶ → ばせる | 遊ぶ → 遊ばせる | |
| む → ませる | 読む → 読ませる | |
| る → らせる | 帰る → 帰らせる | |
| う → わせる | 歌う → 歌わせる | |
| ぐ → がせる | 泳ぐ → 泳がせる | |

### 조사 구조

#### "시키다" (강제)

동작을 시키는 대상에 **を**를 붙입니다:

> **先生は学生**を**走らせた。** — 선생님은 학생들을 달리게 했다.
> **母は子供**を**勉強させた。** — 어머니는 아이를 공부시켰다.

#### "허락하다" (허가)

동작을 허락받은 대상에 **に**를 붙입니다:

> **先生は学生**に**好きな本を読ませた。** — 선생님은 학생들에게 좋아하는 책을 읽게 해주었다.
> **母は子供**に**遊ばせた。** — 어머니는 아이를 놀게 해주었다.

> **원칙**: を = 강제(시키다), に = 허가(하게 해주다).
> 본동사가 타동사여서 이미 を 목적어가 있는 경우, を의 중복을 피하기 위해 강제 맥락에서도 사람에게 に를 사용합니다.

### させてください — "~하게 해주세요"

매우 유용한 정중한 요청 표현입니다:

> **説明させてください。** — 설명하게 해주세요.
> **自己紹介させていただきます。** — 자기소개를 하겠습니다. (겸양)
> **考えさせてください。** — 생각할 시간을 주세요.

### 예문

1. **部長は田中さんを出張させた。** — 부장은 다나카 씨를 출장 보냈다.
2. **子供にゲームをさせてあげた。** — 아이에게 게임을 하게 해주었다.
3. **もう少し考えさせてください。** — 조금 더 생각하게 해주세요.
4. **彼女は彼を3時間も待たせた。** — 그녀는 그를 3시간이나 기다리게 했다.
5. **先生は学生に自由に話させた。** — 선생님은 학생들에게 자유롭게 말하게 해주었다.

## 6. 사역수동형 (使役受身形)

사역수동형은 사역과 수동을 결합한 것입니다: "~하도록 강요받다"라는 의미입니다. 강제로 시킴을 당하는 사람의 관점에서 표현합니다.

### 활용 규칙

| 그룹 | 규칙 | 예시 |
|-------|------|---------|
| 1그룹 (Godan) | 마지막 -u를 -a + せられる로 변경 | 書く → 書**かせられる**, 読む → 読**ませられる** |
| 2그룹 (Ichidan) | -る을 빼고 -させられる 추가 | 食べる → 食べ**させられる** |
| 3그룹 (불규칙) | 암기 | する → **させられる**, 来る → **来させられる** (こさせられる) |

### 축약형 (1그룹만 해당)

구어체 일본어에서 1그룹 사역수동형은 종종 축약됩니다:

| 전체형 | 축약형 | 의미 |
|-----------|---------------|---------|
| 書かせられる | 書かされる | 쓰도록 시킴을 당하다 |
| 読ませられる | 読まされる | 읽도록 시킴을 당하다 |
| 飲ませられる | 飲まされる | 마시도록 시킴을 당하다 |
| 行かせられる | 行かされる | 가도록 시킴을 당하다 |
| 待たせられる | 待たされる | 기다리도록 시킴을 당하다 |

> **예외**: す로 끝나는 동사(話す 등)는 축약형이 **없습니다**.
> 話させられる는 *話さされる가 되지 않습니다.

### 조사 구조

```
[강제당하는 사람]は  [강제하는 사람]に  [행위]させられる

私は    先生に    走らせられた。
나는    선생님에게  달리도록 시킴을 당했다.
(나는 선생님에게 달리도록 시킴을 당했다.)
```

### 예문

1. **毎日野菜を食べさせられた。** — 매일 채소를 먹도록 시킴을 당했다.
2. **子供の頃、ピアノを練習させられた。** — 어렸을 때 피아노 연습을 시킴당했다.
3. **上司に残業させられた。** — 상사에게 야근을 시킴당했다.
4. **嫌いな歌を歌わされた。** — 싫어하는 노래를 부르도록 시킴당했다. (축약형)
5. **3時間も待たされた。** — 3시간이나 기다리도록 시킴당했다. (축약형)

## 7. 활용형 비교 도표

### 食べる (2그룹, 먹다)를 모델로 사용

| 형태 | 일본어 | 의미 |
|------|----------|---------|
| 사전형 | 食べる | 먹다 |
| 가능형 | 食べられる | 먹을 수 있다 |
| 의지형 | 食べよう | 먹자 / 먹겠다 |
| 수동형 | 食べられる | 먹히다 |
| 사역형 | 食べさせる | 먹게 하다 / 먹게 해주다 |
| 사역수동형 | 食べさせられる | 먹도록 시킴당하다 |

### 書く (1그룹, 쓰다)를 모델로 사용

| 형태 | 일본어 | 의미 |
|------|----------|---------|
| 사전형 | 書く | 쓰다 |
| 가능형 | 書ける | 쓸 수 있다 |
| 의지형 | 書こう | 쓰자 / 쓰겠다 |
| 수동형 | 書かれる | 쓰이다 |
| 사역형 | 書かせる | 쓰게 하다 / 쓰게 해주다 |
| 사역수동형 | 書かせられる / 書かされる | 쓰도록 시킴당하다 |

### する (3그룹, 하다)를 모델로 사용

| 형태 | 일본어 | 의미 |
|------|----------|---------|
| 사전형 | する | 하다 |
| 가능형 | できる | 할 수 있다 |
| 의지형 | しよう | 하자 / 하겠다 |
| 수동형 | される | 되다 |
| 사역형 | させる | 시키다 / 하게 해주다 |
| 사역수동형 | させられる | 시킴당하다 |

### 来る (3그룹, 오다)를 모델로 사용

| 형태 | 일본어 | 의미 |
|------|----------|---------|
| 사전형 | 来る (くる) | 오다 |
| 가능형 | 来られる (こられる) | 올 수 있다 |
| 의지형 | 来よう (こよう) | 오자 / 오겠다 |
| 수동형 | 来られる (こられる) | 오게 되다 (드묾) |
| 사역형 | 来させる (こさせる) | 오게 하다 / 오게 해주다 |
| 사역수동형 | 来させられる (こさせられる) | 오도록 시킴당하다 |

## 8. 흔한 실수와 주의점

### 실수 1: 2그룹의 가능형과 수동형 혼동

食べられる(가능)와 食べられる(수동) 모두 형태가 동일합니다.

```
食べられる = 먹을 수 있다 (가능) 또는 먹히다 (수동)
문맥으로 구분합니다:
  ○ 刺身が食べられる。         (사시미를 먹을 수 있다.) — 가능
  ○ 刺身は世界中で食べられている。(사시미는 전 세계에서 먹히고 있다.) — 수동
```

### 실수 2: 동사 그룹 잘못 판별

```
  ✗ 帰る → 帰られる  (2그룹으로 잘못 취급)
  ○ 帰る → 帰れる    (1그룹 가능형: かえ-る → かえ-れる)
```

### 실수 3: 사역형에서 を 중복

```
  ✗ 先生は学生を本を読ませた。  (を 조사가 두 개!)
  ○ 先生は学生に本を読ませた。  (사람에 に를 사용)
```

### 실수 4: す동사에 축약형 사역수동형 사용

```
  ✗ 話さされる  (부정확 — 어색함)
  ○ 話させられる (す동사는 전체형을 유지)
```

## 9. 연습 문제

### 연습 1: 가능형 만들기

가능형으로 변환하세요.

1. 日本語を話す → 日本語が ________
2. この漢字を読む → この漢字が ________
3. 朝早く起きる → 朝早く ________
4. 明日来る → 明日 ________
5. 一人で料理する → 一人で料理 ________

<details>
<summary>정답</summary>

1. 日本語が**話せる** (1그룹: 話す → 話せる)
2. この漢字が**読める** (1그룹: 読む → 読める)
3. 朝早く**起きられる** (2그룹: 起きる → 起きられる)
4. 明日**来られる** (こられる) (3그룹: 来る → 来られる)
5. 一人で料理**できる** (3그룹: する → できる)

</details>

### 연습 2: 의지형 만들기

의지형(반말)으로 변환하세요.

1. 映画を見る → ________
2. 公園を走る → ________
3. コーヒーを飲む → ________
4. 友達に電話する → ________
5. パーティーに行く → ________

<details>
<summary>정답</summary>

1. 映画を**見よう** (2그룹: 見る → 見よう)
2. 公園を**走ろう** (1그룹: 走る → 走ろう)
3. コーヒーを**飲もう** (1그룹: 飲む → 飲もう)
4. 友達に電話**しよう** (3그룹: する → しよう)
5. パーティーに**行こう** (1그룹: 行く → 行こう)

</details>

### 연습 3: 능동문을 수동문으로

수동문으로 다시 쓰세요.

1. 先生が学生を褒めた。(선생님이 학생을 칭찬했다.)
2. 犬が弟を噛んだ。(개가 남동생을 물었다.)
3. 誰かが窓を割った。(누군가가 창문을 깨뜨렸다.)
4. 母が私を起こした。(어머니가 나를 깨웠다.)

<details>
<summary>정답</summary>

1. 学生が先生**に褒められた。** (학생이 선생님에게 칭찬받았다.)
2. 弟が犬**に噛まれた。** (남동생이 개에게 물렸다.)
3. 窓が誰か**に割られた。** (창문이 누군가에게 깨뜨려졌다.)
4. 私が母**に起こされた。** (나는 어머니에게 깨워졌다.)

</details>

### 연습 4: 사역형 만들기

사역형을 사용하여 완성하세요.

1. 母は子供に野菜を ________ (食べる — 먹게 하다)
2. 先生は学生を ________ (走る — 달리게 하다)
3. 部長は私に報告を ________ (する — 시키다)
4. もう少し ________ ください。(考える — 생각하게 해주다)

<details>
<summary>정답</summary>

1. 母は子供に野菜を**食べさせた。** (2그룹: 食べる → 食べさせる)
2. 先生は学生を**走らせた。** (1그룹: 走る → 走らせる)
3. 部長は私に報告を**させた。** (3그룹: する → させる)
4. もう少し**考えさせて**ください。 (2그룹: 考える → 考えさせる)

</details>

### 연습 5: 사역수동형

강제당하는 사람의 관점에서 다시 쓰세요.

1. 先生が私を立たせた。→ 私は先生に ________
2. 母が子供に薬を飲ませた。→ 子供は母に薬を ________
3. 上司が部下を残業させた。→ 部下は上司に ________

<details>
<summary>정답</summary>

1. 私は先生に**立たせられた / 立たされた。** (선생님에게 서도록 시킴당했다.)
2. 子供は母に薬を**飲ませられた / 飲まされた。** (아이는 어머니에게 약을 먹도록 시킴당했다.)
3. 部下は上司に**残業させられた。** (부하는 상사에게 야근을 시킴당했다.)

</details>

### 연습 6: 혼합 활용형 식별

각 문장에서 사용된 동사 활용형을 식별하세요.

1. この本は世界中で読まれている。
2. 日本語が少し話せるようになった。
3. 子供の頃、毎日牛乳を飲まされた。
4. 今日は早く帰ろう。
5. 母に部屋を掃除させられた。

<details>
<summary>정답</summary>

1. **수동형** (読まれている — 읽히고 있다)
2. **가능형** (話せる — 말할 수 있다)
3. **사역수동형 축약** (飲まされた — 마시도록 시킴당했다)
4. **의지형** (帰ろう — 돌아가자)
5. **사역수동형** (掃除させられた — 청소하도록 시킴당했다)

</details>
