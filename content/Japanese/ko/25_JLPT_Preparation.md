## 1. JLPT 개요

### 1.1 JLPT란?

```
+----------------------------------------------------------+
|  JLPT — Japanese Language Proficiency Test               |
|  (日本語能力試験 / にほんごのうりょくしけん)                 |
+----------------------------------------------------------+
|                                                          |
|  시행 기관: Japan Foundation +                             |
|             일본국제교육지원협회 (JEES)                     |
|  인정: 국제적, 유효기간 없음                               |
|  급수: N5 (가장 쉬움) → N1 (가장 어려움)                   |
|  시험일: 7월과 12월 (연 2회)                               |
|  시행 장소: 전 세계 약 90개국                              |
|                                                          |
|  시험 구성:                                               |
|  ┌─────────────────────────────────────────────┐         |
|  │  N5-N4:                                     │         |
|  │  1. 言語知識（文字・語彙）                     │         |
|  │     언어지식 (문자/어휘)                       │         |
|  │  2. 言語知識（文法）・読解                     │         |
|  │     언어지식 (문법) + 독해                     │         |
|  │  3. 聴解                                    │         |
|  │     청해                                     │         |
|  ├─────────────────────────────────────────────┤         |
|  │  N3-N1:                                     │         |
|  │  1. 言語知識（文字・語彙・文法）・読解          │         |
|  │     언어지식 + 독해                           │         |
|  │  2. 聴解                                    │         |
|  │     청해                                     │         |
|  └─────────────────────────────────────────────┘         |
|                                                          |
|  채점:                                                    |
|  • 각 섹션 별도 채점                                      |
|  • 각 섹션 최소 점수를 충족해야 함                          |
|  • 그리고 총점 합격 점수도 충족해야 함                      |
|  • 모든 문제는 객관식                                     |
+----------------------------------------------------------+
```

### 1.2 채점 체계

| 급수 | 총점 | 합격 점수 | 섹션 최소 | 섹션 구성 |
|-------|-----------|---------------|-------------|----------|
| N1 | 180 | 100 | 각 19/60 | 언어지식 + 독해 (0-60), 청해 (0-60) |
| N2 | 180 | 90 | 각 19/60 | 언어지식 + 독해 (0-60), 청해 (0-60) |
| N3 | 180 | 95 | 각 19/60 | 언어지식 (0-60), 독해 (0-60), 청해 (0-60) |
| N4 | 180 | 90 | 38/120 + 19/60 | 언어지식 + 독해 (0-120), 청해 (0-60) |
| N5 | 180 | 80 | 38/120 + 19/60 | 언어지식 + 독해 (0-120), 청해 (0-60) |

> **중요**: 총점이 합격 점수를 넘더라도, 어느 한 섹션이라도 최소 점수에 미달하면 불합격입니다. 모든 섹션에 걸쳐 균형 잡힌 준비를 하십시오.

### 1.3 시험 형식 상세

| 급수 | 시험 시간 | 문항 수 (약) |
|-------|----------|-------------------|
| N1 | 170분 (110 + 60) | 약 100문항 이상 |
| N2 | 155분 (105 + 50) | 약 100문항 이상 |
| N3 | 140분 (70 + 30 + 40) | 약 100문항 이상 |
| N4 | 125분 (60 + 35 + 30) | 약 90문항 이상 |

## 3. 섹션 1: 언어지식 (言語知識)

### 3.1 문자와 어휘 (文字・語彙)

이 하위 섹션에서는 한자 읽기, 어휘 의미, 단어 용법을 테스트합니다.

**문제 유형**:

| 유형 | 평가 내용 | 형식 |
|------|--------------|--------|
| 漢字読み | 한자 읽기 | 밑줄 친 한자의 올바른 읽기 선택 |
| 表記 | 한자 표기 | 밑줄 친 히라가나의 올바른 한자 선택 |
| 文脈規定 | 문맥 속 단어 | 문장에 맞는 단어 선택 |
| 言い換え | 바꿔 말하기 | 가장 가까운 의미의 문장 선택 |
| 用法 | 단어 용법 | 단어가 올바르게 쓰인 문장 선택 |

**예시 — 한자 읽기 (N2 수준)**:

```
問題：この地域は＿交通の便＿がいい。
      「交通の便」の読み方は？

  1. こうつうのべん
  2. こうつうのびん
  3. こうつうのたより
  4. こうづうのべん

정답: 1 (こうつうのべん)
```

**예시 — 문맥 규정 단어 (N2 수준)**:

```
問題：この問題は非常に（　　）で、簡単には解決できない。

  1. 素朴     2. 複雑
  3. 純粋     4. 単純

정답: 2 (複雑 / 복잡한 — 단서는 簡単には解決できない)
```

### 3.2 문법 (文法)

문법 문제는 문형 지식과 적절한 사용을 테스트합니다.

**문제 유형**:

| 유형 | 평가 내용 | 형식 |
|------|--------------|--------|
| 文法形式の判断 | 문법 선택 | 올바른 문법 형태 선택 |
| 文の組み立て | 문장 구성 | 단어를 올바른 순서로 배열 |
| 文章の文法 | 문장 속 문법 | 텍스트에서 올바른 형태 선택 |

**예시 — 문법 선택 (N2 수준)**:

```
問題：彼は忙しい（　　）、いつも笑顔で対応してくれる。

  1. にしては     2. にもかかわらず
  3. としても     4. に関わらず

정답: 2 (にもかかわらず — 바쁜데도 불구하고)
```

**예시 — 문장 구성 (별표 문제 ★)**:

```
問題：
彼女は ___  ___  ★  ___ 人だ。

  1. 困っている   2. 助ける
  3. を          4. いつも

배열: 彼女は いつも 困っている 人 を 助ける 人だ。
(그녀는 항상 어려움에 처한 사람을 돕는 사람이다.)

★ 위치의 답: を (선택지 3)
```

### 3.3 급수별 핵심 문법 포인트

**N3 필수 문법**:

| 문법 | 의미 | 예시 |
|---------|---------|---------|
| ～ようにする | ~하려고 노력하다 / 하도록 하다 | 毎日運動するようにしている |
| ～ことにする | ~하기로 결정하다 | 来年留学することにした |
| ～ばかり | ~뿐 / 방금 ~했다 | テレビばかり見ている |
| ～はずだ | ~일 것이다 / ~할 예정이다 | 彼は来るはずだ |
| ～ために | ~하기 위해 / ~ 때문에 | 合格するために勉強する |

**N2 필수 문법**:

| 문법 | 의미 | 예시 |
|---------|---------|---------|
| ～にもかかわらず | ~에도 불구하고 | 雨にもかかわらず出かけた |
| ～一方（で） | 한편으로 | 便利な一方、危険もある |
| ～に伴い | ~에 따라 | 人口増加に伴い問題が増えた |
| ～に基づいて | ~에 기초하여 | データに基づいて判断する |
| ～ざるを得ない | ~하지 않을 수 없다 | 行かざるを得ない |

**N1 필수 문법**:

| 문법 | 의미 | 예시 |
|---------|---------|---------|
| ～をもって | ~로써 / ~을 기해 | 本日をもって閉店いたします |
| ～ともなると | ~쯤 되면 | 社長ともなると責任が重い |
| ～ないものでもない | ~하지 못할 것도 없다 | 行けないものでもない |
| ～をものともせず | ~에 굴하지 않고 | 批判をものともせず進んだ |
| ～たりとも...ない | 단 ~도 ...않다 | 一日たりとも忘れない |

## 4. 섹션 2: 독해 (読解)

### 4.1 독해 문제 유형

| 유형 | 일본어 | 길이 | 평가 내용 |
|------|----------|--------|---------------|
| 단문 | 短文 | 150-200자 | 요지 파악 |
| 중문 | 中文 | 350-500자 | 세부 사항 이해, 인과관계 |
| 장문 | 長文 | 600-900자 | 전체 이해, 논리 흐름 |
| 통합 | 統合理解 | 두 개의 관련 텍스트 | 관점 비교 |
| 정보 검색 | 情報検索 | 도표, 전단지, 안내문 | 특정 정보 찾기 |

### 4.2 독해 전략

```
+----------------------------------------------------------+
|         JLPT READING STRATEGIES                          |
+----------------------------------------------------------+
|                                                          |
|  지문을 읽기 전에:                                        |
|  1. 질문을 먼저 읽기                                      |
|  2. 무엇을 찾아야 하는지 파악하기                          |
|  3. 질문의 핵심어 메모하기                                 |
|                                                          |
|  읽는 중:                                                 |
|  4. 각 문단의 첫 문장과 마지막 문장에 집중하기              |
|  5. 접속사에 주의하기:                                    |
|     しかし = 대조가 옴                                    |
|     つまり = 요약/바꿔 말하기                              |
|     したがって = 결론                                     |
|  6. 핵심 정보에 밑줄 또는 표시하기                         |
|  7. 모르는 단어에 걸려서 멈추지 않기                       |
|                                                          |
|  답할 때:                                                 |
|  8. 확실히 틀린 답을 먼저 소거하기                         |
|  9. 정답은 보통 지문의 직접 인용이 아니라                   |
|     바꿔 말한 것                                          |
|  10. 사실이지만 질문에 대한 답이 아닌                      |
|      선택지에 주의하기                                    |
+----------------------------------------------------------+
```

### 4.3 자주 출제되는 독해 질문 패턴

| 질문 패턴 | 일본어 | 묻는 내용 |
|-----------------|----------|--------------|
| 주제 | 筆者が最も言いたいことは何か | 저자가 가장 말하고 싶은 것은? |
| 이유 | ○○はなぜか | ○○는 왜인가? |
| 의미 | ○○とはどういう意味か | ○○는 어떤 의미인가? |
| 지시 대상 | ○○とは何を指しているか | ○○는 무엇을 가리키는가? |
| 일치하는 진술 | 本文の内容と合っているものはどれか | 본문 내용과 일치하는 것은? |
| 저자의 의견 | 筆者の考えに合うものはどれか | 저자의 생각과 일치하는 것은? |

### 4.4 단문 예시 (N2 수준)

```
  日本では、電車の中で携帯電話で話すことはマナー違反と
されている。しかし、メールを打ったり、音楽を聴いたりする
ことは一般的に許容されている。このことから分かるように、
日本のマナーにおいて問題とされるのは、周囲の人に
直接的な迷惑をかける行為なのである。

問：筆者が言いたいことは何か。

  1. 日本では電車の中でスマートフォンを使ってはいけない。
  2. 日本のマナーは他の国と比べて厳しすぎる。
  3. 日本のマナーの基準は、他人への直接的な影響があるかどうかだ。
  4. 日本人は電車の中で電話をしないので静かである。
```

<details>
<summary>정답과 해설</summary>

**정답: 3**

핵심 문장은 마지막 문장입니다: 「周囲の人に直接的な迷惑をかける行為なのである」(주위 사람에게 직접적으로 피해를 주는 행위가 문제가 된다). 선택지 3이 이를 올바르게 바꿔 말하고 있습니다. 선택지 1은 너무 포괄적이고 (メール은 괜찮음), 선택지 2는 언급되지 않았으며, 선택지 4는 세부 사항이지 요지가 아닙니다.

</details>

## 5. 섹션 3: 청해 (聴解)

### 5.1 청해 문제 유형

| 유형 | 일본어 | 평가 내용 |
|------|----------|---------------|
| 과제 이해 | 課題理解 | 대화를 듣고 취해야 할 행동 파악 |
| 포인트 이해 | ポイント理解 | 특정 정보 듣고 파악 |
| 개요 이해 | 概要理解 | 전체 메시지나 화자의 의도 이해 |
| 즉시 응답 | 即時応答 | 짧은 발언에 대한 자연스러운 응답 선택 |
| 통합 이해 | 統合理解 | 긴 지문을 듣고 여러 관점 비교 |

### 5.2 청해 전략

```
+----------------------------------------------------------+
|         JLPT LISTENING STRATEGIES                        |
+----------------------------------------------------------+
|                                                          |
|  음성 재생 전에:                                          |
|  1. 질문과 선택지를 빠르게 읽기                            |
|  2. 어떤 종류의 정보가 필요한지 예측하기                    |
|  3. 그림 문제는 모든 선택지를 살펴보기                      |
|                                                          |
|  음성 재생 중:                                            |
|  4. 간단히 메모하기 (숫자, 시간, 핵심어)                   |
|  5. 부정과 계획 변경에 주의하기:                           |
|     「～しようと思ったけど、やっぱり...」                   |
|     「～にしようか。いや、やっぱり...」                     |
|  6. 최종 결정에 주목하기                                  |
|  7. 완곡한 거절에 주의하기:                                |
|     「ちょっと...」「考えておきます」                       |
|                                                          |
|  음성 재생 후:                                            |
|  8. 불확실하면 불가능한 답을 소거하기                       |

+----------------------------------------------------------+
```

### 5.3 자주 출제되는 청해 함정

### 7.3 독해 연습 (N2 수준)

다음을 읽고 질문에 답하십시오:

```
  ペットを飼うことは子供の教育にいいと言われることが多い。
確かに、動物の世話をすることで責任感が育つという面はある。
しかし、それは親が適切に指導した場合の話であって、
ペットを与えれば自動的に子供が成長するわけではない。
むしろ、親自身が動物の命の重さを理解し、
模範を示すことが何より重要なのではないだろうか。
```

問：筆者の意見として最も適切なものはどれか。

1. ペットを飼うことは子供の教育に悪い影響を与える。
2. 子供にペットを与えれば、自然に責任感が身につく。
3. ペットが子供の教育に良い効果をもたらすには、親の役割が大切だ。
4. 子供よりも親のほうがペットの世話が上手だ。

<details>
<summary>정답</summary>

**3** — 저자의 주요 논점은 마지막 문장에 있습니다: 부모 자신이 생명의 무게를 이해하고 모범을 보이는 것이 가장 중요하다. 글은 반려동물 키우기가 교육에 좋은 것은 부모가 적절히 지도할 경우에만 해당된다고 주장합니다. 선택지 1은 지나치게 부정적이고, 선택지 2는 저자가 반대하는 내용이며, 선택지 4는 논의되지 않았습니다.

</details>

## 10. 연습 문제

### 연습 1: 급수 평가

각 문장이 어느 JLPT 급수(N5, N4, N3, N2, N1)에 해당하는지 파악하십시오:

<details>

1. **N5** — 기본 서술문 (これは～です)
2. **N2** — にかかわらず 문법 패턴

5. **N3** — ～そうです (전문), 복합 어휘 (出張)과 결합

</details>

### 연습 2: 문법 패턴 식별

문법 패턴과 의미를 연결하십시오:

| 패턴 | 의미 |
|---------|---------|
| 1. ～ざるを得ない | a. ~하자마자 |
| 2. ～に先立ち | b. ~하지 않을 수 없다 |
| 3. ～たとたん | c. 반드시 ~인 것은 아니다 |
| 4. ～とは限らない | d. ~에 앞서 |
| 5. ～を踏まえて | e. ~을 근거로 / ~을 고려하여 |

<details>
<summary>정답</summary>

1 → b (～ざるを得ない = ~하지 않을 수 없다)
2 → d (～に先立ち = ~에 앞서)
3 → a (～たとたん = ~하자마자 / ~한 순간)
4 → c (～とは限らない = 반드시 ~인 것은 아니다)
5 → e (～を踏まえて = ~을 근거로 / ~을 고려하여)

</details>

### 연습 3: 시간 제한 어휘 드릴

타이머를 2분으로 설정하십시오. 각각의 올바른 답을 선택하십시오:

1. この問題の（　　）を考えなければならない。
   a. 対処　b. 対策　c. 対応　d. 対立

2. 新しい法律が来月（　　）される。
   a. 実行　b. 施行　c. 遂行　d. 実施

3. 会議で彼の意見が（　　）された。
   a. 採用　b. 採択　c. 採集　d. 採点

4. 環境問題に対する（　　）が高まっている。
   a. 意識　b. 意志　c. 意欲　d. 意味

5. この製品は品質に（　　）がある。
   a. 自身　b. 自信　c. 自心　d. 自進

<details>
<summary>정답</summary>

1. **b. 対策 (たいさく)** — 대책/전략. 対策を考える = 대책을 생각하다
2. **b. 施行 (しこう)** — 시행 (법률의). 法律が施行される = 법률이 시행되다
3. **a. 採用 (さいよう)** — 채용/채택. 意見が採用される = 의견이 채택되다
4. **a. 意識 (いしき)** — 의식/인식. 意識が高まる = 인식이 높아지다
5. **b. 自信 (じしん)** — 자신감. 品質に自信がある = 품질에 자신이 있다

</details>

### 연습 4: 청해 시뮬레이션

다음 대화를 읽고 정답을 예측하십시오 (실제 시험에서는 음성으로 듣습니다):

```
女：山田さん、明日の会議、10時からでしたよね。
男：あ、さっき部長から連絡があって、
    11時に変更になったんです。
女：そうですか。場所は同じですか。
男：場所も変わって、3階の大会議室になりました。
女：分かりました。ありがとうございます。
```

問：明日の会議について正しいものはどれか。

1. 10時から3階の大会議室で行う。
2. 11時から元の場所で行う。
3. 11時から3階の大会議室で行う。
4. 10時から元の場所で行う。

<details>
<summary>정답</summary>

**3** — 회의는 11시로 변경되었고 (11時に変更) 장소도 3층 대회의실로 바뀌었습니다 (3階の大会議室). 시간과 장소 모두 변경되었습니다.

이것은 전형적인 JLPT 청해 패턴입니다: 원래 계획을 언급한 후 변경이 이루어집니다. 정답은 항상 최종 결정입니다.

</details>
