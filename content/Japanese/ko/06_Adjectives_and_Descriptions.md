## 1. 일본어 형용사의 두 가지 유형

일본어 형용사는 두 가지 뚜렷한 범주로 나뉩니다. 모든 형용사가 같은 방식으로 동작하는 영어와 달리, 일본어 형용사는 유형에 따라 다른 활용 규칙을 따릅니다.

> **일본어 형용사 유형 (形容詞の種類)**
>
> い형용사 (i-keiyōshi)             な형용사 (na-keiyōshi)
> ──────────────────────────       ────────────────────────────
> 사전형이 い(i)로 끝남             보통 い로 끝나지 않음
> (예외 있음)
>
> 어미 い를 변화시켜 활용           형용사 어간 뒤에
> だ/です를 사용하여 활용
>
> - **예시**: 예시:
> 大きい (ōkii) — 큰               静か (shizuka) — 조용한
> 小さい (chiisai) — 작은          きれい (kirei) — 예쁜
> 高い (takai) — 높은/비싼         元気 (genki) — 건강한
> 新しい (atarashii) — 새로운      好き (suki) — 좋아하는
>
> - **※ 주의**: きれい와 嫌い는 い형용사처럼 보이지만
> 실제로는 な형용사입니다! 반드시 암기해야 합니다.

### 1.1 구별하는 방법

사전형이 핵심입니다:

| 특징 | い형용사 | な형용사 |
|---------|-------------|-------------|
| 사전형 어미 | 항상 い로 끝남 | 다양한 어미 |
| 명사 앞 | 〜い + 명사 | 〜な + 명사 |
| 활용 | い 어미를 변화 | だ/です 체계를 사용 |
| 예시 | 大き**い** 犬 (큰 개) | 静か**な** 部屋 (조용한 방) |

> **중요**: 일부 な형용사는 い로 끝나, い형용사처럼 보입니다. 가장 흔한 함정은 다음과 같습니다:

| 단어 | 읽기 | 유형 | 의미 |
|------|---------|------|---------|
| きれい | kirei | **な형용사** | 예쁜, 깨끗한 |
| 嫌い | kirai | **な형용사** | 싫어하는 |
| 有名 | yūmei | **な형용사** | 유명한 |

## 2. い형용사 활용

い형용사는 마지막 い를 떼고 적절한 어미를 붙여 활용합니다. 이것은 일본어에서 가장 규칙적인 패턴 중 하나입니다.

### 2.1 활용표

高い (takai — 높은, 비싼)를 모델로 사용합니다:

| 형태 | 반말 | 존댓말 | 의미 |
|------|--------|--------|---------|
| 현재 긍정 | 高い | 高いです | 높다/비싸다 |
| 현재 부정 | 高くない | 高くないです / 高くありません | 높지 않다/비싸지 않다 |
| 과거 긍정 | 高かった | 高かったです | 높았다/비쌌다 |
| 과거 부정 | 高くなかった | 高くなかったです / 高くありませんでした | 높지 않았다/비싸지 않았다 |

### 2.2 패턴

> **い형용사 활용 패턴**
>
> 어간 = 형용사에서 마지막 い를 뺀 것
>
> - **현재 (+)**: 어간 + い          高い
> - **현재 (-)**: 어간 + くない      高くない
> - **과거 (+)**: 어간 + かった      高かった
> - **과거 (-)**: 어간 + くなかった  高くなかった
>
> 高  →  어간
> 高い  →  현재 (+)
> 高く  →  부정/부사 어간
> 高か  →  과거 어간

### 2.3 자주 쓰이는 い형용사 예문

**おいしい (oishii — 맛있는)**

```
おいしい           맛있다.
おいしくない       맛없다.
おいしかった       맛있었다.
おいしくなかった   맛없었다.
```

**新しい (atarashii — 새로운)**

```
新しい車           새 차
この車は新しい。   이 차는 새 것이다.
この車は新しくない。 이 차는 새 것이 아니다.
あの車は新しかった。 그 차는 새 것이었다.
```

**寒い (samui — 추운, 날씨/체감)**

```
今日は寒い。       오늘은 춥다.
昨日は寒くなかった。 어제는 춥지 않았다.
```

## 3. な형용사 활용

な형용사는 활용할 때 명사와 비슷하게 동작합니다. 활용에는 판정사(copula) だ(반말) 또는 です(존댓말)를 사용합니다.

### 3.1 활용표

静か (shizuka — 조용한)를 모델로 사용합니다:

| 형태 | 반말 | 존댓말 | 의미 |
|------|--------|--------|---------|
| 현재 긍정 | 静かだ | 静かです | 조용하다 |
| 현재 부정 | 静かじゃない | 静かじゃないです / 静かではありません | 조용하지 않다 |
| 과거 긍정 | 静かだった | 静かでした | 조용했다 |
| 과거 부정 | 静かじゃなかった | 静かじゃなかったです / 静かではありませんでした | 조용하지 않았다 |

### 3.2 패턴

> **な형용사 활용 패턴**
>
> - **현재 (+)**: 어간 + だ/です
> - **현재 (-)**: 어간 + じゃない / ではない
> - **과거 (+)**: 어간 + だった / でした
> - **과거 (-)**: 어간 + じゃなかった / ではなかった
>
> - **명사 앞**: 어간 + な + 명사
> 静かな部屋 (조용한 방)
>
> ※ じゃ는 では의 구어체 축약형

### 3.3 자주 쓰이는 な형용사 예문

**元気 (genki — 건강한, 활기찬)**

```
田中さんは元気だ。         다나카 씨는 건강하다.
田中さんは元気じゃない。   다나카 씨는 건강하지 않다.
田中さんは元気だった。     다나카 씨는 건강했다.
田中さんは元気じゃなかった。 다나카 씨는 건강하지 않았다.
```

**好き (suki — 좋아하는)**

```
猫が好きだ。               고양이를 좋아한다. (직역: 고양이가 좋다.)
猫が好きじゃない。         고양이를 좋아하지 않는다.
猫が好きだった。           고양이를 좋아했다.
```

**きれい (kirei — 예쁜, 깨끗한)** — な형용사라는 것을 기억하세요!

```
この花はきれいだ。         이 꽃은 예쁘다.
きれいな花                 예쁜 꽃 (きれいい花가 아님)
```

## 4. 연체 수식: 명사 앞에서의 형용사

형용사가 명사 바로 앞에 와서 명사를 수식할 때, 유형에 따라 규칙이 다릅니다.

### 4.1 명사 앞의 い형용사

い형용사를 사전형 그대로 명사 앞에 놓으면 됩니다:

```
大きい犬       ōkii inu           큰 개
小さい猫       chiisai neko       작은 고양이
おいしい料理   oishii ryōri       맛있는 음식
新しい本       atarashii hon      새 책
古い家         furui ie           오래된 집
長い髪         nagai kami         긴 머리카락
```

부정 연체 수식의 경우:

```
おいしくない料理   맛없는 음식
新しくない本       새것이 아닌 책
```

### 4.2 명사 앞의 な형용사

형용사 어간과 명사 사이에 な를 붙입니다:

```
静かな部屋     shizuka na heya        조용한 방
きれいな花     kirei na hana          예쁜 꽃
元気な子供     genki na kodomo        활기찬 아이
有名な人       yūmei na hito         유명한 사람
好きな食べ物   suki na tabemono      좋아하는 음식
大切な人       taisetsu na hito      소중한 사람
```

부정 연체 수식의 경우:

```
静かじゃない部屋   조용하지 않은 방
元気じゃない人     건강하지 않은 사람
```

### 4.3 비교표

| 유형 | 긍정 | 부정 |
|------|-------------|----------|
| い형용사 + 명사 | 大きい犬 (큰 개) | 大きくない犬 (크지 않은 개) |
| な형용사 + 명사 | 静かな部屋 (조용한 방) | 静かじゃない部屋 (조용하지 않은 방) |

## 5. 서술 용법: 문장 끝에서의 형용사

형용사가 문장의 주요 서술어(문장 끝에 위치)일 때도, 유형에 따라 규칙이 다릅니다.

### 5.1 서술어로서의 い형용사

```
반말:
  この映画は面白い。       이 영화는 재미있다.
  この映画は面白くない。   이 영화는 재미없다.

존댓말:
  この映画は面白いです。       이 영화는 재미있습니다.
  この映画は面白くないです。   이 영화는 재미없습니다.
```

> **참고**: い형용사는 반말에서 だ를 붙이지 않습니다. 面白いだ라고 하면 **틀립니다**.

### 5.2 서술어로서의 な형용사

```
반말:
  この部屋は静かだ。       이 방은 조용하다.
  この部屋は静かじゃない。 이 방은 조용하지 않다.

존댓말:
  この部屋は静かです。       이 방은 조용합니다.
  この部屋は静かじゃないです。 이 방은 조용하지 않습니다.
```

### 5.3 흔한 실수

```
✗ この映画は面白いだ。     틀림 (い형용사 + だ)
✓ この映画は面白い。       맞음

✗ この部屋は静か。         불완전 (な형용사에는 だ/です가 필요)
✓ この部屋は静かだ。       맞음
```

## 7. 여러 형용사 결합하기

### 7.1 い형용사 연결하기

마지막 い를 떼고 くて를 붙입니다:

```
大きい + おいしい → 大きくて おいしい
ōkii + oishii → ōkikute oishii
크고 맛있는

このりんごは大きくて、おいしい。
Kono ringo wa ōkikute, oishii.
이 사과는 크고 맛있다.

あの映画は長くて、つまらなかった。
Ano eiga wa nagakute, tsumaranakatta.
그 영화는 길고 지루했다.
```

### 7.2 な형용사 연결하기

형용사 어간 뒤에 で를 붙입니다:

```
静か + きれい → 静かで きれい
shizuka + kirei → shizuka de kirei
조용하고 예쁜

この部屋は静かで、きれいです。
Kono heya wa shizuka de, kirei desu.
이 방은 조용하고 깨끗합니다.

田中さんは元気で、親切です。
Tanaka-san wa genki de, shinsetsu desu.
다나카 씨는 활기차고 친절합니다.
```

### 7.3 유형 혼합

```
い형용사 (くて) + な형용사:
  この町は小さくて、静かです。
  Kono machi wa chiisakute, shizuka desu.
  이 마을은 작고 조용합니다.

な형용사 (で) + い형용사:
  この部屋はきれいで、広いです。
  Kono heya wa kirei de, hiroi desu.
  이 방은 깨끗하고 넓습니다.
```

### 7.4 연결형 요약

> **형용사 연결형**
>
> - **い형용사**: い를 떼고 → くて를 붙임
> 大きい → 大きくて
> おいしい → おいしくて
>
> - **な형용사**: 어간 + で
> 静か → 静かで
> きれい → きれいで
>
> 예외:
> いい (좋은) → よくて (いくて가 아님)

## 8. 정도와 강도

### 8.1 자주 쓰이는 정도 부사

| 일본어 | 로마자 | 의미 | 강도 |
|----------|--------|---------|-----------|
| とても | totemo | 매우 | ★★★★ |
| すごく | sugoku | 엄청나게 | ★★★★★ |
| かなり | kanari | 상당히 | ★★★ |
| 少し | sukoshi | 조금 | ★★ |
| ちょっと | chotto | 약간 | ★★ |
| あまり (+부정) | amari | 별로 | ★ |
| 全然 (+부정) | zenzen | 전혀 | ☆ |

### 8.2 사용 예문

```
この料理はとてもおいしいです。
Kono ryōri wa totemo oishii desu.
이 음식은 매우 맛있습니다.

この映画はあまり面白くないです。
Kono eiga wa amari omoshirokunai desu.
이 영화는 별로 재미없습니다.

日本語は全然難しくないです。(겸손/격려)
Nihongo wa zenzen muzukashikunai desu.
일본어는 전혀 어렵지 않습니다.

この部屋はちょっと暗いです。
Kono heya wa chotto kurai desu.
이 방은 약간 어둡습니다.
```

> **참고**: あまり와 全然은 반드시 **부정** 동사/형용사와 함께 사용해야 합니다.

## 9. 특수한 경우와 예외

### 9.1 불규칙 형용사 いい (좋은)

いい (ii — 좋은)는 가장 중요한 불규칙 형용사입니다. 활용할 때 옛 형태인 よい (yoi)를 어간으로 사용합니다:

| 형태 | 활용 | 틀린 예 |
|------|------------|-----|
| 현재 (+) | いい | — |
| 현재 (-) | よくない | ~~いくない~~ |
| 과거 (+) | よかった | ~~いかった~~ |
| 과거 (-) | よくなかった | ~~いくなかった~~ |
| 연결형 | よくて | ~~いくて~~ |
| 명사 앞 | いい + 명사 | — |

```
天気がいい。         날씨가 좋다.
天気がよくない。     날씨가 좋지 않다.
天気がよかった。     날씨가 좋았다.
天気がよくなかった。 날씨가 좋지 않았다.
```

### 9.2 かっこいい (멋진, 잘생긴)

이것은 복합어입니다: かっこ + いい. いい의 불규칙 패턴을 따릅니다:

```
かっこいい   →  かっこよくない  →  かっこよかった
멋진             멋지지 않은         멋졌던
```

### 9.3 두 가지 유형 모두 될 수 있는 형용사

일부 형용사는 い형과 な형 모두 존재하며, 의미나 격식의 차이가 약간 있습니다:

| い형 | な형 | 차이점 |
|---------|---------|-----------|
| 大きい (ōkii) | 大きな (ōki na) | 大きな는 명사 앞에서만 사용, 약간 부드러운 느낌 |
| 小さい (chiisai) | 小さな (chiisa na) | 小さな는 명사 앞에서만 사용, 약간 부드러운 느낌 |

```
大きい犬 = 大きな犬     큰 개 (둘 다 맞음)
この犬は大きい。         이 개는 크다. (い형만 가능)
✗ この犬は大きなだ。     틀림
```

## 10. 연습 문제

### 연습 1: 유형 판별

각 형용사를 い형용사 또는 な형용사로 분류하세요.

```
1. 楽しい (tanoshii — 즐거운)
2. 元気 (genki — 활기찬)
3. 新しい (atarashii — 새로운)
4. きれい (kirei — 예쁜)
5. 難しい (muzukashii — 어려운)
6. 静か (shizuka — 조용한)
7. 嫌い (kirai — 싫어하는)
8. 暑い (atsui — 더운)
```

<details>
<summary>정답 보기</summary>

```
1. 楽しい → い형용사
2. 元気   → な형용사
3. 新しい → い형용사
4. きれい → な형용사 (함정! い형용사처럼 보임)
5. 難しい → い형용사
6. 静か   → な형용사
7. 嫌い   → な형용사 (함정! い형용사처럼 보임)
8. 暑い   → い형용사
```

</details>

### 연습 2: 형용사 활용

각 형용사를 반말로 네 가지 형태(현재 +/-, 과거 +/-)로 활용하세요.

```
1. 面白い (omoshiroi — 재미있는)
2. 好き (suki — 좋아하는)
3. いい (ii — 좋은)
```

<details>
<summary>정답 보기</summary>

```
1. 面白い (い형용사)
   현재 (+): 面白い
   현재 (-): 面白くない
   과거 (+): 面白かった
   과거 (-): 面白くなかった

2. 好き (な형용사)
   현재 (+): 好きだ
   현재 (-): 好きじゃない
   과거 (+): 好きだった
   과거 (-): 好きじゃなかった

3. いい (불규칙 い형용사)
   현재 (+): いい
   현재 (-): よくない
   과거 (+): よかった
   과거 (-): よくなかった
```

</details>

### 연습 3: 빈칸 채우기

올바른 형용사 형태를 선택하세요.

```
1. この花は __________ です。(きれい — 예쁜, 존댓말 현재 +)
2. __________ 部屋が好きです。(静か — 조용한, 명사 앞)
3. 昨日は __________ です。(暑い — 더운, 존댓말 과거 +)
4. あの映画は __________ です。(面白い — 재미있는, 존댓말 현재 -)
5. __________ 本を読みました。(新しい — 새로운, 명사 앞)
6. この問題は __________ です。(簡単 — 쉬운, 존댓말 현재 -)
```

<details>
<summary>정답 보기</summary>

```
1. この花は きれい です。
2. 静かな 部屋が好きです。
3. 昨日は 暑かった です。
4. あの映画は 面白くない です。
5. 新しい 本を読みました。
6. この問題は 簡単じゃない です。
```

</details>

### 연습 4: 형용사 연결

두 형용사를 하나의 문장으로 결합하세요.

```
1. この町は小さい + この町は静かだ
2. この料理はおいしい + この料理は安い
3. 田中さんは親切だ + 田中さんは元気だ
4. あの映画は長い + あの映画はつまらなかった
5. この部屋はきれいだ + この部屋は広い
```

<details>
<summary>정답 보기</summary>

```
1. この町は小さくて、静かです。
   (Kono machi wa chiisakute, shizuka desu.)

2. この料理はおいしくて、安いです。
   (Kono ryōri wa oishikute, yasui desu.)

3. 田中さんは親切で、元気です。
   (Tanaka-san wa shinsetsu de, genki desu.)

4. あの映画は長くて、つまらなかったです。
   (Ano eiga wa nagakute, tsumaranakatta desu.)

5. この部屋はきれいで、広いです。
   (Kono heya wa kirei de, hiroi desu.)
```

</details>

### 연습 5: 일본어로 번역

다음 문장을 존댓말(です) 형태로 번역하세요.

```
1. This room is very quiet.
2. That book was not interesting.
3. Tanaka-san is energetic and kind.
4. The weather was not good yesterday.
5. I like pretty flowers.
6. This town is small but convenient.
```

<details>
<summary>정답 보기</summary>

```
1. この部屋はとても静かです。
   (Kono heya wa totemo shizuka desu.)

2. あの本は面白くなかったです。
   (Ano hon wa omoshirokunakatta desu.)

3. 田中さんは元気で、親切です。
   (Tanaka-san wa genki de, shinsetsu desu.)

4. 昨日は天気がよくなかったです。
   (Kinō wa tenki ga yokunakatta desu.)

5. きれいな花が好きです。
   (Kirei na hana ga suki desu.)

6. この町は小さいですが、便利です。
   (Kono machi wa chiisai desu ga, benri desu.)
```

</details>

## 빠른 참조

> **형용사 요약표 (形容詞まとめ)**
> ├──────────────────────────────────────────────────────────────┤
>
> い형용사
> - **현재 (+)**: 〜い          高い
> - **현재 (-)**: 〜くない      高くない
> - **과거 (+)**: 〜かった      高かった
> - **과거 (-)**: 〜くなかった  高くなかった
> - **명사 앞**: 〜い + 명사   高い山
> - **연결형**: 〜くて        高くて
>
> な형용사
> - **현재 (+)**: 〜だ/です       静かだ
> - **현재 (-)**: 〜じゃない      静かじゃない
> - **과거 (+)**: 〜だった/でした 静かだった
> - **과거 (-)**: 〜じゃなかった  静かじゃなかった
> - **명사 앞**: 〜な + 명사     静かな部屋
> - **연결형**: 〜で            静かで
>
> 예외
> いい → よくない / よかった / よくなかった
> きれい, 嫌い → な형용사 (い로 끝나지만)
> 大きな, 小さな → 연체 수식 전용 변형
>
> 정도 부사
> とても (매우) / あまり〜ない (별로) / 全然〜ない (전혀
> 〜않다) / ちょっと (약간) / すごく (엄청나게)
>
