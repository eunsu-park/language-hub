## 1. 개요: 일본어 경어 체계

일본어에는 敬語(케이고)라 불리는 체계적인 경어 시스템이 있으며, 세 가지 주요 범주로 나뉩니다:

```
+-------------------------------------------------------------------+
|                    敬語 (KEIGO) 체계                                |
+-------------------------------------------------------------------+
|                                                                   |
|  丁寧語 (테이네이고)  — 정중어                                     |
|    목적: 듣는 사람에 대한 일반적 예의                               |
|    도구: です/ます 형태                                            |
|    사용 시기: 모르는 사람, 지인, 격식적 상황에서 기본              |
|                                                                   |
|  尊敬語 (손케이고)  — 존경어                                       |
|    목적: 타인(윗사람, 고객)의 행위를 높임                          |
|    사용 시기: 높은 지위의 사람에 대해 말할 때                      |
|                                                                   |
|  謙譲語 (겐조고)  — 겸양어                                         |
|    목적: 자신의 행위를 낮춤 (타인에 대한 존경 표현)                |
|    사용 시기: 윗사람에게 자신의 행동에 대해 말할 때                |
|                                                                   |
+-------------------------------------------------------------------+
```

### 핵심 원리

```
          ↑ 타인의 행위를 높임 (尊敬語)
          |
  중립    | ← 표준 정중체 (丁寧語 — です/ます)
          |
          ↓ 자신의 행위를 낮춤 (謙譲語)
```

> **핵심 포인트**: 尊敬語와 謙譲語는 화자와 대상 간의 **상대적 위치**에 관한 것입니다. 타인을 위로 올리고 자신을 아래로 내려 존경의 차이를 만듭니다.

## 2. 丁寧語 — 정중어 (です/ます)

丁寧語는 가장 기본적인 경어 수준입니다. 화제와 관계없이 **듣는 사람**에 대한 예의를 표합니다.

### 기본 형태

| 범주 | 보통체 | 정중체 (丁寧語) |
|----------|-------|----------------|
| 동사 (현재) | 食べる | 食べ**ます** |
| 동사 (과거) | 食べた | 食べ**ました** |
| 동사 (부정) | 食べない | 食べ**ません** |
| 동사 (과거 부정) | 食べなかった | 食べ**ませんでした** |
| い형용사 | 高い | 高い**です** |
| な형용사 | 静かだ | 静か**です** |
| 명사 | 学生だ | 学生**です** |
| 명사 (부정) | 学生じゃない | 学生**ではありません** |

### 丁寧語를 사용하는 경우

- 잘 모르는 사람과 대화할 때
- 격식적 상황(학교, 직장, 공공장소)
- 적절한 수준을 잘 모를 때
- 고객 서비스 상황(기본으로)

> **원칙**: 어떤 수준을 써야 할지 모르면 です/ます를 사용하세요. 일본어에서 너무 정중한 것은 절대 실례가 되지 않습니다.

### 예문

1. **明日は東京に行きます。** — 내일 도쿄에 갑니다.
2. **この本は面白いです。** — 이 책은 재미있습니다.
3. **私は田中と申します。** — 저는 다나카라고 합니다. (申す로 더 정중)
4. **すみません、分かりません。** — 죄송합니다, 모르겠습니다.
5. **どうぞよろしくお願いします。** — 잘 부탁드립니다.

## 3. 尊敬語 — 존경어

尊敬語는 **타인**의 행위를 높입니다 — 일반적으로 윗사람, 고객, 또는 존경받는 분들입니다. 자신의 행동에 尊敬語를 사용해서는 **절대 안 됩니다**.

### 활용 방법 1: お + ます형 어간 + になる

일반 동사를 존경어로 만드는 가장 생산적인 패턴입니다:

| 동사 | ます형 어간 | 존경어 | 의미 |
|------|-----------|-----------|---------|
| 読む | 読み | お読みになる | 읽으시다 |
| 使う | 使い | お使いになる | 사용하시다 |
| 書く | 書き | お書きになる | 쓰시다 |
| 帰る | 帰り | お帰りになる | 돌아가시다 |
| 待つ | 待ち | お待ちになる | 기다리시다 |

> **패턴**: お + ます형 어간 + になる → お + ます형 어간 + になります (정중체)

### 활용 방법 2: 수동형을 존경어로 사용

수동형(〜れる/〜られる)도 가벼운 존경어로 기능할 수 있습니다:

| 동사 | 수동/존경형 | 의미 |
|------|------------------|---------|
| 行く | 行かれる | 가시다 |
| 読む | 読まれる | 읽으시다 |
| 帰る | 帰られる | 돌아가시다 |
| 来る | 来られる | 오시다 |

> お〜になる보다 격식성이 떨어지지만 여전히 존경의 표현입니다.

### 활용 방법 3: 특수 존경 동사

일부 자주 쓰이는 동사는 완전히 다른 존경어 형태가 있으며 암기해야 합니다:

| 보통 동사 | 존경 동사 | 의미 |
|-----------|---------------|---------|
| いる (있다) | **いらっしゃる** | 계시다 |
| 行く (가다) | **いらっしゃる** | 가시다 |
| 来る (오다) | **いらっしゃる** | 오시다 |
| 食べる (먹다) | **召し上がる** (めしあがる) | 드시다 |
| 飲む (마시다) | **召し上がる** | 드시다 |
| 言う (말하다) | **おっしゃる** | 말씀하시다 |
| 見る (보다) | **ご覧になる** (ごらんになる) | 보시다 |
| 知っている (알다) | **ご存じだ** (ごぞんじだ) | 알고 계시다 |
| する (하다) | **なさる** | 하시다 |
| くれる (주다) | **くださる** | 주시다 |
| 着る (입다) | **お召しになる** (おめしになる) | 입으시다 |

> **いらっしゃる에 관한 참고**: 이 동사 하나가 いる, 行く, 来る를 모두 포괄합니다. 문맥으로 어떤 의미인지 구분합니다.

### 특수 존경 동사의 활용

대부분의 특수 존경 동사는 -aru 어미를 가진 1그룹(Godan) 동사로 활용됩니다:

| 사전형 | ます형 | 부정형 | て형 |
|-----------|-----------|----------|---------|
| いらっしゃる | いらっしゃいます | いらっしゃらない | いらっしゃって |
| おっしゃる | おっしゃいます | おっしゃらない | おっしゃって |
| くださる | くださいます | くださらない | くださって |
| なさる | なさいます | なさらない | なさって |

### 예문

1. **社長はもうお帰りになりました。** — 사장님은 이미 돌아가셨습니다.
2. **先生がおっしゃったことは正しい。** — 선생님이 말씀하신 것은 맞다.
3. **お客様がいらっしゃいました。** — 손님이 오셨습니다.
4. **どうぞ召し上がってください。** — 어서 드세요.
5. **この資料をご覧になりましたか。** — 이 자료를 보셨습니까?

## 4. 謙譲語 — 겸양어

謙譲語는 말하는 사람 자신의 행위를 낮추어 상대방 또는 언급되는 사람에 대한 존경을 표합니다. **자신의** 행위에 대해 윗사람에게 말할 때 사용합니다.

### 활용 방법 1: お + ます형 어간 + する

| 동사 | ます형 어간 | 겸양어 | 의미 |
|------|-----------|--------|---------|
| 持つ | 持ち | お持ちする | 들겠습니다 |
| 送る | 送り | お送りする | 보내겠습니다 |
| 待つ | 待ち | お待ちする | 기다리겠습니다 |
| 伝える | 伝え | お伝えする | 전하겠습니다 |
| 知らせる | 知らせ | お知らせする | 알려드리겠습니다 |

> **패턴**: お + ます형 어간 + する → お + ます형 어간 + します (정중체)
>
> 한자어(漢語)를 사용하는 する동사에는 お 대신 ご를 사용합니다:
> ご連絡する, ご説明する, ご案内する

### 활용 방법 2: 특수 겸양 동사

| 보통 동사 | 겸양 동사 | 의미 |
|-----------|------------|---------|
| いる (있다) | **おる** | 있습니다 (겸양) |
| 行く (가다) | **参る** (まいる) | 가겠습니다 (겸양) |
| 来る (오다) | **参る** | 오겠습니다 (겸양) |
| 食べる (먹다) | **いただく** | 먹겠습니다 (겸양) |
| 飲む (마시다) | **いただく** | 마시겠습니다 (겸양) |
| もらう (받다) | **いただく** | 받겠습니다 (겸양) |
| 言う (말하다) | **申す** (もうす) | 말씀드리다 (겸양) |
| 見る (보다) | **拝見する** (はいけんする) | 보겠습니다 (겸양) |
| 知っている (알다) | **存じている** (ぞんじている) | 알고 있습니다 (겸양) |
| する (하다) | **いたす** | 하겠습니다 (겸양) |
| あげる (주다) | **差し上げる** (さしあげる) | 드리다 (겸양) |
| 聞く (묻다/듣다) | **伺う** (うかがう) | 여쭙다 (겸양) |
| 訪ねる (방문하다) | **伺う** | 찾아뵙다 (겸양) |
| 会う (만나다) | **お目にかかる** | 뵙다 (겸양) |

> **参る에 관한 참고**: いらっしゃる처럼 参る도 行く와 来る를 모두 포괄합니다. 문맥으로 구분합니다.

### いただく — 가장 유용한 겸양 동사

いただく는 많은 필수 패턴에 등장합니다:

| 패턴 | 의미 | 예시 |
|---------|---------|---------|
| 〜ていただく | ~해 주시다 (겸양) | 教えていただけますか。(가르쳐 주시겠습니까?) |
| 〜ていただけませんか | ~해 주실 수 있겠습니까? (매우 정중한 요청) | 説明していただけませんか。 |
| 〜させていただく | ~하게 해주시다 (겸양) | 自己紹介させていただきます。 |
| いただきます | 식사 전 인사 (관용구) | いただきます！ |

### 예문

1. **田中と申します。** — 다나카라고 합니다.
2. **明日そちらに参ります。** — 내일 그쪽으로 가겠습니다.
3. **お荷物をお持ちします。** — 짐을 들겠습니다.
4. **資料を拝見しました。** — 자료를 보았습니다.
5. **少々お待ちいただけますか。** — 잠시 기다려 주시겠습니까?

## 5. 특수 존경어·겸양어 동사 쌍

### 마스터 참조 표

| 영어 | 보통체 | 尊敬語 (타인에 대해) | 謙譲語 (자신에 대해) |
|---------|-------|---------------------|-------------------|
| 있다 | いる | いらっしゃる | おる |
| 가다 | 行く | いらっしゃる | 参る |
| 오다 | 来る | いらっしゃる | 参る |
| 먹다 | 食べる | 召し上がる | いただく |
| 마시다 | 飲む | 召し上がる | いただく |
| 말하다 | 言う | おっしゃる | 申す |
| 보다 | 見る | ご覧になる | 拝見する |
| 알다 | 知っている | ご存じだ | 存じている |
| 하다 | する | なさる | いたす |
| 주다 (나에게) | くれる | くださる | — |
| 주다 (타인에게) | あげる | — | 差し上げる |
| 받다 | もらう | — | いただく |
| 묻다/방문하다 | 聞く/訪ねる | — | 伺う |
| 만나다 | 会う | — | お目にかかる |
| 입다 | 着る | お召しになる | — |

### 암기 팁

```
+------------------------------------------------------------------+
|  존경어 (尊敬語): 타인의 행위를 높임                               |
|  → "손님께서 드시다" = 召し上がる                                  |
|                                                                  |
|  겸양어 (謙譲語): 자신의 행위를 낮춤                               |
|  → "제가 먹겠습니다" = いただく                                   |
|                                                                  |
|  자신에게 존경어를 절대 사용하지 마세요:                            |
|  ✗ 私がいらっしゃいます (잘못 — 자신에게 존경어)                   |
|  ○ 私が参ります (올바름 — 자신에게 겸양어)                        |
|                                                                  |
|  윗사람에게 겸양어를 절대 사용하지 마세요:                          |
|  ✗ 先生が参りました (잘못 — 선생님에게 겸양어)                    |
|  ○ 先生がいらっしゃいました (올바름 — 선생님에게 존경어)           |
+------------------------------------------------------------------+
```

## 6. 경어 접두사: お와 ご

일본어에서는 접두사 お(o)와 ご(go)를 붙여 단어를 더 정중하게 만듭니다.

### 일반 규칙

| 접두사 | 사용 대상 | 예시 |
|--------|-----------|---------|
| お (o) | 일본 고유어 (和語) | お名前, お水, お茶, お金, お天気 |
| ご (go) | 한자어 (漢語) | ご住所, ご連絡, ご家族, ご意見, ご質問 |

### 자주 쓰이는 お 단어

| 단어 | 정중형 | 의미 |
|------|-------------|---------|
| 名前 | お名前 | 이름 |
| 茶 | お茶 | 차 |
| 水 | お水 | 물 |
| 金 | お金 | 돈 |
| 元気 | お元気 | 건강/안녕 |
| 仕事 | お仕事 | 일 |
| 忙しい | お忙しい | 바쁘시다 |

### 자주 쓰이는 ご 단어

| 단어 | 정중형 | 의미 |
|------|-------------|---------|
| 住所 | ご住所 | 주소 |
| 家族 | ご家族 | 가족 |
| 連絡 | ご連絡 | 연락 |
| 意見 | ご意見 | 의견 |
| 質問 | ご質問 | 질문 |
| 説明 | ご説明 | 설명 |
| 案内 | ご案内 | 안내 |

### 예외

일부 한자어는 お를 사용합니다: お電話, お食事, お時間, お返事

> **팁**: 어떤 것을 써야 할지 모를 때는 お가 더 안전합니다 — ご보다 더 널리 사용됩니다.

## 7. 비즈니스 일본어 화법

### 자주 쓰이는 비즈니스 표현

| 상황 | 표현 | 의미 |
|-----------|-----------|---------|
| 전화 받기 | お電話ありがとうございます。〇〇会社の△△です。 | 전화 감사합니다. ○○회사의 △△입니다. |
| 기다려 달라고 하기 | 少々お待ちください。 | 잠시만 기다려 주십시오. |
| 전화 연결 | 担当者におつなぎします。 | 담당자에게 연결해 드리겠습니다. |
| 사과 | 大変申し訳ございません。 | 대단히 죄송합니다. (가장 격식적) |
| 감사 | 誠にありがとうございます。 | 진심으로 감사드립니다. (가장 격식적) |
| 거절 | 申し訳ございませんが、ちょっと... | 죄송합니다만, 좀... (부드러운 거절) |
| 회의 마무리 | お忙しいところありがとうございました。 | 바쁘신 중에 감사했습니다. |

### 이메일 패턴

| 부분 | 패턴 | 예시 |
|------|---------|---------|
| 시작 | お世話になっております。 | 항상 신세를 지고 있습니다. |
| 요청 | 〜していただけますでしょうか。 | ~해 주실 수 있겠습니까? |
| 사과 | ご迷惑をおかけして申し訳ございません。 | 폐를 끼쳐 죄송합니다. |
| 마무리 | よろしくお願いいたします。 | 잘 부탁드립니다. |

### 정중 접미사: でございます

でございます는 です의 더 격식적인 표현으로 매우 정중한 맥락에서 사용됩니다:

| です 수준 | でございます 수준 | 의미 |
|-----------|-----------------|---------|
| 田中です。 | 田中でございます。 | 다나카입니다. |
| こちらです。 | こちらでございます。 | 이쪽입니다. |
| そうです。 | さようでございます。 | 그렇습니다. |

## 8. 비격식 화법과 보통체 사용

### 비격식/보통체를 사용하는 경우

- 같은 나이의 가까운 친구와
- 가족(특히 아랫사람)과
- 혼잣말이나 속으로 생각할 때
- 일기나 가벼운 글에서
- SNS나 문자 메시지에서

### 비격식 화법의 특징

| 특징 | 격식체 | 비격식체 | 비고 |
|---------|--------|--------|-------|
| 문말 | です/ます | 보통체 | 行きます → 行く |
| 의문문 | ですか? | ? 또는 억양 상승 | 行きますか → 行く？ |
| のです | のです/んです | んだ/の | 何をしているんですか → 何してるの？ |
| ている | ています | てる | 食べています → 食べてる |
| ではない | ではありません | じゃない | 学生ではありません → 学生じゃない |
| けれども | けれども | けど | でも / だけど |

### 성별에 따른 화법 패턴 (전통적)

| 특징 | 남성적 표현 | 여성적 표현 |
|---------|-------------|---------------|
| 문말 종조사 | だ, ぞ, ぜ | わ, の, かしら |
| 1인칭 | 俺 (おれ), 僕 (ぼく) | 私 (わたし), あたし |
| 단정적 | 行くぞ！ | 行くわよ！ |
| 의문 | 行くのか？ | 行くの？ |

> **현대 참고**: 이러한 구분은 특히 젊은 세대에서 점점 덜 엄격해지고 있습니다. 많은 사람들이 성별에 관계없이 전통적으로 성별 구분이 있던 종조사를 혼용합니다.

### 정중도 스펙트럼 예시: "초밥 드시나요?"

```
가장 격식적:  お寿司を召し上がりますか。         (尊敬語)
격식적:       お寿司を食べますか。               (丁寧語)
중립:         寿司、食べる？                     (보통체)
매우 비격식:  寿司食う？                         (거친 반말)
```

## 9. 흔한 실수와 주의점

### 실수 1: 자신에게 존경어 사용

```
✗ 私がいらっしゃいます。 (자신에게 尊敬語 사용)
○ 私が参ります。         (자신에게 謙譲語 사용)

✗ 私が召し上がりました。 (자신의 식사에 존경어)
○ 私がいただきました。   (자신의 식사에 겸양어)
```

### 실수 2: 윗사람에게 겸양어 사용

```
✗ 先生が申しました。   (선생님에게 謙譲語 사용)
○ 先生がおっしゃいました。 (선생님에게 尊敬語 사용)
```

### 실수 3: 이중 경어 (二重敬語)

```
✗ お召し上がりになる (이중 경어 — お〜になる와 召し上がる 모두)
○ 召し上がる          (경어 하나로 충분)

✗ おっしゃられる      (おっしゃる + られる = 이중)
○ おっしゃる          (올바름)
```

### 실수 4: 자신의 회사/가족에게 경어 사용

외부인에게 말할 때는 자기 그룹에 대해 겸양어를 사용합니다:

```
✗ 社長はいらっしゃいません。 (외부인에게 자사 사장을 존경어로)
○ 社長はおりません。         (겸양어 — 사장은 자기 그룹)

✗ 母がおっしゃいました。     (외부인에게 자기 어머니를 존경어로)
○ 母が申しました。           (겸양어 — 어머니는 자기 그룹)
```

> **우치/소토(내부/외부) 규칙**: 외부인과 대화할 때, 자기 그룹 구성원(가족, 회사)에 대해서는 겸양어를, 상대 그룹에 대해서는 존경어를 사용합니다.

### 실수 5: 한 문장에서 화법 혼용

```
✗ 先生、これ見て。             (선생님에게 너무 비격식적)
○ 先生、こちらをご覧ください。 (적절히 존경)

✗ 友達、こちらにいらっしゃってください。 (친구에게 너무 격식적)
○ こっちに来て。                       (친구에게 자연스러움)
```

## 10. 연습 문제

### 연습 1: 존경어(尊敬語)로 변환

적절한 존경어 형태로 다시 쓰세요.

1. 先生が来た。→ 先生が ________。
2. 社長はこの本を読みましたか。→ 社長はこの本を ________。
3. お客様が言いました。→ お客様が ________。
4. 部長はもう食べましたか。→ 部長はもう ________。
5. 先生はこの映画を見ましたか。→ 先生はこの映画を ________。

<details>
<summary>정답</summary>

1. 先生が**いらっしゃいました / 来られました**。
2. 社長はこの本を**お読みになりましたか / 読まれましたか**。
3. お客様が**おっしゃいました**。
4. 部長はもう**召し上がりましたか**。
5. 先生はこの映画を**ご覧になりましたか**。

</details>

### 연습 2: 겸양어(謙譲語)로 변환

자신의 행동에 겸양어를 사용하여 다시 쓰세요.

1. 私が行きます。→ 私が ________。
2. 私はこの資料を見ました。→ 私はこの資料を ________。
3. 私は田中と言います。→ 私は田中と ________。
4. 明日そちらに来ます。→ 明日そちらに ________。
5. この件について聞きたいです。→ この件について ________。

<details>
<summary>정답</summary>

1. 私が**参ります**。
2. 私はこの資料を**拝見しました**。
3. 私は田中と**申します**。
4. 明日そちらに**参ります**。
5. この件について**伺いたいです / お伺いしたいです**。

</details>

### 연습 3: 존경어인가, 겸양어인가?

각 맥락에 맞는 올바른 형태를 선택하세요.

1. (거래처에 자사 사장에 대해) 社長は今 ________ (いない)。
   a. いらっしゃいません b. おりません

2. (고객에 대해) お客様が ________ (来た)。
   a. 参りました b. いらっしゃいました

3. (상사에게 자신에 대해) 私が ________ (する)。
   a. なさいます b. いたします

4. (선생님에 대해) 先生が ________ (言った)。
   a. 申しました b. おっしゃいました

5. (자신이 무언가를 받으며) ________ (もらう)。
   a. いただきます b. くださいます

<details>
<summary>정답</summary>

1. **b. おりません** (외부인에게 말할 때 자사 사장은 내부 그룹 — 겸양어 사용)
2. **b. いらっしゃいました** (고객 = 외부 그룹 — 존경어 사용)
3. **b. いたします** (자신에 대해 — 겸양어 사용)
4. **b. おっしゃいました** (선생님 = 윗사람 — 존경어 사용)
5. **a. いただきます** (자신이 받으므로 — 겸양어 사용)

</details>

### 연습 4: 오류 수정

경어 오류를 찾아 수정하세요.

1. ✗ 私はコーヒーを召し上がります。
2. ✗ 先生が参りました。
3. ✗ お客様、こちらでお待ちしてください。
4. ✗ 社長がおっしゃられました。
5. ✗ (거래처에 자사 부장에 대해) 部長がいらっしゃいます。

<details>
<summary>정답</summary>

1. ○ 私はコーヒーを**いただきます**。(召し上がる는 존경어 — 자신에게 사용 불가)
2. ○ 先生が**いらっしゃいました**。(参る는 겸양어 — 선생님에게 사용 불가)
3. ○ お客様、こちらで**お待ちになってください / お待ちください**。(お待ちする는 겸양어 — 고객에게 사용 부적절)
4. ○ 社長が**おっしゃいました**。(おっしゃられる는 이중 경어)
5. ○ 部長が**おります**。(외부 거래처에 말할 때 자사 부장에게는 겸양어 사용)

</details>

### 연습 5: 화법 선택

각 상황에 가장 적절한 수준을 선택하세요.

1. 카페에서 커피 주문 (고객으로서):
   a. コーヒーをいただけますか。
   b. コーヒーください。
   c. コーヒーくれ。

2. 교수에게 질문:
   a. 先生、ちょっと聞きたいんだけど。
   b. 先生、少しお伺いしたいのですが。
   c. 先生、聞きたいことがある。

3. 친한 친구에게 기다려 달라고 하기:
   a. 少々お待ちいただけますか。
   b. ちょっと待ってて。
   c. お待ちください。

<details>
<summary>정답</summary>

1. **b. コーヒーください。** (고객으로서 자연스러움 — 정중하지만 지나치게 격식적이지 않음. a도 허용 가능하지만 고객 입장에서 매우 겸양적.)
2. **b. 先生、少しお伺いしたいのですが。** (교수에게 말하기에 적절한 겸양어.)
3. **b. ちょっと待ってて。** (친한 친구 사이에 자연스러운 반말. a와 c는 너무 격식적.)

</details>

### 연습 6: 적절한 경어를 사용하여 번역

올바른 정중도 수준으로 일본어로 번역하세요.

1. (상사에게) 이메일을 보내겠습니다. (겸양)
2. (고객에 대해) 고객님이 기다리고 계십니다. (존경)
3. (레스토랑에서, 직원에게) 메뉴를 보고 싶습니다. (정중)
4. (동료에게 사장에 대해) 사장님이 9시에 오라고 말씀하셨습니다. (사장에게 존경어)

<details>
<summary>정답</summary>

1. メールを**お送りいたします / お送りします**。(겸양: お + ます형 어간 + する/いたす)
2. お客様が**お待ちになっています / お待ちです**。(존경: お + ます형 어간 + になる)
3. メニューを**見せていただけますか / 拝見できますか**。(겸양 요청)
4. 社長が9時に来る**ようにおっしゃいました**。(사장의 말씀에 おっしゃる 사용)

</details>
