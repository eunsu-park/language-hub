## 1. Why Japanese Has Three Verbs for "Give"

English has one word for "give" and one for "receive." Japanese has **three verbs** that encode not only the action of giving and receiving, but also the **direction** of the action relative to the speaker and the **social relationship** between the people involved.

```
┌──────────────────────────────────────────────────────────────┐
│          THE JAPANESE GIVING/RECEIVING SYSTEM                 │
│                                                              │
│   English: "give" / "receive"  — 2 verbs, direction-neutral │
│                                                              │
│   Japanese: 3 verbs that encode PERSPECTIVE                  │
│                                                              │
│   あげる (ageru)  — I/someone gives TO someone else          │
│                     Direction: speaker → other               │
│                     (or: third party → third party)          │
│                                                              │
│   もらう (morau)  — I/someone receives FROM someone          │
│                     Direction: other → speaker               │
│                                                              │
│   くれる (kureru) — Someone gives TO me/my in-group          │
│                     Direction: other → speaker               │
│                     (emphasizes the GIVER's action)          │
│                                                              │
│   ※ This system reflects Japanese cultural awareness of      │
│     social positioning and gratitude.                        │
└──────────────────────────────────────────────────────────────┘
```

## 2. あげる — To Give (Speaker → Other)

### 2.1 Basic Pattern

**[Giver] は [Receiver] に [Object] を あげる**

あげる is used when:
- The speaker gives something to someone else
- A third party gives something to another third party
- The direction is **away from** the speaker

```
私は田中さんに本をあげた。
Watashi wa Tanaka-san ni hon o ageta.
I gave a book to Tanaka-san.

姉は友達にプレゼントをあげました。
Ane wa tomodachi ni purezento o agemashita.
My older sister gave a present to her friend.

山田さんは鈴木さんにチョコレートをあげた。
Yamada-san wa Suzuki-san ni chokorēto o ageta.
Yamada-san gave chocolate to Suzuki-san.
```

### 2.2 Important Rule: Never Use あげる When Receiving

```
✗ 田中さんは私に本をあげた。
  (This sounds like you're talking about yourself from a distance.
   Use くれた instead — see Section 4.)

✓ 田中さんは私に本をくれた。
  Tanaka-san gave me a book.
```

### 2.3 Giving to Animals and Plants

あげる is also used when giving to pets, animals, or plants (who cannot reciprocate).

```
犬にえさをあげた。
Inu ni esa o ageta.
I gave food to the dog.

花に水をあげる。
Hana ni mizu o ageru.
I water the flowers. (lit. give water to the flowers)
```

## 3. もらう — To Receive

### 3.1 Basic Pattern

**[Receiver] は [Giver] に/から [Object] を もらう**

もらう is used when someone receives something. The receiver is the subject.

```
私は田中さんに本をもらった。
Watashi wa Tanaka-san ni hon o moratta.
I received a book from Tanaka-san.

弟は母にお小遣いをもらいました。
Otōto wa haha ni okozukai o moraimashita.
My younger brother received allowance from our mother.

誕生日にたくさんプレゼントをもらった。
Tanjōbi ni takusan purezento o moratta.
I received lots of presents on my birthday.
```

### 3.2 に vs から with もらう

Both に and から can mark the giver with もらう, but there is a slight nuance:

```
田中さんに もらった。   I received from Tanaka-san. (standard)
田中さんから もらった。 I received from Tanaka-san. (emphasizes source)
```

> **Note**: から is preferred when the source is an organization or impersonal entity.
> - 学校からもらった。 I received it from the school.
> - 会社からもらった。 I received it from the company.

## 4. くれる — To Give (Other → Speaker)

### 4.1 Basic Pattern

**[Giver] は [Speaker/in-group] に [Object] を くれる**

くれる is used when someone gives something **to the speaker** or to someone in the speaker's **in-group** (family, close friends). It carries a nuance of **gratitude** — the speaker appreciates the giver's action.

```
田中さんは私にプレゼントをくれた。
Tanaka-san wa watashi ni purezento o kureta.
Tanaka-san gave me a present. (I'm grateful.)

先生が本をくれました。
Sensei ga hon o kuremashita.
The teacher gave me a book. (thankful tone)

友達がチケットをくれた。
Tomodachi ga chiketto o kureta.
My friend gave me a ticket.
```

### 4.2 In-group Recipients

くれる is also used when someone gives to a member of the speaker's in-group.

```
山田さんは弟にお菓子をくれた。
Yamada-san wa otōto ni okashi o kureta.
Yamada-san gave sweets to my younger brother.
(Younger brother is in my in-group, so I use くれる.)

先生は妹に辞書をくれました。
Sensei wa imōto ni jisho o kuremashita.
The teacher gave my younger sister a dictionary.
```

### 4.3 あげる vs くれる — The Key Difference

```
┌──────────────────────────────────────────────────────────────┐
│            あげる vs くれる                                    │
│                                                              │
│   Same event seen from different perspectives:               │
│                                                              │
│   私は田中さんに本をあげた。                                 │
│   I gave Tanaka-san a book. (I am the giver → あげる)       │
│                                                              │
│   田中さんは私に本をくれた。                                 │
│   Tanaka-san gave me a book. (I am the receiver → くれる)   │
│                                                              │
│   Think of it as:                                            │
│   あげる = giving AWAY from me  ←── me ──→ (outward)        │
│   くれる = giving TOWARD me     (inward) ──→ me             │
│                                                              │
│   ※ NEVER say: 田中さんは私に本をあげた。                   │
│     (This is grammatically odd — it should be くれた.)       │
└──────────────────────────────────────────────────────────────┘
```

## 5. The Direction System Visualized

### 5.1 The Triangle Model

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                    SPEAKER (私)                               │
│                   /          \                                │
│        くれる   /              \   あげる                     │
│        (to me)/                \  (from me)                  │
│              /                  \                             │
│            ↓                    ↓                             │
│        OTHER PERSON  ────────→  OTHER PERSON                 │
│                       あげる                                  │
│                   (third party to third party)               │
│                                                              │
│        もらう: always RECEIVER is the subject                │
│        (I receive / someone receives)                        │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 5.2 Same Situation, Three Ways to Say It

Imagine: Tanaka-san gives the speaker a book.

```
Perspective 1 — Focus on GIVER's action toward me:
  田中さんが(私に)本をくれた。
  Tanaka-san gave me a book. (くれる — giving toward speaker)

Perspective 2 — Focus on my RECEIVING:
  (私は)田中さんに本をもらった。
  I received a book from Tanaka-san. (もらう — I receive)

NOT possible:
  ✗ 田中さんが私に本をあげた。
  (あげる cannot be used when giving TO the speaker)
```

### 5.3 Decision Flowchart

```
┌──────────────────────────────────────────────────────────────┐
│         WHICH VERB SHOULD I USE?                              │
│                                                              │
│   Am I (or my in-group) the RECEIVER?                        │
│     ├── YES → Is the focus on the GIVER's kindness?         │
│     │         ├── YES → くれる                               │
│     │         └── NO  → もらう                               │
│     │                                                        │
│     └── NO → Am I (or my in-group) the GIVER?               │
│              ├── YES → あげる                                │
│              └── NO  → あげる (third party → third party)   │
│                                                              │
│   Simple version:                                            │
│   Giving outward  → あげる                                   │
│   Getting inward  → もらう / くれる                          │
└──────────────────────────────────────────────────────────────┘
```

## 6. て + あげる — Doing a Favor for Someone

### 6.1 Basic Pattern

**[Doer] は [Beneficiary] に Verb(て-form) + あげる**

This means "I do something for someone" as a favor or kindness.

```
友達に日本語を教えてあげた。
Tomodachi ni nihongo o oshiete ageta.
I taught Japanese to my friend (as a favor).

妹の宿題を手伝ってあげました。
Imōto no shukudai o tetsudatte agemashita.
I helped my younger sister with her homework.

道を教えてあげた。
Michi o oshiete ageta.
I told them the way (as a kindness).
```

### 6.2 Caution: Can Sound Condescending

Using てあげる can sound condescending or self-congratulatory if overused, especially toward superiors.

```
⚠ 先生に教えてあげた。
   I taught the teacher. (sounds arrogant — avoid)

✓ 友達に荷物を持ってあげた。
   I carried my friend's luggage for them. (natural, kind)

✓ 子供に本を読んであげた。
   I read a book to the child. (caring, natural)
```

## 7. て + もらう — Having Someone Do Something

### 7.1 Basic Pattern

**[Beneficiary] は [Doer] に Verb(て-form) + もらう**

This means "I have someone do something for me" or "I get someone to do something." The focus is on the receiver benefiting from the action.

```
友達に手伝ってもらった。
Tomodachi ni tetsudatte moratta.
I had my friend help me. / My friend helped me (and I'm grateful).

先生に説明してもらいました。
Sensei ni setsumei shite moraimashita.
I had the teacher explain it. / The teacher explained it for me.

田中さんに駅まで車で送ってもらった。
Tanaka-san ni eki made kuruma de okutte moratta.
I had Tanaka-san drive me to the station.
```

### 7.2 Making Requests with てもらえますか

A polite way to ask someone to do something:

```
手伝ってもらえますか。
Tetsudatte moraemasu ka.
Could you help me? (lit. Could I receive your helping?)

もう一度説明してもらえますか。
Mō ichido setsumei shite moraemasu ka.
Could you explain it one more time?

写真を撮ってもらえますか。
Shashin o totte moraemasu ka.
Could you take a photo (for me)?
```

## 8. て + くれる — Someone Does Something for Me

### 8.1 Basic Pattern

**[Doer] は/が [Speaker/in-group] に Verb(て-form) + くれる**

This means "someone does something for me" with a sense of gratitude.

```
友達が手伝ってくれた。
Tomodachi ga tetsudatte kureta.
My friend helped me. (I'm grateful.)

母が弁当を作ってくれました。
Haha ga bentō o tsukutte kuremashita.
My mother made me a bento. (thankful)

先生が分かりやすく説明してくれた。
Sensei ga wakariyasuku setsumei shite kureta.
The teacher explained it clearly for me.

田中さんが本を貸してくれた。
Tanaka-san ga hon o kashite kureta.
Tanaka-san lent me a book.
```

### 8.2 てくれる vs てもらう

Both express that someone did something beneficial for the speaker, but from different angles:

```
┌──────────────────────────────────────────────────────────────┐
│          てくれる vs てもらう                                  │
│                                                              │
│   友達が手伝ってくれた。                                     │
│   My friend helped me. (Focus on friend's kind ACTION)       │
│   Subject: 友達 (the doer)                                   │
│                                                              │
│   友達に手伝ってもらった。                                   │
│   I had my friend help me. (Focus on MY receiving benefit)   │
│   Subject: 私 (the beneficiary)                              │
│                                                              │
│   Both express gratitude. The difference is PERSPECTIVE:     │
│   てくれる → "They did it FOR me" (praising the doer)        │
│   てもらう → "I got them to do it" (focus on my benefit)     │
│                                                              │
│   In daily conversation, てくれる is slightly more common    │
│   when simply expressing gratitude.                          │
└──────────────────────────────────────────────────────────────┘
```

## 9. Social Context and Politeness

### 9.1 Humble and Honorific Equivalents

For more formal situations (covered in depth in Lesson 14), there are polite versions:

| Plain | Humble (自分 / own action) | Honorific (相手 / other's action) |
|-------|--------------------------|----------------------------------|
| あげる | さしあげる (sashiageru) | — |
| もらう | いただく (itadaku) | — |
| くれる | — | くださる (kudasaru) |

```
先生にいただきました。
Sensei ni itadakimashita.
I received it from the teacher. (humble もらう)

先生がくださいました。
Sensei ga kudasaimashita.
The teacher gave it to me. (honorific くれる)

部長にさしあげました。
Buchō ni sashiagemashita.
I gave it to the department head. (humble あげる)
```

> **For now**, just be aware these exist. Master あげる / もらう / くれる first.

### 9.2 Social Hierarchy and Verb Choice

```
┌──────────────────────────────────────────────────────────────┐
│         SOCIAL FLOW OF GIVING                                │
│                                                              │
│   Higher status → Lower status:  くださる / くれる           │
│   (teacher → student, boss → employee)                       │
│                                                              │
│   Lower status → Higher status:  さしあげる / あげる         │
│   (student → teacher, employee → boss)                       │
│                                                              │
│   Equal status: あげる / もらう / くれる (standard forms)    │
│                                                              │
│   ※ Using あげる toward a superior can sound presumptuous.   │
│     Use さしあげる in formal contexts.                       │
│   ※ Using くれる about a superior's action is fine casually, │
│     but use くださる in formal contexts.                     │
└──────────────────────────────────────────────────────────────┘
```

### 9.3 Common Mistakes by Learners

```
1. ✗ 先生は私にプレゼントをあげた。
   ✓ 先生は私にプレゼントをくれた。
   (When receiving, use くれる, not あげる.)

2. ✗ 私は田中さんに教えてくれた。
   ✓ 私は田中さんに教えてあげた。
   (When YOU do the favor, use あげる, not くれる.)

3. ✗ 田中さんに手伝ってあげてもらった。
   ✓ 田中さんに手伝ってもらった。
   (Don't combine あげる and もらう — pick one perspective.)
```

│          私は田中さんに本をもらった。                        │
│  くれる  Someone gives TO me/my in-group (inward)            │
│          田中さんが私に本をくれた。                          │
│                                                              │
│  TE-FORM EXTENSIONS (favors)                                 │
│  てあげる  I do a favor FOR someone                          │
│            教えてあげた (I taught them)                       │
│  てもらう  I have/get someone to do FOR me                   │
│            教えてもらった (I got them to teach me)            │
│  てくれる  Someone does a favor FOR me                       │
│            教えてくれた (They taught me — grateful)           │
│                                                              │
│  POLITE REQUEST: てもらえますか (Could you do... for me?)    │
│                                                              │
│  DIRECTION RULE                                              │
│  Speaker → Other:  あげる / てあげる                         │
│  Other → Speaker:  くれる / てくれる / もらう / てもらう     │
│                                                              │
│  COMMON MISTAKES                                             │
│  ✗ 田中さんが私にあげた → ✓ くれた (giving TO me)           │
│  ✗ 私が友達にてくれた → ✓ てあげた (I do for someone)       │
│  ✗ Animals にくれた → ✓ にあげた (giving to animals)        │
│                                                              │
│  FORMAL VERSIONS (Lesson 14)                                 │
│  あげる → さしあげる (humble)                                │
│  もらう → いただく (humble)                                  │
│  くれる → くださる (honorific)                               │
│                                                              │
└──────────────────────────────────────────────────────────────┘
