## 1. Overview of Japanese Writing Systems

### 1.1 Three Scripts, One Language

Japanese is unique among major world languages in using three distinct writing systems simultaneously. A single sentence may contain all three:

> **The Three Japanese Writing Systems**
>
> - **ひらがな Hiragana**: Native Japanese words, grammar
> - **カタカナ Katakana**: Foreign loanwords, emphasis, names
> - **漢字 Kanji**: Chinese-origin characters for meaning
>
> Example sentence using all three:
> 田中さんは コーヒー を 飲みました。
> (Tanaka-san wa koohii o nomimashita.)
> "Mr. Tanaka drank coffee."
>
> - **田中**: kanji (name)
> - **さんは...を...ました**: hiragana (grammar/particles)
> - **コーヒー**: katakana (foreign word: "coffee")
> - **飲み**: kanji (verb stem: "drink")

### 1.2 When to Use Each Script

| Script | Usage | Examples |
|--------|-------|---------|
| Hiragana (ひらがな) | Native words, particles, verb/adjective endings, children's text | たべる (taberu, to eat), は (wa, topic marker) |
| Katakana (カタカナ) | Foreign loanwords, onomatopoeia, emphasis, scientific names | コンピュータ (konpyuuta, computer), ドイツ (Doitsu, Germany) |
| Kanji (漢字) | Nouns, verb/adjective stems, names, meaning-dense content | 食べる (taberu, to eat), 日本 (Nihon, Japan) |
| Romaji (ローマ字) | Latin alphabet — used for signs, input methods, foreign learners | Tokyo, sushi, karate |

### 1.3 Learning Order

```
Step 1: Master hiragana (1-2 weeks)        ← You are here
Step 2: Learn katakana (1-2 weeks)
Step 3: Begin basic kanji (ongoing)
Step 4: Build kanji knowledge continuously
```

> **Important**: Hiragana is absolutely essential. You cannot progress in Japanese without it. Learn it first and learn it well.

## 2. Hiragana: The Foundation

### 2.1 What is Hiragana?

Hiragana is a **syllabary** — each character represents a syllable (a consonant + vowel combination), not a single sound like in English. There are 46 basic characters.

> **Hiragana Key Facts**
>
> - **Characters**: 46 basic + 25 voiced + 33 combo = 104 total
> - **Shape**: Rounded, flowing, cursive-like
> - **Origin**: Simplified from kanji by women in the Heian period
> - **Used for**: Grammar, native words, furigana (kanji readings)
>
> Each character = one syllable (mora)
> Example: さくら (sa-ku-ra) = 3 characters = 3 syllables

### 2.2 The Japanese Sound System

Japanese has 5 vowels and 14 consonants that combine into syllables:

| Vowel | Romaji | IPA | Sound |
|-------|--------|-----|-------|
| あ | a | /a/ | Like "a" in "father" |
| い | i | /i/ | Like "ee" in "feet" |
| う | u | /ɯ/ | Like "oo" in "food" (lips unrounded) |
| え | e | /e/ | Like "e" in "pet" |
| お | o | /o/ | Like "o" in "go" (short, no glide) |

> **Tip**: Japanese vowels are pure — they do not glide like English vowels. Keep them short and crisp.

## 4. Dakuten, Handakuten, and Combo Characters

### 4.1 Dakuten (゛) — Voiced Consonants

Adding two small strokes (゛) to certain hiragana **voices** the consonant:

| Base | + Dakuten | Sound Change |
|------|-----------|-------------|
| か ka → | が ga | k → g |
| き ki → | ぎ gi | k → g |
| く ku → | ぐ gu | k → g |
| け ke → | げ ge | k → g |
| こ ko → | ご go | k → g |
| さ sa → | ざ za | s → z |
| し shi → | じ ji | sh → j |
| す su → | ず zu | s → z |
| せ se → | ぜ ze | s → z |
| そ so → | ぞ zo | s → z |
| た ta → | だ da | t → d |
| ち chi → | ぢ ji* | ch → j |
| つ tsu → | づ zu* | ts → z |
| て te → | で de | t → d |

> ***Note**: ぢ (ji) and づ (zu) are rarely used. じ (ji) and ず (zu) are standard.

## 5. Katakana: The Foreign Script

### 5.1 What is Katakana?

Katakana is the second syllabary. It represents the **exact same sounds** as hiragana but is used in different contexts.

> **Katakana Key Facts**
>
> - **Characters**: 46 basic (same sounds as hiragana)
> - **Shape**: Angular, sharp, geometric
> - **Origin**: Simplified from parts of kanji by monks
> **Used for**:
>   - Foreign loanwords (コンピュータ = computer)
>   - Foreign names (マイケル = Michael)
>   - Country names (アメリカ = America)
>   - Onomatopoeia (ドキドキ = heart beating)
>   - Emphasis (like CAPS or italics in English)
>   - Scientific/technical terms
>   - Company/brand names

### 5.2 Hiragana vs Katakana Comparison

| Hiragana | Katakana | Romaji | Note |
|----------|----------|--------|------|
| あ | ア | a | Same sound, different shape |
| か | カ | ka | Katakana is more angular |
| さ | サ | sa | |
| た | タ | ta | |
| な | ナ | na | |
| は | ハ | ha | |

### 6.2 Commonly Confused Katakana Pairs

These pairs look similar and are a frequent source of mistakes:

| Pair | Characters | Tip to Distinguish |
|------|-----------|-------------------|
| ア / マ | a / ma | ア has a horizontal stroke; マ has two downward strokes |
| シ / ツ | shi / tsu | シ strokes go upward (↗); ツ strokes go downward (↘) |
| ソ / ン | so / n | ソ strokes go downward (↘); ン strokes go upward (↗) |
| ク / タ | ku / ta | タ has an extra horizontal stroke |
| ヌ / ス | nu / su | ヌ has a crossing; ス does not |
| ウ / ワ | u / wa | ウ has a top stroke; ワ is open at top |

> **Critical**: シ/ツ and ソ/ン are the most commonly confused. Practice these until you can distinguish them instantly.

## 7. Introduction to Kanji

### 7.1 What are Kanji?

> **Kanji Key Facts**
>
> - **Origin**: Borrowed from Chinese characters
> - **Total existing**: ~50,000+
> - **Jouyou kanji (常用漢字)**: 2,136 (official "daily use" set)
> - **JLPT N5 kanji**: ~100
> - **JLPT N1 kanji**: ~2,000
>
> Each kanji has:
> - Meaning(s)
> - On'yomi (音読み) — Chinese-derived reading
> - Kun'yomi (訓読み) — Native Japanese reading
> - Stroke order — must be written in correct order
> - Radical(s) — component parts

### 7.2 Examples of Basic Kanji

| Kanji | Meaning | On'yomi | Kun'yomi | Strokes | Example Word |
|-------|---------|---------|----------|---------|-------------|
| 一 | one | イチ (ichi) | ひと (hito) | 1 | 一つ (hitotsu, one thing) |
| 二 | two | ニ (ni) | ふた (futa) | 2 | 二つ (futatsu, two things) |
| 三 | three | サン (san) | み (mi) | 3 | 三つ (mittsu, three things) |
| 人 | person | ジン (jin) | ひと (hito) | 2 | 日本人 (nihonjin, Japanese person) |
| 日 | day/sun | ニチ (nichi) | ひ (hi) | 4 | 日曜日 (nichiyoubi, Sunday) |
| 本 | book/origin | ホン (hon) | もと (moto) | 5 | 日本 (nihon, Japan) |
| 山 | mountain | サン (san) | やま (yama) | 3 | 山田 (Yamada, surname) |
| 川 | river | セン (sen) | かわ (kawa) | 3 | 川 (kawa, river) |
| 大 | big | ダイ (dai) | おお (oo) | 3 | 大きい (ookii, big) |
| 小 | small | ショウ (shou) | ちい (chii) | 3 | 小さい (chiisai, small) |

### 7.3 Why Kanji Exist in Japanese

You might wonder: if hiragana can write everything, why use kanji? Three practical reasons:

1. **Disambiguation**: Japanese has many homophones. 橋 (hashi, bridge) vs 箸 (hashi, chopsticks) vs 端 (hashi, edge)
2. **Reading speed**: Kanji characters are visually dense. 東京都千代田区 is faster to scan than とうきょうとちよだく
3. **Compactness**: Kanji compress meaning into fewer characters. 学校 (gakkou, school) = 2 characters vs がっこう = 4 characters

## 8. Radicals: The Building Blocks of Kanji

### 8.1 What are Radicals?

Radicals (部首, bushu) are the component parts that make up kanji. Think of them like building blocks. Learning radicals helps you:

- **Remember kanji** — break complex characters into familiar parts
- **Look up kanji** — dictionaries are organized by radical
- **Guess meaning** — radicals often hint at the meaning category

### 8.2 Common Radicals

| Radical | Name | Meaning | Found in |
|---------|------|---------|----------|
| 人 (亻) | にんべん | person | 休 (rest), 体 (body), 何 (what) |
| 口 | くちへん | mouth | 食 (eat), 言 (say), 名 (name) |
| 水 (氵) | さんずい | water | 海 (sea), 泳 (swim), 酒 (sake) |
| 木 | きへん | tree/wood | 森 (forest), 本 (book), 校 (school) |
| 日 | にちへん | sun/day | 時 (time), 明 (bright), 曜 (day of week) |
| 火 (灬) | ひへん | fire | 焼 (burn), 点 (point), 熱 (heat) |
| 土 | つちへん | earth | 地 (ground), 場 (place) |
| 女 | おんなへん | woman | 好 (like), 妹 (younger sister) |
| 金 | かねへん | metal/gold | 銀 (silver), 鉄 (iron) |
| 言 (訁) | ごんべん | speech | 話 (talk), 読 (read), 語 (language) |

### 8.3 Radical Position

Radicals appear in specific positions within a kanji:

> **Radical Positions in Kanji**
>
> - **偏 (hen)**: Left side — 亻 in 休
> - **旁 (tsukuri)**: Right side — 力 in 動
> - **冠 (kanmuri)**: Top — 雨 in 雪
> - **脚 (ashi)**: Bottom — 心 in 思
> - **構 (kamae)**: Enclosure — 門 in 間
> - **垂 (tare)**: Top-left drape — 广 in 店
> - **繞 (nyou)**: Bottom-left wrap — 辶 in 道

## 9. On'yomi and Kun'yomi: Two Reading Systems

### 9.1 The Dual Reading Problem

Every kanji has at least two ways to read it — sometimes many more. This is the single most challenging aspect of kanji.

> **On'yomi vs Kun'yomi**
>
> **On'yomi (音読み) — "Sound reading"**
> - Chinese-derived pronunciation
> - Used in compound words (jukugo)
> - Written in katakana in dictionaries
> - Example: 山 = サン (SAN) → 山脈 (sanmyaku, mountain range)
>
> **Kun'yomi (訓読み) — "Meaning reading"**
> - Native Japanese pronunciation
> - Used when kanji stands alone or with hiragana endings
> - Written in hiragana in dictionaries
> - Example: 山 = やま (yama) → 山 (yama, mountain)

### 9.2 When to Use Which Reading

| Context | Reading | Example |
|---------|---------|---------|
| Kanji alone (or + hiragana) | Kun'yomi | 山 (yama), 食べる (taberu) |
| Two+ kanji compound | On'yomi | 火山 (kazan, volcano), 食事 (shokuji, meal) |
| Names | Either | 山田 (Yamada), 山本 (Yamamoto) |

### 9.3 Example: The Kanji 食

| Reading | Type | Context | Word |
|---------|------|---------|------|
| ショク (shoku) | On'yomi | In compounds | 食事 (shokuji, meal), 食堂 (shokudou, cafeteria) |
| た.べる (taberu) | Kun'yomi | With hiragana ending | 食べる (taberu, to eat) |
| く.う (kuu) | Kun'yomi | Casual/male speech | 食う (kuu, to eat) |

> **Tip for beginners**: Don't try to memorize all readings in isolation. Learn readings through vocabulary words. When you learn the word 食べる (taberu), you naturally learn that 食 can be read た.

## 10. Writing Practice Guidance

### 10.1 Stroke Order Rules

Japanese characters must be written in a specific stroke order. The basic rules:

```
Rule 1: Top to bottom              Rule 5: Left-falling before right-falling
        一 (horizontal first)              人 (left \ then right /)

Rule 2: Left to right              Rule 6: Center before wings
        川 (left stroke first)             小 (center | then left, right)

Rule 3: Horizontal before vertical  Rule 7: Enclosure before contents
        十 (— then |)                     国 (outer 口 then inner 玉)

Rule 4: Outside before inside       Rule 8: Bottom closing stroke last
        月 (outer strokes first)           日 (bottom — closes the box)
```

### 10.2 Practice Strategy

> **Recommended Practice Schedule**
>
> **Week 1-2: Hiragana**
> - Learn 5-10 characters per day
> - Write each character 10+ times
> - Use flashcards for recognition
> - Practice reading simple words
>
> **Week 3-4: Katakana**
> - Same approach as hiragana
> - Practice with loanwords you know
> - Compare similar-looking pairs
>
> **Week 5+: Begin Kanji**
> - Learn 3-5 new kanji per day
> - Always learn in context (with vocabulary)
> - Review using spaced repetition
> - Pay attention to radicals
>
> **Tools**: Grid paper (genkouyoushi 原稿用紙), flashcard apps (Anki), writing practice sheets

### 10.3 Common Beginner Mistakes

| Mistake | Problem | Fix |
|---------|---------|-----|
| Ignoring stroke order | Characters look wrong, hard to read | Learn correct order from the start |
| Skipping hiragana | Can't read anything, rely on romaji | Master hiragana before anything else |
| Learning kanji without words | Memorize readings but can't use them | Always learn kanji through vocabulary |
| Mixing up similar kana | は/ほ, わ/れ/ね, シ/ツ, ソ/ン | Practice pairs side by side |
| Using romaji as a crutch | Never develop real reading ability | Switch to kana-only as soon as possible |
