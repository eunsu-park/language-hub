## 1. Overview: Four Ways to Say "If"

Japanese has four main conditional forms, each with different nuances:

```
+------------------------------------------------------------------+
|  CONDITIONAL FORM      CORE MEANING          KEY CHARACTER         |
+------------------------------------------------------------------+
|  ば (ba)               General "if"          Hypothetical          |
|  たら (tara)           "If/when...then"      Completed condition   |
|  なら (nara)           "If it's the case"    Topic/context-based   |
|  と (to)               "Whenever"            Natural consequence   |
+------------------------------------------------------------------+
```

> **Important**: While all four can translate as "if" in English, they are **not** interchangeable. Each has specific contexts where it is most natural.

## 2. ば Conditional (仮定形)

The ば form expresses **general conditions** and **hypothetical situations**: "if X, then Y."

### Formation Rules

| Type | Rule | Example |
|------|------|---------|
| Group 1 verbs | Change final -u to -e + ば | 書く → 書**けば**, 読む → 読**めば**, 行く → 行**けば** |
| Group 2 verbs | Drop -る, add -れば | 食べる → 食べ**れば**, 見る → 見**れば** |
| する | すれば | する → **すれば** |
| 来る | 来れば (くれば) | 来る → **来れば** |
| い-adjective | Drop -い, add -ければ | 高い → 高**ければ**, 安い → 安**ければ** |
| な-adjective | Stem + であれば / なら(ば) | 静かであれば, 元気であれば |
| Noun | Noun + であれば | 学生であれば |
| Negative verb | ない → なければ | 行かない → 行か**なければ** |

### Group 1 Verb ば Form: All Endings

| Final Kana | Change | Example |
|-----------|--------|---------|
| く → けば | 書く → 書けば | |
| す → せば | 話す → 話せば | |
| つ → てば | 待つ → 待てば | |
| ぬ → ねば | 死ぬ → 死ねば | |
| ぶ → べば | 遊ぶ → 遊べば | |
| む → めば | 読む → 読めば | |
| る → れば | 帰る → 帰れば | |
| う → えば | 買う → 買えば | |
| ぐ → げば | 泳ぐ → 泳げば | |

### Uses and Nuances

#### A. General / Hypothetical Conditions

> **安ければ、買います。** — If it's cheap, I'll buy it.
> **時間があれば、映画を見たい。** — If I have time, I want to watch a movie.

#### B. Proverbs and Universal Truths

> **練習すれば、上手になる。** — If you practice, you get better.
> **急がば回れ。** — More haste, less speed. (Lit: If you hurry, go around.)

#### C. Giving Advice (〜ばいい)

> **先生に聞けばいい。** — You should just ask the teacher.
> **早く寝ればいいのに。** — You should go to bed early. (with a nuance of reproach)

### Restriction: ば with Volitional Actions

> **Key rule**: When using ば, the main clause generally should NOT be a volitional action (command, request, invitation) unless the condition clause is a state or adjective.
>
> ✗ 東京に行けば、寿司を食べましょう。 (Unnatural)
> ○ 東京に行ったら、寿司を食べましょう。 (Use たら instead)
>
> ○ 天気がよければ、散歩しましょう。 (OK — condition is a state, not an action)

### Example Sentences

1. **薬を飲めば、すぐ治る。** — If you take the medicine, you'll get better quickly.
2. **もっと勉強すれば、合格できる。** — If you study more, you can pass.
3. **彼に会えば、伝えておきます。** — If I see him, I'll pass on the message.
4. **この道をまっすぐ行けば、駅に着きます。** — If you go straight on this road, you'll reach the station.
5. **暇であれば、手伝ってくれませんか。** — If you're free, could you help me?

## 3. たら Conditional

The たら form expresses conditions that are **completed before** the result occurs: "if/when X happens, then Y." It is the most versatile and commonly used conditional.

### Formation Rules

Simply add ら to the **plain past form** (た-form):

| Type | Past Form | たら Form |
|------|-----------|----------|
| Group 1 verb | 書いた | 書い**たら** |
| Group 1 verb | 行った | 行っ**たら** |
| Group 1 verb | 読んだ | 読ん**だら** |
| Group 2 verb | 食べた | 食べ**たら** |
| する | した | し**たら** |
| 来る | 来た (きた) | 来**たら** (きたら) |
| い-adjective | 高かった | 高かっ**たら** |
| な-adjective | 静かだった | 静かだっ**たら** |
| Noun | 学生だった | 学生だっ**たら** |
| Negative verb | 行かなかった | 行かなかっ**たら** |

> **Formation is simple**: Past form + ら. That's it!

### Uses and Nuances

#### A. Completed Condition ("When/After X happens")

> **家に帰ったら、電話してね。** — When you get home, give me a call.
> **宿題が終わったら、遊びに行こう。** — When/After homework is done, let's go hang out.

#### B. Hypothetical Condition ("If X were to happen")

> **宝くじに当たったら、家を買う。** — If I won the lottery, I'd buy a house.
> **もし明日雨だったら、試合は中止です。** — If it rains tomorrow, the game is cancelled.

#### C. Discovered Result ("When I did X, I found that...")

> **窓を開けたら、猫がいた。** — When I opened the window, there was a cat.
> **駅に着いたら、電車がもう出ていた。** — When I arrived at the station, the train had already left.

#### D. Advice and Suggestions (〜たらどう？)

> **医者に行ったらどうですか。** — How about going to the doctor?
> **もっと休んだら？** — How about resting more? (casual)

### The もし Prefix

もし (what if) can be added before any conditional for emphasis, but pairs most naturally with たら:

> **もし時間があったら、来てください。** — If you happen to have time, please come.
> **もし私だったら、そうはしない。** — If it were me, I wouldn't do that.

### Example Sentences

1. **日本に行ったら、温泉に入りたい。** — If/When I go to Japan, I want to try hot springs.
2. **もし100万円あったら、何をする？** — If you had a million yen, what would you do?
3. **大人になったら、パイロットになりたい。** — When I grow up, I want to be a pilot.
4. **道を曲がったら、コンビニが見えた。** — When I turned the corner, I saw a convenience store.
5. **分からなかったら、聞いてください。** — If you don't understand, please ask.

## 4. なら Conditional

The なら form is **topic-based**: it responds to something the listener said or a known situation. It means "if it's the case that..." or "if we're talking about..."

### Formation Rules

| Type | なら Form |
|------|----------|
| Verb (plain) | 行く**なら** |
| い-adjective | 高い**なら** (or 高いの**なら**) |
| な-adjective | 静か**なら** (or 静かな**のなら**) |
| Noun | 学生**なら** |

> なら attaches to the **plain (dictionary) form**. No conjugation changes needed.

### Uses and Nuances

#### A. Responding to Information ("If that's the case...")

> A: 「明日は暇だ。」(I'm free tomorrow.)
> B: 「暇**なら**、映画を見に行かない？」(If you're free, want to go see a movie?)

> A: 「日本語を勉強している。」(I'm studying Japanese.)
> B: 「日本語を勉強している**なら**、この本がいいよ。」(If you're studying Japanese, this book is good.)

#### B. Giving Advice Based on Context

> **日本に行くなら、秋がいいですよ。** — If you're going to Japan, autumn is great.
> **買うなら、この店が安い。** — If you're going to buy, this store is cheap.

#### C. Qualification / Limitation

> **遊ぶなら、宿題をしてからにしなさい。** — If you're going to play, do it after your homework.
> **お酒を飲むなら、運転してはいけない。** — If you're going to drink alcohol, you must not drive.

### Key Feature: Result Can Precede Condition

Unlike the other conditionals, with なら, the result action can happen **before** the condition:

> **スーパーに行くなら、牛乳を買ってきて。**
> If you're going to the supermarket, buy milk (on your way).
> (The milk-buying happens DURING the going, before arrival.)

Compare with たら:
> **スーパーに行ったら、牛乳を買ってきて。**
> When you go to the supermarket, buy milk.
> (Buying happens AFTER arriving.)

### Example Sentences

1. **北海道に行くなら、ラーメンを食べたほうがいい。** — If you're going to Hokkaido, you should try ramen.
2. **彼が来るなら、私も行く。** — If he's coming, I'll go too.
3. **学生なら、この割引が使えます。** — If you're a student, you can use this discount.
4. **暑いなら、窓を開けてもいい。** — If it's hot, you may open the window.
5. **田中さんのことなら、よく知っていますよ。** — If it's about Tanaka, I know well.

## 5. と Conditional

The と conditional expresses **natural consequences**, **habitual results**, and **discoveries**: "whenever X, Y" or "if X, Y always follows."

### Formation Rules

| Type | と Form |
|------|--------|
| Verb (plain non-past) | 押す**と** |
| い-adjective | 高い**と** |
| な-adjective | 静かだ**と** |
| Noun | 春だ**と** |
| Negative | 行かない**と** |

> と always attaches to the **plain non-past** form.

### Uses and Nuances

#### A. Natural/Automatic Consequences

> **ボタンを押すと、ドアが開く。** — When you press the button, the door opens.
> **春になると、桜が咲く。** — When spring comes, cherry blossoms bloom.

#### B. Habitual Actions

> **毎朝起きると、コーヒーを飲む。** — Whenever I wake up in the morning, I drink coffee.
> **この道をまっすぐ行くと、駅がある。** — If you go straight on this road, there's a station.

#### C. Discovery (Past Tense)

> **窓を開けると、雪が降っていた。** — When I opened the window, it was snowing.
> **家に帰ると、手紙が届いていた。** — When I got home, a letter had arrived.

### Restriction: No Volitional Actions in Main Clause

The main clause with と generally cannot contain requests, commands, intentions, or suggestions:

```
✗ ボタンを押すと、中に入ってください。   (Unnatural)
○ ボタンを押したら、中に入ってください。 (Use たら)

✗ 6時になると、出かけましょう。          (Unnatural)
○ 6時になったら、出かけましょう。        (Use たら)
```

> **Rule**: と implies that the result happens **automatically** — so it cannot be a deliberate choice by the listener.

### Example Sentences

1. **この薬を飲むと、眠くなる。** — When you take this medicine, you get sleepy.
2. **右に曲がると、銀行があります。** — If you turn right, there's a bank.
3. **冬になると、雪がたくさん降る。** — When winter comes, it snows a lot.
4. **ドアを開けると、猫が飛び出してきた。** — When I opened the door, a cat came jumping out.
5. **食べ過ぎると、お腹が痛くなる。** — If you eat too much, your stomach hurts.

## 6. Comparison Chart: Choosing the Right Conditional

### Summary Table

| Feature | ば | たら | なら | と |
|---------|---|------|------|---|
| **Core meaning** | General hypothesis | Completed condition | Topic/context-based | Natural consequence |
| **Volitional result** | Limited | Yes | Yes | No |
| **Past events** | No | Yes | No | Yes (discovery) |
| **Advice** | ばいい | たらどう | Direct advice | No |
| **Habitual** | Yes | Less common | No | Yes |
| **Formality** | Slightly formal | Neutral (most common) | Neutral | Neutral |
| **もし pairing** | OK | Best | OK | Rare |

### Decision Guide

```
+---------------------------------------------------+
|  "If it rains tomorrow..."                        |
|                                                   |
|  General:     雨が降れば...     (ば)              |
|  Specific:    雨が降ったら...   (たら)            |
|  Contextual:  雨なら...         (なら)            |
|  Habitual:    雨が降ると...     (と) — always     |
+---------------------------------------------------+
```

### Practical Comparison Examples

| Situation | Best Form | Sentence | Why |
|-----------|-----------|----------|-----|
| Giving directions | と | まっすぐ行くと、駅がある。 | Automatic result |
| Making plans | たら | 雨が止んだら、出かけよう。 | Volitional result |
| Responding to info | なら | 東京に行くなら、案内する。 | Reacting to topic |
| General truth | ば | 努力すれば、成功する。 | Hypothesis |
| Discovered situation | たら / と | ドアを開けたら/開けると、猫がいた。 | Past discovery |
| Advice | ば / たら | 聞けばいい / 聞いたらどう | Suggestion |

## 7. Common Patterns and Set Phrases

### ば Patterns

| Pattern | Meaning | Example |
|---------|---------|---------|
| 〜ば〜ほど | The more... the more... | 練習すればするほど上手になる。 |
| 〜ばいい | Should just... | 先生に聞けばいい。 |
| 〜ばよかった | I wish I had... (regret) | もっと勉強すればよかった。 |
| 〜なければならない | Must do | 行かなければならない。 |

### たら Patterns

| Pattern | Meaning | Example |
|---------|---------|---------|
| 〜たらどうですか | How about...? | 休んだらどうですか。 |
| 〜たらいいですか | What should I...? | 何を持って行ったらいいですか。 |
| もし〜たら | What if... | もし宝くじが当たったら... |

### なら Patterns

| Pattern | Meaning | Example |
|---------|---------|---------|
| 〜のなら | If it is that... (emphasis) | 行くのなら、早くしなさい。 |
| Noun + なら | Speaking of... / If it's... | 寿司なら、あの店がいい。 |

### と Patterns

| Pattern | Meaning | Example |
|---------|---------|---------|
| 〜と思う | I think that... | 美味しいと思う。 |
| 〜ないといけない | Must do | 行かないといけない。 |
| 〜とき | When... | 日本にいたとき |

## 8. Common Mistakes and Pitfalls

### Mistake 1: Using と with Commands or Requests

```
✗ 駅に着くと、電話してください。
○ 駅に着いたら、電話してください。
```

> と cannot have volitional actions (commands, requests, suggestions) in the result clause.

### Mistake 2: Using ば for One-Time Past Events

```
✗ 昨日、家に帰れば、母がいた。
○ 昨日、家に帰ったら、母がいた。
```

> ば is generally not used for specific past events. Use たら for "when I did X, Y happened."

### Mistake 3: Confusing なら Timing

```
スーパーに行くなら、牛乳を買ってきて。
  = Buy milk on the way (the request is for DURING the trip)

スーパーに行ったら、牛乳を買ってきて。
  = Buy milk when you arrive (the request is for AFTER arriving)
```

### Mistake 4: Overusing たら

While たら is the most versatile, using it everywhere sounds unnatural:

```
△ 春になったら、桜が咲く。 (Acceptable but less natural)
○ 春になると、桜が咲く。   (More natural for habitual/natural events)
```

## 9. Practice Exercises

### Exercise 1: Form the ば Conditional

Convert to ば form.

1. 食べる → ________
2. 安い → ________
3. 行く → ________
4. する → ________
5. 学生 → ________

<details>
<summary>Answers</summary>

1. 食べ**れば** (Group 2: drop る, add れば)
2. 安**ければ** (い-adjective: drop い, add ければ)
3. 行**けば** (Group 1: く → けば)
4. **すれば** (Irregular)
5. 学生**であれば** (Noun + であれば)

</details>

### Exercise 2: Form the たら Conditional

Convert to たら form.

1. 読む → ________
2. 来る (くる) → ________
3. いい → ________
4. 暇だ → ________
5. 食べない → ________

<details>
<summary>Answers</summary>

1. 読ん**だら** (past: 読んだ + ら)
2. 来**たら** (きたら) (past: 来た + ら)
3. よかっ**たら** (past of いい: よかった + ら)
4. 暇だっ**たら** (past: 暇だった + ら)
5. 食べなかっ**たら** (past neg: 食べなかった + ら)

</details>

### Exercise 3: Choose the Best Conditional

Fill in with the most natural conditional form (ば, たら, なら, と).

1. ボタンを押す______、ドアが開きます。(automatic result)
2. もし明日雨だっ______、ピクニックは中止にしよう。(plan change)
3. 日本に行く______、着物を買ったらどう？(responding to someone's plan)
4. もっと練習すれ______、上手になりますよ。(general truth)
5. 冬にな______、雪がたくさん降る。(habitual/seasonal)

<details>
<summary>Answers</summary>

1. ボタンを押す**と** (automatic/natural consequence)
2. もし明日雨だっ**たら** (hypothetical + volitional result)
3. 日本に行く**なら** (responding to someone's stated plan)
4. もっと練習すれ**ば** (general truth/hypothesis)
5. 冬にな**ると** (habitual seasonal event)

</details>

### Exercise 4: Translate Using Conditionals

Translate to Japanese using the most appropriate conditional.

1. If you're free, let's go to karaoke. (responding to what someone said)
2. When I opened the door, no one was there. (past discovery)
3. If you study every day, your Japanese will improve. (general truth)
4. When spring comes, it gets warm. (natural consequence)
5. If you're going to buy a computer, that store is good. (advice based on context)

<details>
<summary>Answers</summary>

1. 暇**なら**、カラオケに行こう。(なら — responding to information)
2. ドアを開け**たら** / 開ける**と**、誰もいなかった。(たら/と — past discovery)
3. 毎日勉強すれ**ば**、日本語が上手になる。(ば — general truth)
4. 春になる**と**、暖かくなる。(と — natural consequence)
5. パソコンを買う**なら**、あの店がいい。(なら — contextual advice)

</details>

### Exercise 5: Error Correction

Find and fix the error in each sentence.

1. ✗ まっすぐ行くと、右に曲がってください。
2. ✗ 昨日家に帰れば、手紙が届いていた。
3. ✗ 春になったら、桜が咲く。(natural seasonal event)
4. ✗ この薬を飲めば、眠くなる。(automatic physical effect)

<details>
<summary>Answers</summary>

1. ○ まっすぐ行っ**たら**、右に曲がってください。(と cannot have commands in the result)
2. ○ 昨日家に帰っ**たら**、手紙が届いていた。(ば is not used for specific past discoveries)
3. ○ 春にな**ると**、桜が咲く。(と is more natural for habitual/natural events)
4. ○ この薬を飲む**と**、眠くなる。(と is more natural for automatic consequences)

</details>
