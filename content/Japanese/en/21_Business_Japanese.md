## 1. Business Honorifics (ビジネス敬語)

Business Japanese demands a level of politeness that goes far beyond the classroom keigo
covered in Lesson 14. In the corporate world, incorrect keigo can damage relationships
and create the impression of incompetence.

```
+----------------------------------------------------------+
|          BUSINESS KEIGO HIERARCHY                         |
+----------------------------------------------------------+
|                                                          |
|  Level 1: 丁寧語 (Polite language)                       |
|  └── です / ます forms                                    |
|  └── Used with peers and familiar colleagues             |
|                                                          |
|  Level 2: 尊敬語 (Respectful language)                    |
|  └── Elevates the OTHER person's actions                 |
|  └── Used for superiors, clients, external contacts      |
|                                                          |
|  Level 3: 謙譲語 (Humble language)                       |
|  └── Lowers YOUR OWN actions                             |
|  └── Used when speaking about yourself to superiors      |
|                                                          |
|  Level 4: ビジネス敬語 (Business honorifics)              |
|  └── Fixed set phrases and ultra-formal expressions      |
|  └── Required for client-facing communication            |
+----------------------------------------------------------+
```

### 1.1 Upgrading Common Verbs to Business Level

In business settings, standard keigo is often insufficient. Here are the business-grade
upgrades for frequently used verbs:

| Plain | Standard Keigo | Business Level |
|-------|---------------|----------------|
| する (do) | なさる / いたす | させていただく |
| 言う (say) | おっしゃる / 申す | 申し上げる |
| 行く (go) | いらっしゃる / 参る | お伺いする / 参上する |
| 来る (come) | いらっしゃる / 参る | お越しになる / お見えになる |
| 見る (see) | ご覧になる / 拝見する | 拝見させていただく |
| 知る (know) | ご存じ / 存じる | 存じ上げる |
| 食べる (eat) | 召し上がる / いただく | 頂戴する |
| もらう (receive) | いただく | 頂戴する / 賜る（たまわる） |
| あげる (give) | 差し上げる | 進呈する（しんていする） |
| 聞く (ask/listen) | お聞きになる / 伺う | 拝聴する（はいちょうする） |

### 1.2 The ～させていただく Construction

This is the most distinctive feature of modern business Japanese. It literally means
"allow me to humbly do..." and is used extensively:

```
Structure: Verb (させ form) + ていただく

Examples:
  確認させていただきます。
  → Allow me to confirm (this).

  説明させていただきます。
  → Allow me to explain.

  ご連絡させていただきました。
  → I have taken the liberty of contacting you.

  担当させていただいております。
  → I have the honor of being in charge of (this).
```

> **Caution**: Overusing ～させていただく is a common mistake. Use it when:
> 1. You are genuinely receiving permission from someone
> 2. You benefit from the action
>
> Incorrect: ✗ 「帰らせていただきます」 (when leaving on time — no permission needed)
> Correct: ○ 「お先に失礼いたします」 (standard leaving phrase)

### 1.3 Double Honorific Errors (二重敬語)

A frequent mistake in business settings is stacking honorifics:

| Incorrect (二重敬語) | Correct |
|---------------------|---------|
| ✗ お召し上がりになられる | ○ 召し上がる |
| ✗ おっしゃられる | ○ おっしゃる |
| ✗ ご覧になられる | ○ ご覧になる |
| ✗ お伺いさせていただく | ○ 伺う / お伺いする |

## 2. Set Phrases for Professional Settings

### 2.1 Greetings and Daily Phrases

| Situation | Phrase | Meaning |
|-----------|--------|---------|
| Morning greeting | おはようございます | Good morning |
| Leaving before others | お先に失礼いたします | Excuse me for leaving first |
| To someone leaving | お疲れ様でした（おつかれさまでした） | Thank you for your hard work |
| Starting work | 本日もよろしくお願いいたします | Please treat me well today also |
| Returning to office | 戻りました | I've returned |
| To someone returning | お帰りなさい / お疲れ様です | Welcome back |
| Going out | 行ってまいります | I'm heading out |
| When someone goes out | 行ってらっしゃい(ませ) | Take care |

### 2.2 Apology Expressions (Ranked by Formality)

```
+----------------------------------------------------------+
|           APOLOGY FORMALITY SCALE                        |
+----------------------------------------------------------+
|                                                          |
|  Casual ──────────────────────────────── Ultra-formal    |
|                                                          |
|  すみません                                               |
|    ↓                                                     |
|  申し訳ありません                                          |
|    ↓                                                     |
|  申し訳ございません                                        |
|    ↓                                                     |
|  大変申し訳ございません                                     |
|    ↓                                                     |
|  誠に申し訳ございませんでした                                |
|    ↓                                                     |
|  深くお詫び申し上げます                                     |
|                                                          |
+----------------------------------------------------------+
```

### 2.3 Requesting and Declining

**Making requests**:
| Formality | Expression |
|-----------|------------|
| Standard polite | ～していただけますか |
| More polite | ～していただけないでしょうか |
| Very formal | ～していただけますと幸いです |
| Ultra-formal | ～賜りますようお願い申し上げます |

**Declining politely**:
| Situation | Expression |
|-----------|------------|
| General decline | 申し訳ございませんが、ちょっと難しい状況でして... |
| Scheduling conflict | あいにく、その日は先約がございまして... |
| Beyond capability | 力不足で申し訳ございませんが... |
| Company policy | 社内規定により、お受けいたしかねます |

> **Cultural note**: Japanese business culture avoids direct refusal. Instead of
> saying "No," use softening expressions like ちょっと... (a bit...) or 難しい
> (difficult) to imply refusal without confrontation.

## 3. Business Email Writing

### 3.1 Email Structure

```
+----------------------------------------------------------+
|  JAPANESE BUSINESS EMAIL STRUCTURE                       |
+----------------------------------------------------------+
|                                                          |
|  件名 (Subject line)                                     |
|  ├── Clear and specific                                  |
|  └── Example: 【ご確認】3月度会議の議題について             |
|                                                          |
|  宛名 (Addressee)                                        |
|  ├── 株式会社○○                                          |
|  ├── ○○部 ○○課                                          |
|  └── ○○様                                               |
|                                                          |
|  挨拶 (Greeting)                                         |
|  ├── External: お世話になっております                      |
|  └── Internal: お疲れ様です                               |
|                                                          |
|  自己紹介 (Self-introduction)                             |
|  └── ○○株式会社の○○でございます                           |
|                                                          |
|  本文 (Body)                                             |
|  └── State purpose, details, and requests                |
|                                                          |
|  結び (Closing)                                          |
|  ├── 何卒よろしくお願い申し上げます                         |
|  └── ご確認のほど、よろしくお願いいたします                  |
|                                                          |
|  署名 (Signature block)                                  |
|  ├── Company name, department                            |
|  ├── Full name                                           |
|  ├── Phone, fax, email                                   |
|  └── Company address                                     |
+----------------------------------------------------------+
```

| Prefix | Usage | Example |
|--------|-------|---------|
| 【ご確認】 | Requesting confirmation | 【ご確認】納品スケジュールについて |
| 【ご連絡】 | General notification | 【ご連絡】来週の出張予定 |
| 【ご依頼】 | Making a request | 【ご依頼】見積書の作成について |
| 【ご報告】 | Reporting something | 【ご報告】プロジェクト進捗状況 |
| 【お詫び】 | Apologizing | 【お詫び】納期遅延のお知らせ |
| 【ご案内】 | Announcement/invitation | 【ご案内】新年会のご案内 |
| 【至急】 | Urgent | 【至急】本日の会議変更について |

### 4.2 Meeting Flow and Key Phrases

```
+----------------------------------------------------------+
|         TYPICAL JAPANESE MEETING FLOW                    |
+----------------------------------------------------------+
|                                                          |
|  1. 開会 (Opening)                                       |
|     「それでは、定刻になりましたので                        |
|      会議を始めさせていただきます。」                       |
|     (It is now the scheduled time, so let us              |
|      begin the meeting.)                                  |
|                                                          |
|  2. 出欠確認 (Attendance check)                           |
|     「本日の出席者は○名です。」                             |
|     (Today we have ○ attendees.)                         |
|                                                          |
|  3. 前回の確認 (Review of previous meeting)               |
|     「前回の議事録をご確認ください。」                       |
|     (Please review the previous minutes.)                |
|                                                          |
|  4. 議題の説明 (Agenda presentation)                      |
|     「本日の議題は3点ございます。」                         |
|     (We have 3 agenda items today.)                      |
|                                                          |
|  5. 討議 (Discussion)                                    |
|     「この件について、ご意見をお聞かせください。」            |
|     (Please share your opinions on this matter.)         |
|                                                          |
|  6. 決議 (Resolution)                                    |
|     「では、この提案を承認ということで                      |
|      よろしいでしょうか。」                                |
|     (Shall we approve this proposal?)                    |
|                                                          |
|  7. 閉会 (Closing)                                       |
|     「以上をもちまして、本日の会議を                        |
|      終了いたします。お忙しい中、                          |
|      ありがとうございました。」                             |
|     (With that, we conclude today's meeting.             |
|      Thank you for your time.)                           |
+----------------------------------------------------------+
```

| Deferring | 持ち帰り検討させていただきます |
| Summarizing | つまり、○○ということでよろしいでしょうか |

## 5. Telephone Manners

### 5.1 Answering and Making Calls

Japanese telephone etiquette has very specific patterns. Errors here are immediately
noticeable and can damage a business relationship.

```
+----------------------------------------------------------+
|        ANSWERING A BUSINESS CALL                         |
+----------------------------------------------------------+
|                                                          |
|  Ring ring...                                            |
|                                                          |
|  「はい、○○株式会社でございます。」                         |
|  (Yes, this is ○○ Company.)                              |
|                                                          |
|  NOTE: Business calls do NOT begin with もしもし.          |
|  もしもし is for personal calls only.                      |
|                                                          |
+----------------------------------------------------------+
|        MAKING A BUSINESS CALL                            |
+----------------------------------------------------------+
|                                                          |
|  「お忙しいところ恐れ入ります。                             |
|    ○○株式会社の○○と申します。                              |
|    ○○部の○○様はいらっしゃいますでしょうか。」                |
|                                                          |
|  (I'm sorry to bother you when you're busy.             |
|   My name is ○○ from ○○ Company.                        |
|   Is Mr./Ms. ○○ from the ○○ department available?)      |
|                                                          |
+----------------------------------------------------------+
```

### 5.2 Telephone Phrases

| Situation | Phrase |
|-----------|--------|
| Answering | ○○株式会社でございます。 |
| Identifying yourself | ○○会社の○○と申します。 |
| Asking for someone | ○○様はいらっしゃいますでしょうか。 |
| Putting on hold | 少々お待ちください(ませ)。 |
| Transferring | ○○におつなぎいたします。 |
| Person is out | あいにく、○○はただいま外出しております。 |
| Person is on another call | ○○はただいま別の電話に出ております。 |
| Offering callback | 折り返しお電話させていただいてもよろしいでしょうか。 |
| Taking a message | ご伝言を承ります。 |
| Confirming details | 念のため、お電話番号を確認させていただけますか。 |
| Ending the call | 失礼いたします。 |

### 5.3 When You Cannot Hear

| Situation | Expression |
|-----------|------------|
| Line is bad | 恐れ入りますが、お電話が少々遠いようでして... |
| Asking to repeat | もう一度おっしゃっていただけますでしょうか。 |
| Confirming name | お名前をもう一度お伺いしてもよろしいでしょうか。 |
| Spelling out | 漢字ではどのように書かれますか。 |

### 5.4 Telephone Conversation Example

```
Receptionist: はい、ABC株式会社でございます。
Caller:       お忙しいところ恐れ入ります。
              XYZ商事の田中と申します。
              営業部の佐藤様はいらっしゃいますでしょうか。
Receptionist: 佐藤でございますね。
              少々お待ちくださいませ。
              ......
Receptionist: お待たせいたしました。
              あいにく、佐藤はただいま会議中でございます。
              3時頃には戻る予定でございますが、
              折り返しお電話させていただきましょうか。
Caller:       それでは、お手数ですが折り返しをお願いできますか。
              電話番号は03-9876-5432でございます。
Receptionist: 03-9876-5432の田中様ですね。
              かしこまりました。佐藤に申し伝えます。
Caller:       よろしくお願いいたします。失礼いたします。
Receptionist: 失礼いたします。
```

## 6. Business Card Exchange

### 6.1 名刺交換 (Meishi Koukan) Protocol

Business card exchange is one of the most ritualized aspects of Japanese business culture.
A card represents the person; mishandling it is equivalent to disrespecting the person.

```
+----------------------------------------------------------+
|       BUSINESS CARD EXCHANGE STEPS                       |
+----------------------------------------------------------+
|                                                          |
|  PREPARATION:                                            |
|  • Cards in a 名刺入れ (card case), not a wallet         |
|  • Standing position, no tables between you              |
|  • Junior person or visitor presents first               |
|                                                          |
|  PRESENTING:                                             |
|  1. Hold the card with BOTH hands                        |
|  2. Face the card toward the recipient (readable)        |
|  3. Hold at chest height                                 |
|  4. Say your introduction:                               |
|     「○○会社の○○と申します。                              |
|      よろしくお願いいたします。」                           |
|                                                          |
|  RECEIVING:                                              |
|  1. Receive with BOTH hands                              |
|  2. Say: 「頂戴いたします」(I humbly receive this)         |
|  3. Read the card carefully (DO NOT skip this)           |
|  4. If unsure of name reading, ask politely:             |
|     「お名前はどのようにお読みすればよろしいでしょうか」      |
|                                                          |
|  SIMULTANEOUS EXCHANGE:                                   |
|  • Both parties present at the same time                 |
|  • Extend your card with RIGHT hand                      |
|  • Receive their card with LEFT hand                     |
|  • Then bring both hands to the received card            |
|                                                          |
|  AFTER EXCHANGE:                                          |
|  • Place the card on the table during the meeting        |
|  • Place the highest-ranking person's card on your       |
|    card case, others lined up on the table               |
|  • NEVER write on the card during the meeting            |
|  • Store carefully in your card case afterward            |
+----------------------------------------------------------+
```

### 6.2 Common Mistakes to Avoid

| Mistake | Why It's Wrong |
|---------|---------------|
| Using one hand | Shows lack of respect |
| Putting card in back pocket | Equivalent to sitting on the person |
| Writing on the card in front of them | Defacing their identity |
| Sliding the card across a table | Lazy, impersonal |
| Not reading the card | Shows you do not care who they are |
| Forgetting your card case | Unprepared, unprofessional |
| Playing with the card during the meeting | Disrespectful fidgeting |

## 7. Workplace Hierarchy and Communication

### 7.1 Titles and How to Address People

| Title | Japanese | Usage |
|-------|----------|-------|
| President / CEO | 社長（しゃちょう） | ○○社長 |
| Vice President | 副社長（ふくしゃちょう） | ○○副社長 |
| Director | 取締役（とりしまりやく） | ○○取締役 |
| Department Head | 部長（ぶちょう） | ○○部長 |
| Section Chief | 課長（かちょう） | ○○課長 |
| Team Leader | 係長（かかりちょう） | ○○係長 |
| Senior staff | 主任（しゅにん） | ○○主任 |
| Regular employee | 社員（しゃいん） | ○○さん |

> **Important rule**: When referring to people at YOUR OWN company while speaking to
> an outsider, you drop the honorific title and use the person's name alone:
>
> ✗ 「部長の田中様はただいま外出しております」
> ○ 「田中はただいま外出しております」
>
> Your company's people are "uchi" (内 = inside), and you humble your own side.

### 7.2 内 (Uchi) vs 外 (Soto) in Business

```
+----------------------------------------------------------+
|       UCHI / SOTO DYNAMIC IN BUSINESS                    |
+----------------------------------------------------------+
|                                                          |
|  When talking TO an outsider ABOUT your company:         |
|  ┌───────────────────────────────────────────────┐       |
|  │  YOUR company = UCHI (humble)                 │       |
|  │  • 弊社 (へいしゃ) = our company (humble)      │       |
|  │  • Drop titles for your own people            │       |
|  │  • Use 謙譲語 for your company's actions       │       |
|  │                                               │       |
|  │  THEIR company = SOTO (respectful)            │       |
|  │  • 御社 (おんしゃ) = your company (spoken)     │       |
|  │  • 貴社 (きしゃ) = your company (written)      │       |
|  │  • Use 尊敬語 for their actions                │       |
|  └───────────────────────────────────────────────┘       |
|                                                          |
|  When talking TO your own superior:                      |
|  ┌───────────────────────────────────────────────┐       |
|  │  Use standard keigo (尊敬語/謙譲語)            │       |
|  │  Address by title: ○○部長, ○○課長              │       |
|  └───────────────────────────────────────────────┘       |
+----------------------------------------------------------+
```

### 7.3 報告・連絡・相談 (Horenso)

The foundation of Japanese workplace communication is ほうれんそう:

| Component | Japanese | Meaning | When |
|-----------|----------|---------|------|
| 報告 | ほうこく | Report | After completing a task or at milestones |
| 連絡 | れんらく | Inform / Notify | When changes or new information arise |
| 相談 | そうだん | Consult | Before making decisions, when unsure |

```
Example phrases:

報告: 「先日ご依頼いただいた件について、ご報告いたします。」
       (I would like to report on the matter you asked about the other day.)

連絡: 「○○の件で、一点ご連絡がございます。」
       (I have one notification regarding the ○○ matter.)

相談: 「お忙しいところ恐れ入りますが、
        一点ご相談させていただいてもよろしいでしょうか。」
       (I'm sorry to bother you when busy, but may I consult you on something?)
```

## 8. Common Business Expressions

### 8.1 Four-Character Idioms in Business (四字熟語)

| Idiom | Reading | Meaning | Business Use |
|-------|---------|---------|-------------|
| 一期一会 | いちごいちえ | Once-in-a-lifetime meeting | Value each client encounter |
| 以心伝心 | いしんでんしん | Telepathy / unspoken understanding | Reading the room |
| 臨機応変 | りんきおうへん | Flexible response | Adapting to situations |
| 前代未聞 | ぜんだいみもん | Unprecedented | Describing unusual situations |
| 試行錯誤 | しこうさくご | Trial and error | Describing development process |
| 一石二鳥 | いっせきにちょう | Kill two birds with one stone | Efficient solutions |

### 8.2 Useful Business Set Phrases

| Japanese | English |
|----------|---------|
| ご検討のほど、よろしくお願いいたします | Please give it your consideration |
| 前向きに検討させていただきます | We will consider it positively |
| 善処いたします | We will do our best to handle it |
| 社内で検討の上、改めてご連絡いたします | We will discuss internally and contact you again |
| 大変恐縮ですが... | I am extremely sorry to impose, but... |
| お手数をおかけして申し訳ございません | I apologize for the trouble |
| ご足労いただきありがとうございます | Thank you for coming all this way |
| 今後ともよろしくお願い申し上げます | I look forward to our continued relationship |

> **Reading between the lines**: Some business phrases are indirect refusals:
> - 「前向きに検討させていただきます」 often means "We're not going to do it"
> - 「善処いたします」 often means "Don't hold your breath"
> - 「ちょっと難しいかもしれません」 often means "No"

## 9. Practice Exercises

### Exercise 1: Keigo Correction

Fix the keigo errors in the following sentences.

1. お客様がおっしゃられた通りです。
2. 社長が申していました。（speaking to an external client about your own president）
3. 先生、こちらの資料を拝見してください。
4. 部長にお会いしたいのですが。（speaking to a receptionist about your own manager）

<details>
<summary>Answers</summary>

1. ✗ おっしゃられた → ○ おっしゃった (double honorific: おっしゃる + られる)
2. This is actually CORRECT. When speaking to an outsider about your own president, you use humble language (申す) and drop the title (社長 → last name alone). However, "社長が" should be "○○（last name）が申しておりました。"
3. ✗ 拝見してください → ○ ご覧ください (拝見 is humble — you can't ask someone ELSE to humble themselves)
4. When asking about your OWN company's manager to a receptionist, use no honorific for the manager: ✗ 「部長にお会いしたい」 → ○ 「○○に会いたいのですが」 (However, if asking to see someone at ANOTHER company, use 尊敬語.)

</details>

### Exercise 4: Business Card Exchange

Which of the following behaviors are correct during 名刺交換? Mark each as ○ (correct) or ✗ (incorrect).

1. Receiving a card with one hand while bowing
2. Placing the received card in your card case on the table
3. Writing notes on the back of the card during the meeting
4. Asking how to read a difficult name on the card
5. The more senior person presents their card first
6. Placing the highest-ranking person's card on top of your card case

<details>
<summary>Answers</summary>

1. ✗ — Always use BOTH hands to receive
2. ○ — Placing the highest-ranking person's card on your card case during a meeting is correct
3. ✗ — NEVER write on the card in front of the person
4. ○ — It is polite and expected: 「お名前はどのようにお読みすればよろしいでしょうか」
5. ✗ — The JUNIOR person or VISITOR presents first
6. ○ — The highest-ranking person's card goes on top of your card case, others in a row on the table

</details>

### Exercise 5: Translate to Business Japanese

Convert these casual statements into proper business Japanese:

1. "I'll check it." (to a client)
2. "I don't know." (to your boss)
3. "Can you send it by email?" (to a client)
4. "Sorry for being late." (arriving late to a meeting)
5. "Let me think about it." (to a client making a proposal)

<details>
<summary>Answers</summary>

1. 確認させていただきます。/ 確認いたします。
2. 申し訳ございません、存じ上げません。/ 勉強不足で申し訳ございません。確認いたします。
3. メールにてお送りいただけますでしょうか。
4. 遅くなりまして、大変申し訳ございません。
5. 社内で検討の上、改めてご連絡させていただきます。

</details>
