## 1. How Kanji Work

Kanji are logographic characters borrowed from Chinese. Each character carries **meaning** and **sound(s)**, and they combine to form words.

### 1.1 Types of Kanji by Origin

| Type | Japanese | Description | Example |
|------|----------|-------------|---------|
| Pictographs | 象形文字 (しょうけいもじ) | Stylized pictures of objects | 山 (mountain), 川 (river), 日 (sun) |
| Ideographs | 指事文字 (しじもじ) | Abstract concepts shown visually | 上 (up), 下 (down), 一 (one) |
| Compound ideographs | 会意文字 (かいいもじ) | Two+ elements combined for meaning | 森 (forest) = 木+木+木 (tree+tree+tree) |
| Phono-semantic | 形声文字 (けいせいもじ) | Meaning part + sound part | 清 (clear) = 氵(water) + 青 (blue/sound) |

> **~80% of all kanji are phono-semantic compounds.** This means most kanji have:
> 1. A **radical** (部首) that hints at the **meaning category**
> 2. A **phonetic component** that hints at the **pronunciation**

### 1.2 Kanji Components

```
┌─────────────────────────────────────────┐
│              漢字 (Kanji)                │
│                                         │
│    ┌──────────┐    ┌──────────────┐     │
│    │  部首     │    │  音符         │     │
│    │ (Radical) │    │ (Phonetic)   │     │
│    │ = meaning │    │ = sound hint │     │
│    │  category │    │              │     │
│    └──────────┘    └──────────────┘     │
│                                         │
│    Example: 語 (language/word)          │
│    言 (speech radical) + 吾 (phonetic)  │
└─────────────────────────────────────────┘
```

## 2. Radicals (部首): Building Blocks

Every kanji has exactly **one official radical** (部首) used for dictionary classification. However, kanji may contain multiple recognizable components.

### 2.1 Radical Positions

Radicals appear in specific positions within a kanji:

```
┌───────────────────────────────────────────┐
│  Position        Name        Example      │
├───────────────────────────────────────────┤
│  Left side       へん (偏)    氵(水)→ 海   │
│  Right side      つくり (旁)  刂(刀)→ 割   │
│  Top             かんむり (冠) 宀 → 家     │
│  Bottom          あし (脚)    灬(火)→ 熱   │
│  Enclosure       かまえ (構)  門 → 間      │
│  Left+bottom     にょう (繞)  辶 → 道      │
│  Top-left        たれ (垂)    广 → 店      │
└───────────────────────────────────────────┘
```

## 4. On'yomi vs. Kun'yomi

Each kanji may have multiple readings. The two main categories are:

### 4.1 On'yomi (音読み) — Sino-Japanese Reading

- Derived from Chinese pronunciation
- Used primarily in **kanji compounds** (two or more kanji together)
- Written in **katakana** in dictionaries

### 4.2 Kun'yomi (訓読み) — Native Japanese Reading

- Original Japanese word mapped to the kanji
- Used when a kanji **stands alone** or is followed by **okurigana** (hiragana endings)
- Written in **hiragana** in dictionaries

### 4.3 Examples

| Kanji | On'yomi | Kun'yomi | Compound (On) | Standalone (Kun) |
|-------|---------|----------|---------------|-----------------|
| 山 | サン | やま | 山脈 (さんみゃく, mountain range) | 山 (やま, mountain) |
| 水 | スイ | みず | 水曜 (すいよう, Wednesday) | 水 (みず, water) |
| 人 | ジン・ニン | ひと | 日本人 (にほんじん, Japanese person) | 人 (ひと, person) |
| 大 | ダイ・タイ | おお(きい) | 大学 (だいがく, university) | 大きい (おおきい, big) |
| 食 | ショク | た(べる) | 食事 (しょくじ, meal) | 食べる (たべる, to eat) |
| 生 | セイ・ショウ | い(きる)・う(まれる)・なま | 学生 (がくせい, student) | 生きる (いきる, to live) |

> **生** has the most readings of any common kanji:
> On: セイ, ショウ | Kun: い(きる), う(まれる), お(う), は(える), なま, き
> This is exceptional — most kanji have 1-2 On readings and 1-2 Kun readings.

## 5. Reading Selection Rules

### 5.1 General Guidelines

```
+----------------------------------------------------------+
|           WHICH READING TO USE?                           |
+----------------------------------------------------------+
|                                                          |
|  COMPOUND (kanji + kanji):                               |
|  → Usually ON'YOMI                                       |
|  e.g., 学校 (がっこう), 電話 (でんわ)                     |
|                                                          |
|  STANDALONE (kanji alone or with okurigana):             |
|  → Usually KUN'YOMI                                      |
|  e.g., 花 (はな), 食べる (たべる)                         |
|                                                          |
|  EXCEPTIONS (must memorize):                             |
|  → Some compounds use kun'yomi:                          |
|    手紙 (てがみ, letter), 場所 (ばしょ, place)            |
|  → Some standalone use on'yomi:                          |
|    本 (ほん, book), 駅 (えき, station)                    |
+----------------------------------------------------------+
```

### 5.2 Special Reading Types

| Type | Japanese | Description | Example |
|------|----------|-------------|---------|
| Jukujikun | 熟字訓 | Compound read as one native word | 大人 = おとな (not だいじん) |
| Nanori | 名乗り | Special readings for names | 美 = よし in names |
| Ateji | 当て字 | Kanji chosen for sound, not meaning | 寿司 = すし |

## 6. Kanji Compounds (熟語) Patterns

Understanding how kanji combine helps you predict the meaning of new words without looking them up.

### 6.1 Formation Patterns

| Pattern | Structure | Examples |
|---------|-----------|---------|
| **Similar meaning** | A + B (both mean similar things) | 道路 (road+road = road), 森林 (forest+grove = forest) |
| **Opposite meaning** | A + B (opposites combined) | 大小 (big+small = size), 売買 (sell+buy = trade) |
| **Modifier + Modified** | Adj/Adv + Noun/Verb | 新聞 (new+hear = newspaper), 高速 (high+speed = high-speed) |
| **Verb + Object** | Action + Target | 読書 (read+book = reading), 登山 (climb+mountain = mountaineering) |
| **Subject + Verb** | Actor/Thing + Action | 地震 (ground+shake = earthquake), 日没 (sun+sink = sunset) |
| **Negation** | 不/無/非/未 + Word | 不可能 (not+possible = impossible), 無料 (no+fee = free) |

### 6.2 Common Prefixes

| Prefix | Meaning | Examples |
|--------|---------|---------|
| 不 (ふ/ぶ) | not, un- | 不安 (uneasy), 不便 (inconvenient), 不足 (insufficient) |
| 無 (む/ぶ) | without, -less | 無料 (free), 無理 (unreasonable), 無視 (ignore) |
| 非 (ひ) | non-, un- | 非常 (emergency), 非公式 (unofficial) |
| 未 (み) | not yet | 未来 (future), 未定 (undecided), 未満 (less than) |
| 再 (さい) | re-, again | 再開 (reopen), 再生 (regeneration), 再利用 (reuse) |
| 超 (ちょう) | super-, ultra- | 超人 (superman), 超高速 (ultra-high-speed) |
| 最 (さい) | most | 最高 (highest/best), 最初 (first), 最新 (latest) |
| 各 (かく) | each | 各国 (each country), 各自 (each person) |

### 6.3 Common Suffixes

| Suffix | Meaning | Examples |
|--------|---------|---------|
| 的 (てき) | -tic, -cal, -like | 国際的 (international), 具体的 (concrete) |
| 化 (か) | -ize, -ification | 国際化 (internationalization), 近代化 (modernization) |
| 性 (せい) | -ity, -ness, nature | 可能性 (possibility), 重要性 (importance) |
| 者 (しゃ) | person who ~ | 研究者 (researcher), 記者 (journalist) |
| 学 (がく) | study of ~, -ology | 科学 (science), 文学 (literature), 医学 (medicine) |
| 力 (りょく) | power of ~, ability | 能力 (ability), 努力 (effort), 体力 (stamina) |
| 感 (かん) | feeling of ~ | 安心感 (sense of security), 達成感 (sense of achievement) |
| 式 (しき) | style, ceremony | 日本式 (Japanese-style), 入学式 (entrance ceremony) |

## 8. Study Strategies

### 8.1 The Radical Method

**Strategy:** Learn radicals first, then build kanji from components.

```
Step 1: Learn the radical    氵= water
Step 2: See it in kanji      海 (sea), 泳 (swim), 洗 (wash), 湖 (lake)
Step 3: Create connections   "All water-related kanji have 氵"
Step 4: Predict new kanji    汚 (dirty) → water + ? → polluted water?
                             Actually: dirty, polluted ✓
```

**Advantages:**
- Reduces 2,000+ kanji to ~200 building blocks
- Makes unfamiliar kanji analyzable
- Helps remember meaning even if you forget the reading

### 8.2 Spaced Repetition (間隔反復)

**Strategy:** Review kanji at increasing intervals to move them into long-term memory.

```
+---------------------------------------------------+
|  SPACED REPETITION SCHEDULE                        |
+---------------------------------------------------+
|                                                   |
|  Day 1:  Learn 10 new kanji                       |
|  Day 2:  Review Day 1 + learn 10 new              |
|  Day 4:  Review Day 1-2 + learn 10 new            |
|  Day 7:  Review Day 1-4 + learn 10 new            |
|  Day 14: Review Day 1-7 + learn 10 new            |
|  Day 30: Review Day 1-14 + learn 10 new           |
|                                                   |
|  Tools: Anki, WaniKani, Kanji Study app           |
+---------------------------------------------------+
```

### 8.3 Context Learning

**Strategy:** Learn kanji through vocabulary words, not in isolation.

| Approach | Method | Example |
|----------|--------|---------|
| Bad | Memorize kanji alone | 食 = eat |
| Good | Learn in compound | 食事 (しょくじ, meal), 食堂 (しょくどう, cafeteria) |
| Best | Learn in sentence | 毎日食堂で食事をする。(I eat at the cafeteria every day.) |

### 8.4 Story/Mnemonic Method

**Strategy:** Create memorable stories linking kanji components to meaning.

| Kanji | Components | Mnemonic |
|-------|-----------|----------|
| 休 | 亻(person) + 木 (tree) | A **person** resting against a **tree** = rest |
| 好 | 女 (woman) + 子 (child) | A **woman** with her **child** = like/love |
| 明 | 日 (sun) + 月 (moon) | **Sun** and **moon** together = bright |
| 岩 | 山 (mountain) + 石 (stone) | **Stones** on a **mountain** = rock/boulder |
| 鳴 | 口 (mouth) + 鳥 (bird) | A **bird** opening its **mouth** = to chirp/cry |

### 8.5 Kanji Journal Method

**Strategy:** Write each new kanji with the following information:

```
┌──────────────────────────────────────┐
│  Kanji: 経                           │
│  Radical: 糸 (thread)               │
│  On: ケイ / キョウ                   │
│  Kun: へ(る) / た(つ)               │
│  Meaning: pass through, experience  │
│  Compounds: 経験 (experience)        │
│             経済 (economy)           │
│             経営 (management)        │
│  Sentence: この経験を通じて学んだ。  │
│  (I learned through this experience)│
│  Mnemonic: Thread (糸) going        │
│  through a loom — passing through   │
└──────────────────────────────────────┘
```

### Exercise 2: On'yomi or Kun'yomi?

For each word, determine whether the kanji use On'yomi or Kun'yomi.

1. 山 (やま) — mountain
2. 登山 (とざん) — mountaineering
3. 食べる (たべる) — to eat
4. 食事 (しょくじ) — meal
5. 大きい (おおきい) — big
6. 大学 (だいがく) — university
7. 花 (はな) — flower
8. 花瓶 (かびん) — vase

<details>
<summary>Answers</summary>

1. **Kun'yomi** — standalone kanji: やま
2. **On'yomi** — kanji compound: とざん (ト + ザン)
3. **Kun'yomi** — with okurigana: た(べる)
4. **On'yomi** — kanji compound: ショク + ジ
5. **Kun'yomi** — with okurigana: おお(きい)
6. **On'yomi** — kanji compound: ダイ + ガク
7. **Kun'yomi** — standalone: はな
8. **On'yomi** — kanji compound: カ + ビン

</details>

### Exercise 3: Compound Meaning Prediction

Using the compound formation patterns, predict the meaning of each word from its kanji components.

1. 火山 (fire + mountain) = ?

3. 出口 (exit + mouth/opening) = ?
4. 不安 (not + peaceful) = ?

### Exercise 4: Suffix and Prefix Practice

Add the correct prefix or suffix to create the target meaning.

1. 可能 + _______ = possibility (性/化/的)
2. _______ + 料 = free of charge (不/無/非)
3. 国際 + _______ = international (adj.) (性/化/的)
4. 研究 + _______ = researcher (者/学/力)
5. _______ + 定 = undecided (不/無/未)
6. 近代 + _______ = modernization (性/化/的)

<details>
<summary>Answers</summary>

</details>
