## 1. What is the Te-form?

The て-form (te-kei) is one of the most versatile and frequently used verb forms in Japanese. By itself, it does not express tense or politeness. Instead, it serves as a **connector** that attaches to helper verbs and expressions to create new meanings.

> **The て-form is a Swiss Army knife**
>
> て + ください      → please do...
> て + いる          → is doing... / has done...
> て + もいい        → may I...? / it's okay to...
> て + はいけない    → must not...
> て + ある          → something has been done (state)
> て + て + て...    → did X, then Y, then Z...
>
> The て-form is THE key to unlocking intermediate Japanese.
> Master it, and dozens of grammar patterns become available.

## 2. Te-form Formation Rules

The て-form is derived from the dictionary form of the verb. The rules differ by verb group.

### 2.1 Group 2: Ichidan Verbs (一段動詞) — The Easy Ones

Drop the final る and add て.

| Dictionary Form | Meaning | て-form |
|----------------|---------|---------|
| 食べる (taberu) | to eat | 食べ**て** |
| 見る (miru) | to see | 見**て** |
| 起きる (okiru) | to wake up | 起き**て** |
| 寝る (neru) | to sleep | 寝**て** |
| 教える (oshieru) | to teach | 教え**て** |

### 2.2 Group 1: Godan Verbs (五段動詞) — The Tricky Ones

The ending changes based on the final kana of the dictionary form. This is where the famous **te-form song** helps.

| Dictionary Ending | Change | Example | て-form |
|-------------------|--------|---------|---------|
| う, つ, る | → って | 買う (kau) | 買**って** |
| | | 待つ (matsu) | 待**って** |
| | | 帰る (kaeru) | 帰**って** |
| む, ぶ, ぬ | → んで | 読む (yomu) | 読**んで** |
| | | 遊ぶ (asobu) | 遊**んで** |
| | | 死ぬ (shinu) | 死**んで** |
| く | → いて | 書く (kaku) | 書**いて** |
| ぐ | → いで | 泳ぐ (oyogu) | 泳**いで** |
| す | → して | 話す (hanasu) | 話**して** |

### 2.3 The Te-form Song (て形の歌)

A mnemonic to remember the godan て-form rules:

> **て-FORM SONG**
>
> ♪ う、つ、る → って  って  って
> (u, tsu, ru → tte)
>
> ♪ む、ぶ、ぬ → んで  んで  んで
> (mu, bu, nu → nde)
>
> ♪ く → いて     ぐ → いで
> (ku → ite)    (gu → ide)
>
> ♪ す → して
> (su → shite)
>
> Sing it, memorize it, love it.

### 2.4 Group 3: Irregular Verbs

| Dictionary Form | Meaning | て-form |
|----------------|---------|---------|
| する (suru) | to do | **して** |
| 来る (kuru) | to come | **来て** (kite) |

### 2.5 Important Exception

行く (iku — to go) is an exception among く-ending verbs:

```
行く → 行って  (NOT 行いて)
iku → itte     (NOT iite)
```

## 3. て + ください — Polite Requests

### 3.1 Basic Usage

て + ください (kudasai) makes a polite request meaning "please do..."

```
待ってください。
Matte kudasai.
Please wait.

ここに座ってください。
Koko ni suwatte kudasai.
Please sit here.

名前を書いてください。
Namae o kaite kudasai.
Please write your name.

もう一度言ってください。
Mō ichido itte kudasai.
Please say it one more time.

ゆっくり話してください。
Yukkuri hanashite kudasai.
Please speak slowly.
```

### 3.2 Negative Requests

To ask someone NOT to do something, use ないで + ください.

```
触らないでください。
Sawaranaide kudasai.
Please don't touch.

心配しないでください。
Shinpai shinaide kudasai.
Please don't worry.

ここで写真を撮らないでください。
Koko de shashin o toranaide kudasai.
Please don't take photos here.
```

### 3.3 Politeness Levels

> **Request Politeness Levels**
>
> 待って。                Very casual (between close friends)
> 待ってください。        Polite (standard)
> 待っていただけますか。  Very polite / humble
> お待ちください。        Formal / honorific
>
> For now, master ください. The others come in Lesson 14.

## 4. て + いる — Progressive and States

て + いる (iru) is one of the most used structures in Japanese. It has several meanings depending on the verb type.

### 4.1 Ongoing Action (Progressive)

With action verbs, て + いる indicates an action currently in progress, like English "-ing."

```
今、本を読んでいます。
Ima, hon o yonde imasu.
I am reading a book right now.

田中さんは電話で話している。
Tanaka-san wa denwa de hanashite iru.
Tanaka-san is talking on the phone.

子供たちが公園で遊んでいます。
Kodomo-tachi ga kōen de asonde imasu.
The children are playing in the park.
```

### 4.2 Resultant State

With certain verbs (change-of-state verbs), て + いる describes the **current state** resulting from a completed action.

```
結婚している。
Kekkon shite iru.
I am married. (not "I am marrying")

知っている。
Shitte iru.
I know. (not "I am knowing")

窓が開いている。
Mado ga aite iru.
The window is open. (result of being opened)

田中さんはもう来ている。
Tanaka-san wa mō kite iru.
Tanaka-san has already come. (is already here)
```

### 4.3 Habitual Action

て + いる can also describe habitual or regular actions.

```
毎日コーヒーを飲んでいます。
Mainichi kōhii o nonde imasu.
I drink coffee every day. (habitual)

東京に住んでいます。
Tōkyō ni sunde imasu.
I live in Tokyo. (ongoing state)

銀行で働いている。
Ginkō de hataraite iru.
I work at a bank. (regular occupation)
```

### 4.4 Conjugation of ている

| Form | Casual | Polite |
|------|--------|--------|
| Present (+) | 食べている | 食べています |
| Present (-) | 食べていない | 食べていません |
| Past (+) | 食べていた | 食べていました |
| Past (-) | 食べていなかった | 食べていませんでした |

> **Spoken contraction**: ている is often shortened to てる in casual speech.
> - 食べている → 食べてる
> - 読んでいる → 読んでる

### 4.5 知っている vs 知らない

An important special case: the negative of 知っている (to know) is 知**ら**ない (NOT 知っていない).

```
知っています。     I know.
知りません。       I don't know. (NOT 知っていません)
```

## 5. て + もいい — Permission

### 5.1 Asking Permission

て + もいいですか asks "May I...?" or "Is it okay if I...?"

```
ここに座ってもいいですか。
Koko ni suwatte mo ii desu ka.
May I sit here?

写真を撮ってもいいですか。
Shashin o totte mo ii desu ka.
May I take a photo?

窓を開けてもいいですか。
Mado o akete mo ii desu ka.
May I open the window?

この本を借りてもいいですか。
Kono hon o karite mo ii desu ka.
May I borrow this book?
```

### 5.2 Granting Permission

```
はい、いいですよ。           Yes, it's fine.
はい、どうぞ。               Yes, please go ahead.
もちろん。                   Of course.
食べてもいいですよ。         You may eat.
```

### 5.3 Denying Permission

To deny permission, use て + は + いけない (see Section 6) or softer alternatives.

```
すみません、ちょっと...       Sorry, that's a bit... (soft refusal)
ここでは撮れません。         You can't take (photos) here. (indirect)
```

## 6. て + はいけない — Prohibition

### 6.1 Basic Usage

て + はいけない (wa ikenai) or て + はいけません (polite) means "must not" or "it is not allowed."

```
ここでタバコを吸ってはいけません。
Koko de tabako o sutte wa ikemasen.
You must not smoke here.

授業中に携帯を使ってはいけない。
Jugyō-chū ni keitai o tsukatte wa ikenai.
You must not use your phone during class.

ここで走ってはいけません。
Koko de hashitte wa ikemasen.
You must not run here.

嘘をついてはいけない。
Uso o tsuite wa ikenai.
You must not tell lies.
```

### 6.2 Permission vs Prohibition Summary

> **PERMISSION vs PROHIBITION**
>
> て + もいいですか         May I ...?
> て + もいいですよ         You may ... / It's okay to ...
>
> て + はいけません         You must not ...
> て + はだめです           You must not ... (slightly casual)
>
> Example pair:
> - **Q**: ここで食べてもいいですか。  May I eat here?
> - **A**: はい、食べてもいいですよ。  Yes, you may eat here.
> - **A**: いいえ、食べてはいけません。No, you must not eat here.
>
> ※ だめ (dame) is a more colloquial alternative to
> いけない for prohibition.
> 食べてはだめです。= You must not eat.

## 7. て + ある — Resultant State

### 7.1 Basic Usage

て + ある describes a state that **resulted from someone's intentional action**. It focuses on the current state, not the action itself.

```
窓が開けてある。
Mado ga akete aru.
The window has been opened. (someone opened it intentionally, and it's still open)

テーブルの上に本が置いてある。
Tēburu no ue ni hon ga oite aru.
A book has been placed on the table. (someone put it there)

壁に絵が掛けてある。
Kabe ni e ga kakete aru.
A painting has been hung on the wall.
```

### 7.2 ている vs てある

> **ている vs てある**
>
> 窓が開いている。
> The window is open. (describes the state, no agent)
> - **Uses intransitive verb**: 開く (aku — to open by itself)
>
> 窓が開けてある。
> The window has been opened. (someone did it on purpose)
> - **Uses transitive verb**: 開ける (akeru — to open something)
>
> Key difference:
> ている = state (natural or otherwise)
> てある = state resulting from someone's INTENTIONAL action

### 7.3 More Examples

```
ドアが閉めてある。         The door has been closed (by someone).
名前が書いてある。         A name has been written (by someone).
料理が作ってある。         The food has been prepared.
ホテルが予約してある。     A hotel has been reserved.
```

## 8. Chaining Actions with て

### 8.1 Sequential Actions

The て-form connects actions in sequence, similar to "and" or "then" in English.

```
朝起きて、顔を洗って、朝ご飯を食べます。
Asa okite, kao o aratte, asagohan o tabemasu.
I wake up in the morning, wash my face, and eat breakfast.

図書館に行って、本を借りました。
Toshokan ni itte, hon o karimashita.
I went to the library and borrowed a book.

電車に乗って、会社に行きます。
Densha ni notte, kaisha ni ikimasu.
I take the train and go to work.
```

> **Note**: Only the **final** verb carries the tense and politeness level. All preceding verbs use the て-form.

### 8.2 Manner or Method

The て-form can describe the manner in which an action is done.

```
歩いて学校に行く。
Aruite gakkō ni iku.
I walk to school. (go by walking)

座って食べてください。
Suwatte tabete kudasai.
Please eat while sitting down.

急いで帰った。
Isoide kaetta.
I hurried home. (returned in a hurry)
```

### 8.3 Cause and Effect

```
風邪をひいて、学校を休んだ。
Kaze o hiite, gakkō o yasunda.
I caught a cold and was absent from school.

お金がなくて、買えなかった。
Okane ga nakute, kaenakatta.
I didn't have money and couldn't buy it.
```

### 8.4 Chain Length

There is no strict limit on how many て-forms you can chain, but 3-4 is typical. More than that becomes unwieldy.

```
朝起きて、シャワーを浴びて、朝ご飯を食べて、学校に行きました。
Asa okite, shawā o abite, asagohan o tabete, gakkō ni ikimashita.
I woke up, took a shower, ate breakfast, and went to school.
```

## 9. Te-form of Adjectives and Nouns

Adjectives and nouns also have て-forms, used for connecting (Lesson 06 review) and in some grammar patterns.

### 9.1 い-Adjectives

Drop い, add くて:

```
大きい → 大きくて   (big, and...)
おいしい → おいしくて (delicious, and...)
いい → よくて (good, and...) ← irregular
```

### 9.2 な-Adjectives

Add で after the stem:

```
静か → 静かで (quiet, and...)
きれい → きれいで (pretty, and...)
元気 → 元気で (healthy, and...)
```

### 9.3 Nouns

Add で after the noun:

```
学生 → 学生で (being a student, and...)
日本人 → 日本人で (being Japanese, and...)
```

### 9.4 Examples in Sentences

```
この部屋は広くて、明るいです。
Kono heya wa hirokute, akarui desu.
This room is spacious and bright.

田中さんは日本人で、東京に住んでいます。
Tanaka-san wa nihonjin de, Tōkyō ni sunde imasu.
Tanaka-san is Japanese and lives in Tokyo.

あの人は親切で、面白い人です。
Ano hito wa shinsetsu de, omoshiroi hito desu.
That person is kind and interesting.
```

> - **Godan む/ぶ/ぬ**: → んで     読む → 読んで
> - **Godan く**: → いて           書く → 書いて
> - **Godan ぐ**: → いで           泳ぐ → 泳いで
> - **Godan す**: → して           話す → 話して
> - **Exception**: 行く → 行って
> - **Irregular**: する → して  /  来る → 来て
>
> GRAMMAR PATTERNS
> て + ください        Please do...  (待ってください)
> ないで + ください    Please don't... (触らないでください)
> て + いる            Ongoing/state/habit (読んでいる)
> て + もいいですか    May I...? (撮ってもいいですか)
> て + はいけない      Must not... (走ってはいけない)
> て + ある            Resultant state (書いてある)
> て + て + て...      Chain actions (起きて、食べて...)
>
> SPECIAL NOTES
> 知っている → 知らない (not 知っていない)
> ている → てる (casual contraction)
>
