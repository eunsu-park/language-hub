## 1. Japanese Verb Groups Review

Before learning advanced conjugations, ensure you can identify the three verb groups:

| Group | Name | Identification | Examples |
|-------|------|---------------|----------|
| Group 1 | 五段動詞 (Godan) | U-verbs: dictionary form ends in -u row kana (except -iru/-eru) | 書く, 話す, 読む, 飲む, 行く |
| Group 2 | 一段動詞 (Ichidan) | Ru-verbs: dictionary form ends in -iru or -eru | 食べる, 見る, 起きる, 寝る |
| Group 3 | 不規則動詞 (Irregular) | Only two verbs | する, 来る (くる) |

> **Warning**: Some verbs look like Group 2 but are actually Group 1:
> 帰る (かえる — to return), 知る (しる — to know), 走る (はしる — to run),
> 切る (きる — to cut), 入る (はいる — to enter).
> These are called "disguised Godan verbs."

## 2. Potential Form (可能形)

The potential form expresses **ability** — "can do" or "be able to do."

### Formation Rules

| Group | Rule | Example |
|-------|------|---------|
| Group 1 (Godan) | Change final -u to -e + る | 書く → 書**ける**, 読む → 読**める**, 話す → 話**せる** |
| Group 2 (Ichidan) | Drop -る, add -られる | 食べる → 食べ**られる**, 見る → 見**られる** |
| Group 3 (Irregular) | Memorize | する → **できる**, 来る → **来られる** (こられる) |

### Complete Conjugation: Group 1 (書く → 書ける)

| Form | Japanese | Romaji |
|------|----------|--------|
| Dictionary | 書ける | kakeru |
| Negative | 書けない | kakenai |
| Polite | 書けます | kakemasu |
| Past | 書けた | kaketa |
| Past Negative | 書けなかった | kakenakatta |

### Complete Conjugation: Group 2 (食べる → 食べられる)

| Form | Japanese | Romaji |
|------|----------|--------|
| Dictionary | 食べられる | taberareru |
| Negative | 食べられない | taberarenai |
| Polite | 食べられます | taberaremasu |
| Past | 食べられた | taberareta |
| Past Negative | 食べられなかった | taberarenakatta |

### Particle Change: を → が

In potential sentences, the object particle を is typically replaced by が:

```
Standard:    日本語を話す。        (I speak Japanese.)
Potential:   日本語が話せる。      (I can speak Japanese.)
```

> **Note**: In casual speech, を is sometimes kept, but が is the grammatically standard choice.

### ら抜き言葉 (Ra-nuki Kotoba) — Colloquial Shortening

In casual spoken Japanese, Group 2 verbs often drop ら from the potential form:

| Standard | Colloquial (ら抜き) | Meaning |
|----------|-------------------|---------|
| 食べ**ら**れる | 食べれる | can eat |
| 見**ら**れる | 見れる | can see |
| 起き**ら**れる | 起きれる | can wake up |

> This is extremely common in everyday speech, especially among younger speakers.
> However, it is considered non-standard in formal writing and JLPT exams.

### Example Sentences

1. **漢字が読めますか。** — Can you read kanji?
2. **明日は来られません。** — I cannot come tomorrow.
3. **この店では日本語が話せる。** — You can speak Japanese at this store.
4. **子供の時、泳げなかった。** — When I was a child, I couldn't swim.
5. **一人でできると思う。** — I think I can do it by myself.

## 3. Volitional Form (意志形)

The volitional form expresses **intention, suggestion, or invitation** — "let's do," "shall we," or "I'll do."

### Formation Rules

| Group | Rule | Example |
|-------|------|---------|
| Group 1 (Godan) | Change final -u to -o + う | 書く → 書**こう**, 読む → 読**もう**, 行く → 行**こう** |
| Group 2 (Ichidan) | Drop -る, add -よう | 食べる → 食べ**よう**, 見る → 見**よう** |
| Group 3 (Irregular) | Memorize | する → **しよう**, 来る → **来よう** (こよう) |

### Group 1 Volitional: All Endings

| Final Kana | Change | Example |
|-----------|--------|---------|
| く → こう | 書く → 書こう | |
| す → そう | 話す → 話そう | |
| つ → とう | 待つ → 待とう | |
| ぬ → のう | 死ぬ → 死のう | |
| ぶ → ぼう | 遊ぶ → 遊ぼう | |
| む → もう | 読む → 読もう | |
| る → ろう | 帰る → 帰ろう | |
| う → おう | 買う → 買おう | |
| ぐ → ごう | 泳ぐ → 泳ごう | |

### Polite Volitional

Use ましょう with the masu-stem:

| Casual | Polite | Meaning |
|--------|--------|---------|
| 行こう | 行きましょう | Let's go |
| 食べよう | 食べましょう | Let's eat |
| しよう | しましょう | Let's do |

### Uses of the Volitional

#### A. Suggestion / Invitation ("Let's...")

> **映画を見よう。** — Let's watch a movie.
> **一緒に勉強しましょう。** — Let's study together.

#### B. Personal Intention ("I'll...")

> **明日から早く起きよう。** — I'll start getting up early from tomorrow.
> **もっと頑張ろう。** — I'll try harder.

#### C. Volitional + と思う ("I think I'll...")

This is a very common pattern for expressing plans:

> **来年、日本に行こうと思っている。** — I'm thinking of going to Japan next year.
> **新しい仕事を探そうと思う。** — I think I'll look for a new job.

> **Grammar point**: Use と思っている (ongoing thought/plan), not と思う (momentary thought), for sustained intentions.

### Example Sentences

1. **今日は早く寝よう。** — I'll go to bed early today.
2. **週末に旅行しようか。** — Shall we travel on the weekend?
3. **何を食べましょうか。** — What shall we eat?
4. **日本語をもっと勉強しようと思っています。** — I'm thinking of studying Japanese more.
5. **帰ろう。もう遅いよ。** — Let's go home. It's already late.

## 4. Passive Form (受身形)

The passive form expresses that the subject **receives** an action — "is done to" or "was done by."

### Formation Rules

| Group | Rule | Example |
|-------|------|---------|
| Group 1 (Godan) | Change final -u to -a + れる | 書く → 書**かれる**, 読む → 読**まれる**, 話す → 話**される** |
| Group 2 (Ichidan) | Drop -る, add -られる | 食べる → 食べ**られる**, 見る → 見**られる** |
| Group 3 (Irregular) | Memorize | する → **される**, 来る → **来られる** (こられる) |

> **Note**: Group 2 passive and potential forms are identical (食べられる).
> Context distinguishes them.

### Group 1 Passive: All Endings

| Final Kana | Change | Example |
|-----------|--------|---------|
| く → かれる | 書く → 書かれる | |
| す → される | 話す → 話される | |
| つ → たれる | 待つ → 待たれる | |
| ぬ → なれる | 死ぬ → 死なれる | |
| ぶ → ばれる | 呼ぶ → 呼ばれる | |
| む → まれる | 読む → 読まれる | |
| る → られる | 取る → 取られる | |
| う → われる | 言う → 言われる | |
| ぐ → がれる | 脱ぐ → 脱がれる | |

### Particle Structure in Passive Sentences

```
Active:   先生が     学生を    褒めた。
          Teacher   student   praised.
          (The teacher praised the student.)

Passive:  学生が     先生に    褒められた。
          Student   teacher   was praised by.
          (The student was praised by the teacher.)
```

Key particle changes:
- The receiver of the action becomes the subject (が)
- The doer of the action is marked by に

### Types of Japanese Passive

#### A. Direct Passive (Standard)

The subject receives the action directly — equivalent to English passive:

> **この本は多くの人に読まれている。** — This book is read by many people.
> **新しい学校が建てられた。** — A new school was built.

#### B. Indirect Passive (Adversity Passive / 迷惑の受身)

Unique to Japanese. Expresses that someone is **negatively affected** by an action:

> **電車で足を踏まれた。** — I had my foot stepped on in the train.
> **雨に降られた。** — I got rained on. (The rain fell on me — annoying!)
> **隣の人にたばこを吸われた。** — The person next to me smoked. (And it bothered me!)

> **Key insight**: The indirect passive often conveys that the speaker is a **victim** of the action. This has no direct equivalent in English.

### Example Sentences

1. **先生に名前を呼ばれた。** — I was called by name by the teacher.
2. **この歌は世界中で歌われている。** — This song is sung around the world.
3. **友達に秘密を言われた。** — My friend told my secret. (And I'm unhappy about it.)
4. **彼は皆に尊敬されている。** — He is respected by everyone.
5. **泥棒に財布を盗まれた。** — I had my wallet stolen by a thief.

## 5. Causative Form (使役形)

The causative form expresses **making** or **letting** someone do something.

### Formation Rules

| Group | Rule | Example |
|-------|------|---------|
| Group 1 (Godan) | Change final -u to -a + せる | 書く → 書**かせる**, 読む → 読**ませる**, 行く → 行**かせる** |
| Group 2 (Ichidan) | Drop -る, add -させる | 食べる → 食べ**させる**, 見る → 見**させる** |
| Group 3 (Irregular) | Memorize | する → **させる**, 来る → **来させる** (こさせる) |

### Group 1 Causative: All Endings

| Final Kana | Change | Example |
|-----------|--------|---------|
| く → かせる | 書く → 書かせる | |
| す → させる | 話す → 話させる | |
| つ → たせる | 待つ → 待たせる | |
| ぬ → なせる | 死ぬ → 死なせる | |
| ぶ → ばせる | 遊ぶ → 遊ばせる | |
| む → ませる | 読む → 読ませる | |
| る → らせる | 帰る → 帰らせる | |
| う → わせる | 歌う → 歌わせる | |
| ぐ → がせる | 泳ぐ → 泳がせる | |

### Particle Structure

#### "Make someone do" (Compulsion)

The person made to act is marked by **を**:

> **先生は学生**を**走らせた。** — The teacher made the students run.
> **母は子供**を**勉強させた。** — The mother made the child study.

#### "Let someone do" (Permission)

The person allowed to act is marked by **に**:

> **先生は学生**に**好きな本を読ませた。** — The teacher let the students read books they liked.
> **母は子供**に**遊ばせた。** — The mother let the child play.

> **Rule of thumb**: を = compulsion (make), に = permission (let).
> When the main verb is transitive and already has a を object, use に for the person even in compulsion contexts to avoid double を.

### させてください — "Please let me..."

A very useful polite request form:

> **説明させてください。** — Please let me explain.
> **自己紹介させていただきます。** — Allow me to introduce myself. (Humble)
> **考えさせてください。** — Please let me think about it.

### Example Sentences

1. **部長は田中さんを出張させた。** — The department head made Tanaka go on a business trip.
2. **子供にゲームをさせてあげた。** — I let the child play games.
3. **もう少し考えさせてください。** — Please let me think a little more.
4. **彼女は彼を3時間も待たせた。** — She made him wait for three whole hours.
5. **先生は学生に自由に話させた。** — The teacher let the students speak freely.

## 6. Causative-Passive Form (使役受身形)

The causative-passive combines causative and passive: "to be **made** to do something." It expresses compulsion from the perspective of the person being forced.

### Formation Rules

| Group | Rule | Example |
|-------|------|---------|
| Group 1 (Godan) | Change final -u to -a + せられる | 書く → 書**かせられる**, 読む → 読**ませられる** |
| Group 2 (Ichidan) | Drop -る, add -させられる | 食べる → 食べ**させられる** |
| Group 3 (Irregular) | Memorize | する → **させられる**, 来る → **来させられる** (こさせられる) |

### Shortened Form (Group 1 only)

In spoken Japanese, Group 1 causative-passive is often shortened:

| Full Form | Shortened Form | Meaning |
|-----------|---------------|---------|
| 書かせられる | 書かされる | be made to write |
| 読ませられる | 読まされる | be made to read |
| 飲ませられる | 飲まされる | be made to drink |
| 行かせられる | 行かされる | be made to go |
| 待たせられる | 待たされる | be made to wait |

> **Exception**: Verbs ending in す (話す, etc.) do **not** have a shortened form.
> 話させられる does NOT become *話さされる.

### Particle Structure

```
[Person forced]は  [Person forcing]に  [action]させられる

私は    先生に    走らせられた。
I       teacher   was made to run.
(I was made to run by the teacher.)
```

### Example Sentences

1. **毎日野菜を食べさせられた。** — I was made to eat vegetables every day.
2. **子供の頃、ピアノを練習させられた。** — As a child, I was made to practice piano.
3. **上司に残業させられた。** — I was made to work overtime by my boss.
4. **嫌いな歌を歌わされた。** — I was made to sing a song I didn't like. (shortened)
5. **3時間も待たされた。** — I was made to wait for three whole hours. (shortened)

## 8. Common Mistakes and Pitfalls

### Mistake 1: Confusing Potential and Passive for Group 2

Both 食べられる (potential) and 食べられる (passive) look identical.

```
食べられる = can eat (potential) OR is eaten (passive)
Context tells you which:
  ○ 刺身が食べられる。         (I can eat sashimi.) — potential
  ○ 刺身は世界中で食べられている。(Sashimi is eaten worldwide.) — passive
```

### Mistake 2: Wrong Verb Group Identification

```
  ✗ 帰る → 帰られる  (treating as Group 2)
  ○ 帰る → 帰れる    (Group 1 potential: かえ-る → かえ-れる)
```

### Mistake 3: Double を in Causative

```
  ✗ 先生は学生を本を読ませた。  (Two を particles!)
  ○ 先生は学生に本を読ませた。  (Use に for the person)
```

### Mistake 4: Shortened Causative-Passive with す-Verbs

```
  ✗ 話さされる  (incorrect — sounds awkward)
  ○ 話させられる (す-verbs keep the full form)
```

## 9. Practice Exercises

### Exercise 1: Form the Potential

Convert to potential form.

1. 日本語を話す → 日本語が ________
2. この漢字を読む → この漢字が ________
3. 朝早く起きる → 朝早く ________
4. 明日来る → 明日 ________
5. 一人で料理する → 一人で料理 ________

<details>
<summary>Answers</summary>

1. 日本語が**話せる** (Group 1: 話す → 話せる)
2. この漢字が**読める** (Group 1: 読む → 読める)
3. 朝早く**起きられる** (Group 2: 起きる → 起きられる)
4. 明日**来られる** (こられる) (Group 3: 来る → 来られる)
5. 一人で料理**できる** (Group 3: する → できる)

</details>

### Exercise 2: Form the Volitional

Convert to volitional form (casual).

1. 映画を見る → ________
2. 公園を走る → ________
3. コーヒーを飲む → ________
4. 友達に電話する → ________
5. パーティーに行く → ________

<details>
<summary>Answers</summary>

1. 映画を**見よう** (Group 2: 見る → 見よう)
2. 公園を**走ろう** (Group 1: 走る → 走ろう)
3. コーヒーを**飲もう** (Group 1: 飲む → 飲もう)
4. 友達に電話**しよう** (Group 3: する → しよう)
5. パーティーに**行こう** (Group 1: 行く → 行こう)

</details>

### Exercise 3: Active to Passive

Rewrite as passive sentences.

1. 先生が学生を褒めた。(The teacher praised the student.)
2. 犬が弟を噛んだ。(The dog bit my younger brother.)
3. 誰かが窓を割った。(Someone broke the window.)
4. 母が私を起こした。(My mother woke me up.)

<details>
<summary>Answers</summary>

1. 学生が先生**に褒められた。** (The student was praised by the teacher.)
2. 弟が犬**に噛まれた。** (My younger brother was bitten by the dog.)
3. 窓が誰か**に割られた。** (The window was broken by someone.)
4. 私が母**に起こされた。** (I was woken up by my mother.)

</details>

### Exercise 4: Form the Causative

Complete using causative form.

1. 母は子供に野菜を ________ (食べる — make eat)
2. 先生は学生を ________ (走る — make run)
3. 部長は私に報告を ________ (する — make do)
4. もう少し ________ ください。(考える — let think)

<details>
<summary>Answers</summary>

1. 母は子供に野菜を**食べさせた。** (Group 2: 食べる → 食べさせる)
2. 先生は学生を**走らせた。** (Group 1: 走る → 走らせる)
3. 部長は私に報告を**させた。** (Group 3: する → させる)
4. もう少し**考えさせて**ください。 (Group 2: 考える → 考えさせる)

</details>

### Exercise 5: Causative-Passive

Rewrite from the perspective of the person being forced.

1. 先生が私を立たせた。→ 私は先生に ________
2. 母が子供に薬を飲ませた。→ 子供は母に薬を ________
3. 上司が部下を残業させた。→ 部下は上司に ________

<details>
<summary>Answers</summary>

1. 私は先生に**立たせられた / 立たされた。** (I was made to stand by the teacher.)
2. 子供は母に薬を**飲ませられた / 飲まされた。** (The child was made to take medicine by the mother.)
3. 部下は上司に**残業させられた。** (The subordinate was made to work overtime by the boss.)

</details>

### Exercise 6: Mixed Form Identification

Identify the verb form used in each sentence.

1. この本は世界中で読まれている。
2. 日本語が少し話せるようになった。
3. 子供の頃、毎日牛乳を飲まされた。
4. 今日は早く帰ろう。
5. 母に部屋を掃除させられた。

<details>
<summary>Answers</summary>

1. **Passive** (読まれている — is being read)
2. **Potential** (話せる — can speak)
3. **Causative-Passive shortened** (飲まされた — was made to drink)
4. **Volitional** (帰ろう — let's go home)
5. **Causative-Passive** (掃除させられた — was made to clean)

</details>
