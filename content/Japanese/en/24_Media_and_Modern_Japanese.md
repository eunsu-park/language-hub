## 2. Reading News Articles

### 2.1 News Writing Style

Japanese news articles use a distinct style that differs from everyday language:

```
+----------------------------------------------------------+
|         NEWS JAPANESE CHARACTERISTICS                    |
+----------------------------------------------------------+
|                                                          |
|  1. Headline compression:                                |
|     • Particles are often omitted                        |
|     • Verbs in dictionary form (not ます form)            |
|     • Example: 「首相、増税見送り表明」                     |
|       Full: 首相が増税の見送りを表明した                    |
|                                                          |
|  2. Formal written style:                                |
|     • ～である / ～だ (not ～です/～ます)                    |
|     • ～とのこと / ～という                                |
|     • ～と見られている (it is thought that)                |
|                                                          |
|  3. Attribution patterns:                                |
|     • ～によると (according to)                           |
|     • ～と述べた (stated)                                 |
|     • 関係者によれば (according to sources)               |
|                                                          |
|  4. Passive voice is very common:                        |
|     • ～が発表された (was announced)                       |
|     • ～が指摘されている (it has been pointed out)         |
|     • ～ことが明らかになった (it became clear that)         |
+----------------------------------------------------------+
```

### 2.2 Sample News Article with Annotations

```
【見出し】
政府、来年度予算案を閣議決定 ← Headline: particles omitted

【本文】
政府は20日、来年度の一般会計予算案を閣議決定した。
         ↑ Date pattern: ○日
総額は約115兆円で、過去最大となった。
              ↑ Approximate number: 約
防衛費は前年度比で16%増の7兆9000億円に上り、
                          ↑ Large numbers
3年連続で過去最高を更新した。

財務大臣は記者会見で「持続可能な財政運営に
                              ↑ Quote
全力を尽くす」と述べた。
             ↑ Attribution

一方、野党からは「国民生活への配慮が不十分だ」
↑ Contrast/transition
との批判の声が上がっている。
     ↑ Passive pattern
```

### 2.3 Common News Patterns

| Pattern | Meaning | Example |
|---------|---------|---------|
| ～と見られている | It is believed that | 犯人は20代と見られている |
| ～ことが分かった | It has been found that | 原因は人為的ミスであることが分かった |
| ～の見通し | Outlook / prospect | 景気は回復する見通しだ |
| ～に踏み切る | To take a decisive step | 政府は増税に踏み切った |
| ～を受けて | In response to | 批判を受けて、計画を修正した |
| ～に先立ち | Prior to | 会談に先立ち、外相が声明を出した |
| ～の一環として | As part of | 改革の一環として新制度を導入する |

## 4. Youth Slang (若者言葉)

### 4.1 Current Youth Expressions

Youth slang (若者言葉 / わかものことば) changes rapidly. Here are established terms
that have persisted:

| Slang | Meaning | Usage |
|-------|---------|-------|
| やばい / ヤバい | Amazing / terrible (context-dependent) | 「このラーメン、やばい！」(This ramen is amazing!) |
| エモい | Emotional / nostalgic / moving | 「この曲エモいわ」(This song is so emotional) |
| ガチ | Serious / for real | 「ガチで言ってる？」(Are you serious?) |
| 推し (おし) | Favorite person (idol, character) | 「推しが尊い」(My fave is precious) |
| 尊い (とうとい) | Precious / too good | Used about beloved characters/idols |
| 沼 (ぬま) | Obsession / deep dive | 「BTS沼にハマった」(Fell into BTS obsession) |
| 陽キャ (ようきゃ) | Extrovert / outgoing person | 陽 = bright + キャラ = character |
| 陰キャ (いんきゃ) | Introvert / quiet person | 陰 = shadow + キャラ = character |
| それな | "That's so true" / agreement | Universal agreement phrase |
| ワンチャン | Maybe / there's a chance | ワンチャンス (one chance) |

### 4.2 Meaning Shift Words

Some words have dramatically changed meaning in youth usage:

| Word | Original Meaning | Youth Meaning |
|------|-----------------|---------------|
| やばい | Dangerous / bad | Amazing (positive or negative) |
| 全然 (ぜんぜん) | Not at all (+ negative) | Totally / absolutely (+ positive) |
| 普通に (ふつうに) | Normally | Quite / genuinely |
| 微妙 (びみょう) | Subtle / delicate | Not great / meh |
| 適当 (てきとう) | Appropriate / suitable | Half-hearted / sloppy |
| 大丈夫 (だいじょうぶ) | All right / safe | No thanks (polite refusal) |

> **Example of 大丈夫 as refusal**:
> 店員：「レジ袋はいりますか。」(Do you need a plastic bag?)
> 客：「大丈夫です。」(No, I'm fine / No thanks.)

### 4.3 Word Formation Patterns in Slang

| Pattern | Examples | How It Works |
|---------|----------|-------------|
| ～み (nominalization) | つらみ, わかりみ, やばみ | Adjective/verb stem + み = abstract noun |
| ～活 (かつ = activity) | 婚活, 就活, 推し活, 朝活 | Abbreviation + 活 = an "activity" |
| Abbreviation | スタバ, ファミマ, バイト | Shortened loanwords |
| ～い (adjective-ification) | エモい, エグい, キモい | Noun/foreign word + い = adjective |
| 逆○○ (ぎゃく) | 逆に好き | "Conversely, I like it" = unexpected positive |

### 4.4 Generational Slang Timeline

| Era | Representative Slang |
|-----|---------------------|
| 1990s | チョベリグ (super good), チョベリバ (super bad), マジ卍 |
| 2000s | KY (空気読めない = can't read the room), ウケる (funny) |
| 2010s | リア充 (fulfilled in real life), ヤバい (shift to positive), 草 |
| 2020s | エモい, 推し, ぴえん (crying), 蛙化現象 (ick), 当たり前体操 |

### 5.2 Online Community Culture

> **JAPANESE INTERNET CULTURE LANDMARKS**
>
> 2ちゃんねる / 5ちゃんねる (Anonymous forums)
> - Enormous influence on Japanese net culture
> - Created the anonymous posting culture
> - Origin of many internet memes and slang
> - **Board categories**: ニュー速, VIP, なんJ
>
> ニコニコ動画 (Video sharing)
> - Comments scroll across the video (弾幕 / danmaku)
> - Birthplace of Vocaloid culture
> - Live streaming culture (生放送)
>
> - **Note**: Japan's internet culture tends toward anonymity,
> unlike Western social media's emphasis on real identity.

1. 関係者以外立入禁止
2. 食べ歩き禁止
3. 只今満席です。しばらくお待ちください。
4. 本日貸切

<details>
<summary>Answers</summary>

1. 関係者以外立入禁止 (かんけいしゃいがいたちいりきんし)
   "No entry except for authorized personnel." Found at construction sites, backstage areas, office buildings.

2. 食べ歩き禁止 (たべあるききんし)
   "No eating while walking." Found in traditional shopping streets, temples, and tourist areas.

3. 只今満席です。しばらくお待ちください。(ただいままんせきです)
   "We are currently full. Please wait a moment." Found at restaurants.

4. 本日貸切 (ほんじつかしきり)
   "Reserved for private event today." Found at restaurants and event venues that are closed to the public.

</details>

### Exercise 4: Youth Slang Context

Choose the correct meaning of the underlined word based on context:

1. 「このケーキ、**やばい**！超おいしい！」
   a) Dangerous  b) Terrible  c) Amazing

2. 「あの映画、**全然**よかったよ。」
   a) Not at all good  b) Quite good  c) A little good

3. 「バイト大変だったけど、**ワンチャン**来月休めるかも。」
   a) One chance  b) Maybe  c) Definitely

4. 「新しいドラマの主人公、**推せる**。」
   a) Can be pushed  b) Is supportable/likable  c) Is strange

<details>
<summary>Answers</summary>

1. c) Amazing — The follow-up 「超おいしい」 confirms the positive meaning.
2. b) Quite good — Youth usage of 全然 with positive adjectives means "totally/quite."
3. b) Maybe — ワンチャン (one chance) is used as "maybe/possibly" in casual speech.
4. b) Is supportable/likable — 推せる (can be an 推し) means the character is worthy of being your favorite.

</details>

### Exercise 5: News Article Comprehension

Read this short news excerpt and answer the questions:

```
政府は5日、来年度の教育予算案を閣議決定した。

一方、教育関係者からは「教員の負担軽減が
先決ではないか」との声も上がっている。
```

2. What is the total budget amount?
3. What did the Minister of Education emphasize?
