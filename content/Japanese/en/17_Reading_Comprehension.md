## 1. Reading Strategies for Japanese Text

Japanese reading presents unique challenges: three writing systems, no spaces between words, and information-dense kanji compounds. The following strategies help manage these challenges.

```
+----------------------------------------------------------+
|         READING STRATEGIES FOR JAPANESE TEXT               |
+----------------------------------------------------------+
|                                                          |
|  BEFORE reading (読む前に):                               |
|  1. Check the title and headings                         |
|  2. Identify the text type (article? email? essay?)      |
|  3. Scan for kanji you recognize                         |
|  4. Note the overall length and structure                |
|  5. Activate background knowledge about the topic        |
|                                                          |
|  DURING reading (読みながら):                             |
|  6. Don't stop at every unknown kanji                    |
|  7. Use context to infer meaning                         |
|  8. Watch for topic markers (は) and subject markers (が) |
|  9. Find the verb at the end of each sentence            |
|  10. Pay attention to conjunctions and connectors         |
|  11. Note the politeness level for context clues          |
|                                                          |
|  AFTER reading (読んだ後で):                              |
|  12. Summarize the main point in one sentence            |
|  13. Identify the author's position or purpose           |
|  14. Review unknown vocabulary in context                |
|  15. Re-read difficult passages with new understanding   |
+----------------------------------------------------------+
```

### 1.1 The "Verb-Last" Principle

In Japanese, the **verb always comes at the end** of a clause. This means the most critical information (the action, the conclusion, the negation) is at the very end. Train yourself to:

1. Identify the topic/subject at the beginning (marked by は or が)
2. Skim the middle for objects and modifiers
3. Focus on the **final verb** for the core meaning
4. Check for negation (ない, ません) at the very end

```
┌─────────┬─────────────────┬──────────────┐
│ Topic/  │    Objects,     │  VERB        │
│ Subject │    Modifiers    │  (key info)  │
│ は/が   │    を/に/で/と   │  ⬅ READ THIS │
└─────────┴─────────────────┴──────────────┘
```

### 1.2 Connector Words (接続詞)

Connector words signal the relationship between sentences. Recognizing them helps you predict what comes next.

| Connector | Meaning | Signal |
|-----------|---------|--------|
| しかし / だが / けれども | However | Contrast is coming |
| したがって / そのため / だから | Therefore | Conclusion follows |
| また / さらに / その上 | Moreover | Additional info |
| 一方 / 他方 / 反面 | On the other hand | Different perspective |
| つまり / すなわち / 要するに | In other words | Rephrasing / summary |
| 例えば / 具体的には | For example | Example follows |
| ところで / さて | By the way | Topic change |
| なぜなら / というのは | Because / The reason is | Explanation follows |

## 2. The Three Writing Systems in Reading

Understanding how the three scripts function in text helps with reading speed and comprehension.

### 2.1 Script Functions in Running Text

| Script | Function | Example |
|--------|----------|---------|
| **漢字** (Kanji) | Content words: nouns, verb/adj stems | 学生 (student), 食べる (eat) |
| **ひらがな** (Hiragana) | Grammar: particles, endings, native words | は, が, を, の, する, できる |
| **カタカナ** (Katakana) | Foreign words, emphasis, sound effects | コンピューター, アメリカ, データ |

### 2.2 Reading Speed Tips

```
+------------------------------------------+
|  VISUAL PARSING STRATEGY                  |
+------------------------------------------+
|                                          |
|  Kanji compounds → key concepts          |
|  (scan these first for main ideas)       |
|                                          |
|  Hiragana between kanji → grammar glue   |
|  (particles tell you WHO did WHAT)       |
|                                          |
|  Katakana blocks → loanwords             |
|  (sound them out in English/Portuguese)  |
|                                          |
|  Sentence-final です/ます → politeness   |
|  Sentence-final だ/である → plain/formal |
+------------------------------------------+
```

> **Pro tip:** When skimming, focus on **kanji compounds**. They carry the core meaning.
> The hiragana between them provides grammatical relationships.
> Example: 「環境問題に関する国際会議が開催された」
> Key kanji: 環境問題 (environmental problem) → 国際会議 (international conference) → 開催 (held)
> → "An international conference on environmental issues was held."

## 3. Newspaper Articles (新聞記事)

### 3.1 Structure of a Japanese News Article

Japanese newspaper articles follow an **inverted pyramid** structure, similar to Western journalism but with specific vocabulary conventions.

```
┌──────────────────────────────────────┐
│           見出し (Headline)           │  ← Short, kanji-heavy, often omits particles
├──────────────────────────────────────┤
│         リード文 (Lead sentence)      │  ← Who, What, When, Where in 1-2 sentences
├──────────────────────────────────────┤
│         本文 (Body)                  │  ← Details, background, quotes
├──────────────────────────────────────┤
│         背景・補足 (Context)          │  ← Additional context, related info
└──────────────────────────────────────┘
```

### 3.2 Headline Vocabulary

Newspaper headlines use a special compressed style. Common patterns:

| Headline Style | Full Form | Meaning |
|----------------|-----------|---------|
| 首相、訪米へ | 首相は訪米する予定だ | PM to visit the US |
| 新法成立 | 新しい法律が成立した | New law established |
| 景気回復の兆し | 景気が回復する兆しがある | Signs of economic recovery |
| ～か | ～かもしれない | Possibly ~ |
| ～見通し | ～見通しである | Outlook / expected to ~ |

| Japanese | Reading | English |
|----------|---------|---------|
| 政府 | せいふ | government |
| 首相 | しゅしょう | prime minister |
| 発表 | はっぴょう | announcement |
| 会談 | かいだん | talks, meeting |
| 対策 | たいさく | countermeasure |
| 被害 | ひがい | damage |

| 減少 | げんしょう | decrease |

### 3.4 Passive Voice in News

Japanese news heavily uses the **passive voice** (受身形) to maintain objectivity:

| Passive (News Style) | English |
|---------------------|---------|
| 計画が発表**された**。 | The plan was announced. |
| 新法が施行**される**見通しだ。 | The new law is expected to be enacted. |
| 容疑者が逮捕**された**。 | The suspect was arrested. |
| 被害が確認**されている**。 | Damage has been confirmed. |

## 4. Essays and Opinion Pieces (論説文)

### 4.1 Structure of Japanese Essays

Japanese essays (especially opinion pieces) follow a classic four-part structure called **起承転結** (きしょうてんけつ):

| Part | Name | Role |
|------|------|------|
| 起 (き) | Introduction | Introduce the topic, provide context |
| 承 (しょう) | Development | Expand on the topic, provide evidence |
| 転 (てん) | Turn/Twist | Present a different angle, counterargument, or deeper point |
| 結 (けつ) | Conclusion | State the main argument, wrap up |

> **Important:** The 転 (turn) section is crucial. Japanese essays often present an unexpected
> angle or counterpoint before reaching the conclusion. Don't assume the development section
> contains the author's final opinion.

### 4.2 Identifying the Author's Opinion

Look for these opinion markers:

| Pattern | Meaning | Strength |
|---------|---------|----------|
| ～と思う / ～と考える | I think ~ | Direct opinion |
| ～べきだ / ～べきである | Should ~ | Strong recommendation |
| ～ではないだろうか | Isn't it ~? | Rhetorical (= I think so) |
| ～のではないか | Could it be that ~? | Suggesting agreement |
| ～に違いない | Must be ~ | Strong conviction |
| ～と言えるだろう | One could say ~ | Moderate claim |
| ～必要がある | There is a need to ~ | Advocating action |
| ～わけにはいかない | Cannot afford to ~ | Strong necessity |

### 4.3 Common Essay Vocabulary

| Japanese | Reading | English |
|----------|---------|---------|
| 主張 | しゅちょう | claim, assertion |
| 根拠 | こんきょ | basis, grounds |
| 傾向 | けいこう | tendency, trend |
| 現状 | げんじょう | current situation |
| 背景 | はいけい | background |
| 課題 | かだい | issue, challenge |
| 観点 | かんてん | viewpoint |
| 矛盾 | むじゅん | contradiction |
| 具体的 | ぐたいてき | concrete, specific |
| 抽象的 | ちゅうしょうてき | abstract |

### 4.4 Logical Flow Markers

| Marker | Function | Example Use |
|--------|----------|-------------|
| まず / 第一に | First | Introducing first point |
| 次に / 第二に | Next / Second | Introducing second point |
| 最後に / 結論として | Finally / In conclusion | Wrapping up |
| 確かに～。しかし～。 | Certainly ~. However ~. | Acknowledging then countering |
| ～という意見もある。しかし～。 | Some say ~. However ~. | Presenting opposing view |

## 6. Scanning vs. Close Reading

### 6.1 Scanning (拾い読み — ひろいよみ)

**Purpose:** Find specific information quickly.

**When to use:**
- Looking for a date, name, number, or keyword
- Answering specific factual questions
- Searching through long documents

**How to scan Japanese text:**

1. Identify your target information (a date? a name? a number?)
2. Scan for visual cues:
   - **Numbers:** 数字 or 漢数字 stand out visually
   - **Names:** Personal names often in kanji with furigana
   - **Katakana:** Foreign words and proper nouns are immediately visible
   - **Quotation marks:** 「」 mark direct quotes and key terms
3. Move your eyes in a **Z-pattern** across the text
4. Don't read every word — let your eyes jump between kanji clusters

### 6.2 Close Reading (精読 — せいどく)

**Purpose:** Understand every detail of a passage.

**When to use:**
- Analyzing the author's argument
- Studying grammar in context
- Preparing for reading comprehension tests
- Reading contracts or legal documents

**How to close-read Japanese text:**

1. Read the full sentence from beginning to verb-end
2. Parse grammatical structure: topic + [modifying clauses] + main predicate
3. Check particle usage (は vs. が, に vs. で, etc.)
4. Identify all conjunctions and their logical function
5. Re-read the sentence with full understanding before moving on

### 6.3 Choosing Your Strategy

```
+-----------------------------------------------------+
|  Question Type       →  Strategy                     |
+-----------------------------------------------------+
|  "When did X happen?"  →  SCAN for dates/時         |
|  "Who said X?"         →  SCAN for 「」 and names   |
|  "What is the main     →  CLOSE READ introduction   |
|   idea?"                  and conclusion             |
|  "Why does the author  →  CLOSE READ the 転 section |
|   argue X?"               (turn/counterpoint)        |
|  "What is implied?"    →  CLOSE READ context clues  |
+-----------------------------------------------------+
```

## 7. Key Comprehension Question Types

JLPT and academic reading tests commonly use these question types:

### 7.1 Main Idea Questions (主旨問題)

**Signal phrases:** 「この文章の主旨は何ですか」「筆者が最も言いたいことは何ですか」

Strategy: Focus on the conclusion (結) section. The author's main point is usually stated or restated near the end.

### 7.2 Detail Questions (詳細問題)

**Signal phrases:** 「～について、正しいものはどれですか」「～とは何ですか」

Strategy: Scan for the relevant keyword, then close-read the surrounding sentences.

### 7.3 Inference Questions (推論問題)

**Signal phrases:** 「～から何がわかりますか」「筆者はどう考えていますか」

Strategy: Look for opinion markers (と思う, べきだ, ではないだろうか). Consider what is implied but not directly stated.

### 7.4 Reason Questions (理由問題)

**Signal phrases:** 「なぜ～ですか」「～の理由は何ですか」

Strategy: Search for causal connectors (なぜなら, から, ため, ので) near the referenced content.

### 7.5 Reference Questions (指示語問題)

**Signal phrases:** 「『それ』は何を指していますか」「『この問題』とは何ですか」

Strategy: Look at the sentence(s) immediately before the demonstrative (それ, これ, この, その). The referent is almost always in the preceding context.

### 7.6 Word Meaning in Context (語彙問題)

**Signal phrases:** 「～はどういう意味ですか」「～に最も近い意味は...」

Strategy: Read the full sentence containing the word. Use surrounding context and kanji components to deduce meaning.

## 9. Practice Text 2: Opinion Essay

Read the following opinion essay excerpt and answer the questions.

---

**「読書離れ」は本当に問題か**

近年、「若者の読書離れ」が社会問題として取り上げられることが多い。確かに、紙の書籍の売り上げは年々減少している。しかし、これをもって「読書離れ」と結論づけるのは早計ではないだろうか。

現代の若者は、スマートフォンやタブレットを通じて、かつてないほど多くのテキストに触れている。SNSの投稿、ニュースアプリの記事、電子書籍など、文字を読む機会はむしろ増えているとも言える。問題は「読む量」ではなく、「何を、どのように読むか」という質の問題なのではないか。

つまり、「読書離れ」という表現自体が、旧来の読書形態にとらわれた見方であり、デジタル時代における新しい「読書」の定義を考え直す必要があるのではないだろうか。

---

### Questions:

1. 「読書離れ」の根拠として挙げられている事実は何ですか。
2. 筆者は「読書離れ」についてどのような立場ですか。
3. 「ではないだろうか」が2回使われていますが、これはどのような効果がありますか。

<details>
<summary>Answers</summary>

1. **紙の書籍の売り上げが年々減少していること**が根拠として挙げられている。
2. 筆者は「読書離れ」とは言い切れないという立場である。デジタル媒体での読書機会は増えており、**問題は量ではなく質である**と主張している。また、**「読書」の定義自体を見直すべきだ**と提案している。
3. 「ではないだろうか」は**修辞的疑問文**（反語）であり、「～と思う」よりも柔らかく自分の意見を提示する効果がある。直接的な主張を避けながらも、読者に同意を促す日本語特有の論説技法である。

</details>

## 10. Practice Text 3: Business Email

Read the following business email and answer the questions.

---

件名: 来月の研修プログラムについて

山田部長

いつもお世話になっております。
人事部の佐藤でございます。

来月開催予定の新入社員研修プログラムについてご連絡いたします。

日時: 4月10日（月）～ 4月14日（金） 9:00～17:00
場所: 本社5階 大会議室
対象: 2026年度新入社員（15名）
講師: 外部講師2名 + 社内講師3名

つきましては、山田部長に4月12日（水）の午後の講義（14:00～16:00）をご担当いただけないかと存じます。テーマは「プロジェクト管理の基礎」を予定しております。

ご多忙のところ恐れ入りますが、3月15日までにご回答いただけますと幸いです。ご不明な点がございましたら、お気軽にお問い合わせください。

何卒よろしくお願い申し上げます。

人事部 佐藤花子
内線: 3456
メール: sato.hanako@company.co.jp

---

### Questions:

1. この文書の目的は何ですか。
2. 山田部長に依頼されている具体的な内容は何ですか。
3. 「ご担当いただけないかと存じます」はどのような表現ですか。なぜ直接的に「担当してください」と言わないのですか。

<details>
<summary>Answers</summary>

1. 新入社員研修プログラムの案内と、**山田部長に講義の担当を依頼すること**が目的である。
2. **4月12日（水）14:00～16:00に「プロジェクト管理の基礎」というテーマで講義を行うこと**が依頼されている。
3. 「ご担当いただけないかと存じます」は**謙譲語を使った間接的な依頼表現**である。「いただけないか」は「もらえないか」の謙譲語で、さらに「と存じます」（「と思います」の謙譲語）を加えることで、命令ではなく依頼のニュアンスを出している。日本のビジネス文化では、目上の人への依頼は直接的な表現を避け、相手に断る余地を残すのが礼儀とされる。

</details>

## 11. Practice Exercises

### Exercise 1: Connector Identification

Identify the connector word in each sentence and explain what type of relationship it signals.

1. 日本語は難しい。**しかし**、とても面白い言語だ。
2. 気温が上がった。**そのため**、桜が早く咲いた。
3. 彼は成績が良い。**さらに**、スポーツも得意だ。
4. 東京は便利だ。**一方**、家賃が高い。
5. **つまり**、この計画は現実的ではない。

<details>
<summary>Answers</summary>

1. **しかし** — Contrast: the second sentence presents an opposing idea
2. **そのため** — Cause-effect: the second sentence is a result of the first
3. **さらに** — Addition: the second sentence adds more information
4. **一方** — Comparison: presenting a different aspect
5. **つまり** — Summary/Rephrasing: restating the conclusion in simpler terms

</details>

### Exercise 2: Question Type Classification

What type of reading comprehension question is each of the following?

1. この文章で筆者が最も言いたいことは何ですか。
2. 「その問題」とは何を指していますか。
3. なぜ著者は反対の立場をとっていますか。
4. 2025年の売上高はいくらですか。
5. 「画期的」に最も近い意味はどれですか。

<details>
<summary>Answers</summary>

1. **主旨問題** (Main idea question) — 「最も言いたいこと」is a signal for the main argument
2. **指示語問題** (Reference question) — 「何を指していますか」asks what a demonstrative refers to
3. **理由問題** (Reason question) — 「なぜ」asks for the reason
4. **詳細問題** (Detail question) — Asks for specific factual information
5. **語彙問題** (Word meaning in context) — 「最も近い意味」asks for the closest meaning

</details>

### Exercise 3: Scanning Practice

The following text contains specific information. Answer each question as quickly as possible using scanning techniques — you do NOT need to read every word.

---

東京スカイツリーは2012年5月22日に開業した。高さは634メートルで、自立式電波塔としては世界一である。建設費は約650億円で、工期は約3年8ヶ月であった。年間来場者数は約600万人に達し、展望台は350メートルと450メートルの2カ所にある。最寄り駅は東武スカイツリーライン「とうきょうスカイツリー駅」と、半蔵門線「押上駅」である。

---

1. スカイツリーの高さは？
2. 開業日はいつ？
3. 展望台は何メートルの場所にある？
4. 最寄り駅の路線名を2つ挙げてください。

<details>
<summary>Answers</summary>

1. **634メートル**
2. **2012年5月22日**
3. **350メートルと450メートル**
4. **東武スカイツリーライン**と**半蔵門線**

</details>

### Exercise 4: News Headline Interpretation

Expand these abbreviated newspaper headlines into full sentences.

1. 首相、来月訪中へ
2. 大型台風接近、交通に影響も
3. 新薬承認、年内にも
4. GDP3期連続プラス

<details>
<summary>Answers</summary>

1. 首相**は**来月**に**中国**を**訪問**する予定だ**。 (The PM is scheduled to visit China next month.)
2. 大型**の**台風**が**接近**しており**、交通**に**影響**が出る可能性もある**。 (A large typhoon is approaching, and there may be an impact on transportation.)
3. 新薬**が**承認**される見通しで**、年内**にも承認される可能性がある**。 (A new drug is expected to be approved, possibly within this year.)
4. GDP**は**3期連続**で**プラス**成長を記録した**。 (GDP recorded positive growth for three consecutive quarters.)

</details>
