## 1. Japanese Verb Basics

### 1.1 How Japanese Verbs Differ from English

```
┌──────────────────────────────────────────────────────────────────┐
│              Japanese Verb Key Facts                              │
│                                                                  │
│   • Verbs always come at the END of the sentence                 │
│   • Verbs do NOT change for person/number                        │
│     (no "I eat" vs "he eats" distinction)                        │
│   • Verbs conjugate for: tense, politeness, mood                 │
│   • Dictionary form always ends in an -u sound                   │
│   • Two politeness levels: plain (casual) and polite (です/ます)  │
│                                                                  │
│   食べる (taberu) = "to eat"                                      │
│   Same form for: I eat, you eat, he/she eats, we eat, they eat  │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 Dictionary Form (辞書形 / じしょけい)

The **dictionary form** (also called plain present affirmative) is the basic form you find in dictionaries. It always ends in an -u vowel sound:

| Ending | Examples |
|--------|---------|
| -る (-ru) | 食べる (taberu), 見る (miru), 起きる (okiru) |
| -う (-u) | 買う (kau), 会う (au) |
| -く (-ku) | 書く (kaku), 聞く (kiku) |
| -ぐ (-gu) | 泳ぐ (oyogu) |
| -す (-su) | 話す (hanasu), 出す (dasu) |
| -つ (-tsu) | 待つ (matsu), 持つ (motsu) |
| -ぬ (-nu) | 死ぬ (shinu) — only one! |
| -ぶ (-bu) | 遊ぶ (asobu), 飛ぶ (tobu) |
| -む (-mu) | 読む (yomu), 飲む (nomu) |

## 2. The Three Verb Groups

### 2.1 Overview

```
┌──────────────────────────────────────────────────────────────────┐
│              The Three Verb Groups                                │
│                                                                  │
│   Group 1: Ichidan verbs (一段動詞) — "ru-verbs"                  │
│     End in -iru or -eru                                          │
│     Drop る, add conjugation ending                              │
│     Examples: 食べる, 見る, 起きる                                  │
│                                                                  │
│   Group 2: Godan verbs (五段動詞) — "u-verbs"                     │
│     End in -u, -ku, -gu, -su, -tsu, -nu, -bu, -mu, -ru          │
│     Change the final vowel, then add ending                      │
│     Examples: 書く, 話す, 読む, 買う                                │
│                                                                  │
│   Group 3: Irregular verbs (不規則動詞)                            │
│     Only TWO verbs:                                              │
│     する (suru, to do) and 来る (kuru, to come)                   │
│     Must be memorized individually                               │
└──────────────────────────────────────────────────────────────────┘
```

### 2.2 Group 1: Ichidan Verbs (る-Verbs)

Ichidan means "one-step" because conjugation requires only one step: drop る and add the new ending.

**How to identify**: The verb ends in -**i**ru or -**e**ru:

| Verb | Romaji | Meaning | Stem (drop る) |
|------|--------|---------|---------------|
| 食べる | taberu | to eat | 食べ (tabe) |
| 見る | miru | to see/watch | 見 (mi) |
| 起きる | okiru | to wake up | 起き (oki) |
| 寝る | neru | to sleep | 寝 (ne) |
| 着る | kiru | to wear | 着 (ki) |
| 出る | deru | to go out | 出 (de) |
| 開ける | akeru | to open | 開け (ake) |
| 閉める | shimeru | to close | 閉め (shime) |
| 教える | oshieru | to teach | 教え (oshie) |
| 答える | kotaeru | to answer | 答え (kotae) |

### 2.3 Group 2: Godan Verbs (う-Verbs)

Godan means "five-step" because the final consonant combines with five different vowels (a, i, u, e, o) in different conjugations.

| Verb | Romaji | Meaning | Ending |
|------|--------|---------|--------|
| 書く | kaku | to write | -ku |
| 聞く | kiku | to listen | -ku |
| 泳ぐ | oyogu | to swim | -gu |
| 話す | hanasu | to speak | -su |
| 待つ | matsu | to wait | -tsu |
| 死ぬ | shinu | to die | -nu |
| 遊ぶ | asobu | to play | -bu |
| 読む | yomu | to read | -mu |
| 飲む | nomu | to drink | -mu |
| 買う | kau | to buy | -u |
| 会う | au | to meet | -u |
| 作る | tsukuru | to make | -ru |
| 乗る | noru | to ride | -ru |
| 帰る | kaeru | to return | -ru |
| 走る | hashiru | to run | -ru |

### 2.4 Tricky る-Ending Godan Verbs

Some verbs end in -iru or -eru but are actually godan (Group 2), not ichidan (Group 1). These must be memorized:

| Verb | Romaji | Meaning | Group | Why tricky |
|------|--------|---------|-------|-----------|
| 帰る | kaeru | to return | Godan | Ends in -eru |
| 走る | hashiru | to run | Godan | Ends in -iru |
| 切る | kiru | to cut | Godan | Ends in -iru |
| 知る | shiru | to know | Godan | Ends in -iru |
| 入る | hairu | to enter | Godan | Ends in -iru |
| 要る | iru | to need | Godan | Ends in -iru |

> **Tip**: There is no foolproof rule to distinguish these. When you learn a new verb, always learn which group it belongs to.

### 2.5 Group 3: Irregular Verbs

Only two verbs are truly irregular:

| Verb | Romaji | Meaning |
|------|--------|---------|
| する | suru | to do |
| 来る (くる) | kuru | to come |

する also forms compound verbs with nouns:

| Compound | Romaji | Meaning |
|----------|--------|---------|
| 勉強する | benkyou suru | to study |
| 料理する | ryouri suru | to cook |
| 運動する | undou suru | to exercise |
| 散歩する | sanpo suru | to take a walk |
| 買い物する | kaimono suru | to go shopping |
| 電話する | denwa suru | to make a phone call |

## 3. ます Form: Polite Present/Future

### 3.1 Formation Rules

The ます form is the standard polite form. It expresses present tense, future tense, and habitual actions:

```
┌──────────────────────────────────────────────────────────────────┐
│              How to Make ます Form                                │
│                                                                  │
│   Group 1 (Ichidan): Drop る → add ます                          │
│     食べる → 食べ + ます → 食べます                                  │
│                                                                  │
│   Group 2 (Godan): Change final -u to -i → add ます              │
│     書く → 書き + ます → 書きます  (ku → ki)                       │
│     話す → 話し + ます → 話します  (su → shi)                      │
│     読む → 読み + ます → 読みます  (mu → mi)                      │
│                                                                  │
│   Group 3 (Irregular): Memorize                                  │
│     する → します                                                  │
│     来る → 来ます (きます)                                         │
└──────────────────────────────────────────────────────────────────┘
```

### 3.2 Group 2 (Godan) Stem Changes

The final kana changes according to the vowel shift (u → i):

| Ending | Change | Dictionary → ます | Example |
|--------|--------|------------------|---------|
| -う (u) | う → い | 買**う** → 買**い**ます | kau → kaimasu |
| -く (ku) | く → き | 書**く** → 書**き**ます | kaku → kakimasu |
| -ぐ (gu) | ぐ → ぎ | 泳**ぐ** → 泳**ぎ**ます | oyogu → oyogimasu |
| -す (su) | す → し | 話**す** → 話**し**ます | hanasu → hanashimasu |
| -つ (tsu) | つ → ち | 待**つ** → 待**ち**ます | matsu → machimasu |
| -ぬ (nu) | ぬ → に | 死**ぬ** → 死**に**ます | shinu → shinimasu |
| -ぶ (bu) | ぶ → び | 遊**ぶ** → 遊**び**ます | asobu → asobimasu |
| -む (mu) | む → み | 読**む** → 読**み**ます | yomu → yomimasu |
| -る (ru) | る → り | 帰**る** → 帰**り**ます | kaeru → kaerimasu |

### 3.3 ます Form Conjugation Table

| Form | Ending | Meaning | Example (食べる) | Example (書く) |
|------|--------|---------|-----------------|---------------|
| Present Aff. | ます | eat/will eat | 食べます | 書きます |
| Present Neg. | ません | don't eat | 食べません | 書きません |
| Past Aff. | ました | ate | 食べました | 書きました |
| Past Neg. | ませんでした | didn't eat | 食べませんでした | 書きませんでした |

## 5. て Form: The Swiss Army Knife

### 5.1 Why て Form Matters

```
┌──────────────────────────────────────────────────────────────────┐
│              The て Form — Most Versatile Conjugation             │
│                                                                  │
│   The て form is used for:                                       │
│   • Connecting sentences ("eat and drink")                       │
│   • Requests (食べてください — "please eat")                       │
│   • Progressive (食べている — "is eating")                        │
│   • Permission (食べてもいい — "may eat")                          │
│   • Prohibition (食べてはいけない — "must not eat")                │
│   • Sequence of actions ("I went and ate")                       │
│                                                                  │
│   It is NOT a tense — it is a CONNECTING form.                   │
│   Learning て form is one of the most important milestones       │
│   in Japanese grammar.                                           │
└──────────────────────────────────────────────────────────────────┘
```

### 5.2 て Form Rules by Group

**Group 1 (Ichidan)**: Drop る → add て

| Dictionary | て Form | Romaji |
|-----------|--------|--------|
| 食べる | 食べて | tabete |
| 見る | 見て | mite |
| 起きる | 起きて | okite |

**Group 3 (Irregular)**: Memorize

| Dictionary | て Form | Romaji |
|-----------|--------|--------|
| する | して | shite |
| 来る | 来て (きて) | kite |

**Group 2 (Godan)**: This is where it gets complex. The ending changes based on the final kana:

### 5.3 Godan て Form: The Song Method

Many learners memorize the て form rules as a song (to the tune of "Santa Claus Is Coming to Town"):

```
┌──────────────────────────────────────────────────────────────────┐
│              The て Form Song for Godan Verbs                     │
│                                                                  │
│   う、つ、る → って      (u, tsu, ru → tte)                      │
│   む、ぶ、ぬ → んで     (mu, bu, nu → nde)                      │
│   く → いて              (ku → ite)                              │
│   ぐ → いで              (gu → ide)                              │
│   す → して              (su → shite)                            │
│                                                                  │
│   Exception: 行く → 行って (itte, NOT いいて)                     │
└──────────────────────────────────────────────────────────────────┘
```

### 5.4 Godan て Form Detailed Table

| Ending | て Form Rule | Dictionary | て Form | Romaji |
|--------|-------------|-----------|--------|--------|
| -う | → -って | 買**う** | 買**って** | katte |
| -つ | → -って | 待**つ** | 待**って** | matte |
| -る | → -って | 帰**る** | 帰**って** | kaette |
| -む | → -んで | 読**む** | 読**んで** | yonde |
| -ぶ | → -んで | 遊**ぶ** | 遊**んで** | asonde |
| -ぬ | → -んで | 死**ぬ** | 死**んで** | shinde |
| -く | → -いて | 書**く** | 書**いて** | kaite |
| -ぐ | → -いで | 泳**ぐ** | 泳**いで** | oyoide |
| -す | → -して | 話**す** | 話**して** | hanashite |

> **Exception**: 行く (iku, to go) → 行**って** (itte), NOT 行いて. This is the only -く verb with this exception.

### 5.5 て Form Uses (Preview)

| Usage | Pattern | Example | Meaning |
|-------|---------|---------|---------|
| Request | て + ください | 食べてください | Please eat |
| Progressive | て + いる/います | 食べています | I am eating |
| Connecting | て + next action | 食べて、飲みました | I ate and drank |
| Permission | て + もいいですか | 食べてもいいですか | May I eat? |

## 6. た Form: Past Tense

### 6.1 た Form = て Form with た/だ

The past tense plain form follows the exact same pattern as the て form, but uses た/だ instead of て/で:

```
Rule: Wherever て form has て → た form has た
      Wherever て form has で → た form has だ
```

### 6.2 た Form Table

**Group 1 (Ichidan)**: Drop る → add た

| Dictionary | た Form | Romaji |
|-----------|--------|--------|
| 食べる | 食べた | tabeta |
| 見る | 見た | mita |

**Group 3 (Irregular)**:

| Dictionary | た Form | Romaji |
|-----------|--------|--------|
| する | した | shita |
| 来る | 来た (きた) | kita |

**Group 2 (Godan)**:

| Ending | て Form | た Form | Example Dictionary → た |
|--------|--------|--------|----------------------|
| -う, -つ, -る | -って | -った | 買う → 買った (katta) |
| -む, -ぶ, -ぬ | -んで | -んだ | 読む → 読んだ (yonda) |
| -く | -いて | -いた | 書く → 書いた (kaita) |
| -ぐ | -いで | -いだ | 泳ぐ → 泳いだ (oyoida) |
| -す | -して | -した | 話す → 話した (hanashita) |
| 行く (exception) | 行って | 行った | 行く → 行った (itta) |

### 6.3 Polite Past Tense

For polite past, simply use ました:

| Dictionary | Polite Past | Romaji | Meaning |
|-----------|------------|--------|---------|
| 食べる | 食べました | tabemashita | ate |
| 行く | 行きました | ikimashita | went |
| する | しました | shimashita | did |
| 来る | 来ました (きました) | kimashita | came |
| 読む | 読みました | yomimashita | read (past) |
| 買う | 買いました | kaimashita | bought |

## 7. Negative Forms: ません and ない

### 7.1 Polite Negative: ません

Simply replace ます with ません:

| Affirmative | Negative | Meaning |
|-------------|----------|---------|
| 食べます | 食べません | don't eat |
| 行きます | 行きません | don't go |
| 書きます | 書きません | don't write |
| します | しません | don't do |
| 来ます | 来ません (きません) | don't come |

### 7.2 Polite Negative Past: ませんでした

| Affirmative Past | Negative Past | Meaning |
|------------------|--------------|---------|
| 食べました | 食べませんでした | didn't eat |
| 行きました | 行きませんでした | didn't go |
| しました | しませんでした | didn't do |

### 7.3 Plain Negative: ない Form

The plain negative is formed differently for each group:

```
┌──────────────────────────────────────────────────────────────────┐
│              ない Form (Plain Negative)                            │
│                                                                  │
│   Group 1 (Ichidan): Drop る → add ない                          │
│     食べる → 食べない (tabenai)                                    │
│                                                                  │
│   Group 2 (Godan): Change -u to -a → add ない                   │
│     書く → 書かない (kakanai)  (ku → ka)                          │
│     話す → 話さない (hanasanai) (su → sa)                         │
│     読む → 読まない (yomanai) (mu → ma)                           │
│                                                                  │
│     Special: -う verbs → -わない (NOT -あない)                     │
│     買う → 買わない (kawanai)                                      │
│                                                                  │
│   Group 3 (Irregular): Memorize                                  │
│     する → しない (shinai)                                        │
│     来る → 来ない (こない, konai)                                  │
│                                                                  │
│   Special: ある → ない (NOT あらない)                              │
└──────────────────────────────────────────────────────────────────┘
```

### 7.4 Godan ない Form: Vowel Shift (u → a)

| Ending | Change | Dictionary → ない | Romaji |
|--------|--------|------------------|--------|
| -う (u) | → -わない | 買**う** → 買**わ**ない | kawanai |
| -く (ku) | → -かない | 書**く** → 書**か**ない | kakanai |
| -ぐ (gu) | → -がない | 泳**ぐ** → 泳**が**ない | oyoganai |
| -す (su) | → -さない | 話**す** → 話**さ**ない | hanasanai |
| -つ (tsu) | → -たない | 待**つ** → 待**た**ない | matanai |
| -ぬ (nu) | → -なない | 死**ぬ** → 死**な**ない | shinanai |
| -ぶ (bu) | → -ばない | 遊**ぶ** → 遊**ば**ない | asobanai |
| -む (mu) | → -まない | 読**む** → 読**ま**ない | yomanai |
| -る (ru) | → -らない | 帰**る** → 帰**ら**ない | kaeranai |

> **Important exception**: Verbs ending in -う use -**わ**ない, not -あない. This is because the historical kana for this position was わ.

### 7.5 Plain Negative Past: なかった

Change ない to なかった:

| Plain Negative | Plain Neg. Past | Meaning |
|----------------|----------------|---------|
| 食べない | 食べなかった | didn't eat |
| 行かない | 行かなかった | didn't go |
| しない | しなかった | didn't do |
| 来ない (こない) | 来なかった (こなかった) | didn't come |

|------|-------------|------------|-------------|------------|
| Present Aff. | します | する | きます | くる |
| Present Neg. | しません | しない | きません | こない |
| Past Aff. | しました | した | きました | きた |
| Past Neg. | しませんでした | しなかった | きませんでした | こなかった |
| て Form | — | して | — | きて |

## 9. Using Verbs in Sentences

### 9.1 Basic Sentence Patterns

```
Pattern 1: [Topic]は [Object]を [Verb]ます。
  私はすしを食べます。
  Watashi wa sushi o tabemasu.
  "I eat sushi."

Pattern 2: [Topic]は [Place]に [Verb]ます。
  私は学校に行きます。
  Watashi wa gakkou ni ikimasu.
  "I go to school."

Pattern 3: [Topic]は [Place]で [Object]を [Verb]ます。
  私は図書館で本を読みます。
  Watashi wa toshokan de hon o yomimasu.
  "I read books at the library."

Pattern 4: [Topic]は [Time]に [Verb]ます。
  私は七時に起きます。
  Watashi wa shichi-ji ni okimasu.
  "I wake up at 7 o'clock."
```

### 9.2 Combining Actions with て Form

```
朝、起きて、シャワーを浴びて、朝ごはんを食べます。
Asa, okite, shawaa o abite, asagohan o tabemasu.
"In the morning, I wake up, take a shower, and eat breakfast."
```

### 9.3 Question Sentences

| Question | Answer |
|----------|--------|
| 何を食べますか。(What do you eat?) | すしを食べます。(I eat sushi.) |
| どこに行きますか。(Where do you go?) | 学校に行きます。(I go to school.) |
| いつ来ましたか。(When did you come?) | 昨日来ました。(I came yesterday.) |
| 日本語を話しますか。(Do you speak Japanese?) | はい、少し話します。(Yes, I speak a little.) |

### 9.4 Expressing Daily Routine

```
私は毎日六時半に起きます。
Watashi wa mainichi roku-ji han ni okimasu.
"I wake up at 6:30 every day."

シャワーを浴びて、朝ごはんを食べます。
Shawaa o abite, asagohan o tabemasu.
"I take a shower and eat breakfast."

八時に学校に行きます。
Hachi-ji ni gakkou ni ikimasu.
"I go to school at 8:00."

学校で日本語を勉強します。
Gakkou de nihongo o benkyou shimasu.
"I study Japanese at school."

五時に家に帰ります。
Go-ji ni ie ni kaerimasu.
"I go home at 5:00."

夜、本を読んで、十一時に寝ます。
Yoru, hon o yonde, juuichi-ji ni nemasu.
"At night, I read a book and go to bed at 11:00."
```

### Exercise 3: Make the て Form

Convert to て form:

```
1. 食べる →
2. 書く →
3. 読む →
4. 買う →
5. 話す →
6. 行く →
7. 泳ぐ →
8. 待つ →
9. する →
10. 遊ぶ →
```

<details>
<summary>Click to reveal answers</summary>

```
1. 食べる → 食べて (tabete) — Ichidan: drop る, add て
2. 書く → 書いて (kaite) — Godan: く → いて
3. 読む → 読んで (yonde) — Godan: む → んで
4. 買う → 買って (katte) — Godan: う → って
5. 話す → 話して (hanashite) — Godan: す → して
6. 行く → 行って (itte) — EXCEPTION! Not 行いて
7. 泳ぐ → 泳いで (oyoide) — Godan: ぐ → いで
8. 待つ → 待って (matte) — Godan: つ → って
9. する → して (shite) — Irregular
10. 遊ぶ → 遊んで (asonde) — Godan: ぶ → んで
```

</details>
