## 1. Writing Systems and Register

### 1.1 Choosing the Right Script

Japanese writing uses three scripts simultaneously, and each carries a tone:

| Script | Usage in Writing | Tone |
|--------|-----------------|------|
| 漢字 (Kanji) | Content words, formal terms | Mature, educated |
| ひらがな (Hiragana) | Grammatical particles, native words without standard kanji | Soft, approachable |
| カタカナ (Katakana) | Foreign loanwords, emphasis, technical terms | Modern, sharp |

### 1.2 Written vs Spoken Register

Japanese has a distinct written language (文語 / 書き言葉) that differs from spoken language
(口語 / 話し言葉). Mixing them is a common error.

| Spoken (話し言葉) | Written (書き言葉) | English |
|------------------|-------------------|---------|
| ～じゃない | ～ではない | is not |
| ～だけど | ～だが / ～であるが | but |
| ～してる | ～している | is doing |
| ～なんだ | ～なのである / ～のだ | it is that |
| すごく | 非常に / 極めて | very |
| ちゃんと | きちんと / 正確に | properly |
| やっぱり | やはり | as expected |
| ～みたいだ | ～のようだ | it seems like |
| ～っていう | ～という | called / that |
| いろんな | さまざまな / 種々の | various |

### 1.3 Formal Writing Markers

The だ/である style (常体) is used for essays and academic writing, while です/ます style
(敬体) is for letters and emails. Never mix them in the same document.

```
+----------------------------------------------------------+
|         WRITING STYLE SELECTION                          |
+----------------------------------------------------------+
|                                                          |
|  敬体 (です/ます style):                                  |
|  • Letters (手紙)                                        |
|  • Emails (メール)                                       |
|  • Business documents                                    |
|  • Self-introduction essays                              |
|                                                          |
|  常体 (だ/である style):                                  |
|  • Academic essays (論文)                                 |
|  • Reports (レポート)                                    |
|  • Newspaper articles                                    |
|  • Diary entries (日記)                                   |
|                                                          |
|  NEVER MIX the two styles in one document.               |
+----------------------------------------------------------+
```

## 2. Punctuation and Formatting

### 2.1 Japanese Punctuation Marks

| Mark | Name | Usage |
|------|------|-------|
| 。 | 句点（くてん） | Period — ends a sentence |
| 、 | 読点（とうてん） | Comma — separates clauses |
| 「」 | 鉤括弧（かぎかっこ） | Quotation marks |
| 『』 | 二重鉤括弧 | Quotes within quotes, book/movie titles |
| ・ | 中黒（なかぐろ） | Separator between items (like a bullet) |
| ～ | 波ダッシュ | Range indicator (9時～5時) |
| … | 三点リーダー | Ellipsis (always use two: ……) |
| ― | ダッシュ | Dash (always use two: ――) |
| （） | 丸括弧 | Parentheses |
| ※ | 米印（こめじるし） | Note/attention marker |

### 2.2 Manuscript Paper Rules (原稿用紙)

Traditional Japanese writing on 原稿用紙 (genko yoshi — 400-character manuscript paper)
follows strict rules:

```
+----------------------------------------------------------+
|         原稿用紙 RULES                                    |
+----------------------------------------------------------+
|                                                          |
|  1. Title: Line 1, leave 2-3 squares blank, then title   |
|  2. Name: Line 2, end before the last square             |
|  3. New paragraph: Indent 1 square (一字下げ)             |
|  4. Punctuation: NEVER at the start of a line            |
|     → Move it to the end of the previous line             |
|     → Or share the last square with the previous char     |
|  5. Small kana (っ, ゃ, ゅ, ょ):                          |
|     → CAN start a line (unlike punctuation)              |
|  6. Numbers:                                              |
|     → Vertical: 漢数字 (一, 二, 三)                       |
|     → Horizontal: Arabic numerals (1, 2, 3)              |
|  7. 「 opening bracket: starts at the beginning of a line |
|  8. Dialogue: Each new speaker starts a new line          |
+----------------------------------------------------------+
```

### 2.3 Horizontal vs Vertical Writing

| Feature | Horizontal (横書き) | Vertical (縦書き) |
|---------|---------------------|-------------------|
| Direction | Left to right | Top to bottom, right to left |
| Used in | Web, science, business | Literature, newspapers, formal letters |
| Numbers | Arabic (1, 2, 3) | Kanji (一、二、三) |
| Punctuation | Standard placement | Same rules apply rotated |

## 3. Essay Structure (序論・本論・結論)

### 3.1 The Three-Part Structure

Japanese essays follow a clear three-part pattern:

```
+----------------------------------------------------------+
|         JAPANESE ESSAY STRUCTURE                         |
+----------------------------------------------------------+
|                                                          |
|  序論 (じょろん) — Introduction                           |
|  ├── State the topic / issue                             |
|  ├── Provide brief background                            |
|  └── Present your thesis / main argument                 |
|      Phrase: ～について論じる / ～を考察する                |
|                                                          |
|  本論 (ほんろん) — Main Body                              |
|  ├── Point 1 + evidence / examples                       |
|  ├── Point 2 + evidence / examples                       |
|  ├── Counterargument + rebuttal (optional)               |
|  └── Phrase: まず... / 次に... / さらに...                |
|                                                          |
|  結論 (けつろん) — Conclusion                              |
|  ├── Summarize main points                               |
|  ├── Restate thesis                                      |
|  └── Final thought / future outlook                      |
|      Phrase: 以上のことから... / 結論として...              |
|                                                          |
+----------------------------------------------------------+
```

### 3.2 Alternative Structure: 起承転結

A traditional Japanese structure often used in shorter pieces:

| Part | Kanji | Role | English |
|------|-------|------|---------|
| 起 | き | Introduction | Introduce the topic |
| 承 | しょう | Development | Expand on it |
| 転 | てん | Twist / Turn | Present a different angle or counterpoint |
| 結 | けつ | Conclusion | Bring it all together |

### 3.3 Essay Opening Patterns

| Pattern | Example |
|---------|---------|
| Question | 「なぜ日本語を学ぶ外国人が増えているのだろうか。」 |
| Statistic | 「2024年、日本語能力試験の受験者数は過去最高を記録した。」 |
| Definition | 「グローバル化とは、国境を越えた交流が活発になる現象である。」 |
| Quotation | 「『言葉は文化の鏡である』と言われている。」 |
| General statement | 「現代社会において、外国語学習の重要性はますます高まっている。」 |

### 3.4 Essay Closing Patterns

| Pattern | Example |
|---------|---------|
| Summary | 「以上のことから、○○であると結論づけられる。」 |
| Restatement | 「本稿で論じたように、○○は○○にとって不可欠である。」 |
| Future outlook | 「今後、○○がさらに発展することが期待される。」 |
| Call to action | 「私たち一人一人が○○について考える必要があるのではないだろうか。」 |
| Open question | 「この問題に対する答えは、まだ見つかっていない。」 |

### 3.5 Sample Essay Paragraph (常体)

```
  現代社会において、人工知能（AI）の発展は目覚ましいものがある。
特に、自然言語処理の分野では、大規模言語モデルの登場により、
機械が人間のような文章を生成できるようになった。しかし、
これは言語学習にどのような影響を与えるのだろうか。

  まず、AI翻訳ツールの精度向上により、外国語を学ぶ必要性を
疑問視する声がある。確かに、日常的なコミュニケーションにおいて、
翻訳アプリは便利なツールとなりつつある。しかしながら、言語学習の
意義は単なる意思疎通にとどまらない。言語を学ぶことは、その文化を
理解し、異なる思考方法を身につけることでもあるのだ。
```

## 4. Formal Letter Writing (手紙の書き方)

### 4.1 Letter Structure

Japanese formal letters follow a highly prescribed format:

```
+----------------------------------------------------------+
|        FORMAL LETTER STRUCTURE (手紙の構成)               |
+----------------------------------------------------------+
|                                                          |
|  ① 前文 (Opening section)                                |
|     ├── 頭語 (Opening salutation): 拝啓                   |
|     ├── 時候の挨拶 (Seasonal greeting)                    |
|     └── 安否の挨拶 (Well-wishing phrase)                  |
|                                                          |
|  ② 主文 (Main body)                                      |
|     ├── 起こし言葉: さて、 / ところで、                    |
|     └── Main content                                     |
|                                                          |
|  ③ 末文 (Closing section)                                |
|     ├── 結びの挨拶 (Closing wishes)                       |
|     └── 結語 (Closing salutation): 敬具                   |
|                                                          |
|  ④ 後付け (Postscript info)                               |
|     ├── 日付 (Date)                                      |
|     ├── 差出人名 (Sender name)                            |
|     └── 宛名 (Recipient name + 様)                       |
|                                                          |
+----------------------------------------------------------+
```

### 4.2 Opening and Closing Pairs (頭語 / 結語)

| Type | Opening (頭語) | Closing (結語) | Usage |
|------|---------------|---------------|-------|
| General | 拝啓 (はいけい) | 敬具 (けいぐ) | Standard formal |
| More formal | 謹啓 (きんけい) | 謹白 (きんぱく) | Very formal occasions |
| Reply | 拝復 (はいふく) | 敬具 (けいぐ) | Replying to a letter |
| Urgent | 急啓 (きゅうけい) | 草々 (そうそう) | Urgent matters |
| Casual letter | 前略 (ぜんりゃく) | 草々 (そうそう) | Skipping formalities |

> **Rule**: 頭語 and 結語 must always be used as a matching pair. Mixing them is incorrect.

### 4.3 Seasonal Greetings (時候の挨拶)

Each month has traditional seasonal greetings:

| Month | Greeting | Translation |
|-------|----------|-------------|
| January (1月) | 新春の候 / 厳寒の候 | Season of new spring / severe cold |
| February (2月) | 立春の候 / 余寒の候 | Season of spring's arrival / lingering cold |
| March (3月) | 早春の候 / 春暖の候 | Season of early spring / spring warmth |
| April (4月) | 陽春の候 / 桜花の候 | Season of warm spring / cherry blossoms |
| May (5月) | 新緑の候 / 薫風の候 | Season of fresh green / fragrant breeze |
| June (6月) | 入梅の候 / 初夏の候 | Season of rainy season / early summer |
| July (7月) | 盛夏の候 / 猛暑の候 | Season of midsummer / extreme heat |
| August (8月) | 残暑の候 / 晩夏の候 | Season of lingering heat / late summer |
| September (9月) | 初秋の候 / 秋涼の候 | Season of early autumn / autumn cool |
| October (10月) | 秋冷の候 / 紅葉の候 | Season of autumn chill / autumn leaves |
| November (11月) | 晩秋の候 / 向寒の候 | Season of late autumn / approaching cold |
| December (12月) | 初冬の候 / 師走の候 | Season of early winter / year-end bustle |

### 4.4 Sample Formal Letter

```
                                          拝啓

  陽春の候、○○様におかれましては
ますますご清祥のこととお慶び申し上げます。
  平素は格別のご高配を賜り、厚く御礼申し上げます。

  さて、このたびは○○の件で
ご連絡させていただきました。
  ○○につきまして、ご検討いただけますと幸いです。

  末筆ながら、皆様のご健康と
ご多幸をお祈り申し上げます。

                                          敬具

  令和八年四月十日
                        山田太郎

  鈴木花子様
```

## 5. Resume / CV Format (履歴書)

### 5.1 Japanese Resume Structure

The Japanese 履歴書 follows an extremely standardized format, very different from
Western-style resumes:

```
+----------------------------------------------------------+
|         日本式履歴書 (JAPANESE RESUME)                     |
+----------------------------------------------------------+
|                                                          |
|  DATE: 令和○年○月○日現在                                 |
|                                                          |
|  PHOTO: Passport-style, 3cm x 4cm, glued upper right    |
|                                                          |
|  ┌──────────────────────────────────────────┐            |
|  │ ふりがな:  やまだ たろう                    │            |
|  │ 氏名:     山田 太郎                        │            |
|  │ 生年月日:  令和○年○月○日生（○歳）          │            |
|  │ 性別:     男                               │            |
|  │ 住所:     〒___-____ ○○県○○市...          │            |
|  │ 電話:     090-XXXX-XXXX                   │            |
|  │ E-mail:   xxxxxx@xxx.com                  │            |
|  └──────────────────────────────────────────┘            |
|                                                          |
|  学歴・職歴 (Education & Work History)                    |
|  ├── Listed in CHRONOLOGICAL order                       |
|  ├── 学歴 header, then education entries                 |
|  ├── 職歴 header, then work entries                      |
|  └── End with 以上 (That is all)                         |
|                                                          |
|  免許・資格 (Licenses & Qualifications)                   |
|  ├── JLPT, driver's license, etc.                        |
|  └── Most recent first or most relevant                  |
|                                                          |
|  志望動機 (Reason for Application)                        |
|  └── 3-5 sentences explaining why this company            |
|                                                          |
|  本人希望記入欄 (Personal Requests)                       |
|  └── 「貴社の規定に従います」 if no special requests      |
|                                                          |
+----------------------------------------------------------+
```

### 5.2 Key Differences from Western Resumes

| Feature | Japanese 履歴書 | Western Resume |
|---------|----------------|----------------|
| Photo | Required (formal photo) | Usually not included |
| Date of birth | Required | Often omitted |
| Gender | Listed | Often omitted |
| Format | Standardized template | Flexible design |
| Length | 1-2 pages (fixed format) | Varies |
| Handwriting | Traditionally handwritten (now typed is OK) | Always typed |
| Education | Chronological (oldest first) | Reverse chronological |
| Personal statement | 志望動機 (mandatory) | Optional cover letter |

### 5.3 Education History Phrases

| Event | Japanese |
|-------|----------|
| Entered school | ○○高等学校 入学 |
| Graduated | ○○高等学校 卒業 |
| Entered university | ○○大学 ○○学部 入学 |
| Graduated university | ○○大学 ○○学部 卒業 |
| Dropped out | ○○大学 ○○学部 中途退学 |
| Currently enrolled | ○○大学 ○○学部 在学中 |

### 5.4 Work History Phrases

| Event | Japanese |
|-------|----------|
| Joined company | ○○株式会社 入社 |
| Transferred dept. | ○○部に異動 |
| Promoted | ○○に昇進 |
| Left company (personal) | 一身上の都合により退職 |
| Left company (company reasons) | 会社都合により退職 |
| Currently employed | 現在に至る |

## 6. Report Writing (レポート)

### 6.1 Report Structure

Academic and business reports follow this structure:

```
+----------------------------------------------------------+
|         REPORT STRUCTURE (レポートの構成)                  |
+----------------------------------------------------------+
|                                                          |
|  表紙 (Cover page)                                       |
|  ├── Title                                               |
|  ├── Course/department name                              |
|  ├── Author name and student/employee number             |
|  └── Date                                                |
|                                                          |
|  目次 (Table of Contents)                                |
|  └── Section headings with page numbers                  |
|                                                          |
|  はじめに / 序論 (Introduction)                            |
|  ├── Background and purpose                              |
|  └── Scope of the report                                 |
|                                                          |
|  本文 (Body)                                              |
|  ├── 第1章: First main section                           |
|  ├── 第2章: Second main section                          |
|  └── Evidence, data, analysis                            |
|                                                          |
|  結論 / おわりに (Conclusion)                              |
|  ├── Summary of findings                                 |
|  └── Future considerations                               |
|                                                          |
|  参考文献 (References / Bibliography)                     |
|  └── Listed in order of citation                         |
|                                                          |
+----------------------------------------------------------+
```

### 6.2 Report Writing Style

Reports use 常体 (だ/である style) and avoid personal opinions unless specifically asked.

| Do | Don't |
|----|-------|
| ～と考えられる (it can be considered) | ～と思う (I think) |
| ～であると言える (it can be said) | ～だと感じた (I felt) |
| ～ことが明らかになった (it became clear) | ～のは面白い (it's interesting) |
| データによれば (according to data) | たぶん (probably) |
| ～と推測される (it is inferred) | ～かもしれない (maybe) |

### 6.3 Citing Sources in Japanese

| Citation Type | Format |
|---------------|--------|
| Book | 著者名（出版年）『書名』出版社 |
| Journal article | 著者名（発表年）「論文タイトル」『雑誌名』巻号, ページ |
| Website | 著者名（更新年）「ページタイトル」URL（閲覧日: ○年○月○日） |
| In-text citation | ○○（2024）によれば… / ○○（2024: 45）は…と述べている |

### 6.4 Useful Report Phrases

| Function | Phrase |
|----------|--------|
| Stating purpose | 本レポートでは○○について考察する。 |
| Background | ○○は近年注目されている問題である。 |
| Defining | ○○とは、○○のことを指す。 |
| Presenting data | 図1に示すように、○○である。 |
| Comparing | ○○と比較すると、○○であることがわかる。 |
| Cause/effect | ○○の結果、○○が生じた。 |
| Concluding | 以上の分析から、○○であると結論づけられる。 |
| Limitation | 本研究の限界として、○○が挙げられる。 |

## 7. Email Writing (メールの書き方)

### 7.1 Casual vs Formal Emails

While Lesson 21 covered business emails in depth, this section covers the broader
spectrum of email writing.

| Element | Casual (to friends) | Semi-formal (to teacher) | Formal (to company) |
|---------|-------------------|------------------------|-------------------|
| Greeting | ○○ちゃん！ / やっほー | ○○先生、こんにちは | ○○様、お世話になっております |
| Tone | タメ口 (casual) | です/ます | ビジネス敬語 |
| Closing | じゃあね！ / またね | よろしくお願いします | 何卒よろしくお願い申し上げます |
| Emoji | OK ✓ | Minimal | Never ✗ |

### 7.2 Email to a Teacher/Professor

```
件名：○○授業のレポート提出について

○○先生

こんにちは。○○学部○年の山田太郎です。

○月○日締め切りのレポートについてご質問がございます。
参考文献の数に指定はありますでしょうか。

お忙しいところ恐れ入りますが、
ご回答いただけますと幸いです。

よろしくお願いいたします。

○○学部○年
山田太郎
学籍番号：12345678
```

### 7.3 Inquiry Email to a Company

```
件名：商品○○についてのお問い合わせ

初めてメールをお送りいたします。
○○と申します。

いくつかお伺いしたい点がございます。

1. ○○の仕様について
2. 納期について
3. 大量注文時の割引について

ご多忙のところ恐縮ですが、
ご回答いただけますと幸いです。

何卒よろしくお願い申し上げます。

山田太郎
TEL: 090-XXXX-XXXX
E-mail: taro.yamada@xxx.com
```

## 8. Common Writing Mistakes

### 8.1 Mistakes by Non-Native Writers

| Mistake | Incorrect | Correct | Explanation |
|---------|-----------|---------|-------------|
| Style mixing | ～です。～である。 | Pick one and stay consistent | Never mix 敬体 and 常体 |
| は vs が confusion | 私が学生です。 | 私は学生です。 | は for topic, が for new info / subject |
| Particle dropping | 東京行く | 東京に行く | Particles cannot be omitted in writing |
| Spoken forms | ～ちゃった / ～じゃない | ～てしまった / ～ではない | Use written forms in formal writing |
| Run-on sentences | ～て、～て、～て、～た。 | Break into shorter sentences | Too many て-form connections |
| Wrong counter | 人が3つ | 人が3人 | Use appropriate counter words |
| Kanji overuse | 出来る事を全て遣る | できることをすべてやる | Some words are conventionally in hiragana |
| Wrong paragraph indent | No indent | 一字下げ (1-space indent) | Always indent new paragraphs |

### 8.2 Words Commonly Written in Hiragana (Not Kanji)

Even though these words have kanji, convention is to write them in hiragana:

| Kanji Form | Hiragana Form | Meaning |
|-----------|---------------|---------|
| 出来る | できる | can do |
| 事 | こと | thing (abstract) |
| 物 | もの | thing (concrete) |
| 所 | ところ | place / aspect |
| 時 | とき | when |
| 為 | ため | for / because |
| 通り | とおり | as / in the way of |
| 下さい | ください | please |
| 頂く | いただく | to receive (humble) |
| 致す | いたす | to do (humble) |

## 9. Useful Connectors and Transition Phrases

### 9.1 Connectors by Function

**Addition**:
| Japanese | Usage Level | English |
|----------|-------------|---------|
| また | Written/Spoken | Also, moreover |
| さらに | Written/Spoken | Furthermore |
| 加えて | Written | In addition |
| そのうえ | Written/Spoken | On top of that |
| なおかつ | Written | Moreover (formal) |

**Contrast**:
| Japanese | Usage Level | English |
|----------|-------------|---------|
| しかし | Written/Spoken | However |
| しかしながら | Written (formal) | Nevertheless |
| 一方（で） | Written/Spoken | On the other hand |
| それに対して | Written | In contrast to that |
| ～にもかかわらず | Written | Despite, regardless of |

**Cause and Effect**:
| Japanese | Usage Level | English |
|----------|-------------|---------|
| したがって | Written | Therefore |
| そのため | Written/Spoken | For that reason |
| その結果 | Written | As a result |
| ～ことから | Written | From the fact that |
| ゆえに | Written (formal) | Hence |

**Example and Explanation**:
| Japanese | Usage Level | English |
|----------|-------------|---------|
| 例えば | Written/Spoken | For example |
| 具体的には | Written | Specifically |
| すなわち | Written | Namely, in other words |
| つまり | Written/Spoken | In short |
| 言い換えれば | Written | In other words |

**Conclusion**:
| Japanese | Usage Level | English |
|----------|-------------|---------|
| 以上のことから | Written | From the above |
| 結論として | Written | In conclusion |
| 要するに | Written/Spoken | In summary |
| このように | Written | In this way |
| 総じて | Written (formal) | Overall |
