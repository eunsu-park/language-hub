## 1. と思う — "I Think"

### Formation

| Type | Connection | Example |
|------|-----------|---------|
| Verb (plain) | + と思う | 行く**と思う** (I think [someone] will go) |
| い-adjective | + と思う | 高い**と思う** (I think it's expensive) |
| な-adjective | **だ** + と思う | 便利**だと思う** (I think it's convenient) |
| Noun | **だ** + と思う | 学生**だと思う** (I think [they're] a student) |
| Negative | ないと思う | 来ない**と思う** (I think [they] won't come) |

### Key Patterns

#### A. と思う vs. と思っている

| Pattern | Meaning | Example |
|---------|---------|---------|
| と思う | Momentary thought / opinion | 明日は雨だ**と思う**。(I think it'll rain tomorrow.) |
| と思っている | Sustained belief / ongoing opinion | 日本語は面白い**と思っている**。(I think Japanese is interesting. — held belief) |

> **Important**: For third person, use と思っている:
> ○ 田中さんは来ない**と思っている**。(Tanaka thinks [someone] won't come.)
> ✗ 田中さんは来ない**と思う**。(Sounds like YOUR thought about Tanaka coming.)

#### B. と思う with Volitional (Plans/Intentions)

> **来年、日本に行こうと思う。** — I think I'll go to Japan next year.
> **新しい車を買おうと思っている。** — I'm thinking of buying a new car.

#### C. Asking Opinions: どう思いますか

> **この映画についてどう思いますか。** — What do you think about this movie?
> **日本の食べ物はどう思う？** — What do you think of Japanese food?

### Nuance: と思う vs. と思わない

"I don't think X will happen" is usually expressed differently from English:

```
English:  I don't think he will come.
Japanese: 彼は来ないと思う。        (I think he won't come.)
          NOT: 彼は来ると思わない。   (Less natural)
```

> In Japanese, negate the content of the thought, not the thinking itself.

### Example Sentences

1. **あの人は日本人だと思う。** — I think that person is Japanese.
2. **この問題は難しいと思います。** — I think this problem is difficult.
3. **彼女はもう帰ったと思う。** — I think she already went home.
4. **将来、教師になろうと思っています。** — I'm thinking of becoming a teacher in the future.
5. **日本の電車は便利だと思いませんか。** — Don't you think Japanese trains are convenient?

## 2. ようだ — "It Seems / It Appears"

ようだ expresses a **judgment based on evidence** the speaker has observed. It is relatively formal.

### Formation

| Type | Connection | Example |
|------|-----------|---------|
| Verb (plain) | + ようだ | 雨が降る**ようだ** |
| い-adjective | + ようだ | 難しい**ようだ** |
| な-adjective | **な** + ようだ | 静か**なようだ** |
| Noun | **の** + ようだ | 風邪**のようだ** |

### Uses

#### A. Judgment Based on Observation

> **外は寒いようだ。**（窓の外を見て）
> It seems cold outside. (Looking out the window.)

> **彼は忙しいようです。** — He seems to be busy.

#### B. Simile / Comparison ("like," "as if")

When ような (before a noun) or ように (before a verb) is used:

> **雪のような白い肌。** — Skin as white as snow.
> **子供のように笑った。** — Laughed like a child.
> **夢のような話だ。** — It's a story like a dream.

#### C. ように + Verb (Purpose / Manner)

> **聞こえるように大きな声で話した。** — I spoke loudly so that [they] could hear.
> **忘れないようにメモした。** — I took notes so I wouldn't forget.

### Conjugation of ようだ

| Form | Japanese |
|------|----------|
| Plain | ようだ |
| Polite | ようです |
| Negative | ようではない |
| Past | ようだった |
| Adverbial (before verb) | ように |
| Attributive (before noun) | ような |

### Example Sentences

1. **道に迷ったようだ。** — It seems like I've gotten lost.
2. **彼女は怒っているようです。** — She seems to be angry.
3. **まるで映画のような景色だ。** — It's scenery just like a movie.
4. **日本語が上手になるように毎日練習している。** — I practice every day so that my Japanese will improve.
5. **試験に合格したようだ。** — It seems like I passed the exam.

## 3. みたいだ — "It Seems Like" (Casual)

みたいだ is the **casual equivalent** of ようだ. It carries the same meanings but is used in everyday conversation.

### Formation

| Type | Connection | Example |
|------|-----------|---------|
| Verb (plain) | + みたいだ | 帰った**みたいだ** |
| い-adjective | + みたいだ | 寒い**みたいだ** |
| な-adjective | + みたいだ | 元気**みたいだ** |
| Noun | + みたいだ | 学生**みたいだ** |

> **Key difference from ようだ**: みたいだ attaches directly — no の or な connector needed.

### Uses

#### A. Judgment Based on Evidence (Same as ようだ but casual)

> **雨が降っているみたいだ。** — It seems like it's raining.
> **彼はもう寝たみたいだよ。** — It looks like he's already asleep.

#### B. Simile / Comparison ("like")

> **子供みたいに笑う。** — Laugh like a child.
> **夢みたいだ！** — It's like a dream!
> **本当のことみたいに聞こえる。** — It sounds like it's true.

### Conjugation of みたいだ

| Form | Japanese |
|------|----------|
| Plain | みたいだ |
| Polite | みたいです |
| Negative | みたいではない / みたいじゃない |
| Past | みたいだった |
| Before verb | みたいに |
| Before noun | みたいな |

### ようだ vs. みたいだ

| Feature | ようだ | みたいだ |
|---------|--------|---------|
| Formality | Formal / written | Casual / spoken |
| Connection to な-adj | 静かな + ようだ | 静か + みたいだ |
| Connection to noun | 学生の + ようだ | 学生 + みたいだ |
| Meaning | Identical | Identical |

### Example Sentences

1. **電車が遅れているみたいだ。** — It seems like the train is delayed.
2. **この料理、ちょっと辛いみたいだね。** — This food seems a bit spicy, huh.
3. **お姫様みたいなドレスだね。** — It's a princess-like dress, isn't it.
4. **田中さん、風邪みたいだよ。** — Tanaka seems to have a cold.
5. **子供みたいにはしゃいでいた。** — He was having fun like a child.

## 4. らしい — "Apparently / I Heard"

らしい expresses information the speaker has **heard from others** or **indirect evidence**. The speaker is reporting, not directly observing.

### Formation

| Type | Connection | Example |
|------|-----------|---------|
| Verb (plain) | + らしい | 来る**らしい** |
| い-adjective | + らしい | 高い**らしい** |
| な-adjective | + らしい | 大変**らしい** |
| Noun | + らしい | 本当**らしい** |

> Like みたいだ, らしい attaches directly to all word types.

### Uses

#### A. Hearsay / Secondhand Information

> **明日は雨が降るらしい。** — Apparently it'll rain tomorrow. (I heard from someone / saw the forecast.)
> **田中さんは来月結婚するらしい。** — I heard Tanaka is getting married next month.
> **あの店はおいしいらしいよ。** — That restaurant is apparently good.

#### B. Typicality: Noun + らしい ("typical of," "-like")

When らしい attaches directly to a noun, it means "characteristic of" or "worthy of":

> **男らしい** — manly, masculine
> **子供らしい** — childlike (in a positive sense)
> **春らしい天気** — spring-like weather (typical spring weather)
> **学生らしくない** — not student-like (not behaving as a student should)

> **Contrast with みたいだ**:
> 子供**らしい** = childlike (positive: natural, expected behavior for a child)
> 子供**みたいだ** = like a child (neutral/negative: could mean immature)

### Conjugation of らしい

| Form | Japanese |
|------|----------|
| Plain | らしい |
| Polite | らしいです |
| Negative | らしくない |
| Past | らしかった |

### Example Sentences

1. **来週テストがあるらしい。** — Apparently there's a test next week.
2. **彼女は有名な歌手らしい。** — She's apparently a famous singer.
3. **今日は春らしい天気ですね。** — It's spring-like weather today, isn't it.
4. **あの映画は面白くないらしいよ。** — I heard that movie isn't interesting.
5. **社長は厳しい人らしい。** — The company president is apparently a strict person.

## 5. そうだ — Two Distinct Meanings

そうだ has **two completely different meanings** depending on how it connects to the preceding word.

### Meaning 1: Appearance (〜そうだ) — "It Looks Like"

Based on **visual impression** or **intuition** — the speaker looks at something and makes a judgment.

#### Formation (Appearance)

| Type | Connection | Example |
|------|-----------|---------|
| Verb masu-stem | + そうだ | 降り**そうだ** (looks like it'll rain) |
| い-adjective (drop い) | + そうだ | おいし**そうだ** (looks delicious) |
| な-adjective (stem) | + そうだ | 元気**そうだ** (looks healthy) |

> **Special cases**:
> いい → **よさそうだ** (looks good) — NOT *いそうだ
> ない → **なさそうだ** (doesn't look like) — NOT *なそうだ

#### Example Sentences (Appearance)

> **このケーキはおいしそうだ。** — This cake looks delicious.
> **雨が降りそうだ。** — It looks like it's going to rain.
> **彼は元気そうだ。** — He looks healthy/energetic.
> **難しそうな本だね。** — That looks like a difficult book.
> **あの人は優しそうだ。** — That person looks kind.

### Meaning 2: Hearsay (〜そうだ) — "I Heard That"

The speaker is **reporting information** from a source (news, someone else, etc.).

#### Formation (Hearsay)

| Type | Connection | Example |
|------|-----------|---------|
| Verb (plain) | + そうだ | 来る**そうだ** (I heard [they]'ll come) |
| い-adjective | + そうだ | 高い**そうだ** (I heard it's expensive) |
| な-adjective + **だ** | + そうだ | 便利**だそうだ** (I heard it's convenient) |
| Noun + **だ** | + そうだ | 先生**だそうだ** (I heard [they're] a teacher) |

#### Example Sentences (Hearsay)

> **明日は雨だそうだ。** — I heard it'll rain tomorrow.
> **あの映画は面白いそうです。** — I heard that movie is interesting.
> **田中さんは来月転勤するそうだ。** — I heard Tanaka is transferring next month.
> **このレストランは有名だそうだ。** — I heard this restaurant is famous.

### Critical Distinction

```
+------------------------------------------------------------------+
|  APPEARANCE vs. HEARSAY — KEY DIFFERENCES                        |
+------------------------------------------------------------------+
|                                                                  |
|  Appearance (looks like):  Verb MASU-STEM + そうだ               |
|                            Adj STEM + そうだ                     |
|                                                                  |
|  Hearsay (I heard):        PLAIN FORM + そうだ                   |
|                                                                  |
+------------------------------------------------------------------+
|                                                                  |
|  おいしそうだ = It LOOKS delicious. (I can see the food.)        |
|  おいしいそうだ = I HEARD it's delicious. (Someone told me.)     |
|                                                                  |
|  降りそうだ = It LOOKS like it'll rain. (I see dark clouds.)     |
|  降るそうだ = I HEARD it'll rain. (I saw the weather forecast.)  |
|                                                                  |
+------------------------------------------------------------------+
```

### Conjugation of そうだ (Appearance)

| Form | Japanese | Example |
|------|----------|---------|
| Plain | そうだ | おいしそうだ |
| Polite | そうです | おいしそうです |
| Negative | そうにない / そうもない | 降りそうにない |
| Before noun | そうな | おいしそうなケーキ |
| Before verb | そうに | おいしそうに食べる |

> **Hearsay そうだ** does not conjugate — it is always そうだ/そうです.

## 6. はずだ — "Should Be / Expected"

はずだ expresses the speaker's **logical expectation** or **strong belief** based on reasoning or evidence.

### Formation

| Type | Connection | Example |
|------|-----------|---------|
| Verb (plain) | + はずだ | 来る**はずだ** (should come) |
| い-adjective | + はずだ | 高い**はずだ** (should be expensive) |
| な-adjective + **な** | + はずだ | 簡単**なはずだ** (should be easy) |
| Noun + **の** | + はずだ | 休み**のはずだ** (should be a holiday) |
| Negative | ないはずだ | 来ない**はずだ** (shouldn't come) |

### Uses

#### A. Logical Expectation

> **もう届いたはずだ。**（先週送ったから）
> It should have arrived by now. (Because I sent it last week.)

> **彼は知っているはずだ。** — He should know. (He was told.)

#### B. "Not Supposed to"

> **こんなに高いはずがない！** — It can't possibly be this expensive!
> **彼が嘘をつくはずがない。** — There's no way he would lie.

> **はずがない** is a strong negation: "there's no way that..."

#### C. Contrast with Reality

> **簡単なはずだったのに、難しかった。**
> It was supposed to be easy, but it was difficult.

### はずだ vs. Other Expressions

| Expression | Certainty Level | Based On |
|-----------|----------------|----------|
| はずだ | High (logical) | Reasoning / evidence |
| ようだ | Moderate | Direct observation |
| らしい | Moderate | Hearsay / indirect |
| かもしれない | Low (maybe) | Possibility |

### Example Sentences

1. **電車は8時に来るはずです。** — The train should come at 8:00.
2. **彼女はもう帰ったはずだ。** — She should have already gone home.
3. **そんなはずはない！** — That can't be! (No way!)
4. **明日は休みのはずだけど。** — Tomorrow should be a day off, but...
5. **このレストランは安いはずだよ。** — This restaurant should be cheap (I read a review).

## 7. ことがある — "There Are Times When"

This pattern expresses that something **occasionally happens** or that one has **had an experience**.

### Formation

| Use | Formation | Example |
|-----|-----------|---------|
| Occasional occurrence | Verb (dictionary form) + ことがある | 忘れる**ことがある** |
| Past experience | Verb (た-form) + ことがある | 行った**ことがある** |

### A. Occasional Occurrence (Dictionary Form + ことがある)

Describes something that sometimes happens:

> **朝ご飯を食べないことがある。** — There are times when I don't eat breakfast.
> **電車が遅れることがある。** — The train is sometimes late.
> **夜中に目が覚めることがある。** — I sometimes wake up in the middle of the night.

### B. Past Experience (た-form + ことがある)

Describes whether one has had a particular experience:

> **日本に行ったことがある。** — I have been to Japan. (I have the experience.)
> **納豆を食べたことがない。** — I have never eaten natto.
> **富士山に登ったことがありますか。** — Have you ever climbed Mt. Fuji?

### Asking About Experience

> Q: **寿司を作ったことがありますか。** — Have you ever made sushi?
> A: **はい、一度だけ作ったことがあります。** — Yes, I've made it once.
> A: **いいえ、一度もありません。** — No, not even once.

### Example Sentences

1. **彼に会ったことがある。** — I have met him before.
2. **朝寝坊することがある。** — I sometimes oversleep.
3. **この映画を見たことがありますか。** — Have you ever seen this movie?
4. **日本語で夢を見たことがある。** — I've dreamed in Japanese before.
5. **授業に遅刻することがある。** — I'm sometimes late for class.

## 8. Comparison Chart

### Evidence vs. Source of Information

| Expression | What it expresses | Evidence source | Certainty |
|-----------|-------------------|----------------|-----------|
| と思う | Personal opinion | Speaker's own thinking | Speaker's belief |
| ようだ | Inference | Direct observation (formal) | Moderate |
| みたいだ | Inference | Direct observation (casual) | Moderate |
| らしい | Hearsay/inference | Heard from others | Moderate |
| そうだ (appearance) | Visual impression | Looking at something | Moderate |
| そうだ (hearsay) | Report | Specific source | Moderate |
| はずだ | Logical expectation | Reasoning/knowledge | High |

### Quick Reference: "It's going to rain"

| Japanese | Nuance |
|----------|--------|
| 雨が降る**と思う** | I think it'll rain. (my opinion) |
| 雨が降る**ようだ** | It seems it'll rain. (I see dark clouds — formal) |
| 雨が降る**みたいだ** | It seems it'll rain. (I see dark clouds — casual) |
| 雨が降る**らしい** | Apparently it'll rain. (someone told me / indirect) |
| 雨が降り**そうだ** | It looks like it'll rain. (the sky looks threatening) |
| 雨が降る**そうだ** | I heard it'll rain. (weather forecast said so) |
| 雨が降る**はずだ** | It should rain. (the forecast said 90% chance) |

## 9. Common Mistakes and Pitfalls

### Mistake 1: Confusing そうだ Appearance vs. Hearsay

```
✗ おいしいそうだ = I heard it's delicious (hearsay)
  おいしそうだ  = It looks delicious (appearance)

Remember: Masu-stem/adj-stem + そうだ = appearance
          Plain form + そうだ = hearsay
```

### Mistake 2: Using と思う for Third Person Without ている

```
✗ 田中さんはそう思う。 (Sounds like YOUR thought)
○ 田中さんはそうと思っている。 (Tanaka thinks so — his ongoing belief)
```

### Mistake 3: いい and ない with Appearance そうだ

```
✗ いそうだ / なそうだ
○ よさそうだ (いい → よさそう)
○ なさそうだ (ない → なさそう)
```

### Mistake 4: Mixing Up らしい (Hearsay) and らしい (Typicality)

```
彼は学生らしい。
  = He's apparently a student. (hearsay)
  OR
  = He's student-like. (typical behavior)
Context distinguishes them.

彼は学生らしい学生だ。 = He's a student worthy of being called a student.
                        (typicality — uses both meanings)
```

## 10. Practice Exercises

### Exercise 1: と思う Practice

Express these opinions using と思う.

1. Japanese is interesting. (I think...)
2. He won't come tomorrow. (I think...)
3. I'm thinking of studying abroad next year. (sustained plan)
4. What do you think about this book? (asking opinion)

<details>
<summary>Answers</summary>

1. 日本語は面白い**と思う / と思います**。
2. 彼は明日来ない**と思う**。(Negate the content, not と思う)
3. 来年留学しよう**と思っている / と思っています**。(Volitional + と思っている for sustained plan)
4. この本についてどう**思いますか**。

</details>

### Exercise 2: Appearance vs. Hearsay そうだ

Choose the correct form of そうだ.

1. (Looking at a cake) This cake looks delicious!
2. (Someone told you) I heard that restaurant is delicious.
3. (Looking at the sky) It looks like it will rain.
4. (Read in the news) I heard there was an earthquake.

<details>
<summary>Answers</summary>

1. このケーキはおいし**そうだ**。(Appearance: おいし + そう)
2. あのレストランはおいしい**そうだ**。(Hearsay: おいしい + そうだ)
3. 雨が降り**そうだ**。(Appearance: 降り + そう)
4. 地震があった**そうだ**。(Hearsay: あった + そうだ)

</details>

### Exercise 3: ようだ / みたいだ / らしい

Fill in with the most appropriate expression.

1. (Looking at someone shivering) 彼は寒い ________ 。(direct observation, casual)
2. (A coworker told you) 来週会議がある ________ 。(hearsay)
3. (Noticing wet umbrellas) 外は雨が降っている ________ 。(direct observation, formal)
4. 今日は春 ________ 天気ですね。(typical of spring)

<details>
<summary>Answers</summary>

1. 彼は寒い**みたいだ**。(Direct observation, casual = みたいだ)
2. 来週会議がある**らしい**。(Hearsay = らしい)
3. 外は雨が降っている**ようだ**。(Direct observation, formal = ようだ)
4. 今日は春**らしい**天気ですね。(Typical of = noun + らしい)

</details>

### Exercise 4: はずだ

Express expectations using はずだ.

1. The store should be open. (It's still business hours.)
2. She should know. (She was at the meeting.)
3. There's no way this is wrong! (strong negation)
4. It was supposed to be easy, but it was hard.

<details>
<summary>Answers</summary>

1. 店は開いている**はずだ**。(まだ営業時間だから。)
2. 彼女は知っている**はずだ**。(会議にいたから。)
3. これが間違っている**はずがない**！(はずがない = no way)
4. 簡単な**はずだったのに**、難しかった。(はずだった = was supposed to be)

</details>

### Exercise 5: ことがある

Complete using ことがある.

1. Have you ever been to Kyoto?
2. I sometimes forget my homework.
3. I've never eaten raw fish.
4. There are times when I can't sleep at night.

<details>
<summary>Answers</summary>

1. 京都に行った**ことがありますか**。(past experience: た + ことがある)
2. 宿題を忘れる**ことがある**。(occasional: dictionary form + ことがある)
3. 生の魚を食べた**ことがない**。(never: た + ことがない)
4. 夜眠れない**ことがある**。(occasional: dictionary form + ことがある)

</details>

### Exercise 6: Mixed — Choose the Best Expression

Select the most natural expression for each context.

1. (You see dark clouds) 雨が ________ 。
   a. 降るそうだ b. 降りそうだ c. 降るはずだ d. 降るらしい

2. (Your friend told you about a restaurant) あの店は安い ________ 。
   a. みたいだ b. はずだ c. らしい d. そうだ (appearance)

3. (You know the schedule) 電車は3時に来る ________ 。
   a. らしい b. はずだ c. みたいだ d. と思う

4. (Giving your personal opinion) この本は面白い ________ 。
   a. ようだ b. そうだ c. と思う d. らしい

<details>
<summary>Answers</summary>

1. **b. 降りそうだ** (Visual impression of the sky = appearance そうだ)
2. **c. らしい** (Information from a friend = hearsay)
3. **b. はずだ** (Based on known schedule = logical expectation)
4. **c. と思う** (Expressing personal opinion)

</details>
