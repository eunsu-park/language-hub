## 1. 제1/제2변화 형용사

### 1.1 개요

라틴어 형용사는 수식하는 명사와 세 가지 면에서 **일치**해야 합니다: **성**, **수**, **격**. 가장 흔한 형용사 유형은 여성에 제1변화 어미를, 남성과 중성에 제2변화 어미를 사용합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│         제1/제2변화 형용사 패턴                                    │
│                                                                 │
│   사전 표제어: bonus, -a, -um (좋은)                              │
│               ↑       ↑   ↑                                     │
│               남성    여성  중성                                  │
│               (제2변화)(제1변화)(제2변화)                           │
│                                                                 │
│   남성 형태 → 제2변화를 따름 (-us/-er 패턴)                       │
│   여성 형태 → 제1변화를 따름 (-a 패턴)                            │
│   중성 형태 → 제2변화를 따름 (-um 패턴)                           │
│                                                                 │
│   이 어미들은 이미 3과에서 모두 배웠습니다!                        │
│   형용사는 수식하는 명사의 성에 따라 적절한                        │
│   어미 세트를 선택할 뿐입니다.                                     │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 전체 패러다임: bonus, -a, -um (좋은)

**남성 (제2변화)**

| 격 | 단수 | 복수 |
|------|----------|--------|
| 주격 | bon**us** | bon**ī** |
| 속격 | bon**ī** | bon**ōrum** |
| 여격 | bon**ō** | bon**īs** |
| 대격 | bon**um** | bon**ōs** |
| 탈격 | bon**ō** | bon**īs** |
| 호격 | bon**e** | bon**ī** |

**여성 (제1변화)**

| 격 | 단수 | 복수 |
|------|----------|--------|
| 주격 | bon**a** | bon**ae** |
| 속격 | bon**ae** | bon**ārum** |
| 여격 | bon**ae** | bon**īs** |
| 대격 | bon**am** | bon**ās** |
| 탈격 | bon**ā** | bon**īs** |
| 호격 | bon**a** | bon**ae** |

**중성 (제2변화)**

| 격 | 단수 | 복수 |
|------|----------|--------|
| 주격 | bon**um** | bon**a** |
| 속격 | bon**ī** | bon**ōrum** |
| 여격 | bon**ō** | bon**īs** |
| 대격 | bon**um** | bon**a** |
| 탈격 | bon**ō** | bon**īs** |
| 호격 | bon**um** | bon**a** |

### 1.3 -er 유형: pulcher, pulchra, pulchrum (아름다운)

일부 형용사는 *puer*와 *ager* 명사처럼 남성 주격 단수에서 **-er**로 끝납니다. 두 가지 하위 유형이 있습니다:

**유형 A — -e-가 탈락 (가장 흔함):**

| | 남성 | 여성 | 중성 |
|---|-----------|----------|--------|
| 주격 단수 | pulch**er** | pulchr**a** | pulchr**um** |
| 속격 단수 | pulchr**ī** | pulchr**ae** | pulchr**ī** |
| 여격 단수 | pulchr**ō** | pulchr**ae** | pulchr**ō** |
| 대격 단수 | pulchr**um** | pulchr**am** | pulchr**um** |
| 탈격 단수 | pulchr**ō** | pulchr**ā** | pulchr**ō** |

**유형 B — -e-를 유지 (소수만):**

| | 남성 | 여성 | 중성 |
|---|-----------|----------|--------|
| 주격 단수 | līber | līber**a** | līber**um** |
| 속격 단수 | līber**ī** | līber**ae** | līber**ī** |

```
┌─────────────────────────────────────────────────────────────────┐
│   어떻게 구별할까? 사전에서 여성형을 확인하세요.                    │
│                                                                 │
│   -e- 탈락:  pulcher, pulchra, pulchrum  (어간 = pulchr-)       │
│              miser, misera, miserum      (어간 = miser-)        │
│              noster, nostra, nostrum     (어간 = nostr-)        │
│                                                                 │
│   -e- 유지:  līber, lībera, līberum     (어간 = līber-)        │
│              tener, tenera, tenerum      (어간 = tener-)        │
│                                                                 │
│   ※ -e-를 유지하는 -er 형용사는 드뭅니다. 대부분은 탈락합니다.    │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 형용사-명사 일치

### 2.1 황금 규칙

```
┌─────────────────────────────────────────────────────────────────┐
│          형용사-명사 일치: 황금 규칙                                │
│                                                                 │
│   형용사는 수식하는 명사와 다음이 일치해야 합니다:                  │
│     1. 성    (남성, 여성, 또는 중성)                              │
│     2. 수    (단수 또는 복수)                                     │
│     3. 격    (주격, 속격, 여격 등)                                │
│                                                                 │
│   그러나 형용사가 명사와 같은 변화에 속할                           │
│   필요는 없습니다!                                                │
│                                                                 │
│   예시:                                                          │
│     agricola bonus    (좋은 농부)                                │
│     ↑         ↑                                                 │
│     제1변화   제2변화   ← 변화가 다름!                            │
│     남성      남성      ← 성 동일 ✓                              │
│     단수      단수      ← 수 동일 ✓                              │
│     주격      주격      ← 격 동일 ✓                              │
│                                                                 │
│   어미가 달라 보일 수 있지만, 성/수/격은                           │
│   반드시 일치해야 합니다. 형용사는 명사의 변화가 아닌               │
│   명사의 성을 따릅니다.                                           │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 성별 일치 예시

```
남성 명사 → 남성 형용사 형태:
  servus bonus         좋은 노예       (제2변화 + 제2변화)
  puer magnus          위대한 소년      (제2변화 + 제2변화)
  agricola bonus       좋은 농부       (제1변화 명사 + 제2변화 형용사!)
  nauta miser          비참한 선원      (제1변화 명사 + 제2변화 형용사!)
  poēta Rōmānus       로마 시인       (제1변화 명사 + 제2변화 형용사!)

여성 명사 → 여성 형용사 형태:
  puella bona          좋은 소녀       (제1변화 + 제1변화)
  terra magna          위대한 땅       (제1변화 + 제1변화)
  via longa            긴 길          (제1변화 + 제1변화)
  fēmina pulchra       아름다운 여자   (제1변화 + 제1변화)
  patria nostra        우리의 조국      (제1변화 + 제1변화)

중성 명사 → 중성 형용사 형태:
  bellum magnum        큰 전쟁        (제2변화 + 제2변화)
  dōnum bonum          좋은 선물       (제2변화 + 제2변화)
  templum antīquum     고대 신전       (제2변화 + 제2변화)
  perīculum parvum     작은 위험       (제2변화 + 제2변화)
  verbum novum         새 단어        (제2변화 + 제2변화)
```

### 2.3 격 전체에 걸친 일치

형용사는 모든 격 변화를 통해 명사를 따릅니다:

```
주격:     puella bona cantat.       좋은 소녀가 노래한다.
속격:     fīlius puellae bonae      좋은 소녀의 아들
여격:     puellae bonae rosam dat.  좋은 소녀에게 장미를 준다.
대격:     puellam bonam videt.      좋은 소녀를 본다.
탈격:     cum puellā bonā ambulat.  좋은 소녀와 함께 걷는다.
호격:     Ō puella bona!            오, 좋은 소녀여!
```

### 2.4 까다로운 경우: 제1변화 남성 명사

명사가 제1변화이지만 남성인 경우(*agricola* 등), 형용사는 **남성**(제2변화) 형태를 사용해야 합니다:

```
┌─────────────────────────────────────────────────────────────────┐
│   agricola bonus (좋은 농부) — *agricola bona가 아님             │
│                                                                 │
│   주격:  agricola bonus         agricolae bonī                  │
│   속격:  agricolae bonī         agricolārum bonōrum             │
│   여격:  agricolae bonō         agricolīs bonīs                │
│   대격:  agricolam bonum        agricolās bonōs                 │
│   탈격:  agricolā bonō          agricolīs bonīs                │
│                                                                 │
│   ※ 명사 어미는 여성처럼 보이지만 (-a, -ae, -am, -ā)             │
│     형용사 어미는 남성입니다 (-us, -ī, -um, -ō)                  │
│     왜냐하면 agricola는 남성 명사이기 때문입니다.                   │
│                                                                 │
│   마찬가지로:                                                    │
│     nauta bonus    (좋은 선원)                                  │
│     poēta magnus   (위대한 시인)                                 │
│     pīrāta malus   (나쁜 해적)                                  │
└─────────────────────────────────────────────────────────────────┘
```

> **한국어 사용자를 위한 참고**: 이것은 한국어에 직접적인 대응이 없는 개념입니다. 한국어에서 형용사(관형사/형용사)는 수식하는 명사에 따라 형태가 변하지 않습니다. 라틴어에서는 항상 머릿속으로 확인해야 합니다: "이 명사의 성은 무엇인가? 어떤 격에 있는가? 어떤 수인가?" 그리고 일치하는 형용사 형태를 사용합니다.

## 3. 실체적 형용사

### 3.1 형용사를 명사로 사용하기

라틴어에서 형용사는 명사 없이 독립적으로 명사처럼 쓸 수 있습니다("실체적" 용법). 성이 어떤 종류의 개체를 가리키는지 알려줍니다:

```
┌─────────────────────────────────────────────────────────────────┐
│              실체적 형용사                                        │
│                                                                 │
│   남성 → 남성 인물을 지칭:                                       │
│     bonī = "좋은 (사람들)" / "선한 이들"                          │
│     malī = "나쁜 (사람들)" / "악한 이들"                          │
│     multī = "많은 (사람들)"                                      │
│     Rōmānī = "로마인들" (로마 사람들)                             │
│                                                                 │
│   여성 → 여성 인물 또는 추상적 개념:                               │
│     bonae = "좋은 (여자들)"                                      │
│     Rōmānae = "로마 여인들"                                      │
│                                                                 │
│   중성 → 사물, 개념, 추상:                                       │
│     bonum = "좋은 것" / "선"                                     │
│     bona = "좋은 것들" / "재산"                                   │
│     malum = "악" / "나쁜 것"                                     │
│     magna = "위대한 것들"                                        │
│     multa = "많은 것들"                                          │
│     vera = "참된 것들" / "진실"                                   │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 문장 속 예시

```
Bonī patriam amant.
"선한 이들이 조국을 사랑한다."

Multa vidēmus.
"우리는 많은 것을 본다."

Rōmānī Graecōs superant.
"로마인들이 그리스인들을 이긴다."

Magna est vēritās.
"진리는 위대한 것이다." / "진리는 위대하다."

Bona et mala in vītā sunt.
"삶에는 좋은 것과 나쁜 것이 있다."
```

## 4. 서술적 vs 한정적 위치

### 4.1 형용사의 두 가지 사용법

라틴어 형용사는 두 가지 주요 역할로 나타납니다:

```
┌─────────────────────────────────────────────────────────────────┐
│        한정적 vs 서술적 형용사                                     │
│                                                                 │
│   한정적 — 명사를 직접 수식 (연결 동사 없이):                      │
│     Puella bona cantat.       좋은 소녀가 노래한다.               │
│     Bonus agricola labōrat.   좋은 농부가 일한다.                 │
│     In magnō perīculō sumus. 우리는 큰 위험에 처해 있다.         │
│                                                                 │
│   서술적 — esse(또는 유사한 동사)로 주어와 연결:                   │
│     Puella est bona.          소녀는 착하다.                      │
│     Perīculum est magnum.     위험은 크다.                       │
│     Agrī sunt magnī.          밭들은 크다.                       │
│                                                                 │
│   핵심 차이:                                                     │
│     한정적: 형용사 + 명사가 함께 (esse 없이)                      │
│     서술적: 주어 + esse + 형용사                                  │
│                                                                 │
│   영어와 비교:                                                    │
│     "The tall man walks." (한정적 — "tall"이 "man"을 수식)       │
│     "The man is tall." (서술적 — "tall"이 "is"로 연결)           │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 한정적 형용사의 위치

라틴어에서 한정적 형용사는 명사의 **앞** 또는 **뒤**에 올 수 있습니다. 선택에 따라 강조가 달라집니다:

```
일반적 위치 — 형용사가 명사 뒤:
  puella bona       "좋은 소녀"
  vir magnus        "위대한 남자"
  via longa         "긴 길"

강조 위치 — 형용사가 명사 앞:
  bona puella       "좋은 소녀" (성질을 강조)
  magnus vir        "위대한 남자"
  magna cum laude   "큰 찬사와 함께"

※ 특정 형용사는 흔히 명사 앞에 옵니다:
  magnus (위대한), bonus (좋은), malus (나쁜), multus (많은),
  parvus (작은), 소유격 (meus, tuus, noster, vester)
  서수 (prīmus, secundus, tertius)
```

### 4.3 분리된 형용사와 명사

라틴어 어순은 유연하므로 형용사와 명사가 다른 단어로 분리될 수 있습니다. 격 어미가 어떤 형용사가 어떤 명사와 맞는지 알려줍니다:

```
Magnam agricola terram arat.
magnus → magnam (대격 여성 단수)은 terram (대격 여성 단수)과 일치
"농부가 넓은 땅을 갈다."

Bonus fīliae dominus dōnum dat.
bonus → bonus (주격 남성 단수)는 dominus (주격 남성 단수)와 일치

## 5. 라틴어의 어순

### 5.1 기본 SOV와 변형

```
┌─────────────────────────────────────────────────────────────────┐
│              라틴어 어순 원칙                                      │
│                                                                 │
│   1. 기본 어순: 주어 — 목적어 — 동사 (SOV)                        │
│      Puella rosam amat.    소녀가 장미를 사랑한다.                │
│      한국어, 일본어의 기본 어순과 동일합니다.                       │
│                                                                 │
│   2. 동사가 절의 끝에 오는 경향                                   │
│      이것이 라틴어 산문의 가장 일관된 특징입니다.                   │
│                                                                 │
│   3. 수식어가 수식 대상 앞에 오는 경향:                            │
│      속격이 명사 앞:  populī Rōmānī lēgēs                        │
│                       ("로마 국민의 법률")                        │
│      형용사가 명사 앞: bonus vir (좋은 남자)                      │
│      그러나 형용사가 명사 뒤에 오는 것도 매우 흔함                  │
│                                                                 │
│   4. 문장의 처음과 끝 위치가 강조를 지님                           │
│      Rosam puella amat.     (장미를 강조 — 바로 장미를!)          │
│      Puella amat rosam.     (amat를 강조 — 사랑한다!)            │
│                                                                 │
│   5. 전치사는 목적어 앞에 옴 (영어와 같음)                         │
│      in agrō, cum amīcō, ad Italiam                             │
│      예외: 대명사와 -cum → mēcum, tēcum, nōbīscum               │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 실용적 어순 지침

| 요소 | 일반적 위치 | 예시 |
|---------|-----------------|---------|
| 주어 | 문장 시작 | **Mārcus** rosam puellae dat. |
| 직접 목적어 | 동사 앞 | Mārcus **rosam** puellae dat. |
| 간접 목적어 | 동사 근처 | Mārcus rosam **puellae** dat. |
| 동사 | 문장 끝 | Mārcus rosam puellae **dat**. |
| 형용사 | 명사 뒤 (강조 시 앞) | puella **bona** / **bona** puella |
| 속격 | 명사의 앞 또는 뒤 | **populī** lēgēs / lēgēs **populī** |
| 전치사구 | 수식하는 단어 근처 | **in agrō** labōrat |
| 부정 (nōn) | 동사 바로 앞 | **nōn** amat |
| 접속사 | 절/단어 사이 | Mārcus **et** Iūlia |

### 5.3 어순 비교

```
라틴어 (SOV 기본):     Agricola terram arat.
                      농부가   땅을   간다.
한국어 (SOV):          농부가   땅을   간다.
영어 (SVO):            The farmer plows the land.
독일어 (V2/SOV):       Der Bauer pflügt das Land.
일본어 (SOV):          農夫が   土地を  耕す。

라틴어와 한국어는 SOV 기본 어순을 공유하지만, 라틴어는
격 어미 덕분에 자유롭게 재배열할 수 있습니다:
  Terram agricola arat.   ← 여전히 명확 (terram = 대격)
  Arat agricola terram.   ← 여전히 명확 (arat = 동사)

한국어도 조사 덕분에 비슷한 유연성을 가집니다:
  땅을 농부가 간다.          ← 여전히 명확 (을 = 목적격 조사)
```

## 6. 등위 접속사

### 6.1 주요 등위 접속사

라틴어에는 단어, 구, 절을 연결하는 여러 접속사가 있습니다:

```
┌─────────────────────────────────────────────────────────────────┐
│              등위 접속사                                          │
│                                                                 │
│   et        /ɛt/          "그리고"                               │
│   -que      /kwɛ/         "그리고" (접어, 두 번째 단어에            │
│                            붙음)                                 │
│   atque/ac  /ˈat.kwɛ/     "그리고" (강조: "그리고 또한")           │
│                                                                 │
│   sed       /sɛd/         "그러나"                               │
│   at        /at/          "그러나" (약한 대조)                    │
│                                                                 │
│   aut       /au̯t/         "또는" (배타적: 둘 중 하나)             │
│   vel       /wɛl/         "또는" (포괄적: 하나 또는 둘 다)        │
│   -ve       /wɛ/          "또는" (접어)                          │
│                                                                 │
│   neque/nec /ˈnɛ.kwɛ/    "그리고 ~않다" / "~도 아니다"           │
│                                                                 │
│   nam       /nam/         "왜냐하면" (이유 제시)                  │
│   enim      /ˈɛ.nɪm/     "왜냐하면, 실로" (후치사:               │
│                            절에서 두 번째 위치)                   │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 ET — "그리고"

가장 흔한 접속사로, 단어, 구, 절을 연결합니다:

```
명사 연결:
  Mārcus et Iūlia ambulant.
  "마르쿠스와 율리아가 걷는다."

형용사 연결:
  Vir est bonus et magnus.
  "그 남자는 착하고 위대하다."

절 연결:
  Puella cantat et puer rīdet.
  "소녀가 노래하고 소년이 웃는다."

여러 항목 연결:
  Servī et puellae et puerī labōrant.
  "노예들과 소녀들과 소년들이 일한다."
  (라틴어는 영어의 "and...and...and"처럼 et를 반복할 수 있음)
```

### 6.3 -QUE — "그리고" (접어)

접어 **-que**는 연결되는 두 항목의 **두 번째**에 붙습니다. 격식체와 문학적 라틴어에서 매우 흔합니다.

```
Senātus Populusque Rōmānus    (SPQR)
"원로원과 로마 국민"
(-que가 두 번째 항목 Populus에 붙음)

Virī fēminaeque ambulant.
"남자들과 여자들이 걷는다."
(-que가 두 번째 항목 fēminae에 붙음)

Bonus fortisque est.
"그는 착하고 용감하다."
(-que가 두 번째 항목 fortis에 붙음)

※ -que는 et보다 격조 있고 문학적입니다.
※ -que는 밀접하게 관련된 쌍을 연결합니다; et는 더 일반적입니다.
```

### 6.4 SED — "그러나"

```
대조되는 절 연결:
  Puella cantat, sed puer nōn cantat.
  "소녀는 노래하지만, 소년은 노래하지 않는다."

  Servus labōrat, sed dominus nōn labōrat.
  "노예는 일하지만, 주인은 일하지 않는다."

  Vīta est brevis, sed ars est longa.
  "인생은 짧지만, 예술은 길다."

  Terra est magna, sed populus est parvus.
  "땅은 넓지만, 국민은 적다."
```

### 6.5 AUT — "또는"

```
배타적 "또는" (둘 중 하나, 둘 다는 아님):
  Aut Caesar aut nihil.
  "카이사르이거나 아무것도 아니거나."

  Aut vincere aut morī.
  "이기거나 죽거나."

  Estne bonus aut malus?
  "그가 좋은가 나쁜가?"

쌍으로 사용 aut...aut = "~이거나 ~이거나":
  Aut amīcus aut inimīcus est.
  "그는 친구이거나 적이다."
```

### 6.6 NEQUE / NEC — "그리고 ~않다" / "~도 아니다"

```
단독 neque = "그리고 ~않다" 또는 "~도 아니다":
  Puella cantat neque labōrat.
  "소녀는 노래하고 일하지는 않는다."

쌍으로 neque...neque = "~도 ~도 아니다":
  Neque puerī neque puellae labōrant.
  "소년들도 소녀들도 일하지 않는다."

  Neque bonus neque malus est.
  "그는 좋지도 나쁘지도 않다."

  Neque terram neque aquam videt nauta.
  "선원은 땅도 물도 보지 못한다."
```

### 6.7 요약표

| 접속사 | 의미 | 예시 |
|------------|---------|---------|
| et | 그리고 | Mārcus **et** Iūlia |
| -que | 그리고 (접어) | Mārcus Iūlia**que** |
| atque / ac | 그리고 (강조) | Mārcus **atque** Iūlia |
| sed | 그러나 | bonus, **sed** parvus |
| aut | 또는 (배타적) | bonus **aut** malus |
| aut...aut | ~이거나 ~이거나 | **aut** bonus **aut** malus |
| vel | 또는 (포괄적) | bonus **vel** magnus |
| neque / nec | 그리고 ~않다, ~도 아니다 | neque bonus |
| neque...neque | ~도 ~도 아니다 | **neque** bonus **neque** malus |

4. 마르쿠스의 딸은 어떤 모습인가?
5. 딸은 무엇을 하고 있는가?
6. 마르쿠스와 아들이 소녀를 보지 못하는 이유는?
```

<details>
<summary>정답 보기</summary>

```
1. 마르쿠스는 농부이다 (agricola).

2. 그는 큰 밭에서 일한다 (in magnō agrō).

3. 그의 아들이 (cum fīliō) 함께 일한다.

4. 그녀는 아름다운 소녀이다 (puella pulchra).

5. 그녀는 친구들과 함께 긴 길 위를 걷고 있다
   (in viā longā cum amīcīs ambulat).

6. 그들이 밭에 머무르고 있기 때문이다
   (nam in agrō manent).

전문 번역:
"마르쿠스는 좋은 농부이다. 큰 밭에서 아들과 함께 일한다.
농부의 아들은 작지만 착한 소년이다. 마르쿠스는 땅을
사랑하고 조국을 칭찬한다. 마르쿠스의 딸은 아름다운 소녀이다.
소녀는 친구들과 함께 긴 길 위를 걷는다. 마르쿠스도
아들도 소녀를 보지 못하는데, 밭에 머무르고 있기 때문이다."
```

</details>
