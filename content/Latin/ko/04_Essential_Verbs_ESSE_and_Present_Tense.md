# 04. 필수 동사: ESSE와 현재 시제 (Verba Necessaria: ESSE et Tempus Praesens)

## 학습 목표
- 불규칙 동사 *esse* (있다/~이다)를 현재 시제로 활용하기
- *esse*와 함께 서술 주격과 서술 형용사를 이해하고 사용하기
- 제1활용 (-āre) 동사를 현재 능동 직설법으로 활용하기
- 제2활용 (-ēre) 동사를 현재 능동 직설법으로 활용하기

- 기본 라틴어 문장 구조 이해하기 (SOV 어순)

## 1. 동사 ESSE (~이다)

### 1.1 ESSE가 특별한 이유

동사 *esse* (~이다)는 라틴어에서 가장 중요하고 가장 불규칙한 동사입니다 — 영어의 "to be"가 불규칙(am, is, are)하고, 독일어의 sein(bin, bist, ist), 한국어의 이다, 그리고 거의 모든 인도유럽어에서 불규칙한 것과 마찬가지입니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    ESSE — ~이다                                  │
│              현재 시제 (Tempus Praesens)                          │
│                                                                 │
│   인칭      단수              복수                               │
│   ──────    ─────────         ──────                            │
│   1인칭     sum               sumus                              │
│             /sʊm/             /ˈsʊ.mʊs/                         │
│             "나는 ~이다"      "우리는 ~이다"                      │
│                                                                 │
│   2인칭     es                estis                              │
│             /ɛs/              /ˈɛs.tɪs/                          │
│             "너는 ~이다"      "너희는 ~이다"                      │
│                                                                 │
│   3인칭     est               sunt                               │
│             /ɛst/             /sʊnt/                              │
│             "그/그녀/그것은    "그들은 ~이다"                      │
│              ~이다"                                               │
│                                                                 │
│   부정사: esse /ˈɛs.sɛ/ "~이다, 있다"                            │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 다른 언어와의 비교

| 인칭 | 라틴어 | 영어 | 스페인어 | 한국어 |
|--------|-------|---------|---------|--------|
| 1인칭 단수 | sum | I am | soy | (나는) ~이다 |
| 2인칭 단수 | es | you are | eres | (너는) ~이다 |
| 3인칭 단수 | est | he/she/it is | es | (그는) ~이다 |
| 1인칭 복수 | sumus | we are | somos | (우리는) ~이다 |
| 2인칭 복수 | estis | you (pl.) are | sois | (너희는) ~이다 |
| 3인칭 복수 | sunt | they are | son | (그들은) ~이다 |

### 1.3 ESSE의 문장 사용

```
Sum discipulus.              나는 학생이다.
Es amīcus meus.             너는 나의 친구이다.
Est bonus magister.          그는 좋은 선생님이다.
Sumus Rōmānī.               우리는 로마인이다.
Estis laetī.                 너희는 행복하다.
Sunt in Italiā.              그들은 이탈리아에 있다.
```

> **참고**: 라틴어는 일반적으로 주어 대명사(ego, tū 등)를 생략합니다. 동사 어미가 주어를 알려주기 때문입니다 — 한국어 동사 어미와 스페인어와 유사합니다. 강조나 대조를 위해서만 대명사를 포함합니다.

## 2. 서술 주격과 서술 형용사

### 2.1 서술 주격

*esse*가 주어를 명사와 연결할 때, 그 명사는 **주격**(대격이 아닌)으로 옵니다. 이 명사를 **서술 주격**이라고 합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│              서술 주격 규칙                                       │
│                                                                 │
│   주어 (주격) + esse + 명사 (주격)                                │
│                                                                 │
│   Mārcus est poēta.        마르쿠스는 시인이다.                   │
│   (주격)     (주격)         *poētam (대격)이 아님                 │
│                                                                 │
│   Iūlia est fēmina.        율리아는 여자이다.                     │
│   (주격)    (주격)                                               │
│                                                                 │
│   Puerī sunt discipulī.    소년들은 학생들이다.                   │
│   (주격 복수) (주격 복수)                                         │
│                                                                 │
│   ※ 이렇게 생각하세요: esse는 등호(=)와 같습니다.                 │
│     양쪽 모두 같은 격(주격)이어야 합니다.                          │
│     Mārcus = poēta → 둘 다 주격                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 서술 형용사

*esse*가 주어를 형용사와 연결할 때, 그 형용사는 주어와 성, 수, 격이 **일치**해야 합니다.

```
서술 형용사 일치:

  Mārcus est bonus.           마르쿠스는 훌륭하다.
  (남성 단수 주격)  (남성 단수 주격)

  Iūlia est bona.             율리아는 훌륭하다.
  (여성 단수 주격)   (여성 단수 주격)

  Bellum est magnum.           전쟁은 크다.
  (중성 단수 주격)  (중성 단수 주격)

  Puellae sunt laetae.        소녀들은 행복하다.
  (여성 복수 주격)   (여성 복수 주격)

  Puerī sunt laetī.           소년들은 행복하다.
  (남성 복수 주격)  (남성 복수 주격)

  Dōna sunt pulchra.          선물들은 아름답다.
  (중성 복수 주격)  (중성 복수 주격)
```

### 2.3 ESSE와 함께 쓰는 일반적인 형용사

| 남성 | 여성 | 중성 | 의미 |
|-----------|----------|--------|---------|
| bonus | bona | bonum | 좋은 |
| malus | mala | malum | 나쁜 |
| magnus | magna | magnum | 큰, 위대한 |
| parvus | parva | parvum | 작은 |
| pulcher | pulchra | pulchrum | 아름다운 |
| laetus | laeta | laetum | 행복한 |
| miser | misera | miserum | 비참한, 불행한 |
| multus | multa | multum | 많은 |
| Rōmānus | Rōmāna | Rōmānum | 로마의 |
| prīmus | prīma | prīmum | 첫 번째의 |

## 3. 라틴어 동사 활용 입문

### 3.1 네 가지 활용

라틴어 규칙 동사는 현재 어간 끝의 모음에 따라 **네 가지 활용**으로 구분됩니다:

```
┌─────────────────────────────────────────────────────────────────┐
│                  라틴어의 네 가지 활용                              │
│                                                                 │
│   활용        어간 모음   부정사 어미      예시                    │
│   ───────────  ──────────  ─────────────  ───────                │
│   제1활용      -ā-         -āre           amāre (사랑하다)        │
│   제2활용      -ē-         -ēre           monēre (경고하다)       │
│   제3활용      -ĕ-         -ĕre           dūcere (이끌다)        │
│   제4활용      -ī-         -īre           audīre (듣다)          │
│                                                                 │
│   이 과에서는 제1활용과 제2활용을 다룹니다.                         │
│   제3활용과 제4활용은 이후 과에서 배웁니다.                         │
│                                                                 │
│   사전 형태 (4가지 기본형):                                       │
│   amō, amāre, amāvī, amātum                                    │
│   ↑    ↑       ↑       ↑                                        │
│   1인칭 부정사  완료    목적분사                                    │
│   단수   현재  1인칭    (지금은 처음 두 가지만 학습)                │
│   현재   현재  단수                                               │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 인칭과 수

한국어의 존칭/평어형, 스페인어의 활용과 마찬가지로, 라틴어 동사는 **인칭**과 **수**에 따라 어미가 변합니다:

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 (나 / 우리) | -ō / -m | -mus |
| 2인칭 (너 / 너희) | -s | -tis |
| 3인칭 (그,그녀,그것 / 그들) | -t | -nt |

> 이 **인칭 어미**는 대부분의 활용과 시제에서 동일합니다. 잘 암기하세요 — 이것이 모든 라틴어 동사를 읽는 열쇠입니다.

### 3.3 현재 어간 찾는 방법

```
현재 시제로 활용하려면:
1. 부정사를 가져옵니다: amāre
2. -re 어미를 제거합니다: amā-  (이것이 현재 어간)
3. 인칭 어미를 추가합니다

부정사         어간
amāre     →    amā-     (제1활용)
monēre    →    monē-    (제2활용)
```

## 4. 제1활용 현재 능동

### 4.1 전체 패러다임: amō, amāre (사랑하다)

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 | am**ō** /ˈa.moː/ | amā**mus** /a.ˈmaː.mʊs/ |
| 2인칭 | amā**s** /ˈa.maːs/ | amā**tis** /a.ˈmaː.tɪs/ |
| 3인칭 | ama**t** /ˈa.mat/ | ama**nt** /ˈa.mant/ |

```
┌─────────────────────────────────────────────────────────────────┐
│        제1활용 현재 능동 형성                                      │
│                                                                 │
│   어간: amā- (amāre에서 -re를 제거)                              │
│                                                                 │
│              어간 + 어미 = 형태                                   │
│   1인칭 단수: amā + ō = amō     (ā가 ō에 흡수됨)                 │
│   2인칭 단수: amā + s = amās                                     │
│   3인칭 단수: amā + t = amat    (ā가 -t 앞에서 단축)              │
│   1인칭 복수: amā + mus = amāmus                                 │
│   2인칭 복수: amā + tis = amātis                                 │
│   3인칭 복수: amā + nt = amant  (ā가 -nt 앞에서 단축)             │
│                                                                 │
│   ※ 1인칭 단수 어미 -ō가 어간 모음 -ā를 "삼킵니다".              │
│   ※ 어간 모음은 -t와 -nt 앞에서 짧아집니다.                       │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 주요 제1활용 동사

| 라틴어 | 부정사 | 의미 |
|-------|-----------|---------|
| amō | amāre | 사랑하다 |
| vocō | vocāre | 부르다 |
| laudō | laudāre | 칭찬하다 |
| labōrō | labōrāre | 일하다, 수고하다 |
| ambulō | ambulāre | 걷다 |
| portō | portāre | 운반하다 |
| nārrō | nārrāre | 이야기하다, 서술하다 |
| spectō | spectāre | 보다, 관찰하다 |
| dō | dare* | 주다 |
| stō | stāre* | 서다 |
| nāvigō | nāvigāre | 항해하다 |
| pugnō | pugnāre | 싸우다 |
| rogō | rogāre | 묻다 |
| servō | servāre | 구하다, 보존하다 |
| superō | superāre | 극복하다, 능가하다 |

> ***dō, dare**와 **stō, stāre**는 부정사에서 짧은 -a-를 갖지만(-āre가 아님) 현재 시제에서는 정상적으로 활용됩니다: dō, dās, dat, damus, datis, dant.

### 4.3 제1활용 예문

```
Amō patriam.                나는 조국을 사랑한다.
Puella rosās portat.        소녀가 장미들을 운반한다.
Nautae ad Italiam nāvigant. 선원들이 이탈리아로 항해한다.
Agricolae in agrīs labōrant. 농부들이 밭에서 일한다.
Poēta fābulam nārrat.       시인이 이야기를 들려준다.
Puerōs et puellās vocāmus.  우리는 소년들과 소녀들을 부른다.
```

## 5. 제2활용 현재 능동

### 5.1 전체 패러다임: moneō, monēre (경고하다, 조언하다)

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 | mon**eō** /ˈmɔ.nɛ.oː/ | mon**ēmus** /mɔ.ˈneː.mʊs/ |
| 2인칭 | mon**ēs** /ˈmɔ.neːs/ | mon**ētis** /mɔ.ˈneː.tɪs/ |
| 3인칭 | mon**et** /ˈmɔ.nɛt/ | mon**ent** /ˈmɔ.nɛnt/ |

```
┌─────────────────────────────────────────────────────────────────┐
│        제2활용 현재 능동 형성                                      │
│                                                                 │
│   어간: monē- (monēre에서 -re를 제거)                            │
│                                                                 │
│              어간 + 어미 = 형태                                   │
│   1인칭 단수: monē + ō = moneō    (ē가 ō 앞에서 단축)            │
│   2인칭 단수: monē + s = monēs                                   │
│   3인칭 단수: monē + t = monet    (ē가 -t 앞에서 단축)            │
│   1인칭 복수: monē + mus = monēmus                               │
│   2인칭 복수: monē + tis = monētis                               │
│   3인칭 복수: monē + nt = monent  (ē가 -nt 앞에서 단축)           │
│                                                                 │
│   ※ 제1활용과 같은 인칭 어미!                                     │
│   ※ 어간 모음 -ē-는 -ō, -t, -nt 앞에서 짧아집니다.               │
│   ※ 제1활용과의 핵심 차이: 어간 모음이 ā가 아닌 ē.                │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 주요 제2활용 동사

| 라틴어 | 부정사 | 의미 |
|-------|-----------|---------|
| moneō | monēre | 경고하다, 조언하다 |
| videō | vidēre | 보다 |
| habeō | habēre | 가지다, 소유하다 |
| doceō | docēre | 가르치다 |
| moveō | movēre | 움직이다 |
| teneō | tenēre | 잡다, 유지하다 |
| timeō | timēre | 두려워하다 |
| dēbeō | dēbēre | ~해야 하다, 빚지다 |
| maneō | manēre | 머무르다, 남다 |
| respondeō | respondēre | 대답하다, 응답하다 |
| rīdeō | rīdēre | 웃다 |
| terreō | terrēre | 겁먹게 하다 |
| valeō | valēre | 건강하다, 힘이 있다 |

### 5.3 제2활용 예문

```
Magister puerōs docet.       선생님이 소년들을 가르친다.
Perīculum vidēmus.           우리는 위험을 본다.
Pecūniam nōn habent.         그들은 돈이 없다.
Puerī magistrum timent.      소년들이 선생님을 두려워한다.
Servī in oppidō manent.      노예들이 마을에 머무른다.
Dēbēmus patriam servāre.     우리는 조국을 구해야 한다.
```

### 5.4 제1활용과 제2활용 비교

| 형태 | 제1활용 (amāre) | 제2활용 (monēre) |
|------|-------------|--------------|
| 1인칭 단수 | am**ō** | mone**ō** |
| 2인칭 단수 | am**ās** | mon**ēs** |
| 3인칭 단수 | am**at** | mon**et** |
| 1인칭 복수 | am**āmus** | mon**ēmus** |
| 2인칭 복수 | am**ātis** | mon**ētis** |
| 3인칭 복수 | am**ant** | mon**ent** |
| 부정사 | am**āre** | mon**ēre** |

> 패턴이 명확합니다: 전체에 걸쳐 **-ā-**를 **-ē-**로 교체합니다. 인칭 어미(-ō, -s, -t, -mus, -tis, -nt)는 동일합니다.

## 6. 주어-동사 일치

### 6.1 규칙

라틴어에서 동사는 주어와 **인칭**과 **수**가 일치해야 합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│              주어-동사 일치                                       │
│                                                                 │
│   단수 주어 → 단수 동사                                           │
│   Puella cantat.        소녀가 노래한다.                          │
│   Servus labōrat.       노예가 일한다.                            │
│                                                                 │
│   복수 주어 → 복수 동사                                           │
│   Puellae cantant.      소녀들이 노래한다.                        │
│   Servī labōrant.       노예들이 일한다.                          │
│                                                                 │
│   복합 주어 → 복수 동사                                           │
│   Mārcus et Iūlia ambulant.   마르쿠스와 율리아가 걷는다.         │
│   Puer et puella rīdent.      소년과 소녀가 웃는다.               │
│                                                                 │
│   주어 생략 — 동사 어미가 주어를 알려줍니다:                       │
│   Labōrō.         나는 일한다.     (1인칭 단수: -ō → "나")       │
│   Labōrās.        너는 일한다.     (2인칭 단수: -s → "너")       │
│   Labōrat.        그/그녀가 일한다. (3인칭 단수: -t → "그/그녀") │
│   Labōrāmus.     우리가 일한다.   (1인칭 복수: -mus → "우리")   │
│   Labōrātis.     너희가 일한다.   (2인칭 복수: -tis → "너희")   │
│   Labōrant.      그들이 일한다.   (3인칭 복수: -nt → "그들")    │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 중요 참고사항

1. **중성 복수 주어**는 일부 맥락에서 **단수** 동사를 취합니다(특히 *esse*와 함께):
   - *Arma sunt bona.* — "무기들은 좋다." (복수 주어에 복수 동사)
   - 그러나 중성 복수 주어와 행위 동사는 시에서 때때로 단수 동사와 함께 나타납니다.

2. **집합 명사**(*populus*, 국민 등)는 **단수** 동사를 취합니다:
   - *Populus Rōmānus pugnat.* — "로마 국민이 싸운다."

## 7. 기본 라틴어 문장 구조

### 7.1 SOV 어순

```
┌─────────────────────────────────────────────────────────────────┐
│           라틴어 기본 어순: S — O — V                               │
│                                                                 │
│   라틴어 기본:    Mārcus    rosam     portat.                     │
│                  (주어)    (목적어)   (동사)                       │
│                                                                 │
│   한국어 대응:    마르쿠스가  장미를    운반한다.                     │
│                  (주어)    (목적어)   (동사)                       │
│                                                                 │
│   영어:          Marcus    carries   a rose.                     │
│                  (주어)    (동사)    (목적어)                      │
│                                                                 │
│   그러나 라틴어 어순은 유연합니다. 격 어미가                        │
│   문법적 기능을 나타내기 때문입니다.                                │
│                                                                 │
│   일반적인 패턴:                                                  │
│   • SOV — 중립적, 기본                                           │
│   • 동사가 끝 — 산문에서 가장 흔함                                 │
│   • 동사가 처음 — 행위 강조 또는 의문문                             │
│   • 목적어가 처음 — 목적어 강조                                    │
│   • 속격이 명사 앞: populī Rōmānī arma                            │
│   • 형용사가 명사 뒤 (강조 시 앞에 올 수 있음)                     │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 일반적인 문장 패턴

```
패턴 1: 주어 + 자동사
  Puella ambulat.              소녀가 걷는다.
  Nautae nāvigant.             선원들이 항해한다.

패턴 2: 주어 + esse + 서술어
  Mārcus est poēta.            마르쿠스는 시인이다.
  Puellae sunt laetae.         소녀들은 행복하다.

패턴 3: 주어 + 직접 목적어 + 동사
  Puer rosam videt.            소년이 장미를 본다.
  Servī terram arant.          노예들이 땅을 갈다.

패턴 4: 주어 + 간접 목적어 + 직접 목적어 + 동사
  Mārcus puellae rosam dat.    마르쿠스가 소녀에게 장미를 준다.
  Magister puerīs librōs dat.  선생님이 소년들에게 책들을 준다.

패턴 5: 주어 + 전치사구 + 동사
  Agricola in agrō labōrat.    농부가 밭에서 일한다.
  Cum amīcīs ambulāmus.       우리는 친구들과 함께 걷는다.
```

### 7.3 어순과 강조

```
중립:       Mārcus rosam portat.         마르쿠스가 장미를 운반한다.
동사 강조:  Portat Mārcus rosam.          마르쿠스가 장미를 운반한다.
목적어 강조: Rosam Mārcus portat.          장미를 마르쿠스가 운반한다.
주어 강조:  Rosam portat Mārcus.          장미를 운반하는 것은 마르쿠스다.

※ 라틴어 문장에서 첫 번째와 마지막 위치가 가장 강한
  강조를 지닙니다. 동사는 종종 마지막 위치를 차지합니다.
```

## 8. NON을 사용한 부정

### 8.1 단순 부정

라틴어 문장을 부정하려면 동사 바로 앞에 **nōn**을 놓습니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    NŌN을 사용한 부정                               │
│                                                                 │
│   긍정문:                  부정문:                                │
│   Puella cantat.            Puella nōn cantat.                  │
│   소녀가 노래한다.           소녀가 노래하지 않는다.                │
│                                                                 │
│   Sum laetus.               Nōn sum laetus.                     │
│   나는 행복하다.             나는 행복하지 않다.                    │
│                                                                 │
│   Pecūniam habēmus.        Pecūniam nōn habēmus.               │
│   우리는 돈이 있다.          우리는 돈이 없다.                      │
│                                                                 │
│   ※ NŌN은 동사(또는 부정하는 단어) 바로 앞에 옵니다.              │
│   ※ 영어의 "do not"에 해당하는 것은 없습니다 — 그냥 nōn.         │
│   ※ 한국어와 비교: ~하지 않는다 (동사 + 않다)                     │
│     라틴어가 더 간단합니다: nōn + 동사                             │
└─────────────────────────────────────────────────────────────────┘
```

### 8.2 부정 예문

| 긍정문 | 부정문 |
|------------|----------|
| Amō. (나는 사랑한다.) | Nōn amō. (나는 사랑하지 않는다.) |
| Est bonus. (그는 훌륭하다.) | Nōn est bonus. (그는 훌륭하지 않다.) |
| Vidēmus. (우리는 본다.) | Nōn vidēmus. (우리는 보지 못한다.) |
| Timent. (그들은 두려워한다.) | Nōn timent. (그들은 두려워하지 않는다.) |
| Magister docet. (선생님이 가르친다.) | Magister nōn docet. (선생님이 가르치지 않는다.) |

### 8.3 기타 부정어 (미리보기)

| 라틴어 | 의미 |
|-------|---------|
| nōn | ~아니다, ~않다 |
| numquam / nunquam | 결코 ~않다 |
| nihil / nīl | 아무것도 ~않다 |
| nēmō | 아무도 ~않다 |
| neque / nec | 그리고 ~않다, ~도 아니다 |
| nōndum | 아직 ~않다 |
| nōn iam | 더 이상 ~않다 |

### 연습 5: 라틴어를 한국어로 번역하기

```
1. Sum discipulus.
2. Puella nōn cantat.
3. Agricolae in agrīs labōrant.
4. Magister puerōs docet.
5. Nautae perīculum timent.
6. Estis laetī.
7. Servī pecūniam nōn habent.
8. Mārcus et Iūlia amīcī sunt.
9. Patriam amāmus.

```

<details>
<summary>정답 보기</summary>

```
1. Sum discipulus.
   → 나는 학생이다.

2. Puella nōn cantat.
   → 소녀가 노래하지 않는다.

3. Agricolae in agrīs labōrant.
   → 농부들이 밭에서 일한다.

4. Magister puerōs docet.
   → 선생님이 소년들을 가르친다.

5. Nautae perīculum timent.
   → 선원들이 위험을 두려워한다.

6. Estis laetī.
   → 너희는 행복하다.

7. Servī pecūniam nōn habent.
   → 노예들은 돈이 없다.

8. Mārcus et Iūlia amīcī sunt.
   → 마르쿠스와 율리아는 친구이다.

9. Patriam amāmus.
   → 우리는 조국을 사랑한다.

10. Fēmina puellam vocat.
    → 여자가 소녀를 부른다.
```

</details>

### 연습 6: 한국어를 라틴어로 번역하기

```
1. 우리는 로마인이다.
2. 시인이 일하지 않는다.
3. 소년들이 선생님을 본다.
4. 나는 책이 있다.
5. 소녀가 선원에게 장미를 준다.
6. 그들은 두려워하지 않는다.
7. 너는 조국을 사랑한다.
8. 노예들이 밭에 머무른다.
```

<details>
<summary>정답 보기</summary>

```
1. 우리는 로마인이다.
   → Rōmānī sumus. (또는: Sumus Rōmānī.)

2. 시인이 일하지 않는다.
   → Poēta nōn labōrat.

3. 소년들이 선생님을 본다.
   → Puerī magistrum vident.

4. 나는 책이 있다.
   → Librum habeō. (또는: Habeō librum.)

5. 소녀가 선원에게 장미를 준다.
   → Puella nautae rosās dat.

6. 그들은 두려워하지 않는다.
   → Nōn timent.

7. 너는 조국을 사랑한다.
   → Patriam amās.

8. 노예들이 밭에 머무른다.
   → Servī in agrō manent.
```

</details>

### 연습 7: 서술 주격인가 대격인가?

빈칸에 올바른 형태를 고르세요. 기억하세요: *esse*는 주격을 취하고, 다른 동사는 직접 목적어에 대격을 취합니다.

```
1. Mārcus est ____________. (poēta / poētam)
2. Puella ____________ videt. (rosa / rosam)
3. Sumus ____________. (Rōmānī / Rōmānōs)
4. Agricolae ____________ amant. (terra / terram)
5. Iūlia est ____________. (bona / bonam)
6. Puerī ____________ timent. (magister / magistrum)
7. Sunt ____________. (laetī / laetōs)
8. Servī ____________ portant. (aqua / aquam)
```

<details>
<summary>정답 보기</summary>

```
1. poēta     (서술 주격 — esse 뒤)
2. rosam     (대격 — videō의 직접 목적어)
3. Rōmānī    (서술 주격 — sumus 뒤)
4. terram    (대격 — amō의 직접 목적어)
5. bona      (서술 형용사 — est 뒤, Iūlia에 일치하여 여성형)
6. magistrum (대격 — timeō의 직접 목적어)
7. laetī     (서술 형용사 — sunt 뒤, 주격 복수 남성)
8. aquam     (대격 — portō의 직접 목적어)
```

</details>
