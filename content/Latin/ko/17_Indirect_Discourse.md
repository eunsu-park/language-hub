# 17. 간접화법 (Ōrātiō Oblīqua)

## 학습 목표
- **대격 + 부정사** 패턴을 사용하여 간접 서술문 구성하기
- 주절 동사에 대한 상대적 시간을 전달하기 위해 올바른 부정사 시제(현재, 완료, 미래) 선택하기
- 간접 서술문을 도입하는 말하기, 생각하기, 알기, 인지하기 동사 식별하기
- 의문사 + **접속법**을 사용하여 간접 의문문 형성하기
- 명령/설득 동사 뒤에 **ut/ne** + 접속법으로 간접 명령문 표현하기
- 간접화법 내에서 재귀대명사(**sē, suus**) 올바르게 적용하기
- 라틴어에서 직접화법과 간접화법 사이를 자유롭게 전환하기

## 1. 간접 서술문

### 1.1 대격 + 부정사 구문

한국어/영어에서 간접 서술문은 "~라고" 절을 사용합니다:
- 직접: 병사들은 **용감하다**.
- 간접: 그는 병사들이 용감하**다고** 말한다.

라틴어에서 간접 서술문은 **대격 주어** + **부정사 동사**를 사용합니다("~라고"에 해당하는 접속사 없음):
- 직접: Mīlitēs **fortēs sunt**.
- 간접: Dīcit mīlitēs **fortēs esse**.

| 구성 요소 | 역할 |
|----------|------|
| **Dīcit** | 주절 동사 (말하기/생각하기 동사) |
| **mīlitēs** | 간접 서술문의 주어 (대격) |
| **fortēs** | 서술 형용사 (대격, 주어에 일치) |
| **esse** | 부정사 (간접 서술문의 주동사) |

### 1.2 부정사의 시제

부정사의 시제는 절대적 시간이 아니라 주절 동사에 대한 **상대적** 시간을 나타냅니다:

| 부정사 시제 | 주절 동사에 대한 시간 | 형태 (능동, 제1활용) |
|-----------|-------------------|-------------------|
| **현재** | 동시 | amāre |
| **완료** | 이전 | amāvisse |
| **미래** | 이후 | amātūrum (-am, -um) esse |

| 부정사 시제 | 형태 (수동, 제1활용) |
|-----------|-------------------|
| **현재 수동** | amārī |
| **완료 수동** | amātum (-am, -um) esse |
| **미래 수동** | amātum īrī |

### 1.3 시제 예시

**주절 동사가 주시제(dīcit — "그는 말한다"):**

| 라틴어 | 번역 | 시간 |
|--------|------|------|
| Dīcit mīlitēs **pugnāre**. | 그는 병사들이 **싸우고 있다고** 말한다. | 동시 |
| Dīcit mīlitēs **pugnāvisse**. | 그는 병사들이 **싸웠다고** 말한다. | 이전 |
| Dīcit mīlitēs **pugnātūrōs esse**. | 그는 병사들이 **싸울 것이라고** 말한다. | 이후 |

**주절 동사가 부시제(dīxit — "그는 말했다"):**

| 라틴어 | 번역 | 시간 |
|--------|------|------|
| Dīxit mīlitēs **pugnāre**. | 그는 병사들이 **싸우고 있다고** 말했다. | 동시 |
| Dīxit mīlitēs **pugnāvisse**. | 그는 병사들이 **싸웠다고** 말했다. | 이전 |
| Dīxit mīlitēs **pugnātūrōs esse**. | 그는 병사들이 **싸울 것이라고** 말했다. | 이후 |

> **핵심 포인트:** 라틴어 부정사 시제는 주절 동사의 시제에 따라 변하지 않습니다. 한국어 번역만 달라집니다.

### 1.4 간접 서술문의 부정

부정사절은 **nōn**으로 부정합니다(부정사나 대격 주어 앞에 놓음):

| 라틴어 | 한국어 |
|--------|--------|
| Dīcit mīlitēs **nōn** pugnāre. | 그는 병사들이 싸우고 있지 **않다고** 말한다. |
| Putō hoc **nōn** vērum esse. | 나는 이것이 사실이 **아니라고** 생각한다. |

## 2. 간접 서술문을 도입하는 동사

### 2.1 말하기 동사

| 라틴어 | 의미 | 예시 |
|--------|------|------|
| **dīcō, dīcere, dīxī, dictum** | 말하다, 이야기하다 | Dīcit sē venīre. |
| **negō, negāre** | ~이 아니라고 말하다, 부정하다 | Negat sē venīre. (= Dīcit sē nōn venīre.) |
| **nārrō, nārrāre** | 서술하다, 이야기하다 | Nārrat rem gestam esse. |
| **referō, referre** | 보고하다 | Refert urbem captam esse. |
| **respondeō, respondēre** | 대답하다 | Respondet sē parātum esse. |
| **trādō, trādere** | 전하다, 전승하다 | Trādit Rōmulum urbem condidisse. |

### 2.2 생각/믿음 동사

| 라틴어 | 의미 |
|--------|------|
| **putō, putāre** | 생각하다, 여기다 |
| **crēdō, crēdere** | 믿다 |
| **exīstimō, exīstimāre** | 판단하다, 여기다 |
| **arbitror, arbitrārī** | 생각하다, 추정하다 |
| **cēnseō, cēnsēre** | 생각하다, 의견을 가지다 |
| **spērō, spērāre** | 희망하다 (미래 부정사를 취함) |

### 2.3 앎/인지 동사

| 라틴어 | 의미 |
|--------|------|
| **sciō, scīre** | 알다 |
| **cognōscō, cognōscere** | 알게 되다, 인식하다 |
| **sentiō, sentīre** | 느끼다, 인지하다 |
| **videō, vidēre** | 보다 |
| **audiō, audīre** | 듣다 |
| **intellegō, intellegere** | 이해하다 |
| **meminī** (결여동사) | 기억하다 |

### 2.4 특별 참고사항

- **spērō**("희망하다")와 **prōmittō**("약속하다")는 일반적으로 **미래** 부정사를 취합니다.
- **negō**는 "~이 아니라고 말하다"를 의미하며 *dīcō...nōn*보다 선호됩니다:
  - **Negat sē venīre.** = 그는 오지 않겠다고 말한다. (*Dīcit sē nōn venīre*보다 자연스러움.)
- **iubeō**("명령하다")는 대격 + 부정사를 취하지만 간접 서술문이 아니라 명령으로 분류됩니다.

## 3. 간접 의문문

### 3.1 형성

간접 의문문은 **접속법**을 사용합니다. 시제는 **시제 조응**(16과 참조)을 따릅니다.

**직접 의문문:** Quid facis? — "너는 무엇을 하고 있니?"
**간접 의문문:** Rogat **quid faciās**. — "그는 네가 **무엇을 하고 있는지** 묻는다."

### 3.2 도입어

| 의문사 | 의미 | 예시 |
|--------|------|------|
| **quis, quid** | 누구, 무엇 | Rogat **quis** vēnerit. |
| **quī, quae, quod** | 어떤, 무슨 | Rogat **quī** librī sint bonī. |
| **ubi** | 어디에 | Nescit **ubi** sit. |
| **unde** | 어디로부터 | Rogat **unde** veniat. |
| **quō** | 어디로 | Quaerit **quō** eat. |
| **quandō** | 언제 | Rogat **quandō** ventūrus sit. |
| **cūr** | 왜 | Nescit **cūr** abierit. |
| **quōmodo** | 어떻게 | Rogat **quōmodo** id factum sit. |
| **num** | ~인지 ("아니오" 예상) | Rogat **num** vērum sit. |
| **nōnne** | ~인지 ("예" 예상) | Rogat **nōnne** hoc vērum sit. |
| **-ne** | ~인지 (중립) | Rogat veniat**ne**. |
| **utrum...an** | ~인지...~인지 | Rogat **utrum** maneat **an** abeat. |

### 3.3 간접 의문문에서의 시제 조응

| 주절 동사 | 종속절 시간 | 접속법 | 예시 |
|----------|-----------|--------|------|
| 주시제 | 동시 | 현재 접속법 | Rogat quid **faciās**. |
| 주시제 | 이전 | 완료 접속법 | Rogat quid **fēceris**. |
| 부시제 | 동시 | 미완료 접속법 | Rogāvit quid **facerēs**. |
| 부시제 | 이전 | 과거완료 접속법 | Rogāvit quid **fēcissēs**. |

### 3.4 간접 의문문을 도입하는 동사

주요 동사: **rogō**(묻다), **quaerō**(묻다, 찾다), **nesciō**(모르다), **sciō**(알다), **dīcō**(말하다), **intellegō**(이해하다), **videō**(살피다), **dubitō**(의심하다, 주저하다).

## 4. 간접 명령문

### 4.1 형성

간접 명령문은 **ut**(긍정) 또는 **ne**(부정) + **접속법**을 사용합니다:

**직접 명령:** Venī! — "와라!"
**간접 명령:** Imperat **ut veniat**. — "그는 **오라고** 명령한다."

| 라틴어 | 한국어 |
|--------|--------|
| Imperāvit **ut mīlitēs prōgrederentur**. | 그는 병사들에게 **진군하라고** 명령했다. |
| Monuit **nē hoc faceret**. | 그는 **이것을 하지 말라고** 경고했다. |
| Ōrāvit **ut sibī ignōsceret**. | 그는 **자신을 용서해 달라고** 간청했다. |
| Persuāsit eī **ut manēret**. | 그는 그에게 **머물도록** 설득했다. |

### 4.2 명령/설득 동사

| 라틴어 | 의미 | 구문 |
|--------|------|------|
| **imperō** (+ 여격) | 명령하다 | ut/ne + 접속법 |
| **moneō** | 경고하다, 충고하다 | ut/ne + 접속법 |
| **hortor** | 격려하다, 촉구하다 | ut/ne + 접속법 |
| **ōrō / rogō** | 간청하다, 요청하다 | ut/ne + 접속법 |
| **persuādeō** (+ 여격) | 설득하다 | ut/ne + 접속법 |
| **petō** | 구하다, 요청하다 | ut/ne + 접속법 |
| **cōgō** | 강요하다 | (보통 대격 + 부정사) |
| **iubeō** | 명령하다 | (대격 + 부정사, ut 아님) |
| **vetō** | 금지하다 | (대격 + 부정사, ne 아님) |

> **중요:** **iubeō**와 **vetō**는 *ut/ne*를 사용하지 않고, 대격 + 부정사 구문을 취합니다:
> - **Iubet eum venīre.** — 그는 그에게 오라고 명령한다.
> - **Vetat eum venīre.** — 그는 그가 오는 것을 금지한다.

## 5. 간접화법에서의 재귀대명사

### 5.1 규칙

간접화법 내에서 재귀대명사 **sē**(와 소유 형용사 **suus, -a, -um**)는 **주절 동사의 주어**(원래의 화자/사고자)를 가리킵니다:

| 라틴어 | 한국어 |
|--------|--------|
| Caesar dīxit **sē** ventūrum esse. | 카이사르는 **자신이** 올 것이라고 말했다. |
| Caesar dīxit **eum** ventūrum esse. | 카이사르는 **그(다른 사람)가** 올 것이라고 말했다. |

### 5.2 직접화법 vs 간접화법 비교

| 직접화법 | 간접화법 |
|---------|---------|
| "Veniam." (나는 올 것이다.) | Dīxit **sē** ventūrum esse. (그는 자신이 올 것이라고 말했다.) |
| "Meum librum legō." (나는 내 책을 읽고 있다.) | Dīxit **sē suum** librum legere. (그는 자신의 책을 읽고 있다고 말했다.) |
| "Tū mē adiuvās." (너는 나를 돕고 있다.) | Dīxit eum **sē** adiuvāre. (그는 그가 자신을 돕고 있다고 말했다.) |

### 5.3 정리표

| 대명사 | 간접화법에서 | 가리키는 대상 |
|--------|-----------|------------|
| **sē** (대격/탈격) | 재귀 | 주절 동사의 주어 |
| **sibi** (여격) | 재귀 | 주절 동사의 주어 |
| **suus, -a, -um** | 재귀 소유격 | 주절 동사의 주어 |
| **eum / eam / id** | 비재귀 | 주절 주어 이외의 사람 |
| **eius** | 비재귀 소유격 | 주절 주어 이외의 사람 |

## 6. 번역 연습

### 6.1 직접화법에서 간접화법으로

다음 직접 서술을 주어진 주절 동사를 사용하여 간접화법으로 전환하시오:

| # | 직접화법 | 주절 동사 | 간접화법 |
|---|---------|----------|---------|
| 1 | "Hostēs fugiunt." | Nūntius dīcit... | Nūntius dīcit hostēs fugere. |
| 2 | "Urbem cēpimus." | Imperātor scrīpsit... | Imperātor scrīpsit sē urbem cēpisse. |
| 3 | "Veniam." | Amīcus prōmīsit... | Amīcus prōmīsit sē ventūrum esse. |
| 4 | "Cūr abīs?" | Māter rogat... | Māter rogat cūr abeās. |
| 5 | "Proficīscere!" | Dux imperāvit... | Dux imperāvit ut proficīscerētur. |

### 6.2 간접화법에서 직접화법으로

| # | 간접화법 | 직접화법 |
|---|---------|---------|
| 1 | Scīmus terram rotundam esse. | "Terra rotunda est." |
| 2 | Caesar dīxit sē exercitum Rhēnum trādūctūrum esse. | "Exercitum Rhēnum trādūcam." |
| 3 | Cicerō rogāvit cūr Catilīna in senātum vēnisset. | "Cūr, Catilīna, in senātum vēnistī?" |

## 7. 연습 문제

### 연습 A: 구문 식별

각 문장이 **간접 서술문**, **간접 의문문**, 또는 **간접 명령문**을 포함하는지 밝히시오:

1. Putō eum bonum virum esse.
2. Rogāvit ubi Caesar esset.
3. Imperāvit ut portās clauderent.
4. Audiō tē Rōmam profectūrum esse.
5. Nescit num amīcī ventūrī sint.
6. Hortātus est nē timērent.
7. Crēdimus exercitum victum esse.
8. Dīcit sē hoc numquam factūrum esse.

<details>
<summary>정답 보기</summary>

1. **간접 서술문** — putō + 대격(eum) + 부정사(esse). "나는 그가 좋은 사람이라고 생각한다."
2. **간접 의문문** — rogāvit + ubi + 접속법(esset). "그는 카이사르가 어디에 있는지 물었다."
3. **간접 명령문** — imperāvit + ut + 접속법(clauderent). "그는 문을 닫으라고 명령했다."
4. **간접 서술문** — audiō + 대격(tē) + 부정사(profectūrum esse). "나는 네가 로마로 출발할 것이라고 듣는다."
5. **간접 의문문** — nescit + num + 접속법(sint). "그는 친구들이 올지 모른다."
6. **간접 명령문** — hortātus est + nē + 접속법(timērent). "그는 두려워하지 말라고 촉구했다."
7. **간접 서술문** — crēdimus + 대격(exercitum) + 부정사(victum esse). "우리는 군대가 패배했다고 믿는다."
8. **간접 서술문** — dīcit + 대격(sē) + 부정사(factūrum esse). "그는 이것을 결코 하지 않겠다고 말한다."

</details>

### 연습 B: 올바른 부정사 선택

빈칸에 올바른 부정사 시제(현재, 완료, 미래)를 넣으시오:

1. Dīcit mīlitēs fortiter ________. (pugn- : 동시)
2. Scīmus Rōmam ā Rōmulō ________. (cond- : 이전)
3. Spērat exercitum ________. (vinc- : 이후)
4. Audīvī tē aegrōtum ________. (sum : audīvī와 동시)
5. Putāmus eōs iam ________. (proficīsc- : 이전)

<details>
<summary>정답 보기</summary>

1. **pugnāre** — 현재 부정사 (주절 동사와 동시). "그는 병사들이 용감하게 싸우고 있다고 말한다."
2. **conditam esse** — 완료 수동 부정사 (이전). "우리는 로마가 로물루스에 의해 건설되었다는 것을 안다."
3. **victūrum esse** — 미래 능동 부정사 (이후). "그는 군대가 승리할 것이라고 희망한다."
4. **esse** — 현재 부정사 (부시제 주절 동사와 동시). "나는 네가 아프다고 들었다."
5. **profectōs esse** — 완료(탈형동사) 부정사 (이전). "우리는 그들이 이미 출발했다고 생각한다."

</details>

### 연습 C: 라틴어로 번역

1. 그는 자신이 오고 있다고 말한다.
2. 우리는 도시가 점령되었다고 믿었다.
3. 그녀는 소년이 왜 울고 있는지 물었다.
4. 집정관은 병사들에게 성벽을 방어하라고 명령했다.
5. 그는 돌아오겠다고 약속했다.

<details>
<summary>정답 보기</summary>

1. **Dīcit sē venīre.** (간접 서술문; sē = 주어를 가리키는 재귀; 현재 부정사 = 동시)
2. **Crēdēbāmus urbem captam esse.** (간접 서술문; 완료 수동 부정사 = 주절 동사보다 이전)
3. **Rogāvit cūr puer flēret.** (간접 의문문; 부시제 → 미완료 접속법)
4. **Cōnsul imperāvit ut mīlitēs mūrōs dēfenderent.** (간접 명령문; ut + 미완료 접속법)
5. **Prōmīsit sē reditūrum esse.** (간접 서술문; 미래 능동 부정사 = 주절 동사보다 이후)

</details>

### 연습 D: 재귀인가 비재귀인가?

빈칸에 **sē/suus** 또는 **eum/eius**를 선택하고 설명하시오:

1. Marcus dīxit ________ venīre. (마르쿠스 자신이 오고 있음)
2. Marcus dīxit ________ venīre. (루키우스가 오고 있음)
3. Fēmina crēdit ________ librum bonum esse. (자신의 책)
4. Fēmina crēdit ________ librum bonum esse. (다른 여인의 책)

<details>
<summary>정답 보기</summary>

1. **sē** — 재귀; 주절 주어인 마르쿠스를 가리킴. "마르쿠스는 자신이 오고 있다고 말했다."
2. **eum** — 비재귀; 마르쿠스가 아닌 다른 사람을 가리킴. "마르쿠스는 그(루키우스)가 오고 있다고 말했다."
3. **suum** — 재귀 소유격; 그 여인(주절 주어)을 가리킴. "그 여인은 자신의 책이 좋다고 믿는다."
4. **eius** — 비재귀 소유격; 다른 여인을 가리킴. "그 여인은 그녀(다른 여인)의 책이 좋다고 믿는다."

</details>
