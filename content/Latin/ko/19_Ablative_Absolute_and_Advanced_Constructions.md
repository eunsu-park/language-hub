# 19. 탈격 독립과 고급 구문 (Ablātīvus Absolūtus et Cōnstrūctiōnēs Prōgredientēs)

## 1. 탈격 독립

### 1.1 탈격 독립이란?

탈격 독립은 **탈격**으로 된 **명사(또는 대명사)** + **분사**(또는 다른 명사/형용사)로 구성되며, 문장의 나머지 부분과 문법적으로 **독립적인** 구문이다. **시간, 원인, 조건, 양보**를 표현하는 종속절로 기능한다.

> **"독립(Absolute)"**은 라틴어 *absolūtus* — "풀린, 해방된"에서 유래한다. 이 구문이 주절에서 통사적으로 "풀려나" 있기 때문이다. 탈격 독립의 명사는 주절의 주어나 목적어가 되어서는 안 된다.

### 1.2 기본 패턴

```
[명사/대명사 (탈격)] + [분사 (탈격)] , [주절]
```

| 라틴어 | 한국어 | 절의 의미 |
|-------|--------|----------|
| **Urbe captā**, mīlitēs gāvīsī sunt. | **도시가 점령된 후**, 군인들이 기뻐했다. | 시간: 이후/~할 때 |
| **Caesare duce**, Rōmānī vīcērunt. | **카이사르가 지휘관인 상태로**, 로마인들이 승리했다. | 상황 |
| **Hostibus fugientibus**, castra cēpimus. | **적이 도주하는 동안**, 우리가 진영을 점령했다. | 시간: ~하는 동안 |

### 1.3 독립성 규칙

탈격 독립의 명사는 주절의 주어, 목적어 또는 기타 문법적 요소로 사용되어서는 안 된다.

**올바른 탈격 독립:**
- **Urbe captā**, mīlitēs gāvīsī sunt. (Urbe는 주절의 주어나 목적어가 아니다.)

**탈격 독립이 아님 (주어와 일치하는 분사를 사용해야 함):**
- ~~Caesare veniente, Caesar dīxit...~~ → **Caesar veniēns dīxit...** (Caesar가 주절의 주어이므로, 주격 분사를 사용한다.)

### 1.4 번역 전략

탈격 독립은 한국어로 다양하게 번역할 수 있다:

| 전략 | *Urbe captā*의 번역 예 |
|------|----------------------|
| 직역 ("~된 후") | 도시가 점령된 후, ... |
| 시간절 | 도시가 점령되었을 때, ... |
| 이유절 | 도시가 점령되었기 때문에, ... |
| 양보절 | 도시가 점령되었음에도 불구하고, ... |
| 전치사구 | 도시의 점령 이후, ... |

문맥에 가장 적합한 전략을 선택한다.

## 2. 탈격 독립의 유형

### 2.1 현재 능동 분사 사용

주동사와 **동시에 일어나는** 행위를 표현한다:

| 라틴어 | 한국어 |
|-------|--------|
| **Sole oriente**, profectī sumus. | **해가 뜨고 있을 때**, 우리는 출발했다. |
| **Omnibus audientibus**, ōrātiōnem habuit. | **모두가 듣고 있는 가운데**, 그는 연설을 했다. |
| **Hostibus appropin­quantibus**, portās clausērunt. | **적이 접근하고 있을 때**, 그들은 성문을 닫았다. |

### 2.2 완료 수동 분사(PPP) 사용

주동사 **이전에 완료된** 행위를 표현한다:

| 라틴어 | 한국어 |
|-------|--------|
| **Bellō cōnfectō**, domum rediit. | **전쟁이 끝난 후**, 그는 집으로 돌아왔다. |
| **Hīs rēbus cognitīs**, Caesar profectus est. | **이 일들을 알게 된 후**, 카이사르는 출발했다. |
| **Signō datō**, mīlitēs impetum fēcērunt. | **신호가 주어진 후**, 군인들이 공격했다. |

### 2.3 명사 + 명사 또는 명사 + 형용사

분사를 사용할 수 없는 경우(예: **esse**에는 현재 분사가 없음), 명사 + 명사 또는 명사 + 형용사 구문이 그 역할을 대신한다:

| 라틴어 | 한국어 | 패턴 |
|-------|--------|------|
| **Caesare duce** | 카이사르가 지휘관인 상태로 | 명사 + 명사 |
| **Cicerōne cōnsule** | 키케로가 집정관이었을 때 | 명사 + 명사 |
| **Mē invītō** | 나의 의지에 반하여 (내가 원하지 않는 상태로) | 대명사 + 형용사 |
| **Caelō serēnō** | 하늘이 맑은 상태로 | 명사 + 형용사 |
| **Nūllā morā factā** | 지체 없이 (아무런 지체가 이루어지지 않은 상태로) | 명사(+형용사) + PPP |

### 2.4 흔한 탈격 독립 구문

이 고정 표현들은 라틴어 산문, 특히 카이사르와 리비우스에서 자주 등장한다:

| 라틴어 | 한국어 |
|-------|--------|
| **hīs rēbus cognitīs** | 이 일들을 알게 되었을 때 |
| **quibus rēbus cognitīs** | 이 일들을 알게 되었을 때 |
| **proeliō commissō** | 전투가 시작되었을 때 |
| **ponte factō** | 다리가 건설되었을 때 |
| **cōnsiliō captō** | 계획이 채택된 후 |
| **omnibus rēbus parātīs** | 모든 것이 준비된 후 |
| **pāce factā** | 평화가 이루어진 후 |
| **operiō prīmā lūce** | 첫 새벽에 |

## 3. 목적 분사

목적 분사(supine)는 제4변화 **동사 명사**로, 네 번째 기본형에서 만들어진다. **대격**과 **탈격** 단수형만 존재한다.

### 3.1 대격 목적 분사 (이동 동사와 함께 목적 표현)

대격 목적 분사(**-um** 어미)는 이동 동사 뒤에서 **목적**을 표현한다:

| 라틴어 | 한국어 |
|-------|--------|
| Vēnit **spectātum**. | 그는 **구경하러** 왔다. |
| Legiōnēs mīsit **frūmentum** **petītum**. | 그는 **곡물을 구하러** 군단을 보냈다. |
| Eō **dormītum**. | 나는 **자러** 간다. |
| Vēnērunt **auxilium** **rogātum**. | 그들은 **도움을 청하러** 왔다. |
| Īvit **lūsum**. | 그는 **놀러** 갔다. |

> **참고:** 대격 목적 분사는 직접 목적어를 취할 수 있다: *mīsit lēgātōs pācem **petītum*** — "그는 평화를 구하러 사절들을 보냈다."

### 3.2 탈격 목적 분사 (관점 / 명세)

탈격 목적 분사(**-ū** 어미)는 특정 형용사와 함께 **어떤 점에서** 그러한지를 표현한다:

| 라틴어 | 한국어 |
|-------|--------|
| Hoc est facile **dictū**. | 이것은 **말하기** 쉽다. |
| Rēs est difficilis **factū**. | 그 일은 **하기** 어렵다. |
| Mīrābile **vīsū**! | **보기에** 놀랍다! |
| Horribile **audītū**! | **듣기에** 끔찍하다! |
| Optimum **cognitū** | **알기에** 가장 좋은 |

탈격 목적 분사와 함께 쓰이는 일반적인 형용사: **facilis** (쉬운), **difficilis** (어려운), **mīrābilis** (놀라운), **horribilis** (끔찍한), **optimus** (가장 좋은), **turpis** (부끄러운), **incrēdibilis** (믿을 수 없는).

## 4. 처소격

처소격은 전치사 없이 **장소**를 표현하는 잔존 격이다. 다음에 대해 남아 있다:

### 4.1 도시와 작은 섬

| 변화 | 처소격 어미 | 예 | 의미 |
|------|-----------|-----|------|
| 제1변화 단수 | **-ae** | **Rōmae** | 로마에서 |
| 제2변화 단수 | **-ī** | **Corinthī** | 코린토스에서 |
| 제3변화 단수 | **-ī** 또는 **-e** | **Carthāginī** / **Carthāgine** | 카르타고에서 |
| 복수 (모든 변화) | **-īs** | **Athēnīs** | 아테네에서 |
| | | **Syrācūsīs** | 시라쿠사에서 |

### 4.2 특수 일반 명사

몇몇 일반 명사가 처소격을 유지한다:

| 라틴어 | 의미 |
|-------|------|
| **domī** | 집에서 |
| **rūrī** (또는 **rūre**) | 시골에서 |
| **humī** | 땅 위에서 |
| **bellī** (고어) | 전쟁 중에 (드묾) |
| **mīlitiae** | 군복무 중에 |

### 4.3 장소 구문 요약

| 개념 | 도시/작은 섬 | 일반 명사 |
|------|------------|----------|
| **~에서 (장소)** | 처소격 (Rōmae) | in + 탈격 (in urbe) |
| **~으로 (방향)** | 대격 단독 (Rōmam) | ad/in + 대격 (ad urbem) |
| **~에서부터 (출발)** | 탈격 단독 (Rōmā) | ā/ab/ē/ex + 탈격 (ex urbe) |
| **특수** | domī, rūrī, humī (처소격) | domum (집으로), domō (집에서부터) |

### 4.4 예문

| 라틴어 | 한국어 |
|-------|--------|
| **Rōmae** habitat. | 그는 **로마에서** 산다. |
| **Rōmam** proficīscitur. | 그는 **로마로** 출발한다. |
| **Rōmā** discessit. | 그는 **로마에서** 떠났다. |
| **Domī** manēbō. | 나는 **집에** 있겠다. |
| **Domum** revertit. | 그는 **집으로** 돌아왔다. |
| **Domō** exiit. | 그는 **집에서** 나갔다. |

## 5. 특수 형용사 및 동사 구문

### 5.1 여격을 지배하는 동사

"돕다," "기쁘게 하다," "해치다," "신뢰하다" 등으로 번역되는 많은 라틴어 동사들은 대격 대신 **여격** 목적어를 취한다:

| 라틴어 | 의미 | 예문 |
|-------|------|------|
| **crēdō** | 믿다, 신뢰하다 | Tibi **crēdō**. — 나는 **너를** 신뢰한다. |
| **noceō** | 해치다 | Nēminī **nocēbō**. — 나는 **아무도** 해치지 않겠다. |
| **pāreō** | 복종하다 | Lēgibus **pārē**. — **법에** 복종하라. |
| **placeō** | 기쁘게 하다 | Hoc mihi **placet**. — 이것이 나를 **기쁘게 한다**. |
| **persuādeō** | 설득하다 | Eī **persuāsit**. — 그는 **그를** 설득했다. |
| **imperō** | 명령하다 | Mīlitibus **imperāvit**. — 그는 **군인들에게** 명령했다. |
| **serviō** | 섬기다 | Patriae **servīmus**. — 우리는 **조국을** 섬긴다. |
| **studeō** | 열중하다 | Litterīs **studet**. — 그는 **문학에** 열중한다. |
| **resistō** | 저항하다 | Hostibus **resistunt**. — 그들은 **적에게** 저항한다. |
| **faveō** | 호의를 보이다 | Populō **favet**. — 그는 **국민에게** 호의를 보인다. |
| **invideō** | 질투하다 | Nēminī **invideō**. — 나는 **아무도** 질투하지 않는다. |
| **ignōscō** | 용서하다 | Mihi **ignōsce**. — **나를** 용서하라. |

### 5.2 탈격을 지배하는 동사

| 라틴어 | 의미 | 예문 |
|-------|------|------|
| **ūtor** | 사용하다 | Gladiō **ūtitur**. — 그는 **칼을** 사용한다. |
| **fruor** | 즐기다 | Vītā **fruimur**. — 우리는 **삶을** 즐긴다. |
| **fungor** | 수행하다 | Officiō **fungitur**. — 그는 **의무를** 수행한다. |
| **potior** | 차지하다 | Urbe **potītus est**. — 그는 **도시를** 차지했다. |
| **vescor** | 먹다 | Carne **vescitur**. — 그는 **고기를** 먹는다. |

> **암기 도움말: UFFPV** — Ūtor, Fruor, Fungor, Potior, Vescor (모두 형식 수동태, 모두 + 탈격).

### 5.3 속격을 지배하는 동사와 형용사

| 유형 | 단어 | 의미 | 예문 |
|------|------|------|------|
| 동사 | **meminī** | 기억하다 | Patriae **meminī**. — 나는 **조국을** 기억한다. |
| 동사 | **oblīvīscor** | 잊다 | Iniūriārum **oblīvīscere**. — **부당함을** 잊어라. |
| 동사 | **misereor** | 불쌍히 여기다 | Nostrī **miserēre**. — **우리를** 불쌍히 여겨라. |
| 형용사 | **plēnus** | ~로 가득 찬 | Plēna **perīculī**. — **위험으로** 가득 찬. |
| 형용사 | **cupidus** | ~을 갈망하는 | Cupidus **glōriae**. — **영광을** 갈망하는. |
| 형용사 | **perītus** | ~에 숙련된 | Perītus **bellī**. — **전쟁에** 숙련된. |
| 형용사 | **memor** | ~을 기억하는 | Memor **beneficiōrum**. — **은혜를** 기억하는. |

## 6. 비인칭 동사

비인칭 동사는 인칭 주어가 없으며, **3인칭 단수**로만 나타난다 (부정사나 접속법 절이 논리적 주어 역할을 한다).

### 6.1 일반적인 비인칭 동사

| 라틴어 | 의미 | 구문 | 예문 |
|-------|------|------|------|
| **licet** | 허락되다 | 여격 + 부정사 | Mihi **licet** abīre. — 나는 떠나도 **된다**. |
| **oportet** | ~해야 하다 | 대격 + 부정사 | Tē **oportet** studēre. — 너는 공부**해야 한다**. |
| **decet** | 어울리다 | 대격 + 부정사 | Tē **decet** esse fortem. — 용감한 것이 너에게 **어울린다**. |
| **dēdecet** | 어울리지 않다 | 대격 + 부정사 | Hoc tē **dēdecet**. — 이것은 너에게 **어울리지 않는다**. |
| **placet** | 마음에 들다 | 여격 + 부정사 | Mihi **placet** manēre. — 나는 머무는 것이 **좋다**. |
| **piget** | 괴롭히다 | 대격 + 속격 | Mē **piget** stultitiae. — 나는 어리석음이 **부끄럽다**. |
| **pudet** | 부끄럽게 하다 | 대격 + 속격 | Mē **pudet** factī. — 나는 그 행위가 **부끄럽다**. |
| **paenitet** | 후회하게 하다 | 대격 + 속격 | Mē **paenitet** errōris. — 나는 그 실수를 **후회한다**. |
| **taedet** | 지겹게 하다 | 대격 + 속격 | Mē **taedet** vītae. — 나는 삶이 **지겹다**. |
| **miseret** | 동정을 느끼게 하다 | 대격 + 속격 | Mē **miseret** tuī. — 나는 너를 **동정한다**. |

### 6.2 Necesse Est와 Opus Est

| 구문 | 의미 | 예문 |
|------|------|------|
| **necesse est** + 여격 + 부정사 | ~해야 한다 | Mihi **necesse est** proficīscī. — 나는 출발**해야 한다**. |
| **necesse est** + 접속법 | ~하는 것이 필요하다 | **Necesse est** ut abeās. — 네가 떠나는 것이 **필요하다**. |
| **opus est** + 탈격 | ~이 필요하다 | Cōnsiliō **opus est**. — 계획이 **필요하다**. |
| **opus est** + 여격 + 탈격 | 누군가에게 무언가가 필요하다 | Mihi pecūniā **opus est**. — 나에게 돈이 **필요하다**. |

### 6.3 비인칭 수동태

모든 자동사는 일반적인 행위를 표현하기 위해 비인칭 수동태로 사용될 수 있다:

| 라틴어 | 직역 | 자연스러운 한국어 |
|-------|------|-----------------|
| **Pugnātum est.** | 싸워졌다. | 전투가 벌어졌다. |
| **Ventum est.** | 와졌다. | 사람들이 도착했다. |
| **Curritur.** | 달려진다. | 사람들이 달린다. |
| **Ā mīlitibus fortiter pugnātum est.** | 군인들에 의해 용감하게 싸워졌다. | 군인들이 용감하게 싸웠다. |

## 7. 연습 문제

### 연습 A: 탈격 독립

가장 자연스러운 한국어로 번역하시오:

1. Hostibus victīs, pāx facta est.
2. Sole occidente, domum rediimus.
3. Caesare cōnsule, multae lēgēs lātae sunt.
4. Hīs rēbus cognitīs, senātus bellum dēcrēvit.
5. Mē invītō, profectus est.
6. Proeliō commissō, mīlitēs fortiter pugnāvērunt.

<details>
<summary>정답 보기</summary>

1. "적이 패배한 후, 평화가 이루어졌다." (PPP 탈격 독립)
2. "해가 지고 있을 때, 우리는 집으로 돌아왔다." (현재 분사 탈격 독립)
3. "키케로가 집정관이었을 때, 많은 법이 통과되었다." (명사 + 명사; *esse*에 분사가 없으므로 분사 불필요)
4. "이 일들을 알게 되었을 때, 원로원은 전쟁을 선포했다." (PPP; 카이사르에게 흔한 표현)
5. "나의 의지에 반하여, 그는 떠났다." (대명사 + 형용사 탈격 독립)
6. "전투가 시작되었을 때, 군인들은 용감하게 싸웠다." (PPP 탈격 독립)

</details>

### 연습 B: 목적 분사와 처소격

한국어로 번역하시오:

1. Eō dormītum.
2. Rēs est difficilis factū.
3. Rōmae habitat.
4. Domum revertit.
5. Mīrābile vīsū!
6. Vēnērunt auxilium rogātum.

<details>
<summary>정답 보기</summary>

1. "나는 자러 간다." (대격 목적 분사 — 이동 동사와 함께 목적 표현)
2. "그 일은 하기 어렵다." (탈격 목적 분사 — 관점)
3. "그는 로마에서 산다." (처소격)
4. "그는 집으로 돌아왔다." (전치사 없는 방향의 대격)
5. "보기에 놀랍다!" (탈격 목적 분사)
6. "그들은 도움을 청하러 왔다." (직접 목적어가 있는 대격 목적 분사)

</details>

### 연습 C: 특수 동사 구문

괄호 안의 단어에 대해 올바른 격을 제시하시오:

1. Tibi (crēdō). — crēdō의 목적어는 ________ 격이어야 한다.
2. Gladiō (ūtor). — ūtor의 목적어는 ________ 격이어야 한다.
3. Patriae (meminī). — meminī의 목적어는 ________ 격이어야 한다.
4. Mihi (licet) abīre. — Mihi는 ________ 격이다.
5. Mē (piget) errōris. — Mē는 ________; errōris는 ________.

<details>
<summary>정답 보기</summary>

1. **여격** — crēdō는 여격 목적어를 취한다. "나는 너를 신뢰한다."
2. **탈격** — ūtor는 탈격 목적어를 취한다. "그는 칼을 사용한다."
3. **속격** — meminī는 속격 목적어를 취한다. "나는 조국을 기억한다."
4. **여격** — licet는 비인칭이다; 허락받는 사람은 여격이다. "나는 떠나도 된다."
5. **대격**; **속격** — piget는 영향받는 사람의 대격과 원인의 속격을 취한다. "나는 실수가 부끄럽다."

</details>

### 연습 D: 라틴어로 번역

1. 다리가 건설된 후, 카이사르는 군대를 건너게 했다.
2. 말하기 쉽고, 하기 어렵다.
3. 그는 집에서 살지만 아테네로 갔다.
4. 너는 법에 복종해야 한다.
5. 여러 시간 동안 전투가 벌어졌다.

<details>
<summary>정답 보기</summary>

1. **Ponte factō, Caesar exercitum trādūxit.** (PPP를 사용한 탈격 독립)
2. **Facile dictū est, difficile factū.** (형용사와 함께 쓰인 탈격 목적 분사)
3. **Domī habitat sed Athēnās iit.** (처소격 *domī*; 방향의 대격 *Athēnās*)
4. **Tē oportet lēgibus pārēre.** (비인칭 *oportet* + 대격 + 부정사; *pārēre* + 여격)
5. **Multās hōrās pugnātum est.** (비인칭 수동태)

</details>

### 연습 E: 구문 식별

각 밑줄 친 요소를 다음 중에서 골라 분류하시오: **탈격 독립**, **목적 분사**, **처소격**, **비인칭 동사**, **특수 격 구문**:

1. _Rēbus omnibus parātīs_, profectī sumus.
2. Vēnit _spectātum_.
3. _Rōmae_ multōs annōs vīxit.
4. Mihi _licet_ loquī.
5. Hostibus _resistunt_.

<details>
<summary>정답 보기</summary>

1. **탈격 독립** — Rēbus omnibus parātīs (명사 + 형용사 + PPP, 모두 탈격)
2. **목적 분사** (대격) — spectātum은 이동 동사 뒤에서 목적을 표현
3. **처소격** — Rōmae, Rōma의 처소격 (제1변화 도시)
4. **비인칭 동사** — licet (허락되다); mihi = 여격
5. **특수 격 구문** — resistō는 여격(hostibus)을 지배

</details>
