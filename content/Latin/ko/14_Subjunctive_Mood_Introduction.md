## 1. 접속법이란?

라틴어에는 세 가지 법이 있습니다:

```
┌────────────────────────────────────────────────────────────────┐
│   직설법     → 사실을 진술하고 직접 질문을 함                     │
│               "카이사르가 갈리아를 정복했다."                      │
│                                                                │
│   접속법     → 가능성, 소원, 목적, 의심, 명령(간접적/                │
│               부드러운), 조건을 표현함                             │
│               "카이사르가 갈리아를 정복하기를."                     │
│               "갈리아를 정복하자."                                │
│               "...카이사르가 갈리아를 정복하도록."                  │
│                                                                │
│   명령법     → 직접적인 명령을 내림                               │
│               "갈리아를 정복하라!"                                │
└────────────────────────────────────────────────────────────────┘
```

접속법은 라틴어에서 **매우 자주** 사용됩니다 — 영어보다 훨씬 많이. 다음에서 나타납니다:
- 독립(주)절: 소원, 부드러운 명령, 가능성
- 종속(부)절: 목적, 결과, 원인, 양보, 간접 의문, 조건

## 2. 현재 접속법 형성

### 2.1 모음 변화 규칙

현재 접속법의 핵심은 **특징적 모음 변화**입니다:

| 활용 | 직설법 모음 | 접속법 모음 | 기억법 |
|------|-----------|-----------|--------|
| 제1 (-āre) | ā | **ē** | "She wEars a diamond." |
| 제2 (-ēre) | ē | **eā** | |
| 제3 (-ere) | e/i | **ā** | |
| 제3-iō | i | **iā** | |
| 제4 (-īre) | ī | **iā** | |

> **전체 활용 기억법**: "wE bEAt A lIAr" — ē, eā, ā, iā.

### 2.2 현재 접속법: 전 활용

| 인칭 | 제1: amō | 제2: moneō | 제3: dūcō | 제3-iō: capiō | 제4: audiō |
|------|----------|-----------|-----------|---------------|-----------|
| 1단 | am**em** | mon**eam** | dūc**am** | cap**iam** | aud**iam** |
| 2단 | am**ēs** | mon**eās** | dūc**ās** | cap**iās** | aud**iās** |
| 3단 | am**et** | mon**eat** | dūc**at** | cap**iat** | aud**iat** |
| 1복 | am**ēmus** | mon**eāmus** | dūc**āmus** | cap**iāmus** | aud**iāmus** |
| 2복 | am**ētis** | mon**eātis** | dūc**ātis** | cap**iātis** | aud**iātis** |
| 3복 | am**ent** | mon**eant** | dūc**ant** | cap**iant** | aud**iant** |

### 2.3 *esse*와 *posse*의 현재 접속법

| 인칭 | esse | posse |
|------|------|-------|
| 1단 | sim | possim |
| 2단 | sīs | possīs |
| 3단 | sit | possit |
| 1복 | sīmus | possīmus |
| 2복 | sītis | possītis |
| 3복 | sint | possint |

### 2.4 현재 접속법과 직설법 구별

| 형태 | 법 | 구별 방법 |
|------|-----|----------|
| *amat* | 직설법 | 제1활용이 -a-를 유지 |
| *amet* | 접속법 | 제1활용이 a → e로 변화 |
| *dūcit* | 직설법 | 제3활용이 -i-를 가짐 |
| *dūcat* | 접속법 | 제3활용이 -a-를 가짐 |

> **주의**: 제3·4활용 현재 접속법(*dūcam, audiam*)은 미래 직설법 1인칭 단수와 동일하게 보입니다. 문맥으로 해결합니다.

## 3. 미완료 접속법 형성

### 3.1 라틴어 문법에서 가장 간단한 규칙

**미완료 접속법 = 현재 능동 부정사 + 인칭 어미**

| 인칭 | 어미 |
|------|------|
| 1단 | -m |
| 2단 | -s |
| 3단 | -t |
| 1복 | -mus |
| 2복 | -tis |
| 3복 | -nt |

### 3.2 미완료 접속법: 전 활용

| 인칭 | 제1: amō | 제2: moneō | 제3: dūcō | 제3-iō: capiō | 제4: audiō |
|------|----------|-----------|-----------|---------------|-----------|
| 1단 | amāre**m** | monēre**m** | dūcere**m** | capere**m** | audīre**m** |
| 2단 | amārē**s** | monērē**s** | dūcerē**s** | caperē**s** | audīrē**s** |
| 3단 | amāre**t** | monēre**t** | dūcere**t** | capere**t** | audīre**t** |
| 1복 | amārē**mus** | monērē**mus** | dūcerē**mus** | caperē**mus** | audīrē**mus** |
| 2복 | amārē**tis** | monērē**tis** | dūcerē**tis** | caperē**tis** | audīrē**tis** |
| 3복 | amāre**nt** | monēre**nt** | dūcere**nt** | capere**nt** | audīre**nt** |

### 3.3 *esse*의 미완료 접속법

| 인칭 | 형태 |
|------|------|
| 1단 | essem |
| 2단 | essēs |
| 3단 | esset |
| 1복 | essēmus |
| 2복 | essētis |
| 3복 | essent |

> 같은 규칙을 따릅니다: *esse*의 부정사는 *esse* → *esse-m, esse-s, esse-t*...

## 4. 접속법의 독립 용법

접속법이 **주절**에 나타날 때(접속사로 도입되지 않을 때), 여러 가지 고유한 용법이 있습니다.

### 4.1 권유적 접속법 ("~하자")

**1인칭 복수, 현재 접속법**

| 라틴어 | 번역 |
|--------|------|
| *Eāmus!* | 가자! |
| *Gaudeāmus igitur!* | 그러므로 기뻐하자! |
| *Patriam dēfendāmus.* | 조국을 지키자. |

> **nē**로 부정합니다: *Nē timeāmus.* — 두려워하지 말자.

### 4.2 명령적 접속법 ("그로 하여금 ~하게 하라")

**3인칭, 현재 접속법**

| 라틴어 | 번역 |
|--------|------|
| *Veniat.* | 그로 하여금 오게 하라. |
| *Fīant lūx.* | 빛이 있으라. (단수: *Fiat lūx.*) |
| *Caveant cōnsulēs.* | 집정관들이 경계하게 하라. |
| *Nē abeat.* | 그가 떠나지 않게 하라. |

### 4.3 기원적 접속법 (소원)

**현재 접속법**은 실현 가능한 소원에, **미완료 접속법**은 현재 사실에 반하는 소원에 사용합니다. 흔히 *utinam*과 함께 씁니다.

| 라틴어 | 번역 | 유형 |
|--------|------|------|
| *Utinam veniat!* | 그가 오기를! / 그가 오면 좋겠다! | 실현 가능한 소원 |
| *Utinam venīret!* | 그가 오고 있다면 좋겠는데! (그러나 오고 있지 않다) | 반사실적 |
| *Utinam nē abiret!* | 그가 떠나고 있지 않다면 좋겠는데! | 반사실적 |

### 4.4 심의적 접속법 ("~해야 하는가?")

**1인칭(가끔 3인칭), 현재 또는 미완료 접속법, 의문문에서**

| 라틴어 | 번역 |
|--------|------|
| *Quid faciam?* | 내가 무엇을 해야 하는가? / 어떻게 해야 하지? |
| *Quō eam?* | 어디로 가야 하는가? |
| *Quid dīcerem?* | 내가 무엇을 말해야 했는가? (과거의 심의) |

### 4.5 가능적 접속법 ("~할 수도 있다/~할 것이다")

**현재 또는 미완료 접속법**, 가능성이나 완곡한 주장을 표현합니다.

| 라틴어 | 번역 |
|--------|------|
| *Aliquis dīcat.* | 누군가가 말할 수도 있다. |
| *Crēdās.* | 믿을 법하다. (= 사람이 믿을 수도 있다.) |
| *Velim.* | 원하겠다. |
| *Nōllem.* | 원하지 않겠다. |

### 4.6 요약표

| 용법 | 인칭 | 시제 | 번역 단서 | 부정 |
|------|------|------|----------|------|
| 권유적 | 1인칭 복수 | 현재 | "~하자" | nē |
| 명령적 | 3인칭 | 현재 | "~하게 하라" | nē |
| 기원적 | 모든 인칭 | 현재 또는 미완료 | "~하기를.../~하면 좋겠는데..." | nē / utinam nē |
| 심의적 | 1인칭 (3인칭) | 현재 또는 미완료 | "~해야 하는가?" | nōn |
| 가능적 | 모든 인칭 | 현재 또는 미완료 | "~할 수도 있다/~하겠다" | nōn |

## 5. 목적절 (ut/nē)

목적절은 **"왜?"** 또는 **"무엇을 위해?"**에 답하며 접속법을 사용합니다.

### 5.1 형성

| 유형 | 접속사 | + 접속법 |
|------|--------|---------|
| 긍정 목적 | **ut** | 현재 또는 미완료 접속법 |
| 부정 목적 | **nē** | 현재 또는 미완료 접속법 |

### 5.2 시제 선택

| 주동사 시제 | 접속법 시제 |
|------------|-----------|
| 현재 또는 미래 (주시칭) | **현재** 접속법 |
| 과거 (부시칭) | **미완료** 접속법 |

### 5.3 예문

| 라틴어 | 번역 |
|--------|------|
| *Venit **ut** videat.* | 그는 보**기 위해** 온다. / 그가 볼 수 있**도록** 온다. |
| *Vēnit **ut** vidēret.* | 그는 보**기 위해** 왔다. / 그가 볼 수 있**도록** 왔다. |
| *Fugit **nē** caperētur.* | 그는 붙잡히**지 않도록** 도망쳤다. / 붙잡힐까 **봐** 도망쳤다. |
| *Labōrāmus **ut** bene vīvāmus.* | 우리는 잘 살 수 있**도록** 일한다. |

> **번역 팁**: 목적절은 "~하기 위해", "~하도록", "~하려고"로 번역할 수 있습니다. *nē*의 부정 목적절은 "~하지 않도록", "~할까 봐"로 번역합니다.

## 6. 결과절 (ut)

결과절은 행위의 **실제 결과**를 표현하며, 흔히 주절의 "표지어"로 예고됩니다.

### 6.1 표지어 (신호어)

| 라틴어 | 의미 |
|--------|------|
| *tam* | 그렇게 (형용사/부사와 함께) |
| *tantus, -a, -um* | 그토록 큰 |
| *tot* | 그토록 많은 |
| *tālis, -e* | 그러한 종류의 |
| *ita / sīc* | 그렇게, 그런 방식으로 |
| *adeō* | 그 정도로 |

### 6.2 형성

| 유형 | 구조 |
|------|------|
| 긍정 결과 | 표지어 ... **ut** + 접속법 |
| 부정 결과 | 표지어 ... **ut nōn** + 접속법 |

> **목적절과의 핵심 차이**: 부정 목적절은 **nē**를 사용하고, 부정 결과절은 **ut nōn**을 사용합니다.

### 6.3 예문

| 라틴어 | 번역 |
|--------|------|
| *Tam fortis erat **ut** omnēs eum laudārent.* | 그는 **너무** 용감해서 모든 사람이 그를 칭찬했다. |
| *Tanta tempestās erat **ut** nāvēs nōn solverent.* | 폭풍이 **너무 거대해서** 배들이 출항하지 못했다. |
| *Ita cucurrit **ut** nēmō eum caperet.* | 그는 **아무도 그를 잡지 못할 정도로** 달렸다. |
| *Tot hostēs erant **ut nōn** resistere possēmus.* | 적이 **너무 많아서** 우리가 저항**할 수 없었다**. |

### 6.4 목적절 vs. 결과절 — 빠른 비교

| 특징 | 목적절 | 결과절 |
|------|--------|--------|
| 답하는 질문 | "왜?" (의도) | "어떤 결과로?" (실제 결과) |
| 부정 | **nē** | **ut nōn** |
| 주절의 표지어 | 없음 | 있음 (tam, tantus, ita 등) |
| 번역 | "~하기 위해 / ~하도록" | "너무 ~해서 ~하다" |

## 7. Cum절

접속사 *cum* ("~할 때/~이므로/~임에도 불구하고")은 세 가지 다른 의미로 접속법을 취합니다.

### 7.1 Cum 시간적 ("~할 때")

| 라틴어 | 번역 |
|--------|------|
| *Cum Caesar pervēnisset, proelium commīsit.* | 카이사르가 도착했을 때, 전투를 시작했다. |
| *Cum haec dīceret, omnēs tacēbant.* | 그가 이것들을 말하고 있었을 때, 모두가 침묵했다. |

### 7.2 Cum 원인적 ("~이므로/~때문에")

| 라틴어 | 번역 |
|--------|------|
| *Cum hostēs adessent, portās clausērunt.* | 적이 있었으므로, 문을 닫았다. |
| *Cum nēmō respondēret, abiit.* | 아무도 대답하지 않았으므로, 그가 떠났다. |

### 7.3 Cum 양보적 ("~임에도 불구하고")

흔히 주절의 **tamen** ("그럼에도")과 짝을 이룹니다:

| 라틴어 | 번역 |
|--------|------|
| *Cum fessī essent, **tamen** pugnāvērunt.* | 피곤했**음에도 불구하고**, **그럼에도** 싸웠다. |
| *Cum paucī essent, **tamen** fortiter resistēbant.* | 수가 적었**음에도 불구하고**, **그래도** 용감히 저항했다. |

### 7.4 어떤 Cum인지 판별하기

```
┌────────────────────────────────────────────────────────────────┐
│   CUM + 직설법   → 순수 시간적 ("~할 때/그 시점에")              │
│   CUM + 접속법   → 문맥 확인:                                   │
│                                                                │
│   1. 주절에 "tamen"이 있는가?  → 양보적 (~임에도 불구하고)        │
│   2. 이유를 설명하는가?        → 원인적 (~이므로)                │
│   3. 장면/시간을 설정하는가?    → 시간적 (~할 때)                │
└────────────────────────────────────────────────────────────────┘
```

> 참고: *Cum* + 직설법은 현재/미래 시간이나 반복적 행위에 사용됩니다(*Cum tacent, clāmant.* — 그들이 침묵할 때, 외치는 것이다). *Cum* + 접속법이 과거 서사에서의 표준입니다.

## 8. 연습 문제

### 연습 A: 접속법 형성
요청된 접속법 형태를 쓰세요.

1. amō — 3인칭 단수 현재 접속법 능동
2. dūcō — 1인칭 복수 현재 접속법 능동
3. moneō — 2인칭 단수 현재 접속법 능동
4. audiō — 3인칭 복수 현재 접속법 능동
5. sum — 3인칭 단수 현재 접속법
6. amō — 1인칭 단수 미완료 접속법 능동
7. capiō — 3인칭 복수 미완료 접속법 능동
8. sum — 3인칭 복수 미완료 접속법

### 연습 B: 독립 용법 판별
각 문장에서 접속법의 유형을 밝히세요.

1. *Gaudeāmus igitur!*
2. *Quid faciam?*
3. *Fiat lūx.*
4. *Utinam venīret!*
5. *Aliquis dīcat.*
6. *Nē cēdāmus!*

### 연습 C: 목적절인가 결과절인가?
각 절이 목적절인지 결과절인지 판별하고 번역하세요.

1. *Vēnit ut rēgem vidēret.*
2. *Tam celeriter cucurrit ut nēmō eum caperet.*
3. *Mīlitēs mīsit nē hostēs urbem caperent.*
4. *Tanta erat tempestās ut nāvēs frangerentur.*
5. *Labōrāmus ut bene vīvāmus.*

### 연습 D: Cum절 번역
번역하고 유형(시간적, 원인적, 양보적)을 밝히세요.

1. *Cum Caesar in Galliam pervēnisset, bellum gessit.*
2. *Cum paucōs mīlitēs habēret, tamen pugnāvit.*
3. *Cum nox esset, iter facere nōn potuērunt.*
4. *Cum haec audīvisset, statim profectus est.*
