# 20. 주제별 어휘 확장 (Vocābulārium Thēmaticum)

## 학습 목표
- 필수 군사 어휘를 익히고 로마 군대 조직 이해하기
- 로마 법정과 법학에서 사용되는 법률 용어 배우기
- 로마 공화정과 제정에 관련된 정치 어휘 숙달하기
- 로마 농촌 생활을 반영하는 농업 어휘 구축하기
- 로마 공학과 건축에 관한 건축 용어 인식하기
- 로마 숭배와 의식을 다루는 종교 어휘 이해하기
- 어형 규칙(접두사와 접미사)을 적용하여 체계적으로 어휘 확장하기

## 7. 어형 형성

라틴어 어형 형성을 이해하면 생소한 단어의 의미를 추론하고 어휘를 기하급수적으로 확장할 수 있다.

### 7.1 일반적인 접두사

| 접두사 | 의미 | 예 | 영어 파생어 |
|--------|------|-----|-----------|
| **ad-** | ~으로, ~을 향하여 | **ad**venīre (도착하다) | advent |
| **con-** (com-, col-, cor-) | 함께, ~와 | **con**venīre (모이다) | convention |
| **dē-** | 아래로, 떠나서 | **dē**dūcere (아래로 이끌다) | deduction |
| **ex-** (ē-) | 밖으로 | **ex**cēdere (나가다) | exceed |
| **in-** (im-) | 안으로 | **in**dūcere (안으로 이끌다) | induce |
| **in-** (im-) | 아닌 (부정) | **in**fēlīx (불행한) | infelicitous |
| **inter-** | 사이에 | **inter**pōnere (사이에 놓다) | interpose |
| **per-** | 통하여, 완전히 | **per**ficere (완성하다) | perfect |
| **prae-** | 앞에, 미리 | **prae**dīcere (예언하다) | predict |
| **prō-** | 앞으로, ~를 위하여 | **prō**cēdere (앞으로 나아가다) | proceed |
| **re-** (red-) | 다시, 뒤로 | **re**dūcere (되돌리다) | reduce |
| **sub-** (suf-, sug-, sup-) | 아래에, 아래에서 위로 | **sub**venīre (도와주러 오다) | subvene |
| **trāns-** (trā-) | 건너서 | **trāns**dūcere (건너 이끌다) | traduce |

### 7.2 일반적인 접미사 — 명사

| 접미사 | 의미 | 예 | 기본 동사/형용사 |
|--------|------|-----|----------------|
| **-tor / -trix** | 행위자 (남/여) | imperā**tor** (지휘관) | imperāre |
| | | victō**rīx** (여성 정복자) | vincere |
| **-tiō / -siō** | 행위, 과정 | ōrā**tiō** (연설) | ōrāre |
| | | mīs**siō** (파견) | mittere |
| **-tās** | 성질, 상태 | lībertās (자유) | līber |
| | | cīvitās (시민권, 국가) | cīvis |
| **-tūdō** | 성질, 상태 | magnitūdō (위대함) | magnus |
| | | fortitūdō (용기) | fortis |
| **-ium** | 장소, 추상 개념 | cōnsilium (계획, 평의회) | cōnsulere |
| | | imperium (지휘권) | imperāre |
| **-men / -mentum** | 도구, 결과 | fundāmentum (기초) | fundāre |
| | | mūnīmentum (요새) | mūnīre |
| **-ārium** | ~을 위한 장소 | grānārium (곡물 창고) | grānum |

### 7.3 일반적인 접미사 — 형용사

| 접미사 | 의미 | 예 | 기본 |
|--------|------|-----|------|
| **-ālis / -āris** | ~에 관한 | mīlitāris (군사적인) | mīles |
| **-ānus** | ~에 속하는 | Rōmānus (로마의) | Rōma |
| **-ōsus** | ~로 가득 찬 | perīculōsus (위험한) | perīculum |
| **-bilis** | ~될 수 있는 | mīrābilis (놀라운) | mīrārī |
| **-ius / -eus** | ~로 만든 | aureus (금의) | aurum |

### 7.4 단어 가족 예시: dūcō

**dūcō, dūcere, dūxī, ductum** (이끌다)에서:

| 합성어 | 의미 | 영어 |
|--------|------|------|
| **ad**dūcere | ~으로 이끌다 | adduce |
| **con**dūcere | 함께 모으다 | conduct |
| **dē**dūcere | 아래로 이끌다 | deduce |
| **in**dūcere | 안으로 이끌다 | induce |
| **prō**dūcere | 앞으로 이끌다 | produce |
| **re**dūcere | 되돌리다 | reduce |
| **trā**dūcere | 건너 이끌다 | traduce |

| 7 | columna | g | 원로원 |
| 8 | sacrificium | h | 쟁기 |
| 9 | augur | i | 수도교 |
| 10 | aquaeductus | j | 조점관 |

<details>
<summary>정답 보기</summary>

1-d, 2-a, 3-f, 4-g, 5-b, 6-h, 7-c, 8-e, 9-j, 10-i

</details>

### 연습 B: 어형 형성

접두사와 기본 동사를 사용하여 라틴어 합성 동사와 그 의미를 쓰시오:

1. **re-** + **venīre** (오다)
2. **con-** + **dūcere** (이끌다)
3. **ex-** + **pellere** (몰다)
4. **prae-** + **mittere** (보내다)
5. **in-** (아닌) + **fēlīx** (행복한)
6. **sub-** + **scrībere** (쓰다)

<details>
<summary>정답 보기</summary>

1. **revenīre** — 돌아오다, 귀환하다
2. **condūcere** — 함께 모으다, 고용하다, 이끌다
3. **expellere** — 쫓아내다, 추방하다
4. **praemittere** — 미리 보내다
5. **infēlīx** — 불행한, 불운한
6. **subscrībere** — 아래에 쓰다, 서명하다

</details>

### 연습 C: 주제별 번역

이 과에서 배운 어휘를 사용하여 다음 문장을 번역하시오:

1. Imperātor legiōnēs in aciem instrūxit.
2. Iūdex reum damnāvit.
3. Cōnsul ōrātiōnem in senātū habuit.
4. Agricola agrum arātrō colit.
5. Sacerdōs in templō sacrificium fēcit.
6. Aquaeductus aquam in urbem dūxit.

<details>
<summary>정답 보기</summary>

1. "총사령관이 군단들을 전투 대열로 배치했다."
2. "재판관이 피고에게 유죄 판결을 내렸다."
3. "집정관이 원로원에서 연설을 했다."
4. "농부가 쟁기로 밭을 경작한다."
5. "사제가 신전에서 희생 제사를 지냈다."
6. "수도교가 도시로 물을 끌어왔다."

</details>

### 연습 D: 주제 영역 식별

각 단어가 어떤 주제 영역(군사, 법률, 정치, 농업, 건축, 종교)에 속하는지 쓰시오:

1. fossa
2. testis
3. comitia
4. vīnea
5. fornix
6. augur
7. centuriō
8. sententia

<details>
<summary>정답 보기</summary>

1. **군사** — fossa (참호, 도랑)
2. **법률** — testis (증인)
3. **정치** — comitia (민회)
4. **농업** — vīnea (포도밭)
5. **건축** — fornix (아치, 궁륭)
6. **종교** — augur (조점관)
7. **군사** — centuriō (백인대장)
8. **법률** (정치도 해당) — sententia (판결, 의견)

</details>

### 연습 E: 접미사 분석

접미사, 그 의미, 그리고 기본 단어를 식별하시오:

1. ōrātor
2. lībertās
3. fortitūdō
4. perīculōsus
5. mīlitāris
6. mīrābilis

<details>
<summary>정답 보기</summary>

1. **-tor** (행위자): *ōrāre* (말하다)에서 → "연설가, 웅변가"
2. **-tās** (성질/상태): *līber* (자유로운)에서 → "자유"
3. **-tūdō** (성질/상태): *fortis* (용감한)에서 → "용기, 강인함"
4. **-ōsus** (~로 가득 찬): *perīculum* (위험)에서 → "위험으로 가득 찬, 위험한"
5. **-āris** (~에 관한): *mīles* (군인)에서 → "군인에 관한, 군사적인"
6. **-bilis** (~될 수 있는): *mīrārī* (놀라다)에서 → "놀라울 만한, 놀라운"

</details>
