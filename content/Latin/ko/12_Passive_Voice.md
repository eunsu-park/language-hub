# 12. 수동태 (Vox Passiva)

## 학습 목표

## 1. 현재 수동태 직설법

### 1.1 수동태 인칭 어미

수동태는 현재 체계에서 능동태와 다른 인칭 어미를 사용합니다:

| 인칭 | 능동태 어미 | 수동태 어미 |
|------|-----------|-----------|
| 1인칭 단수 | -ō / -m | **-or** / **-r** |
| 2인칭 단수 | -s | **-ris** (-re) |
| 3인칭 단수 | -t | **-tur** |
| 1인칭 복수 | -mus | **-mur** |
| 2인칭 복수 | -tis | **-minī** |
| 3인칭 복수 | -nt | **-ntur** |

> 참고: 2인칭 단수 어미 **-ris**에는 대체 형태 **-re**가 있습니다(예: *amāris* 또는 *amāre*). 수동태 *amāre*(너는 사랑받는다)와 부정사 *amāre*(사랑하다)를 혼동하지 않도록 주의하세요 — 문맥으로 구별합니다.

### 1.2 모든 활용에서의 현재 수동태

| 인칭 | 제1: amor | 제2: moneor | 제3: dūcor | 제3-iō: capior | 제4: audior |
|------|-----------|------------|-----------|---------------|------------|
| 1인칭 단수 | amor | moneor | dūcor | capior | audior |
| 2인칭 단수 | amāris | monēris | dūceris | caperis | audīris |
| 3인칭 단수 | amātur | monētur | dūcitur | capitur | audītur |
| 1인칭 복수 | amāmur | monēmur | dūcimur | capimur | audīmur |
| 2인칭 복수 | amāminī | monēminī | dūciminī | capiminī | audīminī |
| 3인칭 복수 | amantur | monentur | dūcuntur | capiuntur | audiuntur |

### 1.3 예문

| 라틴어 | 번역 |
|--------|------|
| *Liber ā discipulō legitur.* | 책이 학생에 의해 읽힌다. |
| *Mīlitēs ab imperātōre dūcuntur.* | 병사들이 지휘관에 의해 이끌린다. |
| *Urbs mūrīs dēfenditur.* | 도시가 성벽으로 방어된다. |

## 2. 미완료 수동태 직설법

### 2.1 형성

미완료 능동태와 동일하되, **수동 어미**를 사용합니다.

**현재 어간 + -bā- (제1/2활용) 또는 -ēbā- (제3/4활용) + 수동 어미**

| 인칭 | 제1: amor | 제2: moneor | 제3: dūcor | 제4: audior |
|------|-----------|------------|-----------|------------|
| 1인칭 단수 | amābar | monēbar | dūcēbar | audiēbar |
| 2인칭 단수 | amābāris | monēbāris | dūcēbāris | audiēbāris |
| 3인칭 단수 | amābātur | monēbātur | dūcēbātur | audiēbātur |
| 1인칭 복수 | amābāmur | monēbāmur | dūcēbāmur | audiēbāmur |
| 2인칭 복수 | amābāminī | monēbāminī | dūcēbāminī | audiēbāminī |
| 3인칭 복수 | amābantur | monēbantur | dūcēbantur | audiēbantur |

### 2.2 예문

- *Fābulae ā mātre narrābantur.* — 이야기들이 어머니에 의해 들려지고 있었다.
- *Oppidum ā Rōmānīs oppugnābātur.* — 마을이 로마인들에 의해 공격당하고 있었다.

## 3. 미래 수동태 직설법

### 3.1 형성

미래 능동태와 동일한 시제 표지에 수동 어미를 붙입니다.

| 인칭 | 제1: amor | 제2: moneor | 제3: dūcor | 제4: audior |
|------|-----------|------------|-----------|------------|
| 1인칭 단수 | amābor | monēbor | dūcar | audiar |
| 2인칭 단수 | amāberis | monēberis | dūcēris | audiēris |
| 3인칭 단수 | amābitur | monēbitur | dūcētur | audiētur |
| 1인칭 복수 | amābimur | monēbimur | dūcēmur | audiēmur |
| 2인칭 복수 | amābiminī | monēbiminī | dūcēminī | audiēminī |
| 3인칭 복수 | amābuntur | monēbuntur | dūcentur | audientur |

> **복습**: 제1/2활용 미래는 -bi-/-bō-를, 제3/4활용은 -a-/-ē-를 사용합니다.

### 3.2 예문

- *Epistula crās mittētur.* — 편지가 내일 보내질 것이다.
- *Ab omnibus laudāberis.* — 너는 모든 사람에게 칭찬받을 것이다.

## 4. 완료 수동태 체계

완료 수동태 체계는 현재 수동태 체계와 **완전히 다르게** 작동합니다. 인칭 어미 대신 **복합 구문**을 사용합니다.

### 4.1 형성

```
┌────────────────────────────────────────────────────────────────┐
│   완료 수동 = PPP + esse의 현재 시제                            │
│   과거완료 수동 = PPP + esse의 미완료 시제                       │
│   미래완료 수동 = PPP + esse의 미래 시제                        │
│                                                                │
│   PPP는 제4 기본형이며 제1/2변화 형용사로 기능합니다:            │
│   주어의 성, 수, 격에 일치합니다.                               │
└────────────────────────────────────────────────────────────────┘
```

### 4.2 완료 수동태 직설법: amō (PPP: amātus, -a, -um)

| 인칭 | 남성 단수 | 여성 단수 | 중성 단수 |
|------|----------|----------|----------|
| 1인칭 | amātus sum | amāta sum | — |
| 2인칭 | amātus es | amāta es | — |
| 3인칭 | amātus est | amāta est | amātum est |

| 인칭 | 남성 복수 | 여성 복수 | 중성 복수 |
|------|----------|----------|----------|
| 1인칭 | amātī sumus | amātae sumus | — |
| 2인칭 | amātī estis | amātae estis | — |
| 3인칭 | amātī sunt | amātae sunt | amāta sunt |

### 4.3 과거완료 수동태 직설법

*sum/es/est*를 *eram/erās/erat*로 대체합니다:

| 인칭 | 단수 (남성) | 복수 (남성) |
|------|-----------|-----------|
| 1인칭 | amātus eram | amātī erāmus |
| 2인칭 | amātus erās | amātī erātis |
| 3인칭 | amātus erat | amātī erant |

### 4.4 미래완료 수동태 직설법

*erō/eris/erit*로 대체합니다:

| 인칭 | 단수 (남성) | 복수 (남성) |
|------|-----------|-----------|
| 1인칭 | amātus erō | amātī erimus |
| 2인칭 | amātus eris | amātī eritis |
| 3인칭 | amātus erit | amātī erunt |

### 4.5 주요 PPP

| 동사 | PPP | 의미 |
|------|-----|------|
| amō | amātus | 사랑받은 |
| moneō | monitus | 경고받은 |
| dūcō | ductus | 이끌린 |
| capiō | captus | 잡힌, 포획된 |
| audiō | audītus | 들린 |
| mittō | missus | 보내진 |
| scrībō | scrīptus | 쓰인 |
| faciō | factus | 만들어진, 행해진 |
| dīcō | dictus | 말해진 |
| videō | vīsus | 보인 |
| vincō | victus | 정복된 |
| ferō | lātus | 운반된, 지탱된 |

## 5. 행위자 구문

### 5.1 행위자 vs. 도구

라틴어는 **행위자**(행위를 수행하는 사람)와 **도구/수단**(행위를 수행하는 데 사용되는 사물)을 구별합니다:

| 구문 | 용도 | 형성 | 예시 |
|------|------|------|------|
| **행위자** | 사람 | *ā/ab* + **탈격** | *ā mīlite* (병사에 의해) |
| **수단/도구** | 사물 | **탈격만** (전치사 없음) | *gladiō* (칼로/칼에 의해) |

### 5.2 예문

| 라틴어 | 번역 | 유형 |
|--------|------|------|
| *Urbs **ā Rōmānīs** capta est.* | 도시가 **로마인들에 의해** 점령되었다. | 행위자 |
| *Urbs **flammīs** dēlēta est.* | 도시가 **화염으로** 파괴되었다. | 수단 |
| *Liber **ab auctōre** scrīptus est.* | 책이 **저자에 의해** 쓰였다. | 행위자 |
| *Liber **atrāmentō** scrīptus est.* | 책이 **잉크로** 쓰였다. | 수단 |

> **규칙**: 자음 앞에서는 *ā*, 모음과 *h* 앞에서는 *ab*를 사용합니다: *ā mīlite* (병사에 의해) / *ab imperātōre* (지휘관에 의해).

## 6. 수동태의 용법

### 6.1 행위자가 알려지지 않았거나 중요하지 않을 때

- *Templum aedificātum est.* — 신전이 건설되었다. (누구에 의해인지 모르거나 중요하지 않음.)

### 6.2 행위의 대상에 초점을 맞출 때

- *Caesar ab hostibus circumventus est.* — 카이사르가 적에게 포위되었다. (카이사르에 초점.)

### 6.3 비인칭 수동

라틴어는 자동사를 비인칭적으로 수동 구문에 사용할 수 있습니다:

- *Pugnātum est ācriter.* — 격렬하게 싸워졌다. (= 격렬한 전투가 있었다.)
- *Itur ad castra.* — 진영으로 가진다. (= 사람들이 진영으로 간다.)

### 6.4 능동 ↔ 수동 전환

```
능동:   Rōmānī    urbem      cēpērunt.
        (주격)    (대격)     (능동 동사)
        로마인들이 도시를 점령했다.

수동:   Urbs    ā Rōmānīs   capta est.
        (주격)  (ā + 탈격)  (수동 동사)
        도시가 로마인들에 의해 점령되었다.
```

## 7. 탈형동사 입문

탈형동사(deponent verbs)는 **수동태 형태**를 갖지만 **능동태 의미**를 가집니다. 능동태 형태를 "벗어놓은"(*dēpōnere*) 동사입니다.

### 7.1 탈형동사 식별법

기본형이 수동태처럼 보입니다:

| 일반 동사 | 탈형동사 |
|----------|---------|
| amō, amāre, amāvī, amātum | cōnor, cōnārī, cōnātus sum |
| (4개의 능동형 기본형) | (수동형으로 보이는 기본형) |

### 7.2 주요 탈형동사

| 동사 | 활용 | 의미 |
|------|------|------|
| cōnor, cōnārī, cōnātus sum | 제1활용 | 시도하다 |
| vereor, verērī, veritus sum | 제2활용 | 두려워하다, 존경하다 |
| sequor, sequī, secūtus sum | 제3활용 | 따르다 |
| loquor, loquī, locūtus sum | 제3활용 | 말하다 |
| morior, morī, mortuus sum | 제3활용-iō | 죽다 |
| patior, patī, passus sum | 제3활용-iō | 겪다, 견디다 |
| ūtor, ūtī, ūsus sum | 제3활용 | 사용하다 (+ 탈격) |
| proficīscor, proficīscī, profectus sum | 제3활용 | 출발하다 |
| hortor, hortārī, hortātus sum | 제1활용 | 격려하다, 촉구하다 |
| mīror, mīrārī, mīrātus sum | 제1활용 | 놀라다, 감탄하다 |

### 7.3 활용 예시: sequor (나는 따른다)

| 시제 | 단수 (1/2/3인칭) | 복수 (1/2/3인칭) |
|------|-----------------|-----------------|
| 현재 | sequor, sequeris, sequitur | sequimur, sequiminī, sequuntur |
| 미완료 | sequēbar, sequēbāris, sequēbātur | sequēbāmur, sequēbāminī, sequēbantur |
| 미래 | sequar, sequēris, sequētur | sequēmur, sequēminī, sequentur |
| 완료 | secūtus sum, es, est | secūtī sumus, estis, sunt |

> **중요**: 탈형동사도 현재 능동 분사(*sequēns* = 따르는)와 미래 능동 부정사를 가집니다. 수동 형태인 것은 정동사(finite verb) 형태뿐입니다.

## 8. 연습 문제

### 연습 A: 능동을 수동으로 전환
다음 능동문을 같은 시제의 수동문으로 바꾸시오.

1. Magister discipulōs docet.
2. Mīlitēs pontem fēcērunt.
3. Rēx cīvēs monēbit.
4. Servī cēnam parāverant.
5. Imperātor cōpiās dūcēbat.

### 연습 B: 형태 식별
각 동사 형태에 대해 시제, 태, 인칭, 수를 밝히시오.

1. capitur
2. ductī erant
3. audīēminī
4. missae sunt
5. monēbāmur
6. amātus erō

### 연습 C: 한국어로 번역

1. Litterae ā poētā scrīptae sunt.
2. Multae urbēs ā Rōmānīs captae erant.
3. Hostēs ā nostrīs mīlitibus victī sunt.
4. Haec verba ab omnibus audiēntur.
5. Fābula ā mātre narrābātur.
6. Pugnātum est usque ad noctem.

### 연습 D: 탈형동사를 사용하여 번역

1. 병사들이 새벽에 출발했다. (proficīscor; "새벽에"는 *prīmā lūce* 사용)
2. 그는 지도자를 따른다. (sequor)
3. 우리는 도시를 정복하려고 시도했다. (cōnor + 부정사)
4. 그녀는 큰 지혜로 말했다. (loquor; *cum magnā sapientiā*)
5. 그들은 용감하게 죽었다. (morior; *fortiter*)

### 연습 E: 행위자 vs. 수단
빈칸을 올바른 구문으로 채우시오(*ā/ab* 사용 여부 결정):

1. Urbs ______ (병사들) dēfēnsa est.
2. Urbs ______ (성벽) dēfēnsa est.
3. Liber ______ (시인) scrīptus est.
4. Liber ______ (펜 / calamus) scrīptus est.

│   PPP (제4 기본형, 변화) + ESSE의 형태                            │
│                                                                  │
│   완료:         PPP + sum, es, est, sumus, estis, sunt            │
│   과거완료:     PPP + eram, erās, erat, erāmus, erātis, erant     │
│   미래완료:     PPP + erō, eris, erit, erimus, eritis, erunt     │
│                                                                  │
│   행위자:  ā/ab + 탈격  (사람)                                    │
│   수단:    탈격만        (사물)                                    │
│                                                                  │
│   탈형동사: 수동 형태, 능동 의미                                   │
│   cōnor, sequor, loquor, morior, patior, ūtor                    │
