## 1. 정부와 정치

### 1.1 로마 정부의 세 시대

| 시대 | 기간 | 국가 수반 | 주요 특징 |
|------|------|----------|----------|
| **Regnum** (왕정) | 기원전 753–509 | *Rex* (왕) | 전설적인 7명의 왕 |
| **Res Publica** (공화정) | 기원전 509–27 | *Consules* (2명의 집정관) | 선출된 행정관, 원로원 |
| **Imperium** (제정) | 기원전 27–서기 476 | *Princeps / Imperator* | 공화정 형식 뒤의 독재 통치 |

### 1.2 관직과 쿠르수스 호노룸

*cursus honorum* ("관직의 순서")은 로마 정치인 지망생이 거치는 공직의 순차적 경로였다:

| 관직 | 라틴어 | 최소 나이 | 인원 | 직무 |
|------|--------|----------|------|------|
| 재무관 | *quaestor* | 30 (술라 시대) | 20 | 재정 행정, 국고 관리 |
| 조영관 | *aedilis* | 36 | 4 | 공공 사업, 경기, 시장 |
| 법무관 | *praetor* | 39 | 8 | 사법, 속주 통치 |
| 집정관 | *consul* | 42 | 2 | 최고 행정관, 군사 지휘 |
| 감찰관 | *censor* | — | 2 | 인구 조사, 공중 도덕, 원로원 명부 |

**특수 관직:**

| 관직 | 라틴어 | 목적 |
|------|--------|------|
| 호민관 | *tribunus plebis* | 평민 보호, 거부권 (*intercessio*) |
| 독재관 | *dictator* | 비상시 단독 통치자 (6개월 임기) |
| 기병대장 | *magister equitum* | 독재관의 부관 |

### 1.3 원로원

*senatus* (*senex*, "노인"에서 유래)는 원로 정치가들의 자문 기관이었다:

- **구성원**: ~300명 (술라 시대 600명, 카이사르 시대 900명으로 확대)
- **권위**: *auctoritas* — 법적 권한이 아닌 도덕적 권위
- **의결**: *senatus consultum* (SC) — 기술적으로 자문이지만 실질적으로 구속력 있음
- **의장**: 집정관 (집정관 부재 시 법무관)
- **유명 구호**: *Senatus Populusque Romanus* (SPQR) — "원로원과 로마 시민"

### 1.4 주요 정치 어휘

| 라틴어 | 의미 |
|-------|------|
| *comitia* | 민회 (투표 기관) |
| *suffragium* | 투표 |
| *lex, legis* (여성) | 법 |
| *ius, iuris* (중성) | 권리, 법, 정의 |
| *imperium* | 최고 지휘권, 제국 |
| *potestas* | 권력, 권한 |
| *provincia* | 속주, 지휘 영역 |
| *res publica* | 공화국, 국가 |
| *civis* | 시민 |
| *peregrinus* | 외국인 |

## 2. 로마법

### 2.1 12표법 (Leges Duodecim Tabularum)

기원전 450년경에 작성된 로마법의 최초 성문화이다. 포럼에 청동판으로 게시되었으며 다음을 다루었다:

| 표 | 주제 |
|-----|------|
| I–III | 재판 절차와 채무 |
| IV–V | 가족법 (*patria potestas*)과 상속 |
| VI–VII | 재산, 소유권, 경계 |
| VIII–IX | 불법 행위(불법행위법), 범죄, 헌법 |
| X | 신성법 (장례) |
| XI–XII | 보충 법률 |

### 2.2 로마법의 분류

| 분류 | 라틴어 | 설명 |
|------|--------|------|
| 시민법 | *ius civile* | 로마 시민에게 적용되는 법 |
| 만민법 | *ius gentium* | 모든 민족에게 공통된 법 |
| 자연법 | *ius naturale* | 보편적 정의의 원칙 |
| 명예법 | *ius honorarium* | 시민법을 보완하는 법무관 고시 |

### 2.3 현재까지 사용되는 법적 원칙

| 라틴어 격언 | 의미 | 현대적 적용 |
|------------|------|-----------|
| *Ignorantia legis neminem excusat* | 법의 무지는 누구도 면책하지 않는다 | 보편적 법적 원칙 |
| *Nulla poena sine lege* | 법 없이는 처벌 없다 | 형법의 토대 |
| *In dubio pro reo* | 의심스러울 때는 피고에게 유리하게 | 무죄 추정 |
| *Pacta sunt servanda* | 약속은 지켜져야 한다 | 계약법 |
| *Res iudicata* | 판결된 사안 | 판례법 / 이중 처벌 금지 |
| *Dura lex, sed lex* | 법은 가혹하나, 법이다 | 법치주의 |

### 2.4 유산

로마법은 유럽 대륙, 라틴 아메리카, 아시아와 아프리카 대부분에서 사용되는 **대륙법 체계**의 토대이다. 유스티니아누스의 *Corpus Iuris Civilis* (서기 529–534)는 로마법을 집대성하여 중세 및 근대 유럽 법률 체계의 기초가 되었다.

## 3. 로마 종교

### 3.1 판테온

로마 신들은 대부분 그리스 신들에서 차용되었다:

| 로마 신 | 그리스 대응 신 | 영역 | 주요 호칭 |
|---------|--------------|------|----------|
| Iuppiter (*Iuppiter*) | 제우스 | 하늘, 천둥, 신들의 왕 | *Optimus Maximus* |
| Iuno (*Iuno*) | 헤라 | 결혼, 여성 | *Regina* |
| Minerva (*Minerva*) | 아테나 | 지혜, 기술, 전쟁 | — |
| Mars (*Mars*) | 아레스 | 전쟁, 농업 | *Pater* |
| Venus (*Venus*) | 아프로디테 | 사랑, 아름다움 | *Genetrix* |
| Apollo (*Apollo*) | 아폴론 | 음악, 예언, 태양 | *Phoebus* |
| Diana (*Diana*) | 아르테미스 | 사냥, 달 | — |
| Neptunus (*Neptunus*) | 포세이돈 | 바다 | — |
| Mercurius (*Mercurius*) | 헤르메스 | 상업, 전령 | — |
| Vulcanus (*Vulcanus*) | 헤파이스토스 | 불, 대장간 | — |
| Ceres (*Ceres*) | 데메테르 | 곡물, 수확 | — |
| Bacchus (*Bacchus*) | 디오니소스 | 포도주, 축연 | *Liber* |

### 3.2 점술과 예언

| 관행 | 라틴어 | 설명 |
|------|--------|------|
| 조점 | *augurium* | 새의 비행과 행동을 해석 |
| 내장점 | *haruspicina* | 동물 내장(특히 간) 조사 |
| 점조 | *auspicium* | 행동 전 징조 관찰 (*avis* + *specere*) |
| 시빌라 서적 | *Libri Sibyllini* | 비상시 참고하는 예언 문서 |

***auspicium melioris aevi*** ("더 나은 시대의 징조")라는 구절이 이 전통을 반영한다.

### 3.3 축제

| 축제 | 날짜 | 기리는 대상 | 활동 |
|------|------|-----------|------|
| 사투르날리아 | 12월 17–23 | 사투르누스 | 선물 교환, 역할 역전, 연회 |
| 루페르칼리아 | 2월 15 | 파우누스 | 정화, 풍요 의식 |
| 플로랄리아 | 4월 28–5월 3 | 플로라 | 봄꽃, 연극 공연 |
| 파렌탈리아 | 2월 13–21 | 조상 | 무덤 방문, 죽은 자에 대한 봉납 |
| 루디 로마니 | 9월 4–19 | 유피테르 | 전차 경주, 연극 공연 |

### 3.4 가정 종교

모든 로마 가정에는 다음이 있었다:

- **라레스** (*Lares*) — 가정과 교차로의 수호신
- **페나테스** (*Penates*) — 식료품 저장실과 가족 양식의 신
- **게니우스** (*Genius*) — *paterfamilias*의 수호 정령
- **라라리움** — 가정 신당, 보통 화덕 근처에 위치

## 5. 건축과 공학

### 5.1 주요 혁신

| 혁신 | 라틴어 | 중요성 |
|------|--------|--------|
| **콘크리트** | *opus caementicium* | 건축에 혁명; 돔과 궁륭 구현 가능 |
| **아치** | *arcus* | 하중 지지 곡선 구조물 |
| **궁륭** | *fornix* | 아치를 3차원으로 확장 |
| **돔** | *tholus* | 1,300년간 최대 비지지 돔 (판테온) |
| **도로** | *viae* | 40만 km 네트워크; 많은 도로가 현재까지 사용 |
| **수도교** | *aquaeductus* | 계곡을 가로지르는 중력식 급수 시스템 |

### 5.2 유명한 건축물

| 건축물 | 날짜 | 특징 |
|--------|------|------|
| 콜로세움 (*Amphitheatrum Flavium*) | 서기 72–80 | 5만 석, 접이식 차양 (*velarium*) |
| 판테온 | 서기 118–125 | 43m 비보강 콘크리트 돔, 오쿨루스 |
| 포룸 로마눔 | 기원전 7세기 이후 | 정치, 종교, 상업 중심지 |
| 아피아 가도 (*Via Appia*) | 기원전 312 | 최초의 주요 로마 도로, 로마에서 브린디시까지 |
| 퐁 뒤 가르 | 기원전 19년경 | 남부 프랑스의 3단 수도교 |

### 5.3 도로 건설

로마 도로(*viae*)는 여러 층으로 건설되었다:

```
표면:     summa crusta (포장석)
깔개:     nucleus (자갈과 석회 시멘트)
기초:     rudus (잔석)
바닥:     statumen (큰 돌)
지면:     fossa (양쪽의 배수 도랑)
```

***Omnes viae Romam ducunt*** ("모든 길은 로마로 통한다")라는 구절은 도로 네트워크의 중심성을 반영한다.

## 6. 사회 구조

### 6.1 사회 계급

| 계급 | 라틴어 | 설명 |
|------|--------|------|
| **귀족** | *patricii* | 세습 귀족, 원래의 지배 계급 |
| **평민** | *plebeii* | 일반 시민, 인구의 대다수 |
| **기사** | *equites* | 비원로원 부유 계급, 사업과 금융 |
| **해방노예** | *liberti / libertini* | 전직 노예, 제한된 정치적 권리 |
| **노예** | *servi* | 법적 권리 없음, 소유자의 재산 |
| **외국인** | *peregrini* | 비시민, 피정복민 |

### 6.2 가족 (Familia)

로마의 *familia*는 *paterfamilias* (가장 연장인 남성)가 이끌었으며, 그는 *patria potestas* (가부장적 권한)를 행사했다:

- 아내 (*manus* 혼인의 경우 — 공화정 말기에 점점 드물어짐)
- 자녀 (아버지가 사망하거나 해방할 때까지 성인 아들 포함)
- 가정의 노예와 해방노예
- 모든 가족 재산

### 6.3 로마의 여성

| 측면 | 지위 |
|------|------|
| 시민권 | 시민이지만 투표권이나 공직 없음 |
| 재산 | 재산 소유 가능 (특히 공화정 말기/제정기) |
| 결혼 | 아버지의 동의 필요; 이혼 주도 가능 (공화정 말기) |
| 교육 | 상류층 여성은 종종 높은 교육 수준 |
| 유명 여성 | 코르넬리아 (그라쿠스 형제의 어머니), 리비아 (아우구스투스의 아내), 아그리피나 |

### 6.4 노예제

- 노예는 전쟁, 해적, 채무, 출생에서 유래
- 가내 노예(*servi domestici*)는 좋은 대우를 받을 수 있었음; 광산 노예는 가혹한 조건
- 해방(*manumissio*)이 흔했음 — 해방된 노예는 *liberti*가 됨
- 해방노예의 자녀는 완전한 시민
- 주요 노예 반란: 스파르타쿠스 (기원전 73–71)

## 7. 일상생활

### 7.1 주거

| 유형 | 라틴어 | 설명 |
|------|--------|------|
| 도시 주택 | *domus* | 단독 가구, 아트리움 중심, 부유층용 |
| 아파트 건물 | *insula* | 다층 임대 주택, 3–8층 |
| 시골 별장 | *villa rustica* | 농업 생산이 있는 농장 영지 |
| 호화 별장 | *villa urbana* | 부유층의 시골 휴양지 |

***domus*의 구조:**

```
거리 입구 (ostium)
    ↓
아트리움 (접견실, 임플루비움 연못)
    ↓
타블리눔 (가장의 사무실)
    ↓
페리스틸리움 (열주 정원)
    → 트리클리니움 (식당)
    → 쿠비쿨룸 (침실)
    → 쿨리나 (주방)
```

### 7.2 목욕 (Thermae)

로마 목욕탕은 핵심적인 사회 기관이었다:

| 방 | 라틴어 | 온도 |
|-----|--------|------|
| 탈의실 | *apodyterium* | — |
| 냉탕 | *frigidarium* | 차가움 |
| 온탕 | *tepidarium* | 따뜻함 |
| 열탕 | *caldarium* | 뜨거움 |
| 운동장 | *palaestra* | — |

***hypocaustum***에 의해 가열 — 화덕에서 가열된 공기가 올려진 바닥 아래 통로를 순환하는 바닥 난방 시스템.

### 7.3 교육

| 단계 | 라틴어 | 나이 | 내용 |
|------|--------|------|------|
| 초등 | *ludus litterarius* | 7–11 | 읽기, 쓰기, 산수 |
| 중등 | *grammaticus* | 12–15 | 문학, 문법, 그리스어 |
| 고등 | *rhetor* | 16+ | 수사학, 웅변술, 철학 |

**필기 도구:**
- *stilus* — 밀랍 서판용 뾰족한 필기구
- *tabula cerata* — 밀랍을 입힌 나무 서판
- *calamus* — 파피루스용 갈대 펜
- *volumen* — 파피루스 두루마리

### 7.4 오락

| 유형 | 라틴어 | 설명 |
|------|--------|------|
| 검투사 경기 | *munera gladiatoria* | 원형 경기장에서의 무장 전투 |
| 전차 경주 | *ludi circenses* | 키르쿠스 막시무스에서 4개 파벌 (청, 녹, 적, 백) |
| 연극 | *ludi scaenici* | 희극 (플라우투스, 테렌티우스)과 비극 |
| 만찬 | *cena* | 여러 코스의 식사, 침상에 기대어 |

## 8. 로마의 유산

### 8.1 로마가 세계에 남긴 것

| 분야 | 유산 |
|------|------|
| **언어** | 라틴어 → 로만스어; 영어 어휘의 60% |
| **법** | 대륙법 체계, 법적 개념, 법률 용어 |
| **정부** | 공화정, 원로원, 권력 분립, *veto* |
| **공학** | 도로, 수도교, 콘크리트, 아치, 돔 |
| **역법** | 율리우스력 → 그레고리력; 달 이름 |
| **문자** | 전 세계에서 사용되는 라틴 문자 |
| **기독교** | 로마가 기독교를 수용하고 전파; 바티칸 라틴어 |
| **문학** | 문학 장르, 수사학, 역사 서술 |

### 8.2 일상에서 사용되는 라틴어 구절

| 구절 | 의미 | 맥락 |
|------|------|------|
| *et cetera* (etc.) | "그리고 나머지" | 일반 |
| *vice versa* | "위치가 바뀌어" | 일반 |
| *ad hoc* | "이것을 위하여" | 비즈니스, 정치 |
| *per capita* | "머리(인원)당" | 경제 |
| *status quo* | "현 상태" | 정치 |
| *bona fide* | "선의로" | 법 |
| *alma mater* | "양육하는 어머니" | 교육 |
| *carpe diem* | "오늘을 잡아라" | 대중 문화 |
| *veni, vidi, vici* | "왔노라, 보았노라, 이겼노라" | 대중 문화 |
| *anno Domini* (AD) | "주의 해에" | 역법 |

## 9. 연습 문제

### 연습 1: 정치 어휘

라틴어 용어를 정의와 짝짓으시오:

| | 라틴어 | | 정의 |
|---|-------|---|------|
| 1 | *cursus honorum* | a | 원로 정치가들의 자문 기관 |
| 2 | *senatus* | b | 최고 행정 권한 |
| 3 | *imperium* | c | 공직의 순서 |
| 4 | *tribunus plebis* | d | 비상시 단독 통치자 |
| 5 | *dictator* | e | 평민 권리의 보호자 |

<details>
<summary>정답 보기</summary>

1-c, 2-a, 3-b, 4-e, 5-d
</details>

### 연습 2: 로마법

각 상황에 어떤 법적 격언이 적용되는가?

**a)** 어떤 사람이 범죄로 기소되었으나 증거가 불확실하다.

**b)** 두 국가가 조약을 체결했는데 한 국가가 파기하려 한다.

**c)** 한 시민이 특정 법의 존재를 몰랐다고 주장한다.

**d)** 정부가 행위 당시 합법이었던 행위에 대해 처벌하려 한다.

<details>
<summary>정답 보기</summary>

- **a)** *In dubio pro reo* — "의심스러울 때는 피고에게 유리하게"
- **b)** *Pacta sunt servanda* — "약속은 지켜져야 한다"
- **c)** *Ignorantia legis neminem excusat* — "법의 무지는 누구도 면책하지 않는다"
- **d)** *Nulla poena sine lege* — "법 없이는 처벌 없다" (즉, 소급 처벌 금지)
</details>

### 연습 3: 군사 조직

다음 군사 부대를 가장 작은 것부터 큰 순서로 정렬하시오:

*cohors, contubernium, centuria, legio*

<details>
<summary>정답 보기</summary>

1. *contubernium* (~8명)
2. *centuria* (~80명)
3. *cohors* (~480명)
4. *legio* (~5,000명)
</details>
