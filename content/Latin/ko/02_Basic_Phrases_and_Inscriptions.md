## 1. 일반 인사와 작별

### 1.1 인사

| 라틴어 | 발음 (고전) | 의미 | 용법 |
|--------|-----------|------|------|
| Salvē! | /ˈsal.weː/ | 안녕하세요! (한 사람에게) | 일반적 인사 |
| Salvēte! | /sal.ˈweː.tɛ/ | 안녕하세요! (여러 사람에게) | 복수 인사 |
| Avē! | /ˈa.weː/ | 만나뵙겠습니다! (한 사람에게) | 더 격식 있는 인사 |
| Avēte! | /a.ˈweː.tɛ/ | 만나뵙겠습니다! (여러 사람에게) | 격식 있는 복수 |
| Salūtem dīcō | /sa.ˈluː.tɛm ˈdiː.koː/ | 인사를 드립니다 | 격식, 서신에서 |
| Quid agis? | /kwɪd ˈa.gɪs/ | 어떻게 지내세요? (단수) | 직역 "무엇을 하고 있습니까?" |
| Quid agitis? | /kwɪd a.ˈgɪ.tɪs/ | 어떻게 지내세요? (복수) | 복수형 |

### 1.2 응답

| 라틴어 | 발음 | 의미 |
|--------|------|------|
| Bene! | /ˈbɛ.nɛ/ | 잘 지냅니다! |
| Optimē! | /ˈɔp.tɪ.meː/ | 아주 잘 지냅니다! |
| Nōn male. | /noːn ˈma.lɛ/ | 나쁘지 않습니다. |
| Bene mē habeō. | /ˈbɛ.nɛ meː ˈha.bɛ.oː/ | 잘 지내고 있습니다. (직역 "나를 잘 유지하고 있습니다") |
| Et tū? | /ɛt tuː/ | 당신은요? |

### 1.3 작별

| 라틴어 | 발음 | 의미 | 용법 |
|--------|------|------|------|
| Valē! | /ˈwa.leː/ | 안녕히 가세요! (한 사람에게) | 일반적 작별 |
| Valēte! | /wa.ˈleː.tɛ/ | 안녕히 가세요! (여러 사람에게) | 복수 작별 |
| Cūrā ut valeās | /ˈkuː.ra ʊt wa.ˈlɛ.aːs/ | 건강하세요! | 직역 "건강하도록 신경 쓰세요" |
| In pace | /ɪn ˈpa.kɛ/ | 평안히 | 격식 있는 작별 |
| Bonā nocte | /ˈbɔ.naː ˈnɔk.tɛ/ | 좋은 밤 되세요 | 저녁 작별 |

> **참고**: *Salvē*는 직역하면 "건강하세요!"입니다(*salveō*의 명령형). 마찬가지로 *valē*는 "강건하세요/건강하세요!"입니다(*valeō*의 명령형). 라틴어 인사는 본질적으로 상대방의 안녕을 비는 말입니다.

### 1.4 자기소개

```
이름 묻기:
  Quid nōmen tibi est?    /kwɪd ˈnoː.mɛn ˈtɪ.bɪ ɛst/
  "이름이 무엇입니까?"     (직역: "당신에게 무슨 이름이 있습니까?")

대답하기:
  Mihi nōmen est Marcus.  /ˈmɪ.hɪ ˈnoː.mɛn ɛst ˈmar.kʊs/
  "제 이름은 마르쿠스입니다."  (직역: "나에게 이름은 마르쿠스입니다.")

  Nōmen mihi est Iūlia.   /ˈnoː.mɛn ˈmɪ.hɪ ɛst ˈjuː.lɪ.a/
  "제 이름은 율리아입니다."

만남:
  Tē salūtō.              /teː sa.ˈluː.toː/
  "인사드립니다."
```

| Grātiās vōbīs agō | /ˈgraː.tɪ.aːs ˈwoː.biːs ˈa.goː/ | 감사합니다 (여러 사람에게) |
| Grātiās maximās agō | /ˈgraː.tɪ.aːs ˈmak.sɪ.maːs ˈa.goː/ | 대단히 감사합니다 |
| Quaesō | /ˈkwae̯.soː/ | 부디 (간청합니다) |
| Sī tibi placet | /siː ˈtɪ.bɪ ˈpla.kɛt/ | 괜찮으시다면 (부디) |

### 2.2 유용한 질문

| 라틴어 | 발음 | 의미 |
|--------|------|------|
| Quid est? | /kwɪd ɛst/ | 무엇입니까? |
| Quis est? | /kwɪs ɛst/ | 누구입니까? |
| Ubi est...? | /ˈʊ.bɪ ɛst/ | ...은(는) 어디에 있습니까? |
| Quōmodo? | /ˈkwoː.mɔ.doː/ | 어떻게? |
| Cūr? | /kuːr/ | 왜? |
| Quantum? | /ˈkwan.tʊm/ | 얼마나? |
| Intellegisne? | /ɪn.tɛl.ˈlɛ.gɪs.nɛ/ | 이해하십니까? |

> **접미 의문 조사 -ne**: 라틴어에서 예/아니오 의문문을 만들려면, 문장 첫 단어에 **-ne**를 붙입니다. *Intellegisne?* = "이해하십니까?" 한국어 의문 조사 -까, -니와 비교해 보세요.

## 4. 로마 비문 읽기

### 4.1 SPQR과 정치 약어

```
┌─────────────────────────────────────────────────────────────────┐
│                        S · P · Q · R                             │
│                                                                 │
│   S(enātus) P(opulus)Q(ue) R(ōmānus)                           │
│   /sɛ.ˈnaː.tʊs pɔ.pʊ.ˈlʊs.kwɛ roː.ˈmaː.nʊs/              │
│                                                                 │
│   "로마의 원로원과 시민"                                         │
│                                                                 │
│   이것은 로마 공화정과 제정의 공식 표어였습니다.                   │
│   군기, 건물, 동전, 공식 문서에 새겨졌습니다.                     │
│   현재에도 로마 시의 상징으로 남아 있으며 — 현대 도시 전역의      │
│   맨홀 뚜껑과 공공 건물에서 볼 수 있습니다.                       │
│                                                                 │
│   -que = "그리고" (접미 접속사, 두 번째 단어에 붙음)              │
│   비교: et = "그리고" (독립 접속사)                               │
│   Senātus et Populus = Senātus Populusque                       │
└─────────────────────────────────────────────────────────────────┘
```

## 5. 현대 생활 속 라틴어

### 5.1 법률 라틴어

| 용어 | 의미 | 용법 |
|------|------|------|
| habeas corpus | "신체를 제출하라" | 법원 출석권 |
| pro bono | "공익을 위해" | 무료 법률 업무 |
| subpoena | "형벌 아래" | 법원 출석 명령 |
| amicus curiae | "법원의 벗" | 제3자 법률 의견서 |
| caveat emptor | "구매자가 주의하라" | 구매자 위험 부담 |
| de facto / de iure | "사실에 의한 / 법에 의한" | 실질적 / 법적 |
| nōlle prōsequī | "기소를 원치 않음" | 기소 취하 |
| prima facie | "첫 얼굴에" | 표면상, 일견하여 |
| quid pro quo | "이것에 대한 저것" | 교환, 호혜 |
| res ipsa loquitur | "사물 자체가 말한다" | 자명한 상황 |

### 5.2 의학 라틴어

| 용어 | 의미 | 용법 |
|------|------|------|
| Rx (recipe) | "가져가라" | 처방전 표제 |
| in vitro | "유리 안에서" | 실험실 실험 (시험관) |
| in vivo | "생체 내에서" | 생체 실험 |
| rigor mortis | "죽음의 경직" | 사후 근육 경직 |
| post mortem | "죽음 이후" | 부검 |
| ante meridiem (a.m.) | "정오 전" | 오전 시간 |
| post meridiem (p.m.) | "정오 후" | 오후/저녁 시간 |
| stat (statim) | "즉시" | 긴급 의료 지시 |

### 5.3 학술 라틴어

| 용어 | 의미 | 용법 |
|------|------|------|
| alma māter | "기르는 어머니" | 출신 대학 |
| alumnus/alumna | "양육받은 자, 학생" | 졸업생 (남/여) |
| cum laude | "칭찬과 함께" | 학업 우등 |
| magna cum laude | "큰 칭찬과 함께" | 더 높은 학업 우등 |
| summa cum laude | "최고의 칭찬과 함께" | 최고 학업 우등 |
| Ph.D. (Philosophiae Doctor) | "철학 박사" | 고급 학위 |
| curriculum vītae (CV) | "생애의 과정" | 이력서 |
| ibid. (ibidem) | "같은 곳에서" | 동일 출처 인용 |
| e.g. (exemplī grātiā) | "예를 위하여" | "예를 들어" |
| i.e. (id est) | "즉" | "다시 말해" |
| N.B. (notā bene) | "잘 기록하라" | "참고" |
| etc. (et cētera) | "그리고 나머지" | "등등" |
| sic | "이와 같이" | 원문 그대로임을 표시 |
| versus (vs.) | "~쪽으로 향한" | ~대(對) |

### 5.4 과학 속 라틴어

```
생물학 명명법 (이명법):
  Homo sapiens        = "지혜로운 인간"
  Tyrannosaurus rex   = "폭군 도마뱀 왕"
  Canis lupus         = "개 늑대" (회색늑대)

화학/천문 용어:
  Sol     = 태양        Luna    = 달
```

## 6. 간단한 문장 패턴

### 6.1 기본 어순: 주어-목적어-동사 (SOV)

```
┌─────────────────────────────────────────────────────────────────┐
│             라틴어 기본 어순: SOV                                   │
│                                                                 │
│   라틴어의 기본 어순은 주어-목적어-동사(SOV)로,                   │
│   한국어, 일본어와 같습니다!                                      │
│                                                                 │
│   한국어:   마르쿠스가   율리아를    사랑한다.                      │
│   라틴어:   Mārcus      Iūliam     amat.                         │
│   영어:     Marcus      loves      Julia.                        │
│             (주)         (목)       (동)                          │
│                                                                 │
│   그러나 라틴어는 격어미를 사용하여 (어순이 아닌)                  │
│   문법적 기능을 나타내므로 어순이 자유롭습니다.                    │
│   다음 문장들은 모두 같은 의미입니다:                              │
│     Mārcus Iūliam amat.    (SOV — 기본)                          │
│     Mārcus amat Iūliam.    (SVO)                                 │
│     Iūliam Mārcus amat.    (OSV)                                 │
│     Amat Mārcus Iūliam.    (VSO)                                 │
│                                                                 │
│   격어미(-us = 주어, -am = 목적어)가 단어 위치에 관계없이          │
│   누가 무엇을 하는지 알려줍니다.                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 패턴 1: 주어 + 동사

가장 간단한 라틴어 문장은 주어와 동사만으로 이루어집니다. 주어 대명사는 종종 생략됩니다(한국어, 스페인어와 같이).

| 라틴어 | 직역 | 번역 |
|--------|------|------|
| Puella cantat. | 소녀가 노래한다. | 소녀가 노래한다. |
| Puer ambulat. | 소년이 걷는다. | 소년이 걷는다. |
| Nauta labōrat. | 선원이 일한다. | 선원이 일한다. |
| Fēmina vocat. | 여인이 부른다. | 여인이 부른다. |
| Agricola arat. | 농부가 밭을 간다. | 농부가 밭을 간다. |

> **관사에 관한 참고**: 라틴어에는 관사가 없습니다("the"나 "a" 없음). *Puella*는 문맥에 따라 "그 소녀," "한 소녀," 또는 단순히 "소녀"를 의미할 수 있습니다 — 한국어에 관사가 없는 것과 유사합니다.

### 6.3 패턴 2: 주어 + 동사 "~이다" + 보어

| 라틴어 | 직역 | 번역 |
|--------|------|------|
| Rōma est magna. | 로마는 위대하다. | 로마는 위대하다. |
| Puella est laeta. | 소녀는 행복하다. | 소녀는 행복하다. |
| Poēta est bonus. | 시인은 훌륭하다. | 시인은 훌륭하다. |
| Aqua est frīgida. | 물은 차갑다. | 물은 차갑다. |
| Vir est Rōmānus. | 남자는 로마인이다. | 남자는 로마인이다. |

### 6.4 패턴 3: 주어 + 목적어 + 동사

| 라틴어 | 직역 | 번역 |
|--------|------|------|
| Puella rosam amat. | 소녀가 장미를 사랑한다. | 소녀가 장미를 사랑한다. |
| Nauta terram videt. | 선원이 땅을 본다. | 선원이 땅을 본다. |
| Agricola aquam portat. | 농부가 물을 나른다. | 농부가 물을 나른다. |
| Poēta fābulam nārrat. | 시인이 이야기를 들려준다. | 시인이 이야기를 들려준다. |
| Fēmina viam mōnstrat. | 여인이 길을 보여준다. | 여인이 길을 보여준다. |

## 7. 연습 문제

### 연습 1: 인사와 응답

라틴어 인사와 적절한 응답을 짝지으세요.

```
인사:                          응답:
1. Salvē!                    a. Bene mē habeō.
2. Quid agis?                b. Salvē!
3. Valē!                     c. Mihi nōmen est Lūcia.
4. Grātiās tibi agō.        d. Valē!
5. Quid nōmen tibi est?     e. Nihil est.
```

<details>
<summary>정답 확인</summary>

```
1. Salvē! → b. Salvē! (안녕하세요! → 안녕하세요!)
2. Quid agis? → a. Bene mē habeō. (어떻게 지내세요? → 잘 지내고 있습니다.)
3. Valē! → d. Valē! (안녕히 가세요! → 안녕히 가세요!)
4. Grātiās tibi agō. → e. Nihil est. (감사합니다. → 별 것 아닙니다.)
5. Quid nōmen tibi est? → c. Mihi nōmen est Lūcia.
   (이름이 무엇입니까? → 제 이름은 루키아입니다.)
```

</details>

### 연습 2: 유명 구문 — 빈칸 채우기

다음 유명한 라틴어 구문을 기억에서 완성하세요.

```
1. Carpe ______.          (이 날을 잡아라.)
2. Vēnī, vīdī, ______.  (왔노라, 보았노라, 이겼노라.)
3. Cogitō ergō ______.   (나는 생각한다, 고로 존재한다.)
4. E plūribus ______.    (다수에서 하나로.)
5. Tempus ______.         (시간은 흐른다.)
6. Alea iacta ______.    (주사위는 던져졌다.)
7. ______ morī.           (죽음을 기억하라.)
8. In vīnō ______.       (술에 진실이 있다.)
```

<details>
<summary>정답 확인</summary>

```
1. Carpe diem.
2. Vēnī, vīdī, vīcī.
3. Cogitō ergō sum.
4. E plūribus ūnum.
5. Tempus fugit.
6. Alea iacta est.
7. Memento morī.
8. In vīnō vēritās.
```

</details>

### 연습 3: 영어에서 사용되는 라틴어 약어

다음 일반적인 약어의 원형과 의미는 무엇입니까?

```
1. e.g.    → _____________ → 의미: _____________
2. i.e.    → _____________ → 의미: _____________
3. etc.    → _____________ → 의미: _____________
4. N.B.    → _____________ → 의미: _____________
5. a.m.    → _____________ → 의미: _____________
6. p.m.    → _____________ → 의미: _____________
7. vs.     → _____________ → 의미: _____________
8. CV      → _____________ → 의미: _____________
```

<details>
<summary>정답 확인</summary>

```
1. e.g.  → exemplī grātiā       → "예를 들어"
2. i.e.  → id est                → "즉" / "다시 말해"
3. etc.  → et cētera             → "그리고 나머지" / "등등"
4. N.B.  → notā bene             → "잘 기록하라" / "참고"
5. a.m.  → ante meridiem         → "정오 전"
6. p.m.  → post meridiem         → "정오 후"
7. vs.   → versus                → "~대(對)" / "~쪽으로 향한"
8. CV    → curriculum vītae      → "생애의 과정" (이력서)
```

</details>

</details>

### 연습 6: 간단한 라틴어 문장

다음 문장들을 한국어로 번역하세요. 어순(SOV)에 주의하세요.

```
1. Puella cantat.
2. Nauta labōrat.
3. Rōma est magna.
4. Agricola aquam portat.
5. Poēta fābulam nārrat.
6. Fēmina est laeta.
7. Puer rosam videt.
8. Mārcus Iūliam amat.
```

<details>
<summary>정답 확인</summary>

```
1. Puella cantat.            → 소녀가 노래한다.
2. Nauta labōrat.            → 선원이 일한다.
3. Rōma est magna.           → 로마는 위대하다.
4. Agricola aquam portat.    → 농부가 물을 나른다.
5. Poēta fābulam nārrat.    → 시인이 이야기를 들려준다.
6. Fēmina est laeta.        → 여인은 행복하다.
7. Puer rosam videt.         → 소년이 장미를 본다.
8. Mārcus Iūliam amat.      → 마르쿠스는 율리아를 사랑한다.
```

</details>

### 연습 7: 발음 연습

각 구문을 고전 발음으로 소리 내어 읽으세요. 특히 C = /k/, V = /w/, AE = /ae̯/에 주의하세요.

```
1. Salvē, amīce!            (안녕, 친구여!)
2. Valē et cūrā ut valeās.  (안녕히 가고 건강하게.)
3. Vēnī, vīdī, vīcī.       (왔노라, 보았노라, 이겼노라.)
4. Caesar Galliam vīcit.     (카이사르가 갈리아를 정복했다.)
5. Caelum est pulchrum.      (하늘은 아름답다.)
```

<details>
<summary>발음 확인</summary>

```
1. /ˈsal.weː a.ˈmiː.kɛ/
   (기억: V = /w/, C = /k/)

2. /ˈwa.leː ɛt ˈkuː.raː ʊt wa.ˈlɛ.aːs/
   (Valē의 V = /w/, valeās의 V = /w/)

3. /ˈweː.niː ˈwiː.diː ˈwiː.kiː/
   (세 개의 V 모두 = /w/, 어미의 ī는 모두 장모음)

4. /ˈkae̯.sar ˈgal.lɪ.am ˈwiː.kɪt/
   (Caesar의 C = /k/, AE = /ae̯/ "아이", vīcit의 V = /w/)

5. /ˈkae̯.lʊm ɛst ˈpʊl.kʰrʊm/
   (caelum의 C = /k/, AE = /ae̯/, pulchrum의 CH = /kʰ/)
```

</details>
