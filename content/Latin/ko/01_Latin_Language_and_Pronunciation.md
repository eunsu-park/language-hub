## 1. 라틴어 입문

### 1.1 역사 속의 라틴어

> **라틴어: 영원한 언어**
>
> - **사용 기간**: 기원전 약 700년 – 현재 (전문 분야에서 사용)
> - **모국어 화자**: 현존하지 않음 (고전어)
> - **공식 지위**: 바티칸 시국 (공동 공용어)
>
> - **어족**: 인도유럽어족 → 이탈리아어파 → 라틴-팔리스키어
> - **후손 언어**: 이탈리아어, 스페인어, 포르투갈어, 프랑스어, 루마니아어, 카탈루냐어 등
>
> - **문화적 영향**: 법률, 의학, 과학, 신학, 철학, 문학, 학문 전통
>
> **역사적 시기:**
> - 고라틴어 (기원전 700–100년)
> - 고전 라틴어 (기원전 100년–기원후 200년) ← 우리의 초점
> - 후기 라틴어 (200–700년)
> - 중세 라틴어 (700–1400년)
> - 르네상스 라틴어 (1400–1600년)
> - 신라틴어 (1600년–현재)

### 1.2 왜 라틴어를 공부하는가?

라틴어는 더 이상 모국어로 사용되지 않지만, 여전히 놀라울 만큼 큰 영향력을 지닙니다:

- **로망스어의 기초**: 스페인어, 프랑스어, 이탈리아어, 포르투갈어, 루마니아어가 라틴어에서 직접 파생되었습니다. 라틴어를 알면 이 모든 언어의 어휘를 이해할 수 있습니다.
- **영어 어휘**: 영어 단어의 60% 이상이 라틴어에서 유래합니다(직접 또는 프랑스어를 통해). 과학 및 법률 영어에서는 그 비율이 90%를 넘습니다.
- **과학 명명법**: 생물학(이명법), 의학, 화학, 천문학 모두 라틴어 용어를 사용합니다.
- **법률 전통**: *habeas corpus*, *pro bono*, *stare decisis* 같은 법률 격언이 전 세계적으로 현재도 사용됩니다.
- **문학 유산**: 베르길리우스, 키케로, 카이사르, 오비디우스, 호라티우스, 세네카를 원문으로 읽을 수 있습니다.
- **논리적 훈련**: 라틴어의 고도의 굴절 문법은 언어 작동 방식에 대한 분석적 사고를 발달시킵니다.

> **한국어 화자를 위한 참고**: 한국어와 라틴어는 놀랍게도 하나의 구조적 유사점을 공유합니다 — 둘 다 기본적으로 SOV(주어-목적어-동사) 어순입니다. 라틴어의 격 체계는 한국어 조사와도 유사합니다: 주격은 이/가, 대격은 을/를, 여격은 에게에 해당합니다.

### 1.3 라틴어가 현대 언어에 미친 영향

```
라틴어 "aqua" (물):
  → 이탈리아어:   acqua
  → 스페인어:     agua
  → 포르투갈어:   água
  → 프랑스어:     eau (/ˈakwa/ → /o/에서 변화)
  → 영어:         aquatic, aquarium, aqueduct

라틴어 "nox, noctis" (밤):
  → 이탈리아어:   notte
  → 스페인어:     noche
  → 프랑스어:     nuit
  → 영어:         nocturnal, equinox

라틴어 "schola" (그리스어 σχολή에서):
  → 이탈리아어:   scuola
  → 스페인어:     escuela
  → 프랑스어:     école
  → 영어:         school
  → 한국어:       스쿨 (영어에서 들어온 외래어)
```

## 2. 라틴어 알파벳

### 2.1 23개 고전 문자

고전 라틴어는 23개의 문자를 사용했습니다. J, U, W는 별도의 문자로 존재하지 않았으며 — I가 /i/와 /j/ 두 음가를, V가 /u/와 /w/ 두 음가를 담당했습니다.

| 문자 | 이름 | IPA (고전) | 예시 |
|------|------|-----------|------|
| A a | ā | /a/, /aː/ | **a**qua (물) |
| B b | bē | /b/ | **b**onus (좋은) |
| C c | cē | /k/ | **c**aput (머리) |
| D d | dē | /d/ | **d**eus (신) |
| E e | ē | /ɛ/, /eː/ | **e**st (~이다) |
| F f | ef | /f/ | **f**ilius (아들) |
| G g | gē | /g/ | **g**loria (영광) |
| H h | hā | /h/ | **h**ora (시간) |
| I i | ī | /i/, /iː/, /j/ | **i**n (~안에), **i**am (이제) |
| K k | kā | /k/ | **K**alendae (초하루) |
| L l | el | /l/ | **l**ux (빛) |
| M m | em | /m/ | **m**ater (어머니) |
| N n | en | /n/ | **n**ox (밤) |
| O o | ō | /ɔ/, /oː/ | **o**mnis (모든) |
| P p | pē | /p/ | **p**ater (아버지) |
| Q q | qū | /k/ (항상 + u) | **qu**i (누구) |
| R r | er | /r/ | **r**ex (왕) |
| S s | es | /s/ | **s**ol (태양) |
| T t | tē | /t/ | **t**erra (땅) |
| V v | ū | /u/, /uː/, /w/ | **v**ia (길), **v**inum (포도주) |
| X x | ix | /ks/ | **r**ex (왕) |
| Y y | ȳ (그리스어) | /y/, /yː/ | **s**ylva (숲) |
| Z z | zēta (그리스어) | /dz/ | **z**ona (지대) |

### 2.2 문자에 관한 주요 사항

> **알파벳 핵심 사항**
>
> - C는 고전 라틴어에서 항상 경음 /k/입니다.
>   Caesar = /ˈkae̯.sar/ "시저"가 아님
>
> - G는 고전 라틴어에서 항상 경음 /g/입니다.
>   genus = /ˈgɛ.nus/ "지너스"가 아님
>
> - V는 모음 /u/와 자음 /w/ 두 가지를 나타냅니다.
>   현대 교재에서는 모음에 U, /w/에 V를 씁니다.
>   VINUM → uīnum /ˈwiː.num/
>
> - I는 모음 /i/와 자음 /j/ 두 가지를 나타냅니다.
>   현대 교재에서는 /j/에 J를 쓰기도 합니다.
>   IAM → iam /jam/ (이제, 이미)
>
> - K는 드물며 — 주로 Kalendae와 일부 외래어에 사용됩니다.
>
> - Y와 Z는 그리스어 차용어에서만 나타납니다.
>   Y = /y/ (독일어 ü 또는 프랑스어 u와 유사)
>   Z = /dz/

### 2.3 현대 표기 관례

현대 라틴어 교재와 판본에서는 흔히 다음과 같은 표기를 볼 수 있습니다:

| 관례 | 고전 | 현대 교재 |
|------|------|----------|
| 모음 u | VRBS | urbs |
| 자음 w | VINUM | vīnum |
| 자음 j | IAM | jam (일부 교재) |
| 장모음 | 표시 없음 | 장음부호(macron) 표시: ā, ē, ī, ō, ū |

> **장음부호(ˉ)**는 원래 라틴어 비문에는 없었지만, 현대 교육에서 장모음을 표시하기 위해 사용됩니다. 올바른 발음과 강세 배치에 필수적입니다.

## 3. 모음 체계

### 3.1 단모음과 장모음

라틴어에는 10개의 모음 음소가 있습니다 — 5개의 단모음과 5개의 장모음. 모음 길이(음량)는 변별적이며 단어의 의미를 바꿀 수 있습니다.

> **라틴어 모음 체계**
>
> 전설 / 중설 / 후설:
> - 고모음: ĭ /ɪ/, ī /iː/ / ŭ /ʊ/, ū /uː/
> - 중모음: ĕ /ɛ/, ē /eː/ / ŏ /ɔ/, ō /oː/
> - 저모음: ă /a/, ā /aː/
>
> 단모음 (ă, ĕ, ĭ, ŏ, ŭ):
> 지속 시간이 짧고 더 개방된 음질
>
> 장모음 (ā, ē, ī, ō, ū):
> 지속 시간이 길고 더 폐쇄된 음질
>
> 장음부호 (ˉ)는 장모음 표시: ā ē ī ō ū
> 단음부호 (˘)는 단모음 표시: ă ĕ ĭ ŏ ŭ (선택적)

| 모음 | IPA | 길이 | 영어 유사음 | 한국어 유사음 | 예시 |
|------|-----|------|-----------|-------------|------|
| ă | /a/ | 단 | "cut"의 "u" | 아 (짧게) | m**a**lus (나쁜) |
| ā | /aː/ | 장 | "father"의 "a" | 아 (길게) | m**ā**lus (사과나무) |
| ĕ | /ɛ/ | 단 | "bet"의 "e" | 에 (짧게) | l**e**gō (나는 읽는다) |
| ē | /eː/ | 장 | "say"의 "ay" (활음 없이) | 에 (길게) | l**ē**x (법) |
| ĭ | /ɪ/ | 단 | "bit"의 "i" | 이 (짧게) | s**i**t (~이기를) |
| ī | /iː/ | 장 | "see"의 "ee" | 이 (길게) | v**ī**ta (생명) |
| ŏ | /ɔ/ | 단 | "pot"의 "o" (영국식) | 오 (짧게) | d**o**minus (주인) |
| ō | /oː/ | 장 | "go"의 "o" (활음 없이) | 오 (길게) | Rōm**ā**nus (로마의) |
| ŭ | /ʊ/ | 단 | "put"의 "u" | 우 (짧게) | d**u**x (지도자) |
| ū | /uː/ | 장 | "food"의 "oo" | 우 (길게) | l**ū**na (달) |

### 3.2 모음 길이에 따른 의미 변화

| 단모음 | 의미 | 장모음 | 의미 |
|--------|------|--------|------|
| m**a**lus | 나쁜, 악한 | m**ā**lus | 사과나무 |
| l**e**vis | 가벼운 (무게) | l**ē**vis | 매끄러운 |
| l**i**ber | 자유로운 | l**ī**ber | 책 |
| p**o**pulus | 민중 | p**ō**pulus | 포플러나무 |
| **o**s | 뼈 | **ō**s | 입 |

> **핵심**: 영어에서는 흔히 강세로 단어를 구별하지만, 라틴어에서는 모음 길이가 같은 종류의 변별적 역할을 합니다. 교재의 장음부호에 항상 주의를 기울이세요.

## 4. 이중모음

### 4.1 고전 라틴어 이중모음

라틴어에는 6개의 이중모음이 있으며, 그중 3개는 일반적이고 3개는 드뭅니다.

> **라틴어 이중모음**
>
> 일반 이중모음:
> - ae /ae̯/ → "아이"와 유사 ("이"가 아님)
> - au /au̯/ → "아우"와 유사
> - oe /oe̯/ → "오이"와 유사
>
> 희귀 이중모음:
> - ei /ei̯/ → "에이"와 유사
> - eu /eu̯/ → 에 + 우 (영어에 해당 음 없음)
> - ui /ui̯/ → "위"와 유사 (huic, cui, hui에서만)
>
> 모든 이중모음은 강세와 운율에서 장음으로 취급됩니다.

| 이중모음 | IPA | 발음 | 예시 |
|----------|-----|------|------|
| ae | /ae̯/ | "아이" | C**ae**sar /ˈkae̯.sar/ (카이사르) |
| au | /au̯/ | "아우" | l**au**dō /ˈlau̯.doː/ (나는 칭찬한다) |
| oe | /oe̯/ | "오이" | p**oe**ta /poe̯.ˈta/ (시인) |
| ei | /ei̯/ | "에이" | d**ei**nde /ˈdei̯n.dɛ/ (그 다음에) |
| eu | /eu̯/ | 에 + 우 (활음) | **Eu**rōpa /eu̯.ˈroː.pa/ (유럽) |
| ui | /ui̯/ | 우 + 이 (빠르게) | h**ui**c /hui̯k/ (이것에게) |

### 4.2 AE와 OE에 관한 중요 사항

후기 라틴어와 교회 발음에서 **ae**는 /ɛː/(긴 "에"와 같은)로, **oe**는 /eː/로 변했습니다. 이것이 라틴어에서 온 많은 영어 단어가 "ae/oe"로 철자되지만 "이"로 발음되는 이유입니다:

```
고전 → 교회 → 영어:
  Caesar   /ˈkae̯.sar/ → /ˈtʃeː.sar/ → "시저"
  poena    /ˈpoe̯.na/  → /ˈpeː.na/   → "penal"
  caelum   /ˈkae̯.lum/ → /ˈtʃeː.lum/ → "celestial"
```

> 이 강좌에서는 별도의 언급이 없는 한 **고전(복원) 발음**을 사용합니다.

## 5. 자음 체계

### 5.1 자음 목록

|-----------|--------|------|----------|--------|

| 유성 폐쇄음 | b /b/ | d /d/ | g /g/ | — |

| 반모음 | v /w/ | — | — | — |
| 반모음 | — | i /j/ | — | — |

### 5.2 주요 자음 규칙 (고전 발음)

> **고전 라틴어 자음 규칙**
>
> - **C** = 항상 /k/, /s/나 /tʃ/가 아님
>   Cicerō = /ˈkɪ.kɛ.roː/ "시서로"가 아님
>   centum = /ˈkɛn.tum/ "센텀"이 아님
>
> - **G** = 항상 /g/, /dʒ/가 아님
>   genus = /ˈgɛ.nʊs/ "제너스"가 아님
>   gerō = /ˈgɛ.roː/
>
> - **V** = 자음일 때 /w/
>   vīnum = /ˈwiː.nʊm/ (포도주)
>   via = /ˈwɪ.a/ (길)
>
> - **I** = 자음일 때 /j/ (모음 앞에서)
>   iam = /jam/ (이제, 이미)
>   iūs = /juːs/ (법, 권리)
>
> - **S** = 항상 무성음 /s/, /z/가 아님
>   rosa = /ˈrɔ.sa/ /ˈrɔ.za/가 아님
>
> - **R** = 혀끝 떨림음 /r/ (이탈리아어나 스페인어처럼)
>   rēx = /reːks/ (왕)
>
> - **H** = /h/ — 고전 라틴어에서 실제 발음됨 (나중에 소실)
>   homō = /ˈhɔ.moː/ (사람)
>
> - **T** = 항상 /t/, i + 모음 앞에서도
>   nātiō = /ˈnaː.tɪ.oː/ "나시오"가 아님
>   ratiō = /ˈra.tɪ.oː/ "라시오"가 아님

### 5.3 겹자음 (장자음)

이탈리아어처럼 라틴어도 단자음과 겹자음을 구별합니다. 겹자음은 더 길게 발음됩니다.

```
겹자음 예시:
  annus /ˈan.nʊs/  (해)         vs  anus /ˈa.nʊs/  (노파)
  agger /ˈag.gɛr/  (둑)
  littera /ˈlɪt.tɛ.ra/ (글자)
  bellum /ˈbɛl.lʊm/ (전쟁)
  occīdō /ɔk.ˈkiː.doː/ (나는 죽인다)
```

### 5.4 특수 조합

| 조합 | 발음 | 예시 |
|------|------|------|
| qu | /kw/ | **qu**i /kwɪ/ (누구) |
| gu (n 뒤) | /gw/ | lin**gu**a /ˈlɪŋ.gwa/ (혀) |
| ph | /pʰ/ (유기음 p) | **ph**ilosophia /pʰɪ.lɔ.sɔ.ˈpʰɪ.a/ |
| th | /tʰ/ (유기음 t) | **th**eātrum /tʰɛ.ˈaː.trʊm/ (극장) |
| ch | /kʰ/ (유기음 k) | **ch**orus /ˈkʰɔ.rʊs/ (합창) |
| x | /ks/ | re**x** /reːks/ (왕) |
| bs | /ps/ | ur**bs** /ʊrps/ (도시) |
| bt | /pt/ | o**bt**ineō /ɔp.tɪ.ˈnɛ.oː/ (나는 얻는다) |

> **ph, th, ch에 관한 참고**: 이들은 그리스어에서 온 음입니다. 고전 라틴어에서는 유기 폐쇄음(/pʰ, tʰ, kʰ/)이며, /f, θ, x/가 아닙니다. "philosophy"를 영어에서 /f/로 발음하는 것은 후대의 변화에서 비롯됩니다.

## 6. 음절 구조와 음량

### 6.1 음절 분리 규칙

```
1. 모음 사이의 단자음 → 두 번째 모음에 붙음:
   a-qua, do-mi-nus, pa-ter, a-mō

2. 모음 사이의 두 자음 → 일반적으로 나뉨:
   mag-nus, ter-ra, pul-cher
   예외: 폐쇄음/마찰음 + 유음(l, r)은 함께 남음:
   pa-tris, li-brum, sa-crum

3. 겹자음은 항상 나뉨:
   an-nus, bel-lum, lit-te-ra

4. 접두사는 경계를 유지:
   ab-est, ad-est, in-est
```

### 6.2 음절 음량 (경음절 vs 중음절)

음절 음량은 강세 배치와 운문(운율)에 매우 중요합니다.

> **음절 무게 (음량)**
>
> **경음절** (syllaba brevis):
> 단모음 + 개음절 (종성 자음 없음)을 포함
> 예: ă-mō (ă는 경음절), dŏ-mi-nus (dŏ는 경음절)
>
> **중음절** (syllaba longa):
> 다음 중 하나:
> - (a) 장모음 또는 이중모음 포함
>   예: Rō-ma (Rō는 중음절), Cae-sar (Cae는 중음절)
> - (b) 자음으로 끝남 (폐음절)
>   예: mag-nus (mag는 중음절), ter-ra (ter는 중음절)
>
> **기억법:**
> - 경음절 = 단모음 + 아무것도 없음
> - 중음절 = 장모음, 또는 이중모음, 또는 단모음 + 자음

### 6.3 음량이 중요한 이유

1. **강세 배치** — 강세 규칙은 끝에서 두 번째 음절의 무게에 따라 결정됩니다.
2. **라틴어 운문** — 운율(장단단육보격, 비가 2행시 등)은 강세가 아닌 중음절과 경음절의 패턴에 기반합니다.
3. **정확한 발음** — 장모음과 단모음은 다르게 들립니다.

## 7. 강세 규칙

### 7.1 세 가지 강세 규칙

라틴어 강세는 음절 음량에서 완전히 예측 가능합니다:

> **라틴어 강세 규칙**
>
> **규칙 1: 단음절어**
> 음절이 하나인 단어는 그 음절에 강세가 옵니다.
> rēx, lūx, nōn, et
>
> **규칙 2: 2음절어**
> 항상 첫 번째 음절에 강세가 옵니다.
> RO-sa, MA-ter, A-mō, TER-ra
>
> **규칙 3: 3음절 이상의 단어**
> 끝에서 두 번째(차말) 음절을 확인:
>
> - (a) 차말이 중음절이면 → 차말에 강세
>   a-MĪ-cus (mī는 장모음 → 중음절 → 강세)
>   ma-GIS-ter (gis는 자음으로 끝남 → 중음절 → 강세)
>   for-TŪ-na (tū는 장모음 → 중음절 → 강세)
>
> - (b) 차말이 경음절이면 → 끝에서 세 번째 음절에 강세
>   DO-mi-nus (mi는 단모음 + 개음절 → 경 → DO에 강세)
>   LIT-te-ra (te는 단모음 + 개음절 → 경 → LIT에 강세)
>   FE-mi-na (mi는 단모음 + 개음절 → 경 → FE에 강세)
>
> ※ 라틴어는 절대 마지막 음절에 강세를 두지 않습니다.
> ※ 라틴어는 끝에서 세 번째보다 앞에 강세를 두지 않습니다.

### 7.2 강세 예시

| 단어 | 음절 | 차말 무게 | 강세 | 발음 |
|------|------|----------|------|------|
| amīcus | a-mī-cus | mī = 중 (장모음 ī) | a-**MĪ**-cus | /a.ˈmiː.kʊs/ |
| dominus | do-mi-nus | mi = 경 (단, 개음절) | **DO**-mi-nus | /ˈdɔ.mɪ.nʊs/ |
| fortūna | for-tū-na | tū = 중 (장모음 ū) | for-**TŪ**-na | /fɔr.ˈtuː.na/ |
| fēmina | fē-mi-na | mi = 경 (단, 개음절) | **FĒ**-mi-na | /ˈfeː.mɪ.na/ |
| magister | ma-gis-ter | gis = 중 (폐음절) | ma-**GIS**-ter | /ma.ˈgɪs.tɛr/ |
| taberna | ta-ber-na | ber = 중 (폐음절) | ta-**BER**-na | /ta.ˈbɛr.na/ |

## 8. 고전 발음 vs 교회 발음

### 8.1 개요

라틴어에는 두 가지 주요 발음 전통이 있습니다:

| 특징 | 고전 (복원) | 교회 (로마 가톨릭) |
|------|-----------|-------------------|
| 시기 | 기원전 약 100년 – 기원후 200년 | 중세 이후 |
| 사용자 | 고전학자, 대학교 | 로마 가톨릭 교회 |
| e, i 앞의 C | 항상 /k/ | /tʃ/ (이탈리아어처럼) |
| e, i 앞의 G | 항상 /g/ | /dʒ/ (이탈리아어처럼) |
| 자음 V | /w/ | /v/ (이탈리아어처럼) |
| AE 이중모음 | /ae̯/ | /ɛː/ ("에"와 같음) |
| OE 이중모음 | /oe̯/ | /eː/ ("에"와 같음) |
| 모음 앞 TI | /tɪ/ | /tsɪ/ 또는 /ʃɪ/ |
| H | /h/ (발음됨) | 묵음 |
| 모음 사이 S | 항상 /s/ | 때때로 /z/ |

### 8.2 비교 예시

| 단어 | 고전 | 교회 | 의미 |
|------|------|------|------|
| Caesar | /ˈkae̯.sar/ | /ˈtʃeː.sar/ | 카이사르 |
| Cicerō | /ˈkɪ.kɛ.roː/ | /ˈtʃiː.tʃɛ.roː/ | 키케로 |
| vēnī | /ˈweː.niː/ | /ˈveː.niː/ | 나는 왔다 |
| genus | /ˈgɛ.nʊs/ | /ˈdʒɛ.nʊs/ | 종류, 종족 |
| caelum | /ˈkae̯.lʊm/ | /ˈtʃeː.lʊm/ | 하늘, 천국 |
| nātiō | /ˈnaː.tɪ.oː/ | /ˈnaː.tsɪ.oː/ | 민족 |
| Iesus | /ˈjɛ.sʊs/ | /ˈjeː.zʊs/ | 예수 |

> **이 강좌는 고전 발음을 사용합니다.** 이것이 고대 로마인들이 실제로 말한 언어를 반영하기 때문입니다. 그러나 중세 및 교회 라틴어를 읽기 위해서는 교회 발음에 대한 이해도 유용합니다.

## 9. 흔한 발음 도전 과제

### 9.1 영어 화자의 경우

| 실수 | 올바른 발음 | 설명 |
|------|-----------|------|
| e, i 앞에서 C를 /s/로 | 항상 /k/ | "Cicero" = /ˈkɪ.kɛ.roː/ "시서로"가 아님 |
| V를 /v/로 | 고전에서 V = /w/ | *vīnum* = /ˈwiː.nʊm/ /ˈviː.nʊm/이 아님 |
| AE를 "이"로 | AE = /ae̯/ ("아이") | *Caesar* = /ˈkae̯.sar/ "시저"가 아님 |
| TI를 "시"로 | TI = /tɪ/ | *nātiō* = /ˈnaː.tɪ.oː/ "나시오"가 아님 |
| 모음 길이 무시 | 장 ≠ 단 | *malus* (나쁜) ≠ *mālus* (사과나무) |
| 비강세 모음 약화 | 모든 모음을 뚜렷하게 | 라틴어에 슈와 /ə/ 없음 |
| 영어식 R | 혀끝 떨림음 /r/ 사용 | 이탈리아어나 스페인어처럼 혀끝 떨림음 |
| 묵음 H | H는 /h/로 발음 | *homō* = /ˈhɔ.moː/ /ˈɔ.moː/가 아님 |

### 9.2 한국어 화자의 경우

| 실수 | 올바른 발음 | 설명 |
|------|-----------|------|
| /r/과 /l/ 구별 안 됨 | 라틴어에는 둘 다 있음 | R = 혀끝 떨림음 /r/; L = 설측음 /l/. 한국어 ㄹ이 아님 |
| 어말 자음 뒤 모음 첨가 | 자음군을 깨끗하게 유지 | *rēx* = /reːks/ /reːkɯsɯ/가 아님 |
| 장 = 단 혼동 | 길이가 중요 | 장모음을 눈에 띄게 더 길게 유지하는 연습 |
| /f/를 /p/로 | 순치 마찰음 /f/ 사용 | 아랫입술이 윗니에 닿음 |
| /w/ 혼동 | V = /w/ (영어 "w"처럼) | 입술을 둥글게 하고 활음: *vīnum* = /ˈwiː.nʊm/ |
| 이중모음 축약 | 두 요소 모두 유지 | AE = /ae̯/, 두 모음이 함께 활음 |
| H 탈락 | 고전 H = /h/ | 실제 기식음, 묵음이 아님 |

## 빠른 참고 카드

> **라틴어 발음 빠른 참고**
>
> **모음** (10개 음소):
> - 단: ă /a/, ĕ /ɛ/, ĭ /ɪ/, ŏ /ɔ/, ŭ /ʊ/
> - 장: ā /aː/, ē /eː/, ī /iː/, ō /oː/, ū /uː/
> - 장음부호 (ˉ) = 장모음 표시
>
> **이중모음**: ae /ae̯/, au /au̯/, oe /oe̯/
> - ae = "아이", au = "아우", oe = "오이"
>
> **주요 자음** (고전):
> - C = 항상 /k/ / G = 항상 /g/
> - V = /w/ (자음) / I = /j/ (모음 앞)
> - S = 항상 /s/ / H = /h/ (발음됨!)
> - R = 혀끝 떨림음 /r/ / T = 항상 /t/
> - QU = /kw/ / X = /ks/
>
> **강세 규칙:**
> - 2음절 → 첫 음절에 강세
> - 3음절 이상 → 차말 확인:
>   - 중음절 차말 → 차말에 강세
>   - 경음절 차말 → 전전말에 강세
> - 마지막 음절에는 절대 강세 없음
>
> **음절 무게:**
> - 중음절 = 장모음 또는 이중모음 또는 폐음절
> - 경음절 = 단모음 + 개음절
