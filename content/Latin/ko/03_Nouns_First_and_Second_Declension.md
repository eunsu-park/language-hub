## 1. 격 체계 소개

### 1.1 격이란 무엇인가?

영어에서는 어순이 누가 무엇을 하는지 알려줍니다: "The dog bites the man"(개가 사람을 문다) vs. "The man bites the dog"(사람이 개를 문다). 라틴어에서는 명사의 **격 어미**가 이 기능을 합니다. 명사의 어미가 어순에 관계없이 문장에서의 역할을 알려줍니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                라틴어의 여섯 가지 격                                │
│                                                                 │
│   격            기능                     한국어 대응              │
│   ──────────    ─────────────────────   ─────────────           │
│   주격          주어                     이/가                    │
│   속격          소유 ("~의")             의                       │
│   여격          간접 목적어 ("~에게")     에게/한테                │
│   대격          직접 목적어              을/를                    │
│   탈격          다양한 용법("~로/에서")   으로/에서                │
│   호격          직접 호칭                아/야                    │
│                                                                 │
│   ※ 한국어 조사와의 유사성이 매우 높습니다. 두 언어 모두           │
│     어순에만 의존하지 않고 어미/조사를 사용합니다.                  │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 격 기능의 상세 설명

| 격 | 약어 | 주요 기능 | 예시 (한국어) |
|------|-------------|-----------------|-------------------|
| **주격** | Nom. | 문장의 주어 | **소녀가** 노래한다. |
| **속격** | Gen. | 소유, "~의" | **농부의** 소녀 (= 농부의 딸) |
| **여격** | Dat. | 간접 목적어, "~에게/~을 위해" | 그는 **소녀에게** 장미를 준다. |
| **대격** | Acc. | 직접 목적어 | 그는 **소녀를** 본다. |
| **탈격** | Abl. | 수단, 방법, 장소, 시간 ("~로/~와 함께/~에서") | 그는 **소녀와 함께** 걷는다. |
| **호격** | Voc. | 직접 호칭 | **소녀야**, 이리 와! |

### 1.3 격 어미의 작동 방식

```
puella (소녀)를 이용한 예시:

  Puella cantat.             소녀가(주격) 노래한다.
  Fīlia agricolae cantat.   농부의(속격) 딸이 노래한다.
  Puellae rosam dat.         소녀에게(여격) 장미를 준다.
  Puellam videt.             소녀를(대격) 본다.
  Cum puellā ambulat.        소녀와 함께(탈격) 걷는다.
  Puella, venī!              소녀야(호격), 이리 와!

주의: "puella"의 어미가 역할에 따라 변합니다:
  puella  (주격)  →  puellae  (속격)  →  puellae  (여격)
  puellam (대격)  →  puellā   (탈격)  →  puella   (호격)
```

### 1.4 수: 단수와 복수

모든 명사 형태는 격과 수(단수 또는 복수)를 모두 가집니다. 이는 각 변화가 총 12개의 형태(6격 x 2수)를 갖는다는 것을 의미하지만, 일부 형태는 동일합니다.

## 2. 라틴어의 성

### 2.1 세 가지 성

라틴어에는 세 가지 문법적 성이 있습니다: **남성**, **여성**, **중성**.

```
┌─────────────────────────────────────────────────────────────────┐
│                  라틴어 문법적 성                                  │
│                                                                 │
│   남성 (M):  servus (노예), amīcus (친구), puer (소년)           │
│   여성 (F):  puella (소녀), rosa (장미), terra (땅)              │
│   중성 (N):  bellum (전쟁), dōnum (선물), templum (신전)         │
│                                                                 │
│   성은 부분적으로 의미적이고, 부분적으로 자의적입니다:               │
│                                                                 │
│   의미적 (논리적):                                               │
│     남성 존재 → 남성: vir (남자), fīlius (아들)                  │
│     여성 존재 → 여성: fēmina (여자), fīlia (딸)                 │
│                                                                 │
│   자의적 (문법적):                                               │
│     왜 mūrus (벽)는 남성일까?                                    │
│     왜 via (길)는 여성일까?                                      │
│     왜 bellum (전쟁)은 중성일까?                                  │
│     → 논리적 이유 없음; 성을 암기해야 합니다.                      │
│                                                                 │
│   변화형이 성 예측에 도움이 됩니다:                                │
│     제1변화 (-a) → 대부분 여성                                   │
│     제2변화 (-us, -er) → 대부분 남성                             │
│     제2변화 (-um) → 항상 중성                                    │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 성이 중요한 이유

성은 다음에 영향을 미칩니다:
1. **형용사 일치** — 형용사는 수식하는 명사의 성, 수, 격과 일치해야 합니다.
2. **대명사 지시** — 대명사는 선행사의 성과 일치해야 합니다.
3. **변화 패턴** — 성이 종종 명사가 사용하는 어미 세트를 결정합니다.

> **한국어 사용자를 위한 참고**: 한국어에는 문법적 성이 없으므로 이 개념이 낯설 수 있습니다. 성을 일종의 분류 체계라고 생각하세요 — 각 명사는 주변 단어의 행동을 결정하는 "그룹"에 속합니다.

## 3. 제1변화

### 3.1 개요

제1변화 명사는 어간 모음 **-a-**가 특징입니다. 대부분 여성이며, 주로 직업과 자연적 남성을 나타내는 소수의 남성 예외가 있습니다.

```
어간 모음: -a-
성: 거의 항상 여성
사전 표제어: puella, -ae, f. (소녀)
              ↑       ↑    ↑
              주격단수  속격단수  성
```

### 3.2 전체 패러다임: puella, -ae, f. (소녀)

| 격 | 단수 | 복수 | 기능 |
|------|----------|--------|----------|
| 주격 | puell**a** | puell**ae** | 주어 |
| 속격 | puell**ae** | puell**ārum** | "~의" / 소유 |
| 여격 | puell**ae** | puell**īs** | "~에게/~을 위해" |
| 대격 | puell**am** | puell**ās** | 직접 목적어 |
| 탈격 | puell**ā** | puell**īs** | "~로/~와 함께/~에서" |
| 호격 | puell**a** | puell**ae** | 직접 호칭 |

```
┌─────────────────────────────────────────────────────────────────┐
│          제1변화 어미 (반드시 암기하세요!)                          │
│                                                                 │
│              단수       복수                                     │
│   주격       -a          -ae                                    │
│   속격       -ae         -ārum                                  │
│   여격       -ae         -īs                                    │
│   대격       -am         -ās                                    │
│   탈격       -ā          -īs                                    │
│   호격       -a          -ae                                    │
│                                                                 │
│   ※ 주격 단수와 호격 단수는 동일 (-a)                             │
│   ※ 속격 단수, 여격 단수, 주격 복수, 호격 복수는 모두 -ae          │
│   ※ 여격 복수와 탈격 복수는 동일 (-īs)                           │
│   ※ 탈격 단수 (-ā)는 장모음 (주격과 구별됨)                       │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 주요 제1변화 명사

| 라틴어 | 속격 단수 | 성 | 의미 |
|-------|----------|--------|---------|
| aqua | aquae | f. | 물 |
| fēmina | fēminae | f. | 여자 |
| fīlia | fīliae | f. | 딸 |
| fortūna | fortūnae | f. | 운, 행운 |
| Itālia | Itāliae | f. | 이탈리아 |
| lingua | linguae | f. | 혀, 언어 |
| lūna | lūnae | f. | 달 |
| patria | patriae | f. | 조국, 나라 |
| pecūnia | pecūniae | f. | 돈 |
| poēta | poētae | **m.** | 시인 (남성!) |
| puella | puellae | f. | 소녀 |
| rosa | rosae | f. | 장미 |
| terra | terrae | f. | 땅, 대지 |
| via | viae | f. | 길, 도로 |
| vīta | vītae | f. | 삶, 생명 |

### 3.4 남성 제1변화 명사

소수의 제1변화 명사는 **남성**입니다 — 주로 남성 직업이나 남성 인물:

| 라틴어 | 의미 | 비고 |
|-------|---------|------|
| agricola, -ae, m. | 농부 | 가장 흔한 예 |
| nauta, -ae, m. | 선원 | |
| poēta, -ae, m. | 시인 | 그리스어 기원 |
| pīrāta, -ae, m. | 해적 | 그리스어 기원 |
| incola, -ae, m. | 주민 | |

> 이들은 *puella*와 정확히 같이 변화합니다 — **어미**는 같고 **성**만 다릅니다. 이것은 형용사 일치에 중요합니다: *bonus agricola* (좋은 농부)는 남성 형용사를 사용합니다.

### 3.5 제1변화의 실제 사용

```
Puella rosam amat.
puella (주격 단수) = "소녀가" → 주어
rosam (대격 단수)  = "장미를" → 직접 목적어
amat              = "사랑한다" → 동사
→ "소녀가 장미를 사랑한다."

Agricolae fīlia cantat.
agricolae (속격 단수) = "농부의"  → 소유
fīlia (주격 단수)     = "딸이"   → 주어
cantat               = "노래한다" → 동사
→ "농부의 딸이 노래한다."

Fēminae pecūniam dant.
fēminae (여격 단수)  = "여자에게"  → 간접 목적어
pecūniam (대격 단수)  = "돈을"     → 직접 목적어
dant                  = "그들이 준다" → 동사
→ "그들이 여자에게 돈을 준다."

Cum puellīs ambulat.
cum (전치사 + 탈격)  = "~와 함께"
puellīs (탈격 복수)  = "소녀들"
ambulat              = "그/그녀가 걷는다"
→ "그/그녀가 소녀들과 함께 걷는다."
```

## 4. 제2변화 남성

### 4.1 개요

제2변화 명사는 어간 모음 **-o-**(때때로 -u-로 나타남)를 갖습니다. 이 변화의 남성 명사는 주격 단수에서 일반적으로 **-us** 또는 **-er**로 끝납니다.

### 4.2 유형 1: -us 명사 (amīcus 패턴)

전체 패러다임: **amīcus, -ī, m.** (친구)

| 격 | 단수 | 복수 | 기능 |
|------|----------|--------|----------|
| 주격 | amīc**us** | amīc**ī** | 주어 |
| 속격 | amīc**ī** | amīc**ōrum** | "~의" / 소유 |
| 여격 | amīc**ō** | amīc**īs** | "~에게/~을 위해" |
| 대격 | amīc**um** | amīc**ōs** | 직접 목적어 |
| 탈격 | amīc**ō** | amīc**īs** | "~로/~와 함께/~에서" |
| 호격 | amīc**e** | amīc**ī** | 직접 호칭 |

```
┌─────────────────────────────────────────────────────────────────┐
│        제2변화 -us 어미 (반드시 암기하세요!)                        │
│                                                                 │
│              단수       복수                                     │
│   주격       -us         -ī                                     │
│   속격       -ī          -ōrum                                  │
│   여격       -ō          -īs                                    │
│   대격       -um         -ōs                                    │
│   탈격       -ō          -īs                                    │
│   호격       -e          -ī                                     │
│                                                                 │
│   ※ 속격 단수, 주격 복수, 호격 복수는 모두 -ī                    │
│   ※ 여격 단수와 탈격 단수는 동일 (-ō)                            │
│   ※ 여격 복수와 탈격 복수는 동일 (-īs)                           │
│   ※ 호격 단수는 -e (변화 중 유일!)                               │
│   ※ 특별: -ius로 끝나는 명사는 호격 단수가 -ī:                   │
│     fīlius → fīlī (오, 아들이여!)                               │
└─────────────────────────────────────────────────────────────────┘
```

### 4.3 유형 2: -er 명사

일부 제2변화 남성 명사는 주격에서 **-er**로 끝납니다. 두 가지 하위 유형이 있습니다:

**하위 유형 A — 전체에 걸쳐 -e-를 유지 (puer 패턴):**

| 격 | 단수 | 복수 |
|------|----------|--------|
| 주격 | pu**er** | pu**erī** |
| 속격 | pu**erī** | pu**erōrum** |
| 여격 | pu**erō** | pu**erīs** |
| 대격 | pu**erum** | pu**erōs** |
| 탈격 | pu**erō** | pu**erīs** |
| 호격 | pu**er** | pu**erī** |

**하위 유형 B — 주격 이후 -e-가 탈락 (ager 패턴):**

| 격 | 단수 | 복수 |
|------|----------|--------|
| 주격 | ag**er** | agr**ī** |
| 속격 | agr**ī** | agr**ōrum** |
| 여격 | agr**ō** | agr**īs** |
| 대격 | agr**um** | agr**ōs** |
| 탈격 | agr**ō** | agr**īs** |
| 호격 | ag**er** | agr**ī** |

```
┌─────────────────────────────────────────────────────────────────┐
│   어떤 -er 하위 유형인지 어떻게 구별할까? 속격 단수를 확인하세요.  │
│                                                                 │
│   puer, puerī → 어간 = puer- (-e- 유지)                        │
│   ager, agrī  → 어간 = agr-  (-e- 탈락)                        │
│                                                                 │
│   속격 단수(사전 표제어의 두 번째 형태)가 진짜 어간을             │
│   보여줍니다. 항상 명사를 속격과 함께 학습하세요!                   │
│                                                                 │
│   -e- 유지:  puer (소년), socer (장인),                         │
│              līber (자유로운 — 형용사), vir* (남자)               │
│   -e- 탈락:  ager (밭), liber (책), magister (선생님),           │
│              minister (하인)                                    │
│                                                                 │
│   *vir, virī는 약간 불규칙 (제2변화, -er 유형은 아님)            │
└─────────────────────────────────────────────────────────────────┘
```

### 4.4 주요 제2변화 남성 명사

| 라틴어 | 속격 단수 | 의미 |
|-------|----------|---------|
| amīcus, -ī | amīcī | 친구 |
| animus, -ī | animī | 정신, 마음 |
| annus, -ī | annī | 해, 년 |
| deus, -ī | deī | 신 |

| fīlius, -ī (호격 fīlī) | fīliī | 아들 |

| liber, -brī | librī | 책 |
| locus, -ī | locī | 장소 |
| magister, -trī | magistrī | 선생님, 스승 |
| mūrus, -ī | mūrī | 벽 |
| numerus, -ī | numerī | 수, 숫자 |
| oculus, -ī | oculī | 눈 |
| populus, -ī | populī | 국민, 민족 |

## 5. 제2변화 중성

### 5.1 개요

제2변화 중성 명사는 주격 단수에서 **-um**으로 끝납니다. -us 남성과 같은 기본 패턴을 따르지만 두 가지 중요한 차이점이 있습니다.

### 5.2 전체 패러다임: bellum, -ī, n. (전쟁)

| 격 | 단수 | 복수 | 기능 |
|------|----------|--------|----------|
| 주격 | bell**um** | bell**a** | 주어 |
| 속격 | bell**ī** | bell**ōrum** | "~의" / 소유 |
| 여격 | bell**ō** | bell**īs** | "~에게/~을 위해" |
| 대격 | bell**um** | bell**a** | 직접 목적어 |
| 탈격 | bell**ō** | bell**īs** | "~로/~와 함께/~에서" |
| 호격 | bell**um** | bell**a** | 직접 호칭 |

```
┌─────────────────────────────────────────────────────────────────┐
│          중성 규칙 (모든 변화에 적용!)                              │
│                                                                 │

│                                                                 │
│   1. 주격 = 대격 = 호격 (항상 동일)                               │
│      단수: -um = -um = -um                                      │
│      복수: -a  = -a  = -a                                       │
│                                                                 │
│   2. 주격/대격 복수는 항상 -a로 끝남                               │
│                                                                 │
│   이것은 라틴어 문법에서 가장 신뢰할 수 있는 규칙 중 하나입니다.    │

│                                                                 │
```

```
┌─────────────────────────────────────────────────────────────────┐
│           격 기능 — 예문을 포함한 요약                              │

│     Puerō librum dat.       소년에게 책을 준다.                    │
│     Patriae labōrat.        조국을 위해 일한다.                    │

│     Prīmō annō             첫 해에 (시간)                        │
│                                                                 │
│   호격 — 직접 호칭                                                │
│     Mārce, venī!            마르쿠스여, 이리 와!                   │

└─────────────────────────────────────────────────────────────────┘
```

### 6.2 격을 지배하는 전치사

라틴어 전치사는 대격 또는 탈격을 취합니다:

| 전치사 | 격 | 의미 |
|------------|------|---------|
| ad | + 대격 | ~으로, ~을 향해 |
| ante | + 대격 | ~전에, ~앞에 |
| contrā | + 대격 | ~에 대항하여 |
| in | + 대격 | ~안으로, ~위로 (이동 방향) |
| per | + 대격 | ~을 통해, ~에 의해 |
| post | + 대격 | ~후에, ~뒤에 |
| trāns | + 대격 | ~을 가로질러 |
| ā / ab | + 탈격 | ~로부터, ~에 의해 (행위자) |
| cum | + 탈격 | ~와 함께 |
| dē | + 탈격 | ~로부터 아래로, ~에 관하여 |
| ex / ē | + 탈격 | ~에서 밖으로, ~로부터 |
| in | + 탈격 | ~안에, ~위에 (위치, 이동 없음) |
| prō | + 탈격 | ~앞에, ~을 대신하여 |
| sine | + 탈격 | ~없이 |

> **참고**: 전치사 *in*은 의미에 따라 **두 가지 다른 격**을 취합니다: *in + 대격* = "~안으로" (이동), *in + 탈격* = "~안에/~위에" (위치). 한국어의 에 (위치) vs 으로 (방향)와 비교해 보세요.

## 7. 간단한 문장 읽기 연습

### 7.1 제1변화 문장

다음 문장을 읽고 번역하세요. 각 명사의 격을 확인하세요.

```
1. Puella rosam portat.
   puella (주격) rosam (대격) portat (동사)
   → 소녀가 장미를 운반한다.

2. Poēta fābulam nārrat.
   poēta (주격, 남성) fābulam (대격) nārrat (동사)
   → 시인이 이야기를 들려준다.

3. Fēminae in viā ambulant.
   fēminae (주격 복수) in viā (탈격) ambulant (동사)
   → 여자들이 길 위를 걷는다.

4. Nautae patriam amant.
   nautae (주격 복수) patriam (대격) amant (동사)
   → 선원들이 조국을 사랑한다.

5. Agricola fīliārum fortūnam laudat.
   agricola (주격) fīliārum (속격 복수) fortūnam (대격) laudat (동사)
   → 농부가 딸들의 행운을 칭찬한다.
```

### 7.2 제2변화 문장

```
1. Servus dominō pecūniam dat.
   servus (주격) dominō (여격) pecūniam (대격) dat (동사)
   → 노예가 주인에게 돈을 준다.

2. Fīlius magistrī est puer bonus.
   fīlius (주격) magistrī (속격) est (동사) puer (주격) bonus (주격)
   → 선생님의 아들은 좋은 소년이다.

3. Bellum est perīculum magnum.
   bellum (주격) est (동사) perīculum (주격) magnum (주격)
   → 전쟁은 큰 위험이다.

4. Populī Rōmānī arma sunt multa.
   populī (속격) Rōmānī (속격) arma (주격 복수 중성) sunt (동사) multa
   → 로마 국민의 무기는 많다.

5. In templō deōrum dōna sunt.
   in templō (탈격) deōrum (속격 복수) dōna (주격 복수 중성) sunt (동사)
   → 신전 안에 신들의 선물이 있다.
```

### 7.3 혼합 변화 문장

```
1. Puella fīliō agricolae rosam dat.
   puella (주격 제1변화) fīliō (여격 제2변화) agricolae (속격 제1변화)
   rosam (대격 제1변화) dat (동사)
   → 소녀가 농부의 아들에게 장미를 준다.

2. Dominus servīs pecūniam et cibum dat.
   dominus (주격 제2변화) servīs (여격 제2변화) pecūniam (대격 제1변화)
   et cibum (대격 제2변화) dat (동사)
   → 주인이 노예들에게 돈과 음식을 준다.

3. Magister puerōs et puellās docet.
   magister (주격 제2변화) puerōs (대격 제2변화) et puellās (대격 제1변화)
   docet (동사)
   → 선생님이 소년들과 소녀들을 가르친다.
```

```
│              단수    복수                                        │
│   주격       -um     -a          속격  -ī      -ōrum            │
│   여격       -ō      -īs         대격  -um     -a               │
│   탈격       -ō      -īs         호격  -um     -a               │
│                                                                 │
│   중성 규칙: 주격 = 대격 = 호격 (항상!)                            │
│              복수 주격/대격/호격은 항상 -a로 끝남                   │
│                                                                 │
│   여섯 격: 주격 (주어), 속격 (~의), 여격 (~에게/~을 위해),         │
│           대격 (목적어), 탈격 (~로/~와 함께/~안에), 호격 (호칭)     │
```
