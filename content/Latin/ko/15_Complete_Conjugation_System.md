# 15. 완전 활용 체계 (Systema Coniugationis Completum)

## 학습 목표
- 능동과 수동 모두에서 완료 접속법과 대과거 접속법 형성하기
- 모든 시제와 법에 걸쳐 이탈동사(deponent verb) 완전 활용 마스터하기
- 반이탈동사(semi-deponent verb) 인식 및 활용하기
- 현재 명령법과 미래 명령법 구성하기
- 모든 시제, 법, 태를 하나의 완전한 동사 패러다임으로 종합하기
- 모든 라틴어 동사 형태를 위한 종합 참고 도표 활용하기

## 1. 완료 접속법

### 1.1 능동 형성

**완료 어간 + -eri- + 인칭 어미**

| 인칭 | 어미 | amō (완료 어간: amāv-) |
|------|------|------------------------|
| 1단 | -erim | amāv**erim** |
| 2단 | -erīs | amāv**erīs** |
| 3단 | -erit | amāv**erit** |
| 1복 | -erīmus | amāv**erīmus** |
| 2복 | -erītis | amāv**erītis** |
| 3복 | -erint | amāv**erint** |

> **주의**: 완료 접속법(*amāverim*)은 미래완료 직설법(*amāverō*)과 유사해 보입니다. 주의 깊게 비교하세요:

| 시제/법 | 1단 | 3단 | 3복 |
|---------|-----|-----|-----|
| 미래완료 직설법 | amāv**erō** | amāv**erit** | amāv**erint** |
| 완료 접속법 | amāv**erim** | amāv**erit** | amāv**erint** |

1인칭 단수 형태가 가장 명확한 구별점입니다: **-erō** (직설법) vs. **-erim** (접속법).

### 1.2 *esse*의 완료 접속법

| 인칭 | 형태 |
|------|------|
| 1단 | fuerim |
| 2단 | fuerīs |
| 3단 | fuerit |
| 1복 | fuerīmus |
| 2복 | fuerītis |
| 3복 | fuerint |

### 1.3 수동 형성

**PPP + *esse*의 현재 접속법**

| 인칭 | 남성 단수 | 여성 단수 |
|------|----------|----------|
| 1단 | amātus sim | amāta sim |
| 2단 | amātus sīs | amāta sīs |
| 3단 | amātus sit | amāta sit |
| 1복 | amātī sīmus | amātae sīmus |
| 2복 | amātī sītis | amātae sītis |
| 3복 | amātī sint | amātae sint |

## 2. 대과거 접속법

### 2.1 능동 형성

**완료 능동 부정사 + 인칭 어미**

이것은 가장 쉽게 형성되는 접속법입니다 — 완료 부정사(*amāvisse*)에 어미만 붙이면 됩니다.

| 인칭 | 어미 | amō (완료 부정사: amāvisse) |
|------|------|---------------------------|
| 1단 | -m | amāvisse**m** |
| 2단 | -s | amāvissē**s** |
| 3단 | -t | amāvisse**t** |
| 1복 | -mus | amāvissē**mus** |
| 2복 | -tis | amāvissē**tis** |
| 3복 | -nt | amāvisse**nt** |

### 2.2 *esse*의 대과거 접속법

| 인칭 | 형태 |
|------|------|
| 1단 | fuissem |
| 2단 | fuissēs |
| 3단 | fuisset |
| 1복 | fuissēmus |
| 2복 | fuissētis |
| 3복 | fuissent |

### 2.3 수동 형성

**PPP + *esse*의 미완료 접속법**

| 인칭 | 남성 단수 | 복수 (남성) |
|------|----------|-----------|
| 1단 | amātus essem | amātī essēmus |
| 2단 | amātus essēs | amātī essētis |
| 3단 | amātus esset | amātī essent |

### 2.4 접속법 체계 완전 개요

| 시제 | 능동 | 수동 |
|------|------|------|
| 현재 | 어간 모음 변화 + 어미 | 어간 모음 변화 + 수동 어미 |
| 미완료 | 부정사 + 어미 | 부정사 + 수동 어미 |
| 완료 | 완료 어간 + -eri- + 어미 | PPP + sim/sīs/sit... |
| 대과거 | 완료 어간 + -isse- + 어미 | PPP + essem/essēs/esset... |

## 3. 이탈동사: 완전한 다루기

이탈동사는 **수동 형태**를 가지지만 **능동 의미**를 가집니다. 모든 시제와 법에서 나타납니다.

### 3.1 기본형

이탈동사는 **세 개의** 기본형만 있습니다(모두 수동 형태):

| 동사 | 제1 기본형 | 제2 기본형 | 제3 기본형 | 의미 |
|------|----------|----------|----------|------|
| cōnor (제1) | cōnor | cōnārī | cōnātus sum | 시도하다 |
| vereor (제2) | vereor | verērī | veritus sum | 두려워하다 |
| sequor (제3) | sequor | sequī | secūtus sum | 따르다 |
| loquor (제3) | loquor | loquī | locūtus sum | 말하다 |
| morior (제3-iō) | morior | morī | mortuus sum | 죽다 |
| patior (제3-iō) | patior | patī | passus sum | 겪다, 허락하다 |
| ūtor (제3) | ūtor | ūtī | ūsus sum | 사용하다 (+ 탈격) |
| proficīscor (제3) | proficīscor | proficīscī | profectus sum | 출발하다 |
| ingredior (제3-iō) | ingredior | ingredī | ingressus sum | 들어가다 |
| ēgredior (제3-iō) | ēgredior | ēgredī | ēgressus sum | 나가다 |
| orior (제4) | orior | orīrī | ortus sum | 일어나다 |

### 3.2 완전 활용: sequor, sequī, secūtus sum

**직설법:**

| 시제 | 1단 | 2단 | 3단 | 1복 | 2복 | 3복 |
|------|-----|-----|-----|-----|-----|-----|
| 현재 | sequor | sequeris | sequitur | sequimur | sequiminī | sequuntur |
| 미완료 | sequēbar | sequēbāris | sequēbātur | sequēbāmur | sequēbāminī | sequēbantur |
| 미래 | sequar | sequēris | sequētur | sequēmur | sequēminī | sequentur |
| 완료 | secūtus sum | secūtus es | secūtus est | secūtī sumus | secūtī estis | secūtī sunt |
| 대과거 | secūtus eram | secūtus erās | secūtus erat | secūtī erāmus | secūtī erātis | secūtī erant |
| 미래완료 | secūtus erō | secūtus eris | secūtus erit | secūtī erimus | secūtī eritis | secūtī erunt |

**접속법:**

| 시제 | 1단 | 3단 | 3복 |
|------|-----|-----|-----|
| 현재 | sequar | sequātur | sequantur |
| 미완료 | sequerer | sequerētur | sequerentur |
| 완료 | secūtus sim | secūtus sit | secūtī sint |
| 대과거 | secūtus essem | secūtus esset | secūtī essent |

### 3.3 이탈동사의 분사와 부정사

이탈동사는 수동 형태임에도 **능동 의미의** 분사와 부정사를 가집니다:

| 형태 | 일반 동사 (수동 의미) | 이탈동사 (능동 의미!) |
|------|---------------------|---------------------|
| 현재 능동 분사 | — | **sequēns** (따르는) |
| 완료 수동 분사 | amātus (사랑받은) | **secūtus** (따른) |
| 미래 능동 분사 | — | **secūtūrus** (따르려고 하는) |
| 현재 부정사 | amārī (사랑받기) | **sequī** (따르기) |
| 완료 부정사 | amātum esse (사랑받았기) | **secūtum esse** (따랐기) |
| 미래 부정사 | — | **secūtūrum esse** (따르게 될 것) |

> **핵심 포인트**: 이탈동사의 PPP는 **능동 의미**입니다: *secūtus* = "따른" ("따라진"이 아님). 이것은 PPP가 수동인 규칙의 유일한 예외입니다.

### 3.4 이탈동사의 특별한 구문

| 동사 | 특별한 점 | 예문 |
|------|----------|------|
| ūtor (+ 탈격) | 대격이 아닌 탈격을 취함 | *Gladiō ūtitur.* (그는 칼을 사용한다.) |
| fruor (+ 탈격) | 탈격을 취함 | *Vītā fruor.* (나는 삶을 즐긴다.) |
| fungor (+ 탈격) | 탈격을 취함 | *Mūnere fungor.* (나는 임무를 수행한다.) |
| potior (+ 탈격/속격) | 탈격 또는 속격을 취함 | *Urbe potītus est.* (그는 도시를 장악했다.) |
| vēscor (+ 탈격) | 탈격을 취함 | *Carne vēscuntur.* (그들은 고기를 먹는다.) |

## 4. 반이탈동사

반이탈동사는 **현재 체계에서는 능동**이지만 **완료 체계에서는 이탈적(수동 형태, 능동 의미)**입니다.

### 4.1 주요 반이탈동사

| 동사 | 현재 (능동) | 완료 (이탈적) | 의미 |
|------|-----------|-------------|------|
| audeō, audēre | audeō, audēs, audet... | ausus sum | 감히 ~하다 |
| gaudeō, gaudēre | gaudeō, gaudēs, gaudet... | gāvīsus sum | 기뻐하다 |
| soleō, solēre | soleō, solēs, solet... | solitus sum | ~하곤 하다 |
| fīdō, fīdere | fīdō, fīdis, fīdit... | fīsus sum | 신뢰하다 (+ 여격) |

### 4.2 활용 예시: audeō

| 시제 | 형태 | 번역 |
|------|------|------|
| 현재 직설법 | audeō | 나는 감히 ~한다 |
| 미완료 직설법 | audēbam | 나는 감히 ~하고 있었다 |
| 미래 직설법 | audēbō | 나는 감히 ~할 것이다 |
| 완료 직설법 | ausus sum | 나는 감히 ~했다 |
| 대과거 직설법 | ausus eram | 나는 감히 ~했었다 |
| 미래완료 직설법 | ausus erō | 나는 감히 ~했을 것이다 |

> 현재 체계는 일반적인 제2활용 동사처럼 보입니다. 완료 체계는 이탈동사처럼 PPP(*ausus*) + *esse*를 사용합니다.

## 5. 명령법

### 5.1 현재 명령법

현재 명령법은 직접 명령의 가장 일반적인 형태입니다.

| | 단수 | 복수 |
|--|------|------|
| **형성** | 순수 어간 (제2 기본형에서 -re를 뺀 것) | 어간 + **-te** |
| 제1 (amō) | **amā** | **amāte** |
| 제2 (moneō) | **monē** | **monēte** |
| 제3 (dūcō) | **dūce** (또는 **dūc**) | **dūcite** |
| 제3-iō (capiō) | **cape** | **capite** |
| 제4 (audiō) | **audī** | **audīte** |
| esse | **es** (있어라!) | **este** |

> **불규칙 제3활용 명령형**: *dīc* (말해라!), *dūc* (이끌어라!), *fac* (해라!), *fer* (운반해라!) — 이들은 단수에서 끝의 -e를 생략합니다.

### 5.2 부정 명령

라틴어에서는 명령법에 단순히 *nōn*을 붙이**지 않습니다**. 대신:

| 구문 | 예문 | 번역 |
|------|------|------|
| **Nōlī** (단수) / **Nōlīte** (복수) + 부정사 | *Nōlī timēre.* | 두려워하지 마라. |
| | *Nōlīte abīre.* | 떠나지 마라. |
| **Nē** + 완료 접속법 (문어체) | *Nē timuerīs.* | 두려워하지 마라. |

### 5.3 미래 명령법

미래 명령법은 드물며, 주로 법률 언어, 격언, 종교 문헌에서 발견됩니다.

| | 단수 | 복수 |
|--|------|------|
| **2인칭** | 어간 + **-tō** | 어간 + **-tōte** |
| **3인칭** | 어간 + **-tō** | 어간 + **-ntō** |

| 형태 | 예시 | 번역 |
|------|------|------|
| 2단 | *amātō* | 너는 (장래에) 사랑해야 한다 |
| 3단 | *amātō* | 그/그녀는 사랑해야 한다 |
| 2복 | *amātōte* | 너희는 사랑해야 한다 |
| 3복 | *amantō* | 그들은 사랑해야 한다 |

법률/격언에서의 예:
- *Mementō morī.* — 죽음을 기억하라. (*mementō* = *meminī*의 미래 명령법)
- *Iūs dīcitō.* — 그가 법을 선포해야 한다.

## 7. 종합 참고 도표

### 7.1 시제 표지 / 법 표지 빠른 찾기

```
┌──────────────────────────────────────────────────────────────────┐
│   직설법                                                         │
│   현재:     어간 + 인칭 어미                                      │
│   미완료:   어간 + -bā- (제1·2) / -ēbā- (제3·4)                  │
│   미래:     어간 + -bi- (제1·2) / -a-/-ē- (제3·4)                │
│   완료:     완료 어간 + -ī/-istī/-it/-imus/-istis/-ērunt         │
│   대과거:   완료 어간 + -era- + 어미                              │
│   미래완료: 완료 어간 + -eri- + 어미                              │
│                                                                  │
│   접속법                                                         │
│   현재:     모음 변화 (ē/eā/ā/iā) + 어미                         │
│   미완료:   부정사 + 어미                                         │
│   완료:     완료 어간 + -erī- + 어미                              │
│   대과거:   완료 어간 + -issē- + 어미                             │
│                                                                  │
│   명령법                                                         │
│   현재:     단수 = 어간; 복수 = 어간 + -te                        │
│   미래:     어간 + -tō (단수) / -tōte, -ntō (복수)               │
└──────────────────────────────────────────────────────────────────┘
```

### 7.2 모든 능동 인칭 어미

| 시제 체계 | 1단 | 2단 | 3단 | 1복 | 2복 | 3복 |
|----------|-----|-----|-----|-----|-----|-----|
| 현재 체계 | -ō/-m | -s | -t | -mus | -tis | -nt |
| 완료 직설법 | -ī | -istī | -it | -imus | -istis | -ērunt |
| 접속법 | -m | -s | -t | -mus | -tis | -nt |

### 7.3 모든 수동 인칭 어미

| | 1단 | 2단 | 3단 | 1복 | 2복 | 3복 |
|--|-----|-----|-----|-----|-----|-----|
| 현재 체계 | -or/-r | -ris | -tur | -mur | -minī | -ntur |
| 완료 체계 | PPP + sum | PPP + es | PPP + est | PPP + sumus | PPP + estis | PPP + sunt |

### 7.4 동사의 다섯 가지 속성

모든 정형 라틴어 동사 형태는 다섯 가지 속성을 부호화합니다:

```
┌──────────────────────────────────────────────────────────────┐
│   1. 인칭:     제1, 제2, 제3                                  │
│   2. 수:       단수, 복수                                     │
│   3. 시제:     현재, 미완료, 미래,                             │
│                완료, 대과거, 미래완료                           │
│   4. 태:       능동, 수동                                     │
│   5. 법:       직설법, 접속법, 명령법                          │
│                                                              │
│   동사당 가능한 형태의 총수:                                   │
│   3 × 2 × 6 × 2 × 2 = 144 정형 형태                         │
│   + 명령법 (4) + 부정사 (6) + 분사 (3-4)                      │
│   + 동명사 (4격) + 동형사 + 목적분사 (2)                       │
│   ≈ 동사당 165개 이상의 고유 형태                              │
└──────────────────────────────────────────────────────────────┘
```

### 연습 E: 영어로 번역

3. Nōlī mē sequī.
4. Tam fortēs fuērunt ut numquam cēderent.
5. Imperātor mīlitēs hortātus est ut fortiter pugnārent.

### 연습 F: 라틴어로 번역

1. 병사들이 출발한 후, 적이 공격했다. (*cum* + 대과거 접속법 사용; *proficīscor*; *oppugnō*)
2. 두려워하지 마라! (단수; *nōlī* + 부정사 사용)
3. 나는 감히 이것을 말하지 않았을 것이다. (*audeō*; 반사실적 대과거 접속법 사용)
4. 군대를 이끌어라, 장군이여! (단수 명령법; *dūcō*; *cōpiae, -ārum*; *imperātor*)
5. 카이사르를 따랐던 자들은 보상을 받았다. (*sequor*; *praemium, -iī*)
