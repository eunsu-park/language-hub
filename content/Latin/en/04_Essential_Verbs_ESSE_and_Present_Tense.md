## 1. The Verb ESSE (To Be)

### 1.1 Why ESSE Is Special

The verb *esse* (to be) is the most important and most irregular verb in Latin — just as "to be" is irregular in English (am, is, are), German (bin, bist, ist), Korean (이다), and nearly every Indo-European language.

```
┌─────────────────────────────────────────────────────────────────┐
│                    ESSE — To Be                                  │
│              Present Tense (Tempus Praesens)                     │
│                                                                 │
│   Person      Singular          Plural                          │
│   ──────      ─────────         ──────                          │
│   1st         sum               sumus                           │
│               /sʊm/             /ˈsʊ.mʊs/                      │
│               "I am"            "we are"                        │
│                                                                 │
│   2nd         es                estis                           │
│               /ɛs/              /ˈɛs.tɪs/                       │
│               "you are"         "you (pl.) are"                 │
│                                                                 │
│   3rd         est               sunt                            │
│               /ɛst/             /sʊnt/                           │
│               "he/she/it is"    "they are"                      │
│                                                                 │
│   Infinitive: esse /ˈɛs.sɛ/ "to be"                           │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Comparison with Other Languages

| Person | Latin | English | Spanish | Korean |
|--------|-------|---------|---------|--------|
| 1sg | sum | I am | soy | (나는) ~이다 |
| 2sg | es | you are | eres | (너는) ~이다 |
| 3sg | est | he/she/it is | es | (그는) ~이다 |
| 1pl | sumus | we are | somos | (우리는) ~이다 |
| 2pl | estis | you (pl.) are | sois | (너희는) ~이다 |
| 3pl | sunt | they are | son | (그들은) ~이다 |

### 1.3 ESSE in Sentences

```
Sum discipulus.              I am a student.
Es amīcus meus.             You are my friend.
Est bonus magister.          He is a good teacher.
Sumus Rōmānī.               We are Romans.
Estis laetī.                 You (pl.) are happy.
Sunt in Italiā.              They are in Italy.
```

> **Note**: Latin typically omits subject pronouns (ego, tū, etc.) because the verb ending tells you who the subject is — similar to Korean verb endings and Spanish. You only include the pronoun for emphasis or contrast.

## 2. Predicate Nominatives and Adjectives

### 2.1 Predicate Nominative

When *esse* links the subject to a noun, that noun is in the **nominative** case (not the accusative). This noun is called the **predicate nominative**.

```
┌─────────────────────────────────────────────────────────────────┐
│              Predicate Nominative Rule                            │
│                                                                 │
│   Subject (Nom.) + esse + Noun (Nom.)                           │
│                                                                 │
│   Mārcus est poēta.        Marcus is a poet.                   │
│   (nom.)     (nom.)        NOT *poētam (acc.)                  │
│                                                                 │
│   Iūlia est fēmina.        Julia is a woman.                   │
│   (nom.)    (nom.)                                              │
│                                                                 │
│   Puerī sunt discipulī.    The boys are students.              │
│   (nom.pl.) (nom.pl.)                                          │
│                                                                 │
│   ※ Think of it this way: esse is like an equals sign (=).     │
│     Both sides must be in the same case (nominative).           │
│     Mārcus = poēta → both nominative                           │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 Predicate Adjective

When *esse* links the subject to an adjective, that adjective must **agree** with the subject in gender, number, and case.

```
Predicate adjective agreement:

  Mārcus est bonus.           Marcus is good.
  (masc. sg. nom.)  (masc. sg. nom.)

  Iūlia est bona.             Julia is good.
  (fem. sg. nom.)   (fem. sg. nom.)

  Bellum est magnum.           The war is great.
  (neut. sg. nom.)  (neut. sg. nom.)

  Puellae sunt laetae.        The girls are happy.
  (fem. pl. nom.)   (fem. pl. nom.)

  Puerī sunt laetī.           The boys are happy.
  (masc. pl. nom.)  (masc. pl. nom.)

  Dōna sunt pulchra.          The gifts are beautiful.
  (neut. pl. nom.)  (neut. pl. nom.)
```

### 2.3 Common Adjectives with ESSE

| Masculine | Feminine | Neuter | Meaning |
|-----------|----------|--------|---------|
| bonus | bona | bonum | good |
| malus | mala | malum | bad |
| magnus | magna | magnum | great, large |
| parvus | parva | parvum | small |
| pulcher | pulchra | pulchrum | beautiful |
| laetus | laeta | laetum | happy |
| miser | misera | miserum | wretched, unhappy |
| multus | multa | multum | much, many |
| Rōmānus | Rōmāna | Rōmānum | Roman |
| prīmus | prīma | prīmum | first |

## 3. Introduction to Latin Verb Conjugation

### 3.1 The Four Conjugations

Latin regular verbs are organized into **four conjugations**, identified by the vowel at the end of the present stem:

```
┌─────────────────────────────────────────────────────────────────┐
│                  The Four Latin Conjugations                     │
│                                                                 │
│   Conjugation   Stem vowel   Infinitive ending   Example        │
│   ───────────   ──────────   ─────────────────   ───────        │
│   1st           -ā-          -āre                amāre (love)   │
│   2nd           -ē-          -ēre                monēre (warn)  │
│   3rd           -ĕ-          -ĕre                dūcere (lead)  │
│   4th           -ī-          -īre                audīre (hear)  │
│                                                                 │
│   In this lesson we cover the 1st and 2nd conjugations.         │
│   The 3rd and 4th will come in later lessons.                    │
│                                                                 │
│   Dictionary form (4 principal parts):                          │
│   amō, amāre, amāvī, amātum                                   │
│   ↑    ↑       ↑       ↑                                        │
│   1sg  infin.  perf.   supine                                   │
│   pres. pres.  1sg     (for now, just learn the first two)     │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Person and Number

Like Korean honorific/plain forms and Spanish conjugations, Latin verbs change their endings based on **person** and **number**:

| Person | Singular | Plural |
|--------|----------|--------|
| 1st (I / we) | -ō / -m | -mus |
| 2nd (you / you all) | -s | -tis |
| 3rd (he,she,it / they) | -t | -nt |

> These **personal endings** are the same across most conjugations and tenses. Memorize them well — they are the key to reading any Latin verb.

### 3.3 How to Find the Present Stem

```
To conjugate in the present tense:
1. Take the infinitive: amāre
2. Remove the -re ending: amā-  (this is the present stem)
3. Add the personal endings

Infinitive     Stem
amāre     →    amā-     (1st conjugation)
monēre    →    monē-    (2nd conjugation)
```

## 4. First Conjugation Present Active

### 4.1 Full Paradigm: amō, amāre (to love)

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | am**ō** /ˈa.moː/ | amā**mus** /a.ˈmaː.mʊs/ |
| 2nd | amā**s** /ˈa.maːs/ | amā**tis** /a.ˈmaː.tɪs/ |
| 3rd | ama**t** /ˈa.mat/ | ama**nt** /ˈa.mant/ |

```
┌─────────────────────────────────────────────────────────────────┐
│        First Conjugation Present Active Formation                │
│                                                                 │
│   Stem: amā- (from amāre, drop -re)                            │
│                                                                 │
│              Stem + Ending = Form                                │
│   1sg:       amā + ō = amō     (ā absorbed into ō)             │
│   2sg:       amā + s = amās                                     │
│   3sg:       amā + t = amat    (ā shortens before final -t)    │
│   1pl:       amā + mus = amāmus                                 │
│   2pl:       amā + tis = amātis                                 │
│   3pl:       amā + nt = amant  (ā shortens before -nt)         │
│                                                                 │
│   ※ The 1sg ending -ō "swallows" the stem vowel -ā.           │
│   ※ The stem vowel shortens before -t and -nt.                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Common First Conjugation Verbs

| Latin | Infinitive | Meaning |
|-------|-----------|---------|
| amō | amāre | to love |
| vocō | vocāre | to call |
| laudō | laudāre | to praise |
| labōrō | labōrāre | to work, toil |
| ambulō | ambulāre | to walk |
| portō | portāre | to carry |
| nārrō | nārrāre | to tell, narrate |
| spectō | spectāre | to watch, look at |
| dō | dare* | to give |
| stō | stāre* | to stand |
| nāvigō | nāvigāre | to sail |
| pugnō | pugnāre | to fight |
| rogō | rogāre | to ask |
| servō | servāre | to save, preserve |
| superō | superāre | to overcome, surpass |

> ***dō, dare** and **stō, stāre** have a short -a- in the infinitive (not -āre) but otherwise conjugate normally in the present tense: dō, dās, dat, damus, datis, dant.

### 4.3 First Conjugation Examples

```
Amō patriam.                I love my country.
Puella rosās portat.        The girl carries roses.
Nautae ad Italiam nāvigant. The sailors sail to Italy.
Agricolae in agrīs labōrant. The farmers work in the fields.
Poēta fābulam nārrat.       The poet tells a story.
Puerōs et puellās vocāmus.  We call the boys and girls.
```

## 5. Second Conjugation Present Active

### 5.1 Full Paradigm: moneō, monēre (to warn, advise)

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | mon**eō** /ˈmɔ.nɛ.oː/ | mon**ēmus** /mɔ.ˈneː.mʊs/ |
| 2nd | mon**ēs** /ˈmɔ.neːs/ | mon**ētis** /mɔ.ˈneː.tɪs/ |
| 3rd | mon**et** /ˈmɔ.nɛt/ | mon**ent** /ˈmɔ.nɛnt/ |

```
┌─────────────────────────────────────────────────────────────────┐
│        Second Conjugation Present Active Formation               │
│                                                                 │
│   Stem: monē- (from monēre, drop -re)                          │
│                                                                 │
│              Stem + Ending = Form                                │
│   1sg:       monē + ō = moneō    (ē shortens before ō)        │
│   2sg:       monē + s = monēs                                   │
│   3sg:       monē + t = monet    (ē shortens before -t)        │
│   1pl:       monē + mus = monēmus                               │
│   2pl:       monē + tis = monētis                               │
│   3pl:       monē + nt = monent  (ē shortens before -nt)       │
│                                                                 │
│   ※ Same personal endings as 1st conjugation!                  │
│   ※ The stem vowel -ē- shortens before -ō, -t, and -nt.       │
│   ※ The key difference from 1st conj.: stem vowel ē not ā.    │
└─────────────────────────────────────────────────────────────────┘
```

| Latin | Infinitive | Meaning |
|-------|-----------|---------|
| moneō | monēre | to warn, advise |
| videō | vidēre | to see |
| habeō | habēre | to have, hold |
| doceō | docēre | to teach |
| moveō | movēre | to move |
| teneō | tenēre | to hold, keep |
| timeō | timēre | to fear |
| dēbeō | dēbēre | to owe, ought |
| maneō | manēre | to remain, stay |
| respondeō | respondēre | to answer, respond |
| rīdeō | rīdēre | to laugh |

### 5.3 Second Conjugation Examples

```
Magister puerōs docet.       The teacher teaches the boys.
Perīculum vidēmus.           We see the danger.
Pecūniam nōn habent.         They do not have money.
Puerī magistrum timent.      The boys fear the teacher.
Servī in oppidō manent.      The slaves remain in the town.
Dēbēmus patriam servāre.     We ought to save our country.
```

### 5.4 Comparing First and Second Conjugation

| Form | 1st (amāre) | 2nd (monēre) |
|------|-------------|--------------|
| 1sg | am**ō** | mone**ō** |
| 2sg | am**ās** | mon**ēs** |
| 3sg | am**at** | mon**et** |
| 1pl | am**āmus** | mon**ēmus** |
| 2pl | am**ātis** | mon**ētis** |
| 3pl | am**ant** | mon**ent** |
| Infinitive | am**āre** | mon**ēre** |

> The pattern is clear: replace **-ā-** with **-ē-** throughout. The personal endings (-ō, -s, -t, -mus, -tis, -nt) are identical.

## 6. Subject-Verb Agreement

### 6.1 The Rule

In Latin, the verb must agree with its subject in **person** and **number**.

```
┌─────────────────────────────────────────────────────────────────┐
│              Subject-Verb Agreement                               │
│                                                                 │
│   Singular subject → singular verb                               │
│   Puella cantat.        The girl sings.                          │
│   Servus labōrat.       The slave works.                        │
│                                                                 │
│   Plural subject → plural verb                                   │
│   Puellae cantant.      The girls sing.                         │
│   Servī labōrant.       The slaves work.                        │
│                                                                 │
│   Compound subjects → plural verb                                │
│   Mārcus et Iūlia ambulant.   Marcus and Julia walk.            │
│   Puer et puella rīdent.      The boy and girl laugh.           │
│                                                                 │
│   Dropped subject — the verb ending tells you who:               │
│   Labōrō.         I work.        (1sg: -ō → "I")              │
│   Labōrās.        You work.      (2sg: -s → "you")            │
│   Labōrat.        He/she works.  (3sg: -t → "he/she/it")      │
│   Labōrāmus.     We work.       (1pl: -mus → "we")            │
│   Labōrātis.     You all work.  (2pl: -tis → "you pl.")       │
│   Labōrant.      They work.     (3pl: -nt → "they")           │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 Important Notes

1. **Neuter plural subjects** take a **singular** verb in some contexts (especially with *esse*):
   - *Arma sunt bona.* — "The weapons are good." (plural verb with plural subject)
   - But neuter plural subjects with verbs of action sometimes appear with singular verbs in poetry.

2. **Collective nouns** (like *populus*, people) take a **singular** verb:
   - *Populus Rōmānus pugnat.* — "The Roman people fights."

## 7. Basic Latin Sentence Structure

### 7.1 SOV Word Order

```
┌─────────────────────────────────────────────────────────────────┐
│           Latin Default Word Order: S — O — V                    │
│                                                                 │
│   Latin default:   Mārcus    rosam     portat.                  │
│                    (S)       (O)       (V)                       │
│                                                                 │
│   Korean parallel: 마르쿠스가  장미를    운반한다.                    │
│                    (S)       (O)       (V)                       │
│                                                                 │
│   English:         Marcus    carries   a rose.                  │
│                    (S)       (V)       (O)                       │
│                                                                 │
│   However, Latin word order is FLEXIBLE because case            │
│   endings indicate grammatical function.                         │
│                                                                 │
│   Typical patterns:                                              │
│   • SOV — neutral, default                                      │
│   • Verb at end — most common in prose                          │
│   • Verb first — emphasis on action, or in questions            │
│   • Object first — emphasis on the object                       │
│   • Genitive before its noun: populī Rōmānī arma               │
│   • Adjective after its noun (but can precede for emphasis)     │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 Common Sentence Patterns

```
Pattern 1: Subject + Intransitive Verb
  Puella ambulat.              The girl walks.
  Nautae nāvigant.             The sailors sail.

Pattern 2: Subject + esse + Predicate
  Mārcus est poēta.            Marcus is a poet.
  Puellae sunt laetae.         The girls are happy.

Pattern 3: Subject + Direct Object + Verb
  Puer rosam videt.            The boy sees the rose.
  Servī terram arant.          The slaves plow the land.

Pattern 4: Subject + Indirect Object + Direct Object + Verb
  Mārcus puellae rosam dat.    Marcus gives a rose to the girl.
  Magister puerīs librōs dat.  The teacher gives books to the boys.

Pattern 5: Subject + Prepositional Phrase + Verb
  Agricola in agrō labōrat.    The farmer works in the field.
  Cum amīcīs ambulāmus.       We walk with friends.
```

### 7.3 Word Order and Emphasis

```
Neutral:     Mārcus rosam portat.         Marcus carries a rose.
Emphasis V:  Portat Mārcus rosam.          Marcus CARRIES a rose.
Emphasis O:  Rosam Mārcus portat.          It's a ROSE Marcus carries.
Emphasis S:  Rosam portat Mārcus.          It's MARCUS who carries the rose.

※ The first and last positions in a Latin sentence carry the most
  emphasis. The verb often occupies the final position.
```

## 8. Negation with NON

### 8.1 Simple Negation

To negate a Latin sentence, place **nōn** directly before the verb.

```
┌─────────────────────────────────────────────────────────────────┐
│                    Negation with NŌN                             │
│                                                                 │
│   Affirmative:              Negative:                           │
│   Puella cantat.            Puella nōn cantat.                  │
│   The girl sings.           The girl does not sing.             │
│                                                                 │
│   Sum laetus.               Nōn sum laetus.                     │
│   I am happy.               I am not happy.                     │
│                                                                 │
│   Pecūniam habēmus.        Pecūniam nōn habēmus.              │
│   We have money.            We do not have money.               │
│                                                                 │
│   ※ NŌN goes directly before the verb (or the word negated).  │
│   ※ There is no equivalent of English "do not" — just nōn.    │
│   ※ Compare Korean: ~하지 않는다 (verb + 않다)                    │
│     Latin is simpler: nōn + verb                                │
└─────────────────────────────────────────────────────────────────┘
```

### 8.2 Negation Examples

| Affirmative | Negative |
|------------|----------|
| Amō. (I love.) | Nōn amō. (I do not love.) |
| Est bonus. (He is good.) | Nōn est bonus. (He is not good.) |
| Vidēmus. (We see.) | Nōn vidēmus. (We do not see.) |
| Timent. (They fear.) | Nōn timent. (They do not fear.) |
| Magister docet. (The teacher teaches.) | Magister nōn docet. (The teacher does not teach.) |

### 8.3 Other Negative Words (Preview)

| Latin | Meaning |
|-------|---------|
| nōn | not |
| numquam / nunquam | never |
| nihil / nīl | nothing |
| nēmō | nobody |
| neque / nec | and not, nor |
| nōndum | not yet |
| nōn iam | no longer |

### Exercise 7: Predicate Nominative or Accusative?


<details>
<summary>Click to reveal answers</summary>


```
┌─────────────────────────────────────────────────────────────────┐
│           Verbs: ESSE and Present Tense Quick Reference           │
│                                                                 │
│   ESSE (to be):                                                  │
│     sum, es, est, sumus, estis, sunt                            │
│     Takes NOMINATIVE (not accusative) for complement            │
│                                                                 │
│   1ST CONJUGATION (-āre):          2ND CONJUGATION (-ēre):      │
│     -ō     -āmus                    -eō    -ēmus                │
│     -ās    -ātis                    -ēs    -ētis                │
│     -at    -ant                     -et    -ent                  │
│                                                                 │
│   PERSONAL ENDINGS (same for both):                              │
│     1sg: -ō/-m     1pl: -mus                                    │
│     2sg: -s        2pl: -tis                                    │
│     3sg: -t        3pl: -nt                                     │
│                                                                 │
│   NEGATION: nōn + verb                                          │
│     Nōn amō. = I do not love.                                   │
│     Nōn est bonus. = He is not good.                            │
│                                                                 │
│   WORD ORDER: SOV default (flexible)                             │
│     S + O + V:  Puella rosam amat.                              │
│     S + V:      Puella cantat.                                  │
│     S + esse + Nom: Puella est bona.                            │
│                                                                 │
│   AGREEMENT:                                                     │
│     Subject-verb: same person and number                        │
│     Predicate adj: same gender, number, case as subject         │
```
