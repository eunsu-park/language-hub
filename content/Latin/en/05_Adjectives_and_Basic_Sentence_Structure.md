## 1. First/Second Declension Adjectives

### 1.1 Overview

Latin adjectives must **agree** with the nouns they modify in three ways: **gender**, **number**, and **case**. The most common adjective type uses First Declension endings for the feminine and Second Declension endings for the masculine and neuter.

```
┌─────────────────────────────────────────────────────────────────┐
│         First/Second Declension Adjective Pattern                │
│                                                                 │
│   Dictionary entry: bonus, -a, -um (good)                       │
│                     ↑       ↑   ↑                               │
│                     masc.  fem. neut.                            │
│                     (2nd)  (1st) (2nd)                           │
│                                                                 │
│   Masculine forms → follow 2nd declension (-us/-er pattern)     │
│   Feminine forms  → follow 1st declension (-a pattern)          │
│   Neuter forms    → follow 2nd declension (-um pattern)         │
│                                                                 │
│   You already know all these endings from Lessons 03!           │
│   The adjective simply picks the right set of endings           │
│   based on the gender of the noun it modifies.                  │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Full Paradigm: bonus, -a, -um (good)

**Masculine (2nd Declension)**

| Case | Singular | Plural |
|------|----------|--------|
| Nom. | bon**us** | bon**ī** |
| Gen. | bon**ī** | bon**ōrum** |
| Dat. | bon**ō** | bon**īs** |
| Acc. | bon**um** | bon**ōs** |
| Abl. | bon**ō** | bon**īs** |
| Voc. | bon**e** | bon**ī** |

**Feminine (1st Declension)**

| Case | Singular | Plural |
|------|----------|--------|
| Nom. | bon**a** | bon**ae** |
| Gen. | bon**ae** | bon**ārum** |
| Dat. | bon**ae** | bon**īs** |
| Acc. | bon**am** | bon**ās** |
| Abl. | bon**ā** | bon**īs** |
| Voc. | bon**a** | bon**ae** |

**Neuter (2nd Declension)**

| Case | Singular | Plural |
|------|----------|--------|
| Nom. | bon**um** | bon**a** |
| Gen. | bon**ī** | bon**ōrum** |
| Dat. | bon**ō** | bon**īs** |
| Acc. | bon**um** | bon**a** |
| Abl. | bon**ō** | bon**īs** |
| Voc. | bon**um** | bon**a** |

### 1.3 The -er Type: pulcher, pulchra, pulchrum (beautiful)

Some adjectives have a masculine nominative singular in **-er**, just like *puer* and *ager* nouns. There are two subtypes:

**Type A — drops the -e- (most common):**

| | Masculine | Feminine | Neuter |
|---|-----------|----------|--------|
| Nom. sg. | pulch**er** | pulchr**a** | pulchr**um** |
| Gen. sg. | pulchr**ī** | pulchr**ae** | pulchr**ī** |
| Dat. sg. | pulchr**ō** | pulchr**ae** | pulchr**ō** |
| Acc. sg. | pulchr**um** | pulchr**am** | pulchr**um** |
| Abl. sg. | pulchr**ō** | pulchr**ā** | pulchr**ō** |

**Type B — keeps the -e- (only a few):**

| | Masculine | Feminine | Neuter |
|---|-----------|----------|--------|
| Nom. sg. | līber | līber**a** | līber**um** |
| Gen. sg. | līber**ī** | līber**ae** | līber**ī** |

```
┌─────────────────────────────────────────────────────────────────┐
│   How to tell? Check the FEMININE form in the dictionary.        │
│                                                                 │
│   Drops -e-:  pulcher, pulchra, pulchrum  (stem = pulchr-)     │
│               miser, misera, miserum      (stem = miser-)       │
│               noster, nostra, nostrum     (stem = nostr-)       │
│                                                                 │
│   Keeps -e-:  līber, lībera, līberum     (stem = līber-)       │
│               tener, tenera, tenerum      (stem = tener-)       │
│                                                                 │
│   ※ The -er adjectives that KEEP -e- are rare. Most drop it.   │
└─────────────────────────────────────────────────────────────────┘
```

## 2. Adjective-Noun Agreement

### 2.1 The Golden Rule

```
┌─────────────────────────────────────────────────────────────────┐
│          ADJECTIVE-NOUN AGREEMENT: The Golden Rule               │
│                                                                 │
│   An adjective must agree with its noun in:                      │
│     1. GENDER  (masculine, feminine, or neuter)                  │
│     2. NUMBER  (singular or plural)                              │
│     3. CASE    (nominative, genitive, dative, etc.)             │
│                                                                 │
│   BUT the adjective does NOT need to be in the same             │
│   DECLENSION as the noun!                                        │
│                                                                 │
│   Example:                                                       │
│     agricola bonus    (good farmer)                              │
│     ↑         ↑                                                  │
│     1st decl. 2nd decl.  ← different declensions!               │
│     masc.     masc.      ← same gender ✓                       │
│     sg.       sg.        ← same number ✓                       │
│     nom.      nom.       ← same case ✓                         │
│                                                                 │
│   The ENDINGS may look different, but gender/number/case        │
│   must match. Adjective follows the noun's GENDER, not its     │
│   declension.                                                   │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 Agreement Examples by Gender

```
MASCULINE nouns → masculine adjective forms:
  servus bonus         good slave        (2nd + 2nd)
  puer magnus          great boy         (2nd + 2nd)
  agricola bonus       good farmer       (1st noun + 2nd adj!)
  nauta miser          wretched sailor   (1st noun + 2nd adj!)
  poēta Rōmānus       Roman poet        (1st noun + 2nd adj!)

FEMININE nouns → feminine adjective forms:
  puella bona          good girl         (1st + 1st)
  terra magna          great land        (1st + 1st)
  via longa            long road         (1st + 1st)
  fēmina pulchra       beautiful woman  (1st + 1st)
  patria nostra        our country       (1st + 1st)

NEUTER nouns → neuter adjective forms:
  bellum magnum        great war         (2nd + 2nd)
  dōnum bonum          good gift         (2nd + 2nd)
  templum antīquum     ancient temple    (2nd + 2nd)
  perīculum parvum     small danger      (2nd + 2nd)
  verbum novum         new word          (2nd + 2nd)
```

### 2.3 Agreement Across Cases

The adjective follows the noun through all its case changes:

```
Nominative:  puella bona cantat.       The good girl sings.
Genitive:    fīlius puellae bonae      The son of the good girl
Dative:      puellae bonae rosam dat.  He gives a rose to the good girl.
Accusative:  puellam bonam videt.      He sees the good girl.
Ablative:    cum puellā bonā ambulat.  He walks with the good girl.
Vocative:    Ō puella bona!            O good girl!
```

### 2.4 Tricky Cases: 1st Declension Masculine Nouns

When a noun is 1st declension but masculine (like *agricola*), the adjective must use **masculine** (2nd declension) forms:

```
┌─────────────────────────────────────────────────────────────────┐
│   agricola bonus (the good farmer) — NOT *agricola bona        │
│                                                                 │
│   Nom:  agricola bonus         agricolae bonī                  │
│   Gen:  agricolae bonī         agricolārum bonōrum             │
│   Dat:  agricolae bonō         agricolīs bonīs                │
│   Acc:  agricolam bonum        agricolās bonōs                 │
│   Abl:  agricolā bonō          agricolīs bonīs                │
│                                                                 │
│   ※ The noun endings look feminine (-a, -ae, -am, -ā)          │
│     but the adjective endings are masculine (-us, -ī, -um, -ō) │
│     because agricola is a MASCULINE noun.                       │
│                                                                 │
│   Similarly:                                                    │
│     nauta bonus    (good sailor)                                │
│     poēta magnus   (great poet)                                 │
│     pīrāta malus   (bad pirate)                                │
└─────────────────────────────────────────────────────────────────┘
```

> **For Korean speakers**: This is a concept without a direct Korean parallel. In Korean, adjectives (관형사/형용사) don't change form based on the noun they modify. In Latin, you must always mentally check: "What gender is this noun? What case is it in? What number?" and then use the matching adjective form.

## 3. Adjectives as Substantives

### 3.1 Using Adjectives as Nouns

In Latin, adjectives can stand alone as nouns (called "substantive" use). The gender tells you what kind of entity is meant:

```
┌─────────────────────────────────────────────────────────────────┐
│              Substantive Adjectives                               │
│                                                                 │
│   Masculine → refers to men/male persons:                       │
│     bonī = "the good (men)" / "good people"                    │
│     malī = "the bad (men)" / "evil people"                     │
│     multī = "many (people)"                                     │
│     Rōmānī = "the Romans" (Roman men/people)                   │
│                                                                 │
│   Feminine → refers to women or can be abstract:                │
│     bonae = "the good (women)"                                  │
│     Rōmānae = "the Roman women"                                │
│                                                                 │
│   Neuter → refers to things, concepts, abstractions:            │
│     bonum = "the good" / "a good thing"                         │
│     bona = "good things" / "goods, possessions"                 │
│     malum = "evil" / "a bad thing"                              │
│     magna = "great things"                                      │
│     multa = "many things"                                       │
│     vera = "true things" / "the truth"                          │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Examples in Sentences

```
Bonī patriam amant.
"The good (people/men) love their country."

Multa vidēmus.
"We see many things."

Rōmānī Graecōs superant.
"The Romans overcome the Greeks."

Magna est vēritās.
"Truth is a great thing." / "The truth is great."

Bona et mala in vītā sunt.
"Good things and bad things are in life."
```

## 4. Predicate vs Attributive Position

### 4.1 Two Ways to Use Adjectives

Latin adjectives appear in two main roles:

```
┌─────────────────────────────────────────────────────────────────┐
│        ATTRIBUTIVE vs PREDICATE Adjectives                       │
│                                                                 │
│   ATTRIBUTIVE — directly modifies the noun (no linking verb):   │
│     Puella bona cantat.       The good girl sings.              │
│     Bonus agricola labōrat.   The good farmer works.            │
│     In magnō perīculō sumus. We are in great danger.           │
│                                                                 │
│   PREDICATE — linked to the subject via esse (or similar):      │
│     Puella est bona.          The girl is good.                 │
│     Perīculum est magnum.     The danger is great.              │
│     Agrī sunt magnī.          The fields are large.             │
│                                                                 │
│   Key difference:                                                │
│     Attributive: adjective + noun together (no esse)            │
│     Predicate: subject + esse + adjective                        │
│                                                                 │
│   Compare English:                                               │
│     "The tall man walks." (attributive — "tall" modifies "man") │
│     "The man is tall." (predicate — "tall" linked by "is")     │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Position of Attributive Adjectives

In Latin, attributive adjectives can appear **before** or **after** their noun. The choice affects emphasis:

```
Normal position — adjective AFTER noun:
  puella bona       "the good girl"
  vir magnus        "the great man"
  via longa         "the long road"

Emphatic position — adjective BEFORE noun:
  bona puella       "the GOOD girl" (emphasizes the quality)
  magnus vir        "a GREAT man"
  magna cum laude   "with GREAT praise"

※ Certain adjectives commonly precede their nouns:
  magnus (great), bonus (good), malus (bad), multus (many),
  parvus (small), possessives (meus, tuus, noster, vester)
  ordinals (prīmus, secundus, tertius)
```

### 4.3 Separated Adjective and Noun

Because Latin word order is flexible, an adjective and its noun can be separated by other words. The case endings tell you which adjective goes with which noun:

```
Magnam agricola terram arat.
magnus → magnam (acc. fem. sg.) agrees with terram (acc. fem. sg.)
"The farmer plows the large land."

Bonus fīliae dominus dōnum dat.
bonus → bonus (nom. masc. sg.) agrees with dominus (nom. masc. sg.)
"The good master gives a gift to his daughter."

※ Read the endings carefully to match adjectives with their nouns!
```

## 5. Word Order in Latin

### 5.1 Default SOV and Variations

```
┌─────────────────────────────────────────────────────────────────┐
│              Latin Word Order Principles                          │
│                                                                 │
│   1. DEFAULT ORDER: Subject — Object — Verb (SOV)               │
│      Puella rosam amat.    The girl loves a rose.               │
│      Same as Korean and Japanese default order.                  │
│                                                                 │
│   2. VERB tends to come LAST in the clause                      │
│      This is the most consistent feature of Latin prose.        │
│                                                                 │
│   3. MODIFIERS tend to PRECEDE what they modify:                │
│      Genitive before noun:  populī Rōmānī lēgēs               │
│                              ("of the Roman people the laws")   │
│      Adjective before noun: bonus vir (good man)                │
│      BUT adjective after noun is also very common               │
│                                                                 │
│   4. FIRST and LAST positions in a sentence carry EMPHASIS      │
│      Rosam puella amat.     (emphasis on rose — IT'S A ROSE)   │
│      Puella amat rosam.     (emphasis on amat — she LOVES it)  │
│                                                                 │
│   5. PREPOSITIONS come BEFORE their objects (like English)      │
│      in agrō, cum amīcō, ad Italiam                           │
│      Exception: -cum with pronouns → mēcum, tēcum, nōbīscum  │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 Practical Word Order Guidelines

| Element | Typical Position | Example |
|---------|-----------------|---------|
| Subject | Beginning | **Mārcus** rosam puellae dat. |
| Direct Object | Before verb | Mārcus **rosam** puellae dat. |
| Indirect Object | Near verb | Mārcus rosam **puellae** dat. |
| Verb | End | Mārcus rosam puellae **dat**. |
| Adjective | After noun (or before for emphasis) | puella **bona** / **bona** puella |
| Genitive | Before or after its noun | **populī** lēgēs / lēgēs **populī** |
| Prepositional phrase | Near the word it modifies | **in agrō** labōrat |
| Negation (nōn) | Directly before verb | **nōn** amat |
| Conjunctions | Between clauses/words | Mārcus **et** Iūlia |

### 5.3 Word Order Comparison

```
Latin (SOV default):     Agricola terram arat.
                         농부가   땅을   간다.
Korean (SOV):            농부가   땅을   간다.
English (SVO):           The farmer plows the land.
German (V2/SOV):         Der Bauer pflügt das Land.
Japanese (SOV):          農夫が   土地を  耕す。

Latin and Korean share the SOV default, but Latin can freely
rearrange because of case endings:
  Terram agricola arat.   ← still clear (terram = acc.)
  Arat agricola terram.   ← still clear (arat = verb)

Korean has similar flexibility with particles:
  땅을 농부가 간다.          ← still clear (을 = object marker)
```

## 6. Coordinating Conjunctions

### 6.1 The Main Coordinating Conjunctions

Latin has several words for connecting words, phrases, and clauses:

```
┌─────────────────────────────────────────────────────────────────┐
│              Coordinating Conjunctions                            │
│                                                                 │
│   et        /ɛt/          "and"                                 │
│   -que      /kwɛ/         "and" (enclitic, attached to 2nd     │
│                            word)                                │
│   atque/ac  /ˈat.kwɛ/     "and" (emphatic: "and also")         │
│                                                                 │
│   sed       /sɛd/         "but"                                 │
│   at        /at/          "but" (milder contrast)               │
│                                                                 │
│   aut       /au̯t/         "or" (exclusive: one or the other)   │
│   vel       /wɛl/         "or" (inclusive: either or both)      │
│   -ve       /wɛ/          "or" (enclitic)                       │
│                                                                 │
│   neque/nec /ˈnɛ.kwɛ/    "and not" / "nor" / "neither"        │
│                                                                 │
│   nam       /nam/         "for" (giving reason)                 │
│   enim      /ˈɛ.nɪm/     "for, indeed" (postpositive:         │
│                            comes 2nd in its clause)             │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 ET — "And"

The most common conjunction, connecting words, phrases, or clauses:

```
Connecting nouns:
  Mārcus et Iūlia ambulant.
  "Marcus and Julia walk."

Connecting adjectives:
  Vir est bonus et magnus.
  "The man is good and great."

Connecting clauses:
  Puella cantat et puer rīdet.
  "The girl sings and the boy laughs."

Connecting multiple items:
  Servī et puellae et puerī labōrant.
  "The slaves and the girls and the boys work."
  (Latin can repeat et, like English "and...and...and")
```

### 6.3 -QUE — "And" (Enclitic)

The enclitic **-que** attaches to the **second** of the two items being joined. It is very common in formal and literary Latin.

```
Senātus Populusque Rōmānus    (SPQR)
"The Senate and the Roman People"
(-que attached to Populus, the second item)

Virī fēminaeque ambulant.
"Men and women walk."
(-que attached to fēminae, the second item)

Bonus fortisque est.
"He is good and brave."
(-que attached to fortis, the second item)

※ -que is more elevated/literary than et.
※ -que connects closely related pairs; et is more general.
```

### 6.4 SED — "But"

```
Connecting contrasting clauses:
  Puella cantat, sed puer nōn cantat.
  "The girl sings, but the boy does not sing."

  Servus labōrat, sed dominus nōn labōrat.
  "The slave works, but the master does not work."

  Vīta est brevis, sed ars est longa.
  "Life is short, but art is long."

  Terra est magna, sed populus est parvus.
  "The land is large, but the people are few."
```

### 6.5 AUT — "Or"

```
Exclusive "or" (one or the other, not both):
  Aut Caesar aut nihil.
  "Either Caesar or nothing."

  Aut vincere aut morī.
  "Either to conquer or to die."

  Estne bonus aut malus?
  "Is he good or bad?"

Paired aut...aut = "either...or":
  Aut amīcus aut inimīcus est.
  "He is either a friend or an enemy."
```

### 6.6 NEQUE / NEC — "And Not" / "Nor"

```
Single neque = "and...not" or "nor":
  Puella cantat neque labōrat.
  "The girl sings and does not work."

Paired neque...neque = "neither...nor":
  Neque puerī neque puellae labōrant.
  "Neither the boys nor the girls work."

  Neque bonus neque malus est.
  "He is neither good nor bad."

  Neque terram neque aquam videt nauta.
  "The sailor sees neither land nor water."
```

### 6.7 Summary Table

| Conjunction | Meaning | Example |
|------------|---------|---------|
| et | and | Mārcus **et** Iūlia |
| -que | and (enclitic) | Mārcus Iūlia**que** |
| atque / ac | and (emphatic) | Mārcus **atque** Iūlia |
| sed | but | bonus, **sed** parvus |
| aut | or (exclusive) | bonus **aut** malus |
| aut...aut | either...or | **aut** bonus **aut** malus |
| vel | or (inclusive) | bonus **vel** magnus |
| neque / nec | and not, nor | neque bonus |
| neque...neque | neither...nor | **neque** bonus **neque** malus |

### Exercise 7: Reading Comprehension

Read this short passage and answer the questions below.

```
Mārcus est agricola bonus. In magnō agrō cum fīliō labōrat.
Fīlius agricolae est puer parvus sed bonus. Mārcus terram
amat et patriam laudat. Fīlia Mārcī est puella pulchra.
Puella in viā longā cum amīcīs ambulat. Neque Mārcus neque
fīlius puellam vident, nam in agrō manent.
```

Questions:
```
1. What is Marcus's occupation?
2. Where does Marcus work?
3. Who works with Marcus?
4. What does Marcus's daughter look like?
5. What is she doing?
6. Why don't Marcus and his son see the girl?
```

<details>
<summary>Click to reveal answers</summary>

```
1. Marcus is a farmer (agricola).

2. He works in a large field (in magnō agrō).

3. His son (cum fīliō) works with him.

4. She is a beautiful girl (puella pulchra).

5. She walks on a long road with friends
   (in viā longā cum amīcīs ambulat).

6. Because they remain in the field
   (nam in agrō manent).

Full translation:
"Marcus is a good farmer. He works in a large field with
his son. The farmer's son is a small but good boy. Marcus
loves the land and praises his country. Marcus's daughter
is a beautiful girl. The girl walks on a long road with
friends. Neither Marcus nor his son sees the girl, for
they remain in the field."
```

```
│        Adjectives and Sentence Structure Quick Reference          │
│                                                                 │
│   1ST/2ND DECLENSION ADJECTIVES (bonus, -a, -um):               │
│     Masc. = 2nd decl. (bonus, bonī, bonō, bonum, bonō, bone)  │
│     Fem.  = 1st decl. (bona, bonae, bonae, bonam, bonā, bona) │
│     Neut. = 2nd decl. (bonum, bonī, bonō, bonum, bonō, bonum)│
│                                                                 │
│   AGREEMENT RULE:                                                │
│     Adjective matches noun in GENDER + NUMBER + CASE            │
│     (NOT necessarily in declension!)                             │
│     agricola bonus = good farmer (1st decl. noun + 2nd adj.)   │
│                                                                 │
│   ADJECTIVE POSITION:                                            │
│     After noun (normal): puella bona                            │
│     Before noun (emphatic): bona puella                         │
│     Predicate: Puella est bona.                                 │
│                                                                 │
│   SUBSTANTIVE ADJECTIVES:                                        │
│     Masc. pl. = people:  bonī (the good people)                 │
│     Neut. sg. = thing:   bonum (the good / a good thing)       │
│     Neut. pl. = things:  bona (good things)                     │
│                                                                 │
│   CONJUNCTIONS:                                                  │
│     et / -que  = and       sed     = but                        │
│     aut        = or        neque   = and not, nor               │
│     aut...aut  = either...or                                    │
│     neque...neque = neither...nor                                │
│                                                                 │
│   WORD ORDER:                                                    │
│     Default: S — O — V (verb last)                              │
│     Flexible — case endings determine function                  │
│     First/last position = emphasis                               │
└─────────────────────────────────────────────────────────────────┘
```
