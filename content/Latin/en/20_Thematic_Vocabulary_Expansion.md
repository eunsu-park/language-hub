# 20. Thematic Vocabulary Expansion (Vocābulārium Thēmaticum)

## Learning Objectives
- Acquire essential military vocabulary and understand Roman army organization
- Learn legal terminology used in Roman courts and jurisprudence
- Master political vocabulary related to the Roman Republic and Empire
- Build agricultural vocabulary reflecting Roman rural life
- Recognize architectural terms for Roman engineering and building
- Understand religious vocabulary covering Roman worship and ritual
- Apply word formation rules (prefixes and suffixes) to expand vocabulary systematically

## 7. Word Formation

Understanding Latin word formation allows you to deduce the meaning of unfamiliar words and expands vocabulary exponentially.

### 7.1 Common Prefixes

| Prefix | Meaning | Example | English Derivative |
|--------|---------|---------|-------------------|
| **ad-** | to, toward | **ad**venīre (to arrive) | advent |
| **con-** (com-, col-, cor-) | together, with | **con**venīre (to assemble) | convention |
| **dē-** | down, away | **dē**dūcere (to lead down) | deduction |
| **ex-** (ē-) | out of | **ex**cēdere (to go out) | exceed |
| **in-** (im-) | in, into | **in**dūcere (to lead in) | induce |
| **in-** (im-) | not (negation) | **in**fēlīx (unhappy) | infelicitous |
| **inter-** | between | **inter**pōnere (to place between) | interpose |
| **per-** | through, thoroughly | **per**ficere (to complete) | perfect |
| **prae-** | before, in front | **prae**dīcere (to predict) | predict |
| **prō-** | forward, for | **prō**cēdere (to go forward) | proceed |
| **re-** (red-) | back, again | **re**dūcere (to lead back) | reduce |
| **sub-** (suf-, sug-, sup-) | under, up from below | **sub**venīre (to come to help) | subvene |
| **trāns-** (trā-) | across | **trāns**dūcere (to lead across) | traduce |

### 7.2 Common Suffixes — Nouns

| Suffix | Meaning | Example | Base Verb/Adj. |
|--------|---------|---------|---------------|
| **-tor / -trix** | agent (m./f.) | imperā**tor** (commander) | imperāre |
| | | victō**rīx** (female conqueror) | vincere |
| **-tiō / -siō** | action, process | ōrā**tiō** (speech) | ōrāre |
| | | mīs**siō** (sending) | mittere |
| **-tās** | quality, state | lībertās (freedom) | līber |
| | | cīvitās (citizenship, state) | cīvis |
| **-tūdō** | quality, state | magnitūdō (greatness) | magnus |
| | | fortitūdō (bravery) | fortis |
| **-ium** | place, abstract | cōnsilium (plan, council) | cōnsulere |
| | | imperium (command) | imperāre |
| **-men / -mentum** | instrument, result | fundāmentum (foundation) | fundāre |
| | | mūnīmentum (fortification) | mūnīre |
| **-ārium** | place for | grānārium (granary) | grānum |

### 7.3 Common Suffixes — Adjectives

| Suffix | Meaning | Example | Base |
|--------|---------|---------|------|
| **-ālis / -āris** | pertaining to | mīlitāris (military) | mīles |
| **-ānus** | belonging to | Rōmānus (Roman) | Rōma |
| **-ōsus** | full of | perīculōsus (dangerous) | perīculum |
| **-bilis** | able to be | mīrābilis (wonderful) | mīrārī |
| **-ius / -eus** | made of | aureus (golden) | aurum |

### 7.4 Word Family Example: dūcō

From **dūcō, dūcere, dūxī, ductum** (to lead):

| Compound | Meaning | English |
|----------|---------|---------|
| **ad**dūcere | to lead to | adduce |
| **con**dūcere | to bring together | conduct |
| **dē**dūcere | to lead down | deduce |
| **in**dūcere | to lead in | induce |
| **prō**dūcere | to lead forth | produce |
| **re**dūcere | to lead back | reduce |
| **trā**dūcere | to lead across | traduce |

### Exercise E: Suffix Analysis

Identify the suffix, its meaning, and the base word:

1. ōrātor
2. lībertās
3. fortitūdō
4. perīculōsus
5. mīlitāris
6. mīrābilis

<details>
<summary>Click to reveal answers</summary>

1. **-tor** (agent): from *ōrāre* (to speak) → "speaker, orator"
2. **-tās** (quality/state): from *līber* (free) → "freedom, liberty"
3. **-tūdō** (quality/state): from *fortis* (brave) → "bravery, fortitude"
4. **-ōsus** (full of): from *perīculum* (danger) → "full of danger, dangerous"
5. **-āris** (pertaining to): from *mīles* (soldier) → "pertaining to a soldier, military"
6. **-bilis** (able to be): from *mīrārī* (to wonder at) → "able to be wondered at, wonderful"

</details>
