# 14. Subjunctive Mood Introduction (Modus Coniunctivus)

## Learning Objectives
- Understand the concept of mood in Latin and when the subjunctive is used
- Form the present subjunctive for all four conjugations and *esse*

- Recognize and translate independent uses of the subjunctive (hortatory, jussive, optative, deliberative, potential)
- Understand dependent uses with *ut/nē* (purpose and result clauses)
- Identify and translate *cum* clauses (temporal, causal, concessive)
- Apply the subjunctive in reading and composition

## 1. What is the Subjunctive?

Latin has three moods:

```
┌────────────────────────────────────────────────────────────────┐
│   INDICATIVE   → states facts, asks direct questions           │
│                  "Caesar conquered Gaul."                       │
│                                                                │
│   SUBJUNCTIVE  → expresses possibility, wish, purpose,         │
│                  doubt, command (indirect/soft), condition      │
│                  "May Caesar conquer Gaul."                     │
│                  "Let us conquer Gaul."                         │
│                  "...so that Caesar might conquer Gaul."        │
│                                                                │
│   IMPERATIVE   → gives direct commands                         │
│                  "Conquer Gaul!"                                │
└────────────────────────────────────────────────────────────────┘
```

The subjunctive is **extremely common** in Latin — far more so than in English. It appears in:
- Independent (main) clauses: wishes, gentle commands, possibilities
- Dependent (subordinate) clauses: purpose, result, cause, concession, indirect questions, conditions

## 2. Present Subjunctive Formation

### 2.1 The Vowel Change Rule

The key to the present subjunctive is a **characteristic vowel change**:

| Conjugation | Indicative Vowel | Subjunctive Vowel | Mnemonic |
|-------------|-----------------|-------------------|----------|
| 1st (-āre) | ā | **ē** | "She wEars a diamond." |
| 2nd (-ēre) | ē | **eā** | |
| 3rd (-ere) | e/i | **ā** | |
| 3rd-iō | i | **iā** | |
| 4th (-īre) | ī | **iā** | |

> **Mnemonic for all conjugations**: "wE bEAt A lIAr" — ē, eā, ā, iā.

### 2.2 Present Subjunctive: All Conjugations

| Person | 1st: amō | 2nd: moneō | 3rd: dūcō | 3rd-iō: capiō | 4th: audiō |
|--------|----------|-----------|-----------|---------------|-----------|
| 1sg | am**em** | mon**eam** | dūc**am** | cap**iam** | aud**iam** |
| 2sg | am**ēs** | mon**eās** | dūc**ās** | cap**iās** | aud**iās** |
| 3sg | am**et** | mon**eat** | dūc**at** | cap**iat** | aud**iat** |
| 1pl | am**ēmus** | mon**eāmus** | dūc**āmus** | cap**iāmus** | aud**iāmus** |
| 2pl | am**ētis** | mon**eātis** | dūc**ātis** | cap**iātis** | aud**iātis** |
| 3pl | am**ent** | mon**eant** | dūc**ant** | cap**iant** | aud**iant** |

### 2.3 Present Subjunctive of *esse* and *posse*

| Person | esse | posse |
|--------|------|-------|
| 1sg | sim | possim |
| 2sg | sīs | possīs |
| 3sg | sit | possit |
| 1pl | sīmus | possīmus |
| 2pl | sītis | possītis |
| 3pl | sint | possint |

### 2.4 Distinguishing Present Subjunctive from Indicative

| Form | Mood | How to Tell |
|------|------|-------------|
| *amat* | Indicative | 1st conj. keeps -a- |
| *amet* | Subjunctive | 1st conj. changes a → e |
| *dūcit* | Indicative | 3rd conj. has -i- |
| *dūcat* | Subjunctive | 3rd conj. has -a- |

> **Warning**: 3rd/4th conjugation present subjunctive (*dūcam, audiam*) looks identical to the future indicative 1sg. Context resolves ambiguity.

## 3. Imperfect Subjunctive Formation

### 3.1 The Simplest Rule in Latin Grammar

**Imperfect subjunctive = present active infinitive + personal endings**

| Person | Ending |
|--------|--------|
| 1sg | -m |
| 2sg | -s |
| 3sg | -t |
| 1pl | -mus |
| 2pl | -tis |
| 3pl | -nt |

### 3.2 Imperfect Subjunctive: All Conjugations

| Person | 1st: amō | 2nd: moneō | 3rd: dūcō | 3rd-iō: capiō | 4th: audiō |
|--------|----------|-----------|-----------|---------------|-----------|
| 1sg | amāre**m** | monēre**m** | dūcere**m** | capere**m** | audīre**m** |
| 2sg | amārē**s** | monērē**s** | dūcerē**s** | caperē**s** | audīrē**s** |
| 3sg | amāre**t** | monēre**t** | dūcere**t** | capere**t** | audīre**t** |
| 1pl | amārē**mus** | monērē**mus** | dūcerē**mus** | caperē**mus** | audīrē**mus** |
| 2pl | amārē**tis** | monērē**tis** | dūcerē**tis** | caperē**tis** | audīrē**tis** |
| 3pl | amāre**nt** | monēre**nt** | dūcere**nt** | capere**nt** | audīre**nt** |

### 3.3 Imperfect Subjunctive of *esse*

| Person | Form |
|--------|------|
| 1sg | essem |
| 2sg | essēs |
| 3sg | esset |
| 1pl | essēmus |
| 2pl | essētis |
| 3pl | essent |

> This follows the same rule: infinitive of *esse* is *esse* → *esse-m, esse-s, esse-t*...

## 4. Independent Uses of the Subjunctive

When the subjunctive appears in a **main clause** (not introduced by a conjunction), it has several distinct uses.

### 4.1 Hortatory Subjunctive ("Let us...")

**1st person plural, present subjunctive**

| Latin | Translation |
|-------|-------------|
| *Eāmus!* | Let us go! |
| *Gaudeāmus igitur!* | Let us therefore rejoice! |
| *Patriam dēfendāmus.* | Let us defend the fatherland. |

> Negated with **nē**: *Nē timeāmus.* — Let us not be afraid.

### 4.2 Jussive Subjunctive ("Let him/them...")

**3rd person, present subjunctive**

| Latin | Translation |
|-------|-------------|
| *Veniat.* | Let him come. |
| *Fīant lūx.* | Let there be light. (= *Fiat lūx.* for singular) |
| *Caveant cōnsulēs.* | Let the consuls beware. |
| *Nē abeat.* | Let him not leave. |

### 4.3 Optative Subjunctive (Wishes)

**Present subjunctive** for wishes that may come true; **imperfect subjunctive** for wishes contrary to present fact. Often with *utinam*.

| Latin | Translation | Type |
|-------|-------------|------|
| *Utinam veniat!* | May he come! / If only he would come! | Possible wish |
| *Utinam venīret!* | If only he were coming! (but he's not) | Contrary to fact |
| *Utinam nē abiret!* | If only he were not leaving! | Contrary to fact |

### 4.4 Deliberative Subjunctive ("Should I...?")

**1st person (occasionally 3rd), present or imperfect subjunctive, in questions**

| Latin | Translation |
|-------|-------------|
| *Quid faciam?* | What should I do? / What am I to do? |
| *Quō eam?* | Where should I go? |
| *Quid dīcerem?* | What was I to say? (past deliberation) |

### 4.5 Potential Subjunctive ("Would/Could/Might")

**Present or imperfect subjunctive**, expressing a possibility or mild assertion.

| Latin | Translation |
|-------|-------------|
| *Aliquis dīcat.* | Someone might say. |
| *Crēdās.* | You would believe. (= One might believe.) |
| *Velim.* | I would like. |
| *Nōllem.* | I would not want. |

### 4.6 Summary Table

| Use | Person | Tense | Translation Clue | Negation |
|-----|--------|-------|-------------------|----------|
| Hortatory | 1st pl. | Present | "Let us..." | nē |
| Jussive | 3rd | Present | "Let him/them..." | nē |
| Optative | Any | Pres. or Impf. | "May.../If only..." | nē / utinam nē |
| Deliberative | 1st (3rd) | Pres. or Impf. | "Should I...?" | nōn |
| Potential | Any | Pres. or Impf. | "Would/might..." | nōn |

## 5. Purpose Clauses (ut/nē)

Purpose clauses answer **"why?"** or **"for what purpose?"** and use the subjunctive.

### 5.1 Formation

| Type | Conjunction | + Subjunctive |
|------|------------|---------------|
| Positive purpose | **ut** | present or imperfect subj. |
| Negative purpose | **nē** | present or imperfect subj. |

### 5.2 Tense Selection

| Main Verb Tense | Subjunctive Tense |
|----------------|-------------------|
| Present or Future (primary) | **Present** subjunctive |
| Past (secondary) | **Imperfect** subjunctive |

### 5.3 Examples

| Latin | Translation |
|-------|-------------|
| *Venit **ut** videat.* | He comes **in order to** see. / He comes **so that** he may see. |
| *Vēnit **ut** vidēret.* | He came **in order to** see. / He came **so that** he might see. |
| *Fugit **nē** caperētur.* | He fled **lest** he be captured. / He fled **so that** he would not be captured. |
| *Labōrāmus **ut** bene vīvāmus.* | We work **so that** we may live well. |

> **Translation tip**: Purpose clauses can be translated with "in order to," "so that...may/might," or "to." The "lest" translation for *nē* is formal English; "so that...not" is more natural.

## 6. Result Clauses (ut)

Result clauses express the **actual outcome** of an action, often signaled by a "flag word" in the main clause.

### 6.1 Flag Words (Signal Words)

| Latin | Meaning |
|-------|---------|
| *tam* | so (with adjectives/adverbs) |
| *tantus, -a, -um* | so great |
| *tot* | so many |
| *tālis, -e* | of such a kind |
| *ita / sīc* | so, in such a way |
| *adeō* | to such a degree |

### 6.2 Formation

| Type | Construction |
|------|-------------|
| Positive result | flag word ... **ut** + subjunctive |
| Negative result | flag word ... **ut nōn** + subjunctive |

> **Key difference from purpose**: Negative purpose uses **nē**; negative result uses **ut nōn**.

### 6.3 Examples

| Latin | Translation |
|-------|-------------|
| *Tam fortis erat **ut** omnēs eum laudārent.* | He was **so** brave **that** everyone praised him. |
| *Tanta tempestās erat **ut** nāvēs nōn solverent.* | The storm was **so great that** the ships did not set sail. |
| *Ita cucurrit **ut** nēmō eum caperet.* | He ran **in such a way that** no one could catch him. |
| *Tot hostēs erant **ut nōn** resistere possēmus.* | There were **so many** enemies **that** we **could not** resist. |

### 6.4 Purpose vs. Result — Quick Comparison

| Feature | Purpose | Result |
|---------|---------|--------|
| Answers | "Why?" (intention) | "With what effect?" (actual outcome) |
| Negative | **nē** | **ut nōn** |
| Flag word in main clause | No | Yes (tam, tantus, ita, etc.) |
| Translation | "so that / in order to" | "so...that" |

## 7. Cum Clauses

The conjunction *cum* ("when/since/although") takes the subjunctive with three different meanings.

### 7.1 Cum Temporal ("when")

| Latin | Translation |
|-------|-------------|
| *Cum Caesar pervēnisset, proelium commīsit.* | When Caesar had arrived, he joined battle. |
| *Cum haec dīceret, omnēs tacēbant.* | When he was saying these things, everyone was silent. |

### 7.2 Cum Causal ("since/because")

| Latin | Translation |
|-------|-------------|
| *Cum hostēs adessent, portās clausērunt.* | Since the enemy were present, they closed the gates. |
| *Cum nēmō respondēret, abiit.* | Since no one was answering, he left. |

### 7.3 Cum Concessive ("although")

Often paired with **tamen** ("nevertheless") in the main clause:

| Latin | Translation |
|-------|-------------|
| *Cum fessī essent, **tamen** pugnāvērunt.* | Although they were tired, **nevertheless** they fought. |
| *Cum paucī essent, **tamen** fortiter resistēbant.* | Although they were few, they **still** resisted bravely. |

### 7.4 Identifying Which Cum

```
┌────────────────────────────────────────────────────────────────┐
│   CUM + INDICATIVE  → purely temporal ("when/at the time")    │
│   CUM + SUBJUNCTIVE → check context:                          │
│                                                                │
│   1. "tamen" in main clause?  → CONCESSIVE (although)         │
│   2. Explains WHY?            → CAUSAL (since)                │
│   3. Sets the scene/time?     → TEMPORAL (when)               │
└────────────────────────────────────────────────────────────────┘
```

> Note: *Cum* + indicative is used for present/future time or repeated actions (*Cum tacent, clāmant.* — When they are silent, they cry out). *Cum* + subjunctive is the norm for past-time narrative.

## 8. Practice Exercises

### Exercise A: Form the Subjunctive
Give the requested subjunctive form.

1. amō — 3sg present subjunctive active
2. dūcō — 1pl present subjunctive active
3. moneō — 2sg present subjunctive active
4. audiō — 3pl present subjunctive active
5. sum — 3sg present subjunctive
6. amō — 1sg imperfect subjunctive active
7. capiō — 3pl imperfect subjunctive active
8. sum — 3pl imperfect subjunctive

### Exercise B: Identify the Independent Use
Name the type of subjunctive in each sentence.

1. *Gaudeāmus igitur!*
2. *Quid faciam?*
3. *Fiat lūx.*
4. *Utinam venīret!*
5. *Aliquis dīcat.*
6. *Nē cēdāmus!*

### Exercise C: Purpose or Result?
Identify each clause as purpose or result, then translate.

1. *Vēnit ut rēgem vidēret.*
2. *Tam celeriter cucurrit ut nēmō eum caperet.*
3. *Mīlitēs mīsit nē hostēs urbem caperent.*
4. *Tanta erat tempestās ut nāvēs frangerentur.*
5. *Labōrāmus ut bene vīvāmus.*

### Exercise D: Translate the Cum Clause
Translate and identify the type (temporal, causal, or concessive).

1. *Cum Caesar in Galliam pervēnisset, bellum gessit.*
2. *Cum paucōs mīlitēs habēret, tamen pugnāvit.*
3. *Cum nox esset, iter facere nōn potuērunt.*
4. *Cum haec audīvisset, statim profectus est.*

### Exercise E: Translate into Latin

1. Let us go to the forum!
2. What should I say?
3. The teacher came so that the students might learn.
4. He was so wise that everyone admired him.
5. Although the road was long, they arrived before night.
