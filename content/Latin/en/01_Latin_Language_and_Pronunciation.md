## 1. Introduction to Latin

### 1.1 Latin in History

```
┌─────────────────────────────────────────────────────────────────┐
│                 Latin: The Eternal Language                       │

│   Period of use:    c. 700 BCE – present (in specialized use)   │
```

### 1.2 Why Study Latin?

Latin is no longer spoken natively, but it remains extraordinarily influential:

- **Foundation of Romance languages**: Spanish, French, Italian, Portuguese, and Romanian descend directly from Latin. Knowing Latin unlocks vocabulary across all of them.
- **English vocabulary**: Over 60% of English words derive from Latin (directly or via French). In scientific and legal English, the figure exceeds 90%.
- **Scientific nomenclature**: Biology (binomial nomenclature), medicine, chemistry, and astronomy all use Latin terminology.
- **Legal tradition**: Legal maxims like *habeas corpus*, *pro bono*, and *stare decisis* remain in active use worldwide.
- **Literary heritage**: Access Virgil, Cicero, Caesar, Ovid, Horace, and Seneca in the original.
- **Logical training**: Latin's highly inflected grammar develops analytical thinking about how language works.

> **For Korean speakers (한국어 화자를 위한 참고)**: Korean and Latin share one surprising structural similarity — both are SOV (Subject-Object-Verb) languages by default. Latin's case system also parallels Korean particles (조사): the nominative case functions like 이/가, the accusative like 을/를, and the dative like 에게.

### 1.3 Latin's Influence on Modern Languages

```
Latin "aqua" (water):
  → Italian:    acqua
  → Spanish:    agua
  → Portuguese: água
  → French:     eau (from /ˈakwa/ → /o/)
  → English:    aquatic, aquarium, aqueduct

Latin "nox, noctis" (night):
  → Italian:    notte
  → Spanish:    noche
  → French:     nuit
  → English:    nocturnal, equinox

Latin "schola" (from Greek σχολή):
  → Italian:    scuola
  → Spanish:    escuela
  → French:     école
  → English:    school
  → Korean:     스쿨 (loanword from English)
```

## 2. The Latin Alphabet

### 2.1 The 23 Classical Letters

Classical Latin used 23 letters. The letters J, U, and W did not exist as separate characters — I served for both /i/ and /j/, and V served for both /u/ and /w/.

| Letter | Name | IPA (Classical) | Example |
|--------|------|-----------------|---------|
| A a | ā | /a/, /aː/ | **a**qua (water) |
| B b | bē | /b/ | **b**onus (good) |
| C c | cē | /k/ | **c**aput (head) |
| D d | dē | /d/ | **d**eus (god) |
| E e | ē | /ɛ/, /eː/ | **e**st (is) |
| F f | ef | /f/ | **f**ilius (son) |
| G g | gē | /g/ | **g**loria (glory) |
| H h | hā | /h/ | **h**ora (hour) |
| I i | ī | /i/, /iː/, /j/ | **i**n (in), **i**am (now) |
| K k | kā | /k/ | **K**alendae (Kalends) |
| L l | el | /l/ | **l**ux (light) |
| M m | em | /m/ | **m**ater (mother) |
| N n | en | /n/ | **n**ox (night) |
| O o | ō | /ɔ/, /oː/ | **o**mnis (all) |
| P p | pē | /p/ | **p**ater (father) |
| Q q | qū | /k/ (always + u) | **qu**i (who) |
| R r | er | /r/ | **r**ex (king) |
| S s | es | /s/ | **s**ol (sun) |
| T t | tē | /t/ | **t**erra (earth) |
| V v | ū | /u/, /uː/, /w/ | **v**ia (road), **v**inum (wine) |
| X x | ix | /ks/ | **r**ex (king) |
| Y y | ȳ (Greek) | /y/, /yː/ | **s**ylva (forest) |
| Z z | zēta (Greek) | /dz/ | **z**ona (zone) |

### 2.2 Notes on Letters

```
┌─────────────────────────────────────────────────────────────────┐
│                     Key Alphabet Notes                            │
│                                                                 │
│   • C is ALWAYS hard /k/ in classical Latin.                    │
│     Caesar = /ˈkae̯.sar/ not "SEE-zer"                          │
│                                                                 │
│   • G is ALWAYS hard /g/ in classical Latin.                    │
│     genus = /ˈgɛ.nus/ not "JEE-nus"                            │
│                                                                 │
│   • V represents both vowel /u/ and consonant /w/.              │
│     Modern textbooks often write U for the vowel, V for /w/.    │
│     VINUM → uīnum /ˈwiː.num/                                   │
│                                                                 │
│   • I represents both vowel /i/ and consonant /j/.              │
│     Modern textbooks sometimes write J for /j/.                 │
│     IAM → iam /jam/ (now, already)                              │
│                                                                 │
│   • K is rare — used mainly in Kalendae and a few loanwords.    │
│                                                                 │
│   • Y and Z appear only in Greek loanwords.                     │
│     Y = /y/ (like German ü or French u)                         │
│     Z = /dz/                                                    │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 Modern Conventions

In modern Latin textbooks and editions, you will commonly see:

| Convention | Classical | Modern Textbook |
|-----------|-----------|-----------------|
| Vowel u | VRBS | urbs |
| Consonant w | VINUM | vīnum |
| Consonant j | IAM | jam (some books) |
| Long vowels | not marked | marked with macrons: ā, ē, ī, ō, ū |

> **Macrons (ˉ)** are not original to Latin inscriptions but are used in modern teaching to indicate long vowels. They are essential for correct pronunciation and stress placement.

## 3. Vowel System

### 3.1 Short and Long Vowels

Latin has 10 vowel phonemes — 5 short and 5 long. Vowel length (quantity) is distinctive and can change word meaning.

```
┌─────────────────────────────────────────────────────────────────┐
│                   Latin Vowel System                              │
│                                                                 │
│           Front         Central         Back                    │
│   High     ĭ /ɪ/  ī /iː/              ŭ /ʊ/  ū /uː/         │
│   Mid      ĕ /ɛ/  ē /eː/              ŏ /ɔ/  ō /oː/         │
│   Low                  ă /a/  ā /aː/                           │
│                                                                 │
│   Short vowels (ă, ĕ, ĭ, ŏ, ŭ):                              │
│     Shorter in duration AND more open in quality                │
│                                                                 │
│   Long vowels (ā, ē, ī, ō, ū):                                │
│     Longer in duration AND more closed in quality               │
│                                                                 │
│   Macron (ˉ) marks long vowels: ā ē ī ō ū                    │
│   Breve (˘) marks short vowels: ă ĕ ĭ ŏ ŭ (optional)        │
└─────────────────────────────────────────────────────────────────┘
```

| Vowel | IPA | Duration | Similar to English | Similar to Korean | Example |
|-------|-----|----------|-------------------|-------------------|---------|
| ă | /a/ | short | "u" in "cut" | 아 (short) | m**a**lus (bad) |
| ā | /aː/ | long | "a" in "father" | 아 (long) | m**ā**lus (apple tree) |
| ĕ | /ɛ/ | short | "e" in "bet" | 에 (short) | l**e**gō (I read) |
| ē | /eː/ | long | "ay" in "say" (no glide) | 에 (long) | l**ē**x (law) |
| ĭ | /ɪ/ | short | "i" in "bit" | 이 (short) | s**i**t (may it be) |
| ī | /iː/ | long | "ee" in "see" | 이 (long) | v**ī**ta (life) |
| ŏ | /ɔ/ | short | "o" in "pot" (British) | 오 (short) | d**o**minus (lord) |
| ō | /oː/ | long | "o" in "go" (no glide) | 오 (long) | Rōm**ā**nus (Roman) |
| ŭ | /ʊ/ | short | "u" in "put" | 우 (short) | d**u**x (leader) |
| ū | /uː/ | long | "oo" in "food" | 우 (long) | l**ū**na (moon) |

### 3.2 Vowel Length Changes Meaning

| Short vowel | Meaning | Long vowel | Meaning |
|------------|---------|------------|---------|
| m**a**lus | bad, evil | m**ā**lus | apple tree |
| l**e**vis | light (weight) | l**ē**vis | smooth |
| l**i**ber | free | l**ī**ber | book |
| p**o**pulus | people | p**ō**pulus | poplar tree |
| **o**s | bone | **ō**s | mouth |

> **Key point**: In English we often distinguish words by stress; in Latin, vowel length carries the same kind of distinctive load. Always pay attention to macrons in your textbook.

## 4. Diphthongs

### 4.1 Classical Latin Diphthongs

Latin has six diphthongs, of which three are common and three are rare.

```
┌─────────────────────────────────────────────────────────────────┐
│                    Latin Diphthongs                               │
│                                                                 │
│   Common diphthongs:                                            │
│     ae  /ae̯/  → like "eye" (not "ee")                          │
│     au  /au̯/  → like "ow" in "cow"                             │
│     oe  /oe̯/  → like "oi" in "oil"                             │
│                                                                 │
│   Rare diphthongs:                                              │
│     ei  /ei̯/  → like "ay" in "day"                             │
│     eu  /eu̯/  → e + u (no English equivalent)                  │
│     ui  /ui̯/  → like "we" (only in huic, cui, hui)             │
│                                                                 │
│   All diphthongs count as LONG for stress and meter.            │
└─────────────────────────────────────────────────────────────────┘
```

| Diphthong | IPA | Pronunciation | Example |
|-----------|-----|---------------|---------|
| ae | /ae̯/ | "eye" | C**ae**sar /ˈkae̯.sar/ (Caesar) |
| au | /au̯/ | "ow" in "cow" | l**au**dō /ˈlau̯.doː/ (I praise) |
| oe | /oe̯/ | "oi" in "oil" | p**oe**ta /poe̯.ˈta/ (poet) |
| ei | /ei̯/ | "ay" in "day" | d**ei**nde /ˈdei̯n.dɛ/ (then) |
| eu | /eu̯/ | e + oo (glide) | **Eu**rōpa /eu̯.ˈroː.pa/ (Europe) |
| ui | /ui̯/ | oo + ee (quick) | h**ui**c /hui̯k/ (to this) |

### 4.2 Important Note on AE and OE

In later Latin and ecclesiastical pronunciation, **ae** became /ɛː/ (like a long "e") and **oe** became /eː/. This is why many English words from Latin spell "ae/oe" but pronounce them as "ee":

```
Classical → Ecclesiastical → English:
  Caesar   /ˈkae̯.sar/ → /ˈtʃeː.sar/ → "SEE-zer"
  poena    /ˈpoe̯.na/  → /ˈpeː.na/   → "penal"
  caelum   /ˈkae̯.lum/ → /ˈtʃeː.lum/ → "celestial"
```

> In this course, we use **classical (restored) pronunciation** unless otherwise noted.

### 5.1 Consonant Inventory

| Manner | Labial | Dental | Velar | Glottal |
|--------|--------|--------|-------|---------|
| Voiceless Stop | p /p/ | t /t/ | c, k /k/ | — |
| Voiced Stop | b /b/ | d /d/ | g /g/ | — |
| Voiceless Fricative | f /f/ | s /s/ | — | h /h/ |
| Nasal | m /m/ | n /n/ | (ng /ŋ/) | — |
| Liquid | — | l /l/, r /r/ | — | — |
| Semivowel | v /w/ | — | — | — |
| Semivowel | — | i /j/ | — | — |

### 5.2 Key Consonant Rules (Classical Pronunciation)

```
┌─────────────────────────────────────────────────────────────────┐
│              Classical Latin Consonant Rules                      │
│                                                                 │
│   C = ALWAYS /k/, never /s/ or /tʃ/                            │
│     Cicerō = /ˈkɪ.kɛ.roː/ NOT "SIS-uh-roh"                   │
│     centum = /ˈkɛn.tum/ NOT "SEN-tum"                          │
│                                                                 │
│   G = ALWAYS /g/, never /dʒ/                                    │
│     genus = /ˈgɛ.nʊs/ NOT "JEE-nus"                            │
│     gerō = /ˈgɛ.roː/                                           │
│                                                                 │
│   V = /w/ when consonantal                                      │
│     vīnum = /ˈwiː.nʊm/ (wine)                                  │
│     via = /ˈwɪ.a/ (road)                                        │
│                                                                 │
│   I = /j/ when consonantal (before a vowel)                     │
│     iam = /jam/ (now, already)                                   │
│     iūs = /juːs/ (law, right)                                   │
│                                                                 │
│   S = ALWAYS voiceless /s/, never /z/                           │
│     rosa = /ˈrɔ.sa/ NOT /ˈrɔ.za/                               │
│                                                                 │
│   R = trilled /r/ (tongue tip trill, like Italian or Spanish)   │
│     rēx = /reːks/ (king)                                        │
│                                                                 │
│   H = /h/ — a real sound in classical Latin (later lost)        │
│     homō = /ˈhɔ.moː/ (man, person)                             │
│                                                                 │
│   T = ALWAYS /t/, even before i + vowel                         │
│     nātiō = /ˈnaː.tɪ.oː/ NOT "NAY-shee-oh"                   │
│     ratiō = /ˈra.tɪ.oː/ NOT "RAY-shee-oh"                     │
└─────────────────────────────────────────────────────────────────┘
```

### 5.3 Double Consonants (Gemination)

Like Italian, Latin distinguishes single from double consonants. Double consonants are held longer.

```
Examples of geminate consonants:
  annus /ˈan.nʊs/  (year)     vs  anus /ˈa.nʊs/  (old woman)
  agger /ˈag.gɛr/  (mound)
  littera /ˈlɪt.tɛ.ra/ (letter)
  bellum /ˈbɛl.lʊm/ (war)
  occīdō /ɔk.ˈkiː.doː/ (I kill)
```

### 5.4 Special Combinations

| Combination | Pronunciation | Example |
|------------|---------------|---------|
| qu | /kw/ | **qu**i /kwɪ/ (who) |
| gu (after n) | /gw/ | lin**gu**a /ˈlɪŋ.gwa/ (tongue) |
| ph | /pʰ/ (aspirated p) | **ph**ilosophia /pʰɪ.lɔ.sɔ.ˈpʰɪ.a/ |
| th | /tʰ/ (aspirated t) | **th**eātrum /tʰɛ.ˈaː.trʊm/ (theater) |
| ch | /kʰ/ (aspirated k) | **ch**orus /ˈkʰɔ.rʊs/ (chorus) |
| x | /ks/ | re**x** /reːks/ (king) |
| bs | /ps/ | ur**bs** /ʊrps/ (city) |
| bt | /pt/ | o**bt**ineō /ɔp.tɪ.ˈnɛ.oː/ (I obtain) |

> **Note on ph, th, ch**: These are Greek-origin sounds. In classical Latin, they are aspirated stops (/pʰ, tʰ, kʰ/), NOT /f, θ, x/. The English pronunciation of "philosophy" with /f/ comes from later changes.

## 6. Syllable Structure and Quantity

### 6.1 Syllable Division Rules

```
1. Single consonant between vowels → goes with second vowel:
   a-qua, do-mi-nus, pa-ter, a-mō

2. Two consonants between vowels → generally split:
   mag-nus, ter-ra, pul-cher
   EXCEPTION: stop/fricative + liquid (l, r) stay together:
   pa-tris, li-brum, sa-crum

3. Double consonants always split:
   an-nus, bel-lum, lit-te-ra

4. Prefixes maintain their boundaries:
   ab-est, ad-est, in-est
```

### 6.2 Syllable Quantity (Light vs. Heavy)

Syllable quantity is crucial for stress placement and poetry (meter).

```
┌─────────────────────────────────────────────────────────────────┐
│              Syllable Weight (Quantity)                           │
│                                                                 │
│   LIGHT syllable (syllaba brevis):                              │
│     Contains a short vowel + open syllable (no closing          │
│     consonant)                                                  │
│     Examples: ă-mō (ă is light), dŏ-mi-nus (dŏ is light)      │
│                                                                 │
│   HEAVY syllable (syllaba longa):                               │
│     Either:                                                     │
│       (a) Contains a long vowel or diphthong                    │
│           Examples: Rō-ma (Rō is heavy), Cae-sar (Cae heavy)   │
│       (b) Ends in a consonant (closed syllable)                 │
│           Examples: mag-nus (mag is heavy), ter-ra (ter heavy)  │
│                                                                 │
│   Rule of thumb:                                                │
│     LIGHT = short vowel + nothing                               │
│     HEAVY = long vowel, OR diphthong, OR short vowel + cons.   │
└─────────────────────────────────────────────────────────────────┘
```

### 6.3 Why Quantity Matters

1. **Stress placement** — The stress rule depends on the weight of the penultimate syllable.
2. **Latin poetry** — Meter (dactylic hexameter, elegiac couplet, etc.) is based on patterns of heavy and light syllables, not stress.
3. **Correct pronunciation** — Long and short vowels sound different.

## 7. Stress Rules

### 7.1 The Three Stress Rules

Latin stress is entirely predictable from syllable quantity:

```
┌─────────────────────────────────────────────────────────────────┐
│                  Latin Stress Rules                               │
│                                                                 │
│   Rule 1: Monosyllables                                         │
│     One-syllable words are stressed on that syllable.           │
│     rēx, lūx, nōn, et                                          │
│                                                                 │
│   Rule 2: Two-syllable words                                    │
│     Always stressed on the FIRST syllable.                      │
│     RO-sa, MA-ter, A-mō, TER-ra                                │
│                                                                 │
│   Rule 3: Words of three or more syllables                      │
│     Look at the PENULTIMATE (second-to-last) syllable:          │
│                                                                 │
│     (a) If the penult is HEAVY → stress the PENULT              │
│         a-MĪ-cus (mī is long vowel → heavy → stressed)         │
│         ma-GIS-ter (gis ends in consonant → heavy → stressed)   │
│         for-TŪ-na (tū is long vowel → heavy → stressed)        │
│                                                                 │
│     (b) If the penult is LIGHT → stress the ANTEPENULT          │
│         DO-mi-nus (mi is short + open → light → stress DO)      │
│         LIT-te-ra (te is short + open → light → stress LIT)     │
│         FE-mi-na (mi is short + open → light → stress FE)       │
│                                                                 │
│   ※ Latin NEVER stresses the last syllable (unlike French).     │
│   ※ Latin NEVER stresses before the antepenult.                 │
└─────────────────────────────────────────────────────────────────┘
```

### 7.2 Stress Examples

| Word | Syllables | Penult weight | Stress | Pronunciation |
|------|-----------|---------------|--------|---------------|
| amīcus | a-mī-cus | mī = heavy (long ī) | a-**MĪ**-cus | /a.ˈmiː.kʊs/ |
| dominus | do-mi-nus | mi = light (short, open) | **DO**-mi-nus | /ˈdɔ.mɪ.nʊs/ |
| fortūna | for-tū-na | tū = heavy (long ū) | for-**TŪ**-na | /fɔr.ˈtuː.na/ |
| fēmina | fē-mi-na | mi = light (short, open) | **FĒ**-mi-na | /ˈfeː.mɪ.na/ |
| magister | ma-gis-ter | gis = heavy (closed) | ma-**GIS**-ter | /ma.ˈgɪs.tɛr/ |
| taberna | ta-ber-na | ber = heavy (closed) | ta-**BER**-na | /ta.ˈbɛr.na/ |

## 8. Classical vs Ecclesiastical Pronunciation

### 8.1 Overview

Two major pronunciation traditions exist for Latin:

| Feature | Classical (Restored) | Ecclesiastical (Church) |
|---------|---------------------|------------------------|
| Period | c. 100 BCE – 200 CE | Medieval onward |
| Used by | Classicists, universities | Roman Catholic Church |
| C before e, i | /k/ always | /tʃ/ (like Italian) |
| G before e, i | /g/ always | /dʒ/ (like Italian) |
| V (consonant) | /w/ | /v/ (like Italian) |
| AE diphthong | /ae̯/ | /ɛː/ (like "e") |
| OE diphthong | /oe̯/ | /eː/ (like "e") |
| TI before vowel | /tɪ/ | /tsɪ/ or /ʃɪ/ |
| H | /h/ (pronounced) | silent |
| S between vowels | /s/ always | sometimes /z/ |

### 8.2 Comparison Examples

| Word | Classical | Ecclesiastical | Meaning |
|------|-----------|---------------|---------|
| Caesar | /ˈkae̯.sar/ | /ˈtʃeː.sar/ | Caesar |
| Cicerō | /ˈkɪ.kɛ.roː/ | /ˈtʃiː.tʃɛ.roː/ | Cicero |
| vēnī | /ˈweː.niː/ | /ˈveː.niː/ | I came |
| genus | /ˈgɛ.nʊs/ | /ˈdʒɛ.nʊs/ | kind, race |
| caelum | /ˈkae̯.lʊm/ | /ˈtʃeː.lʊm/ | sky, heaven |
| nātiō | /ˈnaː.tɪ.oː/ | /ˈnaː.tsɪ.oː/ | nation |
| Iesus | /ˈjɛ.sʊs/ | /ˈjeː.zʊs/ | Jesus |

> **This course uses classical pronunciation** as it reflects the language as the ancient Romans actually spoke it. However, awareness of ecclesiastical pronunciation is valuable for reading medieval and church Latin.

## 9. Common Pronunciation Challenges

### 9.1 For English Speakers

| Mistake | Correct | Explanation |
|---------|---------|-------------|
| C as /s/ before e, i | Always /k/ | "Cicero" = /ˈkɪ.kɛ.roː/ not "SIS-uh-roh" |
| V as /v/ | V = /w/ in classical | *vīnum* = /ˈwiː.nʊm/ not /ˈviː.nʊm/ |
| AE as "ee" | AE = /ae̯/ ("eye") | *Caesar* = /ˈkae̯.sar/ not "SEE-zer" |
| TI as "sh" | TI = /tɪ/ | *nātiō* = /ˈnaː.tɪ.oː/ not "NAY-shee-oh" |
| Ignoring vowel length | Long ≠ short | *malus* (bad) ≠ *mālus* (apple tree) |
| Reducing unstressed vowels | Keep all vowels clear | No schwa /ə/ in Latin |
| English R | Use trilled /r/ | Tongue-tip trill like Italian or Spanish |
| Silent H | H is pronounced /h/ | *homō* = /ˈhɔ.moː/ not /ˈɔ.moː/ |

### 9.2 For Korean Speakers (한국어 화자를 위한 도전)

| Mistake | Correct | Explanation |
|---------|---------|-------------|
| No distinction between /r/ and /l/ | Latin has both | R = trilled /r/; L = lateral /l/. Not Korean ㄹ |
| Adding vowels after final consonants | Keep consonant clusters clean | *rēx* = /reːks/ not /reːkɯsɯ/ |
| Short = Long confusion | Duration matters | Practice holding long vowels noticeably longer |
| /f/ as /p/ | Use labiodental /f/ | Lower lip touches upper teeth for F |
| /w/ confusion | V = /w/ (like English "w") | Round lips and glide: *vīnum* = /ˈwiː.nʊm/ |
| Diphthong reduction | Keep both elements | AE = /ae̯/, two vowels gliding together |
| H omission | Classical H = /h/ | A real breathy sound, not silent |
